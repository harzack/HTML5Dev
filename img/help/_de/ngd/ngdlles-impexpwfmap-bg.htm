Sie haben die Möglichkeit, Workflow-Diagramm-Definitionen in eine Datei auf Ihrem Computer zu exportieren. Auf diese Weise können Sie eine Sicherungskopie dieser Definition erstellen oder die Definition auf einen anderen Content Server verschieben. Nach dem Exportieren eines Workflow-Diagramms können Sie es importieren, um so ein Workflow-Diagramm auf der Grundlage des exportierten Workflow-Diagramms zu erstellen.

Wenn Sie ein Workflow-Diagramm durch Importieren einer Workflow-Diagramm-Definition erstellen möchten, müssen Sie zunächst ein leeres Workflow-Diagramm hinzufügen bzw. ein vorhandenes Workflow-Diagramm verwenden, das nicht gespeichert werden soll. Beim Importieren werden die vorhandenen Workflow-Diagramm-Daten überschrieben. An deren Stelle tritt die Definition aus der importierten Datei.

Beim Exportieren eines Workflow-Diagramms wird die Workflow-Diagramm-Definition als Datei (mit der Dateierweiterung .map) auf Ihrem Computer gespeichert. Anlagen werden nicht zusammen mit den Workflow-Diagrammen exportiert. Wenn das Workflow-Diagramm Anlagen enthält, die mit dem Diagramm gespeichert werden sollen, müssen Sie sie separat speichern.
