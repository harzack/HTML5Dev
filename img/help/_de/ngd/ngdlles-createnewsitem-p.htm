So erstellen Sie Nachrichtenobjekte:

  1. Klicken Sie mit der rechten Maustaste auf den Kanal, an dem das Nachrichtenobjekt erstellt werden soll, und klicken Sie anschließend auf Nachrichten erstellen.

  2. Geben Sie auf der Seite Hinzufügen: Nachrichten im Feld Name einen Namen für die Nachrichten ein.

  3. Wählen Sie mithilfe der Listen für Monat, Tag, Jahr und Uhrzeit ein Gültig ab- und ein Ablaufdatum aus.

  4. Geben Sie im Feld Schlagzeile eine Schlagzeile für das Nachrichtenobjekt ein.

     

    Hinweis:
    Wenn Sie das Feld Schlagzeile leer lassen, wird der Text im Feld Name als Schlagzeile verwendet.

     

  5. Geben Sie im Feld Nachrichtentext den Text Ihres Nachrichtenobjekts ein.

  6. In der folgenden Tabelle wird beschrieben, wie weitere Aufgaben zur Definition des Nachrichtenobjekts durchgeführt werden können.

    Aufgabe Aktion
    Bild zum Nachrichtenobjekt hinzufügen Klicken Sie im Bereich Bild auf die Schaltfläche Content Server durchsuchen, gehen Sie zu dem Bild, das Sie dem Nachrichtenobjekt hinzufügen möchten, und klicken Sie dann auf die Verknüpfung Auswählen.
    Beschreibung des Bildes hinzufügen (für Benutzer, die das Bild nicht sehen können) Geben Sie im Feld ALT-Text eine Beschreibung ein.
    Anlage zum Nachrichtenobjekt hinzufügen Klicken Sie im Bereich Content Server-Anlage hinzufügen auf Content Server durchsuchen, gehen Sie zu dem Objekt, das Sie dem Nachrichtenobjekt hinzufügen möchten, und klicken Sie dann auf die Verknüpfung Auswählen.
    Nachrichtenobjekt eine Datei von Ihrem Computer hinzufügen Klicken Sie im BereichDesktop-Anlage hinzufügen auf Durchsuchen, gehen Sie zu dem Objekt, und klicken Sie auf Öffnen.

  7. Klicken Sie auf Hinzufügen.

 

Tipp:
Sie können Nachrichten auch hinzufügen, indem Sie mit der rechten Maustaste auf ein Objekt klicken und anschließend Nachrichten erstellen wählen. Das Objekt wird standardmäßig als Anlage zum Nachrichtenobjekt hinzugefügt.
