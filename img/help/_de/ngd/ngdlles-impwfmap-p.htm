So importieren Sie Workflow-Diagramme:

  1. Fügen Sie ein Workflow-Diagramm hinzu.

  2. Klicken Sie mit der rechten Maustaste auf das Workflow-Diagramm und anschließend auf Importieren.

  3. Klicken Sie auf Durchsuchen, wählen Sie ein exportiertes Workflow-Diagramm aus, und öffnen Sie es.

  4. Klicken Sie auf eine der folgenden Schaltflächen:

    • Bearbeiter beibehalten

    • Bearbeiter zurücksetzen

  5. Klicken Sie auf Importieren.
