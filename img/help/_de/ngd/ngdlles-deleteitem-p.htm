So löschen Sie ein Objekt:

  1. Klicken Sie mit der rechten Maustaste auf das zu löschende Objekt und anschließend auf Löschen.

  2. Klicken Sie im Dialogfeld Objekt löschen auf Ja.

 

Hinweis:
Wenn Sie das Kontextmenü einer Fassung oder Verknüpfung verwenden, um das Originalobjekt zu löschen, müssen Sie im Dialogfeld Löschen auf Löschen und anschließend im Dialogfeld Objekt wird gelöscht aus auf OK klicken.

 
 

Tipp:
Im Listenfenster können Sie die Objekte auswählen, die Sie löschen möchten, und anschließend auf Delete drücken.
