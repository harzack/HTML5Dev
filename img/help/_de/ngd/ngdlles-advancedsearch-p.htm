So führen Sie eine erweiterte Suche durch:

  1. Klicken Sie im Menü bzw. auf der Registerkarte Enterprise Connect auf Suchen und anschließend auf Content Server durchsuchen.

     

    Tipp:
    Sie können auch auf die Seite Erweiterte Content Server-Suche zugreifen, indem Sie in Windows Explorer oder Microsoft Outlook in der Enterprise Connect-Symbolleiste auf Erweiterte Suche klicken. Zusätzlich können Sie auf „Erweiterte Suche“ auch über das Suchfeld im SchnellZugriff-Bereich sowie über die Dialogfelder Kopieren/Verschieben nach Enterprise Connect, Enterprise Connect Öffnen und Enterprise Connect Speichern unter zugreifen. Die Schaltfläche ist nur verfügbar für Containertypen, die Sie durchsuchen können, und für Container, für die Sie die Berechtigung zum Durchsuchen besitzen. Die Erweiterte Suche ist z. B. nicht verfügbar für das Enterprise Connect-Stammobjekt, für Suchergebnisse, für die Ordner Favoriten und Persönliche und für nicht physische Container, wie z. B. virtuelle Ordner.

     

  2. Mithilfe der Informationen in der folgenden Tabelle können Sie Suchkriterien festlegen.

     

    Hinweis:
    Je nachdem, welche Content Server-Module Ihnen zur Verfügung stehen, sind weitere Optionen verfügbar. Weitere Informationen zum Suchen in Content Server finden Sie unter Content Server durchsuchen - OpenText Content Server Online-Benutzerhilfe (LLESWBB-H-UGD).

     

    Zweck Vorgehensweise
    Legen Sie fest, wo die Suche durchgeführt werden soll Klicken Sie im Bereich Zu Suchformular hinzufügen auf Speicherort. Klicken Sie anschließend im Bereich Speicherort auf Content Server durchsuchen, und wählen Sie das Verzeichnis aus, das durchsucht werden soll.
     

    Hinweise:

    • Wenn Sie nicht festlegen, wo die Suche ausgeführt werden soll, durchsucht Enterprise Connect das gesamte Content Server-Repository.

      Die aktuellen Suchkriterien werden gelöscht, sobald Sie das Dialogfeld Content Server durchsuchen schließen.

    • Bei erweiterten Suchvorgängen in Outlook werden die von Ihnen mit der Feldauswahl gewählten zusätzlichen Spalten immer angezeigt, wenn Sie einen oder mehrere Container desselben Typs durchsuchen. Wenn Sie eine erweiterte Suche in zwei oder mehr Containern unterschiedlichen Typs durchführen, werden nur die Standardspalten angezeigt.

     
     

    Tipp:
    Sie können Ihre Suche weiter definieren, indem Sie auf Duplizieren klicken, den entsprechenden Booleschen Operator aus der Liste auswählen und dann weitere Suchverzeichnisse angeben.

     
    Legen Sie fest, ob die Suche alle Wörter, beliebige Wörter oder die exakte Wortfolge umfassen soll. Klicken Sie im Bereich Gesamter Text eine Option in der Liste Suchen nach, und geben Sie dann den Suchbegriff ein.
    Eine komplexe Abfrage durchführen Klicken Sie im Bereich Gesamter Text in der Liste Suchen nach auf Komplexe Abfrage, und geben Sie dann den Suchbegriff ein. Weitere Informationen finden Sie unter Mithilfe komplexer Abfragen suchen.
    Einen Modifikator zum Erweitern der Suche festlegen Klicken Sie im Bereich Gesamter Text eine Option in der Liste Modifikator, und geben Sie dann den Suchbegriff ein.
    Festlegen, ob die Suche auf den Inhalt des Objekts, die Metadaten oder beides begrenzt werden soll Klicken Sie im Bereich Gesamter Text eine Option in der Liste Innerhalb von, und geben Sie dann den Suchbegriff ein.
    Suche mithilfe eines Satzes oder einer Phrase Klicken Sie im Bereich Zu Suchformular hinzufügen auf Natürlichsprachliche Abfrage. Geben Sie den gewünschten Suchbegriff ein, und wählen Sie dann die entsprechende Option aus der Liste Modus. Weitere Informationen zur natürlichsprachlichen Abfrage finden Sie unter Mit natürlichsprachlichen Abfragen suchen.
    Eine Suche nach Systemattributen durchführen Klicken Sie im Bereich Zu Suchformular hinzufügen auf Systemattribute. Klicken Sie in der Liste Zusätzliche Attribute auf das Attribut, das in der Suche enthalten sein soll, und legen Sie anschließend die zu suchenden Attributwerte fest. Weitere Informationen zur Verwendung von Systemattributen finden Sie unter Mit Kategorien und Attributen arbeiten.
    Eine Kategoriesuche durchführen Klicken Sie im Bereich Zu Suchformular hinzufügen auf Kategorien, und wählen Sie die Kategorie aus, die Sie im Dialogfeld Kategorie auswählen verwenden möchten. Geben Sie im Bereich Kategorie Werte für die Kategorieattribute an, die Sie verwenden möchten. Weitere Informationen zum Verwenden von Kategorien finden Sie unter Mit Kategorien und Attributen arbeiten.
    Eine Classifications-Suche durchführen Klicken Sie im Bereich Zu Suchformular hinzufügen auf Klassifikationen. Klicken Sie im Bereich Classifications auf Classifications durchsuchen, um die Classifications zu finden, die Sie durchsuchen möchten, oder klicken Sie auf Meine Favoriten, um die entsprechenden Classifications aus Ihrem Favoritenordner auszuwählen. Aktivieren Sie das Kontrollkästchen Unterklassifikationen einschließen, wenn Unterklassifikationen bei der Suche eingeschlossen werden sollen. Das Einschließen von Unterklassifikationen ermöglicht die Suche nach den ausgewählten Classifications sowie nach allen ihren untergeordneten Klassifikationen. Weitere Informationen zur Verwendung von Classifications finden Sie unter Klassifikationen verwenden.
     

    Hinweis:
    Diese Option ist nur verfügbar, wenn Classifications in Ihrer Umgebung installiert wurden.

     
    Nach E-Mails suchen (einfache Suche) Klicken Sie im Bereich Zu Suchformular hinzufügen auf E-Mail. Geben Sie Suchkriterien an, wie z. B. OTEmailSubject, OTEmailReceivedDate und OTEmailSentDate. Im Feld Suchen nach können Sie auch Namen von E-Mail-Absendern und Empfängern angeben.
     

    Hinweise:

    • Trennen Sie mehrere Einträge von Suchkriterien im Feld Suchen nach jeweils mit einem Semikolon.

    • Suchen mit Platzhaltern werden nicht unterstützt.

     
    Erweiterte E-Mail-Suche durchführen Klicken Sie im Bereich Zu Suchformular hinzufügen auf E-Mail (Erweitert). Klicken Sie auf die Liste Zusätzliche Attribute, um die Spalten auszuwählen, die Sie als Suchkriterien verwenden möchten, und geben Sie dann die entsprechenden Werte an.
    Nach physischen Objekttypen suchen Klicken Sie im Bereich Erweiterte Optionen auf Physische Objekttypen. Klicken Sie im Bereich Physische Objekttypen auf die Liste Physische Objekttypen auswählen und anschließend auf Physische Objekttypen durchsuchen. Klicken Sie auf der Seite Physische Objekttypen durchsuchen auf Auswählen, um den entsprechenden Objekttyp auszuwählen.
     

    Hinweis:
    Diese Option ist nur verfügbar, wenn physische Objekte in Ihrer Umgebung installiert wurden.

     
    Nach physischen Objektattributen suchen Klicken Sie im Bereich Zu Suchformular hinzufügen auf Attribute physischer Objekte. Geben Sie im Bereich Attribute physischer Objekte die Attribute an, nach denen Sie suchen möchten. Die Liste Ursprünglicher Lagerort wird standardmäßig bereitgestellt. Diese kann jedoch bei Bedarf entfernt werden. Klicken Sie auf die Liste Zusätzliche Attribute, um weitere Attribute zur Suche hinzuzufügen.
     

    Hinweis:
    Diese Option ist nur verfügbar, wenn physische Objekte in Ihrer Umgebung installiert wurden.

     
    Records Management-Attribute durchsuchen Klicken Sie im Bereich Zu Suchformular hinzufügen auf Records Management-Attribute. Klicken Sie im Bereich Records Management-Attribute auf die Liste Zusätzliche Attribute, und wählen und definieren Sie die Attribute, nach denen Sie suchen möchten.
     

    Hinweis:
    Diese Option ist nur verfügbar, wenn Records Management in Ihrer Umgebung installiert wurden.

     
    So führen Sie mithilfe einer Vorlage einen Suchvorgang durch Klicken Sie in der Liste Dieses Suchformular verwenden auf die Vorlage, die Sie verwenden möchten. Die Seite Suchen wird geöffnet, auf der die in der von Ihnen gewählten Vorlage angegebenen Suchkriterien angezeigt werden. Die Suchkriterien können bei Bedarf bearbeitet werden.
    Zusätzliche Kriterien hinzufügen Sofern verfügbar, klicken Sie auf Duplizieren und anschließend in der Liste auf einen Booleschen Operator. Legen Sie dann die zusätzlichen Kriterien fest.
    Kriterien entfernen Sofern verfügbar, klicken Sie für alle zu entfernenden Kriterien auf Entfernen .
    Eine Option aus dem Suchformular entfernen Klicken Sie für die zu entfernende Option auf Komponente entfernen .
    Alle Suchkriterien entfernen Klicken Sie auf Löschen.

  3. Klicken Sie auf Suchen.

 

Hinweise:

  • Wenn Sie einen Datumsbereich für Systemattribute oder benutzerdefinierte Attribute innerhalb einer Kategorie festlegen, können Sie das Von- oder das Bis-Datum unausgefüllt lassen. Wenn Sie in den Listen Von kein Datum angeben, durchsucht Content Server den gesamten Datumsbereich vor dem in den Feldern Bis angegebenen Datum. Sind die Listenfelder Bis leer, durchsucht Content Server den Datumsbereich vom Datum in den Listenfeldern Von bis zum aktuellen Datum.

  • Bei einigen Objekten in der Suchergebnisliste ist im Kontextmenü nur die Option Im Browser starten verfügbar. Dies ist der Fall, wenn Sie keine Zugriffsberechtigung auf den übergeordneten Ordner des zurückgegebenen Objekts haben, oder wenn das Volume nicht mit Enterprise Connect angezeigt werden kann. Klicken Sie auf Im Browser starten, wenn Sie mit dem ausgewählten Objekt in einer separaten Browserinstanz arbeiten möchten. Dieses Verhalten gilt auch für Enterprise Connect-Suchvorgänge und Schnellsuchen.

  • Je nach Umgebung können Sie möglicherweise Anzeigeoptionen für die Suchergebnisse festlegen, indem Sie auf die Verknüpfung Anzeigeoptionen bearbeiten klicken. Weitere Informationen zu den verfügbaren Optionen auf der Seite für Anzeigeoptionen finden Sie unter Anzeigeoptionen festlegen.
