Mit Berechtigungen können Sie festlegen, welche Benutzer oder Gruppen ein Objekt sehen, dessen Attribute ändern oder ein Objekt bearbeiten können. So können Sie z. B. Ihrem Assistenten die Berechtigung erteilen, Dokumente zu bearbeiten, die Sie zu einem Ordner oder Projekt hinzufügen.

Content Server legt anhand des Erstellungsortes des Objekts automatisch Berechtigungen für das Objekt fest. Diese Einstellungen können Sie aber auf der Seite Berechtigungen ändern. Sie können auch Berechtigungen für in Kanälen, Diskussionen oder Aufgabenlisten gespeicherte Objekte zuweisen oder bearbeiten, indem Sie mit der rechten Maustaste auf das Objekt klicken und dann auf Berechtigungen klicken.

Berechtigungen für Benutzer werden in einer Zugriffsliste gespeichert. Beim Erstellen eines Objekts werden der Zugriffsliste automatisch die folgenden Standardeinträge hinzugefügt:

  • Besitzer: Benutzer, der das Objekt erstellt hat

  • Besitzergruppe: Arbeitsgruppe des Objektbesitzers

  • Öffentlicher Zugriff: alle Benutzer

Die Standardeinträge können zwar nicht gelöscht werden, aber Sie können die zugehörigen Berechtigungen ändern.

Diese Funktion steht Ihnen nur zur Verfügung, wenn Sie über die entsprechenden Berechtigungen verfügen.
