Wenn Sie über die entsprechenden Berechtigungen verfügen, können Sie von Ihnen oder Ihrem Systemverwalter erstellte, benutzerdefinierte Ansichtssuchformulare dazu verwenden, auf der Content Server-Web-Benutzerschnittstelle Suchen in Enterprise Connect durchzuführen. Weitere Informationen zum Erstellen und Verwenden von benutzerdefinierten Ansichtssuchformularen finden Sie unter Mit gespeicherten Abfragen arbeiten.

 

Wichtig:
Diese Funktion ist nur in Content Server 10, Update 10 oder höher, verfügbar.

 

Wenn Sie auf einen Ordner zugreifen, der ein benutzerdefiniertes Ansichtssuchformular enthält, ist die Schaltfläche Benutzerdefinierte Suche in der Enterprise Connect-Symbolleiste verfügbar. Je nachdem, welche Version von Content Server Sie verwenden, können Sie auf mehrere benutzerdefinierte Ansichtssuchen zugreifen. Diese Funktion ist in Outlook und Explorer sowie in den Dialogfeldern Enterprise Connect Speichern unter, Enterprise Connect Kopieren/Verschieben und Enterprise Connect Öffnen verfügbar. Wenn Sie eine benutzerdefinierte Ansichtssuche von den Dialogfeldern Enterprise Connect Speichern unter und Enterprise Connect Kopieren/Verschieben durchführen, werden Ordner zurückgegeben. Wenn Sie eine benutzerdefinierte Ansichtssuche von dem Dialogfeld Enterprise Connect Öffnen aus durchführen, werden Ordner und einzelne Objekte zurückgegeben.
