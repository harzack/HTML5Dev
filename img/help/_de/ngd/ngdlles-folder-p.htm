So fügen Sie einen Ordner oder E-Mail-Ordner hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ordner oder den Arbeitsbereich, zu dem Sie den Ordner hinzufügen möchten, klicken Sie dann auf Objekt hinzufügen und anschließend auf eine der folgenden Optionen:

    • Ordner

    • E-Mail-Ordner

  2. Geben Sie auf der Seite Ordner hinzufügen oder E-Mail-Ordner hinzufügen im Feld Name einen Namen für den Ordner ein.

  3. Klicken Sie optional im Feld Erstellen in auf Content Server durchsuchen, navigieren Sie zum Ablageort, in dem der Ordner erstellt werden soll, und klicken Sie dann auf die Verknüpfung Auswählen für diesen Ablageort.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Klicken Sie auf Hinzufügen.
