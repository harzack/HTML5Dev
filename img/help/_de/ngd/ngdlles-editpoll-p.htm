So bearbeiten Sie Umfragen:

  1. Klicken Sie mit der rechten Maustaste auf die zu bearbeitende Umfrage, und klicken Sie anschließend auf Bearbeiten.

  2. Geben Sie auf der Seite Umfragekommentare anzeigen im Feld Name einen Namen für die Umfrage ein.

  3. Legen Sie mithilfe der Listen unter Gültig ab und Ablaufdatum ein Start- und ein Enddatum für die Umfrage fest.

  4. Gehen Sie im Bereich Umfrage wie folgt vor:

    • Geben Sie im Feld Frage eine Frage ein.

    • Geben Sie in die Felder Mögliche Antworten verschiedene Antwortmöglichkeiten ein.

    • Wählen Sie unter Antwortauswahl für jede Frage eine der folgenden Optionen, um die Methode für die Stimmabgabe festzulegen:

      • Optionsschaltfläche

      • Kontrollkästchen

  5. In der folgenden Tabelle sind weitere Methoden zur Definition einer Umfrage beschrieben.

    Gewünschte Aktion Vorgehensweise
    Anweisungen zur Vorgehensweise bei der Stimmabgabe hinzufügen Geben Sie im Feld Anweisungen die entsprechenden Anweisungen ein.
    Fragen und Antworten hinzufügen bzw. löschen Klicken Sie im Bereich Umfrage auf Hinzufügen bzw. Entfernen
    ( / ).
    Umfrageteilnehmern erlauben, Ergebnisse anzuzeigen bzw. Kommentare hinzuzufügen Aktivieren Sie im Bereich Optionen die Option Ergebnisanzeige vor dem Wählen zulassen bzw. Anonyme Kommentare zulassen.
    Umfrage mit Bild versehen Klicken Sie im Bereich Bild auf Content Server durchsuchen, suchen Sie das gewünschte Bild, und klicken Sie auf die Verknüpfung Auswählen für das Bild.
    ALT-Text für Bild hinzufügen Geben Sie im Feld ALT-Text den gewünschten Text ein.

  6. Klicken Sie auf Weiterleiten.
