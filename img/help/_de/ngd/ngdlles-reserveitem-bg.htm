Durch Reservieren von Objekten wird sichergestellt, dass Sie an der aktuellen Version arbeiten und andere Benutzer das betreffende Objekt nicht ändern können, während Sie daran arbeiten. Wenn Sie das Objekt fertig bearbeitet haben, sollten Sie dessen Reservierung aufheben, damit es auch von anderen Benutzern wieder bearbeitet werden kann.

Wenn Sie ein Objekt reservieren, wird das Symbol Reserviert angezeigt, und das Objekt wird in Ihrem Ordner Reservierte Objekte aufgeführt.

Sie können Dokumente, zusammengesetzte Dokumente und deren Äquivalente reservieren. Beachten Sie, dass Sie nur Objekte reservieren können, die nicht bereits von einem anderen Benutzer reserviert wurden.

Diese Funktion steht Ihnen nur zur Verfügung, wenn Sie über die entsprechenden Berechtigungen verfügen.
