Sie können ein Nachrichtenobjekt für einen Kanal erstellen, über den Sie das Objekt für andere Benutzer als Nachricht veröffentlichen können. Ein solches Nachrichtenobjekt könnte z. B. eine Liste mit neuen Kunden sein.

Sie können jedem Kanal auf der Nachrichtenseite im persönlichen Arbeitsbereich Nachrichten hinzufügen.

Ein Nachrichtenobjekt besteht aus einer Schlagzeile und dem Nachrichtentext und verfügt sowohl über ein Gültig-ab- als auch über ein Ablaufdatum, um zu gewährleisten, dass die angezeigten Nachrichten immer aktuell und gültig sind. Nachrichtenobjekte können darüber hinaus auch ein Bild und Anlagen enthalten. Das Symbol Anlage  zeigt an, dass ein Nachrichtenobjekt eine Anlage enthält.

Nachrichtenobjekte können Sie über die Seite Zusammenfassung des Nachrichten-Kanals öffnen und bearbeiten, in dem sie sich befinden.

 

Hinweis:
Nachrichtenobjekte können über mehrere Anlagen verfügen. Wenn Sie ein Nachrichtenobjekt neu erstellen, können Sie eine Anlage aus Content Server und eine Anlage von Ihrem Computer hinzufügen. Weitere Anlagen können dann hinzugefügt werden, indem Sie das Nachrichtenobjekt bearbeiten.

 

Beim Hinzufügen von Nachrichten zu einem Nachrichten-Kanal können Sie eine Beschreibung des Nachrichtenobjekts eingeben, dessen Kategorien und Attribute ändern und das Nachrichtenobjekt einem anderen Container hinzufügen.
