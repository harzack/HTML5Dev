So heben Sie die Reservierung eines Objekts auf:

  1. Klicken Sie mit der rechten Maustaste auf das Objekt, dessen Reservierung Sie aufheben möchten, und klicken Sie dann auf Reservierung aufheben.

  2. Um eine neue Version des Objekts hinzuzufügen, aktiveren Sie auf der Seite Reservierung aufheben das Kontrollkästchen Neue Version hinzufügen, und klicken Sie anschließend auf die Schaltfläche Durchsuchen, um die neue Version festzulegen.

  3. Klicken Sie auf Senden.

     

    Tipp:
    Sie können die Reservierung mehrerer Objekte aufheben, indem Sie die entsprechenden Objekte auswählen, mit der rechten Maustaste darauf klicken und Reservierung aufheben wählen.
