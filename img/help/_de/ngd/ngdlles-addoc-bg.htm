Ein Dokument ist eine beliebige elektronische Datei, die Sie von Ihrem Computer in die Content Server-Datenbank hochladen. Beispiele für Dokumente sind Arbeitsblätter aus Tabellenkalkulationen, Memos, HTML-Seiten, Bilder und Audiodateien. Unter Verwendung von Funktionen arbeiten Sie mit Dokumenten, indem Sie sie dem Content Server hinzufügen, ihren Inhalt bearbeiten, sie zum Anzeigen öffnen sowie Versionen und Fassungen verwalten.

 

Hinweis:
Sie müssen über entsprechende Berechtigungen für die Dokumente verfügen, die Sie hinzufügen, um diese Funktionen ausführen zu können. Weitere Informationen zum Festlegen von Berechtigungen finden Sie unter Berechtigungen festlegen.

 

Dokumente hinzufügen

Sie können Dokumente zu Content Server hinzufügen, indem Sie eine bereits auf Ihrem Computer vorhandenen Dokumentdatei hochladen. Sie können auch ein neues Dokument erstellen.

Wenn Sie über die erforderlichen Berechtigungen verfügen, können Sie ein Dokument als Haupt- oder Nebenversion des Originals hinzufügen. Wenn Sie nicht über die Berechtigung verfügen, ein Dokument als Version hinzuzufügen, wird die Option, ein Dokument als Haupt- oder Nebenversion hinzuzufügen, nicht angezeigt.

Doppelte Dokumente

Wenn Sie ein Dokument zu Content Server hinzufügen und bereits ein Dokument mit demselben Namen im Zielverzeichnis vorhanden ist, können Sie angeben, wie das doppelte Dokument behandelt werden soll. Sie können z. B. ein neues Dokument als Version des bereits im Zielverzeichnis vorhandenen Dokuments erstellen. Wenn Sie mehrere Dokumente zu Content Server hinzufügen, können Sie festlegen, dass alle Duplikate in dem Stapel als neue Versionen von vorhandenen Dokumenten hinzugefügt werden. Sie können auch festlegen, dass nur neue und keine doppelten Dokumente zu Content Server hinzugefügt werden.

Die Seite Dokumentüberblick verwenden

Die Seite Dokumentüberblick bietet Informationen zum Dokument, einschließlich einer Beschreibung und zugehöriger Informationen. Auf der Seite Dokumentüberblick können Sie verschiedene Funktionen am Dokument durchführen, wie z. B. Bearbeiten, Herunterladen, Öffnen oder Hinzufügen einer Version, sofern Sie über die Berechtigungen für das Durchführen dieser Aufgaben verfügen.

Dokumente bearbeiten

Ein Dokument kann auf mehrere verschiedene Weisen bearbeitet werden, wie z. B. mit dem Befehl Bearbeiten. Wenn Sie ein Microsoft Office- oder Adobe Acrobat-Dokument bearbeiten, wird das Dokument in der entsprechenden Anwendung geöffnet. Das Objekt wird in Content Server als reserviert gekennzeichnet, bis Sie die Anwendung schließen. Die aktuelle Version des Dokuments in Content Server wird immer dann aktualisiert, wenn Sie Änderungen speichern.

Ihre Berechtigungen für das Dokument zum Öffnen oder Bearbeiten bestimmen, welche Versionen des Dokuments Ihnen zur Verfügung stehen. Wenn Sie über Berechtigungen zum Ändern eines Dokuments verfügen, können Sie auf die letzte Version eines Dokuments zugreifen. Je nach Status des Dokuments kann dies eine Haupt- oder Nebenversion sein. Wenn Sie nur über eine Leseberechtigung für ein Dokument verfügen, können Sie nur auf die letzte Hauptversion eines Dokuments zugreifen.

 

Hinweis:
Sie können einige Dokumente zum Bearbeiten öffnen, indem Sie mit der rechten Maustaste auf das entsprechende Dokument und anschließend auf Öffnen klicken.

 
 

Tipp:
Je nach Konfiguration können Sie ein Microsoft Office-Dokument möglicherweise zur Bearbeitung öffnen, wenn Sie in Enterprise Connect darauf oder auf eine Verknüpfung zu dem Dokument doppelklicken.
