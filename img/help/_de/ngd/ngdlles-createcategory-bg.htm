Kategorien dienen zum Zusammenfassen von zueinander gehörenden Attributen und definieren deren Typ, die Reihenfolge und die Werte, die die Benutzer für sie eingeben können. Durch Erstellen benutzerdefinierter Kategorien können Sie den Objekten in der Content Server-Datenbank Metadaten hinzufügen, die für Ihr Unternehmen relevant sind.

Zum Erstellen und Bearbeiten von Kategorien müssen die folgenden Voraussetzungen erfüllt sein:

  • Sie müssen die Berechtigung zur Erstellung von Objekten für Kategorien besitzen.

  • Sie müssen die Berechtigung Objekte hinzufügen für den Container bzw. Arbeitsbereich haben, in dem Sie die Kategorie speichern möchten.

Sie können eine Kategorie nahezu jedem Container oder Arbeitsbereich hinzufügen, für den Sie die Berechtigung zum Hinzufügen von Objekten besitzen. Meistens werden die Kategorien aber in einem speziellen Kategoriedatenbereich gespeichert, sodass sie für alle Benutzer verfügbar sind. Wenn Sie eine Kategorie einem anderen Ablageort hinzufügen, z. B. einem Ordner, können Sie sie auch nur für eine kleinere Teilgruppe von Benutzern verfügbar machen und eindeutige Berechtigungen dafür festlegen.

 

Hinweis:
Auf den Kategoriedatenbereich können Sie nur zugreifen, wenn Sie die Berechtigung Systemverwaltungsberechtigungen besitzen. Mit dieser Berechtigung können Sie dem Datenbereich auch Kategorie-Ordner hinzufügen, denen Sie die einzelnen Kategorien zuordnen können. Mit der Berechtigung Systemverwaltungsberechtigungen können Sie auch dem Arbeitsbereich Kategorien hinzufügen.

 

Auch wenn Sie nur ein einzelnes benutzerdefiniertes Attribut hinzufügen, müssen Sie eine Kategorie hinzufügen, in der das Attribut gespeichert werden kann. Nach dem Erstellen einer Kategorie können Sie deren Attribute auf unterschiedliche Weise bearbeiten. So können Sie z. B. das Attribut sperren bzw. dessen Sperre aufheben, die Anzahl der Zeilen ändern und Attribute hinzufügen oder entfernen. Sie können auch die Standardwerte, die Reihenfolge (Sequenz) und das Layout der Attribute in einer Kategorie ändern.

Nachdem Sie eine Kategorie hinzugefügt haben, können Sie sie mithilfe der vielen verfügbaren Attributtypen konfigurieren. Sie wählen dazu die Attributtypen aus, die die Kategorie enthalten soll, definieren deren Eigenschaften und legen deren Reihenfolge (Sequenz) und Layout fest. Sie müssen jeder Kategorie mindestens ein Attribut hinzufügen, eine Höchstbegrenzung für die Anzahl der Attribute gibt es aber nicht. Weitere Informationen zum Arbeiten mit Attributen finden Sie unter Mit Kategorien arbeiten.
