So konfigurieren Sie Optionen für die Inhaltsspeicherung:

  1. Klicken Sie im Enterprise Connect-Baum auf einen Content Server-Container und dann auf Persönlich und Meine Einstellungen.

  2. Führen Sie im Bereich Konfiguration der Metadatenspeicherung in der Liste Beim Hinzufügen von Inhalten nach Metadaten fragen einen der folgenden Schritte aus, um festzulegen, wann Benutzer aufgefordert werden, Metadaten über die Objekte bereitzustellen, die sie Content Server hinzufügen:

    • Bei Auswahl von Nur wenn erforderlich werden Benutzer nur dann aufgefordert, Metadaten einzugeben, wenn die Metadaten als für den Content Server-Zielordner erforderlich gekennzeichnet sind. Dies ist die Standardeinstellung.

    • Bei Auswahl von Immer werden Sie aufgefordert, für alle Objekte Metadaten bereitzustellen, die Sie Content Server hinzufügen.

  3. Klicken Sie auf Senden, um die Änderungen anzuwenden.

     

    Tipp:
    Klicken Sie auf Zurücksetzen, um die für die E-Mail-Benennungskonventionen zuletzt gespeicherten Werte wiederherzustellen. Klicken Sie auf Standardwerte verwenden, um zu den vom Systemverwalter bereitgestellten Einstellungen zurückzukehren.
