Sie können E-Mails senden, die Kopien von Objekten als Anlagen oder Links zu den Objekten enthalten. Je nach den Administrationseinstellungen können Sie über den Link für das Dokument die Seite Überblick oder Eigenschaften und für E-Mails und zusammengesetzte E-Mails die Seite Überblick öffnen. Über Links können auch Objekte geöffnet oder heruntergeladen werden. Wenn Sie einem Objekt einen Link anfügen, muss der Empfänger über die Berechtigungen zum Anzeigen des Objekts in Content Server verfügen.

 

Hinweis:
Für einige Objekttypen können nur Links und keine Kopien als Anlagen verwendet werden.

 
 

Tipp:
Informationen zum Versenden von Links zu Dokumenten per Outlook finden Sie unter OpenText Enterprise Connect Plug-in for Content Server - Benutzerhandbuch (NGDCORE-UGD).
