Beim Erstellen einer Fassung wird eine Referenz zu einer bestimmten Ausgabe oder Revision eines zusammengesetzten Dokuments oder zu einer Dokumentversion angelegt, die von anderen Benutzern nicht bearbeitet oder gelöscht werden kann.

Mithilfe von Fassungen können Sie bestimmte Versionen eines Objekts besser sichtbar machen und den Zugriff erleichtern. Wie Verknüpfungen verweisen Fassungen auf bestimmte, normalerweise ältere Versionen eines Objekts. Wenn Sie eine alte Version eines Objekts besitzen, die noch gebraucht wird, können Sie eine Fassung dieser Version erstellen und diese Fassung in einem beliebigen Container bzw. Arbeitsbereich speichern. Auf diese Weise verhindern Sie, dass die Version, auf die die Fassung verweist, geändert oder gelöscht wird.

Fassungen werden als separate Objekte behandelt, d. h., sie sind nicht mit dem Objekt (bzw. der Version des Objekts) verknüpft, auf dessen (bzw. deren) Basis sie erstellt wurden. Fassungen ermöglichen es, die Versionen, auf die sie verweisen, leichter zu finden und damit zu arbeiten, da sie in den entsprechenden Containern gespeichert werden können.

Beim Erstellen einer Fassung wird das Symbol Fassung für das Objekt angezeigt.

Das Menü Funktionen unterteilt sich bei Fassungen in die folgenden beiden Untermenüs:

  • Das Untermenü Original enthält Funktionen, die Sie gemäß den Ihnen zugewiesenen Berechtigungen am Originalobjekt (der Version, auf die die Fassung verweist) ausführen dürfen. Die Funktionen im Untermenü Original wirken sich nicht auf die Fassung aus.

  • Das Untermenü Fassung enthält Funktionen, die Sie gemäß den Ihnen zugewiesenen Berechtigungen an der Fassung ausführen dürfen. Die Funktionen im Untermenü Fassung wirken sich nicht auf das Originalobjekt aus.
