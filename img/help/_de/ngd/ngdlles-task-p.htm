So fügen Sie Aufgaben hinzu:

  1. Klicken Sie mit der rechten Maustaste auf die Aufgabenliste, zu der die Aufgabe hinzugefügt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf Aufgabe.

  2. Geben Sie auf der Seite Hinzufügen: Aufgabe im Feld Name einen Namen für die Aufgabe ein.

  3. Klicken Sie in der Liste Priorität auf Hoch, Mittel oder Niedrig.

  4. Klicken Sie in der Liste Status auf einen Status.

  5. In der folgenden Tabelle wird beschrieben, wie Sie Aufgaben mithilfe von zusätzlichen Aktionen zuweisen und definieren können.

    Gewünschte Aktion Vorgehensweise
    Aufgabe zuweisen Klicken Sie auf das Symbol Auswählen  , navigieren Sie zum gewünschten Benutzer oder der gewünschten Gruppe, und klicken Sie dann auf die entsprechende Verknüpfung Auswählen.
    Start- und Fälligkeitsdatum für die Aufgabe festlegen Wählen Sie mithilfe der Listen unter Startdatum und Fälligkeitsdatum ein Datum aus.
    Einer Aufgabe einen Meilenstein zuweisen Wählen Sie in der Liste Meilenstein einen Meilenstein aus. Informationen zum Erstellen von Meilensteinen finden Sie unter So fügen Sie Meilensteine hinzu.
    Anweisungen bzw. Kommentare für die Aufgabe hinzufügen Geben Sie die entsprechenden Informationen in den Feldern Anweisungen und Kommentare ein.
    Aufgabe bearbeiten Klicken Sie mit der rechten Maustaste auf eine Aufgabe, und klicken Sie dann auf Bearbeiten. Prüfen Sie die Aufgabe, und klicken Sie anschließend auf Aufgabe aktualisieren.

  6. Klicken Sie auf Hinzufügen.
