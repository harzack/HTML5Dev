So verschieben Sie ein Objekt:

  1. Wählen Sie die Objekte, die Sie verschieben möchten, klicken Sie mit der rechten Maustaste, und wählen Sie Kopieren/Verschieben in Enterprise Connect.

  2. Navigieren Sie im Dialogfeld Enterprise Connect zum Zielverzeichnis, und klicken Sie auf Verschieben.

  3. Je nachdem, wie Ihr Systemverwalter Ihre Umgebung konfiguriert hat, werden Sie aufgefordert, einen der folgenden Schritte auszuführen:

    Zweck Vorgehensweise
    Kategorieoptionen definieren Führen Sie im Bereich Kategorien einen der folgenden Schritte aus:
    •

    Klicken Sie auf Original, wenn Sie die den Objekten ursprünglich zugewiesenen Kategorien beibehalten möchten. Die zugewiesenen Kategorien werden im Feld Kategorien angezeigt. Klicken Sie auf Bearbeiten, um die Kategorien gegebenenfalls zu bearbeiten.

     

    Hinweis:
    Wenn Sie bei mehreren ausgewählten Objekten im Dialogfeld Verschieben auf Bearbeiten klicken, um die zugeordneten Kategorien zu bearbeiten, wird das Dialogfeld Verschieben für das nächste ausgewählte Objekt aufgerufen, und Sie können die Kategorien dieses Objekts bearbeiten. Wenn Sie jedoch nicht auf Bearbeiten klicken, werden die von Ihnen angegebenen Kategorien auf alle verbleibenden ausgewählten Objekte angewendet.

     
    •

    Klicken Sie auf Ziel, wenn Sie Kategorien des neuen Ablageorts auf die Objekte anwenden möchten. Um die dem Ziel zugewiesenen Kategorien anzuzeigen, klicken Sie im Dialogfeld Kategorien bearbeiten auf Bearbeiten und anschließend auf Fertig. Sie können die Kategorien gegebenenfalls bearbeiten.

    •

    Klicken Sie auf Zusammengeführt, um die Kategorien sowohl des ursprünglichen als auch des neuen Ablageorts auf das Objekt oder die Objekte anzuwenden. Um die dem Ziel zugewiesenen Kategorien anzuzeigen, klicken Sie im Dialogfeld Kategorien bearbeiten auf Bearbeiten und anschließend auf Fertig. Sie können die Kategorien gegebenenfalls bearbeiten. Wenn Sie mehrere Objekte ausgewählt haben, wird das Dialogfeld Verschieben für die einzelnen Objekte geöffnet. Hier können Sie die Kategorien gegebenenfalls bearbeiten.

    Festlegen (falls zutreffend), ob das Objekt RM-Klassifikationen übernehmen soll Führen Sie im Bereich RM-Klassifikationen einen der folgenden Schritte aus:
    •

    Klicken Sie auf Ja, wenn das Objekt die dem Zielverzeichnis zugewiesenen RM-Klassifikationen übernehmen soll.

    •

    Klicken Sie auf Nein, wenn das Objekt die dem Zielverzeichnis zugewiesenen RM-Klassifikationen nicht übernehmen soll.

  4. Ist im Zielverzeichnis bereits ein Objekt mit demselben Namen vorhanden, führen Sie im Dialogfeld Enterprise Connect einen der folgenden Schritte aus:

    • Klicken Sie auf Ja, wenn Sie im Zielverzeichnis eine neue Instanz des Objekts mit einem automatisch generierten Namen erstellen möchten.

    • Klicken Sie auf Ja zu allen, wenn Sie eventuelle Namenskonflikte mit anderen Objekten in dem zu verschiebenden Batch beheben möchten, indem Sie im Zielverzeichnis jeweils eine neue Instanz des Objekts mit einem automatisch generierten Namen erstellen.

    • Klicken Sie auf Nein, wenn das aktuelle Objekt nicht in das Zielverzeichnis verschoben werden soll. Die Aktion Verschieben wird nach Auswahl dieser Option fortgeführt.

    • Klicken Sie auf Nein für alle, wenn keine doppelten Objekte in das Zielverzeichnis verschoben werden sollen. Die Aktion Verschieben wird nach Auswahl dieser Option fortgeführt.

    • Klicken Sie auf Abbrechen, wenn die Verschiebeaktion abgebrochen werden soll. Objekte, die erfolgreich verschoben wurden, verbleiben im Zielverzeichnis.

     

    Hinweis:
    Wenn Sie Objekte mit Enterprise Connect in Outlook verschieben, sind die Optionen Nein und Nein für alle aufgrund einer Einschränkung bei der Outlook-Funktionalität nicht verfügbar. Diese Optionen sind jedoch verfügbar, wenn Sie Objekte im Dialogfeld Kopieren/Verschieben von Enterprise Connect verschieben. Diese Optionen sind auch im Enterprise Connect-SchnellZugriff-Bereich verfügbar.

     

  5. Im Dialogfeld Verschieben wird der Fortschritt des Befehls Verschieben angezeigt.
