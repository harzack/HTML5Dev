So bearbeiten Sie Zusammenfassungen:

  1. Klicken Sie mit der rechten Maustaste auf das Objekt, für das Sie eine Zusammenfassung erstellen möchten, und klicken Sie anschließend Bewerten.

  2. Klicken Sie auf das Symbol Zusammenfassung .

  3. Bearbeiten Sie die Zusammenfassung auf der Seite Zusammenfassung bearbeiten.

  4. Klicken Sie auf Aktualisieren.

 

Hinweis:
Das Symbol "Zusammenfassung" wird nur dann eingeblendet, wenn der Systemverwalter die Bearbeitung der Zusammenfassungen durch die Benutzer aktiviert hat und wenn Sie die Berechtigung "Ändern" für das Objekt besitzen.
