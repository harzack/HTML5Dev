So erstellen Sie eine Projektvorlage:

  1. Klicken Sie mit der rechten Maustaste auf das Projekt, vom dem Sie eine Vorlage erstellen möchten, und klicken Sie anschließend auf Vorlage erstellen.

  2. Klicken Sie auf der Seite Hinzufügen: Projektvorlage im Bereich Quelle auf das Optionsfeld Projekt, klicken Sie auf die Schaltfläche Content Server durchsuchen, suchen Sie das Projekt, auf dessen Grundlage Sie eine Vorlage erstellen möchten, und klicken Sie dann auf die Verknüpfung Auswählen für das Projekt.

  3. Geben Sie im Feld Name einen Namen für die Vorlage ein.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Klicken Sie auf Hinzufügen.
