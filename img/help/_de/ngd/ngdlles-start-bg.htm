Mit dieser Anwendung können Sie auf die Inhalte der Content Server-Arbeitsbereiche in der OpenText Enterprise Connect-Baumstruktur zugreifen. Abhängig von den auf Ihrem System installierten Optionen können Sie den Zusatz verwenden, wenn Sie mit Anwendungen wie Windows Explorer, Microsoft Outlook oder Microsoft Office arbeiten.

Dieses Hilfesystem enthält spezifische Informationen für Enterprise Connect. In einigen Fällen bezieht sich das Hilfesystem auf Themen, die im Content Server-Haupthilfesystem enthalten sind, da die Funktionsweise in beiden Anwendungen dieselbe ist. In einigen Fällen können Sie direkt über Enterprise Connect-Fenster und -Dialogfelder auf die Content Server-Hilfe zugreifen.

Inhaltsverzeichnis  

  • Auf den Content Server-Ordner zugreifen

  • Inhalte im Browser anzeigen

  • Von der Web-Benutzeroberfläche von Content Server auf Enterprise Connect zugreifen

  • Objekteigenschaften anzeigen

  • Anmeldeinformationen festlegen
