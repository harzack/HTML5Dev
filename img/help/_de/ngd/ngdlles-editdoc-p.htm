So bearbeiten Sie Dokumente:

  1. Führen Sie einen der folgenden Schritte aus, um ein Dokument zur Bearbeitung zu öffnen:

    • Klicken Sie mit der rechten Maustaste auf das Dokument, und klicken Sie dann auf Bearbeiten.

       

      Hinweis:
      Einige Dokumenttypen lassen sich auch bearbeiten, indem Sie mit der rechten Maustaste auf das entsprechende Dokument und dann auf Öffnen klicken.

       

    • Verwenden Sie das Dialogfeld Enterprise Connect Öffnen für die Anwendung.

    • Verwenden Sie das Dialogfeld Öffnen für die native Anwendung.

    • Wählen Sie das geeignete Dokument aus der Dateiliste Zuletzt verwendet in der Anwendung.

    • Doppelklicken Sie in Enterprise Connect auf das Dokument oder auf eine Verknüpfung zu dem Dokument, falls Ihr Systemverwalter diese Option aktiviert hat.

  2. Speichern Sie das Dokument im Menü Enterprise Connect Speichern unter.
