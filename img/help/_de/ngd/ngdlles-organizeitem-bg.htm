Sie können innerhalb von Content Server einzelne Objekte organisieren. So können Sie z. B. Dokumente in einen anderen Ordner verschieben, Objekte in Content Server einfügen, Verknüpfungen erstellen und nicht mehr benötigte Objekte löschen.

Inhaltsverzeichnis  

  • Objekte hinzufügen

  • Verschieben von Objekten

  • Objekte kopieren

  • Favoriten erstellen

  • Mit Verknüpfungen und Links arbeiten

  • Objekte löschen

  • Objekte umbenennen

  • Ordner und E-Mail-Ordner hinzufügen

  • Mit Sammlungen arbeiten
