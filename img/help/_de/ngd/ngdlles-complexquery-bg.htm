Eine komplexe Abfrage ist eine Anweisung, die in Live Query Language (LQL) erstellt wird und die gesuchten Daten genau definiert. Wenn Sie z. B. nach Objekten suchen, die das Wort Bericht enthalten, nicht jedoch das Wort Ausgaben, können Sie die komplexe Abfrage report AND-NOT expense eingeben.

LQL-Schlüsselwörter sind Wörter, denen in der Livelink-Abfragesprache eine bestimmte Bedeutung zugeordnet ist. Wenn Sie diese LQL-Schlüsselwörter entsprechend den LQL-Konventionen einsetzen, können Sie damit komplexe Abfragen erstellen. Bei LQL-Schlüsselwörtern wird nicht zwischen Groß- und Kleinschreibung unterschieden.

LQL bietet eine Alternative zum Festlegen von Attributen für die Suche nach Objekten. Jedes Objekt besitzt bestimmte Attribute mit zusätzlichen Informationen zum jeweiligen Objekt. Standardmäßig durchsucht Content Server die Objektattribute nicht. Sie können die Attribute jedoch nicht nur in einer einfachen Abfrage nutzen, sondern auch mit LQL nach einem bestimmten Attribut oder einer Gruppe von Attributen suchen.

 

Hinweis:
Bei den Attributsnamen muss die Groß-/Kleinschreibung beachtet werden. Eine Liste der verfügbaren Attribute und deren Schreibweise erhalten Sie von Ihrem Administrator.
