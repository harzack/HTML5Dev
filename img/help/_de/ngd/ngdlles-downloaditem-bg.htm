Je nach Konfiguration können Sie Objekte auf Ihren Computer herunterladen und lokal daran arbeiten. Als Best Practice empfiehlt es sich, alle von Ihnen heruntergeladenen Dokumente zu reservieren. So stellen Sie sicher, dass von Ihnen verwendete Objekte nicht von anderen Benutzern bearbeitet werden können. Weitere Informationen zum Vornehmen und Aufheben einer Reservierung finden Sie unter Reservierung von Objekten vornehmen und aufheben. Mit dem Befehl Offline-Version herunterladen von Enterprise Connect können Sie auch offline mit Objekten arbeiten. Weitere Informationen finden Sie unter OpenText Enterprise Connect Plug-in for Content Server - Benutzerhandbuch (NGDCORE-UGD).
