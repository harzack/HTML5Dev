So fügen Sie Dokumente hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Dokument hinzugefügt werden soll, klicken Sie dann auf Objekt hinzufügen und anschließend auf Dokument.

     

    Tipp:
    Sie können auch im Menü Objekt hinzufügen auf Dokument klicken.

     

  2. Klicken Sie im Bereich Dokument der Seite Dokument hinzufügen auf Durchsuchen, um zu der Datei zu navigieren, die Sie hinzufügen möchten, und klicken Sie dann auf Öffnen .

  3. Geben Sie in das Feld Name einen Namen für das Dokument und optional eine Beschreibung ein.

  4. Wählen Sie im Bereich Versionssteuerung einen Versionierungstyp aus, indem Sie auf Standard - Lineare Versionierung oder Erweitert - Versionierung in Haupt-/Nebenversionen klicken.

    Legen Sie optional Einstellungen für Klassifikationen und Kategorien fest.

  5. Klicken Sie im Bereich Erstellen in auf Content Server durchsuchen, um anzugeben, wo das Dokument gespeichert werden soll.

  6. Klicken Sie auf Hinzufügen.

 

Hinweis:
Wenn sich im Zielverzeichnis bereits ein Dokument mit demselben Namen befindet, wird das Dialogfeld Enterprise Connect geöffnet. Wählen Sie eine der folgenden Optionen:

  • Ja, wenn Sie das neue Dokument als eine Version des vorhandenen Dokuments hinzufügen möchten.

  • Ja zu allen, wenn alle doppelten Dokumente zu Content Server hinzugefügt werden sollen.

  • Nein, wenn das Dokument nicht zu Content Server hinzugefügt werden soll.

  • Nein für alle, wenn keine doppelten Dokumente zu Content Server hinzugefügt werden sollen. Daraufhin werden nur neue Dokumente hinzugefügt.

  • Abbrechen, um die Aktion zu stoppen.
