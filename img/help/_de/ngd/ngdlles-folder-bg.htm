Mithilfe von Ordnern können Sie Ihre Arbeit genau so organisieren, wie Sie dies mit den Ordnern auf Ihrem Computer tun. Ordner können fast jeden Objekttyp umfassen, einschließlich Projekte und andere Ordner. Ein Ordner innerhalb eines anderen Ordners wird als Unterordner bezeichnet.

E-Mail-Ordner sind ähnlich konzipiert wie normale Ordner. E-Mail-Ordner umfassen E-Mail-Spalten, mit deren Hilfe Sie E-Mail-Metadaten für alle E-Mails im Ordner anzeigen können. Wenn Sie E-Mails in einem E-Mail-Ordner speichern, werden standardmäßig die folgenden zusätzlichen Informationsspalten bereitgestellt.

  • Von

  • An

  • Betreff

  • Sendedatum

  • Empfangsdatum

Verwenden Sie E-Mail-Ordner, wenn die vorrangig im Ordner gespeicherten Inhalte E-Mail-Nachrichten sind.

Sie können einen Ordner oder E-Mail-Ordner auf allen Arbeitsbereich- oder Container-Seiten hinzufügen, für die Sie über eine Berechtigung zum Hinzufügen von Objekten verfügen. Weitere Informationen zum Verschieben und Kopieren von Objekten finden Sie unter Verschieben von Objekten und Objekte kopieren.

Wenn Sie einen Ordner in die Enterprise Connect-Struktur ziehen und bereits ein Ordner mit demselben Namen vorhanden ist, haben Sie die Möglichkeit, den neuen Ordner mit dem vorhandenen zusammenzuführen.
