So fügen Sie Meilensteine hinzu:

  1. Klicken Sie mit der rechten Maustaste auf die Aufgabenliste, zu der die Aufgabe hinzugefügt werden soll, und klicken Sie auf Objekt hinzufügen und anschließend auf Meilenstein.

  2. Geben Sie auf der Seite Hinzufügen: Meilenstein im Feld Name einen Namen für den Meilenstein ein.

  3. Geben Sie mithilfe der Listen unter Zieldatum das Zieldatum für die Aufgabe an.

    Sie können zudem den Meilenstein beschreiben und allgemeine Einstellungen angeben.

  4. Klicken Sie auf Hinzufügen.
