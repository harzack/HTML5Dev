So greifen Sie von der Web-Benutzeroberfläche von Content Server auf Enterprise Connect zu:

  • Wählen Sie auf der Web-Benutzeroberfläche von Content Server das geeignete Objekt, klicken Sie auf das Menü Funktion, und wählen Sie Enterprise Connect hier öffnen oder den von Ihrem Systemverwalter erstellten benutzerdefinierten Namen.
