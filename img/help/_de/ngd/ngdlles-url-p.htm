So fügen Sie URLs hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein URL hinzugefügt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf URL.

  2. Geben Sie auf der Seite Hinzufügen: URL im Feld Name einen Namen für den URL ein.

  3. Geben Sie im Feld URL den URL ein.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Klicken Sie auf Hinzufügen.

 

Hinweis:
Sie können einen URL bearbeiten, indem Sie auf dessen Eigenschaften zugreifen.
