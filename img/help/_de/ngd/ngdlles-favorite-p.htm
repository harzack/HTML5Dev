So erstellen Sie einen Favoriten:

  • Klicken Sie mit der rechten Maustaste auf das Objekt, für das Sie einen Favoriten erstellen möchten, und klicken Sie anschließend auf Zu Favoriten hinzufügen.

 

Tipp:
Sie können Objekte aus Favoriten entfernen, indem Sie mit der rechten Maustaste auf das betreffende Objekt klicken und den Befehl Favoriten entfernen wählen.
