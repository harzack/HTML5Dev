So fügen Sie Aufgabengruppen hinzu:

  1. Klicken Sie mit der rechten Maustaste auf die Aufgabenliste, zu der die Aufgabengruppe hinzugefügt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf Aufgabengruppe.

  2. Geben Sie auf der Seite Hinzufügen: Aufgabengruppe im Feld Name einen Namen für die Aufgabengruppe ein.

    Klicken Sie optional in der Liste Standard-Meilenstein für Aufgaben auf einen Meilenstein, um der Aufgabengruppe einen Meilenstein zuzuweisen.

  3. Klicken Sie auf Hinzufügen.

 

Hinweis:
Es ist einfacher, einer vorhandenen Aufgabengruppe neue Aufgaben hinzuzufügen, als vorhandenen Aufgaben eine neue Aufgabengruppe zuzuweisen. Wenn Sie beabsichtigen, eine Aufgabenliste mithilfe von Aufgabengruppen zu organisieren, erstellen Sie erst die Aufgabengruppe, und fügen Sie dieser Aufgabengruppe anschließend die gewünschten Aufgaben hinzu.
