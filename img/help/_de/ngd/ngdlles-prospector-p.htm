So fügen Sie einen Prospector hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ordner, zu dem Sie einen Prospector hinzufügen möchten, wählen Sie Objekt hinzufügen und anschließend Prospector.

  2. Geben Sie auf der Seite Hinzufügen: Prospector im Feld Name einen Namen für den Prospector ein.

    Geben Sie optional im Feld Beschreibung eine Beschreibung für den Prospector ein.

  3. Klicken Sie in der Liste Im Feld auf ein oder mehrere Segment(e).

  4. Geben Sie im Feld Suchen eine Abfrage ein.

  5. Geben Sie im Feld Optionale Kriterien zusätzliche Suchkriterien ein, um die Abfrage einzugrenzen oder zu erweitern.

  6. Legen Sie die folgenden Parameter fest:

    • Zustand

    • Ergebnis-Optionen

    • Benachrichtigungsoptionen

    • Benachrichtigungsplan

  7. Klicken Sie auf die Schaltfläche Content Server durchsuchen neben dem Feld Erstellen in, um den Prospector zu einem anderen Ablageort als dem aktuellen Container hinzuzufügen.

  8. Klicken Sie auf Hinzufügen.

 

Tipp:
Sie können die Einstellungen für einen vorhandenen Prospectors anpassen, indem Sie mit der rechten Maustaste auf den Prospector klicken und anschließend Bearbeiten wählen.
