So erstellen Sie eine Verknüpfung durch Ziehen:

  1. Wählen Sie die Objekte aus, für die Sie eine Verknüpfung erstellen möchten.

  2. Klicken Sie mit der rechten Maustaste, und ziehen Sie die Objekte zu dem Ablageort, in dem die Verknüpfung sich befinden soll.

    Wenn Sie die Maustaste loslassen, wird ein Kontextmenü angezeigt.

  3. Klicken Sie im Kontextmenü auf Verknüpfung hier erstellen.

  4. Geben Sie Metadaten ein, falls erforderlich.

     

    Tipp:
    Wenn sich im Zielspeicherort bereits eine Verknüpfung mit demselben Namen befindet, wird die neue Verknüpfung mit einem numerischen Suffix erstellt, um sie von der bereits vorhandenen zu unterscheiden.
