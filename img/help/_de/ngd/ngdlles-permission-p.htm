So legen Sie Berechtigungen fest:

  1. Klicken Sie mit der rechten Maustaste auf das Objekt, für das Sie Berechtigungen festlegen möchten, und klicken Sie anschließend auf Berechtigungen.

  2. Klicken Sie auf der Seite Berechtigungen in der Liste Standard-Zugriff bzw. Zugewiesener Zugriff auf die Verknüpfung zu einem Benutzer oder einer Gruppe.

  3. Ändern Sie im Bereich Berechtigungen bearbeiten die Berechtigungen für den jeweiligen Benutzer bzw. die Gruppe, und klicken Sie dann auf Aktualisieren.

  4. Klicken Sie auf Fertig.

 

Hinweis:
Wenn Sie nicht im Besitz der Berechtigungen für ein bestimmtes Objekt sind, können Sie die Informationen auf der Seite Berechtigungen nicht ändern.
