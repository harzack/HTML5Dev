Ein Erscheinungsbild stellt einen Container dar, in dem Dokumente und HTML-Code gespeichert werden. Mithilfe von Erscheinungsbildern können Sie bestimmte Ablageorte anpassen. Dies gleicht dem Entwerfen einer Webseite.

 

Wichtig:
Obwohl Sie Erscheinungsbilder mit Enterprise Connect erstellen und bearbeiten können, werden Sie in Enterprise Connect nicht angezeigt. Dies gilt auch für Erscheinungsbilder, die auf der Content Server-Web-Benutzeroberfläche erstellt wurden.

 

Es gibt zwei Arten von Erscheinungsbildern, die Sie hinzufügen können, um die Oberfläche anzupassen:

  • Allgemeine Erscheinungsbilder gelten für bestimmte Benutzer innerhalb des Content Server. Sie können ausschließlich dem Erscheinungsbild-Datenbereich hinzugefügt werden, sodass sie systemweit zur Verfügung stehen.

  • Ablageortspezifische Erscheinungsbilder:

Ablageortspezifische Erscheinungsbilder können auf zwei Arten zugewiesen werden:

  • Nicht durchgängig: Das Erscheinungsbild wird nur dem Arbeitsbereich, Projekt oder Ordner zugewiesen, in dem es hinzugefügt wurde.

  • Durchgängig: Das Erscheinungsbild wird dem übergeordneten Arbeitsbereich, Projekt oder Ordner, in dem es hinzugefügt wurde, und allen untergeordneten Projekten oder Ordnern zugewiesen. Diese Art von Erscheinungsbild kann nur bestimmten Benutzern oder Gruppen zugewiesen werden.

Erscheinungsbilder bestehen aus den beiden Seiten Überblick und Arbeitsbereich, mit denen Sie das Erscheinungsbild definieren bzw. bearbeiten können. Nachdem Sie ein Erscheinungsbild hinzugefügt haben, definieren Sie die Einstellungen und das Layout auf der Seite Überblick, bevor Sie dem Arbeitsbereich Inhalt, z. B. Bilder, hinzufügen. Das Hinzufügen und Bearbeiten von Erscheinungsbildern ist im Normalfall auf bestimmte Benutzer beschränkt.
