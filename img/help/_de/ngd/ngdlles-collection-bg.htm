Eine Sammlung ist ein einzelner Ablageort, an dem Sie Zeiger zu ausgewählten Objekten speichern können, die sich an verschiedenen Content Server-Ablageorten befinden. Mit diesen Zeigern können Sie mühelos auf die ursprünglichen Objekte zugreifen. Sie können Objekte von Ihrem Arbeitsbereich oder von einer Liste von Objekten, die bei einer Suche angezeigt wurden, zu Sammlungen hinzufügen. So könnten Sie eine Sammlung erstellen, die Zeiger zu häufig verwendeten Dokumenten, Aufgabenlisten und URLs enthält, und diese Sammlung anschließend für die Mitglieder Ihres Teams freigeben. Wenn Sie über die entsprechenden Berechtigungen verfügen und Ihr Administrator diese Funktion aktiviert hat, können Sie auch Objekte aus Sammlungen entfernen.

Weitere Informationen zum Arbeiten mit Sammlungen in Content Server finden Sie unter Sammlungen verwenden.
