Wenn Sie nach Informationen suchen, können Sie Ihre Suche mithilfe von Attributen eingrenzen. Attribute sind Metadaten zu einem Objekt in der Content Server-Datenbank. Attribute, die automatisch hinzugefügt werden, wie z. B. die Objekt-ID eines Objekts, werden als Systemattribute bezeichnet. Attribute, die einem Objekt durch einen Benutzer zugewiesen werden, heißen benutzerdefinierte Attribute. Mithilfe benutzerdefinierter Attribute können Sie eindeutige Metadaten hinzufügen, die für Ihre Organisation aussagekräftig sind.

Solche benutzerdefinierten Attribute werden besonderen Containern hinzugefügt, den so genannten Kategorien. Diese Kategorien können entweder nur ein Attribut oder mehrere zusammengehörige Attribute enthalten. Ein großer Vorteil von Kategorien besteht darin, dass Sie mit ihrer Hilfe mehrere Attribute auf einmal zuweisen und effizientere Suchvorgänge ausführen können.

Da Attribute nur innerhalb einer Kategorie existieren können, bedeutet das Zuweisen einer Kategorie zu einem Objekt, dass die in dieser Kategorie enthaltenen Attribute auf das Objekt angewendet werden. Kategorien legen die Reihenfolge (Sequenz), das Layout, den Typ und die Standardwerte für die in ihnen enthaltenen Attribute fest. Es ist möglich, eine Kategorie nur mit einem Attribut zu versehen, und Sie können ein und demselben Objekt mehrere Kategorien zuweisen.

Alle Benutzer haben die Möglichkeit, beim Hinzufügen, Bearbeiten oder Verschieben von Objekten Kategorien zuzuweisen. Die Seite Hinzufügen der meisten Objekttypen enthält einen Bereich Kategorien, in dem eine Kategorie zum Zuweisen ausgewählt werden kann. Die meisten Objekte besitzen auf der Seite Eigenschaften auch eine Registerkarte Kategorien. Mit den entsprechenden Berechtigungen können Sie die Informationen einer einem vorhandenen Objekt zugewiesenen Kategorie auch ändern bzw. eine Kategoriezuweisung ganz entfernen. Sie können jede Kategorie anwenden, für die Sie die entsprechenden Berechtigungen besitzen. Meistens werden Kategorien dem Kategoriendatenbereich hinzugefügt. Auf Kategorien, die sich hier befinden, können alle Benutzer zugreifen.

Wenn ein Container, dem Sie ein Objekt hinzufügen, über eine Kategorie mit erforderlichen zugehörigen Attributen verfügt, müssen Sie die erforderlichen Informationen eingeben. Eine Kategorie mit erforderlichen Attributen ist durch das Symbol Erforderlich  gekennzeichnet.
