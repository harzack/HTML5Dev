So fügen Sie neue Dokumente hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Dokument hinzugefügt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf Dokument.

     

    Tipp:
    Sie können auch im Menü Objekt hinzufügen auf Dokument klicken.

     

  2. Klicken Sie im Bereich Dokument auf der Seite Hinzufügen: Dokument auf Neu und anschließend auf den Dokumenttyp, der der Liste hinzugefügt werden soll.

  3. Geben Sie im Feld Name einen Namen für das Dokument ein.

  4. Wählen Sie im Bereich Versionssteuerung einen Versionierungstyp aus, indem Sie auf Standard - Lineare Versionierung oder Erweitert - Versionierung in Haupt-/Nebenversionen klicken.

  5. Legen Sie weitere allgemeine Einstellungen fest.

  6. Klicken Sie auf Hinzufügen.

  7. Fügen Sie Inhalt zu dem Dokument hinzu, das in dem mit ihm verknüpften Desktop-Programm geöffnet wird, und schließen Sie es danach wieder.

  8. Klicken Sie im Dialogfeld Bearbeitung beenden auf Änderungen in Content Server speichern und anschließend auf OK.
