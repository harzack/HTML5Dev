Ein Workflow-Diagramm ist eine vordefinierte Implementierung eines Geschäftsprozesses, der normalerweise mehrere Benutzer und Aufgaben umfasst. So kann z. B. ein einfaches Workflow-Diagramm einem Mitarbeiter die Möglichkeit bieten, einen Urlaubstag zu beantragen und genehmigt zu bekommen, und aus einigen wenigen Aufgaben bestehen, die von dem Mitarbeiter, einem Vorgesetzten und einem Vertreter der Personalabteilung zu erledigen sind. Komplexe Workflow-Diagramme können viele Schritte enthalten und die Teilnahme Dutzender Benutzer erfordern.

Um ein Workflow-Diagramm zu erstellen, definieren Sie zunächst ein Workflow-Diagramm-Objekt und anschließend das Workflow-Diagramm mit dem Workflow-Editor. Informationen zum Arbeiten mit Workflow-Editor finden Sie unter Workflow-Diagramm-Designer kennenlernen. Ein Workflow-Diagramm kann an jedem Ablageort erstellt werden, für den Sie die entsprechenden Berechtigungen besitzen. Wenn Sie ein Workflow-Diagramm hinzugefügt haben, können Sie damit auf dieselbe Weise wie mit anderen Objekten arbeiten. So können Sie ein solches Workflow-Diagramm z. B. löschen, umbenennen, kopieren oder verschieben. Sie können auch eine Verknüpfung erstellen, das Workflow-Diagramm zu einem Nachrichtenobjekt machen oder es Ihrer Seite "Favoriten" hinzufügen. Darüber hinaus können Sie eine Workflow-Diagramm-Fassung erstellen.
