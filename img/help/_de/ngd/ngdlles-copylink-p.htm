So kopieren Sie eine Verknüpfung in die Zwischenablage:

  1. Klicken Sie mit der rechten Maustaste auf das entsprechende Objekt. Klicken Sie dann auf Link zur Zwischenablage kopieren und anschließend auf eine der folgenden Optionen. Die verfügbaren Befehle variieren abhängig vom Objekttyp:

    • Klicken Sie auf Öffnen, um eine Verknüpfung zu kopieren, über die das Objekt geöffnet werden kann.

    • Klicken Sie auf Herunterladen, um eine Verknüpfung zu kopieren, über die das Objekt heruntergeladen werden kann.

    • Klicken Sie auf Eigenschaften, um eine Verknüpfung zu kopieren, mit der Sie die Seite Eigenschaften für das Objekt öffnen können.

    • Klicken Sie auf Überblick, um eine Verknüpfung zu kopieren, mit der Sie die Seite Überblick für das Objekt öffnen können.

  2. Fügen Sie die Verknüpfung in das Zielobjekt ein, wie z. B. in eine E-Mail, eine Desktop-Anwendung (z. B. Word oder Excel) oder einen Webbrowser.
