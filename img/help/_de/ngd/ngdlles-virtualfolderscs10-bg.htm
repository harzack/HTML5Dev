Mit Enterprise Connect können Sie auf Inhalt zugreifen, der in virtuellen Ordnern gespeichert ist, und mehrere Aufgaben mit diesem Inhalt ausführen. Ein Banner direkt unter der Enterprise Connect-Symbolleiste gibt an, ob Sie in einem virtuellen Ordner arbeiten. Weitere Informationen zum Erstellen, Konfigurieren und Verwenden von virtuellen Ordnern finden Sie unter Virtuelle Ordner verwenden.

 

Hinweis:
Facets werden in dieser Version von Enterprise Connect nicht unterstützt.

 
 

Tipp:
Sie können zum Originalspeicherort eines Objekts in einem virtuellen Ordner wechseln, wenn der Systemverwalter diese Option aktiviert hat. Klicken Sie mit der rechten Maustaste auf das Objekt, und wählen Sie dann Zu ursprünglichem Ablageort.

 

Die folgenden Abschnitte enthalten einen Überblick über die in Enterprise Connect verfügbare Funktion für virtuelle Ordner.

Virtuelle Ordner

Sie können die folgenden Aufgaben beim Arbeiten mit virtuellen Ordnern in Enterprise Connect ausführen:

  • Durchsuchen Sie den Inhalt virtueller Ordner im Enterprise Connect-Baum.

  • Öffnen und bearbeiten Sie Dokumente in einem virtuellen Ordner.

  • Löschen Sie virtuelle Ordner, und benennen Sie sie um.

  • Zeigen Sie eine Vorschau des Inhalts eines virtuellen Ordners im Vorschaubereich an.

  • Fügen Sie einem echten Ordner innerhalb eines virtuellen Ordners mit der Kontextmenüaktion Objekt hinzufügen Inhalt hinzu. Im Banner wird angezeigt, ob Sie auf einen virtuellen Ordner oder dessen Inhalt zugreifen.

  • Erstellen Sie eine Desktopverknüpfung zu einem virtuellen Ordner.

     

    Hinweis:
    Da der Inhalt in virtuellen Ordnern dynamisch ist, kann eine Verknüpfung zu einem virtuellen Ordner oder dessen Inhalt ungültig werden, wenn der Inhalt geändert wird.

     

  • Öffnen und bearbeiten Sie Dokumente mit demselben Namen, die in demselben virtuellen Ordner angezeigt werden.

     

    Tipp:
    Anhand der Spalte Ablageort können Sie zwischen Dokumenten mit demselben Namen unterscheiden, indem Sie den Namen des übergeordneten Ordners des gewählten Dokuments prüfen.

     

Beachten Sie die folgenden Ausnahmen beim Arbeiten mit virtuellen Ordnern in Enterprise Connect:

  • Sie können virtuelle Ordner nicht offline setzen.

  • Sie können keinen Inhalt per Drag and Drop oder mit dem Enterprise Connect-Dialogfeld Speichern unter zu einem virtuellen Ordner hinzufügen. Für virtuelle Ordner steht die Liste Objekt hinzufügen oder das Kontextmenü Objekt hinzufügen zur Verfügung. Sie können aber nur ein Erscheinungsbild, einen anderen virtuellen Ordner oder eine benutzerdefinierte Ansicht zu einem virtuellen Ordner hinzufügen.

  • Sie können den Inhalt eines virtuellen Ordners nicht durchsuchen.

Echte Ordner

Wenn Sie auf einen echten Ordner innerhalb eines virtuellen Ordners zugreifen, gilt das folgende Verhalten:

  • Sie können echte Ordner oder deren Unterordner nicht innerhalb des virtuellen Ordners offline setzen. Sie können die Ordner aber in ihrem tatsächlichen Speicherort im Content Server offline setzen.

  • Wenn Sie mit der rechten Maustaste auf einen echten Ordner klicken, können Sie auf das vollständige Kontextmenü zugreifen.

  • Sie können echten Ordnern innerhalb von virtuellen Ordnern Inhalt hinzufügen.

Dokumente

Wenn Sie mit Dokumenten innerhalb eines virtuellen Ordners in Enterprise Connect arbeiten, gilt das folgende Verhalten:

  • Sie können schreibgeschützte Versionen von Dokumenten öffnen und Dokumente über das Enterprise Connect-Dialogfeld Öffnen öffnen und bearbeiten.

  • Mit der Funktion Als neue Version speichern können Sie eine neue Version eines Dokuments in einem virtuellen Ordner speichern.

  • Wenn zwei Dokumente mit demselben Dateinamen im Listenbereich angezeigt werden, können Sie eines davon öffnen und bearbeiten.

     

    Tipp:
    Anhand der Spalte Ablageort können Sie zwischen Dateien mit demselben Namen unterscheiden, indem Sie den Namen des übergeordneten Ordners des gewählten Dokuments prüfen.

     

  • Sie können ein Dokument innerhalb eines virtuellen Ordners löschen und umbenennen.

  • Sie können ein Dokument von einem virtuellen Ordner in Ihren Desktop ziehen.

  • Sie können Dokumente aus einem virtuellen Ordner in echte Ordner ziehen.

  • Sie können Dokumente innerhalb von virtuellen Ordnern nicht offline setzen.

  • Wenn Sie mit der rechten Maustaste auf ein Dokument in einem virtuellen Ordner klicken, erhalten Sie dieselben Kontextmenüaktionen wie bei Verwendung der Content Server-Web-Benutzeroberfläche.

  • Sie können ein Dokument innerhalb eines virtuellen Ordners mit dem Enterprise Connect Viewer anzeigen.

  • Der Name des übergeordneten Ordners eines Dokuments wird im Listenbereich in der Spalte Ablageort angezeigt.
