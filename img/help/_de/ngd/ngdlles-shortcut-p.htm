So erstellen Sie eine Verknüpfung:

  1. Klicken Sie mit der rechten Maustaste auf das Objekt, für das Sie eine Verknüpfung erstellen möchten, und klicken Sie anschließend auf Verknüpfung erstellen.

  2. Geben Sie auf der Seite Hinzufügen: Verknüpfung im Feld Name einen Namen für die Verknüpfung ein.

  3. Klicken Sie im Feld Erstellen in auf Content Server durchsuchen, navigieren Sie zum Ablageort, in dem die Verknüpfung erstellt werden soll, und klicken Sie dann auf die Verknüpfung Auswählen für diesen Ablageort.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Klicken Sie auf Hinzufügen.
