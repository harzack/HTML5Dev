Um den Zugriff auf häufig verwendete Objekte zu erleichtern, können Sie diese als Favoriten kennzeichnen. So können Sie z. B. Favoriten für alle derzeit von Ihnen bearbeiteten Projekte erstellen. Wenn Sie ein Objekt als Favoriten kennzeichnen, erstellen Sie eine Art Verweis auf das Objekt, der unter Meine Favoriten gespeichert wird.
