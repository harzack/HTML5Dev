Das Erstellen einer Verknüpfung empfiehlt sich, wenn Sie von verschiedenen Ablageorten in Content Server aus auf ein Objekt zugreifen möchten. Sie können beispielsweise in Ihrem persönlichen Arbeitsbereich eine Verknüpfung für die Ordner erstellen, auf die Sie am häufigsten zugreifen. Sie können Verknüpfungen erstellen, indem Sie die Option Verknüpfung erstellen im Kontextmenü verwenden oder indem Sie in Content Server mit der rechten Maustaste auf ein Objekt klicken, die Option Verknüpfung hier erstellen verwenden und das Objekt in einen Container ziehen. Die Zieh-Option ist nur in Windows Explorer verfügbar. Diese Option steht nur für unterstützte Containertypen wie Ordner, Projekte und Aufgaben zur Verfügung. Für Aufgabenlisten oder Taskgroups steht sie z. B. nicht zur Verfügung. Je nachdem, wie Ihr System eingerichtet ist, werden Sie eventuell aufgefordert, Metadaten für die neue Verknüpfung anzugeben.

Mit dem Befehl Link zur Zwischenablage kopieren können Sie eine Verknüpfung für ein Objekt in die Windows-Zwischenablage kopieren. Anschließend können Sie die Verknüpfung in E-Mail-Nachrichten oder Desktop-Anwendungen, wie Word, Excel oder Webbrowser, einfügen. Sie können den Typ der zu sendenden Verknüpfung wählen, und die verfügbaren Optionen hängen vom Typ des gewählten Objekts ab. Beispiel: Sie können festlegen, dass die Verknüpfung die Seite Überblick oder Eigenschaften für Dokumente öffnet bzw. dass die Verknüpfung das Dokument öffnet oder herunterlädt. Bei E-Mails, zusammengesetzten E-Mails, Ordnern und anderen Objekttypen wie Diskussionen, Umfragen und Aufgabenlisten können Sie festlegen, dass die Verknüpfung das Objekt oder die Seite Eigenschaften öffnet.
