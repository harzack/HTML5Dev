So erstellen Sie Workflow-Status-Objekte:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Workflow-Status erstellt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf Workflow-Status.

  2. Geben Sie in den Feldern Name und Beschreibung einen Namen bzw. eine Beschreibung für den Workflow-Status ein.

  3. Klicken Sie in der Liste Beziehung auf eine der folgenden Optionen:

    • Alle(s): Es werden nur die Workflows aufgeführt, deren Manager oder Initiator Sie sind.

    • Initiiert: Es werden nur die Workflows angezeigt, die Sie initiiert haben.

    • Verwaltet: Es werden nur die Workflows aufgeführt, die Sie verwalten.

    • System: Nur zugänglich für Systemverwalter bzw. Benutzer mit Bypass-Berechtigungen.

  4. Klicken Sie in den Bereichen Workflows auf eine der folgenden Optionen:

    • Nicht archiviert: Es werden nur die noch in Ausführung befindlichen Workflows angezeigt.

    • Archiviert: Es werden nur die archivierten Workflows angezeigt.

  5. Sie können dann optional die folgenden Schritte ausführen:

    • Klicken Sie auf das Symbol Ausdruck bearbeiten , um die Abfrage mithilfe von Expression Builder zu bearbeiten.

    • Klicken Sie in der Liste Sortierreihenfolge auf einen Eintrag, um die Workflows in der gewünschten Reihenfolge zu sortieren.

    • Klicken Sie auf Content Server durchsuchen, um festzulegen, wo der Workflow-Status erstellt werden soll.

  6. Klicken Sie auf Hinzufügen.

 

Hinweis:
Sie können einen Workflow-Status bearbeiten, indem Sie mit der rechten Maustaste auf den zu ändernden Workflow-Status klicken und Abfrage bearbeiten wählen.
