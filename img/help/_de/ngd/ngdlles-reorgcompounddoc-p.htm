So reorganisieren Sie Inhalte zusammengesetzter Dokumente:

  1. Klicken Sie mit der rechten Maustaste auf das zusammengesetzte Dokument, dessen Inhalte reorganisiert werden sollen, und klicken Sie anschließend auf Reorganisieren.

  2. Geben Sie auf der Seite Neuorganisation des zusammengesetzten Dokuments im Feld Neu für jedes Objekt, dessen Position in der Reihenfolge Sie ändern möchten, eine Nummer ein.

    Aktivieren Sie optional die Option Hauptdokument für das Objekt, das zum Hauptdokument werden soll.

    Aktivieren Sie optional die Option Fortlaufende Neunummerierung, um die anderen Objekte im zusammengesetzten Dokument neu zu nummerieren.

  3. Klicken Sie auf Aktualisieren.
