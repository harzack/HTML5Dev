So erstellen Sie Erscheinungsbilder:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Erscheinungsbild erstellt werden soll. Klicken Sie dann auf Objekt hinzufügen und anschließend auf Erscheinungsbild.

  2. Geben Sie auf der Seite Hinzufügen: Erscheinungsbild im Feld Name einen Namen für das Erscheinungsbild ein.

  3. Legen Sie weitere allgemeine Einstellungen fest.

  4. Klicken Sie auf Hinzufügen.

  5. Klicken Sie mit der rechten Maustaste auf das neue Erscheinungsbild und anschließend auf Hier durchsuchen.

     

    Hinweis:
    Das Kontextmenü Hier durchsuchen ist zwar standardmäßig verfügbar, kann jedoch von Ihrem Systemverwalter bei Bedarf entfernt werden. Wenn Hier durchsuchen nicht verfügbar ist, wenden Sie sich an Ihren Systemverwalter.

     

  6. Klicken Sie auf das entsprechende Symbol Bearbeiten , um den Inhalt und die Einstellungen für das Erscheinungsbild festzulegen.
