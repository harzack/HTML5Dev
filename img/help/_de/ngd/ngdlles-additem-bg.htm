Das Menü Objekt hinzufügen ist überall dort verfügbar, wo Sie berechtigt sind, Objekte hinzuzufügen. Die beim Hinzufügen von Objekten zur Verfügung stehenden Optionen sind vom jeweiligen Objekttyp abhängig. Informationen zum Hinzufügen eines Objekts finden Sie im Hilfethema für den speziellen Objekttyp.

Beachten Sie beim Hinzufügen von Objekten, dass Berechtigungen bestimmen, auf welche Objekte Sie Zugriff haben. Standardmäßig übernimmt jedes Objekt, die Berechtigungen des Containers, dem Sie das Objekt hinzufügen. Wenn eine Gruppe z. B. über die Berechtigung Inhalt anzeigen für einen Ordner verfügt, sind die Mitglieder dieser Gruppe berechtigt, den Inhalt aller Dokumente anzuzeigen, die Sie diesem Ordner hinzugefügt haben. Die Berechtigungen für einzelne Objekte in einem Ordner können jedoch geändert werden. Sie sind der Besitzer der Objekte, die Sie hinzufügen, und haben uneingeschränkte Berechtigungen für diese Objekte. Weitere Informationen zum Festlegen von Berechtigungen finden Sie unter Berechtigungen festlegen.

Metadaten bereitstellen

Je nach der Konfiguration Ihrer Umgebung müssen Sie eventuell Metadaten, z. B. Kategorieattribute, für die Objekte eingeben, die Sie zu einem Content Server-Ordner oder einem anderen Containertyp hinzufügen. In einigen Fällen werden bestimmte Metadaten automatisch aus dem Objekt extrahiert und in den Feldern im Dialogfeld Hinzufügen angezeigt. Je nach Konfiguration können Sie diese Felder eventuell bearbeiten.

Wenn Sie eine Gruppe von Objekten zu einem Content Server-Container hinzufügen, ist das Kontrollkästchen Auf alle Objekte mit ähnlichem Typ anwenden möglicherweise sichtbar. Wenn Sie dieses Kontrollkästchen aktivieren, werden die für dieses Objekt festgelegten Metadaten auf alle nachfolgenden Objekte desselben Typs in der Gruppe angewendet.

Im Dialogfeld Hinzufügen gibt es einige Parameter, die bei fast allen hinzugefügten Objekttypen gleich sind. Diese Parameter sind in der folgenden Tabelle aufgeführt und unter Umständen nicht erforderlich, um ein Objekt erfolgreich zu einem Content Server-Container hinzuzufügen.

Parameter Beschreibung
Name Content Server schlägt für jedes Objekt, das Sie hinzufügen, einen Standardnamen vor. Bei Dokumenten wird dazu zum Beispiel der Name der hinzuzufügenden Datei verwendet. Sie können jedoch im Feld Name einen eigenen Namen eingeben, der mehr zum Inhalt und Zweck des Objekts verrät. Die Vergabe eines möglichst aussagekräftigen Namens hilft auch, das jeweilige Objekt im Rahmen von Suchabfragen oder in einem LiveReport zu finden.
Beschreibung Durch Hinzufügen einer Beschreibung erhöhen sich die Chancen, dass das Objekt im Rahmen von Suchabfragen oder in einem LiveReport gefunden wird. Content Server indiziert den Text im Feld Beschreibung als Teil der Metadaten des Objekts. Je mehr Metadaten für ein Objekt vorliegen, desto besser kann Content Server dessen Inhalt und Zweck feststellen.
Kategorien Mithilfe des Feldes Kategorien können Sie den hinzuzufügenden Objekten ein oder mehrere Attribute hinzufügen. Weitere Informationen zur Arbeit mit Kategorien finden Sie unter Kategorien verwenden.
Erstellen in Standardmäßig wird das Objekt dem jeweils aktuellen Container hinzugefügt. Sie können jedoch festlegen, dass das Objekt einem anderen Container hinzugefügt werden soll.

Das folgende Beispiel einer Vorgehensweise erläutert die Felder, die Sie beim Hinzufügen eines Objekts zu einem Content Server-Container unter Umständen vorfinden. Je nach Konfiguration und den zur Verfügung stehenden Modulen sehen Sie eventuell weitere Felder.

  1. Wählen Sie die hinzuzufügenden Objekte, und ziehen Sie sie anschließend in den entsprechenden Content Server-Ordner.

  2. Führen Sie auf der Seite Hinzufügen eine der folgenden Aktionen aus:

    • Beschreiben Sie das Objekt im Feld Beschreibung.

    • Wählen Sie, falls verfügbar, die entsprechende Klassifikation für das Objekt.

    • Klicken Sie optional auf Bearbeiten, und geben Sie die Kategorieinformationen ein.

    • Aktivieren Sie das Kontrollkästchen Auf alle Objekte mit ähnlichem Typ anwenden, wenn diese Werte auf alle nachfolgenden Objekte desselben Typs in der Gruppe angewendet werden sollen.

       

      Hinweis:
      Wenn Sie auf Abbrechen klicken, während die Objekte zum Container hinzugefügt werden, wird der Vorgang angehalten, und es werden keine weiteren Objekte zum Zielverzeichnis hinzugefügt. Bereits hinzugefügte Dateien werden nicht gelöscht.

       

  3. Klicken Sie auf OK.
