Sie können angeben, in welchen Situationen Sie aufgefordert werden möchten, Metadaten zu in Content Server eingefügten Objekten einzugeben. Mögliche Einstellungen: Metadaten werden nur eingegeben, wenn Attribute oder andere Informationen erforderlich sind oder Metadaten werden für alle Objekte eingegeben, die Sie Content Server hinzufügen. Wählen Sie die Option, die Ihren Best Practices bei der Erfassung von Metadaten entspricht.

 

Hinweis:
Sie werden nicht aufgefordert, Metadaten einzugeben, wenn Sie Inhalte in einer Content Server-Instanz hinzufügen, verschieben oder kopieren.
