So erstellen Sie benutzerdefinierte Ansichten:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem eine benutzerdefinierte Ansicht erstellt werden soll, und klicken Sie dann auf Objekt hinzufügen und anschließend auf Benutzerdefinierte Ansicht.

  2. Klicken Sie auf der Seite Hinzufügen: Benutzerdefinierte Ansicht auf Durchsuchen, wählen Sie die von Ihnen erstellte HTML-Datei aus, in der die benutzerdefinierte Ansicht definiert wird, und klicken Sie dann auf Öffnen.

  3. Legen Sie weitere allgemeine Einstellungen fest.

  4. Klicken Sie auf Hinzufügen.

 

Hinweis:
Wenn die HTML-Datei Verweise auf andere Dateien (wie z. B. Bilder) enthält, müssen Sie diese Dateien separat hinzufügen.
