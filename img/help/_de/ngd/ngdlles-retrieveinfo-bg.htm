Sie können auf dem Content Server Suchvorgänge durchführen, um Dokumente, Ordner, Diskussionen und andere Objekte zu lokalisieren. Sie können konfigurieren, wie viele Suchergebnisse pro Seite angezeigt werden sollen. Wenn Sie über die entsprechenden Berechtigungen verfügen, können Sie zur Automatisierung des Suchvorgangs Prospectoren erstellen.

 

Tipp:
Wenn Sie mit der Schnellsuche von Enterprise Connect suchen, erhalten Sie dieselben Suchergebnisse wie mit der Websuche von Content Server. Weitere Informationen zur Suchfunktion in Enterprise Connect finden Sie unter OpenText Enterprise Connect Plug-in for Content Server - Benutzerhandbuch (NGDCORE-UGD).

 

Inhaltsverzeichnis  

  • Suchen ausführen

  • Objekte ohne übergeordnete Objekte im Ordner Zusätzliche Objekte anzeigen

  • Mit benutzerdefinierten Ansichtssuchformularen suchen

  • Anzahl der Suchergebnisse pro Seite angeben

  • Prospectoren erstellen
