So aktualisieren Sie eine Kategorie:

  1. Klicken Sie mit der rechten Maustaste auf die zu aktualisierende Kategorie, und klicken Sie anschließend auf Objekte aktualisieren.

    Aktivieren Sie optional auf der Seite Objekte aktualisieren das Kontrollkästchen Version hinzufügen, um eine neue Version der Kategorie zu erstellen.

  2. Klicken Sie auf Senden.
