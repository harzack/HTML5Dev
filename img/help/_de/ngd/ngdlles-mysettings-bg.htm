Wenn Sie E-Mails mit Enterprise Connect in Content Server einfügen, werden bestimmte Benennungskonventionen angewendet. Hat Ihr Systemverwalter den Knoten Meine Einstellungen im Enterprise Connect-Baum aktiviert, können Sie Regeln für E-Mail-Namen angeben. Die Ausgangswerte auf der Seite Namenserstellung für E-Mails wurden von Ihrem Systemverwalter eingestellt. Auf der Seite Namenserstellung für E-Mails können Sie die folgenden Aufgaben ausführen:

  • Angeben, welche Nachrichteneigenschaften bei der Generierung eines Content Server-Namens für E-Mails verwendet werden sollen.

  • Definieren Sie das Trennzeichen, das zwischen den einzelnen Nachrichteneigenschaften verwendet werden soll, die den E-Mail-Namen bilden.

  • Alle Zeichenfolgen angeben, die aus dem generierten E-Mail-Namen entfernt werden sollen

  • Format für Datum und Uhrzeit angeben

  • Handhabung von doppelten E-Mails angeben

 

Hinweis:
Wenn Enterprise Connect einen Namen für die E-Mail generiert, der bereits im Content Server-Container verwendet wurde, werden Sie aufgefordert, den Namenskonflikt zu lösen. Weitere Informationen zur Auflösung von Dateinamenskonflikten finden Sie unter Mit E-Mails arbeiten.

  • So legen Sie das Datum- und Uhrzeitformat in E-Mail-Namen fest
