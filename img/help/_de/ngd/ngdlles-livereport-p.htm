So erstellen Sie einen LiveReport:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem der LiveReport erstellt werden soll, und klicken Sie auf Objekt hinzufügen und anschließend LiveReport.

  2. Geben Sie auf der Seite Hinzufügen: LiveReport im Feld Name einen Namen für den Bericht ein.

  3. Geben Sie im Feld Datensatzgrenze die maximal zulässige Anzahl von Ergebnissen ein.

  4. Geben Sie im Feld SQL eine SQL-Anweisung für den Bericht ein.

  5. Klicken Sie in der Liste Parameter auf die mit der SQL-Anweisung verknüpften Parametertypen.

  6. Wählen Sie in der Liste Berichtsformat einen Status für den LiveReport aus.

  7. In der folgenden Tabelle ist beschrieben, wie Sie zusätzliche Aktionen zum Definieren des LiveReport ausführen.

    Zweck Vorgehensweise
    Titel hinzufügen, der auf der LiveReport-Ergebnisseite angezeigt wird Geben Sie im Feld Titel einen Namen ein.
    Benutzer auffordern, bei der Verwendung des LiveReport als Unterbericht neue Parameterwerte bereitzustellen Aktivieren Sie die Option Eingabeaufforderungen anzeigen.
    Eingabetypen definieren Klicken Sie im Bereich Eingaben in der Liste Typ auf einen Eingabetyp, und geben Sie dann im Feld Eingabeaufforderung den Text ein, der angezeigt werden soll, um die Benutzer zu einer Eingabe aufzufordern.
    Zulassen, dass die SQL-Anweisung den Inhalt der Content Server-Datenbank ändert Aktivieren Sie die Option Datenbankänderung zulassen.
    Unterbericht mit diesem LiveReport verknüpfen Klicken Sie im Bereich Bericht auswählen auf Durchsuchen, suchen Sie den LiveReport, der als Unterbericht angezeigt werden soll, und klicken Sie dann auf die entsprechende Verknüpfung Auswählen.
    Einem Unterbericht Parameter zuweisen Geben Sie in jedem Unterberichtparameter-Feld den Namen eines vorhandenen Parameters ein.
    Festlegen, welche Informationen aus der Datenbank auf der LiveReports-Ergebnisseite angezeigt werden Geben Sie im Feld Feld den Namen der Datenbankspalte ein, und geben Sie dann im Feld Spaltentitel einen entsprechenden Anzeigenamen für die Spalte ein.
    Erscheinungsbild des ausgewählten LiveReport-Formats konfigurieren Wählen Sie ein Format für den LiveReport, klicken Sie auf Konfigurieren, nehmen Sie im Fenster Berichtformat konfigurieren die gewünschten Änderungen vor, und klicken Sie anschließend auf Auswählen.

  8. Klicken Sie auf Hinzufügen.

 

Tipp:
Sie können die Einstellungen für einen vorhandenen LiveReport anpassen, indem Sie mit der rechten Maustaste auf den LiveReport klicken und anschließend auf Bearbeiten klicken.
