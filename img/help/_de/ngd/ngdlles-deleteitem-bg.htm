Objekte, die nicht mehr benötigt werden (beispielsweise eine Diskussion, die nicht mehr aktiv ist), können gelöscht werden.

Beachten Sie beim Löschen von Objekten Folgendes:

  • Sie müssen über die Berechtigung zum Löschen des Objekts verfügen.

  • Beim Löschen von Containerobjekten, wie z. B. Ordnern, Projekten und zusammengesetzten Dokumenten, werden auch alle im Container enthaltenen Objekte gelöscht.

  • Je nach Ihrer Konfiguration können Sie Objekte, die Sie gelöscht haben, möglicherweise wiederherstellen. Weitere Informationen zum Wiederherstellen gelöschter Objekte erhalten Sie von Ihrem Systemverwalter.

 

Tipps:

  • Sie können mehrere Objekte löschen, indem Sie sie auswählen, mit der rechten Maustaste darauf klicken und anschließend Löschen wählen. In einer Meldung werden Objekte aufgelistet, die Sie nicht löschen können, da Sie z. B. nicht über die entsprechenden Berechtigungen verfügen.

  • Wenn das Content Server-Modul Papierkorb in Ihrer Umgebung verfügbar ist, können Sie gelöschte Objekte wiederherstellen. Navigieren Sie dazu in Enterprise Connect zum Datenbereich Papierkorb, wählen Sie die wiederherzustellenden Objekte aus, und geben Sie den Ablageort für die wiederhergestellten Objekte an. Weitere Informationen zum Arbeiten mit dem Papierkorb finden Sie in der Online-Hilfe zum OpenText-Papierkorb.
