So fügen Sie ein Objekt zu einer Sammlung hinzu:

 

Hinweis:
Die verfügbaren Optionen sowie die Namen von Dialogfeldern und Listen hängen von der Konfiguration Ihrer Umgebung und der Version von Content Server ab, die Sie verwenden.

 

  1. Wählen Sie die Objekte aus, die Sie einer Sammlung hinzufügen möchten, klicken Sie mit der rechten Maustaste darauf, und wählen Sie dann Sammeln.

  2. Aktivieren Sie im Dialogfeld Sammeln das Kontrollkästchen Aktuelle Version sammeln, falls dies verfügbar ist.

  3. Führen Sie im Bereich Sammlung einen der folgenden Schritte aus, um anzugeben, welcher Sammlung Sie die ausgewählten Objekte hinzufügen möchten:

    • Falls verfügbar, wählen Sie eine Sammlung in der Liste kürzlich verwendeter Sammlungen aus.

    • Falls verfügbar, wählen Sie eine Sammlung aus den Ordnern Favoriten oder Persönliche aus.

    • Erstellen Sie eine neue Sammlung. Geben Sie im Dialogfeld Sammlung hinzufügen Namen, Beschreibung, Klassifikationen oder Kategorien (falls erforderlich) und den Ablageort für die neue Sammlung an, und klicken Sie anschließend auf Hinzufügen.

    • Durchsuchen Sie Enterprise Connect, um die Sammlung zu finden, der Sie die Objekte hinzufügen möchten.

  4. Klicken Sie auf Senden und anschließend auf OK.
