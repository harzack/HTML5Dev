So erstellen Sie ein Workflow-Diagramm:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Workflow-Diagramm hinzugefügt werden soll, klicken Sie auf Objekt hinzufügen und anschließend auf Workflow-Diagramm.

  2. Geben Sie auf der Seite Hinzufügen: Workflow-Diagramm im Feld Name einen Namen für das Workflow-Diagramm ein.

  3. Legen Sie weitere allgemeine Einstellungen fest.

  4. Klicken Sie auf Hinzufügen.

  5. Klicken Sie mit der rechten Maustaste auf das neue Workflow-Diagramm, und klicken Sie anschließend auf Bearbeiten, um den Workflow-Editor zu starten, mit dessen Hilfe Sie das Workflow-Diagramm definieren.
