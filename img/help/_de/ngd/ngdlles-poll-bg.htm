Umfragen ermöglichen es Benutzern, sich an Entscheidungsfindungsprozessen zu beteiligen, indem bestimmte Fragen zur Abstimmung gestellt werden.

Eine Umfrage ist ein anfügbarer Objekttyp, d. h., Sie können eine Umfrage an ein Nachrichtenobjekt, ein Diskussionsthema bzw. eine Antwort auf ein solches Thema oder an eine Aufgabe anhängen und damit anderen Benutzern sofortigen Zugang gewähren.

Eine Umfrage können Sie erstellen, indem Sie sie an einem beliebigen Ablageort hinzufügen, für den Sie über die Berechtigung "Objekt hinzufügen" verfügen. Beim Hinzufügen einer Umfrage können Sie eine Beschreibung der Umfrage eingeben, deren Kategorien und Attribute ändern und die Umfrage einem anderen Container hinzufügen.

Nachdem Sie einem Container eine Umfrage hinzugefügt haben, müssen Sie die Umfrage bearbeiten, um ihr Fragen und Antworten hinzuzufügen, Methoden für die Stimmabgabe festzulegen sowie andere Umfrageoptionen zu definieren. Standardmäßig werden nur ein Frage- und ein Antwortfeld angezeigt. Sie können aber bei Bedarf weitere Felder hinzufügen. Eine Umfrage kann bis zu 10 Fragen und pro Frage maximal 10 Antworten enthalten.

Für jede Frage müssen Sie eine der folgenden Methoden für die Stimmabgabe festlegen:

  • Optionsfeld: Die Umfrageteilnehmer können pro Frage nur eine Antwort auswählen.

  • Kontrollkästchen: Die Umfrageteilnehmer können pro Frage mehrere Antworten auswählen.

Darüber hinaus können Sie festlegen, dass die Umfrageteilnehmer einer Umfrage anonyme Kommentare hinzufügen und vor der Stimmabgabe die aktuellen Ergebnisse anzeigen können. Sie können einer Umfrage außerdem ein Bild (mit entsprechendem beschreibendem ALT-Text) hinzufügen. Dieses Bild sehen die Umfrageteilnehmer auf der Seite Umfrage, wenn sie ihre Stimme abgeben oder sich die Ergebnisse ansehen.

Jeder Umfrageteilnehmer hat eine Stimme. Nach der Teilnahme an einer Umfrage können Sie die Umfrageergebnisse und Kommentare für die Umfrage nur anzeigen.

 

Achtung
Sie können eine Umfrage bearbeiten, nachdem Benutzer ihre Stimme abgegeben haben. Jedoch gehen alle bisherigen Stimmabgaben verloren.
