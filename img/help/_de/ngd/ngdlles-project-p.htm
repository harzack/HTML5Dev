So erstellen Sie ein Projekt:

  1. Klicken Sie mit der rechten Maustaste auf den Ordner oder Arbeitsbereich, in dem das Projekt erstellt werden soll, und klicken Sie auf Objekt hinzufügen und anschließend Projekt.

  2. Gehen Sie im Bereich Allgemeine Info auf der Seite Hinzufügen: Projekt wie folgt vor, und klicken Sie anschließend auf Weiter:

    • Geben Sie im Feld Name einen Namen für das Projekt ein.

    • Klicken Sie im Bereich Erstellen aus auf Leeres Projekt.

    • Legen Sie alle anderen allgemeinen Schritte fest.

  3. Führen Sie im Bereich Inhalt folgende Schritte aus, und klicken Sie anschließend auf Weiter:

    • Wählen Sie in der Liste Status einen Status für das Projekt aus.

    • Wählen Sie in den Listen Startdatum und Zieldatum jeweils einen Monat, einen Tag und ein Jahr.

    • Führen Sie alle weiteren optionalen Schritte im Bereich Inhalt aus.

  4. Führen Sie im Bereich Teilnehmer folgende Schritte aus, und klicken Sie anschließend auf Weiter:

    • Klicken Sie im Bereich Teilnehmer auf die entsprechende Option.

      Klicken Sie optional auf Teilnehmer hinzufügen bzw. Teilnehmer bearbeiten, wenn Sie Projektteilnehmer hinzufügen oder bearbeiten möchten.

    • Klicken Sie im Bereich Öffentlicher Zugriff auf Aktiviert oder Deaktiviert.

    • Klicken Sie im Bereich E-Mail-Benachrichtigungen auf die entsprechende Option, um festzulegen, welche Gruppen von Projektteilnehmern Benachrichtigungen per E-Mail erhalten sollen.

  5. Führen Sie im Bereich Präsentation folgende Schritte aus, und klicken Sie anschließend auf Fertig stellen:

    • Klicken Sie im Bereich Anzahl der hervorgehobenen Unterobjekte auf eine Option.

    • Führen Sie alle weiteren optionalen Schritte im Bereich Präsentation aus.
