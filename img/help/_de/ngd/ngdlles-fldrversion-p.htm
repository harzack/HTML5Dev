So legen Sie Optionen für die Erstellung von Versionen für Ordner und Projekte fest:

  1. Klicken Sie mit der rechten Maustaste auf den Ordner oder das Projekt, für den oder das Optionen zur Versionserstellung festgelegt werden sollen, und klicken Sie anschließend auf Versionssteuerung.

  2. Klicken Sie im Bereich Versionssteuerung auf eine Option, um festzulegen, ob Benutzer eine Methode für die Erstellung von Versionen auswählen können oder ob sie Haupt- und Nebenversionen verwenden müssen.

  3. Klicken Sie im Bereich Anwenden auf auf eine Option, um festzulegen, ob die Einstellung Versionskontrolle nur für das aktuelle Objekt, nur für Ordner und Projekte im aktuellen Objekt oder für beide gilt.

  4. Klicken Sie auf Senden.
