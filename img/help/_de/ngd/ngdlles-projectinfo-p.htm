So greifen Sie auf Projektinformationen zu:

  1. Klicken Sie mit der rechten Maustaste auf das Projekt, mit dem Sie arbeiten möchten, und klicken Sie anschließend auf Projekt.

  2. Wählen Sie den entsprechenden Informationstyp.

Aufgabe Aktion
Zugreifen auf einen detaillierten Projektüberblick, einschließlich Projektzusammenfassung, aktuelle Nachrichten und Aufgaben. Wählen Sie Überblick.
Zugreifen auf den Projekt-Arbeitsbereich. Wählen Sie Arbeitsbereich.
Anzeigen einer Liste der Ihnen zugewiesenen Projektaufgaben. Wählen Sie Meine Aufgaben.
Zugreifen auf mit einem Projekt verwandte Nachrichtenobjekte. Wählen Sie Nachrichten.
Definieren Sie eine Projektgliederung. Wählen Sie Gliedern.
Zugreifen auf eine Liste der Projektteilnehmer. Wählen Sie Teilnehmer.
Zugreifen auf Projektabfragen, Schnappschüsse und Berichte. Wählen Sie Berichte.
