Wenn Ihr Systemverwalter diese Funktion aktiviert hat, können Sie bei der Arbeit mit angegebenen Inhaltstypen auf der Web-Benutzeroberfläche von Content Server auf Enterprise Connect zugreifen. Wenn diese Funktion verfügbar ist, können Sie auf das Menü Funktion für ein unterstütztes Objekt zugreifen und Enterprise Connect hier öffnen auswählen. Eine neue Instanz von Enterprise Connect wird geöffnet, in der das ausgewählte Objekt angezeigt wird. Ihr Systemverwalter legt fest, für welche Untertypen diese Funktion verfügbar ist, Standardmäßig ist diese Aktion mit Enterprise Connect hier öffnen beschriftet. Der Systemverwalter kann jedoch auch eine benutzerdefinierte Beschriftung erstellen. Weitere Informationen zu dieser Funktion erhalten Sie bei Ihrem Systemverwalter.

 

Hinweis:
Die Funktion ist nur für Instanzen der Web-Benutzeroberfläche von Content Server im Internet Explorer verfügbar.
