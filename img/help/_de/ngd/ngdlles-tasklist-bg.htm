Aufgabenlisten sind Container, die eine einfache Möglichkeit zum Verwalten und Verfolgen der Details einzelner und zusammengefasster Arbeitszuweisungen bieten. Aufgabenlisten können Aufgaben, Meilensteine und Aufgabengruppen enthalten, und sie können an jedem beliebigen Ort hinzugefügt werden. Die Verwendung von Aufgabenlisten zum Verwalten von Zuweisungen in einem Projekt empfiehlt sich insbesondere deshalb, weil sich auf diese Weise alle Mitglieder eines Projekts stets über den Status einer Aufgabe informieren können.

Mithilfe von Aufgaben können Sie einem Benutzer bzw. einem Benutzerteam eine Priorität bzw. ein Fälligkeitsdatum für eine zugewiesene Aufgabe sowie detaillierte Anweisungen zur Erledigung einer zugewiesenen Aufgabe bereitstellen. Bevor Sie Aufgaben oder Aufgabengruppen hinzufügen können, müssen Sie zunächst eine Aufgabenliste erstellen. Sie können einer Aufgabenliste bzw. einer anderen Aufgabe oder Aufgabengruppe unbeschränkt viele zusätzliche Aufgaben hinzufügen.

Aufgabengruppen dienen dazu, mehrere zueinander gehörige Aufgaben zusammenzufassen. Sie können Funktionen an Gruppen von Aufgaben innerhalb einer Aufgabengruppe ausführen. Darüber hinaus können Sie mehrere Aufgaben ein und derselben Aufgabengruppe sowie anschließend mehrere Aufgabengruppen ein und derselben Aufgabenliste hinzufügen. Außerdem können Sie spezielle Aufgabengruppen für jeden Benutzer erstellen und jedem Benutzer separate Meilensteine zuweisen.

Meilensteine sind Zieldaten innerhalb eines Projektes, wie z. B. Termine, zu denen bestimmte Aufgaben erledigt sein sollen. Mithilfe solcher Meilensteine lassen sich der Status und die Zieldaten für alle Aufgaben verfolgen, die einem bestimmten Meilenstein zugewiesen sind. Wenn Sie einer Aufgabe oder Aufgabengruppe einen Meilenstein zuweisen möchten, ist es am effizientesten, wenn Sie diesen der Aufgabenliste hinzufügen, bevor Sie eine Aufgabe bzw. Aufgabengruppe hinzufügen. Sie können auch jederzeit danach Meilensteine zuweisen bzw. Änderungen an Meilensteinen vornehmen, indem Sie die Attribute der jeweiligen Meilensteine bearbeiten.

Über verknüpfte Informationsseiten erhalten Sie Zugriff auf spezielle Informationen zu den Aufgabenlisten. So können Sie z. B. die Seite Aufgabenlistenzusammenfassung, die eine Zusammenfassung aller Objekte in einer Aufgabenliste bietet, oder die Seite Meilensteine-Details, die Informationen über die mit einer Aufgabenliste verknüpften Meilensteine bietet, anzeigen lassen.
