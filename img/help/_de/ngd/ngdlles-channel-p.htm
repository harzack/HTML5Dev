So erstellen Sie Kanäle:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem der Kanal erstellt werden soll, und klicken Sie dann auf Objekt hinzufügen und anschließend auf Kanal.

  2. Geben Sie auf der Seite Hinzufügen: Kanal im Feld Name einen Namen für den Kanal ein.

  3. Führen Sie dann optional die folgenden Schritte aus:

    • Geben Sie im Feld Beschreibung eine Beschreibung des Kanals ein.

    • Klicken Sie neben dem Feld Bild auf Content Server durchsuchen, gehen Sie zu dem Objekt, das Sie dem Kanal hinzufügen möchten, und klicken Sie dann auf die Verknüpfung Auswählen.

    • Geben Sie im Feld ALT-Text für Benutzer, die das Bild nicht sehen können, eine Beschreibung des Bildes ein.

    • Legen Sie im Feld Erstellen in fest, wo Sie den Kanal erstellen möchten.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Klicken Sie auf Hinzufügen.

 

Tipp:
Sie können den Inhalt eines vorhandenen Kanals anzeigen, indem Sie mit der rechten Maustaste auf den Kanal klicken und anschließend Öffnen wählen.
