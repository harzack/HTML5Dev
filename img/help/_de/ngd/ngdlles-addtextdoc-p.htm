So fügen Sie Text-Dokumente hinzu:

  1. Klicken Sie mit der rechten Maustaste auf den Ablageort, an dem ein Text-Dokument hinzugefügt werden soll. Klicken Sie dann auf Objekt hinzufügen und anschließend auf Text-Dokument.

  2. Geben Sie auf der Seite Hinzufügen: Text-Dokument im Feld Name einen Namen für das Text-Dokument ein.

  3. Geben Sie im Feld Text den gewünschten Inhalt des Text-Dokuments ein.

    Falls verfügbar, klicken Sie auf das Symbol Rechtschreibung prüfen , um die Rechtschreibung des Textes zu prüfen.

  4. Legen Sie weitere allgemeine Einstellungen fest.

  5. Wählen Sie in der Liste MIME-Typ einen MIME-Typ für das Text-Dokument aus. Der Standardwert ist text/plain)

  6. Klicken Sie auf Hinzufügen.

 

Tipp:
Je nach der Version von Content Server, die Sie verwenden, können Sie u. U. HTML-MIME-Typen festlegen und bearbeiten.
