So erstellen Sie eine Fassung:

  1. Klicken Sie mit der rechten Maustaste auf das Dokument oder das zusammengesetzte Dokument, für das Sie eine Fassung erstellen möchten, und klicken Sie anschließend auf Fassung erstellen.

  2. Geben Sie auf der Seite Hinzufügen: Fassung im Feld Name einen Namen für die Fassung ein.

    Klicken Sie optional in der Liste Version auf eine Versionsnummer, um die Fassung auf eine vorherige Version verweisen zu lassen.

  3. Legen Sie weitere allgemeine Einstellungen fest.

  4. Klicken Sie auf Hinzufügen.
