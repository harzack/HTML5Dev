So fügen Sie einen E-Mail-Ordner hinzu

  1. Klicken Sie im Menü Objekt hinzufügen auf E-Mail-Ordner.

  2. Geben Sie auf der Seite E-Mail-Ordner hinzufügen unter Name einen Namen für den E-Mail-Ordner ein. Klicken Sie optional auf Symbol auswählen, wenn Sie ein anderes Symbol für den Ordner festlegen möchten.

  3. Legen Sie bei Bedarf weitere Optionen fest.

  4. Klicken Sie auf Hinzufügen.
