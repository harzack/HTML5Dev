Wenn Sie eine Lotus-Nachricht in Content Server öffnen, wird die Nachricht zunächst in die Lotus Notes-Datenbank importiert und dann geöffnet (im Gegensatz zum Öffnen von Outlook-E-Mails in Content Server). Alle Lotus Notes-Nachrichten, die Sie in Content Server öffnen, verbleiben so lange in der Datenbank, bis Sie sie wieder löschen.
