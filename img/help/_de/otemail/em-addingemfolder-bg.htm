Mit E-Mail-Ordnern können Sie Ihre Arbeit in Content Server auf die gleiche Weise organisieren wie bei der Verwendung von Ordnern in einem E-Mail-System.

E-Mail-Ordner sind ähnlich konzipiert wie normale Content Server-Ordner. Bei E-Mail-Ordnern können Sie allerdings zahlreiche E-Mails gleichzeitig auf einer Seite darstellen lassen. Wenn Sie E-Mails in einem E-Mail-Ordner speichern, werden standardmäßig die folgenden zusätzlichen Informationsspalten bereitgestellt. Diese Spalten können auch in regulären Content Server-Ordnern angezeigt werden, sofern der Systemverwalter die entsprechende Option aktiviert hat.

  • Von

  • Gewünschte Aktion

  • Betreff

  • Sendedatum

  • Empfangsdatum

Verwenden Sie E-Mail-Ordner, wenn hauptsächlich E-Mail-Nachrichten im Ordner gespeichert werden sollen und der Ordner zahlreiche E-Mail-Nachrichten aufnehmen soll.

So fügen Sie einen E-Mail-Ordner hinzu

Sie können E-Mail-Ordner auf jeder Arbeitsbereich- oder Container-Seite hinzufügen, für die Sie über eine Berechtigung zum Hinzufügen von Objekten verfügen.

Beim Hinzufügen von E-Mail-Ordnern in Content Server können Sie eine Beschreibung des E-Mail-Ordners hinzufügen, die zugehörigen Kategorien/Attribute ändern und den E-Mail-Ordner zu einem anderen Container hinzufügen. Weitere Informationen zum Hinzufügen von Objekten finden Sie unter Mit allgemeinen Objekten arbeiten.

Berechtigungen

Standardmäßig werden alle Berechtigungen eines E-Mail-Ordners auf das Objekt übertragen, das Sie innerhalb des Ordners erstellen. Wenn eine Gruppe etwa über die Berechtigung zum Anzeigen von Inhalten für einen E-Mail-Ordner verfügt, haben die Mitglieder dieser Gruppe auch die Berechtigung, alle in diesem Ordner erstellten Objekte anzuzeigen. Es ist aber auch möglich, die Berechtigungen für einzelne Objekte innerhalb eines Ordners zu ändern. Weitere Informationen finden Sie unter Berechtigungen.
