Een URL toevoegen:

  1. Klik met de rechtermuisknop op de locatie waar u een URL wilt toevoegen, klik op Item toevoegen en klik vervolgens op URL.

  2. Op de pagina URL toevoegen typt u een naam voor de URL in het vak Naam.

  3. Typ de URL in het vak URL.

  4. Geef eventuele andere algemene instellingen op voor het item.

  5. Klik op Toevoegen.

 

Opmerking:
U kunt een URL bewerken door de eigenschappen van de URL te bewerken.
