To define a Flag Checkbox attribute:

  1. Click a Category.

  2. On the Category page, click Flag: Checkbox in the Add Attribute drop-down list.

  3. On the Add Attribute To page, type a name for the attribute in the Name field.

  4. Do one of the following to sequence the attributes:

    • Click the name of the attribute after which you want the new attribute to appear on the Category page in the After drop-down list. Click <None> to set the attribute to appear first.

    • Select or clear the Show in Search check box.

    • Click the OK button.

    • On the Category page, select or clear the attribute's check box to set its default value.

    • Click the Submit button.
