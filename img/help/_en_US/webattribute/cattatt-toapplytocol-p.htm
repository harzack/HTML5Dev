To apply categories to collected items:

  1. On the Global Menu bar, from the Personal menu, click Collections.

  2. On the My Collections page, select the check box for each item to which you want to apply Categories.

  3. From the More Actions list, click Apply Categories.

  4. If available, on the Apply Categories page, complete the Notification area as follows:

    1. In the Enabled field, select the check box if you want to send email notification.

    2. In the Recipient field, if defined, your email address appears. You can change this to send the notification to an alternate email address.

    3. In the Message Subject field, you can change the default email message subject.

    4. In the 3rd Party Notifications area, select one of the two available option buttons:

      1. Select No Additional Notification if you do not want to add additional email recipients.

      2. Select Group to add additional email recipients.

        Next, do any, or all, of the following:

        • In the User field, click Find a user to select a Content Server user to whom you want to send an email notification.

        • In the Group field, click Find a group to select a Content Server group to whom you want to send an email notification.

        • In the Other Email field, enter an email address to send an email notification to an email address not available in Content Server.

  5. On the Apply Categories page, in the Categories field, click Edit.

  6. On the Edit Categories page, do one of the following:

    • Click the Add Categories button, , to add a Category, and then click Apply.

    • Click Upgrade to update existing Categories to the latest version.

  7. Optional Select the Add Version check box to add a Version to any versionable items, then select the Clear Existing Categories check box to remove all existing Categories from the items.

  8. Click Submit.

 

Note:
Email notification is only available when applying Categories to all items in the Collection.
