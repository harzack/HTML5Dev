The following email attributes can be configured to appear as facets in the Content Filter Sidebar:

  • BCC

  • CC

  • From

  • Has Attachments

  • On Behalf Of

  • Received Date

  • Sent Date

  • To

In order to add these attributes as facets, you must be a Content Server administrator or Knowledge Manager with permission to access the Facets Volume. The email attributes appear in the list of available data sources that you can assign when adding a new facet. For more information on adding facets, see Working with the Facets Volume.

If these attributes are added as facets, you can use them to filter the view of folders that store emails. For more information on using facets to filter the folder view, see Working with the Content Filter Sidebar.

 

Note:
Email facets only apply to emails. They do not apply to email attachments.

 

The following email attributes can be configured to appear as custom columns in Content Server:

  • BCC

  • CC

  • From

  • Has Attachments

  • On Behalf Of

  • Received Date

  • Sent Date

  • Subject

  • To

In order to add these attributes as custom columns, you must be a Content Server administrator or Knowledge Manager with permission to access the Facets Volume. The email attributes appear in the list of available data sources that you can assign when adding a new custom column. For more information on adding custom columns, see Working with the Facets Volume.
