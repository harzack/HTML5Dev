To create a Transport Box:

  1. Click Transport Warehouse on the Enterprise menu.

  2. On the Transport Warehouse page, open the Transport Boxes container.

  3. On the Transport Boxes page, click Transport Box on the Add Item menu.

  4. On the Add Transport Box page, type a name for the Transport Box in the Name field.

     

    Note:
    If a Transport Box already exists with the name you enter, an error message will appear.

     

  5. Optional Type a description for the contents of the Box in the Description field.

  6. Optional To add an existing Transport Box to the default box folder, click the Browse button in the Transport Box File section, navigate to the box file you want to upload, and then click Open.

     

    Note:
    The file must be a ZIP file, and contain only box items.

     

  7. Click Add.

 

Tip:
If you have the proper permission, you can also create a box from within the Workbench. To do this, select the check box for each item that will be added to the box, and then click the Add all to Transport Box button.
