To import a Box:

  1. On the target system, click Transport Warehouse on the Enterprise menu.

  2. In the Transport Warehouse, click the Transport Boxes link.

  3. On the Transport Boxes page, click the Add Item menu, and then click Transport Box.

  4. On the Add page, type a name for the box in the Name field.

  5. Optional Type a description for the box in the Description field.

  6. In the Transport Box File section, click the Browse button, navigate to the file you want to add, and then click Open.

  7. Click Add.
