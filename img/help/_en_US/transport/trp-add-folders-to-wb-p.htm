To add Folders to a Workbench:

  1. Navigate to the folder you want to add to the Workbench, click its Functions menu, and then click Add to Warehouse.

  2. Optional On the Add page, select the Include contents check box.

  3. In the Add to Workbench section, click the Browse Content Server button, navigate to the Workbench you want to add the folder, and then click its Select link.

  4. In the Duplicate Handling section, select one of the following:

    • Replace, which replaces any duplicate items in the Workbench with the items you are adding.

    • Skip, which ignores duplicate items and leaves the existing item in the Workbench.

  5. Click the Add to Warehouse button.
