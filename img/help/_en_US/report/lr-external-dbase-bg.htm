To report on an external database, a connection must be established in the WebForms Database Lookup function in the Content Server administration pages. These connections can only be setup by a Content Server administrator. Navigate to the Web Forms Database Lookup Administration section, and then select Add Database Connection Information. Complete the form for each external database required.

 

Important:
The optional module WebForms must be installed in your Content Server instance in order to connect to external databases.
