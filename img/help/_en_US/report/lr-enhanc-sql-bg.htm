LiveReport Extensions provides some improvements to the SQL editing offered within the LiveReport.

  1. The size of the editing window has been increased and a vertical and horizontal scroll bar provided.

  2. The SQL query may be formatted with spaces and carriage returns to improve readability and aid maintenance of the query. This formatting is retained by Content Server when a developer returns to edit the LiveReport.

  3. Tab characters within the SQL editing field are inserted into the text rather that being interpreted (and moving the cursor out of the edit field).

  4. An Undo and a Redo button have been provided to aid in text editing.
