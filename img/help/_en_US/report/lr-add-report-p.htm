To add a LiveReport:

  1. From the Add Item menu, select LiveReport.

  2. On the Add: LiveReport page, in the Name field, type a name for the report.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the names in the other, enabled, languages.

  3. Optional In the Description field, type a description for the LiveReport.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the descriptions in the other, enabled, languages.

  4. Optional If you want to add a title that appears on the LiveReport Results page, in the Title field type a name.

  5. In the Record Limit field, type the maximum number of results allowed.

  6. Optional Select the Show Prompts check box if you want to require users to supply new parameter values when using the LiveReport as a Subreport.

  7. Optional In the Inputs section, click an input type from the Type list, and then type the text that you want Content Server to display when requesting user input in the corresponding Prompt field.

  8. Type an SQL statement for the report in the SQL text box.

  9. Optional Select the Allow Database Modification check box if you want to allow the SQL statement to alter the contents of the Content Server database.

     

    Note:
    The Allow Database Modification check box does not appear if your administrator has enabled LiveReports Security.

     

  10. In the Param fields, select the parameter types that are associated with the SQL statement in the parameter lists. For more information about parameter types, see Adding and Editing LiveReports.

  11. From the Report Format list, select a format for the LiveReport. For more information about formats, see Understanding LiveReport Formats.

    If you want to configure the appearance of the selected LiveReport format, click Configure. Make changes in the Configure Report Format window, then click Select.

  12. In the Click-thru Sub-Report field, to associate a Subreport with this LiveReport, click Browse Content Server.... Navigate to the LiveReport you want to appear as a subreport, then click that LiveReport's Select link.

  13. To assign parameters to a Subreport, type the name of an existing parameter in each Sub-report Param # field.

  14. In the Display Columns area:

    1. In the Field text box, if you want to specify what information from the database is shown on the LiveReport's Results page, type the name of the database column.

    2. In the Column Title text box, type a corresponding display name for the column.

  15. Optional In the Categories field, click Edit to either select or add a Category to apply to this LiveReport.

  16. Optional Content Server will save your new LiveReport, by default, in the Enterprise Workspace. To change the default location, in the Create In field, click Browse Content Server.... Choose the Select link next to a new location.

  17. Click Add.
