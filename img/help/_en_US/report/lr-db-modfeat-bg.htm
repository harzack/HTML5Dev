By default, LiveReports provides a checkbox to allow database modification from a LiveReport. Anyone who edits a LiveReport can normally enable this feature by selecting this checkbox. It is possible for your administrator to disable this feature by editing the AllowDBModification field in the [report] section of the opentext.ini file. For more information, see [report].

There is also a feature called ModificationSQLTerms that allows you administrator to define terms that, if used, would require the Allow Database Modification option. If any of these terms are found in the SQL (or SQL templates) then an error is generated. For more information, see the ModificationSQLTerms field in the [report] section of [report].
