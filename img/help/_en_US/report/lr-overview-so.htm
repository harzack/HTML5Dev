LiveReports provide access to information in your organization's Content Server database, and enable you to find, organize and present the information stored there. LiveReports obtain information by issuing Structured Query Language (SQL) statements to the database. Since SQL statements can alter or delete critical data, the permission to add or edit LiveReports is restricted to the administrator by default.
