To view user information:

  1. From the Global Menu bar, choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, find the user whose information you want to view. For information on searching for users, see To Find a User or Group.

  3. Once you have found the user whose information you want to view, in the Name column, click on the user's name.

  4. Depending on your permissions to create and edit users and groups, you may also see links in the Actions column:

    • Edit, which opens that user's General Info page in edit mode.

    • Browse, which opens that user's My Workspace page.

    • Groups, which displays information about the groups to which that user belongs.
