Every Content Server user is a member of at least one group, and may be a member of many groups. A group can contain subgroups, which are groups within a group. Organizing users into groups is essential to setting up a Content Server system that is easy to use and maintain. A well-organized user community helps the administrator to effectively:

  • administer system privileges,

  • manage permissions on items in Content Server,

  • manage work assignments, such as Tasks from Workflow steps and Task Lists, and

  • set up default Notification reports for new users.
