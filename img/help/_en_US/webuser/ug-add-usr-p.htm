To create a user:

  1. Click the Enterprise global menu, and then click Users & Groups.

  2. On the Users and Groups page, click Add Item and then click User.

  3. On the Content Server Create User page, type a unique user name in the Log-in Name box.

  4. Do one of the following:

    • If a list appears next to the Department field, click the name of the Department to which you want to add the user.

    • If a list does not appear, click the Select button, click the Find button on the Select Department page to locate the group you want to assign as the user's department, and then click the group's Select link.

  5. In the Password field, type a password the new user must use to sign in to Content Server. Type the password again in the Verify Password field.

  6. Optional To set a password expiry date, select a date in the Expire Password section.

     

    Tip:
    The Expire Password section appears only if password expiration is enabled in your system.

     

  7. Optional To disable password expiration for this user, enable Password Never Expires.

     

    Tip:
    The Expire Password section appears only if password expiration is enabled in your system, and you have the User administration rights privilege.

     

  8. Optional To specify additional information about the user, type information in the First Name, Middle Initial, Last Name, Title, E-mail, Phone, Fax, and Office Location fields.

  9. Optional To set the language that will appear for this user, click a language in the Preferred Language list.

     

    Tip:
    The Preferred Language list appears only if multiple language selections are available on your system.

     

  10. Optional To specify a time zone for the user, click a time zone in the Time Zone list.

  11. Optional To grant privileges to the user, select any of the appropriate check boxes in the Privileges section.

  12. Click Submit.
