To edit user information:

  1. From the Global Menu bar, choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, find the user whose information you want to edit. For information on searching for users, see To Find a User or Group.

  3. Click the user's Edit link in the Actions column.

  4. On the General Info page, add or modify general user information, including password settings and system privileges.

  5. Click the Personal tab, and then add or modify any user information.

  6. Click Update.

 

Tip:
To view and edit your own user information, from the Global Menu bar, choose My Profile on the Personal menu.
