To delete a user:

  1. From the Global Menu bar, choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, find the user you want to delete. For information on searching for users, see To Find a User or Group.

  3. Click the user's Browse link in the Actions column.

  4. In the user's Personal Workspace, clean up the contents by deleting unwanted items and transferring important items to other locations. It is important to do this before you delete the user because it can be difficult to access the Personal Workspace of a deleted user.

  5. Once you have cleaned up the contents of the user's Personal Workspace, return to the Users and Groups page and click the user's Edit link in the Actions column.

     

    Tip:
    If you want to retrieve the userid of the user you are about to delete, once you are on that user's Content Server Edit User page, and prior to deleting that user, copy the user's userid from the address bar of your web browser. The userid is found in the page's URL in the form: userID=<number>.

     

  6. On the Content Server Edit User page, click Delete User.

  7. In the dialog box, click OK.
