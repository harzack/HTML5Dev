To add a user or subgroup to a group:

  1. From the Global Menu bar, choose Users & Groups on the Enterprise menu.

  2. Find the group to which you want to add a user or subgroup. For information on searching for groups, see To Find a User or Group

  3. On the Users and Groups page, click the group's Edit link in the Actions column.

  4. On the Edit Group page, find the user or group that you want to add.

  5. Select the user or group's Add to Group check box.

  6. Click Submit.

  7. Click Done.
