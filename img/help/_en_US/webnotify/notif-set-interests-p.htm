To define specific interests:

  1. Click an item's Functions icon, and then choose Set Notification.

  2. For each interest on the Set Notification page, click the name of the report where you want the Notification sent in the item's list.

  3. Click Submit.

 

Note:
Projects have their own Notification reports. You modify Project interests the same way you modify general interests. For more information, see Defining Notification Interests.
