You set Notification interests to monitor types of events that occur. Notifications monitor two types of interests: general and specific. General interests monitor types of events that occur throughout Content Server. They apply to all items that you have been assigned the See permission. For example, if you set Notification for the An Item is Added interest, you will be notified every time an item is added to Content Server, provided that you have been assigned the See permission for that item.

Specific interests monitor events that occur in specific containers or on specific items. For example, you can set Notification on a specific Task List to monitor when a deadline for the Task List changes. Each item in Content Server has its own set of specific interests.

If you have the proper privileges, you can define the default Notification interests for users in a group. By default, your interests are copied from your default group interests; however, if you make changes to your personal interests, they will override your default group interests.

 

Note:
Only the Administrator, or users with User Administration Rights, can mange the default interests for a Group.
