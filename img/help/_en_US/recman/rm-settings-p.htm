To change Records Management settings:

  1. Click Records Management on the Enterprise menu.

  2. On the Records Management Administration page, click the System Settings link.

  3. On the System Settings page, click the RM Settings tab.

  4. For each setting, click the Yes or No radio button.

  5. In the Hide the Option to Inherit RM Classification on Copy and Move area, click the Always keep RM Classification or Always inherit RM Classification radio button to specify what to do with the RM Classification on an item when it is copied or moved and the Inherit RM Classification option is hidden.

  6. In the Delete Workflow area, click the Select Workflow Map button to choose a Workflow map that will be used when RM Classified objects are deleted.

  7. In the Enable Record Versions area, choose the default Record Version Code that is used when objects are created with a new version.

  8. Click the Submit button.
