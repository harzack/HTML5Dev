To create a code:

  1. Choose Records Management on the Enterprise Menu.

  2. On the Records Management page, click the Records Management Administration link.

  3. On the Records Management Administration page, click the Table Maintenance link.

  4. On the Records Management Table Maintenance page, click the link for the table you want to add a code to.

  5. Click a code type on the Add Item menu.

  6. Type a code in the Code field.

  7. Specify any of the remaining settings.

  8. Click the Submit button.

 

Tip:
You can also create a Table Maintenance code on the page where it is to be used, by clicking <Create New Code> in the code's drop down list.
