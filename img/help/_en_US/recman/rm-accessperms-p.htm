To set permission settings for classified objects:

  1. Click Records Management on the Enterprise menu.

  2. On the Records Management Administration page, click the System Settings link.

  3. On the System Settings page, click the Access Permissions tab.

  4. On the Configure Access on Classify and Mark Official page, click one of the following radio buttons in the Classify and Mark Official sections:

    • Use existing access restrictions

    • Add to access list

    • Replace access list

  5. In the Permissions section, select the check box for each permission type that users must have to perform the Mark Official, Remove Official, and Finalize Record functions.

  6. Click the Submit button.
