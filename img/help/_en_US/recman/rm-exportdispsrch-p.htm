To export Disposition Search results:

  1. Click the Functions icon for a Disposition Search, and then choose View Snapshots.

  2. On the Disposition Results page, click the Functions icon for a Disposition snapshot, and then choose Export Result Metadata.

  3. In the Export to section of the Export Disposition Results page, click the Delimited Text File or Excel File radio button.

  4. Select additional RM and Physical Objects metadata to be included in the report by highlighting fields in the Available lists and using arrow buttons to add desired metadata to the Selected lists.

    Optionally, select the Export selected items only check box to only include RM and Physical Objects that you have selected on the Disposition Search Results page.

  5. Type a name for the report in the Name field.

  6. Click the Browse Content Server button, navigate to a location in Content Server where you want to export the results, and then click its Select link.

  7. Click the Export button.

 

Tip:
The parameters specified on the Disposition’s Settings tab determine whether the item’s current version or all versions are exported.
