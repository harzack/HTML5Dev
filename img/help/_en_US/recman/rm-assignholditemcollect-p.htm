To assign a Hold to items in a Collection:

  1. In a Collection, select the check box next to each item you want held.

  2. Click Apply Hold in the More Actions drop-down list.

  3. On the Search Holds page, type a Hold name in the Search Hold Name for field, and then click the Search button.

  4. Click the Select link for a Hold.

 

Tip:
You can assign a Hold to all items in a Collection by selecting the Entire Collection check box in the detail view of a Collection.

To create a new Hold and assign it to the item in a Collection, click Create New Hold in the Apply: Hold drop-down list in the Search Holds section of the Search Holds page.
