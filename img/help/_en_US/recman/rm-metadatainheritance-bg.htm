On the Metadata Inheritance tab, you can select which Records Management metadata will be inherited during a copy or move operation, and which metadata will be added to classified items during a reclassification or metadata update. The following metadata can be chosen:

  • Status

  • Status Date

  • Essential

  • Storage Medium

  • RSI

  • Mark Official

 

Note:
When multiple Classifications are turned on, metadata is not inherited from Classifications to managed objects.
