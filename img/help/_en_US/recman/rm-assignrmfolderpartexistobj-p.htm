To assign an RM folder or part to an existing object:

  1. Click the Functions icon for an item, choose Properties, and then choose Records Detail.

  2. On the Records Detail page, click Browse Classifications in the Select Classification drop-down list.

  3. On the Browse Classifications page, locate an RM folder or part.

  4. Click the Select link for the RM folder or part.

 

Note:
An RM folder must contain an open RM part to have objects assigned to it.
