To assign an RM classification to a new object:

  1. In the Enterprise or Personal Workspace, click an item type on the Add New Item menu.

  2. Click Search RM Classifications in the Classify drop-down list.

  3. On the Search RM Classification page, specify any search settings.

  4. Click the Search button.

  5. Select the check box for each RM classification you want to assign to the item.

  6. Click the Submit button.
