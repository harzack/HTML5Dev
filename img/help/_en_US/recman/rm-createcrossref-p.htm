To create a Cross-Reference or record version:

  1. Click the Functions icon for a managed object, choose Properties, and then choose XReference.

  2. On the XReference tab of the object's Properties page, click New Cross-Reference or New Record Version in the Add: Cross-Reference drop-down list.

     

    Note:
    If object you are adding as a version is already a version of another object, the New Record Version option will not appear. This option also does not appear unless it is enabled by the administrator.

     

  3. Click the Browse Content Server button, browse for the Content Server object you want to Cross-Reference, and then click the Select link for the object.

  4. On the Add Cross-Reference page, click a Cross-Reference type in the Cross-Reference Type drop-down list.

     

    Note:
    If you are adding a Record Version and the administrator selected a default record type, this field is automatically populated and cannot be changed.

     

  5. Optional Type a reason for the association in the Comment field.

  6. Click the Add button.
