To remove RM Classifications from Multiple Items:

  1. In a Collection, Search Result, or the Browse View of a Workspace, select the check box for each item that you want the RM Classification removed.

  2. In the More Actions menu, click Remove Classifications.

  3. In the Remove Classifications window, click RM Classification in the Classify drop-down list, then select the RM Classification that you want to remove.

  4. Click the Submit button.
