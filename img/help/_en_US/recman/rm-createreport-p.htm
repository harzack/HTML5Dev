To create a report:

  1. Choose Records Management on the Enterprise menu.

  2. On the Records Management page, click the Reports link.

  3. On the Reports page, click the link for the report you want to create.

  4. Specify any of the settings.

  5. Click the Run Report button.

Table 1-8: Optional Steps

Start a Classification report for a specific classification Click the Browse Classification button, and then click the Select link for a classification.
Set the disposition by which a Disposition report is filtered Click a search option in the Disposition Search drop-down list.
