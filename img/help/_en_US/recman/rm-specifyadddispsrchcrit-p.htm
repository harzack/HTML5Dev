Table 1-7: To specify additional Disposition Search criteria:

To...
 On the Specific tab of the disposition search, do any of the following...
Use only Maximum Retention schedules Select the Maximum Retention check box.
Assign RSI's to the Disposition Search Click Select RSI in the RSI drop-down list, select the check box for each RSI you want to include in the Search criteria on the Select RSI page, and then click the Submit button.
 

Note:
To include all RSIs in the Disposition Search, leave the RSI field blank.

To remove RSIs from the Disposition Search, highlight the RSI and then click the Remove button.

 
Specify a default process
 

Note:
Specifying a default process is optional for Delete Electronic Format, Export, and Close processes; it is required for the Change Status process. For a description of the default processes, see RSI Schedule Actions.

 
 Click a radio button in the Default Process section. If you click the Delete Electronic Format or Change Status radio buttons, click a status in the New Status drop-down list.
Define the scope of the search in another container In the Container section, click the Browse Enterprise button, navigate to the container where you want to initiate the search, and then click its Select link.
Exclude a container from the search results Clear the Include Container in Search Results check box.
Restrict the search to specific Content Server object types Click an object type in the Object Type drop-down list. To select multiple object types, Hold down the CTRL key while you click the object types you want to select.
Allow multiple snapshots of disposition search results Select the Multiple Snapshots check box.
Automatically action the results of a Disposition Search Select the Auto Process Search Results check box.
Use a saved Advanced Search Query as part of disposition search processing In the Advanced Search Query section, click the Browse button, navigate to the Advanced Search Query that you want included, and then click its Select link. This parameter only appears if it enabled by the Administrator.
 

Note:
You cannot select both RM classification object types (RM classification, RM folder, and RM part) and managed object types as disposition search criteria.

If you do not restrict the search to specific Content Server object types, it will return classified objects by default.
