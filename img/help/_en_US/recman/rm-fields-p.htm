To configure Records Management Fields:

  1. Click Records Management on the Enterprise menu.

  2. On the Records Management Administration page, click the System Settings link.

  3. On the System Settings page, click the RM Fields tab.

  4. In the Configure Records Details Fields area of the Configure Records Management Fields page, select the check boxes for the parameters you want to appear on the Add Item page or on the Records Detail tab.

  5. In the Configure Required RM Classification Fields area, select the check box for any RM Classification field that you want to designate as a required field.

  6. In the Include Cross-Reference on Add Item Page area, select the Cross-Reference check box to allow users to add a Cross-Reference on the Add Item page when they are adding a managed object to Content Server.

  7. Click the Submit button.
