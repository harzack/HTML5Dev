To change default Records Management settings:

  1. Click Records Management on the Enterprise menu.

  2. On the Records Management Administration page, click the System Settings link.

  3. On the System Settings page, click the Default Settings tab.

  4. On the Default Settings page, click the Select Classification button, browse to the Classification that you want to set as the default, and then click its Select link.

  5. In the Default Location for Items Created within the Classification Workspace area, click the Browse Content Server button, navigate to the location in Content Server where you want items created, and then click its Select link.

  6. In the RM Classification Titling area, select the appropriate button to enable titling and set the title append options.

  7. In the Number of Search Results to Return area, type the number of results that you want to appear per page in the Number of Results text box.

  8. In the File Number Separator area, select the default character that appears in file number in the Separator list.

  9. In the RM Classification Display Mode area, click the List or Featured radio button to specify how RM Classifications appear when browsing the Classifications volume.

  10. In the RSI and Holds Browsing area, click the Standard Browse or Advanced Search radio button to specify how users will browse RSIs or Holds.

  11. In the Hide Icons area, click the Yes or No radio button to choose whether to hide the Official and Classified item icons when browsing.

  12. In the Records Management Actions in Browse View, select the check box for each Records Management function for which you want multi-select actions enabled.
