To...
 On the Records Management Administration page, click the Table Maintenance link, click the link for the table you want to modify, and then do the following...
Edit a code Click the Edit link for a code, modify available settings, and then click the Submit button.
Delete a code Click the Delete link for a code.
Set a code as the default option Click the Set Default link for a code.
Remove a code as the default option Click the Remove Default link for the code that is set as the default option.

 

Note:
You cannot delete codes that are in use.
