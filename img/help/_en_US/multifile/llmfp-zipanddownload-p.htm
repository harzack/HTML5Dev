To Zip & Download objects:

  1. Select the check box beside one or more object names and then click Zip & Download.

  2. In the Items to be Zipped and Downloaded page, the Archive Name box is automatically populated with a name generated by Content Server. You can edit the name to something more appropriate.

    The Items area contains a summary of the items in your archive. The Supported Item Count area details the number of objects contained in the ZIP file.

  3. Click Zip & Download.

    The status of the events associated with the Zip and Download action display in the Action column of the Zip & Download Status page.

  4. When all actions are completed, click Download the zip file. You will be prompted to select a save destination and filename, following your operating systems standard functionality.

     

    Tip:
    Click Abort to skip any incomplete processes.

     

  5. Click OK to return to My Workspace.

To Zip & Download minor versions:

  1. From the Functions menu of the document to be zipped and downloaded, select Properties and then Versions.

  2. In the Versions tab, select the Functions menu of the minor version of the document to be zipped and downloaded.

  3. Select Zip & Download, and then follow the steps outlined in To Zip & Download Objects.
