To Zip & E-mail objects:

  1. Select the check box beside one or more object names and then click Zip & E-mail.

  2. In the Items to be Zipped and Downloaded page, the Archive Name box is automatically populated with a name generated by Content Server. You can edit the name to something more appropriate.

    The Items area contains a summary of the items in your archive. The Supported Item Count area details the number of objects contained in the ZIP file.

  3. To send a URL reference, you must save your archive to a location in Content Server. In the Archive To area, click Browse Content Server to select the location.

     

    Notes:

    • Users to whom you are sending the URL must have at least See Contents permissions within this area to access the link.

    • If you are sending the ZIP file as an attachment, archiving to Content Server is not an option. This field is not present if you are not sending a URL reference.

     

  4. In the From box, your email address, as stored in Content Server, appears by default.

  5. In the To box, type the email addresses of the users who will receive the email, or click User to select other Content Server users.

  6. In the Cc box, type the email addresses of the users who will be cc'd on the email, or click User to select other Content Server users.

  7. In the Subject box, type the subject of your message.

  8. In the Message box, type the body of your message.

  9. If your Content Server Administrator has configured Multi-File Output so that users are allowed to choose the method by which they reference their archives, the radio buttons for the Attach Archive area will appear.

    • Select As File to send the ZIP archive as an email attachment.

    • Select As URL reference to send a URL link in the body of the email.

    • If your Content Server Administrator has configured Multi-File Output such that only one of those options is available, only that option will be displayed in the Attach Archive area.

  10. Click Zip & E-mail.

    The Zip & E-mail Status page appears, displaying the status of all actions associated with the Zip & E-mail operation.

    • If the archive is attached as a file, then the users to whom the email is sent will receive a message with the ZIP archive as an attachment.

    • If the archive is attached as a URL reference, then the users to whom the email is sent will receive a message with a hyperlink that points to the archive in Content Server.

To Zip & E-mail minor versions:

  1. From the Functions menu of the document to be zipped and emailed, select Properties and then Versions.

  2. In the Versions tab, select the Functions menu of the minor version of the document to be zipped and emailed.

  3. Select Zip & E-mail and the follow the steps outlined in To Zip & E-mail Objects.
