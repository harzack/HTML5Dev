To print objects using the default printer:

  1. Select the check box beside one or more object names and then click Print.

  2. The Items to be Printed page displays a summary of the documents selected for printing. Click Print To Default.

  3. In the confirmation window that appears, click OK.

    The applications associated with the objects selected for printing open temporarily in response to the print request. The Print Status screen which contains a Printing Process window, also appears when the documents are sent to the default printer.

     

    Note:
    If any of the documents selected for printing cannot not be printed, a message displays the documents that will not print in a separate window. Click OK to close the window.

     

  4. When the print job is complete, click OK.

To print objects to a specific printer:

  1. Select the check box beside one or more object names and then click Print.

  2. The Items to be Printed page displays a summary of the documents selected for printing. Click Print.

  3. In the confirmation window that appears, click OK.

    The applications associated with the objects selected for printing open temporarily in response to the print request. The Print Status screen which contains a Printing Process window, also appears when the documents are sent to the default printer.

     

    Note:
    If any of the documents selected for printing cannot not be printed, a message displays the documents that will not print in a separate window. Click OK to close the window.

     

  4. In the Print window, select a printer from the Name list, and then click OK.

  5. When the print job is complete, click OK.

 

Note:
The first time that you use the print functionality of Multi-File Output, you will be prompted to download a printdoc.cab file. Click Yes to install it. Certain security settings may prevent this dialog box from appearing. If the dialog box fails to appear, temporarily disable the security setting, so you can download this control.

 
 

Tip:
When you attempt to print a Microsoft Power Point file, a Printer Properties dialog box might appear behind the browser page. If this occurs, change focus to the Printer Properties dialog box and click OK to proceed.
