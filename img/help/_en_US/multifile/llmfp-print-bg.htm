The Print portion of Multi-File Output enables the selecting of one or more Content Server to send directly to the printer. Not all Content Server objects can be printed. For example, workflows and discussions cannot be printed. It is only possible to print those documents that your desktop is configured to print. If your local system cannot print a document, an error message generates.

 

Note:
Multi-File Output does not support Compound Documents for Enterprise Library Services objects.
