Warning
When you edit a Poll after votes are cast, all existing votes are deleted.

 

To edit a Poll:

  1. Click a Poll's Functions icon, and then choose Edit.

  2. On the Poll page, type a name for the Poll in the Name field.

  3. Optional To add instructions to help voters complete the Poll, type them in the Instructions field.

  4. Click a start and end date for the Poll in the Effective Date and Expiration Date drop-down lists.

  5. In the Poll section, do the following:

    • Type a question in the Question text box of the Question column. To add multiple questions, click the Add Choice button found under the column labelled Row. To delete a question, click the Delete Choice button found under the column labelled Row.

    • Type answer choices for the question in the Choices text boxes of the Choices column. To add multiple answers, click the Add Choice button in the Choices column for each new answer you want to input. To delete an answer, click the Delete Choice button in the Choices column.

    • In the Choice Selection area, select one of the following radio buttons for each question to determine the method of voting:

      • Radio Button

      • Checkbox

  6. Optional To allow voters to view results or add comments, check the Allow viewing the results before voting or the Allow anonymous comments check boxes.

  7. To add an image to the Poll, click the Browse Content Server button, navigate to an image, and then click its Select link. To add alternate text to the image you selected, type the text to the ALT Text field.

  8. Click the Submit button to save your changes. Click the Reset button to reset all edited fields.
