To create a Template from a Project:

  1. Click a Project's Functions icon, and then click Make Template.

  2. Type a Name for the Template in the Name field.

  3. In the Source section of the Add Project Template page, click Project, and click Browse Content Server. Navigate to the Project from which you want to create the Template, and then click Select.

  4. Specify any other general item settings.

  5. Click Add.
