To create a Template from an XML file:

  1. Click a Project's Functions icon, and then choose Make Template.

  2. Type a Name for the Template in the Name field.

  3. In the Source section of the Add Project Template page, click File (*.xml), click Browse, navigate to the XML file you want to use as a Template, and then click Open.

  4. Specify any other general item settings.

  5. Click Add.
