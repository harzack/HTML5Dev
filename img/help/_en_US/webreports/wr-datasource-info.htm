WebReport data sources can be manipulated using special parameters called data source parameters. Examples of WebReport data sources include LiveReports, Search Queries or Forms. These parameters are passed in the URL when a WebReport is executed and they control things like the maximum number of results on each page.

Pagination

One useful application of data source parameters is pagination. This is discussed in full detail in WebReports DSstartrow and DSendrow.
