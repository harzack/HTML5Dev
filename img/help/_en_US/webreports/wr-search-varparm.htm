WebReports support the use of user defined parameters in the Full Text section of the search query. This is designed to work in a similar way to the parameters used in LiveReports. Parameters are specified in the format: %n where n is a number from 1 to 99. These parameters can be added to multiple Full Text boxes. When the webreport is run, parameters are passed by adding &inputlabeln to the URL.
