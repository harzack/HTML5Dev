This section details the column data tags: LANGUAGE and LIBPATH.

LANGUAGE

Syntax

[LL_REPTAG_LANGUAGE /]

Description

Inserts the system language currently being used.

LIBPATH

Syntax

[LL_REPTAG_LIBPATH /]

Description

Provides a path to the WebReports library folder. This is primarily used to to store useful Javascript files that can be used for WebReports applications. The path returned can be used for URLs that need to reference these files.

Example

For example, a typical path returned looks like this: /Livelinksupport/webreports/library/

The syntax to use this tag to include a Javascript file in the WebReports library looks like this: <SCRIPT SRC="[LL_REPTAG_LIBPATH /]page.js"></SCRIPT>
