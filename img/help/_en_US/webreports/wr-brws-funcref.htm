The browse.js file packaged with WebReports includes a list of functions designed to aid in page browse design. The browse.js file is accessed using a SCRIPT include like this: <SCRIPT SRC="/<support dir>/webreports/library/browse.js"></SCRIPT>

If this file is being included into a WebReport or ActiveView the LIBPATH tag can be used like this: <SCRIPT SRC="[LL_REPTAG_LIBPATH /]browse.js"></SCRIPT>
