This section details the sub-tag: ZIP.

ZIP

Syntax

ZIP

Description

The ZIP is used to merge two Lists of related data into a single List. The main data tag and the sub-tag parameter must both be Lists, and the return value is a List of Lists.

Example

[LL_REPTAG_"{1,2}" ZIP:"{'one','two'}" /] will return {{1,'one'},{2,'two'}}.
