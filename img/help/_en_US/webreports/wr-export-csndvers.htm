Exporting report data to a new document version is similar to Exporting to a Content Server New Document with the following exceptions:

  • Users are required to select an existing document or Custom View rather than a container.

  • Set the Export Type option to Content Server Version.

  • In the Add to field, select the document to which you want to add this exported report. You must select an existing document or Custom View rather than a container.

  • In the Append Results field, you can choose where, in the document to which you are adding a version, you want to insert the exported report. You can insert the exported report at the start, at the end, or, if using tags, anywhere in the body of the document.

  • If the document selected has advanced versioning enabled, you can choose to add either a Minor or a Major version.

When you click Export, a new version is added to the selected document. The new version will have the new mime type, description and categories as you specified on the Export page.

For more information, see Exporting to a Content Server Version - Parameters and Example.
