This section details the column data tags: SOURCENAME, SOURCEID, SOURCEROWNUM, SUBVERSION and SUPPORTDIR.

SOURCENAME

Syntax

[LL_REPTAG_SOURCENAME /]

Description

Returns the name of the data source.

SOURCEID

Syntax

[LL_REPTAG_SOURCEID /]

Description

Returns the objectid of the data source. This tag might be used in conjunction with the NODEINFO sub-tag to retrieve the name of the data source.

Example

For example, [LL_REPTAG_SOURCEID NODEINFO:NAME /]

SOURCEROWNUM

Syntax

[LL_REPTAG_SOURCEROWNUM /]

Description

Returns the data source row number. This tag differs from the ROWNUM tag in that it will always return the original row number from the data source regardless of how many rows are actually returned by the WebReport and regardless of any WebReport SORTs that may occur. This makes it useful for INCLUDEIF conditions that need to relate to the original data source rows before any WebReports row processing occurs.

This tag is similar to the ORIGROWNUM tag; however, the ORIGROWNUM tag provides the row number reference after any sorting has occurred.

Related Information

For more information, see ORIGROWNUM and ROWNUM.

SUBVERSION

Syntax

[LL_REPTAG_SUBVERSION /]

Description

Returns subversion details for WebReports and related modules, which is of use to administrators and when raising issues with support.

Example

Example output: (Content Server 10.5.0) (activeview 10.5.0.P05) (webreports 10.5.0.P04)

SUPPORTDIR

Syntax

[LL_REPTAG_SUPPORTDIR /]

Description

Provides the correct path to reference the Content Server support directory. This directory and/or folder contains standard Content Server support files such as images.

Example

For example, on many instances, the tag will return: /LivelinkSupport/.

Related Information

For more information, see LIBPATH.
