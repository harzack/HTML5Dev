It is possible to make use of sub-tags without also invoking a static data tag using the following dot function:

Function Parameters Returns
String ._subTag(String data, String subtags)
data: a String value to be operated on by the sub-tags.
subtags: a String containing the sub-tags to use on the data.
 A String value containing the result of the data transformed by the sub-tags.

 

Example: String s = ._subTag('3','DECODE:1:Beginning:2:Middle:3:End:Other UPPER')

In this case the string “s” is set to END.

Note that not all sub-tags are supported when called by ._subTag. Generally sub-tags that cause some other action besides returning formatted data do not work. The following sub-tags are not supported: ADDVAR, CONCATVAR, NEXT, PREV, SETFORM, SETVAR, SETWFATTR, SETWFFORM.
