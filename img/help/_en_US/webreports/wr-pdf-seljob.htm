The interface for selecting a Job Ticket to be used when sending a file for conversion is found alongside the Use External Conversion Engine check box in the Destination tab for any WebReport. The interface will only be displayed if the Use External Conversion Engine checkbox is checked and XML Job Tickets have been enabled in the admin pages.

  • To select a Job Ticket to use during conversion, click the Browse Content Server button, and browse for a job ticket file that a user has created and placed in Content Server.

  • If you would like to change a WebReport's settings to ensure that it is still sent for conversion but not used a job ticket anymore, press the clear button and update the page.

  • If a job ticket has been selected and the destination page has been updated, there is a function menu available to access the functions available to the job ticket file selected.
