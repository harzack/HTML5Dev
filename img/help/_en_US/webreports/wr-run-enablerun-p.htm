1. Select the WebReport's functions menu. Select Properties then select Specific.

  2. On the Specific tab of the WebReport, click the user icon, , next to the Run As field.

  3. In the User selection window, choose the user that will be used to run this report.

  4. Click Update to save your changes.

     

    Note:
    The next time this report is run, it will be run as though the user in the Run As field executed it. All permissions and any tags within the WebReport will be run as though it was executed by the Run As user. This includes tags like USERID. No matter who runs the report, this tag will always resolve to the USERID of the Run As user for that WebReport.
