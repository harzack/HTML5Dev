This section details the column data tags: ACTUALROWS, ACTUALUSERID, and APPSUPPORTDIR.

ACTUALROWS

Syntax

[LL_REPTAG_ACTUALROWS /]

Description

This tag returns the number of rows included in the WebReport by the INCLUDEIF tag. This differs from the total number of rows provided by the data source. For example, if the WebReport is executed in conjunction with a data source returning ten rows of data, but the INCLUDEIF restricts the output to three rows then TOTALROWS = 10 and ACTUALROWS = 3. This feature is only supported in the header and footer sections of the WebReport.

Related Information

For more information, see FILTEREDROWS, TOTALROWS, and TOTALSOURCEROWS.

ACTUALUSERID

Syntax

[LL_REPTAG_ACTUALUSERID /]

Description

This tag returns the user ID of the user that ran the WebReport, regardless of whether the RunAs feature is set.

Example

For example, if User1 runs a WebReport that has RunAs set to User2, the ACTUALUSERID tag will return the ID of User1, whereas the USERID tag will return the ID of User2.

APPSUPPORTDIR

Syntax

[LL_REPTAG_APPSUPPORTDIR /]

Description

Provides the correct path to reference the support directory used by Content Server Applications Management.
