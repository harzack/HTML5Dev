This section details the content control tags: APPEAR, APPEARBOTTOMOFF, APPEARLEFTOFF, APPEARRIGHTOFF, APPEARTOPHEADEROFF and APPEARTOPOFF.

APPEAR

Syntax

[LL_WEBREPORT_APPEAR /]

or

[LL_WEBREPORT_APPEAR TITLE1:“<title1>” TITLE2:“<title2>” /]

Description

Overrides the titles displayed in the report title bar. The TITLE1 parameter overrides the top component of the title, while the TITLE2 parameter overrides the bottom component.

Example

The tag:

[LL_WEBREPORT_APPEAR TITLE1:"Here is Title1" TITLE2:"Here is Title2" /]

will set the titles in the title bar as follows:

APPEARBOTTOMOFF

Syntax

[LL_WEBREPORT_APPEARBOTTOMOFF /]

Description

Excludes any Appearance HTML elements at the bottom, bottom left and bottom right.

APPEARLEFTOFF

Syntax

[LL_WEBREPORT_APPEARLEFTOFF /]

Description

Excludes any Appearance HTML elements to the left.

APPEARRIGHTOFF

Syntax

[LL_WEBREPORT_APPEARRIGHTOFF /]

Description

Excludes any Appearance HTML elements to the right.

APPEARTOPHEADEROFF

Syntax

[LL_WEBREPORT_APPEARTOPHEADEROFF /]

Description

Excludes any Appearance HTML elements at the very top.

APPEARTOPOFF

Syntax

[LL_WEBREPORT_APPEARTOPOFF /]

Description

Excludes any Appearance HTML elements at the top, top left and top right.
