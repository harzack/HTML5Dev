This section details the sub-tags: ORIGROWNUM, OVERRIDEOBJID and OWNERID.

ORIGROWNUM

Syntax

[LL_REPTAG_ORIGROWNUM /]

Description

Returns the data source row number after any WebReports sorting has occurred, but before any WebReports filtering has occurred. This tag differs from the ROWNUM tag in that it will always return the original row number from the data source regardless of how many rows are actually returned by the WebReport.

It is useful for INCLUDEIF conditions that need to react to a row number as the normal ROWNUM tag can't be used in an INCLUDEIF statement.

This tag is similar to the SOURCEROWNUM tag; however, it differs from the SOURCEROWNUM tag in that it reflects the order of rows after any WebReport SORT tags have been processed.

Related Information

For more information, see ROWNUM and SOURCEROWNUM.

OVERRIDEOBJID

Syntax

[LL_REPTAG_OVERRIDEOBJID /]

Description

Returns the dataID of the Content Server object that is currently overridden by an ActiveView template.

OWNERID

Syntax

[LL_REPTAG_OWNERID /]

Description

Returns the ownerID of the Content Server object.
