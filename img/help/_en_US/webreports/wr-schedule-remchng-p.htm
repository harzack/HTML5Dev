To remove or change a schedule:

  1. To remove a schedule, from the WebReport's functions menu select Properties then select Destination.

  2. On the Destination page, clear the Set Schedule box.

  3. Click Update.

     

    1. Other schedule settings will be retained, but the report will no longer run until the Set Schedule checkbox is selected. To alter settings, make your changes then click Update.

    2. If a report has been scheduled to run a fixed number of times, the remaining number of times and next run time will be updated by Content Server so you can see how many more runs are expected and when the next one will happen.

    3. Note that the Content Server administrator has the power to disable or delete schedules created by other users. This is achieved in the Content Server administration pages. See the WebReports Installation and Administration Guide for more details.
