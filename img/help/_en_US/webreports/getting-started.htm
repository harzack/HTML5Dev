The WebReports module provides a new Content Server node type called WebReport, which can be added to a Content Server workspace from the Add New Item drop down menu. When creating a new WebReport, the user is prompted to:

  1. Choose a data source on which the WebReport will be based.

  2. Select a reportview to use as the presentation template for the data source.

  3. A WebReport has all the standard properties of other Content Server nodes, such as a document node, but it also includes functions appropriate to reporting. For example, Edit Reportview.
