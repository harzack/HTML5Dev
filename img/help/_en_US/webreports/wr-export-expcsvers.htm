Example 1-23: Example of a URL invocation when exporting to a Content Server Version:

?func=ll&objId=1234&objAction=RunReport&export_location=livelink&livelink_node=livelinkversion&newVersion_ID=5678&mimetype=text%2Fplain&nexturl=...

In this example, WebReport 1234 adds a new Content Server document version with a Mime type of text/plain to the document id 5678.

For more information, see Exporting to a Content Server New Document Version.
