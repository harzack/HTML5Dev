This section details the sub-tags: HIDE, HTMLCR, and HUMANIZE.

HIDE

Syntax

HIDE

Description

This sub-tag is used to hide the output of a tag.

HTMLCR

Syntax

HTMLCR

Description

Inserts the HTML code wherever a carriage return and line feed character are found together, a single carriage return character is found or a single linefeed character is found. This sub-tag is useful for correctly formatting long text fields in HTML that would otherwise be missing carriage returns.

HUMANIZE

Syntax

HUMANIZE:APNUMBER

Description

The HUMANIZE sub-tag is used to format dates and integers to ensure they are more readable. The sub-tag has one required parameter, which determines how to format the tag.

Parameters

Parameter
Description
HUMANIZE:APNUMBER
If the main tag is a number from 1-10 then return the number spelled out. Otherwise, return the input unmodified.
HUMANIZE:INTCOMMA
This tag reformats an integer with commas separating every three digits. An optional parameter controls the separater character.
HUMANIZE:NATURALDAY
This tag reformats a date as “yesterday”, “today”, or “tomorrow” if appropriate. Otherwise, the date is formatted according to the system settings.
HUMANIZE:ORDINAL
This tag reformats an integer as an ordinal string.

Example

Examples using HUMANIZE:APNUMBER:

  • [LL_REPTAG_'1' HUMANIZE:APNUMBER /] returns “one”.

  • [LL_REPTAG_'5' HUMANIZE:APNUMBER /] returns “five”.

  • [LL_REPTAG_'11' HUMANIZE:APNUMBER /] returns “11”.

  • [LL_REPTAG_'some string' HUMANIZE:APNUMBER /] returns “some string”.

Example

Examples using HUMANIZE:INTCOMMA:

  • [LL_REPTAG_'100' HUMANIZE:INTCOMMA /] returns “100”.

  • [LL_REPTAG_'1000' HUMANIZE:INTCOMMA:"'" /] returns “1'000”.

  • [LL_REPTAG_'-1000000' HUMANIZE:INTCOMMA /] returns “-1,000,000”.

Example

Examples using HUMANIZE:NATURALDAY:

  • [LL_REPTAG_DATE HUMANIZE:NATURALDAY /] returns “today”.

  • [LL_REPTAG_'May 2 2010' HUMANIZE:NATURALDAY /] returns “2010/May/05”, according to the system date format.

Example

Examples using HUMANIZE:ORDINAL:

  • [LL_REPTAG_'1' HUMANIZE:ORDINAL /] returns “1st”.

  • [LL_REPTAG_'2' HUMANIZE:ORDINAL /] returns “2nd”.

  • [LL_REPTAG_'23' HUMANIZE:ORDINAL /] returns “23rd”.

  • [LL_REPTAG_'some string' HUMANIZE:ORDINAL /] returns “some string”.
