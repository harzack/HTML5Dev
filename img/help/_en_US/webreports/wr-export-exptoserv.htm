Example 1-27: Example of a URL invocation when exporting to the Server:

?func=ll&objId=1234&objAction=RunReport&export_location=server&serverFilePath=C:\export_testing\test_export.xml&nexturl=...

In this example, WebReport 1234 adds a new file to the Content Server file system. This new file is exported to: C:\export_testing\test_export.xml.
