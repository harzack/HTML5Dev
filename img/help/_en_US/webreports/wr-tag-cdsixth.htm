This section details the column data tags: INDEXTOTALHITS and ISSUBWEBREPORT.

INDEXTOTALHITS

Syntax

[LL_REPTAG_INDEXTOTALHITS /]

Description

Inserts the total number of hits from a search data source. This tag is useful when working with Using a Search Query as a Data Source as it can provide the user with an indication of the total result size without having to display them all. This tag can be used in combination with the &maxrows URL parameter to limit the results to a low number and get a value for the number of search hits very quickly with the minimum of performance impact.

ISSUBWEBREPORT

Syntax

[LL_REPTAG_ISSUBWEBREPORT /]

Description

Returns true if the WebReport is being run as a sub-WebReport, else returns false.
