To automatically add a new constant for the WebReport:

  1. Select the Constants tab for the WebReport. From the WebReport's Functions menu select Properties. From the Properties menu, select Constants.

  2. On the WebReport's Properties' page's Constants tab, select the button to add all the constants that are used in the reportview.

     

    Note:
    This icon only appears when the reportview contains constants that are not defined on the constants tab.

     

  3. For all the new constants:

    1. The Constant Name field is pre-populated.

    2. In the Constant Value field, type a value for this new constant. You can also select Browse to find a Content Server object, then click that object's Select link. Once you have clicked a Content Server object's Select link, that object's ID is automatically entered to the Constant Value field.

    3. Where possible, WebReports will determine the Constant Type of each constant programmatically. Where this is not possible, the constant will default to type String. You can optionally change the setting of Constant Type.

    4. Optional In the Constant Description field, you can, if you want, enter a description for this constant.

  4. Click Update.
