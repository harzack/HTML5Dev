To set a WebReport to run on the same date or day:

  1. Create a WebReport as normal and set its output destination to the desired location.

  2. Create a second WebReport that has no data source and set it to run on a daily schedule at the time you want the report to execute on the 25th.

  3. Make the second WebReport call the first one by using [LL_WEBREPORT_SUBWEBREPORT ... /].

  4. Finally, wrapper the call to the sub-WebReport with a [LL_WEBREPORT_IF ... /] statement that checks the day of the month.

     

    Note:
    Your second WebReport, the master report, should contain code something like this:

    [LL_WEBREPORT_IF "[LL_REPTAG_DATE DATE:'%d' /]" == "25" /]
    [LL_WEBREPORT_SUBWEBREPORT NODEID:[LL_REPTAG_$main_rep /] /]
[LL_WEBREPORT_ENDIF /]
