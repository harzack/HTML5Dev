The WebReports export feature allows users to export report data to email. The Export to E-Mail screen allows the user to specify email addresses in three different ways in the E-Mail Address field:

  • Use the first text field to manually type all email addresses separated by a comma.

     

    Note:
    Each address is separated using whatever symbol has been defined as a system preference. The instruction for this field will indicate this symbol. Two examples of these symbols are comma (,)and semi-colon (;).

     

  • Use the second field to select either a Content Server User or Group.

  • Use the third field to select a CSV file of addresses from within Content Server.

 

Notes:

  • It is possible to use these three methods in any combination.

  • If selecting a Content Server User or Group, no email will be sent if an email address is not specified in the user's Content Server profile.

  • Emails are only sent to direct members of the selected Content Server groups.

  • The total address list must not exceed 10,000 characters, roughly 500 email addresses.

 

For more information, see Exporting to E-Mail - Parameters and Example.
