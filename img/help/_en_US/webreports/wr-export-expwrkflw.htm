Example 1-26: Example of a URL invocation when exporting to a Workflow:

?func=ll&objId=1234&objAction=RunReport&export_location=workflow&workflow_ID=5678&attach=on&nodeName=report%20data&mimetype=text/html&workflow_title=my%20new%20workflow&nexturl=...

In this example, WebReport 1234 initiates an instance of workflow map 5678 and sets the title to “my new workflow”. The report itself is attached to the workflow with a name of “report data” and a Mime type of text/html.

For more information, see Exporting to a Workflow.
