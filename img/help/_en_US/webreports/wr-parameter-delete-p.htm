To delete a WebReport parameter:

  1. From the WebReport's functions menu select Properties, and then select Parameters.

  2. On the Parameters tab:

    1. Select the minus, -, button associated with the parameter to be deleted.

    2. Confirm you want to delete this parameter by selecting OK.

  3. Click Update.

     

    Tip:
    Select the red X button to delete all the parameter definitions for the entire WebReport, and then click Update.
