This section details the column data tags: REQUEST, RESPONSE, RKTENGINEENABLED, ROWDATA, and ROWNUM.

REQUEST

Syntax

[LL_REPTAG_REQUEST /]

Description

Returns the request as a Record.

Related Information

For more information, see RESPONSE and RECORD.

RESPONSE

Syntax

[LL_REPTAG_RESPONSE /]

Description

Returns the response as an Assoc.

Related Information

For more information, see REQUEST and ASSOC.

RKTENGINEENABLED

Syntax

[LL_REPTAG_RKTENGINEENABLED /]

Description

Returns true.

ROWDATA

Syntax

[LL_REPTAG_ROWDATA /]

Description

This tag can only be used in the row section and returns the Record for the current row of data.

Related Information

For more information, see FIELDS, FORMAT, and RECORD.

ROWNUM

Syntax

[LL_REPTAG_ROWNUM /]

Description

Returns the current row number. This tag is most useful when used with the ODD or EVEN sub-tags.

This tag cannot be used as a term in the INCLUDEIF content control tag as it changes according to the number of rows that are being returned by the INCLUDEIF tag.

Example

For example, [LL_REPTAG_ROWNUM ODD /] will return true for rows 1, 3, 5, 7, and so on. This tag can be used to alternate row colours.

Related Information

For more information, see ORIGROWNUM and SOURCEROWNUM.
