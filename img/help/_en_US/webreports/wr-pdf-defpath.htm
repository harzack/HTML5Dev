When used as part of a Job Ticket file in the FOLDER attribute,$DEFAULT_PATH will dynamically insert the appropriate file path as specified in the Manage WebReports Conversion administration page.

There are two sets of directories that can be specified in the administration page, and if the set of directories for use with XML Job Tickets has been specified, the $DEFAULT_PATH value will return those file paths; otherwise, if not specified, it will return the first set of conversion directories specified for conversion.

Because this is not a constant, you can use this value as part of the FOLDER attribute in both DOCINPUT and DOCOUTPUT tags. Using the $DEFAULT_PATH value will return the proper file path where used.

Example

<JOB:DOCINPUT FILENAME="" FOLDER="$DEFAULT_PATH" />
<JOB:DOCOUTPUT FILENAME="" FOLDER="$DEFAULT_PATH" DOCTYPE="PDF" />
