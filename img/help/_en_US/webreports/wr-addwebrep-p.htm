To create a WebReport:

  1. In the Name field, type an appropriate title for the new WebReport.

  2. Optional As per all Content Server items, an optional Description can be added. This information does not appear anywhere in the WebReport output.

  3. Optional The Data Source field is an optional field that allows a report source to be selected.

    1. To populate the Data Source field, click Browse Content Server, navigate to an appropriate data source, and then click its select link.

       

      Note:
      If no data source is selected, the row section of the Reportview is ignored.

       

    2. If the Go To Source Tab option is selected, after clicking the Add button, the Source tab is displayed so that other non-Content Server data sources can be selected, or more specific data source settings can be modified.

  4. The Reportview File is a mandatory field.

    1. You can select a reportview from the Use a Default Reportview list, to use a simple pre-written reportview. This default reportview can be edited using the Edit Reportview function option once the WebReport has been created.

    2. If a default reportview has not been selected, the Choose File button can be used to select a pre-written reportview from the user's desktop. Once a file has been selected, the file details are automatically inserted into the corresponding form field.

  5. Optional The Categories field is optionally used to associate any pre-defined attributes with the newly created WebReport.

  6. Optional The Create In field optionally allows the user to select an alternative Content Server location to store the new WebReport.

  7. Optional Click Add.
