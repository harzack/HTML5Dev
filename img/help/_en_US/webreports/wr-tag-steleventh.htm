This section details the sub-tags: MAX, MIN, MODULUS and MULTIPLY.

MAX

Syntax

MAX

Description

This sub-tag returns the maximum value in a list of integers or reals.

Example

[LL_REPTAG_'{-4,2,4}' MAX /] will return 4.

Related Information

For more information, see AVERAGE, LENGTH, MIN, and SUM.

MIN

Syntax

MIN

Description

This sub-tag returns the minimum value in a list of integers or reals.

Example

[LL_REPTAG_'{-4,2,4}' MIN /] will return -4.

Related Information

For more information, see AVERAGE, LENGTH, MAX, and SUM.

MODULUS

Syntax

MODULUS:<modulus number>

Description

Returns the value of a MODULUS operation on the data returned by the main tag.

Example

[LL_REPTAG_'2001' MODULUS:7 /] will return 6.

Related Information

For more information, see ADD, DIVIDE, MULTIPLY, SUBTRACT and TIMESINCE.

MULTIPLY

Syntax

MULTIPLY:<value>

Description

Performs a multiplication operation. Multiplies the main tag by <value>.

Example

ROUND will provide greater precision, [LL_REPTAG_"10" MULTIPLY:2 ROUND:2 /] will resolve to 20.00.

Related Information

For more information, see ADD, DIVIDE, MODULUS, SUBTRACT, and TIMESINCE.
