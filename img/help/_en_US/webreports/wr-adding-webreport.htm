A WebReport is added by selecting WebReport from the standard Content Server Add New Item menu. This option brings up the Add: WebReport page. You will be taken to the Add Items... WebReport screen:

Data source valid options are:

  • LiveReport: the results of a LiveReport query are used.

  • Saved Query: the results of a saved query are used. A saved query is created from advanced search.

  • Form Template: the results of a query to return all the SQL table data associated with the selected template are used.

  • Form: the results of a query to return all SQL table data owned by the selected form are used.

  • Document: a document with either CSV or HTML table based content is used.

  • Folder, Compound Document, Task List, Task Group, Channel, Discussion: all show the children when selected.

  • External Application, typically via HTTP.

For detailed information about how to create a WebReport, see To Create a WebReport.
