To delete a WebReport constant:

  1. Select the Constants tab for the WebReport. From the WebReport's Functions menu select Properties. From the Properties menu, select Constants.

  2. On the WebReport's Properties' page's Constants tab, select the minus button, - next to the the constant you want to delete.

  3. Click OK to confirm you want to delete this constant.

  4. Click Update.

     

    Tip:
    If you want to delete all constant definitions for the entire WebReport, click the red X button, then select Update.
