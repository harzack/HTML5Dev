The ajax.js file packaged with WebReports includes a list of functions designed to aid in the use of WebReport services. The ajax.js file is accessed using a SCRIPT include like this:

<SCRIPT SRC="/<support_dir>/webreports/library/ajax.js"></SCRIPT>

If this file is being included into a WebReport or ActiveView, the LIBPATH tag can be used like this:

<SCRIPT SRC="[LL_REPTAG_LIBPATH /]ajax.js"></SCRIPT>
 

Note:
When using [LL_WEBREPORT_EXCLUDEHTML /] in a WebReport that is also using the AJAX Library Functions the following variable will need to be defined in a script block. The reason is the baseURL variable is defined within the Content Server page code that is inserted into the WebReport output but this is excluded by the tag. The following code will ensure this variable is defined and prevent the AJAX library from breaking.

<script>
	baseURL = "[LL_REPTAG_URLPREFIX /]";
</script>
