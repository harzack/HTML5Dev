There are two ways to create a new version of a reportview. One is by editing a WebReport using the Edit Reportview option on the function menu. This procedure is described in Editing a Reportview Online.

The other way to create a new version is by editing the reportview as a standalone file on the desktop. To retrieve the current version of a reportview, the Download Reportview option can be used.

The reportview can be edited with any text or HTML editor. These editors are used to create the appropriate look and feel for the WebReport and to insert selected WebReport tags into the code. When the necessary changes have been made to the file, it should be saved to the user's desktop and then added to the WebReport using the Add Reportview Version.

It is possible to run versions of a specific WebReport by going to the Properties page and clicking the Versions tab for that report then either clicking the File Name or selecting the appropriate action from the function menu.
