This section details the column data tag NEXTURL.

NEXTURL

Syntax

[LL_REPTAG_NEXTURL /]

Description

A URL which will take the user to the previous page.

Example

It can be used to implement a Return button using HTML. For example: <INPUT TYPE="BUTTON" NAME="return" VALUE="Return" ONCLICK="document.location='[LL_REPTAG_NEXTURL /]' ">
