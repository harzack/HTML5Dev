This section details the column data tags: WEBDAVAUTH, WEBDAVDIR and WEBREPORTNAME.

WEBDAVAUTH

Syntax

[LL_REPTAG_WEBDAVAUTH /]

Description

Returns a personal authorization code for WebDav usage. For more information see the WEBDAV entries for the LLURL tag.

WEBDAVDIR

Syntax

[LL_REPTAG_WEBDAVDIR /]

Description

The Virtual Directory used by the webserver, for example IIS, to handle WebDAV requests within Content Server. For more information see the WEBDAV entries for the LLURL tag.

WEBREPORTNAME

Syntax

[LL_REPTAG_WEBREPORTNAME /]

Description

Inserts the name of the executing WebReport.
