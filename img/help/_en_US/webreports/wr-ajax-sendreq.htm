• This is a generic function that can send an AJAX request to invoke a WebReport. This function is also used by both the updatePage and getJSData functions.

  • Takes the arguments:

    • URLreference (number/String): a required parameter. Either a URL or the numeric objectId (dataId) or the nickname (String) of the WebReport. If the nickname is numeric then it should be enclosed in quotes.

    • responseFunction (function): a required parameter. Function to be executed when the AJAX request returns.

    • reqType ('string','json','xml'): an optional parameter. The type of request to send to the web server (Content Server). Valid types are either GET or POST. The default value is GET.

    • optionalParms (String): an optional parameter. Any parameters to be passed to the target WebReport being executed. optionalParms has the format: &parm1=value1,&parm2=value2...

  • If an error is encountered a descriptive alert message will appear in the browser.

  • For an example, see sendRequest( WebReportId, responseFunction, reqType, optionalParms ) Example.
