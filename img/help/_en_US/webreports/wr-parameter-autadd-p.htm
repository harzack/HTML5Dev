To automatically add a WebReport parameter:

  1. From the WebReport's functions menu select Properties, and then select Parameters.

  2. On the Parameters tab:

    1. Select the button to add all the parameters that are used in the reportview or LiveReport data source. This icon only appears when the reportview or LiveReport contains parameters that are not defined on the Parameters tab.

    2. For all the new parameters, add a parameter name, display text, prompt order, default value and other settings. For information about parameter options, see Parameter Tab Settings.

  3. Select Update.
