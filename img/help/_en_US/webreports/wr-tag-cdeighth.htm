This section details the column data tags: MYCACHEURL, MYID and MYURL.

MYCACHEURL

Syntax

[LL_REPTAG_MYCACHEURL /]

Description

Returns a URL for the currently running WebReport but any URL parameters that would ordinarily be included in the URL, are cached and the cache ID is added to the URL instead. The cache ID is added to the URL in the format: &reqcacheid=<nnnnnnnn>

If this URL is used to execute a request to Content Server, all of the cached parms are added on to the request prior to any WebReports processing.

MYID

Syntax

[LL_REPTAG_MYID /]

Description

Returns the ObjectID of the WebReport that is being run.

MYURL

Syntax

[LL_REPTAG_MYURL /]

Description

Returns the URL that was used to run the executing WebReport. In conjunction with the ESCAPEURL sub-tag, this tag could be used when including a hyperlink to another Content Server item to allow the original WebReport to be re-run. Whenever this tag is used to specify a &nexturl parameter, the ESCAPEURL sub-tag should be used to generate the correct URL escaped format.

Example

The last part of a hyperlink could include: &nexturl=[LL_REPTAG_MYURL ESCAPEURL /]
