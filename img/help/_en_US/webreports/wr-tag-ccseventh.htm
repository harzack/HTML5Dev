This section details the content control tag RUNIF.

RUNIF

Syntax

[LL_WEBREPORT_RUNIF /]

[LL_WEBREPORT_RUNIF (expression) [:USECASE] /]

Description

Causes the processing of a WebReport to be aborted if a condition is met. If the specified expression evaluates to FALSE, then the data source is not run and no processing of the WebReport occurs.

Due to the order in which the tags are processed, the RUNIF tag conditions are limited to constants, parameters, literals, and any of the static Data Tags that are valid in the Header Section.

Variables are not supported in the conditions of the RUNIF tag, since the WebReport will need to run in order to process those variables.

Parameters

Format for (expression): "parmA" <operator> "parmB" <optional clauses>

Example

[LL_WEBREPORT_RUNIF "[LL_REPTAG_$GROUPA USERINGROUP /]" == "TRUE" /]

In this example, if the user running the report is not in group A, as referenced by a constant, then the report does not run. This tag only works in the header section. In other words, before the [LL_WEBREPORT_STARTROW /] tag.

Related Information

For more information, see INCLUDEIF.
