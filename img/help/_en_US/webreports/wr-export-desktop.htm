Select Desktop as the export destination, then select the Export Mime Type for the resulting export file. The Mime type will be used by WebReports to determine the file extension, and therefore the file type, to be downloaded. This is ultimately used to determine the application that will open the exported file. When you click Export, you are presented with a download dialogue box. At this point, it is possible to change the default filename that will be set to the name of the corresponding WebReport.

In the Export to Desktop screen:

  1. From the Output Destination list, select Desktop.

  2. In the Download File Name field, type the export filename.

  3. From the Export MIME Type list, select the export format.

For more information, see Exporting to the Desktop - Parameters and Example.
