To manually add a WebReport parameter:

  1. From the WebReport's functions menu select Properties, and then select Parameters.

  2. On the Parameters tab:

    1. Select the + button to add a new parameter.

       

      Tip:
      If no other parameters have been defined, the Click here to add new parameter option can also be used.

       

    2. In the new, blank row that is created, add a parameter name, display text, prompt order, default value, and other settings. For information about parameter options, see Parameter Tab Settings.

    3. Select Update.
