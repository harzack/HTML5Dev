In order to convert a document, all a user needs to do is check the Use External Conversion Engine on the Destination tab of any WebReport and it will be converted into PDF format.

 

Note:
PDF conversion is not instantaneous. The user may need to wait for several minutes for a report to appear at the destination location.

 

 

Note:
Before PDF conversion can be used, your Content Server administrator must configure options in the Content Server admin pages:

 

If you are an administrator, see Manage WebReports Conversion in the administration help.
