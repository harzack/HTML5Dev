An example of a simple reportview file follows, in which tags appear in bold. In this example the report generates a summary of customer issues that have been captured using workflows and forms. The data source in this case is a LiveReport. Note how the tags in the row template section specify which LiveReport column to obtain the data from. They do this by specifying the column number [LL_REPTAG_1 /] or by specifying the LiveReport column name [LL_REPTAG=Customer /]:
