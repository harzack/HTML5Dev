To access the dynamic tag guide:

  1. From any WebReport's functions menu, do one of the following:

    • Select Open Tag Guide directly from the functions menu.

    • Select Edit Reportview. On the Edit WebReport page, on the WebReport's tool bar, select Tag Guide.

  2. The dynamic tag guide will open in a new page. On the WebReports Tag Guide page, double click anywhere on a row to expand or contract the specific description.

  3. When you have finished using the dynamic tag guide, close the page.
