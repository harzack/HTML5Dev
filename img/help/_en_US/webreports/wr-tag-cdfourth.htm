This section details the column data tags: DATASOURCE, DATE, DATEFORMATS, DATETIME, and DBTYPE.

DATASOURCE

Syntax

[LL_REPTAG_DATASOURCE /]

Description

Returns the data source as a RecArray. This can be used for special processing outside the main row section.

Example

The following example shows how the LL_WEBREPORT_FOR loop tag can be used to iterate each row.

[LL_WEBREPORT_FOR DATA:[LL_REPTAG_DATASOURCE /] VAR:row /]
   [LL_REPTAG_%row CURRENTVAL RECORD:NAME /]
[LL_WEBREPORT_ENDFOR /]

Related Information

See also SETDATASOURCE, [LL_WEBREPORT_FOR VAR:varName DATA:collection /], RECORD, and RECARRAY.

DATE

Syntax

[LL_REPTAG_DATE /]

Description

Inserts the date on which the WebReport was executed.

Related Information

See also DATETIME and TODATE. For formatting options see the DATE sub-tag.

DATEFORMATS

Syntax

[LL_REPTAG_DATEFORMATS /]

Description

Content Server has three configurable date masks: Long, Short, and Input. This data tag returns an ASSOC with the system configured date masks, which can be individual retrieved with the ASSOC sub-tag.

The date masks are compatible with Javascript.

Example

For example, [LL_REPTAG_DATEFORMATS /] returns:

A<1,?,'INPUT'='%m/%d/%Y %I:%M %p','LONG'='%m/%d/%Y %I:%M %p','SHORT'='%m/%d/%Y'>

DATETIME

Syntax

[LL_REPTAG_DATETIME /]

Description

Inserts the date and time when the WebReport was executed. For formatting options see the DATE sub-tag.

Related Information

See also DATE and TODATE.

DBTYPE

Syntax

[LL_REPTAG_DBTYPE /]

Description

Returns the type of database being used on the server. Valid return values are 'ORACLE', 'MSSQL', and 'UNKNOWN'.

Related Information

See also DATE and TODATE.
