This section details the content control tag DEBUGON.

DEBUGON

Syntax

[LL_WEBREPORT_DEBUGON /]

Description

The way in which tags are created and specified in the reportview is validated each time a new version is added to the WebReport. This validation is provided to aid in finding any syntax errors that may exist in the reportview. Sometimes, this validation may interfere with more complicated tag structures.

Advanced users can use the DEBUGON tag to disable the validation in order to allow the use of complex scenarios.

 

Important:
The DEBUGON tag should only be used after basic testing, with validation on, and preferably on a test system, to identify any possible run time errors that have been introduced.
