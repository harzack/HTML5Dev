To reference parameters in the WebReport reportview:

  1. You can drag and drop parameters by selecting the appropriate parameter from the dropdown list and then dragging the icon to the point in the reportview where you want to insert the parameter.

  2. You can open the Parameters tab in a new window by selecting the Parameters hyperlink before the dropdown list.

  3. You can update the list of parameters in the dropdown by clicking the icon. This has the effect of updating the list without reloading the page.
