This section details the column data tag FAVORITES and FILTEREDROWS.

FAVORITES

Syntax

[LL_REPTAG_FAVORITES /]

Description

Returns a RecArray containing the Favorites of the logged-in user.

Example

For example, the following could be used to get the favorites as a comma-separated list:

[LL_REPTAG_FAVORITES PLUCK:NAME FROMLIST /]

Related Information

For more information, see FROMLIST and PLUCK.

FILTEREDROWS

Syntax

[LL_REPTAG_FILTEREDROWS /]

Description

Returns the number of rows remaining in the row section after any WebReports filtering, but before the INCLUDERANGE tag has reduced the number of rows. This differs from ACTUALROWS which is affected by any INCLUDERANGE filtering. This feature is only supported in the header and footer sections of the WebReport.
