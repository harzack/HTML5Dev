Example 1-15: DSstartrow

If the data source has 100 rows of data, and a DSstartrow of 21 is specified, without any DSendrow or DSmaxrows parameters, then only rows 21 through 100 would be used.

Example 1-16: DSendrow

If the data source has 100 rows of data, and a DSendrow of 75 is specified, without any DSstartrow or DSmaxrows parameters, then only rows 1 through 75 will be used.

Example 1-17: DSstartrow, DSendrow, and DSmaxrows

If the parameters DSstartrow=1, DSendrow=75, and DSmaxrows=20 are specified, only rows 1 through 20 will be used.

Further, if the parameters DSstartrow=1, DSendrow=20, and DSmaxrows=100 are specified, only rows 1 through 20 will be used.

Example 1-18: Example of URL invocation

Consider the following:

?func=ll&objId=1234&objAction=RunReport&sourceid=12345&DSstartrow=11&DSendrow=100&DSmaxrows=50

In this example, the data source identified by nodeid 12345 will be used to supply data for the WebReport.

The data used will end at row 50, despite the DSendrow parameter being set to 100, because the DSmaxrows parameter is set to 50.
