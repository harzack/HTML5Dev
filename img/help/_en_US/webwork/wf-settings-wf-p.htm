To specify Workflow settings:

  1. Choose Settings on the Tools menu and then click the Workflow tab.

  2. On the My General Settings page, modify the settings as described in the following table.

    To… Do the following…
    Choose the default page displayed when you initiate a Workflow or view an assignment In the Workflow Assignments area, click one of the following in the Starting Page list:
    •

    General to display the General page.

    •

    Overview to display the Overview page.

    Display Workflows and process instances on the My Workflow and Workflow Status pages by relationship
    		 In the Workflow Statuses area, click one of the following in the Show list:
    •

    All, displays all Workflows for which you are the manager or the initiator, but not System Workflows.

    •

    Initiated, displays all Workflows that you initiate.

    •

    Managed, displays all Workflows that you manage.

    •

    System, displays all system Workflows (available only to the administrator or users with Bypass privileges).

    Display Workflows based on archived state Click the Not Archived radio button to display non-archived Workflows or the Archived radio button to display archived Workflows.
    Display Workflows based on an Expression Builder query In the Workflow Statuses section, click the Expression Builder icon and then create a query.
    Specify how to sort Workflows Click a sort order in the Sort Order list.
    Choose the default tab displayed when you access the My Workflows page In the Default "My Workflows" Tab list, click Statues to display the Workflow Statues or Assignments to display your Workflow Assignments.
    Specify the maximum number of Workflows displayed In the Maximum Items Per Page box, type the default number of Workflows to display.

For information related to this procedure, see Specifying Workflow and Assignment Settings.
