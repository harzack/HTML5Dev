To send an Action Completed message using a Send On button:

  1. In the Action Requested email message, click Send On. The Action Completed email message is automatically generated.

  2. In the Action Completed email message, do any of the following:

    • Attach to the message any documents that you want to submit. If you submit an attachment to a step that cannot accept attachments, your attachment is ignored.

    • Type any comments that you want to add to the step in the email message body.

  3. Send the Action Completed message.

    If Content Server receives and processes the Action Completed message, it returns a confirmation message. The confirmation message lists any attachments successfully received by Content Server. If Content Server cannot process the Action Completed message, it returns an error message.

To send an Action Completed message using a disposition button:

  1. In the Action Requested message, click a disposition button. The Action Completed email message is automatically generated.

  2. In the Action Completed email message, do one of the following:

    • If the Action Requested message instructs you to include attachments, attach to the message any documents that you want to submit to the step. If you submit an attachment to a step that cannot accept attachments, your attachments will be ignored.

    • Type any comments that you want to add to the step in the email message body.

  3. Send the Action Completed message.

    If Content Server receives and processes the notification message, it returns a confirmation message. The confirmation message lists any attachments successfully received by Content Server. If your request cannot be processed, Content Server returns an error message explaining the problem encountered.
