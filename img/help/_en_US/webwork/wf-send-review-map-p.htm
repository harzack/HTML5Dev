To send a step to a sub-Workflow for review:

  1. Choose Assignments on the Personal menu.

  2. Click the name of the Workflow step you want to send for review.

  3. On the Workflow Step page, click the Send for Review link.

  4. In the Send for Review area, click Choose Sub-map.

  5. In the Select Sub-map area, navigate to the Workflow Map, and then click its Select link.

  6. Click the Attachments tab, do one of the following, and then click Add to Workflow Definition:

    • Select the None radio button if you do not want to pass attachments between the Workflow and the sub-Workflow.

    • Select the All radio button, and then type a name in the Folder Name box to pass all the attachments between the Workflow and the sub-Workflow.

    • Select the Selected Items radio button, and then select the check box of each attachment you want to pass between the Workflow and the sub-Workflow.

  7. Click the Comments tab, select the check box indicating how you want to pass comments between the Workflow and the sub-Workflow, and then click Add to Workflow Definition.

  8. Click the Attributes tab, map any attributes to the sub-Workflow, and then click Add to Workflow Definition.

  9. Click Send to Sub-map.

 

Tip:
You can also find and open a step by choosing Workflows on the Personal menu, clicking the Assignments tab on the My Workflows page, and then clicking the step name link.

 

For information related to this procedure, see Sending a Step for Review.
