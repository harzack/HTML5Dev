To view Workflow and Assignment information:

  1. Do one of the following:

    • To display all Workflows and EPS processes you have permissions to see, choose Workflows on the Personal menu, and then on the My Workflows page, click the Statuses tab.

    • To display a custom list of Workflows and EPS processes, click the name link of a Content Server Workflow Status item, if available.

    • To display only Workflows initiated from a particular Workflow Map, click the Workflow Map's Functions icon, and then choose Status.

  2. In the Show section, select one of the following from the list:

    • All, displays all Workflows for which you are the manager or the initiator, but not system Workflows.

    • Initiated, displays all Workflows that you initiate.

    • Managed, displays all Workflows that you manage.

    • Initiated, displays all Workflows and EPS processes that you initiate.

    • Managed, displays all Workflows and EPS processes that you manage.

    • Workflow Managed, displays only those Workflows you can manage.

    • Workflow Initiated, displays only those Workflows you have initiated.

    • EPS Managed, displays all EPS process instances that you can manage.

    • EPS Initiated, displays all EPS process instances that you have initiated.

    • System, displays all system Workflows (available only to the administrator or users with Bypass privileges).

  3. In the Show section, click one of the following radio buttons:

    • Not Archived, to see non-completed Workflows.

    • Archived to see only archived Workflows.

  4. Optional In the Show section, click the Express Builder icon to further limit the display of Workflows using a conditional expression. For details, see Using the Expression Builder.

  5. In the Workflows section, click a Workflow name link to view the Workflow details.

  6. Optional To view specific information about a step, do one of the following in the Workflow Status page:

    • Click the Step List link, and then click a step name.

    • Click the Map View link, and then double-click a step icon.

 

Tips:

  • You can also view a custom list of Workflows by clicking the name link of a Workflow status or all Workflows initiated from a Workflow Map by clicking a Workflow Map's Functions icon, and then choosing Status.

  • You can view step information by clicking the Step List link on the Workflow Status page, and then clicking a step name link.

 

For information related to this procedure, see Monitoring Workflows and Assignments.
