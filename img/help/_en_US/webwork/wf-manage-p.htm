To manage Workflows:

  1. Choose Workflows on the Personal menu.

  2. On the My Workflows page, modify the settings as described in the following table:

    To… Do the following…
    Stop a Workflow Select the check box next to the Workflow you want to stop, and then click Stop.
    Suspend a Workflow Select the Workflows Functions icon of the Workflow you want to suspend, and then select Suspend.
    Resume a Workflow Select the Workflows Functions icon of the Workflow you want to resume, and then select Resume.
    Archive a Workflow Select the check box next to the Workflow you want to archive, and then click Archive.
    Delete a Workflow Select the check box next to the Workflow you want to delete, and then click Delete.

For information related to this procedure, see Managing Workflows.
