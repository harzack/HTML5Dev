To create a Workflow status item:

  1. In the container where you want to create the Workflow status, click Workflow Status on the Add Item list.

  2. On the Add Workflow Status page, type the name and description of the Workflow status.

  3. Click one of the following in the Relationship list:

    • All, displays all Workflows for which you are the manager or the initiator, but not system Workflows.

    • Initiated, displays all Workflows that you initiate.

    • Managed, displays all Workflows that you manage.

    • Initiated, displays all Workflows and EPS processes that you initiate.

    • Managed, displays all Workflows and EPS processes that you manage.

    • Workflow Managed, displays only those Workflows you can manage.

    • Workflow Initiated, displays only those Workflows you have initiated.

    • EPS Managed, displays all EPS process instances that you can manage.

    • EPS Initiated, displays all EPS process instances that you have initiated.

    • System, displays all system Workflows (available only to the administrator or users with Bypass privileges).

  4. Click one of the following radio buttons:

    • Not Archived, to view only executing Workflows.

    • Archived, to view only archived Workflows.

  5. Optional Click the Expression Builder icon to specify additional search criteria. For more information, see Using the Expression Builder.

  6. Optional Click an order in the Sort Order list to sort the list of Workflows. For more information on the sort order options, see Sort Order.

  7. Click Add.

For information related to this procedure, see Using a Content Server Workflow Status Item.
