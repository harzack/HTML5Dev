To specify assignment settings:

  1. Choose Settings on the Tools menu, click the Workflow tab.

  2. On the My General Settings page, modify the settings as described in the following table.

    To… Do the following…
    Choose the default page displayed when you initiate a Workflow or view an assignment In the Workflow Assignments area, click one of the following in the Starting Page list:
    •

    General to display the General page.

    •

    Overview to display the Overview page.

    Set a Workflow proxy who will receive your assignments when you are unavailable In the Proxy area, click the Choose User or Group icon , and then find and select the user you want to act as your proxy.
    Display Workflow assignments based on an Expression Builder query In the Workflow Assignments area, click the Expression Builder icon , and then create a query.
    Display subsequent Workflow assignments assigned to you on your My Assignments page or the Assignments tab Select the Automatically open the next assignment in the same Workflow if assigned to me check box.
    Display tasks assigned to you (the default) on your My Assignments page Select the Include Workflow assignments on "My Assignments" page check box.
    Choose the default tab displayed when you access the My Workflows page In the Default "My Workflows" Tab list, click Statues to display the Workflow statues or Assignments to display your Workflow assignments.
    Specify the maximum number of assignments displayed In the Maximum Items Per Page box, type the default number of Workflow assignments to display.

For information related to this procedure, see Specifying Workflow and Assignment Settings.
