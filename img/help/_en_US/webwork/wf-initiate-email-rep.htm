To initiate an email-enabled Workflow:

  1. Create an email message and address it to an email-enabled Workflow Map.

  2. Specify a Workflow name in the subject line.

  3. Optional Type Start step comments in the message body and attach any documents you want to include in the Workflow.

  4. Send the message. If errors occur, see Troubleshooting.

For information related to this procedure, see Working with Email-Enabled Workflows.
