You must use system tools called eLink alerts and notifications to help you monitor the Workflows in which you are participating. When a Workflow event occurs that corresponds to a notification that you have configured, Content Server periodically sends you a report in an email message along with any other notifications you have configured. When a Workflow event occurs that corresponds to an eLink setting that you have configured, Content Server notifies you by sending an email message that contains an alert about the Workflow event only. You can configure notifications and alerts for the following Workflow events:

  • A Workflow you manage is completed or late.

  • A step assigned to you is ready or late.

Working with eLink Settings

If you do not manage your personal eLink settings, the Content Server administrator can control eLink settings for you and all system users by configuring global eLink Workflow settings. The Content Server administrator's global eLink Workflow settings override your eLink Workflow settings.

You must configure your personal eLink settings in a browser. For more information about configuring your eLink Workflow event settings and global eLink Workflow settings, contact your system administrator.

 

Important:
You or the Content Server administrator must enable eLink alerts for Workflow events before you will receive messages from email-enabled Workflows.
