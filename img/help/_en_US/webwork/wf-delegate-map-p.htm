To delegate a step to a sub-Workflow:

  1. Choose Workflows on the Personal menu, click the Assignments tab on the My Workflows page, and then click the Workflow name link.

  2. Click the Delegate to link.

  3. In the Delegate To area, click Choose Sub-map.

  4. In the Select Sub-map area, navigate to the Workflow Map to which you want to delegate your task, and then click its Select link.

  5. Click the Attachments tab, do one of the following, and then click Add to Workflow Definition:

    • Select the None radio button if you do not want to pass attachments between the Workflow and the sub-Workflow.

    • Select the All radio button, and then type a name in the Folder Name field to pass all the attachments between the Workflow and the sub-Workflow.

    • Select the Selected Items radio button, and then select the check box of each attachment you want to pass between the main Workflow and the sub-Workflow.

  6. Click the Comments tab, select the check boxes indicating how you want to pass comments between the Workflow and the sub-Workflow, and then click Add to Workflow Definition.

  7. Click the Attributes tab, and then do one of the following:

    • Click Auto Link if you want Content Server to automatically map compatible attribute data types.

    • In the Sub-Map lists, click the name of each attribute that you want to map to those in the Main Map column.

  8. Optional Map attributes to the sub-Workflow, and then click Add to Workflow Definition.

  9. Click Send to Sub-map.

For information related to this procedure, see Delegating a Step.
