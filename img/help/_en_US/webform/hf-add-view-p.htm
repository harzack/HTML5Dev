To add an HTML file as a view:

  1. Click the Functions icon of the Form Template to which you want to add the custom view, choose Properties, and then choose Views.

  2. On the Views tab of the Properties page, click HTML in the Add View menu.

  3. In the HTML File field, click Browse, and then select the HTML file you want to add as the view.

  4. Type a name in the Name field.

  5. Type a description in the Description field.

  6. Click Add View.

For information related to this procedure, see Adding an HTML File as a View.
