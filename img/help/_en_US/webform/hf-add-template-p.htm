To add an HTML Form Template:

  1. Click Form Template on the Add Item menu.

  2. If the optional PDF Forms module is installed, a Template Type page will appear. Click the HTML radio button, and then click Continue.

  3. On the Add Template page, type the Form Template name in the Name box.

  4. Type a description of the Form Template in the Description box.

  5. Optional Click Browse Content Server, navigate to the container where you want to store your template, and then click the container's Select link.

  6. Click Add.

For information related to this procedure, see Adding an HTML Form Template.
