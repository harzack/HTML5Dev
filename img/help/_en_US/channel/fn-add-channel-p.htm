To add a Channel:

  1. On the Add Item menu, click Channel.

    The Add: Channel page appears.

  2. The Name field is required. The remaining fields are optional.

  3. If you wish to attach an image to your new Channel, click the Browse Content Server button and navigate to that image. Click its Select link.

  4. Optionally, type a description of the image in the ALT Text field.

  5. Optionally, specify any other general item settings.

  6. Click the Add button.
