To add News to a Channel:

Do one of the following:

  • Click any Channel's Functions icon then choose Make News. You will see the Add: News page.

    If you use the Make News function on an existing item in Content Server, the Add Content Server Attachment field is pre-populated with the item's location.

  • Open the Channel to which you want to add the News item. For information on how to open a channel, see To Open a Channel.

    Click the Add Item button on the Channel browse view toolbar. Click News. You will see the Add: News page.

Next, enter the following information:

  1. On the Add: News page, type a name for the News item in the Name field.

  2. Optional Type a description of this News item in the Description field.

  3. Click an Effective Date and Expiration Date in the corresponding month, day, year, and time lists. If you do not put values in for the start and end dates of this News item, the News item will not be visible to users.

  4. Type a headline for the News item in the Headline field. If you leave the Headline field blank, the text in the Name field is used as the headline.

  5. Optional Add Highlights for the News item. You can enter multiple lines of text. Each line of the Highlights field will be a separate bullet point on the Channel summary page.

  6. Type the text of your News item in the Story field. You can also copy and paste text, images, and links from a web page.

  7. If you want to attach an image from Content Server to the News item, in the Image field, click the Browse Content Server button. Navigate to the image and click its Select link.

  8. If you want to add a description of the image, for users who are unable to see it, for example, type a description in the ALT Text field.

  9. If you want to add an attachment from Content Server to the News item, in the Add Content Server Attachment field, click the Browse Content Server button. Navigate to the item and click its Select link.

  10. If you want to add an attachment from your computer to the News item, in the Add Desktop Attachment field, click the Browse button. Navigate to the item and click the Open button.

  11. Click the Add button to save your changes and return to the previous page. Click the Reset button to clear all fields.
