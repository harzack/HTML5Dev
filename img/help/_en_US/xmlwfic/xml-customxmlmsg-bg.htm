All actions that send and accept XML messages have filter and format properties that can be used to customize the messages that are sent and received.

You can use filter properties to include or exclude data items from messages it accepts or produces. When customizing the messages that are sent and received, information from the Workflow can be broken down into individual Workflow items, each of which can be controlled individually.

The format properties let you apply XSLT stylesheets and XML namespaces to change the basic XML representation of the data items.
