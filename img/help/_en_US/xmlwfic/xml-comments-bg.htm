The comments package contains the comments that are associated with the steps of the Workflow when they are performed.

Comments from previous steps include the step name, user name, and comment date.

XML Representation

<comments>

<comment>…</comment>

<comment stepname="…" username="…" datesaved="…">…</comment>

</comments>

In XML, the comments package is a comments element with one comment child element for each step.

XML Namespaces (if used)

<comments xmlns="http://www.opentext.com/llx/workflow/">

HTML Representation

comments_text_1=…>

comments_text_2=…>

comments_username_2=…>

comments_stepname_2=…>

comments_datesaved_2=…>

In HTML, comments fields are grouped and numbered by step. The comments for the current step are always in the first group.
