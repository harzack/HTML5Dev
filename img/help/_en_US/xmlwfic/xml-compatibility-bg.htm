This versions of XML Workflow Interchange is easily customized and has slightly different default message formats than previous versions. If you want to exchange XML messages with previous versions of XML Workflow Interchange consider the following:

  • Previous versions always use XML namespaces. In the Format page for each action that will exchange messages with a pre-1.1.0 version of XML Workflow Interchange, always select the Use and require XML namespaces check box.

  • Previous versions convert spaces to underscores in Content Server multi-valued attribute names. You should ensure that any attributes exchanged with pre-1.1.0 versions have names without spaces.

  • Previous versions always produce attribute schemas on outgoing messages but do not require them to be present on incoming messages. If you are transmitting XML messages to custom processes that were written to work with a previous version and if these processes require this information, then select the Include attribute schemas in outgoing messages check box for actions that send such messages.

  • If you use or import a Workflow Map that includes Workflow Interchange steps created with previous versions, the appropriate options will be selected automatically to maintain compatibility.
