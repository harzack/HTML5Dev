For every event that causes a message to be sent and every event corresponding to a message receipt, you can choose and configure the message processing action that takes place.

Different actions send or accept messages in various data formats. Each type of message supports a different set of configurable properties that control how the action operates.

All actions that send and accept HTML form data messages have a Format page that lets you filter and customize the messages that are sent and received.

When customizing the messages that are sent and received, information from the Workflow can be broken down into individual Workflow items, each of which can be controlled individually.
