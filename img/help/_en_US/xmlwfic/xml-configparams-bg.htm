To configure the general parameters:

  1. On the Workflow Map, double-click the Workflow Interchange step.

  2. On the General tab, edit the name of the task in the Step Name field.

    The name that you specify is used to identify the name of the step on the Workflow Map, as well as the name of the task in the assignee's assignments.

  3. Type instructions for completing the task in the Instructions field.

    The instructions that you specify are displayed to the assignee and are transmitted in any initiation messages sent by the task when it becomes active.

  4. In the Assignee field, click the Choose User icon to search for and select an assignee.

  5. Click Add to Workflow Definition.

To... Do the following...
Display comments from previous steps and send them with messages Select the Target/Assignee can see all comments check box.
Set an expected duration for the task Type a value in the Duration field.
Start a task at a particular time Click a start date and time in the lists in the Start Date area.

For more information related to this procedure, see Configuring Workflow Interchange General Parameters.
