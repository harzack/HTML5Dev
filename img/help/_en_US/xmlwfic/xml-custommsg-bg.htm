To customize HTML form data messages:

  1. Double-click the Workflow Interchange step and click the Actions tab.

  2. Click the Edit icon for Send an HTML Initiation Message.

  3. Click the Format tab.

  4. In the Format area, select Custom Format in the list. Two types of Workflow items will appear in the breakdown: fields and field groups. A field group is a set of fields and sub-groups that forms a logical unit.

  5. Optional Clear the TaskStatus check box to omit a field from all messages that the action sends and ignores the field in all messages the action receives.

  6. Select the TaskTitle check box to add the task title to all messages sent and received.

     

    Note:
    The Prompt for title at initiation check box in the Start step of the Workflow Map must be selected for this Workflow item.

     

  7. Click a format for each field group in the lists. A breakdown of all the Workflow items that are valid in messages will appear.

  8. Optional Type a name in the HTML form field area for each Workflow item, including Workflow items in field groups, to specify the HTML form field used to hold the field data in messages.

  9. In the Dates area, click one of the following:

    • Single Field, which is a complex value written into a single HTML form field.

    • Multi-field, which is six integer fields to specify the year, month, day, hour, minute, and second.

  10. Click a format in the Booleans area.

  11. Optional If a field or field group has multiple values, numeric indexes are appended to the corresponding HTML form fields to distinguish between the multiple occurrences. For example, the third part of the second section might be named section_part_2_3. If you want to change the numbering scheme for the numeric index, select one of the following in the Format of multi-value names list:

    • name_1_1, name_1â€¦, which includes indexes in the names for all fields that might have multiple values.

    • name, name_1_2, which omits indexes when they would all be 1.

  12. Optional In the Fields area, type a name and value in the Name and Value fields, and then click the Add value icon if you are specifying a custom format for the send action and you want to add HTML form fields to outgoing messages.

  13. Click Apply.

 

Note:
The name of a form field is derived from the base name that you specify, the type of data the field carries, and the value of the Dates and Names properties. For more information about name fields as they appear in form data, see Message Fields and Formats.
