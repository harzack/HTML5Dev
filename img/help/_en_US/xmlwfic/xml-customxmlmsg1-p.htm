To filter XML message content:

  1. Double-click the Workflow Interchange step and click the Actions tab.

  2. Click the Edit icon for Accept an XML Initiation Message.

  3. Click the Filter tab.

  4. Select the check box next to each field that you want to include. Two types of items appear in the breakdown, fields and field groups. A field group is a set of fields and subgroups that forms a logical unit.

     

    Note:
    Fields that you include appear in messages as an XML element or attribute in its default format. For more information about the default format of XML fields, see Message Fields and Formats.

     

  5. Optional Clear the TaskStatus check box to omit a field from all messages that the action sends and ignore the field in all messages the action receives.

  6. Select the TaskTitle check box to add the task title to all messages sent and received.

     

    Note:
    The Prompt for title at initiation check box in the Start step of the Workflow Map must be selected for this Workflow item.

     

  7. Click a format in the list for each field group. A breakdown of all the Workflow items that are valid in messages appears.

  8. Optional Select the Include attribute schemas in outgoing messages check box to specify that every set of Content Server multi-valued attributes contain additional XML content that describes the type and number of values that are allowed for each attribute.

  9. Click Apply.

For more information related to this procedure, see Customizing XML Messages.
