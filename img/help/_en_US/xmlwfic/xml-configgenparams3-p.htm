To configure the XML response to remote initiation action properties:

  1. Double-click the Workflow Interchange step and click the Actions tab.

  2. In the Step Ready area, select the Perform Action list and select Send an XML response to a Remote Initiator.

  3. Click an action's Edit icon .

  4. Type the name of the action in the Name field.

     

    Note:
    The name appears in the list of actions configured for the step but does not appear in any messages. You can use it to describe the purpose of a send action or to distinguish between multiple actions of the same type that are configured on the same step.

     

  5. Select the Complete the initiating task check box if you want the task status field set to “done” in the response message that is created.

     

    Note:
    If this check box is selected in the message customization options, the source system will complete and send on the task that initiated the current Workflow. For more information about initiating tasks, see Message Fields and Formats.

     

  6. Click Apply.

For more information related to this procedure, see Configuring General Action Parameters.
