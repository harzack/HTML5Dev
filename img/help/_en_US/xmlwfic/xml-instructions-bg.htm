An initiation message includes the general instructions provided for the source task.

XML Representation

<instructions startdate="…" duedate="…">…</instructions>

In XML, instructions are formatted as an instructions element. The content of the element is the text of the instructions. If the Workflow step has a start date and due date, they are provided in attributes.

XML Namespaces (if used)

<instructions xmlns="http://www.opentext.com/llx/workflow/">

HTML Representation:

instructions_startdate=…

instructions_duedate=…

instructions_text=…

In HTML form data, the start date, due date, and instructions are separate fields.
