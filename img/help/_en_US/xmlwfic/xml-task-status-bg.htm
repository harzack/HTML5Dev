A task status field indicates whether the task it refers to is active or completed. The value of the field is either active or done.

A task edit message or a reply to an initiation message can send on a task by setting its status to done.

XML Representation

<… status= "…">

In XML, the task status field is an attribute that can appear on initiate, edittask, or OK elements.

HTML Representation

taskstatus=…

In HTML form data, the task status is carried in a single field.
