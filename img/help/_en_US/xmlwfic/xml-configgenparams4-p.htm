To configure XML accept action properties:

  1. Double-click the Workflow Interchange step and click the Actions tab.

  2. In the Step Completion area, select the Reject It list and select Accept an HTML Task Edit Message or Accept an XML Task Edit Message.

  3. Click an action's Edit icon .

  4. Click one of the following options in the When errors are encountered while processing the request list:

    • Return HTTP error codes, which returns a standard HTTP error status code if any errors prevent successful processing of the request.

    • Return XML error messages, which reports processing errors using XML error messages. Errors that prevent successful routing of the request return a report with HTTP codes.

  5. Click Apply.

For more information related to this procedure, see Configuring General Action Parameters.
