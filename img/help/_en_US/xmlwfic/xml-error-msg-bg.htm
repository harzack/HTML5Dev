An error message can be returned in response to an HTTP request and indicates that the request could not be processed successfully.

XML Representation

<error message="â€¦"/>

When an XML accepting action is configured to generate XML error messages, it returns the error message as error elements wherever possible. Otherwise, errors are returned with HTTP error codes.

A message attribute on the element can be used to describe the error.

XML Namespaces (if used)

<error xmlns="http://www.opentext.com/llx/">

HTML Representation

HTML accepting actions can only return errors as HTTP error codes.
