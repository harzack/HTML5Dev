To create and configure an FAQ:

  1. Select FAQ on the Add Item menu.

  2. On the Add: FAQ page, type the title of the FAQ in the Title box.

  3. Type a description to the FAQ in the Description box.

  4. Specify any other general item settings.

  5. Click Create.

  6. On the Advanced Options page, do any of the following:

    • Select or clear the Enable Comments check box.

    • Select or clear the Enable E-mail Subscriptions check box.

    • Type a topic name in the FAQ Topics box. Click the Add FAQ topics button for each additional topic you want to add.

  7. Click Submit.
