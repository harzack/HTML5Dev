To transfer content to an FAQ:

  1. Open the forum.

  2. Select a topic title.

  3. In a topic reply region, click the Add to FAQ link.

  4. On the Choose FAQ page, click Browse Content Server, navigate to the FAQ to which you want to transfer the content, and then click its Select link.

  5. Click Next.

  6. On the Edit Answer page, click an FAQ topic in the FAQ Topic list, if available.

  7. Edit the question and answer so that they conform with the rest of the FAQ.

  8. Click Next.

  9. On the FAQ Detail page, do any of the following:

    • In the Expiration area, click the Date selector button and then select a month, day, and year to set an expiration date for the entry.

    • In the Refer To area, click the Choose User or Group button and then select the user you want to add as a reference to the entry.

    • To view the FAQ after clicking Finish, select the Go to newly created object check box.

    • In the Desktop Attachment area, click Browse, navigate to the file that you want to add as an attachment, and then click Open. Click Upload.

    • In the Content Server Attachment area, click Browse Content Server, navigate to the item that you want to add as an attachment, and then click the item's Select link.

    • In the URL Attachment area, type the name of the URL in the Name box and type the actual URL in the URL box. Click Add.

  10. Click Finish.
