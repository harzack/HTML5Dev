You can create an FAQ anywhere in Content Server where you have permission to add items. When you create an FAQ, you also configure it. Configuration consists of the following tasks:

  • Writing a title and description that explains the focus of the FAQ.

  • Allowing users to comment on entries.

  • Specifying whether to allow users to subscribe to the FAQ.

  • Specifying FAQ topics. FAQ topics can be applied to entries as a method of categorization. For more information about FAQ topics, see Adding New Content to FAQs.

If you want to configure an FAQ so that other users can add content, you must grant them the appropriate permissions. For more information about permissions, see Managing Permissions.

You can change the configuration settings of an FAQ at any time.
