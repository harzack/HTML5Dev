When you add new content to an FAQ, you add, edit, or comment on an entry. You can add and edit entries if you have the proper permissions. You can only comment on entries if the FAQ has been configured to allow comments. For more information about configuring FAQs, see Creating and Configuring FAQs.

When you add an entry, specify the following features:

  • Add the question and answer for the FAQ.

  • Select which FAQ topics to apply to the entry, if the FAQ has topics added. FAQ topics are broad terms that categorize entries on the FAQ's main page.

  • Specify an expiration date, if available.

  • Add any user references. User references are users who can provide additional information about the entry.

  • Add desktop attachments or Content Server attachments. Desktop attachments are files that you upload from your computer to attach to the FAQ entry. Content Server attachments are Content Server items that you associate with the FAQ entry.

When you edit an entry, you can change any of these elements.

Commenting on an entry allows you to provide feedback on the FAQ's contents. Others can also reply to your comment.
