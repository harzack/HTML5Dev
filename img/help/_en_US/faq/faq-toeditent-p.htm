To edit an FAQ entry:

  1. On the FAQ's main page, click the See Details link for the entry.

  2. On the FAQ Entry page, click the Edit link.

  3. On the Edit FAQ Entry page, do any of the following:

    • Edit the question and answer.

    • Click a FAQ topic in the FAQ Topic list, if available.

    • In the Expiration area, click the Date selector button and then select a month, day, and year to set an expiration date for the entry.

    • In the Refer To area, click the Choose User or Group button and then select the user you want to add as a reference to the entry.

    • Select an item type in the Add Item list to add an item to the entry as a reference. Complete the details and then click Add.

  4. Click Submit.
