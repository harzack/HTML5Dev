To change the presentation settings for a Workspace or Container:

  1. Click a Workspace or Container's Functions icon, choose Properties, and then choose Presentation.

  2. Optional In the # of Featured Sub-Items field, click a radio button to specify the number of Featured Sub-Items that appear as links below a Featured Item.

  3. Optional In the 1st Banner or 2nd Banner area, do the following:

    1. In the Image field, to add or change a banner image, click the Browse Content Server button. Navigate to an image, and then click that image's Select link.

    2. In the ALT Text field, to add a description for the image, type the description.

    3. In the Target URL, to link the image to a Web site, type the URL.

  4. Click Update.

 

Note:
The Presentation Properties page for a Project is unique. For more information, see Customizing a Project Workspace.
