Your personal settings let you specify certain preferences for your own Workspace or system. By default, the following tabs are available on the Settings pages:

  • General, which lets you customize the way you view information. For example:

    • From the Default Start Page list, you can select the page which displays when you first sign in to Content Server.

    • If descriptions are provided in the Description field of the General tab of any item's Properties page, you can choose to display those descriptions when in the Detail Browse view. For more information, see Understanding the User Interface and Viewing and Modifying Item Properties.

    • You can replace the Navigation list with a trail of hyperlinks that represent the hierarchical path to the currently displayed item. For more information, see Understanding the User Interface.

    • You can determine how many days the new, , and modified, , icons appear beside those new or modified items in Content Server. By default, the New icon appears for two days after an item is added to Content Server. The Modified icon appears for seven days after an item has been modified in Content Server.

    • You can use Enhanced Keyboard Accessibility Mode, which allows you to work using keyboard commands instead of a mouse or other pointing device. For more information, see Using Assistive Technology and Java.

    • You can choose to have the Sidebar appear in the browse view. If you choose to display the Sidebar, you can also choose on which side of the page the Sidebar will display. By default, the sidebar will appear on the left.

    • If your administrator has enabled other languages for Content Server, you can change the Metadata Language for your system. This will display names and descriptions which have been entered in that language.

  • Color, which lets you pick a color scheme for your interface.

  • Discussion, which lets you specify your default view of Discussions. For more information, see Changing Discussion Settings.

  • Enterprise Process Services, which lets you change the default Enterprise Process Services settings such as the display duration, viewer, and the option to delegate processes when you are out of the office. For more information, see Changing Enterprise Process Services Settings.

  • Workflow, which lets you manage how you participate in automatically executing work processes. For more information, see Specifying Workflow and Assignment Settings.

  • Password, which lets you maintain your password.

 

Note:
Your administrator may not permit you to change some of these settings.
