To move one or more items:

  1. From the Detail View of any Container, under the Select/Deselect column located to the left of the Type column, select the check box for each item you want to move.

  2. Click the Move button.

  3. Optional On the Move page, in the Items area:

    1. Under the Name heading, type a new name for the item(s).

      If your system has multiple languages installed and enabled, click the edit multilingual values icon to edit the name in the other, enabled, languages.

    2. Under the Options heading, select the Add Version check box.

       

      Note:
      The Add Version check box only appears for items that are versionable. For more information, see Managing Versions and Generations.

       

  4. In the Move to field, click Browse Content Server.... Navigate to the location where you want the item(s) moved, and then click that location's Select link.

  5. In the Categories area:

    1. Click one of the following radio buttons:

      • Original

      • Destination

      • Merged

    2. Optional Click the Edit... button to modify the item's Categories.

  6. Click Move.

  7. Click OK.

 

Note:
For information related to this procedure, see Working with General Items.

For more information about Detail View, see Understanding Workspaces and Containers.

 
 

Tip:
You can move a single item by clicking its Functions icon, and then choosing Move.
