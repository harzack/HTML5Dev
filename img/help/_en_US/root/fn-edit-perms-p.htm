To edit an item's permissions:

  1. Click the item's Functions icon, and then choose Permissions.

  2. On the Permissions page, do one of the following:

    1. If you want to edit the permissions of one of the Default Access groups, in the Default Access area on the left side of the page, click the name of any group whose permissions you want to edit. Now proceed to step 3.

    2. If you want to assign access to an individual or group, in the Assigned Access area, click the name of any user or click the Grant Access... button.

      1. In the Grant Access area on the right side of the page, navigate to the user or group whose permissions you want to edit.

         

        Tip:
        For information about how to find a user or group, see Working with Users and Groups.

         

      2. Select that user or group's Grant Access check box.

      3. Click Submit. Now proceed to step 3.

  3. On the right side of the page, in the Edit <user/group> Permissions dialog box, in the Access area:

    1. Select the check boxes next to the permissions you want to grant to that user or group.

    2. Click Update.

  4. Optional If you want to edit the permissions of another user or group, proceed to step 2. If you have finished, proceed to the next step.

  5. On the left side of the page, click Done.

 

Note:
If you do not have the Edit Permissions permission on an item, you cannot view that item's Permissions page.
