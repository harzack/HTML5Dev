To map Form fields in sub-workflow steps:

  1. In the Workflow Painter, double-click a Sub-workflow step.

  2. Click the Forms tab.

  3. For each Form field in the Main Map column, click a field to map to in the Sub-Map list. Optionally, click Auto Link to automatically map all Form fields which have the same name and are of the same data type in the main Workflow Map and the sub-Workflow Map.

  4. Click Add to Workflow Definition.

  5. Save the Workflow Map.

For information related to this procedure, see Mapping Form Fields in Sub-Workflow Steps.
