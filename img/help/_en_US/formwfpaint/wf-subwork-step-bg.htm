If a Workflow Map using Forms contains a Sub-workflow step, you can map Form fields in the main Workflow to Form fields in the sub-Workflow so that field values pass between the two Workflows. When the sub-Workflow ends, the values of its mapped fields pass back to the main Workflow. You specify how Form fields in the main Workflow correspond to Form fields in the sub-Workflow on the Sub-Workflow.

For more information about defining Sub-workflow steps, see Defining a Sub-Workflow Map Step.
