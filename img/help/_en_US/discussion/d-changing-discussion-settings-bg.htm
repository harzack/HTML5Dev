Content Server gives you the option of changing filter settings for all Discussions that you have permission to open. Filter settings apply to all Discussions, but you can change some filters for an individual Discussion. For more information about changing settings for an individual Discussion, see Adding and Viewing Discussions.

To make changes to filter settings for all Discussions, you modify the following parameters:

  • Posting Preference, which uses the beginning of a Topic's context as the subject of a Reply.

  • Reply Preference, which places the contents of Topics and Replies in quotation marks as you compose a Reply.

  • View Item Options, which shows a Topic's entire thread in context as you compose a Reply.
