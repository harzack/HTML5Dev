To change Discussion settings:

  1. Choose Settings on the Tools menu.

  2. On the My General Settings page, click the Discussion tab.

  3. On the My Discussion Settings page, click an option in the Default Period From list. The default setting is No Time Limit.

  4. In the Default View list, click Nested View or Report View. The default setting is Nested View.

  5. In the Filter Preference field, click one of the following radio buttons: Topics Only, Unread Only, or All. The default setting is Unread Only.

  6. In the Posting Preference field, clear the Use Content Summary as Subject check box if you don't want an excerpt from the beginning of untitled topics and replies used as the subject.

  7. In the Reply Preference field, clear the Quote Comments check box if you don't want to display the contents of the entire thread on the Post Reply page.

  8. In the View Item Options field, clear the Show Threads in Context check box if you don't want to display the Threads in Context section below the text of the current topic or reply.

  9. Click Update.
