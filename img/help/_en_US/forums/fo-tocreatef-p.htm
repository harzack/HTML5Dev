To create and configure a forum:

  1. Click Forum on the Add Item menu.

  2. On the Add: Forum page, type the title of the forum in the Title box.

  3. In the Directories area, select the check boxes of the directories that you want the forum to be associated with.

  4. Specify any other general item settings.

  5. Click Create.

  6. On the Advanced Options page, type the terms of use in the Terms of use box.

  7. Type a post warning in the Post Warning box.

  8. In the Header area, do the following:

    • Select or clear the Display Header in All Views check box.

    • Type and format the header text in the Header box.

  9. Select the Enable Ratings check box to enable ratings.

  10. Select the Enable E-mail Subscriptions check box and type the email address in the E-mail Address for this Forum box.

  11. Click the Choose User or Group button in the Moderator area, and then select a user or group to specify a moderator for the forum.

  12. Select the Require Approval of All Posts check box to require the moderator to approve all posts.

  13. Select the Allow the Forum to Receive and List Questions check box to allow the forum to receive and list questions.

  14. Select the List Urgent Questions Only check box to set the forum to list urgent questions only.

  15. Type a number in the Total Number of Questions Listed in the Forum (Answered and Unanswered) box to specify the total number of questions to list in the forum.

  16. Select Date View, Reply View, or Thread View from the Default View list to specify the default view of the forum. For more information, see Managing Your View of Forums.

  17. Click Submit.
