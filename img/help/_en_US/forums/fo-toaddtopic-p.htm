To add a forum topic:

  1. On the forum's main page, click the New Topic button .

  2. On the Add Forum Topic page, type the title of the topic in the Title box.

  3. Type the body of the topic in the Topic box.

  4. Choose an icon for the topic other than the default icon by selecting a radio button of the icon in the Icon area.

  5. Specify any other general item settings.

  6. Click Next.

  7. On the More Options page, do any of the following:

    • In the Refer To area, click the Choose User or Group icon and then select the user you want to add as a reference to the post.

    • In the Desktop Attachment area, click Browse, navigate to the file that you want to add as an attachment, and then click Open. Click Upload.

    • In the Content Server Attachment area, click Browse Content Server, navigate to the item that you want to add as an attachment, and then click the item's Select link.

    • In the URL Attachment area, type the name of the URL in the Name box and type the actual URL in the URL box.

  8. Click Finish.
