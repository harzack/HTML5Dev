In addition to adding forums as favorites, which allows you to manage a list of your favorite forums along with other Content Server items, you can view and manage your list of personal forums, which includes all the forums to which you have contributed.

You can also change your view of a forum thread as follows:

  • Date view, which sorts the thread by date.

  • Reply view, which indents the posts to clearly reflect which posts are forum topics and which are replies.

  • Thread view, which collapses the thread into a more compact view.

  • Print view, which arranges the thread in a printer-friendly view in a new window.
