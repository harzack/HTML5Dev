You can create a forum anywhere in Content Server where you have permission to add items. When you create a forum, you also configure it. This configuration includes the following options:

  • Defining the forum's terms of use, which users will see when viewing the forum.

  • Defining the forum's post warning, which warns users of possible infractions of the terms of use when they make posts.

  • Allowing users to rate posts.

  • Allowing users to subscribe to the forum.

  • Specifying an email address for the forum's eLink functions. For information about eLink, see Understanding eLink.

  • Specifying a moderator and specifying whether posts must be approved by the moderator before other users can see them. For more information about specifying moderators, see Tips for Managing Moderators.

     

    Note:
    When specifying a moderator, you can select only users who have sufficient permissions to carry out the moderator role in the new forum. Therefore, you must give the user who you want to be the moderator the proper permissions in advance. Similarly, you must remove permissions from a user who has been removed as the moderator. For more information about permissions, see Managing Permissions.

     

  • Specifying whether the forum will receive and list questions. For more information about questions, see Using Questions and Answers.

To specify which users can contribute to a forum, you must grant them the proper permissions.

You can change the configuration settings of a forum at any time. When you do so, you can also define a header for the forum. This header consists of text that is edited and formatted with a rich-text editor.

Tips for Managing Moderators

The simplest way to manage forum moderators and their permissions is to create a moderator role for communities, with sufficient permissions to allow users assigned to that role to be moderators. You can then easily apply permissions by assigning members to the moderator role for all forums to which the role has access.

 

Note:
In this case, all users in the moderator role receive full permissions for all forums to which the role has access. This may not be desirable.

 

If you have users that you only want to have access to specific forums, you can create individual moderator roles for different forums. However, in most cases only one moderator is necessary per forum. Therefore it is advisable to grant one individual user full permissions on the forum.

For more information about moderators, see Moderating Forums.
