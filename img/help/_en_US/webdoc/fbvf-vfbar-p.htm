To use the browse banner:

  1. Once you select a facet, the Dynamic View browse banner appears for all users, located above the sidebar panel. To remove all filters, click the Remove all filters link at the right hand side of the browse bar.

  2. Users who have permission to create a Virtual Folder will also see the Save as Virtual Folder link on the browse banner. To create a Virtual Folder from your facets selection, click the Save as Virtual Folder link. See To Create a Virtual Folder for instructions on how to create a Virtual Folder.

  3. Users who navigate to a previously created Virtual Folder will see the Virtual Folder View browse banner. If you have permission to edit this Virtual Folder, you will see an Edit Virtual Folder link on the browse banner. To edit a Virtual Folder, click the Edit Virtual Folder link. See To Review or Edit a Virtual Folder's Settings for instructions on how to edit a Virtual Folder's settings.
