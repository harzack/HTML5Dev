The Document Overview Page

The Document Overview page provides information about the Document. By default, you are directed to a Document Overview page when you click a Document's link, or an e-mail short link. Your administrator can alter the default behavior of Content Server so that the Document Overview page is not automatically accessed. The Document Overview page is also accessible through a Document's Functions menu.

The information provided by the Document Overview page can include a description and Version information. You can also perform a number of functions on the Document from the Overview page. These functions include:

  • Opening a Document: to open the document, click Edit. For more information about opening a document, see Accessing Documents and Text Documents.

  • Downloading a Document: to download a document, click Download. For more information about downloading a document, see Accessing Documents and Text Documents.

  • E-mailing a link to a Document: to email a link to a document, click E-mail this link. For more information about e-mailing a link to a document, see Assigning Nicknames to Items.

  • Editing a Document: to edit the document, click Edit. For more information about editing documents, see Adding and Editing Documents.

  • Adding Versions: to add a version, click Add Version. For more information about versions, see Working with Versions.

  • Accessing previous Versions: to access a previous version, select the More Actions menu. From the Properties menu, select Versions. For more information about versions, see Working with Versions.

  • Adding a Rating to a Document: to add a rating to a document, click Rate this item. For more information about ratings, see Working with Ratings.

  • Applying Notifications to a Document: to apply notifications to a document, click Notify me of changes. For more information about notifications, see Using Notifications.

  • Accessing a Document's Properties pages: to access a document's Properties page, click View Properties. For more information about Properties pages, see Viewing and Modifying Item Properties.

  • Editing a Document's description: to add or edit a document's description, click Edit description.

     

    Tip:
    The description field for a document can also be accessed from that document's functions menu. From the functions menu's Properties menu, select General. On the General tab of the Properties page, you can add or edit the Description field.

     

The functions available to you on the Document Overview page are based on your permissions and the type of versioning, whether standard or advanced, set by the administrator.Advanced versioning changes the available functions, depending on whether the Document's latest Version is a major or minor Version. For example, if the most recent Version of a Document is a minor Version, you cannot edit the associated major Version; you will only be able to edit and add Versions to the current minor Version. For more information about standard and advanced versioning, see Working with Versions.

 

Note:
Advanced versioning Documents are available to read-only users through the Document Overview page, only if a major Version exists.

 

The Version Overview Page

By default, you are directed to a Version Overview page when you click a Version's link on the Versions tab of a Document's Properties page. It is also accessible through a Version's Functions menu. The Version Overview page provides a description of the Version, and enables you to perform a number of functions on the Version. These functions include:

  • Opening a Version. For more information, see Working with Versions.

  • Downloading a Version. For more information, see Working with Versions.

  • Accessing a Version's Properties pages. For more information, see Viewing and Modifying Item Properties.
