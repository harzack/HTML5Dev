To purge Versions:

  1. Click an item's Functions icon, choose Properties, and then choose Versions.

  2. On the Versions tab of the item's Properties page, in the Purge all versions except the most recent field, type an integer greater than 0.

     

    Note:
    If this item has advanced versioning applied, the field name may read: Purge all minor versions except the most recent.

     

  3. Click Purge Versions.

  4. Click OK.
