To create a Custom View in a Virtual Folder:

  1. Navigate to the Virtual Folder in which you want to create a Custom View.

  2. From the Add Item menu, select Custom View.

  3. On the Add: Custom View page, do one of the following:

    • Click the Existing radio button, next click the Browse button to navigate to the HTML file you created that defines the Custom View, and then click Open.

    • Click the Template radio button, next click the Browse Content Server... button to navigate to the template file that defines the Custom View, and then click Open.

  4. In the Name field, type a name for your Custom View.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the name in the other, enabled, languages.

  5. Optional In the Description field, type a description for your Custom View.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the description in the other, enabled, languages.

  6. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Custom View.

  7. Optional If you want to place the Custom View in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Custom View, then click its Select link.

  8. Click Add.

 

Note:
If the HTML file contains references to other files, such as images, you must add those files separately.

For information about editing a Custom View, see Creating and Administering Custom Views.
