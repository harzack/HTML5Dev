To use the Content Filter Sidebar text field:

  1. Navigate to the location that you know contains the item you want to find. If you are unsure which Folder, Project or Workspace contains your item, select Enterprise from the Global Menu Bar, then select Workspace.

  2. In the Filter by name text field, type the name, or part of the name, of the item you want to find.

  3. Click the Submit Search button to the right of the input box.

  4. If an item, or items, is found, the Browse View now contains a list of that item or items. Click the name of an item to view that item.

  5. If an item of that name is not found in the container from which you ran your search, an error message will appear along with a link to allow you to Show all items. Click Show all items to return to the previous view.

To use the Content Filter Sidebar panel filters:

  1. Navigate to the location that you know contains the item you want to find. If you are unsure which Folder, Project or Workspace contains your item, select Enterprise from the Global Menu Bar, then select Workspace.

  2. Select a facet value through which you want to filter all items, and sub-items, contained within this location.

  3. If you see a More... link below the list of any facet's displayed values, this facet is not displaying all available facet values. Click the More... link to display a dialog box which allows you to select from all facet values. Click the check boxes next to all facet values through which you want to filter the items and sub-items. Click Select to apply your facet values selections to filter your current view.

  4. After you have selected the first facet value through which you want to filter all items contained in this location, click your next facet value to use as a filter.

  5. As you select each filter, it will disappear from the filter panel and appear as text under the Filter by name text field. If you want to remove each filter individually, click the X to the right of each facet value. If you want to remove all facet filters at once, select the Remove all filters link.

  6. Continue to click facet filters until you have narrowed your search results to the point that the Browse View is now displaying a short list of results which match all search filters. Click any item's name to open that item. To return to the beginning and start again, select the Remove all filters link.
