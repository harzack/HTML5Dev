To add an existing Document:

  1. From the Add Item menu, select Document.

  2. On the Add: Document page, in the Document section, select the Existing radio button. Click the Browse button to navigate to the file you want to add, and then click the Open button.

     

    1. The Existing radio button displays only on Internet Explorer browsers.

    2. If you want to add a new Document, select the New radio button, then follow the instructions located in To Add a New Document.

     

  3. In the Name field, type a name for the Document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the name in the other, enabled, languages.

  4. Optional In the Description field, type a description for your Document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the description in the other, enabled, languages.

  5. If you are using a Macintosh computer which is not running the OSX operating system, and the file you are composing has a relevant resource fork that you need to preserve, select the MacBinary (set before selecting a file) check box.

     

    Note:
    For more information about adding Macintosh files, see Adding Macintosh Files to Content Server.

     

  6. In the Version Control field, choose a type of versioning by clicking either the Standard - linear versioning or Advanced - major/minor versioning radio button.

  7. Optional If you want to apply a Classification to this Document, from the Classifications list, select Browse Classifications or My Favorites.

    1. In the dialog box which appears, select the classification you will be applying to this Document.

    2. Ensure the Action check box next to the Classification is checked.

    3. Click Submit.

  8. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Document.

  9. Optional If you want to place the Document in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Document, then click its Select link.

  10. Click Add.

  11. If the Edit Attributes page appears after you add the Document, select the check box for each Category you want to assign to the Document, and then click Submit.

 

Tip:
If it has been made available by your administrator, you can also add a document to Content Server by clicking the Add Document button on a Workspace page.
