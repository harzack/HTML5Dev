To make a Document Generation:

  1. Click a Document's Functions icon, and then choose Make Generation.

  2. On the Add: Generation page, in the Name field, type a name for the Generation.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the Generation's name in the other, enabled, languages.

  3. Optional In the Description field, type a description of the Generation you are creating.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the Generation's description in the other, enabled, languages.

  4. Optional From the Version list, if you want to point the Generation to a previous Version, select that previous Version's number.

  5. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Generation.

  6. Optional If you want to place the Generation in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Generation, then click its Select link.

  7. Click Add.
