To add a Column:

  1. On the Global Menu bar, from the Tools menu, select Facets Volume.

  2. Navigate to the custom facet folder in which you want to create your new column, or create a facet folder in the Facets Volume then navigate to that new folder.

  3. From the Add Item menu, select Column.

  4. In the Name field, type a unique name for your new column.

  5. Optional In the Description field, type a description for your new column.

  6. From the Data Source list, choose a data source for your new column.

  7. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this column.

  8. Optional If you want to place the column in a location other than that which appears in the Create In field, click Browse Content Server..., navigate to the container where you want to locate the column, then click its Select link. Only those areas in Content Server that can store a column will be selectable.

  9. Click Add to save your new column and return to the Facets Volume.
