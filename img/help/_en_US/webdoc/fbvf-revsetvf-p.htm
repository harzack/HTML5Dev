To review or edit a Virtual Folder's settings:

  1. Navigate to the container in Content Server where the Virtual Folder is located.

  2. There are two ways for users with permissions to review or edit a Virtual Folder's settings. Do one of the following:

    • from the Virtual Folder's Functions menu, select Properties, then select Specific.

    • click the Virtual Folder's name. On the <your_virtual_folder's_name> page, on the Browse Banner, click the Edit Virtual Folder link.

    You are now in the Specific tab of the Virtual Folder's Properties page.

  3. The Selected Location field details the location from which the Virtual Folder begins its facet browse. This field is not editable.

  4. The Selected Facets field details the facets that the Virtual Folder will display. Once this field is edited by changing the selected facets, a visual indicator appears next to Selected Facets on the Specific tab. If you have the necessary permissions, this field can be edited by selecting the Update Dynamic Contents link:

    1. When you select Update Dynamic Contents, you will see that the browse banner has changed color, and that the title in the browse banner reads Virtual Folder Edit View.

      If you have edit permissions to this Virtual Folder, but you don't have see contents permission to view all the facets which make up this Virtual Folder, the Update Dynamic Contents link will not be displayed. Instead, you will see a message indicating that the Virtual Folder must be updated with the adjusted selected facets before the dynamic contents can be updated.

       

      Caution
      OpenText recommends that you only perform the Virtual Folder update if you are certain that the facets you cannot see are incidental to the definition of this Virtual Folder.

       

    2. After you make changes to your Virtual Folder, select Return to Save Changes.

       

      Note:
      Clicking Return to Save Changes does not save your changes. You need to save your changes in the next step. Clicking Return to Save Changes brings you back to the Specific tab.

       

    3. If you made changes in the Virtual Folder Edit View, you will no longer see the Update Dynamic Contents link.

       

      Note:
      To save the changes made in the Virtual Folder Edit View, you need to select Update on the Specific tab.

       

  5. Optional Select the Show Selected Location check box to ensure that any user accessing the Virtual Folder can see the location from which the Virtual Folder begins its facet browse.

  6. Optional Select the Show Selected Facets check box to ensure that any user accessing the Virtual Folder can see the names of the facets that the Virtual Folder displays.

  7. Optional Select the Show Location Column check box to ensure that any user accessing the Virtual Folder can see the location, in Content Server, of each item that the Virtual Folder displays.

  8. Click Update.

  9. Optional If your Virtual Folder is stored in a parent Virtual Folder, it is listed as a featured item of that parent Virtual Folder by default. To change that:

     

    Important:
    If you change the default setting, you will need to access this sub-Virtual Folder from the Configure page of its parent Virtual Folder. For more information about the Configure page of a Virtual Folder, see To Configure a Virtual Folder.

     

    1. From the sub-Virtual Folder's Functions menu, select Properties, then select General.

    2. On the General tab, in the Display field, select one of List, Featured or Hidden.

    3. Click Update.
