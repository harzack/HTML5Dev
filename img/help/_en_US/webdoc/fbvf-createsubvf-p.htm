To Create a Virtual Folder within a Virtual Folder:

  • When Virtual Folder “A” is located within Virtual Folder “B”, Virtual Folder “A” is called the sub-Virtual Folder and Virtual Folder “B” is called the parent Virtual Folder.

    There are two ways to create a sub-Virtual Folder within a parent Virtual Folder. Choose from one of the following:

    • To create a sub-Virtual Folder from the Add Item menu:

      1. Click the parent Virtual Folder's Functions icon. From the Functions menu, select Properties. From the parent Virtual Folder's Properties page, select theSpecific tab.

      2. In the Selected Facets field, select the Update Dynamic Contents link.

      3. From the Add Item menu, select Virtual Folder.

      4. See To Create a Virtual Folder for instructions on how to fill in the Add: Virtual Folder page.

      5. Click Return to Save Changes, then click Update.

    • To create a sub-Virtual Folder using the Create In field, during the creation of the sub-Virtual Folder as detailed in To Create a Virtual Folder:

      1. In the Create In field, click Browse Content Server....

      2. Navigate to the parent Virtual Folder you want to designate as the storage location for this sub-Virtual Folder, then click the parent Virtual Folder's Select link.
