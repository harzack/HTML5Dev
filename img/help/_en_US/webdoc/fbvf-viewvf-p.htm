To view a Virtual Folder:

  1. Navigate to the container in Content Server where the Virtual Folder is located.

     

    Note:
    Your administrator, or a user who has created a Virtual Folder, may provide you with a link to a Virtual Folder.

     

  2. Click the Virtual Folder's name. In the main view, you will see those items which conform to the settings of the Virtual Folder, provided you have See Contents permission to view them.

  3. In the Content Filter panel of the sidebar, you may see the location in Content Server that the Virtual Folder is set to browse, and the names of the facets being browsed.
