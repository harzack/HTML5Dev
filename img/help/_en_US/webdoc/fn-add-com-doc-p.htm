To add a Compound Document:

  1. From the Add Item menu select Compound Document.

  2. On the Add: Compound Document page, in the Name field, type a name for the Compound Document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the name in the other, enabled, languages.

  3. Optional In the Description field, type a description for your Compound Document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the description in the other, enabled, languages.

  4. Optional If you want to apply a Classification to this Compound Document, from the Classifications list, select Browse Classifications or My Favorites.

    1. In the dialog box which appears, select the classification you will be applying to this Compound Document.

    2. Ensure the Action check box next to the Classification is checked.

    3. Click Submit.

  5. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Compound Document.

  6. Optional If you want to place the Compound Document in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Compound Document, then click its Select link.

  7. Click Add.
