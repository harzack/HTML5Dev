To create an Appearance in a Virtual Folder:

  1. Navigate to the Virtual Folder in which you want to create an Appearance.

  2. From the Add Item menu, select Appearance.

  3. On the Add: Appearance page, in the Name field, type a name for your new Appearance.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the names in the other, enabled, languages.

  4. Optional In the Description field, type a description for your new Appearance.

    If your system has multiple languages installed and enabled, click the click to edit multilingual values button to edit the descriptions in the other, enabled, languages.

  5. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Appearance.

  6. Optional If you want to place the Appearance in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Appearance, then click its Select link.

  7. Click Add.
