A Document is any electronic file that you upload from your computer to the Content Server database. Examples of documents include spreadsheets, memos, HTML pages, images and audio files. Using Content Server functions, you work with documents by adding them to Content Server, editing their contents, opening them for viewing, and managing versions and generations.

A text document is a document whose content is written in plain text; for example, a document created in Windows Notepad. Using Content Server functions, you work with text documents using many of the same functions that you use with other document types; for example, you can add them to Content Server and edit their contents.
