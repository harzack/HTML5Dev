You access the faceted browsing capability from the Content Filter panel of the sidebar. The default setting at install enables the sidebar for all users. It appears on the left side of each page for which it is available. For information about how to customize your sidebar, see To Customize the Content Filter Sidebar.

The Content Filter panel of the sidebar works by dynamically filtering the facets of all items contained within the view from which you use the sidebar. As you navigate through Content Server, the Content Filter panel continuously updates itself to display the facets that are available for you to browse from that location. This includes the facets that apply to items in the view itself and sub-items.

The elements which make up the Content Filter panel are:

  • The Filter by name text field and Submit Search button.

  • The name of the current view which indicates whether you are in a dynamic view or viewing a container, such as a Folder or the Enterprise Workspace View.

    If you are in a dynamic view, you will also see the Virtual Folder Browse Banner.

  • The list of facet filters which have been applied to the location, and links to remove the filters. The list only appears when one or more facet filters have been applied.

  • All filter panels that are available for you to apply to that location. Each filter panel displays the name of the filter as well as a list of facet values. For example, the Owner facet may be listed. Underneath the Owner facet is a list of all userids, or names of individuals, who own items contained within this location. Click one of the userids, or names, and the Browse View will change to show you all items owned by that userid or name.

    The list of facets that appear in this section depends on how your Knowledge Managers have set up faceted browsing. Each filter panel can be collapsed or expanded by the user by clicking on the filter panel title bar. Once the user has collapsed or expanded a filter panel, that filter panel's state is saved as your setting. Your settings are retained as you navigate through Content Server and are saved after you log out.

    Each facet which is available for the location you are viewing will appear as a collapsible bar in the Content Filter panel.

     

    Note:
    Although not part of the Content Filter panel of the sidebar, the Virtual Folder Browse Banner will display by default when a user selects a facet to display. If the Browse Banner is visible, your available options depend on your permissions. By default, the Remove all filters link is available on the Browse Banner. For more information about the Virtual Folder Browse Banner, see Using Virtual Folders.

     

Customizing and Using the Content Filter Sidebar

Some locations in Content Server will neither display the sidebar, nor the ability to activate the sidebar. Generally, the sidebar is available within Folders, Projects and Workspace pages.

There are two ways to use the Content Filter sidebar. The first is to search using the Filter by name text field to type your search terms. The second is to filter your search by clicking the available facets. See To Use the Content Filter Sidebar below for more information.

Unless you are browsing within your own personal Workspace, the sidebar panel will not display items found in personal workspaces.

For information about your Content Filter settings, see To Customize the Content Filter Sidebar.

Working with Personal Columns

Each user may have permission to customize their column settings in their Browse View. If you do not have the necessary permission, contact your administrator.

All of the following must be true in order for a column to appear in the Available Columns box:

  1. The user must have permission to see the column object.

  2. The user must have permission to the column data source.

  3. The setting for Column Availability must be set to Available everywhere for the column object.

  4. The column is not displayed in the Displayed Columns list.

  5. The column is not in the list of Global columns to display.
