To promote a Version to Major:

  1. Click a Document's Functions icon, choose Properties, and then choose Versions.

  2. On the Versions tab of the Document Properties page, click the Version's Functions icon.

  3. From the Version's Functions menu, select Promote to Major.

     

    Note:
    The Promote to Major menu option is only available if the Document was created with a Version Control of Advanced - major/minor versioning.

     

  4. On the Promote to Major page:

    1. Optional In the Description field, type a description for this Major Version.

      The Description field on the Promote to Major page allows you to provide a description of the version you are adding. It is not the description of the object on which this version is based. Version descriptions are not multilingual.

    2. The Versioning field is informational only. It describes the specific version change which will take place when you click Submit.

    3. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Document.

  5. Click Submit.
