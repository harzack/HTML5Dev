To unreserve a Document and add a Version:

  1. Click a reserved Document's Functions icon, and then choose Unreserve.

  2. On the Unreserve page, select the Add New Version check box.

    1. Optional In the Description field, type a description for your new Version.

    2. In the File section, click the Browse button. Navigate to the file you want to add as the latest Version, and then click the Open button.

    3. Optional If the Versioning section is available, click the Add this as a minor Version or Add this as a major Version radio button.

       

      Note:
      The Versioning section is only available if the Document on which you are basing this Version uses advanced versioning.

       

    4. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Version.

  3. Click Submit.

     

    Note:
    For more information related to this procedure, see Working with Versions.
