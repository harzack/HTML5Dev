To Create a Virtual Folder:

  1. There are two ways to create a Virtual Folder. Choose from one of the following:

    • To create a Virtual Folder from the Add Item menu:

      1. Navigate to the container in which you want to create the Virtual Folder.

      2. From the Add Item menu, select Virtual Folder.

    • To create a Virtual Folder by selecting facets:

      1. Navigate to the location in Content Server at which you want to start your facet browse.

      2. Using the Content Filter panel on the sidebar, select the facets you want applied to your Virtual Folder.

        In the event your sidebar is not visible, see To Customize the Content Filter Sidebar.

      3. Once you have completed selecting your facets, in the Dynamic View browse bar, click Save as Virtual Folder.

  2. On the Add: Virtual Folder page, in the Name field, type a name for your Virtual Folder.

  3. Optional In the Description field, type a description for your Virtual Folder.

  4. The Selected Location and Selected Facets fields are not editable on this page.

    If you created your Virtual Folder from the Add Item menu, these fields read: No Location selected and No Facets selected. This will change once a location is selected and facets are applied to this Virtual Folder.

    If you created your new Virtual Folder by selecting facets, the Selected Location field shows the location of the container from which you selected those facets. The Selected Facets field lists the facets you selected.

  5. Optional Select the Show Selected Location check box to display to any user who accesses this Virtual Folder the location of the container from which you selected the facets applied to this Virtual Folder.

    This check box is selected by default.

  6. Optional Select the Show Selected Facets check box to display to any user who accesses this Virtual Folder the facets which comprise the Virtual Folder.

    This check box is selected by default.

  7. Optional Select the Show Location Column check box to display to any user who accesses this Virtual Folder a column which shows the location, in Content Server, of each item which satisfies that Virtual Folder's criteria.

    This check box is selected by default.

  8. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Virtual Folder.

  9. Optional If you want to place the Virtual Folder in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Virtual Folder, then click its Select link.

     

    Note:
    You can create a Virtual Folder within a Virtual Folder using the Create In field of any Add: Virtual Folder page. Navigate to the Virtual Folder you want to designate as the storage location for this sub-Virtual Folder, then click that parent Virtual Folder's Select link.

     

  10. Click Add.
