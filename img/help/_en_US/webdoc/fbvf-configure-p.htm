To configure a Virtual Folder

  1. Navigate to the container in Content Server where the Virtual Folder is located.

  2. From the Virtual Folder's Functions menu, select Configure.

  3. On the <your_virtual_folder_name> page:

    1. If your Virtual Folder has no sub-Virtual Folders, Appearances or Custom Views saved to it, the Type Name column will read: “There are no items.”.

    2. If your Virtual Folder has an item saved to it, you will see the item's name listed under the Type Name heading.

      1. To change the display of the item, select one of the List, Featured or Hidden buttons.

        The default for a sub-Virtual Folder is Featured. Currently, if you select either List or Hidden for a sub-Virtual Folder, the sub-Virtual Folder will be hidden from view. OpenText recommends that you leave the value for a sub-Virtual Folder set to Featured.

      2. Optional To edit the settings of any sub-Virtual Folder, select the sub-Virtual Folder's Functions menu. From the sub-Virtual Folder's Functions menu, select Properties, then select Specific. For more information, see To Review or Edit a Virtual Folder's Settings.

      3. Optional To edit the settings of any Appearance or Custom View, select the item's Functions menu. From the item's Functions menu, select Open. For more information, see Creating and Administering Custom Views and Adding and Editing Appearances.

      4. Optional To delete an item, select the Delete button next to that item. Click Apply, then click OK to confirm you want to delete this item.

  4. Click Update.
