To add a Version:

  1. Click any item's Functions icon, and then choose Add Version.

  2. Optional On the Add: Version page, in the Description field, type a description for this Version.

     

    Note:
    The Description field on the Add: Version page allows you to provide a description of the version you are adding. It is not the description of the object to which you are adding the version.Version descriptions are not multilingual.

     

  3. In the File field, click Browse.... Navigate to the local copy of the file that you will be adding as the new version of this item. Click to highlight that file, and then click Open.

  4. If you are using a Macintosh computer which is not running the OSX operating system, and the file you are uploading has a relevant resource fork that you need to preserve, select the MacBinary (set before selecting a file) check box.

     

    Note:
    For more information about adding Macintosh files, see Adding Macintosh Files to Content Server.

     

  5. Optional Click the Add this as a minor Version or Add this as a major Version radio button.

     

    Note:
    This option only appears for Documents and Text Documents, and only if the Document uses advanced versioning.

     

  6. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this item.

  7. Optional Specify any other general item settings.

  8. Click Add Version.

     

    Tip:
    You can add a Version to a text document by clicking the text document's Edit link. On the Edit: <text_document_name> page, perform the edits to the text document as required, then click Add Version. For information about the Edit link, see Promoted Actions in Understanding the User Interface.
