To specify eLink settings:

  1. Choose eLink on the Personal menu.

  2. On the personal eLink page, click the Modify Settings button .

    To... Do the following...
    Specify vacation settings In the Vacation section, select the month, day and year that you want to resume receiving eLink messages.
    Specify email content format In the E-mail Contents section, select either Text or HTML from the drop-down list.
    Receive an email for topics and replies that you post Select the check box for E-mail me topics/replies that I post.
    Receive eLink notification for events In the Global Events section, select the check box of each event that you want to receive eLink notification for.

  3. Click the Submit button.
