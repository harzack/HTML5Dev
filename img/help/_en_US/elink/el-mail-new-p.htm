To compose an email message:

  1. Choose My Mailbox on the Personal menu.

  2. On the My Mailbox page, click the New Message button .

  3. On the E-mail This Document page, type the email address of each recipient in the To: field, separating addresses with a comma or semicolon.

  4. Optional Type the email address of each recipient you want to send a copy of this email message to in the CC: field. Separate addresses with a comma or semicolon.

  5. Select the Send a copy of sent messages to My Mailbox check box if you want to automatically receive a copy of messages that you send.

  6. Type a subject for this email message in the Subject field.

  7. Type the message text in the Message field.

  8. Click the Send button.

 

Tip:
You can also choose users by clicking the Choose User icon and selecting users from the Add recipients page.
