To add an item using eLink:

  1. Browse to the eLink-enabled project, folder, compound document, or custom container that you want to add an item to.

  2. Click the item's Function icon, and then choose eLink to obtain the email address for the container in which you want to add the item.

  3. In your email program, compose an email using the container's email address, attach the item that you want to add to Content Server, and then send the email.

 

Note:
If you email an item that already exists in the specified container, the system adds a version to the item if you have Reserve permissions to the item.
