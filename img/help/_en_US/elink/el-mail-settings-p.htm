To set up the eLink mailbox:

  1. Choose My Mailbox on the Personal menu.

  2. On the My Mailbox page, click the Configure Settings button.

  3. On the Configure Mail Settings page, do the following:

    • Type the DNS name for your mail server in the POP Server field; for example, mail.co.company.com.

    • Type the Port number for your mail server in the POP Port field; for example, 110.

    • Type the user name that you use to log into your mail server in the User Name field.

    • Type the password that you use to log into your mail server in the Password field.

    • Optionally, select the Send a copy of sent messages to My Mailbox check box if you want to automatically receive a copy of messages that you send

  4. Click the Submit button.
