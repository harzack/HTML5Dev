You can use the eLink My Mailbox functionality to send and manage email just as you would with any email program. My Mailbox provides a basic email client that you can use in Content Server with any POP3 mail server. Once you set up My Mailbox to access your mail server, you can use it to compose and manage email messages. When you manage email messages, you access your mailbox to view, reply to, forward, and delete them. You set up My Mailbox by providing the DNS name and Port number for any POP3 mail server that you have access to and specifying the user name and password that you use for the mail server.
