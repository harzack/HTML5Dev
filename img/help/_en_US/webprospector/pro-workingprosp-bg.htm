When you create Prospectors, you specify parameters that define search criteria. If a Prospector identifies information that matches your search criteria, it adds the corresponding item to a result list where you can access it. Working with Prospectors also involves organizing and deleting results. For more information, see Prospector Parameters or Managing Prospectors Results.

Creating Prospectors

You can create a prospector by using the Live Query Language (LQL) to construct a query that locates the type of information you want to retrieve. For more information about constructing Queries, see Searching Content Server.

After you construct the prospector's Query, you identify the slice that you want the prospector to scan. The prospector then executes its Query to see if new items that are added to the index match the specified search criteria. If a match is made, the prospector adds a link to the corresponding item in its result list. The prospector continues updating its result list as items are added to or removed from the index. Prospectors can scan only those indexes that a Content Server administrator has set up for prospecting.

 

Notes:

  • If your Administrator created Content Server Enterprise Library data sources, you can identify the Enterprise Library [All Versions] slice for a prospector to scan and include Content Server Enterprise Library items in your search results. Prospectors ignore the Enterprise Library [All Versions] slice, and search Enterprise Library [All Variants] slice.

  • Prospecting other Enterprise Library slices will not produce expected results.

 

When you create a prospector, you also set its result list options, which include the number of results displayed and the order in which the results are listed. You can also choose whether Content Server shows or hides summaries of the result list items. If you want Content Server to e-mail you each time the result list is updated, you can set up Notification to deliver e-mail alerts if your administrator has enabled Notification. For more information about Notification , see Using Notifications. For more information about whether or not your system has been configured to notify you by e-mail, contact your administrator.

You can create prospectors in most Content Server locations. For example, you can create them in the Enterprise Workspace, the Project Workspace, and your personal workspace (My Workspace).

Editing Prospectors

When you edit a prospector, you modify the parameters that you originally defined.

Organizing Prospectors

You can organize prospectors in most Content Server locations by moving them or arranging them on tabs. For example, on the Prospectors page, which is accessed from the Personal menu, you can arrange prospectors on the My Prospectors tab (the default tab), or on custom tabs that you create. When you create custom tabs, you can specify the number of prospector results that the tab should display. For more information about managing tabs, see Managing Report Page Tabs.

Deleting Prospectors

If you want to permanently remove a prospector from Content Server, you can delete it the same way you delete other items. For information about deleting items, see Working with General Items.
