To create a prospector:

  1. Click Prospector on the Add Item menu.

  2. Type a name for the prospector in the Name box.

  3. Optional Type a description of the prospector in the Description box.

  4. Click one or more slices in the In list.

  5. Type a query in the Find box.

     

    Note:
    For more information on constructing queries and the Live Query Language (LQL), see Searching Content Server.

     

  6. To expand or narrow your Query, type additional search criteria in the Optional Criteria box.

  7. Specify the following parameters:

    • State

    • Result Options

    • Notification Options

    • Notification Schedule

  8. To add the prospector to a location other than the current container, click the Browse Content Server button beside the Create In box.

  9. Click the Add button.

 

Note:
For information about the parameters you set when creating a prospector, see Prospector Parameters.

 
 

Tip:
You can also create a prospector by clicking the Create Prospector icon on the Prospectors page.
