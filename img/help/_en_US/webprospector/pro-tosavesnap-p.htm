To save a prospector Snapshot:

  1. Click a Prospector's Functions icon, and then choose Open.

  2. Click the Save as Snapshot icon .

  3. On the Add Prospector Snapshot page, type a name for the Snapshot in the Name box.

  4. Optional Type a description of the Snapshot in the Description box.

  5. Click the Browse Content Server button, browse to the location where you want to store the Snapshot, and then click its Select link.

  6. Click the Add button.
