When Prospectors identify information that match your search criteria, they add the corresponding items to result lists, where you can access them. Prospectors result lists change as items that match your search criteria are added or removed from Content Server indexes.

When you work with Prospectors results, you specify how Prospectors information is sorted in them, whether Prospectors results are locked, whether a result list is saved as a Snapshot, or whether prospector results are deleted.

Sorting Results

When you create or edit Prospectors, you can specify whether to sort results by highest score or latest date. However, when you view prospector results, you can temporarily sort results by the following criteria:

  • Score, which lists results in order, starting with the best match.

  • Type, which groups results based on their document type.

  • Name, which lists results alphabetically based on their document name.

  • Date Found, which lists results by the date they were added to the result list.

  • Size, which lists results based on their document size.

  • Location, which groups results based on their Content Server location.

 

Note:
An item's score may vary slightly between Prospectors and Search. For example, if you run a Query in Prospectors and then run the same Query in Search, the score for an individual item may vary slightly between the two results. This is due to a slight modification in the implementation of relevancy ranking between Prospectors and Search.

 

Locking Results

If you want a result to remain on a result list, you can lock the result. Locking results is useful when a prospector Query retrieves more results than the maximum number you specify for that prospector and you do not want results with lower scores to be removed from a result list as items with higher scores are found. When you lock a result, Content Server displays a lock icon beside the result. You can unlock a result at any time.

Saving Snapshots

You save a prospector result list by creating a prospector Snapshot. A prospector Snapshot is a static representation of a prospector's result list at a particular time. It does not change as results are added to or removed from the result list later.

Deleting Results

You can delete a single prospector result, or you can delete all prospector results.
