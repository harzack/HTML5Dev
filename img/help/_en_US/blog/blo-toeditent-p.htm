To edit a blog entry:

  1. On the blog's main page, click the entry's Functions icon, and then click Edit Entry.

  2. On the Edit Blog Entry page, do any of the following:

    • Edit the title and content of the entry.

    • Select or clear any of the topic check boxes in the Topic area, if available.

    • Click the Publish or Draft radio button to change the publishing status.

    • Select or clear the Notify subscribers that this entry has changed check box, if available.

    • Select an item type in the Add Item list to add an attachment to the entry. Complete the required information and then click Add.

  3. Click Submit.
