You can create a blog anywhere in Content Server where you have permission to add items. When you create a blog, you also configure the following options:

  • Set the read and write permissions for the blog. These settings determine who can read the blog and who can add entries.

  • Allow users to rate and comment on entries.

  • Allow users to subscribe to the blog.

  • Specify an email address for the blog's eLink functions. For information about eLink, see Understanding eLink.

  • Add URLs to the blog's main page.

  • Specify blog topics. These blog topics can be applied to entries as a method of categorization. For more information about topics, see Adding Content to Blogs.

 

Note:
When you add a URL to a blog's main page, you are adding a hyperlink to the blog's interface. The name that you type when you add the URL is the name that appears in the blog's Links sidebar.

 

You can change the configuration settings of a blog at any time.
