A blog creator can add and edit blog entries, as well as comment on these entries. A blog reviewer can also comment on a blog entry, or rate the blog entry using a five-point scale. You can perform these tasks only if the creator of the blog has configured it accordingly.

When you add an entry, you can specify the following features:

  • You can select which blog topics to apply to the entry, if the blog has topics added.

    Blog topics are broad terms that categorize entries on the blog's main page. Users can then see this information when viewing entries, and can filter their view of the blog to only show entries on a certain blog topic.

  • You can choose whether to publish the entry, making it immediately visible to other users, or to save it as a draft, making it visible only to yourself until you choose to publish it.

  • You can add desktop attachments or Content Server attachments. Desktop attachments are files that you upload from your computer to attach to the blog entry. Content Server attachments are Content Server items that you associate with the blog entry.

When you edit an entry, you can change the content, change the topics associated with the entry, change the publishing status to publish or draft, and add or delete attachments. Also, for an edited entry marked as published, you can specify that Content Server notify the blog's subscribers that the entry has changed.

Rating and commenting on blog entries allow you to provide feedback to the blog creator. A rating is an evaluation of the entry on a five-point scale. A comment is written feedback to which others can then reply.
