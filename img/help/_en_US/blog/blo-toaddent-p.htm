To add a blog entry:

  1. On the blog's main page, click the Post an Entry link in the Blog Console sidebar.

  2. On the Add: Blog Entry page, type the title of the entry in the Title box.

  3. Type the body of the entry in the Entry box.

     

    Note:
    You can copy text from a Word document and paste it in a blog entry. For more information, see To Paste Content from Other Text Editors.

     

  4. To apply one or more topics to the entry, select the corresponding check boxes in the Topic area.

     

    Note:
    In order to select a topic, topics must be added in the Blog's Settings page. For more information, see To Create and Configure a Blog.

     

  5. Click one of the following radio buttons:

    • Publish, to publish the entry.

    • Draft, to save the entry as a draft.

  6. Specify any other general item settings.

  7. Click Finish.
