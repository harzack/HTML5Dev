To comment on a blog entry:

  1. On the blog's main page, click the Add A Comment link for the entry.

  2. Type the body of the comment in the Comments box.

  3. Click the icon that you want to appear next to the comment in the Comment Icon area.

  4. Specify any other general item settings.

  5. Click Submit.

 

Tips:

  • The Add Comments window stays open after you add comments, allowing you to view other comments for the blog. To return the main blog page, you must close the Add Comments window by clicking the Close button in the top right corner of the window.

  • If you want to edit a comment after you already added it, you can do so by clicking the comment's Functions icon, and then selecting Edit Entry. Make your edits to the comment, and then click Submit.
