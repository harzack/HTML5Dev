To create and configure a blog:

  1. Click Blog on the Add Item menu.

  2. On the Add: Blog page, type the title of the blog in the Name box.

  3. Specify any other general item settings.

  4. Click Create.

  5. On the Blog Settings page, select the appropriate permissions in the Read and Write lists.

  6. Select or clear either of the following check boxes:

    • Enable Comments

    • Enable Ratings

  7. In the Subscriptions area, do the following:

    • Select the Enable E-mail Subscriptions check box.

    • Type an email address in the E-mail Address for this Blog field.

  8. In the Show last area, do one of the following to specify how many entries to show on the blog's main page:

    • Type a number in the Show last box, and then click day's entries or entries in the list.

    • Select the Show all entries check box.

  9. In the Order by area, click one of the following options to specify how the displayed entries will be ordered:

    • date created

    • date last updated

    • contributor

    • title

  10. In the URL area, type the name of a URL you want to add to the blog's main page in the URL box, and then type the address of the URL in the Link box. To add more URLs, click the Add URL button .

  11. Type the name of a blog topic in the Entry Topics box. To add more topics, click the Add Entry topics button .

  12. Click Submit.
