To add Task Groups:

  1. Right-click the Task List to which you want to add the Task Group, click Add Item, and then click Task Group.

  2. On the Add Task Group page, type a name for the Task Group in the Name box.

    Optionally, click a Milestone in the Default Milestone for Tasks list to assign a Milestone to the Task Group.

  3. Click Add.

 

Note:
It is easier to add new Tasks to an existing Task Group than to assign a new Task Group to existing Tasks. If you plan to use a Task Group to organize a Task List, create the Task Group first, and then add the Tasks.
