To set versioning options for Folders and Projects:

  1. Right-click the Folder or Project for which you want to set versioning options, and then click Version Control.

  2. In the Version Control area, click an option to specify whether users can choose a versioning method, or if they are required to use major/minor versioning.

  3. In the Apply To area, click an option to specify whether the Version Control setting applies only to the current item, only to Folders and Projects in the current item, or to both.

  4. Click Submit.
