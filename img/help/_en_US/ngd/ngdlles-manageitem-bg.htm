You can perform a number of Content Server functions from within Enterprise Connect. For example, you may want to manage an item's categories, reserve an item, or create a task list.

 

Notes:

  • If you have hidden an item, such as a Folder, Document, or Task List, it will still appear in the List Pane of both Outlook and Explorer. In addition, hidden items are included in search results.

  • Enterprise Connect does not support the Content Server Featured function.
