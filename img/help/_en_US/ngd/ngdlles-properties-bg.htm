When you view an item in the Reading Pane, a series of tabs appear at the bottom of the pane. These tabs represent the available properties for the selected item. You can also launch a separate Window for a Property by right-clicking an item, choosing Properties, and then choosing the appropriate Property. If you have permissions, you may also be able to edit some of the boxes in the Property tabs. For more information about editing these boxes, see Viewing and Modifying Item Properties.

Depending on the type of item you select, some of the following tabs appear:

  • General, which displays the Name, Type, Create Date, Created By, Display, Description, Size, Modified Date, and Ownership information.

     

    Notes:

    • Depending on your configuration, items that have been hidden such as Folders, Documents, or Task Lists may be fully accessible in the Enterprise Connect tree. You will be able to view hidden items in the List Pane for both Outlook and Explorer and hidden items will be included in search results.

    • Enterprise Connect does not support the Content Server Featured function.

     

  • Specific, which displays specific information about the item, depending on the item type. For example, a Compound Document's release date or a Document's MIME type.

  • Audit, which displays the events, dates, and usernames for actions performed on an item.

  • Categories, which displays the item category and attribute settings. You can modify the Category information from this page.

  • Classifications, which displays the Classifications for the selected items. You can browse for available Classifications from this page.

  • Columns, which displays options for displaying custom columns.

     

    Note:
    You will need to refresh the Enterprise Connect client to apply any changes that you make.

     

  • Link, which displays only for an alias or shortcut of a Compound Document. The tab shows the version type (Current release, Current revision, or Work in process) of the Compound Document that the alias or shortcut references.

  • Presentation, which displays presentation options for Workspaces and Folders. From this tab, you can display images, a news ticker, and sub-catalog items.

  • References, which displays Alias or Generation items that were created from the original item, including Item Type, Name, and Create Date, Time, and Username.

  • Releases, which displays all releases for a corresponding Compound Document, including the release icon, the name and number of the release, the date and time it was created, and the name of the user who created it.

  • Statistics, which displays all the posted topics and replies for a Discussion item, including usernames, number of topics posted, number of replies posted, total number posted, and date of the most recent posting.

  • Versions, which displays the item's versions, including the version number, name and size; the date and time it was created; and username of the user who created it.

  • View, which displays a preview of the selected item.

  • XReference, which displays any cross-references to other Content Server objects.
