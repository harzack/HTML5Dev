To create a Project Template:

  1. Right-click the Project from which you want to create a template, and then click Make Template.

  2. In the Source area of Add Project Template page, click Project, click Browse Content Server, navigate to the Project from which you want to create the Template, and then click its Select link.

  3. Type a Name for the Template in the Name box.

  4. Specify any other general item settings.

  5. Click Add.
