To add a Prospector:

  1. Right-click the Folder to which you want to add a Prospector, click Add Item, and then click Prospector.

  2. On the Add Prospector page, type a name for the Prospector in the Name box.

    Optionally, type a description of the Prospector in the Description box.

  3. Click one or more slices in the In list.

  4. Type a query in the Find box.

  5. Type additional search criteria in the Optional Criteria box to expand or narrow your Query.

  6. Specify the following parameters:

    • State

    • Result Options

    • Notification Options

    • Notification Schedule

  7. Click Browse Content Server beside the Create In box to add the Prospector to a location other than the current container.

  8. Click Add.

 

Tip:
You can modify the settings for existing Prospectors by right-clicking the Prospector, and then choosing Edit.
