To add Text Documents:

  1. Right-click the location where you want to add a Text Document, click Add Item, and then click Text Document.

  2. On the Add Text Document page, type a name for the Text Document in the Name box.

  3. Type the contents of the Text Document in the Text box.

    If available, click the Check Spelling icon to check for spelling errors.

  4. Specify any other general item settings.

  5. Click a MIME type for the text document in the MIME Type list. The default value is text/plain.

  6. Click Add.

 

Tip:
Depending on the version of Content Server that you are using, you may be able to specify and edit HTML MIME types.
