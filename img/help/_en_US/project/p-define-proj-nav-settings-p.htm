To define Project Navigation settings:

  1. Click a Project's Functions icon, choose Properties, and then choose Presentation.

  2. On the Presentation tab of the Project Properties page, click Navigation.

  3. In the Navigation section of the Presentation tab, specify any settings, and then click Submit.

To... Do the following...
Set the default start page Click Overview or Workspace in the Start Page section.
Select the Project navigation style Select the Project Menu or Project Icon Bar, or both.
Select Project Icon Bar items Select the check box for each item you want to appear in the Project Icon Bar.
