To add a Project using a Template:

  1. Click a Project Template's Functions icon, and then choose Create From.

  2. In the General Info section of the Add Project page, type a new name for the Project in the Name field.

  3. Optional Click Next to add or modify information in the other sections of the Add Project page.

  4. Click Finish.

 

Tip:
You can also add a Project using a Template by clicking Project on the Add Item menu, clicking Template in the Create From section, and then navigating to a Project Template by clicking Browse Content Server.
