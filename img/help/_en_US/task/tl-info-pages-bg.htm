You can view information about Task Lists, Tasks, Milestones, and Assignees on the following information pages:

  • Task List Summary, which provides summary information for all items in a Task List.

  • Milestone Information pages, which provide information about individual Milestones and Milestones in a Task List.

    The Tasks Not In Any Milestones page is accessible from the Milestone Information page.

  • Detail Resource List, which provides a summary of every user or group in a Task List that has assigned Tasks.

    The Assignees Detail page is accessible from the Detail Resource List page.

 

Note:
Only a Task's creator, a Project Coordinator, and a Task's assignee can modify information in a Task List and on its information pages.

 

To View Information Pages

To view the... Do the following...
Task List Summary page Click a Task List link. You will be taken to that Task List's page. From the main menu bar, located at the top of the page, click the Task List menu item. Select Summary.
Detail Milestone List page Click a Task List link. You will be taken to that Task List's page. From the main menu bar, located at the top of the page, click the Task List menu item. Select Detail Milestone List.
Tasks Not in Any Milestones page On the Detail Milestone List page, click the Tasks Not in Any Milestones link.
Detail Resource List page Click a Task List link. You will be taken to that Task List's page. From the main menu bar, located at the top of the page, click the Task List menu item. Select Detail Resource List.
Assignee Detail page On the Detail Resource List page, click the View Detail icon located in the Assignee field.
