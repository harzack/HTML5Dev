To add a Milestone to a Task List:

  1. Click a Task List link.

  2. On the Task List page, click the Add Item button, and then select Milestone.

  3. On the Add: Milestone page, type a name for the Milestone in the Name field.

  4. The Target Date field has the current month, date, year, and time filled by default. Optionally, click a month, date, year, and time in the Target Date lists to specify the Task's target completion date.

  5. By default, the Create In field contains the name of the Task List you selected in the first step. To change the Task List within which you will create this Milestone, in the Create In field, click Browse Content Server to select another Task List.

  6. Optional Add a description of the new Milestone in the Description field. Also, in the Categories field, click Edit to select a Category to apply to this Milestone.

  7. Click the Add button.
