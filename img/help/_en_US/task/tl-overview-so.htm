A Task List is a container that allows users to manage and track the details of individual and collective work assignments. A Task List can contain Tasks, Milestones, and Task Groups, and it can be added anywhere in Content Server. Using Task Lists to manage assignments in a Project is especially useful, since it allows the members of a Project to remain informed of a Task's status.
