To add a Task:

  1. Click a Task List, Task, or Task Group link.

  2. On the Task List, Task, or Task Group page, on the Add Item menu, click Task.

  3. On the Add: Task page, type a name for the Task in the Name field.

  4. Click High, Medium, or Low in the Priority drop-down list.

  5. Click a status in the Status drop-down list.

  6. Optional Specify any other general item settings.

  7. Click the Add button.

To... Do the following...
Assign a Task In the Assigned To field, click the Choose User or Group icon. Navigate to a user or group, click the Select > link.
Specify start and due dates for the Task In the Start Date and Due Date lists, click a month, date, year, and time. By default, the date on which you create the task will be placed in the start date.
Specify a priority In the priority field, click low, medium, or high. By default, the field will read medium.
Specify a status In the status field, click pending, in process, issue, on hold, completed, or cancelled. By default, the field will read pending.
Assign a Milestone to a Task If Milestones have been created, you will be able to click a Milestone in the Milestone list.
Add instructions or comments for the Task Type information in the Instructions and Comments text boxes.
Add a Content Server Attachment In the Add Content Server Attachment field, click Browse Content Server. Click the Select > text next to any item you want to attach to your Task.
Add a Desktop Attachment In the Add Desktop Attachment field, click Browse . Navigate to the file on your computer you want to attach, highlight that file and click Open.
Assign a Category In the Categories field, click Edit . In the Edit Categories box, click Add Categories. If your administrator has created categories, navigate to the category and click the Select > text next the category you want to assign to this Task. Click Done.
