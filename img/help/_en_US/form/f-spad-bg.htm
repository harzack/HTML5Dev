If your Form contains a field that has an identical value for many users, you can lock the value into the field by saving the Form as a Stationery Pad. Users can access a Stationery Pad like any Form, but they can complete only the fields that are not locked. When users submit a completed Stationery Pad, Content Server prompts them to save their copy of the Form in a different location.

 

Note:
You cannot use a Stationery Pad if the revision mechanism is set to <None>.

 

When you convert a Form into a Stationery Pad, Content Server locks the data entered in fields on the Form. You can modify and delete data in a Stationery Pad. Also, you can convert a Stationery Pad into a Form.

Administer permission is required to create and maintain Stationery Pads. You must have Edit Stationery Data permission for the Form. For more information about permissions, see Understanding Form Template and Form Permissions.
