To add a Form:

  1. Click Form on the Add Item menu.

  2. On the Add Form page, click Browse Content Server and then find the Form Template on which you want to base the Form.

  3. Click a revision mechanism in the Revision Mechanism list.

  4. Click a submission mechanism in the Submission Mechanism list.

  5. Click Continue.

  6. Type a new name in the Name box.

  7. Optional Type a Form description in the Description box. This description appears on the Form's General Info page.

  8. Optional If available, select a custom view in the Custom View list.

  9. If you selected Initiate Workflow as the submission mechanism, in the Workflow Map field, click Browse Content Server, and then find and select a Workflow Map.

  10. On the Add Form page, click Add.

 

Tip:
Click the Functions icon for a Form Template, and then click Make Form.

 

For information related to this procedure, see Defining a Form.
