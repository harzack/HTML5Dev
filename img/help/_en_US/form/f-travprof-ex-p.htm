Suppose your organization asks employees to fill out a travel profile Form. The company travel agent refers to the profile when making travel arrangements for employees. Occasionally, the travel preferences, such as a window or aisle airplane seat, favorite car rental company, and so on may change.

To set up a travel profile Form:

  1. Create a Form Template that contains the text and fields that you want to include in the travel profile. For more information about defining a Form Template, see Defining a Form Template.

  2. Add a Form based on the Form Template. When you define the Form, use the following settings:

    • In the Revision Mechanism list, click <None>.

    • In the Submission Mechanism list, click User Revisable Records (Content Server) .

    For more information about defining Forms, see Defining a Form.

Using the Travel Profile Example

Open the travel profile Form, supply your preferences, and then submit the current Version of the Form. The next time you open the Form, it displays the previously saved values. If you want to update the profile, resubmit the Form to save the data.

For information related to this procedure, see Using Forms Examples.
