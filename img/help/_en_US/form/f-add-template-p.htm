To add a Form Template:

  1. On the Add Item menu, click Form Template.

  2. Optional If you are prompted for a Template Type, see the online help for the optional PDF Forms module to learn how to install an optional Form Template type.

  3. On the Add Template page, type the Form Template name in the Name box.

  4. Type a description of the Form Template in the Description box.

  5. Optional Click Browse Content Server, navigate to the container where you want to store the Template, and then click the container's Select link

  6. Click Add.

For information related to this procedure, see Defining a Form Template.
