To add a custom view to a Form Template:

  1. Click the Functions icon of the Form Template to which you want to add the view, choose Properties, and then choose Views.

  2. On the Views tab, click the view type you want to add in Add View list.

  3. Optional Type a description of the view in the Description box.

  4. Click Browse and then find and select the file you want to add as the view.

  5. Click Add View.

For information related to this procedure, see Defining Custom Views.
