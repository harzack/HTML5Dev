You can control the behavior of Form Templates and Forms by specifying their permissions. While Form Template permissions are similar to the permissions for other Content Server items, the permissions for Forms fall into the following categories:

  • See, which allows you to view the Form.

  • Use "Content Server Versions" Revision Mechanism, which allows you to store interim Form data by activating the Form's revision mechanism. This permission is available only for Forms that have a revision mechanism. It also requires See permission.

  • Edit Stationery Data, which allows you to edit the Form's data that was locked in as default settings. This permission is available only for Forms that are Stationery Pads. It also requires See permission.

  • Use "..." Submission Mechanism, which allows you to submit final Form data by activating the Form's submission mechanism. This permission is available only for Forms that have a submission mechanism. It also requires See permission.

  • Administer Form, which allows you to perform all other functions on this Form and its stored data. This permission also requires all other Form permissions.

    The revision and submission mechanisms are defined when adding a Form in Content Server.
