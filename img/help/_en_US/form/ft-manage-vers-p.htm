To manage Form Template versions:

  1. Click the Form Template's Functions icon, choose Properties, and then choose Versions.

  2. On the Properties page, modify any of the settings as described in the following table.

    To… Do the following…
    Edit a Version Click the File Name link of the version you want to edit.
    Lock or unlock a Version Click the Functions icon of the Version you want to lock or unlock, and then click Lock or Unlock.
    Delete a Version Click the Functions icon of the Version you want to delete, and then click Delete.
    Purge Versions In the Purge all versions except the most recent field, type the number of the most recent versions you want to keep, and then click Purge Versions to remove the other versions.

For information related to this procedure, see Working with Form Templates.
