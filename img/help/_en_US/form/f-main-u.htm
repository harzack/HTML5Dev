Forms are electronic documents you can use to collect and store data online. You can use Forms to make your organization's forms, such as survey information, questionnaires, and other types available to users. In addition, you can store the data in Content Server or a database where it can be accessed by LiveReports or external applications. For more information about LiveReports, see Working With LiveReports.

When you create a Form, you base it on a Form Template that you or other users have created. The Form derives its layout and design from the template on which it is based, but you can select processing and data-storage options for each individual Form. The configurations you choose depend on how users will use the Form and how you want to store the data.

If you install the optional Web Forms, PDF Forms, or eForms modules, you can create Form Templates and Forms in HTML, Adobe PDF and eForms formats.

For more information about Web Forms, see Using HTML Web Forms.

For more information about PDF Forms, see Using PDF Forms.

For more information about eForms, see Using eForms Management.
