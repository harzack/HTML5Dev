To add an answer to an FAQ:

  1. On a question's main page, click the answer's Add to FAQ link.

     

    Note:
    The answer must be accepted before you can add it to an FAQ.

     

  2. On the Choose FAQ page, click Browse Content Server, navigate to the FAQ to which you want to add the answer, and then click the FAQ's Select link.

  3. Click Next.

  4. On the Edit Answer page, do any of the following:

    • Click an FAQ topic in the FAQ Topic list, if available.

    • Edit the question in the Question box.

    • Edit the answer in the Answer box.

  5. Click Next.

  6. On the FAQ Detail page, do any of the following:

    • In the Expiration area, click Date selector button and then select a month, day, and year to set an expiration date for the entry.

    • In the Refer to area, click the Choose User or Group button and then select the user you want to add as a reference to the entry.

    • In the Desktop Attachment area, click Browse, navigate to the file that you want to add as an attachment, and then click Open. Click Upload.

    • In the Content Server Attachment area, click Browse Content Server, navigate to the item that you want to add as an attachment, and then click the item's Select link.

    • In the URL Attachment area, type the name of the URL in the Name box and type the actual URL in the URL box.

  7. Click Finish.
