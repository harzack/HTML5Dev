To classify items in search results:

  1. On the Search Result page, select the check box for each item you want to classify. In the Select Action list, click Apply Classifications.

  2. On the Classify Search Results page, click one of the following options in the Classify list:

    • Browse Classifications

    • My Favorites

  3. In the Browse Classifications window, or the My Classification Favorites window, navigate to the Classification that you want to assign to the item. Select its check box.

  4. Click the Submit button.

  5. On the Classify Search Results page, click the Apply button.
