To add a Classification Shortcut:

  1. In the Classification Workspace of a Classification Tree, on the Add Item menu, click Classification Shortcut.

  2. On the Add Classification Shortcut page, type a name for the Classification Shortcut in the Name field. Optionally, type a description of the Classification Shortcut in the Description field.

  3. Click the Browse Classification button, navigate to the Classification to which you want the Shortcut to point, and click its Select link. Optionally, click the Edit button to modify the categories or attributes associated with the Classification Shortcut. Optionally, click the Browse Content Server button to select an alternative location in which to add the Shortcut.

  4. Click the Add button.
