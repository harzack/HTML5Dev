To search Classifications using the From Here option:

  1. In the Classifications Workspace, type your Query in the text field on the search bar.

  2. Click From Here in the Select Search Type list.

  3. Click the Go button.

    Your search is applied against all Content Server and Enterprise Library data sources.

     

    Tips:

    • The From Here search option restricts your search to within the Classification or Classification Tree from which you initiate your search. If your search returns no results, click the Refine Your Search button to expand or modify your search criteria. This will open the Advanced Search page.

    • If you want to add additional search criteria to your search, but you want to still restrict your search to within the Classification or Classification Tree, you can use the From Here (Expanded) option in the Select Search Type list instead. For instructions, see To Search Classifications Using the From Here (Expanded) Option
