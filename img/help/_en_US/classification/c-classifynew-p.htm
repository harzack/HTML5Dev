To classify a new item:

  1. In the Enterprise or Personal Workspace, click the item type that you want to add on the Add Item menu.

  2. On the Add page for the item, specify the item settings. Then click one of the following options in the Classify list:

    • Browse Classifications

    • My Favorites

  3. In the Browse Classifications window, or the My Classification Favorites window, navigate to each Classification that you want to assign to the item. Select the corresponding check box.

  4. Click the Submit button.

  5. On the Add page for the item, click the Add button.
