To search Classifications using the From Here (Expanded) option:

  1. In the Classifications Workspace type your Query in the text field on the search bar.

  2. Click From Here (Expanded) in the Select Search Type list.

  3. On the Advanced Search page, the text entered in the search bar appears in the Look For field in the Full Text section If necessary, add additional criteria. Optionally, click All Words, Any Words, Exact Phrase, or Complex Query from the Look For list.

  4. In the Scope section, Within <Classification> appears preselected. Select any additional search slices that you want to run the search against.

  5. In the Browse Content Server section, the Classification location you initiated the search from appears preselected. Optionally, click Browse Content Server to select additional Classifications to which you want to constrain the search.

  6. Select and enter any additional search criteria.

  7. Click the Search button.

 

Tips:

  • When you initiate a search using the From Here (Expanded) option, before the search is performed, the Advanced Search page appears to allow you to add any additional criteria to your search query. By default, the Advanced Search settings are pre-populated based on the Classification or Classification Tree from which you initiated the search, which limits the search results to the specified Classification or Classification Tree location. For more information, see Browsing and Searching Classifications.

  • If you want to restrict your search to a specific Classification or Classification Tree location, but do not need to add additional search criteria, perform your search using the From Here option. For instructions, see To Search Classifications Using the From Here Option
