To remove a defined rule:

  1. Click an itemâ€™s Functions icon, click Properties, and then choose Security Clearance.

  2. On the Security Clearance tab, click the Defined Rules link.

  3. On the Defined Rules tab, locate the defined rule you want to remove in the Action to Perform column, and then click its Remove link.

  4. If you are removing a defined rule that is currently assigned to a container, click one of the following radio buttons in the Update Options window, and then click OK:

    • Apply changes to current item only, which removes the defined rule from the container only.

    • Apply changes to current item and sub-items, which removes the defined rule from the container and all items within the container.
