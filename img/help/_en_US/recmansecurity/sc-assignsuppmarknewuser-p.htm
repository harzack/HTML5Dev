To assign supplemental markings to a new user:

  1. Choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, click User on the Add Item menu.

  3. On the Add New User page, type a name in the Log-in Name field.

  4. Click a department in the Department drop-down list.

  5. In the Available list in the Supplemental Markings section, select the Supplemental Markings that you want to assign to the user, and then click the Add Arrow button.

  6. Specify any of the remaining settings.

  7. Click the Submit button.

 

Note:
For more information about adding a user in Content Server, see Working with Users and Groups.

 
 

Tip:
To add multiple Supplemental Markings from the Available list to the Selected list, press the CTRL key while highlighting the Supplemental Markings you want to add.
