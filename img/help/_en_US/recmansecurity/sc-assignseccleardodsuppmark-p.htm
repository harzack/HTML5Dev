To add a new item and assign security clearance when the DoD Supplemental Markings setting is enabled:

  1. Click an object on the Add New Item menu.

  2. In the Supplemental Markings section on the Add page, select any supplemental markings in the Available list, then click the Right Arrow to move the supplemental marking to the Selected list.

     

    Note:
    Only the supplemental markings that are assigned to you display in the Supplemental Markings section.

     

  3. Specify any remaining settings.

  4. Click the Add button.
