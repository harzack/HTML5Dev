Security Clearance lets you assign Security Clearance Levels and Supplemental Markings codes to new or existing objects in Content Server.

When you assign Security Clearance to an object, you are essentially hiding the object from any user who does not have the same Security Clearance Level. The following rules apply:

  • If you have no DoD setting enabled, or both the DoD Supplemental Markings and Security Clearance Levels enabled, you can assign both Security Clearance Levels and Supplemental Markings.

  • If you have the DoD Supplemental Markings setting enabled, you can assign only Supplemental Markings.

The DoD Supplemental Markings and Security Clearance setting also provides you with additional required fields for Security Clearance Levels.

For information about assigning security clearance to users, see Assigning Security Clearance and Supplemental Markings to Users.

 

Note:
The Administrator specifies which object types support security clearance settings.

 

If your administrator enabled the option to inherit Security Clearance Levels and Supplemental Markings on items that are moved or copied, and you have the Edit permission on the item or belong to the designated Security Officers group, the Inherit Supplemental Markings and Security Level field appears when you move or copy the item.

The following actions are available:

  • Supplemental Markings

    • Ignore, which does not update the Supplemental Marking when it is copied or moved

    • Merge, which merges the Supplemental Markings on the item being copied or moved with the Supplemental Marking of the new container

    • Replace, which replaces the Supplemental Marking on the item being copied or moved with the Supplemental Marking of the new container

  • Security Level

    • Ignore, which does not change the Security Level

    • Upgrade, which upgrades the Security Level of the item being copied or moved to the Security Level of the new container

 

Note:
If your administrator set the Security Clearance configuration settings to either DoD chapter 2 or DoD chapter 2 & 3, the Inheritance options do not appear.
