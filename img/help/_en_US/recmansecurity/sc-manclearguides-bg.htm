Security Clearance lets you create clearance guides that users can select when setting security clearance on objects. Clearance guides provide users with pre-populated information about how the security clearance was derived. Once you create a clearance guide, you can edit or delete it.

When creating a clearance guide, you can give the guide a title, date, and name of the organization from where the guide originated. You can also specify where the information is derived from, a security clearance level, a reason for the security clearance, remarks about the clearance guide, and when and under what circumstance the declassification should be performed.

Note:
Only the Security Officer or Administrator can create, edit, and delete clearance guides.
