To upgrade the security clearance level on an object:

  1. Click an itemâ€™s Functions icon, click Properties, and then choose Security Clearance.

  2. In the Security Clearance Level section of the Security Clearance tab, click the original security clearance level and the level you are upgrading to in the Initial Security Clearance Level and Current Security Clearance Level drop-down lists.

  3. Click the Update button.

 

Note:
You can update the Security Clearance level on multiple items at one time in the Browse View by selecting the check box for each item you want to update, and then clicking the Update Security Clearance Level button.

You can update the Security Clearance level on multiple items in a Collection or Search result at one time by selecting the check box for each item you want to update, and then clicking Update Security Clearance Level in the Actions drop-down list.
