To assign supplemental markings to an existing user:

  1. Choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, find the user whose information you want to edit, and then click the Edit link for that user in the Actions column.

  3. In the Supplemental Markings section (on the General tab of the My Profile page), select the Supplemental Markings that you want assigned to the user in the Available list, and then click the Right Arrow button to move it to the Selected list.

  4. Click the Update button.

 

Note:
For more information about finding a user, see Working with Users and Groups.

 
 

Tip:
To add multiple Supplemental Markings from the Available list to the Selected list, press the CTRL key while highlighting the Supplemental Markings you want to add.
