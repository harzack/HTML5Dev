To create a clearance guide:

  1. Type the following URL in your Web browser: 

    http://server/content_server_instance/livelink.exe?func=RecmanSecurity.SecurityGuide
    where

    • server is the name of your server

    • content_server_instance is the name of your Content Server instance

    For example, 

    http://localhost/ContentServer2010/livelink.exe?func=RecmanSecurity.SecurityGuide
    If prompted, type the appropriate password in the Administrator password field, and then click the Log-in button.

    Optionally, on the Set Security Clearance Guide Definitions page, enter a title, date, and organization where the guide originated from in the appropriate fields.

  2. Set Supplemental Markings and specify any additional metadata.

  3. Click the Save button.
