To edit a defined rule:

  1. Click an itemâ€™s Functions icon, click Properties, and then choose Security Clearance.

  2. On the Security Clearance tab, click the Defined Rules link.

  3. On the Defined Rules tab, locate the defined rule you want to edit in the Action to Perform column, and then click its Edit link.

  4. In the Edit Defined Rule window, click a new month, day, year, and time for the action to be performed in the Date to Perform lists.

  5. Optional Change the reason why you are performing the action on the item in the Reason for Rule field.

  6. If you are editing a rule that is assigned to a container, click one of the following radio buttons:

    • Apply changes to current item only

    • Apply changes to current item and sub-items

  7. Click OK.
