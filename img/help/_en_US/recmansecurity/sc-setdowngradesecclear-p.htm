To set the downgrade of a security clearance level on an object:

  1. Click an itemâ€™s Functions icon, choose Properties, and then choose Security Clearance.

  2. On the Security Clearance tab, select any of the following check boxes:

    • Date, which lets you specify the date on which to downgrade the record

    • Event, which lets you specify the event that needs to occur before the record can be downgrade

  3. Optional Type instructions on how to complete the downgrade procedure in the Downgrade Instructions field.

  4. Click the Update button.

 

Note:
If the document is restricted, the information in the Downgrade On section is disregarded.
