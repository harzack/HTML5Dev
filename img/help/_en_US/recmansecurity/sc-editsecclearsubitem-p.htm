To edit the security clearance on a container with sub-items:

  1. Click the Functions icon for a container, click Properties, and then choose Security Clearance.

  2. Click the Set Security Clearance sub-tab, and then do the following:

    • In the Supplemental Markings section, select the Supplemental Markings you want to assign in the Available field, click the Right Arrow button to move it to the Assign field.

    • In the Security Clearance Level section, click a security level in the Current Security Clearance Level drop-down list.

  3. Click one of the following in the Apply To drop-down list:

    • This item, which applies the changes to only the parent object

    • This Item and Sub-items, which applies the changes to the parent object and all child objects

      If you choose This Item and Sub-items, a Sub Items Options section appears.

  4. In the Sub Items Options section, select an action in the Supplemental Marking and Security Clearance drop-down lists.

  5. Click the Update button.

 

Note:
The settings in step three and four display only if the Inherit From setting is enabled for the supported object type you are editing. Supported object type settings are set by the Administrator.

If you change the Security Clearance Level on a parent container to a level that is lower than the level assigned to containers or items within the parent container, those items will retain their current, higher, Security Clearance Level.
