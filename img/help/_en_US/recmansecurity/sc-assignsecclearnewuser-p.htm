To assign a security clearance level to a new user:

  1. Choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, click User on the Add Item menu.

  3. On the Add New User page, type a name in the Log-in Name field.

  4. Click a department in the Department drop-down list.

  5. Click a level in the Security Clearance Level drop-down list.

  6. Specify any of the remaining settings.

  7. Click the Submit button.

 

Note:
For more information about adding a user in Content Server, see Working with Users and Groups.
