To assign a security clearance level to an existing user:

  1. Choose Users & Groups on the Enterprise menu.

  2. On the Users and Groups page, find the user whose information you want to edit, and then click the Edit link for that user in the Actions column.

  3. On the General tab of the My Profile page, click a level in the Security Clearance Level drop-down list.

  4. Click the Update button.

 

Note:
For more information about finding a user, see Working with Users and Groups.
