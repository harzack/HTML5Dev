To update a Security Clearance level on an item using a defined rule:

  1. Click an item’s Functions icon, click Properties, and then choose Security Clearance.

  2. On the Security Clearance tab, click the Defined Rules link.

  3. On the Defined Rules tab, click Update Security Clearance Level in the Action to Perform list.

  4. Click the Security Clearance level you want the item updated with in the Security Clearance Level list.

     

    Note:
    If you click None, the current Security Clearance level will be removed from the item.

     

  5. Select the month, day, year, and time that you want the item’s Security Clearance level updated in Date to Perform lists.

  6. Optional Type the reason why you are performing the action on the item in the Reason for Rule field.

  7. Click Update.

  8. If you are updating the Security Clearance level on a container with sub-items, click one of the following radio buttons in the Update Options window, and then click Close:

    • Apply changes to current item only

    • Apply changes to current item and sub-items
