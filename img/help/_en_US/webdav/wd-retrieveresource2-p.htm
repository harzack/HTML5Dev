To retrieve a Content Server document resource using Microsoft Word as the WebDAV client:

  1. In Content Server, click the Functions icon for a document, select Properties, and then select WebDAV to obtain the document resource's URL.

  2. Open Microsoft Word, and then click Open on the File menu.

  3. Type the document resource's URL in the File name field, and then click Open.

  4. Navigate to the document, and then double-click its name.
