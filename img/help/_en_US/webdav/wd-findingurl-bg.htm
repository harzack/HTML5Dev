An item in Content Server that is accessible through WebDAV will have a WebDAV tab on its Properties page in Content Server. The WebDAV tab displays the WebDAV URL for the Content Server item.

If the WebFolders component is installed on your computer and you click the WebDAV URL link on the WebDAV Properties page for a Content Server Folder, Microsoft Internet Explorer 5.5 (or later) will recognize it as a WebDAV collection and open it with the appropriate WebDAV client. The following scenario illustrates how a WebDAV client is used to open a Content Server Document.

  • You open a Content Server Document that was originally created in Microsoft Word, by following the WebDAV URL link on the WebDAV Properties page for a Content Server Folder that contains it. Because WebFolders is installed on your computer, Microsoft Internet Explorer 5.5 (or later) will open the Folder as a Web folder. From the Web folder, you can double-click to open the Document in Microsoft Word and edit it, rather than opening the Document as a Web page that has read-only permissions.

 

Note:
All browsers may not automatically recognize WebDAV resources. For information, see Retrieving a Content Server Document Resource.
