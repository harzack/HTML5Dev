WebDAV resources consist of collections and documents, which are often displayed by most WebDAV-enabled software as folders and files. A collection resource, like a folder, can contain other resources, including documents and other collections. A document resource, like a file, contains content.

URLs for WebDAV resources reflect this hierarchy in the same way that file paths reflect the folder hierarchy on a local hard drive. For example, to access the resource named somedoc.doc in the Content Server Enterprise Workspace, you append somedoc.doc to the WebDAV URL for the Enterprise Workspace, separated by a forward slash (/):

http://localhost/livelink91dav/enterprise/somedoc.doc.

Every type of item in Content Server is accessible through WebDAV. Each is classified as a document or a collection resource, as shown in the following table.

Document Resources Collections Resources
Documents Folders
News items Compound Documents
Discussion Topics and Replies Channels
LiveReports Discussions
Workflow Maps Projects
