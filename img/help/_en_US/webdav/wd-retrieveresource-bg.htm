WebDAV lets you directly edit documents through WebDAV. WebDAV Edit is available for documents created in Microsoft Word, Microsoft PowerPoint, and Microsoft Excel. It requires the Microsoft SharePoint Client API and Microsoft WebFolders to be installed, and a browser with ActiveX scripting enabled. The Microsoft Office application for the document you are editing must be installed on your computer. The action will fail if the application is set to install upon first use.

Document resources can also be retrieved in other ways, depending on the WebDAV client being used to open the resource. As an example, users can open a Content Server document using Microsoft Word as the WebDAV client.
