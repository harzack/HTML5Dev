Since WebDAV supports numerous applications, working with documents over the Web is as simple as working with documents on your local hard drive. The Content Server WebDAV module allows you to copy files into Content Server, create files in Content Server, or edit existing Content Server files. WebDAV automatically reserves a file when a user opens it, so that changes cannot be made simultaneously by two or more people. When the user closes the file, WebDAV automatically unreserves it and adds a new version. You can view the file structure in WebDAV Folder view, Content Server view, or Microsoft WebFolder view.

WebDAV Folder view requires you to have Microsoft Internet Explorer 5.5 or later installed, along with Microsoft WebFolders. You should also set the Internet Explorer security setting that enables the opening of items from an IFRAME. If you set this setting to Prompt, Internet Explorer will prompt you for confirmation as you perform actions.

The WebFolders client is embedded in an IFRAME in Windows Explorer. This could cause extra authentication prompts. Saving your password at all such prompts will make this feature more convenient to use.

 

Note:
Content Server permissions limit your ability to see or access every object in WebDAV, just as they do in Content Server. When you list the contents of Content Server's main server in WebDAV, you will not see the personal workspaces of other users, even though you may be able to access them in Content Server.
