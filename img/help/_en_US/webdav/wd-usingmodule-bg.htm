Several desktop operating systems and applications support WebDAV. When Content Server is installed on a system, there are two servers: the main server and the admin server. When the WebDAV module is installed on Content Server, the main server becomes a WebDAV collection that contains all the content on that server.

Using WebDAV to access Content Server does not require downloading or installing any Content Server client software. All objects that are accessible through Content Server will have a WebDAV Properties page in Content Server that gives the item's URL. For more information about WebDAV URLs, see Finding the WebDAV URL for Content Server Items.
