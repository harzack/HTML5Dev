Ordinarily, you should not change a URL name. However, because a collection's URL name appears in the URL for each item within it, providing a short URL name for a collection with a long display name can help manage URLs within collections.

You can change an item's URL name manually on the WebDAV Properties page without changing the display name.

 

Note:
After you change a URL name using the WebDAV Properties page, the URL name does not automatically synchronize with the display name.
