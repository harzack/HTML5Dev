When a User Managed mapping is in effect, you can use this page to edit the Principal Names that are mapped to a Content Server user for access to WebDAV resources.

To add a mapping:

  1. Enter a Principal Name in the Principal Name field.

  2. Enter your Content Server password in the Content Server Password for field.

  3. Click Add.

     

    Note:
    If prompted for your user name and password, enter the Principal Name in the User name field e.g. me@THEDOMAIN.NET. For this particular example, you would enter the password for the logon 'me' in the 'THEDOMAIN.NET' domain in the Password field. Press OK to complete the authentication.

     

To delete a mapping:

  1. Select the check box of a Principal Name.

  2. Enter your Content Server password in the Content Server Password for field.

  3. Click Delete.

 

Note:
The Content Server Password for field is not present for users with Administrative privileges.
