WebDAV (Web-based Distributed Authoring and Versioning) is a set of extensions to the HTTP protocol that enables users to collaboratively manage files on remote Web servers.

WebDAV supports virtual development teams by providing an infrastructure that makes client software accessible via the Internet. The Content Server WebDAV module enables authors and editors to directly access Content Server repositories, using a WebDAV client. A WebDAV client is any third-party product that is able to communicate with Content Server through the WebDAV server. These products include: Microsoft Office 2003/2007/2010/2013, Microsoft WebFolders, Adobe products such as Photoshop, Acrobat, and GoLive, and Riverfront's WebDrive.
