You can view information specific to you on your personal report pages. Many of the report pages are divided into tabs, which allow you to organize items based on your own preferences. For more information about report page tabs, see Managing Report Page Tabs.

The following table describes each personal report page.

Report Page Name This report page... For more information, see...
My Assignments
Contains a complete list of Tasks that are assigned to you, including Project Tasks, , and Workflow Tasks, .
 See Working with Tasks and Task Lists and Monitoring Workflows and Assignments.
My Collections Stores the collected shortcuts to Content Server items. See Using Collections.
Discussions Lists all the Discussions that you have been assigned the See permission. See Communicating with Discussions.
My Favorites Stores links to items that you have added as Favorites. See Managing Favorites.
My Groups Lists every group of which you are a member. See Working with Groups.
My Profile Lists information associated with your user account. If you have the Edit Self privilege for a tab, denoted by the Edit icon, , you can edit user information. See Working with Users.
News Channels Lists all Channels that you have been assigned the See permission. See Working with Channels and News.
My Notification Reports Lists your personal Notification reports and provides options for configuring them. See Using Notifications.
My Projects Lists all Projects and Subprojects for which you are a Participant. See Working with Projects.
My Prospectors Lists the Prospectors that you have created. See Using Prospectors.
Recommendations Lists items that Content Server determines are of interest to you. See Using Recommendations and Ratings.
My Reports Lists all LiveReports that you can open, and all Search Queries and Snapshots that you have created, in your Personal Workspace. See Working with LiveReports and Searching Content Server.
My Reserved Items Lists items that are currently reserved by you or by one of your groups. See Working with Documents and Text Documents.
Task Lists Lists all Task Lists that you have been assigned the See permission. See Working with Tasks and Task Lists.
My Workflows Displays the Workflows and Workflow Assignments that you have been assigned the See permission. This includes those initiated from particular Workflow Maps. See Monitoring Workflows and Assignments.
