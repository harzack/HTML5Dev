Each user can access the Recycle Bin from the Personal menu (if so configured) in the Toolbar. The Recycle Bin has a folder structure of the format year, month, and day. The first folder is the year folder. Deleted items are shown in the folder of the day they were deleted. If an item with the same name is deleted several times in the same day, the name is automatically appended with a numerical value (this is also true for documents with the same name that are located in different locations on Content Server). For example, a second instance of readme.txt will be renamed to readme.txt2.
