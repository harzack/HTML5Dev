To delete a code in a table:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Objects Administration link.

  3. On the Physical Objects Administration page, click the Table Maintenance link.

  4. On the Physical Objects Table Maintenance page, click the link for a table.

  5. Click the Delete link in the Actions column of the code you want to delete.
