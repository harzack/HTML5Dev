To request a physical item:

  1. Click the Functions icon for a physical item, choose Properties, and then choose Circulation.

  2. On the Circulation tab of the Physical Item Properties page, click the item's Functions icon, and then choose Request.

    Optionally, on the Request information page, click the Choose User icon in the Requestor section, and then choose a user for whom you want to request the item.

  3. Click a date in the Request Date drop-down lists.

  4. Specify any remaining settings.

  5. Click the Submit button.

 

Note:
If you select the Recurring check box, you must also supply time interval information in the Repeat every drop down lists. If you do not provide this information, you will receive an error message asking you to supply the recurring information.

 
 

Tip:
You can also request multiple physical items in a Collection, advanced Search, or the Browse View of a workspace by selecting the check box next to each item you want to request, and then clicking Request Physical Items in the More Actions drop-down list.
