To generate labels for a storage locator:

  1. Choose Physical Objects on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Storage Management link.

  3. On the Storage Management page, click the Locators link.

  4. On the Locators page, specify any search criteria, and then click the Search button.

  5. In the list of search results, click the link for a locator.

  6. Select the Generate Label check box.

  7. Click a label type in the Label Type drop-down list.

  8. Type the number of copies you want to generate in the Number of Copies field.

  9. Click the Update button.
