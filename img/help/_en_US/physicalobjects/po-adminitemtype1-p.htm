To create a physical item type:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Item Types link.

  3. On the Physical Item Types page, click Physical Item Type on the Add Item menu.

  4. On the Add Physical Item Type page, click a radio button in the Select Icon section.

    Optionally, type a unique ID format in the Unique ID Format field to specify a unique ID for the physical item type.

    Optionally, select the Cannot be borrowed check box to specify that this physical item type cannot be borrowed.

  5. Click one of the following radio buttons:

    • Non Container, which lets you add a physical item type that cannot contain other physical items. Non-containers can have multiple copies.

    • Container, which lets you add a physical item type that can contain multiple physical items. Containers cannot have multiple copies.

    • Box, which lets you add a physical item type that can have assigned physical items but not physical items created within. Boxes cannot have multiple copies.

    Optionally, you can restrict the physical item types that can be stored in the container or box you are creating by selecting the corresponding check boxes in the Physical Item Type section.

  6. Specify any of the remaining setting.

  7. Click the Add Item button.

 

Note:
Conflicts could arise if multiple unique ID formats that start with the same format exist. For example, if a physical item type named Physical Folder is assigned a unique ID format of AB001 and a physical item type named Physical Document is assigned a unique ID format of AB010, the uniquely generated IDs will eventually meet and cause delays to the system.

The icon you choose displays next to each physical item assigned that physical item type.
