To search with physical item attribute constraints:

  1. On the Content Server Search page, click the Physical Item Attributes link in the Advanced Options list.

  2. In the Physical Item Attributes section, click a home location code in the Home Location drop-down list.

    Optionally, click the Duplicate button , choose an operator, and then repeat step 2 to use a Boolean operator.

  3. Click Current Location in the Additional Attributes drop-down list, and then click a current location code in the Current Location drop-down list.

    Optionally, click the Duplicate button , choose an operator, and then repeat step 3 to use a Boolean operator.

  4. Optionally, set a date range by selecting date options from the From /To Dates drop-downs. The date range field is also available for Collections.
