To generate labels from a Collection or Search results:

  1. In a Collection or set of Search results, select the check box beside each physical item for which you want to generate a label.

  2. In the More Actions drop-down list, click Generate Labels.

  3. On the Generate Labels page, select the type of label you want generated for the items in the Collection from the Label Type drop-down list, type the number of copies that you want generated in the Number of Copies field, and then click the Generate button.
