To generate labels for physical items and locators in batches:

  1. Click Physical Objects on the Enterprise menu.

  2. In the Physical Objects Workspace, click the Batch Labels link, and then click the Generate New Labels link.

  3. On the Generate Labels page, click the Physical Items, Locators, or Users radio button to select the object type.

  4. Specify any of the search criteria used to generate the labels.

     

    Note:
    For more information about the available search criteria, see Working with Labels

     

  5. Enter a list of unique IDs that are used to generate labels in the Unique ID List field, or browse to the location of a file with a list of unique IDs.

  6. Select a label type and stock from the Label Type and Label Stock drop-down lists.

  7. Type the starting position for the labels in the Starting position for printing field.

  8. Click the Generate button.
