To edit personal physical objects settings:

  1. Choose Settings on the Tools menu.

  2. On the My General Settings page, click the Physical Objects tab.

  3. On the My Physical Objects Settings page, select the Hide Labels Tab or Hide Circulation Tab check boxes to hide these labels from a Physical Item's Properties page.

  4. Select the Return Items to Home Location check box.

  5. Select the Always send e-mail on request check box to enable automatic e-mail notifications to be sent every time a Physical Item is requested.

  6. In the Default Property Tab drop-down list, choose the Property tab that you want to appear when users click on a Physical Item link.

  7. In the Default settings section, do the following:

    • Select a location for the Physical Item in the Location drop-down list.

    • Select a Locator Type for the Physical Item in the Locator Type drop-down list.

    • Select a Reference Rate for the Physical Item in the Reference Rate drop-down list.

    • In the Box Client field, click the Choose User or Choose Box Client icon to populate the field with a user or box client, or click the Remove icon to delete the current user or box client.

    • Select the Generate Label check box to create a label for the Physical Item.

    • If you selected the Generate Label check box, click a type of label in the Label Type drop-down list.

  8. Click the Update button.

 

Note:
If you want to specify a default value for a code that has not yet been created, you can create a code on the My Physical Objects Settings page by clicking Create New Code in any of the default settings drop-down lists.
