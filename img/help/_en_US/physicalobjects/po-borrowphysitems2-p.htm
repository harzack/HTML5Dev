To borrow multiple physical items:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Circulation link.

  3. On the Circulation page, click the Borrow Physical Items link.

  4. On the Borrow Physical Items page, click the Choose User icon in the Borrowed By section, and then choose a user.

  5. Click the Choose User icon in the Obtained By section, and then choose a user. Specify this field if the physical item or physical item copy is borrowed on behalf of the user identified in the Borrowed By section.

  6. Click a date in the drop-down lists in the following sections:

    • Borrow Date, which let you specify a date for borrowing the physical items

    • Return Date, which let you specify a date for returning the physical items

    Optionally, select the Pre-Approve Security Overrides check box to automatically approve any security overrides.

  7. Type unique ID in the Unique ID field, or scan a list of bar codes in the Unique ID List field or browse to a list of unique IDs.

  8. Specify any of the remaining settings.

  9. Click the Search button.

  10. In the list of search results, select the check box for each physical item you want to borrow.

  11. In the Acknowledgement drop-down list, click an option to specify how borrowers acknowledge receipt of the physical object when it is passed to them or borrowed by them.

  12. Click the Borrow button.

 

Note:
You must have Circulation page permissions to borrow multiple physical items.

 
 

Tip:
You can also borrow multiple physical items in the Browse View of a Workspace by selecting the check box next to each item you want to borrow, and then clicking Borrow Physical Items button (or from the More Actions drop-down list).

If an item is a box or container, you can view any borrowed children by clicking the View link in the Has Borrowed Children column.
