To assign a physical item to a box:

  1. Click the Functions icon for a physical item, choose Properties, and then choose Circulation.

  2. On the Circulation tab of the Physical Item Properties page, click the Functions icon in the Actions column, and then choose Assign Box.

  3. On the Assign box page, type the box's unique ID in blank field or click the Browse Content Server button, browse for the box you want to assign, and then click the box's Select link.

     

    Note:
    If you browse to a box and type the box's unique ID in the filed, the box that you browsed to will take precedence over the box with the unique ID.

     

    Optionally, select the Update Status of all Sub-Items check box to update the status of any sub-items the physical item has.

    Optionally, select the Update Location of all Sub-Items check box to update the location of any sub-items the physical item has.

  4. Click the Update button.

  5. On the Boxing Information page, select the check box next to the physical item you want to assign to the box.

  6. Click the Assign button.

 

Tip:
You can assign multiple physical items to a box from the Browse View of a workspace by selecting the check box for each physical item you want to assign, then clicking Assign Box from the More drop-down list.
