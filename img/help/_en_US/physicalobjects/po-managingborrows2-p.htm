To return multiple physical items:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Circulation link.

  3. On the Circulation page, click the Borrowed Physical Items link.

  4. On the Borrowed Physical Items page, click Return in the Action drop-down list.

  5. Do any of the following:

    • In the Return Date section, click the date from which to begin the search in the From drop-down lists.

    • In the Return Date section, click the date at which to end the search in the To drop-down lists.

    • Select the Ready for Pick Up check box.

    • Type unique IDs in the Unique ID List field.

  6. To assign a Locator to physical items, select the Assign Locator check box, choose the Facility and Area where the Locator resides in the appropriate drop-down lists, and then click the Search Locators button to select a Locator for the item you are returning.

  7. Specify any of the remaining settings, and then click the Search button.

  8. On the Borrowed Physical Items page, click the date the physical items are being returned in the Return Date drop-down list.

  9. Do one of the following:

    • Select the Return to Home Location check box to return the physical items to the home location.

    • Type a location name in the Current Location field to return the physical items to a location other than the home location.

  10. Select the check box next to each borrowed physical item you want to return.

  11. Click the Return Borrowed Items button, and then click OK in the dialog window.

  12. On the Circulation Status page, click OK.

 

Tip:
To return all borrowed items in the list, click the Select All button. This button allows you to select all borrowed items, even when you have a long list of items that span over multiple pages.

You can also specify search criteria for borrowed physical items by scanning a list of bar codes in the Unique ID List field or by clicking the Browse button, and then browsing to a list of unique IDs.
