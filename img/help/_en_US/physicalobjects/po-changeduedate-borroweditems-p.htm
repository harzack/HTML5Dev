To change the due date for borrowed Physical Items:

  1. Choose Physical Objects on the Enterprise menu.

  2. On the Physical Objects page, click the Circulation link.

  3. On the Circulation page, click the Borrowed Physical Items link.

  4. On the Borrowed Physical Items page, select Change Return Date in the Action drop-down list.

    Optionally, enter the user ID or any other optional search criteria to narrow the results that are returned.

  5. In the Unique ID List, enter borrowed item's unique IDs, or browse to a file that contains the list of the unique IDs.

  6. Click the Search button.

  7. In the New Return Date field, select a new date from the appropriate drop-down lists, or extend the date by a specific number of days by clicking the Extend by radio button and entering a number of days in the field.

  8. Select the check box beside each item for which you want the return due date changed.

  9. Click the Change Return Date button.
