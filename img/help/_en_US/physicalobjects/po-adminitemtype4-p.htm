To edit the unique ID format for a physical item type:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Item Types link.

  3. On the Physical Item Types page, click the Functions icon for a physical item type, choose Properties, and then choose Specific.

  4. On the Specific tab of the Physical Item Type Properties page, type a new unique ID format in the Unique ID Format field.

  5. Click the Update button.
