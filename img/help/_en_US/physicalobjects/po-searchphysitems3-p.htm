To return physical items:

  1. On the Content Server Search Results page (or from a Collection), select one or more physical items and from the More Actions drop-down list, select Return Physical Item.

  2. On the Return Physical Items page, click Cancel Borrow, Pass or Return on the Action drop-down list.

  3. Click Return.

  4. The Physical Items page appears with fields specific to the action selected in step 2. Update the fields and click Cancel, Pass or Return.

  5. The metadata entered on the previous page is now displayed as read only and the status of each physical item is displayed. Click Cancel, Pass or Return.

  6. When the action is complete, click OK.

 

Tip:
You can return multiple physical items at one time in the Browse View of a workspace by selecting the check box for each item you want to return, and then clicking Return Physical Items on the More drop-down list.
