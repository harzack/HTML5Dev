The Circulation tab of an item's Properties page contains information such as the item's location, unique ID, borrower information, and a return date for a borrowed item. This page also indicates if an item was borrowed using the Security Override feature, and provides a message if a physical item is in a container that is borrowed, or if the physical item or its parent is in a box that is borrowed.

You can view the circulation history for a physical item. The search for the circulation history of an item is performed based on the unique ID for the physical item, but you can also include other search criteria such as the borrow or return date. You can also specify whether to search for deleted items. The search results provide information for items found using the search parameters. Information on the Circulation History results page includes date the items were borrowed and returned, borrower comments, the name of the user who obtained the item, whether a security override was used to borrow the item, and comments the user added when he or she returned the item.
