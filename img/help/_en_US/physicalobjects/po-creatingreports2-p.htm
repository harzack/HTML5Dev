To create a Request Listing report:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Reports link.

  3. On the Reports page, click the Request Listing Report link.

  4. Optional In the Filter Report by section of the Run WebReport page, filter the results by any of the following options:

    • Requestor

    • Request From and To Dates

    • Request Type

    • Offsite Storage ID

    • Home Location

    • Current Location

  5. In the Sort Report By section, click the check box for the method by which you want the report sorted, and then choose Ascending or Descending in the associated drop-down list.

  6. Click the Run Report button.
