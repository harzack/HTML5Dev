To create a physical item:

  1. Click Physical Item on the Add Item menu.

  2. On the Add Physical Item page, click the radio button for a physical item type, and then click Next.

  3. Type a name and description for the item in the Name and Description fields.

  4. Type a location for the physical item in the Home Location field.

    Optionally, type a temporary location for the item in the Current Location field to specify a temporary location for a physical item whose parent is borrowed.

  5. Type the number of copies to create in the Number of Copies field.

     

    Note:
    Containers and boxes cannot have multiple copies.

     

    Optionally, click a locator type in the Locator Type drop-down list to specify the type of place the physical item can be stored.

  6. Specify any of the remaining settings.

  7. Click the Add button.

  8. Optional Click the Create Another button, and then type the number of additional items you want to create using the same metadata.

 

Note:
If the unique ID for the Physical item appears after you create it, click the Continue button.
