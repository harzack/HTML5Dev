To assign physical items in a Collection to a box:

  1. In a Collection, select the check box next to every physical item you want to assign.

  2. Click Assign Box in the More Actions drop-down list.

    Optionally, select the Update Status of all Sub-Items check box to update the status of any sub-items the physical item has.

    Optionally, select the Update Location of all Sub-Items check box to update the location of any sub-items the physical item has.

  3. Specify any of the remaining settings, and then click the Search button.

  4. In the list of search results, click the Select link for a box.

  5. On the Boxing Information page, click the Assign button.
