To create a physical item copy:

  1. Click the Functions icon for a physical item, choose Properties, and then choose Circulation.

  2. On the Circulation tab of the Physical Item Properties page, click the Add New Copy button.

  3. On the Copy Information page, type an ID in the Unique ID field. Optionally, type a location name in the Current Location field to specify a current location that is different from the parent's current location.

     

    Note:
    The Current Location field is available only if the parent of the new copy is borrowed.

     

  4. Click the Add Item button.

 

Tip:
You can automatically generate the unique ID by leaving the Unique ID field blank.
