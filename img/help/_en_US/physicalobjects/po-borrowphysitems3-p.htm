To borrow physical items in a Collection:

  1. In a Collection, select the check box next to every physical item you want borrow.

  2. Click Borrow Physical Items in the More Actions drop-down list.

  3. On the Borrow Physical Items page, click the Choose User icon in the Borrowed By section, and then choose a user.

  4. Click the Choose User icon in the Obtained By section, and then choose a user. Specify this field if the physical item or physical item copy is borrowed on behalf of the user identified in the Borrowed By section.

  5. Click a date in the drop-down lists in the following sections:

    • Borrow Date, which lets you specify a date for borrowing the physical items

    • Return Date, which lets you specify a date for returning the physical items

    Optionally, select the Pre-Approved Security Override check box to automatically approve any security overrides.

  6. Click the Borrow button.
