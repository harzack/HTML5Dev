To add physical items to a box:

  1. Click the link of the box container to which you want to add physical items.

  2. In the Detail View of the container, click the Create Box Contents button.

  3. On the Physical Item page, click the radio button next to the type of physical item you want to add, and then click the Next button.

  4. On the Add Physical Item page, specify settings the same way you would if you were adding a physical item to a workspace.

  5. Click the Add button to add only this physical item to the box, or click the Create Another button to add another physical item to the box.

 

Note:
When adding a physical item to a box, the Locator Type attribute must be set to <None>.
