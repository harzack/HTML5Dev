To assign objects in a Collection to a transfer:

  1. Browse to a Collection.

  2. Select the check box next to every object you want to transfer.

  3. Click Assign To Transfer in the More Actions drop-down list.

  4. On the Assign to Transfer page, specify any search criteria, and then click the Search Transfers button.

  5. In the list of search results, click the Select link for a transfer.
