You can export the Physical Objects Table Maintenance and Locator codes from the Content Server source and then import them into a different server. This is the useful when you have Content Server installed on multiple computers or set up in more than one environment. The codes will be exported to an XML file and saved to the following location:

C:\contentserver_installation_path\temp\filename.xml

where contentserver_installation_path is the location Content Server is installed, and filename.xml is the XML file name you specified during the export process.

The XML file is also automatically uploaded to your Personal Workspace.
