In Physical Objects, you can request physical items, notifying others that a user intends to borrow an item on a certain day but has not yet borrowed it. You can request a single physical item or mutliple physical items in a Collection or in the Browse View of a workspace. When you request one or more physical items, you can choose a request type code, and you can send an e-mail notification if an e-mail address exists for the home location of the physical item. You can also add any comments about the request, and, if you are requesting items for a different Content Server user, you can include the name, address, phone and fax number, and e-mail address of a contact person.

 

Note:
Your Administrator can choose to send e-mail notifications for each Physical Item request by default. If your Administrator has selected this option, you cannot choose whether to send an e-mail notification for requested Physical Items, an e-mail will automatically be sent upon requesting the item.

 

After you request physical items, you can cancel the request. The cancellation of the request can be performed on a single physical item or multiple physical items. When canceling the request of multiple physical items, you can specify the start and end date for the search, the user who requested the physical items, the location where the physical items are originally stored, and the current location where the physical items are stored.

You can also view the contact information for a user who has requested an item.
