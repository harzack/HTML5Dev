To create a Box Listing Report:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Reports link.

  3. On the Reports page, click the Box Listing Report or Container Listing Report link.

  4. On the Run WebReports page, do one of the following:

    • To create a report for a single physical item box or container, click the Browse Content Server button to select the box or container.

    • To create a report for multiple boxes or containers, enter the unique IDs of the containers or boxes in the Unique ID List, or scan a list of barcodes.

  5. Click the Run Report button.
