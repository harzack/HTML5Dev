To import Table Maintenance or Locator codes:

  1. Choose Physical Objects on the Enterprise menu.

  2. On the Physical Objects page, click the Physical Objects Administration link.

  3. On the Physical Objects Administration page, click the Import/Export link.

  4. In the Import section, click the Import Table Maintenance Codes or Import Locators link.

  5. On the Import page, click the Browse button to navigate to the XML file where the Table Maintenance codes or Locators are saved, click Open, and then click the Import button.
