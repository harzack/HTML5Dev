To print labels in batches:

  1. Choose Physical Objects on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Batch Labels link, and then click the Search and Print Generated Labels link.

  3. On the Search and Print Generated Labels page, click one of the following radio buttons:

    • Physical Items, which lets you search labels for physical items

    • Locators, which lets you search labels for locators

    • Users, which lets you search labels for users

  4. Specify any remaining search criteria, and then click the Search button.

  5. Select the check box in the first column for each label you want to print.

    Optionally, click the Preview button to view the batch of labels before printing them.

  6. Click the Print button.
