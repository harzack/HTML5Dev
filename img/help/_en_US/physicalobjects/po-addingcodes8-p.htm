To add a code to the Reference Rate table:

  1. Choose Physical Objects on the Enterprise menu.

  2. On the Physical Objects page, click the Physical Objects Administration link.

  3. On the Physical Objects Administration page, click the Table Maintenance link.

  4. On the Physical Objects Table Maintenance page, click the Reference Rate link.

  5. Click Reference Rate on the Add Item menu.

  6. Type a code in the Code field.

  7. Specify any of the remaining settings.

  8. Click the Submit button.

 

Tip:
You can also create a Table Maintenance code on the page where it is to be used, by clicking <Create New Code> in the code's drop-down list.
