To create a storage locator:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Objects Administration link.

  3. On the Storage Management page, click the Locators link.

  4. On the Locators page, click Locator on the Add Item menu.

  5. On the Add Locator page, click a code in the following drop-down lists:

    • Facility, which lets you specify a facility code for the storage locator

    • Area, which lets you specify an area code for the storage locator

  6. Type a name for the locator in the Locator field.

  7. Type a value in the following fields:

    • Total Space, which lets you specify the total size of the locator

    • Free Space, which lets you specify the size of available space for the locator

    Optionally, specify any of the remaining settings in the Information for Managed Facilities section. Information in that section is only used if the assigned facility is managed.

  8. Click the New button.

 

Note:
If the size of the locator exceeds the size of the available space and the facility is non-managed, the facility must be set to automatically increase total space. For more information, see Adding Codes to Physical Object Tables.
