To generate labels for a physical item:

  1. Click the Functions icon for a physical item, choose Properties, and then choose Labels.

  2. On the Labels tab of the Physical Item Properties page, click a label type in the Label Type drop-down list.

  3. Type the number of copies you want to generate in the No of Copies field.

  4. Click the Generate button.

 

Tip:
You can generate labels for multiple physical items from a Collection or set of Search results, and also from the Browse View of a workspace by selecting the check box for each item, and then clicking the Generate Labels button (or clicking it on the More drop-down list).
