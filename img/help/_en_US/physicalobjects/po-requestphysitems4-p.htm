To cancel the request of multiple physical items:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Circulation link.

  3. On the Circulation page, click the Requested Physical Items link.

  4. On the Request Physical Items page, click Cancel Request in the Action drop-down list.

    Optionally, select the Pre-Approve Security Overrides check box to automatically approve any security overrides.

  5. Specify any of the remaining settings, and then click the Search button.

  6. Select the check box for each requested physical item you want to cancel.

  7. Click the Cancel Request button.
