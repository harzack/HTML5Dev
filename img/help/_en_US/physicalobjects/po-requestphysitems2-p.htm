To request multiple physical items:

  1. In a Collection or advanced search workspace, select the check box next to each physical item you want to request.

  2. Click Request Physical Items in the More Actions drop-down list.

    Optionally, on the Request Info page, click the Choose User icon in the Requestor section, and then choose a user for whom you want to request the item.

  3. Click the Choose User icon in the Obtained By section, and then choose a user. Specify this field if the physical item or physical item copy is borrowed on behalf of the user identified in the Borrowed By section.

  4. Click a date in the Request Date drop-down list.

  5. Specify any remaining settings.

  6. Click the Request button.
