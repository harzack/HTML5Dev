To edit the icon for a physical item type:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Item Types link.

  3. On the Physical Item Types page, click the Functions icon for a physical item type, choose Properties, and then choose Specific.

  4. On the Specific tab of the Physical Item Type Properties page, click a radio button in the Icon section.

  5. Click the Update button.

 

Note:
Changing the icon for an existing physical item type will not change the icon for any physical items that were assigned the physical item type before the icon was changed.
