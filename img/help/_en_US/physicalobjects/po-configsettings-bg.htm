You can edit your personal settings for physical objects. For example, you can hide or display the Labels and Circulation tabs, choose to always send an e-mail when a Physical Item is requested, and set the Property tab to be opened when a Physical Item link is clicked. You can also choose the default location, locator type, reference rate, and label type codes that display when you add physical items. If a value has not yet been created for the code, and you want to specify a default setting, you can create the code using the appropriate drop-down list.

From this page, you can also select or remove a Box Client or user and enable Physical Items to be returned to the home location. By default, the Return to Home Location check box is cleared. When this option is enabled, items returned from the Circulation Workspace and Collections and Circulation tabs are returned to their home locations.
