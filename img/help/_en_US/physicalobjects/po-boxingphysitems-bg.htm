Boxing physical items physically groups the physical items without virtually grouping them. You can add Physical Items to a box that is already created, or you can assign a box to an item as you are creating it. In order to add a physical item to a box, you must not specify a locator type for the new physical item. The To and From dates for a box automatically update to correspond to the To and From dates of the contents within the box, which means if you specify dates and then add or remove items that have earlier or later To and From dates, the original dates for the box will change. For more information about creating physical items, see Adding Physical Items.

You can assign existing physical items to a box one at a time using the Circulation page, or you can assign single items or several items at a time using the Detail View page.

When assigning physical items in a Collection to a box or when using the Circulation page to assign physical items to a box, you can search for a box by unique ID, name, or locator type.

 

Note:
You must have a physical item type specified as a box type in order to add items to it. For more information about physical item types, see Administering Physical Item Types.
