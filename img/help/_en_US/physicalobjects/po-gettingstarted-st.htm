The Physical Objects module is used to store and manage physical items within Content Server. As a Content Server user, you can create virtual objects in Content Server that represent physical objects, such as videocassettes, paper records, and CD-ROMs. You can then use the virtual objects to manage the lifecycle of the physical objects. For example, you can borrow and request physical items, prepare listing reports, and generate and print labels for placing on the physical objects which you can then use to track inventory and manage space in your storage facilities.
