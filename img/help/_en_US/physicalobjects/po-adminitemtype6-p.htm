To hide Add Physical Item page and Specific tab fields for a physical item type:

  1. Choose Physical Objects Workspace on the Enterprise menu.

  2. On the Physical Objects Workspace page, click the Physical Item Types link.

  3. On the Physical Item Types page, click the Functions icon for a physical item type, choose Properties, and then choose Hide Fields.

  4. On the Hide Fields tab of the Physical Item Properties page, select the check boxes for any fields you want to hide. Optionally, if you selected the Generate Label check box, select a default label type from the drop-down list, and then type the number of copies in the field to generate the label silently.

  5. Click the Update button.
