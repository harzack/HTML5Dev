The Physical Objects Workspace lets you perform tasks on or in conjunction with physical objects or items. The following six areas are accessible through links in the Physical Objects Workspace:

  • Batch Labels, which lets you print labels that have been batched in the system

  • Circulation, which lets you perform circulation actions on a batch of physical items

  • Physical Item Types, which lets you create physical item types

  • Physical Objects Administration, which lets you perform administrative tasks relating to physical items, such as configuring default settings

  • Reports, which lets you run reports relating to physical items

  • Storage Management, which lets you manage locators and transfers

 

Note:
Each link has Content Server permissions access.
