To search with physical item type constraints:

  1. On the Content Server Search page, click the Physical Item Types link in the Advanced Options list.

  2. In the Physical Item Types section, click Browse Physical Item Type in the drop-down list, browse for a physical item type, and then click the Select link for a physical item type. Optionally, click the Duplicate button , choose an operator, and then repeat step 2 to use a Boolean operator.

 

Tip:
You can also specify a physical item type by clicking Browse Physical Item Type in the drop-down list, browsing for a physical item type, and then clicking the Select link for a physical item type.
