To manage generated labels for a physical item:

  1. Click the Functions icon for a physical item, choose Properties, and then choose Labels.

  2. On the Labels tab of the Physical Item Properties page, do one of the following:

    • Select the check box next to each label you want to preview, and then click the Preview button.

    • Select the check box next to each label you want to print, and then click the Print button.

    • Select the check box next to each label you want to remove, and then click the Remove button.
