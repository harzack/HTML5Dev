To edit physical item metadata for multiple items:

  1. In a Collection, Search result, or Browse View, select the check box beside each item for which you want to the Physical Item metadata changed.

  2. Click the Physical Items Metadata button (or from the More Actions drop-down list).

  3. On the Physical Items Metadata page, modify any of the settings.

  4. Click the Update button.

 

Tip:
You can also edit Physical Item Metadata for all items in a Collection by selecting the Entire Collection check box in the Detail View of the Collections page.
