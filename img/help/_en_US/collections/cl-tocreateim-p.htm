To create a Disk Image:

  1. On the Global Menu bar, from the Personal menu, click Collections.

  2. On the My Collections page, click the name of the Collection from which you want to select items to create your disk image.

  3. On the <your_collection_name> page, select the check box for each item you want to include in the Disk Image.

    Optionally, select the Select/Deselect All check box to include all items in the Collection.

  4. From the More Actions list, select Make Disk Image.

  5. On the Make Disk Image page, the Summary and Destination Path fields provide information about the items to be included in the disk image and the path where Content Server will create the disk image.

  6. From the Available Disk Sizes list, click the appropriate disk size.

  7. From the Disk File System Format list, click the appropriate format type.

  8. Optional In the Collection field, select the check box(es) for the type of information about the Collection that you want to include in the Disk Image.

  9. Optional In the Content field, select the check box(es) for the type of information about the items in the Collection that you want to include in the Disk Image.

  10. In the Current Version Handling field, select the option to determine how current Versions should be handled during the Disk Creation process:

    • Do not force inclusion: unless the user has specifically selected the current version, it will not be included.

    • Include current version: the current versions of all items will be included.

  11. In the Metadata Language field, select your preferred language from the list.

  12. In the Notification area:

    1. Depending on how your administator has set up Make Disk Image, you may be able to select the point when notification should be sent from the When list.

    2. In the Recipient field, if defined, your email address appears. You can change this to send the notification to an alternate email address.

    3. In the Message Subject field, you can change the default email message subject.

    4. In the 3rd Party Notifications area, select one of the available option buttons:

      1. Select No Additional Notification if you do not want to add additional email recipients.

      2. Select Group to add additional email recipients.

        Next, do any, or all, of the following:

        • In the User field, click Find a user to select a Content Server user to whom you want to send an email notification.

        • In the Group field, click Find a group to select a Content Server group to whom you want to send an email notification.

        • In the Other Email field, enter an email address to send an email notification to an email address not available in Content Server.

  13. Click Submit.
