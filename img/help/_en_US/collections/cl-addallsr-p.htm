To add all search results to a Collection:

  1. From a Search Results page, from the Search Tools list, select Collect All Search Results.

  2. On the Collect All Search Results page, in the Notification area:

    1. Optional If you want to send email notification, in the Enabled field, select the check box.

    2. Optional In the Recipient field, if defined on your My Profile page, your email address appears. To change this and send the notification to an alternate email address, enter the new email address in the text field.

    3. Optional In the Message Subject field, you can accept the default email message subject, or you can enter a new message subject in the text field.

    4. In the 3rd Party Notifications area, select one of the two available option buttons:

      1. Select No Additional Notification if you do not want to add additional email recipients.

      2. Select Group to add additional email recipients.

        Next, do any, or all, of the following:

        • In the User field, click Find a user to select a Content Server user to whom you want to send an email notification.

        • In the Group field, click Find a group to select a Content Server group to whom you want to send an email notification.

        • In the Other Email field, enter an email address to send an email notification to an email address not available in Content Server.

  3. In the Collection field, click one of the following:

    • Click Create New... to create a new Collection. Enter a name for your new Collection, optionally edit other fields as desired, then click Add.

    • Click Browse Content Server to locate an existing Collection. Browse to the Collection you want to select, then click that Collection's Select link.

  4. Click Submit.
