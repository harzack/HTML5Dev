To add items to a Collection from a Search Results page:

  1. From a Search Results page, select the multi-select check box(es) next to the search result(s) you want to collect.

  2. From the Select Action... list, select Add to Collection.

  3. On the Collect Search Results page, in the Collection field, choose one of the following:

    • To create a new Collection, click Create New.... Enter a name for your new Collection. Optionally edit other fields as desired, then click Add.

    • To locate an existing Collection, click Browse Content Server. Browse to the Collection you want to select, then click that Collection's Select link.

  4. Click Submit.
