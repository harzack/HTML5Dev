There are two ways to add a collection. The first is to create an empty Collection, then add items to your Collection later. The second is to select items and then create a Collection.

To create an empty Collection:

  1. From the Add Item menu, click Collection.

  2. On the Add: Collection page, in the Name field, type a name for the Collection.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the name in the other, enabled, languages.

  3. Optional In the Description field, add a description for the Collection.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the description in the other, enabled, languages.

  4. Optional If available, you can apply a Classification to this Collection by selecting Browse Content Server from the Classifications list.

    1. In the dialog box which appears, select the Classification you will be applying to this Document.

    2. Ensure the Action check box next to the Classification is checked.

    3. Click Submit.

  5. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Collection.

  6. Optional If you want to place the Collection in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Collection, then click its Select link.

  7. Click Add.

To select items then create a Collection:

  1. From any workspace or folder, select the multi-select check box(es) next to each item that you want to add to your new Collection.

  2. Select Collect from the toolbar. The Collect button may be available explicitly on the toolbar, or it may be listed under the More... toolbar option.

  3. Optional On the Items to Collect page, in the Items area, if you want to ensure that the current version of any item is collected, select the Collect Current Version check box next to that item.

  4. In the Collection field, select Create New Collection from the list.

  5. In the Add: Collection dialog box:

    1. In the Name field, type a name for your new Collection.

      If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the name in the other, enabled, languages.

    2. Optional In the Description field, add a description for the Collection.

      If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the description in the other, enabled, languages.

    3. Optional If available, you can apply a Classification to this Collection by selecting Browse Content Server from the Classifications list.

      1. In the dialog box which appears, select the Classification you will be applying to this Document.

      2. Ensure the Action check box next to the Classification is checked.

      3. Click Submit.

    4. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Collection.

    5. Optional If you want to place the Collection in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Collection, then click its Select link.

    6. Click Add.

  6. On the Items to Collect page, click Submit.

     

    Note:
    If you select the Cancel button, an empty Collection may have been created if you selected Add in the previous step. You will need to manually delete the empty Collection.

     

  7. On the Collecting Items to this collection page, click OK.
