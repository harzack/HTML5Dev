To move collected items to another Collection:

  1. On the Global Menu bar, from the Personal menu, click Collections.

  2. On the My Collections page, click the name of the Collection which contains the items you want to move. On the <collection_name> page, select the multi-select check box(es) for each item you want to move.

  3. From the More Actions... list, select Move Items to Another Collection.

  4. On the Move Items to Another Collection page, from the Select list, choose one of the following:

    • Click Create New Collection to create a new Collection. Enter a name for your new Collection. Optionally, edit other fields as desired, then click Add.

    • Click Browse Collections to locate an existing Collection. Browse to the Collection you want to select, then click that Collection's Select link.

    • Click My Favorite Collections to select a Collection listed in Favorites. Find the Collection you want to select, then click that Collection's Select link.

  5. Click Move.
