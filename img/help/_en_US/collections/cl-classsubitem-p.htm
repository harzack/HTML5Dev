To classify sub-items:

  1. On the Detail View page, click the Function button for any container. For example, click any folder's Function button. From the Function menu, select Properties, and then select Classifications.

  2. On the Properties page, click Apply Classifications to Sub-Items.

  3. When the confirmation dialog box appears, click OK.

  4. On the Apply Classifications to Sub-Items screen, wait for the Status line to display Done. Click OK.

 

Note:
The Apply Classifications to Sub-Items button does not appear if the container does not have any Classifications applied.
