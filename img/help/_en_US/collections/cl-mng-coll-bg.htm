Once you have added items to a Collection, you can manage both the original item and the item pointer from within the Collection. You can browse Collections like other Content Server containers and view information about the Collection and its contents. You can also perform basic Content Server functions on the collected items, and manage the pointers to items within the Collection.

You can also apply Categories to all items within a Collection by selecting Apply Categories from the More Actions list. For more information about Categories, see Working With Categories and Attributes.

In addition, you can classify your Collection by selecting Apply Classification from the More Actions list. For more information about Classifications, see Using Classifications.

To see detailed information about the Collection, and the activities that have occurred within it, you can view the Collection Items Audit page by selecting any Collection's Functions icon. From that Collection's Functions menu, select Properties, then select Collection Items Audit.
