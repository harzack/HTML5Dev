To add items to a Collection from a Workspace:

  1. In the Detail view of any workspace, select the multi-select check box(es) next to each item that you want to add to your new Collection.

  2. Select Collect from the toolbar. The Collect button may be available explicitly on the toolbar, or it may be listed under the More... toolbar option.

  3. On the Items to Collect page, in the Items area, if you want to ensure that the current version of any item is collected, select the Collect Current Version check box next to that item.

  4. In the Collection field, from the Select list, click one of the following:

    • Create New to add these items to a new Collection you create. Enter a name for your new Collection, optionally edit other fields as desired, then click Add.

    • Browse Content Server to add these items to an existing Collection you locate. Browse to the Collection you want to select, then click that Collection's Select link.

  5. Click Submit.
