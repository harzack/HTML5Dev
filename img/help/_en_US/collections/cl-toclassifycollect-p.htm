To classify items in a collection:

  1. On the Global Menu Bar, from the Personal menu, click Collections.

  2. On the My Collections page, select a Collection.

  3. Select the check box for each item you want to classify. From the More Actions list, select Apply Classifications.

  4. On the Classify Collection Items page, from the Classify list, click one of the following options:

    • Browse Classifications

    • My Favorites

  5. In the Browse Classifications window, or the My Classification Favorites window, navigate to each Classification that you want to assign to the item. Select the corresponding check box.

  6. Click Submit.

  7. On the Classify Collection Items page, click Apply.

 

Note:
The Apply Classifications menu option is only available if your administrator has given you the appropriate usage privileges.
