To download a collection as a spreadsheet:

  1. Navigate to the container in Content Server that contains the Collection you want to download as a spreadsheet. Click that Collection's name

  2. Once you are in the Collection, using the multi-select check box(es), select the check box next to each item you want to download. You can also choose to download the entire collection.

  3. From the More Actions... list, select Download as Spreadsheet.

  4. In the File Download box, select one of:

    • Open to open the downloaded spreadsheet

    • Save to save the downloaded spreadsheet to your computer

    • Cancel to cancel the operation

  5. If you selected Save above, in the Save As box which appears, do the following:

    1. Navigate to the location on your computer to save the spreadsheet.

    2. Optional You can either choose to accept the default name of the spreadsheet or enter a new name in the File name field.

    3. Click Save.
