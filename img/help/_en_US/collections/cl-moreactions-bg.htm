After you have added an item to a Collection, you can remove a specific item's pointer from the Collection using the item's Functions menu. You can also copy, move and remove one or more collected pointers from one Collection to another using the More Actions list on the Collection page. These actions include:

  • Copy Items to Another Collection

  • Move Items to Another Collection

  • Remove Items from Collection

 

Note:
If you select an item where specific versions have been collected, when you select one of these options, all collected versions of that item are copied, moved or removed.

 

If you want to remove a specific version of an item, you can remove just that version from the Collection using that version's Functions menu. If you remove all versions of an item from that Collection, the item remains in the Collection until you select Remove from Collection from the Functions menu.

 

Note:
The menu options available to you depend on the privileges assigned by your administrator. Therefore, some of the menu options described in this section may not be available.
