To print a Workflow Map:

  1. In the Workflow Map Designer, click the Print icon .

  2. Optional In the Paper Settings area, alter the page size and orientation.

  3. Optional In the Scaling Settings area, select the Scale up to the given number of pages check box and enter the number of pages in the Pages text box.

  4. Optional Click Preview to view the print settings before sending the print job to the printer.

  5. Click Print.

For information related to this procedure, see Verifying Workflow Maps.
