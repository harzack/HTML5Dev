To define an expression:

  1. Double-click an Evaluate, User, or Form Task step.

  2. If you double-clicked a User or Form Task step, click the Assignees tab.

  3. Click the Expression Builder icon .

  4. Click a Workflow Map search criterion or parenthesis (if you are creating a complex expression) in the Add New Row list.

  5. Specify the expression using the Expression Builder.

  6. Optional To specify other criteria, repeat steps 2 and 3.

  7. Click Submit.

For information related to this procedure, see Working with the Expression Builder.
