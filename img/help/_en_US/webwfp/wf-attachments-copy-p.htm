To copy a Content Server item as an attachment:

  1. Choose Attachments on the Map menu.

  2. On the Attachments tab, do one of the following:

    • Click the Copy item from Content Server link.

    • Click a location in the navigation list to browse Content Server or the Attachments folder.

  3. Select the Select check box next to the item you want to add.

  4. Click Submit.

For information related to this procedure, see Defining Attachments.
