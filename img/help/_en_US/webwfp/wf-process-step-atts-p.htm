To modify Workflow Map attributes:

  1. In the Workflow Map Designer, double-click a Process step.

  2. Click the Attributes tab.

  3. For each attribute you want to modify, select the check box next to the attribute, and then specify a value in the attribute Value field.

  4. Click Add to Workflow Map Definition.

  5. Save the Workflow Map.

For information related to this procedure, see Defining a Process Step.
