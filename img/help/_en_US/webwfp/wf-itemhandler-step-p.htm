To define an Item Handler step:

  1. In the Workflow Map Designer, double-click an Item Handler step.

  2. On the General tab, specify a step name, provide instructions that will display in the General window, specify date and priority settings, a step assignee, and the Workflow Agent processing option, and then click Add to Workflow Map Definition.

  3. Optional If attachments are enabled for the Workflow Map, click the Attachments tab, provide Attachments window instructions, and then click Add to Workflow Map Definition.

  4. Click the Comments tab, provide Comments window instructions, and then click Add to Workflow Map Definition.

  5. Optional If attributes are enabled for the Workflow Map, click the Attributes tab, provide Attributes window instructions, specify an access option for each attribute, and then click Add to Workflow Map Definition.

  6. On the Folder Definitions, Categories, Versioning, and Move/Copy tabs, specify the operations you want to perform.

  7. On the Records Management tab, click the Add Records Management Definition icon, select the Finalize Record check box to automatically finalize Records Management items as part of the step, select or remove an associated RM Classification, and then specify any of the standard operations you want to perform.

     

    Note:
    An error message displays if you attempt to save a Records Management Definition without adding an Item to the step and either updating an RM Classification or enabling the Finalize Record option. You are required to add an Item and choose an Update option.

     

  8. Save the Workflow Map.

For more information about Item Handler operations, see Defining Item Handler Step Operations.
