To set Workflow Map performers:

  1. On the Map menu, choose Edit Map Performers, click the Action icon of the step you want to edit.

  2. On the Edit Map Performers page, modify the settings as described in the following table.

    To… Do the following…
    Assign a step to a User or Group Find the user or group, click the Select link next to the user or group you want to assign to this step, and then click Add to Workflow Map Definition.
    Assign a step to the Initiator or a role Click the Initiator or role's Select link, and then click Add to Workflow Map Definition.

For information related to this procedure, see Setting Workflow Map Performers.
