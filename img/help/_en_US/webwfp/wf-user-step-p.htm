To define the User or Initiator step:

  1. In the Workflow Map Designer, double-click a User or Initiator step.

  2. On the General tab, specify the following information and then click Add to Workflow Map Definition.

    • In the Step Name box, type a name for the step.

    • In the Instructions box, provide instructions that will display in the General window.

    • Click the Choose User or Group icon to select a step assignee.

    • Select a group requirement in the Group Options list. For more information, see Specifying the Group Assignment Method.

    • Select and type the date and priority settings. For more information, see Understanding Dates and Priorities.

    • Select the Recalculate Due Dates check box to set this step to recalculate the due date of the Workflow Map at the time of its completion.

    • Select the Allow the Workflow Map Agent to perform 'Send On' processing check box to enable background processing.

    • Select the Enable Action E-mail check box to send an email to the step assignee advising them of the Workflow Map assignment.

    • Select the Single Page View check box to display all selected packages in a single page. To display a package for the step assignee, highlight the package in the Packages region and click the right arrow button. To change the order of the packages for the step assignee, highlight a package in the Display Packages region and click the up or down arrow button. For more information, see Using the Single Page View. Clearing the Single Page View check box displays all packages in the map package in a double pane view. For more information, see Understanding the Display of the User and Initiator Step.

  3. Click the Assignees tab, do one of the following, and then click Add to Workflow Map Definition:

    • Click the User or Group radio button, click the Choose User or Group icon , and then find and select the user or group to which you want to assign the step.

    • Click the Multiple Users/Groups radio button, click the Choose User or Group icon , and then find and select the users or groups to which you assign the step.

    • Click the Assignee from Prior Step radio button, and then click a step name in the list.

    • Click the Assignee from Attribute radio button, if available, and then click an attribute in the list.

    • Click the Expressions radio button, define the conditional statements used to determine the step assignee (click the Edit Expression icon to create the conditional statements using the Expression Builder), click the Choose User or Group icon , and then find and select the users or groups you want to assign the step to if the conditional statement is true.

  4. Click the Permissions tab, specify any step permissions or dispositions, and then click Add to Workflow Map Definition.

  5. If attachments are enabled for the Workflow Map, click the Attachments tab, provide Attachments window instructions, and then click Add to Workflow Map Definition.

  6. Click the Comments tab, provide Comments window instructions, and then click Add to Workflow Map Definition.

  7. If attributes are enabled for the Workflow Map, click the Attributes tab, provide Attributes window instructions, specify an access option for each attribute, and then click Add to Workflow Map Definition.

  8. Save the Workflow Map.

 

Note:
For more information about the Expression Builder, see Working with the Expression Builder.
