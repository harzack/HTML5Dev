To define an Evaluate step:

  1. In the Workflow Map Designer, double-click a Evaluate step.

  2. Specify a step name.

  3. To set this step to recalculate the due date of the Workflow Map at the time of its completion, select the Recalculate Due Dates check box.

  4. Click the Edit Expression icon and use the Expression Builder to create the conditional statements that are evaluated at each step to route Workflow Map.

  5. In the Step list, click the name of the step to which the processing is routed if the expression is evaluated as true.

  6. Click Add to Workflow Map Definition.

  7. Save the Workflow Map.

 

Note:
For more information about the Expression Builder, see Working with the Expression Builder.

 

For information related to this procedure, see Defining an Evaluate Step.
