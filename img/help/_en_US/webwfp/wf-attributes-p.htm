To add attributes:

  1. On the Map menu, click Attributes.

  2. On the Workflow Map Attributes tab, click an attribute type in the Add Attribute list.

  3. Specify the attribute properties.

  4. Click OK.

  5. Optional Select the Disable Versioning check box to prevent Content Server from storing attribute value versions.

  6. Click Add to Workflow Map Definition.

For information related to this procedure, see Defining Attributes.
