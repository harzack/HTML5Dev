If you have enabled attributes in the Workflow Map, you can specify how attributes are displayed on a Start, User, Initiator, or Item Handler step's Attribute page. You can assign one of the following settings for each attribute:

  • Not Available, which hides the attribute.

  • Editable, which allows the step assignee to modify the value of the attribute.

  • Entry Required, which requires the step assignee to modify the attribute.

  • Read Only, which allows the step assignee to view the attribute and its value.

  • Ignore Action, which appears on the Physical Items tab and controls whether the Physical Item metadata used to create a Physical Item during a Workflow Create step is passed to subsequent steps without triggering another Create action. If the Ignore Option is selected, the action step is not available on a Workflow step and the Mandatory field requirements for the step are enforced. If the Action value is set to Create, either set as the default value in design mode or passed from another step, then Physical Item(s) are not created.

    If the action is set to Editable or Entry Required, the mandatory field requirements on a step execution are enforced if the value is set to Reject; if the action is set to Not Available or Read Only, the mandatory field requirements on a step execution are enforced if the value is set to Reject.
