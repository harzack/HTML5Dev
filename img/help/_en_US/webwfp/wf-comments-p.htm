To allow users to view all comments:

  1. Double-click a User or Initiator step.

  2. Click the Permissions tab.

  3. Select the See all comments check box.

  4. Click Add To Workflow Map Definition.

 

Note:
If the Single Page View check box is selected in the General tab, ensure the Comments package is listed in the Display Packages area.

 

For information related to this procedure, see Allowing Users to View Comments.
