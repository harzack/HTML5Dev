To define the Start step:

  1. In the Workflow Map Designer, double-click the Start step.

  2. On the General tab, specify a step name, select step options, provide instructions that will display in the General window, and then click Add to Workflow Map Definition.

  3. Optional If attachments are enabled for the Workflow Map, click the Attachments tab, provide Attachments window instructions, and then click Add to Workflow Map Definition.

  4. Click the Comments tab, provide Comments window instructions, and then click Add to Workflow Map Definition.

  5. Optional If attributes are enabled for the Workflow Map, click the Attributes tab, provide Attributes window instructions, specify an access option for each attribute, and then click Add to Workflow Map Definition.

  6. Optional If the optional Forms module is installed and enabled, select the Form and Form View, and then click Add to Workflow Map Definition.

  7. Save the Workflow Map.

For information related to this procedure, see Defining a Start Step.
