You define Workflow Map steps to specify the work, events, and participants in a Workflow Map process. You use links to control the order in which steps are processed. For more information about the steps and links, see Understanding Steps and Links. For a list of standard steps, see Choosing Steps.

The optional XML Workflow Extension module offers additional steps to specify the work, events, and participants in a Workflow process. XML Workflow Extension steps provide a means to automate certain processes and extend the functionality of a standard Content Server Workflow. For more information, see Defining XML Workflow Extensions Steps and Links.
