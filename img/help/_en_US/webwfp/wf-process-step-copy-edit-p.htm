To edit a copy attachment request:

  1. In the Workflow Map Designer, double-click a Process step.

  2. Click the Attachments tab.

  3. Click the Edit link in the Actions column.

  4. In the Copy to box, click Browse Content Server, navigate to the location to which you want to copy the item, and then click its Select link.

  5. Type the name under which you want Content Server to save the copy of the attachment in the Name of Copy box.

  6. Specify the method used to process categories and attributes on the copied item.

  7. Click Submit.

  8. Click Add to Workflow Map Definition.

For information related to this procedure, see Defining a Process Step.
