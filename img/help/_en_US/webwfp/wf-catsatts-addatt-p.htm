To choose an attribute:

  1. Double-click an Item Handler step.

  2. Click the Categories tab, and then click the Definitions link.

  3. On the Definitions tab, click one of the following links next to the category that contains the attribute you want to choose:

    • Add Attributes, to choose attributes from a category for the first time.

    • Edit Attributes, to modify an attribute you have already chosen.

  4. In the Select Category Attributes window, select the check box for each attribute you want.

  5. Click Submit.

For information related to this procedure, see Working with Categories and Attributes in Workflow Maps.
