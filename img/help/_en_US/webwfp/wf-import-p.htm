To import a Workflow Map:

  1. Add a Workflow Map.

  2. Click the Workflow Map's Functions icon, and then choose Import.

  3. Click Browse, navigate to an exported Workflow Map on your computer or network file system and then select it.

  4. Click one of the following radio buttons:

    • Maintain Performers

    • Reset Performers

  5. Click Import.

For information related to this procedure, see Importing and Exporting Workflow Maps.
