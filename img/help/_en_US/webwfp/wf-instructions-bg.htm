You can add instructions to Start, User, Initiator, Item Handler, and Process steps that explain what actions you want step assignees to complete on the Overview, Comments, Attributes, and Attachments step pages. For example, if users are required to attach a document, you can provide instructions that display a note at the bottom of the step's main and Attachments page. Comments, attributes, and attachments must be enabled in the Workflow Map before you can add instructions for them.

 

Note:
The instructions you provide can include basic HTML formatting codes. If you use special characters, you may be required to use HTML character codes so that the special characters are not interpreted as HTML. For example, if you want to use less than or greater than symbols (< or >), you must enter the HTML character codes for each symbol (&lt; or &gt;).
