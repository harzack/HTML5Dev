To Add a Text Document:

  1. From the Add Item menu, select Text Document.

  2. On the Add: Text Document page, in the Name field, type a name for the text document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the name in the other, enabled, languages.

  3. Optional In the Description field, type a description for your Text Document.

    If your system has multiple languages installed and enabled, click the Click to edit multilingual values button to edit the description in the other, enabled, languages.

  4. In the Text field, type the contents of the text document.

  5. Optional Also in the Text field, click the Check Spelling icon to check for spelling errors. The Check Spelling icon is located in the Text field to the left of the box you use to enter your content.

     

    Note:
    Spell checking is not available when Java is disabled. For more information, see Using Assistive Technology and Java.

     

  6. From the MIME Type list, select a MIME type for the text document. The default value is text/plain.

  7. Optional If available, you can apply a Classification to this Text Document. From the Classifications list, select Browse Classifications or My Favorites.

    1. In the dialog box which appears, select the Classification you will be applying to this Document.

    2. Ensure the Action check box next to the Classification is checked.

    3. Click Submit.

  8. Optional In the Categories field, click Edit... to either select, or add, a Category to apply to this Text Document.

  9. Optional If you want to place the Text Document in a location other than that which appears in the Create In field, click the Browse Content Server... button. Navigate to the container where you want to locate the Text Document, then click its Select link.

  10. Click Add.
