To add a URL module to the community homepage:

  1. In the Navigation Menu sidebar, point to Facilitator Controls, and then click Setup.

  2. On the Configure Community Settings page, click the Edit URLs button .

  3. On the Edit URLs page, click the Add URL button .

  4. On the Add URL page, do the following:

    • Type a name for the URL in the Name box.

    • Type the URL of the desired page in the URL box.

  5. Click Add.

  6. On the Edit URLs page, click Done.

  7. On the Configure Community Settings page, select the URL in the Content Modules list, and then click Add.

  8. Drag the URL content module to the desired position among the other active content modules.

  9. Click Finished.

 

Note:
For more information about adding URLs, see Working with Shortcuts and URLs.
