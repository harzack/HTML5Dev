To reject membership requests:

  1. In the Navigation Menu sidebar, click the Member Info link.

  2. On the Community Participants page, click the There are membership requests pending link.

  3. Select the check box for each user you want to reject.

  4. Click Reject.

  5. On the Create Email Message page, do one of the following:

    • Type a message explaining the reasons for rejecting the membership requests in the Message box, and then click Send.

    • Click Continue Without Sending.
