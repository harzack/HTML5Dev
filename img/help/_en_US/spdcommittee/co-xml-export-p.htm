To export a community or selected content:

  1. Click the item's Functions icon, and then click Export.

  2. On the Export page, click one of the following options in the Scope list:

    • This node only

    • This node and only the first level children

    • This node and all child nodes

  3. To include an item's category and attribute information in the exported data, click the include check box in the Categories & Attributes area.

  4. In the Versions list, click one of the following options:

    • Export all versions of the node

    • Export only the current version

  5. In the Content list, click one of the following options:

    • Encoded text (base-64)

    • Plain text

  6. To include an item's permissions information in the exported data, click the include check box in the Permissions area.

  7. Click Download and then save the XML file to your computer.

  8. Click Done.
