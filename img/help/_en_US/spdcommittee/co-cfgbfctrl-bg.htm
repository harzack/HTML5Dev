When you promote blogs, you make certain blogs more prominent in the community. You can promote blogs to their own content modules on the Community Homepage so that they appear with either the blog name or author name displayed prominently in a featured blog or featured author report style. You can also promote blogs to the headlines content module so that recent entries from one or more blogs appear in the Headlines area on the Community Homepage.

For blogs, you can format the title of a blog in a community headline to change the size of the font and how much text to display. For more information about blogs, see Using Blogs.

For more information about making content modules appear on the Community Homepage and making links active in a community's Navigation Menu sidebar, see Configuring Communities.
