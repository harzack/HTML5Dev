To search for members in a community:

  1. In the Navigation Menu sidebar, click the Member Info link.

  2. On the Community Participants page, type the search criteria for the member you are searching for in the Search in List box.

 

Note:
You can search by entering a user's first name, last name, sign-on name, or email address. You cannot search by group name or role. However, you can filter the list by type of role or expert group using the list beside the Search in List box.

If you filter the search results by expert group, expert group names are searched first. If no expert groups match the search criteria you typed in the Search in List box, user details such as first name, last name, sign-on name, and email address will then be searched.
