To set notifications for a role:

  1. In the Navigation Menu sidebar, click the Notification link.

  2. On the Community Notification page, click the link of the role for which you want to set notifications.

  3. Click the Modify Interests button and modify the settings and interests for the role.

  4. Click Submit.

 

Tip:
You can also set notifications for a role from the Member Info page by placing the mouse over the Tools menu, selecting Set Notification, and then selecting a role.

 
 

Note:
For more information about setting notifications, see Using Notifications.
