The community directory provides a list of communities and related items, such as its questions and answers, organized in a hierarchy of directories. It provides a way for users to browse and find communities, and identify the best place to ask questions and find answers. Directories are set up and maintained by users with system administration privileges. For more information about questions and answers, see Using Questions and Answers.

You can subscribe to a directory in the community directory, which means that you will receive email reports of updates to that directory. When you subscribe, you specify which item types you want to be notified about, for example, communities and questions. In the case of questions, you will receive notification of all question activity for questions listed in that directory, delivered in a question activity report. For other item types, you will only be notified of new items, delivered in a community activity report. For more information about subscribing, see Subscribing to Communities and Community Content.

Configuring email activity reports includes the following tasks:

  • Setting how frequently the reports will be delivered.

  • Setting the email message format for the reports.

The report frequency and message format settings for email reports also apply to rated items reports. For more information about rated items reports, see Using Rated Items Reports.

You can change the directories associated with an item at any time, provided you have the proper permissions.
