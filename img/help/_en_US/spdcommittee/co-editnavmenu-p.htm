To edit the navigation menu sidebar options:

  1. In the Navigation Menu sidebar, point to Facilitator Controls, and then click Setup.

  2. On the Configure Community Settings page, click the Navigation Menu tab.

     

    Note:
    To reposition a link, drag and drop the link anywhere within the Navigation Menu region.

     

  3. To insert community links, select the Menu Items radio button, select an item from the Add Menu Items list and drag it to the Navigation Menu region.

  4. To insert a shortcut to a Content Server item, select the Content Server Items radio button and then do the following:

    • Type a new title for the link.

    • Click Browse and to navigate and select the Content Server item of interest.

    • Click Add.

    To insert an external link, select the External Links radio button, type a title for the link in the Title box and then type the URL of the Web site in the URL box. Click Add.

  5. To insert a separator between links, select the Separator radio button and then click Add. Drag and drop the separator in the Navigation Menu.

  6. To delete an entry from the Navigation Menu, highlight then entry and click Delete.

  7. Click Finished.
