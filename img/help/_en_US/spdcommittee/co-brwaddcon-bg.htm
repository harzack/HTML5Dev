In a community, you can browse content in the following places:

  • The Community Homepage, which displays small reports of content, such as, recently added items.

  • The Library, which is the main storage location for community content.

  • The Outline, which displays the contents of the community in a hierarchical list.

    When viewing the outline, you can choose whether to include several item types in the list.

  • The Digest, which presents a summary of recent community activity, such as lists of new members and most viewed items.

  • My Assignments, which lists your assignments from workflows and task lists in the community. For more information about assignments, see To View Personal Report Pages.

  • The Notification page, which lists notification reports that the facilitator has set up for members by role. For more information about notifications, see Using Notifications.

  • The My Workspace page, which is your personal container within the community. You can add your own private content to this page, such as drafts of important documents.

  • The Community Questions page, which is a list of questions and other items in the community, organized in a hierarchy of community topics. For more information about questions, see Using Questions and Answers.

 

Note:
The contents of the Community Homepage, such as the availability of the library or the outline pages in the Navigation Menu sidebar, are determined by the facilitator of your specific community. If you are the facilitator, for information on editing the Navigation Menu sidebar options, see To Edit Navigation Menu Sidebar Options.

 

Adding Items to a Community

You add items to a community, such as documents, folders, and blogs, by adding them to the community library.

 

Note:
The facilitator must give you permission to add items to the community, and you must have the privilege to add certain item types in Content Server.

 

When you add an item to a community, you can assign it one or more community topics. You can change the community topics associated with an item at any time, provided you have the proper permissions. For more information about adding items to Content Server, see Working with General Items.
