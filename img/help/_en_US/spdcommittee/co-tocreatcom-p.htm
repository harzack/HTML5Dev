To create a community:

  1. On the Add Item menu, click Community.

  2. On the Add Community page, click one of the following radio buttons to specify the community type:

    • Public

    • Private

    • Unlisted

  3. Click Next.

  4. In the Name box, type the name of the community.

  5. In the Unique Name box, type the unique name for your community.

  6. In the Mission & Purpose box, type the mission and purpose of the community.

  7. In the Template area, click Browse Content Server, navigate to the community template you want, and then click the template's Select link. Contact your administrator to create a community template.

  8. In the Location area, click Browse Content Server, navigate to the container where you want to add the item, and then click the container's Select link.

  9. In the Directories area, select the check box for each item, such as questions and answers, you want associated with this community.

  10. Click Next.

  11. On the Subscriptions tab, select the Subscribe check boxes for the roles that you want subscribed to the community.

  12. Select the Create check box in the Submission Folder area to create a submission folder.

  13. Select the Allow check box in the External Questions area to allow the community to receive external questions.

  14. Click Next.

  15. On the Members tab, click one of the following radio buttons in the Members area:

    • Not now, I'll add members later.

    • Inherit members based on folder permissions.

    • Add the following users and then click the Add Members link to add the users and groups.

  16. Click one of the following radio buttons in the Send e-mail to area:

    • All members of enabled groups.

    • Only direct members of enabled groups.

  17. Click Finish.

 

Tip:
You can also add a community by clicking the Create Community button on your Communities page.
