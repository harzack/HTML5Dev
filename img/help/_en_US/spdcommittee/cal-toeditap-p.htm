To edit an appointment:

  1. In the Navigation Menu sidebar, click the Calendar link.

  2. Click the link for the appointment that you want to edit.

  3. On the Edit Event page, do any of the following:

    • In the Attendees area, add or remove roles by selecting or clearing the corresponding check boxes.

    • In the Title box, edit the title of the appointment.

    • In the Start and End areas, click the Date selector icon to edit the dates and times for the start and end of your appointment.

    • In the Description box, type a description of the appointment.

  4. Click Update.
