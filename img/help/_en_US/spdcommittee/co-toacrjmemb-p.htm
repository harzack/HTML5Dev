To accept membership requests:

  1. In the Navigation Menu sidebar, click the Member Info link.

  2. On the Community Participants page, click the There are membership requests pending link.

  3. Select the check box for each user that you want to accept as a member.

  4. Select a role in the Role list to assign an accepted user to a role.

  5. Click Accept.

  6. On the Create Email Message page, do one of the following:

    • Type a message welcoming the new members to the community in the Message box, and then click Send.

    • Click Continue Without Sending.
