Moon
"hors*"
="jumping jacks"
[qlregion "OTData"] "home"
[qlregion "OTFileType"] qlrange "pdf|word|excel"
[qlregion "OTFileSize"] qlrange "80000~120000"
[qlregion "OTIndexErrors"] !="2"
[qlregion "OTFileName"] *

Operators are used to join Search Terms. If not specified, AND is assumed.

AND, OR, NOT, XOR, SOR
PROX[5] - second term within 5 words of first term
NEAR or QLNEAR - same as PROX[10,F]

To search for operators, enclose in quotes:

"twist" "not" "shout"

Order of evaluation is left to right. Brackets can be used to clarify or change this.

Moon and (phobos or deimos)

Comparators are applied to a single term. You can use a comparator or a modifier, not both.

=, !=, >, <, >=, <=

Modifiers request special handling for a single term.

qlthesaurus - also match synonyms
qlrange - match within a range or from a list
qlstem - match singular or plural forms
qlregex - match based on a pattern (regular expression)
qllefttruncation term (easier usage: *term)
qlrighttruncation term (easier usage: term*)

Metadata fields. If not specified, search terms match the full text body and the 'default' search regions. Specify a metadata field for a search term:

[qlregion "fieldname"] "search phrase"

To restrict matches to fields with metadata attributes. Only available with a few metadata fields in Content Server, usually for languages:

[qlregion "OTName"][qlattribute "lang"="en"] "English phrase"

To search for a term in full text only (not metadata):

[qlregion "OTData"] "term"

To search for special characters in metadata, precede with a backslash character:

[qlregion "url"] "c\:\\root\\folder"
