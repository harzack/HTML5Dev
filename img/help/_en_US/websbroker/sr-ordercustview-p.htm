To order Custom Views

  1. Navigate to the location where your Custom Views or saved Searches are located in Content Server, by clicking the container's Functions menu, and then choose Order Custom Views.

     

    Tip:
    You can display all the Custom Views that are stored in your Personal Workspace or your Project Workspace by choosing Reports on the Personal or Project menu.

     

  2. On the Order Custom Views page, click each Custom View object, and drag it up or down to define its display priority. The Custom View object at the top will be displayed as the default item in the Use this Custom View Search drop-down list in the location or the container.

     

    Note:
    When a Custom View object appears dimmed, it indicates that a Custom View item exists, but you cannot open it since you only have see permissions. You cannot drag this object up or down but you can arrange other objects around it.

    You cannot see the Content Server items for which you do not have see permissions.

    For more information about permissions, see Understanding Permissions.

     

  3. Click OK.

For more information about Custom Views, see Working with Custom Views.
