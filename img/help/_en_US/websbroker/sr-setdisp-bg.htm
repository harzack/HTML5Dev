Setting display options lets you customize how the results of your search query are displayed on the Search Results page. You can choose a results page layout style and set other specific options, such as how many results are displayed on one page. Choosing a page style allows you to select how much information, and in what format, you want to display elements on the page. You can apply these display options for the current search only, or create or modify a Search Form so that results appear using these options by default. For more information on Search Results Forms, see Working with Forms and the Search Results Page.

In the Results Page Layout section, there are four available page styles from which you can choose:

  1. Classic

  2. Standard

  3. Tabular

  4. Web Search, which is the default

The default style is normally recommended. The layout elements are sticky for your Content Server sessions, and are modified from the Search Results page for features such as display style.

The following table describes each page style and its default elements:

For more information about finding similar items, see Finding Similar Documents.

A Show items list at the bottom of the page, which allows you to specify the number of search results you want to appear on a results page.

•

A list at the top of the page which allows you to collect your search results and save your search.

•

A list of search results in the middle of the page including elements such as a breadcrumb trail that provides the location path of the item; a link to the Properties page of each item; and links to add items to your Favorites, or a Collection, or Share them. A Show items list at the bottom of the page, which allows you to specify the number of search results you want to appear on a results page.

A pagination bar at the bottom of the page, which allows you to navigate through search results pages.

The Preview area shows how the page layout style you select will look.

Depending on the selected layout style, the following controls may also apply:

Number of Results You can specify the number of search results you want to appear on the search results page by default. If the number of search results exceed the specified number, additional search results pages are added.
Show Summaries/Descriptions You can also show or hide search result summaries and descriptions. You have the option to select summaries, descriptions, or both as the preferred search result information for Content Server to display. For example, if you select Summaries Preferred, a summary for each search result, is displayed if available. If a summary is not available, a description of the search result is displayed if available. For more information about search results, see Working with Search Results.
Show Key Phrases You can show or hide key phrases. Key phrases are recurring words and word combinations, especially those involving unusual words.
Show Full Path in Location You can choose to show the full path of the location of each search result.
Result Fields You can specify which result fields you want to display on the Search Results page, such as the size or location of the items listed and remove the result fields you do not want to appear.
 

Note:
Your administrator may have configured specific search fields as required. If Required fields have been configured, an additional Required section appears on the Display Options page with these fields listed. You cannot remove required fields or change the position of the fields from the Search Results page.

 
Sort By You can configure the default sort order to be used when search results are returned. For more information about sort options, see Sorting Search Results.
