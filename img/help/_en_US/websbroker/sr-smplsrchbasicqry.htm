The following examples use the Content Server Search page, which is accessed from the Global Menu Bar. For more information about the Global Menu Bar, see Understanding the User Interface.

 

Tip:
You can also access the Content Server Search page by clicking the Open Search Panel down arrow from the Content Server Search Bar, then clicking Advanced Search.

 

For general information about searching with Basic Queries, see Searching with Basic Queries.

  • Search for all Documents that contain the exact phrase Release 5 and were created by the user WSmith.

  • Search for all Discussions related to deadlines in the New Release project.

  • Search for Content Server items in the Enterprise slice, that contain the word supply.

  • Search for all Content Server items that have Français associated with them in the Name and Description fields on the General tab of an item's Properties page.

To search for all Documents that contain the exact phrase Release 5 and were created by the user WSmith:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Exact Phrase.

    2. In the Enter Search Terms text field, type Release 5.

  3. In the Advanced Options section, click System Attributes. In the System Attributes section, do the following:

    1. Click Documents in the Content Type list.

    2. Type WSmith in the Created By field.

  4. Click Search.

To find all Discussions related to deadlines in the Next Release project:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Enter Search Terms text field, type deadlines.

    2. In the Modifier list, click Related To.

  3. In the Advanced Options section, click System Attributes.

  4. In the System Attributes section, in the Content Type list, click Discussions.

  5. In the Advanced Options section, click Location.

  6. In the Location section, click Browse Content Server..., locate the Next Release project, and then click its Select link.

  7. Click Search.

To search for Content Server items that contain the word supply in the Enterprise slice:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, type supply in the Enter Search Terms text field.

  3. In the Advanced Options section, click Slices.

  4. In the Slices section, click Enterprise.

  5. Click Search.

To search for all Content Server items that have Français associated with them in the Name and Description fields on the General tab of an item's Properties page:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click System Attributes.

  3. In the System Attributes section, do the following:

    1. In the Additional Attributes list, select Name.

    2. In the Name text box, type *.

    3. Click the Any list beside Name, and select fr.

    4. In the Additional Attributes list, select Description.

    5. In the Description text box, type *.

    6. Click the Any list beside Description, and select fr.

  4. Click Search.
