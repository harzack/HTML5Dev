Although most of these Queries can also be constructed using the elements on the Content Server Search page, the following samples demonstrate how to use the Content Server Query Language to submit complex Queries.

These Queries use the Content Server Search page, which is accessed from the Global Menu Bar. For more information about the Global Menu Bar, see Understanding the User Interface.

 

Tip:
You can also access the Content Server Search page by clicking the Open Search Panel down arrow from the Content Server Search Bar, then clicking Advanced Search.

 

For general information about searching using Complex Queries, see Searching with Complex Queries.

  • Search for all help pages that mention projects using the Content Server Search page.

  • Find all Content Server items that contain the phrase next release date.

  • Search for Content Server items that contain words that sound like video and contain at least one word derived from the stem of the word music.

  • Search for Content Server items that contain thesaurus entries associated with the word concert or performance and at least one word derived from the stem of the word music.

  • Search for Content Server items that contain words whose prefixes match perform or sing and words that sound like band.

  • Search for Content Server items that contain the words choreography and dance or words that sound like performer.

  • Search for Content Server items that contain words whose prefixes match air or sky or words whose prefixes match dome or center.

  • Search for Content Server items with French names containing the word voiture.

  • Search for all Content Server items that do not have a language associated with them in the Description field on the General tab of an item's Properties page.

  • Search for Content Server items that were created by the user jsmith.

  • Search for Content Server items created by the user whose name matches the suffix son.

  • Search for Content Server items that were reserved by the user jsmith and not by the user rdonald.

  • Search for Content Server items that contain image/GIF MIME type and not the application/MSWord MIME type.

To use the Content Server Search page to search for all help pages that mention Projects:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, type project* in the Enter search terms text field.

  3. In the Advanced Options section, click Slices.

  4. In the Slices section, click Help.

  5. Click Search.

To find all Content Server items that contain the phrase next release date:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type the following Content Server Query Language statement: "next release date".

  3. Click Search.

To search for Content Server items that contain words that sound like video and contain at least one word derived from the stem of the word music:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type (qlphonetic "video") AND (qlstem "music").

  3. Click Search.

To search for Content Server items that contain thesaurus entries associated with the words concert or performance and at least one word derived from the word music:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type (qlthesaurus "concert" OR qlthesaurus "performance") AND (qlstem "music").

  3. Click Search.

To search for Content Server items that contain words whose prefixes match perform or sing and words that sound like band:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type (qlright-truncation "perform" OR qlright-truncation "sing") AND (qlphonetic "band").

  3. Click Search.

To search for Content Server items that contain the words choreography and dance or words that sound like performer:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type ("choreography" & "dance" ) OR (qlphonetic "performer").

  3. Click Search.

To search for Content Server items that contain words whose prefixes match sky or air or items that contain words whose suffixes match dome or center:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type (qlright-truncation "sky" OR qlright-truncation "air") OR (qlleft-truncation "dome" OR domeqlleft-truncation "center").

  3. Click Search.

To search for Content Server items with French names containing the word voiture:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTName"] [qlattribute "lang" = "fr"] "voiture".

  3. Click Search.

To search for all Content Server items that do not have a language associated with them in the Description field on the General tab of an item's Properties page:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTDComment"] [qlattribute "lang" is null].

  3. Click Search.

To search for Content Server items created by the user jsmith:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTCreatedByName"] "jsmith".

  3. Click Search.

To search for Content Server items created by a user whose name matches the suffix son:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTCreatedByName"] qlleft-truncation "son".

  3. Click Search.

To search for Content Server items that were reserved by the user jsmith and not by the user rdonald:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTReservedByName"] "jsmith" NOT [qlregion "OTReservedByName"] "rdonald".

  3. Click Search.

To search for Content Server items that contain image/GIF MIME types and not the application/MSWord MIME type:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Full Text section, do the following:

    1. In the Look For list, click Complex Query.

    2. In the Enter search terms text field, type [qlregion "OTMIMEType"] "image/GIF" NOT [qlregion "OTMIMEType"] "application/MSWord".

  3. Click Search.
