The Content Server Search Bar provides fast and convenient searching capability on almost every Content Server page. The table below describes the Content Server Search Bar. However, the Search Bar can be fully customized by your administrator, so the options that appear at your site may differ from the ones discussed in this section. For more information about the Search Bar at your site, contact your administrator.

The Content Server Search Bar appears on almost every page, by default, at the far right side of the Global Menu bar. Examining the Search Bar from left to right, you will see:

  • the text Search,

  • the “Search From Here” input box,

  • the “Perform Search” button,

  • and the “Open Search Panel” down arrow which expands the Search Bar to a search panel. If the search panel is expanded, then the down arrow becomes an up arrow called “Close Search Panel”.

If you enter text to the Search field and click the search button, Content Server will search that page for the text that you entered.

The Search Bar can be expanded to a search panel by clicking the down arrow to the right of the search button. Depending on how your administrator has set up your search function, you may see the following items in the search panel:

Content Server Item Function
Search field Performs a full text search on the text that you enter to the field.
Search Type list This allows you to select one of: Full Text, Nickname, and Natural Language Query. Depending on your choice, the search panel will dynamically display the options available for that search type. For more information see Assigning Nicknames to Items and Searching with Natural Language Queries.
Slices list. This is available when performing a Full Text or Natural Language Query search. This allows you to select one of: From Here, From Here (Expanded), Help, Enterprise, and Enterprise [All Versions]. This option allows you to specify the Content Server slice that you want to search. You can search from the Folder or Workspace you are currently in by selecting From Here, or you can search the entire enterprise.
 

Note:
If a slice is deleted, it may still appear in the Slices drop-down list, and may be returned in a func=search.getslices function call. This is because of the caching characteristics in Content Server. Once the cache is cleared by all active threads the slice will no longer be returned.

 
Action list. This is available when performing a Nickname search. This allows you to select either Open or Properties. If you select Open, your search will bring up the item. If you select Properties, your search will bring up the item's Properties page. For more information see Assigning Nicknames to Items.
Mode list. This is available when performing a Full Text or Natural Language Query search. When performing a Full Text search, Mode allows you to narrow your search by selecting one of: All Words, Any Words, Exact Phrase, or Complex Query. When performing a Natural Language Query search, Mode allows you to narrow your search by selecting one of: Keyword Extraction or Content Server Aware. For more information see Searching with Natural Language Queries.
Modifier list. This is available when performing a Full Text search. Allows you to narrow your search by selecting one of: None, Synonyms Of, Related To, Sounds Like, Word Begins With, or Word Ends With.
Content Type list. This is available when performing a Full Text search. When performing a Full Text search, Content Type allows you to specify the type of item for which you want to search. For example, you can choose from particular file formats or Content Server item types. Your administrator sets the object types which are searchable for your system.
Created By list. This is available when performing a Full Text search. Allows you to narrow your search by selecting one of: Anyone, Created by me, or Created by my group. If this search option is not available, contact your administrator.
Last modified list. This is available when performing a Full Text search. Lets you limit the scope of your search to a definite time period, or Slice, such as Modified in the past day or Modified in the past month. If this search option is not available, contact your administrator.
Help link. Brings up the help page associated with the search panel. This link is visible on the Search Bar only when it is enabled by your administrator.
Last Search Results link. Opens your last Search Results page. This link is visible on the Search Bar only when it is enabled by your administrator.
Advanced Search link. This is available when performing a Full Text or Natural Language Query search. Opens the Advanced Search page. This is visible on the Search Bar only when it is enabled by your administrator.

 

Note:
When you are in a Workspace that contains OpenText Enterprise Library items, you can use the From Here option on the Search Bar to return OpenText Enterprise Library Folder, Variants, and Items from your current Workspace. For more information, see Using the Search Bar to Find Open Text Enterprise Library Services Items.

 

When you use the Search Bar, the search terms that you specify are retained in the Search Bar on the Search Results page. This allows you to refine the terms and run a new search without leaving the Search Results page. For samples of Search Bar Queries, see the Searching from the Content Server Search Bar section in the Sample Queries reference topic.

 

Note:
Because you use the Content Server Search page to construct more complex Queries, the search terms that you specify on the Content Server Search page are not retained in the Search Bar on the Search Results page.
