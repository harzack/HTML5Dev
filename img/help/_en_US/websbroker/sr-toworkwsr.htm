To... Do the following...
Sort search results On the Search Results page, click a sort results option in the drop-down list beside the Sort button, and then click the Sort button.
Sort search results using Multi-Field On the Search Results page, click Mutli-Field in the Sort list. Next, select the criteria that should be applied first. Then, click the Plus button, , to add additional sort criteria. Once you have selected the sort criteria you want to apply, click OK. Finally, click Sort.
Change the Number of Search Results on a Page On the Search Results page, click a value in the Show items list.
Filter Search Results according to metadata values On the Search Results page, click one or more values in the Search Filters panel.
View instances of search terms in a search result On the Search Results page, click an item's Functions menu, choose Hit Highlight, and then click the link you want in the Highlight window.
Save a Search Results Snapshot On the Search Results page, select the check box for each result you want to add to the Search Result Snapshot, and then click Save Your Results in the Select Action drop-down list. Next, specify the name of the Search Result Snapshot, the location where you want to save it, and any other information you want, and then click the Add button.
Open a Search Result Snapshot Navigate to the location where the Search Result Snapshot is located in Content Server, click the Snapshot's Functions menu, and then choose Open.
Collect All Search Results
•

On the Search Results page, in the Search Tools drop-down list, click Collect All Search Results.

•

On the Collect All Search Results page, edit the email notification parameters, click the Collection setting you want from the Select drop-down list, and then click the Submit button.

Download a Collection as a Spreadsheet After you add your Search Results to a Collection:
•

Browse to any Collection in Content Server, and select the check box for each item you want to download, or the entire collection.

•

From the More Actions drop-down list, click Download as Spreadsheet.

 

Tip:
You can see all the Search Result Snapshots that are stored in your Personal Workspace or your Project Workspace by choosing Reports on the Personal menu.
