The types of searches that you can perform from the Content Server Search Bar depend on how your administrator has configured it. For more information about how the Search Bar is configured at your Content Server site, contact your administrator.

The following examples are based on the default settings for the Content Server Search Bar, and therefore may, or may not, reflect the settings for the Content Server Search Bar at your site. For general information about using the Content Server Search Bar, see Using the Content Server Search Bar.

  • Search for Content Server items that contain the URL www.opentext.com.

  • Search for all User Online Help pages about projects.

To use the Content Server Search Bar to search for Content Server items that contain the URL www.opentext.com:

  1. On the Content Server Search Bar, click Ask Content Server a question in the drop-down list.

  2. Type Find me items containing the URL www.opentext.com in the Search From Here text field on the Content Server Search Bar.

  3. In the Slices list, click Enterprise.

  4. Click Go.

To use the Content Server Search Bar to search for all User Online Help pages about Projects:

  1. On the Content Server Search Bar, click Search Content Server for in the drop-down list.

  2. Type project* in the Search From Here text field.

  3. In the Slices list, click Help.

  4. Click Go.
