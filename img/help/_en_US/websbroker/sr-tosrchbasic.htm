To search using a basic Query:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, type your Query in the Full Text field.

  3. Specify the search criteria that you want.

    If you want to use a Boolean operator, click the Duplicate button, , and then choose an operator.

  4. Click Search.

  5. If you want to configure how results are displayed on the Search Results page, click the Select Display Style drop-down menu, and select Edit the Display Options to open the Display Options page to choose how your search results are displayed.

 

Note:
When you specify a date range for system attributes or custom attributes within a Category, you can leave the date in either the From or the To lists blank. If you leave the From list blank, Content Server searches all dates prior to the date in the To list. If you leave the To list blank, Content Server searches from the date in the From list to the present.

When searching for dates in the System Attributes and Categories components, you can choose the “blank” option. This is the default setting, which excludes the date field from the search query.

When searching for dates in the System Attributes component, if you specify Past 3 Months or Past 6 Months, in the Date drop-down list, the date range specified by the query is not exactly 3 or 6 months. Instead, the calculation is performed using 90 days and 180 days, respectively. Since some months may have more or less than 30 days, the range over which the query operates may miss some items that would otherwise be found.

You cannot search XML files through a System Attributes search. To perform a search on an XML region, use the XML Types component to select the DTD region.
