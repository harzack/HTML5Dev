To save a Search:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, construct a Search.

  3. In the Search Tools drop-down list, click Save Your Search.

  4. Click the button.

  5. On the Save Search Query page, specify the name of the Query, the location where you want to save it, and any other information that you want to specify.

  6. Click Add.

 

Tip:
You can also save a Query by clicking Save Query on the Search Results page.
