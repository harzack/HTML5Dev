Content Server's searching capabilities provide a way to find information stored in Content Server that is relevant to you. You can locate Documents, Folders, Discussions, and other Content Server items. You can also locate items that have a language associated with them. A search can be as basic as finding Documents that are similar to a Document in a Workspace. You can also create more complex searches using Queries. A Query is a Content Server item that stores search criteria and display options for results.

 

Note:
You can search for OpenText Enterprise Library items, provided you have the appropriate permissions.
