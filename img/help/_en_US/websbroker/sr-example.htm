The following sample Queries demonstrate the different ways to search Content Server. The examples are listed in order of complexity, beginning with finding similar documents, which is a basic type of Content Server Search, and concluding with the more advanced searching capabilities of the Content Server Query Language.

Each title in the Related Subtopics section below, relates to a specific Searching Content Server help topic. For more information about a particular type of search, see the corresponding help topic in Searching Content Server.

 

Note:
The Search Bar can be fully customized by your administrator, so the options that appear at your site may differ from the ones discussed in this section. For more information about the Search Bar at your site, contact your administrator.
