To choose a Search Form:

  1. Choose Search on the Tools menu.

  2. In the Use this Search Form drop-down list, click the name of a Form in one of the following sections:

    • Personal Search Forms

    • System Search Forms

 

Tip:
To make a Form your default Search Form, click the Save as Search Form button, select the Set as Default Search Form check box, and then click Save. If you log in as the Admin user, your default Form is always the System Default Template and the Set as Default Search Form check box is not available.

You can return to the default Search Form at any time by clicking Revert to Default on the Use this Search Form drop-down list.
