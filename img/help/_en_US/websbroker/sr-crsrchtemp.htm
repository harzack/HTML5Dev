To create a Search Form:

  1. Choose Search on the Tools menu.

  2. On the Content Server Search page, modify the search criteria.

  3. Click the Save as Search Form button.

  4. On the Save as Search Form page, type a name in the Search Form Name field to name a new Form.

  5. Click Save.

 

Tip:
On the Use this Search Form drop-down list you can also click a Search Form name in the Personal Search Forms or the System Search Forms lists to overwrite an existing Search Form.
