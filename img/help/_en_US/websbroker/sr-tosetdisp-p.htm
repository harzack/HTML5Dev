To set display options:

  1. Specify your search criteria by typing your query in the Search field on the Global Menu Bar.

  2. Click the Perform Search button.

  3. On the Search Results page, click the Select Display Style drop-down menu, and select Edit the Display Options to open the Display Options page.

  4. On the Display Options page, choose how to display your search results. For detailed information, see Setting Display Options.

  5. Click the Select Action drop-down menu, and select Save Your Results to add a Search Result Snapshot.
