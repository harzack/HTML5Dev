To search using a Natural Language Query:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click Natural Language Query.

  3. Type your Query as a question or text.

  4. Click a Natural Language Query mode in the Mode list.

  5. Click Search.

 

Tip:
You can also perform a Natural Language Query from the Content Server Search Bar that appears at the top of most Content Server pages. To find out if this is enabled at your Content Server site, contact your administrator. For more information about the Content Server Search Bar, see Using the Content Server Search Bar.
