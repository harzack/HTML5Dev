The following examples use the Content Server Search page, which is accessed from the Global Menu Bar. For more information about the Global Menu Bar, see Understanding the User Interface.

 

Tip:
You can also access the Content Server Search page by clicking the Open Search Panel down arrow from the Content Server Search Bar, then clicking Advanced Search.

 

For general information about searching using Natural Language Queries, see Searching with Natural Language Queries.

  • Search for the exact phrase Show me documents about OpenText.

  • Search for Documents about the user conference created approximately two weeks ago.

  • Search for Discussions about sales that occurred after the summer of 2013.

  • Search for Folders with the title compensation.

To search for the exact phrase Show me documents about OpenText:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click Natural Language Query.

  3. In the Natural Language Query section, do the following:

    1. In the Mode list, select Keyword Extraction or Content Server Aware.

    2. In the Enter a question or text field, type "Show me documents about OpenText", including the quotation marks.

  4. Click Search.

To search for Documents about the user conference created approximately two weeks ago:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click the Natural Language Query link.

  3. In the Natural Language Query section, do the following:

    1. In the Mode list, click Keyword Extraction or Content Server Aware.

    2. In the Enter a question or text field, type Find documents about the user conference created two weeks ago.

  4. Click Search.

To search for Discussions about sales that occurred after the summer of 2013:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click Natural Language Query.

  3. In the Natural Language Query section, do the following:

    1. In the Mode list, click Keyword Extraction or Content Server Aware.

    2. In the Enter a question or text field, type Show me all discussions about sales after the summer of 2013.

  4. Click Search.

     

    Note:
    Content Server defines the summer as any date between June 21st and September 21st.

     

To search for Folders with the title compensation:

  1. On the Global Menu Bar, select Search from the Tools menu.

  2. On the Content Server Search page, in the Advanced Options section, click Natural Language Query.

  3. In the Natural Language Query section, do the following:

    1. In Mode list, click Keyword Extraction or Content Server Aware.

    2. In the Enter a question or text field, type Where are folders that have the title compensation?.

  4. Click Search.

     

    Note:
    Content Server recognizes compensation as the title of the Folder because compensation is typed after title. Content Server specifically searches for the word compensation in titles instead of in summaries, descriptions, or the body of Documents. Summaries and descriptions are also specifically searchable.
