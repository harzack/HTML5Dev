Supposons que votre entreprise demande aux employés de remplir un formulaire de profil de déplacement. L'agent de voyages de l'entreprise se réfère au profil lorsqu'il organise des voyages pour les employés. Dans certains cas, les préférences de voyage (par exemple, siège près du hublot ou de l'allée ou agence de location de voitures préférée, etc.) peuvent changer.

Pour configurer un formulaire de profil de déplacement :

  1. Créez un modèle de formulaire qui contient le texte et les champs que vous souhaitez inclure dans le profil de déplacement. Pour plus d'informations sur la définition d'un modèle de formulaire, reportez-vous à la rubrique Définition d'un modèle de formulaire.

  2. Ajoutez un formulaire basé sur un modèle de formulaire. Lorsque vous définissez un formulaire, utilisez les paramètres suivants :

    • Dans la liste Mécanisme de révision, cliquez sur <Aucun>.

    • Dans la liste Mécanisme d'envoi, cliquez sur Enregistrement révisable par l'utilisateur (Content Server).

    Pour plus d'informations sur la définition des formulaires, reportez-vous à la rubrique Définition d'un formulaire.

Utilisation de l'exemple du profil de déplacement

Ouvrez le formulaire de profil de déplacement, indiquez vos préférences, puis envoyez la version actuelle du formulaire. La prochaine fois que vous ouvrirez le formulaire, il affichera les valeurs précédemment enregistrées. Si vous souhaitez mettre à jour le formulaire, renvoyez-le afin d'enregistrer les nouvelles données.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des exemples de formulaires.
