Pour supprimer toutes les données de formulaire stockées :

  1. Cliquez sur l'icône Fonctions du formulaire qui contient les données envoyées à supprimer, puis sélectionnez Supprimer les données envoyées.

  2. Dans la boîte de dialogue de confirmation, cliquez sur OK.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des données de formulaire.
