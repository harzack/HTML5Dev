Pour gérer des versions de modèle de formulaire :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire, choisissez Propriétés, puis Versions.

  2. Sur la page Propriétés, modifiez les paramètres comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Modifier une version Cliquez sur le lien Nom du fichier de la version que vous souhaitez modifier.
    Verrouiller ou déverrouiller une version Cliquez sur l'icône Fonctions de la version que vous souhaitez verrouiller ou déverrouiller, puis cliquez sur Verrouiller ou Déverrouiller.
    Supprimer une version Cliquez sur l'icône Fonctions de la version que vous souhaitez supprimer, puis cliquez sur Supprimer.
    Elimination des versions Dans le champ Eliminer toutes les versions à l'exception de la plus récente, saisissez le numéro de la version la plus récente que vous souhaitez conserver, puis cliquez sur Eliminer les versions pour éliminer les autres versions.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des modèles de formulaire.
