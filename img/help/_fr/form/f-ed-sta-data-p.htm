Pour… Effectuez les opérations suivantes…
Convertir un formulaire en modèle prérenseigné Cliquez sur l'icône Fonctions du formulaire, choisissez Propriétés, puis Infos spécifiques. Dans l'onglet Infos spécifiques, activez la case à cocher Modèle prérenseigné, puis cliquez sur Envoyer.
Convertir un modèle prérenseigné en un formulaire Cliquez sur l'icône Fonctions du formulaire, choisissez Propriétés, puis Infos spécifiques. Dans l'onglet Infos spécifiques, désactivez la case à cocher Modèle prérenseigné, puis cliquez sur Envoyer.
Modifier les données du modèle prérenseigné Cliquez sur l'icône Fonctions du modèle prérenseigné, sélectionnez Modifier les données préinsérées. Sur la page Modifier, modifiez les données du champ, puis cliquez sur Appliquer.

Pour plus d'informations sur cette procédure, consultez la section Définition d'un modèle prérenseigné.
