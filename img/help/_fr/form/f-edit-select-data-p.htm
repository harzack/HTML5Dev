Pour modifier un ensemble de données de formulaire :

  1. Sur l'icône Fonctions du formulaire qui contient l'ensemble de données que vous souhaitez modifier, cliquez sur Répertorier les données.

  2. Sur la page Répertorier les données du formulaire, cliquez sur le lien Modifier de l'ensemble de données que vous souhaitez supprimer.

  3. Sur la page Modifier, cliquez sur Appliquer ou sur un autre bouton optionnel qui active le mécanisme d'envoi.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des données de formulaire.
