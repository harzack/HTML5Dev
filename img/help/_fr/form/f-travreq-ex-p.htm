Supposons que votre entreprise demande aux employés de remplir un formulaire de demande de déplacement avant qu'ils effectuent un voyage d'affaires. Le service de déplacement doit traiter les demandes et réalise des statistiques annuelles sur les besoins en déplacement de votre entreprise.

Pour configurer un formulaire de demande de déplacement :

  1. Créez un modèle de formulaire qui contient le texte, les champs et les boutons que vous souhaitez voir apparaître sur le formulaire.

  2. Associez le modèle à une table de base de données SQL nommée TravelProfile. Pour ce faire, cliquez sur l'icône Fonctions du modèle de formulaire et sélectionnez Gérer la table relationnelle. Définissez les paramètres suivants :

    • Dans la zone Nom de la table SQL, entrez TravelProfile.

    • Dans la liste Créer une table SQL, cliquez sur Créer une table de base de données avec le nom indiqué ci-dessus.

    Pour plus d'informations sur la définition d'un modèle de formulaire, reportez-vous à la rubrique Définition d'un modèle de formulaire.

  3. Ajoutez un formulaire en sélectionnant le modèle de formulaire :

    • Dans la liste Mécanisme de révision, cliquez sur Versions Content Server.

    • Dans la liste Mécanisme d'envoi, cliquez sur Table SQL.

    Pour plus d'informations sur la définition des formulaires, reportez-vous à la rubrique Définition d'un formulaire.

Utilisation de l'exemple du profil de déplacement

Supposez que vos préférences de voyage évoluent et mettez votre formulaire de demande de déplacement à jour. Chaque fois que vous ouvrez le formulaire, il affiche les valeurs précédemment enregistrées. Vous pouvez afficher l'historique du profil de déplacement en accédant aux précédentes versions du formulaire. Pour plus d'informations sur l'accès aux versions du formulaire, reportez-vous à la section Utilisation des formulaires.

Content Server stocke les données envoyées dans une table distincte de la base de données Content Server. Le service des déplacements peut accéder aux données via des applications externes pour traiter les demandes de déplacement. Ils peuvent également utiliser des rapports LiveReport pour réaliser des statistiques annuelles sur les besoins en déplacement.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des exemples de formulaires.
