Pour modifier un champ dans un modèle de formulaire :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire dans lequel vous souhaitez modifier un champ, puis sélectionnez Modifier.

  2. Sur la page Modèle de formulaire, cliquez sur le lien du nom du formulaire que vous souhaitez modifier.

  3. Dans la page Modifier un attribut, définissez les propriétés du champ, puis cliquez sur OK.

  4. Sur la page Modèle de formulaire, cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Définition d'un modèle de formulaire.
