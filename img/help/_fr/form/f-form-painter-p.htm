Pour ajouter un champ à un modèle de formulaire :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire, puis sélectionnez Modifier.

  2. Sur la page Modèle de formulaire, dans la liste Ajouter un attribut, cliquez sur le Type d'attribut du champ de formulaire que vous souhaitez ajouter.

  3. Sur la page Ajouter un attribut, définissez les propriétés du champ, puis cliquez sur OK.

  4. Sur la page Modèle de formulaire, cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Définition d'un modèle de formulaire.
