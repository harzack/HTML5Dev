Supposons que votre entreprise effectue un suivi du nombre d'heures de travail des employés par l'intermédiaire de feuilles de présence électroniques. Chaque employé remplit une feuille de présence séparée.

Pour définir un formulaire de feuille de présence électronique :

  1. Pour chaque employé, créez un modèle de formulaire qui contient le texte, les champs et les boutons que vous souhaitez voir apparaître sur la feuille de présence.

  2. Associez chaque modèle à une table de base de données SQL nommée du nom de l'employé. Pour ce faire, cliquez sur l'icône Fonctions du modèle de formulaire et sélectionnez Gérer la table relationnelle. Définissez les paramètres suivants :

    • Dans la zone Nom de la table SQL, saisissez le nom de la table, identique au nom de l'employé.

    • Dans la liste Créer une table SQL, cliquez sur Créer une table de base de données avec le nom indiqué ci-dessus.

    Pour plus d'informations sur la définition d'un modèle de formulaire, reportez-vous à la rubrique Définition d'un modèle de formulaire.

  3. Pour chaque employé, ajoutez un formulaire basé sur le modèle de formulaire de l'employé. Lorsque vous définissez un formulaire, utilisez les paramètres suivants :

    • Dans la liste Mécanisme de révision, cliquez sur Table SQL.

    • Dans la liste Mécanisme d'envoi, cliquez sur <Aucun>.

    Pour plus d'informations sur la définition des formulaires, reportez-vous à la rubrique Définition d'un formulaire.

Utilisation de l'exemple de feuille de présence

Dans plusieurs feuilles de présence, enregistrez différentes heures que des employés auraient pu passer sur diverses tâches en remplissant le formulaire, puis envoyez les données en tant que version. Les données étant stockées dans la table de base de données Content Server réservée à l'employé, le service en charge des salaires peut y accéder en utilisant un programme externe qui calcule la paye de l'employé en extrayant les informations du formulaire de la table. Chaque employé peut afficher les versions de sa feuille afin de voir les précédentes feuilles de présence. Pour plus d'informations sur l'accès aux versions du formulaire, reportez-vous à la section Utilisation des formulaires.
