Pour ajouter un Form :

  1. Cliquez sur Form dans le menu Ajouter un élément.

  2. Sur la page Ajouter un formulaire, cliquez sur Parcourir Content Server puis recherchez le modèle de formulaire sur lequel vous voulez baser le formulaire.

  3. Dans la liste Mécanisme de révision, cliquez sur un mécanisme de révision.

  4. Dans la liste Mécanisme d'envoi, cliquez sur un mécanisme d'envoi.

  5. Cliquez sur Continuer.

  6. Saisissez un nouveau nom dans la zone Nom.

  7. Facultatif Saisissez une description du formulaire dans la zone Description. Cette description s'affiche sur la page Infos générales du formulaire.

  8. Facultatif S'il en existe, sélectionnez une vue personnalisée dans la liste Vue personnalisée.

  9. Si vous avez sélectionné Lancer le workflow comme mécanisme d'envoi, dans le champ Schéma de workflow, cliquez sur Parcourir Content Server, puis recherchez et sélectionnez un schéma de workflow.

  10. Sur la page Ajouter un formulaire, cliquez sur Ajouter.

 

Astuce :
Cliquez sur l'icône Fonctions d'un modèle de formulaire, puis sur Créer un formulaire.

 

Pour plus d'informations sur cette procédure, consultez la section Définition d'un formulaire.
