Pour gérer la table de base de données :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire et sélectionnez Gérer la table relationnelle.

  2. Sur la page Gérer la table relationnelle, modifiez les paramètres comme indiqué dans le tableau suivant.

    Pour... Effectuez les opérations suivantes…
    Créer une table de base de données à associer avec le modèle de formulaire Saisissez le nom de la table de base de données dans la zone Nom de la table SQL, cliquez sur Créer une table de base de données avec le nom indiqué ci-dessus dans la liste Créer une table SQL, puis cliquez sur Envoyer.
    Associer une table de base de données existante au modèle de formulaire Saisissez le nom de la table de base de données dans la zone Nom de la table SQL, cliquez sur Ne pas créer une table de base de données, utiliser la table existante dont le nom est indiqué ci-dessus dans la liste Créer une table SQL, puis cliquez sur Envoyer.
    Vérifier si un schéma de table SQL correspond à un modèle de formulaire Saisissez le nom de la base de données dans la zone Nom de la table SQL, cliquez sur Vérifier le schéma de correspondance entre le modèle et la table nommée nom_de_la_table dans la liste Créer une table SQL, puis cliquez sur Envoyer.
    Afficher les instructions SQL nécessaires à la création d'une table pour le modèle de formulaire Saisissez le nom de la table de base de données dans la zone Nom de la table SQL, cliquez sur Emettre des commandes SQL afin de créer la table requise pour ce modèle dans la liste Créer une table SQL, puis cliquez sur Envoyer.
    Supprimer le lien entre une table de base de données et un modèle de formulaire Saisissez le nom de la base de données dans la zone Nom de la table SQL, cliquez sur Supprimer le lien entre ce modèle et la table nom_de_la_table dans la liste Créer une table SQL, puis cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Définition d'un modèle de formulaire.
