Si votre formulaire contient un champ qui a une valeur identique pour plusieurs utilisateurs, vous pouvez verrouiller cette valeur dans le champ en enregistrant le formulaire en tant que modèle prérenseigné. Les utilisateurs peuvent accéder à un modèle prérenseigné comme à n'importe quel formulaire, mais ils ne peuvent compléter que les champs qui ne sont pas verrouillés. Lorsque les utilisateurs ont rempli le formulaire prérenseigné, Content Server les invite à enregistrer une copie du formulaire à un emplacement différent.

 

Note :
Vous ne pouvez pas utiliser un modèle prérenseigné si le mécanisme de révision est défini sur <Aucun>.

 

Lorsque vous convertissez un formulaire en modèle prérenseigné, Content Server verrouille les données entrées dans les champs du formulaire. Vous pouvez modifier et supprimer les données d'un modèle prérenseigné. Vous pouvez également convertir un modèle prérenseigné en formulaire.

L'autorisation Gérer est nécessaire à la création et à la maintenance des modèles prérenseignés. Vous devez disposer de l'autorisation Modifier les données préinsérées pour ce formulaire. Pour de plus amples informations sur les autorisations, reportez-vous à la section Description des autorisations pour les modèles de formulaire et les formulaires.
