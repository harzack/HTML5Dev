Pour supprimer un ensemble de données de formulaire :

  1. Cliquez sur l'icône Fonctions du formulaire qui contient l'ensemble de données à supprimer, puis sélectionnez Répertorier les données.

  2. Sur la page Répertorier les données du formulaire, cliquez sur le lien Supprimer de l'ensemble de données que vous souhaitez supprimer.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des données de formulaire.
