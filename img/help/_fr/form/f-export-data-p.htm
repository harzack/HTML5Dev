Pour exporter des données de formulaire :

  1. Cliquez sur l'icône Fonctions du formulaire qui contient les données envoyées à exporter, puis sélectionnez Exporter les données.

  2. Dans la boîte de dialogue Téléchargement de fichier, cliquez sur Enregistrer, accédez à l'emplacement auquel vous souhaitez enregistrer les données de formulaire exportées, puis cliquez sur le bouton OK.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des données de formulaire.
