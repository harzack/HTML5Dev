En définissant leurs autorisations, vous pouvez contrôler le comportement des modèles de formulaire et des formulaires. Les autorisations pour les modèles de formulaire sont similaires aux autorisations pour les autres éléments Content Server. En revanche, les autorisations pour les formulaires sont réparties dans les catégories suivantes :

  • Voir, qui vous autorise à afficher le formulaire.

  • Utiliser le mécanisme de révision Versions Content Server, qui vous permet de stocker les données intermédiaires du formulaire en activant le mécanisme de révision du formulaire. Cette autorisation ne peut être utilisée que pour des formulaires ayant un mécanisme de révision. Elle nécessite également l'autorisation Voir.

  • Modifier les données préinsérées, qui vous permet de modifier les données du formulaire qui étaient verrouillées comme paramètres par défaut. Cette autorisation ne peut être utilisée que pour des formulaires prérenseignés. Elle nécessite également l'autorisation Voir.

  • Utiliser le mécanisme d'envoi, qui vous permet d'envoyer les données finales du formulaire en activant le mécanisme d'envoi du formulaire. Cette autorisation ne peut être utilisée que pour des formulaires ayant un mécanisme d'envoi. Elle nécessite également l'autorisation Voir.

  • Gérer le formulaire, qui vous permet d'effectuer toutes les autres fonctions sur ce formulaire et sur les données stockées. Cette autorisation nécessite également toutes les autres autorisations sur les formulaires.

    Les mécanismes de révision et d'envoi sont définis lors de l'ajout d'un formulaire dans Content Server.
