Pour ajouter une version de modèle de formulaire :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire auquel vous souhaitez ajouter une version, puis sélectionnez Modifier.

  2. Ajoutez ou supprimez des attributs dans le modèle de formulaire.

  3. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des modèles de formulaire.
