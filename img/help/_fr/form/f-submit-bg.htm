Pour traiter un formulaire rempli utilisant un mécanisme d'envoi, vous devez l'envoyer. En fonction des mécanismes configurés pour le formulaire, la fonction Envoyer traite ou stocke les données du formulaire.

Pour envoyer un formulaire, vous devez disposer de l'autorisation Utiliser le mécanisme d'envoi pour ce formulaire. Pour de plus amples informations sur les autorisations, reportez-vous à la section Description des autorisations pour les modèles de formulaire et les formulaires.

Les options suivantes d'envoi du formulaire peuvent être disponibles en fonction des mécanismes de ce formulaire et du modèle de formulaire sur lequel il est basé.

  • Si le formulaire n'a pas de mécanisme de révision, le bouton sur le formulaire (tel que Enregistrer ou Appliquer) envoie les données sur le serveur HTTP.

  • Si le formulaire a un mécanisme de révision et d'envoi, la fonction Envoyer.

  • Un lien Envoyer optionnel sur la page Content Server qui répertorie le formulaire.

  • Un bouton optionnel sur le formulaire qui active le mécanisme d'envoi (en plus de celui qui active le mécanisme de révision).
