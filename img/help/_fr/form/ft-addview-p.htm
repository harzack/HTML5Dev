Pour ajouter une vue personnalisée à un modèle de formulaire :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire auquel vous voulez ajouter la vue personnalisée, choisissez Propriétés, puis Vues.

  2. Dans l'onglet Vues, cliquez sur le type de vue que vous souhaitez ajouter à la liste Ajouter une vue.

  3. Facultatif Saisissez une description de la vue dans la zone Description.

  4. Cliquez sur Parcourir, puis recherchez et sélectionnez le fichier que vous souhaitez ajouter en tant que vue.

  5. Cliquez sur Ajouter une vue.

Pour plus d'informations sur cette procédure, consultez la section Définition des vues personnalisées.
