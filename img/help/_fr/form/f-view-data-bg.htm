Vous pouvez afficher les données actuelles du formulaire en l'ouvrant. Si vous essayez d'ouvrir un formulaire que vous n'êtes pas autorisé à modifier et pour lequel aucune vue <Lecture seule> n'est définie, vous ne pourrez qu'afficher le formulaire. Lorsque vous avez fini de visualiser un formulaire, utilisez le bouton Retour (ou équivalent) de votre navigateur pour quitter le formulaire. Si vous cliquez sur le bouton Enregistrer ou Appliquer, une erreur risque de se produire.

Affichage des formulaires sans vues personnalisées

Si vous essayez d'ouvrir un formulaire pour lequel aucune vue personnalisée n'est disponible, Content Server affiche la vue par défaut du formulaire.

Utilisation des données de formulaire stockées

Si un formulaire utilise un mécanisme d'envoi qui stocke les données, tel que Versions ou Table SQL, vous pouvez effectuer les opérations suivantes sur les données du formulaire :

  • Répertorier des données

  • Exporter des données

  • Modifier un ensemble de données

  • Supprimer un ensemble de données

  • Supprimer toutes les données stockées

Répertorier les données de formulaire

La fonction Répertorier les données exécute un rapport LiveReport qui fournit une liste sous forme de tableau des données envoyées par les utilisateurs. Le nom de chaque champ correspond au titre d'une colonne du rapport. Pour obtenir la liste des données envoyées d'un formulaire, vous devez disposer de l'autorisation Gérer pour ce formulaire.

Exportation des données de formulaire

Content Server exporte les données du formulaire dans un fichier au format .csv (valeurs séparées par des virgules). Si le formulaire comporte des sous-tables, le fichier d'exportation contient les données de la table principale dans la première section, suivies de lignes vierges. Les sections supplémentaires suivantes correspondent à chaque sous-table du formulaire. Chaque section contient les données de chaque sous-table et est séparée de la section suivante par des lignes vierges.

Modification d'un ensemble de données de formulaire

Vous pouvez modifier un ensemble de données de la collection des données de formulaire.

Suppression des données de formulaire

Vous pouvez supprimer un ensemble de données de la collection de données de formulaire ou bien vous pouvez supprimer toutes les données de formulaire stockées. Lorsque vous supprimez toutes les données stockées, Content Server supprime toutes les données envoyées via le formulaire. Cette fonction nécessite que vous ayez l'autorisation Gérer sur le formulaire dont vous souhaitez supprimer les données.

 

Note :
Vous ne pouvez pas supprimer les données d'un formulaire stockées dans Content Server si vous supprimez le formulaire en premier.
