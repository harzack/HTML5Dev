Pour ajouter un groupe :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, choisissez Groupe dans le menu Ajouter un élément.

  3. Saisissez un nom unique pour le nouveau groupe dans le champ Nom de groupe.

  4. Cliquez sur Envoyer.
