Pour rechercher un utilisateur ou un groupe :

  1. Choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, dans la liste Rechercher, sélectionnez la méthode de recherche souhaitée. Les options possibles sont : Nom de l'utilisateur, Prénom de l'utilisateur, Identification de l'utilisateur, E-mail de l'utilisateur ou Nom du groupe.

  3. Dans la boîte de dialogue qui commence par, saisissez les premières lettres du nom de l'utilisateur ou du groupe.

  4. Cliquez sur Rechercher.

 

Astuce :
Sur la page Utilisateurs et groupes, si vous souhaitez générer une liste de tous les utilisateurs dans Content Server, laissez le champ qui commence par vide et cliquez sur le bouton Rechercher. Pour les grandes organisations, une telle recherche peut prendre beaucoup de temps et solliciter énormément les ressources du système.
