Pour enlever un utilisateur ou un sous-groupe d'un groupe :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez le groupe duquel vous souhaitez supprimer un utilisateur ou un sous-groupe. Pour plus d'informations sur la recherche de groupes, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Modifier du groupe dans la colonne Actions.

  4. Sur la page Modifier le groupe, cliquez sur le lien du nom de l'utilisateur ou du groupe que vous souhaitez enlever.

  5. Cliquez sur Enlever du groupe.

  6. Cliquez sur Terminé.
