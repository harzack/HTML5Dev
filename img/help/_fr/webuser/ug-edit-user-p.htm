Pour modifier les informations sur les utilisateurs :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez l'utilisateur dont vous souhaitez modifier les informations. Pour plus d'informations sur la recherche d'utilisateurs, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Modifier de l'utilisateur dans la colonne Actions.

  4. Sur la page Infos générales, ajoutez ou modifiez les informations générales sur l'utilisateur, y compris les paramètres du mot de passe et les privilèges système.

  5. Cliquez sur l'onglet Personnel, puis ajoutez ou modifiez les informations sur l'utilisateur.

  6. Cliquez sur Mettre à jour.

 

Astuce :
Pour afficher et modifier vos propres informations utilisateur, dans la barre du menu générale, choisissez Mon profil dans le menu Personnel.
