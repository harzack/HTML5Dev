Pour supprimer un utilisateur :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez l'utilisateur que vous souhaitez supprimer. Pour plus d'informations sur la recherche d'utilisateurs, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Parcourir de l'utilisateur dans la colonne Actions.

  4. Eliminez le contenu de l'espace de travail Personnel de l'utilisateur en supprimant les éléments indésirables et en transférant les éléments importants vers d'autres emplacements. Il est important d'effectuer cette opération avant de supprimer l'utilisateur, car il peut être difficile d'accéder à l'espace de travail Personnel d'un utilisateur supprimé.

  5. Une fois que vous avez éliminé le contenu de l'espace de travail Personnel de l'utilisateur, revenez sur la page Utilisateurs et groupes et cliquez sur le lien Modifier de l'utilisateur, dans la colonne Actions.

     

    Astuce :
    Si vous souhaitez récupérer l'ID de l'utilisateur que vous allez supprimer, lorsque vous êtes sur la page Modifier l'utilisateur Content Server de l'utilisateur, et avant de supprimer cet utilisateur, copiez l'ID d'utilisateur à partir de la barre d'adresse de votre navigateur Web. L'ID d'utilisateur se trouve dans l'URL sous la forme suivante : ID_utilisateur =<nombre>.

     

  6. Sur la page Content Server Modifier l'utilisateur, cliquez sur Supprimer l'utilisateur.

  7. Dans la boîte de dialogue, cliquez sur OK.
