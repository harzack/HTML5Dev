Pour créer un utilisateur :

  1. Cliquez sur le menu général Entreprise, puis cliquez sur Utilisateurs et groupes.

  2. Sur la page Utilisateurs et groupes, cliquez sur Ajouter un élément, puis sur Utilisateur.

  3. Sur la page Content Server Créer un utilisateur, saisissez un nom d'utilisateur unique dans le champ Nom de connexion.

  4. Procédez de l'une des manières suivantes :

    • Si une liste s'affiche en regard du champ Service, cliquez sur le nom du service auquel vous souhaitez ajouter l'utilisateur.

    • Dans le cas contraire, cliquez sur le bouton Sélectionner, cliquez sur le bouton Rechercher sur la page Sélectionner le service pour rechercher le groupe que vous souhaitez désigner comme le service de l'utilisateur, puis cliquez sur le lien Sélectionner du groupe.

  5. Dans le champ Mot de passe, saisissez un mot de passe que le nouvel utilisateur doit utiliser pour se connecter à Content Server. Saisissez une nouvelle fois le mot de passe dans le champ Confirmation.

  6. Facultatif Pour définir la date d'expiration d'un mot de passe, sélectionnez une date dans la section Faire expirer le mot de passe.

     

    Astuce :
    La section Faire expirer le mot de passe apparaît uniquement si l'expiration de mot de passe est activée sur votre système.

     

  7. Facultatif Pour désactiver l'expiration de mot de passe pour cet utilisateur, activez Le mot de passe n'expire jamais.

     

    Astuce :
    La section Faire expirer le mot de passe apparaît uniquement si l'expiration de mot de passe est activée sur votre système, et si vous disposez du privilège Droits d'administration utilisateur.

     

  8. Facultatif Pour indiquer des informations supplémentaires sur l'utilisateur, saisissez des informations dans les champs Prénom, Initiale du second prénom, Nom, Titre, E-mail, Téléphone, Télécopie et Emplacement de bureau.

  9. Facultatif Pour définir la langue affichée pour l'utilisateur, cliquez sur une langue dans la liste Langue préférée.

     

    Astuce :
    La liste Langue préférée apparaît uniquement si plusieurs langues sont disponibles sur votre système.

     

  10. Facultatif Pour définir un fuseau horaire pour l'utilisateur, cliquez sur un fuseau horaire dans la liste Fuseau horaire.

  11. Facultatif Pour accorder des privilèges à l'utilisateur, sélectionnez les cases à cocher adéquates dans la section Privilèges.

  12. Cliquez sur Envoyer.
