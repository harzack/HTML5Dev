Pour afficher des informations sur l'utilisateur :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez l'utilisateur dont vous souhaitez afficher les informations. Pour plus d'informations sur la recherche d'utilisateurs, voir Pour rechercher un utilisateur ou un groupe.

  3. Une fois que vous avez trouvé l'utilisateur dont vous souhaitez consulter les informations, cliquez sur le nom de l'utilisateur dans la colonne Nom.

  4. En fonction des autorisations de création et de modifications d'utilisateurs et de groupes dont vous disposez, des liens peuvent également apparaître dans la colonne Actions :

    • Modifier, qui affiche la page Infos générales d'un utilisateur en mode édition.

    • Parcourir, qui affiche la page Mon espace de travail d'un utilisateur.

    • Groupes, qui affiche les informations sur les groupes auxquels l'utilisateur appartient.
