Pour modifier les informations sur les groupes :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez le groupe dont vous souhaitez modifier les informations. Pour plus d'informations sur la recherche de groupes, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Modifier du groupe dans la colonne Actions.

  4. Sur la page Modifier le groupe, ajoutez ou modifiez les informations sur le groupe. Vous pouvez également cliquer sur le nom d'un utilisateur ou d'un groupe dans la section Groupe actuel, puis cliquer sur Nommer comme chef de groupe pour définir cet utilisateur ou ce groupe en tant que chef de groupe.

  5. Cliquez sur Terminé.

 

Astuce :
Pour supprimer le chef de groupe sans en nommer un nouveau, cliquez sur le nom du chef de groupe dans la section Membres du groupe actuel, puis sur le bouton Révoquer comme chef de groupe.
