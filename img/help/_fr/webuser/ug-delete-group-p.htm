Pour supprimer un groupe :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez le groupe que vous souhaitez supprimer. Pour plus d'informations sur la recherche de groupes, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Modifier du groupe dans la colonne Actions.

  4. Cliquez sur Supprimer le groupe.

  5. Cliquez sur Terminé.

 

Note :
Vous ne pouvez pas supprimer un groupe utilisé en tant que service.
