Pour afficher des informations sur les groupes :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, recherchez le groupe dont vous souhaitez afficher les informations. Pour plus d'informations sur la recherche de groupes, voir Pour rechercher un utilisateur ou un groupe.

  3. Cliquez sur le lien Afficher du groupe dans la colonne Actions.

 

Astuce :
Pour afficher vos propres informations de groupe, dans la barre du menu générale, choisissez Mes groupes dans le menu Personnel.
