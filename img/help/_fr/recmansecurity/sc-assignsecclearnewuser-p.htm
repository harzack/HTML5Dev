Pour assigner un niveau d'habilitation de sécurité à un nouvel utilisateur :

  1. Choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Dans la page Utilisateurs et groupes, cliquez sur Utilisateur dans le menu Ajouter : Elément.

  3. Dans le champ Nom de connexion de la page d'ajout de nouvel utilisateur, saisissez un nom.

  4. Dans la liste déroulante Service, cliquez sur un service.

  5. Dans la liste déroulante Niveau d'habilitation de sécurité, cliquez sur un niveau.

  6. Spécifiez les autres paramètres.

  7. Cliquez sur le bouton Envoyer.

 

Note :
Pour plus d'informations sur l'ajout d'un utilisateur dans Content Server, reportez-vous à Utilisation des utilisateurs et des groupes.
