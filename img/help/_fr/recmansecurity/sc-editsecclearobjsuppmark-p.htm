Pour modifier l'habilitation de sécurité sur un objet avec les paramètres Marquages supplémentaires DoD activés :

  1. Cliquez sur l'icône Fonctions d'un élément, cliquez sur Propriétés, puis sélectionnez Habilitation de sécurité.

  2. A la section Marquages supplémentaires de l'onglet Habilitation de sécurité, sélectionnez les marquages supplémentaires que vous souhaitez assigner dans le champ Disponibles, puis cliquez sur la flèche droite pour les placer dans le champ Sélectionnés.

  3. Cliquez sur Mettre à jour.
