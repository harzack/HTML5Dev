Pour assigner des marquages supplémentaires à des objets :

  1. En mode navigation, cochez la case des éléments auxquels vous souhaitez assigner un marquage supplémentaire, puis cliquez sur le bouton Assigner des marquages supplémentaires.

  2. A la page Assigner des marquages supplémentaires, sélectionnez les marquages supplémentaires à assigner dans le champ Disponibles, cliquez sur la flèche droite pour les placer dans le champ Assigner, puis cliquez sur le bouton Assigner.

  3. Cliquez sur OK.

 

Astuce :
Vous pouvez aussi assigner des marquages supplémentaires pour un ou plusieurs éléments dans une collection ou un résultat de recherche en cochant les cases correspondantes, puis en cliquant sur Assigner des marquages supplémentaires dans la liste déroulante Actions.

 
 

Note :
L'administrateur peut activer ou désactiver le bouton Assigner des marquages supplémentaires. S'il est désactivé, le bouton ne s'affiche pas en mode navigation et vous devez assigner les marquages supplémentaires individuellement à l'aide de l'icône Fonctions de l'élément.
