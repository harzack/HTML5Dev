Pour supprimer des marquages supplémentaires d'objets :

  1. En mode navigation, cochez la case des éléments pour lesquels vous souhaitez supprimer des marquages supplémentaires, puis cliquez sur le bouton Supprimer des marquages supplémentaires.

  2. A la page Supprimer des marquages supplémentaires, sélectionnez les marquages supplémentaires à supprimer dans le champ Disponibles, cliquez sur la flèche gauche pour les placer dans le champ Supprimer, puis cliquez sur le bouton Supprimer.

  3. Cliquez sur OK.

 

Astuce :
Vous pouvez aussi supprimer des marquages supplémentaires pour un ou plusieurs éléments d'une collection ou d'un résultat de recherche en cochant les cases correspondantes, puis en cliquant sur Supprimer des marquages supplémentaires dans la liste déroulante Actions.

 
 

Note :
L'administrateur peut activer ou désactiver le bouton Supprimer des marquages supplémentaires. S'il est désactivé, le bouton ne s'affiche pas en mode navigation et vous devez supprimer les marquages supplémentaires individuellement à l'aide de l'icône Fonctions de l'élément.
