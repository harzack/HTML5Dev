Pour créer un guide d'habilitation :

  1. Saisissez l'URL suivante dans votre navigateur Web : 

    http://server/content_server_instance/livelink.exe?func=RecmanSecurity.SecurityGuide
    où

    • serveur est le nom de votre serveur ;

    • content_server_instance est le nom de votre instance Content Server.

    Par exemple, 

    http://localhost/ContentServer2010/livelink.exe?func=RecmanSecurity.SecurityGuide
    Si un message vous y invite, entrez le mot de passe approprié dans le champ Mot de passe de l'administrateur, puis cliquez sur le bouton Connexion.

    Sur la page de définitions de guide d'habilitation de sécurité, vous pouvez également saisir un titre, une date et une organisation d'origine du guide dans les champs appropriés.

  2. Définissez des marquages supplémentaires et spécifiez des métadonnées supplémentaires selon les besoins.

  3. Cliquez sur le bouton Enregistrer.
