Pour définir le bridage d'un niveau d'habilitation de sécurité sur un objet :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Habilitation de sécurité.

  2. Dans l'onglet Habilitation de sécurité, cochez l'une des cases suivantes :

    • Date permet de définir la date de bridage du document.

    • Evénement permet de spécifier l'événement qui doit avoir lieu avant le bridage du document.

  3. Facultatif Saisissez des instructions relatives à la procédure de bridage à exécuter dans le champ Instructions de bridage.

  4. Cliquez sur le bouton Mettre à jour.

 

Note :
Si le document est restreint, les informations de la section Bridé le sont ignorées.
