Pour mettre à jour le niveau de sécurité d'un élément à l'aide d'une règle définie :

  1. Cliquez sur l'icône Fonctions d'un élément, cliquez sur Propriétés, puis sélectionnez Habilitation de sécurité.

  2. A l'onglet Habilitation de sécurité, cliquez sur le lien Règles définies.

  3. Dans l'onglet Règles définies, cliquez sur Mettre à jour le niveau d'habilitation de sécurité dans la liste Action à exécuter.

  4. Cliquez sur le niveau d'habilitation de sécurité souhaité pour l'élément dans la liste Niveau d'habilitation de sécurité.

     

    Note :
    Si vous cliquez sur Aucun, le niveau d'habilitation de sécurité actuel est supprimé de l'élément.

     

  5. Sélectionnez le mois, le jour, l'année et l'heure à laquelle le niveau d'habilitation de sécurité doit être mis à jour pour l'élément dans la liste Date d'exécution.

  6. Facultatif Indiquez un motif d'exécution de l'action sur l'élément dans le champ Motif de la règle.

  7. Cliquez sur Mettre à jour.

  8. Si vous supprimez le niveau d'habilitation de sécurité d'un recueil avec des sous-éléments, cliquez sur l'un des boutons radio suivants dans la fenêtre Options de mise à jour, puis cliquez sur Fermer :

    • Appliquer les modifications à l'élément actuel uniquement

    • Appliquer les modifications à l'élément actuel et aux sous-éléments
