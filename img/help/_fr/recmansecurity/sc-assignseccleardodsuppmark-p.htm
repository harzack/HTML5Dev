Pour ajouter un nouvel élément et assigner une habilitation de sécurité lorsque le paramètre Marquages supplémentaires DoD est activé :

  1. Dans le menu Ajouter un nouvel élément, cliquez sur un objet.

  2. A la section Marquages supplémentaires de la page Ajouter, sélectionnez les marquages supplémentaires dans la liste Disponibles, puis cliquez sur la flèche droite pour les placer dans la liste Sélectionnés.

     

    Note :
    Seuls les marquages supplémentaires qui vous sont assignés s'affichent dans la section Marquages supplémentaires.

     

  3. Spécifiez les autres paramètres.

  4. Cliquez sur le bouton Ajouter.
