Pour déclasser le niveau d'habilitation de sécurité sur un objet :

  1. Cliquez sur l'icône Fonctions d'un élément, cliquez sur Propriétés, puis sélectionnez Habilitation de sécurité.

  2. A l'onglet Habilitation de sécurité, cliquez sur le niveau d'habilitation de sécurité minimum disponible dans la liste déroulante Niveau d'habilitation de sécurité actuel.

  3. Dans la section Informations de vérification et de notation, cliquez sur une date dans les listes déroulantes Déclassé le.

  4. Saisissez le nom de la personne ou de l'agence ayant effectué le déclassement dans le champ Déclassé par.

  5. Cliquez sur le bouton Mettre à jour.
