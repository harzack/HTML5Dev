Pour ajouter un nouvel élément et assigner une habilitation de sécurité lorsqu'aucun paramètre DoD n'est activé :

  1. Dans le menu Ajouter un nouvel élément, cliquez sur un objet.

  2. Dans la liste déroulante Niveau d'habilitation de sécurité de la page d'ajout, cliquez sur un niveau.

  3. Spécifiez les autres paramètres.

  4. Cliquez sur le bouton Ajouter.
