Pour modifier l'habilitation de sécurité sur un recueil avec des sous-éléments :

  1. Cliquez sur l'icône Fonctions d'un recueil, cliquez sur Propriétés, puis sélectionnez Habilitation de sécurité.

  2. Cliquez sur le sous-onglet Définir l'habilitation de sécurité, puis procédez comme suit :

    • A la section Marquages supplémentaires, sélectionnez les marquages supplémentaires à assigner dans le champ Disponibles, puis cliquez sur la flèche droite pour les placer dans le champ Assigner.

    • A la section Niveau d'habilitation de sécurité, cliquez sur un niveau de sécurité dans la liste déroulante Niveau d'habilitation de sécurité actuel.

  3. Sélectionnez l'une des options suivantes dans la liste déroulante Appliquer à :

    • Cet élément : applique les modifications à l'objet parent uniquement.

    • Cet élément et les sous-éléments : applique les modifications à l'objet parent et à tous les objets enfants.

      Si vous sélectionnez Cet élément et les sous-éléments, une section Options de sous-éléments s'affiche.

  4. A la section Options de sous-éléments, sélectionnez une action dans les listes déroulantes Marquages supplémentaires et Habilitation de sécurité.

  5. Cliquez sur le bouton Mettre à jour.

 

Note :
Les paramètres des étapes 3 et 4 apparaissent uniquement si le paramètre Hériter à partir de est activé pour le type d'objet pris en charge modifié. Les paramètres du type d'objet pris en charge sont définis par l'administrateur.

Si vous faites passer le niveau d'habilitation de sécurité d'un recueil parent à un niveau inférieur à celui assigné aux recueils ou éléments du recueil parent, ces éléments conservent leur niveau d'habilitation de sécurité supérieur.
