Pour accéder à la page Intervenants à un projet :

  • Ouvrez un projet, puis choisissez Intervenants dans le menu Projet.

 

Astuce :
Si la barre d'icônes du projet est activée, vous pouvez également accéder à la page Intervenants sur un projet en cliquant sur l'icône Intervenants, dans la barre d'icônes du projet. Pour plus d'informations sur l'activation de la barre d'icônes de projet, reportez-vous à la rubrique Pour définir les paramètres de navigation du projet.
