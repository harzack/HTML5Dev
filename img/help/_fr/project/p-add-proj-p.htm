Note :
Après avoir indiqué si vous ajoutez le projet à l'aide d'un modèle vierge ou d'un modèle existant, vous pouvez mettre fin à la procédure sans ajouter d'autres informations en cliquant sur Terminer.

 

Pour ajouter un projet :

  1. Dans le menu Ajouter un élément, cliquez sur Projet.

  2. Dans la section Infos générales de la page Ajouter : Projet, procédez de la façon suivante :

    • Saisissez un nom de projet dans le champ Nom.

    • Dans la section Créer à partir de, sélectionnez Projet vierge.

    • Indiquez les autres informations générales, puis cliquez sur Suivant.

  3. Dans la section Contenu, procédez de la façon suivante :

    • Cliquez sur un statut pour le projet dans la liste Statut.

    • Sélectionnez un mois, un jour et une année dans les listes Date de commencement et Date prévue.

    • Indiquez les autres informations facultatives dans la section Contenu, puis cliquez sur Suivant.

  4. Dans la section Intervenants, procédez de la façon suivante :

    • Sélectionnez la case d'option appropriée dans la section Intervenants.

      Vous pouvez également cliquer sur Ajouter des intervenants ou Modifier les intervenants pour ajouter des intervenants à un projet ou les modifier.

    • Dans la section Accès public, cliquez sur Activé ou Désactivé.

    • Dans la section Notification par e-mail, cliquez sur le bouton radio approprié pour indiquer les types d'intervenants du projet qui recevront des notifications par e-mail.

       

      Note :
      Si les notifications ne sont pas activées ou si aucune adresse e-mail n'est indiquée, vous ne pouvez pas configurer de notifications par e-mail.

       

    • Cliquez sur Suivant.

  5. Dans la section Présentation, procédez de la façon suivante :

    • Sélectionnez une case d'option dans la section N° de sous-éléments mis en valeur.

    • Indiquez les autres informations facultatives dans la section Présentation, puis cliquez sur Terminer.

 

Astuce :
Vous pouvez également ajouter un projet en choisissant Projets dans le menu Personnel, puis en cliquant sur Créer un projet dans la page Mes projets.
