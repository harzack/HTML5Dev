Pour définir les options d'apparence du projet :

  1. Cliquez sur l'icône Fonctions du projet, choisissez Propriétés, puis Présentation.

  2. Dans l'onglet Présentation de la page Propriétés du projet, cliquez sur Options.

  3. Dans la section Options, cliquez sur le bouton radio correspondant au nombre maximum de sous éléments mis en valeur que vous souhaitez afficher dans la section N° de sous-éléments mis en valeur.

  4. Cliquez sur Envoyer.

Pour... Effectuez les opérations suivantes...
Ajouter une bannière d'images Dans la section 1ère bannière ou 2ème bannière, cliquez sur Parcourir Content Server, recherchez une image et cliquez sur Sélectionner.
Ajouter du texte de remplacement à une bannière d'images Saisissez le texte dans le champ Texte ALT.
Associer une URL à la bannière d'images Saisissez l'URL dans le champ URL choisie.
Indiquer une chaîne de nouvelles Dans la liste Bandeau de nouvelles, cliquez sur la chaîne dans laquelle se trouve la nouvelle que vous souhaitez lire.
