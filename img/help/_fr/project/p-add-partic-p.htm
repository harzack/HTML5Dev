Pour ajouter un intervenant ou un groupe à un projet :

  1. Ouvrez un projet, puis sélectionnez Intervenants dans le menu Projet.

  2. Sur la page Intervenants sur un projet, cliquez sur Ajouter des intervenants, .

  3. Sur la page Ajouter des intervenants du projet, recherchez l'utilisateur ou le groupe que vous souhaitez ajouter, cliquez sur un rôle pour l'utilisateur ou le groupe dans la liste Rôle, puis cliquez sur Envoyer. Répétez l'opération pour chaque intervenant ou groupe à ajouter.

  4. Cliquez sur Terminé.
