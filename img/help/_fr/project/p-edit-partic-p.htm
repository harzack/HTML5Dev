Pour modifier les informations sur les intervenants :

  1. Ouvrez un projet, puis sélectionnez Intervenants dans le menu Projet.

  2. Sur la page Intervenants sur un projet, cliquez sur Modifier les intervenants, .

  3. Sur la page Modifier les intervenants sur un projet, modifiez les informations, puis cliquez sur Envoyer.

Pour... Effectuez les opérations suivantes...
activer ou désactiver l'accès public. Cliquez sur Activer ou Désactiver dans la colonne Actions.
Modifier le rôle d'un intervenant Cliquez sur un nouveau rôle dans la liste Rôle.
Supprimer les intervenants d'un projet Cochez la case de chaque intervenant que vous souhaitez supprimer.
