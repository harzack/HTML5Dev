Pour définir les paramètres de navigation du projet :

  1. Cliquez sur l'icône Fonctions du projet, choisissez Propriétés, puis Présentation.

  2. Dans l'onglet Présentation de la page Propriétés du projet, cliquez sur Navigation.

  3. Dans la section Navigation de l'onglet Présentation, définissez les paramètres, puis cliquez sur Envoyer.

Pour... Effectuez les opérations suivantes...
Définir la page de démarrage par défaut Cliquez sur Vue d'ensemble ou Espace de travail dans la section Page de démarrage.
Sélectionner le style de navigation du projet Sélectionnez Menu Projet ou Barre d'icônes du projet ou les deux.
Sélectionner les éléments du bandeau d'icônes de projet Cochez la case de chaque élément que vous souhaitez voir apparaître dans le bandeau d'icônes de projet.
