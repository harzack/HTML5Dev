Pour filtrer les éléments dans la page Aperçu du projet :

  1. Ouvrez un projet, puis sélectionnez Aperçu dans le menu Projet.

  2. Dans la section Inclure les types d'élément de la page Aperçu du projet, sélectionnez la case à cocher de chaque élément que vous souhaitez afficher dans l'aperçu, puis cliquez sur Appliquer.

  3. Facultatif Répétez la procédure pour ajouter d'autres éléments à l'aperçu.
