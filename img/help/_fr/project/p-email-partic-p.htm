Pour envoyer un e-mail aux intervenants :

  1. Sur la page Intervenants sur un projet, cliquez sur Envoyer un e-mail aux intervenants, .

  2. Facultatif Vous pouvez modifier ou supprimer les adresses e-mail dans la fenêtre Modifier les adresses des intervenants.

  3. Cliquez sur Courrier.

  4. Dans votre programme de messagerie électronique, saisissez le message, puis envoyez-le.

  5. Dans la fenêtre Modifier les adresses des intervenants, cliquez sur Annuler.
