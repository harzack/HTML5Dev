Pour personnaliser la page Vue d'ensemble du projet :

  1. Cliquez sur l'icône Fonctions du projet, choisissez Propriétés, puis Présentation.

  2. Dans l'onglet Présentation de la page Propriétés du projet, cliquez sur Vue d'ensemble.

  3. Dans la section Vue d'ensemble de l'onglet Présentation, sélectionnez la case à cocher de chaque élément que vous souhaitez afficher sur la page Vue d'ensemble du projet.

  4. Cliquez sur le nombre maximum d'éléments pouvant s'afficher sur la page Vue d'ensemble dans la liste Nb d'éléments.

  5. Cliquez sur Envoyer.

 

Astuce :
Vous pouvez également accéder à la page Vue d'ensemble en cliquant sur Vue d'ensemble dans le menu Projet, puis en cliquant sur l'icône Modifier, , de la page Vue d'ensemble du projet.
