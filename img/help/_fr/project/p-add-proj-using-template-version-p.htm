Pour ajouter un projet à l'aide d'une version de modèle :

  1. Cliquez sur l'icône Fonctions du modèle de projet, choisissez Propriétés, puis Versions.

  2. Dans l'onglet Versions de la page Propriétés Modèle de projet, cliquez sur le numéro de Version du modèle de projet que vous souhaitez utiliser.

  3. Dans la section Infos générales de la page Ajouter : Projet, saisissez un nouveau nom pour le projet dans le champ Nom.

  4. Facultatif Vous pouvez également cliquer sur Suivant pour ajouter des informations ou les modifier dans les autres sections de la page Ajouter : Projet.

  5. Cliquez sur Terminer.
