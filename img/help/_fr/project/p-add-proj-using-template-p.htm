Pour ajouter un projet à l'aide d'un modèle :

  1. Cliquez sur l'icône Fonctions du modèle de projet et choisissez Créer à partir de.

  2. Dans la section Infos générales de la page Ajouter : Projet, saisissez un nouveau nom pour le projet dans le champ Nom.

  3. Facultatif Vous pouvez également cliquer sur Suivant pour ajouter des informations ou les modifier dans les autres sections de la page Ajouter : Projet.

  4. Cliquez sur Terminer.

 

Astuce :
Vous pouvez également ajouter un projet en utilisant un modèle : cliquez sur Projet dans le menu Ajouter un élément, cliquez sur Modèle dans la section Créer à partir de, puis cliquez sur Parcourir Content Server pour accéder à un modèle de projet.
