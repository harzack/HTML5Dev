Les coordinateurs de projets et les membres peuvent personnaliser l'apparence d'un espace de travail Projet. Cet espace est divisé en trois sections : Options, Navigation et Vue d'ensemble.

 

Note :
Vous ne pouvez pas personnaliser l'apparence de l'espace de travail Projet d'un utilisateur ; les modifications apportées à ces sections s'appliquent à tous les intervenants du projet.

 

Définition des options d'apparence du projet

La section Options permet d'indiquer le nombre maximum de sous-éléments qui s'affichent sous les éléments mis en valeur dans un projet. Vous pouvez également ajouter des bannières et indiquer une chaîne de nouvelles pour le projet.

Une bannière est une image qui s'affiche dans l'espace de travail Projet au-dessus des autres éléments du projet. Vous pouvez ajouter jusqu'à deux bannières et indiquer un texte de remplacement et une URL associée pour chaque bannière.

Lorsque vous activez le bandeau de nouvelles dans un projet, l'en-tête de chaque nouvelle que vous avez sélectionnée défile en haut de l'espace de travail Projet, au-dessus de la bannière et des autres éléments. Pour pouvoir faire défiler les nouvelles, vous devez sélectionner une chaîne. De plus, la chaîne dans laquelle se trouve la nouvelle doit appartenir à l'espace de travail Projet. Pour en savoir plus, voir Utilisation des chaînes et des nouvelles.

Définition des paramètres de navigation du projet

La section Navigation permet de sélectionner une page de démarrage par défaut, qui correspond à la première page qui s'affiche lorsque vous accédez au projet. Vous pouvez choisir d'afficher la page Vue d'ensemble du projet ou l'espace de travail Projet par défaut.

Vous pouvez également modifier le style de navigation actuel du projet qui est, par défaut, le menu Projet. Le style de navigation d'un projet peut inclure le bandeau d'icônes de projet, le menu Projet ou les deux. Lorsque vous sélectionnez le style de navigation avec le bandeau d'icônes de projet, vous pouvez choisir les éléments que vous souhaitez afficher comme raccourcis dans le bandeau d'icônes. Ces éléments sont identiques à ceux qui s'affichent dans le menu Projet. Cependant, votre administrateur Content Server contrôle l'affichage des éléments dans la section Navigation, c'est pour cette raison que certains menus ne sont pas toujours disponibles.

 

Note :
L'administrateur Content Server peut désactiver les styles de navigation. Lorsqu'un style de navigation est désactivé, il ne s'affiche pas dans la section Navigation de l'onglet Présentation de la page Propriétés.

 

Personnalisation de la page Vue d'ensemble du projet

La page Vue d'ensemble du projet contient des informations sur le projet réparties en sous-catégories, comme Tâches non assignées ou Eléments récemment consultés. La section Vue d'ensemble de l'onglet Présentation de la page Propriétés du projet permet d'indiquer les sections qui s'affichent sur la page Vue d'ensemble et de spécifier une limite pour le nombre de sections affichées.

Votre administrateur Content Server contrôle les éléments ajoutés à la page Vue d'ensemble. Par défaut, une page Vue d'ensemble peut contenir les sections suivantes :

  • Fiche projet

  • Mes tâches

  • Sous-projets

  • Nouvelles actuelles

  • Sondages récents

  • Etapes clés actuelles

  • Tâches en retard

  • Tâches non assignées

  • Eléments récemment consultés

  • Eléments les plus actifs

  • Documents intéressants

  • Les mieux notés

Pour plus d'informations sur la page Vue d'ensemble du projet, voir Navigation dans l'espace de travail Projet.
