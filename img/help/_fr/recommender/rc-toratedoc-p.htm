Pour évaluer un élément :

  1. Dans l'espace de travail Entreprise, cliquez sur l'icône Fonctions d'un élément, puis sélectionnez l'option Evaluer.

  2. Dans l'onglet Evaluations de la page Propriétés d'un élément dans la section Ma révision, cliquez sur une évaluation de l'élément dans la liste Evaluation.

     

    Astuce :
    Pour supprimer votre opinion sur l'élément, envoyez l'évaluation <Aucun>.

     

  3. Facultatif Vous pouvez ajouter un commentaire ou une explication en les saisissant dans le champ Commentaire ou Explication.

  4. Facultatif Pour supprimer votre opinion sur l'élément, envoyez l'évaluation <Aucun>.

  5. Cliquez sur Envoyer.

 

Astuces :

  • Vous pouvez également accéder à l'onglet Evaluations de la page Propriétés de l'élément en cliquant sur l'icône Fonctions de l'élément, en choisissant Propriétés, puis Evaluations.
