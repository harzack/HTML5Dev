Les recommandations sont des listes d'éléments ou d'utilisateurs qu'Content Server juge intéressants pour vous. Les recommandations figurent sur la page Recommandations et la page Vue d'ensemble du projet de chaque projet.

Il existe trois types de recommandations :

  • Usuel : elle est basée sur les entrées système de tous les utilisateurs.

  • Personnel : elle est basée sur votre activité dans le système.

  • Historique : il s'agit d'un enregistrement de votre activité récente.

Chaque type de recommandation consiste en un ou plusieurs composants, qui sont donc des sous-ensembles du type de recommandation.

Tableau 1-1 : Composants Assistant de recommandation

Type Composant Description
Usuel Nouveautés Eléments récemment ajoutés à Content Server.
  Eléments les plus actifs Eléments les plus actifs dans Content Server. L'appréciation se fait en fonction du nombre de consultations par élément.
  Les mieux notés Eléments Content Server les mieux classés. Cette appréciation est basée sur les trois critères suivants : l'évaluation globale faite par les utilisateurs, la date de création ou de dernière mise à jour des éléments et le nombre d'évaluations par élément.
Personnel Personnes ayant des centres d'intérêt similaires Utilisateurs ayant des habitudes de navigation similaires. L'appréciation se fait sur la base des éléments que vous et les autres utilisateurs avez consultés.
  Documents intéressants Eléments Content Server consultés par des personnes ayant des intérêts similaires.
Historique Eléments récemment consultés Eléments que vous avez récemment consultés.

Affichage des recommandations

Vous pouvez afficher les recommandations Usuel, Personnel et Historique sur votre page Recommandations.

Les recommandations peuvent aussi être consultées sur la page Vue d'ensemble du projet. Chaque projet comporte les composants suivants, dont la portée reste limitée au projet : Eléments récemment consultés, Eléments les plus actifs, Documents intéressants et Les mieux notés. Pour plus d'informations sur les vues d'ensemble des projets, reportez-vous à la rubrique Navigation dans l`espace de travail Projet.

Configuration des composants de recommandation

La configuration d'un composant de recommandation permet de déterminer le nombre d'éléments et de type d'éléments (le cas échéant) qu'il affiche. La liste des types d'éléments disponibles à la sélection dépend des types d'éléments que votre administrateur a rendus disponibles.
