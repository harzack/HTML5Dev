Pour générer un rapport sur une base de données externe, une connexion doit être établie dans la fonction Consultation de bases de données formulaires Web dans les pages d'administration de Content Server. Seul un administrateur de Content Server peut configurer ces connexions. Naviguez jusqu'à la section Administration des consultations de bases de données formulaires Web, puis sélectionnez Ajouter les informations de connexion de base de données. Remplissez le formulaire pour chaque base de données externe requise.

 

Important :
Le module facultatif formulaires Web doit être installé dans votre instance Content Server pour pouvoir établir une connexion à des bases de données externes.
