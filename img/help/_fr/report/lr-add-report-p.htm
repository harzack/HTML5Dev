Pour ajouter un rapport LiveReport :

  1. Dans le menu Ajouter un élément, sélectionnez LiveReport.

  2. Sur la page Ajouter : Rapport LiveReport, saisissez un nom pour le rapport dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description du rapport LiveReport.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Facultatif Si vous souhaitez ajouter un titre qui s'affiche sur la page Résultats LiveReport, saisissez un nom dans le champ Titre.

  5. Dans le champ Limite d'enregistrement, saisissez le nombre maximum de résultats autorisés.

  6. Facultatif Sélectionnez la case Afficher les invites si vous souhaitez que les utilisateurs saisissent de nouvelles valeurs de paramètre lorsqu'ils utilisent le rapport LiveReport en tant que sous-rapport.

  7. Facultatif Dans la section Entrées, cliquez sur un type d'entrée dans la liste Type, puis saisissez le texte que Content Server doit afficher pour l'invite dans le champ Invite correspondant.

  8. Saisissez une instruction SQL pour le rapport dans la zone de texte SQL.

  9. Facultatif Sélectionnez la case à cocher Autoriser les modifications de la base de données si vous souhaitez que l'instruction SQL puisse modifier le contenu de la base de données Content Server.

     

    Note :
    La case à cocher Autoriser les modifications de la base de données n'apparaît pas si votre administrateur a activé la sécurité des rapports LiveReport.

     

  10. Dans les champs Paramètre, sélectionnez les types de paramètres associés à l'instruction SQL dans les listes de paramètres. Pour plus d'informations sur les types de paramètres, voir Ajout et modification de rapports LiveReport.

  11. Dans la liste Format du rapport, sélectionnez le format du rapport LiveReport. Pour plus d'informations sur les formats, voir Formats des rapports LiveReport.

    Si vous souhaitez configurer l'apparence du format de rapport LiveReport sélectionné, cliquez sur Configurer. Apportez vos modifications dans la fenêtre Configurer le format du rapport, puis cliquez sur Sélectionner.

  12. Dans le champ Sélectionner le sous-rapport, pour associer un sous-rapport à ce rapport LiveReport, cliquez sur Parcourir Content Server... . Naviguez jusqu'au rapport LiveReport à afficher comme sous-rapport, puis cliquez sur le lien Sélectionner de ce rapport LiveReport.

  13. Pour assigner des paramètres à un sous-rapport, saisissez le nom d'un paramètre existant dans chaque champ Paramètre du sous-rapport n°.

  14. Dans la zone Afficher les colonnes :

    1. Dans la zone de texte Champ, si vous souhaitez préciser les informations issues de la base de données présentes dans la page Résultats du rapport LiveReport, saisissez le nom de la colonne de base de données.

    2. Dans la zone de texte Titre de la colonne, saisissez un nom d'affichage correspondant pour la colonne.

  15. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à ce rapport LiveReport.

  16. Facultatif Par défaut, Content Server enregistre votre nouveau rapport LiveReport dans l'Espace de travail Entreprise. Pour modifier l'emplacement par défaut, cliquez sur Parcourir Content Server... dans le champ Créer dans. Choisissez le lien Sélectionner en regard d'un nouvel emplacement.

  17. Cliquez sur Ajouter.
