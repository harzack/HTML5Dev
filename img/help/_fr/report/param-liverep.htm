Lorsque vous ajoutez un rapport LiveReport, vous devez indiquer les paramètres et spécifier jusqu'à cinq types d'entrées. Les paramètres déterminent les informations renvoyées par le rapport LiveReport et sont utilisés avec l'instruction SQL. L'administrateur indique les types de paramètres lors de l'ajout d'un rapport LiveReport. Le tableau suivant décrit les types de paramètres les plus courants.

Paramètre Renvoie...
Filtrer les types d'<éléments> Les enregistrements du type d'élément spécifié ou du sous-type de base de données.
Filtrer les autorisations Le rapport LiveReport renvoie uniquement les résultats des éléments que l'utilisateur est autorisé à voir.
Filtrer les <volumes> Uniquement les enregistrements du volume spécifié.
Type d'objet L'un des types d'éléments disponibles.
Code du statut Le statut d'un élément, tel que En attente ou Annulé pour une tâche.
<Heure> Les date et heure spécifiées relatives à l'ouverture du rapport LiveReport.
UserID Le numéro d'ID utilisateur Content Server.
Entrée utilisateur <n°> La valeur des paramètres d'entrée définis dans la section Entrées.

Exigences relatives aux types d'entrées – Pour les types d'entrées, l'utilisateur qui ouvre le rapport doit fournir une valeur, un type d'élément ou un autre type d'entrée à l'invite que vous spécifiez. Pour chaque entrée que vous ajoutez, vous devez définir un paramètre d'entrée et inclure une référence au paramètre dans votre instruction SQL. Le tableau suivant contient les types d'entrées disponibles pour les rapports LiveReport.

Type d'entrée Invite les utilisateurs à...
Recueil Rechercher et sélectionner un élément pouvant contenir d'autres éléments, tels qu'un dossier, une chaîne ou une discussion.
Date Fournir une date dans le champ correspondant.
ID de nœud Fournir l'ID de l'élément.
Numéro Fournir un nombre entier dans le champ correspondant.
Objet Sélectionner un type d'élément dans la liste fournie. Les types d'objets qui s'affichent dans cette liste sont mis à jour de manière dynamique en fonction des types d'objets disponibles.
Projet Sélectionner un projet.
Chaîne Saisir une chaîne de caractères, qui peut contenir des caractères spéciaux, tels que les tirets, dans le champ correspondant.
Utilisateur Sélectionnez un utilisateur.
