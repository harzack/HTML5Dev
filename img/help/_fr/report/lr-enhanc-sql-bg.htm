LiveReport Extensions apporte des améliorations aux fonctions d'édition SQL offertes avec le rapport LiveReport.

  1. La taille de la fenêtre d'édition a été augmentée et une barre de défilement vertical et horizontal a été ajoutée.

  2. La requête SQL peut comporter des espaces et des retours chariot pour améliorer la lisibilité et faciliter la maintenance. Cette mise en forme est conservée par Content Server lorsqu'un développeur revient modifier le rapport LiveReport.

  3. Les caractères de tabulation du champ d'édition SQL sont insérés dans le texte au lieu d'être interprétés (et le curseur sort du champ d'édition).

  4. Un bouton Annuler et un bouton Restaurer ont été ajoutés pour faciliter l'édition du texte.
