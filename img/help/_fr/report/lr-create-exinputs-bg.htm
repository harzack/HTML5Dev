Cliquez sur les symboles plus pour créer d'autres entrées utilisateur ou paramètres de substitution. Lorsque vous ajoutez de nouvelles entrées utilisateur à la liste des paramètres, sélectionnez Entrée utilisateur dans la liste déroulante Type, puis sélectionnez le nombre d'invites dans les options. Notez que le bouton Entrée utilisateur sélectionne automatiquement le paramètre Entrée utilisateur dans la liste déroulante Type. Le nombre d'invites identifie les entrées utilisateur définies dans la section Entrées qui seront remplacées.
