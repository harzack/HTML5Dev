Pour ajouter un sous-rapport :

  1. Cliquez sur l'icône Fonctions d'un rapport LiveReport, puis sélectionnez Modifier.

  2. Sur la page Modifier, dans le champ Sélectionner le sous-rapport, cliquez sur Parcourir.

  3. Recherchez un sous-rapport, puis cliquez sur son lien Sélectionner.

  4. Dans les champs Paramètre du sous-rapport <n°>, saisissez chaque paramètre que vous avez indiqué dans l'instruction SQL du sous-rapport.

  5. Cliquez sur Enregistrer et exécuter.
