Pour modifier un graphique circulaire :

  1. Pour faire pivoter tout le graphique, cliquez et faites glisser la souris ou appuyez sur les touches de direction de votre clavier.

  2. Pour séparer les différents secteurs du graphique circulaire, appuyez sur la touche Page précédente.

  3. Pour regrouper les secteurs du graphique circulaire, appuyez sur la touche Page suivante.

  4. Pour séparer un secteur du graphique, cliquez avec le bouton droit et faites glisser la souris.
