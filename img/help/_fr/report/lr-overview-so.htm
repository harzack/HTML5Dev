Les rapports LiveReport permettent d'accéder aux informations de la base de données de Content Server et vous permettent de rechercher, d'organiser et de présenter les informations qui y sont stockées. Les rapports LiveReport obtiennent des informations en envoyant des requêtes SQL (Structured Query Language) à la base de données. Les instructions SQL pouvant modifier ou supprimer les données critiques, l'autorisation Ajouter ou Modifier les rapports LiveReport est réservée à l'administrateur par défaut.
