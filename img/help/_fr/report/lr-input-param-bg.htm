Dans le module LiveReports, tous les paramètres transmis pour la substitution dans le code du rapport sont nommés à l'aide du préfixe : inputlabel. Ainsi, « Input1 » est associé à inputlabel1, « Input2 » est associé à inputlabel2, etc. A partir de LiveReport Extensions Version 3.0.2, il est possible de définir un autre paramètre URL pour chaque entrée. Dès qu'une entrée est ajoutée, le champ Paramètre URL est défini par défaut sur le paramètre inputlabel<x> approprié, mais ce champ permet de définir n'importe quelle chaîne de caractères plutôt que la chaîne par défaut.
