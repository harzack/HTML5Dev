Pour ouvrir un rapport LiveReport :

  1. Dans la barre de menu générale, sélectionnez Personnel, puis Rapports.

  2. Sur la page Mes rapports, sélectionnez l'onglet Rapports LiveReport.

  3. Cliquez sur le nom d'un rapport LiveReport pour ouvrir ce rapport.

     

    Astuce :
    Vous pouvez également ouvrir un rapport LiveReport en cliquant sur l'icône Fonctions d'un rapport, puis en sélectionnant Ouvrir.
