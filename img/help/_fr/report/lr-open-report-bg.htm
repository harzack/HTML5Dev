L'obtention d'informations à l'aide des rapports LiveReport est différente de l'utilisation de la fonction Recherche Content Server. Les rapports LiveReport interrogent la base de données Content Server, au lieu de rechercher dans le référentiel Content Server. Par exemple, le rapport LiveReport Eléments réservés par dresse la liste des éléments de la base de données qu'un utilisateur spécifique a réservés. Lorsque vous l'ouvrez, le rapport LiveReport vous invite automatiquement à saisir le paramètre (un utilisateur Content Server), puis exécute l'instruction SQL.

Vous pouvez ouvrir un rapport LiveReport si vous disposez de l'autorisation Voir le contenu pour ce document. Pour de plus amples informations sur les autorisations, reportez-vous à la section Types d`autorisations.

Le volume Rapports LiveReport contient un ensemble de rapports LiveReport par défaut. Vous pouvez afficher tous les rapports du volume Rapports LiveReport pour lequel vous disposez au moins de l'autorisation Voir dans l'onglet Rapports LiveReport de votre page Mes rapports.

 

Note :
Les rapports LiveReport peuvent se trouver dans n'importe quel recueil, mais seuls les rapports LiveReport stockés dans le volume Rapports LiveReport s'affichent dans la page Mes rapports.

 

Vous pouvez configurer un rapport LiveReport pour qu'à son ouverture une invite vous demande de fournir des informations supplémentaires, telles que votre nom. La manière dont vous fournissez les informations supplémentaires dépend des paramètres LiveReport. Dans certains cas, vous pouvez rechercher les informations dont vous avez besoin. Dans d'autres cas, vous devez saisir les informations dans un champ. Dans d'autres cas encore, Content Server fournit les informations pour vous.

Certains formats LiveReport permettent d'afficher les informations de manière différente après l'ouverture du rapport. Par exemple, si les informations sont présentées dans un graphique circulaire, vous pouvez faire pivoter le graphique, séparer et regrouper chaque secteur du graphique et séparer les secteurs du graphique. Pour plus d'informations sur les formats LiveReport, reportez-vous à la rubrique Formats des rapports LiveReport.
