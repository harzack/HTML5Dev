Si Formulaires Web est installé, lorsque vous ajoutez ou modifiez un rapport LiveReport, une nouvelle option vous permet de choisir une base de données sur laquelle exécuter le rapport. Les choix représentent les connexions établies dans la fonction Consultation de bases de données formulaires Web. Le choix par défaut correspond à la base de données de l'instance Content Server hébergeant le rapport LiveReport.

Notez que, lorsque vous générez un rapport sur une base de données externe, certaines options LiveReport telles qu'Autoriser les modifications de la base de données, Sélectionner le sous-rapport et Afficher les colonnes ne sont pas appropriées ou prises en charge. Une alerte est alors affichée dans LiveReport Developer et ces options sont supprimées du formulaire Ajouter/Modifier un rapport LiveReport. De plus, le choix Format du rapport « LiveReport » n'est pas pris en charge pour les bases de données externes.
