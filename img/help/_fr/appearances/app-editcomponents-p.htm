Pour modifier les composants de l'apparence :

  1. Cliquez sur l'icône Fonctions de l'apparence, puis sélectionnez Ouvrir.

  2. Sur la page Vue d'ensemble Apparence, cliquez sur Composants Content Server.

  3. Sur la page Modifier les composants de l'apparence, désélectionnez tout élément suivant pour exclure les composants d'interface de l'apparence :

    • En-tête

    • Barre de recherche

      Si la case à cocher Barre de recherche est sélectionnée, vous devez également faire une sélection dans la liste Barre de recherche.

    • Menu Entreprise

    • Bandeau de nouvelles

    • Ajouter un élément

      Si la case à cocher Ajouter un élément est sélectionnée, vous pouvez également sélectionner les cases à cocher Bouton Ajouter document et/ou Bouton Ajouter dossier.

    • navigation

    • Barre latérale

    • Pied de page

  4. Cliquez sur Envoyer.
