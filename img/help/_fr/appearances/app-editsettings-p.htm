Pour modifier les paramètres de l'apparence :

  1. Cliquez sur l'icône Fonctions de l'apparence, puis sélectionnez Ouvrir.

  2. Sur la page Vue d'ensemble Apparence, cliquez sur Paramètres.

  3. Si vous modifiez une apparence basée sur l'emplacement, cliquez sur l'une des options suivantes dans la liste Type :

    • Pas de cascade

    • En cascade

  4. Cliquez sur le bouton radio Activé ou Désactivé.

  5. Cliquez sur Envoyer.
