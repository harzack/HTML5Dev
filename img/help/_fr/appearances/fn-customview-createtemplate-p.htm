Pour créer un modèle de vue personnalisée :

  1. Cliquez sur l'icône Fonctions pour le fichier customview.html pour lequel vous souhaitez créer un modèle, puis cliquez sur Créer un modèle.

  2. Sur la page Ajouter : Modèle de vue personnalisée, saisissez un nom pour le modèle dans le champ Nom.

  3. Dans le champ Source, effectuez l'une des opérations suivantes :

    1. Cliquez sur Vue personnalisée, puis sur Parcourir pour accéder au fichier customview.html à partir duquel vous souhaitez créer le modèle. Cliquez ensuite sur son lien Sélectionner.

    2. Cliquez sur Fichier, puis sur Parcourir pour accéder au fichier .xml à partir duquel vous souhaitez créer le modèle. Cliquez ensuite sur Ouvrir.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez également créer un modèle à partir d'un modèle existant en cliquant sur l'icône Fonctions du modèle d'affichage personnalisé, puis cliquez sur Créer à partir de. Lorsque vous créez un modèle de cette façon, vous devez lui spécifier un nouveau nom en cliquant sur le bouton Ajouter.
