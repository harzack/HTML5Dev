Pour ajouter une fonction Javascript à un affichage personnalisé :

  • Ouvrez le fichier customview.html dans un éditeur, puis ajoutez les fonctions Javascript à intégrer une fois l'affichage personnalisé activé.

 

Note :
Vous pouvez ajouter les fonctions Javascript à un fichier texte, puis ajouter ce fichier en tant qu'affichage personnalisé.

 

Pour ajouter une fonction Javascript à une apparence générale :

  1. Cliquez sur l'icône Fonctions de l'apparence générale, puis sur Ouvrir.

  2. Sur la page Apparence générale, cliquez sur l'icône Modifier de la section à laquelle vous voulez ajouter une fonction Javascript.

  3. Sur la page Modifier, saisissez les fonctions Javascript que vous voulez afficher dans le champ de texte, puis cliquez sur Ajouter une version.
