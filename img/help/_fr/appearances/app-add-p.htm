Pour ajouter une apparence :

  1. Procédez de l'une des manières suivantes :

    1. Pour ajouter une apparence basée sur l'emplacement, cliquez sur Ajouter un élément. Dans la liste, sélectionnez Dossier Apparence.

    2. Pour ajouter une apparence générale, cliquez sur Ajouter un élément. Dans la liste, sélectionnez Apparence générale.

  2. Sur la page Ajouter : Apparence, saisissez un nom pour l'apparence dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description de l'apparence.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à l'apparence.

  5. Content Server enregistre par défaut votre nouvelle apparence dans l'Espace de travail Entreprise. Pour modifier l'emplacement par défaut, cliquez sur Parcourir Content Server dans le champ Créer dans. Choisissez Sélectionner en regard d'un nouvel emplacement.

  6. Cliquez sur Ajouter.
