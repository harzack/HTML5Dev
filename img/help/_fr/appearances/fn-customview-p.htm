Pour ajouter une vue personnalisée :

  1. Dans le menu Ajouter un élément, cliquez sur Vue personnalisée.

  2. Sur la page Ajouter : Vue personnalisée, procédez de l'une des façons suivantes :

    1. Cliquez sur Existant, puis sur Parcourir pour accéder au fichier HTML que vous avez créé et qui définit la vue personnalisée. Cliquez ensuite sur Ouvrir.

    2. Cliquez sur Modèle, puis sur Parcourir Content Server... pour accéder au fichier de modèle qui définit la vue personnalisée. Cliquez ensuite sur Ouvrir.

  3. Dans le champ Nom, saisissez un nom pour l'affichage personnalisé.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  4. Facultatif Dans le champ Description, saisissez une description de la vue personnalisée.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à la vue personnalisée.

  6. Content Server enregistre par défaut votre nouvelle vue personnalisée dans l'Espace de travail Entreprise. Pour modifier l'emplacement par défaut, cliquez sur Parcourir Content Server dans le champ Créer dans. Choisissez Sélectionner en regard d'un nouvel emplacement.

  7. Cliquez sur Ajouter.

 

Note :
Si le fichier HTML renvoie à d'autres fichiers (par exemple, des fichiers image), vous devez ajouter ces fichiers séparément.
