Pour rechercher des classements à l'aide de l'option A partir d'ici (étendue) :

  1. Saisissez votre requête dans le champ texte de la barre de recherche de l'Espace de travail Classements.

  2. Dans la liste Sélectionner le type de recherche, cliquez sur A partir d'ici (étendue).

  3. Sur la page Recherche avancée, le texte entré dans la barre de recherche s'affiche dans le champ Rechercher de la section Texte intégral. Vous pouvez ajouter des critères si nécessaire. Vous pouvez également cliquer sur Tous les mots, N'importe quel mot, Locution exacte ou Requête complexe dans la liste Rechercher.

  4. Dans la section Portée, Au sein du <classement> est présélectionné. Vous pouvez sélectionner des vues supplémentaires dans lesquelles effectuer la recherche.

  5. Dans la section Parcourir Content Server, l'emplacement du classement d'où vous avez lancé la recherche est présélectionné. Pour sélectionner des classements supplémentaires auxquels limiter la recherche, cliquez sur le bouton Parcourir Content Server (facultatif).

  6. Sélectionnez et saisissez tout critère de recherche supplémentaire.

  7. Cliquez sur le bouton Rechercher.

 

Astuces :

  • Lorsque vous lancez une recherche à l'aide de l'option A partir d'ici (étendue), avant que la recherche ne s'exécute, la page Recherche avancée s'ouvre pour vous permettre d'ajouter des critères supplémentaires à votre requête. Par défaut, les paramètres de recherche avancée sont prérenseignés sur la base du classement ou de l'Arbre de classement à partir duquel vous avez lancé la recherche, ce qui limite les résultats de recherche à l'emplacement du classement ou de l'Arbre de classement spécifié. Pour en savoir plus, voir Navigation et recherche dans les classements.

  • Pour limiter votre recherche à un emplacement de classement ou d'Arbre de classement sans ajouter de critères de recherche supplémentaires, lancez votre recherche via l'option A partir d'ici. Pour des informations sur la marche à suivre, reportez-vous à la rubrique Pour rechercher des classements à l'aide de l'option A partir d'ici.
