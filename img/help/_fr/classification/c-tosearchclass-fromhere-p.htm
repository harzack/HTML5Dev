Pour rechercher des classements à l'aide de l'option A partir d'ici :

  1. Saisissez votre requête dans le champ texte de la barre de recherche de l'Espace de travail Classements.

  2. Dans la liste Sélectionner le type de recherche, cliquez sur A partir d'ici.

  3. Cliquez sur le bouton Aller.

    Votre recherche est appliquée à toutes les sources de données Content Server et Enterprise Library.

     

    Astuces :

    • L'option de recherche A partir d'ici limite votre recherche au classement ou à l'Arbre de classement à partir duquel vous avez lancé votre recherche. Si votre recherche ne renvoie aucun résultat, cliquez sur le bouton Affinez votre recherche afin d'étendre ou de modifier vos critères de recherche. La page Recherche avancée s'ouvre.

    • Pour ajouter des critères de recherche sans étendre votre recherche au-delà du classement ou de l'Arbre de classement, utilisez plutôt l'option A partir d'ici (étendue) de la liste Sélectionner le type de recherche. Pour des informations sur la marche à suivre, reportez-vous à la rubrique Pour rechercher des classements à l'aide de l'option A partir d'ici (étendue).
