Pour dissocier un classement d'un élément :

  1. Cliquez sur l'icône Fonctions de l'élément classé, choisissez Propriétés, puis Classements.

  2. Sur la page Propriétés de l'élément, cliquez sur le lien Supprimer en regard du classement que vous souhaitez supprimer.

  3. Dans la boîte de dialogue de confirmation, cliquez sur OK.
