Pour classer des sous-éléments :

  1. Sur la page Vue détaillée, cliquez sur le bouton Fonction correspondant à un recueil ; par exemple, un dossier. Sélectionnez Propriétés, puis Classements.

  2. Sur la page Propriétés, cliquez sur le bouton Appliquer des classements aux sous-éléments.

  3. Dans la boîte de dialogue de confirmation qui s'affiche, cliquez sur OK.

  4. Sur l'écran Application des classements aux sous-éléments, attendez que la ligne Statut affiche la mention Terminé. Cliquez sur le bouton OK.

 

Note :
Le bouton Appliquer des classements aux sous-éléments ne s'affiche pas si aucun classement n'est appliqué au recueil.
