Pour ajouter un arbre de classement :

  1. Dans le menu Entreprise de la barre de menus générale, sélectionnez Classements.

  2. Dans l'espace de travail Classement, cliquez sur Arbre de classement dans le menu Ajouter un élément.

  3. Sur la page Ajouter : un classement, saisissez un nom pour l'arbre de classement dans le champ Nom.

  4. Facultatif Dans le champ Description, saisissez une description de l'arbre de classement.

  5. Cliquez sur l'une des options Type de gestion suivantes :

    • Manuel. Pour la description, reportez-vous à Configuration du type de gestion.

    • Assisté. Pour la description, reportez-vous à Configuration du type de gestion.

    • Automatique. Pour la description, reportez-vous à Configuration du type de gestion.

  6. Désactivez la case Sélectionnable pour empêcher les utilisateurs d'affecter l'arbre de classement aux éléments.

  7. Facultatif Dans le champ Catégories, si vous souhaitez appliquer ou modifier des catégories et des attributs Content Server, cliquez sur Modifier pour afficher le champ Modifier les catégories. Ajoutez ou modifiez les catégories que vous voulez appliquer, puis cliquez sur Terminé.

  8. Facultatif Si vous souhaitez créer l'arbre de classements à un autre emplacement, dans le champ Créer dans, cliquez sur Parcourir Content Server. Recherchez le nouvel emplacement, puis cliquez sur le lien Sélectionner correspondant.

  9. Cliquez sur Ajouter.
