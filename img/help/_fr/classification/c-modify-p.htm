Pour modifier les propriétés d'un classement :

  1. Dans Espace de travail Classement, cliquez sur l'icône Fonctions de l'Arbre de classement ou du classement à modifier. Choisissez Propriétés, puis Extras.

  2. Dans l'onglet Extras de la page Propriétés, effectuez l'une des opérations suivantes :

    • Cliquez sur une option Type de gestion.

    • Activez ou désactivez la case à cocher Sélectionnable.

  3. Cliquez sur le bouton Appliquer.
