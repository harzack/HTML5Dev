Pour classer un nouvel élément :

  1. Dans Espace de travail Entreprise ou Personnel, cliquez sur le type d'élément que vous souhaitez ajouter dans le menu Ajouter un élément.

  2. Sur la page Ajouter de l'élément, spécifiez les paramètres de l'élément. Ensuite, dans la liste Classer, cliquez sur l'une des options suivantes :

    • Parcourir les classements

    • Mes favoris

  3. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement que vous souhaitez assigner à l'élément, Cochez la case correspondante.

  4. Cliquez sur le bouton Envoyer.

  5. Sur la page Ajouter de l'élément, cliquez sur le bouton Ajouter.
