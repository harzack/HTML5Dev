Pour parcourir les classements :

  1. Dans le menu Entreprise, cliquez sur Classements.

  2. Dans Espace de travail Classement, cliquez sur l'icône Fonctions d'un Arbre de classement ou d'un classement, puis choisissez Ouvrir.

  3. Lorsqu'Enterprise Library est installé, cliquez sur Tous, Enterprise ou Enterprise Library dans le menu à bascule en haut à droite pour sélectionner ces types d'élément.
