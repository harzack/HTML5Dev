Pour classer des éléments dans les résultats de recherche :

  1. Sur la page Résultat de la recherche, cochez la case correspondant à chaque élément à classer. Dans la liste Sélectionner l'action, cliquez sur Appliquer les classements.

  2. Sur la page Classer les résultats de recherche, cliquez sur l'une des options suivantes de la liste Classer :

    • Parcourir les classements

    • Mes favoris

  3. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement que vous souhaitez assigner à l'élément. Cochez la case correspondante.

  4. Cliquez sur le bouton Envoyer.

  5. Sur la page Classer les résultats de recherche, cliquez sur le bouton Appliquer.
