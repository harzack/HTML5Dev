Si vous disposez des autorisations adéquates, vous pouvez modifier les propriétés d'un classement après l'avoir ajouté à l'Espace de travail Classement. Par exemple, sur la page Propriétés des extras d'un Arbre de classement ou d'un classement, vous pouvez :

  • changer le type de gestion ou

  • définir si le classement sera ou non disponible à la sélection, c'est-à-dire qu'il pourra être affecté à des éléments.

Pour plus d'informations sur les propriétés d'un classement, reportez-vous à la rubrique Ajout de classements.

Configuration des colonnes pour l'affichage des métadonnées

L'onglet Colonnes de la page Propriétés vous permet de configurer les colonnes qui afficheront les métadonnées des classements, arbres de classement et éléments classés. Les métadonnées s'affichent en regard d'un élément lorsque vous naviguez dans la vue Parcourir. Les métadonnées affichées peuvent être configurées de deux façons : globalement, c'est-à-dire qu'elles apparaissent pour tous les classements, arbres de classement et éléments classés du système, ou localement, ce qui signifie que les métadonnées supplémentaires s'affichent uniquement pour l'élément spécifié. Par défaut, les colonnes de métadonnées Type et Nom sont configurées globalement. Les autres colonnes s'affichent selon la configuration définie par l'utilisateur. Les colonnes de métadonnées pouvant être ajoutées localement dépendent des colonnes créées par un utilisateur. Les colonnes que vous ajoutez s'affichent pour tous les sous-éléments d'un recueil. Elles doivent être configurées dans le volume Facettes de Content Server. Pour plus d'informations, reportez-vous à la rubrique Utilisation du panneau de configuration du volume à facettes.
