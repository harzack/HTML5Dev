Pour retirer des classements de plusieurs éléments :

  1. Dans une collection, des résultats de recherche ou la vue Parcourir d'un espace de travail, cochez la case correspondant à chaque élément dont vous souhaitez supprimer le classement.

  2. Dans le menu Autres actions, cliquez sur Enlever le classement.

  3. Dans la fenêtre Enlever le classement, sélectionnez le classement que vous souhaitez retirer dans la liste déroulante Classements.

  4. Cliquez sur le bouton Envoyer.
