Pour effectuer une recherche dans les classements :

  1. Cliquez sur le lien Recherche avancée de la barre de recherche.

  2. Dans la section Options avancées de la page Recherche, cliquez sur le lien Classements.

  3. Dans la liste déroulante Classer, cliquez sur l'une des options suivantes :

    • Parcourir les classements

    • Mes favoris

  4. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement voulu pour la recherche, puis cliquez sur le lien Sélectionner correspondant.

  5. Spécifiez les critères de recherche supplémentaires de votre choix. Si vous le souhaitez, sélectionnez la case à cocher Inclure les sous-classements pour rechercher les éléments classés dans le classement sélectionné ainsi que tous ses descendants.

  6. Cliquez sur le bouton Rechercher.
