Pour classer l'ensemble d'une collection d'éléments :

  1. Ouvrez une Collection dans la page de navigation.

  2. Sur la page Vue détaillée, cochez la case Collection complète.

  3. Choisissez Appliquer les classements dans la liste Autres actions.

  4. Sur la page Classer la collection complète, cliquez sur l'une des options suivantes de la liste Classer :

    • Parcourir les classements

    • Mes favoris

  5. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement que vous souhaitez assigner à la collection complète. Cochez la case correspondante.

  6. Cliquez sur le bouton Envoyer.

  7. Sur la page Classer les éléments d'une collection, saisissez les informations suivantes : adresse e-mail du destinataire, objet du message et notifications tierces, le cas échéant.

  8. Cliquez sur le bouton Envoyer.

  9. Sur la page de statut du classement de l'ensemble d'une collection, cliquez sur le bouton OK une fois que le statut affiche la mention Terminé.

 

Note :
L'option Appliquer les classements n'est disponible que si votre administrateur vous a accordé les privilèges d'utilisation appropriés.
