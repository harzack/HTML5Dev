Pour classer un élément existant :

  1. Cliquez sur l'icône Fonctions de l'élément que vous souhaitez classer, choisissez Propriétés, puis Classements.

  2. Sur la page Propriétés de l'élément, cliquez sur l'une des options de la liste déroulante Classer :

    • Parcourir les classements

    • Mes favoris

  3. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement que vous souhaitez assigner à l'élément, Cochez la case correspondante.

  4. Cliquez sur le bouton Envoyer.
