Pour configurer les colonnes de métadonnées à afficher :

  1. Cliquez sur le menu Fonctions d'un classement, d'un arbre de classement ou d'un élément classé, cliquez sur Propriétés, puis sélectionnez Colonnes.

  2. Dans la section Colonnes locales de l'onglet de propriétés Colonnes, mettez en surbrillance les colonnes que vous souhaitez afficher, puis cliquez sur la flèche droite pour ajouter ces colonnes à la section Colonnes affichées.

  3. Cliquez sur Mettre à jour.
