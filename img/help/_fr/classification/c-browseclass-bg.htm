Vous pouvez parcourir les Classements pour rechercher des Arbres de classements (hiérarchies de groupes de classements), des classements et des éléments classés dans l'espace de travail Classement. En parcourant les Arbres de classement, vous obtenez la liste de tous les classements et éléments classés associés, ainsi que leur emplacement d'origine dans Content Server. De même, en parcourant les classements, vous obtenez la liste de tous les éléments classés associés, ainsi que leur emplacement d'origine dans Content Server.

Vous pouvez également configurer les classements pour qu'ils apparaissent sous forme de facettes dans le bandeau Filtre de contenu. Si vous sélectionnez une facette de classement, les facettes enfants existantes seront affichées. Si vous utilisez la fonction de recherche de facette pour affiner les résultats, les facettes de classement et les facettes de classement enfants existantes apparaissent dans les résultats. Les facettes doivent être configurées dans le volume Facettes de Content Server. Pour en savoir plus, voir Utilisation du volume à facettes.

Navigation dans Enterprise Library Services

Lorsqu'Enterprise Library est installé, un menu à bascule apparaît dans la section de navigation Elément classé. Ce menu contient les types d'élément Tous, Enterprise et Enterprise Library. Le mode sélectionné est conservé d'une page de navigation de classements à la suivante. Toutefois, lorsque vous naviguez hors d'une page de navigation de classements (par exemple, vers la page Recherche avancée), le filtre de l'élément est redéfini sur le mode par défaut. La prochaine fois que vous naviguez dans un classement, le mode par défaut affiche les types d'élément Enterprise et Enterprise Library.

Le modèle de recherche utilisé pour la pagination de grands ensembles d'éléments classés inclut une liste Portée avec des vues Content Server et Enterprise Library. La liste Portée est requise pour localiser les éventuels éléments classés, puisque les données Enterprise Library sont indexées séparément des données de Content Server. Si Enterprise Library n'est pas installé, les vues Enterprise Library ne sont pas disponibles dans la liste Portée.

 

Note :
Les opérations Copier, Déplacer et Supprimer ne sont pas prises en charge pour les éléments Enterprise Library. Si vous cliquez sur l'un de ces boutons lorsque plusieurs éléments sont sélectionnés, Content Server ignore les éléments Enterprise Library pendant l'opération. Si vous sélectionnez un élément Enterprise Library et appliquez une seule opération, un message d'erreur s'affiche.

 

Recherche

Lorsque vous recherchez des éléments dans l'espace de travail à l'aide de la barre de recherche Content Server, les éléments sont renvoyés dans les résultats de la recherche qui correspondent à votre requête. Lorsque vous lancez une recherche sur des classements, vous obtenez des éléments auxquels des classements spécifiques sont assignés. En incluant les sous-classements, vous pouvez rechercher le classement sélectionné ainsi que tous ses descendants.

 

Note :
Pour plus d'informations sur l'utilisation de la barre de recherche Content Server et sur la recherche dans Content Server, reportez-vous à la rubrique Recherche Content Server.

 

Si vous recherchez fréquemment des éléments au sein des mêmes classements, vous pouvez ajouter ces classements à vos favoris. Cela vous permet de sélectionner les classements à partir de la page Mes favoris lors de vos recherches. Pour plus d'informations sur l'ajout d'éléments comme favoris, reportez-vous à la rubrique Gestion des favoris.

Vous pouvez affiner les résultats d'une recherche en prenant uniquement en compte les éléments ayant le même thème de classement. Les thèmes identifient les auteurs, les emplacements, les dates, les sujets et les classements qui apparaissent le plus fréquemment dans vos résultats de recherche. Lorsque vous affinez vos résultats de recherche à l'aide d'un thème de classement, Content Server inclut uniquement les éléments avec le classement que vous avez choisi.

 

Note :
Pour plus d'informations sur les résultats de recherche, reportez-vous à la rubrique Utilisation des résultats de la recherche.

 

Recherche de classements à partir de la barre de recherche

Lorsque vous êtes dans un espace de travail de classement et que vous lancez une recherche à l'aide de l'option A partir d'ici, la recherche s'effectue dans les vues Enterprise et Enterprise Library. Lorsque vous lancez une recherche à l'aide de l'option A partir d`ici (étendue), la page Recherche avancée s'ouvre et présente les sections et les critères de recherche prédéfinis suivants :

  • Texte intégral, contenant le texte du champ de la barre de recherche

  • Portée, mettant en surbrillance les vues Enterprise et Enterprise Library.

  • Parcourir Content Server, limitant la recherche à l'Espace de travail Classement actuel.
