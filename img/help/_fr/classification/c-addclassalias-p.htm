Pour ajouter un raccourci de classement :

  1. Dans l'Espace de travail Classement d'un arbre de classement, cliquez sur Raccourci de classement dans le menu Ajouter un élément.

  2. Sur la page Ajouter un raccourci de classement, saisissez un nom pour le Raccourci de classement dans le champ Nom. Si vous le souhaitez, vous pouvez saisir une description du raccourci de classement dans le champ Description.

  3. Cliquez sur le bouton Parcourir le classement, naviguez jusqu'au classement que le raccourci doit désigner, puis cliquez sur son lien Sélectionner. Vous pouvez également cliquer sur le bouton Modifier pour changer les catégories ou attributs associés au raccourci de classement. Si vous le souhaitez, cliquez sur le bouton Parcourir Content Server pour sélectionner un autre emplacement où ajouter le raccourci.

  4. Cliquez sur le bouton Ajouter.
