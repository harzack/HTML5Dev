Pour enlever une attente d'éléments d'une collection :

  1. Dans la collection, activez la case à cocher en regard des éléments dont vous souhaitez retirer l'attente.

  2. Cliquez sur Enlever une attente dans la liste déroulante Autres actions.

  3. Sur la page Rechercher les attentes, spécifiez le nom de l'attente dans le champ Rechercher un nom d'attente pour, puis cliquez sur le bouton Recherche.

  4. Cliquez sur le lien Sélectionner d'une attente.

    Vous pouvez également spécifier un motif pour la suppression de l'attente dans la zone de texte de la fenêtre Confirmer la suppression de l'attente.

  5. Cliquez sur le bouton OK.
