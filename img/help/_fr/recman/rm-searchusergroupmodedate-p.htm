Pour rechercher des modifications d'utilisateur/de groupe par date :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records Management, cliquez sur le lien Audit des modifications d'utilisateur/de groupe.

  4. Sur la page Informations sur l'audit d'utilisateur/de groupe, cliquez sur une option dans la première liste déroulante de la section Date.

  5. Cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  6. Cliquez sur le bouton Recherche par date.
