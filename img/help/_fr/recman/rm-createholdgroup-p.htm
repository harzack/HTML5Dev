Pour créer un groupe d'attentes :

  1. Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes.

  2. Sur la page Gestion des attentes, cliquez sur l'option Groupe d'attentes du menu Ajouter un élément.

  3. Sur la page Groupe d'attentes, spécifiez un nom et une description pour le groupe d'attentes dans les champs appropriés, puis cliquez sur le bouton Ajouter.
