Pour configurer les paramètres de sort final :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Paramètres de sort final.

  4. A la section Réviseur de sorts finals de la page Paramètres de sort final, cliquez sur un bouton radio pour indiquer comment le réviseur de sorts finals est déterminé pour les éléments obtenus à l'issue d'une recherche de sorts finals.

  5. A la section Décision de révision de sorts finals par défaut, cliquez sur une valeur par défaut dans la liste Décision de révision.

  6. Cliquez sur le bouton radio Oui ou Non pour chaque paramètre restant, puis cliquez sur Envoyer.
