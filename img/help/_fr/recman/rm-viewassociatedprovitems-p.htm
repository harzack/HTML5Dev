Pour afficher les éléments de provenance associés :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions d'une provenance, puis choisissez Afficher sous forme d'arborescence.

  2. Sur la page Arborescence, cliquez sur l'icône Afficher les éléments associés située en regard de la provenance dont vous souhaitez afficher les éléments associés.
