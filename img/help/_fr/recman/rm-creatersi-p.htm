Pour créer un ISA :

  1. Dans le menu Entreprise, sélectionnez Records management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur ISA dans le menu Ajouter un élément.

  4. Sur la page d'ajout des ISA, spécifiez un nom pour l'ISA dans le champ ISA.

  5. Facultatif Vous pouvez également spécifier un titre, une description et un objet pour l'ISA en les saisissant dans les champs appropriés.

  6. Cliquez sur un statut pour l'ISA dans la liste déroulante Statut.

  7. Facultatif Activez la case Sous contrôle de disposition pour empêcher la suppression des éléments avec l'ISA assigné.

  8. Cliquez sur le bouton Ajouter.
