Pour assigner des utilisateurs à une attente :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Gestion des attentes.

  3. Dans la page Gestion des attentes, cliquez sur l'icône Fonctions de l'attente à laquelle vous souhaitez assigner des utilisateurs, puis cliquez sur Maintenance d'attente utilisateur et sélectionnez Ajouter des utilisateurs.

  4. Dans l'onglet Ajouter des utilisateurs, recherchez un utilisateur ou un groupe d'utilisateurs, activez la case à cocher de chaque utilisateur que vous souhaitez assigner à l'attente, puis cliquez sur le bouton Ajouter.

  5. Cliquez sur l'onglet Traiter l'attente.

  6. Saisissez un intervalle de dates dans les listes déroulantes A partir du et Jusqu'au pour les éléments créés.

    Facultatif : saisissez un commentaire pour l'attente ou une raison expliquant la modification effectuée dans le champ Commentaire.

  7. Cliquez sur le bouton Traiter l'attente.
