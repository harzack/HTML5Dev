Pour créer un rapport :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Rapports.

  3. Sur la page Rapports, cliquez sur le lien correspondant au rapport que vous voulez créer.

  4. Spécifiez les autres paramètres.

  5. Cliquez sur le bouton Exécuter le rapport.
