Pour rechercher des modifications d'utilisateur/de groupe par utilisateur supprimé :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records Management, cliquez sur le lien Audit des modifications d'utilisateur/de groupe.

  4. Sur la page Informations sur l'audit d'utilisateur/de groupe, cliquez sur le bouton Rechercher un utilisateur supprimé.

  5. Sur la page Sélectionner un utilisateur supprimé, spécifiez des critères de recherche, puis cliquez sur le bouton Rechercher.

  6. Cliquez sur le lien Sélectionner d'un utilisateur ou d'un groupe.

  7. Cliquez sur le bouton Recherche par nom.
