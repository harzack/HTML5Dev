Pour afficher les informations sur l'approbation d'ISA :

  1. Dans l'espace de travail Entreprise, cliquez sur Records Management.

  2. Dans l'espace de travail de Records Management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur l'icône Fonctions de l'ISA dont vous souhaitez afficher les informations d'approbation, puis cliquez sur Afficher les informations d'approbation.

 

Note :
L'option Afficher les informations d'approbation de l'icône Fonctions ne s'affiche que si l'administrateur l'a activée.
