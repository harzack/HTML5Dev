Pour créer une recherche manuelle de sorts finals :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Sorts finals.

  3. Sur la page Sort final, cliquez sur Recherche de sorts finals dans le menu Ajouter : Elément.

  4. Sur la page Recherche de sorts finals, spécifiez les paramètres.

  5. Cliquez sur le bouton Ajouter.

  6. Sur la page Sorts finals, cliquez sur l'icône Fonctions pour une recherche de sorts finals, choisissez Propriétés puis Infos spécifiques.

  7. Sur l'onglet Infos spécifiques de la page Propriétés pour une recherche de sorts finals, cliquez sur une phase dans la liste déroulante Phase.

  8. Spécifiez vos critères de recherche.

  9. Dans la section Spécifications de date calculée - Sélectionnez Manuel ou Programmé, cliquez sur le bouton radio Manuel (Sélectionner l'intervalle de date).

  10. Cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  11. Cliquez sur le bouton Ajouter.

 

Note :
Pour l'intervalle de dates, une seule date est requise ; cependant, vous pouvez définir les dates de début et de fin.

 
 

Astuce :
Vous pouvez ajouter une recherche de sorts finals à un groupe de sort final en cliquant sur le lien d'un groupe de ce type, puis sur Recherche de sorts finals dans le menu Ajouter : Elément.
