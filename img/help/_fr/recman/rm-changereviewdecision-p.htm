Pour modifier une décision de révision de sort final :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. A la page Records Management, cliquez sur le lien Sorts finals.

  3. A la page Recherche de sorts finals, cochez la case du lot de résultats pour lequel vous souhaitez changer la décision de révision, puis cliquez sur le bouton Changer la décision de révision.

  4. Dans la fenêtre Changer la décision de révision, cliquez sur le bouton radio En attente, Approuver ou Rejeter.

  5. Facultatif Saisissez les informations relatives à la décision dans le champ Commentaires.

  6. Dans la liste Appliquer à, activez l'un des boutons d'option suivants :

    • Eléments sélectionnés sur cette page : la décision de révision et les commentaires mis à jour sont appliqués aux éléments sélectionnés sur la page actuelle.

    • Eléments sélectionnés dans le lot de résultats complet : la décision de révision et les commentaires mis à jour sont appliqués aux éléments sélectionnés sur toutes les pages du lot de résultats.

  7. Cliquez sur OK.

 

Astuce :
Vous pouvez modifier la décision de révision pour un élément particulier en cliquant sur son lien dans la colonne Décision de révision.
