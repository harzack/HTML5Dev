Pour définir l'accès fonctionnel et privilégié :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. A la page Records Management, cliquez sur le lien Administration de Records Management, puis sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Accès fonctionnel.

  4. Dans la zone Gestionnaires de documents, cliquez sur le bouton radio Département du propriétaire, Groupe de propriétaires ou Classement RM pour déterminer comment les groupes de gestionnaires de documents sont sélectionnés pour les objets gérés.

  5. Dans la zone Groupe de gestionnaires de documents, cliquez sur le bouton Sélectionner un groupe, accédez au groupe auquel vous voulez appliquer les autorisations de Groupe de gestionnaires de documents, puis cliquez sur son lien Sélectionner.

  6. Dans la zone Listes de sélection d'utilisateur, cliquez sur le bouton Oui pour autoriser la création de listes de sélection de groupe pour des utilisateurs spécifiques.

  7. Dans la zone Accès fonctionnel, cliquez sur le bouton Oui pour autoriser le filtrage des actions Records Management par accès fonctionnel.

  8. Dans la zone Groupe d'accès fonctionnel global, cliquez sur le bouton Sélectionner un groupe, accédez au groupe auquel vous voulez appliquer les paramètres d'accès fonctionnel, puis cliquez sur son lien Sélectionner.

  9. Dans la section Configurer l'accès fonctionnel à Records Management (RM), utilisez les flèches vers la gauche et vers la droite pour ajouter ou enlever des fonctions pour les utilisateurs disposant des autorisations Voir ou Voir le contenu.

  10. Dans la section Configurer l'accès privilégié à Records Management (RM), cliquez sur le lien Assigner de la colonne Actions de la fonction que vous souhaitez assigner.

  11. Sur la page Accès privilégié pour les objets gérés, recherchez le groupe auquel vous souhaitez assigner un accès privilégié, sélectionnez le bouton Ajouter/Enlever correspondant, puis cliquez sur le bouton Mettre à jour.

  12. Cliquez sur le bouton Envoyer.
