Pour modifier les paramètres de Records management :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Paramètres RM.

  4. Pour chaque paramètre, cliquez sur le bouton radio Oui ou Non.

  5. Dans la zone Masquer l'option permettant d'hériter du classement RM sur copier et déplacer, cliquez sur le bouton radio Toujours conserver le classement RM ou Toujours hériter du classement RM pour spécifier l'action à prendre avec le classement RM sur un élément lorsqu'il est copié ou déplacé et que l'option Hériter du classement RM est masquée.

  6. Dans la zone Workflow de suppression, cliquez sur Sélectionner un schéma de workflow pour choisir un schéma de workflow qui sera utilisé lors de la suppression des objets du classement RM.

  7. Dans la zone Activer une version de document, choisissez le code de version de document par défaut qui est utilisé lorsque les objets sont créés avec une nouvelle version.

  8. Cliquez sur le bouton Envoyer.
