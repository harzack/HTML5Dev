Pour modifier les paramètres personnels de Records Management :

  1. Sélectionnez Paramètres dans le menu Outils.

  2. A la page Mes paramètres généraux, cliquez sur l'onglet Records Management.

  3. Pour masquer les onglets Records Management de la page Propriétés des éléments gérés, activez les cases à cocher Masquer l'onglet Attentes, Masquer l'onglet Détails du document ou Masquer l'onglet Renvoi.

  4. Pour indiquer un classement RM par défaut pour les éléments gérés, cliquez sur le bouton Ajouter dans la section Classements RM par défaut, accédez au classement que vous souhaitez utiliser par défaut, puis cliquez sur son lien Sélectionner.

  5. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Vous pouvez supprimer le classement par défaut en cliquant sur le bouton Enlever dans la section Classement RM par défaut.
