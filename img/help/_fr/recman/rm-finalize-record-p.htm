Pour finaliser des enregistrements :

  1. Dans le menu Entreprise, cliquez sur Records management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système, puis sur l'onglet Finaliser l'enregistrement.

  3. Dans la page Administration Finaliser l'enregistrement, cliquez sur Oui pour activer les options Eliminer toutes les versions, Détruire les éléments verrouillés ou transformés en génération ou Mettre à jour la date de document.

  4. Dans la liste Mettre à jour le statut, sélectionnez la valeur à appliquer à l'élément une fois finalisé.

  5. Dans la section Interdire la fonctionnalité Finaliser l'enregistrement pour les documents de version secondaire, cliquez sur Oui pour empêcher que l'élément soit finalisé si sa dernière version est une version secondaire.

  6. Cliquez sur Envoyer.

 

Astuce :
Si vous disposez des autorisations appropriées, vous pouvez finaliser simultanément plusieurs enregistrements dans la vue Parcourir d'un espace de travail. Pour ce faire, cochez la case correspondant à chaque élément, cliquez sur Finaliser l'enregistrement, puis sur Finaliser. Si le gestionnaire de documents a activé cette fonction, vous pouvez également finaliser les enregistrements dans la vue Parcourir d'un espace de travail, les résultats de recherche de sort final ou une collection, ainsi qu'en cliquant sur l'option Finaliser l'enregistrement dans le menu Fonctions de l'élément.
