Pour créer un terme pour un terme racine :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Termes du classement RM.

  3. Sur la page Termes du classement RM, cliquez sur le lien d'un terme racine.

  4. Sur la page Plan des termes du classement RM, cliquez sur le bouton Ajouter un nouveau terme.

  5. Sur la page Ajouter un nouveau terme, saisissez le nom d'un terme dans le champ Nouveau terme.

  6. Pour définir la relation entre le terme et le terme racine, cliquez sur une relation dans la liste déroulante Relation.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Créer.
