Pour éliminer l'audit des journaux de suppression :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records Management, cliquez sur le lien Audit des journaux de suppression.

  4. Sur la page Audit de suppression pour les objets classés, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes de la section Eliminer avant la date.

    Facultatif : pour créer un fichier d'élimination d'audit, cochez l'option Créer un fichier d'élimination d'audit.

  5. Cliquez sur le bouton Eliminer l'audit de suppression.

 

Note :
Si l'élimination des journaux d'audit est désactivée dans Content Server, l'option d'élimination du journal d'audit des suppressions n'est pas disponible.
