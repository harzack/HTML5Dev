Pour mettre à jour une provenance :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions de la provenance que vous souhaitez mettre à jour, cliquez sur Propriétés, puis sélectionnez Infos spécifiques.

  2. Dans l'onglet Infos spécifiques de la page Propriétés de provenance, modifiez les paramètres disponibles.

  3. Pour enregistrer l'historique de mise à jour dans le journal de l'audit, activez la case à cocher Enregistrer dans l'historique.

  4. Cliquez sur le bouton Mettre à jour.
