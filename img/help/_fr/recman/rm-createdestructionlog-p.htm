Pour afficher et créer un journal de destruction d'un lot de résultats :

  1. Dans l'espace de travail de Records management, cliquez sur le lien Sorts finals.

  2. Cliquez sur l'icône Fonctions d'un sort final dont vous souhaitez afficher un lot de résultats, puis sélectionnez Afficher les lots de résultats.

  3. Cliquez sur l'icône Fonctions du lot de résultats, puis choisissez Afficher le journal de destruction.

  4. Pour enregistrer le journal de destruction dans Content Server, procédez de la façon suivante :

    • Cliquez sur le bouton Parcourir Content Server, naviguez jusqu'au recueil de Content Server dans lequel vous souhaitez enregistrer le fichier journal, puis cliquez sur le lien Sélectionner correspondant.

    • Cliquez sur le bouton Créer un journal.

  5. Cliquez sur le lien Revenir au sort final.
