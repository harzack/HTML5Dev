Pour ajouter une provenance :

  1. Dans le menu Entreprise, cliquez sur Records management.

  2. Dans l'espace de travail de Records management, cliquez sur le lien Classement.

  3. Dans l'espace de travail Classement, cliquez sur Provenance dans le menu Ajouter : Elément.

  4. Dans la page d'ajout de provenance, saisissez le nom de la provenance dans le champ Nom.

    Vous pouvez aussi saisir une description de la provenance dans le champ Description.

  5. Activez la case à cocher Sélectionnable pour autoriser l'assignation de nouvelles provenances à des objets.

  6. Dans la liste déroulante Type de provenance, effectuez l'une des opérations suivantes :

    • Cliquez sur un type de provenance existant.

    • Pour créer un nouveau type de provenance, cliquez sur <Créer un nouveau code>, puis saisissez un code, une description et une description complémentaire dans les champs appropriés de la boîte de dialogue d'ajout de type de provenance.

  7. Spécifiez un ou plusieurs des paramètres facultatifs suivants :

    • Autre nom

    • Autre description

    • Date de début

    • Date de fin

  8. Activez la case à cocher Désactiver pour montrer si le corps de provenance est toujours en vigueur.

  9. Dans le champ Texte, saisissez les informations à afficher dans l'onglet Infos spécifiques et dans l'historique Provenance.

  10. Pour assigner une catégorie à la provenance, cliquez sur le bouton Modifier de la section Catégories, accédez à la catégorie à assigner à la provenance, puis cliquez sur le bouton Terminé.

  11. Pour créer la provenance dans un autre emplacement que celui défini par défaut, cliquez sur le bouton Parcourir Content Server, accédez à l'emplacement où vous souhaitez créer l'élément, puis cliquez sur le lien Sélectionner correspondant.

  12. Cliquez sur le bouton Ajouter.
