Pour afficher des objets Content Server assignés à une attente :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Gestion des attentes.

  3. Cliquez sur l'icône Fonctions d'une attente, puis sur Afficher les objets Content Server.

 

Note :
En l'absence d'éléments Content Server, le message suivant apparaît : Aucun élément Content Server n'existe pour cette ATTENTE.
