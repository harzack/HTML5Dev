Pour enlever une relation :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Termes du classement RM.

  3. Sur la page Termes du classement RM, cliquez sur le lien correspondant à un terme.

  4. Sur la page Plan des termes du classement RM, cliquez sur le bouton Relation.

  5. Sur la page Ajouter des relations pour, cliquez sur le lien Enlever d'un terme.

  6. Cliquez sur le bouton Terminé.
