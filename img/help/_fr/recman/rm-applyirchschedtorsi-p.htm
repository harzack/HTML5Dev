Pour appliquer un programme IRCH à un programme ISA :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur l'icône Fonctions d'un ISA, puis choisissez Appliquer un programme IRCH.

  4. Sur la page Appliquer un programme IRCH, activer le bouton radio qui correspond au programme IRCH que vous souhaitez appliquer.

     

    Note :
    Si le programme IRCH a déjà été appliqué au programme ISA, une coche s'affiche en regard du programme.

     

  5. Cliquez sur le bouton Appliquer.
