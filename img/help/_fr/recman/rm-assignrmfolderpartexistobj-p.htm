Pour assigner un dossier ou une partie RM à un objet existant :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Détails du document.

  2. Sur la page Détails du document, cliquez sur Parcourir les classements dans la liste déroulante Sélectionner un classement.

  3. Sur la page Parcourir les classements, localisez un dossier ou une partie RM.

  4. Cliquez sur le lien Sélectionner du dossier ou de la partie RM.

 

Note :
Un dossier RM doit présenter une partie RM ouverte pour que des objets puissent lui être assignés.
