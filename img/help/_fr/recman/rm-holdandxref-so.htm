Le module Records management vous permet d'effectuer certaines opérations à partir des objets que vous créez. Vous pouvez attribuer des attentes aux objets afin que les objets en attente ne soient pas supprimés. Si vous déplacez un objet auquel une attente a été attribuée, cette attente sera transférée avec l'objet déplacé. Vous pouvez également lier les éléments à l'aide de renvois.
