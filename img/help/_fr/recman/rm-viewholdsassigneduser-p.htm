Pour afficher les attentes assignées à un utilisateur :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur l'icône Fonctions Gestion des attentes, puis sur Afficher les attentes attribuées aux utilisateurs.

  3. Sur la page de recherche des attentes des utilisateurs, saisissez le nom de l'utilisateur ou du groupe pour lequel vous souhaitez afficher les attentes assignées, puis cliquez sur le bouton Rechercher.

  4. Dans les résultats de la recherche, cliquez sur le lien dans la colonne Actions pour afficher les attentes assignées à un utilisateur.

 

Note :
L'option Afficher les attentes attribuées aux utilisateurs apparaît uniquement lorsque l'accès fonctionnel est activé et que l'option Objet géré – Gérer les attentes de l'utilisateur est activée, ou si vous appartenez au groupe principal de gestionnaires de documents.
