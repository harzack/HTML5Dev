Pour définir un archiviste :

  1. Dans le menu Entreprise, sélectionnez Records management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Définir les archivistes.

  4. Utilisez la barre de recherche située en haut de la page Sélectionner un groupe pour l'archiviste afin de trouver un groupe.

  5. Cliquez sur le lien Ajouter d'un groupe.

  6. Sur la page Sélectionner un archiviste pour le groupe, cliquez sur le bouton Rechercher de la section Nouvel archiviste.

  7. Sur la page Sélectionner un utilisateur, recherchez un utilisateur, puis cliquez sur son lien Sélectionner.

    Facultatif : pour que l'utilisateur sélectionné devienne l'archiviste principal, activez la case à cocher Rendre principal de la page Sélectionner un archiviste pour le groupe.

  8. Cliquez sur le bouton Mettre à jour.
