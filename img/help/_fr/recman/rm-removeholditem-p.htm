Pour enlever une attente d'un élément :

  1. Cliquez sur l'icône Fonctions d'un objet géré et sélectionnez Propriétés, puis Attentes.

  2. Dans l'onglet Attentes de la page Propriétés de l'objet, cliquez sur le lien Enlever d'une attente.

    Vous pouvez également spécifier un motif pour la suppression de l'attente dans la zone de texte de la fenêtre Confirmer la suppression de l'attente.

  3. Cliquez sur le bouton OK.

 

Astuce :
Pour enlever une attente de plusieurs éléments de façon simultanée, activez la case à cocher située en regard de chaque élément pour lequel vous souhaitez enlever l'attente, puis cliquez sur Enlever l'attente dans la liste déroulante Autres actions.

Vous pouvez enlever une attente à tous les éléments d'une collection en activant la case à cocher Collection complète dans la vue de détail d'une collection.
