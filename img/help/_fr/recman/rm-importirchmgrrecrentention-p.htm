Pour importer les séries d'archives du gestionnaire de rétention et les programmes de rétention des archives IRCH, puis les mapper aux ISA et aux programmes :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Importer/Exporter.

  4. Sur la page Importer/Exporter de Records management, cliquez sur le lien Importation de gestionnaire de rétention IRCH.

  5. Une fois sur la page Importation de gestionnaire de rétention IRCH, procédez comme suit :

    • Cliquez sur le bouton Parcourir en regard du champ Linkrr.txt, accédez au fichier Linkrr.txt, et cliquez sur le bouton Ouvrir.

    • Cliquez sur le bouton Parcourir en regard du champ Linkrs.txt, accédez au fichier Linkrs.txt, et cliquez sur le bouton Ouvrir.

     

    Note :
    Les fichiers que vous importez doivent se nommer Linkrr.txt et Linkrs.txt.

     

  6. Cliquez sur le bouton Importer.

  7. Sur la page Mappage du gestionnaire de rétention IRCH, cliquez sur une phase dans la liste déroulante Phase.

  8. Cliquez sur un statut de l'ISA dans la liste déroulante Statut de l'ISA.

  9. Si les événements IRCH s'affichent dans la section Conditions d'événement, cliquez sur les codes d'état Content Server dans les listes correspondantes afin de les mapper à chaque élément.

  10. Dans la section Fin de l'exercice fiscal, saisissez le mois de fin dans le champ Mois, puis le jour de fin dans le champ Jour.

  11. Cliquez sur le bouton Continuer. Une fois le mappage achevé, la page Statut de mappage IRCH s'affiche.
