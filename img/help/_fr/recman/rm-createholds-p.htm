Pour créer une attente :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Gestion des attentes.

  3. Sur la page Gestion des attentes, cliquez sur l'option Attente du menu Ajouter un élément. Si vous voulez ajouter une attente à un groupe d'attentes, recherchez d'abord le groupe d'attente avant de cliquer sur l'option Attente du menu Ajouter un élément.

  4. Sur la page Ajouter une attente, saisissez un nom unique pour l'attente dans le champ Nom de l'attente.

     

    Note :
    Les noms des attentes doivent être uniques. Si vous entrez un nom qui existe déjà, un message d'avertissement s'affiche et vous devez renommer l'attente pour pouvoir continuer.

     

  5. Dans la section Date d'application, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  6. Cliquez sur un type d'attente dans la liste déroulante Type d'attente.

    Facultatif : pour spécifier un autre ID d'attente, saisissez un ID dans le champ Autre ID d'attente.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Ajouter.

 

Astuce :
Pour créer une attente pour un élément existant, cliquez sur l'icône Fonctions d'un élément, puis sur Propriétés et Attentes. Dans l'onglet de propriétés des attentes, cliquez sur Créer une nouvelle attente dans la liste déroulante Appliquer : Attente.
