Pour...
 Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes, puis procédez comme suit...
Modifier une attente Cliquez sur le lien d'une attente, modifiez les paramètres souhaités, puis cliquez sur le bouton Mettre à jour.
Supprimer une attente Cliquez sur l'icône Fonctions d'une attente, puis choisissez Supprimer.
Interrompre une attente Cliquez sur l'icône Fonctions d'une attente, choisissez Interrompre, cliquez sur une date dans la liste déroulante de la section Date d'interruption, puis cliquez sur le bouton Mettre à jour.
Réactiver une attente Cliquez sur l'icône Fonctions d'une attente, choisissez Réactiver, puis cliquez sur le bouton Mettre à jour.
