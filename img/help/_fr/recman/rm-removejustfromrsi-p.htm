Pour enlever une justification d'un ISA :

  1. Dans le menu Entreprise, sélectionnez Records management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur le lien d'un ISA.

  4. Sur la page Infos spécifiques pour, cliquez sur l'onglet Justifications.

  5. Sur la page Justifications, cliquez sur le lien Enlever d'une justification.
