Pour créer une partie RM supplémentaire dans un dossier RM :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Classement.

  3. Cliquez sur un lien de classement RM.

  4. Sur la page Classement RM, cliquez sur un lien de dossier RM.

  5. Sur la page Dossier RM, cliquez sur Partie RM dans le menu Ajouter : Elément.

  6. Dans la section Date de création, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Ajouter.
