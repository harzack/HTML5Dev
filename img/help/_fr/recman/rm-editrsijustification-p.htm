Pour modifier une justification :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Justifications.

  4. Sur la page Justifications, cliquez sur le lien Modifier correspondant à une justification.

  5. Sur la page Modifier la justification, effectuez les modifications souhaitées.

  6. Cliquez sur le bouton Mettre à jour.
