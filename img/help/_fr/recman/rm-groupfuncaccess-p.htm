Pour configurer un accès fonctionnel de groupe :

  1. Cliquez sur Utilisateurs et groupes dans le menu Entreprise.

  2. Dans la section Définir les fonctions disponibles de la page Modifier le groupe, cliquez sur le lien Accès fonctionnel du groupe.

  3. Dans la section Sélectionner des fonctions pour un groupe, ajoutez les fonctions à assigner au groupe dans la liste Fonctions assignées, puis cliquez sur le bouton Envoyer.
