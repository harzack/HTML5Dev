Pour exporter le contenu d'une attente :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Gestion des attentes.

  3. Sur la page Gestion des attentes, cliquez sur l'icône Fonctions d'une attente, puis choisissez Exportation du contenu en attente.

     

    Note :
    En l'absence d'éléments Content Server, le message suivant apparaît :

    Aucun élément dans cette attente.

     

  4. Dans la fenêtre Exportation du contenu en attente, cliquez sur le bouton Version actuelle ou Toutes les versions, puis cliquez sur le bouton Exporter.
