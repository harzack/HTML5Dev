Pour créer un programme ISA :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur l'icône Fonctions d'un ISA, puis sélectionnez Créer un programme.

  4. Sur la page d'ajout de programme ISA, cliquez sur le bouton radio Classements RM ou Objets classés.

  5. Dans la liste déroulante Phase, cliquez sur une phase.

  6. Dans la section Type de règle, cliquez sur le bouton radio d'un type de règle.

    Vous pouvez aussi cliquer sur un code pour la règle dans la liste déroulante Code de règle, entrer une raison pour l'action dans le champ Commentaire sur la règle, puis cliquer sur le bouton Continuer.

  7. Pour créer un programme ISA, procédez comme suit selon le type de règle :

    • Définitif : Sur la page Programme ISA - Action à exécuter, cliquez sur une action de décision dans la liste déroulante Décision, puis cliquez sur le bouton Ajouter.

    • Evénementiel : passez à l'étape 9.

    • Date précise : cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes de la section Date précise.

    • Calcul selon le calendrier : Effectuez les opérations suivantes :

      • Cliquez sur une valeur dans la liste déroulante Date à utiliser. Si vous sélectionnez Date de la catégorie, cliquez sur le lien Sélectionner une catégorie et sélectionnez la catégorie que vous souhaitez utiliser.

      • Dans la section Définition de la fin de l'année, saisissez dans les champs correspondants le mois et le jour à utiliser comme définition de fin d'année.

      • Pour définir la fréquence des intervalles du calcul selon le calendrier, cliquez sur une valeur dans la liste déroulante Nombre d'intervalles.

      • Pour définir la période de conservation, saisissez des valeurs dans les champs Années, Mois et Jours de la section Période.

    • Date calculée : cliquez sur une valeur dans la liste déroulante Date à utiliser, puis saisissez des valeurs dans les champs Années, Mois et Jours de la section Période pour définir la période de conservation. Si vous sélectionnez Date de la catégorie, cliquez sur le lien Sélectionner une catégorie et sélectionnez la catégorie que vous souhaitez utiliser.

  8. Cliquez sur le bouton Continuer.

  9. Sur la page Programme d'ISA – Conditions d'événement pour, choisissez des propriétés de condition dans le menu Propriété, puis définissez les propriétés des conditions d'événement.

    Pour définir la relation entre plusieurs propriétés, cliquez sur un opérateur booléen dans la liste déroulante d'une propriété (facultatif).

  10. Sur la page Programme d'ISA - Action à exécuter pour, cliquez sur l'un des boutons radio pour spécifier une action par défaut.

  11. Cliquez sur le bouton Ajouter un élément.
