Pour définir les paramètres d'autorisation des objets classés :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Autorisations d'accès.

  4. Dans les sections Classer et Officialiser de la page Configurer l'accès sur Classer et Officialiser, cliquez sur l'un des boutons radio suivants :

    • Utiliser les restrictions d'accès existantes

    • Ajouter à la liste d'accès

    • Remplacer la liste d'accès

  5. Dans la section Autorisations, cochez la case correspondant à chaque type d'autorisation que les utilisateurs doivent posséder pour exécuter les fonctions Officialiser, Supprimer l'officialisation et Finaliser l'enregistrement.

  6. Cliquez sur le bouton Envoyer.
