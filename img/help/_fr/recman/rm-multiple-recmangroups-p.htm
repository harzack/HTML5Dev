Pour assigner un groupe de gestionnaires de documents à un groupe principal Content Server :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Dans l'espace de travail de Records management, cliquez sur le lien Administration de Records management, puis cliquez sur le lien Groupes de gestionnaires de documents multiples.

  3. Sur la page Sélectionner un groupe pour les gestionnaires de documents, spécifiez des critères de recherche, puis cliquez sur le bouton Rechercher.

  4. Cliquez sur le lien Ajouter dans la colonne Actions du groupe souhaité.

  5. Sur la page Sélectionner un utilisateur, cliquez sur Nom de groupe dans la liste déroulante Rechercher, spécifiez des critères, puis cliquez sur le bouton Rechercher.

  6. Cliquez sur le lien Sélectionner d'un groupe.
