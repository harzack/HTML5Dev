Pour importer des documents :

 

Note :
Pour pouvoir réimporter des documents dans Content Server, votre administrateur système doit avoir activé le paramètre DoD. Si le paramètre DoD n'est pas activé, l'option d'importation de documents n'est pas disponible.

 

  1. Dans l'e-mail que vous recevez après exécution d'une recherche de sorts finals, cliquez sur le lien ici pour poursuivre le transfert d'éléments exportés.

     

    Note :
    Vous pouvez également poursuivre le transfert en cliquant sur l'option Continuer le transfert du menu Fonctions du lot de résultats de sorts finals.

     

  2. Sur la page Continuer le transfert DoD, activez la case à cocher de chaque attribut au sein d'une catégorie assignée à un objet exporté pour lequel vous souhaitez assigner un nom d'élément XML, puis cliquez sur le bouton Continuer.

  3. Sur la page Réussite de la transformation des fichiers exportés, vérifiez l'emplacement des fichiers transformés afin que vous puissiez aller les chercher et les copier vers un disque ou un emplacement externe, si nécessaire.

    Facultatif : pour détruire les objets, revenez au lot de résultats de sorts finals, puis cliquez sur le bouton Exécution d'actions.

  4. Dans l'espace de travail de Records management, cliquez sur le lien Administration de Records management.

  5. Sur la page Administration de Records management, cliquez sur le lien Importer/Exporter.

  6. Dans la section Importer de la page Importer/Exporter de Records management, cliquez sur le lien Importation de documents.

  7. Sur la page Importation de documents, spécifiez le chemin absolu du dossier sur le serveur Content Server sur lequel les fichiers d'importation XML compatibles DoD sont situés, puis cliquez sur le bouton Continuer.

  8. Sur la page Mappage d'importation, spécifiez les valeurs par défaut des champs de classement RM obligatoires ainsi que les emplacements de création des classements RM et objets gérés sur Content Server, puis mettez en correspondance les métadonnées DoD avec les codes Records management existants.

  9. Activez la case à cocher située en regard de chaque propriété pour laquelle vous souhaitez qu'une nouvelle catégorie soit créée. La propriété est une métadonnée définie par l'entreprise. Pour chaque propriété que vous sélectionnez, une nouvelle catégorie est créée et assignée aux objets importés.

  10. Cliquez sur le bouton Importer.
