Pour assigner un dossier ou une partie RM à un nouvel objet :

  1. Dans l'espace de travail Entreprise ou Personnel, cliquez sur un type d'élément dans le menu Ajouter : Elément.

  2. Cliquez sur Parcourir les classements dans la liste déroulante Classer.

  3. Sur la page Parcourir les classements, localisez un dossier ou une partie RM.

  4. Sélectionnez la case à cocher du dossier ou de la partie RM.

  5. Cliquez sur le bouton Envoyer.

 

Note :
Un dossier RM doit présenter une partie RM ouverte pour que des objets puissent lui être assignés.
