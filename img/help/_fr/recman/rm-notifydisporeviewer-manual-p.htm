Pour envoyer manuellement une notification de révision de sorts finals :

  1. Cliquez sur l'icône Fonctions d'une recherche de sorts finals, puis sélectionnez Afficher les résultats par réviseur de sorts finals.

  2. A la page Afficher les résultats par réviseur de sorts finals, cliquez sur le lien Envoyer une notification en regard du réviseur souhaité.

  3. Cliquez sur OK.

 

Astuce :
Vous pouvez cliquer sur le bouton Envoyer une notification à la page Afficher les résultats par réviseur de sorts finals pour envoyer une notification à tous les réviseurs de sorts finals.
