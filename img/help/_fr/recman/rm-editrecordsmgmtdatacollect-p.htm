Pour modifier les métadonnées Records Management d'un élément collecté :

  1. Sélectionnez Collections dans le menu Personnel.

  2. Sur la page Mes collections, activez la case à cocher de l'élément dont vous souhaitez modifier les métadonnées Records management.

  3. Cliquez sur Métadonnées Records management dans la liste déroulante Autres actions.

  4. Sur la page Métadonnées Records management, modifiez les paramètres.

  5. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Vous pouvez modifier les données Records management de tous les éléments d'une collection en sélectionnant la case à cocher Collection complète dans la vue de détail d'une collection.

 

Pour modifier les métadonnées Records management de plusieurs éléments :

  1. Dans la vue Parcourir d'un espace de travail, activez la case à cocher située en regard de chaque élément pour lequel vous souhaitez modifier les métadonnées Records management, puis cliquez sur Métadonnées Records management dans la liste déroulante Autres actions.

  2. Sur la page Métadonnées Records management, modifiez les paramètres.

  3. Cliquez sur le bouton Mettre à jour.
