Pour définir les adresses e-mail de notification de sorts finals :

  1. Dans l'espace de travail de Records Management, cliquez sur le lien Sorts finals.

  2. Sur la page Sorts finals, cliquez sur l'icône Fonctions pour une recherche de sorts finals, puis cliquez sur Propriétés et enfin sur Notification de sorts finals.

  3. Dans l'onglet Notification de sorts finals, sélectionnez un utilisateur ou un groupe Content Server auquel vous voulez que les notifications de sorts finals soient envoyées, saisissez des adresses e-mail supplémentaires dans le champ Autres destinataires, puis cliquez sur le bouton Mettre à jour.

 

Note :
Les adresses e-mail n'étant pas vérifiées lors de cette procédure, prenez bien soin de saisir des adresses e-mail valides dans le champ Autres destinataires. Si une adresse non valide est spécifiée dans le profil d'un utilisateur ou d'un membre de groupe Content Server, cette adresse ne sera pas prise en compte pour l'envoi de notifications.
