Pour exporter des codes de gestion des tables, des ISA ou des termes du classement RM :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Importer/Exporter.

  4. Sur la page Importer/Exporter de Records management, cliquez sur l'un des liens suivants :

    • Exporter les codes de gestion des tables

    • Exporter les ISA

    • Exporter les termes du classement RM

  5. Sur la page Exporter de l'élément, saisissez un nom de fichier dans le champ Nom du fichier à exporter.

  6. Cliquez sur le bouton Exporter.

 

Note :
Le fichier XML créé est enregistré dans le répertoire d'installation temporaire de l'espace Content Server ainsi que dans l'espace de travail personnel de l'utilisateur.
