To create an RSI schedule based on the Event Based rule type:

  1. Choose Records Management on the Enterprise menu.

  2. On the Records Management page, click the RSI link.

  3. On the RSI page, click the Functions icon for an RSI, and then choose Create Schedule.

  4. On the Add RSI Schedule page, click one of the following radio buttons:

    • RM Classifications, which lets you apply the RSI schedule to RM Classifications

    • Classified Objects, which lets you apply the RSI schedule to RM classified objects

  5. Click a stage code in the Stage drop-down list.

  6. Click the Event Based radio button in the Rule Type section.

  7. Specify any of the remaining settings, and then click the Continue button.

  8. On the RSI Schedule – Event Conditions page, choose any condition properties on the Property menu, and then set the event condition properties. Optionally, click a Boolean operator in the drop-down list for a property to specify a relationship between properties.

  9. Click the Continue button.

  10. On the RSI Schedule – Action to Perform page, click one of the following radio buttons to specify a default process:

    • None if you do not want to define a default process

    • Delete Electronic Format, and then click a status in the corresponding New Status drop-down list to delete the electronic format and change the status of the objects

    • Destroy to destroy the objects

    • Export to export the objects

    • Close, and then click a status in the corresponding New Status drop-down list to close the objects if they are RM classification object types

    • Change Status, and then click a status in the corresponding New Status drop-down list to change the status of the objects

    • Mark Official to remove all permissions except See and See Contents if they are RM classified objects

    • Reevaluate Storage Rules to mark the object for reevaluation when RM metadata included in the storage provider rules changes. Optionally, click a status in the corresponding New Status drop-down list to change the status of the objects.

       

      Note:
      The Reevaluate Storage Rules radio button is only available if the Content Move module has been installed.

       

  11. Click the Add Item button.
