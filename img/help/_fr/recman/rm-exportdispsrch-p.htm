Pour exporter les résultats des recherches de sorts finals :

  1. Cliquez sur l'icône Fonctions d'une recherche de sorts finals, puis sélectionnez Afficher les lots de résultats.

  2. Sur la page Résultats du sort final, cliquez sur l'icône Fonctions d'un lot de résultats de sorts finals, puis sélectionnez Exporter les métadonnées des résultats.

  3. Dans la section Exporter vers de la page Exporter les résultats de sort final, cliquez sur le bouton radio Fichier texte délimité ou Fichier Excel.

  4. Sélectionnez des métadonnées RM et d'objets physiques supplémentaires à inclure dans le rapport en mettant en surbrillance des champs dans les listes Disponibles et en ajoutant les métadonnées souhaitées dans les listes Sélectionnés à l'aide des boutons fléchés.

    Facultatif : pour inclure uniquement les RM et objets physiques que vous avez sélectionnés sur la page Résultats de recherche de sorts finals, activez la case à cocher Exporter uniquement les éléments sélectionnés.

  5. Saisissez un nom pour le rapport dans le champ Nom.

  6. Cliquez sur le bouton Parcourir Content Server, choisissez un emplacement dans Content Server dans lequel vous souhaitez exporter les résultats, puis cliquez sur le lien Sélectionner.

  7. Cliquez sur le bouton Exporter.

 

Astuce :
Les paramètres définis dans l'onglet Paramètres du sort final déterminent si seule la version actuelle de l'élément ou toutes les versions de l'élément doivent être exportées.
