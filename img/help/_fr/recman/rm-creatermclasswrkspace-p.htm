Pour créer un classement RM dans l'espace de travail de classement :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Classement.

  3. A la page Classement, cliquez sur Classement RM dans le menu Ajouter un élément.

  4. Saisissez un numéro de fichier dans le champ Numéro de fichier.

  5. Dans la section Date de création, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  6. Cliquez sur un code dans les listes déroulantes suivantes :

    • Etat

    • Essentiel

    • Support de stockage

  7. Dans la section Date du statut, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

    Facultatif : pour spécifier que le classement RM peut être appliqué aux objets Content Server, activez la case à cocher Sélectionnable.

    Facultatif : pour spécifier une période de contrôle du cycle de mise à jour des documents vitaux, cliquez sur une période dans la liste déroulante Période de cycle de mise à jour.

  8. Dans la section Groupe de gestionnaires de documents, cliquez sur l'icône Sélectionner un groupe pour sélectionner un groupe pour le classement RM. Il s'agira du groupe Groupe de gestionnaires de documents pour le classement RM.

  9. Spécifiez les autres paramètres.

  10. Cliquez sur le bouton Ajouter.

 

Note :
Vous ne pouvez pas créer de classement RM dans un dossier RM. Pour plus d'informations sur les dossiers RM, reportez-vous à la section Utilisation des dossiers et parties RM.

Une période de cycle de mise à jour est requise si l'administrateur considère que le code essentiel est déterminant.
