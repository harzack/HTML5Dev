Pour modifier un terme racine :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Termes du classement RM.

  3. Sur la page Termes du classement RM, cliquez sur le lien Informations d'un terme.

  4. Dans l'onglet Terme de la page Modifier, cliquez sur l'un des boutons radio suivants :

    • Préféré associe le terme à l'arbre de classement RM ;

    • Non préféré dissocie le terme de l'arbre de classement RM.

  5. Modifiez les autres paramètres.

  6. Cliquez sur le bouton Mettre à jour.
