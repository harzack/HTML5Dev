Pour copier un dossier Records Management :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Classement.

  3. Cliquez sur l'icône Fonctions du classement RM que vous souhaitez copier, puis cliquez sur Copier.

  4. Sur la page Copier le classement RM, cliquez sur le bouton Parcourir Content Server, naviguez jusqu'à l'emplacement où vous souhaitez copier le dossier RM, puis cliquez sur le lien Sélectionner.

  5. Cliquez sur le bouton Copier.
