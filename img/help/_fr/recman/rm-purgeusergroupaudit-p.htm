Pour éliminer l'audit d'utilisateur/de groupe :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records Management, cliquez sur le lien Audit des modifications d'utilisateur/de groupe.

  4. Sur la page Informations sur l'audit d'utilisateur/de groupe, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes de la section Eliminer avant la date.

  5. Cliquez sur le bouton Eliminer l'audit d'utilisateur/de groupe.

 

Note :
Si l'élimination des journaux d'audit est désactivée dans Content Server, l'option d'élimination du journal d'audit des utilisateurs et groupes n'est pas disponible.
