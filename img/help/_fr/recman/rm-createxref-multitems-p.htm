Pour ajouter un renvoi pour plusieurs éléments :

  1. Dans une collection, un ensemble de résultats de recherche ou dans la vue Parcourir d'un espace de travail, cochez la case située en regard de chaque élément auquel vous souhaitez appliquer un renvoi.

  2. Dans la liste déroulante Autres actions, cliquez sur Assigner un renvoi.

  3. Cliquez sur le bouton Parcourir Content Server, recherchez l'objet auquel vous souhaitez appliquer un renvoi, puis cliquez sur son lien Sélectionner.

  4. Sur la page Assigner un renvoi, sélectionnez un type de renvoi dans la liste déroulante Type de renvoi.

  5. Facultatif Saisissez une raison pour l'association dans le champ Commentaire.

  6. Cliquez sur le bouton Ajouter.
