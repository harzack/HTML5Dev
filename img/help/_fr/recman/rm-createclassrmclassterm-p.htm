Pour créer un classement pour un terme du classement RM :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Termes du classement RM.

  3. Sur la page Termes du classement RM, cliquez sur le lien d'un terme racine.

  4. Sur la page Plan des termes du classement RM, cliquez sur le lien Générer d'un terme racine.

  5. Spécifiez les autres paramètres.

  6. Cliquez sur le bouton Créer.
