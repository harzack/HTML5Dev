Vous pouvez ajouter des types de document dans Records management. Vous pouvez créer des types de document en fonction des besoins de votre entreprise. Par exemple, vous pouvez créer des types de document pour les factures, contrats, budgets, etc. Les types de document que vous créez peuvent être affectés à des documents gérés ou des éléments physiques. Lorsque vous ajoutez un type de document, vous pouvez spécifier un ISA ; vous pouvez choisir si l'ISA de ce type de document s'applique à l'élément créé. Pour plus d'informations sur les ISA, reportez-vous à la section Gestion des identificateurs de séries d'archives.

 

Note :
Lors de la création d'un objet géré, si vous lui assignez à la fois un classement RM et un type de document, et qu'un ISA est assigné à ce dernier, alors l'ISA du type de document est appliqué à l'objet géré. La seule exception à cette règle est si le type de document est utilisé par défaut, auquel cas c'est l'ISA du classement RM qui est appliqué à l'objet géré.
