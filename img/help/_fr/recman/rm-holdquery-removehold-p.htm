Pour supprimer une attente d'une requête d'attente :

  1. Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes.

  2. Sur la page Gestion des attentes, cliquez sur le menu Fonctions d'une attente, cliquez sur Attente avec recherche, puis sur Critères de l'attente.

  3. Sur la page de gestion d'attente avec recherche, cliquez sur le lien Enlever l'attente de la colonne Actions pour les résultats de la requête dont vous souhaitez retirer l'attente.

  4. Dans la boîte de dialogue qui s'affiche, cliquez sur OK.

 

Note :
Le lien Enlever l'attente n'apparaît pas lorsque l'action Programmé est activée. Pour retirer manuellement les attentes des éléments d'une requête d'attente, vous devez d'abord la déprogrammer.

La même attente peut toujours être appliquée pour d'autres requêtes d'attente aux éléments renvoyés dans les résultats de recherche de cette requête d'attente.
