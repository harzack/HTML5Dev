Pour programmer une requête d'attente automatisée :

  1. Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes.

  2. Sur la page Gestion des attentes, cliquez sur le menu Fonctions d'une attente, cliquez sur Attente avec recherche, puis sur Critères de l'attente.

  3. Sur la page de gestion d'attente avec recherche, cliquez sur le bouton radio Oui de la colonne Actions pour les résultats de la requête que vous souhaitez exécuter automatiquement.

  4. Saisissez le motif de modification dans le champ Commentaires, puis cliquez sur Mettre à jour.
