Tableau 1-7 : Pour spécifier des critères de recherche de sorts finals supplémentaires :

Pour...
 Dans l'onglet Infos spécifiques de la recherche de sorts finals, effectuez l'une des opérations suivantes :
Utiliser uniquement des programmes de conservation maximale Activez la case à cocher Conservation maximale.
Assigner des ISA à la recherche de sorts finals Cliquez sur Sélectionner un ISA dans la liste déroulante ISA, activez la case à cocher pour chaque ISA que vous souhaitez inclure aux critères de recherche sur la page Sélectionner un ISA, puis cliquez sur le bouton Envoyer.
 

Note :
Pour inclure tous les ISA dans la recherche de sorts finals, ne renseignez pas le champ ISA.

Pour enlever des ISA de la recherche de sorts finals, mettez le ou les ISA concernés en surbrillance, puis cliquez sur le bouton Enlever.

 
Spécifier un processus par défaut
 

Note :
La spécification d'un processus par défaut est facultative pour les processus de suppression d'un format électronique, d'exportation et de fermeture. En revanche, elle est obligatoire pour le processus de modification du statut. Pour obtenir une description des processus par défaut, reportez-vous à la section Actions liées au programme ISA.

 
 Cliquez sur un bouton radio dans la section Processus par défaut. Si vous cliquez sur les boutons radio Supprimer le format électronique ou Changer le statut, cliquez sur un statut dans la liste déroulante Nouveau statut.
Définir la portée de la recherche dans un autre recueil Dans la section Recueil, cliquez sur le bouton Parcourir l'espace Entreprise, accédez au recueil dans lequel vous souhaitez lancer la recherche, puis cliquez sur le lien Sélectionner correspondant.
Exclure un recueil des résultats de la recherche Désactivez la case à cocher Inclure le recueil dans les résultats de recherche.
Limiter la recherche à des types d'objets Content Server spécifiques Cliquez sur un type d'objet dans la liste déroulante Type d'objet. Pour sélectionner plusieurs types d'objets, maintenez la touche CTRL enfoncée tout en cliquant sur les types d'objet que vous souhaitez sélectionner.
Autoriser les lots de résultats multiples dans les recherches de sorts finals Activez la case à cocher Lots de résultats multiples.
Afficher automatiquement les résultats d'une recherche de sorts finals Activez la case à cocher Résultats de la recherche de processus automatique.
Utiliser une requête de recherche avancée enregistrée dans le cadre du processus de recherche de sorts finals Dans la section Requête de recherche avancée, cliquez sur le bouton Parcourir, accédez à la requête de recherche avancée que vous souhaitez inclure, puis cliquez sur le lien Sélectionner correspondant. Ce paramètre apparaît uniquement si l'administrateur réseau l'a activé.
 

Note :
Vous ne pouvez pas sélectionner à la fois les types d'objets de classement RM (classement RM, dossier RM et partie RM) et les types d'objet géré en tant que critères de recherche de sorts finals.

Si vous ne limitez pas la recherche à des types d'objet Content Server spécifiques, le résultat comprend les objets classés par défaut.
