Pour créer un renvoi ou une version de document :

  1. Cliquez sur l'icône Fonctions d'un objet géré et sélectionnez Propriétés, puis Renvoi.

  2. Dans l'onglet Renvoi de la page Propriétés de l'objet, cliquez sur Nouveau renvoi ou sur Nouvelle version de document dans la liste déroulante Ajouter : Renvoi.

     

    Note :
    Si l'objet que vous ajoutez en tant que version est déjà une version d'un autre objet, l'option Nouvelle version de document ne s'affiche pas. En outre, cette option ne s'affiche que si elle a été activée par l'administrateur système.

     

  3. Cliquez sur le bouton Parcourir Content Server, recherchez l'objet Content Server à associer au renvoi, puis cliquez sur le lien Sélectionner correspondant.

  4. Sur la page Ajouter un renvoi, cliquez sur un type de renvoi dans la liste déroulante Type de renvoi.

     

    Note :
    Si vous ajoutez une version de document et que l'administrateur a sélectionné un type d'enregistrement par défaut, ce champ est renseigné automatiquement et ne peut pas être modifié.

     

  5. Facultatif Saisissez une raison pour l'association dans le champ Commentaire.

  6. Cliquez sur le bouton Ajouter.
