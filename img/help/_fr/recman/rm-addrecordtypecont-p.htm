Pour ajouter un type de document :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien Types de document.

  3. Sur la page Types de document, cliquez sur Type de document dans le menu Ajouter un élément.

  4. Sur la page d'ajout des types de document, spécifiez les paramètres.

  5. Cliquez sur le bouton Ajouter.
