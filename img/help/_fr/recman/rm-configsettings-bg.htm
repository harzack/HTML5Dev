Vous pouvez modifier vos paramètres personnels de Records Management. Vous pouvez choisir de masquer les onglets Attentes, Détails du document et Renvoi depuis la page Propriétés des éléments gérés. Vous pouvez également sélectionner un Classement de gestion des enregistrements (classement RM) par défaut pour les éléments gérés. Le classement RM par défaut indiqué sur la page des paramètres s'affiche automatiquement sur la page Créer des éléments gérés que vous ajoutez à Content Server. Vous pouvez affecter un classement RM à un élément géré au moment où vous le créez ou vous pouvez affecter des classements RM à des éléments gérés existants.

Les classements RM par défaut peuvent être affectés à des groupes et les éléments peuvent avoir des classements RM par défaut. Les éléments héritent d'un classement par défaut reposant sur l'ordre suivant :

  • Le classement RM indiqué sur la page des paramètres personnels de l'utilisateur. Si ce paramètre n'est pas précisé, le classement RM suivant est utilisé :

  • Le classement RM indiqué dans le service de l'utilisateur. Si ce paramètre n'est pas précisé, le classement RM suivant est utilisé :

  • Le classement RM indiqué par le type d'objet de la page Objets gérés. Si ce paramètre n'est pas précisé, le classement RM suivant est utilisé :

  • Le classement RM indiqué comme classement global par défaut.
