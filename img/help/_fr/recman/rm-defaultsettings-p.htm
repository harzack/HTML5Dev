Pour modifier les paramètres par défaut de Records Management :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Paramètres par défaut.

  4. Dans la zone Paramètres par défaut, cliquez sur Sélectionner un classement, accédez au classement que vous voulez définir comme étant le classement par défaut, puis cliquez sur son lien Sélectionner.

  5. Dans la zone Emplacement par défaut pour les éléments créés dans l'espace de travail Classement, cliquez sur le bouton Parcourir Content Server, accédez à l'emplacement de Content Server dans lequel vous voulez créer les éléments, puis cliquez sur son lien Sélectionner.

  6. Dans la zone Titrage du classement RM, sélectionnez le bouton approprié afin d'activer le titrage et spécifiez les options d'ajout de titres.

  7. Dans la zone Nombre de résultats de recherche à renvoyer, entrez le nombre de résultats que vous voulez voir apparaître par page dans la zone de texte Nombre de résultats.

  8. Dans la zone Séparateur de numéro de fichier, sélectionnez le caractère par défaut qui apparaît dans le numéro de fichier dans la liste Séparateur.

  9. Dans la zone Mode d'affichage du classement RM, cliquez sur le bouton liste ou intégré pour spécifier la façon dont les classements RM s'affichent lorsque vous parcourez le volume des classements.

  10. Dans la zone Parcourir l'ISA et les attentes, cliquez sur le bouton Navigation standard ou Recherche avancée pour spécifier la façon dont les utilisateurs parcourront les ISA ou les attentes.

  11. Dans la zone Masquer les icônes, cliquez sur le bouton Oui ou Non pour choisir de masquer ou non les icônes Officiel et Eléments classés durant la navigation.

  12. Dans la section Actions de gestion des documents en vue Parcourir, activez la case à cocher située en regard de chaque fonction de gestion de documents pour laquelle vous souhaitez activer les actions de sélection multiple.
