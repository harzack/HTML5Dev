Pour configurer les champs Records Management :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Champs RM.

  4. Dans la section Configurer les champs Détails des documents de la page Configurer les champs Records management, sélectionnez les cases à cocher des paramètres à afficher sur la page Ajouter un élément ou dans l'onglet Détails du document.

  5. Dans la section Configurer les champs de classement RM requis, sélectionnez la case à cocher des champs de classement RM à définir en tant que champ requis.

  6. Dans la section Inclure le renvoi sur la page Ajouter un élément, sélectionnez la case à cocher Renvoi pour permettre aux utilisateurs d'ajouter un renvoi sur la page Ajouter un élément lors de l'ajout d'un objet géré à Content Server.

  7. Cliquez sur le bouton Envoyer.
