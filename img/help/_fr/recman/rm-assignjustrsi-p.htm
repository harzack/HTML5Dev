Pour assigner une justification à un ISA :

  1. Dans le menu Entreprise, sélectionnez Records management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur le lien d'un ISA.

  4. Sur la page Infos spécifiques pour, cliquez sur l'onglet Justifications.

  5. Sur la page Justifications, cliquez sur Justification dans le menu Attribuer un nouveau.

  6. Sur la page Recherche de justifications, cochez l'une des options de la liste déroulante :

    • Juridiction permet d'effectuer une recherche sur le champ juridiction des justifications ;

    • Description permet d'effectuer une recherche sur le champ description des justifications ;

    • Citation permet d'effectuer une recherche sur le champ citation des justifications.

  7. Cliquez sur le bouton Rechercher.

  8. Dans la liste des résultats de recherche, activez les cases à cocher en regard de toutes les justifications que vous souhaitez assigner.

  9. Cliquez sur le bouton Assigner des justifications, puis sur Mettre à jour.
