Pour créer une liste de sélection de classement à partir de classements recherchés :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Listes de sélection de classement.

  4. Sur la page Sélectionner un groupe pour la liste de sélection, spécifiez des critères de recherche, puis cliquez sur le bouton Rechercher.

  5. Cliquez sur le lien Sélectionner d'un groupe.

  6. Sur la page Liste de sélection de groupe, cliquez sur Parcourir les classements dans la liste déroulante Sélectionner un classement.

  7. Sur la page Parcourir le module classements, cliquez sur le lien Sélectionner d'un classement.

  8. Cliquez sur le bouton Ajouter à la liste de sélection.
