Pour créer une recherche programmée de sorts finals :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Sorts finals.

  3. Sur la page Sort final, cliquez sur Recherche de sorts finals dans le menu Ajouter : Elément.

  4. Sur la page Recherche de sorts finals, entrez un nom pour la recherche de sorts finals dans le champ Nom.

  5. Facultatif : fournissez une description pour la recherche de sorts finals, ou modifiez l'emplacement où la recherche de sorts finals va être créée.

  6. Cliquez sur le bouton Ajouter.

  7. Sur la page Sorts finals, cliquez sur l'icône Fonctions pour une recherche de sorts finals, choisissez Propriétés puis Infos spécifiques.

  8. Sur la page Propriétés pour une recherche de sorts finals, cliquez sur une phase dans la liste déroulante Phase.

  9. Spécifiez vos critères de recherche.

  10. Dans la section Spécifications de date calculée - Sélectionnez Manuel ou Programmé, cliquez sur le bouton radio Programmé.

  11. Dans la section Régler la récurrence du programme, cliquez sur l'un des boutons radio suivants :

    • Quotidien permet de créer une recherche de sorts finals quotidienne et d'indiquer le nombre de jours dans le champ.

    • Hebdomadaire permet de créer une recherche de sorts finals hebdomadaire et d'indiquer le nombre de jours dans le champ, puis de sélectionner la case du jour de la semaine où la recherche doit être répétée.

    • Mensuelle permet de créer une recherche de sorts finals mensuelle et d'indiquer une date et le nombre de mois dans les champs appropriés.

    • Annuel permet de créer une recherche de sorts finals annuelle et d'indiquer un mois à l'aide d'une liste déroulante et une date dans le champ approprié.

  12. Dans la section Régler l'intervalle de dates flottantes par rapport à la date programmée, cliquez sur + ou - dans les listes déroulantes, puis saisissez le nombre d'années, de mois et de jours dans les champs correspondants.

  13. Cliquez sur le bouton Ajouter.

 

Note :
Pour l'intervalle de dates flottantes, une seule date est requise; cependant, vous pouvez définir les dates de début et de fin.

 
 

Astuce :
Si vous souhaitez laisser la date de début modifiable, activez la case à cocher Date de début modifiable. Si vous souhaitez laisser la date de fin modifiable, activez la case à cocher Date de fin modifiable.
