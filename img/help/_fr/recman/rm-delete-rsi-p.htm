Pour supprimer un ISA :

  1. Dans le menu Entreprise, cliquez sur Records management.

  2. Dans l'espace de travail de Records Management, cliquez sur le lien ISA.

  3. Sur la page ISA, cochez la case située en regard de chaque ISA que vous souhaitez supprimer, puis cliquez sur le bouton Supprimer.

  4. Cliquez sur OK dans la boîte de dialogue qui s'affiche.

 

Astuce :
Vous pouvez également supprimer les ISA en cliquant sur l'icône Fonctions de chaque ISA, puis sur Supprimer.
