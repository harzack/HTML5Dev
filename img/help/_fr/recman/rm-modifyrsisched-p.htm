Tableau 1-6 : Pour modifier un programme ISA :

Pour... Cliquez sur l'icône Fonctions d'un ISA, choisissez Afficher les programmes, puis procédez comme suit...
Modifier un programme ISA
 Cliquez sur le lien Modifier de la section Action, modifiez les champs que vous souhaitez et cliquez sur le bouton Continuer pour passer à la page suivante, puis cliquez sur le bouton Mettre à jour.
 

Note :
Lorsque vous modifiez un programme ISA, vous pouvez fournir une raison expliquant pourquoi vous avez procédé à la modification dans le champ Raison de la mise à jour. Les informations saisies dans ce champ sont suivies dans l'historique d'expertise.

 
Supprimer un programme ISA Dans la section Action, cliquez sur le lien Supprimer.
