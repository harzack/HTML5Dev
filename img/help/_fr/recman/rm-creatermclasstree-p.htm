Pour créer un classement RM en tant que partie intégrante d'un arbre de classement :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Classement.

  3. Sur la page Classement, cliquez sur le lien d'un arbre de classement.

  4. Sur la page Arbre de classement, cliquez sur Classement RM dans le menu Ajouter : Elément.

  5. Saisissez un numéro de fichier dans le champ Numéro de fichier.

  6. Dans la section Date de création, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes.

  7. Cliquez sur un code dans les listes déroulantes suivantes :

    • Etat

    • Essentiel

    • Support de stockage

  8. Dans la section Date du statut, cliquez sur un mois, un jour et une année dans les listes déroulantes correspondantes. Facultatif : pour spécifier que le classement RM peut être appliqué aux objets Content Server, activez la case à cocher Sélectionnable. Facultatif : pour spécifier une période de contrôle du cycle de mise à jour des documents vitaux, cliquez sur une période dans la liste déroulante Période de cycle de mise à jour.

  9. Spécifiez les autres paramètres.

  10. Cliquez sur le bouton Ajouter.

 

Note :
Une période de cycle de mise à jour est requise si l'administrateur considère que le code essentiel est déterminant.
