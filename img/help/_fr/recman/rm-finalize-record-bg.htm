L'onglet Finaliser l'enregistrement permet de marquer un élément comme officiel. Cet onglet propose des actions supplémentaires qui peuvent être appliquées aux éléments lors de l'exécution de la fonction Finaliser l'enregistrement. Les actions suivantes sont disponibles :

  • Eliminer toutes les versions, qui exécute le processus de suppression des versions précédentes d'un élément.

  • Détruire les éléments verrouillés ou transformés en génération, qui détruit les éléments verrouillés ou pour lesquels il existe des générations. Les générations de l'élément sont également détruites lorsque cette option est activée.

     

    Note :
    Vous ne pouvez activer le paramètre Détruire les éléments verrouillés ou transformés en génération que si le paramètre Eliminer toutes les versions est également activé.

    Si un utilisateur n'est pas autorisé à éliminer des générations, ou si une génération est en attente, l'action Finaliser l'enregistrement échoue. Un message affiche le motif d'échec de l'action.

     

  • Mettre à jour la date de document, qui remplace la date de document existante par la date du jour.

  • Mettre à jour le statut, qui vous permet de choisir un statut pour l'élément.

    Lorsque le paramètre Mettre à jour le statut est mis à jour à l'aide de la fonction Finaliser l'enregistrement, les autorisations de la liste de contrôle d'accès de l'élément sont également mises à jour si le code Statut est configuré pour la mise à jour de ces autorisations. Pour plus d'informations sur les autorisations de la liste de contrôle d'accès, reportez-vous à la rubrique Autorisations.

  • Créer la présentation, qui crée des présentations des éléments versionables au moment de l'exécution de l'action Finaliser l'enregistrement. Les présentations des éléments gérés sont créées sur la base de toutes les règles de présentation applicables, à l'exception des règles désignées globalement. Si des erreurs se produisent lors de la création de présentations avec la fonctionnalité Finaliser l'enregistrement, elles sont consignées dans le fichier journal des erreurs de présentation.

  • Interdire la fonctionnalité Finaliser l'enregistrement pour les documents de version secondaire, qui vous permet d'interdire la finalisation d'un document dont la version la plus récente est une version secondaire. Cette option est désactivée par défaut. Activez cette option uniquement si vous ne souhaitez finaliser que des documents dont la version la plus récente est une version principale.

 

Note :
La fonctionnalité de l'onglet Finaliser l'enregistrement est contrôlée par des autorisations d'accès. Pour vérifier le niveau minimum d'autorisations nécessaires pour exécuter les fonctions de cet onglet, cliquez sur le lien Autorisations d'accès. Pour plus d'informations sur les autorisations d'accès, reportez-vous à Définition des restrictions d'accès.

 

Vous pouvez également finaliser des enregistrements dans la vue Parcourir d'un espace de travail, des résultats de recherche ou une collection à partir de la liste Autres actions..., ou à l'aide de l'option Finaliser l'enregistrement du menu Fonctions d'un élément (si celui-ci est disponible).

 

Note :
La fonction Finaliser l'enregistrement est disponible uniquement dans la vue Parcourir, les résultats de recherche et les collections, et dans le menu Fonctions d'un élément si celui-ci a été activé par l'administrateur de Records Management.
