Pour assigner une attente aux éléments d'une collection :

  1. Dans la collection, activez la case à cocher en regard de chaque élément que vous souhaitez mettre en attente.

  2. Cliquez sur Appliquer une attente dans la liste déroulante Autres actions.

  3. Sur la page Rechercher les attentes, spécifiez le nom de l'attente dans le champ Rechercher un nom d'attente pour, puis cliquez sur le bouton Rechercher.

  4. Cliquez sur le lien Sélectionner correspondant à une attente.

 

Astuce :
Vous pouvez assigner une attente à tous les éléments d'une collection en sélectionnant la case à cocher Collection complète dans la vue de détail d'une collection.

Pour créer une nouvelle attente et l'assigner à un élément d'une collection, cliquez sur Créer une nouvelle attente dans la liste déroulante Appliquer : Attente figurant dans la section Rechercher les attentes de la page Rechercher les attentes.
