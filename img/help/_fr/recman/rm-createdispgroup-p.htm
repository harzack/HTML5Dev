Pour créer un groupe de sort final :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Sorts finals.

  3. Sur la page Sorts finals, cliquez sur Groupe de sort final dans le menu Ajouter un élément.

  4. Sur la page Groupe de sort final, spécifiez les paramètres.

  5. Cliquez sur le bouton Ajouter.

 

Astuce :
Vous pouvez créer un groupe de sort final au sein d'un autre groupe de ce type en cliquant sur le lien d'un groupe, puis sur Groupe de sort final dans le menu Ajouter un élément.
