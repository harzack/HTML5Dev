Attention
Les éléments dont la décision de révision est définie sur En attente ou Rejeté ne seront pas traités.

 

Pour traiter des objets sur la page Résultats du sort final :

  1. Procédez de l'une des manières suivantes :

    • Cliquez sur l'icône Fonctions d'un sort final, puis sélectionnez Afficher les lots de résultats.

    • Cliquez sur l'icône Fonctions d'un lot de sorts finals, puis sélectionnez Afficher tous les résultats.

  2. Facultatif Pour filtrer les résultats des recherches de sorts finals, cliquez sur le bouton Filtrer les résultats de la recherche, saisissez les critères de recherche, puis cliquez sur le bouton OK.

  3. Sur la page Résultats du sort final, cochez la case correspondant à chaque élément à modifier, effectuez l'une des actions suivantes, puis cliquez sur OK :

    • Pour appliquer une acquisition aux éléments, cliquez sur le bouton Appliquer une acquisition, sélectionnez un code d'acquisition dans la liste déroulante, puis cliquez sur le bouton radio Eléments sélectionnés sur cette page ou Eléments sélectionnés dans l'ensemble de résultats complet.

    • Pour changer l'action par défaut qui sera exécutée pour les éléments sélectionnés, cliquez sur le bouton Changer les actions, sélectionnez le bouton radio de l'action par défaut de votre choix, puis cliquez sur le bouton radio Eléments sélectionnés sur cette page ou Eléments sélectionnés dans l'ensemble de résultats complet dans la section Appliquer à.

    • Pour traiter les éléments sélectionnés, cliquez sur le bouton Exécuter des actions, puis cliquez sur le bouton radio Eléments sélectionnés sur cette page ou Eléments sélectionnés dans l'ensemble de résultats complet.

    • Pour déplacer les éléments sélectionnés, cliquez sur le bouton Déplacer, sélectionnez l'emplacement de destination, spécifiez des critères supplémentaires, puis cliquez sur Déplacer.

       

      Note :
      Le bouton Déplacer apparaît uniquement s'il a été activé sur la page Paramètres de sort final. Contrairement aux autres éléments de Content Server, les éléments d'un lot de résultats ne sont pas supprimés du lot de résultats lorsque vous les déplacez vers un autre emplacement.

       
 

Note :
Il est impossible de détruire les types d'éléments suivants : Mon espace de travail (142), Volume Workflow (161), EL TypeVolume (820) et ELVolume (822).

 
 

Astuce :
Vous pouvez effectuer toutes les actions pour les résultats d'une recherche de sort final. Pour ce faire, cliquez sur le bouton Exécuter les actions, puis sélectionnez l'option permettant d'exécuter les actions sur toutes les pages. Vous pouvez également cliquer sur le menu Fonctions d'une recherche de sort final, puis sélectionner Exécuter les actions. A la fin du processus, l'utilisateur reçoit un e-mail incluant toutes les erreurs qui se sont produites pendant le processus, ainsi qu'un lien vers le fichier journal. L'option Afficher le journal des erreurs s'affiche également dans le menu Fonctions du lot de résultats.

Pour inclure les présentations des éléments dans le fichier d'exportation, cliquez sur l'icône Fonctions d'une recherche de sort final, cliquez sur Paramètres, puis cochez la case Inclure la(les) présentation(s).
