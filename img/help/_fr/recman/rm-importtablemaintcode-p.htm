Pour importer des codes de gestion des tables, des ISA ou des termes du classement RM :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Importer/Exporter.

  4. Sur la page Importer/Exporter de Records management, cliquez sur l'un des liens suivants :

    • Importer des codes de gestion des tables

    • Importer des ISA

    • Importer les termes du classement RM

  5. Si vous importez des codes de gestion des tables, procédez comme suit :

    • Sur la page Importer, cliquez sur le bouton Parcourir dans la section Importer des codes XML Records management, accédez au fichier XML, puis cliquez sur le bouton Ouvrir.

  6. Si vous importez des ISA, procédez comme suit :

    • Sur la page Importer, cliquez sur le bouton Parcourir dans la section Importer des ISA, accédez au fichier XML, puis cliquez sur le bouton Ouvrir.

    • Activez la case Mise à jour des ISA et programmes existants pour mettre à jour les ISA et programmes lors de l'importation.

    • Activez la case Supprimer les programmes qui n'existent pas dans le fichier d'importation XML pour supprimer les fichiers qui ne figurent pas dans le fichier XML que vous importez.

  7. Si vous importez des termes du classement RM, procédez comme suit :

    • Sur la page Importer, cliquez sur le bouton Parcourir dans la section Importer les termes du classement RM XML, accédez au fichier XML, puis cliquez sur le bouton Ouvrir.

    • Dans la liste déroulante Date de démarrage, cliquez sur la date d'importation du fichier.

    • Saisissez l'origine des termes dans le champ Source. Vous pouvez par exemple saisir le nom de l'utilisateur qui a créé le fichier.

  8. Cliquez sur le bouton Importer.
