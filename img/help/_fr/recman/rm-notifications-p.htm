Pour activer des notifications :

  1. Dans le menu Entreprise, cliquez sur Records Management.

  2. Sur la page Administration de Records Management, cliquez sur le lien Paramètres du système.

  3. Sur la page Paramètres du système, cliquez sur l'onglet Notifications.

  4. Dans la zone Notification activée, cliquez sur le bouton Oui pour activer l'envoi de notifications lorsque les documents doivent être révisés.

  5. Dans la zone Destinataire de l'e-mail, cliquez sur un bouton pour déterminer si les notifications sont envoyés au gestionnaire de documents ou au propriétaire des documents.

  6. Cliquez sur le bouton Envoyer.
