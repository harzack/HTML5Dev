Pour créer un code :

  1. Dans le menu Entreprise, sélectionnez Records management.

  2. Sur la page Records Management, cliquez sur le lien Administration de Records Management.

  3. Sur la page Administration de Records management, cliquez sur le lien Gestion des tables.

  4. Sur la page Gestion de table de Records management, cliquez sur le lien de la table à laquelle vous souhaitez ajouter un code.

  5. Cliquez sur un type de code dans le menu Ajouter un élément.

  6. Renseignez le champ Code.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Envoyer.

 

Astuce :
Vous pouvez également créer un code de maintenance de table sur la page où elle sera utilisée, en cliquant sur <Créer un nouveau code> dans la liste déroulante du code.
