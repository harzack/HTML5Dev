Pour créer une requête de recherche d'attentes :

 

Note :
Vous devez disposer de l'autorisation Ajouter un élément pour créer une requête à partir de l'espace de travail Gestion des attentes.

 

  1. Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes.

  2. Sur la page Gestion des attentes, cliquez sur le menu Fonctions d'une attente, cliquez sur Attente avec recherche, puis sur Critères de l'attente.

  3. Sur la page de gestion d'attente avec recherche, cliquez sur le bouton Créer une requête.

  4. Sur la page Recherche avancée, sélectionnez les paramètres de recherche à utiliser pour la recherche des éléments, puis cliquez sur le bouton Enregistrer en tant que Requête pour attente dans la zone d'options d'enregistrement.

  5. Sur la page d'enregistrement de la requête de recherche, saisissez un nom pour la requête, entrez d'éventuelles informations complémentaires, puis cliquez sur le bouton Ajouter.

 

Astuce :
Les critères que vous saisissez pour une requête de recherche d'attentes sont les mêmes que ceux utilisés pour une requête de recherche standard dans Content Server. Pour plus d'informations sur les requêtes de recherche, reportez-vous à Recherche à l`aide de requêtes de base.

Lorsque vous avez créé une requête de recherche d'attentes et que vous l'avez enregistrée, l'onglet Critères d'attente réapparaît.

La recherche n'a pas lieu automatiquement lorsque vous créez une requête. Vous devez cliquer sur le bouton Rechercher dans l'onglet Infos spécifiques pour exécuter la recherche ou cliquer sur le menu Fonctions de la requête d'attente, puis sur Exécuter.
