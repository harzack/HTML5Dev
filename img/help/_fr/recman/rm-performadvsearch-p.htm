Pour effectuer une recherche avancée d'ISA :

  1. Dans la liste déroulante Sélectionner un ISA, cliquez sur Rechercher un ISA.

  2. Sur la page Sélectionner un ISA, procédez comme suit :

    • Dans le champ ISA, saisissez une valeur d'ISA.

    • Cliquez sur Mot clé dans la liste déroulante Attributs, puis saisissez le mot clé dans le champ Mot clé.

    • Cliquez sur Statut dans la liste déroulante Attributs, puis cliquez sur un code de statut ISA dans la liste déroulante Statut.

     

    Note :
    Lorsque vous cliquez sur un attribut de la liste déroulante Attributs, un champ de recherche correspondant à l'attribut s'affiche et Content Server enlève le nom de l'attribut de la liste déroulante Attributs.

     

    Facultatif : pour définir la relation entre plusieurs valeurs d'attributs, cliquez sur un bouton Doubler  , puis sur un opérateur booléen de la liste déroulante.

    Facultatif : pour enlever le champ de recherche d'un attribut, cliquez sur le bouton Enlever  en regard du champ de recherche concerné.

  3. Cliquez sur le bouton Rechercher.

  4. Sur la page Sélectionner un ISA, cliquez sur le lien Sélectionner d'un ISA.
