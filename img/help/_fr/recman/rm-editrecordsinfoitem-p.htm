Pour modifier les informations sur les documents d'un élément :

  1. Cliquez sur l'icône Fonctions d'un objet géré, choisissez Propriétés, puis Détails du document.

  2. Dans l'onglet Détails du document de la page Propriétés de l'objet, modifiez les paramètres souhaités.

  3. Cliquez sur le bouton Mettre à jour.
