Pour retirer des classements RM de plusieurs éléments :

  1. Dans une collection, des résultats de recherche ou la vue Parcourir d'un espace de travail, cochez la case correspondant à chaque élément dont vous souhaitez supprimer le classement RM.

  2. Dans le menu Autres actions, cliquez sur Enlever le classement.

  3. Dans la fenêtre Enlever des classements, cliquez sur Classement RM dans la liste déroulante Classer, puis sélectionnez le classement RM à retirer.

  4. Cliquez sur le bouton Envoyer.
