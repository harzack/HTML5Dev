Pour appliquer une attente à une requête d'attente :

  1. Dans l'espace de travail de Records management, cliquez sur le lien Gestion des attentes.

  2. Sur la page Gestion des attentes, cliquez sur le menu Fonctions d'une attente, cliquez sur Attente avec recherche, puis sur Critères de l'attente.

  3. Sur la page de gestion d'attente avec recherche, cliquez sur le lien Appliquer l'attente de la colonne Actions pour les résultats de la requête à laquelle vous souhaitez appliquer l'attente.

     

    Note :
    Le lien Appliquer l'attente n'apparaît pas lorsque l'action Programmé est activée.

     

  4. Dans la boîte de dialogue qui s'affiche, cliquez sur OK.
