Pour créer une recherche d'acquisitions :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. A la page Records Management, cliquez sur le lien Sorts finals.

  3. Sur la page Sort final, cliquez sur Recherche d'acquisitions dans le menu Ajouter : Elément.

  4. Sur la page Recherche d'acquisitions, cliquez sur un code d'acquisition dans la liste déroulante Acquisition. Pour définir la portée de la recherche dans un autre recueil, cliquez sur le bouton Parcourir l'espace Entreprise, accédez au recueil dans lequel vous souhaitez lancer une recherche, puis cliquez sur son lien Sélectionner (facultatif).

  5. Facultatif Pour limiter la recherche à un seul type d'objet Content Server, cliquez sur un type d'objet dans la liste déroulante Type d'objet (facultatif).

  6. Renseignez les autres champs.

  7. Cliquez sur le bouton Ajouter.

 

Astuce :
Vous pouvez ajouter une recherche d'acquisitions à un groupe de sorts finals en cliquant sur le lien d'un groupe de ce type, puis sur Recherche d'acquisitions dans le menu Ajouter un élément.
