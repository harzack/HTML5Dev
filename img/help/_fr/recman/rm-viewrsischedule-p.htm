Pour afficher un programme ISA :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Sur la page ISA, cliquez sur l'icône Fonctions d'un ISA, puis sélectionnez Afficher le programme.

 

Astuce :
Vous pouvez également créer un programme pour l'ISA en cliquant sur le bouton Créer un programme. Pour plus d'instructions sur la création du programme, reportez-vous à la rubrique Pour créer un programme ISA.
