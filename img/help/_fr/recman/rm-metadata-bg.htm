Vous pouvez modifier les métadonnées Records management pour les éléments d'une collection ainsi que pour plusieurs éléments lorsque vous êtes en vue Parcourir. Pour plus d'informations sur les collections, consultez l'aide en ligne de Content Server. Lorsque vous modifiez les métadonnées, vous pouvez choisir d'enlever l'ISA ou l'officialisation de l'élément.
