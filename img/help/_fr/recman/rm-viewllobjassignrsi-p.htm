Pour afficher des objets Content Server assignés à un ISA :

  1. Dans le menu Entreprise, sélectionnez Records Management.

  2. Sur la page Records management, cliquez sur le lien ISA.

  3. Cliquez sur l'icône Fonctions d'un ISA, puis sur Afficher les objets Content Server.

 

Note :
En l'absence d'éléments Content Server, le message suivant apparaît : Aucun élément Content Server pour cet ISA.
