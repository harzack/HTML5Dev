Le gestionnaire de documents ou les utilisateurs disposant des autorisations requises peuvent créer des attentes dans le recueil Gestion des attentes ou à la volée, à partir de l'onglet de propriétés des attentes. Les attentes reposent sur les codes créés. Pour en savoir plus, voir Gestion des codes.

Lorsque vous créez des attentes, vous pouvez spécifier des commentaires et une date à laquelle l'attente doit être retirée. Une fois que vous avez créé des attentes, vous pouvez les modifier dans le recueil Gestion des attentes. Modifier une attente signifie l'éditer, l'interrompre, la réactiver ou la supprimer. Lorsque vous interrompez une attente, celle-ci devient inactive. Autrement dit, les métadonnées de l'attente s'appliquent encore aux éléments, mais les éléments auxquels l'attente est appliquée peuvent être supprimés.

Vous pouvez exporter le contenu d'une attente vers un répertoire spécifié par votre administrateur. Vous pouvez exporter la version actuelle uniquement ou toutes les versions de l'attente. Si vous exportez plus d'une version d'un document, les versions et numéros de version correspondants s'affichent sous le document ; si seule la version actuelle est exportée, le document est affiché. Les versions sont triées par ordre décroissant de la plus récente à la plus ancienne. Le fichier index.html du dossier d'exportation est une liste des attentes et contenus exportés. Vous pouvez cliquer sur un nom de fichier pour être redirigé vers le fichier exporté actuel. Si l'objet géré contient des métadonnées multilingues dans les champs Nom et Description, le contenu de ces champs est exporté de la même manière que les métadonnées RM. Pour plus d'informations sur l'éditeur de métadonnées multilingues, consultez Utilisation des éléments généraux.

Utilisation des groupes d'attentes

Vous pouvez organiser les attentes en créant des groupes d'attentes. Tout comme les attentes, les groupes d'attentes peuvent être ajoutés sur la page Gestion des attentes de l'espace de travail de Records management. Les groupes d'attentes ne peuvent être ajoutés qu'au premier niveau de la page Gestion des attentes, et ne peuvent contenir que des attentes. En d'autres termes, seul un niveau de regroupement est autorisé par groupe d'attentes. Vous ne pouvez pas déplacer ou copier des attentes existantes dans des groupes d'attentes.

Une fois que vous avez créé un groupe d'attentes et que celui-ci contient des attentes, vous pouvez exporter tous les documents associés au groupe d'attentes.

 

Note :
Le mode d'affichage des attentes dépend du mode de navigation défini par votre administrateur système (Navigation standard ou Recherche avancée). Si le mode Recherche avancée est sélectionné et qu'un utilisateur accède à la page Gestion des attentes, tous les groupes d'attentes s'afficheront mais l'utilisateur devra lancer une recherche pour afficher les attentes. Les utilisateurs pourront toutefois ajouter une attente directement à partir de la page de recherche ou accéder à un groupe d'attentes et y ajouter une attente. Si vous appliquez une attente à un objet à partir de l'onglet Attentes, d'une collection ou des résultats d'une recherche, les groupes d'attentes n'apparaissent pas dans la liste d'attentes.
