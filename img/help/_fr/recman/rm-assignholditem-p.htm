Pour assigner une attente à un élément :

  1. Cliquez sur l'icône Fonctions d'un objet géré et sélectionnez Propriétés, puis Attentes.

  2. Dans l'onglet Attentes de la page Propriétés de l'objet, cliquez sur Rechercher les attentes dans la liste déroulante Appliquer : Attente.

  3. Sur la page Rechercher les attentes, effectuez l'une des opérations suivantes :

    • Pour rechercher une attente, spécifiez le nom de l'attente dans le champ Rechercher un nom d'attente pour, puis cliquez sur le bouton Rechercher.

    • Cliquez sur le lien Sélectionner correspondant à une des attentes répertoriées dans la liste.

 

Astuce :
Pour assigner une attente à plusieurs éléments de façon simultanée, activez la case à cocher située en regard de chacun des éléments auxquels vous souhaitez assigner l'attente, puis cliquez sur le bouton Appliquer une attente ou cliquez sur Appliquer une attente dans la liste déroulante Autres actions.

Pour créer une nouvelle attente et l'assigner à l'élément, cliquez sur Créer une nouvelle attente dans la liste déroulante Appliquer : Attente figurant dans la section Rechercher les attentes de la page Rechercher les attentes. Cette option est disponible uniquement lors de la création d'attentes à la volée à partir d'une collection. Vous pouvez également créer une nouvelle attente à partir de l'onglet Attente.
