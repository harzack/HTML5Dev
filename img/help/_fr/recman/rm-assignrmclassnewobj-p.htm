Pour assigner un classement RM à un nouvel objet :

  1. Dans l'espace de travail Entreprise ou Personnel, cliquez sur un type d'élément dans le menu Ajouter un nouvel élément.

  2. Cliquez sur Recherche dans les classements RM dans la liste déroulante Classer.

  3. Sur la page Recherche dans les classements RM, spécifiez des paramètres de recherche.

  4. Cliquez sur le bouton Rechercher.

  5. Activez la case à cocher en regard de chaque classement RM que vous souhaitez assigner à l'élément.

  6. Cliquez sur le bouton Envoyer.
