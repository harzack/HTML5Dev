Pour rendre un élément confidentiel :

  1. Cliquez sur l'icône Fonctions d'un élément, puis sur Rendre confidentiel.

  2. Sur la page Rendre confidentiel, activez la case à cocher Rendre confidentiel.

    Vous pouvez rendre confidentiels les sous-éléments des recueils en activant la case à cocher Appliquer aux sous-éléments.

  3. Cliquez sur le bouton Envoyer.

 

Note :
Pour rendre un élément confidentiel, vous devez disposer de l'autorisation Modifier sur tous les éléments concernés.

Si l'administrateur a désactivé le paramètre de confidentialité, le menu Rendre confidentiel n'apparaît pas.
