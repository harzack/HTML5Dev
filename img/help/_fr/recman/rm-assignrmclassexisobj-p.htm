Pour assigner un classement RM à un objet existant :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Détails du document.

  2. Dans l'onglet Détails du document de la page Propriétés de l'élément, cliquez sur Recherche dans les classements RM dans la liste déroulante Sélectionner un classement.

  3. Sur la page Recherche dans les classements RM, spécifiez des paramètres de recherche.

  4. Cliquez sur le bouton Rechercher.

  5. Activez la case à cocher en regard de chaque classement RM que vous souhaitez assigner à l'élément.

  6. Cliquez sur le bouton Envoyer.

 

Astuce :
Vous pouvez assigner un classement RM à plusieurs éléments de façon simultanée dans la vue Parcourir. Vous devez pour cela activer la case à cocher située en regard de chaque élément auquel vous souhaitez appliquer le classement RM, puis cliquer sur le bouton Appliquer le classement ou cliquer sur Appliquer le classement dans la liste déroulante Autres actions.
