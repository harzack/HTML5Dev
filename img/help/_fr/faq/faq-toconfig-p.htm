Pour modifier les paramètres d'une FAQ :

  1. Cliquez sur l'icône Fonctions de la FAQ, puis sur Modifier les paramètres.

  2. Sur la page Paramètres de FAQ, effectuez l'une des actions suivantes :

    • Modifiez le titre ou la description de la FAQ.

    • Sélectionnez ou désélectionnez la case à cocher Activer les commentaires.

    • Sélectionnez ou désélectionnez la case à cocher Activer les abonnements par e-mail.

    • Modifiez le nom d'un sujet dans la zone Sujets de la FAQ.

    • Cliquez sur le bouton Ajouter des sujets d'entrée pour chaque nouveau sujet à ajouter, puis saisissez le nom du sujet dans le nouveau champ vierge.

  3. Cliquez sur Soumettre.

 

Astuce :
Pour modifier les paramètres d'une FAQ sur sa page principale, cliquez sur le lien Modifier les paramètres dans le bandeau Console FAQ.
