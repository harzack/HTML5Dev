Lorsque vous ajoutez un nouveau contenu à une FAQ, vous ajoutez, modifiez ou commentez une entrée. Il est possible d'ajouter et de modifier les entrées à condition de détenir les autorisations appropriées. Vous ne pouvez commenter les entrées que si la FAQ a été configurée pour autoriser les commentaires. Pour de plus amples informations sur la configuration des FAQ, reportez-vous à la section Création et configuration des FAQ.

Lorsque vous ajoutez une entrée, spécifiez les points suivants :

  • Ajoutez la question et la réponse associée pour la FAQ.

  • Sélectionnez les sujets de FAQ à appliquer à l'entrée, si des sujets ont été ajoutés à cette FAQ. Les sujets de FAQ sont utilisés pour classer les entrées sur la page principale de la FAQ.

  • Définissez une date d'expiration, le cas échéant.

  • Ajoutez des références utilisateur. Les références utilisateur sont des utilisateurs qui fournissent des informations supplémentaires sur l'entrée.

  • Ajoutez des pièces jointes sur disque local ou Content Server. Les pièces jointes sur disque local sont des fichiers que vous téléchargez depuis votre ordinateur pour les joindre à l'entrée de FAQ ; les pièces jointes Content Server sont des éléments Content Server que vous associez à l'entrée de FAQ.

Lorsque vous modifiez une entrée, vous pouvez modifier n'importe lequel de ces éléments.

Le commentaire d'une entrée vous permet d'apporter un retour d'informations sur le contenu de la FAQ. Les autres peuvent aussi répondre à votre commentaire.
