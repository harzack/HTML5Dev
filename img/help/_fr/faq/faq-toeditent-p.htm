Pour modifier une entrée de FAQ :

  1. Sur la page principale de la FAQ, cliquez sur le lien Voir les détails de l'entrée.

  2. Sur la page Entrée de FAQ, cliquez sur le lien Modifier.

  3. Sur la page Modifier l'entrée FAQ, effectuez l'une des actions suivantes :

    • Modifiez la question et la réponse.

    • Cliquez sur un sujet de FAQ dans la liste Sujet de FAQ, si cette liste est disponible.

    • Dans la zone Expiration, cliquez sur le bouton Outil de sélection de la date , puis sélectionnez un mois, un jour et une année pour configurer la date d'expiration de l'entrée.

    • Dans la zone S'adresser à, cliquez sur le bouton Choisir un utilisateur ou un groupe , puis sélectionnez l'utilisateur à ajouter comme référence à l'entrée.

    • Sélectionnez un type d'élément dans la liste Ajouter un élément pour ajouter un élément à l'entrée comme référence. Complétez les détails, puis cliquez sur Ajouter.

  4. Cliquez sur Soumettre.
