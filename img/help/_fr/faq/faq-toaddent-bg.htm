Pour ajouter une entrée à une FAQ :

  1. Sur la page principale de la FAQ, cliquez sur le lien Créer une entrée de FAQ dans le bandeau Console FAQ.

  2. Sur la page Ajouter une entrée de FAQ, cliquez sur un sujet de FAQ dans la liste Sujet de FAQ, si cette liste est disponible.

  3. Rédigez la question dans le champ Question.

  4. Saisissez la réponse à la question dans le champ Réponse.

  5. Cliquez sur Suivant.

  6. Sur la page Informations de la FAQ, effectuez l'une des actions suivantes :

    • Dans la zone Expiration, cliquez sur le bouton Outil de sélection de la date , puis sélectionnez un mois, un jour et une année pour configurer la date d'expiration de l'entrée.

    • Dans la zone S'adresser à, cliquez sur le bouton Choisir un utilisateur ou un groupe , puis sélectionnez l'utilisateur à ajouter comme référence à l'entrée.

    • Dans la zone Pièce jointe sur disque local, cliquez sur Parcourir, accédez au fichier à ajouter comme pièce jointe, puis cliquez sur le bouton Ouvrir. Cliquez sur Charger.

    • Dans la zone Pièce jointe Content Server, cliquez sur Parcourir Content Server, naviguez jusqu'à l'élément à ajouter en tant que pièce jointe, puis cliquez sur le lien Sélectionner de l'élément.

    • Dans la zone URL en tant que pièce jointe, saisissez un nom pour l'URL dans le champ Nom, puis saisissez l'adresse URL dans le champ URL.

  7. Cliquez sur Terminer.
