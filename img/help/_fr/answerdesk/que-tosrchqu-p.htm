Pour rechercher une question globale :

  1. Dans l'annuaire des communautés ou sur votre page d'accueil personnelle, cliquez sur le lien Mes questions dans le bandeau Console de navigation.

  2. Cliquez sur l'un des onglets suivants :

    • Mes questions, pour restreindre la recherche à vos questions.

    • Questions et Réponses, pour étendre la recherche aux questions de l'ensemble des utilisateurs.

  3. Saisissez des mots clés dans le champ Rechercher des Questions et Réponses.

  4. Cliquez sur l'une des options suivantes dans la liste :

    • Toutes les questions

    • Questions ayant obtenu une réponse

    • Questions n'ayant pas obtenu de réponse

  5. Cliquez sur Rechercher.
