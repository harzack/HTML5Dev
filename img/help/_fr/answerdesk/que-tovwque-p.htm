Pour afficher les questions globales :

  1. Dans l'annuaire des communautés ou sur votre page d'accueil personnelle, cliquez sur le lien Mes questions dans le bandeau Console de navigation.

  2. Cliquez sur l'un des onglets suivants :

    • Mes questions, pour afficher une liste de vos questions.

    • Questions et Réponses, pour afficher une liste des questions de l'ensemble des utilisateurs.
