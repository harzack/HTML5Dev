Pour configurer les paramètres des abonnements :

  1. Sélectionnez Paramètres dans le menu Outils.

  2. Sur la page Mes paramètres généraux, cliquez sur l'onglet Rapports par e-mail.

  3. Sur la page Mes paramètres de rapports par e-mail, effectuez l'une des actions suivantes :

    • Cliquez sur la fréquence à laquelle vous souhaitez recevoir les rapports d'activité des questions dans la liste Fréquence des rapports par e-mail.

    • Cliquez sur le format de votre choix pour les rapports d'activité des questions dans la liste Format de l'e-mail.

  4. Cliquez sur Mettre à jour.
