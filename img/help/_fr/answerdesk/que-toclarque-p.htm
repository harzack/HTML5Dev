Pour clarifier une question :

  1. Sur la page principale d'une question, cliquez sur le lien Clarifier la question dans la Console Question.

  2. Sur la page Ajouter une clarification, saisissez votre clarification dans le champ Réponse.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Soumettre.
