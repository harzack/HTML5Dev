Pour afficher des questions spécifiques à une communauté :

  1. Dans une communauté, cliquez sur le lien Questions de communauté dans le bandeau Console de navigation.

  2. Cliquez sur l'un des onglets suivants :

    • Mes questions, pour afficher une liste de vos questions.

    • Questions et Réponses, pour afficher une liste des questions de l'ensemble des membres.

     

    Astuce :
    Sélectionnez un Auditoire dans la liste pour restreindre les résultats de recherche.
