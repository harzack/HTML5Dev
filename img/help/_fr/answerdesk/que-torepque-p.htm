Pour répondre à une question :

  1. Sur la page principale de la question, cliquez sur le lien Ajouter une réponse dans le bandeau Console Question.

  2. Sur la page Ajouter une réponse, cliquez sur l'un des boutons radio suivants dans la section Type de réponse :

    • Réponse

    • Commentaire

  3. Saisissez le texte de votre réponse dans le champ Réponse.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Soumettre.
