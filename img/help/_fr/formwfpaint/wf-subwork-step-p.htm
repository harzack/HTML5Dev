Pour mapper les champs de formulaire dans les étapes de sous-workflow :

  1. Dans l'éditeur de workflow, double-cliquez sur une étape Sous-workflow.

  2. Cliquez sur l'onglet Forms.

  3. Pour chaque champ Formulaire de la colonne Schéma principal, cliquez sur un champ à mapper dans la liste Sous-schéma. Vous pouvez également cliquer sur Lien automatisé pour mapper automatiquement tous les champs de formulaire portant le même nom et contenant le même type de données dans le schéma de workflow principal et dans le schéma de sous-workflow.

  4. Cliquez sur Ajouter à la définition du workflow.

  5. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Mappage des champs de formulaire dans les étapes de sous-workflow.
