Pour rendre un formulaire disponible dans une étape :

  1. Dans l'éditeur de workflow, double-cliquez sur une étape de départ, sur une étape Utilisateur ou Initiateur ou sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Forms.

  3. Sélectionnez la case à cocher Disponible pour rendre disponible le formulaire choisi.

  4. Procédez de l'une des manières suivantes :

    • Affichez une vue personnalisée du formulaire en sélectionnant son nom dans la liste Vue du formulaire.

    • Affichez la vue par défaut du formulaire en sélectionnant <Aucun> dans la liste Vue du formulaire.

  5. Cliquez sur Ajouter à la définition du workflow.

  6. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Ajout de formulaires au lot de travail.
