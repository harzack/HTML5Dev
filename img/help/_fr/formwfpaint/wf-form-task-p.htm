Pour définir une étape Tâche de formulaire :

  1. Dans l'éditeur de workflow, double-cliquez sur une étape Tâche de formulaire.

  2. Dans l'onglet Général, indiquez un nom d'étape.

  3. Dans la liste Form à afficher, cliquez sur le nom du formulaire que doit voir l'utilisateur assigné lorsqu'il effectue la tâche de formulaire.

  4. Dans la liste Vue personnalisée, cliquez sur le nom de la vue personnalisée du formulaire que vous voulez utiliser pour cette tâche.

  5. Cliquez sur l'onglet Utilisateurs avec une tâche assignée et procédez de l'une des façons suivantes :

    • Cliquez sur le bouton radio Utilisateur ou groupe, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez l'utilisateur ou le groupe auquel vous souhaitez assigner l'étape.

    • Cliquez sur le bouton radio Utilisateurs/groupes multiples, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez les utilisateurs ou les groupes auxquels vous souhaitez assigner l'étape.

    • Activez la case d'option Utilisateur avec une tâche assignée à partir d'une étape antérieure, puis cliquez sur un nom d'étape dans la liste.

    • Activez la case d'option Utilisateur ayant une tâche assignée à partir du formulaire, puis cliquez sur un attribut dans la liste.

    • Cliquez sur le bouton radio Expressions et définissez les conditions à utiliser pour déterminer l'utilisateur avec une tâche assignée pour l'étape. Cliquez ensuite sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez les utilisateurs ou les groupes auxquels vous souhaitez assigner l'étape si la condition est vérifiée.

  6. Facultatif Ajoutez des instructions générales, spécifiez les paramètres de date et de priorité et sélectionnez l'option de traitement en arrière-plan, ainsi que l'option d'action par e-mail.

  7. Cliquez sur Ajouter à la définition du workflow.

  8. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition des étapes Tâche de formulaire.
