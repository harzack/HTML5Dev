Pour modifier un formulaire dans le lot de travail :

  1. Dans l'éditeur de workflow, sélectionnez Formulaires dans le menu Schéma.

  2. Dans l'onglet Formulaires, modifiez les paramètres comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes...
    Spécifier les valeurs par défaut des champs du formulaire Cliquez sur le lien Modifier, spécifiez les valeurs des champs, puis cliquez sur le bouton Appliquer.
    Modifier les paramètres du formulaire Cliquez sur le lien du nom du formulaire, modifiez les paramètres du formulaire, puis cliquez sur Mettre à jour le formulaire .

Pour plus d'informations sur cette procédure, consultez la section Ajout de formulaires au lot de travail.
