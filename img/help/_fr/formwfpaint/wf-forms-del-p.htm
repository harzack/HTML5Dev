Pour supprimer un formulaire du lot de travail :

  1. Dans l'éditeur de workflow, sélectionnez Formulaires dans le menu Schéma.

  2. Dans la page Formulaires, cliquez sur le nom du formulaire que vous souhaitez supprimer.

  3. Dans la page Formulaires du workflow, cliquez sur Supprimer.

  4. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Ajout de formulaires au lot de travail.
