Pour ajouter un formulaire à un lot de travail :

  1. Dans l'éditeur de workflow, sélectionnez Formulaires dans le menu Schéma.

  2. Dans l'onglet Formulaires, cliquez sur l'icône Ajouter un nouveau formulaire .

  3. Sur la page Formulaires de workflow, cliquez sur Parcourir Content Server puis sur le lien Sélectionner d'un modèle de formulaire sur lequel est basé le formulaire que vous souhaitez ajouter.

  4. Dans la zone Nom du formulaire, saisissez le nom du formulaire.

  5. Dans la liste Vue du statut, sélectionnez la vue qui apparaît lorsqu'un responsable du workflow affiche le formulaire dans un workflow en cours d'exécution.

  6. Dans la liste Mécanisme de stockage, cliquez sur le mécanisme de stockage avec lequel vous souhaitez stocker les données que les utilisateurs fournissent dans le formulaire.

  7. Facultatif Cochez la case Désactiver la création de versions pour désactiver les versions du formulaire.

  8. Facultatif Cochez la case Formulaire obligatoire pour obliger les utilisateurs auxquels l'étape est affectée à afficher le formulaire.

  9. Facultatif Cochez la case Afficher les pièces jointes pour permettre aux utilisateurs d'accéder aux pièces jointes du workflow à partir du formulaire.

  10. Cliquez sur AjouterForm.

  11. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Ajout de formulaires au lot de travail.
