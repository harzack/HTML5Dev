Le nom de cet exemple de projet est « Equipe des ventes ». Ce projet contient plusieurs références au mois de « Janvier » et vous souhaitez remplacer toutes les occurrences de « Janvier » par « Février ». Vous devez donc procéder comme suit :

  • Créez un modèle à partir du projet : Equipe des ventes.

  • Définissez les valeurs de la règle de substitution dans le nouveau modèle.

  • Créez un projet à partir du modèle mis à jour.

L'exemple suivant explique comment remplacer toutes les occurrences de « Janvier » par « Février » dans le projet Equipe des ventes.

  1. Plusieurs éléments du projet Equipe des ventes contiennent « Janvier » dans leur nom.

  2. Etant donné que vous souhaitez remplacer toutes les occurrences de « Janvier » par « Février », vous devez créer un modèle de projet à partir du projet Equipe des ventes d'origine.

  3. Une fois le modèle créé, vous devez saisir les valeurs des règles de substitution dans les champs appropriés.

  4. Une fois les règles établies dans le modèle de projet, vous devez créer un projet à partir du modèle. Les informations sur les règles de substitution s'affichent dans la section Contenu de la page Ajouter : Projet. Dans cet exemple, le libellé contient la valeur Mois. La valeur de substitution par défaut, Février, est automatiquement renseignée dans le champ associé au libellé. De même, lorsque vous cliquez sur la mention Aide en regard du libellé, la description du libellé s'affiche dans une fenêtre distincte.

  5. Le nouvel espace de travail Projet affiche les noms d'éléments avec la valeur « Février » à la place de « Janvier ».

Pour en savoir plus, voir Utilisation des projets.
