Pour créer un modèle à partir d'un projet :

  1. Cliquez sur l'icône Fonctions du projet, puis cliquez sur Créer un modèle.

  2. Saisissez un nom pour le modèle dans le champ Nom.

  3. Dans la section Source de la page Ajouter : Modèle de projet, cliquez sur Projet, puis sur Parcourir Content Server. Naviguez jusqu'au projet à partir duquel vous souhaitez créer le modèle, puis cliquez sur Sélectionner.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
