Pour ajouter une version à un modèle :

  1. Cliquez sur l'icône Fonctions d'un modèle de projet et choisissez Ajouter une version.

  2. Dans la section Source, cliquez sur Parcourir Content Server, naviguez jusqu'au modèle de projet à partir duquel vous voulez créer une version, puis cliquez sur Sélectionner.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter une version.
