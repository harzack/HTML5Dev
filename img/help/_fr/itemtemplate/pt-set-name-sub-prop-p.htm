Pour définir les propriétés des règles de substitution :

  1. Cliquez sur l'icône Fonctions du modèle de projet, choisissez Propriétés, puis Règle de substitution.

  2. Dans l'onglet Règle de substitution de la page Propriétés Modèle de projet, cliquez sur ici.

  3. Dans le champ Chaîne à remplacer, saisissez le nom de la chaîne que vous souhaitez remplacer.

  4. Dans le champ Valeur de replacement par défaut, saisissez le texte devant remplacer la chaîne spécifiée à l'étape précédente.

  5. Dans le champ Afficher le libellé, saisissez le texte du libellé pour la valeur de remplacement par défaut dans la section Contenu de la page Ajouter : Projet.

  6. Cliquez sur Mettre à jour.

Pour... Effectuez les opérations suivantes...
Ajouter une description du libellé affiché Saisissez une description dans le champ Afficher la description.
Ajouter des valeurs de règle de substitution supplémentaires Cliquez sur l'icône Ajouter une ligne, , dans la colonne Ligne.
Supprimer des valeurs de règle de substitution Cliquez sur l'icône Supprimer, , dans la colonne Ligne de la valeur à supprimer, puis cliquez sur OK dans la boîte de dialogue.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des modèles de projet.
