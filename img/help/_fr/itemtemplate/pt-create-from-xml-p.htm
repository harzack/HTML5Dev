Pour créer un modèle à partir d'un fichier XML :

  1. Cliquez sur l'icône Fonctions du projet et choisissez Créer un modèle.

  2. Saisissez un nom pour le modèle dans le champ Nom.

  3. Dans la section Source de la page Ajouter : Modèle de projet, cliquez sur Fichier (*.xml), puis sur Parcourir, naviguez jusqu'au fichier XML que vous souhaitez utiliser comme modèle, puis cliquez sur Ouvrir.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
