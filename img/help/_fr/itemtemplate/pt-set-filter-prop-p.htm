Pour définir les propriétés de filtrage :

  1. Cliquez sur l'icône Fonctions du modèle de projet, choisissez Propriétés, puis Filtrage.

  2. Dans l'onglet Filtrage de la page Propriétés Modèle de projet, indiquez les éléments que vous souhaitez inclure lors de la création d'un projet à partir du modèle.

  3. Cliquez sur Mettre à jour.
