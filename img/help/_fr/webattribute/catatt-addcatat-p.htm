Pour ajouter ou modifier un attribut :

  1. Cliquez sur une catégorie que vous souhaitez modifier.

  2. Sur la page Catégorie, procédez comme suit :

    • Pour ajouter un attribut, cliquez sur un type Utilisation des attributs dans la liste déroulante Ajouter un attribut, ajoutez les valeurs d'attribut appropriées sur la page Ajouter l'attribut à, puis cliquez sur le bouton OK.

    • Pour modifier un attribut, cliquez sur le lien de Utilisation des attributs, modifiez l'attribut sur la page Modifier l'attribut pour, puis cliquez sur le bouton OK.

  3. Sur la page Catégorie, cliquez sur le bouton Envoyer.
