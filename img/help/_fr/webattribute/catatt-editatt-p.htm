Pour modifier les valeurs d'attribut d'un élément :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Catégories.

  2. Dans l'onglet Catégories de la page Propriétés de l'élément, cliquez sur le nom de la catégorie dont vous voulez modifier les attributs.

  3. Cliquez sur le lien d'un attribut, puis modifiez ses paramètres sur la page Modifier l'attribut.

  4. Cliquez sur le bouton OK.

 

Astuce :
Vous pouvez ajouter un nouvel attribut à une catégorie assignée à l'élément en cliquant sur un attribut dans la liste déroulante Ajouter un attribut. Pour en savoir plus, voir Utilisation des attributs.
