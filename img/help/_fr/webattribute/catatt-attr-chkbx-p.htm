Pour définir un attribut Indicateur : Case à cocher :

  1. Cliquez sur une catégorie.

  2. Sur la page Catégorie, cliquez sur Indicateur : Case à cocher dans la liste déroulante Ajouter un attribut.

  3. Sur la page Ajouter l'attribut à, tapez un nom pour l'attribut dans le champ Nom.

  4. Pour séquencer les attributs, procédez de l'une des façons suivantes :

    • Dans la liste déroulante Après, cliquez sur le nom de l'attribut après lequel vous souhaitez afficher le nouvel attribut sur la page de catégorie. Cliquez sur <Aucun(e)> pour définir l'attribut s'affichant en premier.

    • Activez ou désactivez la case à cocher Afficher dans la recherche.

    • Cliquez sur le bouton OK.

    • Sur la page Catégorie, activez ou désactivez la case à cocher de l'attribut pour définir sa valeur par défaut.

    • Cliquez sur le bouton Envoyer.
