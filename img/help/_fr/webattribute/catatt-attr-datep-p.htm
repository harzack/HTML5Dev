Pour définir un attribut Date : Liste déroulante :

  1. Cliquez sur une catégorie.

  2. Sur la page Catégorie, cliquez sur Date : dans la liste déroulante Ajouter un attribut.

  3. Sur la page Ajouter l'attribut à, tapez un nom pour l'attribut dans le champ Nom.

  4. Dans la section Lignes, procédez de la façon suivante :

    • Activez ou désactivez la case à cocher Verrouillé(e).

    • Cliquez sur une valeur dans les listes déroulantes N° max. et N° par défaut.

       

      Note :
      Lorsque vous augmentez le nombre de lignes par défaut, le nombre maximum de lignes augmente automatiquement. Vous pouvez définir un nombre maximal de lignes supérieur au nombre de lignes par défaut uniquement si vous désactivez la case à cocher Verrouillé(e).

       

  5. Dans la liste déroulante Après, cliquez sur le nom de l'attribut après lequel vous souhaitez afficher le nouvel attribut sur la page de catégorie. Cliquez sur <Aucun(e)> pour définir l'attribut s'affichant en premier.

  6. Activez ou désactivez les cases à cocher suivantes :

    • Afficher dans la recherche

    • Obligatoire

  7. Pour définir chacune des valeurs qui s'affiche dans la liste de zone déroulante de l'attribut :

    • Cliquez sur l'icône Ajouter une valeur .

    • Pour chaque valeur valable, sélectionnez un mois, un jour et une année dans les listes déroulantes de date.

  8. Cliquez sur le bouton OK.

  9. Sur la page Catégorie, pour définir une valeur par défaut à l'attribut, cliquez sur un mois, une date et une année (ainsi qu'une heure le cas échéant) dans les listes déroulantes appropriées.

  10. Cliquez sur le bouton Envoyer.
