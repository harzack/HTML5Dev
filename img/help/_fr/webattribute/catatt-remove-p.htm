Pour supprimer une catégorie d'un élément :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Catégories.

  2. Dans l'onglet Catégories de la page Propriétés, cliquez sur le nom de la catégorie que vous souhaitez enlever.

  3. Cliquez sur le bouton Enlever.

  4. Cliquez sur le bouton OK dans la boîte de dialogue.

 

Astuce :
Vous pouvez également enlever une catégorie lors de la copie ou du déplacement d'un élément. Pour en savoir plus, voir Utilisation des éléments généraux.
