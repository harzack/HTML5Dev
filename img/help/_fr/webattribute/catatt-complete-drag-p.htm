Pour compléter les informations obligatoires des éléments glissés-déposés :

  1. Procédez de l'une des manières suivantes :

    • Cliquez sur le bouton Afficher la vue Glisser-déposer dans le recueil contenant des éléments incomplets, puis cliquez sur le bouton Continuer.

    • Dans le recueil contenant des éléments incomplets, cliquez sur le lien Afficher les éléments incomplets.

  2. Sur la page des éléments incomplets, effectuez l'une des opérations suivantes :

    • Pour compléter les attributs d'un élément individuel, cliquez sur le lien Compléter cet élément.

    • Pour compléter les attributs de plusieurs éléments, cochez la case correspondant à chacun des éléments que vous souhaitez compléter, puis cliquez sur le bouton Compléter les éléments sélectionnés.

     

    Note :
    Si vous souhaitez mettre à jour des éléments incomplets dans les sous-dossiers, cochez la case Appliquer les informations requises aux éléments des sous-recueils avant de cliquer sur le bouton Compléter les éléments sélectionnés.

     

  3. Sur la page Modifier les attributs, entrez une valeur pour chaque attribut obligatoire, puis cliquez sur le bouton Envoyer.

    Vous pouvez éventuellement changer le nom de l'élément dans le champ Nom sur la page Mettre à jour.

  4. Cliquez sur le bouton Accepter.

  5. Cliquez sur le bouton OK.

 

Astuce :
Vous pouvez également compléter les éléments comportant des attributs obligatoires en accédant au recueil contenant des éléments incomplets sur votre page Tâches assignées.
