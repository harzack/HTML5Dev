Pour mettre à niveau les affectations de catégories :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Catégories.

  2. Cliquez sur le lien d'une catégorie pouvant être mise à niveau.

  3. Cliquez sur le bouton Mettre à niveau. Si nécessaire, modifiez les valeurs d'attribut. Si nécessaire, lorsque l'élément appartient à un recueil, cliquez sur l'icône Appliquer aux sous-éléments  pour appliquer la mise à niveau aux sous-éléments.

  4. Cliquez sur le bouton Envoyer.
