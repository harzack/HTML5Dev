Pour ajouter une catégorie :

  1. Dans le menu Ajouter un élément, cliquez sur Catégorie.

  2. Sur la page Ajouter : Catégorie, saisissez un nom pour la catégorie dans le champ Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur le bouton Ajouter.

 

Note :
Après avoir créé une catégorie, vous devez lui ajouter des attributs. Pour en savoir plus, voir Utilisation des attributs.
