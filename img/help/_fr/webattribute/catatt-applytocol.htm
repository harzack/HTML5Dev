Vous pouvez appliquer des catégories à un ou plusieurs éléments d'une collection. Dans une page Collection, dans la liste Autres actions, sélectionnez Appliquer les catégories. Lorsque vous appliquez une catégorie à des éléments spécifiques, la page Appliquer les catégories qui s'affiche est composée de deux sections : une section Objets pris en charge répertoriant les éléments sélectionnés auxquels vous pouvez appliquer des catégories et une section Objets non pris en charge répertoriant ceux auxquels vous ne pouvez pas appliquer de catégories.

Les informations suivantes sont disponibles sur la page Appliquer les catégories :

  • Type

  • Nom

  • Emplacement

 

Note :
Dans la section Objets non pris en charge, une explication présentant la raison invoquée pour la non-application d'une catégorie sur un élément donné est disponible dans la colonne Motif pour chaque élément.

 

En plus des informations ci-dessus, les options suivantes sont à votre disposition :

  • Le champ Catégories qui permet de sélectionner les catégories à appliquer.

  • La case à cocher Ajouter une version qui permet de créer des versions d'éléments multiversions une fois les catégories appliquées.

  • La case à cocher Vider les catégories existantes qui permet de supprimer les catégories existantes des éléments.

Lorsque vous voulez appliquer des catégories à tous les éléments de la collection, il faut savoir que les zones Objets pris en charge et Objets non pris en charge de la page Appliquer les catégories contiennent un certain nombre de types d'éléments auxquels vous pouvez appliquer ou non des catégories. En plus des options répertoriées ci-dessus, vous pouvez également envoyer une notification par e-mail aux utilisateurs, aux groupes ou aux destinataires tiers pour les informer qu'une ou plusieurs catégories ont été appliquées aux éléments de la collection. Par exemple, si la collection est utilisée comme emplacement central pour un projet d'équipe, vous pouvez utiliser la notification par e-mail pour informer les autres membres de l'équipe de l'application des catégories.

Si vous appliquez des catégories à une collection pour laquelle des catégories ont déjà été appliquées, ces dernières sont répertoriées comme valeurs par défaut à la page Modifier les catégories. S'il existe des versions plus récentes que les catégories actuellement appliquées, le bouton Mettre à niveau s'affiche, vous permettant d'utiliser la dernière version de la catégorie sur les éléments. Si vous choisissez de ne pas appliquer la dernière version de la catégorie, les éléments disposant déjà de la dernière version ne sont pas affectés. Si vous appliquez une catégorie différente à un élément pour lequel une catégorie a été appliquée, la nouvelle catégorie est ajoutée.

 

Note :
Si pour une raison quelconque l'application de catégorie à des éléments sélectionnés échoue, et si la notification par e-mail est activée, les destinataires spécifiés seront informés de l'erreur.
