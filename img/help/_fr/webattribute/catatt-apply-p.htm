Pour appliquer une catégorie à des sous-éléments :

  1. Cliquez sur l'icône Fonctions d'un recueil, choisissez Propriétés, puis Catégories.

  2. Cliquez sur l'icône Appliquer aux sous-éléments .

  3. Sur la page Appliquer les catégories aux sous-éléments, cliquez sur l'une des listes déroulantes suivantes pour chaque catégorie :

    • <Aucun(e)>, qui n'a aucune action sur cette catégorie.

    • Ajouter, qui applique la catégorie à tous les sous-éléments mais ne modifie pas les informations de catégorie des éléments associés à la catégorie.

    • Mettre à niveau, qui met à jour uniquement les définitions d'attributs des éléments associés à la catégorie.

    • Ajouter/Mettre à niveau, qui applique la catégorie à tous les sous-éléments et met automatiquement à niveau les éléments associés à la catégorie vers la version la plus récente.

    • Enlever, qui supprime la catégorie des sous-éléments.

    Pour que Content Server ajoute une version à tous les sous-éléments multi-version lorsque vous mettez à jour leurs catégories, vous pouvez activer la case à cocher Ajouter une version. Pour enlever toutes les catégories des sous-éléments du recueil, vous pouvez activer la case à cocher Vider les catégories existantes.

  4. Cliquez sur le bouton Envoyer.

  5. Cliquez sur le bouton OK dans la fenêtre en cours.

  6. Dans l'onglet Catégories de la page Propriétés, cliquez sur le bouton Envoyer.

 

Astuce :
La case à cocher Ajouter une version permet d'annuler les changements apportés à un élément en mettant à niveau ou à jour les catégories associées.
