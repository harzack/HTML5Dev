Vous pouvez lancer un schéma de workflow à l'aide de messages HTTP à partir d'un système distant si le lot de travail de lancement à distance est activé et si vous ajoutez des tickets de lancement à distance au schéma de workflow.

 

Note :
les messages de lancement à distance peuvent uniquement échanger le type de document normal (sous-type 144) sous forme de pièce jointe de workflow.
