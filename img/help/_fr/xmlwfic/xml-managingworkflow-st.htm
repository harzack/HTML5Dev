Lorsqu'une tâche Workflow Interchange est active, elle apparaît dans la liste de tâches assignées du propriétaire de la tâche.

Vous pouvez ouvrir une liste de tâches assignées à partir de n'importe quelle page Content Server. Une fois une liste de tâches assignées ouverte, vous pouvez en modifier n'importe quelle tâche. Vous pouvez également afficher et modifier les données, et envoyer une tâche immédiatement sans envoyer ni recevoir d'autres messages.

Le lot de travail général permet de gérer le comportement d'échange de la tâche. Outre les informations courantes sur la tâche, cette page contient les sections suivantes :

  • URL d'accès externe, qui affiche l'URL que les systèmes externes doivent utiliser pour accéder à la tâche. Si la tâche est configurée de façon à accepter les messages de modification de tâche, ils sont envoyés à cette URL. L'URL est incluse par défaut dans tous les messages de lancement envoyés par la tâche. L'URL est affichée sous forme de lien, de telle sorte qu'elle peut facilement être copiée et collée dans d'autres applications. Si la tâche est configurée de façon à accepter les messages de modification de tâche, vous pouvez cliquer sur ce lien pour récupérer les valeurs courantes de tous les champs susceptibles d'être définis dans un message de modification de tâche et pour récupérer le format que l'action accepte.

  • Statut d'acceptation, qui affiche les informations relatives aux cinq dernières requêtes envoyées à l'URL d'accès externe (notamment l'heure de réception de la requête) et un message d'erreur ou une indication que le traitement de la requête a réussi.

  • Statut de l'envoi, qui affiche le statut des actions d'envoi configurées de façon à se produire une fois l'étape prête. Pour réexécuter les actions, sélectionnez les cases à cocher Sélectionner des actions souhaitées et cliquez sur Nouvel essai sélectionné.

D'autres lots de travail permettent à l'utilisateur avec tâche assignée d'effectuer des modifications manuelles et d'utiliser les mêmes procédures qu'une tâche utilisateur standard.
