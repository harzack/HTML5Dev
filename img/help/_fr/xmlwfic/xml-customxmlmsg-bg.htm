Toutes les actions qui envoient et acceptent des messages XML disposent de propriétés de filtre et de format qui peuvent être utilisées pour personnaliser les messages envoyés ou reçus.

Vous pouvez utiliser les propriétés de filtre pour inclure ou exclure des éléments de données des messages acceptés ou produits. Lorsque vous personnalisez les messages envoyés et reçus, les informations du workflow peuvent être divisées en différents éléments, chacun pouvant être contrôlé individuellement.

Les propriétés de format permettent d'appliquer des feuilles de style XSLT et des espaces de noms XML pour changer la représentation XML de base des éléments de données.
