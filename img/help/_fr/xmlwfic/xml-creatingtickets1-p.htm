Pour ajouter un ticket de lancement à distance :

  1. Sélectionnez Lancement à distance dans le menu Schéma.

  2. Dans l'onglet Lancement à distance, cliquez sur Ajouter un ticket.

  3. Modifiez le nom du ticket dans le champ Nom du ticket.

  4. Sélectionnez une date dans les listes Date d'expiration.

     

    Note :
    Tous les messages envoyés à l'URL d'accès à la date d'expiration ou ultérieurement sont rejetés.

     

  5. Dans la zone Action de lancement, sélectionnez l'une des options suivantes :

    • Accepter un message de lancement XML, qui spécifie que les messages de lancement doivent être au format XML. Vous pouvez par exemple utiliser ce format pour permettre à une étape Workflow Interchange d'un autre système Content Server (configuré pour envoyer un message de lancement XML une fois prêt) de lancer le schéma de workflow.

    • Accepter un message de lancement HTML, qui spécifie que les messages de lancement doivent être au format de données de formulaire HTML. Vous pouvez par exemple utiliser ce format pour accepter les messages de systèmes hérités, les formulaires HTML ou les documents joints au workflow par d'autres systèmes.

  6. Dans la zone IP sécurisées, sélectionnez l'un des boutons radio suivants :

    • Toutes les adresses IP, qui permet d'accepter les messages de lancement de n'importe quelle adresse IP.

    • A partir des adresses IP suivantes uniquement, qui permet de limiter les adresses IP autorisées à envoyer des messages de lancement. Pour spécifier des adresses IP, cliquez sur l'icône Ajouter une valeur  et saisissez une adresse IP dans le champ prévu à cet effet.

  7. Dans la zone Initiateur, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez un utilisateur.

    Le rôle d'initiateur est affecté à l'utilisateur lorsque le workflow est lancé avec ce ticket.

  8. Cliquez sur Accepter.

  9. Cliquez sur Ajouter à la définition du workflow, puis enregistrez le workflow.

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Création de tickets de lancement à distance.
