Pour configurer les paramètres généraux :

  1. Sur le schéma de workflow, double-cliquez sur l'étape Workflow Interchange.

  2. Dans l'onglet Général, modifiez le nom de la tâche dans le champ Nom de l'étape.

    Le nom spécifié permet d'identifier l'étape dans le schéma de workflow ainsi que la tâche dans les tâches assignées de l'utilisateur.

  3. Saisissez les instructions nécessaires à l'exécution de la tâche dans le champ Instructions.

    Les instructions spécifiées sont visibles pour l'utilisateur avec une tâche assignée et transmises dans tous les messages de lancement envoyés par la tâche une fois celle-ci active.

  4. Dans le champ Utilisateur avec une tâche assignée, cliquez sur l'icône Choisir l'utilisateur  afin de rechercher et de sélectionner un utilisateur avec une tâche assignée.

  5. Cliquez sur Ajouter à la définition du workflow.

Pour... Effectuez les opérations suivantes...
Afficher les commentaires des étapes précédentes et les envoyer avec les messages Cochez la case La cible/l'utilisateur avec une tâche assignée peut consulter tous les commentaires.
Définir la durée de la tâche Saisissez une valeur dans le champ Durée.
Lancer une tâche à une heure précise Sélectionnez une date et une heure de début dans les listes de la zone Date de début.

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Configuration des paramètres généraux de Workflow Interchange.
