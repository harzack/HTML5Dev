Pour ajouter une étape Workflow Interchange à un schéma de workflow, utilisez l'éditeur de workflow Content Server comme vous le feriez pour toute autre étape de workflow. Vous pouvez ensuite configurer les paramètres Général et Action de l'étape. Vous pouvez également accéder à d'autres paramètres de configuration du comportement des différents lots de travail.
