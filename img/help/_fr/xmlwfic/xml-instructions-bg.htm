Un message de lancement comprend les instructions générales pour la tâche source.

Représentation XML

<instructions startdate="…" duedate="…">…</instructions>

En XML, les instructions sont formatées sous forme d'un élément instructions. Le contenu de l'élément est le texte des instructions. Si une date de début et une date d'échéance ont été définies pour l'étape du workflow, elles sont indiquées dans les attributs.

Espaces de noms XML (le cas échéant)

<instructions xmlns="http://www.opentext.com/llx/workflow/">

Représentation HTML :

instructions_startdate=…

instructions_duedate=…

instructions_text=…

Dans les données de formulaires HTML, la date de début, la date de fin et les instructions sont des champs séparés.
