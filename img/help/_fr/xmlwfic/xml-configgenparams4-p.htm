Pour configurer les propriétés d'une action d'acceptation XML :

  1. Double-cliquez sur l'étape Workflow Interchange, puis cliquez sur l'onglet Actions.

  2. Dans la zone Exécution de l'étape, sélectionnez la liste Rejeter, puis sélectionnez Accepter un message de modification de la tâche HTML ou Accepter un message de modification de la tâche XML.

  3. Cliquez sur l'icône Modifier  d'une action.

  4. Cliquez sur l'une des options suivantes dans la liste Si des erreurs sont rencontrées lors du traitement de la requête :

    • Renvoyer des codes d'erreur HTTP, qui renvoie un code de statut d'erreur HTTP si des erreurs empêchent le traitement correct de la requête ;

    • Renvoyer des messages d'erreur XML, qui rapporte les erreurs de traitement au moyen de messages d'erreur XML. Les erreurs qui empêchent le routage correct de la requête renvoient un rapport contenant des codes HTTP.

  5. Cliquez sur Appliquer.

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Configuration des paramètres d'action généraux.
