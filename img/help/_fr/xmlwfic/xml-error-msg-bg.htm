Un message d'erreur peut être renvoyé en réponse à une requête HTTP. Il indique un échec de traitement de la requête.

Représentation XML

<error message="…"/>

Lorsqu'une action d'acceptation XML est configurée de façon à générer des messages d'erreur XML, elle renvoie le message d'erreur sous forme d'éléments error dans la mesure du possible. Sinon, les erreurs sont renvoyées avec les codes d'erreur HTTP.

Un attribut de message peut être utilisé sur l'élément pour décrire l'erreur.

Espaces de noms XML (le cas échéant)

<error xmlns="http://www.opentext.com/llx/">

Représentation HTML

Les actions d'acceptation HTML peuvent uniquement renvoyer des erreurs sous forme de codes d'erreur HTTP.
