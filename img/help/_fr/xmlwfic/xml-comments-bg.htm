Le lot de commentaires contient les commentaires associés aux étapes du workflow lorsqu'elles sont réalisées.

Les commentaires des étapes précédentes comprennent le nom de l'étape, le nom de l'utilisateur et la date du commentaire.

Représentation XML

<comments>

<comment>…</comment>

<comment stepname="…" username="…" datesaved="…">…</comment>

</comments>

En XML, le lot de commentaires est un élément comments ayant un élément comment enfant pour chaque étape.

Espaces de noms XML (le cas échéant)

<comments xmlns="http://www.opentext.com/llx/workflow/">

Représentation HTML

comments_text_1=…>

comments_text_2=…>

comments_username_2=…>

comments_stepname_2=…>

comments_datesaved_2=…>

Dans HTML, les champs de commentaires sont groupés et numérotés par étape. Les commentaires de l'étape en cours se trouvent toujours dans le premier groupe.
