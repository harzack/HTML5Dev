Un champ de statut de tâche indique si la tâche à laquelle il se réfère est active ou terminée. La valeur du champ est soit actif, soit terminé.

Un message de modification de tâche ou une réponse à un message de lancement peut transférer une tâche en définissant son statut sur terminé.

Représentation XML

<… status= "…">

Dans XML, le champ de statut de tâche est un attribut qui peut apparaître dans les éléments initiate, edittask ou OK.

Représentation HTML

taskstatus=…

Dans les données de formulaire HTML, le statut de tâche est présent dans un seul champ.
