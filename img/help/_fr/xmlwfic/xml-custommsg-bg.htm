Pour personnaliser les messages de données de formulaire HTML :

  1. Double-cliquez sur l'étape Workflow Interchange, puis cliquez sur l'onglet Actions.

  2. Cliquez sur l'icône Modifier  pour Envoyer un message de lancement HTML.

  3. Cliquez sur l'onglet Format.

  4. Dans la liste de la zone Format, sélectionnez Format personnalisé. Deux types d'éléments de workflow apparaissent : les champs et les groupes de champs. Un groupe de champs est un ensemble de champs et de sous-groupes qui forment une unité logique.

  5. Facultatif Désactivez la case à cocher Statut de la tâche pour ne pas inclure un champ dans tous les messages que l'action envoie et ignorer le champ dans tous les messages que l'action reçoit.

  6. Sélectionnez la case à cocher Titre de la tâche pour ajouter le titre de la tâche à tous les messages envoyés et reçus.

     

    Note :
    Dans l'Etape de départ du schéma de workflow, la case à cocher Demander un titre à l'initialisation doit être sélectionnée pour cet élément du workflow.

     

  7. Cliquez sur un format pour chaque groupe de champs dans les listes. Tous les éléments du workflow valides dans les messages apparaissent de façon individuelle.

  8. Facultatif Saisissez un nom pour chaque élément du workflow dans la zone Champ de formulaire HTML, notamment pour les éléments compris dans les groupes de champs, afin de spécifier le champ de formulaire HTML utilisé pour contenir les données du champ dans les messages.

  9. Dans la zone Dates, cliquez sur l'une des options suivantes :

    • Champ unique, qui est une valeur complexe écrite dans un champ de formulaire HTML unique ;

    • Champ multiple, qui correspond à des champs de six entiers indiquant l'année, le mois, le jour, l'heure, les minutes et les secondes.

  10. Dans la zone Booléens, cliquez sur un format.

  11. Facultatif Si un champ ou un groupe de champs contient plusieurs valeurs, des index numériques sont ajoutés aux champs de formulaire HTML correspondants pour distinguer les occurrences multiples. Par exemple, le nom de la troisième partie de la seconde section peut être section_partie_2_3. Si vous souhaitez changer le schéma de numérotation de l'index numérique, sélectionnez l'une des options suivantes dans la liste Format des noms à valeurs multiples :

    • nom_1_1, nom_1… : ajoute des chiffres aux noms de tous les champs pouvant avoir des valeurs multiples

    • nom, nom_1_2 : n'utilise pas de chiffres si ceux-ci correspondent tous à 1

  12. Facultatif Saisissez un nom et une valeur dans les champs Nom et Valeur de la zone Champs, puis cliquez sur l'icône Ajouter une valeur  si vous spécifiez un format personnalisé pour l'action d'envoi et si vous souhaitez ajouter des champs de formulaire HTML aux messages sortants.

  13. Cliquez sur Appliquer.

 

Note :
le nom d'un champ de formulaire provient du nom de base que vous spécifiez, du type de données qu'il contient et de la valeur des propriétés Dates et Noms. Pour plus d'informations sur les champs de nom tels qu'ils apparaissent dans les données de formulaire, reportez-vous à Champs et formats de messages.

 

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Personnalisation des messages du formulaire HTML..
