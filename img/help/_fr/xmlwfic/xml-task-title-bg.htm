Un titre de tâche indique le titre de la tâche.

Représentation XML

<… title= "…">

En XML, le champ de titre de la tâche est un attribut qui peut apparaître dans les éléments initiate ou edittask.

Représentation HTML

tasktitle=…

Dans les données de formulaire HTML, le titre de la tâche est présent dans un seul champ.
