Pour... Effectuez les opérations suivantes...
Ouvrir votre liste de tâches assignées Sélectionnez Tâches assignées dans le menu Personnel.
Ouvrir la page de modification d'une tâche Cliquez sur le nom de la tâche dans la liste des tâches assignées.
Afficher et modifier les données Cliquez sur les éléments de travail disponibles dans la liste située à gauche.
Envoyer une tâche immédiatement sans envoyer ni recevoir d'autres messages Cliquez sur le bouton permettant de sauter l'étape.

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Utilisation des listes de tâches assignées.
