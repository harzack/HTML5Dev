Pour configurer la réponse XML aux propriétés de l'action de lancement à distance :

  1. Double-cliquez sur l'étape Workflow Interchange, puis cliquez sur l'onglet Actions.

  2. Dans la zone Etape prête, sélectionnez la liste Exécuter l'action, puis sélectionnez Envoyer une réponse XML à l'initiateur à distance.

  3. Cliquez sur l'icône Modifier  d'une action.

  4. Saisissez le nom de l'action dans le champ Nom.

     

    Note :
    le nom apparaît dans la liste des actions configurées pour l'étape mais pas dans les messages. Vous pouvez l'utiliser pour décrire l'objectif d'une action d'envoi ou pour faire la distinction entre les différentes actions du même type configurées pour la même étape.

     

  5. Sélectionnez la case à cocher Exécuter la tâche de lancement si vous voulez que le champ de statut de la tâche soit configuré comme « terminé » dans le message de réponse créé.

     

    Note :
    si cette case à cocher est sélectionnée dans les options de personnalisation du message, le système source exécute et transfère la tâche qui a lancé le workflow en cours. Pour plus d'informations sur le lancement des tâches, reportez-vous à Champs et formats de messages.

     

  6. Cliquez sur Appliquer.

Pour plus d'informations sur cette procédure, reportez-vous à la rubrique Configuration des paramètres d'action généraux.
