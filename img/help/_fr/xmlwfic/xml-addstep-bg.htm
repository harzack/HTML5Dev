Pour ajouter une étape Workflow Interchange

  1. Dans l'éditeur de schéma de workflow, glissez-déposez l'étape Workflow Interchange sur le schéma de workflow.

  2. Placez le pointeur de la souris sur l'étape Workflow Interchange, puis faites glisser la flèche qui apparaît sur l'étape à laquelle vous souhaitez la connecter.

pour plus d'informations sur les étapes de travail, reportez-vous à Description des workflows.
