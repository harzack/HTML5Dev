Pour créer une sous-communauté :

  1. Sur la page Accueil Communauté, cliquez sur le lien Sous-communautés dans le bandeau Console de navigation.

     

    Note :
    Le lien Sous-communautés apparaît seulement s'il est activé par le modérateur.

     

  2. Sur la page Sous-communautés, cliquez sur le lien Ajouter une sous-communauté.

  3. Créez la communauté, tel que décrit dans la section Pour créer une communauté.
