Pour ajouter des éléments à une communauté :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Bibliothèque.

  2. Sur la page Bibliothèque, cliquez sur le menu Ajouter un élément, puis cliquez sur <type_d'élément>.

  3. Sur la page Ajouter de l'élément, spécifiez les paramètres nécessaires pour le type d'élément, puis cliquez sur Ajouter.

 

Note :
Pour de plus amples informations sur l'ajout d'éléments à Content Server, reportez-vous à la section Ajout d`éléments.
