Pour ajouter un groupe d'experts :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le lien Domaine d'expertise dans le bandeau Liens connexes.

  3. Sur la page Experts de la communauté, saisissez le nom du groupe dans le champ Nom du groupe.

  4. Saisissez une description du groupe dans la zone Description.

  5. Cliquez sur Ajouter.
