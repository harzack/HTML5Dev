Pour envoyer une alerte :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le lien Envoyer une alerte pour l'utilisateur à qui vous souhaitez envoyer un message.

  3. Sur la page Envoyer une alerte, saisissez un message dans le champ Message.

  4. Cliquez sur Envoyer.

 

Astuce :
Pour envoyer une alerte à tous les membres de la communauté, cliquez sur Envoyer un message, puis sélectionnez Envoyer une alerte.
