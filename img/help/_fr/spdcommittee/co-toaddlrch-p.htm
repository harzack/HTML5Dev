Pour ajouter un rapport LiveReport à la page d'accueil de la communauté :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur le bouton Sélectionner des rapports LiveReport .

  3. Sur la page Sélectionner des rapports LiveReport, cliquez sur le bouton Ajouter un rapport LiveReport .

  4. Sur la page Ajouter, recherchez le rapport LiveReport que vous souhaitez ajouter, activez la case à cocher Sélectionner du rapport, puis cliquez sur Envoyer.

     

    Astuce :
    Vous pouvez ajouter plusieurs rapports LiveReport simultanément.

     

  5. Sur la page Sélectionner des rapports LiveReport, cliquez sur Terminé.

  6. Sur la page Configurer le paramétrage de la communauté, double-cliquez sur le rapport LiveReport dans le bandeau Modules de contenu.

  7. Placez le rapport LiveReport parmi les modules de contenu actifs.

  8. Cliquez sur Terminé.

 

Note :
Seuls les rapports LiveReport qui fournissent une liste de nœuds sont pris en charge sur la page d'accueil de la communauté.
