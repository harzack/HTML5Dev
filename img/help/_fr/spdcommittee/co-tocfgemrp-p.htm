Pour configurer les rapports par e-mail :

  1. Dans le menu Outils, cliquez sur Paramètres.

  2. Sur la page Mes paramètres généraux, cliquez sur l'onglet Rapports par e-mail.

  3. Sur la page Mes paramètres de rapports par e-mail, effectuez l'une des actions suivantes :

    • Dans la liste Fréquence des rapports par e-mail, cliquez sur la fréquence à laquelle vous souhaitez recevoir des rapports par e-mail.

    • Dans la liste Format de l'e-mail, cliquez sur le format souhaité pour les rapports par e-mail.

    • Dans la zone Rapports sur les éléments évalués, activez la case à cocher Activer pour inclure les rapports sur les éléments évalués dans un rapport par e-mail.

  4. Cliquez sur Mettre à jour.
