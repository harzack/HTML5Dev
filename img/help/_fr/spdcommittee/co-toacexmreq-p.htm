Pour accepter les demandes d'abonnement aux groupes d'experts :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le lien Il existe des demandes d'adhésion en attente, puis activez les cases à cocher correspondant aux utilisateurs dont vous souhaitez accepter la demande d'adhésion.

  3. Cliquez sur Accepter.

  4. Sur la page Créer un e-mail, effectuez l'une des actions suivantes :

    • Tapez un message de bienvenue pour les nouveaux membres des groupes d'experts dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
