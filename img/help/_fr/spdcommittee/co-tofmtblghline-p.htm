Pour formater un titre de blog en un titre A la Une :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Contrôles du blog.

  2. Sur la page Contrôles du blog, cliquez sur l'onglet A la Une.

  3. Sur l'onglet A la Une, procédez comme suit dans les deux sections Titre principal et Tous les autres titres :

    • Cliquez sur une taille de police dans la liste Taille de police.

    • Saisissez un nombre de caractères dans le champ Longueur du message.

  4. Cliquez sur Terminé.
