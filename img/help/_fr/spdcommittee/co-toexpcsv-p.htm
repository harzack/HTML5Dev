Pour exporter un abonnement au format CSV :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur la liste Outils, puis sélectionnez Exporter vers CSV.

  3. Sur la page Exporter des membres vers un fichier CSV, sélectionnez ou désélectionnez les cases à cocher correspondant aux utilisateurs que vous voulez exporter au format CSV.

  4. Cliquez sur Exporter.

  5. Enregistrez le fichier CSV sur votre ordinateur.

  6. Cliquez sur Terminé.
