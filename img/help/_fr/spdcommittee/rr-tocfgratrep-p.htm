Pour configurer les rapports sur les éléments évalués :

  1. Sélectionnez Paramètres dans le menu Outils.

  2. Sur la page Mes paramètres généraux, cliquez sur l'onglet Rapports par e-mail.

  3. Sur l'onglet Rapports par e-mail, cliquez sur l'une des options suivantes dans la liste Fréquence des rapports par e-mail :

    • Immédiatement

    • Hebdomadaire

    • Journalier

  4. Cliquez sur l'une des options suivantes dans la liste Format de l'e-mail :

    • Message texte brut uniquement

    • Corps HTML uniquement

    • Message texte brut avec attachement HTML

  5. Activez la case à cocher Activer dans la zone Rapports sur les éléments évalués.

  6. Facultatif Cliquez sur le lien Afficher les éléments auditables pour afficher les types d'éléments inclus dans les rapports sur les éléments évalués.

  7. Cliquez sur Mettre à jour.
