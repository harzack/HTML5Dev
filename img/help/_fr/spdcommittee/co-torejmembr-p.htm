Pour rejeter des demandes d'abonnement :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur lien Il existe des demandes d'adhésion en attente.

  3. Activez la case à cocher correspondant à chaque utilisateur que vous souhaitez rejeter.

  4. Cliquez sur Rejeter.

  5. Sur la page Créer un message électronique, effectuez l'une des actions suivantes :

    • Saisissez un message expliquant les raisons pour lesquelles vous avez rejeté les demandes d'abonnement dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
