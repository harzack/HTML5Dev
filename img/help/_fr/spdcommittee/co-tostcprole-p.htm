Pour définir les autorisations de communauté pour un rôle :

  1. Dans le bandeau Console de navigation, sélectionnez Contrôles du modérateur, puis cliquez sur Modifier les rôles et les autorisations.

  2. Sur la page Autorisations, cliquez sur le nom du rôle pour lequel vous voulez définir les autorisations de communauté.

  3. Configurez les autorisations pour le rôle.

  4. Cliquez sur Mettre à jour, sur OK, puis sur Terminé.
