Pour configurer votre page d'accueil personnelle :

  1. Sur votre page d'accueil personnelle, cliquez sur le lien Page d'accueil personnelle sur le bandeau Console de navigation, puis cliquez sur Configuration.

  2. Sur la page Configuration, procédez comme suit :

    • Cliquez sur un nom de module dans le bandeau Modules de contenu, puis cliquez sur Ajouter.

    • Cliquez sur le bouton Modifier pour modifier les fonctionnalités de chaque module de contenu.

    • Cliquez sur le bouton Supprimer pour supprimer un module de contenu à partir de votre page d'accueil personnelle.

    • Glissez et déposez les modules de contenu en haut, en bas, à gauche ou à droite de la page pour repositionner les modules de contenu sur votre page d'accueil personnelle.

  3. Cliquez sur Terminé.
