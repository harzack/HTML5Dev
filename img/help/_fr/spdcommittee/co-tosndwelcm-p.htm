Pour envoyer un message de bienvenue aux nouveaux membres :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, sélectionnez le lien Cliquez ici dans le bandeau Nouveaux membres.

  3. Rédigez l'objet et le message, puis cliquez sur Envoyer.
