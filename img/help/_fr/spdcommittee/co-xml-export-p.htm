Pour exporter une communauté ou un contenu sélectionné :

  1. Cliquez sur l'icône Fonctions de l'élément, puis sur Exporter.

  2. Sur la page Exporter, cliquez sur l'une des options suivantes de la liste Portée :

    • Ce nœud uniquement

    • Ce nœud et le premier niveau enfant uniquement

    • Ce nœud et tous les nœuds enfant

  3. Pour inclure les informations sur la catégorie et les attributs d'un élément dans les données exportées, activez la case à cocher Inclure dans la zone Catégorie & Attributs.

  4. Dans la liste Versions, cliquez sur l'une des options suivantes :

    • Exporter toutes les versions du nœud

    • Exporter uniquement la version en cours

  5. Dans la liste Contenu, cliquez sur l'une des options suivantes :

    • Texte codé (en base 64)

    • Texte brut

  6. Pour inclure les informations sur les autorisations d'un élément dans les données exportées, cliquez sur la case à cocher Inclure dans la zone Autorisations.

  7. Cliquez sur Télécharger, puis enregistrez le fichier XML sur votre ordinateur.

  8. Cliquez sur Terminé.
