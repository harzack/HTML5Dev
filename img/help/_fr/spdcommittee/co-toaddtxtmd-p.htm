Pour ajouter un module Texte à la page d'accueil de la communauté :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur le lien Modifier les modules Texte.

  3. Sur la page Modules Texte, sélectionnez Module Texte dans le menu Ajouter un élément.

  4. Effectuez l'une des actions suivantes :

    • Saisissez l'intitulé du module Texte dans le champ Titre.

    • Activez la case à cocher Afficher le titre pour afficher le titre du module Texte et son contenu sur la page d'accueil de la communauté.

    • Tapez le corps de texte du module texte dans le champ Corps.

  5. Cliquez sur Ajouter.

  6. Sur la page Modules Texte, dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  7. Sur la page Configurer le paramétrage de la communauté, double-cliquez sur le module Texte dans le bandeau Modules de contenu.

  8. Placez le module Texte parmi les modules de contenu actifs.

  9. Cliquez sur Terminé.
