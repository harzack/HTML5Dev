Pour accepter les demandes d'annulation d'abonnement :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, sélectionnez les cases à cocher correspondant aux utilisateurs dont vous voulez accepter les demandes d'annulation d'abonnement, puis cliquez sur le lien accepter.

  3. Sur la page Créer un e-mail, effectuez l'une des actions suivantes :

    • Saisissez un message notifiant les utilisateurs que leurs demandes d'annulation d'abonnement ont été acceptées dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
