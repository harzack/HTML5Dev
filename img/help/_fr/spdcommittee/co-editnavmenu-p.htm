Pour modifier les options du bandeau Console de navigation :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur l'onglet Console de navigation.

     

    Note :
    Pour repositionner un lien, glissez et déposez ce lien à l'emplacement de votre choix dans la zone de la Console de navigation.

     

  3. Pour insérer des liens de communauté, activez le bouton radio Eléments de menu, sélectionnez un élément dans la liste Ajouter des éléments de menu, puis déplacez-le vers la zone de la Console de navigation.

  4. Pour insérer un raccourci vers un élément Content Server, activez le bouton radio Eléments Content Server, puis procédez comme suit :

    • Saisissez un nouveau titre pour le lien.

    • Cliquez sur Parcourir, puis recherchez et sélectionnez l'élément Content Server de votre choix.

    • Cliquez sur Ajouter.

    Pour insérer un lien externe, activez le bouton radio Liens externes, saisissez un titre pour le lien dans le champ Titre, puis indiquez l'URL du site Web dans le champ URL. Cliquez sur Ajouter.

  5. Pour insérer un séparateur entre les liens, activez le bouton radio Séparateur, puis cliquez sur Ajouter. Glissez et déposez le séparateur dans la Console de navigation.

  6. Pour supprimer une entrée dans la Console de navigation, mettez-la en surbrillance, puis cliquez sur Supprimer.

  7. Cliquez sur Terminé.
