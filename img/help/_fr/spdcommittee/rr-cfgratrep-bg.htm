La configuration des rapports sur les éléments évalués inclut les tâches suivantes :

  • Configuration de la fréquence de réception des rapports

  • Configuration du format d'e-mail pour les rapports

Fréquence des rapports et format des messages

Les paramètres de fréquence des rapports et de format des messages pour les rapports sur les éléments évalués s'appliquent uniquement aux rapports d'activité liés aux questions et aux rapports d'activité de la communauté. Pour de plus amples informations sur ces rapports par e-mail, reportez-vous à la section Utilisation de l'annuaire des communautés.

Types d'élément

Les types d'élément pouvant être inclus dans les rapports sur les éléments évalués sont configurés par votre administrateur.
