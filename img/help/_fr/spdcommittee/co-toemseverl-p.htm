Pour envoyer un e-mail à plusieurs membres de la communauté :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le bouton Envoyer un message, puis sélectionnez Envoyer un e-mail.

  3. Sur la page Créer un message électronique, activez ou désactivez les cases à cocher correspondant aux utilisateurs auxquels vous voulez envoyer un e-mail.

  4. Facultatif Saisissez les adresses e-mail de chaque nouveau destinataire en les séparant par des virgules dans le champ Adresses e-mail externes (par exemple tom@opentext.com, jeanne@opentext.com).

  5. Dans le champ Objet, saisissez l'objet du message.

  6. Dans la zone Message, saisissez le corps de l'e-mail.

  7. Facultatif Dans la zone Pièces jointes, cliquez sur le bouton Parcourir correspondant, sélectionnez un fichier, puis cliquez sur Ouvrir pour ajouter jusqu'à trois pièces jointes à cet e-mail.

  8. Cliquez sur Envoyer.
