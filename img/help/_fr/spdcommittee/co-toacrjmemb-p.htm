Pour accepter les demandes d'abonnement :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur lien Il existe des demandes d'adhésion en attente.

  3. Activez la case à cocher correspondant à chaque utilisateur dont vous souhaitez accepter l'abonnement.

  4. Sélectionnez un rôle dans la liste Rôle pour assigner un rôle à un utilisateur accepté.

  5. Cliquez sur Accepter.

  6. Sur la page Créer un e-mail, effectuez l'une des actions suivantes :

    • Tapez un message de bienvenue pour les nouveaux membres de la communauté dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
