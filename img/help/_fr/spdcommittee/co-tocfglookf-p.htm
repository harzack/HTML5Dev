Pour configurer une communauté :

  1. Ouvrez une communauté.

  2. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  3. Spécifiez les paramètres de votre choix sur les onglets suivants :

    • Page d'accueil

    • Bannière de la page d'accueil

    • ‎Console de navigation

    • Configuration

    • Résumé

  4. Cliquez sur Terminé.
