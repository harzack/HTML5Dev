Pour importer une communauté ou un contenu sélectionné :

  1. Cliquez sur l'icône Fonctions de l'élément de destination, puis sur Importer.

  2. Sur la page Importer, cliquez sur Parcourir dans la zone Importer le fichier, sélectionnez un fichier, puis cliquez sur Ouvrir.

  3. Dans la zone Destination, cliquez sur Parcourir Content Server, naviguez jusqu'à l'élément de destination, puis cliquez sur son lien Sélectionner.

  4. Cliquez sur Importer, puis sur Terminé.
