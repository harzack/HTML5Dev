Pour activer les questions externes :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur l'onglet Configuration.

  3. Activez la case à cocher Permettre dans la zone Questions externes.

  4. Cliquez sur Terminé.
