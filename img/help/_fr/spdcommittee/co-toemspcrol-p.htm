Pour envoyer un e-mail à un rôle spécifique :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le bouton Envoyer un message, puis sélectionnez Envoyer un e-mail.

  3. Sur la page Créer un message électronique, activez la case à cocher de chaque rôle auquel vous souhaitez envoyer des e-mails, puis activez l'une des cases à cocher suivantes pour chaque rôle :

    • Direct

    • Indirect

     

    Note :
    L'affichage des membres indirects peut être désactivé par le modérateur.

     

  4. Facultatif Saisissez les adresses e-mail des nouveaux destinataires en les séparant par des virgules dans le champ Adresses e-mail externes (par exemple tom@opentext.com, jeanne@opentext.com).

  5. Dans le champ Objet, saisissez l'objet du message.

  6. Dans la zone Message, saisissez le corps de l'e-mail.

  7. Facultatif Pour vérifier l'orthographe du corps de votre message, cliquez sur le bouton Vérifier l'orthographe .

  8. Pour inclure jusqu'à trois pièces jointes à l'e-mail, cliquez sur le bouton Parcourir correspondant dans la section Pièces jointes, sélectionnez un fichier, puis cliquez sur Ouvrir.

  9. Cliquez sur Envoyer.
