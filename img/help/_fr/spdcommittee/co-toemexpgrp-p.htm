Pour envoyer un e-mail à un groupe d'experts :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le lien Domaine d'expertise dans le menu Liens connexes.

  3. Sur la page Experts de la communauté, cliquez sur le nom du groupe d'experts auquel vous voulez envoyer un e-mail.

  4. Sur la page Groupes d'experts, cliquez sur l'icône Envoyer un e-mail aux experts .

  5. Facultatif Pour ajouter les adresses e-mail de destinataires externes à la liste, saisissez ces adresses en les séparant par des virgules dans le champ Adresses e-mail externes (par exemple, tom@opentext.com, jeanne@opentext.com).

  6. Sur la page Envoyer un e-mail aux experts, saisissez l'objet de votre e-mail dans le champ Objet et le corps de votre e-mail dans le champ Message.

  7. Facultatif Pour inclure jusqu'à trois pièces jointes à l'e-mail, cliquez sur le bouton Parcourir correspondant dans la section Pièces jointes, sélectionnez un fichier, puis cliquez sur Ouvrir.

  8. Cliquez sur Envoyer.
