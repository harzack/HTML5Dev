Pour configurer le résumé de la communauté :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur l'onglet Résumé.

  3. Sur la page Résumé, sélectionnez une plage de dates dans la liste Plage de dates à afficher.

  4. Pour activer une catégorie, sélectionnez les cases d'option Activé suivantes :

    • Identifiants

    • Nouveaux membres

    • Membres supprimés

    • Eléments les plus affichés

  5. Pour spécifier les détails à afficher, activez les cases à cocher suivantes pour chaque catégorie (sauf Eléments les plus consultés) :

    • Date

    • Nom

    • Rôle

    • E-mail

    • Téléphone

  6. Si vous avez activé la catégorie Eléments les plus consultés, cliquez sur le nombre d'éléments à afficher dans le résumé de la communauté dans la liste déroulante Nombre d'éléments à afficher.

  7. Cliquez sur Terminé.
