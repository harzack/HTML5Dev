Pour rejeter des demandes d'annulation d'abonnement :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, sélectionnez les cases à cocher correspondant aux utilisateurs pour lesquels vous voulez rejeter les demandes d'annulation d'abonnement, puis cliquez sur le lien rejeter.

  3. Sur la page Créer un message électronique, effectuez l'une des actions suivantes :

    • Saisissez un message expliquant les raisons pour lesquelles vous avez rejeté les demandes d'annulation d'abonnement dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
