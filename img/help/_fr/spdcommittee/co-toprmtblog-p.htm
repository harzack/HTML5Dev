Pour promouvoir un blog en page d'accueil de communauté :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Contrôles du blog.

  2. Sur la page Contrôles du blog, cliquez sur le bouton Ajouter un blog .

  3. Dans la fenêtre Ajouter, naviguez jusqu'au blog que vous voulez promouvoir, puis cliquez sur le lien Sélectionner du blog.

  4. Sur la page Contrôles du blog, cliquez sur l'une des options suivantes dans la liste Style du rapport :

    • Blog à la une

    • Auteur à la une

  5. Cliquez sur Terminé.

  6. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  7. Sur la page Configurer le paramétrage de la communauté, double-cliquez sur le nom du blog dans la liste Modules de contenu.

  8. Cliquez sur Terminé.
