Pour définir des notifications pour un rôle :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Notification.

  2. Sur la page Notification de la communauté, cliquez sur le lien du rôle pour lequel vous voulez définir des notifications.

  3. Cliquez sur le bouton Modifier les centres d'intérêt , puis modifiez les paramètres et les centres d'intérêt pour le rôle.

  4. Cliquez sur Soumettre.

 

Astuce :
Vous pouvez également définir des notifications pour un rôle sur la page Informations sur les membres. Pour cela, cliquez sur le menu Outils, choisissez Définir les notifications, puis sélectionnez un rôle.

 
 

Note :
Pour de plus amples informations sur la définition de notifications, reportez-vous à la section Utilisation des notifications.
