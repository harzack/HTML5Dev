Pour télécharger un rendez-vous dans Microsoft Outlook :

  1. Cliquez sur le lien correspondant au rendez-vous à télécharger.

  2. Sur la page Modifier l'évènement, cliquez sur le lien Télécharger vers votre calendrier.

  3. Dans la boîte de dialogue Téléchargement de fichier, cliquez sur Ouvrir.

  4. Dans la fenêtre Rendez-vous de Microsoft Outlook, changez les options de votre choix, puis cliquez sur Enregistrer et fermer.
