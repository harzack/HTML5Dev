Pour ajouter des rôles :

  1. Dans le bandeau Console de navigation, sélectionnez Contrôles du modérateur, puis cliquez sur Modifier les rôles et les autorisations.

  2. Sur la page Autorisations, cliquez sur l'icône Ajouter un rôle .

  3. Saisissez le nom du rôle dans le champ Nom du rôle.

  4. Configurez les autorisations pour le rôle.

  5. Cliquez sur Ajouter, sur OK, puis sur Terminé.
