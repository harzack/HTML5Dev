Pour définir votre photo personnelle :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur votre nom dans la liste des membres.

  3. Sur la page Informations sur les membres de la communauté, cliquez sur le lien Mettre à jour la photo.

  4. Sur la page Mettre à jour la photo personnelle, effectuez l'une des actions suivantes :

    • Pour télécharger un fichier depuis votre ordinateur, cliquez sur Parcourir, recherchez le fichier image de votre choix, puis cliquez sur Ouvrir.

    • Pour télécharger une image depuis Content Server, cliquez sur Parcourir Content Server, recherchez le fichier image de votre choix, puis cliquez sur le lien Sélectionner correspondant.

     

    Note :
    Par défaut, la taille des photos doit être inférieure à 50 000 octets. Les dimensions optimales pour une image sont 80 par 80 pixels.

     

  5. Cliquez sur Mettre à jour.
