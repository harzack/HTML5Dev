Pour promouvoir un blog en module de contenu A la Une

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Contrôles du blog.

  2. Sur la page Contrôles du blog, cliquez sur l'onglet A la Une.

  3. Sur la page A la Une, cliquez sur le bouton Ajouter un blog .

  4. Dans la page Ajouter, naviguez jusqu'au blog que vous voulez promouvoir, puis cliquez sur le lien Sélectionner du blog.

  5. Cliquez sur Terminé.
