Pour effectuer une recherche dans l'archive de messagerie :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Archive de messagerie, si celui-ci est disponible.

  2. Sur la page Archive de messagerie, effectuez l'une des actions suivantes :

    • Dans le champ Rechercher, saisissez vos critères de recherche pour lancer une recherche par objet, nom de connexion, nom de famille ou prénom de l'expéditeur.

      Vous pouvez restreindre vos paramètres de recherche à l'aide du champ Rechercher par date et de la liste Par.

    • Dans le champ Rechercher du contenu pour, saisissez vos critères de recherche pour faire porter la recherche sur tout le contenu des e-mails.

      Vous pouvez recibler votre recherche en cliquant sur une cible différente à partir du champ Rechercher par date et de la liste A partir d'ici.

  3. Cliquez sur le bouton Aller correspondant à votre type de recherche.

 

Astuce :
Si vous laissez le champ De vierge lorsque vous effectuez une recherche par date, Content Server effectue la recherche en incluant les éléments dont la date de création est la plus ancienne. De la même manière, si vous laissez le champ A vierge, Content Server effectue la recherche en incluant les éléments les plus récents.
