Pour rechercher des membres dans une communauté :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, saisissez les critères de recherche pour le membre que vous recherchez dans le champ Liste Rechercher dans.

 

Note :
Vous pouvez rechercher en saisissant le prénom, nom complet, nom de connexion ou adresse e-mail d'un utilisateur. Vous ne pouvez pas effectuer de recherches à partir d'un rôle ou d'un nom du groupe. Vous pouvez toutefois filtrer la liste par type de rôle ou groupe d'experts en utilisant la liste en regard du champ Recherche dans la liste.

Si vous filtrez les résultats de la recherche par groupe d'experts, les noms de groupe d'experts sont traités en premier. Si aucun groupe d'experts ne correspond aux critères de recherche que vous avez saisis dans le champ Recherche dans la liste, les détails de l'utilisateur (par exemple son prénom, son nom de famille, son nom de connexion et son adresse e-mail) seront alors recherchés.
