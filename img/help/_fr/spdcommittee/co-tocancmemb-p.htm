Pour demander l'annulation de l'abonnement à une communauté :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur <votre_nom> dans la liste des membres.

  3. Sur la page Informations sur les membres, cliquez sur le lien Annuler l'abonnement.

  4. Sur la page Annuler l'abonnement, saisissez la raison pour laquelle vous souhaitez annuler votre abonnement dans le champ Motif.

  5. Cliquez sur Envoyer.
