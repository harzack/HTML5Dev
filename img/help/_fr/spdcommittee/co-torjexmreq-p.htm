Pour rejeter des demandes d'abonnement aux groupes d'experts :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Sur la page Intervenants de la communauté, cliquez sur le lien Il existe des demandes d'adhésion en attente, puis activez les cases à cocher correspondant aux utilisateurs dont vous souhaitez rejeter la demande d'adhésion.

  3. Cliquez sur Rejeter.

  4. Sur la page Créer un message électronique, effectuez l'une des actions suivantes :

    • Saisissez un message expliquant les raisons pour lesquelles vous avez rejeté les demandes d'abonnement aux groupes d'experts dans le champ Message, puis cliquez sur Envoyer.

    • Cliquez sur Continuer sans envoyer.
