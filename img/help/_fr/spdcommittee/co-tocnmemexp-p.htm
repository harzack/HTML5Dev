Pour annuler l'abonnement à un groupe d'experts :

  1. Dans le bandeau Console de navigation, cliquez sur le lien Informations sur les membres.

  2. Dans le menu Liens associés, cliquez sur le lien Domaine d'expertise.

  3. Sur la page Groupes d'experts, cliquez sur le nom du groupe d'experts pour lequel vous voulez annuler votre abonnement.

  4. Cliquez sur le lien Me supprimer de ce groupe d'experts.

  5. Sur la page Demande d'abonnement au groupe d'experts, saisissez la raison pour laquelle vous voulez annuler votre abonnement dans le champ Motif.

  6. Cliquez sur Envoyer.
