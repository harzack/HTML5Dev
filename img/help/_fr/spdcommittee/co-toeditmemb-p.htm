Pour modifier un abonnement :

  1. Dans le bandeau Console de navigation, cliquez sur Informations sur les membres.

  2. Dans le menu Outils, sélectionnez Modifier les membres.

  3. Sélectionnez un membre dans la zone Membres actuels, puis modifiez le rôle du membre de votre choix dans la liste Rôle.

  4. Cliquez sur Mettre à jour, puis sur Terminé.
