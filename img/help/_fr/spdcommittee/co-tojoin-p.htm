Pour faire une demande d'abonnement à une communauté :

  1. Cliquez sur l'icône Fonctions d'une communauté, puis cliquez sur Demande d'abonnement.

  2. Sur la page Demande d'abonnement, cliquez sur le rôle que vous souhaitez dans la communauté dans la liste Rôle de préférence.

  3. Facultatif Pour entrer plusieurs adresses e-mail, saisissez les adresses de vos parrains en les séparant par des virgules dans le champ Parrain(s).

  4. Facultatif Saisissez les raisons de votre demande d'abonnement à la communauté dans le champ Message facultatif.

  5. Cliquez sur Soumettre.
