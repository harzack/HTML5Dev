Pour ajouter un module URL à la page d'accueil de la communauté :

  1. Dans le bandeau Console de navigation, cliquez sur Contrôles du modérateur, puis sur Configuration.

  2. Sur la page Configurer le paramétrage de la communauté, cliquez sur le bouton Modifier les URL .

  3. Sur la page Modifier les URL, cliquez sur le bouton Ajouter une URL .

  4. Sur la page Ajouter une URL, procédez comme suit :

    • Saisissez un nom pour l'URL dans la zone Nom.

    • Saisissez l'URL de la page souhaitée dans le champ URL.

  5. Cliquez sur Ajouter.

  6. Sur la page Modifier les URL, cliquez sur Terminé.

  7. Sur la page Configurer les paramètres de la communauté, sélectionnez l'URL dans la liste Modules de contenu, puis cliquez sur Ajouter.

  8. Faites glisser le module de contenu de l'URL dans la position de votre choix parmi les autres modules de contenu actifs.

  9. Cliquez sur Terminé.

 

Note :
Pour de plus amples informations sur l'ajout d'URL, reportez-vous à la section Utilisation des raccourcis et des adresses URL.
