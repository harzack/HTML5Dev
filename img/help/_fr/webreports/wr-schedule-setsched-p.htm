Pour définir un programme :

  1. A partir du menu Fonctions d'un WebReport pour lequel vous souhaitez définir un programme, sélectionnez Propriétés, puis Destination.

  2. Sur la page Destination, choisissez la destination de sortie souhaitée, Content Server ou e-mail, puis remplissez les informations requises pour cette destination de sortie.

  3. Sélectionnez la case à cocher Définir le programme. Un ensemble d'options supplémentaires apparaît ensuite :

    1. Dans les champs Prochaine exécution après, définissez la date et l'heure de la première exécution du rapport. Notez que l'exécution peut avoir lieu jusqu'à 5 minutes après l'heure prévue. En général, les rapports sont exécutés à l'heure dite, puis 5 minutes après l'heure.

    2. Dans le champ Exécuter, sélectionnez Toujours ou, dans le champ Nombre de fois, saisissez le nombre de fois que le rapport doit s'exécuter.

    3. Définissez l'intervalle entre les exécutions du rapport en minutes, heures, jours et semaines.

  4. Cliquez sur Mettre à jour pour définir le programme.
