Lorsqu'il est utilisé au sein d'un fichier Bon de travail dans l'attribut FOLDER, $DEFAULT_PATH insert de manière dynamique le chemin d'accès approprié comme indiqué dans la page Administration Gérer la conversion WebReports.

Il existe deux ensembles de répertoires qui peuvent être spécifiés dans la page Administration, et si l'ensemble de répertoires pouvant être utilisé avec les bons de travail XML a été spécifié, la valeur $DEFAULT_PATH renverra ces chemins d'accès. S'il n'est pas spécifié, elle renverra le premier ensemble de répertoires de conversion spécifié pour conversion.

Etant donné qu'il ne s'agit pas d'une constante, vous pouvez utiliser cette valeur au sein de l'attribut FOLDER dans les deux balises DOCINPUT et DOCOUTPUT. L'utilisation de la valeur $DEFAULT_PATH renverra le chemin d'accès adéquat vers les dossiers parents.

Exemple

<JOB:DOCINPUT FILENAME="" FOLDER="$DEFAULT_PATH" />
<JOB:DOCOUTPUT FILENAME="" FOLDER="$DEFAULT_PATH" DOCTYPE="PDF" />
