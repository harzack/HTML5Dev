Cette section apporte des informations sur la sous-balise BITCHECK.

BITCHECK

Syntaxe

BITCHECK:<valeur du masque de bits>

Description

Teste une valeur d'entier renvoyée par la balise pour déterminer si un bit binaire particulier a été défini. La valeur du masque de bits doit être un nombre décimal ordinaire qui est utilisé pour déterminer les bits binaires à tester. S'il est déterminé que les bits attendus sont définis, cette sous-balise renvoie alors true, sinon, elle renvoie false.

Exemple

Pour tester si les premier et quatrième bits, de droite à gauche, ont été définis, le masque doit être 1001 en binaire. A l'aide d'un convertisseur décimal-binaire, le nombre 1001 est converti au nombre décimal 9. Par conséquent, la syntaxe BITCHECK:9 doit être utilisée. Si la balise correspondante a renvoyé une valeur de 20, décimale, alors la séquence de bits testée est : 10100. Le masque de 9 détermine que ni le bit 0 ni le bit 3 ne sont définis, si bien que BITCHECK:9 renvoie « FALSE ». Pour vérifier si le troisième bit en partant de la droite a été défini, le masque requis est le nombre binaire 100, soit le nombre décimal 4. Si la balise correspondante renvoie un décimal 15, soit le nombre binaire 1111, alors BITCHECK:4 renvoie « TRUE ».
