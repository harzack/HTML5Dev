Vous pouvez utiliser des sous-balises sans invoquer également la balise de donnée statique, au moyen de la fonction point suivante :

Fonction Paramètres Renvoie
String ._subTag(String data, String subtags)
data : valeur de chaîne contenant la donnée traitée par les sous-balises.
subtags : valeur de chaîne contenant les sous-balises à utiliser sur la donnée.
 Une valeur de chaîne contenant le résultat de la donnée transformée par les sous-balises.

 

Exemple: String s = ._subTag('3','DECODE:1:Beginning:2:Middle:3:End:Other UPPER')

Dans ce cas, la chaîne « s » est définie sur END.

Notez que certaines sous-balises ne sont pas prises en charge par ._subTag. En général, les sous-balises qui exécutent une autre action que renvoyer des données formatées ne fonctionnent pas. Les sous-balises suivantes ne sont pas prises en charge : ADDVAR, CONCATVAR, NEXT, PREV, SETFORM, SETVAR, SETWFATTR, SETWFFORM.
