• Renvoie un tableau JS (multidimensionnel) comportant tous les fieldId et valeurs d'attributs. Par exemple, [ ('_1_1_2_1', '2008/2/5:0:0:0'), ('_1_1_3_1', true), ... ]

  • Traite les arguments suivants :

    • categoryID : paramètre obligatoire.

    • nodeID : paramètre obligatoire.

  • En fait, tout ce dont vous avez besoin pour effectuer l'envoi des données de catégorie.

  • Renvoie la valeur null si une erreur survient ou si les paires fieldId/valeur sont introuvables.
