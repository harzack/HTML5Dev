Les sources de données WebReport peuvent être manipulées à l'aide de paramètres spéciaux appelés paramètres de source de données. Voici quelques exemples de sources de données WebReport : rapports LiveReports, requêtes de recherche ou formulaires. Ces paramètres sont transmis dans l'URL lorsqu'un WebReport est exécuté et contrôlent des éléments tels que le nombre maximum de résultats sur chaque page :
