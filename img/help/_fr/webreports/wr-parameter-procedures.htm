Les procédures suivantes décrivent l'ajout manuel et automatique d'un paramètre WebReport, le référencement d'un paramètre WebReport dans un affichage de rapport et la suppression d'un paramètre WebReport.

Vous devez ajouter manuellement un paramètre WebReport lorsque vous souhaitez définir des paramètres que vous n'avez pas encore utilisés dans le WebReport. Lorsqu'ils ont été définis, vous pouvez les référencer dans l'affichage de rapport. Pour en savoir plus, voir Pour ajouter manuellement un paramètre WebReport.

Vous devez ajouter automatiquement un paramètre WebReport lorsque vous avez fait référence à un paramètre, ou à plusieurs, dans l'affichage de rapport, et que vous souhaitez les remplir rapidement dans l'onglet Paramètres. En outre, les paramètres définis dans les sources de données de rapport LiveReport sont également remplis dans l'onglet. Cette méthode est plus rapide que la méthode manuelle décrite ci-dessus, car elle permet de renseigner tous les paramètres en une seule fois. Le cas échéant, elle permet également de déterminer le type de paramètre par programmation. Lorsque ce n'est pas possible, le type de paramètre sera string. Pour en savoir plus, voir Pour ajouter manuellement un paramètre WebReport.

Lorsqu'un paramètre a été défini, la valeur correspondante peut être insérée dans l'affichage de rapport autant de fois que nécessaire. Il n'est pas indispensable de définir un paramètre avant d'ajouter la balise correspondante dans l'affichage de rapport. Il peut être plus rapide de les définir via la méthode automatique ci-dessus lorsque vous avez terminé de modifier l'affichage de rapport. Si vous travaillez dans l'éditeur en ligne WebReports, il est possible d'utiliser l'interface glisser-déposer pour accélérer certaines activités. Pour en savoir plus, voir Pour référencer des paramètres dans l'affichage de rapport WebReport.
