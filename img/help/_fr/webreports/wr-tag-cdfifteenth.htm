Cette section apporte des informations sur les balises de données de colonne : URLDOMAIN, URLPREFIX, URLPREFIXFULL, URLPROTOCOL, USERFULLNAME,USERID et USERNAME.

URLDOMAIN

Syntaxe

[LL_REPTAG_URLDOMAIN /]

Description

Renvoie le domaine ou le nom d'hôte utilisé pour accéder à l'instance. Par exemple : www.theDomain.com.

URLPREFIX

Syntaxe

[LL_REPTAG_URLPREFIX /].

Description

URL requise pour appeler le CGI Content Server. Par exemple : Livelink/livelink.exe. Peut être utilisé pour créer un lien hypertexte dans le WebReport qui exécute une commande Content Server. Par exemple, ouvrir un formulaire pour les données dans le WebReport.

URLPREFIXFULL

Syntaxe

[LL_REPTAG_URLPREFIXFULL /]

Description

Renvoie l'URL complète requise pour appeler le CGI Content Server comprenant le protocole et le domaine. Par exemple : http://www.theDomain.com/Livelink/livelink.exe. Peut être utilisé pour créer un lien hypertexte dans le WebReport qui exécute une commande Content Server. Par exemple, ouvrir un formulaire pour les données dans le WebReport.

URLPROTOCOL

Syntaxe

[LL_REPTAG_URLPROTOCOL /]

Description

Renvoie le protocole utilisé pour accéder à l'instance. Par exemple : http:// ou https://.

USERFULLNAME

Syntaxe

[LL_REPTAG_USERFULLNAME /]

Description

Nom complet de l'utilisateur exécutant le WebReport.

USERID

Syntaxe

[LL_REPTAG_USERID /]

Description

ID d'utilisateur pour l'utilisateur exécutant le WebReport. Si la fonction RunAs est définie, cette balise renverra l'ID d'utilisateur de l'utilisateur RunAs. Utilisez la balise [LL_REPTAG_ACTUALUSERID /] pour obtenir l'ID d'utilisateur actuel.

Exemple

Par exemple, si l'utilisateur 1 exécute un WebReport dont le paramètre RunAs est défini sur Utilisateur 2, la balise ACTUALUSERID renverra l'ID de l'utilisateur 1 tandis que la balise USERID renverra l'ID de l'utilisateur 2.

USERNAME

Syntaxe

[LL_REPTAG_USERNAME /]

Description

Nom de l'utilisateur exécutant le WebReport.
