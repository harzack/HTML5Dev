La fonction Constantes WebReports permet de prédéfinir des données qui peuvent être insérées dans la sortie du rapport. Cette fonction est généralement utilisée dans les situations suivantes :

  • Lorsque du texte ou des informations du WebReport nécessitent des modifications périodiques, une constante peut être utilisée afin de faciliter la maintenance qui peut être effectuée par une personne non spécialisée.

  • Lorsque des valeurs doivent être définies manuellement, comme des numéros d'attribut, les constantes simplifient la définition et la gestion de ces paramètres.

  • Lorsqu'un même affichage de rapport est utilisé pour plusieurs WebReports de différentes manières, chaque WebReport peut avoir différents paramètres de constante.

  • Lorsqu'un affichage de rapport est créé avec des références qui peuvent varier selon les systèmes, les constantes permettent de gérer la transparence du système.

     

    Note :
    La fonction d'importation/exportation XML tente de traduire des références aux objets Content Server entre les systèmes.
