Pour supprimer ou modifier un programme :

  1. Pour supprimer un programme, à partir du menu Fonctions du WebReport, sélectionnez Propriétés, puis Destination.

  2. Sur la page Destination, désactivez la case à cocher Définir le programme.

  3. Cliquez sur Mettre à jour.

     

    1. D'autres paramètres de programme seront conservés, mais le rapport ne s'exécutera plus jusqu'à ce que la case à cocher Définir le programme soit sélectionnée. Pour modifier les paramètres, effectuez vos changements, puis cliquez sur Mettre à jour.

    2. Si l'exécution d'un rapport est programmée pour un certain nombre de fois, le nombre restant et l'heure suivante d'exécution seront mis à jour par Content Server, vous permettant de voir combien d'exécutions sont attendues et quand aura lieu la prochaine.

    3. Notez que l'administrateur Content Server a le pouvoir de désactiver ou supprimer les programmes créés par d'autres utilisateurs. Pour ce faire, il se rend sur les pages Administration de Content Server. Pour plus d'informations, voir le Manuel d'installation et d'administration WebReports.
