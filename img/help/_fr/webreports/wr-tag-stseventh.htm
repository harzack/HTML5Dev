Cette section apporte des informations sur les sous-balises suivantes : GETCVID et GETTEXT.

GETCVID

Syntaxe

GETCVID

Description

Obtient l'ID de nœud d'un Affichage personnalisé s'il existe dans le recueil spécifié par son ID de nœud fourni à cette sous-balise.

GETTEXT

Syntaxe

GETTEXT

Description

Renvoie le contenu d'un document basé sur du texte spécifié par l'ID de nœud fourni à cette sous-balise.
