Exemple 1-11 : DSendrow

Si la source de données possède 100 lignes de données et si un paramètre DSendrow de 75 lignes est spécifié sans aucun paramètre DSstartrow ou DSmaxrows, alors seules les lignes de 1 à 75 sont utilisées.

Exemple 1-12 : DSmaxrows

Si les paramètres suivants sont configurés :

DSstartrow=1, DSendrow=75 et DSmaxrows=20

seules les lignes de 1 à 20 sont utilisées.

Exemple 1-13 : DSendrow et DSmaxrows

Si les paramètres suivants sont configurés :

DSstartrow=1, DSendrow=20 et DSmaxrows=100

seules les lignes de 1 à 20 sont utilisées.

Exemple 1-14 : Exemple d'appel d'une URL

Examinez les éléments suivants :

?func=ll&objId=1234&objAction=RunReport&sourceid=12345&DSstartrow=11&DSendrow=100&DSmaxrows=50

Dans cet exemple, la source de données identifiée par nodeid 12345 est utilisée afin de fournir des données pour le WebReport. Les données utilisées se terminent à la ligne 50, même si le paramètre DSendrow est configuré sur 100, car le paramètre DSmaxrows est configuré sur 50.
