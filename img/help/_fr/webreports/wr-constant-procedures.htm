Les procédures suivantes indiquent comment :

  1. Ajouter manuellement ou automatiquement une constante : utilisez cette procédure lorsque vous souhaitez définir une constante que vous n'avez pas encore utilisée dans le WebReport. Lorsqu'elle a été définie ici, vous pouvez la référencer dans l'affichage de rapport.

  2. Faire glisser et déposer une constante dans un affichage de rapport : utilisez cette procédure lorsque vous avez référencé une ou plusieurs constantes dans l'affichage de rapport et que vous souhaitez les renseigner rapidement dans l'onglet Constantes. Cette méthode est plus rapide que la méthode manuelle décrite dans Pour ajouter manuellement une constante pour le WebReport, car elle permet de renseigner toutes les constantes en une seule fois.

  3. Supprimer une constante.
