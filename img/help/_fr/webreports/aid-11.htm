Pour modifier la source de données associée d'un WebReport à partir du menu Fonctions du WebReport, sélectionnez Propriétés, puis Source. L'écran Source permet de sélectionner une nouvelle source de données à l'aide du navigateur Content Server standard. Une fois ce formulaire mis à jour, toutes les exécutions à venir de ce WebReport utiliseront la source de données qui a été sélectionnée à l'aide du navigateur.
