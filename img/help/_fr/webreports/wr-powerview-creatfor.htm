Les développeurs de formulaires ont la possibilité d'ajouter un WR Power View à un modèle de formulaire en plus de la traditionnelle vue HTML. Cette nouvelle vue de formulaire se comporte comme une vue HTML avec quelques nouveautés utiles :

  • Une prise en charge native pour les balises de formulaire existantes comme [LL_FormTag_1_1_2_1 /] et [LL_FUNC /], c'est-à-dire que les Power Views sont rétrocompatibles avec les vues HTML. Afin de tirer parti de cette fonction, il vous suffit de télécharger la dernière version de votre vue de formulaire et de l'ajouter en tant que version d'affichage de rapport à un nouveau WebReport ajouté à partir de l'option WR Power View.

  • Prise en charge complète pour les balises statiques WebReports, constantes et paramètres qui peuvent être utilisés associés aux balises de formulaire.

  • Appelez un nombre illimité de WebReports, via des appels de sous-rapports WebReports, dans la vue activant le remplissage dynamique des champs de recherche à partir de sources de données à la fois internes et externes à Content Server.

  • Sélectionnez une WR Power View pour un formulaire de workflow et utilisez les balises de paramètre WebReports, [LL_REPTAG_& /], pour accéder aux informations concernant le workflow en cours d'exécution. Par exemple, workid, subworkid , taskid, mapid, puis utilisez-les pour rechercher plus d'informations spécifiques à l'étape actuelle, à l'instance du workflow ou de l'utilisateur.

  • Utilisez des formulaires provisoires pour exécuter des rapports dans un workflow.

Pour une utilisation instantanée de ces fonctions :

  • Depuis le menu Fonctions du modèle de formulaire, sélectionnez Exporter au format HTML, puis enregistrez le fichier sur votre bureau.

  • Allez dans Propriétés, sélectionnez ensuite l'onglet Vues depuis le modèle et sélectionnez Ajouter une vue suivi de WR Power View.

  • Lorsque vous sélectionnez l'affichage de rapport, recherchez le fichier enregistré sur votre bureau lors de la première étape.

  • Ouvrez le guide sur les balises WebReports et commencez à utiliser instantanément plus de 100 balises dans votre vue du formulaire.

 

Note :
Le module WebForms doit être installé pour utiliser cette fonction.
