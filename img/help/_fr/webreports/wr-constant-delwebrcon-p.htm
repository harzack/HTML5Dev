Pour supprimer une constante WebReport :

  1. Sélectionnez l'onglet Constantes pour le WebReport. Dans le menu Fonctions du WebReport, sélectionnez Propriétés. Dans le menu Propriétés, sélectionnez Constantes.

  2. Dans l'onglet Constantes de la page Propriétés du WebReport, sélectionnez le bouton moins - en regard de la constante que vous souhaitez supprimer.

  3. Cliquez sur OK pour confirmer que vous souhaitez supprimer cette constante.

  4. Cliquez sur Mettre à jour.

     

    Astuce :
    Si vous souhaitez supprimer toutes les définitions de constantes de l'ensemble du WebReport, cliquez sur le bouton X rouge, puis sélectionnez Mettre à jour.
