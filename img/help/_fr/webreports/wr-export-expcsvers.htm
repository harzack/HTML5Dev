Exemple 1-19 : Exemple de l'appel d'une URL lors d'une exportation vers une version Content Server :

?func=ll&objId=1234&objAction=RunReport&export_location=livelink&livelink_node=livelinkversion&newVersion_ID=5678&mimetype=text%2Fplain&nexturl=

Dans cet exemple, WebReport 1234 ajoute une nouvelle version de document Content Server avec un type Mime texte normal à l'ID de document 5678.
