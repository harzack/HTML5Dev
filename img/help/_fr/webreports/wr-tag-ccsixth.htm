Cette section apporte des informations sur la balise de contrôle de contenu JSLIBS.

JSLIBS

Syntaxe

[LL_WEBREPORT_JSLIBS /]

ou

[LL_WEBREPORT_JSLIBS <liste de bibliothèques> /]

Description

Cette balise prend en charge un ou plusieurs sélecteurs de bibliothèque, comme par exemple COREUTIL qui déterminent quelles bibliothèques JavaScript insérer dans la sortie.

Paramètres

Les bibliothèques actuellement disponibles pour les versions Content Server 10.0.0 et supérieures sont :

 

Note :
COREMENU et PAGINATION sont obsolètes. Utilisez plutôt BROWSE et COREAJAX.

 

BASE
(webdav/global.js, menu.js)
Bibliothèques de base utilisées pour les menus et les fonctions Content Server standard.
BASICBROWSE
(checkall.js)
Fonctionnalité liée à la sélection d'objets multiples.
BROWSE
(webnode/browsecoretable_en_US.js, webnode/browse.js)
Fonctions utilisées pour browseview.
COREUTIL
(core/ot_utilities.js)
Utilitaires divers.
COREAJAX
(core/ajax_dhtml_util.js)
Utilitaires Ajax.
SEARCHBROKER
(websbroker/search.js, websbroker/sortwidget.js)
Utilitaires de résultats de recherche

Les bibliothèques actuellement disponibles pour les versions Livelink 9.7.1 et supérieures sont :

BASE
(global.js,globalmenu.js,globalmenus.js,newsticker.js,functionmenu.js,menu.js)
Bibliothèques de base utilisées pour les menus et les fonctions Livelink standard.
BASICBROWSE
(checkall.js)
Fonctionnalité liée à la sélection d'objets multiples.
BROWSE
(browsecoretable_en_US.js, browsecoretable.js)
Nouvelles fonctions, à partir de Livelink 9.7.1, utilisées pour browseview.
COREMENU
(browsecorermenu.js)
Nouvelle fonctionnalité de menu, à partir de Livelink 9.7.1.
COREUTIL
(otutilities.js)
Utilitaires divers.
COREAJAX
(otajaxsupport.js)
Utilitaires Ajax.
PAGINATION
(minipaginationcontrol.js)
Les contrôles et/ou les fonctions utilisées pour afficher les options de pagination, à partir de Livelink 9.7.1.
SEARCHBROKER
(search.js, sortwidget.js)
Utilitaires de résultats de recherche.
