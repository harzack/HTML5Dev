Cette section apporte des informations sur les balises de contrôle de contenu : // et [/* ... */].

//

Syntaxe

[//

Description

Utilisée pour commenter le restant de la ligne. Utiliser cette balise plutôt que des balises de commentaire HTML permet d'éviter que le commentaire ne soit inscrit dans la sortie. Pour des rapports importants, cela présente des avantages considérables pour la restriction de la quantité de données envoyées vers chaque navigateur. La conversion de tous les commentaires afin d'utiliser les balises de commentaire WebReports peut apporter des avantages en matière de performances pour les rapports importants.

Exemple

[LL_REPTAG=dueDate DATEDEC:6:DAYS /] [// Returns Wed May 18 00:00:00 2011

[/* ... */]

Syntaxe

[/* ... */]

Description

Utilisée pour indiquer un commentaire de bloc pouvant s'étendre à des lignes multiples. Utiliser cette balise plutôt que des balises de commentaire HTML ou JavaScript permet d'éviter que le commentaire ne soit inscrit dans la sortie envoyée au navigateur Web de l'utilisateur. Pour des rapports importants, cela présente des avantages considérables pour la restriction de la quantité de données envoyées vers chaque navigateur. La conversion de tous les commentaires afin d'utiliser les balises de commentaire WebReports peut apporter des avantages en matière de performances pour les rapports importants.
