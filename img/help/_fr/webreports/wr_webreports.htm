The WebReports online help section provides useful information for different types of users. This section describes some useful documentation sections for each type of user.

Basic procedures for WebReports end users

The Procedures section of the WebReports online help provides basic procedures for using the WebReports object. For end users of WebReports there are important items related to the execution of WebReports to display or export data. The most useful items for WebReport users are:

  • Exporting WebReport Output
  • Running WebReports
  • Scheduling WebReports

Basic procedures for WebReports Developers

The Procedures section of the WebReports online help provides basic procedures for using the WebReports object. For WebReports developers some of the most useful procedures are:

  • Adding a New WebReport
  • Creating/Editing Reportviews
  • Data Source Configuration
  • Editing a Reportview Online
  • Scheduling WebReports
  • Using Constants
  • Setting Parameter Defaults

Tag Guide for WebReports Developers

The most important resource for a WebReports developer is the online tag guide which provides complete syntax information with drag and drop capability. Select this link to open the tag guide in a new window.

Procedures for WebReports Administrators

The Procedures section of the WebReports online help provides basic procedures for using the WebReports object. In some cases the administrator of WebReports is not necessarily the same person developing WebReports or on production systems a specific individual may be responsible for certain WebReport procedures. Some procedures that are useful for administrators are listed below. (The WebReports administrator guide is also important for administrators and is normally available as part of the WebReports module documentation on the OpenText Knowledge Centre.)

  • Adding a New WebReport
  • Data Source Configuration
  • Scheduling WebReports
  • Permissions Setting

Advanced Information for WebReports Developers

Besides the basic procedural help provided for WebReports there are also several sections provided with more advanced information that can be used by WebReport developers. A list of the current sections can be found under Advanced Information
