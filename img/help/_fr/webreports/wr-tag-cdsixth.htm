Cette section apporte des informations sur les balises de données de colonne : INDEXTOTALHITS et ISSUBWEBREPORT.

INDEXTOTALHITS

Syntaxe

[LL_REPTAG_INDEXTOTALHITS /]

Description

Insère le nombre total de résultats à partir d'une source de données de recherche. Cette balise est utile lors de l'utilisation de Utilisation d'une recherche de requête comme source de données, car elle peut fournir à l'utilisateur une indication sur la taille totale des résultats sans les afficher tous. Cette balise peut être utilisée en même temps que le paramètre d'URL &maxrows afin de limiter le nombre de résultats et d'obtenir très rapidement une valeur pour le nombre de résultats de recherche tout en minimisant l'impact sur les performances.

ISSUBWEBREPORT

Syntaxe

[LL_REPTAG_ISSUBWEBREPORT /]

Description

Renvoie true si le WebReport est exécuté en tant que sous-rapport WebReport, renvoie false dans le cas contraire.
