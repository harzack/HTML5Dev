Cette section apporte des informations sur les balises de données de colonne : TOTALROWS, TOTALSOURCEROWS, TRIGGER, TRIGGERDESTPARENTID, TRIGGERNEWID, TRIGGERID, TRIGGERNEWNAME, TRIGGEROLDNAME et TRIGGERSOURCEPARENTID.

TOTALROWS

Syntaxe

[LL_REPTAG_TOTALROWS /]

Description

Le nombre total de lignes renvoyées par la source de données, après DSstartrow et DSendrow. Cette balise renvoie le nombre de lignes initialement disponibles, quelle que soit la balise INCLUDEIF. Si l'utilisateur a besoin d'un compteur pour le nombre de lignes ayant été intégrées après INCLUDEIF, le paramètre ACTUALROWS doit être utilisé. Cette balise ne prend pas en compte l'effet de paramètres spéciaux source des données DSstartrow et DSendrow, voir TOTALSOURCEROWS.

TOTALSOURCEROWS

Syntaxe

[LL_REPTAG_TOTALSOURCEROWS /]

Description

Le nombre total de lignes renvoyées par la source de données, avant DSstartrow et DSendrow. Cette balise est en relation étroite avec TOTALROWS et ACTUALROWS. La différence est qu'elle prend en compte les paramètres spéciaux de la source des données DSstartrow et DSendrow, ce qui est très utile pour la pagination.

Informations associées

Pour plus d'informations, consultez Paramètres de source de données.

TRIGGER

Syntaxe

[LL_REPTAG_TRIGGER /]

Description

L'événement de déclenchement qui a lancé le WebReport. Cette balise ne renverra une valeur que lorsqu'un WebReport aura été déclenché en utilisant la fonction WRTrigger. La fonction WRTrigger doit être activée dans les pages Administration sur une base par sous-type. Par défaut, elle est désactivée pour tous les sous-types. Si la fonction n'est pas activée pour un sous-type donné, aucun WebReport ne sera déclenché même si le nœud se trouve au sein de la hiérarchie pour laquelle un déclenchement en cascade est appliqué.

Paramètres

Les valeurs de déclenchement possibles sont :

Valeur de déclenchement
Description
Ajouter une version
Déclenchée lorsqu'une nouvelle version est ajoutée à un document existant ou lorsqu'un nouveau document est ajouté.
Mettre à jour la catégorie
Déclenchée lorsqu'une catégorie Content Server est appliquée, supprimée ou mise à jour sur un nœud donné.
Destination de la copie
Déclenchée lorsqu'un élément est copié. Peut aussi être associée à un recueil où elle se déclenchera pour tout élément copié vers le recueil. En d'autres termes, le recueil est la destination pour l'action de copie.
Source de la copie
Déclenchée lorsqu'un élément est copié. Peut aussi être associée à un recueil où elle se déclenchera pour tout élément copié à partir du recueil. En d'autres termes, le recueil est la source pour l'action de copie.
Créer
Déclenchée lorsqu'un nœud est créé.
Supprimer
Déclenchée lorsqu'un nœud est supprimé.
Déplacer la destination
Déclenchée lorsqu'un élément est déplacé. Peut aussi être associée à un recueil où elle se déclenchera pour tout élément déplacé vers le recueil. En d'autres termes, le recueil est la destination pour l'action de déplacement.
Déplacer la source
Déclenchée lorsqu'un élément est déplacé. Peut aussi être associée à un recueil où elle se déclenchera pour tout élément déplacé à partir du recueil. En d'autres termes, le recueil est la source pour l'action de déplacement.
Renommer
Déclenchée lorsqu'un nœud est renommé.
Mettre à jour
Déclenchée lorsqu'un nœud est mis à jour, dont un exemple change la description.
Workflow lancé
Déclenchée lorsqu'un workflow est lancé.
Workflow achevé
Déclenchée lorsqu'un workflow est achevé.
Workflow interrompu
Déclenchée lorsqu'un workflow est interrompu.
Workflow repris
Déclenchée lorsqu'un workflow est repris.
Workflow arrêté
Déclenchée lorsqu'un workflow est arrêté.
Workflow supprimé
Déclenchée lorsqu'un workflow est supprimé.
Workflow archivé
Déclenchée lorsqu'un workflow est archivé.

Exemple

[LL_REPTAG_&PARM1 MODULUS:7 /] renvoie 6 si PARM1 contient « 2001 ».

TRIGGERDESTPARENTID

Syntaxe

[LL_REPTAG_TRIGGERDESTPARENTID /]

Description

Le nodeid du recueil de destination dans une opération de copie ou de déplacement.

TRIGGERID

Syntaxe

[LL_REPTAG_TRIGGERID /]

Description

Le nodeid de l'élément qui a lancé l'évènement de déclenchement. Si l'evénement de déclenchement a été lancé depuis un workflow, le subworkid de l'instance du workflow sera renvoyé à sa place.

TRIGGERNEWID

Syntaxe

[LL_REPTAG_TRIGGERNEWID /]

Description

Le nouveau nodeid de l'élément copié lorsqu'un évènement de copie se produit.

TRIGGERNEWNAME

Syntaxe

[LL_REPTAG_TRIGGERNEWNAME /]

Description

Le nouveau nom de l'élément renommé lorsqu'un évènement de nouvelle attribution de nom se produit.

TRIGGEROLDNAME

Syntaxe

[LL_REPTAG_TRIGGERNEWNAME /]

Description

L'ancien nom de l'élément renommé lorsqu'un évènement de nouvelle attribution de nom se produit.

TRIGGERSOURCEPARENTID

Syntaxe

[LL_REPTAG_TRIGGERSOURCEPARENTID /]

Description

Le nodeid du recueil source dans une opération de copie ou de déplacement.
