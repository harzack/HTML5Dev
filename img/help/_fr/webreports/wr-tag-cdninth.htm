Cette section apporte des informations sur la balise de données de colonne NEXTURL.

NEXTURL

Syntaxe

[LL_REPTAG_NEXTURL /]

Description

Renvoie une URL qui redirigera l'utilisateur vers la page précédente.

Exemple

Elle peut être utilisée pour créer un bouton Retour utilisant HTML comme suit : <INPUT TYPE="BUTTON" NAME="return" VALUE="Return" ONCLICK="document.location='[LL_REPTAG_NEXTURL /]' ">
