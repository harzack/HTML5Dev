Vous pouvez utiliser les fonctions suivantes dans vos applications de production de rapports ActiveView/WebReport. Elles fournissent des utilitaires pratiques de parcours de pages pour améliorer et simplifier la conception par les développeurs. Au moyen des fonctions ci-dessous, vous pouvez ajouter des paramètres d'URL, créer des colonnes pouvant être triées ou créer des boutons utiles comme COPIER, DEPLACER ou SUPPRIMER dans vos rapports.

Fonctions de navigation

  • setFilter( name, value )

  • setupContext( support_Dir, url_Prefix, my_URL, next_URL, def_SortCol, sort_Dir, form_name, style_Type )

  • setupContextPOST( support_Dir, def_SortCol, sort_Dir, form_name, style_Type )

  • setColumn( title, parm2, parm3, parm4, parm5, parm6 )

  • setColumnPOST( title, parm2, parm3 )

  • multiButton( gifName, label, reqHandler, reqName )
