Cette section apporte des informations sur la balise de données de colonne COLNAME.

COLNAME

Syntaxe

[LL_REPTAG_COLNAME<n> /]

ou

[LL_REPTAG_COLNAME++ /]

Description

Cette balise renvoie le nom d'une colonne à partir de la source de données associée, en fonction du numéro fourni après COLNAME. Par exemple, [LL_REPTAG_COLNAME1 /] renvoie le mot : « DATAID » dans un WebReport archivant le DTREE Content Server à l'aide d'un rapport LiveReport automatique.

Cette balise est utile lorsqu'un affichage de rapport générique est développé. Plutôt que de coder en dur un nom de colonne dans l'affichage de rapport, cette balise fournit toujours le nom de colonne correct. L'affichage de rapport par défaut utilise cette balise pour tous les noms de colonne.

La variation COLNAME++ de cette balise fournit automatiquement le numéro de colonne suivant.

Exemple

Par exemple :

  • Si la balise colname précédente est COLNAME3, alors COLNAME++ renvoie la valeur de COLNAME4.

  • Si COLNAME++ est utilisée à nouveau, alors la valeur de COLNAME5 est renvoyée.

  • Si COLNAME1 était alors utilisée, COLNAME++ continue à partir du dernier numéro de colonne et renvoie donc la valeur de COLNAME2.
