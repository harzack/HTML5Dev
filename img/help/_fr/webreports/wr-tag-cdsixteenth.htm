Cette section apporte des informations sur les balises de données de colonne : WEBDAVAUTH, WEBDAVDIR et WEBREPORTNAME.

WEBDAVAUTH

Syntaxe

[LL_REPTAG_WEBDAVAUTH /]

Description

Renvoie un code d'autorisation personnel pour l'utilisation de WebDav. Pour plus d'informations, voir les entrées WEBDAV pour la balise LLURL.

WEBDAVDIR

Syntaxe

[LL_REPTAG_WEBDAVDIR /]

Description

Le répertoire virtuel utilisé par le serveur Web, par exemple IIS, pour gérer les requêtes WebDAV dans Content Server. Pour plus d'informations, voir les entrées WEBDAV pour la balise LLURL.

WEBREPORTNAME

Syntaxe

[LL_REPTAG_WEBREPORTNAME /]

Description

Insère le nom du WebReport en cours d'exécution.
