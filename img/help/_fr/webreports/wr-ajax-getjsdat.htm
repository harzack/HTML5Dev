• Cette fonction appelle une URL spécifique ou un WebReport. Elle attend le renvoi du code JavaScript à exécuter dans la page Web appelante.

  • Traite les arguments suivants :

    • URLreference (number/String) : paramètre obligatoire. URL, objectId numérique (dataId) ou surnom (String) du WebReport. Si le surnom est numérique, il doit être mis entre guillemets.

    • optionalParms (String) : paramètre facultatif. Tout paramètre à transmettre au WebReport cible en cours d'exécution. optionalParms présente le format suivant : &parm1=value1,&parm2=value2...

    • JSfunction : paramètre facultatif. Fonction facultative qui est appelée après le renvoi d'un code JavaScript par le WebReport cible et son exécution. Tout code JavaScript à exécuter uniquement après le renvoi d'une réponse doit être appelé à partir d'ici.

  • D'autres informations peuvent être transmises à JSfunction sous forme de paramètres.

  • En cas d'erreur, un message d'alerte descriptif apparaît dans le navigateur.

  • Par exemple, consultez Exemple d'utilisation de la fonction getJSData.
