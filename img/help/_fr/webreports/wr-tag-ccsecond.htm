Cette section apporte des informations sur la balise de contrôle de contenu CALL.

CALL

Syntaxe

[LL_WEBREPORT_CALL NAME:<nom de script> PARM:<arg1>:<arg2>:<argN> /]

Description

Appelle une fonction Oscript côté serveur définie par les balises STARTSCRIPT et ENDSCRIPT. Le script particulier à appeler est identifié par le paramètre <nom de script>. Le script peut également être transmis à une liste d'arguments séparés par des points-virgules. Notez que le mode script côté serveur doit être activé par un administrateur, au niveau du serveur et pour chaque WebReport, toutes les fois où un nouvel affichage de rapport est ajouté. Les scripts sont rendus inactifs, excepté lorsqu'ils sont activés. Le mode script côté serveur est un outil puissant, qu'il faut toutefois utiliser avec beaucoup de précaution. Avant de s'en servir, il est recommandé aux utilisateurs de lire attentivement l'ensemble de la documentation disponible à ce sujet.
