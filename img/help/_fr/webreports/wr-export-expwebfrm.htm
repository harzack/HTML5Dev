Exemple 1-21 : Exemple de l'appel d'une URL lors d'une exportation vers un WebForm :

?func=ll&objId=1234&objAction=RunReport&export_location=form&form_ID=5678&appendForm=overwrite&nexturl=

Dans cet exemple, le WebReport 1234 supprime le contenu du WebForm 5678 et le renseigne avec des données spécifiées dans le champ SETFORM du WebReport.
