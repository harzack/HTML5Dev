La fonction Exportation WebReports permet aux utilisateurs d'exporter les données du rapport vers un e-mail. L'écran Exporter vers e-mail permet à l'utilisateur d'indiquer des adresses e-mail de trois manières différentes dans le champ Adresse e-mail :

  • Utilisez le premier champ texte pour saisir manuellement toutes les adresses e-mail séparées par une virgule.

     

    Note :
    Chaque adresse est séparée au moyen du symbole défini comme préférence système. L'instruction pour ce champ indique ce symbole, Deux exemples de ces symboles sont la virgule et le point-virgule.

     

  • Utilisez le second champ pour sélectionner un utilisateur ou un groupe Content Server.

  • Utilisez le troisième champ pour sélectionner un fichier CSV d'adresses à partir de Content Server.

 

Notes :

  • Vous pouvez combiner ces trois méthodes en fonction de vos besoins.

  • En sélectionnant un utilisateur ou un groupe Content Server, aucun e-mail ne sera envoyé si aucune adresse e-mail n'est indiquée dans le profil Content Server de l'utilisateur.

  • Les e-mails ne sont envoyés qu'aux membres directs des groupes Content Server sélectionnés.

  • La liste complète d'adresses ne doit pas dépasser 10 000 caractères, soit environ 500 adresses e-mail.
