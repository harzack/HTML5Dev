Sélectionnez Bureau comme destination d'exportation, puis Exporter le type MIME pour le fichier d'exportation obtenu. Le type MIME peut être utilisé par les WebReports pour déterminer l'extension du fichier, et par conséquent le type du fichier, à télécharger. Il est enfin utilisé pour déterminer l'application qui va ouvrir le fichier exporté. Lorsque vous cliquez sur Exporter, une boîte de dialogue de téléchargement apparaît. A ce stade, vous pouvez modifier le nom de fichier par défaut qui sera défini sur le nom du WebReport correspondant.

Sur l'écran Exporter vers le bureau :

  1. Dans la liste Destination de sortie, sélectionnez Bureau.

  2. Dans le champ Télécharger le nom de fichier, saisissez le nom du fichier d'exportation.

  3. Dans la liste Exporter le type MIME, sélectionnez le format d'exportation.
