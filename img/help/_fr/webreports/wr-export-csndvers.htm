L'exportation des données du rapport vers une nouvelle version de document est identique à Exportation vers un nouveau document Content Server, à deux exceptions près :

  • Définissez l'option Type d'exportation sur Version Content Server.

  • Dans le champ Ajouter à, sélectionnez le document auquel vous souhaitez ajouter ce rapport exporté. Vous devez sélectionner un document ou un affichage personnalisé existant plutôt qu'un recueil.

  • Dans le champ Ajouter des résultats, vous pouvez sélectionner l'endroit, dans le document auquel vous ajoutez une version, où vous souhaitez insérer le rapport exporté. Vous pouvez insérer le rapport exporté au début, à la fin, ou, en utilisant des balises, n'importe où dans le corps du document.

Lorsque vous cliquez sur Exporter, une nouvelle version est ajoutée au document sélectionné. La nouvelle version contiendra le nouveau type MIME, la description et les catégories spécifiés sur la page Exporter.
