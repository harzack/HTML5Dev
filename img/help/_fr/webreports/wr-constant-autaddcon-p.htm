Pour ajouter automatiquement une constante pour le WebReport :

  1. Sélectionnez l'onglet Constantes pour le WebReport. Dans le menu Fonctions du WebReport, sélectionnez Propriétés. Dans le menu Propriétés, sélectionnez Constantes.

  2. Dans l'onglet Constantes de la page Propriétés du WebReport, sélectionnez le bouton pour ajouter toutes les constantes utilisées dans l'affichage de rapport.

     

    Note :
    Cette icône apparaît uniquement lorsque l'affichage de rapport contient des constantes non définies dans l'onglet Constantes.

     

  3. Pour toutes les nouvelles constantes :

    1. Le champ Nom de la constante est prérenseigné.

    2. Dans le champ Valeur de la constante, saisissez une valeur pour cette nouvelle constante. Vous pouvez également sélectionner Parcourir pour rechercher un objet Content Server, puis cliquer sur le lien Sélectionner de cet objet. Lorsque vous avez cliqué sur le lien Sélectionner d'un objet Content Server, l'ID de cet objet est automatiquement saisi dans le champ Valeur de la constante.

    3. Dans la mesure du possible, WebReports détermine le Type de constante de chaque constante par programmation. Lorsque ce n'est pas possible, le type de constante sera Chaîne par défaut. Vous pouvez éventuellement modifier le paramètre du Type de constante.

    4. Facultatif Dans le champ Description de la constante, vous pouvez saisir une description pour cette constante si vous le souhaitez.

  4. Cliquez sur Mettre à jour.
