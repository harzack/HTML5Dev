Pour ajouter automatiquement un paramètre WebReport :

  1. Dans le menu Fonctions du WebReport, sélectionnez Propriétés. Sélectionnez ensuite Paramètres.

  2. Dans l'onglet Paramètres :

    1. Sélectionnez le bouton pour ajouter tous les paramètres utilisés dans l'affichage de rapport ou la source de données de rapport LiveReport. Notez que cette icône apparaît uniquement lorsque l'affichage de rapport ou le rapport LiveReport contient des paramètres qui ne sont pas définis dans l'onglet Paramètres.

    2. Pour tous les nouveaux paramètres, ajoutez un nom de paramètre, un texte d'affichage, un ordre d'invite, une valeur par défaut et d'autres paramètres. Pour plus d'informations sur les options des paramètres, voir Réglages de l'onglet Paramètres.

  3. Sélectionnez Mettre à jour.
