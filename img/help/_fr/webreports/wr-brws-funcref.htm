Le fichier browse.js fourni avec WebReports inclut une liste de fonctions étudiées pour faciliter la conception du parcours de pages. Le fichier browse.js est accédé par l'inclusion d'un SCRIPT comme suit : <SCRIPT SRC="/<support dir>/webreports/library/browse.js"></SCRIPT>

Si ce fichier est inclus dans un WebReport ou ActiveView, la balise LIBPATH peut être utilisée comme suit : <SCRIPT SRC="[LL_REPTAG_LIBPATH /]browse.js"></SCRIPT>
