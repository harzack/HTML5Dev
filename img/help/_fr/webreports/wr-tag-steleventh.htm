Cette section apporte des informations sur les sous-balises suivantes : MODULUS et MULTIPLY.

MODULUS

Syntaxe

MODULUS:<nombre module >

Description

Renvoie la valeur d'une opération MODULUS sur la donnée renvoyée par la balise principale.

Exemple

[LL_REPTAG_&PARM1 MODULUS:7 /] renvoie 6 si PARM1 contient « 2001 ».

MULTIPLY

Syntaxe

MULTIPLY:<valeur>

Description

Multiplie un entier renvoyé dans la balise principale par la valeur définie. D'autres sous-balises peuvent être utilisées en association avec cette balise.

Exemple

ROUND apporte plus de précision, [LL_REPTAG_"10" MULTIPLY:2 ROUND:2 /] devient 20.00.
