Pour accéder au guide sur les balises dynamiques :

  1. Dans le menu Fonctions d'un WebReport, effectuez l'une des opérations suivantes :

    • Sélectionnez Ouvrir le guide sur les balises directement dans le menu Fonctions.

    • Sélectionnez Modifier l'affichage du rapport. Dans la page Modifier WebReport, dans la barre d'outils du WebReport, sélectionnez Guide sur les balises.

  2. Le guide sur les balises dynamiques s'ouvre dans une nouvelle page. Dans la page Guide sur les balises WebReports, double-cliquez sur une ligne pour développer ou réduire la description correspondante.

  3. Lorsque vous avez fini d'utiliser le guide sur les balises dynamiques, fermez la page.
