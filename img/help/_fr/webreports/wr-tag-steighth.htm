Cette section apporte des informations sur les sous-balises suivantes : HIDE et HTMLCR.

HIDE

Syntaxe

HIDE

Description

Cette sous-balise est utilisée pour masquer la sortie d'une balise.

HTMLCR

Syntaxe

HTMLCR

Description

Insère le code HTML dès qu'un retour chariot et un caractère de saut de ligne sont trouvés ensemble, dès qu'un seul caractère de retour chariot est trouvé ou dès qu'un seul caractère de saut de ligne est trouvé. Cette sous-balise est utile pour formater correctement de longs champs de texte en HTML qui autrement ne prendraient pas en compte les retours chariot.
