Vous pouvez exporter la sortie du rapport vers un nouveau document Content Server. Cela peut se faire de deux manières :

  1. Sélectionnez Exporter vers Content Server à partir de l'option Exportation de l'objet WebReport.

  2. Définissez la Destination de sortie par défaut sur Content Server, puis exécutez le WebReport.

Sur l'écran Exporter vers un nouveau document Content Server :

  1. Dans la liste Destination de sortie, sélectionnez Content Server.

  2. Définissez l'option Type d'exportation sur Nœud Content Server.

  3. Dans le champ Nom, assurez-vous de saisir un nom de fichier unique. Si le nom fourni n'est pas un nom unique, un suffixe unique sera ajouté. Par exemple, « Nouveau Cliché WebReport 5 ».

     

    Note :
    Le nom par défaut sera le nom du WebReport suivi de « Cliché ».

     

  4. Si vous le souhaitez, vous pouvez saisir une description du document obtenu dans le champ Description.

  5. Dans le champ Créer dans, cliquez sur Parcourir Content Server, recherchez le recueil dans lequel vous souhaitez stocker le document exporté, puis cliquez sur le lien Sélectionner du recueil.

     

    Note :
    L'emplacement par défaut sera le dossier qui contient le WebReport.

     

  6. Si vous le souhaitez, vous pouvez sélectionner une catégorie à appliquer à ce document exporté dans le champ Catégories.

  7. Dans le champ Exporter le type MIME, vous pouvez spécifier le type MIME du document obtenu.

  8. Si vous le souhaitez, vous pouvez sélectionner Utiliser un moteur de conversion externe.

  9. Pour retourner immédiatement sur votre navigateur, sélectionnez Exécuter en arrière-plan.
