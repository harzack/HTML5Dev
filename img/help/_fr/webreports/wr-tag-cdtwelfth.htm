Cette section apporte des informations sur les balises de données de colonne : RKTENGINEENABLED et ROWNUM.

RKTENGINEENABLED

Syntaxe

[LL_REPTAG_RKTENGINEENABLED /]

Description

Renvoie Vrai si le moteur RKT est installé et activé, renvoie Faux dans le cas contraire.

ROWNUM

Syntaxe

[LL_REPTAG_ROWNUM /]

Description

Renvoie le numéro de ligne actuel. Cette balise est particulièrement utile lorsqu'elle est utilisée avec les sous-balises ODD ou EVEN. Par exemple, [LL_REPTAG_ROWNUM ODD /] renvoie true pour les lignes 1, 3, 5, 7, etc. Cela peut être utilisé afin d'alterner les couleurs de ligne. Cette balise ne peut pas être utilisée en tant que terme dans la balise de contrôle de contenu INCLUDEIF, car elle est modifiée en fonction du nombre de lignes renvoyé par la balise INCLUDEIF.
