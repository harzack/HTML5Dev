Ce tableau illustre les différents types de paramètres qui peuvent être sélectionnés dans l'onglet Paramètres et leur effet sur le rapport final.

Type
Description
String
Le type de paramètre String invite l'utilisateur à saisir un champ de texte libre.
ObjectID
Le type de paramètre ObjectID permet au développeur du rapport de spécifier les types d'objets Content Server que l'utilisateur peut sélectionner dans l'écran d'invite. Un ou plusieurs éléments peuvent être sélectionnés dans la liste à valeurs multiples et au moins un type doit être sélectionné pour que le développeur puisse rechercher une valeur par défaut.
User
Le type de paramètre User permet au développeur du rapport de créer une invite utilisateur. Le développeur peut décider si l'utilisateur est autorisé ou non à rechercher des utilisateurs, des groupes ou les deux.
Number
Le type de paramètre Number est similaire au paramètre String. Toutefois, le type Number oblige l'utilisateur à saisir une valeur numérique.
Object
Le type de paramètre Object permet à l'utilisateur de sélectionner un sous-type Content Server. Par exemple, sélectionnez Document (144) ou Dossier (0).
Date
Le type de paramètre Date invite l'utilisateur à saisir une date. Le développeur peut décider si l'heure est affichée ou non. Le développeur peut également décider si l'utilisateur est invité à saisir une date vide, la date du jour ou une date spécifique.
Custom
Le type de paramètre Custom permet au développeur de rechercher un objet WebReport servant à fournir une invite personnalisée. Le développeur peut l'utiliser pour créer une liste de sélection personnalisée par exemple.
