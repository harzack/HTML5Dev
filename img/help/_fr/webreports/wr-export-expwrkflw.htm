Exemple 1-22 : Exemple de l'appel d'une URL lors d'une exportation vers un workflow :

?func=ll&objId=1234&objAction=RunReport&export_location=workflow&workflow_ID=5678&attach=on&nodeName=report%20data&mimetype=text/html&workflow_title=my%20new%20workflow&nexturl=

Dans cet exemple, le WebReport 1234 lance une instance de schéma de workflow 5678 et définit le titre sur « mon nouveau workflow ». Le rapport lui-même est joint au workflow avec un nom de « données de rapport » et un type Mime text/html.
