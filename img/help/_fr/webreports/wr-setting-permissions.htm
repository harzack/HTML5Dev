Les autorisations WebReport présentent certaines caractéristiques uniques. WebReports : ils sont en effet un regroupement de trois types d'objets différents dans Content Server :

  • Documents : utilisés pour stocker et gérer les affichages de rapport.

  • Documents texte : utilisés pour prendre en charge la modification d'affichage de rapport en ligne.

  • Rapports LiveReport : utilisés pour générer la sortie du rapport.

Le tableau suivant indique les rôles les plus courants pour les utilisateurs WebReports et ce que doivent être les autorisations recommandées.
