La pagination peut être réalisée de différentes manières avec les WebReports. La méthode particulière utilisée par votre entreprise va dépendre de plusieurs facteurs :

  • Les performances de la base de données par rapport aux performances Content Server par rapport aux performances client

  • Base de données utilisée (SQL Server ou Oracle)

  • Taille de l'ensemble de résultats (maximum, minimum et probable)

  • Compétences du développeur, généralement SQL, WebReports et HTML/Javascript, et délai de développement disponible

  • Le cas échéant, emplacement dans lequel l'ensemble de résultats est filtré

Il n'est pas possible d'établir les instructions définitives relatives à la meilleure approche sans une description détaillée du système de destination. Ce document va décrire certaines approches générales, mais la meilleure méthode sera différente d'un système à un autre.
