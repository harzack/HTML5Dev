WebReports prend en charge l'utilisation de paramètres définis par l'utilisateur dans la section Texte intégral de la requête de recherche. Ils sont conçus pour fonctionner de la même manière que les paramètres utilisés dans les rapports LiveReports. Les paramètres sont spécifiés au format : n % où n est nombre entre 1 et 99. Ces paramètres peuvent être ajoutés aux champs Texte intégral multiples. Lorsque le WebReport est exécuté, les paramètres sont transmis en ajoutant &inputlabeln à l'URL.

Exemple 1-29 : Si le champ Texte intégral contenait « test %1 » :

Le paramètre &inputlabel1=cat dans l'URL entraînerait la recherche des mots test et cat par la requête de recherche.
