Des données peuvent être transmises à un script de plusieurs façons : en utilisant l'Assoc Context, en transmettant des paramètres explicites et en appelant des balises de données statiques WebReports dans la section de script. Cette dernière méthode est étudiée séparément dans la section Appel de balises de données statiques.
