Cette section apporte des informations sur les sous-balises suivantes : IMEMBERSHIP, INC et ISMODULEINSTALLED.

IMEMBERSHIP

Syntaxe

IMEMBERSHIP

Description

Renvoie une liste séparée par des virgules des groupes Content Server dont l'utilisateur est un membre direct ou indirect.

INC

Syntaxe

INC

ou

INC:<nombre d'incrémentation>

Description

Incrémente une valeur numérique de 1 ou d'un nombre d'incrémentation spécifié. Par défaut, cette sous-balise renvoie la valeur de balise actuelle plus un et des erreurs si la balise ne contient pas un entier valide. Si le nombre d'incrémentation facultatif a été spécifié, alors la balise actuelle à laquelle le nombre d'incrémentation est ajouté est renvoyée.

ISMODULEINSTALLED

Syntaxe

ISMODULEINSTALLED

Description

Cette sous-balise renvoie Vrai si le nom du module dans la balise principale est installé, sinon elle renvoie Faux.

Exemple

Par exemple, [LL_REPTAG_'WebReports' ISMODULEINSTALLED /] renvoie Vrai, alors que [LL_REPTAG_'NonExistentModule' ISMODULEINSTALLED /] renvoie Faux.
