Les colonnes renvoyées par une recherche enregistrée sont décrites dans le tableau ci-dessous :

Nom de la colonne
Description
DataID
Le DataID de l'objet tel que stocké dans la table de base de données DTREE Content Server.
ParentID
L'ID de l'objet du recueil parent tel que stocké dans la table de base de données DTREE Content Server.
Nom
Le nom de l'objet tel que stocké dans la table de base de données DTREE Content Server.
CreatedBy
La date de création de l'objet telle que stockée dans la table de base de données DTREE Content Server.
ModifyDate
La date de dernière modification de l'objet telle que stockée dans la table de base de données DTREE Content Server.
Enregistrement
Un rapport complet de tous les champs de l'objet stocké dans la table de base de données DTREE Content Server. Ces champs sont stockés dans une structure RECORD spécifique à laquelle il est possible d'accéder à l'aide de la sous-balise RECORD. Par exemple : [LL_REPTAG=RECORD RECORD:SUBTYPE /]
[Optional Fields]
Outre les champs fixes ci-dessus, tous les autres attributs système ou régions activés pour l'affichage de recherche peuvent être référencés à l'aide d'une balise de données WebReports. Comme ces champs dépendent du système, il est nécessaire de demander à l'administrateur de fournir une liste de ces attributs ou régions.
