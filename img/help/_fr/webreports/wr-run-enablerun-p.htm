1. Sélectionnez le menu Fonctions de WebReports. Sélectionnez Propriétés, puis Spécifiques.

  2. Dans l'onglet Spécifique du WebReport, cliquez sur l'icône d'utilisateur, , située en regard du champ Exécuter en tant que.

  3. Dans la fenêtre de sélection Utilisateur, choisissez l'utilisateur employé pour exécuter ce rapport.

  4. Cliquez sur Mettre à jour pour enregistrer les modifications.

     

    Note :
    La prochaine exécution de ce rapport s'effectue comme si l'utilisateur indiqué dans le champ Exécuter en tant que l'avait demandée. Toutes les autorisations et toutes les balises du WebReport sont traitées comme si celui-ci était exécuté par l'utilisateur choisi pour la fonction Exécuter en tant que. Cela inclut des balises telles que [LL_REPTAG_USERID /]. Quel que soit l'utilisateur exécutant ce WebReport, cette balise fait toujours référence à l'USERID de l'utilisateur indiqué pour la fonction Exécuter en tant que.
