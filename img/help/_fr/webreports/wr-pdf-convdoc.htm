Afin de convertir un document, l'utilisateur doit seulement sélectionner Utiliser un moteur de conversion externe dans l'onglet Destination d'un WebReport et il sera converti au format PDF.

 

Note :
La conversion PDF n'est pas instantanée. L'utilisateur devra éventuellement attendre quelques minutes avant qu'un rapport n'apparaisse à son emplacement de destination.

 

 

Note :
Avant que votre conversion PDF ne soit utilisable, votre administrateur Content Server doit configurer les options dans les pages Administration Content Server :

 

Si vous êtes un administrateur, consultez la section Gérer la conversion WebReports dans l'aide en ligne administrateur.
