• Crée et insère un bouton dans une page ActiveView/WebReport. Affiche le style de bouton approprié, selon l'instance Content Server.

  • Traite les arguments suivants :

    • gifName : paramètre obligatoire. Chaîne indiquant le nom de l'image dans le répertoire de support.

    • label : paramètre obligatoire. Chaîne indiquant le libellé à afficher sur le bouton dans la page.

    • reqHandler : paramètre obligatoire. Chaîne indiquant l'action que le gestionnaire de requête doit effectuer lorsque l'utilisateur clique sur le bouton.

    • reqName : paramètre obligatoire. Chaîne indiquant le nom de la requête.

  • Cette fonction exige que la fonction 'setupContext' ait été exécutée au préalable, car elle fait référence à la variable 'supportDir'.

  • Exemple :

    <SCRIPT>
	// Create the buttons Copy, Move, Delete, Zip & Download, Zip & Email, and Print on the page
	multiButton('copy', 'Copy', 'll.ProcessMultiCopy','copy');
	multiButton('move', 'Move', 'll.ProcessMultiMove','move');
	multiButton('del', 'Delete', 'll.ProcessMultiDelete','delete');
	multiButton('multifile/zipdwnld', 'Zip & Download', 'multifile.zipdwnldmulti', 'zipdwnld');
	multiButton('multifile/zipemail', 'Zip & Email', 'multifile.zipemailmulti', 'zipemail');
	multiButton('multifile/print', 'Print', 'multifile.printmulti', 'print');
</SCRIPT>
