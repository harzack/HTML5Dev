Pour supprimer un paramètre WebReport :

  1. Dans le menu Fonctions du WebReport, sélectionnez Propriétés. Sélectionnez ensuite Paramètres.

  2. Dans l'onglet Paramètres :

    1. Sélectionnez le bouton moins, -, à côté du paramètre à supprimer.

    2. Confirmez la suppression du paramètre en sélectionnant OK.

  3. Cliquez sur Mettre à jour.

     

    Astuce :
    Sélectionnez le bouton X rouge pour supprimer toutes les définitions de paramètre dans tout WebReport, puis cliquez sur Mettre à jour.
