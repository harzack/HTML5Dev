Le fichier ajax.js contient une liste de fonctions utiles qui font appel à la technologie AJAX (Asynchronous JavaScript and XML) qui peut être utilisée dans WebReports. L'un des avantages d'AJAX est sa capacité à fournir au navigateur des petits éléments d'informations, lorsque l'utilisateur les demande, et ce sans redéfinir toute la page. Outre cette capacité, les fonctions décrites ci-dessous facilitent également la conception d'applications Web performantes à l'aide de WebReports et de la technologie AJAX.

Les sections suivantes décrivent en détail les fonctions, les paramètres obligatoires et facultatifs et certains exemples d'utilisation des fonctions dans le fichier ajax.js.

Fonctions de la bibliothèque AJAX

  • getURLParm( parm )

  • updatePage( URLreference, HTMLelementID, optionalParms, JSfunction )

  • getJSData( URLreference, optionalParms, JSfunction )

  • sendRequest( URLreference, responseFunction, reqType, optionalParms )

  • executeWRService( serviceType, responseTarget, parmList, responseType, getPost, secure )
