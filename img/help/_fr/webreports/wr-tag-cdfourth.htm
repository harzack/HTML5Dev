Cette section apporte des informations sur les balises de données de colonne : DATE et DATETIME.

DATE

Syntaxe

[LL_REPTAG_DATE /]

Description

Insère la date à laquelle le WebReport a été exécuté. Pour les options de formatage, voir la sous-balise DATE.

Voir également DATETIME et TODATE.

DATETIME

Syntaxe

[LL_REPTAG_DATETIME /]

Description

Insère la date et l'heure auxquelles le WebReport a été exécuté. Pour les options de formatage, voir la sous-balise DATE.
