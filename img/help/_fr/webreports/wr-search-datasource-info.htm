Un WebReport peut utiliser une requête de recherche enregistrée comme source de données. Les requêtes enregistrées sont le plus souvent créées en enregistrant une recherche avancée, ce qui crée un objet dans Content Server. Le navigateur de la source de données WebReports reconnaît les requêtes de données enregistrées en tant qu'objets sélectionnables.

Les requêtes de recherche peuvent utiliser des spécifications Texte intégral standard, par exemple Tous les mots, ou complexes comme les requêtes en langage Content Server.

Lorsqu'une source de données est sélectionnée, la spécification de recherche correspondante sera exécutée en même temps que le WebReport.
