Le fichier ajax.js fourni avec WebReports comprend une liste de fonctions conçues pour faciliter l'utilisation des services WebReport. Le fichier ajax.js est accédé par l'inclusion d'un SCRIPT comme suit : 

<SCRIPT SRC="/<support_dir>/webreports/library/ajax.js"></SCRIPT>

Si le fichier est inclus dans un WebReport ou ActiveView, la balise LIBPATH peut être utilisée comme suit : 

<SCRIPT SRC="[LL_REPTAG_LIBPATH /]ajax.js"></SCRIPT>
 

Note :
Lorsque vous utilisez [LL_WEBREPORT_EXCLUDEHTML /] dans un WebReport qui utilise également les fonctions de la bibliothèque AJAX, la variable suivante doit être définie dans un bloc de script, car la variable baseURL est définie dans le code de page Content Server qui est inséré dans la sortie WebReport, mais elle est exclue à l'aide de la balise. Le code suivant garantit la définition de la variable et empêche la rupture de la bibliothèque AJAX. 

<script>
	baseURL = "[LL_REPTAG_URLPREFIX /]";
</script>
 

Les fonctions disponibles dans le fichier ajax.js et leurs paramètres sont décrits ci-dessous :

Table des matières  

  • getURLParm( parm )

  • updatePage( URLreference, HTMLelementID, optionalParms, JSfunction )

  • getJSData( URLreference, optionalParms, JSfunction )

  • sendRequest( URLreference, responseFunction, reqType, optionalParms )

  • executeWRService( serviceType, responseTarget, parmList, responseType, getPost, secure )
