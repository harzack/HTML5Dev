Exemple 1-18 : Exemple de l'appel d'une URL lors d'une exportation vers un nœud Content Server :

?func=ll&objId=1234&objAction=RunReport&export_location=livelink&livelink_node=livelinknode&newNode_ID=2000&nodeName=%5BLL_REPTAG_DATE%20%2F%5D&mimetype=text%2Fplain&nexturl=

Dans cet exemple, WebReport 1234 exporte vers un nouveau nœud Content Server avec un type Mime texte normal. Le fichier obtenu sera placé dans le dossier avec un ID de 2 000 et le nom de ce dernier sera la date actuelle.
