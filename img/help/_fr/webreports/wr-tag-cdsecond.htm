Cette section apporte des informations sur les balises de données de colonne : ACTUALROWS et ACTUALUSERID.

ACTUALROWS

Syntaxe

[LL_REPTAG_ACTUALROWS /]

Description

Cette balise renvoie le nombre de lignes contenues dans le WebReport via la balise INCLUDEIF. Ceci diffère du nombre total de lignes fournies par la source de données. Par exemple, si le WebReport est exécuté en association avec une source de données renvoyant dix lignes de données, mais que la balise INCLUDEIF restreint la sortie à trois lignes, alors TOTALROWS = 10 et ACTUALROWS = 3. Cette fonction est uniquement prise en charge à partir des sections en-tête et pied de page du WebReport.

ACTUALUSERID

Syntaxe

[LL_REPTAG_ACTUALUSERID /]

Description

Cette balise renvoie l'ID d'utilisateur de l'utilisateur qui a exécuté le WebReport, que la fonction RunAs soit définie ou non. Par exemple, si l'utilisateur 1 exécute un WebReport dont le paramètre RunAs est défini sur Utilisateur 2, la balise ACTUALUSERID renverra l'ID de l'utilisateur 1 tandis que la balise USERID renverra l'ID de l'utilisateur 2.
