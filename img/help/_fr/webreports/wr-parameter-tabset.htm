Ce tableau fournit des détails sur les différentes options disponibles dans l'onglet Paramètres et leur effet sur le rapport final.

Option
Description
Nom du paramètre
Il s'agit du nom du paramètre tel qu'il apparaît dans l'URL une fois que l'utilisateur a sélectionné des valeurs et choisi Exécuter sur la page d'invite. Le nom du paramètre ne doit contenir aucune espace ni autre caractère ne pouvant être transmis dans une URL. L'utilisateur final ignore généralement cette valeur.
Texte d'affichage
Il s'agit du texte d'invite de l'utilisateur. Il diffère du nom du paramètre puisqu'il est plus descriptif et peut contenir des espaces.
Invite
Sélectionnez cette case à cocher si vous voulez que l'utilisateur voie l'invite. Si l'utilisateur n'est pas invité à saisir un paramètre spécifique, la valeur par défaut est utilisée.
 

Important :
Si vous désactivez l'invite d'un paramètre, veillez à ne pas placer l'utilisateur en situation difficile. Cela pourrait être le cas si vous rendez un paramètre obligatoire, désactivez l'invite et ne sélectionnez aucune valeur par défaut.

 
Obligatoire
Sélectionnez cette case à cocher pour rendre un paramètre obligatoire. Si un paramètre est obligatoire, l'utilisateur ne peut pas passer outre l'écran d'invite sans sélectionner une valeur pour le paramètre, sauf si ce dernier comporte une valeur par défaut. Les paramètres obligatoires sont identifiés dans l'écran d'invite à l'aide de l'icône en regard du texte d'invite.
Ordre d'invite
Ce champ détermine l'ordre d'affichage des paramètres sur l'écran d'invite. Les valeurs peuvent être alphanumériques.
Type
Ce champ détermine le type de paramètre pour lequel l'utilisateur reçoit une invite. Les champs disponibles sont String, ObjectID, User, Number, Object, Date ou Custom. Pour en savoir plus, voir Types de paramètres.
Valeur par défaut
Il s'agit de la valeur utilisée si l'utilisateur ne sélectionne aucune valeur dans l'écran d'invite. Elle apparaît préremplie dans l'écran d'invite. Les WebReports exécutés sur un programme utilisent la valeur.
Description
La description est un champ de texte libre qui offre deux options au développeur WebReport. Il peut placer des commentaires de façon à mentionner un détail de mise en œuvre. La seconde option consiste à sélectionner la case à cocher Afficher les descriptions en haut de la page Paramètres. Toutes les descriptions s'affichent dans l'écran d'invite à l'endroit où elles peuvent servir à fournir des instructions à l'utilisateur final. Ce champ peut contenir des balises HTML.
Afficher les descriptions
Ce champ détermine si le texte de description de tous les paramètres apparaît sur la page d'invite.
Fichier d'invite
Lorsque cette option est sélectionnée, le contenu de ce fichier apparaît en haut de la page Invites. Les développeurs peuvent ainsi appliquer rapidement la même personnalisation ou les mêmes instructions en haut des écrans d'invite. Un document texte ou un WebReport distinct peut être sélectionné pour fournir ce texte. Si le développeur utilise un WebReport, il peut renvoyer le contenu dynamique vers l'écran d'invite. Cette option peut servir à effectuer des comptes récapitulatifs sur les données que l'utilisateur est sur le point de sélectionner dans l'écran d'invite.
