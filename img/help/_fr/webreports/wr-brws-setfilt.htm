• Fonction utilisée pour enregistrer tout paramètre et l'ajouter à l'URL actuelle. Cette fonction assure que les paramètres continuent d'exister, même si une colonne est triée à nouveau, et qu'il est possible d'y accéder plus tard au cours d'une session.

  • Traite les arguments suivants :

    • name : paramètre obligatoire. name est une chaîne représentant le nom du nouveau paramètre à ajouter à l'URL.

    • value : paramètre obligatoire. value est une chaîne ou un nombre représentant la valeur du paramètre.

  • Exemple : setFilter( 'count', 2 );

    Le paramètre &count=2 est ajouté à l'URL actuelle.
