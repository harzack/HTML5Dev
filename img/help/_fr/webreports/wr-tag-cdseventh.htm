Cette section apporte des informations sur les balises de données de colonne : LANGUAGE et LIBPATH.

LANGUAGE

Syntaxe

[LL_REPTAG_LANGUAGE /]

Description

Insère le langage système actuellement utilisé.

LIBPATH

Syntaxe

[LL_REPTAG_LIBPATH /]

Description

Fournit un chemin d'accès au dossier de la bibliothèque WebReports. Il est généralement utilisé pour stocker les fichiers Javascript utiles qui peuvent servir aux applications WebReports. Le chemin d'accès renvoyé peut être utilisé pour les URL qui doivent référencer ces fichiers.

Exemple

Par exemple, un chemin d'accès renvoyé type se présente comme suit : /Livelinksupport/webreports/library/

La syntaxe pour utiliser cette balise afin d'inclure un fichier Javascript dans la bibliothèque WebReports se présente comme suit : <SCRIPT SRC="[LL_REPTAG_LIBPATH /]page.js"></SCRIPT>
