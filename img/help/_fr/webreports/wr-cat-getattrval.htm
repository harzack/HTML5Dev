• Renvoie la valeur d'un attribut.

  • S'il s'agit d'un attribut à valeurs multiples, cette fonction renvoie un tableau. Par exemple, [1,2,3]

  • S'il s'agit d'un attribut à valeurs multiples dans un ensemble à valeurs multiples, elle renvoie un tableau de tableaux. Par exemple, [ [1,2,3], [2], [4,6,8] ]

  • Traite les arguments suivants :

    • categoryID : paramètre obligatoire.

    • attribute : paramètre obligatoire. Peut prendre les valeurs suivantes :

      • attributeId - '2','3','4',etc. (string) ou 2,3,4,etc. (integer)

      • Libellé d'affichage de l'attribut. Par exemple, 'Date of Test'. Doit être sensible à la casse pour que la fonction renvoie une ou plusieurs valeurs.

        Il est également possible de passer le nom de l'ensemble, par exemple 'Tester Details', qui renvoie toutes les valeurs des attributs de l'ensemble dans un tableau multidimensionnel.

        Chaque index du tableau représente la ligne des valeurs d'attributs. Par exemple, [ [row1 attr values], [row2 attr values], [row3 attr values], ...]

      • fieldId : est représenté sous la forme '_1_1_2_1', où '2' est l'attributeId. Il s'agit d'un identificateur unique qui renvoie toujours 1 valeur d'attribut.

    • nodeID : paramètre obligatoire.

  • La fonction détermine si un nom ou un id est passé en paramètre, puis effectue l'action appropriée.

  • Renvoie la valeur null si une erreur survient ou si la valeur de l'attribut est introuvable.
