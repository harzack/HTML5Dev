• Renvoie des informations sur l'attribut, selon le nom du champ spécifié.

  • Traite les arguments suivants :

    • categoryID : paramètre obligatoire.

    • attributeID : paramètre obligatoire. Au format '2','3','4',... (string) ou 2,3,4,... (integer).

    • fieldName : paramètre obligatoire. Doit prendre l'une des valeurs suivantes : 'req', 'type', 'len', 'maxlen', 'rows', 'maxrows', 'multi', 'set'. Il doit être sensible à la casse.

    • nodeID : paramètre facultatif. S'il n'est pas spécifié, il prend la valeur 'null', '' ou vide.

  • Renvoie la valeur null si une erreur survient ou si le nom du champ est introuvable.
