• C'est une fonction générique qui peut envoyer une requête AJAX pour appeler un WebReport. Cette fonction est également utilisée par les fonctions updatePage et getJSData.

  • Traite les arguments suivants :

    • URLreference (number/String) : paramètre obligatoire. URL, objectId numérique (dataId) ou surnom (String) du WebReport. Si le surnom est numérique, il doit être mis entre guillemets.

    • responseFunction (function) : paramètre obligatoire. Fonction à exécuter lorsque la requête AJAX est renvoyée.

    • reqType ('string','json','xml') : paramètre facultatif. Type de requête à envoyer au serveur Web (Content Server). Les types valides sont GET ou POST. La valeur par défaut est GET.

    • optionalParms (String) : paramètre facultatif. Tout paramètre à transmettre au WebReport cible en cours d'exécution. optionalParms présente le format suivant : &parm1=value1,&parm2=value2...

  • En cas d'erreur, un message d'alerte descriptif apparaît dans le navigateur.

  • Par exemple, consultez Exemple d'utilisation de la fonction sendRequest( URLreference, responseFunction, reqType, optionalParms ).
