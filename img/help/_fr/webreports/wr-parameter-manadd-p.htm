Pour ajouter manuellement un paramètre WebReport :

  1. Dans le menu Fonctions du WebReport, sélectionnez Propriétés. Sélectionnez ensuite Paramètres.

  2. Dans l'onglet Paramètres :

    1. Sélectionnez le bouton + pour ajouter un nouveau paramètre.

       

      Astuce :
      Dans le cas où aucun autre paramètre n'a été défini, l'option Cliquer ici pour ajouter un paramètre peut également être utilisée.

       

    2. Dans la nouvelle ligne vide créée, ajoutez un nom de paramètre, un texte d'affichage, un ordre d'invite, une valeur par défaut et d'autres paramètres. Pour plus d'informations sur les options des paramètres, voir Réglages de l'onglet Paramètres.

    3. Sélectionnez Mettre à jour.
