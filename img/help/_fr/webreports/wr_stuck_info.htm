Other software available from Resonate KT:

WebReports Workflow Extensions

WebReports Workflow Extensions integrates WebReports powerful reporting and presentation capabilities into Open Text Content Server's workflow engine. Workflow steps can now be completely re-skinned and reports can be viewed and executed without ever leaving the workflow assignment you are working on.

ActiveView

ActiveView is the leading tool for customization and simplification of Open Text's Content Server user interface. Unlike alternative approaches, ActiveView's attraction is its ability to entirely customize the browse interface whilst retaining optimal system performance. With a minimum of effort ActiveView can readily make Content Server (Livelink Enterprise Server) objects look and operate the way you need them to.
