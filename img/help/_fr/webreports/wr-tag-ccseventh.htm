Cette section apporte des informations sur la balise de contrôle de contenu RUNIF.

RUNIF

Syntaxe

[LL_WEBREPORT_RUNIF (expression) /]

Description

Entraîne l'annulation du traitement d'un WebReport si l'une des conditions est remplie. Si l'expression spécifiée est définie sur FALSE, alors la source de données n'est pas exécutée et aucun traitement du WebReport n'est effectué.

Paramètres

Format pour (expression) : "parmA" <opérateur> "parmB" <clauses facultatives>

Exemple

[LL_WEBREPORT_RUNIF "[LL_REPTAG_$GROUPA USERINGROUP /]" == "TRUE" /]

Dans cet exemple, si l'utilisateur exécutant le rapport n'appartient pas au groupe A, comme référencé par une constante, alors le rapport n'est pas exécuté. Cette balise fonctionne uniquement dans la section d'en-tête. En d'autres termes, devant la balise [LL_WEBREPORT_STARTROW /].
