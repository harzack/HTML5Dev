Pour afficher l'audit des éléments de collection :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Cliquez sur l'icône Fonctions de la collection dont vous souhaitez consulter l'audit.

  3. Depuis le menu Fonctions de la collection, cliquez sur Propriétés. Depuis le sous-menu Propriétés, cliquez sur Audit des éléments de collection.
