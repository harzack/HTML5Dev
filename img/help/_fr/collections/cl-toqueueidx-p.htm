Pour mettre en attente une collection pour l'indexation :

 

Attention
L'indexation d'une collection doit être utilisée avec précaution, car il s'agit d'une opération dont les performances sont parfois coûteuses et pour laquelle plusieurs heures ou plusieurs jours peuvent être nécessaires.

 

  1. Pour accéder à la collection à indexer à nouveau, procédez de l'une des façons suivantes :

    • Depuis l'espace de travail ou le dossier de votre choix contenant une collection, cliquez sur le nom de la collection à indexer à nouveau.

    • Dans la barre de menu générale, dans le menu Personnel, sélectionnez Collections. Sur la page Gérer les collections, cliquez sur le nom de la collection à indexer à nouveau.

  2. Sur la page <nom_collection>, à l'aide des cases à cocher de sélection multiple, sélectionnez la case à cocher en regard de chaque élément de la collection à indexer à nouveau. Vous pouvez également cocher la case Sélectionner/Désélectionner tout pour sélectionner tous les éléments de la collection.

  3. Depuis le menu Autres actions..., sélectionnez File d'attente pour l'indexation.

  4. Dans la boîte de dialogue Placer la collection dans la file d'attente pour confirmation de l'indexation, les champs Description, Nom de la collection et Résumé donnent uniquement des informations et ne sont pas modifiables.

  5. Dans la zone Notification :

    1. Dans le champ Destinataire, saisissez l'adresse e-mail de la personne qui recevra cette notification.

    2. Dans le champ Objet du message, saisissez l'objet du message.

    3. Facultatif Dans la zone Notifications tierces, sélectionnez l'un des deux boutons d'options disponibles :

      1. sélectionnez Aucune notification supplémentaire si vous ne voulez pas ajouter de destinataires d'e-mails supplémentaires.

      2. Sélectionnez Groupe pour ajouter des destinataires d'e-mails supplémentaires. Effectuez ensuite une ou plusieurs des actions suivantes :

        • Dans le champ Utilisateur, cliquez sur Rechercher un utilisateur et sélectionnez un utilisateur Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Groupe, cliquez sur Rechercher un groupe et sélectionnez un groupe Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Autre adresse e-mail, saisissez une adresse e-mail pour envoyer une notification à une adresse e-mail non disponible dans Content Server.

  6. Cliquez sur Envoyer.
