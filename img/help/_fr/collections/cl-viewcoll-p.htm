Pour afficher une collection :

  • Procédez de l'une des manières suivantes :

    • Depuis l'espace de travail ou le dossier de votre choix contenant une collection, cliquez sur le nom de la collection à afficher.

    • Dans la barre de menu générale, dans le menu Personnel, sélectionnez Collections.

      Sur la page Gérer les collections, cliquez sur le nom de la collection à afficher.
