Pour créer une image de disque :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes collections, cliquez sur le nom de la collection à partir de laquelle sélectionner des éléments afin de créer votre image de disque.

  3. Sur la page <votre_nom_de_collection>, sélectionnez la case à cocher correspondant aux éléments à inclure dans l'image de disque.

    Vous pouvez également sélectionner la case à cocher Sélectionner/Désélectionner tout pour inclure tous les éléments de la collection.

  4. Dans la liste Autres actions, sélectionnez Créer une image de disque.

  5. Sur la page Créer une image de disque, les champs Résumé et Chemin de destination fournissent les informations sur les éléments à inclure dans l'image de disque et sur le chemin où Content Server crée l'image de disque.

  6. Dans la liste Tailles de disque disponibles, cliquez sur la taille de disque appropriée.

  7. Dans la liste Format du système de fichiers du disque, cliquez sur le type de format approprié.

  8. Facultatif Dans le champ Collection, sélectionnez les cases à cocher correspondant aux types d'informations relatifs à la Collection que vous voulez inclure dans l'image de disque.

  9. Facultatif Dans le champ Contenu, sélectionnez les cases à cocher correspondant aux types d'informations relatifs aux Eléments de la collection que vous voulez inclure dans l'image de disque.

  10. Dans le champ Gestion de la version actuelle, sélectionnez l'option pour déterminer le traitement des versions actuelles lors du processus de création du disque.

    • Ne pas forcer l'inclusion : à moins que l'utilisateur ait sélectionné la version actuelle, cette dernière ne sera pas incluse.

    • Inclure la version actuelle : les versions actuelles de tous les éléments seront incluses.

  11. Dans le champ Langue des métadonnées, sélectionnez votre langue préférée dans la liste.

  12. Dans le paramètre Zone de Notification :

    1. Selon la manière dont votre administrateur a défini Créer une image de disque, vous pouvez préciser le moment où doit être envoyée la notification à l'aide de la liste Lorsque.

    2. Dans le champ Destinataire, si définie, votre adresse e-mail s'affiche. Vous pouvez modifier ce paramètre pour que la notification soit envoyée à une autre adresse e-mail.

    3. Dans le champ Objet du message, vous pouvez modifier l'objet du message par défaut.

    4. Dans la zone Notifications tierces, sélectionnez l'un des boutons d'options disponibles :

      1. sélectionnez Aucune notification supplémentaire si vous ne voulez pas ajouter de destinataires d'e-mails supplémentaires.

      2. Sélectionnez Groupe pour ajouter des destinataires d'e-mails supplémentaires.

        Effectuez ensuite une ou plusieurs des actions suivantes :

        • Dans le champ Utilisateur, cliquez sur Rechercher un utilisateur et sélectionnez un utilisateur Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Groupe, cliquez sur Rechercher un groupe et sélectionnez un groupe Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Autre adresse e-mail, saisissez une adresse e-mail pour envoyer une notification à une adresse e-mail non disponible dans Content Server.

  13. Cliquez sur Envoyer.
