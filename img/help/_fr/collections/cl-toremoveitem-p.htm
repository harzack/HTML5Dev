Pour enlever des éléments d'une collection :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes Collections, sélectionnez les cases à cocher de sélection multiple pour chaque élément à enlever.

  3. Dans la liste Autres actions..., sélectionnez Enlever des éléments d'une collection.

  4. Sur la page Enlever des éléments d'une collection, cliquez sur Enlever.
