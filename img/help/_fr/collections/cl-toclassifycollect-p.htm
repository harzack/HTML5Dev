Pour classer des éléments dans une collection :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes Collections, sélectionnez une collection.

  3. Sélectionnez la case à cocher de chaque élément que vous souhaitez classer. Dans la liste Autres actions, sélectionnez Appliquer les classements.

  4. Sur la page Classer les éléments d'une collection, cliquez sur l'une des options suivantes de la liste Classer :

    • Parcourir les classements

    • Mes favoris

  5. Dans la fenêtre Parcourir les classements ou Mes classements favoris, recherchez le classement que vous souhaitez assigner à l'élément, Cochez la case correspondante.

  6. Cliquez sur Envoyer.

  7. Sur la page Classer les éléments d'une collection, cliquez sur Appliquer.

 

Note :
L'option Appliquer les classements n'est disponible que si votre administrateur vous a accordé les privilèges d'utilisation appropriés.
