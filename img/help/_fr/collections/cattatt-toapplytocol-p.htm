Pour appliquer des catégories à des éléments collectés :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes Collections, sélectionnez la case à cocher en regard des éléments auxquels appliquer des catégories.

  3. Dans la liste Autres actions, cliquez sur Appliquer les catégories.

  4. Le cas échéant, renseignez la zone Notification de la page Appliquer les catégories comme suit :

    1. Dans le champ Activé, sélectionnez la case à cocher si vous voulez envoyer une notification par e-mail.

    2. Dans le champ Destinataire, si définie, votre adresse e-mail s'affiche. Vous pouvez modifier ce paramètre pour que la notification soit envoyée à une autre adresse e-mail.

    3. Dans le champ Objet du message, vous pouvez modifier l'objet du message par défaut.

    4. Dans la zone Notifications tierces, sélectionnez l'un des deux boutons d'options disponibles :

      1. sélectionnez Aucune notification supplémentaire si vous ne voulez pas ajouter de destinataires d'e-mails supplémentaires.

      2. Sélectionnez Groupe pour ajouter des destinataires d'e-mails supplémentaires.

        Effectuez ensuite une ou plusieurs des actions suivantes :

        • Dans le champ Utilisateur, cliquez sur Rechercher un utilisateur et sélectionnez un utilisateur Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Groupe, cliquez sur Rechercher un groupe et sélectionnez un groupe Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Autre adresse e-mail, saisissez une adresse e-mail pour envoyer une notification à une adresse e-mail non disponible dans Content Server.

  5. Sur la page Appliquer les catégories, cliquez sur Modifier du champ Catégories.

  6. Sur la page Modifier les catégories, effectuez l'une des actions suivantes :

    • Cliquez sur le bouton Ajouter des catégories pour ajouter une catégorie, puis cliquez sur Appliquer.

    • Cliquez sur Mettre à niveau pour mettre à niveau des catégories existantes vers la version la plus récente.

  7. Facultatif Sélectionnez la case à cocher Ajouter une version pour ajouter une version à des éléments multiversions, puis sélectionnez la case à cocher Vider les catégories existantes pour supprimer toutes les catégories existantes des éléments.

  8. Cliquez sur Envoyer.

 

Note :
La notification par e-mail est disponible uniquement lorsque vous appliquez des catégories à tous les éléments de la collection.
