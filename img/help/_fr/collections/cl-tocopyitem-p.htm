Pour copier des éléments collectés dans une autre collection :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes collections, cliquez sur le nom de la collection contenant les éléments à copier. Sue la page <nom_collection>, sélectionnez les cases à cocher de sélection multiple pour chaque élément à copier.

  3. Dans la liste Autres actions..., cliquez sur Copier les éléments dans une autre collection.

  4. Sur la page Copier les éléments dans une autre collection, choisissez l'une des opérations suivantes dans la liste Sélectionner :

    • Cliquez sur Créer une nouvelle collection pour créer une collection. Saisissez un nom pour votre nouvelle collection, modifiez d'autres champs à votre convenance, puis cliquez sur Ajouter.

    • Cliquez sur Parcourir les collections pour rechercher une collection existante. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

    • Cliquez sur Mes collections favorites pour sélectionner une collection répertoriée dans les Favoris. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

  5. Cliquez sur Copier.
