Il existe deux façons d'ajouter une collection. La première est de créer une collection vide, puis d'ajouter ultérieurement des éléments à votre collection. La seconde est de sélectionner des éléments, puis de créer la collection.

Pour créer une collection vide :

  1. Depuis le menu Ajouter un élément, cliquez sur Collection.

  2. Sur la page Ajouter : Collection, saisissez un nom pour la collection dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

  3. Facultatif Dans le champ Description, ajoutez une description à la collection.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

  4. Facultatif Le cas échéant, vous pouvez appliquer un classement à la collection. Depuis la liste des Classements, sélectionnez Parcourir les classements ou Mes favoris.

    1. Dans la boîte de dialogue qui apparaît, sélectionnez le classement que vous souhaitez appliquer au document.

    2. Assurez-vous que la case à cocher Action située en regard du classement est bien activée.

    3. Cliquez sur Envoyer.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette collection.

  6. Facultatif Si vous souhaitez placer la collection à un autre emplacement que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel vous souhaitez rechercher la collection, puis cliquez sur son lien Sélectionner.

  7. Cliquez sur Ajouter.

Pour sélectionner des éléments et créer une collection :

  1. Depuis le dossier ou l'espace de travail de votre choix, sélectionnez les cases à cocher de sélection multiple en regard de chaque élément à ajouter à votre nouvelle collection.

  2. Sélectionnez Collecter dans la barre d'outils. Le bouton Collecter peut être disponible dans la barre d'outils ou être répertorié sous l'option Plus... de la barre d'outils.

  3. Facultatif Sur la page Eléments à collecter, dans la zone Eléments, pour vous assurer que la version actuelle de tout élément est collectée, sélectionnez la case à cocher Collecter la version actuelle en regard de cet élément.

  4. Dans le champ Collection, sélectionnez Créer une nouvelle collection dans la liste.

  5. Dans la boîte de dialogue Ajouter : Collection :

    1. Dans le champ Nom, saisissez un nom pour votre nouvelle collection.

      Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

    2. Facultatif Dans le champ Description, ajoutez une description à la collection.

      Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

    3. Facultatif Le cas échéant, vous pouvez appliquer un classement à la collection. Depuis la liste des Classements, sélectionnez Parcourir les classements ou Mes favoris.

      1. Dans la boîte de dialogue qui apparaît, sélectionnez le classement que vous souhaitez appliquer au document.

      2. Assurez-vous que la case à cocher Action située en regard du classement est bien activée.

      3. Cliquez sur Envoyer.

    4. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette collection.

    5. Facultatif Si vous souhaitez placer la collection à un autre emplacement que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel vous souhaitez rechercher la collection, puis cliquez sur son lien Sélectionner.

    6. Cliquez sur Ajouter.

  6. Sur la page Eléments à collecter, cliquez sur Envoyer.

     

    Note :
    Si vous sélectionnez le bouton Annuler, il se peut qu'une collection vide ait été créée si vous avez sélectionné Ajouter à l'étape précédente. Vous devez supprimer manuellement la collection vide.

     

  7. Sur la page Collecte d'éléments vers cette collection, cliquez sur OK.
