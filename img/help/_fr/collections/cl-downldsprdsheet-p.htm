Pour télécharger une collection comme feuille de calcul :

  1. Accédez à un recueil dans Content Server contenant la collection à télécharger comme feuille de calcul. Cliquez sur le nom de cette collection.

  2. Une fois dans la collection, sélectionnez la case à cocher située en regard de chaque élément à télécharger à l'aide des cases à cocher de sélection multiple. Vous pouvez également télécharger l'ensemble de la collection.

  3. Dans la liste Autres actions..., sélectionnez Télécharger comme feuille de calcul.

  4. Dans la boîte de dialogue Téléchargement de fichier sélectionnez l'une des options suivantes :

    • Ouvrir pour ouvrir la feuille de calcul téléchargée.

    • Enregistrer pour enregistrer la feuille de calcul téléchargée sur votre ordinateur.

    • Annuler pour annuler l'opération.

  5. Si vous avez sélectionné Enregistrer comme ci-dessus, procédez comme suit dans la boîte de dialogue Enregistrer sous qui apparaît :

    1. Accédez à l'emplacement sur votre ordinateur pour enregistrer la feuille de calcul.

    2. Facultatif Vous pouvez soit accepter le nom par défaut de la feuille de calcul, soit saisir un nouveau nom dans le champ Nom de fichier.

    3. Cliquez sur Enregistrer.
