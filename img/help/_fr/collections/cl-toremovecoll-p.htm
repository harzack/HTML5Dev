Pour supprimer une collection :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Collections.

  2. Sur la page Mes collections, cliquez sur l'icône Fonctions de la collection à supprimer. Depuis le menu Fonctions, sélectionnez Supprimer.

  3. Sur la page Supprimer les collections, cliquez sur Supprimer.

  4. Sur la page Suppression de l'élément dans, cliquez sur OK.

 

Note :
Lorsque vous supprimez une collection, les éléments de la collection ne sont pas supprimés car les collections ne contiennent que des pointeurs vers les éléments. Pour plus d'informations sur l'utilisation des éléments dans une collection, reportez-vous à la rubrique Collection d'éléments.
