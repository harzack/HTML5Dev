Lorsqu'une collection a été créée et que les éléments ont été enregistrés vers cette collection, vous pouvez la télécharger comme feuille de calcul à l'aide de l'option Télécharger comme feuille de calcul disponible dans la liste Autres actions.

Vous pouvez utiliser les cases à cocher de sélection multiple pour sélectionner des éléments spécifiques dans la collection à télécharger, ou télécharger l'ensemble de la collection. Si vous sélectionnez l'ensemble de la collection, vous pouvez spécifier le type d'élément de la collection à télécharger à l'aide du filtre Type d'objet.

 

Note :
La fonction Télécharger comme feuille de calcul dépend des privilèges attribués par votre administrateur et peut ne pas être disponible.

 

Lors du téléchargement d'une collection importante, une barre de progression indique le pourcentage réalisé et le temps restant pour terminer l'opération.

 

Note :
Seuls les éléments correspondant aux critères du filtre seront inclus dans la feuille de calcul.

Seules les métadonnées des éléments dans la collection sont placées dans la feuille de calcul. Les éléments réels ne sont pas extraits de Content Server.

 

A chaque fois qu'une collection est téléchargée comme feuille de calcul, les informations sont enregistrées dans l'onglet Audit des éléments de collection pour la collection. Vous pouvez ainsi afficher tous les événements passés liés à la collection, notamment l'intégration des éléments dans les feuilles de calcul actuelles et passées. Les administrateurs peuvent obtenir plus d'informations sur l'audit des éléments de collection dans Envoi de l'audit des éléments de collection pour révision.

Feuille de calcul téléchargée

La feuille de calcul téléchargée d'une collection est écrite au format CSV (valeurs séparées par des virgules) et enregistrée avec le suffixe .csv. La convention de dénomination utilisée pour enregistrer la feuille de calcul sera Collection export <your_collection_name> < YYYY>-<MM>-<DD> < HHMM>.csv, où :

  • <votre_nom_de_collection> correspond au nom que vous avez saisi pour votre collection.

  • <AAAA>-<MM>-< JJ> correspond à l'année sous forme de nombre à quatre chiffres, au mois à deux chiffres et au jour à deux chiffres auxquels la feuille de calcul téléchargée d'une collection a été créée.

  • <HHMM> correspond à l'heure et aux minutes, affiché au format 24 heures, auxquelles cette feuille de calcul téléchargée d'une collection a été créée. Par exemple, 7:15 du matin est affiché sous la forme 0715, alors que 7:15 du soir est affiché sous la forme 1915.

Un exemple de nom de fichier d'une feuille de calcul téléchargée d'une collection est : Collection export My_Collection 2012-01-14 1605.csv

Seules les métadonnées des éléments dans la collection sont placées dans la feuille de calcul. Les éléments réels ne sont pas extraits de Content Server.

La feuille de calcul affiche les informations dans plusieurs colonnes dans deux formats différents.

Premier format : En-tête

Le premier format, l'en-tête, s'affiche en haut de la feuille de calcul comme suit :

Le titre apparaît en haut de la feuille de calcul : Content Server Rapport de collection exporté

Nom de la collection : <nom_collection> ID de la collection : <xxx>
Date d'exportation : JJ/MM/AAAA Heure d'exportation : HH:MM matin/soir
ID de l'utilisateur :
<id_utilisateur>
L'ID d'utilisateur de la personne qui a créé la collection.
 Nom de l'utilisateur :
<nom_utilisateur>
Le nom d'utilisateur de la personne qui a créé la collection.
Eléments dans la collection :
<n>
Nombre d'éléments dans la collection.
 Eléments sélectionnés :
<n>
Le nombre d'éléments sélectionnés.
Nom du filtre :
<nom_filtre>
Le nom du filtre utilisé pour filtrer les éléments de la collection.
 Type de filtre :
<type_filtre>
Le type de filtre utilisé pour filtrer les éléments de la collection.
Section :
<n>
Le numéro de la section.
 Elément de démarrage :
<n>
Le numéro de l'élément de démarrage dans la collection.

Second format : Corps

Dans le second format trouvé dans la feuille de calcul, une ligne est créée pour chaque élément exporté vers la feuille de calcul. Chaque ligne contient toutes les métadonnées pertinentes importées de la collection.

Le second format, le corps de la feuille de calcul, affiche les informations suivantes :

  • ID d'objet

    L'ID de l'objet dans Content Server.

  • Nom

    Le nom de l'objet dans Content Server.

  • ID de type

    L'ID de type de l'objet dans Content Server.

  • Type

    Le type de l'objet dans Content Server.

  • Créé

    La date à laquelle l'objet a été créé dans Content Server, au format JJ/MM/AAAA.

  • Modifié

    La dernière date à laquelle l'objet a été modifié dans Content Server, au format JJ/MM/AAAA.

  • Créé par

    L'ID de l'utilisateur ayant créé l'objet dans Content Server.

  • Modifié par

    L'ID de l'utilisateur ayant modifié en dernier l'objet dans Content Server.

  • Propriétaire

    L'ID d'utilisateur du propriétaire de l'objet dans Content Server.

  • Description

    La description de l'objet dans Content Server, si elle a bien été saisie pour l'objet.

  • Taille

    La taille de l'objet dans Content Server.

  • Chemin complet

    Le chemin de l'emplacement de l'objet dans Content Server.

  • Version

    Le numéro de version de l'objet dans Content Server.

  • Current

    « OUI » ou « NON » pour indiquer s'il s'agit de la version actuelle de l'objet dans Content Server.

  • Nom du fichier

    Le nom du fichier de l'objet dans Content Server.

  • Taille du fichier (en octets)

    La taille de l'objet, en octets, dans Content Server.

  • Version créée

    La date à laquelle la version de l'objet a été créée dans Content Server, au format JJ/MM/AAAA.

  • Version créée par

    L'ID d'utilisateur de la personne ayant créé la version de l'objet dans Content Server.

  • Commentaire de version

    S'il s'agit d'une version, et si un commentaire de version a été créé pour cette version, le commentaire de version est répertorié ici.

La ligne de pied de page de la feuille de calcul

La ligne de pied de page de la feuille de calcul comprend la colonne suivante :

  • Eléments exportés

En pied de page, la valeur du titre « Eléments exportés » correspond au nombre de lignes d'éléments dans le corps de la feuille de calcul.
