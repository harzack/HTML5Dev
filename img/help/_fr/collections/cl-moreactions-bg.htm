Après avoir ajouté un élément à une collection, vous pouvez retirer le pointeur d'un élément donné de la collection grâce au menu Fonctions. Vous pouvez également copier, déplacer et enlever un ou plusieurs pointeurs d'une collection à une autre à l'aide de la liste Autres actions de la page Collection. Ces actions permettent :

  • la copie des éléments dans une autre collection ;

  • le déplacement des éléments vers une autre collection ;

  • le retrait des éléments d'une collection.

 

Note :
Si vous sélectionnez un élément pour lequel des versions spécifiques ont été collectées, lorsque vous sélectionnez l'une de ces options, toutes les versions collectées de cet élément sont copiées, déplacées ou enlevées.

 

Si vous voulez enlever une version spécifique d'un élément, vous pouvez enlever uniquement cette version de la collection à l'aide du menu Fonctions associé à cette version. Si vous enlevez toutes les versions d'un élément de cette collection, l'élément reste dans la collection jusqu'à ce que vous sélectionniez Enlever de la collection depuis le menu Fonctions.

 

Note :
Les options de menu à votre disposition dépendent des privilèges qui vous ont été attribués par votre administrateur. Par conséquent, certaines des options de menu décrites dans cette section ne sont pas disponibles.
