Pour ajouter des éléments à une collection à partir d'une page Résultats de recherche :

  1. Sur une page Résultats de recherche, sélectionnez les cases à cocher de sélection multiple en regard des résultats de recherche que vous voulez collecter.

  2. Dans la liste Sélectionner l'action..., sélectionnez Ajouter à la collection.

  3. Sur la page Collecter les résultats de recherche, dans le champ Collection, choisissez l'une des opérations suivantes dans la liste Sélectionner... :

    • Pour créer une nouvelle collection, sélectionnez Créer une nouvelle collection. Saisissez un nom pour votre nouvelle collection. Vous pouvez également modifier d'autres champs à votre convenance, cliquez ensuite sur Ajouter.

    • Pour rechercher une collection existante, sélectionnez Parcourir les collections. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

    • Pour sélectionner une collection répertoriée dans les favoris, sélectionnez Mes collections favorites. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

  4. Cliquez sur Envoyer.
