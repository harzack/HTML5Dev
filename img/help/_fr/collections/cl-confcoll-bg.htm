Collections peuvent être ajoutées à l'espace de travail de Content Server, à votre espace de travail personnel ou à un dossier de Content Server.

Lorsque vous ajoutez une collection à Mon espace de travail, celle-ci est ajoutée à votre page Mes collections. Une collection peut être ajoutée, renommée, supprimée ou enlevée.

 

Note :
La capacité de créer une collection dépend des privilèges qui vous ont été attribués par votre administrateur. Par conséquent, il se peut que vous n'ayez pas accès à cette option.

 

Vous pouvez accéder à votre page personnelle Mes Collections dans la barre de menu générale depuis le menu Personnel en sélectionnant Collections. Sur la page Mes collections, l'onglet Mes collections affiche les colonnes d'informations suivantes sur les collections :

  • Type

  • Nom

  • Emplacement

  • Taille

  • Modifié

Une fois que vous avez ajouté une collection personnelle, vous pouvez commencer à ajouter des éléments collectés. Pour plus d'informations sur la collection des éléments, reportez-vous à la rubrique Collection d'éléments.

Si l'administrateur vous a accordé le niveau d'autorisation voulu, vous pourrez non seulement classer des éléments spécifiques dans une collection, mais également classer des éléments multiples. Vous pourrez, par exemple, appliquer un classement à tous les dossiers d'une collection. De plus, lorsque vous classez une collection entière, vous pouvez envoyer une notification par e-mail pour informer différentes personnes que l'ensemble d'une collection a été classé. Pour plus d'informations sur l'application des classements, voir Utilisation de classements.
