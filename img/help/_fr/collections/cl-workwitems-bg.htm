Sur la page Collections, chaque élément collecté dispose d'un menu Fonctions. Selon le type d'objet et vos autorisations par rapport à l'élément, vous pouvez exécuter des fonctions de base pour les éléments d'origine (pas les pointeurs) de la collection. Les exemples de fonctions de base incluent télécharger, ouvrir et supprimer. Ces fonctions vous permettent d'accéder facilement aux éléments d'une collection et de les gérer sans avoir à rechercher l'emplacement de ces éléments.

Si vous avez regroupé les versions d'un élément, chaque version est répertoriée et dispose d'un menu Fonctions restreint qui permet d'exécuter certaines des fonctions de base de cette version. En outre, le numéro de version indiqué dans la colonne Version de la page Collections correspond à un lien hypertexte qui peut ouvrir soit la page Vue d'ensemble de la version, soit la version spécifique de l'élément, selon les paramètres définis.

 

Note :
Les options à votre disposition dépendent également des privilèges qui vous ont été attribués par votre administrateur. Par conséquent, vous pouvez ne pas avoir accès à certaines options de menu décrites dans cette rubrique, et ce même si vous disposez des autorisations appropriées pour l'élément.

 

Pour plus d'informations sur la copie, le déplacement et la suppression d'éléments, voir Utilisation des éléments généraux.
