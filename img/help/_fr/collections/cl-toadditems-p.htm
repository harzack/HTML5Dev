Pour ajouter des éléments à une collection à partir d'un espace de travail :

  1. Dans la vue détaillée de tout espace de travail, sélectionnez les cases à cocher de sélection multiple en regard de chaque élément à ajouter à votre nouvelle collection.

  2. Sélectionnez Collecter dans la barre d'outils. Le bouton Collecter peut être disponible dans la barre d'outils ou être répertorié sous l'option Plus... de la barre d'outils.

  3. Sur la page Eléments à collecter, dans la zone Eléments, pour vous assurer que la version actuelle de tout élément est collectée, sélectionnez la case à cocher Collecter la version actuelle en regard de cet élément.

  4. Dans le champ Collection, dans la liste Sélectionner, cliquez sur l'un des éléments suivants :

    • Créer une nouvelle collection pour ajouter ces éléments à la nouvelle collection que vous avez créée. Saisissez un nom pour votre nouvelle collection, modifiez d'autres champs à votre convenance, puis cliquez sur Ajouter.

    • Parcourir les collections pour ajouter ces éléments à une collection existante que vous recherchez. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

    • Mes collections favorites pour ajouter ces éléments à une collection sélectionnée dans une collection répertoriée dans les favoris. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

  5. Cliquez sur Envoyer.
