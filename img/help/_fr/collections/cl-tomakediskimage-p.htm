To make a disk image:

  1. Select the collected items, and click Make Disk Image from the More Actions drop-down list.

  2. On the Make Disk Image page, click the appropriate disk size in the Available Disk Sizes drop-down list.

  3. Click the appropriate format type in the Disk File System Format drop-down list.

  4. In the Notification section, specify when to send notification.

  5. Type your e-mail address in the Recipient field.

  6. Type a description for the e-mail in the Message Subject filed.

  7. In the 3rd Party Notification section, select one of the following radio buttons:

    • No Additional Notification

    • Group, which allows you to select a User, Group, or an external e-mail address

  8. Click the Submit button.
