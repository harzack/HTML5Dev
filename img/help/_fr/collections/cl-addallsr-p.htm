Pour ajouter tous les résultats de recherche à une collection :

  1. Depuis une page Résultats de recherche, sélectionnez Collecter tous les résultats de recherche dans la liste Outils de recherche.

  2. Sur la page Collecter tous les résultats de recherche, dans la zone Notification :

    1. Facultatif Si vous voulez envoyer une notification par e-mail, dans le champ Activé, sélectionnez la case à cocher.

    2. Facultatif Dans le champ Destinataire, si définie sur votre page Mon profil, votre adresse e-mail s'affiche. Pour modifier ce paramètre et pour que la notification soit envoyée à une autre adresse e-mail, saisissez la nouvelle adresse e-mail dans le champ de texte.

    3. Facultatif Dans le champ Objet du message, vous pouvez accepter l'objet du message par défaut ou saisir un nouvel objet dans le champ de texte.

    4. Dans la zone Notifications tierces, sélectionnez l'un des deux boutons d'options disponibles :

      1. sélectionnez Aucune notification supplémentaire si vous ne voulez pas ajouter de destinataires d'e-mails supplémentaires.

      2. Sélectionnez Groupe pour ajouter des destinataires d'e-mails supplémentaires.

        Effectuez ensuite une ou plusieurs des actions suivantes :

        • Dans le champ Utilisateur, cliquez sur Rechercher un utilisateur et sélectionnez un utilisateur Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Groupe, cliquez sur Rechercher un groupe et sélectionnez un groupe Content Server auquel vous souhaitez envoyer une notification par e-mail.

        • Dans le champ Autre adresse e-mail, saisissez une adresse e-mail pour envoyer une notification à une adresse e-mail non disponible dans Content Server.

  3. Dans le champ Collection, dans la liste Sélectionner..., cliquez sur l'un des éléments suivants :

    • Cliquez sur Créer une nouvelle collection pour créer une collection. Saisissez un nom pour votre nouvelle collection, modifiez d'autres champs à votre convenance, puis cliquez sur Ajouter.

    • Cliquez sur Parcourir les collections pour rechercher une collection existante. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

    • Cliquez sur Mes collections favorites pour sélectionner une collection répertoriée dans les Favoris. Recherchez la collection que vous souhaitez sélectionner, puis cliquez sur le lien Sélectionner de cette collection.

  4. Cliquez sur Envoyer.
