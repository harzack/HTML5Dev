Pour modifier un message publié sur un forum :

  1. Sur la page principale du forum, cliquez sur l'icône Fonctions de la publication, puis sur Modifier l'entrée.

  2. Sur la page Modifier l'entrée Forum, effectuez l'une des actions suivantes :

    • Modifiez le titre et le contenu de la publication dans les champs Titre et Sujet.

    • Cliquez sur un bouton radio Icône pour changer l'icône par défaut pour la publication.

    • Sélectionnez un type d'élément dans la liste Ajouter un élément pour ajouter une pièce jointe à l'entrée. Complétez les informations requises, puis cliquez sur Ajouter.

  3. Cliquez sur Soumettre.
