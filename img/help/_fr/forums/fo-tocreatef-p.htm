Pour créer et configurer un forum :

  1. Cliquez sur Forum dans le menu Ajouter un élément.

  2. Sur la page Ajouter un forum, saisissez le titre du forum dans le champ Titre.

  3. Dans la zone Annuaires, activez les cases à cocher correspondant aux annuaires auxquels le forum devra être associé.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Créer.

  6. Sur la page Options avancées, saisissez les limites d'utilisation dans le champ Limites d'utilisation.

  7. Saisissez un avertissement de publication dans le champ Avertissement de publication.

  8. Dans la zone En-tête, procédez de la façon suivante :

    • Sélectionnez ou désélectionnez la case à cocher Afficher l'en-tête dans toutes les vues.

    • Saisissez et formatez le texte de l'en-tête dans le champ En-tête.

  9. Activez la case à cocher Activer les évaluations pour permettre l'attribution d'évaluations.

  10. Activez la case à cocher Activer les abonnements par e-mail et saisissez une adresse e-mail dans le champ Adresse e-mail de ce forum.

  11. Cliquez sur le bouton Choisir un utilisateur ou un groupe dans la zone Modérateur, puis sélectionnez un utilisateur ou un groupe pour le définir comme modérateur du forum.

  12. Activez la case à cocher Exiger une approbation de toutes les publications pour exiger une approbation de toutes les publications par le modérateur.

  13. Activez la case à cocher Permettre au forum de recevoir et répertorier les questions pour permettre au forum de recevoir et répertorier les questions.

  14. Activez la case à cocher Lister uniquement les questions urgentes pour configurer le forum afin qu'il ne répertorie que les questions urgentes.

  15. Saisissez un nombre dans le champ Nombre total de questions répertoriées dans ce forum (avec ou sans réponse) pour indiquer le nombre total de questions à afficher dans le forum.

  16. Sélectionnez Affichage par date, Visualisation des réponses ou Visualisation du thread dans la liste Vue par défaut pour définir l'affichage par défaut du forum. Pour plus d'informations, voir Gestion de votre affichage des forums.

  17. Cliquez sur Soumettre.
