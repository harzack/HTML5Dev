Pour répondre à un message publié sur un forum :

  1. Sur la page principale du forum, cliquez sur le nom du sujet de forum qui contient la publication à laquelle vous souhaitez répondre.

  2. Cliquez sur l'un des liens suivants :

    • Publier une réponse, pour publier une réponse qui n'inclura pas le texte de la publication à laquelle vous répondez ;

    • Publier une réponse avec citation, pour publier une réponse qui inclura le texte de la publication à laquelle vous répondez.

  3. Sur la page Ajouter un sujet de forum, saisissez le contenu de la réponse dans le champ Sujet.

  4. Dans la zone Icône, cliquez sur l'icône que vous souhaitez utiliser pour ce sujet.

  5. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  6. Cliquez sur Suivant.

  7. Sur la page Options supplémentaires, effectuez l'une des actions suivantes :

    • Dans la zone S'adresser à, cliquez sur l'icône Choisir un utilisateur ou un groupe , puis sélectionnez l'utilisateur à ajouter comme référence à la publication.

    • Dans la zone Pièce jointe sur disque local, cliquez sur Parcourir, accédez au fichier à ajouter comme pièce jointe, puis cliquez sur le bouton Ouvrir. Cliquez sur Charger.

    • Dans la zone Pièce jointe Content Server, cliquez sur Parcourir Content Server, naviguez jusqu'à l'élément à ajouter en tant que pièce jointe, puis cliquez sur le lien Sélectionner de l'élément.

    • Dans la zone URL en tant que pièce jointe, saisissez un nom pour l'URL dans le champ Nom, puis saisissez l'adresse URL dans le champ URL.

  8. Cliquez sur Terminer.
