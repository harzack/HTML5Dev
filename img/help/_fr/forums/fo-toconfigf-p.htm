Pour modifier les paramètres d'un forum :

  1. Cliquez sur l'icône Fonctions du forum, puis sur Modifier les paramètres.

  2. Sur la page des paramètres du forum, effectuez l'une des actions suivantes :

    • Modifiez le titre ou la description du forum.

    • Modifiez les limites d'utilisation du forum.

    • Modifiez l'avertissement de publication du forum.

    • Modifiez et formatez le texte de l'en-tête dans le champ En-tête.

    • Modifiez si besoin les autres options.

  3. Cliquez sur Soumettre.

 

Astuce :
Pour modifier les paramètres d'un forum sur sa page principale, cliquez sur le lien Modifier les paramètres.
