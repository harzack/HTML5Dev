Pour rejeter une publication :

  1. Cliquez sur l'icône Fonctions de la publication, puis sur Rejeter la publication.

  2. Sur la page Rejeter la publication, saisissez un objet pour le rejet de la publication dans le champ Objet.

  3. Tapez la raison du rejet dans le champ Message.

  4. Cliquez sur Envoyer.
