Outre l'ajout des forums aux favoris, qui permet de gérer une liste de vos forums favoris ainsi que d'autres éléments Content Server, vous pouvez afficher et gérer votre liste de forums personnels. Cette liste comporte tous les forums auxquels vous avez participé.

Vous pouvez également modifier l'affichage du thread d'un forum à l'aide des vues suivantes :

  • Affichage par date, qui classe le thread par date ;

  • Visualisation des réponses, qui affiche les publications en indentation pour différencier clairement les sujets de forum et les réponses ;

  • Visualisation du thread, qui réduit le thread à un affichage plus compact ;

  • Affichage pour impression, qui affiche une version imprimable du thread dans une nouvelle fenêtre.
