Lorsque vous souhaitez créer une vue personnalisée HTML d'un modèle de formulaire, vous commencez par exporter le modèle de formulaire dans un fichier HTML. Après avoir exporté le modèle de formulaire, vous pouvez créer la vue personnalisée en modifiant le fichier HTML. Pour plus d'informations sur la création de vues personnalisées, reportez-vous à la rubrique Définition des vues personnalisées
