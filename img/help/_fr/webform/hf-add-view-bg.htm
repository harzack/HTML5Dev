Après avoir exporté un modèle de formulaire sous forme de fichier HTML et l'avoir modifié, vous pouvez ajouter le fichier HTML comme vue personnalisée du modèle de formulaire. Pour plus d'informations sur les modèles de formulaire, reportez-vous à Utilisation de formulaires.

Création de boutons d'envoi JavaScript

Si vous souhaitez que les utilisateurs envoient les formulaires en cliquant sur le bouton du formulaire plutôt que sur le bouton Appliquer de Content Server, vous pouvez créer un bouton JavaScript dans le fichier HTML qui devient le modèle de formulaire Livelink. Cela n'est nécessaire que pour les formulaires ayant un mécanisme de révision et d'envoi. Si un formulaire n'a pas de mécanisme de révision, le bouton requis pour envoyer les données du formulaire vers le serveur HTTP active le mécanisme d'envoi.
