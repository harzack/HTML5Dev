Le module Web Forms permet de créer une vue personnalisée des modèles de formulaires en les exportant sous forme de fichiers HTML pour modification avant ajout en tant que vue personnalisée. Par exemple, vous pouvez masquer les champs que vous ne souhaitez pas afficher ou ajouter un formatage spécifique, des graphiques et autres caractéristiques HTML. Les formulaires basés sur la vue personnalisée que vous créez reflèteront les modifications que vous apportez au fichier HTML.

Pour plus d'informations sur les formulaires, reportez-vous à la rubrique Utilisation de formulaires.
