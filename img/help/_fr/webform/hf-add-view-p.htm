Pour ajouter un fichier HTML sous forme de vue :

  1. Cliquez sur l'icône Fonctions du modèle de formulaire auquel vous voulez ajouter la vue personnalisée, choisissez Propriétés, puis Vues.

  2. Dans l'onglet Vues de la page Propriétés, cliquez sur HTML dans le menu Ajouter une vue.

  3. Dans le champ Fichier HTML, cliquez sur Parcourir, puis sélectionnez le fichier HTML que vous souhaitez ajouter comme vue.

  4. Saisissez un nom dans le champ Nom.

  5. Saisissez une description dans le champ Description.

  6. Cliquez sur Ajouter une vue.

Pour plus d'informations sur cette procédure, consultez la section Ajout d'un fichier HTML sous forme de vue.
