Pour ajouter un modèle de formulaire HTML :

  1. Dans le menu Ajouter un élément, cliquez sur Modèle de formulaire.

  2. Si le module facultatif PDF Forms est installé, une page Type de modèle s'affiche. Activez la case d'option HTML, puis cliquez sur Continuer.

  3. Sur la page Ajouter un modèle, saisissez un nom de modèle de formulaire dans la zone Nom.

  4. Saisissez une description du modèle de formulaire dans la zone Description.

  5. Facultatif Cliquez sur Parcourir Content Server, accédez au recueil dans lequel vous souhaitez stocker le modèle, puis cliquez sur le lien Sélectionner du recueil.

  6. Cliquez sur Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Ajout d'un modèle de formulaire HTML.
