Vous pouvez configurer un schéma de workflow pour exécuter automatiquement des opérations de catégories et d'attributs sur des éléments Content Server, des pièces jointes du schéma de workflow, des attributs du schéma de workflow et des champs dans les formulaires ajoutés au schéma de workflow. Lorsque le processus du schéma de workflow atteint une étape Gestionnaire d'élément, il effectue les opérations de catégories et d'attributs que vous avez définies. Les actions que vous pouvez effectuer sont :

  • L'ajout, la modification ou la suppression de catégories et d'attributs d'un élément Content Server

  • La modification des valeurs actuelles des attributs du schéma de workflow ou des champs dans un formulaire ajouté au schéma de workflow

 

Note :
Avant de pouvoir exécuter des opérations de catégories et d'attributs dans des champs de formulaire, votre administrateur doit installer le module Content Server Forms sur votre serveur Content Server.

 

Les opérations que vous définissez sont basées sur un mappage entre les valeurs d'un attribut de catégorie Content Server et un attribut du schéma de workflow ou un champ de formulaire (et non les définitions d'attributs). Les actions de transfert de données que vous indiquez s'appliquent uniquement aux valeurs contenues dans chaque instance de l'attribut ou du champ. Prenons l'exemple d'un schéma de workflow contenant l'attribut de workflow de type Entier : Liste déroulante suivant :

Nom Valeurs légales
Prix 5, 10, 15, 20, 25, 30

Dans notre exemple, votre serveur Content Server contient un attribut de catégorie de type Entier : Liste déroulante :

Nom Valeurs légales
Coût 0, 10, 20, 30

Vous pouvez créer une définition de mappage entre les attributs Prix et Coût mais les opérations que vous effectuez peuvent uniquement s'appliquer aux valeurs légales communes de chaque attribut (10, 20, 30). Il n'est pas possible de définir une opération qui modifiera l'attribut Coût pour accepter la valeur 5. Si les attributs n'ont pas de valeurs communes, vous pouvez également définir un mappage mais vous ne pouvez pas exécuter d'opération entre les attributs.

 

Note :
Les champs de formulaire de type Ensemble ne sont pas pris en charge dans les opérations de catégories.

 

Choix des catégories et des attributs et configuration des mappages

Pour définir des opérations de catégories pour votre schéma de workflow, vous devez effectuer les tâches suivantes :

  • Choisir les catégories Content Server que vous souhaitez utiliser. Les catégories que vous choisissez sont affichées dans l'onglet Définitions.

Pour définir les opérations d'attributs, vous devez exécuter ces tâches supplémentaires :

  • Choisir les attributs de catégorie que vous souhaitez utiliser. Vous pouvez choisir un ou plusieurs attributs pour chaque catégorie.

  • Définir un mappage entre les attributs de catégorie que vous avez sélectionnés et les attributs du schéma de workflow ou les champs de formulaire.

Définition d'un mappage entre les attributs de catégorie et les attributs du schéma de workflow ou les champs de formulaire

Une fois que vous avez choisi les catégories et les attributs que vous souhaitez utiliser, vous créez une relation, appelée mappage, entre chaque attribut de catégorie et un attribut du schéma de workflow ou un champ du formulaire. Vous ne pouvez sélectionner que les attributs du schéma de workflow ou les champs du formulaire qui possèdent le même type d'attribut que celui de la catégorie. Par exemple, si un attribut de catégorie est de type Texte : Liste déroulante, vous pouvez créer un mappage uniquement avec les attributs du schéma de workflow ou les champs du formulaire qui sont du même type.

Spécification de l'élément et de l'opération

Une fois que vous avez sélectionné les catégories et les attributs, vous devez indiquer l'élément sur lequel l'opération de transfert de données est effectuée et le type d'opération que l'étape va exécuter. L'élément sur lequel l'opération de transfert de données est effectuée peut être l'un des suivants :

  • Une pièce jointe du schéma de workflow, un élément Content Server, un dossier créé par une définition de dossier sur la même étape ou un attribut de référence d'élément qui renvoie vers un élément Content Server.

  • Les attributs du schéma de workflow ou les champs du formulaire, ce qui permet la mise à jour des valeurs des attributs du schéma de workflow avec les valeurs des attributs de catégorie auxquels ils sont mappés.

Pour plus d'informations sur les définitions de dossier et les attributs de référence d'élément, voir Création de dossier et Définition des attributs.

 

Notes :

  • Les définitions de dossier sont uniquement disponibles si vous avez défini plus d'une définition de dossier pour la même étape Gestionnaire d'élément.

  • Un attribut de référence d'élément est uniquement disponible si vous en avez configuré un pour le schéma de workflow.

 

Une fois que vous avez spécifié un élément, sélectionnez les opérations de transfert de données à accomplir pour l'étape. Vous pouvez effectuer les opérations suivantes :

  • Mettre à jour les valeurs de l'attribut du schéma de workflow ou du champ du formulaire, qui permet de mettre à jour les attributs du schéma de workflow ou les champs du formulaire avec les valeurs des attributs de catégories auxquelles ils sont mappés.

  • Enlever des catégories, qui permet de supprimer les catégories et les attributs sélectionnés pour l'élément spécifié.

  • Ajouter des catégories, qui permet d'ajouter des catégories et des attributs que vous avez sélectionnés pour l'élément spécifié.

  • Définir des valeurs d'attribut, qui permet de mettre à jour les valeurs d'attribut de la catégorie avec les attributs du schéma de workflow ou des champs du formulaire mappés.

Par exemple, pour mettre à jour un attribut de schéma de workflow avec la valeur de l'attribut de la catégorie sur lequel vous avez créé un mappage, spécifiez un document comme élément, puis choisissez les attributs du schéma de workflow comme opération sur les données. Pour ajouter les catégories et les attributs que vous avez sélectionnés, spécifiez le document comme élément, puis choisissez l'opération Ajouter des catégories.
