Pour exporter un schéma de workflow :

  1. Cliquez sur l'icône Fonctions du schéma de workflow, puis choisissez Exporter.

  2. Procédez de l'une des manières suivantes :

    • Si vous utilisez Microsoft Internet Explorer, activez la case d'option Enregistrer ce fichier sur le disque local de la boîte de dialogue Téléchargement de fichier. Dans la boîte de dialogue Enregistrer sous, recherchez l'emplacement où vous souhaitez enregistrer le schéma de workflow exporté, puis cliquez sur Enregistrer.

    • Si vous utilisez le navigateur Netscape, recherchez l'emplacement où vous souhaitez enregistrer le schéma de workflow exporté, puis cliquez sur Enregistrer de la boîte de dialogue Enregistrer sous.

Pour plus d'informations sur cette procédure, consultez la section Importation et exportation des schémas de workflow.
