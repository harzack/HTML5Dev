Lorsque vous définissez une étape Départ, vous devez spécifier les informations suivantes :

  • des informations générales comme le nom de l'étape, les options de l'initiateur et des instructions générales ;

  • les instructions des commentaires si les commentaires ont été activés pour le lot de travail du schéma de workflow spécifié dans les propriétés Général ;

  • l'accès aux attributs et leurs instructions si les attributs ont été activés pour le lot de travail du schéma de workflow dans les propriétés Général ;

  • les instructions des pièces jointes si les pièces jointes ont été activées pour le lot de travail du schéma de workflow dans les propriétés Général.

Pour plus d'informations sur l'étape Départ, voir Etape Départ.

Définition des informations générales

Les informations générales que vous pouvez définir pour les étapes Départ sont le nom de l'étape, les instructions affichées dans la section Général de la page Lancer le workflow et les options d'étape suivantes :

  • Afficher au lancement, qui entraîne l'affichage de l'étape si sélectionnez cette option.

  • Demander un titre à l'initialisation, qui demande à l'initiateur d'indiquer un nom pour le schéma de workflow dans le champ Lancer sous de la fenêtre Général.

  • L'initiateur peut spécifier une échéance pour le workflow, qui permet à l'initiateur d'indiquer la date d'échéance du schéma de workflow dans le champ Date d'échéance de la fenêtre Général.

  • Authentifier, qui implique que l'initiateur fournisse un mot de passe Content Server valide pour authentifier l'action. L'authentification de l'étape Départ utilise les règles de connexion Content Server de la même manière que pour les étapes Utilisateur et Initiateur. Pour plus d'informations sur les règles de connexion Content Server, reportez-vous à la rubrique Etapes Utilisateur et Initiateur.

Définition des instructions de la section Commentaires

Si les commentaires sont activés dans le lot de travail du schéma de workflow, vous pouvez indiquer des instructions qui s'affichent dans la fenêtre Commentaires de la page Lancer le workflow. Pour plus d'informations sur la définition des instructions, reportez-vous à la rubrique Configuration des instructions.

Définition de l'accès aux attributs et des instructions

Si les attributs sont activés dans le lot de travail du schéma de workflow, vous pouvez indiquer des commentaires qui s'affichent dans la fenêtre Attributs de l'étape du schéma de workflow de l'utilisateur avec une tâche assignée. Vous pouvez également indiquer la manière dont chaque attribut s'affiche et s'il peut être modifié à cette étape.

Pour plus d'informations sur la définition de l'accès aux attributs, reportez-vous à la rubrique Configuration de l'accès aux attributs. Pour plus d'informations sur la définition des instructions, reportez-vous à la rubrique Configuration des instructions.

Définition des instructions de la section Pièces jointes

Si les pièces jointes sont activées dans le lot de travail du schéma de workflow, vous pouvez indiquer des instructions qui s'affichent dans la section Pièces jointes de la page Lancer le workflow. Pour plus d'informations sur la définition des instructions, reportez-vous à la rubrique Configuration des instructions.
