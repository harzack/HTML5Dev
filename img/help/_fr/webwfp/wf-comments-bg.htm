Si vous activez les commentaires dans Définition des propriétés générales du schéma de workflow, les utilisateurs avec une tâche assignée ont la possibilité d'ajouter des commentaires sur leurs travaux dans l'onglet Commentaires de l'étape. Par défaut, les utilisateurs avec une tâche assignée pour les étapes Utilisateur et Initiateur ne peuvent pas voir les commentaires portant sur les tâches qui ne sont pas les leurs. Vous pouvez toutefois les autoriser à voir tous les commentaires du schéma de workflow. Pour plus d'informations sur les propriétés générales, reportez-vous à la rubrique Définition des propriétés générales.
