Pour gérer des rôles :

  1. Dans le menu Schéma, choisissez Rôles.

  2. Sur la page Paramètres, modifiez les paramètres comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Ajouter un rôle Saisissez un nom dans la zone Rôle, cliquez sur Ajouter un rôle, puis sur Ajouter à la définition du schéma de workflow.
    Enlever un rôle Cliquez sur le rôle, puis sur et cliquez ensuite sur Ajouter à la définition du schéma de workflow.
    Organiser les rôles Cliquez sur le rôle, puis sur les flèches Haut ou Bas pour déplacer le rôle dans la liste. Ensuite, cliquez sur Ajouter à la définition du schéma de workflow.

 

Note :
L'onglet Rôles est disponible uniquement si vous avez choisi Fondé sur schéma dans la liste Mise en oeuvre du rôle sur la page Général du schéma de workflow.

 

Pour plus d'informations sur cette procédure, consultez la section Utilisation des rôles.
