Pour configurer l'étape Utilisateur ou Initiateur :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Utilisateur ou Initiateur.

  2. Dans l'onglet Général, définissez les informations suivantes, puis cliquez sur Ajouter à la définition du schéma de workflow.

    • Dans la zone Nom de l'étape, saisissez un nom pour l'étape.

    • Dans la zone Instructions, spécifiez les instructions qui s'afficheront dans la fenêtre Général.

    • Cliquez sur l'icône Choisir un utilisateur ou un groupe pour sélectionner un utilisateur avec une tâche assignée.

    • Sélectionnez une condition requise du groupe dans la liste Options du groupe. Pour en savoir plus, voir Spécification de la méthode d'assignation des tâches à un groupe.

    • Sélectionnez et saisissez les paramètres de date et de priorité. Pour en savoir plus, voir Dates et priorités.

    • Pour que cette étape recalcule la date d'échéance du schéma de workflow à la fin de son exécution, sélectionnez la case à cocher Recalculer les dates prévues.

    • Sélectionnez la case à cocher Permettre à l'agent de schéma de workflow d'exécuter le processus 'Faire suivre' pour activer le traitement en arrière-plan.

    • Sélectionnez la case à cocher Activer les e-mails relatifs aux actions pour envoyer un e-mail à l'utilisateur avec une tâche assignée pour l'informer de l'assignation du schéma de workflow.

    • Sélectionnez la case à cocher Vue de page unique pour afficher tous les lots de travail sélectionnés sur une même page. Pour afficher un lot pour l'utilisateur avec une tâche assignée, mettez le lot en surbrillance dans la zone Lots, puis cliquez sur la flèche droite. Pour modifier l'ordre des lots de l'utilisateur avec une tâche assignée, mettez un lot en surbrillance dans la zone Afficher les lots, puis cliquez sur la flèche vers le haut ou le bas. Pour en savoir plus, voir Utilisation de la vue une seule page. Désactivez la case à cocher Vue une seule page pour afficher tous les lots d'un lot de travail du schéma dans une vue à deux cadres. Pour en savoir plus, voir Affichage des étapes Utilisateur et Initiateur.

  3. Cliquez sur l'onglet Utilisateurs avec une tâche assignée, effectuez l'une des opérations suivantes, puis cliquez sur Ajouter à la définition du schéma de workflow :

    • Cliquez sur le bouton radio Utilisateur ou groupe, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez l'utilisateur ou le groupe auquel vous souhaitez assigner l'étape.

    • Cliquez sur le bouton radio Utilisateurs/groupes multiples, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez les utilisateurs ou les groupes auxquels vous souhaitez assigner l'étape.

    • Activez la case d'option Utilisateur avec une tâche assignée à partir d'une étape antérieure, puis cliquez sur un nom d'étape dans la liste.

    • Cliquez sur le bouton radio Utilisateur avec une tâche assignée à partir d'un attribut (si elle est désactivée), puis cliquez sur un attribut dans la liste.

    • Cliquez sur le bouton radio Expressions, définissez les conditions à utiliser pour déterminer l'utilisateur de la tâche (cliquez sur l'icône Modifier l'expression  pour créer les conditions à l'aide du générateur d'expressions), cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez les utilisateurs ou les groupes auxquels vous souhaitez assigner l'étape si la condition est vérifiée.

  4. Cliquez sur l'onglet Autorisations, indiquez les autorisations ou les décisions de l'étape, puis cliquez sur Ajouter à la définition du schéma de workflow.

  5. Si les pièces jointes sont activées pour le schéma de workflow, cliquez sur l'onglet Pièces jointes, remplissez la fenêtre d'instructions relatives aux pièces jointes, puis cliquez sur Ajouter à la définition du schéma de workflow.

  6. Cliquez sur l'onglet Commentaires, remplissez la fenêtre d'instructions relatives aux commentaires, puis cliquez sur Ajouter à la définition du schéma de workflow.

  7. Si les attributs sont activés pour le schéma de workflow, cliquez sur l'onglet Attributs, remplissez la fenêtre d'instructions relatives aux attributs, précisez une option d'accès pour chaque attribut, puis cliquez sur Ajouter à la définition du schéma de workflow.

  8. Enregistrez le schéma de workflow.

 

Note :
Pour plus d'informations sur le générateur d'expressions, reportez-vous à la rubrique Utilisation du générateur d'expressions.

 

Pour plus d'informations sur cette procédure, consultez la section Configuration d\'une étape Utilisateur ou Initiateur.
