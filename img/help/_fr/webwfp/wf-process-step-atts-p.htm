Pour modifier les attributs du schéma de workflow :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Processus.

  2. Cliquez sur l'onglet Attributs.

  3. Pour chaque attribut que vous souhaitez modifier, activez la case à cocher située à côté de l'attribut, puis indiquez une valeur dans le champ Valeur de l'attribut.

  4. Cliquez sur Ajouter à la définition du schéma de workflow.

  5. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition d'une étape de processus.
