Lorsque vous configurez une étape Processus, vous devez spécifier les informations suivantes :

  • le nom de l'étape ;

  • des instructions de commentaires ;

  • les opérations de l'étape Processus ;

Définition des instructions de la section Commentaires

Si les commentaires sont activés dans le lot de travail du schéma de workflow, vous pouvez faire des commentaires qui s'affichent dans la section Commentaires de la page de l'étape du schéma de workflow. Pour plus d'informations sur la définition des instructions, reportez-vous à la rubrique Configuration des instructions.

Configuration des opérations de l'étape Processus

Content Server effectue automatiquement les opérations que vous configurez dans les onglets suivants :

  • Général, qui vous permet d'écrire un e-mail (et d'en choisir le destinataire) qui est envoyé lorsque le traitement de schéma de workflow atteint l'étape.

  • Pièces jointes, qui vous permet de copier et de renommer des pièces jointes à un schéma de workflow dans un recueil Content Server. Lorsque vous copiez une pièce jointe, vous devez décider si l'élément copié conserve les catégories et les attributs d'origine qui lui ont été attribués, s'il hérite des catégories et des attributs de l'élément de destination ou si ces deux ensembles de catégories et d'attributs sont fusionnés.

  • Attributs, qui vous permet de choisir les attributs du schéma de workflow que vous souhaitez mettre à jour et d'indiquer de nouvelles valeurs.
