Pour définir une étape Evaluer :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Evaluer.

  2. Indiquez un nom d'étape.

  3. Pour que cette étape recalcule la date d'échéance du schéma de workflow à la fin de son exécution, sélectionnez la case à cocher Recalculer les dates prévues.

  4. Cliquez sur l'icône Modifier l'expression et utilisez le générateur d'expressions pour créer les conditions à évaluer pour chaque étape vers laquelle vous souhaitez envoyer le schéma de workflow.

  5. Dans la liste Etapes, cliquez sur le nom de l'étape vers laquelle envoyer le traitement si l'expression est vraie.

  6. Cliquez sur Ajouter à la définition du schéma de workflow.

  7. Enregistrez le schéma de workflow.

 

Note :
Pour plus d'informations sur le générateur d'expressions, reportez-vous à la rubrique Utilisation du générateur d'expressions.

 

Pour plus d'informations sur cette procédure, consultez la section Configuration d'une étape Evaluer.
