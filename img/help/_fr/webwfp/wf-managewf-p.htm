Pour… Effectuez les opérations suivantes…
Ajouter un schéma de workflow Dans le menu Ajouter un élément, choisissez Schéma de workflow.
Modifier un schéma de workflow Cliquez sur l'icône Fonctions du schéma du workflow, puis choisissez Modifier.
Supprimer un schéma de workflow Cliquez sur l'icône Fonctions du schéma du workflow, puis choisissez Supprimer.
Créer une génération à partir d'un schéma de workflow Cliquez sur l'icône Fonctions du schéma de workflow, puis cliquez sur Créer une génération. Sur la page Ajouter : Génération, saisissez un nom pour la génération et indiquez la version du schéma de workflow à partir de laquelle vous souhaitez créer une génération, puis cliquez sur Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Gestion de schémas de workflow.
