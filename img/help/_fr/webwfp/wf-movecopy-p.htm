Pour créer ou modifier une définition déplacer/copier :

  1. Double-cliquez sur une étape Gestionnaire d'élément, puis cliquez sur l'onglet Déplacer/copier.

  2. Procédez de l'une des manières suivantes :

    • Cliquez sur l'icône Ajouter une définition déplacer/copier pour ajouter ou modifier une définition déplacer/copier.

    • Cliquez sur le lien Modifier situé à côté de la définition déplacer/copier que vous souhaitez modifier.

  3. Dans la section Source, effectuez l'une des opérations suivantes pour spécifier l'élément à déplacer ou à copier :

    • Cliquez sur le bouton radio Pièce jointe de schéma de workflow, puis sur Parcourir les pièces jointes pour naviguer dans le dossier Pièces jointes du schéma de workflow. Cliquez ensuite sur le lien Sélectionner d'une pièce jointe de schéma de workflow ou d'un dossier.

    • Cliquez sur le bouton radio Elément Content Server, puis sur Parcourir Content Server pour naviguer dans Content Server. Cliquez ensuite sur le lien Sélectionner d'un élément Content Server.

    • Cliquez sur le bouton radio Définition du dossier, puis sur Parcourir les définitions du dossier pour naviguer dans les définitions du dossier de l'onglet Définitions de dossier. Ensuite, cliquez sur le lien Sélectionner d'une définition de dossier.

    • Activez la case d'option Attribut de référence d'élément, puis cliquez sur l'attribut de référence d'élément de l'élément dans la liste.

  4. Dans la section Cible, effectuez l'une des opérations suivantes pour spécifier l'emplacement où copier ou déplacer l'élément :

    • Cliquez sur le bouton radio Pièce jointe de schéma de workflow, puis sur Parcourir les pièces jointes pour naviguer dans le dossier Pièces jointes du schéma de workflow. Cliquez ensuite sur le lien Sélectionner d'une pièce jointe de schéma de workflow ou d'un dossier.

    • Cliquez sur le bouton radio Recueil Content Server, puis sur Parcourir Content Server pour naviguer dans Content Server. Cliquez ensuite sur le lien Sélectionner d'un recueil Content Server.

    • Cliquez sur le bouton radio Définition du dossier, puis sur Parcourir les définitions du dossier pour naviguer dans les définitions du dossier de l'onglet Définitions de dossier. Ensuite, cliquez sur le lien Sélectionner d'une définition de dossier.

       

      Note :
      Le bouton radio Définition de dossier est uniquement disponible si vous avez défini plusieurs définitions de dossier pour l'étape. Pour plus d'informations sur la configuration des définitions de dossier, voir Création de dossier.

       

    • Activez la case d'option Attribut de référence d'élément, puis cliquez sur l'attribut de référence d'élément de l'élément dans la liste.

  5. Dans la zone Type de transfert, activez la case d'option située à côté du type d'opération à effectuer.

  6. Dans la zone Autorisations, indiquez les autorisations à appliquer à l'élément déplacé ou copié.

  7. Dans la zone Catégories, indiquez comment seront traitées les catégories de l'élément déplacé ou copié.

  8. Cliquez sur Ajouter ou sur Mettre à jour.

 

Astuce :
Pour sélectionner toutes les pièces jointes, cliquez sur le lien Sélectionner de la liste Pièces jointes.

 

Pour… Effectuez les opérations suivantes…
Déplacer ou copier uniquement le contenu d'un recueil Activez la case à cocher Contenus SEULEMENT.
Renommer l'élément déplacé ou copié Activez la case à cocher Renommer la source, puis saisissez un nouveau nom pour l'élément déplacé ou copié.
Synchroniser le contenu du recueil cible avec celui du recueil source Activez la case à cocher Synchroniser le contenu du recueil.
Créer des versions principales de documents dans le recueil cible Activez la case à cocher Créer une version principale. Vous ne pouvez activer cette case à cocher que si vous avez également activé les cases à cocher Synchroniser le contenu du recueil et Contenus SEULEMENT.

Pour plus d'informations sur cette procédure, consultez la section Copie ou déplacement d'éléments.
