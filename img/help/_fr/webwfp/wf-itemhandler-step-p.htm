Pour configurer une étape Gestionnaire d'élément :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Gestionnaire d'élément.

  2. Dans l'onglet Général, indiquez un nom d'étape, des instructions qui seront affichées dans la fenêtre Général, des paramètres de date et de priorité, un utilisateur avec une tâche assignée et précisez si vous souhaitez travailler avec l'agent de workflow, puis cliquez sur le bouton Ajouter à la définition du schéma de workflow.

  3. Facultatif Si les pièces jointes sont activées pour le schéma de workflow, cliquez sur l'onglet Pièces jointes, remplissez la fenêtre d'instructions relatives aux pièces jointes, puis cliquez sur Ajouter à la définition du schéma de workflow.

  4. Cliquez sur l'onglet Commentaires, remplissez la fenêtre d'instructions relatives aux commentaires, puis cliquez sur Ajouter à la définition du schéma de workflow.

  5. Facultatif Si les attributs sont activés pour le schéma de workflow, cliquez sur l'onglet Attributs, remplissez la fenêtre d'instructions relatives aux attributs, précisez une option d'accès pour chaque attribut, puis cliquez sur Ajouter à la définition du schéma de workflow.

  6. Dans les onglets Définitions de dossier, Catégories, Création de version et Déplacer/copier, spécifiez les opérations que vous souhaitez effectuer.

  7. Dans l'onglet Records Management, cliquez sur l'icône Ajouter une définition Records Management, sélectionnez la case à cocher Finaliser l'enregistrement pour finaliser automatiquement les éléments Records Management dans le cadre de l'étape, sélectionnez ou supprimez un classement RM associé, puis spécifiez les éventuelles opérations standard que vous souhaitez exécuter.

     

    Note :
    Un message d'erreur s'affiche si vous tentez d'enregistrer une définition Records Management sans ajouter d'élément à l'étape et si vous mettez à jour un classement RM ou activez l'option Finaliser l'enregistrement. Vous devez impérativement ajouter un élément et choisir une option de mise à jour.

     

  8. Enregistrez le schéma de workflow.

Pour plus d'informations sur les opérations Gestionnaire d'élément, reportez-vous à la rubrique Configuration des opérations de l'étape Gestionnaire d'élément.

Pour plus d'informations sur cette procédure, consultez la section Configuration d\'une étape Gestionnaire d\'élément.
