Pour définir des propriétés générales :

  1. Dans le menu Schéma, choisissez Général.

  2. Dans la page Paramètres, modifiez les paramètres souhaités, comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Renommer le schéma de workflow Saisissez un nom dans le champ Titre.
    Décrire le schéma de workflow Saisissez une description dans le champ Description.
    Activer le schéma de workflow pour l'utilisation des rôles Dans la liste Validation du rôle, cliquez sur Fondé sur schéma.
    Choisir les paramètres d'exécution du schéma de workflow Dans la liste Actions suivant l'exécution, cliquez sur l'une des options suivantes :
    •

    Aucune permet au responsable du schéma de workflow de choisir la façon dont il souhaite achever le schéma de workflow.

    •

    Archiver après exécution attribue automatiquement aux schémas de workflow achevés le statut Archivés.

    •

    Supprimer après exécution supprime automatiquement les schémas de workflow achevés, ainsi que toutes les informations qui leur sont associées.

    Indiquer comment traiter les week-ends dans le calcul de l'échéance Activez la case à cocher Ignorer les week-ends pour qu'ils ne soient pas pris en compte dans le calcul de l'échéance pour que Livelink compte uniquement les jours de semaine dans le calcul des échéances ou désactivez la case pour inclure les week-ends dans le calcul.
    Choisir un message personnalisé pour le lancement Activez la case d'option Message personnalisé et saisissez votre message dans le champ.
    Activer les éléments dans le lot de travail
    Activez la case à cocher située à côté des éléments que vous souhaitez rendre disponibles dans le lot de travail.
    Si le module Physical Objects est installé, un champ Eléments physiques est également disponible. Si vous avez sélectionné la case à cocher Eléments physiques, l'onglet Eléments physiques apparaît pour une étape de workflow, en tant qu'élément de travail, lors du lancement et de l'exécution du workflow.
    Lorsque vous sélectionnez l'onglet Eléments physiques en mode conception, vous pouvez saisir et configurer les métadonnées spécifiques aux éléments physiques. Vous pouvez ainsi entrer les instructions utilisateur qui s'afficheront lors de l'exécution d'une étape de workflow. Ou encore choisir le type d'élément physique défini pour le système. Lorsque vous sélectionnez un type d'élément physique, la page s'actualise avec les attributs et les métadonnées d'objets physiques correspondants, s'ils sont disponibles.
    Lors de l'exécution d'une étape de workflow, le lien Eléments physiques est disponible et vous permet de modifier les instructions, les champs de métadonnées configurés durant la conception du workflow et les champs de métadonnées obligatoires. Les utilisateurs ne peuvent modifier le type d'élément physique à créer, et le champ de métadonnées Attribut est obligatoire.
     

    Note :
    Si vous cliquez sur Créer pour finaliser l'étape de workflow, tous les éléments physiques spécifiés seront créés de la même manière que s'ils avaient été créés via la page Ajouter : Elément physique. Si vous ne spécifiez aucune valeur dans le champ Nombre d'exemplaires, Nombre de libellés ou Nombre d'éléments à créer, la valeur 1 sera automatiquement attribuée.

     

Pour plus d'informations sur cette procédure, consultez la section Définition des propriétés générales.
