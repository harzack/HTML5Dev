Pour ajouter un fichier en pièce jointe :

  1. Sélectionnez Pièces jointes dans le menu Schéma.

  2. Cliquez sur l'icône Ajouter un document .

  3. Sur la page Ajouter : Document, cliquez sur Parcourir, puis naviguez jusqu'à un fichier enregistré sur votre ordinateur ou sur le système de fichiers réseau et sélectionnez-le.

  4. Indiquez les autres paramètres généraux pour l'élément, puis cliquez sur Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Définition des pièces jointes.
