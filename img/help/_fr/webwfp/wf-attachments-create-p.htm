Pour créer une pièce jointe d'élément :

  1. Sélectionnez Pièces jointes dans le menu Schéma.

  2. Sur la page Pièces jointes, choisissez un élément dans le menu Ajouter un élément.

  3. Sur la page Ajouter du type d'élément, indiquez les paramètres de l'élément, puis cliquez sur le bouton Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Définition des pièces jointes.
