Pour modifier une demande de copie de pièce jointe :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Processus.

  2. Cliquez sur l'onglet Pièces jointes.

  3. Dans la colonne Actions, cliquez sur le lien Modifier.

  4. Dans la zone Copier vers, cliquez sur Parcourir Content Server, naviguez jusqu'à l'emplacement vers lequel vous voulez copier l'élément, puis cliquez sur son lien Sélectionner.

  5. Saisissez le nom sous lequel vous souhaitez que Content Server enregistre la copie de la pièce jointe dans la zone Nom de la copie.

  6. Choisissez la méthode à utiliser pour traiter les catégories et les attributs pour l'élément copié.

  7. Cliquez sur Envoyer.

  8. Cliquez sur Ajouter à la définition du schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition d'une étape de processus.
