Pour configurer les opérations sur les versions pour les pièces jointes au schéma de workflow :

  1. Double-cliquez sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Création de version, puis modifiez les paramètres souhaités, comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes...
    Créer une version d'un document ou d'un document composite ajouté comme pièce jointe à un schéma de workflow ajoutée à partir de Content Server Dans la liste Ajouter une version à la version Content Server d'origine, cliquez sur la méthode de création de version.
    Déverrouiller les documents ou les documents composites d'origine Activez la case à cocher Annuler la réservation des originaux.
    Créer une version principale de tous les documents ou des documents modifiés ajoutés en tant que pièces jointes au schéma de workflow Activez la case à cocher Créer une version principale.
    Créer une génération à partir de documents Dans la liste Documents, cliquez sur Créer une génération.
    Créer une version, une révision ou une génération à partir de documents composites Dans la liste Documents composites, cliquez sur le type de création de version à effectuer par l'étape.

Pour plus d'informations sur cette procédure, consultez la section Opérations de création de versions.
