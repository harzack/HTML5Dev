Pour configurer des instructions pour... Double-cliquez sur l'étape, puis effectuez les opérations suivantes...
Etape Cliquez sur l'onglet Général, puis saisissez les instructions dans la zone Instructions.
Pièces jointes Cliquez sur l'onglet Pièces jointes, puis saisissez les instructions dans la zone Instructions.
Attributs Cliquez sur l'onglet Attributs, puis saisissez les instructions dans la zone Instructions.
Commentaires Cliquez sur l'onglet Commentaires, puis saisissez les instructions dans la zone Instructions.

 

Note :
Vous pouvez configurer des instructions pour les étapes de départ, Utilisateur, Initiateur, Gestionnaire d'élément et Processus.

 

Pour plus d'informations sur cette procédure, consultez la section Configuration des instructions.
