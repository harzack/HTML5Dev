Pour indiquer l'importance d'une étape Utilisateur, Initiateur ou Gestionnaire d'élément, vous pouvez lui attribuer une valeur de priorité. Vous pouvez également utiliser une priorité pour acheminer automatiquement le workflow ou déterminer l'utilisateur avec une tâche assignée d'une étape Utilisateur. Les priorités que vous pouvez sélectionner sont :

  • Elevée

  • Moyenne

  • Faible

Les responsables du workflow et les intervenants peuvent afficher les tâches d'étape en fonction de leur priorité. Ils peuvent, par exemple, configurer leur onglet Tâches assignées pour afficher uniquement les étapes à priorité élevée. Ils peuvent également trier les étapes par ordre croissant ou décroissant de priorité.
