Pour gérer des versions de schéma de workflow :

  1. Cliquez sur le menu Fonctions du schéma du workflow, choisissez Propriétés, puis cliquez sur Versions.

  2. Modifiez les paramètres souhaités, comme indiqué dans le tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Modifier une version de schéma de workflow Cliquez sur le menu Fonctions de la version que vous souhaitez modifier et choisissez Modifier.
    Exporter une version de schéma de workflow Cliquez sur le menu Fonctions de la version que vous souhaitez exporter et choisissez Exporter.
    Lancer une version de schéma de workflow Cliquez sur le menu Fonctions de la version que vous souhaitez lancer et choisissez Lancer.
    Verrouiller ou déverrouiller une version de schéma de workflow Cliquez sur le menu Fonctions de la version que vous souhaitez verrouiller ou déverrouiller, puis choisissez Verrouiller ou Déverrouiller.
    Supprimer une version de schéma de workflow Cliquez sur le menu Fonctions de la version que vous souhaitez supprimer et choisissez Supprimer.
    Eliminer des versions de schéma de workflow Cliquez sur le menu Fonctions du schéma du workflow, choisissez Propriétés, puis cliquez sur Versions. Dans le champ Eliminer toutes les versions à l'exception de la plus récente, saisissez le numéro de la version la plus récente que vous souhaitez conserver, puis cliquez sur Eliminer les versions pour éliminer les autres versions.

Pour plus d'informations sur cette procédure, consultez la section Gestion de schémas de workflow.
