Pour configurer des opérations sur les versions pour les cibles spécifiques :

  1. Double-cliquez sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Création de version, puis sur le lien Cibles spécifiques.

  3. Dans la section Cibles spécifiques, effectuez une des actions suivantes :

    • Cliquez sur l'icône Ajouter une définition de la création de versions pour créer une demande de création de version.

    • Cliquez sur le lien Modifier pour modifier une demande de création de version.

  4. Dans la zone Source, indiquez le document ou le document composite sur lequel l'opération de création de version sera effectuée.

  5. Dans la zone Cible, indiquez le document ou le document composite sur lequel l'opération de création de version sera effectuée.

  6. Cliquez sur Ajouter ou sur Mettre à jour.

Pour… Effectuez les opérations suivantes…
Annuler la réservation d'éléments cibles Activez la case à cocher Annuler la réservation.
Ajouter un document source en tant que version principale d'un document cible Activez la case à cocher Créer une version principale.
Créer une génération à partir d'un document composite cible Sélectionnez la case à cocher Créer une génération, puis saisissez le nom de la génération dans la zone Remplacer le nom de génération par défaut.
Créer une version ou une révision d'un document composite cible Cliquez sur Créer une révision ou Créer une version dans la liste, puis saisissez le nom de la version ou de la révision dans la zone Remplacer le nom de la version/révision par défaut.

Pour plus d'informations sur cette procédure, consultez la section Opérations de création de versions.
