Si vous souhaitez créer une copie de sauvegarde d'une définition de schéma de workflow ou la déplacer d'un serveur Content Server à un autre, vous pouvez l'exporter vers un fichier sur votre ordinateur. Une fois le schéma de workflow exporté, vous pouvez importer un schéma de workflow lorsque vous souhaitez créer un schéma de workflow à partir du schéma de workflow exporté.

Importation d'un schéma de workflow

Si vous souhaitez créer un schéma de workflow en important une définition de schéma de workflow, vous devez d'abord ajouter un schéma de workflow vide ou utiliser un schéma de workflow existant que vous ne souhaitez pas enregistrer. L'importation remplace les données du schéma de workflow existant par la définition du fichier importé. Si vous importez un schéma de workflow à partir de l'ancien éditeur de schéma de workflow, il peut être nécessaire de repositionner les étapes et les liens, car les nouvelles étapes sont plus grandes que les anciennes et risquent de se chevaucher. Pour plus d'informations sur l'ajout d'un schéma de workflow, reportez-vous à la rubrique Gestion de schémas de workflow. Pour plus d'informations sur le repositionnement des étapes, voir Définition de liens entre les étapes.

Si le schéma de workflow importé comporte des pièces jointes, vous devez les ajouter manuellement. Pour plus d'informations sur l'ajout de pièces jointes, reportez-vous à la rubrique Définition des pièces jointes.

Vous devez également décider si vous souhaitez assigner les mêmes tâches aux mêmes utilisateurs dans le schéma de workflow importé ou si vous souhaitez réinitialiser toutes les tâches assignées. Si vous réinitialisez les tâches assignées, vous devez modifier le schéma de workflow pour ajouter des intervenants au schéma de workflow.

Exportation d'un schéma de workflow

Lorsque vous exportez un schéma de workflow, la définition de ce schéma de workflow est enregistrée sous la forme d'un fichier (avec l'extension .map) sur votre ordinateur. Les pièces jointes ne sont pas exportées avec un schéma de workflow. Si le schéma de workflow comporte des pièces jointes que vous souhaitez enregistrer en même temps que celui-ci, enregistrez-les séparément.
