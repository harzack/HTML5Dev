Pour spécifier un élément et une opération :

  1. Double-cliquez sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Catégories.

  3. Cliquez sur le lien Opérations.

  4. Dans la section Opérations, effectuez l'une des actions suivantes :

    • Cliquez sur l'icône Ajouter une opération pour spécifier un nouvel élément et une nouvelle opération.

    • Cliquez sur le lien Modifier situé à côté d'une définition d'élément et d'opération pour la modifier.

  5. Dans la section Elément, effectuez l'une des actions suivantes pour préciser l'élément sur lequel l'opération doit porter :

    • Cliquez sur le bouton radio Pièce jointe de schéma de workflow, puis sur Parcourir les pièces jointes pour naviguer dans le dossier Pièces jointes du schéma de workflow. Cliquez ensuite sur le lien Sélectionner d'une pièce jointe de schéma de workflow.

    • Cliquez sur le bouton radio Elément Content Server, puis sur Parcourir Content Server. Cliquez ensuite sur le lien Sélectionner d'un élément. Si la fenêtre de navigation de la cible affiche des catégories, cliquez sur Content Server dans la liste de navigation pour aller dans Content Server.

    • Cliquez sur le bouton radio Définition du dossier, puis sur Parcourir les définitions du dossier pour naviguer dans les définitions que vous avez configurées dans l'onglet Définitions de dossier. Ensuite, cliquez sur le lien Sélectionner d'une définition de dossier.

    • Activez la case d'option Attribut de référence d'élément, puis cliquez sur l'attribut de référence de l'élément Content Server dans la liste.

  6. Dans la zone Options de mise à jour, activez l'une des cases d'option suivantes pour préciser les opérations de données à accomplir :

    • Valeurs d'attributs ou de champs, pour mettre à jour les attributs du schéma de workflow ou les champs du formulaire mappés.

    • Elément, pour mettre à jour l'élément, puis indiquez quelles actions l'étape doit effectuer.

  7. Cliquez sur Ajouter ou sur Mettre à jour.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des catégories et des attributs dans les schémas de workflow.
