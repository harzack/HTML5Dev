Pour ajouter des attributs :

  1. Dans le menu Schéma, choisissez Attributs.

  2. Dans l'onglet Attributs du schéma de workflow, cliquez sur un type d'attribut dans la liste Ajouter un attribut.

  3. Précisez les propriétés de l'attribut.

  4. Cliquez sur OK.

  5. Facultatif Activez la case à cocher Désactiver la création de versions pour empêcher Content Server de stocker les versions des valeurs d'attribut.

  6. Cliquez sur Ajouter à la définition du schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition des attributs.
