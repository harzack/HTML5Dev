Pour permettre aux utilisateurs de visualiser les commentaires :

  1. Double-cliquez sur une étape Utilisateur ou Initiateur.

  2. Cliquez sur l'onglet Autorisations.

  3. Activez la case à cocher Voir tous les commentaires.

  4. Cliquez sur Ajouter à la définition du schéma de workflow.

 

Note :
Si la case à cocher Vue de page unique est sélectionnée dans l'onglet Général, vérifiez que le lot de travail Commentaires apparaît dans la zone Afficher les lots de travail.

 

Pour plus d'informations sur cette procédure, consultez la section Autoriser les utilisateurs à visualiser les commentaires.
