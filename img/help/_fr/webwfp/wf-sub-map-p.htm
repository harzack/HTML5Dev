Pour configurer une étape Sous-workflow :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Sous-Workflow.

  2. Dans l'onglet Général, saisissez un nom de l'étape dans la zone Nom de l'étape.

  3. Cliquez sur Parcourir Content Server, recherchez le schéma de workflow que vous souhaitez intégrer dans le schéma de workflow principal, puis cliquez sur son lien Sélectionner.

  4. Dans la zone Lancer sous, saisissez le nom appliqué à chaque instance de schéma de workflow basée sur le sous-workflow.

  5. Dans le champ Durée, cliquez sur le bouton radio Jours ou Heures, saisissez le nombre d'heures ou de jours nécessaires pour exécuter le schéma de workflow intégré, puis sélectionnez la case à cocher Recalculer les dates prévues si vous souhaitez que Content Server recalcule l'échéance du schéma de workflow principal à la fin de l'exécution du schéma de workflow intégré.

  6. Cliquez sur Ajouter à la définition du schéma de workflow.

  7. Dans chaque onglet du lot de travail et du type de données configurés pour votre schéma, indiquez les mappages et la disponibilité des données, puis cliquez sur Ajouter à la définition du schéma de workflow.

  8. Enregistrez le schéma de workflow.

Pour… Effectuez les opérations suivantes…
Echanger des pièces jointes entre les schémas de workflow Cliquez sur l'onglet Pièces jointes et effectuez l'une des opérations suivantes :
•

Activez la case d'option Aucun(e) pour interdire tout échange de pièces jointes.

•

Activez la case d'option Tous pour échanger toutes les pièces jointes.

•

Activez la case d'option Eléments sélectionnés, puis activez la case à cocher des pièces jointes que vous souhaitez échanger. Saisissez ensuite le nom du raccourci que vous souhaitez utiliser dans la zone Raccourci.

Créer un mappage automatique entre les attributs du même type qui ont le même nom Cliquez sur l'onglet Attributs, puis sur Lien automatisé.
Echanger des attributs entre les schémas de workflow Cliquez sur l'onglet Attributs, puis cliquez sur l'attribut Sous-workflow sur lequel l'attribut est mappé dans la liste située à côté de chaque attribut du schéma de workflow principal.
Partager des commentaires entre les schémas de workflow Cliquez sur l'onglet Commentaires, puis activez la case à cocher Commentaires transmis du workflow principal au sous-workflow, activez la case Commentaires transmis du sous-workflow au workflow principal ou les deux.
Mapper des rôles entre les schémas de workflow Cliquez sur l'onglet Rôles, puis cliquez sur le rôle du schéma de workflow principal sur lequel le rôle est mappé dans la liste située à côté de chaque rôle du sous-workflow.

Pour plus d'informations sur cette procédure, consultez la section Configuration d'une étape Sous-workflow.
