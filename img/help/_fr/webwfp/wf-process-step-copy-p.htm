Pour copier une pièce jointe :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Processus.

  2. Cliquez sur l'onglet Pièces jointes.

  3. Cliquez sur l'icône Ajouter une pièce jointe pour créer une demande de copie.

  4. Dans la zone Fichier, cliquez sur Parcourir les pièces jointes, puis sur le lien Sélectionner de la pièce jointe de schéma de workflow à ajouter à cette étape.

  5. Dans le champ Copier vers, cliquez sur Parcourir Content Server, naviguez jusqu'à l'emplacement vers lequel vous voulez copier l'élément, puis cliquez sur son lien Sélectionner.

  6. Saisissez le nom sous lequel vous souhaitez que Content Server enregistre la copie de la pièce jointe dans la zone Nom de la copie.

  7. Choisissez la méthode à utiliser pour traiter les catégories et les attributs pour l'élément copié.

  8. Cliquez sur Envoyer.

  9. Cliquez sur Ajouter à la définition du schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition d'une étape de processus.
