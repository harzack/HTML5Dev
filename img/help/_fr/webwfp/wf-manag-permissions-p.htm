Pour configurer les autorisations de gestion :

  1. Dans le menu Schéma, choisissez Gestion, puis cliquez sur l'icône Modifier les autorisations de gestion .

  2. Sur la page Gestion des workflows, modifiez les paramètres comme indiqué dans le tableau suivant.

    Pour... Effectuez les opérations suivantes...
    Nommer le responsable principal Cliquez sur l'icône Définir le gestionnaire principal , puis recherchez et cliquez sur le lien Sélectionner de l'utilisateur.
    Rétablir l'initiateur en tant que responsable principal Cliquez sur l'icône Définir le gestionnaire principal , puis sur le lien Sélectionner de l'initiateur.
    Ajouter un responsable Cliquez sur l'icône Ajouter un gestionnaire , recherchez l'utilisateur que vous souhaitez ajouter en tant que gestionnaire, sélectionnez la case à cocher Accorder l'accès de l'utilisateur, puis cliquez sur Ajouter à la définition du schéma de workflow.
    Définir des autorisations pour un responsable Cliquez sur le nom du responsable du schéma de workflow dont vous souhaitez définir les autorisations, sélectionnez les cases à cocher des autorisations que vous souhaitez lui attribuer, puis cliquez sur Mettre à jour.
    Enlever un responsable Cliquez sur le nom du responsable du schéma de workflow que vous souhaitez enlever, puis cliquez sur Enlever.

 

Note :
Les autorisations de gestion apparaissent imbriquées sur la page Modifier des autorisations de gestion, et affichent leurs dépendances. Vous ne pouvez pas assigner une autorisation imbriquée sans assigner l'autorisation dont elle dépend.

 

Pour plus d'informations sur cette procédure, consultez la section Configuration des autorisations de gestion.
