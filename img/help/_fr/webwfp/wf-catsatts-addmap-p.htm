Pour définir un mappage d'attribut :

  1. Double-cliquez sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Catégories, puis sur le lien Définitions.

  3. Dans la section Définitions, cliquez sur un des liens suivants :

    • Ajouter un schéma, pour définir un schéma.

    • Modifier le schéma, pour modifier un schéma.

  4. Dans la fenêtre Sélectionner un type de données de schéma, cliquez sur le lien Sélectionner des attributs du schéma de workflow ou du formulaire qui contient le champ pour lequel vous souhaitez créer un schéma.

  5. Dans la fenêtre Sélectionner un champ de schéma, cliquez sur le lien Sélectionner de l'attribut du schéma de workflow ou du formulaire auquel vous souhaitez faire correspondre l'attribut de catégorie.

Pour plus d'informations sur cette procédure, consultez la section Utilisation des catégories et des attributs dans les schémas de workflow.
