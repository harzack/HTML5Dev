Pour définir une étape Point de repère :

  1. Dans l'éditeur de schéma de workflow, double-cliquez sur une étape Clé.

  2. Indiquez un nom d'étape.

  3. Procédez de l'une des manières suivantes :

    • Cliquez sur le bouton radio Calculer la date prévue.

    • Activez la case d'option Saisir la date prévue, puis indiquez le mois, le jour, l'année et l'heure de la date prévue.

  4. Cliquez sur Ajouter à la définition du schéma de workflow.

  5. Enregistrez le schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Configuration d'une étape Point de repère.
