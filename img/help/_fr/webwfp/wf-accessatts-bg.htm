Si vous avez activé des attributs dans le schéma de workflow, vous pouvez préciser le mode d'affichage de ces attributs pour les pages Attribut des étapes Départ, Utilisateur Initiateur ou Gestionnaire d'élément. Vous pouvez assigner l'un des paramètres suivants pour chaque attribut :

  • Non disponible masque l'attribut.

  • Modifiable permet à l'utilisateur avec une tâche assignée de modifier la valeur de l'attribut.

  • Saisie obligatoire signifie que l'utilisateur avec une tâche assignée doit modifier l'attribut.

  • Lecture seule permet à l'utilisateur avec une tâche assignée d'afficher l'attribut et sa valeur.

  • Ignorer l'action figure dans l'onglet Eléments physiques et détermine si les métadonnées d'élément physique utilisées pour créer un élément physique, lors de l'étape de création d'un workflow, sont transmises aux étapes suivantes, sans déclencher une autre action de création. Si vous sélectionnez l'option Ignorer, l'étape d'action n'est pas disponible dans une étape de workflow et les champs Obligatoire relatifs à l'étape sont appliqués. Si la valeur Action est définie sur Créer, soit comme valeur par défaut en mode conception ou héritée d'une autre étape, les éléments physiques ne sont pas créés.

    Si la valeur Action est définie sur Modifiable ou Saisie obligatoire, les champs obligatoires d'une exécution d'étape sont appliqués si la valeur est définie sur Rejeter. Si la valeur Action est définie sur Non disponible ou Lecture seule, les champs obligatoires d'une exécution d'étape sont appliqués si la valeur est définie sur Rejeter.
