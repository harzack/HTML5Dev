Vous avez la possibilité d'ajouter des instructions aux étapes de départ, Utilisateur, Initiateur, Gestionnaire d'élément et Processus, afin d'expliquer les actions que vous souhaitez voir réaliser par les utilisateurs assignés à l'étape, dans les pages Vue d'ensemble, Commentaires, Attributs et Pièces jointes. Par exemple, si les utilisateurs doivent joindre un document, vous pouvez leur fournir des instructions qui s'affichent dans une note en bas de la page principale et de la page Pièces jointes de l'étape. Les commentaires, attributs et pièces jointes doivent être activés dans le schéma de workflow pour que vous puissiez y ajouter des instructions.

 

Note :
Les instructions que vous indiquez peuvent comporter du code HTML basique. Si vous utilisez des caractères spéciaux, vous devrez peut-être utiliser leurs codes HTML pour qu'ils ne soient pas interprétés comme du code HTML. Par exemple, si vous souhaitez utiliser les symboles inférieur à ou supérieur à (< ou >), vous devez saisir les codes HTML de chaque symbole (&lt; ou &gt;).
