Pour modifier des attributs :

  1. Dans le menu Schéma, choisissez Attributs.

  2. Dans l'onglet Attributs du schéma de workflow, cliquez sur le lien Eléments d'attribut.

  3. Sur la page Modifier les attributs, modifiez les propriétés de l'attribut.

  4. Cliquez sur OK.

  5. Facultatif Activez la case à cocher Désactiver la création de versions pour empêcher Content Server de procéder au suivi des versions de valeurs d'attribut.

  6. Cliquez sur Ajouter à la définition du schéma de workflow.

Pour plus d'informations sur cette procédure, consultez la section Définition des attributs.
