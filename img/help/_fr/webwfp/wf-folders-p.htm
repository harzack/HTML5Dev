Pour ajouter ou modifier une définition de dossier :

  1. Double-cliquez sur une étape Gestionnaire d'élément.

  2. Cliquez sur l'onglet Définitions de dossier.

  3. Procédez de l'une des manières suivantes :

    • Cliquez sur l'icône Ajouter une définition de dossier pour créer une définition de dossier.

    • Cliquez sur le nom d'une définition pour modifier la définition de dossier.

    • Cliquez sur le lien Ajouter un sous-dossier pour créer un dossier à l'intérieur d'un dossier.

  4. Saisissez le nom du dossier dans la zone Nom.

  5. Dans la zone Option de conflit de nom, cliquez sur le bouton radio de l'action à effectuer si un dossier du même nom existe déjà.

  6. Dans la section Cible, effectuez l'une des actions suivantes pour indiquer l'emplacement où créer un dossier :

    • Cliquez sur le bouton radio Pièces jointes du schéma de workflow, puis sur Parcourir les pièces jointes pour naviguer dans le volume des pièces jointes du schéma de workflow. Cliquez ensuite sur le lien Sélectionner du dossier où le dossier doit être créé.

    • Cliquez sur le bouton radio Recueil Content Server, puis sur ParcourirContent Server pour naviguer dans Content Server. Cliquez ensuite sur le lien Sélectionner du dossier où le dossier doit être créé.

    • Cliquez sur le bouton radio Définition du dossier, puis sur Parcourir les définitions du dossier pour naviguer dans les définitions que vous avez configurées dans l'onglet Définitions de dossier. Ensuite, cliquez sur le lien Sélectionner d'une définition de dossier.

    • Cliquez sur le bouton radio Attribut de référence d'élément, puis, dans la liste, cliquez sur l'attribut de référence du recueil Content Server où le dossier doit être créé.

  7. Cliquez sur Ajouter ou sur Mettre à jour.

 

Astuce :
Vous pouvez créer plusieurs définitions de dossier dans une même étape Gestionnaire d'élément.

 

Pour plus d'informations sur cette procédure, consultez la section Création de dossier.
