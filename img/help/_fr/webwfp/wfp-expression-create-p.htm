Pour définir une expression :

  1. Double-cliquez sur une étape Evaluation, Utilisateur ou Tâche de formulaire.

  2. Si vous avez double-cliqué sur une étape Utilisateur ou Tâche de formulaire, cliquez sur l'onglet Utilisateurs avec une tâche assignée.

  3. Cliquez sur l'icône Générateur d'expressions .

  4. Dans la liste Ajouter une nouvelle ligne, cliquez sur un critère de recherche de schéma de workflow ou sur les parenthèses (si vous souhaitez créer une expression complexe).

  5. Spécifiez l'expression à l'aide du Générateur d'expressions.

  6. Facultatif Répétez les étapes 2 et 3 pour préciser d'autres critères.

  7. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Utilisation du générateur d'expressions.
