Pour définir une étape Clé, vous devez préciser le nom de l'étape et la date prévue.

Spécification d'une date prévue

Vous pouvez demander à Content Server de calculer la date prévue de l'étape Point de repère ou saisir vous-même une date. Lorsque Content Server calcule la date prévue, il se base sur la durée totale des étapes situées avant le point de repère ou sur l'échéance de l'étape qui précède directement le point de repère. Si vous saisissez la date manuellement, vous devez préciser le jour, le mois, l'année et l'heure de la date prévue.
