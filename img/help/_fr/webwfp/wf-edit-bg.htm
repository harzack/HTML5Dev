Pour modifier un schéma de workflow, vous pouvez utiliser les outils suivants :

  • L'éditeur de schéma de workflow, qui vous permet de créer une représentation graphique du processus de traitement.

  • Le menu Schéma, qui vous permet de définir les propriétés générales du schéma de workflow, de préciser les lots de travail et d'effectuer les opérations du schéma de workflow.

Utilisation de l'éditeur de schéma de workflow

L'éditeur de schéma de workflow est un outil de création que vous pouvez utiliser pour créer et gérer des schémas de workflow. L'éditeur de schéma de workflow est une grille en pointillé qui s'affiche lorsque vous modifiez un schéma de workflow. Vous pouvez ajouter, dupliquer ou supprimer des étapes et assigner des exécutants pour les étapes et créer des liens entre les étapes. Pour plus d'informations sur l'utilisation des étapes et des liens, reportez-vous à la rubrique Configuration des étapes et des liens.

Par défaut, l'éditeur de schéma de workflow contient les éléments suivants :

  • l'icône de l'étape Départ, dans la grille en pointillé ;

  • la palette Etape, dans le volet de gauche ;

  • la vue Zoom, dans le volet de gauche ;

  • les icônes d'impression et de zoom de l'éditeur de schéma de workflow.

Les barres de défilement de l'éditeur de schéma de workflow vous permettent d'agrandir le schéma de workflow au-delà de la zone actuellement visible.

Utilisation de la palette Etape

La palette Etape permet d'afficher toutes les étapes disponibles pour les modules que vous avez installés. Si vous installez le module facultatif Forms, une étape Tâche de formulaire apparaît. Lorsque vous placez le curseur de la souris sur une étape, une vue miniature agrandie de cette étape s'affiche.

Utilisation de la vue Zoom

La vue Zoom montre un aperçu général du schéma de workflow et vous permet de naviguer dans une portion du schéma de workflow invisible depuis l'éditeur de schéma de workflow.

Utilisation des icônes de l'éditeur de schéma de workflow

Les icônes de l'éditeur de schéma de workflow sont situées au-dessus de la palette Etape, dans le cadre de gauche.

Tableau 5-1 : Icônes de l'éditeur de schéma de workflow

Icône Description
L'icône Imprimer vous permet d'imprimer le schéma de workflow.
L'icône 1:1 vous permet de visualiser rapidement le schéma de workflow à 100 % de zoom, sur la base de la taille de votre navigateur.
L'icône Zoom avant vous permet d'effectuer un zoom avant sur une zone donnée, dans l'éditeur de schéma de workflow. Cette action a le même effet que lorsque vous faites glisser vers la gauche le coin inférieur droit du rectangle bleu, dans la vue Zoom.
L'icône Zoom arrière vous permet d'effectuer un zoom arrière sur une zone donnée, dans l'éditeur de schéma de workflow. Cette action a le même effet que lorsque vous faites glisser vers la droite le coin inférieur droit du rectangle bleu, dans la vue Zoom.

Utilisation du menu Schéma

Lorsque vous affichez l'éditeur de schéma de workflow, le menu Schéma apparaît en regard des menus généraux en haut de votre page Content Server. Elle contient les options suivantes :

  • Editeur vous permet de retourner à l'éditeur de schéma de workflow depuis une autre page.

  • Général, Attributs, Pièces jointes, Gestion et Rôles vous permettent de configurer le lot de travail et les paramètres du schéma de workflow. Rôles apparaît uniquement si vous avez choisi une validation du rôle Fondé sur schéma dans les propriétés générales du schéma de workflow. Pour plus d'informations sur les paramètres du schéma de workflow, reportez-vous à la rubrique Configuration de schémas de workflow.

  • Modifier les exécutants du schéma vous permet d'assigner des étapes à des groupes ou des utilisateurs particuliers. Pour plus d'informations sur la configuration d'exécutants, reportez-vous à la rubrique Définition des exécutants du schéma de workflow.

  • Vérifier la définition du schéma vous permet de vous assurer que le schéma de workflow ne contient aucune erreur. Pour plus d'informations sur la vérification d'un schéma de workflow, reportez-vous à la rubrique Vérification des schémas de workflow.

  • Lancer vous permet de lancer le schéma de workflow. Pour plus d'informations sur le lancement d'un schéma de workflow basé sur votre schéma de workflow, voir Lancement des schémas de workflow.

  • Enregistrer et Enregistrer sous vous permettent d'enregistrer le schéma de workflow. Pour plus d'informations sur l'utilisation des schémas de workflow, voir Gestion de schémas de workflow.
