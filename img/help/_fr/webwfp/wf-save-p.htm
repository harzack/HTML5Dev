Pour… Dans le Workflow Map, procédez comme suit...
Enregistrer le schéma de workflow Dans le menu Schéma, choisissez Enregistrer.
Enregistrez le schéma comme un nouvel élément de schéma de workflow Dans le menu Schéma, choisissez Enregistrer sous. Saisissez le nom du schéma de workflow dans la zone Nom du nouveau schéma, cliquez sur Parcourir Content Server et accédez à l'emplacement où vous souhaitez enregistrer le nouveau schéma de workflow, puis cliquez sur Enregistrer le schéma.

Pour plus d'informations sur cette procédure, consultez la section Enregistrement de schémas de workflow.
