Pour copier un élément Content Server comme pièce jointe :

  1. Sélectionnez Pièces jointes dans le menu Schéma.

  2. Dans l'onglet Pièces jointes, effectuez l'une des actions suivantes :

    • Cliquez sur le lien Copier l'élément à partir de Content Server.

    • Cliquez sur un emplacement dans la liste de navigation pour parcourir Content Server ou le dossier Pièces jointes.

  3. Activez la case à cocher Sélectionner située à côté de l'élément à ajouter.

  4. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Définition des pièces jointes.
