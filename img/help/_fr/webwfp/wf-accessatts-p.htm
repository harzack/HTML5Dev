Pour configurer l'accès aux attributs :

  1. Double-cliquez sur une étape Départ, Utilisateur, Initiateur ou Gestionnaire d'élément.

  2. Cliquez sur l'onglet Attributs.

  3. Pour chaque attribut, cliquez sur un paramètre d'accès dans la liste située à côté de l'attribut.

  4. Cliquez sur Ajouter à la définition du schéma de workflow.

 

Note :
Si la case à cocher Affichage page unique est activée dans l'onglet Général, vérifiez que le module Attributs apparaît dans la zone Afficher les modules.

 

Pour plus d'informations sur cette procédure, consultez la section Configuration de l'accès aux attributs.
