Une étape Tâche de formulaire vous permet d'ajouter un formulaire Content Server à un processus de schéma de workflow. Les informations collectées au cours d'une étape Tâche de formulaire peuvent être utilisées dans le schéma de workflow.

Pour plus d'informations sur l'utilisation des étapes Tâche de formulaire dans les schémas de workflow, voir Utilisation de formulaires dans les schémas de workflow.

 

Note :
Le module Forms doit être installé dans Content Server pour pouvoir utiliser l'étape Tâche de formulaire.
