Pour rendre plusieurs éléments physiques :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques empruntés.

  4. Sur la page Eléments physiques empruntés, cliquez sur Retour dans la liste déroulante Action.

  5. Effectuez l'une des actions suivantes :

    • Dans la section Date du retour, cliquez sur la date de début de la recherche dans les listes déroulantes A partir de.

    • Dans la section Date du retour, cliquez sur la date de fin de la recherche dans les listes déroulantes A.

    • Activez la case à cocher Prêt pour sélection.

    • Spécifiez des ID uniques dans le champ Liste d'ID uniques.

  6. Pour assigner un localisateur à des éléments physiques, activez la case à cocher Assigner un localisateur, choisissez l'Infrastructure et la Zone où se trouve le localisateur dans les listes déroulantes correspondantes, puis cliquez sur le bouton Rechercher des localisateurs et sélectionnez un localisateur pour l'élément que vous rendez.

  7. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  8. Sur la page Eléments physiques empruntés, cliquez sur la date correspondant au retour des éléments physiques dans la liste déroulante Date du retour.

  9. Procédez de l'une des manières suivantes :

    • Pour remettre à leur emplacement d'origine les éléments physiques empruntés, activez la case à cocher Retour à l'emplacement d'origine.

    • Pour remettre les éléments physiques à un emplacement autre que leur emplacement d'origine, saisissez un nom d'emplacement dans le champ Emplacement actuel.

  10. Activez la case à cocher en regard des éléments physiques empruntés que vous souhaitez rendre.

  11. Cliquez sur le bouton Rendre les éléments empruntés, puis sur le bouton OK dans la boîte de dialogue.

  12. Sur la page Statut de la circulation, cliquez sur OK.

 

Astuce :
Pour rendre tous les éléments empruntés de la liste, cliquez sur le bouton Sélectionner tout. Ce bouton vous permet de sélectionner tous les éléments empruntés, même si la liste s'étend sur plusieurs pages.

Vous pouvez également spécifier des critères de recherche pour les éléments physiques empruntés en scannant une liste de codes-barres dans le champ Liste d'ID uniques ou en cliquant sur le bouton Parcourir et en parcourant une liste d'ID uniques.
