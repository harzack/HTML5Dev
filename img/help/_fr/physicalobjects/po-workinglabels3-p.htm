Pour créer des libellés pour un localisateur de stockage :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Localisateurs.

  4. Sur la page Localisateurs, spécifiez les critères de recherche, puis cliquez sur le bouton Recherche.

  5. Dans la liste des résultats de la recherche, cliquez sur le lien d'un localisateur.

  6. Activez la case à cocher Générer un libellé.

  7. Cliquez sur un type de libellé dans la liste déroulante Type de libellé.

  8. Indiquez le nombre d'exemplaires à créer dans le champ Nombre d'exemplaires.

  9. Cliquez sur le bouton Mettre à jour.
