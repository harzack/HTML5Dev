Pour restreindre la recherche à certains types d'éléments physiques :

  1. Sur la page Recherche Content Server, cliquez sur le lien Types d'élément physique dans la liste Options avancées.

  2. Dans la section Types d'élément physique, cliquez sur Parcourir les types d'élément physique dans la liste déroulante, recherchez un type d'élément physique, puis cliquez sur le lien Sélectionner correspondant. Facultatif : pour utiliser un opérateur booléen, cliquez sur le bouton Doubler  , choisissez un opérateur et recommencez l'étape 2.

 

Astuce :
Vous pouvez également spécifier un type d'élément physique en cliquant sur Parcourir les types d'éléments physiques dans la liste déroulante, en recherchant un type d'élément physique, puis en cliquant sur le lien Sélectionner correspondant.
