Pour marquer pour sélection vos éléments physiques empruntés :

  1. Choisissez Eléments empruntés dans le menu Personnel.

  2. Sur la page Eléments physiques empruntés, activez la case à cocher pour chaque élément physique emprunté que vous voulez marquer pour sélection.

  3. Cliquez sur le bouton Marquer pour sélection.
