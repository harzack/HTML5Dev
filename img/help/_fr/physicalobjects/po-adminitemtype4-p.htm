Pour modifier le format d'ID unique d'un type d'élément physique :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Types d'élément physique.

  3. Sur la page Types d'élément physique, cliquez sur l'icône Fonctions d'un type d'élément physique, puis sélectionnez Propriétés et Infos spécifiques.

  4. Dans l'onglet Infos spécifiques de la page Propriétés Type d'élément physique, saisissez un nouveau format d'ID unique dans le champ Format d'ID unique.

  5. Cliquez sur le bouton Mettre à jour.
