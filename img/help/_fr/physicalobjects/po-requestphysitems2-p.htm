Pour demander plusieurs éléments physiques :

  1. Dans une collection ou un espace de travail de recherche avancée, activez la case à cocher en regard de chaque élément physique que vous souhaitez demander.

  2. Cliquez sur Demander des éléments physiques dans la liste déroulante Autres actions.

    Facultatif : sur la page Informations sur la requête pour, cliquez sur l'icône Choisir l'utilisateur dans la section Générateur de requêtes, puis sélectionnez l'utilisateur pour lequel vous souhaitez demander l'élément.

  3. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  4. Dans la liste déroulante Date de la requête, cliquez sur une date.

  5. Spécifiez les autres paramètres.

  6. Cliquez sur le bouton Requête.
