Pour rendre des éléments physiques :

  1. Sur la page de résultats de recherche Content Server (ou dans une collection), sélectionnez un ou plusieurs éléments physiques puis, dans la liste déroulante Autres actions, sélectionnez Rendre un élément physique.

  2. Sur la page Rendre l'élément physique, cliquez sur Annuler l'emprunt, Transmettre ou Rendre dans la liste déroulante Action.

  3. Cliquez sur Rendre.

  4. La page Eléments physiques s'affiche avec les champs spécifiques à l'action sélectionnée à l'étape 2. Actualisez les champs et cliquez sur Annuler, Transmettre ou Rendre.

  5. Les métadonnées saisies sur la page précédente s'affichent en lecture seule et le statut de chaque élément physique apparaît. Cliquez sur Annuler, Transmettre ou Rendre.

  6. Une fois l'action terminée, cliquez sur OK.

 

Astuce :
Il est possible de rendre plusieurs éléments physiques simultanément depuis la vue Parcourir d'un espace de travail. Vous devez pour cela activer la case à cocher pour chacun des éléments à rendre, puis cliquer sur Rendre un élément physique dans la liste déroulante Autres actions.
