Pour créer un rapport de liste de boîtes :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Rapports.

  3. Sur la page Rapports, cliquez sur le lien Rapport de liste de boîtes ou Rapport de liste de recueils.

  4. Sur la page Exécuter WebReports, effectuez l'une des opérations suivantes :

    • Pour créer un rapport pour une seule boîte ou un seul recueil d'éléments physiques, cliquez sur le bouton Parcourir Content Server afin de sélectionner la boîte ou le recueil.

    • Pour créer un rapport pour plusieurs boîtes ou recueils, saisissez les ID uniques des boîtes ou recueils dans la Liste d'ID uniques, ou scannez une liste de codes-barres.

  5. Cliquez sur le bouton Exécuter le rapport.
