Pour créer un type d'élément physique :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Types d'élément physique.

  3. Sur la page Types d'élément physique, cliquez sur Type d'élément physique dans le menu Ajouter un élément.

  4. Dans la section Sélectionner l'icône de la page Ajouter : Type d'élément physique, cliquez sur un des boutons radio.

    Facultatif : pour spécifier un ID unique pour le type d'élément physique, saisissez un format d'ID unique dans le champ Format d'ID unique.

    Facultatif : activez la case à cocher Ne peut pas être emprunté pour spécifier que ce type d'élément physique ne peut pas être emprunté.

  5. Cliquez sur l'un des boutons radio suivants :

    • Non recueil : permet d'ajouter un type d'élément physique ne pouvant pas contenir d'autres éléments physiques. Un non-recueil peut exister en plusieurs exemplaires.

    • Recueil : permet d'ajouter un type d'élément physique pouvant contenir plusieurs éléments physiques. Un recueil ne peut exister qu'en un seul exemplaire.

    • Boîte : permet d'ajouter un type d'élément physique auquel des éléments physiques peuvent être assignés, mais dans lequel il est impossible d'en créer. Une boîte ne peut exister qu'en un seul exemplaire.

    Facultatif : vous pouvez restreindre les types d'élément physique susceptibles d'être enregistrés dans le recueil ou la boîte que vous créez, en activant les cases à cocher correspondantes dans la section Type d'élément physique.

  6. Spécifiez les autres paramètres.

  7. Cliquez sur le bouton Ajouter un élément.

 

Note :
Des conflits peuvent se produire lorsqu'il existe plusieurs formats d'ID unique commençant par le même format. Par exemple, si vous assignez le format d'ID unique AB001 au type d'élément physique nommé Dossier physique et le format d'ID unique AB010 au type d'élément physique nommé Document physique, les ID uniques générés entreront en conflit à un moment donné et ralentiront le système.

L'icône choisie s'affiche en regard de chaque élément physique auquel est assigné ce type d'élément physique.
