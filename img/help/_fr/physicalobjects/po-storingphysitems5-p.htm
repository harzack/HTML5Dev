Pour assigner un localisateur de stockage aux boîtes d'un transfert :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Transfert.

  4. Sur la page Transferts, cliquez sur le bouton radio Transferts reçus.

    Facultatif : pour spécifier un transfert, saisissez un ID dans le champ ID de transfert.

  5. Cliquez sur le bouton Rechercher.

  6. Dans la liste de résultats de la recherche, cliquez sur l'icône Fonctions d'un transfert, puis sélectionnez Assigner un localisateur.

  7. Sur la page Ajouter : Localisateur, activez la case à cocher en regard de chaque boîte que vous souhaitez assigner au localisateur.

  8. Spécifiez des critères de recherche pour les localisateurs de stockage, puis cliquez sur le bouton Rechercher des localisateurs.

  9. Dans la liste des localisateurs, cliquez sur le lien Assigner dans la colonne Action du localisateur que vous souhaitez assigner.

 

Note :
Seules les infrastructures gérées sont disponibles dans la liste déroulante Infrastructure.

Si la boîte figurait déjà dans un localisateur et que vous lui en avez assigné un autre, l'espace libre de l'ancien localisateur est mis à jour.
