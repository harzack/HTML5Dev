Pour supprimer un code dans une table :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Gestion des tables.

  4. Sur la page Maintenance de la table des objets physiques, cliquez sur le lien d'une table.

  5. Cliquez sur le lien Supprimer dans la colonne Actions du code que vous souhaitez supprimer.
