Pour reconnaître des éléments empruntés via une notification par e-mail :

  1. Dans la notification par e-mail de reconnaissance, cliquez sur le lien de l'élément physique.

  2. Connectez-vous à Content Server.

  3. Sur la page Accusé de réception, cliquez sur le bouton Reconnaître.

 

Note :
Si vous transférez l'e-mail à une autre personne, celle-ci sera en mesure de le reconnaître, mais vous figurerez toujours en tant qu'emprunteur de l'élément.

 

Pour reconnaître des éléments empruntés à partir de la page Eléments empruntés :

  1. Dans le menu Personnel, sélectionnez Eléments empruntés.

  2. Sur la page Eléments physiques empruntés, cochez la case située en regard de chaque élément que vous voulez reconnaître, puis cliquez sur Reconnaître dans la liste déroulante Choisissez l'action à effectuer.
