Pour modifier les paramètres personnels d'objets physiques :

  1. Sélectionnez Paramètres dans le menu Outils.

  2. Sur la page Mes paramètres généraux, cliquez sur l'onglet Objets physiques.

  3. Sur la page Mes paramètres d'objets physiques, activez les cases à cocher Masquer l'onglet Libellés ou Masquer l'onglet Circulation pour masquer ces libellés sur la page Propriétés Elément physique.

  4. Activez la case à cocher Remettre les éléments à leur emplacement d'origine.

  5. Cochez la case Toujours envoyer un e-mail sur demande pour activer l'envoi automatique de notifications par e-mail chaque fois qu'un élément physique est demandé.

  6. Dans la liste déroulante Onglet de propriété par défaut, choisissez l'onglet Propriété que vous souhaitez voir apparaître lorsqu'un utilisateur clique sur un lien d'élément physique.

  7. Dans la section Paramètres par défaut, procédez de la façon suivante :

    • Sélectionnez un emplacement pour l'élément physique dans la liste déroulante Emplacement.

    • Sélectionnez un type de localisateur pour l'élément physique dans la liste déroulante Type de localisateur.

    • Sélectionnez un taux de référence pour l'élément physique dans la liste déroulante Taux de référence.

    • Dans le champ Client de boîte, cliquez sur l'icône Choisir l'utilisateur ou Choisir le client de boîte pour compléter le champ avec un utilisateur ou un client de boîte, ou cliquez sur l'icône Enlever pour supprimer l'utilisateur ou le client de boîte actuel.

    • Activez la case à cocher Générer un libellé pour créer un libellé pour l'élément physique.

    • Si vous avez activé la case à cocher Générer un libellé, cliquez sur un type de libellé dans la liste déroulante Type de libellé.

  8. Cliquez sur le bouton Mettre à jour.

 

Note :
Si vous souhaitez spécifier une valeur par défaut pour un code qui n'a pas encore été créé, vous pouvez le faire sur la page Mes paramètres d'objets physiques en cliquant sur Créer un nouveau code dans une des listes de paramètres par défaut.
