Le module Physical Objects permet d'utiliser les éléments physiques pour gérer les objets physiques. Les éléments physiques sont des entités électroniques d'une base de données qui représentent un objet physique, comme un livre ou un CD-ROM, ou un recueil d'objets physiques, comme une boîte ou un dossier. Une fois que vous avez créé des éléments physiques, vous pouvez contrôler leur utilisation lorsqu'ils sont empruntés, groupés et déplacés dans les espaces de travail Content Server. Vous pouvez afficher l'historique de circulation d'un élément physique. Pour rechercher un élément physique, utilisez les fonctions de recherche avancées de Content Server. Vous pouvez également configurer vos paramètres personnels d'objets physiques.
