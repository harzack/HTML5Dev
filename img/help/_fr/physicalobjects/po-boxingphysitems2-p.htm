Pour assigner un ou plusieurs éléments physiques à une boîte depuis la page Vue détaillée :

  1. Cliquez sur le lien d'une boîte.

  2. Sur la page Vue détaillée, cliquez sur le bouton Assigner à une boîte.

  3. Procédez de l'une des manières suivantes :

    • Pour assigner un seul élément physique, saisissez le nom d'un élément physique dans le champ Assigner un seul élément.

    • Pour assigner plusieurs éléments physiques, saisissez des ID uniques dans le champ Liste d'ID uniques.

    • Recherchez l'emplacement d'un fichier avec une liste d'ID uniques et saisissez-le dans le champ Lire à partir d'un fichier.

    Facultatif : pour mettre à jour l'emplacement des sous-éléments de l'élément ou des éléments physiques, activez la case à cocher Mettre à jour l'emplacement de tous les sous-éléments.

    Facultatif : pour mettre à jour le statut des sous-éléments de l'élément ou des éléments physiques, activez la case à cocher Mettre à jour le statut de tous les sous-éléments.

  4. Cliquez sur le bouton Mettre à jour.

 

Astuces :

  • Vous pouvez également assigner un seul élément en cliquant sur le bouton Parcourir Content Server pour rechercher un élément physique dans .

  • Vous pouvez également assigner plusieurs éléments physiques en scannant une liste de codes-barres dans le champ Liste d'ID uniques ou en cliquant sur le bouton Parcourir et en parcourant une liste d'ID uniques.
