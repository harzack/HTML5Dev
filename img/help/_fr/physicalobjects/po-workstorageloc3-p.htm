Pour afficher les éléments physiques assignés à un localisateur :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Localisateurs.

  4. Sur la page Localisateurs, spécifiez les critères de recherche, puis cliquez sur le bouton Recherche.

  5. Cliquez sur l'icône Fonctions d'un localisateur, puis sélectionnez Afficher le contenu.

 

Astuce :
Pour ajouter des critères de recherche dans le champ Localisateur, cliquez sur le bouton Ajouter une ligne , puis saisissez le texte de recherche dans les champs supplémentaires. Pour supprimer un champ de recherche Localisateur, cliquez sur le bouton Supprimer la ligne .
