Pour enlever un localisateur assigné à un élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique assigné à un localisateur, puis sélectionnez Propriétés et Infos spécifiques.

  2. Dans l'onglet Infos spécifiques de la page Propriétés Elément physique, cliquez sur le bouton Enlever le localisateur.

 

Note :
Il est possible d'enlever un localisateur assigné à plusieurs éléments physiques dans une collection ou un ensemble de résultats de recherche. Vous devez pour cela activer la case à cocher en regard de chaque élément pour lequel vous souhaitez enlever le localisateur, puis cliquer sur Enlever le localisateur dans le menu Autres actions. Pour plus d'informations sur les collections, reportez-vous à la rubrique Utilisation de collections.

Vous pouvez également enlever un localisateur assigné à plusieurs éléments physiques dans la vue Parcourir d'un espace de travail en cliquant sur Enlever le localisateur dans la liste déroulante Autres actions.
