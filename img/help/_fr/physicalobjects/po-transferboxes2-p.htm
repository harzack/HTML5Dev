Pour assigner plusieurs objets à un transfert :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur l'icône Fonctions du lien Transferts, puis sélectionnez Assigner du contenu.

  4. Sur la page Assigner à un transfert, saisissez un ID de transfert dans le champ ID de transfert.

  5. Spécifiez des ID uniques dans le champ Liste d'ID uniques.

  6. Cliquez sur le bouton Assigner.

 

Astuces :

  • Vous pouvez également spécifier un ID de transfert en cliquant sur le bouton Rechercher des transferts de la page Transferts. Pour cela, spécifiez des critères de recherche sur la page Sélectionner un transfert, puis cliquez sur le bouton Rechercher des transferts.

  • Il est également possible d'assigner plusieurs objets en scannant une liste de codes-barres dans le champ Liste d'ID uniques ou en cliquant sur le bouton Parcourir, puis en recherchant une liste d'ID uniques. A partir d'une collection, d'un ensemble de résultats de recherche ou encore de la vue Parcourir d'un espace de travail, vous devez activer la case à cocher pour chaque élément, puis cliquer sur le bouton Assigner à un transfert (ou dans la liste déroulante Autres actions).
