Pour masquer des champs de la page Ajouter : Elément physique et de l'onglet Infos spécifiques pour un type d'élément physique :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Types d'élément physique.

  3. Sur la page Types d'élément physique, cliquez sur l'icône Fonctions d'un type d'élément physique, puis sélectionnez Propriétés et Masquer les champs.

  4. Dans l'onglet Masquer les champs de la page Propriétés Élément physique, cochez les cases en regard des champs à masquer. Facultatif : si vous avez coché la case Générer un libellé, sélectionnez un type de libellé par défaut dans la liste déroulante, puis indiquez le nombre d'exemplaires dans le champ prévu à cet effet de manière à créer discrètement le libellé.

  5. Cliquez sur le bouton Mettre à jour.
