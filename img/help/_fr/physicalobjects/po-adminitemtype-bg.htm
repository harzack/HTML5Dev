Les types d'éléments physiques sont des définitions que les utilisateurs associent aux éléments physiques. Ainsi l'utilisateur peut identifier le type de l'élément physique géré. Lorsque vous créez un type d'élément physique, vous pouvez spécifier un nom et une description.

Lorsque vous avez créé un type d'élément physique, vous pouvez modifier ses propriétés générales, notamment son nom, sa description et ses paramètres d'affichage, ainsi que son icône et son ID unique. Vous pouvez également spécifier qu'un type d'élément physique ne peut pas être emprunté et masquer les champs de la page Ajouter : Elément physique et de l'onglet Infos spécifiques d'un type d'élément physique.

 

Note :
Si le champ ID unique, A partir du ou Jusqu'au est obligatoire, vous ne pouvez pas le masquer.

 

Dans l'onglet Infos spécifiques de la page Propriétés d'un élément physique, vous pouvez afficher et modifier les informations d'un élément physique telles que son type, ses dates de début/fin, son emplacement actuel, son type de localisateur ainsi que ses informations de stockage et son ID temporaire, s'il a été spécifié. Vous pouvez également synchroniser manuellement les dates des recueils en cliquant sur l'icône Dates du champ Dates de début/fin. La synchronisation manuelle est utile lorsque les données sont désynchronisées à cause de la survenue d'évènements tels que le déplacement ou la suppression d'éléments. Lorsque vous synchronisez manuellement les dates des recueils, tous les éléments du recueil seront mis à jour avec les valeurs de date maximales et minimales. La synchronisation manuelle de dates ne met pas à jour l'arbre complet, mais seulement le recueil actuel.

Vous pouvez également ajouter des propriétés physiques aux types d'éléments physiques que vous créez. Les propriétés physiques sont des champs supplémentaires associés à un type d'élément physique spécifique. Ils s'affichent lorsqu'un utilisateur crée un élément physique et lui assigne ce type d'élément physique. Il est possible d'afficher et de modifier les propriétés d'élément physique dans l'onglet Propriétés physiques.

 

Note :
Lorsque vous ajoutez une propriété supplémentaire à un type d'élément physique, vous devez cliquer sur le bouton Ajouter avant de cliquer sur le bouton Mettre à jour.

 

L'onglet Propriétés physiques vous permet de saisir des informations relatives à un type d'élément physique spécifique. Par exemple, si vous modifiez les propriétés physiques d'un type d'élément livre et que vous lui ajoutez un auteur, un titre et un numéro ISBN, les champs de la page Créer sont automatiquement renseignés chaque fois que vous créez un élément physique de type livre.
