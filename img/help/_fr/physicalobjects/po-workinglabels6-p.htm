Pour enlever des libellés générés par lots :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Lot de libellés.

  3. Sur la page Lot de libellés, cliquez sur l'un des boutons radio suivants :

    • Eléments physiques permet de rechercher des libellés pour les éléments physiques

    • Localisateurs permet de rechercher des libellés pour les localisateurs

  4. Spécifiez les autres critères de recherche, puis cliquez sur le bouton Recherche.

  5. Dans la première colonne, activez la case à cocher de chaque libellé que vous souhaitez enlever.

  6. Cliquez sur le bouton Enlever.
