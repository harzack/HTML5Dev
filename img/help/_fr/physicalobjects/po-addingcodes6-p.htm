Pour ajouter un code à la table Client externe :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Gestion des tables.

  4. Sur la page Maintenance de la table des objets physiques, cliquez sur le lien Client externe.

  5. Cliquez sur Client externe dans le menu Ajouter un élément.

  6. Renseignez le champ Code.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Envoyer.
