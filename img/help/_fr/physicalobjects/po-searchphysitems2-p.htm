Pour restreindre la recherche à certains attributs d'élément physique :

  1. Sur la page Recherche Content Server, cliquez sur le lien Attributs d'élément physique dans la liste Options avancées.

  2. Dans la section Attributs d'élément physique, cliquez sur un code d'emplacement d'origine dans la liste déroulante Emplacement d'origine.

    Facultatif : pour utiliser un opérateur booléen, cliquez sur le bouton Doubler  , choisissez un opérateur et recommencez l'étape 2.

  3. Cliquez sur Emplacement actuel dans la liste déroulante Attributs supplémentaires, puis cliquez sur un code d'emplacement actuel dans la liste déroulante Emplacement actuel.

    Facultatif : pour utiliser un opérateur booléen, cliquez sur le bouton Doubler  , choisissez un opérateur et recommencez l'étape 3.

  4. Facultatif : pour spécifier un intervalle de dates, sélectionnez les options de date dans les listes déroulantes Dates de début / fin. Le champ d'intervalle de dates est également disponible dans les collections.
