Pour emprunter des éléments physiques requis :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques requis.

  4. Sur la page Demander des éléments physiques, cliquez sur Emprunter dans la liste déroulante Action. Facultatif : pour approuver automatiquement les annulations de sécurité, activez la case à cocher Préapprouver les annulations de sécurité.

  5. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  6. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  7. Cliquez sur une date dans les listes déroulantes des sections suivantes :

    • Date d'emprunt : permet d'indiquer une date d'emprunt des éléments physiques.

    • Date du retour : permet d'indiquer une date de retour des éléments physiques.

  8. Spécifiez les autres paramètres.

  9. Cliquez sur le bouton Emprunter.
