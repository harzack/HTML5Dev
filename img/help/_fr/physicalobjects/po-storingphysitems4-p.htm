Pour assigner un localisateur de stockage aux éléments physiques d'une collection :

  1. Dans une collection, activez la case à cocher en regard de chaque élément physique que vous souhaitez assigner.

  2. Cliquez sur Assigner un localisateur dans la liste déroulante Autres actions.

  3. Sur la page Assigner un localisateur, cliquez sur une infrastructure dans la liste déroulante Infrastructure.

  4. Cliquez sur une zone de l'infrastructure dans la liste déroulante Zone.

  5. Spécifiez un localisateur de stockage dans le champ Localisateur. Facultatif : vous pouvez spécifier dans le champ ID de réserve hors site un ID pour l'infrastructure de stockage hors site.

  6. Cliquez sur le bouton Rechercher des localisateurs.

  7. Dans la liste des résultats de la recherche, cliquez sur le lien Sélectionner d'un localisateur.
