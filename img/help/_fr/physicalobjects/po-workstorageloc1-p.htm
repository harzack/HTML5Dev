Pour créer un localisateur de stockage :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Gestion du stockage, cliquez sur le lien Localisateurs.

  4. Sur la page Localisateurs, cliquez sur Localisateur dans le menu Ajouter un élément.

  5. Sur la page Ajouter : Localisateur, cliquez sur un code dans les listes déroulantes suivantes :

    • Infrastructure permet d'indiquer un code d'infrastructure pour le localisateur de stockage.

    • Zone permet d'indiquer un code de zone pour le localisateur de stockage.

  6. Saisissez le nom du localisateur dans le champ Localisateur.

  7. Saisissez une valeur dans les champs suivants :

    • Espace total permet d'indiquer la taille totale du localisateur.

    • Espace libre permet d'indiquer la quantité d'espace disponible pour le localisateur.

    Facultatif : dans la section Informations sur les infrastructures gérées, spécifiez les autres paramètres éventuels. Les informations de cette section sont utilisées uniquement si l'infrastructure assignée est gérée.

  8. Cliquez sur le bouton Nouveau.

 

Note :
Si la taille du localisateur dépasse la taille de l'espace disponible et que l'infrastructure est non gérée, l'infrastructure doit être configurée pour augmenter automatiquement l'espace total. Pour en savoir plus, voir Ajout de codes aux tables Objets physiques.
