Pour ajouter un code d'infrastructure non gérée à une table Infrastructure :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Gestion des tables.

  4. Sur la page Maintenance de la table des objets physiques, cliquez sur le lien Infrastructure.

  5. Cliquez sur Infrastructure dans le menu Ajouter un élément.

  6. Renseignez le champ Code.

  7. Cliquez sur le bouton radio Non géré. Facultatif : pour augmenter automatiquement l'espace total disponible d'une infrastructure non gérée, activez la case à cocher Augmenter automatiquement l'espace total.

  8. Spécifiez les autres paramètres.

  9. Cliquez sur le bouton Envoyer.

 

Astuce :
Vous pouvez également créer un code de maintenance de table sur la page où celui-ci sera utilisé, en cliquant sur <Créer un nouveau code> dans la liste déroulante du code.
