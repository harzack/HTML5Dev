Pour vérifier le contenu d'un localisateur :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur l'icône Fonctions du lien Localisateurs, puis sélectionnez Vérifier le contenu du localisateur.

  4. Sur la page Vérifier le contenu du localisateur, cliquez sur le bouton Parcourir, recherchez le fichier contenant les données du localisateur, puis cliquez sur le bouton Ouvrir. Facultatif : dans le champ Créer un fichier journal sous, saisissez le chemin d'accès du répertoire dans lequel vous souhaitez enregistrer le fichier journal des données du localisateur. Facultatif : dans la table de résultats, assurez-vous que toutes les cases des éléments auxquels vous voulez assigner un localisateur sont cochées, puis cliquez sur le bouton Tout mettre à jour.

  5. Cliquez sur le bouton Vérifier.

     

    Note :
    Chaque rangée du document doit inclure les données suivantes séparées par des virgules : infrastructure, zone, localisateur et ID unique.
