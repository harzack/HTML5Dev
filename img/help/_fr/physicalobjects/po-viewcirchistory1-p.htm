Pour afficher l'historique de circulation d'un élément :

  1. Cliquez sur l'icône Fonctions d'un élément physique, sélectionnez Propriétés, puis Circulation.

  2. Dans l'onglet Circulation de la page Propriétés Elément physique, cliquez sur le lien Historique de circulation.

  3. Sur la page Historique de circulation, spécifiez les paramètres.

  4. Cliquez sur le bouton Rechercher.
