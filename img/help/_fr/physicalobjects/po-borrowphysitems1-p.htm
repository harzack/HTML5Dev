Pour emprunter un élément physique ou un exemplaire d'élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique ou d'un exemplaire d'élément physique, sélectionnez Propriétés, puis Circulation.

  2. Dans l'onglet Circulation de la page Propriétés Elément physique, cliquez sur l'icône Fonctions de la colonne Actions, puis sélectionnez Circuler.

    Dans la page Informations sur la circulation, cliquez sur un code d'emplacement dans le champ Emplacement actuel pour spécifier l'emplacement dans lequel l'élément physique emprunté doit être stocké.

  3. Dans la section Emprunté par, cliquez sur l'icône Choisir un utilisateur, puis sélectionnez un utilisateur.

  4. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

    Facultatif : vous pouvez aussi saisir une raison pour l'emprunt dans le champ Commentaire.

  5. Cliquez sur la date d'emprunt de l'élément physique dans les listes déroulantes de la section Date d'emprunt.

  6. Cliquez sur la date de retour de l'élément physique dans les listes déroulantes de la section Date du retour.

  7. Dans la liste déroulante Reconnaître, cliquez sur une option afin de spécifier la manière dont les emprunteurs accusent réception de l'objet physique lorsque celui-ci leur est transmis ou lorsqu'ils l'empruntent.

  8. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Si l'élément est une boîte ou un recueil, vous pouvez afficher tous les enfants empruntés en cliquant sur le bouton Afficher les enfants empruntés.
