Pour annuler la demande de plusieurs éléments physiques :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques requis.

  4. Sur la page Demander des éléments physiques, cliquez sur Annuler la requête dans la liste déroulante Action.

    Facultatif : pour approuver automatiquement les annulations de sécurité, activez la case à cocher Préapprouver les annulations de sécurité.

  5. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  6. Activez la case à cocher en regard des éléments physiques demandés que vous souhaitez annuler.

  7. Cliquez sur le bouton Annuler la requête.
