Pour importer des codes de gestion de table ou de localisateur :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Importer/Exporter.

  4. Dans la section Importer, cliquez sur le lien Importer les codes de gestion des tables ou Importer les localisateurs.

  5. Sur la page Importer, cliquez sur le bouton Parcourir afin de rechercher le fichier XML dans lequel les codes de gestion de table ou les localisateurs doivent être enregistrés, puis cliquez sur Ouvrir et sur le bouton Importer.
