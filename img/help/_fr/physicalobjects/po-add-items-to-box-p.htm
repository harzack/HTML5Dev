Pour ajouter des éléments physiques à une boîte :

  1. Cliquez sur le lien du recueil de boîtes dans lequel vous souhaitez ajouter des éléments physiques.

  2. Dans la vue détaillée du recueil, cliquez sur le bouton Créer du contenu de boîte.

  3. Sur la page Elément physique, cliquez sur le bouton radio situé en regard du type d'élément physique que vous souhaitez ajouter, puis cliquez sur le bouton Suivant.

  4. Sur la page Ajouter : Elément physique, définissez les paramètres de la même façon que pour ajouter un élément physique à un espace de travail.

  5. Cliquez sur le bouton Ajouter pour n'ajouter que cet élément physique à la boîte ou cliquez sur le bouton Créer un autre pour ajouter un autre élément physique à la boîte.

 

Note :
Lorsque vous ajoutez un élément physique à une boîte, l'attribut Type de localisateur doit être défini sur <Aucun>.
