Pour assigner un localisateur de stockage à un élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique, puis sélectionnez Propriétés et Infos spécifiques.

  2. Dans l'onglet Infos spécifiques de la page Propriétés Elément physique, cliquez sur le bouton Assigner un localisateur.

  3. Sur la page Assigner un localisateur, cliquez sur une infrastructure dans la liste déroulante Infrastructure.

  4. Cliquez sur une zone de l'infrastructure dans la liste déroulante Zone.

  5. Spécifiez un localisateur de stockage dans le champ Localisateur.

    Facultatif : vous pouvez spécifier dans le champ ID de réserve hors site un ID pour l'infrastructure de stockage hors site.

  6. Cliquez sur le bouton Assigner.

 

Note :
Si vous n'assignez aucun type de localisateur à l'élément physique auquel vous assignez un localisateur de stockage, le bouton Assigner un localisateur ne s'affiche pas.

 
 

Astuce :
Il est possible d'assigner un localisateur à plusieurs éléments physiques à partir de la vue Parcourir d'un espace de travail. Vous devez pour cela activer la case à cocher pour chaque élément, puis cliquer sur Assigner un localisateur dans la liste déroulante Autres actions.

Vous pouvez également assigner un localisateur de stockage à un élément physique en cliquant sur le bouton Rechercher des localisateurs, puis sur le lien Sélectionner d'un localisateur de stockage, sur la page Sélectionner un localisateur.
