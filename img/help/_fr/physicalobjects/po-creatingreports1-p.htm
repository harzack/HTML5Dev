Pour créer un rapport de liste de retour :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Rapports.

  3. Sur la page Rapports, cliquez sur le lien Rapport de liste de retour.

  4. Facultatif Dans la section Filtrer le rapport par de la page d'exécution des rapports WebReport, filtrez les résultats en fonction des options suivantes :

    • Emprunteur

    • Dates de début et de fin de l'emprunt

  5. Cliquez sur le bouton Exécuter le rapport.
