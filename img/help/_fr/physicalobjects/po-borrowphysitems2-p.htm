Pour emprunter plusieurs éléments physiques :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Emprunter des éléments physiques.

  4. Sur la page Emprunter des éléments physiques, cliquez sur l'icône Choisir un utilisateur dans la section Emprunté par, puis sélectionnez un utilisateur.

  5. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  6. Cliquez sur une date dans les listes déroulantes des sections suivantes :

    • Date d'emprunt : permet d'indiquer une date d'emprunt des éléments physiques

    • Date du retour : permet d'indiquer une date de retour des éléments physiques

    Facultatif : pour approuver automatiquement les annulations de sécurité, activez la case à cocher Préapprouver les annulations de sécurité.

  7. Saisissez un ID unique dans le champ ID unique, analysez une liste de codes-barres dans le champ Liste d'ID uniques ou parcourez une liste d'ID uniques.

  8. Spécifiez les autres paramètres.

  9. Cliquez sur le bouton Rechercher.

  10. Dans la liste des résultats de la recherche, cochez les options en regard des éléments physiques que vous souhaitez emprunter.

  11. Dans la liste déroulante Reconnaître, cliquez sur une option pour spécifier comment les emprunteurs accusent réception de l'objet physique lorsque celui-ci leur est transmis ou lorsqu'ils l'empruntent.

  12. Cliquez sur le bouton Emprunter.

 

Note :
Vous devez disposer des autorisations d'accès à la page Circulation pour pouvoir prêter plusieurs éléments physiques.

 
 

Astuce :
Vous pouvez également emprunter plusieurs éléments physiques à partir de la vue Parcourir d'un espace de travail. Vous devez pour cela activer la case à cocher située en regard de chacun des éléments à emprunter, puis cliquer sur le bouton Emprunter des éléments physiques (ou dans la liste déroulante Autres actions).

Si un élément est une boîte ou un recueil, vous pouvez afficher tous les enfants empruntés en cliquant sur le lien Afficher dans la colonne A des enfants empruntés.
