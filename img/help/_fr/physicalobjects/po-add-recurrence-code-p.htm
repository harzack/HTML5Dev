Pour ajouter un code à la table Récurrence :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Gestion des tables.

  4. Sur la page Maintenance de la table des objets physiques, cliquez sur le lien Récurrence.

  5. Dans le menu Ajouter un élément, cliquez sur Récurrence.

  6. Saisissez un code de récurrence dans le champ Code.

    Si vous le souhaitez, vous pouvez saisir une description du code de récurrence dans le champ Description.

  7. Dans le champ Nombre de jours, saisissez un nombre entier pour désigner le délai entre les requêtes, puis cliquez sur le bouton Envoyer.

 

Astuce :
Vous pouvez également créer un code de maintenance de table sur la page où celui-ci sera utilisé, en cliquant sur <Créer un nouveau code> dans la liste déroulante du code.
