Pour modifier la date de retour prévue d'éléments physiques empruntés :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques empruntés.

  4. Sur la page Eléments physiques empruntés, cliquez sur Changer la date du retour dans la liste déroulante Action.

    Facultatif : saisissez l'ID utilisateur ou tout autre critère de recherche disponible afin d'affiner les résultats renvoyés par la recherche.

  5. Dans la Liste d'ID uniques, saisissez des ID uniques d'éléments empruntés. Sinon, recherchez un fichier contenant la liste d'ID uniques.

  6. Cliquez sur le bouton Rechercher.

  7. Dans le champ Nouvelle date de retour, sélectionnez une nouvelle date dans les listes déroulantes appropriées, ou différez la date d'un nombre de jours spécifique en cliquant sur le bouton radio Différer de et en indiquant le nombre de jours souhaité dans le champ prévu à cet effet.

  8. Activez la case à cocher située en regard de chaque élément pour lequel vous souhaitez modifier la date de retour.

  9. Cliquez sur le bouton Changer la date du retour.
