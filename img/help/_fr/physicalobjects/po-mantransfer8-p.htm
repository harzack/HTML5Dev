Pour assigner des éléments physiques à un transfert :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Transferts.

  4. Dans la section Rechercher des transferts de la page Transferts, spécifiez des critères de recherche, puis cliquez sur le bouton Recherche.

  5. Dans la liste de résultats de la recherche, cliquez sur l'icône Fonctions d'un transfert, puis choisissez Assigner du contenu.

  6. Sur la page Assigner à un transfert, saisissez un ID dans le champ ID de transfert.

  7. Dans le champ Liste d'ID uniques, attribuez des ID uniques aux éléments physiques.

  8. Cliquez sur le bouton Assigner.

 

Astuce :
Vous pouvez également assigner des éléments physiques en scannant une liste de codes-barres dans le champ Liste d'ID uniques ou en cliquant sur le bouton Parcourir et en parcourant une liste d'ID uniques.
