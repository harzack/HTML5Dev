Pour supprimer un localisateur de stockage :

  1. Dans le menu Personnel, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Localisateurs.

  4. Sur la page Localisateurs, spécifiez les critères de recherche, puis cliquez sur le bouton Recherche.

  5. Cliquez sur l'icône Fonctions d'un localisateur, puis choisissez Supprimer.

 

Note :
Vous ne pouvez pas supprimer les localisateurs auxquels ont été assignés des éléments physiques. Pour plus d'informations sur la suppression d'éléments physiques assignés ou de localisateurs d'éléments physiques, reportez-vous à la rubrique Stockage d'éléments physiques.
