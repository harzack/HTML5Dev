Pour modifier l'icône d'un type d'élément physique :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Types d'élément physique.

  3. Sur la page Types d'élément physique, cliquez sur l'icône Fonctions d'un type d'élément physique, puis sélectionnez Propriétés et Infos spécifiques.

  4. Dans l'onglet Infos spécifiques de la page Propriétés Type d'élément physique, cliquez sur un des boutons radio de la section Icône.

  5. Cliquez sur le bouton Mettre à jour.

 

Note :
Lorsque vous modifiez l'icône d'un type d'élément physique existant, la modification ne s'applique pas à l'icône des éléments physiques assignés à ce type avant la modification.
