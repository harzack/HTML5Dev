Pour imprimer des libellés par lots :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Lot de libellés puis sur le lien Rechercher et imprimer les libellés générés.

  3. Sur la page Rechercher et imprimer des libellés générés, cliquez sur l'un des boutons radio suivants :

    • Eléments physiques permet de rechercher des libellés pour les éléments physiques

    • Localisateurs permet de rechercher des libellés pour les localisateurs

    • Utilisateurs permet de rechercher des libellés pour les utilisateurs

  4. Spécifiez les autres critères de recherche, puis cliquez sur le bouton Recherche.

  5. Dans la première colonne, activez la case à cocher de chaque libellé que vous souhaitez imprimer.

    Facultatif : pour afficher les lots de libellés avant de les imprimer, cliquez sur le bouton Aperçu.

  6. Cliquez sur le bouton Imprimer.
