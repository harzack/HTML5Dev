Pour transmettre plusieurs éléments physiques empruntés :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques empruntés.

  4. Sur la page Eléments physiques empruntés, cliquez sur Transmettre dans la liste déroulante Action.

  5. Dans la section Transmettre à, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur.

  6. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  7. Cliquez sur une date dans les listes déroulantes des sections suivantes :

    • Nouvelle date d'emprunt : permet d'assigner une nouvelle date d'emprunt aux éléments physiques empruntés.

    • Nouvelle date de retour : permet d'assigner une nouvelle date de retour aux éléments physiques empruntés.

    Facultatif : pour approuver automatiquement les annulations de sécurité, activez la case à cocher Préapprouver les annulations de sécurité.

  8. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  9. Dans la liste des résultats de la recherche, cochez les options en regard des éléments physiques empruntés que vous souhaitez transmettre.

  10. Cliquez sur le bouton Transmettre.
