Pour recalculer l'espace de stockage :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Gestion du stockage, cliquez sur le lien Localisateurs.

  4. Sur la page Localisateurs, spécifiez les critères de recherche, puis cliquez sur le bouton Recherche.

  5. Cliquez sur l'icône Fonctions d'un localisateur, puis sélectionnez Nouveau calcul de localisateur.

  6. Sur la page Recalculer le localisateur, cliquez sur une infrastructure de stockage dans la liste déroulante Infrastructure.

  7. Cliquez sur une zone dans la liste Zone.

    Facultatif : pour augmenter l'espace total dans des infrastructures non gérées, activez la case à cocher Incrémenter l'espace total.

  8. Cliquez sur le bouton Recalculer.

 

Astuce :
Vous pouvez sélectionner plusieurs zones en appuyant sur CTRL avant de cliquer sur une zone de la liste Zone.

Vous pouvez également recalculer l'espace de stockage en cliquant sur l'icône Fonctions du lien Localisateur, puis en sélectionnant Nouveau calcul de localisateur.
