Lorsque vous avez emprunté un élément physique, vous pouvez le rendre, annuler l'emprunt ou le transmettre à un autre utilisateur. Vous pouvez réaliser toutes ces tâches sur un ou plusieurs éléments physiques.

Marquage d'éléments pour sélection

Vous pouvez également marquer pour sélection un ou plusieurs éléments physiques, voire tous les éléments physiques que vous avez empruntés. Marquer pour sélection signifie qu'un utilisateur a emprunté l'élément et spécifié une date de retour à laquelle il est supposé le rendre. Si l'utilisateur souhaite rendre l'élément avant la date prévue, il peut marquer l'élément comme étant « prêt pour sélection », ce qui signifie que l'élément est prêt pour être rendu. Si le gestionnaire de documents a activé cette fonctionnalité, un e-mail est envoyé aux adresses e-mail spécifiées lorsque les éléments sont marqués pour sélection.

Renvoi, annulation et transfert des objets physiques

Lorsque vous rendez un élément physique emprunté, vous pouvez modifier son emplacement de stockage, indiquer un commentaire sur le retour, définir la date de retour ou définir si le retour doit s'effectuer à l'emplacement d'origine.

Lorsque vous rendez, que vous annulez ou que vous transmettez l'emprunt de plusieurs éléments physiques, vous pouvez spécifier le nom de l'utilisateur qui a emprunté les éléments physiques, les dates de début et de fin de la recherche, l'emplacement dans lequel les éléments sont stockés, l'emplacement de stockage actuel des éléments physiques et les ID uniques des éléments physiques spécifiques.

Vous pouvez également assigner des localisateurs aux éléments empruntés que vous rendez sur la page Eléments physiques empruntés. Cette option n'est disponible que si vous sélectionnez Rendre ou Annuler l'emprunt dans la liste déroulante Actions.

Modification de la date de retour d'éléments physiques empruntés

Vous pouvez modifier la date de retour d'éléments physiques empruntés en effectuant une recherche à partir des ID uniques leur correspondant. Il est possible d'affiner les résultats de la recherche en spécifiant certains critères tels que l'ID de l'utilisateur, la date de retour initiale ou encore l'emplacement actuel et d'origine de l'élément physique.

Vous disposez de deux options pour modifier la date de retour prévue. Vous pouvez spécifier une nouvelle date à laquelle vous souhaitez rendre le ou les éléments, ou différer la date de retour initiale d'un certain nombre de jours.

Lorsque vous modifiez la date de retour d'un élément, le processus est consigné dans l'historique d'expertise de l'élément concerné. Tout commentaire ajouté au cours de ce processus est également consigné dans l'historique d'expertise.

Reconnaissance des objets physiques

Pour assurer la responsabilité relative aux objets physiques empruntés, les gestionnaires de documents peuvent activer la fonctionnalité de reconnaissance. Lorsqu'elle est activée, vous êtes invité à accuser réception d'un élément lorsqu'il vous est affecté ou transmis à partir du système ou d'un autre utilisateur. Lorsqu'un objet physique vous est affecté ou transmis, vous avez cinq jours pour en accuser réception. Si vous omettez d'accuser réception de l'objet physique, il sera renvoyé à l'utilisateur qui l'a transmis. Les utilisateurs externes sont également invités à reconnaître un objet physique lorsqu'il leur est envoyé. Des notifications pour reconnaître un élément emprunté seront envoyées par e-mail.
