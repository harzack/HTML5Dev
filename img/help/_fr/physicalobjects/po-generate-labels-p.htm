Pour créer des libellés pour les éléments physiques, localisateurs et utilisateurs par lots :

  1. Dans le menu Entreprise, cliquez sur Objets physiques.

  2. Dans l'espace de travail Objets physiques, cliquez sur le lien Lot de libellés, puis sur le lien Générer de nouveaux libellés.

  3. Sur la page Générer des libellés, cliquez sur le bouton radio Eléments physiques, Localisateurs ou Utilisateurs pour sélectionner le type d'objet.

  4. Spécifiez les critères de recherche utilisés pour la création des libellés.

     

    Note :
    Pour plus d'informations sur les critères de recherche disponibles, reportez-vous à la section Utilisation de libellés.

     

  5. Saisissez une liste des ID uniques devant être utilisés pour la création de libellés dans le champ Liste d'ID uniques. Sinon, recherchez l'emplacement d'un fichier contenant la liste des ID uniques.

  6. Sélectionnez un type de libellé dans les listes déroulantes Type de libellé et Papier de libellés.

  7. Saisissez la position de départ des libellés dans le champ Position de départ pour l'impression.

  8. Cliquez sur le bouton Générer.
