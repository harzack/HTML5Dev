Pour assigner un objet à un transfert :

  1. Cliquez sur l'icône Fonctions d'un objet, choisissez Propriétés, puis Infos spécifiques.

  2. Dans l'onglet Infos spécifiques de la page Propriétés Elément physique, cliquez sur le bouton Assigner à un transfert.

  3. Sur la page Assigner à un transfert, saisissez un ID dans le champ ID de transfert.

  4. Cliquez sur le bouton Assigner.

 

Note :
Si vous n'assignez pas de type de localisateur à l'objet transféré, le bouton Assigner ne s'affiche pas.

 
 

Astuce :
Vous pouvez également spécifier un ID de transfert en cliquant sur le bouton Rechercher des transferts de la page Transferts. Pour cela, spécifiez des critères de recherche sur la page Sélectionner un transfert, puis cliquez sur le bouton Rechercher des transferts.
