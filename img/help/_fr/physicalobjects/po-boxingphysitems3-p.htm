Pour assigner des éléments physiques d'une collection à une boîte :

  1. Dans une collection, activez la case à cocher en regard de chaque élément physique que vous souhaitez assigner.

  2. Cliquez sur Assigner une boîte dans la liste déroulante Autres actions.

    Facultatif : pour mettre à jour le statut des sous-éléments de l'élément physique, activez la case à cocher Mettre à jour le statut de tous les sous-éléments.

    Facultatif : pour mettre à jour l'emplacement des sous-éléments de l'élément physique, activez la case à cocher Mettre à jour l'emplacement de tous les sous-éléments.

  3. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  4. Dans la liste des résultats de la recherche, cliquez sur le lien Sélectionner d'une boîte.

  5. Sur la page Informations sur la mise en boîte, cliquez sur le bouton Assigner.
