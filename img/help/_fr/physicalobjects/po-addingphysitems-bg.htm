Le module Physical Objects permet de créer un élément physique et de l'utiliser pour organiser des objets physiques. Un élément physique est une représentation virtuelle d'un objet physique associé à un type d'élément physique.

Lorsque vous créez un élément physique, vous pouvez spécifier un nom, une description, un ID unique, des mots clés, des catégories et l'emplacement dans lequel vous créez l'élément physique. L'emplacement dans lequel vous créez l'élément physique est appelé emplacement d'origine. Si l'emplacement d'origine est modifié après la création de l'élément physique, l'emplacement actuel n'est mis à jour que si la valeur est la même que l'ancien emplacement.

Si vous créez un élément physique dont le type est une boîte ou un recueil, vous pouvez également spécifier les dates de début et de fin pour les éléments qui sont contenus dans la boîte ou le recueil ou qui leur sont assignés. Les dates de début et de fin du recueil se mettent à jour automatiquement pour correspondre aux dates de début et de fin du contenu du recueil. Ainsi, si vous spécifiez des dates, puis ajoutez ou retirez des éléments dont les dates de début et de fin sont antérieures ou postérieures, les dates spécifiées initialement pour le recueil seront modifiées.

Vous pouvez spécifier la fréquence d'accès à une boîte, un ID pour les boîtes stockées hors site et le nom de l'utilisateur responsable de la boîte. Si l'élément physique est géré, vous pouvez le classer.

Vous pouvez créer un autre élément physique qui contient les métadonnées définies pour un élément physique existant. De plus, lorsque vous avez créé un élément physique, vous pouvez en faire des copies.

 

Note :
Lorsque vous ajoutez un élément physique, vous pouvez spécifier un ID unique pour cet élément. Si vous ne spécifiez pas d'ID unique, un ID unique sera automatiquement assigné à l'élément. Si votre administrateur vous le permet, l'ID unique s'affiche une fois que vous avez créé l'élément physique.
