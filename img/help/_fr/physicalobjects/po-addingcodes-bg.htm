Le module Physical Objects vous permet de saisir des codes associés aux éléments physiques. Ces codes permettent de définir des options spécifiques que les utilisateurs peuvent sélectionner lorsqu'ils utilisent ou gèrent des éléments physiques. Vous pouvez ajouter des codes aux tables suivantes sur la page Gestion des tables :

  • Infrastructure : permet de saisir des codes permettant d'effectuer le suivi des boîtes envoyées vers des infrastructures de stockage hors site. Les codes d'infrastructure peuvent être utilisés pour des infrastructures gérées ou non gérées. Lorsque vous ajoutez des codes à la table Infrastructure, vous pouvez spécifier une description et une adresse. Vous pouvez définir des règles de correspondance d'infrastructures gérées pour le taux de référence, la longueur, la profondeur, la hauteur et le type de réserve. Les règles de correspondance permettent de faire en sorte que les propriétés d'une infrastructure et les propriétés des boîtes qui lui sont assignées sont identiques.

  • Zone : permet d'ajouter des codes qui définissent des zones spécifiques dans une infrastructure de stockage. Lorsque vous ajoutez un code à la table Zone, vous pouvez spécifier une description.

  • Site gardien : autorise la saisie de codes permettant à l'utilisateur de faire circuler des éléments à l'intérieur de son site gardien, mais limitant l'affichage d'éléments à ceux des autres sites gardiens.

  • Type de localisateur : permet d'ajouter des codes qui renvoient à un emplacement spécifique d'une tablette pouvant contenir une seule boîte, à la totalité d'une tablette pouvant contenir plusieurs boîtes ou à un autre emplacement physique d'une zone de stockage de boîtes. Lorsque vous ajoutez un code à la table Type de localisateur, vous pouvez spécifier une description et les dimensions de l'emplacement.

  • Client externe : permet de saisir des codes à utiliser lors de l'emprunt d'éléments physiques. Cette table sert à assigner des éléments physiques à des utilisateurs non-Content Server sans saisir les informations dans un formulaire libre. Lorsque vous ajoutez des codes à la table Client externe, vous pouvez spécifier une description, ainsi que l'adresse e-mail (ou plusieurs adresses e-mail séparées par un point-virgule), l'adresse physique et le numéro de téléphone du client. Vous pouvez indiquer si client externe désigne Emprunteur externe ou Client de boîte. Client de boîte peut être assigné à une boîte en tant que personne ou société responsable d'une boîte.

  • Emplacement : permet de saisir des codes spécifiant l'emplacement d'éléments physiques. Vous pouvez utiliser cette table pour configurer votre système pour qu'il n'autorise pas les entrées de formulaire libre pour les champs d'emplacement. Lorsque vous ajoutez des codes à la table Emplacement, vous pouvez spécifier une description et une ou plusieurs adresses e-mail séparées par un point-virgule. Vous pouvez également ajouter un site gardien ou désactiver le code d'emplacement afin de le rendre indisponible dans la liste déroulante Emplacement d'origine des éléments physiques.

  • Récurrence : désigne le délai entre deux requêtes. Pour plus d'informations sur les requêtes d'éléments physiques, consultez la section Demande d'éléments physiques.

  • Taux de référence : permet d'ajouter des codes qui décrivent la fréquence d'accès à une boîte ou à un localisateur de stockage. Lorsque vous ajoutez des codes à la table Taux de référence, vous pouvez spécifier une description du code.

  • Type de requête : permet d'ajouter des codes spécifiant le type de requête. Lorsque vous ajoutez un code de type de requête, vous pouvez spécifier une description du code.

 

Note :
En plus de pouvoir ajouter des codes sur la page Maintenance de la table, vous pouvez également créer des codes de maintenance de table sur la page sur laquelle le code est utilisé. Cette fonction est particulièrement utile, car elle vous permet de créer des codes à la volée si ce code n'existe pas ou lorsque le code requis n'a pas été créé. La fonction qui permet la création d'un code sur la page destinée à l'utiliser n'est pas disponible sur les pages suivantes :

  • Rechercher des localisateurs

  • Pages Circulation – Emprunter des éléments physiques, Eléments physiques empruntés, Mise en boîte et Eléments physiques requis.
