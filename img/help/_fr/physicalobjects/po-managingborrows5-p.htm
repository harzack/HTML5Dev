Pour transmettre un élément physique emprunté :

  1. Cliquez sur l'icône Fonctions d'un élément physique emprunté, sélectionnez Propriétés, puis Circulation.

  2. Dans l'onglet Circulation de la page Propriétés Elément physique, cliquez sur l'icône Fonctions de la colonne Actions, puis sélectionnez Transmettre.

  3. Sur la page Transmettre les informations, saisissez un nom d'emplacement dans le champ Emplacement actuel.

  4. Dans la section Transmettre à, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Facultatif : pour spécifier un commentaire sur la transmission de l'élément physique emprunté, saisissez un commentaire dans le champ Commentaire.

  5. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  6. Cliquez sur une date dans les listes déroulantes des sections suivantes :

    • Nouvelle date d'emprunt : permet d'assigner une nouvelle date d'emprunt aux éléments physiques empruntés.

    • Nouvelle date de retour : permet d'assigner une nouvelle date de retour aux éléments physiques empruntés.

  7. Cliquez sur le bouton Mettre à jour.
