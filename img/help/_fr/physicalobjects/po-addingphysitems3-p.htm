Pour créer un exemplaire d'élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique, sélectionnez Propriétés, puis Circulation.

  2. Dans l'onglet Circulation de la page Propriétés Elément physique, cliquez sur le bouton Ajouter un nouvel exemplaire.

  3. Sur la page Informations sur la copie, saisissez un ID dans le champ ID unique. Facultatif : pour spécifier un emplacement actuel différent de celui du parent, saisissez un nom d'emplacement dans le champ Emplacement actuel.

     

    Note :
    Le champ Emplacement actuel n'est disponible que si le parent du nouvel exemplaire est prêté.

     

  4. Cliquez sur le bouton Ajouter un élément.

 

Astuce :
Si vous ne renseignez pas le champ ID unique, l'ID unique est généré automatiquement.
