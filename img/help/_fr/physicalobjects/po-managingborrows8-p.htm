Pour marquer pour sélection plusieurs éléments physiques empruntés :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Circulation.

  3. Sur la page Circulation, cliquez sur le lien Eléments physiques empruntés.

  4. Sur la page Eléments physiques empruntés, cliquez sur Marquer pour sélection dans la liste déroulante Action.

  5. Spécifiez les autres paramètres, puis cliquez sur le bouton Recherche.

  6. Dans la liste des résultats de la recherche, cochez la case en regard des éléments physiques empruntés que vous souhaitez marquer pour sélection.

  7. Cliquez sur le bouton Marquer pour sélection.
