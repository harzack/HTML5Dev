Vous pouvez exporter les codes de maintenance de table d'objets physiques et de localisateurs depuis la source Content Server, puis les importer vers un autre serveur. Ceci est particulièrement utile lorsque Content Server est installé sur plusieurs ordinateurs ou configuré sur plusieurs environnements. Les codes sont alors exportés vers un fichier XML et enregistrés à l'emplacement suivant :

C:\contentserver_installation_path\temp\filename.xml

où contentserver_installation_path est l'emplacement d'installation de Content Server et filename.xml le nom du fichier XML spécifié par vos soins au cours du processus d'exportation.

Le fichier XML est par ailleurs automatiquement chargé sur votre espace de travail personnel.
