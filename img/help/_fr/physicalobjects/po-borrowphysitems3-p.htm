Pour emprunter les éléments physiques d'une collection :

  1. Dans la collection, cochez les cases en regard des éléments physiques que vous souhaitez emprunter.

  2. Cliquez sur Emprunter des éléments physiques dans la liste déroulante Autres actions.

  3. Sur la page Emprunter des éléments physiques, cliquez sur l'icône Choisir un utilisateur dans la section Emprunté par, puis sélectionnez un utilisateur.

  4. Dans la section Obtenu par, cliquez sur l'icône Choisir l'utilisateur, puis sélectionnez un utilisateur. Renseignez ce champ si l'élément physique ou l'exemplaire de l'élément physique est emprunté au nom de l'utilisateur identifié dans la section Emprunté par.

  5. Cliquez sur une date dans les listes déroulantes des sections suivantes :

    • Date d'emprunt : permet d'indiquer une date d'emprunt des éléments physiques.

    • Date du retour : permet d'indiquer une date de retour des éléments physiques.

    Facultatif : pour approuver automatiquement les annulations de sécurité, activez la case à cocher Préapprouver les annulations de sécurité.

  6. Cliquez sur le bouton Emprunter.
