Pour exporter des codes de gestion de table ou de localisateur :

  1. Dans le menu Entreprise, choisissez Objets physiques.

  2. Sur la page Objets physiques, cliquez sur le lien Administration d'objets physiques.

  3. Sur la page Administration d'objets physiques, cliquez sur le lien Importer/Exporter.

  4. Dans la section Exporter, cliquez sur le lien Exporter les codes de gestion des tables ou Exporter les localisateurs.

  5. Dans le champ Nom du fichier d'exportation, saisissez le nom du fichier dans lequel les codes seront exportés, puis cliquez sur le bouton Exporter.
