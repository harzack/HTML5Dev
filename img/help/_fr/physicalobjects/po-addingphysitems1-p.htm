Pour créer un élément physique :

  1. Cliquez sur Elément physique dans le menu Ajouter un élément.

  2. Sur la page Ajouter : Elément physique, cliquez sur le bouton radio du type d'élément physique souhaité, puis cliquez sur Suivant.

  3. Saisissez un nom et une description pour l'élément dans les champs Nom et Description.

  4. Dans le champ Emplacement d'origine, indiquez l'emplacement de l'élément physique.

    Facultatif : pour spécifier un emplacement temporaire pour un élément physique dont le parent a été emprunté, saisissez un emplacement temporaire dans le champ Emplacement actuel.

  5. Dans le champ Nombre d'exemplaires, indiquez le nombre d'exemplaires devant être créés.

     

    Note :
    Les recueils comme les boîtes ne peuvent pas exister en plusieurs exemplaires.

     

    Facultatif : pour indiquer le type d'emplacement où l'élément physique peut être stocké, cliquez sur un type de localisateur dans la liste déroulante Type de localisateur.

  6. Spécifiez les autres paramètres.

  7. Cliquez sur le bouton Ajouter.

  8. Facultatif Cliquez sur le bouton Créer un autre, puis entrez le nombre d'éléments supplémentaires que vous souhaitez créer avec les mêmes métadonnées.

 

Note :
Si l'ID unique de l'élément physique s'affiche après sa création, cliquez sur le bouton Continuer.
