Pour créer un transfert :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur le lien Transferts.

  4. Sur la page Transferts, cliquez sur Transfert dans le menu Ajouter : Elément.

  5. Sur la page Ajouter : Transfert, spécifiez un ID dans le champ ID de transfert.

  6. Dans la section Créé, cliquez sur une date dans les listes déroulantes.

  7. Spécifiez les autres paramètres.

  8. Cliquez sur le bouton Ajouter un élément.

 

Note :
Dans la liste déroulante Créé, la date est définie par défaut sur la date actuelle et ne peut pas être ultérieure à cette date.
