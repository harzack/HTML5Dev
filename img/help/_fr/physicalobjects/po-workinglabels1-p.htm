Pour créer des libellés pour un élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique, puis sélectionnez Propriétés et Libellés.

  2. Dans l'onglet Libellés de la page Propriétés Elément physique, cliquez sur un type de libellé dans la liste déroulante Type de libellé.

  3. Indiquez le nombre d'exemplaires à créer dans le champ Nombre d'exemplaires.

  4. Cliquez sur le bouton Générer.

 

Astuce :
Il est possible de générer des libellés pour plusieurs éléments physiques à partir d'une collection, d'un ensemble de résultats de recherche ou de la vue Parcourir d'un espace de travail. Vous devez pour cela activer la case à cocher située en regard de chaque élément, puis cliquer sur le bouton Générer des libellés (ou dans la liste déroulante Autres actions).
