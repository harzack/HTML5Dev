Pour modifier les propriétés générales des types d'éléments physiques :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Types d'élément physique.

  3. Sur la page Types d'élément physique, cliquez sur l'icône Fonctions d'un type d'élément physique et sélectionnez Propriétés, puis Général.

  4. Dans l'onglet Général de la page Propriétés Type d'élément physique, modifiez les paramètres souhaités.

  5. Cliquez sur le bouton Mettre à jour.
