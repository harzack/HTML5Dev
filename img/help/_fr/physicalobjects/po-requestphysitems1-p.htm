Pour demander un élément physique :

  1. Cliquez sur l'icône Fonctions d'un élément physique, sélectionnez Propriétés, puis Circulation.

  2. Dans l'onglet Circulation de la page de Propriétés Elément physique, cliquez sur l'icône Fonctions de l'élément, puis sélectionnez Requête.

    Facultatif : sur la page Informations sur la requête pour, cliquez sur l'icône Choisir l'utilisateur dans la section Générateur de requêtes, puis sélectionnez l'utilisateur pour lequel vous souhaitez demander l'élément.

  3. Dans les listes déroulantes Date de la requête, cliquez sur une date.

  4. Spécifiez les autres paramètres.

  5. Cliquez sur le bouton Envoyer.

 

Note :
Si vous activez la case à cocher Récurrent, vous devez également sélectionner un intervalle de temps dans les listes déroulantes Répéter tou(te)s les. Si vous n'effectuez pas cette action, vous recevrez un message d'erreur vous demandant de fournir les informations relatives à la récurrence.

 
 

Astuce :
Pour demander plusieurs éléments physiques à partir d'une collection, d'une recherche avancée ou de la vue Parcourir d'un espace de travail, activez la case à cocher en regard des éléments que vous souhaitez demander, puis cliquez sur Demander des éléments physiques dans la liste déroulante Autres actions.
