Pour assigner des éléments physiques à un localisateur de stockage :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Gestion du stockage.

  3. Sur la page Gestion du stockage, cliquez sur l'icône Fonctions du lien Localisateurs, puis sélectionnez Assigner du contenu.

  4. Sur la page Assigner un localisateur, cliquez sur une infrastructure dans la liste déroulante Infrastructure.

  5. Cliquez sur une zone de l'infrastructure dans la liste déroulante Zone.

  6. Spécifiez un localisateur de stockage dans le champ Localisateur.

  7. Dans le champ Liste d'ID uniques, attribuez des ID uniques aux éléments physiques.

  8. Cliquez sur le bouton Assigner.

 

Astuces :

Il est possible d'assigner un localisateur à plusieurs éléments physiques à partir d'une collection, d'un ensemble de résultats de recherche ou de la vue Parcourir d'un espace de travail. Vous devez pour cela activer la case à cocher pour chaque élément, puis cliquer sur le bouton Assigner un localisateur (ou dans la liste déroulante Autres actions).

  • Vous pouvez rechercher un localisateur de stockage en cliquant sur le bouton Rechercher des localisateurs, puis sur le lien Sélectionner d'un localisateur de stockage, sur la page Sélectionner un localisateur.

  • Vous pouvez également assigner plusieurs éléments physiques en scannant une liste de codes-barres dans le champ Liste d'ID uniques ou en cliquant sur le bouton Parcourir et en parcourant une liste d'ID uniques.
