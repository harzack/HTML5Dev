Les libellés sont des blocs d'informations utilisés pour les objets physiques. Ils peuvent contenir plusieurs types d'informations, par exemple le nom ou l'ID unique de l'élément physique, du localisateur de stockage ou de l'utilisateur.

Vous pouvez générer ou imprimer un libellé unique pour un élément physique, localisateur ou utilisateur. Un groupe d'objets ou d'utilisateurs correspondant aux critères de recherche spécifiés est appelé un lot. La première étape consiste à générer un libellé ou lot de libellés. Vous pouvez ensuite imprimer les libellés nécessaires. Il est également possible de générer un libellé pour les éléments d'une collection ou d'un ensemble de résultats de recherche.

Les critères qui s'affichent lorsque vous recherchez ou générez des libellés par lots dépendent du type d'objet que vous avez spécifié. Pour chaque type d'objet, vous pouvez spécifier le nombre d'exemplaires que vous souhaitez générer. La liste ci-dessous présente de manière détaillée les paramètres pouvant être spécifiés pour chaque type d'objet :

  • Eléments physiques : Généré par, Date de génération, ID de stockage hors site, ID de transfert, Type d'élément physique, Infrastructure, Zone, Localisateur, Limites d'ID unique de/à, Papier de libellés et Type de libellé, et Position de départ pour l'impression des libellés.

  • Localisateurs : infrastructure, zone, localisateur et type de libellé.

  • Utilisateurs : utilisateurs ou groupes et type de libellé

Pour imprimer ou enlever des libellés par lots, vous pouvez rechercher les libellés d'élément physique ou de localisateur en spécifiant le papier et le type de libellés.

 

Note :
Si vous êtes un gestionnaire de documents (ou un utilisateur disposant de passe-droits) et que vous ne spécifiez aucun critère de recherche lors de la génération ou de l'impression de libellés par lots, ces derniers seront générés ou imprimés pour tous les éléments physiques de Content Server. Pour tout autre type d'utilisateur, les libellés seront générés pour les éléments physiques pour lesquels vous disposez de l'autorisation Voir. En outre, seuls les libellés que vous aurez générés pourront être imprimés.

 

Pour plus d'informations sur la création du contenu généré sur les libellés, consultez le guide de création de libellés du module OpenText Physical Objects.
