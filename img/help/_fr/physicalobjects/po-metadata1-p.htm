Pour modifier les métadonnées d'éléments physiques pour plusieurs éléments :

  1. Une fois en vue Collection, Résultats de recherche ou Parcourir, activez la case à cocher située en regard de chacun des éléments physiques dont vous souhaitez modifier les métadonnées.

  2. Cliquez sur le bouton Métadonnées d'éléments physiques ou dans la liste déroulante Autres actions.

  3. Sur la page Métadonnées d'éléments physiques, modifiez les paramètres appropriés.

  4. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Vous pouvez également modifier les métadonnées d'éléments physiques d'une collection en activant la case à cocher Collection complète dans la vue de détail d'une page Collections.
