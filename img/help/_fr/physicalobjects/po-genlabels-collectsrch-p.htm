Pour créer des libellés à partir d'une collection ou d'un ensemble de résultats de recherche :

  1. Depuis une collection ou un ensemble de résultats de recherche, activez la case à cocher située en regard de chaque élément physique pour lequel vous souhaitez générer un libellé.

  2. Dans la liste déroulante Autres actions, cliquez sur Générer des libellés.

  3. Sur la page Générer des libellés, sélectionnez le type de libellé que vous souhaitez générer pour les éléments de la collection dans la liste déroulante Type de libellé, indiquez le nombre d'exemplaires à générer dans le champ Nombre d'exemplaires, puis cliquez sur le bouton Générer.
