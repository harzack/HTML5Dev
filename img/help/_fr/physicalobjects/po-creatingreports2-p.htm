Pour créer un rapport de liste de requêtes :

  1. Dans le menu Entreprise, sélectionnez Espace de travail Objets physiques.

  2. Sur la page Espace de travail Objets physiques, cliquez sur le lien Rapports.

  3. Sur la page Rapports, cliquez sur le lien Rapport de liste de requêtes.

  4. Facultatif Dans la section Filtrer le rapport par de la page d'exécution des rapports WebReport, filtrez les résultats en fonction des options suivantes :

    • Générateur de requêtes

    • Dates de début et de fin de requête

    • Type de requête

    • ID de stockage hors site

    • Emplacement d'origine

    • Emplacement actuel

  5. Dans la section Trier le rapport par, activez la case correspondant à la méthode en fonction de laquelle trier le rapport, puis sélectionnez Croissante ou Décroissante dans la liste déroulante correspondante.

  6. Cliquez sur le bouton Exécuter le rapport.
