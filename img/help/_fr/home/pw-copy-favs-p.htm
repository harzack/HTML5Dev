Pour copier un favori dans un autre onglet :

  1. Dans la barre de menu générale, dans le menu Personnel, cliquez sur Favoris.

  2. Cliquez sur l'onglet dans lequel l'élément favori est situé.

  3. Cliquez sur l'icône Modifier/Organiser, .

  4. Pour chaque élément que vous souhaitez copier, cliquez sur un nouvel emplacement d'onglet dans la liste Onglet Favoris de l'élément.

  5. Cliquez sur Envoyer.
