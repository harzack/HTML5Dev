Avertissement
Lorsque vous modifiez un sondage après que les votes ont été enregistrés, tous les votes existants sont supprimés.

 

Pour modifier un sondage :

  1. Cliquez sur l'icône Fonctions du sondage, puis sélectionnez Modifier.

  2. Sur la page Sondage, saisissez un nom pour le sondage dans le champ Nom.

  3. Facultatif Pour ajouter des instructions pour aider les votants à remplir le sondage, saisissez-les dans le champ Instructions.

  4. Cliquez sur une date de début et une date de fin pour le sondage dans les listes déroulantes Date d'entrée en vigueur et Date d'expiration.

  5. Dans la section Sondage, procédez de la façon suivante :

    • Saisissez une question dans la zone de texte Question de la colonne Question. Pour ajouter plusieurs questions, cliquez sur le bouton Ajouter un choix situé en dessous de la colonne libellée Ligne. Pour supprimer une question, cliquez sur le bouton Supprimer un choix situé en dessous de la colonne libellée Ligne.

    • Saisissez des choix de réponse à la question dans la zone de texte Choix de la colonne Choix. Pour ajouter plusieurs réponses, cliquez sur Ajouter un choix de la colonne Choix pour chaque nouvelle réponse à entrer. Pour supprimer une réponse, cliquez sur le bouton Supprimer un choix de la colonne Choix.

    • Pour déterminer la méthode de vote, dans la zone Sélection de choix, sélectionnez l'une des cases d'option suivantes pour chaque question :

      • Case d'option

      • Case à cocher

  6. Facultatif Pour autoriser les votants à afficher les résultats ou à ajouter des commentaires, sélectionnez les cases à cocher Autoriser l'affichage des résultats avant le vote ou Autoriser les commentaires anonymes.

  7. Pour ajouter une image au sondage, cliquez sur le bouton Parcourir Content Server, naviguez jusqu'à une image, puis cliquez sur son lien Sélectionner. Pour ajouter du texte de remplacement à l'image sélectionnée, saisissez le texte dans le champ Texte ALT.

  8. Cliquez sur le bouton Envoyer pour enregistrer les paramètres. Cliquez sur le bouton Réinitialiser pour réinitialiser tous les champs modifiés.
