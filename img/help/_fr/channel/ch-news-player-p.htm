Pour activer un bandeau de nouvelles :

  1. Cliquez sur l'icône Fonctions d'un espace de travail ou d'un recueil, choisissez Propriétés, puis Présentation.

  2. Dans la section Bandeau de nouvelles, cliquez sur le bouton Sélectionner, puis cliquez sur le lien Sélectionner de la chaîne dont vous voulez assigner les titres au bandeau de nouvelles.

  3. Cliquez sur le bouton Mettre à jour.

 

Note :
Le bandeau de nouvelles ne s'affiche que s'il existe des nouvelles n'ayant pas encore expiré dans les chaînes assignées.

 
 

Astuce :
Cliquez sur le bouton Supprimer dans la section Bandeau de nouvelles pour supprimer les chaînes que vous ne voulez plus afficher dans le bandeau de nouvelles.
