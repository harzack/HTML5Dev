Pour ajouter des nouvelles à une chaîne :

Procédez de l'une des manières suivantes :

  • Cliquez sur l'icône Fonctions d'une chaîne et choisissez Créer une nouvelle. La page Ajouter : Nouvelle s'affiche.

    Si vous avez utilisé la fonction Créer une nouvelle pour un élément existant de Content Server, le champ Ajouter une pièce jointe Content Server contient déjà l'emplacement de l'élément.

  • Ouvrez la chaîne à laquelle vous voulez ajouter la nouvelle. Pour plus d'informations sur l'ouverture d'une chaîne, reportez-vous à la rubrique Pour ouvrir une chaîne.

    Cliquez sur le bouton Ajouter un élément de la barre d'outils de la fenêtre de recherche de chaîne. Sélectionnez Nouvelles. La page Ajouter : Nouvelle s'affiche.

Saisissez ensuite les informations suivantes :

  1. Sur la page Ajouter : Nouvelle, tapez un nom pour la nouvelle dans le champ Nom.

  2. Facultatif Saisissez une description de la nouvelle dans le champ Description.

  3. Cliquez sur une Date d'entrée en vigueur et une Date d'expiration dans les listes Mois, Jour, Année et Heure correspondantes. Si vous ne saisissez pas de valeurs pour les dates d'entrée en vigueur et d'expiration de cette nouvelle, les utilisateurs ne pourront pas la visualiser.

  4. Saisissez un titre pour la nouvelle dans le champ Titre. Si vous ne remplissez pas le champ Titre, le texte se trouvant dans le champ Nom sera utilisé comme titre.

  5. Facultatif Ajoutez des mises en évidence à la nouvelle. Vous pouvez entrer plusieurs lignes de texte. Chaque ligne du champ Mises en évidence correspondra à une puce sur la page de résumé de la chaîne.

  6. Saisissez le texte de votre nouvelle dans le champ Contenu. Vous pouvez également copier et coller du texte, des images et des liens à partir d'une page Web.

  7. Si vous voulez ajouter une image de Content Server à une nouvelle, cliquez sur le bouton Parcourir Content Server dans le champ Image. Recherchez l'image, puis cliquez sur son lien Sélectionner.

  8. Pour ajouter une description à l'image pour les utilisateurs qui seront dans l'impossibilité de la voir, par exemple, saisissez une description dans le champ Texte ALT.

  9. Pour ajouter une pièce jointe de Content Server à la nouvelle, cliquez sur le bouton Parcourir Content Server dans le champ Ajouter une pièce jointe Content Server. Recherchez l'élément, puis cliquez sur son lien Sélectionner.

  10. Pour ajouter une pièce jointe de votre ordinateur à la nouvelle, cliquez sur le bouton Parcourir dans le champ Ajouter une pièce jointe à partir du disque local. Recherchez l'élément, puis cliquez sur le bouton Ouvrir.

  11. Cliquez sur le bouton Ajouter pour enregistrer vos modifications et revenir à la page précédente. Cliquez sur le bouton Réinitialiser pour effacer tous les champs.
