Pour ajouter une chaîne :

  1. Dans le menu Ajouter un élément, cliquez sur Chaîne.

    La page Ajouter : Chaîne s'affiche.

  2. Le champ Nom est obligatoire. Les champs restants sont facultatifs.

  3. Si vous voulez joindre une image à votre nouvelle chaîne, cliquez sur le bouton Parcourir Content Server et recherchez cette image. Cliquez sur son lien Sélectionner.

  4. Saisissez une description de l'image dans le champ Texte ALT.

  5. Vous pouvez également spécifier les autres paramètres généraux nécessaires pour l'élément.

  6. Cliquez sur le bouton Ajouter.
