Il existe deux façons d'ajouter des documents dans Content Server :

  • En chargeant un fichier qui existe déjà sur votre ordinateur.

  • En créant un nouveau document dans Content Server.

     

    Note :
    La création d'un nouveau document nécessite l'utilisation du navigateur Web Microsoft Internet Explorer.

     

Lorsque vous ajoutez un document existant dans Content Server, vous pouvez spécifier un nom et une description, appliquer des catégories et des attributs et choisir l'emplacement de stockage du document dans Content Server. Si le document que vous ajoutez comporte plusieurs catégories associées au type de document, vous accédez à une seconde page dans laquelle vous pouvez choisir la catégorie à assigner au document. Par exemple, supposons que votre administrateur a associé des fichiers JPG et TIF à des catégories appelées « Image numérisée » et « Photo numérique ». Lorsque vous ajoutez des fichiers portant l'extension .jpg ou .tif, vous devez sélectionner la catégorie Image numérisée et/ou Photo numérique sur une seconde page après avoir ajouté l'élément.

 

Note :
Cette fonction n'est disponible que si votre administrateur a activé le processus d'Ajout avancé des éléments et si le type de document que vous ajoutez est un type MIME que votre administrateur a associé à plusieurs catégories. Vous êtes invité à choisir une catégorie ou plusieurs si diverses catégories sont associées au type MIME.

 

Lorsque vous ajoutez un nouveau document à Content Server, vous devez sélectionner un type de document, spécifier des informations et composer son contenu dans une application externe. Les types de documents que vous pouvez ajouter sont définis par l'administrateur mais peuvent comprendre des fichiers Microsoft Word, Excel ou PowerPoint. Après avoir ajouté le nouveau document, une version vierge s'ouvre dans le programme de bureau correspondant ; vous pouvez alors composer son contenu.

Si vous disposez des autorisations nécessaires, vous pouvez ajouter un document en tant que version principale ou secondaire du document original. Si vous ne disposez pas des autorisations nécessaires, vous ne pouvez pas ajouter un document en tant que version principale ou secondaire du document original. Pour plus d'informations sur la création de versions avancée, reportez-vous à la rubrique Utilisation des versions.

 

Note :
Pour plus d'informations sur l'ajout de documents à Content Server sur un ordinateur Macintosh avec Netscape, reportez-vous à la rubrique Ajout de fichiers Macintosh dans Content Server.

 

Si vous utilisez Microsoft Internet Explorer, vous pouvez également modifier des documents dans Content Server. Pour cela, vous devez lancer une session de modification. Une session de modification exécute automatiquement pour vous les opérations suivantes :

  • réservation du document et téléchargement d'un fichier temporaire sur votre ordinateur et

  • ouverture du fichier temporaire dans son programme de bureau associé afin que vous puissiez le modifier.

Une fois toutes les modifications apportées, vous pouvez terminer la session de modification. A l'issue de la session de modification, vous avez le choix entre les options suivantes :

  • enregistrer les modifications dans Content Server en tant que version secondaire du document original ;

  • enregistrer les modifications sur le fichier local de votre ordinateur, laissant ainsi le document dans Content Server tel quel ;

  • rejeter l'ensemble des modifications.

 

Note :
Lorsque vous travaillez dans une session de modification de Content Server, vous ne devez fermer le navigateur Web qu'après avoir terminé de modifier le document. Si vous fermez le navigateur sans enregistrer vos modifications, celles-ci seront perdues.

 
 

Astuce :
Pour afficher vos modifications dans Content Server, actualisez le navigateur Web lorsque vous terminez une session de modification. La case à cocher Actualiser le navigateur qui héberge cette session est sélectionnée par défaut. Laissez cette case à cocher sélectionnée pour actualiser le navigateur Web à la fin de votre session de modification.
