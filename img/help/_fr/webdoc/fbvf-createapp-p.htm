Pour créer une apparence dans un dossier virtuel :

  1. Naviguez jusqu'au dossier virtuel dans lequel vous souhaitez créer une apparence.

  2. Dans le menu Ajouter un élément, sélectionnez Apparence.

  3. Sur la page Ajouter : Apparence, dans le champ Nom, saisissez un nom pour votre nouvelle apparence.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  4. Facultatif Dans le champ Description, saisissez la description de la nouvelle apparence.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette apparence.

  6. Facultatif Pour placer l'apparence dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer l'apparence, puis cliquez sur son lien Sélectionner.

  7. Cliquez sur Ajouter.
