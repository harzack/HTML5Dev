Lorsqu'un dossier virtuel est créé, les utilisateurs disposant des autorisations Voir le contenu sur ce dossier virtuel peuvent l'utiliser pour parcourir ses facettes prédéfinies.

Une bannière de navigation apparaît au-dessus de la barre latérale quand :

  • un utilisateur sélectionne une facette parmi les facettes disponibles dans le panneau Filtre de contenu de la barre latérale ou

  • un utilisateur sélectionne un dossier virtuel

Bannière de navigation

La bannière de navigation s'affiche sur toute la page, au-dessus de la barre latérale. Elle s'affiche chaque fois que vous accédez à une vue de navigation à facettes du contenu de Content Server. Les options de la bannière de navigation changent de façon dynamique, en fonction de vos autorisations.

  • La bannière de navigation affiche l'intitulé Vue de dossier virtuel lorsqu'un utilisateur clique sur le nom d'un dossier virtuel. Cette vue affiche les noms des éléments dans Content Server qui sont conformes aux paramètres définis pour ce dossier virtuel.

    Les paramètres du dossier virtuel, définis par le créateur du dossier virtuel, peuvent être affichés dans le panneau Filtre de contenu de la barre latérale, en fonction du mode de configuration du dossier virtuel.

    En fonction de vos autorisations, la bannière de navigation Vue de dossier virtuel peut également afficher le lien Modifier le dossier virtuel. Il s'affiche uniquement si l'utilisateur est autorisé à modifier le dossier virtuel. Pour plus d'informations sur la modification d'un dossier virtuel, voir Pour réviser ou modifier les paramètres d'un dossier virtuel.

  • La bannière de navigation affiche le titre Vue dynamique lorsqu'un utilisateur sélectionne une facette dans le panneau Filtre de contenu de la barre latérale.

    La bannière de navigation Vue dynamique peut également afficher :

    • Le lien Supprimer tous les filtres. Il s'affiche lorsqu'un utilisateur sélectionne une facette dans le panneau Filtre de contenu de la barre latérale.

    • Le lien Enregistrer comme dossier virtuel. Il s'affiche uniquement si l'utilisateur est autorisé à créer un dossier virtuel. Pour plus d'informations sur la création d'un dossier virtuel, voir Pour créer un dossier virtuel.

Règles destinées aux utilisateurs qui créent et gèrent des dossiers virtuels

Seuls les utilisateurs auxquels l'administrateur a accordé des autorisations peuvent créer des dossiers virtuels. Si l'option de création ou de gestion d'un dossier virtuel n'est pas disponible, consultez votre administrateur.

Afin de faciliter la gestion des dossiers virtuels conçus pour être utilisés dans toute l'entreprise, l'administrateur a peut-être désigné un recueil de dossiers virtuels dans lequel vous devez stocker les dossiers virtuels que vous créez.

  1. Vous pouvez créer un dossier virtuel dans tout recueil dans lequel :

    • Vous êtes autorisé à créer un objet.

    • Vous pouvez créer un dossier.

       

      Note :
      Vous pouvez créer un dossier virtuel dans un autre dossier virtuel dans le menu Ajouter un élément ou à l'aide du champ Créer dans d'une page Ajouter : Dossier virtuel. Pour en savoir plus, voir Pour créer un dossier virtuel dans un autre dossier virtuel.

       

  2. Vous pouvez uniquement stocker un dossier virtuel, une apparence ou une vue personnalisée dans un dossier virtuel. Aucun autre élément ne peut être stocké dans un dossier virtuel.

  3. Vous pouvez stocker un dossier virtuel dans un autre dossier virtuel. Le sous-dossier virtuel s'affiche, par défaut, sous la forme d'un élément mis en valeur lorsqu'un utilisateur affiche le dossier virtuel parent. L'utilisateur autorisé à modifier le dossier virtuel peut modifier ce paramètre par défaut. Voir Pour réviser ou modifier les paramètres d'un dossier virtuel pour plus d'informations.

  4. Lorsqu'un dossier virtuel apparaît dans une liste de répertoires, il n'affiche aucune valeur de taille. Le nombre d'éléments dans le dossier virtuel ne s'affiche pas. Lorsque l'utilisateur clique sur le dossier virtuel, chaque élément affiché par le dossier virtuel affiche une valeur de taille.

  5. Lorsque vous créez un dossier virtuel pour vos utilisateurs, ils ne voient que l'emplacement du recueil dans lequel vous avez sélectionné les facettes enregistrées si vous sélectionnez la case à cocher Afficher l'emplacement sélectionné.

    Si cette case à cocher est sélectionnée, l'emplacement s'affiche dans le panneau Filtre de contenu de la barre latérale lorsque l'utilisateur final sélectionne ce dossier virtuel.

  6. Lorsque vous créez un dossier virtuel pour vos utilisateurs, ils ne voient que la liste des facettes sélectionnées pour la navigation à facettes enregistrée si vous sélectionnez la case à cocher Afficher les facettes sélectionnées.

    Si cette case à cocher est sélectionnée, les facettes sont répertoriées dans le panneau Filtre de contenu de la barre latérale lorsque l'utilisateur sélectionne ce dossier virtuel.

  7. Lorsque vous créez un dossier virtuel pour vos utilisateurs, ils ne voient que l'emplacement, dans Content Server, de chaque élément conforme aux critères du dossier virtuel si vous sélectionnez la case à cocher Afficher la colonne d'emplacement.

    Si cette case à cocher est sélectionnée, l'emplacement de chaque élément s'affiche dans le mode navigation principal sous le titre de colonne Emplacement. Si cette case à cocher n'est pas sélectionnée, la colonne Emplacement n'apparaît pas.

  8. Si vous accordez à un utilisateur l'autorisation de modifier le dossier virtuel que vous avez créé et si cet utilisateur n'est pas autorisé à afficher l'une des facettes définies pour le dossier virtuel, lorsque l'utilisateur affiche l'onglet Spécifique de votre dossier virtuel, il voit uniquement les facettes qu'il est autorisé à voir. Si l'utilisateur tente de modifier ce dossier virtuel, un message l'informe qu'il n'a pas accès à une ou plusieurs des facettes définissant le dossier virtuel.

    Autrement dit, un utilisateur auquel vous avez accordé l'autorisation de modifier votre dossier virtuel peut modifier la définition du dossier virtuel, mais pas afficher la liste complète des facettes définies pour ce dossier virtuel.

    Dans ce cas, l'utilisateur peut supprimer toutes les facettes dont il n'est pas autorisé à voir le contenu, à partir de la définition du dossier virtuel.

    OpenText vous recommande de vérifier que les utilisateurs auxquels vous accordez l'autorisation de modifier votre dossier virtuel disposent de l'autorisation Voir le contenu pour toutes les facettes qui définissent le dossier virtuel.

  9. Il est possible d'accorder à un utilisateur l'autorisation Voir le contenu pour un dossier virtuel, puis de définir les facettes de ce dossier virtuel que l'utilisateur n'est pas autorisé à voir. Dans ce cas, l'utilisateur pourra accéder au dossier virtuel, mais celui-ci sera vide.

     

    Attention
    Etant donné que les autorisations des dossiers virtuels sont différentes de celles des facettes qui les définissent, vous pouvez, en théorie, autoriser un utilisateur à modifier votre dossier virtuel, sans qu'il puisse voir le contenu des facettes qui définissent ce dossier virtuel. Dans ce cas, si un utilisateur autorisé à modifier le dossier virtuel le met à jour en mode édition, les facettes qui définissent ce dossier virtuel seront supprimées et le dossier virtuel sera vide.
