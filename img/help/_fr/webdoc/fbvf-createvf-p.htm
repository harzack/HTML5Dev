Pour créer un dossier virtuel :

  1. Il existe deux façons de créer un dossier virtuel. Choisissez l'une des options suivantes :

    • Pour créer un dossier virtuel à partir du menu Ajouter un élément :

      1. Naviguez jusqu'au recueil dans lequel vous souhaitez créer le dossier virtuel.

      2. Dans le menu Ajouter un élément, sélectionnez Dossier virtuel.

    • Pour créer un dossier virtuel en sélectionnant des facettes :

      1. Naviguez jusqu'à l'emplacement de Content Server dans lequel vous souhaitez démarrer la navigation à facettes.

      2. A l'aide du panneau Filtre de contenu de la barre latérale, sélectionnez les facettes à appliquer au dossier virtuel.

        Si la barre latérale n'est pas visible, voir Pour personnaliser le bandeau du filtre de contenu.

      3. Une fois la sélection des facettes terminée, dans la barre de navigation Vue dynamique, cliquez sur Enregistrer comme dossier virtuel.

  2. Sur la page Ajouter : Dossier virtuel, dans le champ Nom, saisissez un nom pour le dossier virtuel.

  3. Facultatif Dans le champ Description, saisissez la description du dossier virtuel.

  4. Les champs Emplacement sélectionné et Facettes sélectionnées ne sont pas modifiables dans cette page.

    Si vous avez créé le dossier virtuel à partir du menu Ajouter un élément ces champs indiquent : Aucun emplacement sélectionné et Aucune facette sélectionnée. Ces valeurs seront différentes une fois qu'un emplacement est sélectionné et que des facettes sont appliquées au dossier virtuel.

    Si vous avez créé le dossier virtuel en sélectionnant des facettes, le champ Emplacement sélectionné indique l'emplacement du recueil à partir duquel vous avez sélectionné ces facettes. Le champ Facettes sélectionnées répertorie les facettes sélectionnées.

  5. Facultatif Sélectionnez la case à cocher Afficher l'emplacement sélectionné pour que les utilisateurs qui accèdent au dossier virtuel puissent voir l'emplacement du recueil à partir duquel vous avez sélectionné les facettes appliquées à ce dossier virtuel.

    Cette case est cochée par défaut.

  6. Facultatif Sélectionnez la case à cocher Afficher les facettes sélectionnées pour que les utilisateurs qui accèdent au dossier virtuel puissent voir les facettes qui constituent le dossier virtuel.

    Cette case est cochée par défaut.

  7. Facultatif Sélectionnez la case à cocher Afficher la colonne d'emplacement pour que les utilisateurs qui accèdent au dossier virtuel puissent voir une colonne qui présente l'emplacement, dans Content Server, de chaque élément conforme aux critères du dossier virtuel.

    Cette case est cochée par défaut.

  8. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce dossier virtuel.

  9. Facultatif Pour placer le dossier dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer le dossier virtuel, puis cliquez sur son lien Sélectionner.

     

    Note :
    Vous pouvez créer un dossier virtuel dans un autre dossier virtuel à l'aide du champ Créer dans d'une page Ajouter : Dossier virtuel. Naviguez jusqu'au dossier virtuel à désigner comme emplacement de stockage du sous-dossier virtuel, puis cliquez sur le lien Sélectionner du dossier virtuel parent.

     

  10. Cliquez sur Ajouter.
