Pour ajouter un document existant :

  1. Dans le menu Ajouter un élément, sélectionnez Document.

  2. Sur la page Ajouter : Document, dans la section Document, sélectionnez le bouton radio Existant. Cliquez sur le bouton Parcourir pour rechercher le fichier que vous souhaitez ajouter, puis cliquez sur le bouton Ouvrir.

     

    1. Le bouton radio Existant ne s'affiche que si vous utilisez Internet Explorer.

    2. Pour ajouter un nouveau document, sélectionnez le bouton radio Nouveau, puis suivez les instructions dans Pour ajouter un nouveau document.

     

  3. Saisissez un nom pour le document dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

  4. Facultatif Dans le champ Description, saisissez une description pour le document.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

  5. Si vous travaillez sur un ordinateur Macintosh qui n'exécute pas le système d'exploitation OSX et que le fichier que vous créez contient des ressources importantes que vous souhaitez préserver, sélectionnez la case à cocher MacBinary (définir avant de choisir un fichier).

     

    Note :
    Pour obtenir plus d'informations sur l'ajout de fichiers Macintosh, reportez-vous à la section Ajout de fichiers Macintosh dans Content Server.

     

  6. Dans le champ Contrôle de version, choisissez un type de création de versions en cliquant sur le bouton radio Standard - création linéaire de versions ou Avancée - création de versions principales/secondaires.

  7. Facultatif Si vous appliquez un classement à ce document, dans la liste Classements, sélectionnez Parcourir les classements ou Mes favoris.

    1. Dans la boîte de dialogue qui apparaît, sélectionnez le classement que vous souhaitez appliquer au document.

    2. Assurez-vous que la case à cocher Action située en regard du classement est bien activée.

    3. Cliquez sur Envoyer.

  8. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce document.

  9. Facultatif Pour placer le document dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer le document, puis cliquez sur son lien Sélectionner.

  10. Cliquez sur Ajouter.

  11. Si la page Modifier les attributs s'affiche après l'ajout du document, sélectionnez la case à cocher de chaque catégorie à laquelle vous souhaitez assigner le document, puis cliquez sur Envoyer.

 

Astuce :
Si votre administrateur l'a mis à votre disposition, vous pouvez également ajouter un document à Content Server en cliquant sur le bouton Ajouter un document sur la page d'un espace de travail.
