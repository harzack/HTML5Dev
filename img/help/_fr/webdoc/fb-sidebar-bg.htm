Vous accédez à la fonction de navigation à facettes via le panneau Filtre de contenu de la barre latérale. Le paramètre par défaut lors de l'installation active le bandeau pour tous les utilisateurs. Il s'affiche à gauche de chaque page dans laquelle il est activé. Pour plus d'informations sur la personnalisation de votre bandeau, reportez-vous à la rubrique Pour personnaliser le bandeau du filtre de contenu.

Le panneau Filtre de contenu de la barre latérale filtre dynamiquement les facettes de tous les éléments contenus dans la vue dans laquelle vous l'utilisez. Lorsque vous parcourez Content Server, le panneau Filtre de contenu est en permanence mis à jour pour afficher les facettes disponibles à partir de cet emplacement. Ceci inclut les facettes appliquées aux éléments de la vue et aux sous-éléments.

Les éléments constituant le panneau Filtre de contenu sont les suivants :

  • La zone de texte Filtrer par nom et le bouton Envoyer la recherche.

  • Le nom de la vue actuelle qui indique si vous vous trouvez dans une vue dynamique ou dans un recueil comme un dossier ou la vue de l'espace de travail Entreprise.

    Dans une vue dynamique, vous pouvez également visualiser la Bannière de navigation dans le dossier virtuel.

  • La liste des filtres de facette appliqués à l'emplacement ainsi que des liens pour supprimer les filtres. La liste n'apparaît que si un ou plusieurs filtres de facette sont appliqués.

  • Tous les filtres que vous pouvez appliquer à cet emplacement. Chaque filtre indique son nom et une liste de valeurs de facette. La facette Propriétaire, par exemple, peut apparaître. Sous la facette Propriétaire se trouve une liste de tous les ID utilisateur, les noms des personnes, qui détiennent des éléments contenus à cet emplacement. Cliquez sur l'un des ID utilisateur ou nom, le mode navigation est alors actualisé pour afficher tous les éléments détenus par cet ID utilisateur ou nom.

    La liste des facettes qui s'affiche dans cette section dépend de la configuration de la navigation à facette par vos Gestionnaires des connaissances. L'utilisateur peut réduire ou développer chaque filtre en cliquant sur sa barre de titre. Lorsque l'utilisateur réduit ou développe un filtre, l'état de ce dernier est enregistré et devient votre paramètre. Vos paramètres sont conservés lorsque vous naviguez dans Content Server puis enregistrés lorsque vous vous déconnectez.

    Chaque facette disponible pour un emplacement affiché se présente sous la forme d'une barre pouvant être réduite dans Filtre de contenu.

     

    Note :
    Bien qu'elle n'appartienne pas au panneau Filtre de contenu de la barre latérale, la Bannière de navigation dans le dossier virtuel s'affiche par défaut lorsqu'un utilisateur sélectionne une facette à afficher. Si la bannière de navigation est visible, les options disponibles dépendent de vos autorisations. Par défaut, le lien Supprimer tous les filtres est disponible sur la bannière de navigation. Pour plus d'informations sur la bannière de navigation dans le dossier virtuel, voir Utilisation des dossiers virtuels.

     

Personnalisation et utilisation du bandeau du filtre de contenu

Certains emplacements dans Content Server n'afficheront jamais le bandeau et vous ne pourrez pas l'activer. La barre latérale s'applique généralement aux pages Dossiers, Projets et Espace de travail.

Vous pouvez utiliser le bandeau Filtre de contenu de deux manières. La première consiste à effectuer une recherche à l'aide de la zone de texte Filtrer par nom dans laquelle vous saisissez les mots à rechercher. La seconde consiste à filtrer votre recherche en cliquant sur les facettes disponibles. Reportez-vous à Pour utiliser le bandeau Filtre de contenu ci-dessous pour plus d'informations.

Sauf si vous naviguez dans votre espace de travail personnel, le bandeau n'affiche pas les éléments trouvés dans les espaces de travail personnels.

Pour plus d'informations sur vos paramètres de Filtre de contenu, reportez-vous à la rubrique Pour personnaliser le bandeau du filtre de contenu.

Utilisation des colonnes personnelles

Chaque utilisateur peut être autorisé à personnaliser ses paramètres de colonne dans sa Vue du parcours. Si vous ne disposez pas des autorisations nécessaires, contactez votre administrateur.

Toutes les conditions suivantes doivent être remplies pour qu'une colonne s'affiche dans la zone Colonnes disponibles :

  1. L'utilisateur doit être autorisé à afficher l'objet colonne.

  2. L'utilisateur doit disposer de l'autorisation correspondant à la source de données de colonne.

  3. Le paramètre Disponibilité de la colonne de l'objet colonne doit être défini sur Disponible partout.

  4. La colonne ne figure pas dans la liste Colonnes affichées.

  5. La colonne ne figure pas dans la liste des colonnes globales à afficher.
