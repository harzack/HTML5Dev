Pour utiliser la zone de texte du bandeau du filtre de contenu :

  1. Accédez à l'emplacement contenant l'élément que vous souhaitez rechercher. Si vous ne savez pas quel dossier, projet ou espace de travail contient l'élément, sélectionnez Entreprise dans la barre de menu générale, puis Espace de travail.

  2. Dans la zone de texte Filtrer par nom, saisissez le nom (ou une partie du nom) de l'élément à rechercher.

  3. Cliquez sur le bouton Envoyer la recherche situé à droite de la zone de saisie.

  4. Si un ou des éléments sont trouvés, la Vue du parcours contient désormais la liste de cet ou ces éléments. Cliquez sur le nom d'un élément pour afficher ce dernier.

  5. Si un élément avec ce nom n'est pas trouvé dans le conteneur dans lequel vous effectuez la recherche, un message d'erreur s'affiche ainsi qu'un lien Afficher tous les éléments. Cliquez sur Afficher tous les éléments pour revenir à la vue précédente.

Pour utiliser les filtres du bandeau du filtre de contenu :

  1. Accédez à l'emplacement contenant l'élément que vous souhaitez rechercher. Si vous ne savez pas quel dossier, projet ou espace de travail contient l'élément, sélectionnez Entreprise dans la barre de menu générale, puis Espace de travail.

  2. Sélectionnez une valeur de facette sur laquelle vous souhaitez filtrer tous les éléments et sous-éléments de cet emplacement.

  3. Si un lien Plus... apparaît sous la liste des valeurs d'une facette, cela signifie que toutes les valeurs disponibles de cette facette ne sont pas affichées. Cliquez sur le lien Plus... pour afficher une boîte de dialogue dans laquelle toutes les valeurs de la facette peuvent être sélectionnées. Activez les cases à cocher des valeurs de facette sur lesquelles vous souhaitez filtrer les éléments et sous-éléments. Cliquez sur Sélectionner pour appliquer les valeurs de facette sélectionnées pour filtrer la vue actuelle.

  4. Après avoir sélectionné la première valeur de facette sur laquelle vous souhaitez filtrer tous les éléments de cet emplacement, cliquez sur la valeur de facette suivante à utiliser comme filtre.

  5. Lorsque vous sélectionnez un filtre, celui-ci disparaît du bandeau et apparaît sous forme de texte sous la zone de texte Filtrer par nom. Pour supprimer chaque filtre séparément, cliquez sur le X situé à droite de chaque valeur de facette. Pour supprimer tous les filtres de facette simultanément, cliquez sur le lien Supprimer tous les filtres.

  6. Cliquez ensuite sur des filtres de facette jusqu'à ce que les résultats de votre recherche soient suffisamment affinés et que le mode navigation affiche une petite liste de résultats correspondant à tous les filtres de la recherche. Cliquez sur le nom d'un élément pour l'ouvrir. Pour revenir au début et recommencer, cliquez sur le lien Supprimer tous les filtres.
