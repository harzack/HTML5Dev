Pour créer un dossier virtuel dans un autre dossier virtuel :

  • Lorsque le dossier virtuel « A » se trouve dans le dossier virtuel « B », le dossier virtuel « A » correspond au sous-dossier virtuel et le dossier virtuel « B » correspond au dossier virtuel parent.

    Vous pouvez créer un sous-dossier virtuel dans un dossier virtuel parent de deux façons. Choisissez l'une des options suivantes :

    • Pour créer un sous-dossier virtuel à partir du menu Ajouter un élément :

      1. Cliquez sur l'icône Fonctions du dossier virtuel parent. Dans le menu Fonctions, sélectionnez Propriétés. Sur la page Propriétés du dossier virtuel parent, sélectionnez l'onglet Spécifique.

      2. Dans le champ Facettes sélectionnées, sélectionnez le lien Mettre à jour le contenu dynamique.

      3. Dans le menu Ajouter un élément, sélectionnez Dossier virtuel.

      4. Voir Pour créer un dossier virtuel pour plus d'instructions sur comment compléter la page Ajouter : Dossier virtuel.

      5. Cliquez sur Revenir à l'enregistrement des modifications, puis sur Mettre à jour.

    • Pour créer un sous-dossier virtuel à l'aide du champ Créer dans, pendant la création du sous-dossier virtuel comme décrit dans la Pour créer un dossier virtuel :

      1. Dans le champ Créer dans, cliquez sur Parcourir Content Server....

      2. Naviguez jusqu'au dossier virtuel parent à désigner comme emplacement de stockage du sous-dossier virtuel, puis cliquez sur le lien Sélectionner du dossier virtuel parent.
