Pour réviser ou modifier les paramètres d'un dossier virtuel :

  1. Naviguez jusqu'au recueil de Content Server contenant le dossier virtuel.

  2. Il existe deux façons pour les utilisateurs autorisés de réviser ou modifier les paramètres d'un dossier virtuel. Procédez de l'une des manières suivantes :

    • Dans le menu Fonctions du dossier virtuel, sélectionnez Propriétés, puis Spécifique.

    • Cliquez sur le nom du dossier virtuel. Sur la page <nom_du_dossier_virtuel>, dans la bannière de navigation, cliquez sur le lien Modifier le dossier virtuel.

    Vous êtes à présent dans l'onglet Spécifique de la page Propriétés du dossier virtuel.

  3. Le champ Emplacement sélectionné indique l'emplacement à partir duquel le dossier virtuel commence la navigation des facettes. Ce champ n'est pas modifiable.

  4. Le champ Facettes sélectionnées indique les facettes affichées par le dossier virtuel. Lorsque vous modifiez ce champ en modifiant les facettes sélectionnées, un indicateur visuel apparaît en regard de Facettes sélectionnées dans l'onglet Spécifique. Si vous disposez des autorisations nécessaires, ce champ peut être modifié à l'aide du lien Mettre à jour le contenu dynamique :

    1. Lorsque vous sélectionnez Mettre à jour le contenu dynamique, vous voyez que la bannière de navigation a changé de couleur et que son titre indique Vue Modifier le dossier virtuel.

      Si vous êtes autorisé à modifier ce dossier virtuel, mais pas à voir le contenu pour visualiser toutes les facettes qui constituent ce dossier virtuel, le lien Mettre à jour le contenu dynamique ne s'affiche pas. A la place, un message vous indique que le dossier virtuel doit être mis à jour avec les facettes sélectionnées modifiées pour que le contenu dynamique puisse être mis à jour.

       

      Attention
      OpenText vous recommande d'effectuer uniquement la mise à jour du dossier virtuel si vous êtes certain que les facettes non visibles peuvent influer sur la définition de ce dossier virtuel.

       

    2. Une fois les modifications apportées au dossier virtuel, sélectionnez Revenir à l'enregistrement des modifications.

       

      Note :
      Si vous cliquez sur Revenir à l'enregistrement des modifications, vos modifications ne sont pas enregistrées. Vous devez enregistrer vos modifications à l'étape suivante. Si vous cliquez sur Revenir à l'enregistrement des modifications, vous revenez à l'onglet Spécifique.

       

    3. Si vous avez apporté des modifications à la Vue Modifier le dossier virtuel, vous ne pouvez plus voir le lien Mettre à jour le contenu dynamique.

       

      Note :
      Pour enregistrer les modifications apportées à la Vue Modifier le dossier virtuel, vous devez sélectionner Mettre à jour dans l'onglet Spécifique.

       

  5. Facultatif Sélectionnez la case à cocher Afficher l'emplacement sélectionné pour vérifier que chaque utilisateur accédant au dossier virtuel peut voir l'emplacement à partir duquel le dossier virtuel commence la navigation des facettes.

  6. Facultatif Sélectionnez la case à cocher Afficher les facettes sélectionnées pour vérifier que chaque utilisateur accédant au dossier virtuel peut voir les noms des facettes affichées dans le dossier virtuel.

  7. Facultatif Sélectionnez la case à cocher Afficher la colonne d'emplacement pour vérifier que chaque utilisateur accédant au dossier virtuel peut voir l'emplacement, dans Content Server, de chaque élément affiché dans le dossier virtuel.

  8. Cliquez sur Mettre à jour.

  9. Facultatif Si votre dossier virtuel est stocké dans un dossier virtuel parent, il figure comme élément mis en valeur de ce dossier virtuel parent par défaut. Pour modifier cela :

     

    Important :
    Si vous modifiez le paramètre par défaut, vous devrez accéder au sous-dossier virtuel à partir de la page Configurer du dossier virtuel parent. Pour plus d'informations sur la page Configurer d'un dossier virtuel, voir Pour configurer un dossier virtuel.

     

    1. Dans le menu Fonctions du sous-dossier virtuel, sélectionnez Propriétés, puis Général.

    2. Dans l'onglet Général, dans le champ Afficher, sélectionnez Liste, Mis en valeur ou Masqué.

    3. Cliquez sur Mettre à jour.
