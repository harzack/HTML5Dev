Pour créer une génération de document :

  1. Cliquez sur l'icône Fonctions d'un document et choisissez Créer une génération.

  2. Sur la page Ajouter : Génération, dans le champ Nom, saisissez un nom pour la génération.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom de la génération dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description de la génération en cours de création.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description de la génération dans les autres langues activées.

  4. Facultatif Dans la liste Version, si vous voulez diriger la génération vers une version précédente, sélectionnez le numéro de la version précédente.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette génération.

  6. Facultatif Si vous souhaitez placer la génération à un autre emplacement que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel vous souhaitez placer la génération, puis cliquez sur son lien Sélectionner.

  7. Cliquez sur Ajouter.
