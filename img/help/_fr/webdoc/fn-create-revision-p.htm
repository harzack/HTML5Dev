Pour créer une révision :

  1. Cliquez sur l'icône Fonctions du document composite et choisissez Créer une révision.

  2. Sur la page Créer une révision, saisissez un nom dans le champ Nom de la révision.

  3. Cliquez sur Envoyer.

 

Note :
Vous ne pouvez pas créer de révision si le document composite ou des sous-éléments de ce document composite sont réservés par un autre utilisateur.
