Pour réorganiser les éléments dans un document composite :

  1. Cliquez sur l'icône Fonctions d'un document composite et choisissez Réorganiser.

  2. Sur la page Réorganiser le contenu, dans le champ Nouveau de chaque élément dont vous voulez modifier l'ordre, saisissez un nombre entier positif.

  3. Facultatif Sélectionnez la case à cocher Document principal de l'élément que vous souhaitez traiter comme document principal.

  4. Facultatif Sélectionnez la case à cocher Renumérotation séquentielle pour réorganiser les autres éléments dans le document composite.

  5. Cliquez sur Mettre à jour.
