Page Vue d'ensemble d'un document

La page Vue d'ensemble d'un document fournit des informations sur le document. La page Vue d'ensemble d'un document s'affiche par défaut lorsque vous cliquez sur le lien d'un document ou sur un lien abrégé dans un e-mail. Votre administrateur peut modifier le comportement par défaut de Content Server de façon à empêcher l'accès automatique à la page Vue d'ensemble d'un document. La page Vue d'ensemble d'un document est également accessible à l'aide du menu Fonctions d'un document.

Les informations contenues sur la page Vue d'ensemble d'un document peuvent inclure une description et des informations sur la version. Vous pouvez également effectuer plusieurs fonctions dans le document à partir de la page Vue d'ensemble. Ces fonctions sont :

  • Ouverture d'un document : pour ouvrir le document, cliquez sur Modifier. Pour plus d'informations sur l'ouverture d'un document, voir Accès aux documents et aux documents texte.

  • Téléchargement d'un document : pour télécharger un document, cliquez sur Télécharger. Pour plus d'informations sur le téléchargement d'un document, voir Accès aux documents et aux documents texte.

  • Envoi par e-mail d'un lien vers un document : pour envoyer par e-mail un lien vers un document, cliquez sur Envoyer ce lien par e-mail. Pour plus d'informations sur l'envoi par e-mail d'un lien vers un document, voir Attribution de surnoms aux éléments.

  • Modification d'un document : pour modifier le document, cliquez sur Modifier. Pour plus d'informations sur la modification de documents, voir Ajout et modification de documents.

  • Ajout de versions : pour ajouter une version, cliquez sur Ajouter une version. Pour de plus amples informations sur les versions, reportez-vous à la rubrique Utilisation des versions.

  • Accès aux versions précédentes : pour accéder à une version précédente, sélectionnez le menu Autres actions. Dans le menu Propriétés, sélectionnez Versions. Pour de plus amples informations sur les versions, reportez-vous à la rubrique Utilisation des versions.

  • Ajout d'une évaluation à un document : pour ajouter une évaluation à un document, cliquez sur Evaluer cet élément. Pour plus d'informations sur les évaluations, voir Utilisation des évaluations.

  • Application de notifications à un document : pour appliquer des notifications à un document, cliquez sur Me prévenir des changements. Pour plus d'informations sur les notifications, voir Utilisation des notifications.

  • Accès aux pages Propriétés d'un document : pour accéder aux pages Propriétés d'un document, cliquez sur Afficher les propriétés. Pour plus d'informations sur la page Propriétés, reportez-vous à la rubrique Affichage et modification des propriétés d`un élément.

  • Modification de la description d'un document : pour ajouter ou modifier la description d'un document, cliquez sur Modifier la description.

     

    Astuce :
    Le champ de description d'un document est également accessible dans le menu Fonctions du document. Sur la page Propriétés du menu Fonctions, sélectionnez Général. Dans l'onglet Général de la page Propriétés, vous pouvez ajouter ou modifier le champ Description.

     

Les fonctions auxquelles vous pouvez accéder sur la page Vue d'ensemble d'un document dépendent de vos autorisations et du type de création de versions (Standard ou Avancé) définis par l'administrateur. La création de versions avancée modifie les fonctions disponibles, selon que la dernière version du document est une version principale ou secondaire. Par exemple, si la dernière version d'un document est une version secondaire, vous ne pouvez pas modifier la version principale qui y est associée ; vous pourrez uniquement modifier et ajouter des versions à la version secondaire actuelle. Pour plus d'informations sur la création de versions standard et avancée, reportez-vous à la rubrique Utilisation des versions.

 

Note :
Les utilisateurs en lecture seule peuvent consulter les documents de version avancée par l'intermédiaire de la page Vue d'ensemble d'un document uniquement si une version principale existe.

 

Page Vue d'ensemble de la version

La page Vue d'ensemble de la version s'affiche par défaut lorsque vous cliquez sur le lien d'une version dans l'onglet Versions de la page Propriétés d'un document. Vous pouvez également y accéder à l'aide du menu Fonctions d'une version. La page Vue d'ensemble de la version fournit une description de la version. Elle vous permet d'effectuer un grand nombre de fonctions dans la version. Ces fonctions sont :

  • Ouverture d'une version. Pour en savoir plus, voir Utilisation des versions.

  • Téléchargement d'une version. Pour en savoir plus, voir Utilisation des versions.

  • Accès aux pages Propriétés d'une version. Pour en savoir plus, voir Affichage et modification des propriétés d`un élément.
