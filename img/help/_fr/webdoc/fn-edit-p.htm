Pour modifier un document :

  1. Cliquez sur le lien Modifier du document à modifier.

     

    Note :
    Cette procédure nécessite l'utilisation du navigateur Web Microsoft Internet Explorer. Le lien Modifier s'affiche uniquement dans Internet Explorer.

     

  2. Une fois le document ouvert dans son programme de bureau associé, modifiez son contenu et fermez-le.

  3. Dans la boîte de dialogue Terminer la session de modification, cliquez sur l'une des cases d'option suivantes :

    • Enregistrer les modifications dans Content Server, pour ajouter le document modifié dans Content Server en tant que version secondaire du document d'origine.

    • Enregistrer une copie sur votre bureau, qui vous permet de spécifier un nouveau nom pour le document et de l'enregistrer sur votre ordinateur.

    • Rejeter les modifications, pour fermer le document sans l'enregistrer.

  4. Facultatif Pour afficher vos modifications dans Content Server, vous devez actualiser le navigateur Web lorsque vous terminez votre session de modification. Si vous ne souhaitez pas actualiser le navigateur Web, décochez la case Actualiser le navigateur qui héberge cette session.

 

Note :
Reportez-vous à l'aide en ligne de OpenText Explorer pour plus d'informations sur la modification de documents en utilisant OpenText Explorer.

 
 

Astuce :
Vous pouvez également modifier un document à partir de sa page Vue d'ensemble. Dans le menu Fonctions d'un document, sélectionnez Vue d'ensemble. Sur la page Vue d'ensemble d'un document, cliquez sur Modifier.
