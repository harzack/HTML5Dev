Un document est un fichier électronique que vous chargez de votre ordinateur vers la base de données Content Server. Les feuilles de calcul, les mémos, les pages HTML et les fichiers audio et image sont des exemples de documents. Les fonctions Content Server vous permettent d'ajouter des documents à Content Server, de modifier leur contenu, de les ouvrir, de les visualiser et de gérer des versions et des générations.

Un document texte est un type de document contenant du texte brut (un fichier créé dans le Bloc-notes Windows par exemple). La majorité des fonctions Content Server disponibles pour les autres types de documents le sont également pour les documents texte. Vous pouvez par exemple les ajouter à Content Server et modifier leur contenu.
