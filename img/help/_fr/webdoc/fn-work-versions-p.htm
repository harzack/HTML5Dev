Pour utiliser des versions :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Versions.

  2. Dans l'onglet Versions de la page Propriétés de l'élément, effectuez l'une des opérations suivantes :

    1. Facultatif Pour ouvrir une version, cliquez sur l'icône Fonctions de la version et choisissez Ouvrir.

    2. Facultatif Pour afficher une version, cliquez sur l'icône Fonctions de la version, choisissez Télécharger, puis enregistrez le fichier sur votre ordinateur.

    3. Facultatif Pour télécharger une version, cliquez sur l'icône Fonctions de la version et choisissez Ouvrir.

    4. Facultatif Pour supprimer une version, cliquez sur l'icône Fonctions de la version, choisissez Supprimer, puis cliquez sur le bouton OK.

    5. Facultatif Pour verrouiller ou déverrouiller une version, cliquez sur l'icône Fonctions de la version et choisissez Verrouiller ou Déverrouiller.

       

      Note :
      Les versions verrouillées ne peuvent pas être supprimées.
