Pour transformer une version en version principale :

  1. Cliquez sur l'icône Fonctions d'un document, sélectionnez Propriétés, puis Versions.

  2. Dans l'onglet Versions de la page Propriétés du document, cliquez sur l'icône Fonctions de la version.

  3. Dans le menu Fonctions de la version, sélectionnez Transformer en version principale.

     

    Note :
    L'option de menu Transformer en version principale est disponible uniquement si le document a été créé avec le contrôle de version Avancée - création de versions principales/secondaires.

     

  4. Sur la page Transformer en version principale :

    1. Facultatif Dans le champ Description, saisissez une description pour cette version principale.

      Le champ Description de la page Transformer en version principale vous permet de fournir une description de la version ajoutée. Il ne s'agit pas de la description de l'objet sur lequel repose cette version. Les descriptions de version ne sont pas multilingues.

    2. Le champ Création de version est fourni uniquement à titre informatif. Il décrit la modification de version effectuée lorsque vous cliquez sur Envoyer.

    3. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce document.

  5. Cliquez sur Envoyer.
