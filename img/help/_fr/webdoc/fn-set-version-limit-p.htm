Pour configurer une limite de version :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Infos spécifiques.

  2. Dans l'onglet Spécifique de la page Propriétés de l'élément, saisissez un nombre entier supérieur à 0 dans le champ Nbre max. de versions.

  3. Cliquez sur Mettre à jour.

 

Astuce :
Pour que Content Server ne conserve que la version la plus récente de l'élément, saisissez 1 dans le champ Nbre max de versions.
