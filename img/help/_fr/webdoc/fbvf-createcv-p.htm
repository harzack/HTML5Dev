Pour créer une vue personnalisée dans un dossier virtuel :

  1. Naviguez jusqu'au dossier virtuel dans lequel vous souhaitez créer une vue personnalisée.

  2. Dans le menu Ajouter un élément, sélectionnez Vue personnalisée.

  3. Sur la page Ajouter : Vue personnalisée, procédez de l'une des façons suivantes :

    • Cliquez sur le bouton radio Existant, puis sur le bouton Parcourir pour accéder au fichier HTML que vous avez créé et qui définit la vue personnalisée. Cliquez ensuite sur Ouvrir.

    • Cliquez sur le bouton radio Modèle, puis sur le bouton Parcourir Content Server... pour accéder au fichier de modèle qui définit la vue personnalisée. Cliquez ensuite sur Ouvrir.

  4. Dans le champ Nom, saisissez un nom pour la vue personnalisée.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

  5. Facultatif Dans le champ Description, saisissez une description pour la vue personnalisée.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

  6. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à la vue personnalisée.

  7. Facultatif Pour placer la vue personnalisée dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer la vue personnalisée, puis cliquez sur son lien Sélectionner.

  8. Cliquez sur Ajouter.

 

Note :
Si le fichier HTML renvoie à d'autres fichiers (par exemple, des fichiers image), vous devez ajouter ces fichiers séparément.

Pour plus d'informations sur la modification d'une vue personnalisée, voir Création et gestion des affichages personnalisés.
