Important :
Tout utilisateur disposant de l'autorisation Voir le contenu pour un dossier virtuel afin d'afficher ce dossier virtuel. Toutefois, vous ne pouvez créer des dossiers virtuels que si l'administrateur vous a accordé l'autorisation de créer des dossiers virtuels. Si vous ne pouvez pas créer de dossier virtuel, contactez votre administrateur.

 

Un dossier virtuel est un objet de Content Server qui définit :

  1. une sélection de facettes et

  2. un emplacement de Content Server dans lequel cette sélection va être appliquée.

Les utilisateurs disposant de l'autorisation Voir le contenu sur un dossier virtuel peuvent le sélectionner pour afficher une Vue de dossier virtuel pour la navigation à facettes définie par l'utilisateur qui a créé le dossier virtuel.

Les dossiers virtuels étant des sélections de navigation à facettes enregistrées, il est nécessaire de bien comprendre Utilisation de la navigation à facettes avant de commencer à afficher, créer ou modifier des dossiers virtuels.

 

Note :
Les administrateurs trouveront des instructions sur l'octroi d'une autorisation utilisateur pour créer un dossier virtuel dans Configuration de dossiers virtuels. Développez la rubrique d'aide Configuration du serveur, puis sélectionnez Configuration des dossiers virtuels.

 

Les dossiers virtuels fournissent deux vues :

  • La Vue de dossier virtuel d'un dossier virtuel affiche les noms de tous les éléments de Content Server :

    1. qui sont conformes aux paramètres que le créateur du dossier virtuel a définis pour celui-ci, et

    2. que l'utilisateur dispose de l'autorisation Voir le contenu.

  • La Vue statique affiche les paramètres appliqués au dossier virtuel. Seuls les gestionnaires des connaissances peuvent y accéder. Cette vue correspond à l'onglet Spécifique sur la page Propriétés du dossier virtuel. La vue statique permet aux utilisateurs disposant des autorisations appropriées de modifier les paramètres du dossier virtuel.

Exemple 5-1 : Exemple de dossier virtuel

A l'aide de l'exemple dans Utilisation de la navigation à facettes, l'administrateur a autorisé l'utilisateur à créer un dossier virtuel. Cet utilisateur a créé un dossier virtuel nommé MémoService, avec les paramètres suivants :

  • MémoService affiche uniquement les éléments enregistrés dans le dossier Administration ou dans un dossier enfant du dossier Administration

  • MémoService affiche tous les éléments de Content Server où la valeur Document est attribuée à la facette Type

  • MémoService affiche tous les éléments de Content Server où la valeur Jean Dupont est attribuée à la facette Propriétaire

Lorsqu'un utilisateur clique sur le nom du dossier virtuel MémoService, la Vue de dossier virtuel de MémoService affiche le nom de tous les Documents possédés par Jean Dupont qui ont été enregistrés dans le dossier Administration, à condition que :

  1. l'utilisateur dispose de l'autorisation Voir le contenu pour visualiser le dossier virtuel et

  2. l'utilisateur dispose de l'autorisation Voir le contenu pour visualiser les documents possédés par Jean Dupont qui ont été enregistrés dans le dossier Administration.

Sous réserve qu'un utilisateur soit autorisé à accéder à la Vue statique, la Vue statique du dossier virtuel MémoService affiche :

  1. l'Emplacement sélectionné à partir duquel la navigation commence : « Administration »

  2. une liste des noms des facettes à parcourir : « Type de contenu : Document » et « Propriétaire : Jean Dupont ».

  3. les options que l'utilisateur avec des autorisations de modification peut appliquer à ce dossier virtuel. Pour plus d'informations sur ces options, voir Pour réviser ou modifier les paramètres d'un dossier virtuel.
