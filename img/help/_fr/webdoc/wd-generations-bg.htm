Utilisez des générations pour rendre certaines versions d'un élément plus visibles et plus accessibles dans Content Server. Comme un raccourci, une génération attire l'attention sur une version particulière d'un élément, une version précédente généralement. Si vous possédez une ancienne version d'un élément dont vous avez toujours besoin, vous pouvez créer une génération pour cette version et la stocker dans n'importe quel recueil ou dans l'espace de travail. Ainsi, la version visée par la génération ne sera ni modifiée, ni supprimée.

La génération est traitée comme un élément distinct. Elle n'est pas associée à l'élément (ni à la version de l'élément) à partir duquel elle a été créée. Grâce aux générations, les versions visées sont faciles à localiser et à utiliser car elles peuvent être stockées dans n'importe quel recueil approprié.

Menu Fonctions d'une génération

Le menu Fonctions d'une génération se divise en deux sous-menus :

  • Le sous-menu Original contient les fonctions que vous êtes autorisé à effectuer sur l'élément d'origine. L'élément d'origine est la version vers laquelle la génération pointe. Les fonctions du sous-menu Original ne concernent pas la génération.

  • Le sous-menu Génération contient les fonctions que vous êtes autorisé à utiliser sur la génération. Les fonctions du sous-menu Génération ne concernent pas l'élément d'origine.
