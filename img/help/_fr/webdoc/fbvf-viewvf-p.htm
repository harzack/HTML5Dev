Pour afficher un dossier virtuel :

  1. Naviguez jusqu'au recueil de Content Server contenant le dossier virtuel.

     

    Note :
    Votre administrateur ou un utilisateur qui a créé un dossier virtuel peut vous fournir un lien vers un dossier virtuel.

     

  2. Cliquez sur le nom du dossier virtuel. Dans la vue principale, vous verrez les éléments conformes aux paramètres du dossier virtuel, à condition que vous disposiez de l'autorisation Voir le contenu.

  3. Dans le panneau Filtre de contenu de la barre latérale, vous pouvez voir l'emplacement de Content Server que le dossier virtuel peut parcourir, ainsi que le nom des facettes parcourues.
