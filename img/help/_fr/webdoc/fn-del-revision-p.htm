Pour supprimer une version ou une révision :

  1. Cliquez sur l'icône Fonctions d'un document composite, choisissez Propriétés, puis Versions.

  2. Dans l'onglet Versions de la page Propriétés du document composite, cliquez sur l'icône Fonctions d'une version ou d'une révision, puis choisissez Supprimer.

  3. Cliquez sur OK.

 

Note :
Les versions et les révisions verrouillées ne peuvent pas être supprimées.
