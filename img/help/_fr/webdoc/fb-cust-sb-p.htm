Pour modifier les paramètres du bandeau à partir de la boîte de dialogue Paramètres :

  1. Dans la barre de menu générale, sélectionnez Outils, puis Paramètres.

  2. Sur la page Mes paramètres : Général, cliquez sur l'onglet Général.

  3. Dans l'onglet Général, dans la liste Emplacement de la barre latérale, sélectionnez Gauche ou Droite pour déterminer l'emplacement par défaut de la barre latérale.

  4. Activez la case à cocher Afficher le bandeau dans Parcourir pour que le bandeau s'affiche par défaut.

  5. Cliquez sur Mettre à jour.

Pour modifier dynamiquement les paramètres du bandeau dans une Vue :

  1. Lorsque le bandeau est disponible mais qu'il n'apparaît pas, déplacez le curseur à gauche de la page jusqu'à ce qu'une barre orange s'affiche.

    Survolez cette barre pour afficher le texte Cliquer pour afficher le bandeau. Cliquez sur la barre orange pour afficher le bandeau Filtre de contenu. Si la barre orange n'apparaît pas à gauche de la page, il est possible que le paramètre par défaut du bandeau ait changé. Déplacez le curseur à droite de la page jusqu'à ce que la barre orange s'affiche, puis cliquez sur celle-ci.

  2. Si le bandeau s'affiche sur votre page mais que vous souhaitez la masquer, déplacez le curseur entre la Vue du parcours et le bandeau jusqu'à ce que la barre orange s'affiche.

    Le texte Cliquer pour masquer le bandeau s'affiche lorsque vous survolez la barre orange. Cliquez sur la barre orange pour masquer le bandeau.

  3. Cette méthode d'affichage ou de masquage du bandeau sur votre page devient votre paramètre par défaut.

     

    Astuce :
    Pour afficher les paramètres par défaut du Filtre de contenu, dans la barre de menu générale, sélectionnez Outils puis Paramètres.
