Pour utiliser la bannière de navigation :

  1. Lorsque vous sélectionnez une facette, la bannière de navigation Vue dynamique s'affiche pour tous les utilisateurs, au-dessus du panneau de la barre latérale. Pour supprimer tous les filtres, cliquez sur le lien Supprimer tous les filtres à droite de la barre de navigation.

  2. Les utilisateurs autorisés à créer un dossier virtuel peuvent également voir le lien Enregistrer comme dossier virtuel dans la bannière de navigation. Pour créer un dossier virtuel à partir de votre sélection de facettes, cliquez sur le lien Enregistrer comme dossier virtuel. Voir Pour créer un dossier virtuel pour plus d'instructions sur la création d'un dossier virtuel.

  3. Les utilisateurs qui naviguent vers un dossier virtuel précédemment créé peuvent voir la bannière de navigation Vue de dossier virtuel. Si vous êtes autorisé à modifier ce dossier virtuel, vous verrez un lien Modifier le dossier virtuel dans la bannière de navigation. Pour modifier un dossier virtuel, cliquez sur le lien Modifier le dossier virtuel. Voir Pour réviser ou modifier les paramètres d'un dossier virtuel pour plus d'instructions sur la modification des paramètres d'un dossier virtuel.
