Pour éliminer des versions :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Versions.

  2. Dans l'onglet Versions de la page Propriétés de l'élément, saisissez un nombre entier supérieur à 0 dans le champ Eliminer toutes les versions à l'exception de la plus récente.

     

    Note :
    Si une création de version avancée a été appliquée à cet élément, le nom du champ indique : Eliminer toutes les versions secondaires à l'exception de la plus récente.

     

  3. Cliquez sur Eliminer les versions.

  4. Cliquez sur OK.
