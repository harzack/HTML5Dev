Pour ajouter un arbre de facettes :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Volume Facettes.

  2. Accédez au dossier des facettes personnalisé dans lequel vous souhaitez créer une arborescence de facettes ou créez un dossier des facettes dans le volume Facettes, puis accédez à ce nouveau dossier.

  3. Dans le menu Ajouter un élément, sélectionnez Arborescence de facettes.

  4. Dans le champ Nom, saisissez un nom unique pour la nouvelle arborescence de facettes.

  5. Facultatif Dans le champ Description, saisissez la description de la nouvelle arborescence de facettes.

  6. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette arborescence de facettes.

  7. Facultatif Pour placer l'arborescence de facettes dans un emplacement autre que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server..., naviguez jusqu'au recueil dans lequel vous voulez placer l'arborescence de facettes, puis cliquez sur le lien Sélectionner. Seules les zones de Content Server permettant d'enregistrer une facette peuvent être sélectionnées.

  8. Cliquez sur le bouton Ajouter pour valider et créer votre nouvelle arborescence de facettes et revenir au volume Facettes.
