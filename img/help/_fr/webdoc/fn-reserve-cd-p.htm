Pour réserver un document composite :

  1. Cliquez sur l'icône Fonctions du document composite et choisissez Réserver.

  2. Sur la page Réserver, dans le champ Réserver par, effectuez l'une des opérations suivantes :

    • Pour réserver le document composite pour vous-même, sélectionnez votre nom d'utilisateur.

    • Pour réserver le document composite pour un groupe auquel vous appartenez, sélectionnez le nom du groupe.

  3. Cliquez sur Envoyer.
