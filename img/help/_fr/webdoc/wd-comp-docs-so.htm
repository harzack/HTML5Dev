Un document composite est un recueil conçu pour le traitement d'un groupe de documents dans son ensemble, avec un ordre et une structure spécifiques. Par exemple, vous pouvez utiliser un document composite pour gérer :

  • les fichiers individuels de traitement de texte constituant les chapitres d'un livre ;

  • un groupe de fichiers HTML et de fichiers d'image pour la réalisation d'un site Web ;

  • une proposition avec des feuilles de calcul, des graphiques et une lettre de présentation.
