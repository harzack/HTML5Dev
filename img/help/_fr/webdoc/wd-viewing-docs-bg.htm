Selon ce que vous souhaitez faire, de vos autorisations et de la configuration de Content Server, vous pouvez avoir accès à des documents dans Content Server de différentes façons. En règle générale, vous pouvez afficher le contenu d'un document, y effectuer des modifications, le télécharger, le réserver et annuler sa réservation.

Vous pouvez afficher un document dans votre navigateur Web, ce qui vous permet de visualiser rapidement le contenu sans avoir à vous préoccuper de la mise en forme. Vous pouvez également ouvrir une copie d'un document en lecture seule dans le programme de bureau qui lui est associé. Cette opération vous permet de visualiser le contenu d'un document, sa mise en forme et sa disposition.

 

Note :
Votre administrateur doit activer la fonction qui vous permet d'afficher un document en tant que page Web.

 

Téléchargez des documents à partir de Content Server lorsque vous souhaitez les enregistrer sur votre ordinateur ou y effectuer des modifications. Lorsque vous téléchargez un document, vous pouvez l'ouvrir dans le programme de bureau qui lui est associé ou l'enregistrer sur votre ordinateur. Si vous téléchargez un document afin de le modifier, vous devez le réserver afin que personne ne puisse le modifier en même temps. Pour plus d'informations sur la Réservation d'un document, reportez-vous à la rubrique Réservation et annulation de la réservation d`éléments.

 

Note :
Les modules Content Server peuvent avoir un impact sur l'affichage, l'ouverture et le téléchargement de vos documents.

 

Si cette fonction est activée par votre administrateur, vous pouvez également accéder aux documents via la page Vue d'ensemble d'un document. La page Vue d'ensemble d'un document fournit des informations sur le document, notamment une description et des informations sur la version. Elle vous permet d'effectuer un grand nombre de fonctions dans le document. Pour en savoir plus, voir Pages de vue d'ensemble d'un document et d'une version.
