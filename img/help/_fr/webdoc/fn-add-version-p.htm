Pour ajouter une version :

  1. Cliquez sur l'icône Fonctions d'un élément et choisissez Ajouter une version.

  2. Facultatif Sur la page Ajouter : Version, dans le champ Description, saisissez une description pour cette version.

     

    Note :
    Le champ Description de la page Ajouter : Version vous permet de saisir une description de la version en cours d'ajout. Il ne s'agit pas de la description de l'objet auquel vous ajoutez la version. Les descriptions de version ne sont pas multilingues.

     

  3. Dans le champ Fichier, cliquez sur Parcourir.... Naviguez jusqu'à la copie locale du fichier que vous ajoutez comme nouvelle version de cet élément. Cliquez sur le fichier pour le mettre en surbrillance, puis sur Ouvrir.

  4. Si vous travaillez sur un ordinateur Macintosh qui n'exécute pas le système d'exploitation OSX et que le fichier que vous chargez contient des ressources importantes que vous souhaitez préserver, sélectionnez la case à cocher MacBinary (définir avant de choisir un fichier).

     

    Note :
    Pour obtenir plus d'informations sur l'ajout de fichiers Macintosh, reportez-vous à la section Ajout de fichiers Macintosh dans Content Server.

     

  5. Facultatif Cliquez sur le bouton radio L'ajouter en tant que version secondaire ou L'ajouter en tant que version principale.

     

    Note :
    Cette option apparaît seulement pour les documents et les documents texte, et uniquement si le document utilise la fonction de création avancée des versions.

     

  6. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cet élément.

  7. Facultatif Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  8. Cliquez sur Ajouter une version.

     

    Astuce :
    Vous pouvez ajouter une version à un document texte en cliquant sur le lien Modifier du document texte. Sur la page Modifier : <nom_du_document_texte>, apportez les modifications requises au document texte, puis cliquez sur Ajouter une version. Pour plus d'informations sur le lien Modifier, voir Actions promues dans Interface utilisateur.
