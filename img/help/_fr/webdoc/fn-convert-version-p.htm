Pour convertir une version standard ou avancée :

  1. Cliquez sur l'icône Fonctions d'un document, sélectionnez Propriétés, puis Infos spécifiques.

  2. Dans l'onglet Spécifique de la page Propriétés du document, cliquez sur l'un des boutons radio suivants dans la section Versions :

    • Standard - création linéaire de versions

    • Avancée - création de versions principales/secondaires

  3. Cliquez sur Mettre à jour.

     

    Note :
    Pour plus d'informations sur le champ Nbre max. de versions, voir Pour configurer une limite de version.
