Dans la plupart des cas, Content Server peut identifier automatiquement le type de fichier ajouté à la base de données. Dès qu'un document est ajouté à Content Server, son type MIME est stocké dans ses métadonnées. MIME (Multipurpose Internet Media Extensions) est une norme utilisée pour décrire les types de fichiers (texte, image ou audio par exemple).

Par exemple, le type MIME application/msword indique qu'il s'agit d'un fichier MS Word et le type MIME text/richtext peut désigner un document texte normal. Dans Content Server, des icônes servent à identifier les types de fichiers. Content Server utilise par exemple l'icône Microsoft Word, l'icône Document texte et l'icône Document pour identifier les types de fichiers.

Exemples de types MIME courants :

  • application/msword

  • application/pdf

  • audio/x-wav

  • image/jpeg

  • text/richtext

  • video/mpeg
