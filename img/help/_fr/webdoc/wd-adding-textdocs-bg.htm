Content Server comporte un éditeur de texte intégré, que vous pouvez utiliser pour créer et modifier des documents texte dans Content Server. L'éditeur de texte permet d'ajouter des fichiers texte simples à Content Server, sans avoir à les créer préalablement dans une application distincte.

Pour ajouter un document texte se trouvant sur votre ordinateur dans Content Server, vous devez l'ajouter en tant que document. Content Server le reconnaît en tant que fichier texte et le traite dès lors comme un document texte. Lorsque vous créez ou ajoutez un document texte, vous pouvez fournir une description de ce document, modifier ses catégories et attributs et l'ajouter à un autre recueil. Pour plus d'informations sur l'ajout d'éléments, reportez-vous à la section Utilisation des éléments généraux.

Lorsque vous modifiez un document texte, Content Server l'ouvre dans l'éditeur de texte, quelle que soit la manière dont il a été ajouté. L'éditeur de texte fonctionne avec les documents existants ou les documents en texte clair, notamment les formats HTML, XML et SGML. Le type MIME par défaut assigné par le système est texte normal (text/plain). Pour plus d'informations sur les types MIME, reportez-vous à la section Types MIME.
