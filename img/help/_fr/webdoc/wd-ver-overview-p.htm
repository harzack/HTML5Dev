Pour afficher la page Vue d'ensemble d'une version à partir du menu Fonctions :

  1. Cliquez sur l'icône Fonctions d'un document.

  2. Dans le menu Fonctions, choisissez Propriétés, puis Versions.

  3. Cliquez sur l'icône Fonctions de la version dont vous souhaitez afficher la page Vue d'ensemble.

  4. Dans le menu Fonctions, choisissez Vue d'ensemble.
