Pour accéder à une apparence dans un dossier virtuel :

  1. Naviguez jusqu'au dossier virtuel contenant l'apparence.

  2. Cliquez sur le bouton Fonctions du dossier virtuel. Sélectionnez Configurer dans le menu Fonctions.

  3. Cliquez sur le bouton Fonctions de l'apparence. Sélectionnez Ouvrir dans le menu Fonctions.

     

    Astuce :
    Vous pouvez également localiser votre nouvelle apparence en sélectionnant Apparence dans la barre latérale Filtre de contenu. Sur la page Vue dynamique, recherchez votre nouvelle apparence, puis cliquez sur son nom.

     

  4. Facultatif Sur la page <nom_de_votre_apparence>, modifiez les champs requis. Pour plus d'informations sur la modification d'une apparence, voir Ajout et modification d`apparences.

  5. Facultatif Vous devez activer votre nouvelle apparence pour l'afficher dans le mode navigation d'un dossier virtuel :

    1. Sur la page <nom_de_votre_apparence>, cliquez sur Paramètres.

    2. Sur la page Modifier les paramètres de l'apparence :

      1. Dans le champ Type, sélectionnez le type d'apparence, En cascade ou Pas de cascade.

        Si vous choisissez En cascade, votre apparence est appliquée à tous les sous-éléments du dossier virtuel.

      2. Dans le champ Statut actuel, pour activer l'apparence, sélectionnez Activé.

      3. Cliquez sur Envoyer.
