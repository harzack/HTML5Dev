Pour ajouter un document composite :

  1. Dans le menu Ajouter un élément, sélectionnez Document composite.

  2. Sur la page Ajouter : Document composite, saisissez le nom du document composite dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description pour le document composite.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

  4. Facultatif Si vous appliquez un classement à ce document composite, dans la liste Classements, sélectionnez Parcourir les classements ou Mes favoris.

    1. Dans la boîte de dialogue qui s'affiche, sélectionnez le classement à appliquer à ce document composite.

    2. Assurez-vous que la case à cocher Action située en regard du classement est bien activée.

    3. Cliquez sur Envoyer.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce document composite.

  6. Facultatif Pour placer le document composite dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer le document composite, puis cliquez sur son lien Sélectionner.

  7. Cliquez sur Ajouter.
