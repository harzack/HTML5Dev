Pour configurer un dossier virtuel

  1. Naviguez jusqu'au recueil de Content Server contenant le dossier virtuel.

  2. Dans le menu Fonctions du dossier virtuel, sélectionnez Configurer.

  3. Sur la page <nom_du_dossier_virtuel> :

    1. Si le dossier virtuel ne comporte aucun sous-dossier virtuel, apparence ou vue personnalisée, la colonne Nom de type indique : « Aucun élément. ».

    2. Si le dossier virtuel contient un élément, le nom de cet élément est répertorié sous l'intitulé Nom de type.

      1. Pour modifier l'affichage de l'élément, sélectionnez l'un des boutons Liste, Mis en valeur ou Masqué.

        Par défaut, un sous-dossier virtuel est Mis en valeur. Actuellement, si vous sélectionnez Liste ou Masqué pour un sous-dossier virtuel, le sous-dossier virtuel est masqué. OpenText vous recommande de laisser la valeur d'un sous-dossier virtuel définie sur Mis en valeur.

      2. Facultatif Pour modifier les paramètres d'un sous-dossier virtuel, sélectionnez le menu Fonctions du sous-dossier virtuel. Dans le menu Fonctions du sous-dossier virtuel, sélectionnez Propriétés, puis Spécifique. Pour en savoir plus, voir Pour réviser ou modifier les paramètres d'un dossier virtuel.

      3. Facultatif Pour modifier les paramètres d'une apparence ou d'une vue personnalisée, sélectionnez le menu Fonctions de l'élément. Dans le menu Fonctions de l'élément, sélectionnez Ouvrir. Pour en savoir plus, voir Création et gestion des affichages personnalisés et Ajout et modification d`apparences.

      4. Facultatif Pour supprimer un élément, sélectionnez le bouton Supprimer en regard de cet élément. Cliquez sur Appliquer, puis sur OK pour confirmer la suppression de l'élément.

  4. Cliquez sur Mettre à jour.
