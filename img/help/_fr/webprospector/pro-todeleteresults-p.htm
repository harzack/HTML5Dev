Pour supprimer un prospecteur :

  1. Cliquez sur l'icône Fonctions du prospecteur et choisissez Ouvrir.

  2. Sur la page de résultats d'un prospecteur, cliquez sur Supprimer dans la liste Fonctions correspondant aux résultats que vous souhaitez supprimer.

  3. Cliquez sur le bouton Mettre à jour.

 

Note :
Dans un résultat, si vous cliquez sur l'icône Fonctions d'un élément et que vous choisissez Supprimer, vous supprimez l'élément de Content Server.

 
 

Astuce :
Pour supprimer tous les résultats du prospecteur (à l'exception des résultats verrouillés), cliquez sur le bouton Définir sur Supprimer sur la page de résultats d'un prospecteur, puis cliquez sur le bouton Mettre à jour.
