Pour créer un prospecteur :

  1. Sélectionnez Prospecteur dans le menu Ajouter un élément.

  2. Saisissez un nom pour le prospecteur dans le champ Nom.

  3. Facultatif Saisissez une description du prospecteur dans le champ Description.

  4. Cliquez sur une ou plusieurs vues de recherche dans la liste Dans.

  5. Saisissez une requête dans la zone Rechercher.

     

    Note :
    Pour plus d'informations sur la création de requêtes et le langage LQL (Livelink Query Language), reportez-vous à la rubrique Recherche Content Server.

     

  6. Pour étendre ou limiter votre requête, ajoutez des critères de recherche supplémentaires dans le champ Critères facultatifs.

  7. Renseignez les champs des paramètres suivants :

    • Etat

    • Options de résultats

    • Options de notification

    • Programme de notification

  8. Pour insérer le prospecteur à un autre emplacement que celui indiqué dans le recueil actuel, cliquez sur le bouton Parcourir Content Server situé à côté du champ Créer dans.

  9. Cliquez sur le bouton Ajouter.

 

Note :
Pour plus d'informations sur les paramètres que vous pouvez configurer lors de la création d'un prospecteur, reportez-vous à la rubrique Paramètres du prospecteur.

 
 

Astuce :
Vous pouvez également créer un prospecteur en cliquant sur l'icône Créer un prospecteur sur la page des prospecteurs.
