Lorsque les prospecteurs identifient des informations qui répondent à vos critères de recherche, ils ajoutent les éléments correspondants aux listes de résultats pour vous permettre d'y accéder. Les listes des résultats des prospecteurs changent lorsque des éléments correspondant à vos critères de recherche sont ajoutés ou supprimés des index Content Server.

Lorsque vous utilisez les résultats de prospecteurs, vous indiquez la manière dont vous souhaitez trier les informations qu'ils contiennent et si vous souhaitez verrouiller les résultats, les enregistrer dans un lot de résultats ou les supprimer.

Tri des résultats

Lorsque vous créez ou modifiez des prospecteurs, vous pouvez choisir de trier les résultats en fonction du taux le plus élevé ou du résultat le plus récent. Toutefois, lorsque vous affichez les résultats d'un prospecteur, vous pouvez provisoirement trier les résultats selon les critères suivants :

  • Taux : pour afficher en premier le résultat qui correspond le mieux aux critères de recherche.

  • Type : pour afficher les résultats de groupe en fonction du type de document.

  • Nom : pour afficher les résultats par ordre alphabétique en fonction du nom des documents.

  • Date trouvée : pour afficher les résultats selon la date où ils ont été ajoutés à la liste de résultats.

  • Taille : pour afficher les résultats en fonction de la taille des documents.

  • Emplacement : pour regrouper les résultats en fonction de leur emplacement Content Server.

 

Note :
Le score d'un élément veut varier légèrement entre Prospecteurs et Recherche. Par exemple, si vous exécutez une même requête dans un prospecteur, puis dans la recherche, le taux de résultats d'un élément peux varier légèrement. Cette différence est due à une légère modification dans l'implémentation du classement de la pertinence entre les Prospecteurs et la Recherche.

 

Verrouillage de résultats

Pour conserver un résultat dans une liste de résultats, vous avez la possibilité de le verrouiller. Le verrouillage de résultats est utile lorsqu'une requête du prospecteur récupère plus de résultats que le nombre maximum que vous avez choisi, mais que vous ne souhaitez pas supprimer les résultats à faible taux de votre liste uniquement parce que des résultats à taux plus élevé ont été trouvés. Lorsque vous verrouillez un résultat, Content Server affiche une icône de verrouillage à côté du résultat. Vous pouvez verrouiller un résultat à tout moment.

Enregistrement de lot de résultats

Pour enregistrer une liste de résultats de prospecteur, vous devez créer un lot de résultats. Un lot de résultats du prospecteur est la représentation statique d'une liste de résultats d'un prospecteur à un moment donné. Il n'est pas modifié par la suite, lorsque des résultats sont ajoutés ou enlevés de la liste de résultats.

Suppression des résultats

Il est possible de supprimer les résultats de prospecteur un par un ou tous ensemble.
