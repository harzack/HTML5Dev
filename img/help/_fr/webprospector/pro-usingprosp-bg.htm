Les Prospectors sont des éléments Content Server qui aident les utilisateurs à localiser et utiliser les informations. Ils agissent en tant que filtres, en analysant les données indexées par Content Server afin de localiser les informations susceptibles d'intéresser les utilisateurs. Les prospecteurs sont exécutés en continu, de sorte que les listes de résultats qu'ils gèrent sont toujours précises et à jour.

Les prospecteurs peuvent analyser toute source de données indexée par Content Server. Ils peuvent, par exemple, effectuer des recherches sur le référentiel Content Server, sur des répertoires locaux ou sur une base de données Lotus Notes. Pour plus d'informations sur les sources de données indexées sur votre site Content Server, contactez votre administrateur.

Si votre administrateur a créé des sources de données Enterprise Library, le prospecteur peut rechercher des éléments Enterprise Library.
