Pour enregistrer un lot de résultats du prospecteur :

  1. Cliquez sur l'icône Fonctions du prospecteur et choisissez Ouvrir.

  2. Cliquez sur l'icône Enregistrer comme Lot de résultats .

  3. Sur la page Ajouter : Lot de résultats du prospecteur, saisissez le nom du lot dans le champ Nom.

  4. Facultatif Saisissez une description du lot de résultats dans le champ Description.

  5. Cliquez sur le bouton Parcourir Content Server, recherchez l'emplacement où vous souhaitez ajouter le lot, puis cliquez sur son lien Sélectionner.

  6. Cliquez sur le bouton Ajouter.
