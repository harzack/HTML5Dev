Si votre administrateur supprime une source de données de prospecteur, vous ne pouvez pas cliquer sur l'élément (les prospecteurs effectuent des recherches dans des sources de données pour localiser les éléments dans Content Server). Bien que la source de données de cet élément n'existe plus, vous pouvez peut-être encore accéder aux résultats de l'élément. Pour plus d'informations sur la manière d'accéder à cet élément, contactez votre administrateur.
