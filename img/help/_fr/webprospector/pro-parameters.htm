Pour créer un prospecteur de base, vous devez indiquer des critères de recherche et préciser les index sur lesquels vous souhaitez faire porter la recherche. Pour créer un prospecteur plus complexe, vous pouvez modifier des paramètres supplémentaires à tout moment. Par exemple, vous pouvez choisir le mode d'affichage des résultats, ainsi que le moment où vous serez informé que de nouveaux résultats ont été trouvés. Le tableau suivant décrit les paramètres que vous pouvez définir lorsque vous créez ou modifiez un prospecteur.

Tableau 1-1 : Onglet Infos spécifiques de la page Propriétés Prospecteur

Paramètre Description
dans Spécifie les vues de recherche que le prospecteur inspecte pour trouver des modifications.
Recherche Indique les critères de recherche de la requête du prospecteur.
Critères facultatifs Précise des critères de recherche supplémentaires pour étendre ou limiter la portée de la requête du prospecteur.
Etat Indique si le prospecteur est actif, ainsi que la date à laquelle il expire. Les trois états possibles sont les suivants :
•

Actif jusqu'au : permet d'indiquer la date à laquelle le prospecteur expirera.

•

Actif (n'expire pas) : permet au prospecteur de s'exécuter sans jamais expirer.

•

Inactif : permet d'ajouter le prospecteur à Content Server sans le lancer.

Options de résultats Indique la manière dont les résultats seront affichés et le type de résultats à afficher. Précise également si les informations de résumé pour chaque prospecteur sont affichées ou masquées.
Les résultats sont conservés en fonction des options suivantes :
•

Conserver : indique le nombre de résultats que le prospecteur affiche.

•

Taux minimum : précise le taux minimum d'un résultat du prospecteur.

Les résultats peuvent être ordonnés de deux façons différentes :
•

Taux le plus élevé : les éléments sont ordonnés en fonction du taux de correspondance. Le prospecteur conserve uniquement les résultats qui correspondent le mieux aux critères de recherche. Il s'agit de la valeur par défaut.

•

Dernière date : les éléments sont classés par ordre chronologique. Le prospecteur conserve uniquement les résultats les plus récents.

Vous pouvez également afficher ou masquer des résumés et exclure les éléments supprimés de la recherche. Par défaut, les éléments supprimés ne sont pas inclus dans la recherche.
Options de notification Indique si vous serez informé lorsque de nouveaux résultats seront trouvés et le type de notification que vous recevrez.
Vous pouvez recevoir trois types de messages de notification :
•

Notification seulement : envoie un e-mail au format texte qui vous informe simplement que de nouveaux résultats de prospecteur sont disponibles.

•

Inclure les nouveaux résultats : envoie un e-mail au format HTML qui inclut les nouveaux résultats.

•

Inclure tous les résultats : envoie un e-mail au format HTML qui comprend tous les résultats du prospecteur.

 

Note :
L'e-mail de notification inclut les résumés des résultats uniquement si vous sélectionnez Afficher les résumés dans la section Options des résultats.

 
Programme de notification Indique la fréquence à laquelle le prospecteur vous informe que de nouveaux résultats ont été trouvés. Pour choisir la fréquence des notifications, vous pouvez combiner les paramètres suivants :
•

Les jours suivants : détermine les jours où le prospecteur effectue une nouvelle recherche.

•

Aux heures suivantes : détermine les heures auxquelles le prospecteur effectue une nouvelle recherche.

•

Aux minutes suivantes : détermine à quel moment de l'heure indiquée le prospecteur effectue la nouvelle recherche.

Derniers résultats ajoutés Précise la dernière fois que des résultats ont été ajoutés au prospecteur. Vous ne pouvez pas modifier ce champ.
