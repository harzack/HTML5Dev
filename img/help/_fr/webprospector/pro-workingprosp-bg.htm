Lorsque vous créez des prospecteurs, vous indiquez des paramètres qui permettent de définir des critères de recherche. Si un prospecteur identifie des informations qui répondent à vos critères de recherche, il ajoute l'élément correspondant à une liste de résultats pour vous permettre d'y accéder. L'utilisation des prospecteurs implique également de les organiser et de les supprimer le cas échéant. Pour plus d'informations, reportez-vous à la rubrique Paramètres du prospecteur ou Gestion des résultats des prospecteurs.

Création de prospecteurs

Pour créer un prospecteur, vous utilisez le langage LQL (Livelink Query Language) pour construire une requête qui vous permet de localiser le type d'informations que vous souhaitez récupérer. Pour plus d'informations sur la création de requêtes, reportez-vous à la rubrique Recherche Content Server.

Une fois la requête du prospecteur créée, vous choisissez la vue de recherche que le prospecteur devra analyser. Le prospecteur exécute ensuite sa requête pour voir si les nouveaux éléments ajoutés à l'index correspondent aux critères de recherche indiqués. Si c'est le cas, le prospecteur ajoute un lien à l'élément correspondant dans sa liste de résultats. Il continue de mettre à jour sa liste de résultats lorsque des éléments sont ajoutés ou enlevés de l'index. Un prospecteur peut uniquement analyser les index créés pour la recherche par un administrateur Content Server.

 

Notes :

  • Si votre administrateur a créé des sources de données Enterprise Library de Content Server, vous pouvez identifier la vue Enterprise Library [Toutes les versions] d'un prospecteur pour analyser et inclure des éléments Enterprise Library de Content Server dans les résultats de votre recherche. Les prospecteurs ignorent les vues de recherche Enterprise Library [Toutes les versions] et effectue une recherche dans les vues de recherche Entreprise Library [Toutes les variantes].

  • Les recherches dans les autres vues de recherche Enterprise Library ne donnent pas les résultats attendus.

 

Lorsque vous créez un prospecteur, vous définissez également les options de la liste de résultats : le nombre de résultats et leur ordre d'affichage. Vous pouvez également choisir d'afficher ou de masquer les résumés des éléments de la liste des résultats dans Content Server. Pour recevoir un e-mail de Content Server chaque fois que la liste des résultats est mise à jour, vous pouvez configurer une notification à condition que votre administrateur ait activé cette fonction. Pour plus d'informations sur la notification, reportez-vous à la rubrique Utilisation des notifications. Contactez votre administrateur afin de savoir si votre système a été configuré de façon à ce que vous receviez des e-mails.

Il est possible de créer des prospecteurs dans la plupart des emplacements Content Server. Vous pouvez, par exemple, les créer dans l'espace de travail Entreprise, dans un espace de travail Projet ou dans votre propre espace de travail (Mon espace de travail).

Modification des prospecteurs

Pour modifier un prospecteur, changez les paramètres que vous avez définis au moment de sa création.

Organisation des prospecteurs

Vous pouvez organiser des prospecteurs dans la plupart des emplacements Content Server en les déplaçant et en les organisant dans des onglets. Par exemple, sur la page des prospecteurs, à laquelle vous accédez depuis le menu Personnel, vous pouvez organiser les prospecteurs dans l'onglet Mes prospecteurs (l'onglet par défaut) ou dans des onglets personnalisés que vous créez. Lorsque vous créez des onglets personnalisés, vous pouvez préciser le nombre de résultats à afficher dans l'onglet. Pour plus d'informations sur la gestion des onglets, reportez-vous à la rubrique Gestion des onglets de la page de rapports.

Suppression des prospecteurs

Si vous souhaitez supprimer définitivement un prospecteur d'Content Server, vous pouvez le supprimer de la même manière que tout autre élément. Pour plus d'informations sur la suppression des éléments, reportez-vous à la rubrique Utilisation des éléments généraux.
