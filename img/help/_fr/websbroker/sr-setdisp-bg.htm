La définition des options d'affichage vous permet de personnaliser l'affichage des résultats de vos requêtes de recherche sur la page Résultats de recherche. Vous pouvez choisir un style de page de résultats et définir d'autres options spécifiques, telles que le nombre de résultats à afficher sur une page. Le choix d'un style de page vous permet de définir la quantité d'informations à afficher et le format d'affichage des éléments sur la page. Vous pouvez appliquer ces options d'affichage à la recherche actuelle uniquement ou bien créer ou modifier un formulaire de recherche afin que les résultats s'affichent selon ces options par défaut. Pour plus d'informations sur les formulaires de résultats de recherche, voir Utilisation de la page Résultats de recherche et des formulaires.

Dans la rubrique Disposition de la page de résultats, il existe quatre styles de page possible :

  1. Classique

  2. Standard

  3. Tabulaire

  4. Recherche Web, style par défaut

Le style par défaut est généralement recommandé. Les éléments de disposition sont figés pour vos sessions Content Server et modifiés à partir de la page Résultats de recherche pour les fonctionnalités comme le style d'affichage.

La zone Aperçu affiche le style de disposition que vous avez sélectionné pour la page.

Selon le style de disposition sélectionné, les contrôles suivant peuvent également s'appliquer :

Nombre de résultats Vous pouvez indiquer le nombre de résultats de recherche que vous souhaitez afficher par défaut sur la page de résultats de recherche. Si le nombre de résultats de recherche est supérieur au nombre indiqué, des pages de résultats de recherche supplémentaires sont ajoutées.
Afficher les résumés/descriptions Vous pouvez également afficher ou masquer les résumés et descriptions des résultats de recherche. Vous avez la possibilité de sélectionner les résumés, les descriptions, ou les deux comme informations de résultat de recherche préférées à afficher dans Content Server. Par exemple, si vous choisissez d'afficher les résumés, le résumé de chaque résultat de recherche est affiché, si celui-ci est disponible. Dans le cas contraire, une description du résultat de recherche est affichée, si celle-ci est disponible. Pour plus d'informations sur les résultats de la recherche, reportez-vous à la rubrique Utilisation des résultats de la recherche.
Afficher les locutions clés Vous pouvez afficher ou masquer des locutions clés. Les locutions clés sont des mots ou des combinaisons de mots récurrents, notamment des combinaisons contenant des mots inhabituels.
Afficher le chemin complet de l'emplacement Vous pouvez décider d'afficher le chemin complet de l'emplacement de chaque résultat de recherche.
Champs de résultat Vous pouvez indiquer les champs de résultat à afficher sur la page Résultats de recherche, comme la taille ou l'emplacement des éléments répertoriés, et enlever les champs de résultat que vous ne souhaitez pas afficher.
 

Note :
Il est possible que votre administrateur ait spécifié des champs de recherche obligatoires. Si les champs requis ont été configurés, une section Obligatoire supplémentaire contenant ces champs s'affiche sur la page Options d'affichage. Vous ne pouvez pas supprimer des champs obligatoires ou les déplacer de la page Résultats de recherche.

 
Trier par Vous pouvez configurer l'ordre de classement par défaut lorsque des résultats de recherche sont renvoyés. Pour plus d'informations sur les options de classement, reportez-vous à la rubrique Tri des résultats de la recherche.
