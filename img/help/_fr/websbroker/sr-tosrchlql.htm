Pour effectuer une recherche à l'aide d'une requête complexe :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche Content Server, choisissez Requête complexe dans la liste Rechercher.

  3. Saisissez votre requête dans le champ Texte intégral.

  4. Cliquez sur Rechercher.
