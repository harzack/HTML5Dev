Pour définir les options d'affichage :

  1. Définissez vos critères de recherche en saisissant votre requête dans le champ Rechercher de la barre du menu global.

  2. Cliquez sur le bouton Exécuter la recherche.

  3. Sur la page Résultats de recherche, cliquez sur le menu déroulant Sélectionner le style d'affichage, puis sélectionnez Modifier les options d'affichage pour ouvrir la page Options d'affichage.

  4. Sur la page Options d'affichage, choisissez le mode d'affichage de vos résultats de recherche. Pour plus d'informations, voir Définition des options d'affichage.

  5. Cliquez sur le menu déroulant Sélectionner l'action puis sélectionnez Enregistrez vos résultats pour ajouter un lot de résultats de recherche.
