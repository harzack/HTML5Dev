Vous pouvez enregistrer une recherche que vous avez formulée pour réutiliser un ensemble spécifique de critères de recherche. Vous pouvez ensuite relancer la recherche que vous avez enregistrée pour mettre à jour les résultats de recherche obtenus pour cette recherche. Vous pouvez modifier une recherche enregistrée pour effectuer des modifications sur une recherche sans avoir à la formuler une nouvelle fois. Si vous souhaitez vous appuyer sur des modèles pour la formulation de vos recherches, utilisez les formulaires de recherche. Pour plus d'informations sur les formulaires de recherche, voir Utilisation des formulaires de recherche. Pour plus d'informations sur la formulation d'une recherche, voir Recherche à l'aide de requêtes de base.

Lorsque vous enregistrez une recherche, vous l'ajoutez à un dossier ou à un espace de travail dans Content Server. Vous pouvez ensuite effectuer de nombreuses opérations identiques à celles que vous réalisez sur d'autres types d'éléments Content Server. Par exemple, vous pouvez définir des autorisations pour la recherche afin que d'autres utilisateurs et groupes Content Server puissent y accéder. Pour plus d'informations sur les opérations possibles sur les éléments Content Server, reportez-vous à la rubrique Utilisation des éléments généraux.

Vous pouvez également créer une recherche de vue personnalisée dans un dossier spécifique. Cela permet :

  • à d'autres utilisateurs de l'ajouter à leurs favoris ;

  • à d'autres utilisateurs de réutiliser vos critères de recherche ;

  • de définir la priorité d'affichage des formulaires de recherche personnalisés.

Pour plus d'informations, voir les rubriques Pour créer une recherche de vue personnalisée, Pour classer les vues personnalisées et Utilisation des affichages personnalisés.

Si vous créez une requête d'attente dans Records management, vous pouvez enregistrer la requête sur la page Critères d'attente ou sur la page Recherche avancée. Le bouton Enregistrer en tant que Requête pour attente s'affiche dans la section Options d'enregistrement uniquement si vous disposez des autorisations adéquates ou si vous êtes un utilisateur désigné de Records management. Après l'enregistrement d'une requête d'attente sur la page Recherche avancée, vous serez automatiquement redirigé vers la page Critères d'attente dans Records management. Pour plus d'informations sur l'utilisation des attentes avec recherche dans Records management, voir Gestion des attentes.
