Pour modifier un formulaire de recherche :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche avancée, dans la liste déroulante Utiliser ce formulaire de recherche, cliquez sur le nom d'un formulaire de recherche dans l'une des listes suivantes :

    • Formulaires de recherche personnels

    • Formulaires de recherche système

  3. Cliquez sur le bouton Enregistrer en tant que formulaire.

  4. Sur la page Enregistrer en tant que formulaire, cliquez sur le bouton radio Mettre à jour un formulaire de recherche.

  5. Dans la liste déroulante Formulaires de recherche actuels, cliquez sur le nom du formulaire de recherche que vous souhaitez modifier.

  6. Cliquez sur Enregistrer.

 

Astuce :
Vous pouvez également modifier un formulaire de recherche en sélectionnant Modèles de recherche dans le menu Personnel, puis en cliquant dans le menu Fonctions du formulaire. Cliquez sur Modifier ou sur Propriétés dans le menu Fonctions, puis sur Spécifique. Modifiez la configuration du formulaire de recherche à partir de l'onglet Spécifique de la page Propriétés du formulaire, puis cliquez sur le bouton Mettre à jour pour appliquer vos modifications.
