Pour choisir un formulaire de recherche :

  1. Sélectionnez Rechercher dans le menu Outils.

  2. Dans la liste déroulante Utiliser ce formulaire de recherche, cliquez sur le nom d'un formulaire dans l'une des sections suivantes :

    • Formulaires de recherche personnels

    • Formulaires de recherche système

 

Astuce :
Pour définir un Formulaire de recherche par défaut, cliquez sur le bouton Enregistrer en tant que formulaire, cochez la case Définir comme formulaire de recherche par défaut puis cliquez sur Enregistrer. Si vous vous connectez en tant qu'utilisateur Admin, votre formulaire par défaut est toujours le modèle système par défaut et la case à cocher Définir comme formulaire de recherche par défaut n'est pas disponible.

Vous pouvez revenir au Formulaire de recherche par défaut à tout moment en cliquant sur Rétablir les valeurs par défaut dans la liste déroulante Utiliser ce formulaire de recherche.
