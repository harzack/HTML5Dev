Pour enregistrer une recherche :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche Content Server, formulez une recherche.

  3. Dans la liste déroulante Outils de recherche,cliquez sur Enregistrer votre recherche.

  4. Cliquez sur le bouton.

  5. Sur la page Enregistrer la requête de recherche, spécifiez le nom de la requête, l'emplacement dans lequel vous souhaitez l'enregistrer et toute autre information souhaitée.

  6. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez également enregistrer une requête en cliquant sur le bouton Enregistrer la requête de la page Résultats de recherche.
