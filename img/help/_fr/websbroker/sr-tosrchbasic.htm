Pour effectuer une recherche à l'aide d'une requête de base :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche Content Server, saisissez votre requête dans le champ Texte intégral.

  3. Spécifiez les critères de recherche souhaités.

    Si vous souhaitez utiliser un opérateur booléen, cliquez sur le bouton Dupliquer , puis choisissez un opérateur.

  4. Cliquez sur Rechercher.

  5. Pour configurer l'affichage des résultats sur la page Résultats de recherche, cliquez sur le menu déroulant Sélectionner le style d'affichage, puis sélectionnez Modifier les options d'affichage pour ouvrir la page Options d'affichage et sélectionner l'affichage de vos résultats de recherche.

 

Note :
Lorsque vous spécifiez un intervalle de dates pour les catégories ou les attributs personnalisés d'une catégorie, vous pouvez laisser vide la date dans l'une des listes Du ou Au. Si vous ne mentionnez rien dans la liste Du, Content Server recherche toutes les dates antérieures à la date figurant dans la liste Au. Si vous ne mentionnez rien dans la liste Au, Content Server effectue la recherche à partir de la date sélectionnée dans la liste Du jusqu'à aujourd'hui.

Lorsque vous recherchez des dates dans les composants Attributs système et Catégories, vous pouvez sélectionner l'option « vide ». Il s'agit du paramètre par défaut qui exclut le champ date de la requête de recherche.

Lorsque vous recherchez des dates dans le composant Attributs système, si vous spécifiez Les 3 derniers mois ou Les 6 derniers mois dans la liste déroulante Date, l'intervalle de dates défini dans la requête ne correspond pas exactement à 3 ou 6 mois. Le calcul s'effectue plutôt avec, respectivement, 90 jours et 180 jours. Certains mois comptant plus ou moins de 30 jours, la requête effectuée sur cet intervalle de dates peut ne pas afficher certains éléments qui devraient normalement apparaître.

Vous ne pouvez pas rechercher de fichiers XML via une recherche Attributs système. Pour effectuer une recherche sur une région XML, utilisez le composant Types XML pour sélectionner la région DTD.
