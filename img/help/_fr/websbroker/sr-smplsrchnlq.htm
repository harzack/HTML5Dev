Les exemples suivants utilisent la page Recherche Content Server accessible à partir de la barre de menu générale. Pour plus d'informations sur la barre de menu générale, voir Interface utilisateur.

 

Astuce :
Vous pouvez également accéder à la page Recherche Content Server en cliquant sur la flèche vers le bas Ouvrir le panneau de recherche à partir de la barre de recherche Content Server, puis en cliquant sur Recherche avancée.

 

Pour des informations générales sur la recherche à l'aide de requêtes en langage naturel, voir Recherche à l'aide de requêtes en langage naturel.

  • Rechercher la locution exacte Me montrer des documents relatifs à OpenText.

  • Rechercher les documents relatifs à la conférence utilisateur créés il y a environ deux semaines.

  • Rechercher les discussions relatives aux ventes effectuées après l'été 2003.

  • Rechercher les dossiers qui ont pour titre compensation.

Pour rechercher la locution exacte Me montrer des documents relatifs à OpenText :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Options avancées de la page de recherche Content Server, cliquez sur Requête en langage naturel.

  3. Dans la section Requête en langage naturel, procédez comme suit :

    1. Dans la liste Mode, sélectionnez Extraction de mots-clés ou Content ServerAware .

    2. Dans le champ Saisir une question ou du texte, saisissez "Me montrer des documents relatifs à OpenText", avec les guillemets.

  4. Cliquez sur Rechercher.

Pour rechercher les documents relatifs à la conférence utilisateur créés il y a environ deux semaines :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Options avancées de la page Recherche Content Server, cliquez sur Requête en langage naturel.

  3. Dans la section Requête en langage naturel, procédez comme suit :

    1. Dans la liste Mode, cliquez sur Extraction de mots-clés ou Content ServerAware .

    2. Dans le champ Saisir une question ou du texte, saisissez Trouver des documents relatifs à la conférence utilisateur créés il y a deux semaines.

  4. Cliquez sur Rechercher.

Pour rechercher les discussions relatives aux ventes effectuées après l'été 2003 :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Options avancées de la page de recherche Content Server, cliquez sur Requête en langage naturel.

  3. Dans la section Requête en langage naturel, procédez comme suit :

    1. Dans la liste Mode, cliquez sur Extraction de mots-clés ou Content ServerAware .

    2. Dans le champ Saisir une question ou du texte, saisissez Me montrer toutes les discussions relatives aux ventes effectuées après l'été 2003.

  4. Cliquez sur Rechercher.

     

    Note :
    Pour Content Server, l'été est défini comme toute date comprise entre le 21 juin et le 21 septembre.

     

Pour rechercher les dossiers qui ont pour titre compensation :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Options avancées de la page de recherche Content Server, cliquez sur Requête en langage naturel.

  3. Dans la section Requête en langage naturel, procédez comme suit :

    1. Dans la liste Mode, cliquez sur Extraction de mots-clés ou Content ServerAware .

    2. Dans le champ Saisir une question ou du texte, saisissez Où sont les dossiers qui ont pour titre compensation ? .

  4. Cliquez sur Rechercher.

     

    Note :
    Content Server reconnaît que compensation est le titre du dossier car compensation est saisi après le mot titre. Content Server recherche spécifiquement le mot compensation dans les titres plutôt que dans les résumés, les descriptions ou le corps des documents. Il est également possible d'effectuer des recherches uniquement dans les résumés ou dans les descriptions.
