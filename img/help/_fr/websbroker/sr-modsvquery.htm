Pour modifier une recherche enregistrée :

  1. Recherchez l'emplacement de Content Server où se trouve la recherche précédemment enregistrée, cliquez dans le menu Fonctions de la recherche, sélectionnez Propriétés, puis Spécifique.

  2. Modifiez la requête dans l'onglet Spécifique de la page Propriétés de la requête de recherche.

  3. Cliquez sur Mettre à jour.

 

Astuce :
Cliquez sur Rechercher pour faire une recherche à partir de l'onglet Spécifique.
