Pour créer une recherche de vue personnalisée :

  1. Recherchez l'emplacement de Content Server où se trouve une recherche précédemment enregistrée, cliquez dans le menu Fonctions de la recherche, puis sélectionnez Créer la recherche avec vue personnalisée.

     

    Astuce :
    Pour afficher l'ensemble des recherches répertoriées dans votre espace de travail Personnel ou dans votre espace de travail Projet, cliquez sur le menu Personnel ou Projet, puis sélectionnez Rapports.

     

  2. Pour activer la zone de texte dans laquelle saisir les termes de recherche, dans la section Texte intégral de la page Créer la recherche avec vue personnalisée, sélectionnez la case à cocher Afficher.

  3. Pour activer la zone de texte dans laquelle saisir une question ou du texte, dans la section Requête en langage naturel, sélectionnez la case à cocher Afficher.

  4. Pour activer la vue de recherche Content Server que vous souhaitez rechercher, dans la section Vues de recherche, sélectionnez la case à cocher Afficher. Si vous ne sélectionnez pas explicitement une vue de recherche Content Server, la vue de recherche Entreprise est recherchée par défaut.

     

    Note :
    Si une vue de recherche est supprimée, elle apparaîtra toujours dans la liste déroulante des vues de recherche et pourra être renvoyée dans un appel de fonction func=search.getslices. Ceci est dû aux caractéristiques de mémoire cache dans Content Server. Lorsque la mémoire cache sera nettoyée de toutes les threads actives, la vue de recherche n'apparaîtra plus.

     

  5. Pour choisir un emplacement Content Server à saisir dans la zone de texte, dans la section Emplacement, sélectionnez la case à cocher Afficher.

  6. Dans la section Options, sélectionnez toutes les cases à cocher Afficher pour :

    • Afficher un Titre de recherche avec vue personnalisée. Saisir le texte à afficher.

    • Afficher les Instructions pour la recherche avec vue personnalisée. Saisir les instructions à afficher.

    • Afficher l'option Afficher les en-têtes de composants de recherche. Il s'agit des titres des sections décrites précédemment.

    • Sélectionnez le Format des résultats de recherche dans la liste déroulante Sélectionner le style d'affichage. Pour en savoir plus, voir Définition des options d'affichage.

  7. Cliquez sur Enregistrer.

Pour plus d'informations sur les affichages personnalisés, reportez-vous à Utilisation des affichages personnalisés.
