Pour... Effectuez les opérations suivantes...
Trier des résultats de recherche Sur la page Résultats de recherche, cliquez sur une option de tri des résultats dans la liste déroulante en regard du bouton Trier, puis cliquez sur le bouton Trier.
Trier les résultats de recherche à l'aide de Champ multiple Sur la page Résultats de recherche, cliquez sur Champ multiple dans la liste Trier. Sélectionnez ensuite les critères à appliquer en premier. Cliquez ensuite sur le bouton Plus, , pour ajouter d'autres critères de tri. Une fois les critères de tri à appliquer sélectionnés, cliquez sur OK. Enfin, cliquez sur Trier.
Modifier le nombre de résultats de recherche sur une page Sur la page Résultats de recherche, cliquez sur une valeur dans la liste Afficher les éléments.
Filtrer les résultats de recherche en fonction des valeurs de métadonnées Sur la page Résultats de recherche, cliquez sur une ou plusieurs valeurs dans le panneau Filtres de recherche.
Afficher des exemples de termes de recherche dans un résultat de recherche Sur la page Résultats de recherche, cliquez sur le menu Fonctions d'un élément, choisissez Mise en évidence des résultats, puis cliquez sur le lien souhaité dans la fenêtre Mise en évidence.
Enregistrer un lot de résultats de recherche Sur la page Résultats de recherche, sélectionnez la case à cocher correspondant à chaque résultat à ajouter au lot de résultats de recherche, puis cliquez sur Enregistrez vos résultats dans la liste déroulante Sélectionner l'action. Spécifiez ensuite le nom du lot de résultats de recherche, l'emplacement où l'enregistrer et toute autre information que vous souhaitez ajouter, puis cliquez sur le bouton Ajouter.
Ouvrir un lot de résultat de recherche Recherchez l'emplacement dans Content Server où le lot de résultats est enregistré, cliquez sur le menu Fonctions du lot de résultats, puis choisissez Ouvrir.
Collecter tous les résultats de la recherche
•

Sur la page Résultats de recherche, cliquez sur Collecter tous les résultats de recherche dans la liste déroulante Outils de recherche.

•

Sur la page Collecter tous les résultats de recherche, modifiez les paramètres de notification par e-mail, cliquez sur l'option Collection dans la liste déroulante Sélectionner, puis cliquez sur le bouton Envoyer.

Télécharger une collection comme feuille de calcul Après l'ajout de vos résultats de recherche à une collection :
•

Recherchez une collection dans Content Server et sélectionnez la case à cocher correspondant à chaque élément que vous souhaitez télécharger, ou l'ensemble de la collection.

•

Cliquez sur Télécharger comme feuille de calcul dans la liste déroulante Autres actions.

 

Astuce :
Vous pouvez visualiser l'ensemble des lots de résultats de recherche répertoriés dans votre espace de travail Personnel ou dans votre espace de travail Projet en choisissant Rapports dans le menu Personnel.
