Pour effectuer une recherche à l'aide d'une recherche enregistrée précédemment :

  • Recherchez l'emplacement de Content Server où se trouve la recherche précédemment enregistrée, cliquez dans le menu Fonctions de la recherche, puis sélectionnez Exécuter.

 

Astuce :
Pour afficher l'ensemble des recherches répertoriées dans votre espace de travail Personnel ou dans votre espace de travail Projet, cliquez sur le menu Personnel ou Projet, puis sélectionnez Rapports.
