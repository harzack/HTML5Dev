Pour utiliser des filtres de recherche avec la recherche avancée :

  1. Dans le menu Outils de la barre de menu générale, choisissez Rechercher.

  2. Dans le composant Texte intégral (et d'autres composants de votre choix) de la page Recherche avancée Content Server, définissez vos critères de recherche. Pour en savoir plus, consultez le document Pour effectuer une recherche à l'aide de la recherche avancée.

     

    Note :
    Votre administrateur peut personnaliser les filtres de recherche. Par conséquent, les options affichées sur votre site peuvent être différentes de celles mentionnées dans cette section. Pour plus d'informations sur les filtres de recherche de votre site, contactez votre administrateur.

     

  3. Cliquez sur le bouton Rechercher.

  4. Dans le panneau Filtres de recherche de la page Résultats de recherche, sous le filtre Date, cliquez sur J (Jour), S (Semaine), M (Mois) ou A (Année) et sélectionnez une période pour affiner votre recherche.

  5. Sous d'autres types de filtres de recherche, sélectionnez une ou plusieurs valeurs incluant des restrictions supplémentaires. Le cas échéant, cliquez sur les liens Plus... ou Moins... pour étendre ou réduire les choix disponibles.

  6. Cliquez sur le lien Recherche avancée pour revenir à la page Recherche avancée.

  7. Dans le composant Filtres de recherche, affinez davantage votre recherche avec tout autre critère de votre choix. Vous pouvez enlever un filtre de recherche en cliquant sur la petite icône x en regard du composant.

  8. Cliquez à nouveau sur le bouton Rechercher. Si nécessaire, continuez d'affiner votre recherche jusqu'à ce que vous trouviez les résultats souhaités.
