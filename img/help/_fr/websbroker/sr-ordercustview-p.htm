Pour classer les vues personnalisées

  1. Recherchez l'emplacement de Content Server où se trouvent les vues personnalisées ou les recherches précédemment enregistrées, en cliquant dans le menu Fonctions du recueil, puis sélectionnez Organiser les vues personnalisées.

     

    Astuce :
    Pour afficher l'ensemble des vues personnalisées répertoriées dans votre espace de travail Personnel ou dans votre espace de travail Projet, cliquez sur le menu Personnel ou Projet, puis sélectionnez Rapports.

     

  2. Sur la page Organiser les vues personnalisées, cliquez sur chaque objet de vue personnalisée, puis faites-les glisser vers le haut ou vers le bas pour définir leur priorité d'affichage. L'objet de vue personnalisée situé en haut sera affiché en tant qu'élément par défaut dans la liste déroulante Utiliser cette recherche avec vue personnalisée dans le recueil ou l'emplacement.

     

    Note :
    Lorsqu'un objet de vue personnalisée s'affiche en grisé, cela signifie qu'un élément de vue personnalisée existe, mais que vous ne pouvez l'ouvrir puisque vous ne disposez que des autorisations voir. Vous ne pouvez pas faire glisser cet objet vers le haut ou vers le bas, mais vous pouvez classer d'autres objets autour de celui-ci.

    Vous ne pouvez pas voir les éléments Content Server pour lesquels vous ne disposez pas des autorisations voir.

    Pour de plus amples informations sur les autorisations, reportez-vous à la section Autorisations.

     

  3. Cliquez sur OK.

Pour plus d'informations sur les affichages personnalisés, reportez-vous à Utilisation des affichages personnalisés.
