Les exemples de requête suivants illustrent les différentes façons d'effectuer des recherches dans Content Server. Ils sont présentés dans un ordre de complexité croissant de la recherche de documents similaires (un type de recherche Content Server de base) aux fonctions de recherche les plus avancées du langage de requête Content Server.

Chaque titre de la section Sous-rubriques connexes ci-dessous est lié à une rubrique de l'aide de la recherche Content Server. Pour plus d'informations sur un type de recherche spécifique, voir d'aide correspondante dans Recherche Content Server.

 

Note :
Votre administrateur peut personnaliser la barre de recherche. Par conséquent, les options affichées sur votre site peuvent être différentes de celles mentionnées dans cette section. Pour plus d'informations sur la barre de recherche de votre site, contactez votre administrateur.
