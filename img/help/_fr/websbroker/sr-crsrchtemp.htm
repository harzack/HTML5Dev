Pour créer un formulaire de recherche :

  1. Sélectionnez Rechercher dans le menu Outils.

  2. Sur la page de recherche Content Server, modifiez le critère de recherche.

  3. Cliquez sur le bouton Enregistrer en tant que formulaire.

  4. Sur la page Enregistrer en tant que formulaire, saisissez un nom dans le champ Nom du formulaire de recherche pour attribuer un nom au nouveau formulaire.

  5. Cliquez sur Enregistrer.

 

Astuce :
Dans la liste déroulante Utiliser ce formulaire de recherche, vous pouvez également cliquer sur le nom d'un formulaire de recherche dans les listes Formulaires de recherche personnels ou Formulaires de recherche système pour remplacer un modèle de recherche existant.
