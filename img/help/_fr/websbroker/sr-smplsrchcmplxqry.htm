Bien que la plupart de ces requêtes puissent également être constituées en utilisant les éléments figurant sur la page de recherche Content Server, les exemples ci-dessous illustrent comment utiliser le langage de requêtes Content Server pour soumettre des requêtes complexes.

Ces requêtes utilisent la page Recherche Content Server accessible à partir de la barre de menu générale. Pour plus d'informations sur la barre de menu générale, voir Interface utilisateur.

 

Astuce :
Vous pouvez également accéder à la page Recherche Content Server en cliquant sur la flèche vers le bas Ouvrir le panneau de recherche à partir de la barre de recherche Content Server, puis en cliquant sur Recherche avancée.

 

Pour des informations générales sur la recherche à l'aide de requêtes complexes, voir Recherche à l'aide de requêtes complexes.

  • Rechercher toutes les pages d'aide qui font référence à des projets à l'aide de la page de recherche Content Server.

  • Rechercher tous les éléments Content Server contenant l'expression date de la version suivante.

  • Rechercher des éléments Content Server contenant des mots ressemblant au terme vidéo et au moins un mot dérivé de la racine du terme musique.

  • Rechercher des éléments Content Server contenant les synonymes associés aux mots concert ou interprétation et au moins un mot dérivé de la racine du terme musique.

  • Rechercher des éléments Content Server contenant des mots dont le préfixe correspond à interpréter ou à chanter et à des mots ressemblant au terme groupe.

  • Rechercher des éléments Content Server contenant les mots chorégraphie et danse ou des mots ressemblant au terme interprète.

  • Rechercher des éléments Content Server contenant des mots dont le préfixe correspond au terme air ou ciel ou des mots dont le préfixe est le même que dôme ou centre.

  • Rechercher des éléments Content Server dont le nom français contient le mot voiture.

  • Rechercher tous les éléments Content Server n'ayant aucun langage associé dans le champ Description, onglet Général de la page Propriétés d'un élément.

  • Rechercher des éléments Content Server créés par l'utilisateur pdurant.

  • Rechercher des éléments Content Server créés par un utilisateur dont le nom correspond au suffixe net.

  • Rechercher des éléments Content Server réservés par l'utilisateur pdurant et non par l'utilisateur dleroi.

  • Rechercher des éléments Content Server contenant le type MIME image/GIF et non le type MIME application/MSWord.

Pour rechercher toutes les pages d'aide mentionnant des projets à l'aide de la page de recherche Content Server :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche Content Server, dans la section Texte intégral, saisissez projet* dans le champ texte Saisir des termes de recherche.

  3. Dans la section Options avancées, cliquez sur Vues de recherche.

  4. Dans la section Vues de recherche, cliquez sur Aide.

  5. Cliquez sur Rechercher.

Pour rechercher tous les éléments Content Server contenant l'expression date de la version suivante :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez la requête suivante en langage de requêtes Content Server : "date de la version suivante".

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Servercontenant des mots ressemblant au terme vidéo et au moins un mot dérivé de la racine du terme musique :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez (qlphonetic "vidéo") AND (qlstem "musique").

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server contenant les synonymes de concert ou d'interprétation et au moins un mot dérivé du terme musique :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez (qlthesaurus "concert" OR qlthesaurus "interprétation") AND (qlstem "musique").

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server contenant des mots dont le préfixe correspond à interpréter ou à chanter et à des mots ressemblant au terme groupe :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez (qlright-truncation "interpréter" OR qlright-truncation "chanter") AND (qlphonetic "groupe").

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server contenant les mots chorégraphie et danse ou des mots ressemblant au terme interprète :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez ("chorégraphie" et "danse") OR (qlphonetic "interprète").

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server contenant des mots dont le préfixe correspond à ciel ou air ou dont le suffixe correspond à dôme ou à centre :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez (qlright-truncation "ciel" OR qlright-truncation "air") AND (qlleft-truncation "dôme" OR domeqlleft-truncation "centre").

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server dont le nom français contient le mot voiture :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez [qlregion "OTName"] [qlattribute "lang" = "fr"] "voiture".

  3. Cliquez sur Rechercher.

Pour rechercher tous les éléments Content Server n'ayant aucun langage associé dans le champ Description, onglet Général de la page Propriétés d'un élément :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Requête complexe.

    2. Dans le champ texte Saisir des termes de recherche, saisissez [qlregion "OTDComment"] [qlattribute "lang" est nul] .

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server créés par l'utilisateur rduval :

    2. Dans le champ texte Saisir des termes de recherche, saisissez [qlregion "OTCreatedByName"] "pdurant".

  3. Cliquez sur Rechercher.

  3. Cliquez sur Rechercher.

Pour rechercher des éléments Content Server réservés par l'utilisateur pdurant et non par l'utilisateur dleroi :
