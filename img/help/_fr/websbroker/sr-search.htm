Les fonctionnalités de recherche de Content Server permettent de trouver les informations stockées dans Content Server qui vous intéressent. Vous pouvez rechercher l'emplacement de documents, de dossiers, de discussions et d'autres éléments Content Server. Vous pouvez également rechercher l'emplacement d'éléments ayant un langage associé. Une recherche peut simplement consister à trouver des documents similaires à un document dans un espace de travail donné. ou des recherches plus complexes, effectuées par l'intermédiaire de requêtes. Une requête est un élément Content Server qui stocke tous les critères de recherche et les options d'affichage des résultats.

 

Note :
Vous pouvez rechercher des éléments OpenText Enterprise Library à condition de disposer des autorisations appropriées.
