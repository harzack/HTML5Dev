Pour modifier ou supprimer une valeur de meilleurs résultats :

  1. Sur la page Résultats de recherche, cliquez sur le menu Fonctions d'un élément, choisissez Propriétés, puis Général.

  2. Dans l'onglet Général de la page Propriétés de l'élément, modifiez ou supprimez les valeurs dans le champ Valeur de meilleurs résultats, dans la liste Expiration des meilleurs résultats ou dans les deux.

  3. Cliquez sur Mettre à jour.

 

Note :
Une date d'expiration ne s'applique qu'aux valeurs de meilleurs résultats spécifiées.

 
 

Astuce :
Vous pouvez également modifier ou supprimer une valeur de meilleurs résultats directement dans l'élément d'un recueil.
