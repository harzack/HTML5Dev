Les exemples suivants utilisent la page Recherche Content Server accessible à partir de la barre de menu générale. Pour plus d'informations sur la barre de menu générale, voir Interface utilisateur.

 

Astuce :
Vous pouvez également accéder à la page Recherche Content Server en cliquant sur la flèche vers le bas Ouvrir le panneau de recherche à partir de la barre de recherche Content Server, puis en cliquant sur Recherche avancée.

 

Pour des informations générales sur la recherche à l'aide de requêtes de base, voir Recherche à l'aide de requêtes de base.

  • Rechercher tous les documents contenant la locution Version 5 et créés par l'utilisateur SMartin.

  • Rechercher toutes les discussions liées aux échéances dans le projet Nouvelle version.

  • Dans la vue de recherche Entreprise, rechercher les éléments Content Server contenant le terme fournir.

  • Rechercher tous les éléments Content Server contenant le terme Français dans les champs Nom et Description dans l'onglet Général de la page Propriétés d'un élément.

Pour rechercher tous les documents contenant la locution Version 5 et créés par l'utilisateur SMartin :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans la liste Rechercher, cliquez sur Locution exacte.

    2. Dans le champ texte Saisir des termes de recherche, saisissez Version 5.

  3. Dans la section Options avancées, cliquez sur Attributs système. Dans la section Attributs système, procédez comme suit :

    1. Dans la liste Type de contenu, cliquez sur Documents.

    2. Dans le champ Créé par, tapez SMartin.

  4. Cliquez sur Rechercher.

Pour rechercher toutes les discussions liées aux échéances dans le projet Version suivante :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Texte intégral de la page Recherche Content Server, procédez comme suit :

    1. Dans le champ texte Saisir des termes de recherche, saisissez échéances.

    2. Dans la liste Modificateur, cliquez sur Relatif à.

  3. Dans la section Options avancées, cliquez sur Attributs système.

  4. Dans la section Attributs système, puis dans la liste Type de contenu, cliquez sur Discussions.

  5. Dans la section Options avancées, cliquez sur Emplacement.

  6. Dans la section Emplacement, cliquez sur le bouton Parcourir Content Server..., recherchez le projet Version suivante, puis cliquez sur son lien Sélectionner.

  7. Cliquez sur Rechercher.

Pour rechercher les éléments Content Server contenant le mot fournir dans la vue de recherche Entreprise :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Sur la page Recherche Content Server, dans la section Texte intégral, saisissez fournir dans le champ texte Saisir des termes de recherche.

  3. Dans la section Options avancées, cliquez sur Vues de recherche.

  4. Dans la section Vues de recherche, cliquez sur Entreprise.

  5. Cliquez sur Rechercher.

Pour rechercher tous les éléments Content Server contenant le terme Français dans les champs Nom et Description dans l'onglet Général de la page Propriétés d'un élément :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Recherche.

  2. Dans la section Options avancées de la page Recherche Content Server, cliquez sur Attributs système.

  3. Dans la section Attributs système, procédez comme suit :

    1. Dans la liste Attributs supplémentaires, sélectionnez Nom.

    2. Dans la zone de texte Nom, saisissez *.

    3. Cliquez sur la liste N'importe lequel située en regard de Nom et sélectionnez fr.

    4. Dans la liste Attributs supplémentaires, sélectionnez Description.

    5. Dans la zone de texte Description, saisissez *.

    6. Cliquez sur la liste N'importe lequel située en regard de Description et sélectionnez fr.

  4. Cliquez sur Rechercher.
