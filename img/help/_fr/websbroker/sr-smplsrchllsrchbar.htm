Les types de recherches que vous pouvez effectuer à partir de la barre de recherche Content Server dépendent de la configuration définie par votre administrateur. Pour plus d'informations sur la manière dont est configurée la barre de recherche sur votre site Content Server, contactez votre administrateur.

Les exemples suivants sont basés sur les paramètres par défaut de la barre de recherche Content Server. Il est donc possible qu'ils ne reflètent pas les paramètres de la barre de recherche Content Server sur votre site. Pour des informations générales sur l'utilisation de la barre de recherche Content Server, voir Utilisation de la barre de recherche Content Server.

  • Rechercher des éléments Content Server qui comprennent l'URL www.opentext.com.

  • Rechercher toutes les pages de l'aide utilisateur en ligne faisant référence à des projets.

Pour rechercher des éléments Content Server contenant l'URL www.opentext.com à l'aide de la barre de recherche de Content Server :

  1. Dans la barre de recherche Content Server, cliquez sur Poser une question à Content Server dans la liste déroulante.

  2. Dans le champ texte Rechercher à partir d'ici de la barre de recherche Content Server, saisissez Rechercher les éléments contenant l'URL www.opentext.com.

  3. Dans la liste Vues de recherche, cliquez sur Entreprise.

  4. Cliquez sur Aller.

Pour rechercher toutes les pages de l'aide utilisateur en ligne faisant référence à des projets à l'aide de la barre de recherche Content Server :

  1. Dans la liste déroulante de la barre de recherche Content Server, cliquez sur Rechercher Content Server.

  2. Saisissez projet* dans le champ texte Rechercher à partir d'ici.

  3. Dans la liste Vues de recherche, cliquez sur Aide.

  4. Cliquez sur Aller.
