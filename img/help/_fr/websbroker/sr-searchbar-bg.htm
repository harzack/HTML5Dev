La barre de recherche Content Server est un outil de recherche pratique et rapide vous permettant d'effectuer des recherches sur quasiment toutes les pages Content Server. Le tableau ci-dessous décrit la barre de recherche Content Server. Votre administrateur peut néanmoins personnaliser la barre de recherche. Par conséquent, les options affichées sur votre site peuvent être différentes de celles mentionnées dans cette section. Pour plus d'informations sur la barre de recherche de votre site, contactez votre administrateur.

La barre de recherche Content Server apparaît, par défaut, sur la plupart des pages à l'extrême droite de la barre de menu générale. Observez les éléments suivants de gauche à droite de la barre de recherche :

  • le texte Rechercher,

  • la zone de saisie « Rechercher à partir d'ici »,

  • le bouton « Exécuter la recherche »,

  • et la flèche vers le bas « Ouvrir le panneau de recherche » qui permet de développer la barre de recherche en panneau de recherche. Si vous développez le panneau de recherche, la flèche vers le bas devient une flèche vers le haut appelée « Fermer le panneau de recherche ».

Si vous saisissez du texte dans le champ Rechercher et que vous cliquez sur le bouton de recherche, Content Server recherche le texte saisi dans cette page.

La barre de recherche peut être développée en panneau de recherche en cliquant sur la flèche vers le bas située à droite du bouton de recherche. Selon la configuration de la fonction de recherche définie par votre administrateur, les éléments suivants peuvent apparaître dans le panneau de recherche :

Elément Content Server Fonction
Champ Rechercher Permet d'exécuter une recherche de texte intégral du texte saisi dans le champ.
Liste Type de recherche Permet de sélectionner une des options suivantes : Texte intégral, Surnom et Requête en langage naturel. Selon votre sélection, le panneau de recherche affiche dynamiquement les options applicables au type de recherche choisi. Pour plus d'informations, reportez-vous aux rubriques Attribution de surnoms aux éléments et Recherche à l'aide de requêtes en langage naturel.
Liste Vues de recherche. Disponible lors d'une recherche Texte intégral ou Requête en langage naturel. Permet de sélectionner une des options suivantes : A partir d'ici, A partir d'ici (étendue), Aide, Entreprise et Entreprise [Toutes versions]. Cette option vous permet de spécifier la vue de recherche Content Server dans laquelle vous souhaitez effectuer une recherche. Vous pouvez effectuer votre recherche dans le dossier ou l'espace de travail dans lequel vous vous trouvez en sélectionnant l'option A partir d'ici, ou sur l'ensemble de l'espace Entreprise.
 

Note :
Si une vue de recherche est supprimée, elle apparaîtra toujours dans la liste déroulante des vues de recherche et pourra être renvoyée dans un appel de fonction func=search.getslices. Ceci est dû aux caractéristiques de mémoire cache dans Content Server. Lorsque la mémoire cache sera nettoyée de toutes les threads actives, la vue de recherche n'apparaîtra plus.

 
Liste Action. Disponible lors d'une recherche Surnom. Cette option vous permet de sélectionner Ouvrir ou Propriétés. Si vous sélectionnez Ouvrir, votre recherche renvoie l'élément. Si vous sélectionnez Propriétés, votre recherche renvoie la page Propriétés de l'élément. Pour plus d'informations, consultez la section Attribution de surnoms aux éléments.
Liste Mode. Disponible lors d'une recherche Texte intégral ou Requête en langage naturel. Lors d'une recherche Texte intégral, Mode vous permet d'affiner votre recherche en sélectionnant l'une des options suivantes : Tous les mots, N'importe quel mot, Locution exacte ou Requête complexe. Lors d'une recherche Requête en langage naturel, Mode vous permet d'affiner votre recherche en sélectionnant l'une des options suivantes : Extraction de mots clés ou Content Server Aware. Pour plus d'informations, consultez la section Recherche à l'aide de requêtes en langage naturel.
Liste Modificateur. Disponible lors d'une recherche Texte intégral. Permet d'affiner votre recherche en sélectionnant l'une des options suivantes : Aucun, Synonymes de, Relatif à, Recherche phonétique, Mot commençant par ou Mot se terminant par.
Liste Type de contenu. Disponible lors d'une recherche Texte intégral. Lors d'une recherche en texte intégral, le Type de contenu vous permet de spécifier le type d'élément que vous souhaitez rechercher. Par exemple, vous pouvez sélectionner des formats de fichier particuliers ou des types d'élément Content Server. Votre administrateur définit les types d'objets pouvant être recherchés sur votre système.
Liste Créé par. Disponible lors d'une recherche Texte intégral. Permet d'affiner votre recherche en sélectionnant l'une des options suivantes : Tous, Créé par moi-même ou Créé par mon groupe. Si cette option de recherche n'est pas disponible, contactez votre administrateur.
Liste Date de dernière modification. Disponible lors d'une recherche Texte intégral. Vous permet de limiter le champ de votre recherche à une période précise, ou vue de recherche, du type Modifié hier ou Modifié le mois dernier. Si cette option de recherche n'est pas disponible, contactez votre administrateur.
Lien Aide Affiche la page d'aide associée au panneau de recherche. Ce lien apparaît sur la barre de recherche uniquement si votre administrateur a activé cette option.
Lien Résultats de la dernière recherche. Ouvre la page Résultats de votre dernière recherche. Ce lien apparaît sur la barre de recherche uniquement si votre administrateur a activé cette option.
Lien Recherche avancée Disponible lors d'une recherche Texte intégral ou Requête en langage naturel. Ouvre la page de recherche avancée. Ce lien apparaît sur la barre de recherche uniquement si votre administrateur a activé cette option.

 

Note :
Lorsque vous vous trouvez dans un espace de travail qui contient des éléments OpenText Enterprise Library, vous pouvez utiliser l'option A partir d'ici de la barre de recherche pour renvoyer un dossier, des variantes et des éléments OpenText Enterprise Library à partir de votre espace de travail actuel. Pour en savoir plus, voir Using the Search Bar to Find Open Text Enterprise Library Services Items.

 

Lorsque vous utilisez la barre de recherche, les termes de recherche spécifiés sont conservés dans la barre de recherche sur la page Résultats de recherche. Cela vous permet d'affiner la recherche en utilisant des termes plus précis sans quitter la page Résultats de recherche. Pour des exemples de requêtes à l'aide de la barre de recherche, voir la section Recherche à l'aide de la barre de rechercheContent Server dans la rubrique de référence Exemples de requête.

 

Note :
En utilisant la page de Recherche Content Server pour formuler des requêtes plus complexes, les termes de recherche spécifiés sur la page de Recherche Content Server ne sont pas conservés dans la barre de recherche de la page Résultats de recherche.
