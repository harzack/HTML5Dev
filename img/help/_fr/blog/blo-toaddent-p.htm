Pour ajouter une entrée de blog :

  1. Sur la page principale d'un blog, cliquez sur le lien Publier une entrée dans le bandeau Console du blog.

  2. Sur la page Ajouter une entrée de blog, saisissez l'intitulé de l'entrée dans le champ Titre.

  3. Saisissez le contenu de l'entrée dans le champ Entrée.

     

    Note :
    Vous pouvez copier le texte d'un document Word et le coller dans une entrée de blog. Pour plus d'informations, voir Pour coller du contenu à partir d`autres éditeurs de texte.

     

  4. Pour appliquer un ou plusieurs sujets à l'entrée, vous pouvez sélectionner les cases à cocher correspondantes dans la section Sujet.

     

    Note :
    Pour pouvoir sélectionner un sujet, des sujets doivent avoir été ajoutés à la page des paramètres du blog. Pour plus d'informations, voir Pour créer et configurer un blog.

     

  5. Cliquez sur l'un des boutons radio suivants :

    • Publier, pour publier l'entrée.

    • Brouillon, pour enregistrer l'entrée sous la forme d'un brouillon.

  6. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  7. Cliquez sur Terminer.
