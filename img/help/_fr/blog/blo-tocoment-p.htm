Pour commenter une entrée de blog :

  1. Sur la page principale du blog, cliquez sur le lien Ajouter un commentaire pour l'entrée.

  2. Saisissez le contenu du commentaire dans le champ Commentaires.

  3. Cliquez sur l'icône que vous souhaitez afficher en regard du commentaire dans la zone Icône Commentaire.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Soumettre.

 

Astuces :

  • Une fois que vous avez ajouté vos commentaires, la fenêtre Ajouter des commentaires reste ouverte. Vous pouvez ainsi consulter les autres commentaires du blog. Pour revenir à la page principale du blog, vous devez fermer la fenêtre Ajouter des commentaires en cliquant sur le bouton Fermer en haut à droite de la fenêtre.

  • Pour modifier un commentaire après l'avoir inséré, cliquez sur l'icône Fonctions du commentaire, puis sélectionnez Modifier l'entrée. Apportez les modifications nécessaires, puis cliquez sur Envoyer.

  • Pour afficher tous les commentaires publiés, cliquez sur le lien Afficher les commentaires sur la page principale du blog.
