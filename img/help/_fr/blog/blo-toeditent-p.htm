Pour modifier une entrée de blog :

  1. Sur la page principale du blog, cliquez sur l'icône Fonctions de l'entrée, puis sur Modifier l'entrée.

  2. Sur la page Modifier l'entrée de blog, effectuez l'une des actions suivantes :

    • Modifiez le titre et le contenu de l'entrée.

    • Activez ou désactivez une ou plusieurs cases à cocher de la section Sujet, le cas échéant.

    • Cliquez sur la case d'option Publier ou Brouillon pour changer le statut de publication.

    • Activez ou désactivez la case à cocher Avertir les abonnés que cette entrée a été modifiée, le cas échéant.

    • Sélectionnez un type d'élément dans la liste Ajouter un élément pour ajouter une pièce jointe à l'entrée. Complétez les informations requises, puis cliquez sur Ajouter.

  3. Cliquez sur Soumettre.
