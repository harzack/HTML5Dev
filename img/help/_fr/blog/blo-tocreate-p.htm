Pour créer et configurer un blog :

  1. Cliquez sur Blog dans le menu Ajouter un élément.

  2. Sur la page Ajouter un blog, saisissez le titre du blog dans le champ Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Créer.

  5. Sur la page Paramètres du blog, sélectionnez les autorisations appropriées dans les listes Lecture et Ecriture.

  6. Sélectionnez ou désélectionnez les cases à cocher suivantes :

    • Activer les commentaires

    • Activer les évaluations

  7. Dans la zone Abonnements, procédez de la façon suivante :

    • Sélectionnez la case à cocher Activer les abonnements par e-mail.

    • Saisissez une adresse e-mail dans le champ Adresse e-mail de ce blog.

  8. Dans la section Afficher le dernier, effectuez l'une des actions suivantes pour indiquer le nombre d'entrées à afficher sur la page principale du blog :

    • Saisissez un nombre dans le champ Afficher les dernières, puis cliquez sur entrées du jour ou entrées dans la liste.

    • Sélectionnez la case à cocher Afficher toutes les entrées.

  9. Dans la zone Ordonner par, cliquez sur l'une des options suivantes pour indiquer l'ordre des entrées affichées :

    • date de création ;

    • date de dernière mise à jour ;

    • auteur ;

    • titre.

  10. Dans la section URL, saisissez le nom d'une URL à ajouter à la page principale du blog dans le champ URL, puis saisissez l'adresse de l'URL dans le champ Lien. Pour ajouter d'autres URL, cliquez sur le bouton Ajouter une URL .

  11. Tapez le nom d'un sujet de blog dans le champ Sujets de l'entrée. Pour ajouter d'autres sujets, cliquez sur le bouton Ajouter des sujets d'entrée .

  12. Cliquez sur Soumettre.
