Pour modifier les paramètres du blog :

  1. Cliquez sur l'icône Fonctions du blog, puis cliquez sur Modifier les paramètres.

  2. Sur la page Paramètres du blog, renommez le blog en tapant un nouveau nom dans le champ Nom.

  3. Ajoutez ou modifiez la description du blog dans le champ Description.

  4. Modifiez les paramètres d'options avancées de votre choix.

  5. Cliquez sur Soumettre.

 

Astuce :
Pour modifier les paramètres d'un blog depuis sa page principale, cliquez sur le lien Modifier les paramètres du bandeau Console du blog.
