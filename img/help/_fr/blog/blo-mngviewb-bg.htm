Outre l'ajout des blogs aux favoris, qui vous permet de gérer une liste de vos blogs favoris parallèlement aux autres éléments favoris, vous pouvez afficher et gérer votre liste de blogs personnels. Cette liste comporte tous les blogs que vous avez créés.

Lorsque vous consultez un blog, vous pouvez le filtrer, ce qui signifie qu'il n'affichera que les entrées qui remplissent certains critères. Par exemple, vous pouvez effectuer le filtrage par sujet de blog, entrée, archive hebdomadaire (toutes les entrées d'une semaine donnée) et mots clés (une série de mots que vous spécifiez et qui doivent apparaître dans les entrées filtrées).

Vous pouvez également générer une sortie RSS (Really Simple Syndication) pour un blog. RSS est un format de données XML normalement utilisé pour la publication de contenu numérique fréquemment mis à jour.

 

Note :
Votre administrateur doit activer la sortie RSS pour les blogs. Contactez votre administrateur si cette option n'est pas disponible.
