Lorsque vous vous abonnez à un blog, vous vous inscrivez pour être notifié par e-mail de toute modification de son contenu (par exemple, lorsque de nouvelles entrées sont ajoutées ou lorsque des entrées existantes sont modifiées). Le créateur d'un blog doit activer les abonnements pour les rendre possibles. Si vous disposez de l'autorisation de suppression sur le blog, vous pouvez afficher une liste des utilisateurs abonnés à ce blog.

 

Note :
Pour pouvoir utiliser les abonnements, le module eLink doit être installé et activé. Contactez votre administrateur pour activer eLink. Pour des informations sur eLink, reportez-vous à la section Présentation d`eLink.

Pour être notifié des nouvelles entrées de blog, modifiez vos paramètres de notifications. Pour de plus amples informations sur les notifications, reportez-vous à la section Utilisation des notifications.
