Normalement, vous ne devez pas modifier un nom d'URL. Toutefois, comme le nom d'URL d'une collection apparaît dans l'URL de chaque élément qu'il contient, donner un nom d'URL court à une collection ayant un libellé d'affichage long peut simplifier la gestion des URL dans les collections.

Vous pouvez modifier manuellement un nom d'URL d'un élément sur la page Propriétés de WebDAV sans modifier le libellé d'affichage.

 

Note :
Une fois que vous avez modifié un nom d'URL sur la page Propriétés de WebDAV, le nom d'URL n'est plus synchronisé automatiquement avec le libellé d'affichage.
