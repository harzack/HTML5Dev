Pour récupérer une ressource document Content Server à l'aide de WebDAV :

  1. Accédez à un document dans Content Server.

  2. Cliquez sur l'icône Fonctions du document, puis sélectionnez Modifier. Le document s'ouvre dans son application d'origine.

 

Note :
La connexion de cette application à Content Server peut requérir la saisie d'informations d'authentification supplémentaires. Pour simplifier l'utilisation de cette fonction, enregistrez votre mot de passe pour toutes les invites de ce type.
