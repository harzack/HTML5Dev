A chaque élément Content Server accessible via WebDAV correspond un onglet WebDAV sur la page Propriétés de cet élément dans Content Server. L'onglet WebDAV indique l'URL WebDAV pour l'élément Content Server.

Si le composant Dossiers Web est installé sur votre ordinateur et que vous cliquez sur le lien URL WebDAV sur la page Propriétés WebDAV d'un dossier Content Server, Microsoft Internet Explorer 5.5 (ou version supérieure) le reconnaît comme collection WebDAV et l'ouvre avec le client WebDAV adapté. Le scénario suivant illustre comment un client WebDAV est utilisé pour ouvrir un document Content Server.

  • Vous ouvrez un document Content Server créé à l'origine dans Microsoft Word, à l'aide du lien URL WebDAV de la page Propriétés WebDAV du dossier Content Server qui le contient. Comme Dossiers Web est installé sur votre ordinateur, Microsoft Internet Explorer 5.5 (ou version supérieure) ouvre le dossier en tant que dossier Web. Dans le dossier Web, vous pouvez double-cliquer sur le document pour l'ouvrir dans Microsoft Word et le modifier plutôt que d'ouvrir le document comme une page Web en lecture seule.

 

Note :
Tous les navigateurs peuvent ne pas reconnaître automatiquement les ressources WebDAV. Pour plus d'informations, reportez-vous à la section Récupération d'une ressource document Content Server.
