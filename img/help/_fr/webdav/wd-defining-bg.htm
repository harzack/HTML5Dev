WebDAV (Web-based Distributed Authoring and Versioning, création de versions et de contenu distribuée sur le Web) est un ensemble d'extensions au protocole HTTP qui permet aux utilisateurs de gérer conjointement des fichiers situés sur des serveurs Web distants.

WebDAV prend en charge des équipes de développement virtuelles en fournissant une infrastructure qui rend le logiciel client accessible via Internet. Le module Content Server WebDAV permet aux rédacteurs et aux éditeurs d'accéder directement aux référentiels Content Server grâce à un client WebDAV. Un client WebDAV est tout produit tiers pouvant communiquer avec Content Server via le serveur WebDAV. Il peut s'agir de : Microsoft Office 2003/2007/2010/2013, Dossiers Web Microsoft, les produits Adobe tels que Photoshop, Acrobat et GoLive, ainsi que Riverfront WebDrive.
