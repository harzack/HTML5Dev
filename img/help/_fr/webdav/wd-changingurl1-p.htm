Pour modifier un nom d'URL sans modifier le libellé d'affichage :

  1. Dans Content Server, cliquez sur l'icône Fonctions d'un élément, sélectionnez Propriétés, puis WebDAV.

  2. Dans l'onglet WebDAV de la page Propriétés de l'élément, saisissez un nouveau nom dans le champ Nom de l'URL.

  3. Cliquez sur Mettre à jour.
