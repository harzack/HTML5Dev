Pour récupérer une ressource document Content Server en utilisant Microsoft Word comme client WebDAV :

  1. Dans Content Server, cliquez sur l'icône Fonctions d'un document, choisissez Propriétés, puis sélectionnez WebDAV pour obtenir l'URL de la ressource document.

  2. Ouvrez Microsoft Word, puis cliquez sur Ouvrir dans le menu Fichier.

  3. Saisissez l'URL de la ressource document dans le champ Nom du fichier, puis cliquez sur Ouvrir.

  4. Recherchez le document et double-cliquez sur son nom.
