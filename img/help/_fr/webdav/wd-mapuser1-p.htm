Lorsqu'un mappage Géré par l'utilisateur est appliqué, vous pouvez utiliser cette page pour modifier les noms principaux mappés sur un utilisateur Content Server pour accéder à des ressources WebDAV.

Pour ajouter un mappage :

  1. Saisissez un nom principal dans le champ Nom principal.

  2. Saisissez votre mot de passe Content Server dans le champ Mot de passe Content Server pour.

  3. Cliquez sur Ajouter.

     

    Note :
    Si vous êtes invité à saisir votre nom d'utilisateur et mot de passe, saisissez le nom principal dans le champ Nom de l'utilisateur, par exemple moi@LEDOMAINE.NET. Dans cet exemple, il faut saisir le mot de passe correspondant au nom d'utilisateur « moi » du domaine « LEDOMAINE.NET » dans le champ Mot de passe. Cliquez sur OK pour terminer l'authentification.

     

Pour supprimer un mappage :

  1. Cochez la case d'un nom principal.

  2. Saisissez votre mot de passe Content Server dans le champ Mot de passe Content Server pour.

  3. Cliquez sur Supprimer.

 

Note :
Le champ Mot de passe Content Server pour n'est pas visible pour les utilisateurs disposant des privilèges d'administration.
