Les ressources WebDAV se présentent sous forme de pages Web et l'interface Content Server est essentiellement une page Web. Cependant, les ressources WebDAV diffèrent des pages Web Content Server car elles permettent aux utilisateurs disposant des autorisations appropriées d'ouvrir et de modifier un élément Content Server dans son programme d'origine.

Récupération des ressources document Content Server par les clients WebDAV

Un client WebDAV récupère les ressources document Content Server avec WebDAV exactement de la même manière qu'un navigateur Web récupère les pages Web. Vous pouvez utiliser n'importe quel navigateur Web pour télécharger des éléments Content Server. Il suffit pour cela de saisir l'URL WebDAV de l'élément Content Server dans la barre d'adresses du navigateur ou de cliquer sur le lien de l'élément sur la page Propriétés WebDAV de l'élément Content Server.

Pour un navigateur Web, une ressource document ressemble à n'importe quelle autre page Web. Malheureusement, comme les navigateurs Web ne font pas la différence entre les ressources document et les pages Web, le navigateur Web ne saura pas toujours qu'un document se trouve sur un serveur WebDAV. Lorsque cette information n'est pas transmise au client, il est possible que l'élément ne s'ouvre pas automatiquement dans son application d'origine.

Récupération des ressources collection Content Server par les clients WebDAV

Une fois le module WebDAV installé sur un serveur Content Server, ce dernier devient une collection WebDAV qui comporte tout le contenu du serveur Content Server. Dans WebDAV, une ressource collection correspond aux éléments suivants : dossiers, documents composites, chaînes, discussions, projets, tâches et listes de tâches.

L'accès aux ressources collection WebDAV diffère de l'accès aux ressources document. Lorsque vous récupérez une collection en saisissant son URL dans un navigateur Web, votre navigateur vous informe que l'élément est une collection WebDAV et que vous devez y accéder via un client WebDAV.

La page Web affiche le même lien URL WebDAV que celui qui apparaît sur la page Propriétés WebDAV de l'élément. Certains navigateurs ouvrent automatiquement la ressource avec WebDAV lorsque vous cliquez sur le lien URL WebDAV.

Avec la plupart des clients WebDAV, vous vous connectez au serveur WebDAV à l'aide de l'URL d'une collection qui contient les éléments que vous souhaitez récupérer.
