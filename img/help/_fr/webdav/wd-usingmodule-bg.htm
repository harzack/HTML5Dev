WebDAV est pris en charge par plusieurs systèmes d'exploitation et applications bureautiques. Lorsque Content Server est installé sur un système, il existe deux serveurs : le serveur principal et le serveur d'administration. Une fois le module WebDAV installé sur Content Server, le serveur principal devient une collection WebDAV qui comporte tout le contenu de ce serveur.

L'utilisation de WebDAV pour accéder à Content Server ne nécessite ni le téléchargement ni l'installation d'un logiciel client Content Server. Tous les objets accessibles via Content Server comprennent une page Propriétés WebDAV dans Content Server fournissant l'URL de l'élément. Pour plus d'informations sur les URL WebDAV, reportez-vous à la rubrique Recherche de l'URL WebDAV pour les éléments Content Server.
