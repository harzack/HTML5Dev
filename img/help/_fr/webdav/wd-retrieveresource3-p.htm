Pour récupérer un élément Content Server à l'aide d'un autre client WebDAV :

  1. Dans Content Server, cliquez sur l'icône Fonctions d'un élément, sélectionnez Propriétés, puis sélectionnez WebDAV pour obtenir l'URL WebDAV de l'élément dans l'onglet WebDAV.

  2. Procédez de l'une des manières suivantes :

    • Cliquez sur URL WebDAV dans l'onglet WebDAV sur la page Propriétés de l'élément.

    • Saisissez l'URL WebDAV de l'élément dans la fenêtre de votre navigateur Internet.

  3. Double-cliquez sur l'élément. Si le client WebDAV approprié est disponible pour WebDAV, l'élément s'ouvre dans son application d'origine.
