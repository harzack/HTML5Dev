WebDAV vous permet de modifier des documents directement via WebDAV. La fonction Modification WebDAV est disponible pour les documents créés dans Microsoft Word, Microsoft PowerPoint et Microsoft Excel. Pour l'utiliser, vous devez installer l'API client Microsoft SharePoint et Dossiers Web Microsoft et activer les scripts ActiveX de votre navigateur. L'application Microsoft Office du document que vous modifiez doit également être installée sur votre ordinateur. L'action échouera si l'application est définie sur une installation à la première utilisation.

Il est également possible de récupérer des ressources document de différentes manières en fonction du client WebDAV utilisé pour ouvrir la ressource. Par exemple, les utilisateurs peuvent ouvrir un document Content Server en utilisant Microsoft Word comme client WebDAV.
