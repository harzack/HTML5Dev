Comme WebDAV prend en charge de nombreuses applications, travailler sur des documents sur le Web devient aussi simple que de travailler sur disque dur. Le module Content Server WebDAV vous permet de copier et de créer des fichiers dans Content Server, ainsi que de modifier des fichiers Content Server existants. WebDAV réserve automatiquement un fichier lorsqu'un utilisateur l'ouvre pour qu'aucune modification ne soit apportée au même moment par d'autres utilisateurs. Lorsque l'utilisateur ferme le fichier, WebDAV le déverrouille automatiquement et ajoute une nouvelle version. Vous pouvez afficher la structure de fichier en mode d'affichage dossier WebDAV, Content Server ou Microsoft Dossiers Web.

L'affichage du dossier WebDAV requiert l'installation de Microsoft® Internet Explorer version 5.5 ou ultérieure et des dossiers Web Microsoft sur votre ordinateur. Vous devez également définir le paramètre de sécurité Internet Explorer qui active l'ouverture des éléments à partir d'un IFRAME (Lancement des applications et des fichiers dans un IFRAME). Si vous définissez ce paramètre sur Demander, vous serez invité par Internet Explorer à confirmer chaque action que vous entreprendrez.

Le client Dossiers Web est intégré à un IFRAME dans l'Explorateur Windows. Cela peut entraîner des invites d'authentification supplémentaires. Pour simplifier l'utilisation de cette fonction, enregistrez votre mot de passe pour toutes les invites de ce type.

 

Note :
De la même manière que dans Content Server, les autorisations Content Server limitent vos possibilités d'afficher les objets de WebDAV ou d'y accéder. Lorsque vous répertoriez le contenu du serveur Content Server dans WebDAV, les espaces de travail personnels des autres utilisateurs ne s'affichent pas, même si vous pouvez y accéder depuis Content Server.
