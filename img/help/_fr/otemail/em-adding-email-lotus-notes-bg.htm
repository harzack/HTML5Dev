Contrairement à l'ouverture d'un e-mail provenant d'Outlook dans Content Server, lorsque vous ouvrez un message Lotus Notes dans Content Server, celui-ci est d'abord importé dans la base de données Lotus Notes, puis ouvert. Les messages Lotus Notes que vous ouvrez dans Content Server restent dans votre base de données jusqu'à leur suppression.
