Pour rechercher un ensemble personnalisé de propriétés d'e-mails :

  1. Dans le menu Outils de Content Server, cliquez sur Recherche.

  2. Dans Ajouter au formulaire de recherche, cliquez sur E-mail (recherche avancée).

  3. Dans la liste Attributs supplémentaires, cliquez sur la propriété que vous souhaitez rechercher.

  4. Saisissez dans le champ de texte le critère de recherche pour la propriété.

  5. Facultatif Cliquez sur une option dans la liste Modificateur, si celle-ci s'affiche, pour préciser votre recherche. Par exemple, vous pouvez choisir de rechercher des synonymes des termes recherchés.

  6. Facultatif Pour ajouter des ensembles de critères de recherche supplémentaires pour une propriété donnée, cliquez sur le bouton Dupliquer en regard du critère existant, puis saisissez le critère supplémentaire dans le champ de texte qui s'affiche. Vous devez sélectionner l'opérateur OU, ET ou SAUF dans la liste pour déterminer le mode de traitement des critères pour la propriété concernée. Par exemple, si vous ajoutez deux ensembles de critères de recherche pour une propriété et que vous cliquez sur ET dans la liste, la recherche donnera uniquement les résultats correspondant aux deux critères à la fois.

    Pour supprimer un ensemble supplémentaire de critères de recherche pour une propriété, cliquez sur le bouton Enlever en regard de l'ensemble concerné.

  7. Cliquez sur Rechercher.
