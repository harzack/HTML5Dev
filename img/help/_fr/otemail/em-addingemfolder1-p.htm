Pour ajouter un dossier d'e-mails

  1. Cliquez sur Dossier des e-mails dans le menu Ajouter un élément.

  2. Sur la page Ajouter le dossier des e-mails, saisissez le nom du dossier dans le champ Nom. Vous pouvez également cliquer sur le bouton Sélectionner l'icône pour choisir une autre icône pour le dossier.

  3. Spécifiez les autres paramètres de votre choix.

  4. Cliquez sur Ajouter.
