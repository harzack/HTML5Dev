Vous pouvez utiliser la page Recherche avancée de Content Server pour rechercher des propriétés spécifiques, telles que la date d'envoi ou le nom de l'expéditeur. Les options suivantes de recherche dans les e-mails sont disponibles sur la page Recherche avancée Content Server :

E-mail

Permet d'effectuer une recherche dans les propriétés standard des e-mails, telles que l'objet, l'expéditeur, le destinataire et les dates d'envoi et de réception.

E-mail (recherche avancée)

Permet d'effectuer une recherche avec un ensemble de propriétés que vous pouvez sélectionner.

 

Note :
Votre administrateur détermine les champs disponibles. Seules les propriétés pour lesquelles les cases Pour recherche et Pour affichage sont cochées dans l'onglet Régions de la page d'administration Enterprise Search Manager peuvent être sélectionnées dans la section E-mail (recherche avancée) de la page Recherche avancée. Le nom de la propriété correspond au Nom d'affichage de la région d'index.
