Pour spécifier les paramètres des tâches assignées :

  1. Choisissez Paramètres dans le menu Outils, puis cliquez sur l'onglet Workflow.

  2. Sur la page Mes paramètres : Général, modifiez les paramètres comme décrit au tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Choisir la page qui s'affiche par défaut lorsque vous lancez un workflow ou que vous visualisez une tâche assignée Dans la zone Tâches assignées du workflow, cliquez sur l'une des options suivantes dans la liste Page de départ :
    •

    Général pour afficher la page Général ;

    •

    Vue d'ensemble pour afficher la page Vue d'ensemble.

    Définir un mandataire de workflow qui reçoit vos tâches assignées lorsque vous n'êtes pas disponible Dans la zone Mandataire, cliquez sur l'icône Choisir un utilisateur ou un groupe  , puis recherchez et sélectionnez l'utilisateur que vous souhaitez nommer comme mandataire.
    Afficher les tâches assignées à l'aide d'une requête du générateur d'expressions Dans la section Tâches assignées du workflow, cliquez sur l'icône Générateur d'expressions , puis créez votre requête.
    Afficher vos prochaines tâches assignées pour le workflow sur votre page Mes tâches assignées ou dans l'onglet Tâches assignées Sélectionnez la case à cocher Ouvrir automatiquement la tâche suivante dans le même workflow si elle m'est assignée.
    Afficher les tâches qui vous sont assignées (option par défaut) sur votre page Mes tâches assignées Sélectionnez la case à cocher Inclure les tâches assignées du workflow dans la page 'Mes tâches assignées'.
    Choisir l'onglet qui s'affiche par défaut lorsque vous accédez à la page Mes workflows Dans la liste Onglet « Mes workflows » par défaut, cliquez sur Statuts pour afficher les statuts du workflow ou sur Tâches assignées pour afficher vos tâches assignées du workflow.
    Indiquer le nombre maximum de tâches assignées à afficher Dans le champ Nombre maximum d'éléments par page, saisissez le nombre de tâches assignées du workflow à afficher par défaut.

Pour plus d'informations sur cette procédure, consultez la section Spécification des paramètres du workflow et des tâches assignées.
