Pour faire suivre une étape :

  1. Sélectionnez Tâches assignées dans le menu Personnel.

  2. Cliquez sur le nom de l'étape de workflow qui vous a été assignée, puis lisez les instructions pour la tâche.

  3. Révisez et modifiez le lot de travail selon ces instructions.

  4. Sur la page de l'étape, cliquez sur un des boutons suivants :

    • Faire suivre, pour indiquer que vous avez achevé la tâche assignée et diriger le traitement du workflow vers l'étape suivante.

    • Un bouton de décision s'affiche si une décision doit être prise pour l'étape.

 

Astuces :

  • Sur la page Tâches assignées de votre étape du workflow, sélectionnez les cases à cocher Terminé pour faciliter le suivi du travail que vous avez effectué sur cette étape.

  • Vous pouvez ouvrir une étape en choisissant Workflows dans le menu Personnel, puis en cliquant sur l'onglet Tâches assignées de la page Mes workflows.

 

Pour plus d'informations sur cette procédure, consultez la section Exécution des tâches assignées d'un workflow.
