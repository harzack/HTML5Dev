Pour déléguer une étape de workflow à un utilisateur ou à un groupe :

  1. Sélectionnez Tâches assignées dans le menu Personnel.

  2. Cliquez sur le lien Déléguer à.

  3. Cliquez sur Choisir l'utilisateur.

  4. Dans la section Utilisateurs et groupes, recherchez et sélectionnez l'utilisateur ou le groupe auquel vous souhaitez déléguer l'étape.

  5. Cliquez sur Déléguer.

Pour plus d'informations sur cette procédure, consultez la section Délégation d'une étape.
