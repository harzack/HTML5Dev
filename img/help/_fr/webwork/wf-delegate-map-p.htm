Pour déléguer une étape à un sous-workflow :

  1. Choisissez Workflows dans le menu Personnel, cliquez sur l'onglet Tâches assignées de la page Mes workflows, puis cliquez sur le nom du workflow.

  2. Cliquez sur le lien Déléguer à.

  3. Dans la section Déléguer à, cliquez sur Choisir le sous-schéma.

  4. Dans la section Sélectionner un sous-schéma, accédez au schéma de workflow auquel vous souhaitez déléguer votre tâche, puis cliquez sur le lien Sélectionner.

  5. Cliquez sur l'onglet Pièces jointes, effectuez l'une des opérations suivantes, puis cliquez sur Ajouter à la définition du workflow :

    • Sélectionnez le bouton radio Aucune si vous ne voulez pas autoriser la transmission de pièces jointes entre le workflow et le sous-workflow.

    • Sélectionnez le bouton radio Tous, puis saisissez un nom dans le champ Nom du dossier si vous voulez transmettre toutes les pièces jointes entre le workflow et le sous-workflow.

    • Sélectionnez le bouton radio Eléments sélectionnés, puis les cases à cocher des pièces jointes que vous souhaitez transmettre entre le workflow principal et le sous-workflow.

  6. Cliquez sur l'onglet Commentaires, sélectionnez les cases à cocher indiquant la façon dont vous souhaitez transmettre les commentaires entre le workflow et le sous-workflow, puis cliquez sur Ajouter à la définition du workflow.

  7. Cliquez sur l'onglet Attributs et effectuez l'une des opérations suivantes :

    • Cliquez sur Lien automatisé si vous souhaitez que Content Server associe automatiquement les types de données d'attributs compatibles.

    • Dans les listes Sous-schéma, cliquez sur le nom de chaque attribut que vous souhaitez mapper sur ceux de la colonne Schéma principal.

  8. Facultatif Mappez les attributs sur le sous-workflow, puis cliquez sur Ajouter à la définition du workflow.

  9. Cliquez sur Envoyer au sous-schéma.

Pour plus d'informations sur cette procédure, consultez la section Délégation d'une étape.
