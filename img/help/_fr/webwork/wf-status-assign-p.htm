Pour afficher les tâches assignées d'un workflow :

  1. Dans le menu Personnel, sélectionnez Workflows.

  2. Sur la page Mes workflows, cliquez sur l'onglet Tâches assignées.

  3. Dans la section Afficher, cliquez sur l'icône Générateur d'expressions , puis créez une condition pour restreindre l'affichage des tâches assignées. Pour en savoir plus, consultez le document Utilisation du générateur d'expressions.

  4. Cliquez sur le nom de la tâche assignée pour l'afficher.

 

Astuces :

  • Vous pouvez également afficher les tâches assignées en cliquant sur Tâches assignées dans le menu Personnel.

 

Pour plus d'informations sur cette procédure, consultez la section Gestion des workflows et des tâches assignées.
