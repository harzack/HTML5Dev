Pour modifier l'itinéraire du workflow :

  1. Dans le menu Personnel, sélectionnez Workflows.

  2. Sur la page Mes workflows, cliquez sur le lien correspondant au nom du workflow.

  3. Cliquez sur Interrompre pour interrompre le workflow.

  4. Cliquez sur Modifier l'itinéraire.

  5. Modifiez le workflow dans l'éditeur de workflow.

  6. Dans le menu Schéma, choisissez Modifier.

  7. Revenez à la page Mes workflows et cliquez sur Reprendre.

 

Astuce :
Vous pouvez également afficher un workflow en ouvrant un élément de statut de workflow Content Server configuré pour afficher le workflow.

 
 

Note :
Pour plus d'informations sur l'interruption des workflows, reportez-vous à la rubrique Gestion des workflows.
