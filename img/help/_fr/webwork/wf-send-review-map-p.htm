Pour envoyer une étape à un sous-workflow pour vérification :

  1. Sélectionnez Tâches assignées dans le menu Personnel.

  2. Cliquez sur le nom de l'étape du workflow que vous souhaitez envoyer pour vérification.

  3. Sur la page Etape du workflow, cliquez sur le lien Envoyer pour révision.

  4. Dans la section Envoyer pour révision, cliquez sur Choisir le sous-schéma.

  5. Dans la section Sélectionner un sous-schéma, accédez au schéma de workflow, puis cliquez sur le lien Sélectionner.

  6. Cliquez sur l'onglet Pièces jointes, effectuez l'une des opérations suivantes, puis cliquez sur Ajouter à la définition du workflow :

    • Sélectionnez le bouton radio Aucune si vous ne voulez pas autoriser la transmission de pièces jointes entre le workflow et le sous-workflow.

    • Sélectionnez le bouton radio Tous, puis saisissez un nom dans le champ Nom du dossier si vous voulez transmettre toutes les pièces jointes entre le workflow et le sous-workflow.

    • Sélectionnez le bouton radio Eléments sélectionnés, puis les cases à cocher des pièces jointes que vous souhaitez transmettre entre le workflow et le sous-workflow.

  7. Cliquez sur l'onglet Commentaires, sélectionnez la case à cocher indiquant la façon dont vous souhaitez transmettre les commentaires entre le workflow et le sous-workflow, puis cliquez sur Ajouter à la définition du workflow.

  8. Cliquez sur l'onglet Attributs, mappez les attributs sur le sous-workflow, puis cliquez sur Ajouter à la définition du workflow.

  9. Cliquez sur Envoyer au sous-schéma.

 

Astuce :
Vous pouvez également rechercher et ouvrir une étape en choisissant Workflows dans le menu Personnel, en cliquant sur l'onglet Tâches assignées de la page Mes workflows, puis en cliquant sur le nom de l'étape.

 

Pour plus d'informations sur cette procédure, consultez la section Envoi d'une étape pour révision.
