Pour afficher les informations sur un workflow et une tâche assignée :

  1. Procédez de l'une des manières suivantes :

    • Pour afficher tous les workflows et processus EPS que vous avez l'autorisation de visualiser, choisissez Workflows dans le menu Personnel, puis cliquez sur l'onglet Statuts sur la page Mes workflows.

    • Pour afficher une liste personnalisée de workflows et de processus EPS, cliquez sur le lien d'un nom d'élément de statut de workflow Content Server, si présent.

    • Pour afficher uniquement les workflows lancés à partir d'un schéma de workflow particulier, cliquez sur l'icône Fonctions du schéma de workflow, puis choisissez Statut.

  2. Dans la section Afficher, sélectionnez l'une des options suivantes dans la liste :

    • Tous, qui affiche tous les workflows dont vous êtes le responsable ou l'initiateur, mais non les workflows système.

    • Lancés, qui affiche tous les workflows que vous avez lancés.

    • Gérés, qui affiche tous les workflows que vous gérez.

    • Lancés, qui affiche tous les workflows et processus EPS que vous avez lancés.

    • Gérés, qui affiche tous les workflows et processus EPS que vous gérez.

    • Workflows gérés, qui affiche uniquement les workflows que vous pouvez gérer.

    • Workflows lancés, qui affiche uniquement les workflows que vous avez lancés.

    • EPS gérés, qui affiche toutes les instances de processus EPS que vous pouvez gérer.

    • EPS lancés, qui affiche toutes les instances de processus EPS que vous avez lancées.

    • Système, qui affiche tous les workflows système (disponible uniquement pour l'administrateur ou les utilisateurs disposant de passe-droits).

  3. Dans la liste Afficher, activez l'un des boutons d'option suivants :

    • Non archivés, pour afficher uniquement les workflows en cours d'exécution

    • Archivés, pour afficher uniquement les workflows archivés.

  4. Facultatif Dans la section Afficher, cliquez sur l'icône Générateur d'expressions pour créer une condition qui restreint l'affichage des workflows. Pour en savoir plus, consultez le document Utilisation du générateur d'expressions.

  5. Dans la section Workflows, cliquez sur le lien correspondant au nom du workflow pour afficher des informations détaillées sur le workflow.

  6. Facultatif Pour afficher des informations spécifiques sur une étape, vous pouvez procéder de l'une des façons suivantes sur la page Statut du workflow :

    • Cliquez sur le lien Liste des étapes, puis sur un nom d'étape.

    • Cliquez sur le lien Afficher le schéma, puis double-cliquez sur une icône d'étape.

 

Astuces :

  • Vous pouvez également afficher une liste personnalisée des workflows en cliquant sur le lien d'un nom de statut de workflow ou afficher tous les workflows lancés à partir d'un schéma de workflow en cliquant sur une icône Fonctions d'un schéma de workflow, puis en choisissant Statut.

  • Vous pouvez afficher les informations d'une étape en cliquant sur le lien Liste des étapes sur la page Statut du workflow, puis sur un lien de nom d'étape.

 

Pour plus d'informations sur cette procédure, consultez la section Gestion des workflows et des tâches assignées.
