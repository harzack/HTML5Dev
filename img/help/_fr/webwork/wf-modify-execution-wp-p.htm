Pour modifier le lot de travail :

  1. Dans le menu Personnel, sélectionnez Workflows.

  2. Sur la page Mes workflows, exécutez l'une des opérations suivantes :

    • Cliquez sur l'icône Fonctions d'un workflow, sélectionnez Propriétés, puis Pièces jointes et modifiez ensuite les pièces jointes sur la page Pièces jointes.

    • Cliquez sur l'icône Fonctions d'un workflow, sélectionnez Propriétés, puis Commentaires et modifiez ensuite les attributs sur la page Attributs.

    • Cliquez sur l'icône Fonctions d'un workflow, sélectionnez Propriétés, puis Général et modifiez ensuite les attributs sur la page Attributs.

    • Cliquez sur l'icône Fonctions d'un workflow, sélectionnez Propriétés, puis Attributs et modifiez ensuite les attributs sur la page Attributs.

 

Astuce :
Vous pouvez également afficher un workflow en ouvrant un élément du statut de workflow configuré pour afficher le workflow.

 

Pour plus d'informations sur cette procédure, consultez la section Modification d'un workflow en cours d'exécution.
