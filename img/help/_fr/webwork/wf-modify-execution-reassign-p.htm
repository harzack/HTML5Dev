Pour réassigner une étape :

  1. Dans le menu Personnel, sélectionnez Workflows.

  2. Sur la page Mes workflows, cliquez sur le lien correspondant au nom du schéma de workflow.

  3. Sur la page de Statut du workflow, cliquez sur le lien Liste des étapes.

  4. Cliquez sur l'icône Fonctions de l'étape que vous souhaitez réassigner, puis choisissez Général.

  5. Sur la page Général, cliquez sur le lien Utilisateur ou groupe dans le champ Réassigner à.

  6. Recherchez et sélectionnez l'utilisateur ou le groupe auquel vous souhaitez réassigner l'étape.

  7. Cliquez sur Terminé.

 

Astuce :
Vous pouvez également afficher un workflow en ouvrant un élément de statut de workflow Content Server configuré pour afficher le workflow.

 

Pour plus d'informations sur cette procédure, consultez la section Modification d'un workflow en cours d'exécution.
