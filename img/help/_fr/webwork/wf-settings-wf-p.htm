Pour spécifier les paramètres du workflow :

  1. Choisissez Paramètres dans le menu Outils, puis cliquez sur l'onglet Workflow.

  2. Sur la page Mes paramètres : Général, modifiez les paramètres comme décrit au tableau suivant.

    Pour… Effectuez les opérations suivantes…
    Choisir la page qui s'affiche par défaut lorsque vous lancez un workflow ou que vous visualisez une tâche assignée Dans la zone Tâches assignées du workflow, cliquez sur l'une des options suivantes dans la liste Page de départ :
    •

    Général pour afficher la page Général ;

    •

    Vue d'ensemble pour afficher la page Vue d'ensemble.

    Afficher les instances de workflow et de processus sur les pages Mes workflows et Statut du workflow en fonction des relations
    		 Dans la zone Statuts du workflow, cliquez sur l'une des options suivantes dans la liste Afficher :
    •

    Tous, qui affiche tous les workflows dont vous êtes le responsable ou l'initiateur, mais non les workflows système.

    •

    Lancés, qui affiche tous les workflows que vous avez lancés.

    •

    Gérés, qui affiche tous les workflows que vous gérez.

    •

    Système, qui affiche tous les workflows système (disponible uniquement pour l'administrateur ou les utilisateurs disposant de passe-droits).

    Afficher les workflows en fonction du statut archivé Activez le bouton radio Non archivés pour afficher les workflows qui ne sont pas archivés ou le bouton radio Archivés pour afficher les workflows archivés.
    Afficher les workflows à l'aide d'une requête du générateur d'expressions Dans la section Statuts du workflow, cliquez sur l'icône Générateur d'expressions , puis créez votre requête.
    Choisir comment classer les workflows Cliquez sur un mode de classement dans la liste Ordre de classement.
    Choisir l'onglet qui s'affiche par défaut lorsque vous accédez à la page Mes workflows Dans la liste Onglet 'Mes workflows' par défaut, cliquez sur Statuts pour afficher les statuts du workflow ou sur Tâches assignées pour afficher vos tâches assignées du workflow.
    Indiquer le nombre maximum de workflows à afficher Dans la zone Nombre maximum d'éléments par page, saisissez le nombre de workflows à afficher par défaut.

Pour plus d'informations sur cette procédure, consultez la section Spécification des paramètres du workflow et des tâches assignées.
