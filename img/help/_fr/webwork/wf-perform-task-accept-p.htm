Pour accepter ou refuser une tâche assignée à un groupe :

  1. Sélectionnez Tâches assignées dans le menu Personnel.

  2. Cliquez sur le nom de l'étape de workflow qui a été assignée à votre groupe.

  3. Cliquez sur l'un des boutons suivants :

    • Accepter la tâche assignée

    • Ne pas accepter

 

Astuce :
Vous pouvez également rechercher une tâche assignée du workflow en choisissant Workflows dans le menu Personnel, puis en cliquant sur l'onglet Tâches assignées sur la page Mes workflows.

 

Pour plus d'informations sur cette procédure, consultez la section Exécution des tâches assignées d'un workflow.
