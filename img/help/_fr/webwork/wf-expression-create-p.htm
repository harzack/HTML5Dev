Pour créer une expression :

  1. Cliquez sur l'icône Générateur d'expressions .

  2. Dans la liste Ajouter une nouvelle ligne, cliquez sur un critère de recherche de workflow ou sur les parenthèses (si vous souhaitez créer une expression complexe).

  3. Indiquez l'expression.

  4. Facultatif Répétez les étapes 2 et 3 pour préciser d'autres critères.

  5. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Utilisation du générateur d'expressions.
