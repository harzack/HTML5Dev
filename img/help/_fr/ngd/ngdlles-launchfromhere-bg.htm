Si votre administrateur a activé cette fonction, vous pouvez accéder à Enterprise Connect lorsque vous utilisez des types de contenu spécifiques dans l'interface Web de Content Server. Lorsque cette fonction est disponible, vous pouvez accéder au menu Fonction pour un élément pris en charge, choisir Enterprise Connect depuis ici, et une nouvelle instance d'Enterprise Connect s'ouvre et affiche l'élément que vous avez sélectionné. Votre administrateur détermine les sous-types pour lesquels cette fonction sera disponible. Par défaut, cette action est intitulée Enterprise Connect depuis ici, mais votre administrateur peut créer un intitulé personnalisé pour cette fonction. Pour plus d'informations sur cette fonction, contactez votre administrateur.

 

Note :
Cette fonctionnalité est disponible uniquement pour les instances de l'interface Web de Content Server dans Internet Explorer.
