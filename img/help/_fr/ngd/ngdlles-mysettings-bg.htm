Lorsque vous ajoutez des e-mails à Content Server à l'aide d'Enterprise Connect, des conventions de dénomination sont appliquées à l'e-mail. Si votre administrateur a activé le nœud Mes paramètres dans l'arborescence d'Enterprise Connect, vous pouvez définir des règles pour les noms d'e-mails. Les valeurs initiales de la page Génération de nom pour les e-mails ont été définies par votre administrateur. Vous pouvez effectuer les opérations suivantes sur la page Génération de nom pour les e-mails :

  • Spécifier les propriétés de message à utiliser lors de la génération d'un nom Content Server pour un e-mail.

  • Définir le séparateur à utiliser entre les propriétés de message qui composent le nom de l'e-mail.

  • Spécifier les chaînes que vous souhaitez supprimer du nom d'e-mail généré.

  • Spécifier le format utilisé pour les dates et heures.

  • Spécifier une méthode de gestion pour les e-mails en double.

 

Note :
Si Enterprise Connect génère un nom pour l'e-mail déjà utilisé dans le recueil Content Server, vous êtes invité à résoudre le conflit de dénomination pour pouvoir ajouter l'e-mail. Pour plus d'informations sur la résolution des conflits de dénomination des fichiers, consultez la section Utilisation de l'e-mail.
