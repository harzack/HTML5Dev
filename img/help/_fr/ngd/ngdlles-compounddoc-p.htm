Pour créer un document composite :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter un document composite, cliquez sur Ajouter un élément, puis sur Document composite.

  2. Dans la page Ajouter un document composite, saisissez le nom du document composite dans la zone Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.
