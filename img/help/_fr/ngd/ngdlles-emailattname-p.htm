Pour configurer la façon dont Enterprise Connect gère les pièces jointes d'e-mail portant des noms identiques :

  1. Dans l'arborescence d'Enterprise Connect, cliquez sur un recueil Content Server, Personnel, puis sur Mes paramètres.

  2. Dans la zone Génération de nom pour les pièces jointes des e-mails, dans la liste Quand une pièce jointe d'e-mail portant le même nom est ajoutée à un dossier, effectuez l'une des actions suivantes :

    • Choisissez Inviter l'utilisateur à ajouter une pièce jointe en tant que nouvelle version du document pour permettre aux utilisateurs de choisir s'ils veulent ajouter des pièces jointes portant le même nom qu'un document existant dans la destination en tant que nouvelle version du document existant. C'est le paramètre par défaut.

    • Choisissez Inviter l'utilisateur à ajouter des pièces jointes en tant que nouveau document avec un suffixe unique pour permettre aux utilisateurs de choisir s'ils souhaitent ajouter des pièces jointes portant le même nom qu'un document existant dans la destination en tant que nouveau document du même nom auquel est ajouté un suffixe numérique pour le distinguer du document existant.

    • Choisissez Ajouter automatiquement une pièce jointe en tant que nouvelle version du document pour ajouter automatiquement des pièces jointes portant le même nom qu'un document existant dans la destination en tant que nouvelle version du document existant.

    • Choisissez Ajouter automatiquement des pièces jointes en tant que nouveau document avec un suffixe unique pour ajouter automatiquement des pièces jointes portant le même nom qu'un document existant dans la destination en tant que nouveau document auquel est ajouté un suffixe numérique pour le distinguer du document existant.

  3. Cliquez sur Envoyer pour appliquer les modifications.

     

    Astuce :
    Cliquez sur Réinitialiser pour rétablir le dernier paramètre enregistré pour la gestion de pièces jointes d'e-mail avec des noms en double. Cliquez sur Utiliser les paramètres par défaut pour rétablir les paramètres définis par l'administrateur.
