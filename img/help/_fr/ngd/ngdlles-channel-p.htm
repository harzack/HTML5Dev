Pour créer des chaînes :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer la chaîne, cliquez sur Ajouter un élément, puis cliquez sur Chaîne.

  2. Sur la page Ajouter une chaîne, saisissez un nom pour la chaîne dans la zone Nom.

  3. Vous pouvez également exécuter les actions suivantes :

    • Saisir une description de la chaîne dans la zone Description.

    • Cliquer sur Parcourir Content Server en regard de la zone Image, rechercher une image que vous souhaitez joindre à la chaîne, puis cliquer sur son lien Sélectionner.

    • Saisir une description de l'image (pour les utilisateurs ne pouvant pas la voir) dans la zone Texte ALT.

    • Spécifier l'emplacement où vous souhaitez créer la chaîne dans la zone Créer dans.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez afficher le contenu d'une chaîne existante en cliquant avec le bouton droit de la souris sur la chaîne, puis en sélectionnant Ouvrir.
