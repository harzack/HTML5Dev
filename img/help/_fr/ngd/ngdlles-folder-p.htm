Pour ajouter un dossier ou un dossier d'e-mails :

  1. Avec le bouton droit de la souris, cliquez sur le dossier ou sur l'Espace de travail dans lequel vous souhaitez ajouter le dossier, cliquez sur Ajouter un élément, puis cliquez sur l'une des options suivantes :

    • Dossier

    • Dossier des e-mails

  2. Sur la page Ajouter un dossier ou Ajouter le dossier des e-mails, saisissez le nom du dossier dans la zone Nom.

  3. Vous pouvez également cliquer sur Parcourir Content Server en regard du champ Créer dans, accéder à l'emplacement au niveau duquel vous souhaitez créer le dossier, puis cliquer sur le lien Sélectionner.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
