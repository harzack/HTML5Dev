Pour créer des affichages personnalisés :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer l'affichage personnalisé, cliquez sur Ajouter un élément, puis sur Affichage personnalisé.

  2. Sur la page Ajouter : Affichage personnalisé, cliquez sur Parcourir, recherchez le fichier HTML que vous avez créé et qui définit l'affichage personnalisé, puis cliquez sur Ouvrir.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.

 

Note :
Si le fichier HTML renvoie à d'autres fichiers (par exemple, des fichiers image), vous devez ajouter ces fichiers séparément.
