Pour ajouter des documents texte :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter un document texte, cliquez sur Ajouter un élément, puis sur Document texte.

  2. Dans la page Ajouter un document texte, saisissez le nom du document texte dans la zone Nom.

  3. Tapez le contenu du document texte dans la zone Texte.

    Le cas échéant, cliquez sur l'icône Vérifier l'orthographe pour corriger d'éventuelles fautes d'orthographe.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur un type MIME pour le document texte dans la liste Type MIME La valeur par défaut est texte normal.

  6. Cliquez sur Ajouter.

 

Astuce :
Selon la version de Content Server que vous utilisez, vous pouvez spécifier et modifier les types MIME HTML.
