Selon la configuration de votre serveur Content Server, il se peut que vous deviez spécifier vos informations de connexion la première fois que vous vous connectez à Enterprise Connect. Si vous avez modifié votre mot de passe Content Server via l'interface Web de Content Server, Enterprise Connect vous invitera à fournir le nouveau mot de passe lors de votre prochaine connexion.

 

Note :
vous ne pouvez pas modifier votre mot de passe sur la page de connexion à Enterprise Connect.
