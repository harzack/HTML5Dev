La création d'une génération crée une référence pour la version ou la révision spécifique d'un document composite ou une version de document et empêche les autres utilisateurs de la modifier ou de la supprimer.

Utilisez des générations pour rendre certaines versions d'un élément plus visibles et plus accessibles. Comme un raccourci, une génération attire l'attention sur une version particulière d'un élément, une version précédente généralement. Si vous possédez une ancienne version d'un élément dont vous avez toujours besoin, vous pouvez créer une génération pour cette version et la stocker dans n'importe quel recueil ou dans l'Espace de travail. Ainsi, la version visée par la génération ne sera ni modifiée, ni supprimée.

La génération est traitée comme un élément distinct. Elle n'est pas associée à l'élément (ni à la version de l'élément) à partir duquel elle a été créée. Grâce aux générations, les versions visées sont faciles à localiser et à utiliser car elles peuvent être stockées dans n'importe quel recueil approprié.

Lorsque vous créez une génération, l'icône Génération apparaît pour l'élément.

Le menu Fonctions d'une génération se divise en deux sous-menus :

  • Le sous-menu Original contient les fonctions que vous êtes autorisé à utiliser pour l'élément d'origine (la version vers laquelle la génération pointe). Les fonctions du sous-menu Original ne concernent pas la génération.

  • Le sous-menu Génération contient les fonctions que vous êtes autorisé à utiliser sur la génération. Les fonctions du sous-menu Génération ne concernent pas l'élément d'origine.
