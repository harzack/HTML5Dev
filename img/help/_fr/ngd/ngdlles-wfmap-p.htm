Pour créer un Schéma de Workflow :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter un Schéma de Workflow, cliquez sur Ajouter un élément, puis sur Schéma de Workflow.

  2. Sur la page Ajouter un Schéma de Workflow, saisissez un nom pour le Schéma de Workflow dans la zone Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.

  5. Cliquez avec le bouton droit de la souris sur le nouveau Schéma de Workflow, puis cliquez sur Modifier pour lancer l'Editeur de workflow que vous devez utiliser pour définir le Schéma de Workflow.
