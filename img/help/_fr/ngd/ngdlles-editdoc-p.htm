Pour modifier des documents :

  1. Pour ouvrir un document pour modification, effectuez l'une des opérations suivantes :

    • Cliquez avec le bouton droit de la souris sur le document, puis cliquez sur Modifier.

       

      Note :
      Pour ouvrir certains types de documents pour modification, vous pouvez cliquer avec le bouton droit de la souris sur le document approprié, puis cliquer sur Ouvrir.

       

    • Utilisez la boîte de dialogue Ouvrir d'Enterprise Connect pour l'application.

    • Utilisez la boîte de dialogue Ouvrir pour l'application native.

    • Sélectionnez le document approprié dans la liste de fichiers Utilisés récemment dans l'application.

    • Double-cliquez sur le document ou un raccourci vers celui-ci dans Enterprise Connect, si votre administrateur a activé cette option.

  2. Enregistrez le document à l'aide du menu Enregistrer sous d'Enterprise Connect.
