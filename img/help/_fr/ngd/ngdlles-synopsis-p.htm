Pour modifier des synthèses :

  1. Avec le bouton droit de la souris, cliquez sur l'élément dont vous souhaitez modifier une synthèse, puis cliquez sur Evaluer.

  2. Cliquez sur l'icône Synthèse .

  3. Dans la page Modifier la synthèse, modifiez la synthèse.

  4. Cliquez sur Mettre à jour.

 

Note :
L'icône de synthèse est disponible uniquement si votre administrateur a autorisé les utilisateurs à modifier les synthèses et si vous disposez vous-même de l'autorisation Modifier pour cet élément.
