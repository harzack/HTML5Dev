Pour ajouter des documents :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter un document, cliquez sur Ajouter un élément, puis sur Document.

     

    Astuce :
    Vous pouvez également cliquer sur Document dans le menu Ajouter un élément.

     

  2. Dans la zone Document de la page Ajouter un document, cliquez sur Parcourir pour accéder au fichier que vous souhaitez ajouter, puis cliquez sur Ouvrir.

  3. Saisissez le nom du document dans la zone Nom et entrez éventuellement une description.

  4. Dans la zone Contrôle de version, choisissez un type de création de versions en cliquant sur Standard - création linéaire de versions ou Avancée - création de versions principales/secondaires.

    Vous pouvez spécifier les paramètres des classements et des catégories.

  5. Dans la zone Créer dans, cliquez sur Parcourir Content Server pour spécifier où vous souhaitez enregistrer le document.

  6. Cliquez sur Ajouter.

 

Note :
Si un document de même nom existe déjà dans l'emplacement de destination, la boîte de dialogue Enterprise Connect s'ouvre. Sélectionnez l'une des options suivantes :

  • Oui si vous souhaitez ajouter le nouveau document en tant que version du document existant.

  • Oui pour tous si vous souhaitez ajouter tous les doublons de document à Content Server.

  • Non si vous ne souhaitez pas ajouter le document à Content Server.

  • Non pour tous si vous ne souhaitez pas ajouter les doublons de document à Content Server. Seuls les nouveaux documents seront ajoutés.

  • Annuler pour arrêter l'opération.
