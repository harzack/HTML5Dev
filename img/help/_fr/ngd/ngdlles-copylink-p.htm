Pour copier un lien vers le Presse-papiers :

  1. Cliquez avec le bouton droit de la souris sur l'élément approprié, cliquez sur Copier le lien dans le Presse-papiers, puis sur l'un des éléments suivants. Les commandes disponibles dépendent du type d'élément sélectionné :

    • Cliquez sur Ouvrir pour copier un lien vous permettant d'ouvrir l'élément.

    • Cliquez sur Télécharger pour copier un lien vous permettant de télécharger l'élément.

    • Cliquez sur Propriétés pour copier un lien vous permettant d'ouvrir la page Propriétés de l'élément.

    • Cliquez sur Vue d'ensemble pour copier un lien vous permettant d'ouvrir la page Vue d'ensemble de l'élément.

  2. Collez le lien dans la cible, qui peut être un e-mail, une application telle que Word ou Excel, ou encore un navigateur Web.
