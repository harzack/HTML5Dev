Pour créer une version ou une révision d'un document composite :

  1. Cliquez avec le bouton droit de la souris sur le document composite pour lequel vous souhaitez créer une version ou une révision, puis cliquez sur Aperçu.

  2. Sur la page Aperçu, cliquez sur le menu Fonction du document composite, puis cliquez sur Créer une version ou Créer une révision.

  3. Attribuez un nom à la version ou à la révision, puis cliquez sur Envoyer.
