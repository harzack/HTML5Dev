Pour configurer les autorisations :

  1. Avec le bouton droit de la souris, cliquez sur l'élément pour lequel vous souhaitez configurer des autorisations, puis cliquez sur Autorisations.

  2. Sur la page Autorisations, cliquez sur le lien d'un utilisateur ou d'un groupe dans la liste Accès par défaut ou Accès assigné.

  3. Dans la zone Modifier les autorisations de l'utilisateur ou du groupe, modifiez les paramètres des autorisations, puis cliquez sur Mettre à jour.

  4. Cliquez sur Terminé.

 

Note :
Si vous ne disposez pas des autorisations de modification pour un élément, la page Autorisations s'affiche en mode lecture seule.
