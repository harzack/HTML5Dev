En utilisant Enterprise Connect, vous pouvez accéder au contenu qui réside dans des dossiers virtuels et effectuer un certain nombre de tâches sur ce contenu. Une bannière juste en dessous de la barre d'outils Enterprise Connect vous indique lorsque vous travaillez dans un dossier virtuel. Pour plus d'informations sur la création, la configuration et l'utilisation des dossiers virtuels, reportez-vous à la rubrique Utilisation des dossiers virtuels.

 

Note :
Cette version d'Enterprise Connect ne prend pas en charge les facettes.

 
 

Astuce :
Vous pouvez accéder à l'emplacement d'origine d'un élément dans un dossier virtuel si votre administrateur a activé cette option. Cliquez avec le bouton droit de la souris sur l'élément, puis choisissez Aller à l'emplacement d'origine.

 

Les sections suivantes présentent la fonction Dossier virtuel d'Enterprise Connect.

Dossiers virtuels

Lorsque vous travaillez dans des dossiers virtuels dans Enterprise Connect, vous pouvez effectuer les opérations suivantes :

  • Parcourir le contenu d'un dossier virtuel dans l'arborescence d'Enterprise Connect.

  • Ouvrir et modifier des documents résidant dans un dossier virtuel.

  • Supprimer et renommer un dossier virtuel.

  • Prévisualiser dans le volet Aperçu le contenu résidant dans un dossier virtuel.

  • Ajouter du contenu à un dossier réel résidant dans un dossier virtuel via l'action de menu contextuel Ajouter un élément. La bannière indique lorsque vous accédez à un dossier virtuel ou à son contenu.

  • Créer sur le bureau un raccourci de dossier virtuel.

     

    Note :
    en raison de la nature dynamique du contenu résidant dans les dossiers virtuels, un raccourci menant vers un dossier virtuel ou son contenu peut devenir obsolète en cas de modification du contenu.

     

  • Ouvrir et modifier des documents de même nom apparaissant dans un même dossier virtuel.

     

    Astuce :
    Pour distinguer des documents de même nom, consultez le nom du dossier parent du document sélectionné dans la colonne Emplacement.

     

Quand vous travaillez dans des dossiers virtuels dans Enterprise Connect, les exceptions suivantes s'appliquent :

  • Vous ne pouvez pas utiliser les dossiers virtuels hors ligne.

  • Vous ne pouvez ajouter de contenu à un dossier virtuel ni en le faisant glisser ni via la boîte de dialogue Enregistrer sous d'Enterprise Connect. La liste Ajouter un élément ou le menu contextuel Ajouter un élément sont disponibles pour les dossiers virtuels, mais vous pouvez ajouter une apparence, un autre dossier virtuel ou une vue personnalisée uniquement à un dossier virtuel.

  • Vous pouvez rechercher le contenu d'un dossier virtuel.

Dossiers réels

Quand vous accédez à un dossier réel résidant dans un dossier virtuel, vous pourrez noter les comportements suivants :

  • Vous ne pouvez pas utiliser les dossiers réels ou leurs sous-dossiers hors ligne à partir du dossier virtuel. Cependant, vous pouvez utiliser les dossiers hors ligne à partir de leur emplacement réel dans Content Server.

  • Lorsque vous cliquez avec le bouton droit de la souris sur un dossier réel, vous pouvez accéder au menu contextuel complet.

  • Vous pouvez ajouter du contenu à des dossiers réels résidant dans un dossier virtuel.

Documents

Quand vous travaillez sur des documents résidant dans un dossier virtuel dans Enterprise Connect, vous pourrez noter les comportements suivants :

  • Vous pouvez ouvrir des versions en lecture seule des documents, et ouvrir et modifier des documents via la boîte de dialogue Ouvrir d'Enterprise Connect.

  • Vous pouvez enregistrer une nouvelle version d'un document dans un dossier virtuel en utilisant la fonction Enregistrer en tant que nouvelle version.

  • Si deux documents avec un nom de fichier identique apparaissent dans le volet Liste, vous pouvez ouvrir et modifier l'un des deux.

     

    Astuce :
    pour distinguer des fichiers de même nom, regardez dans la colonne Emplacement le nom du dossier parent du document sélectionné.

     

  • Vous pouvez supprimer et renommer un document résidant dans un dossier virtuel.

  • Vous pouvez faire glisser un document d'un dossier virtuel vers votre bureau.

  • Vous pouvez faire glisser des documents d'un dossier virtuel vers des dossiers réels.

  • Vous ne pouvez pas utiliser hors ligne des documents résidant dans des dossiers virtuels.

  • Lorsque vous cliquez avec le bouton droit de la souris sur un document résidant dans Dossiers virtuels, vous avez accès aux mêmes actions de menu contextuel que via l'interface utilisateur Web de Content Server.

  • Vous pouvez afficher un document résidant dans un dossier virtuel à l'aide du visualiseur Enterprise Connect.

  • Le nom du dossier parent dans lequel réside un document s'affiche dans la colonne Emplacement du volet Liste.
