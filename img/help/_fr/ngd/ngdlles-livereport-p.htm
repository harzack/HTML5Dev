Pour créer un Rapport LiveReport :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer le Rapport LiveReport, cliquez sur Ajouter un élément, puis sur LiveReport.

  2. Sur la page Ajouter un rapport LiveReport, saisissez un nom pour le rapport dans la zone Nom.

  3. Saisissez le nombre maximum de résultats autorisés dans la zone Limite d'enregistrement.

  4. Saisissez une instruction SQL pour le rapport dans la zone SQL.

  5. Cliquez sur les types de paramètres associés à l'instruction SQL dans les listes Paramètre.

  6. Sélectionnez un format pour le Rapport LiveReport dans la liste Format du rapport.

  7. Le tableau suivant décrit comment effectuer des tâches supplémentaires pour définir le rapport LiveReport.

    Pour Procéder ainsi
    Ajouter un titre qui s'affiche sur la page de résultats du rapport LiveReport Saisissez un nom dans la zone Titre.
    Inviter les utilisateurs à saisir de nouvelles valeurs de paramètre lors de l'utilisation du Rapport LiveReport en tant que sous-rapport Cochez la case Afficher les invites.
    Définir les types d'entrées Dans la zone Entrées, cliquez sur un type d'entrée dans la liste Type, puis saisissez le texte que vous souhaitez afficher pour l'invite dans la zone Invite.
    Autoriser l'instruction SQL à modifier le contenu de la base de données Content Server Cochez la case Autoriser les modifications de la base de données.
    Associer un sous-rapport à ce Rapport LiveReport Dans la zone Sélectionner le sous-rapport, cliquez sur Parcourir, recherchez le Rapport LiveReport que vous souhaitez afficher comme sous-rapport, puis cliquez sur son lien Sélectionner.
    Assigner des paramètres à un sous-rapport Saisissez le nom d'un paramètre existant dans chaque zone Paramètre du sous-rapport n°.
    Indiquer les informations de la base de données que vous souhaitez afficher sur la page de résultats du rapport LiveReport Saisissez le nom de la colonne de la base de données dans la zone Champ, puis saisissez un nom d'affichage correspondant pour la colonne dans la zone Titre de la colonne.
    Configurer l'apparence du format Rapport LiveReport sélectionné Choisissez un format pour le Rapport LiveReport, cliquez sur Configurer, effectuez les changements dans la fenêtre Configurer le format du rapport, puis cliquez sur Sélectionner.

  8. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez modifier les paramètres d'un Rapport LiveReport existant en cliquant sur le Rapport LiveReport avec le bouton droit de la souris et en cliquant sur Modifier.
