Pour ajouter des listes de tâches :

  1. Avec le bouton droit de la souris, cliquez sur le dossier ou l'Espace de travail dans lequel vous souhaitez ajouter la liste de tâches, cliquez sur Ajouter un élément, puis sur Liste de tâches.

  2. Dans la page Ajouter une liste de tâches, saisissez le nom de la liste de tâches dans la zone Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.
