Pour définir les propriétés de message à utiliser dans les noms des e-mails :

  1. Dans l'arborescence d'Enterprise Connect, cliquez sur un recueil Content Server, Personnel, puis sur Mes paramètres.

  2. Dans la section Génération de nom pour les e-mails, accédez à la zone Propriétés.

  3. Cliquez sur la propriété de message que vous souhaitez utiliser dans la liste Propriétés disponibles, puis cliquez sur Déplacer vers la droite pour la déplacer dans la liste Propriétés sélectionnées. Objet est la valeur par défaut.

  4. Pour définir l'ordre dans lequel les propriétés de message apparaissent dans le nom de l'e-mail, cliquez sur une propriété de message dans la liste Propriétés sélectionnées, puis cliquez sur Vers le haut ou Vers le bas .

  5. Cliquez sur Envoyer pour appliquer les modifications.

     

    Astuce :
    Cliquez sur Réinitialiser pour rétablir les dernières valeurs enregistrées pour les conventions de dénomination des e-mails. Cliquez sur Utiliser les paramètres par défaut pour rétablir les paramètres définis par l'administrateur.
