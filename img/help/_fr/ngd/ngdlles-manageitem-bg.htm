Vous pouvez exécuter diverses fonctions de Content Server à partir d'Enterprise Connect. Par exemple, il est possible que vous souhaitiez gérer une catégorie d'élément, réserver un élément ou créer une liste de tâches.

 

Notes :

  • Si vous avez masqué un élément (Dossier, Document ou Liste de tâches par exemple), il apparaît toujours dans le Volet Liste d'Outlook et de l'Explorateur. Les éléments masqués sont également inclus dans les résultats d'une recherche.

  • Enterprise Connect ne prend pas en charge la fonction Content Server Mis en valeur.
