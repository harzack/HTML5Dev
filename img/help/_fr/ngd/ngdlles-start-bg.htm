Cette application vous permet d'accéder au contenu des espaces de travail Content Server de la Structure d'arborescence d'OpenText Enterprise Connect. En fonction des options installées sur votre système, vous pouvez utiliser le module d'extension lorsque vous utilisez les applications telles que l'Explorateur Windows, Microsoft Outlook ou Microsoft Office.

Ce système d'aide contient des informations spécifiques à Enterprise Connect. Dans certains cas, ce système d'aide vous renvoie vers des rubriques du système d'aide Content Server principal, la fonctionnalité étant la même dans les deux applications. Vous pouvez parfois accéder à l'aide Content Server directement à partir des boîtes de dialogue et des fenêtres Enterprise Connect.
