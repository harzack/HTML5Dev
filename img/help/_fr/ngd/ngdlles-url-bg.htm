Les URL sont similaires aux favoris et aux signets des navigateurs Web. Lorsque vous ouvrez une URL, l'adresse Internet est transmise à votre navigateur qui affiche la page Web.

Vous pouvez enregistrer les éléments suivants en tant qu'URL :

  • les adresses du World Wide Web ;

  • les sites FTP ;

  • les adresses e-mail.

Vous pouvez utiliser une URL pour renvoyer à un autre élément Content Server, mais il est préférable d'utiliser un raccourci.
