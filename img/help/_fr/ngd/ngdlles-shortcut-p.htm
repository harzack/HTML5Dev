Pour créer un raccourci :

  1. Avec le bouton droit de la souris, cliquez sur l'élément pour lequel vous souhaitez créer un raccourci, puis cliquez sur Créer un raccourci.

  2. Sur la page Ajouter : Raccourci, saisissez un nom pour le raccourci dans la zone Nom.

  3. Cliquez sur Parcourir Content Server en regard de la zone Créer dans, naviguez jusqu'à l'emplacement au niveau duquel vous souhaitez créer le raccourci, puis cliquez sur son lien Sélectionner.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
