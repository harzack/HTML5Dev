Vous pouvez procéder à des recherches dans Content Server pour localiser des documents, des dossiers, des discussions et d'autres éléments. Vous pouvez définir le nombre de résultats de recherche qui s'affichent sur chaque page. Si vous disposez des autorisations suffisantes, vous pouvez créer des prospecteurs pour automatiser le processus de recherche.

 

Astuce :
Si vous effectuez une recherche via la fonction de recherche rapide d'Enterprise Connect, vous obtiendrez les mêmes résultats de recherche qu'avec la recherche Web Content Server. Pour plus d'informations sur les recherches dans Enterprise Connect, reportez-vous à la section OpenText Enterprise Connect Plug-in for Content Server - Manuel de l'utilisateur (NGDCORE-UGD).
