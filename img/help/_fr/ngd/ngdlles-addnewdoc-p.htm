Pour ajouter de nouveaux documents :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter un nouveau document, cliquez sur Ajouter un élément, puis sur Document.

     

    Astuce :
    Vous pouvez également cliquer sur Document dans le menu Ajouter un élément.

     

  2. Dans la zone Document de la page Ajouter un document, cliquez sur Nouveau, puis cliquez sur le type de document à ajouter dans la liste.

  3. Saisissez un nom pour le document dans la zone Nom.

  4. Dans la zone Contrôle de version, choisissez un type de création de versions en cliquant sur Standard - création linéaire de versions ou Avancée - création de versions principales/secondaires.

  5. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  6. Cliquez sur Ajouter.

  7. Ajoutez du contenu au document qui s'ouvre dans le programme de bureau qui lui est associé, puis fermez-le.

  8. Dans la boîte de dialogue Terminer la modification, cliquez sur Enregistrer les modifications dans Content Server, puis sur OK.
