Pour modifier les attributs d'une liste de tâches :

  1. Avec le bouton droit de la souris, cliquez sur la liste de tâches dont vous souhaitez modifier les attributs, puis cliquez sur Modifier des éléments.

  2. Sur la page Modifier les éléments, cochez la case de chaque attribut que vous souhaitez modifier dans la colonne Modifier.

  3. Cochez la case de chaque élément auquel vous souhaitez appliquer les modifications dans la colonne Appliquer à.

  4. Cliquez sur Mettre à jour.
