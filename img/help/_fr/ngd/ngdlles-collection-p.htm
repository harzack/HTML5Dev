Pour ajouter des éléments à une collection :

 

Note :
Selon la façon dont votre environnement est configuré et la version de Content Server que vous utilisez, vous verrez peut-être des variantes en ce qui concerne les options disponibles et le nom des boîtes de dialogue et listes.

 

  1. Sélectionnez le ou les éléments que vous voulez ajouter à une collection, cliquez dessus avec le bouton droit, puis choisissez Collecter.

  2. Dans la boîte de dialogue Collecter, le cas échéant, sélectionnez la case à cocher Collecter la version active.

  3. Dans la zone Collection, effectuez l'une des actions suivantes pour spécifier la collection à laquelle vous voulez ajouter le ou les éléments :

    • Le cas échéant, sélectionnez une collection dans la liste des collections récemment utilisées.

    • Le cas échéant, sélectionnez une collection dans vos dossiers Favoris ou Personnel.

    • Créez une nouvelle collection. Dans la boîte de dialogue Ajouter une collection, indiquez le nom, la description, les classements ou les catégories si nécessaire, ainsi que l'emplacement de la nouvelle collection, puis cliquez sur Ajouter.

    • Parcourez Enterprise Connect pour trouver la collection à laquelle vous voulez ajouter les éléments.

  4. Cliquez sur Envoyer, puis sur OK.
