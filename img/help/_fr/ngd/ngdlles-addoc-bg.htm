Un document est un fichier électronique que vous chargez de votre ordinateur vers la base de données Content Server. Les feuilles de calcul, les mémos, les pages HTML et les fichiers audio sont des exemples de documents. Les fonctions Livelink vous permettent d'ajouter des documents à Content Server, de modifier leur contenu, de les ouvrir, de les visualiser et de gérer des versions et des générations.

 

Note :
Vous devez disposer des autorisations appropriées relatives aux documents que vous ajoutez pour exécuter les actions suivantes : Pour plus d'informations sur la définition des autorisations, consultez la section Définition des autorisations.

 

Ajout de documents

Vous pouvez ajouter des documents à Content Server en chargeant le fichier d'un document qui existe déjà sur votre ordinateur. Vous pouvez également créer un document.

Si vous disposez des autorisations nécessaires, vous pouvez ajouter un document en tant que version principale ou secondaire du document original. Si vous ne disposez pas des autorisations nécessaires, vous ne pouvez pas ajouter de document en tant que version principale ou secondaire du document original.

Dupliquer des documents

Si vous ajoutez un document à Content Server et qu'un document de même nom existe déjà dans l'emplacement de destination, vous pouvez spécifier la façon selon laquelle le doublon du document doit être traité. Vous pouvez par exemple ajouter le nouveau document en tant que version du document déjà présent dans l'emplacement de destination. Si vous ajoutez plusieurs documents à Content Server, vous pouvez spécifier que tous les doublons du lot doivent être ajoutés en tant que nouvelles versions de documents existants. Vous pouvez spécifier que seuls les nouveaux documents soient ajoutés à Content Server et que les doublons soient ignorés.

Utilisation de la page Vue d'ensemble d'un document

La page Vue d'ensemble fournit des informations sur le document, notamment une description et des informations associées. La page Vue d'ensemble d'un document vous permet d'effectuer un grand nombre de fonctions dans le document comme la modification, le téléchargement, l'ouverture ou l'ajout d'une version, si vous disposez des autorisations nécessaires pour effectuer ces tâches.

Modification de documents

Plusieurs méthodes permettent de modifier un document, notamment l'utilisation de la commande Modifier. Si vous modifiez un document Microsoft Office ou Adobe Acrobat, le document s'ouvre dans l'application appropriée. L'élément est marqué comme étant réservé dans Content Server jusqu'à ce que vous fermiez l'application. La version actuelle du document dans Content Server est mise à jour à chaque fois que vous enregistrez les modifications apportées.

Votre niveau d'autorisations sur le document que vous ouvrez ou modifiez détermine les versions du document auxquelles vous avez accès. Si vous disposez d'autorisations Modifier sur un document, vous pourrez accéder à la dernière version de ce document. Selon l'état du document, il peut s'agir d'une version majeure ou mineure. Si vous disposez uniquement des autorisations de lecture seule sur un document, vous pourrez uniquement accéder à la dernière version majeure de ce document.

 

Note :
Vous pouvez ouvrir certains documents pour modification en cliquant avec le bouton droit de la souris sur le document approprié, puis en cliquant sur Ouvrir.

 
 

Astuce :
En fonction de votre configuration, vous pourrez peut-être ouvrir un document Microsoft Office pour modification en double-cliquant dessus ou sur un raccourci vers celui-ci dans Enterprise Connect.
