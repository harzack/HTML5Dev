Une recherche peut parfois renvoyer des éléments auxquels vous ne pouvez pas accéder car vous ne disposez pas des autorisations d'accès aux dossiers dans lesquels se trouvent ces éléments. Ces types d'élément sont appelés sans parent. Lorsque vous sélectionnez un élément sans parent depuis la liste des résultats d'une recherche pour l'ouvrir, cet élément apparaît dans la structure du dossier Eléments supplémentaires de l'arborescence d'Enterprise Connect.

Prenez garde aux points suivants lorsque vous utilisez des éléments sans parent en conjonction avec le dossier Eléments supplémentaires :

  • Si vous exécutez la commande Ouvrir ou Aller à l'emplacement d'origine sur un élément sans parent depuis la liste des résultats d'une recherche, celle-ci se base sur l'emplacement des Eléments supplémentaires.

  • Si vous exécutez la commande Enterprise Connect depuis ici sur un élément sans parent dans l'interface utilisateur Web de Content Server, Enterprise Connect ouvre l'élément situé dans le dossier Eléments supplémentaires.

  • Le dossier Eléments supplémentaires apparaît dans l'arborescence d'Enterprise Connect uniquement si nécessaire.

  • Une fois créé, le dossier Eléments supplémentaires reste disponible dans Enterprise Connect tant que votre session Enterprise Connect est ouverte.

  • Si vous disposez de plusieurs référentiels Content Server, chacun d'eux contient une instance du dossier Eléments supplémentaires.

  • Il est impossible d'ajouter directement des éléments au dossier Eléments supplémentaires. Cela dit, vous avez la possibilité d'ajouter des éléments aux dossiers qui apparaissent dans le dossier Eléments supplémentaires.

  • Le dossier Eléments supplémentaires est supprimé de l'arborescence d'Enterprise Connect lorsque vous fermez votre session.

  • Vous ne pouvez pas supprimer le dossier Eléments supplémentaires.

  • Si vous disposez des autorisations suffisantes, vous pouvez supprimer des éléments du dossier Eléments supplémentaires. Cette action entraîne la suppression de l'élément du dossier Eléments supplémentaires ainsi que de son emplacement d'origine dans Content Server.
