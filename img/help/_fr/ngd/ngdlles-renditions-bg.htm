Le module de présentations est un module en option de Content Server qui vous permet de créer et de stocker plusieurs versions en lecture seule de documents. Si votre administrateur a activé ce module et si les autorisations définies pour vous le permettent, vous pouvez créer, afficher et vous abonner aux présentations à l'aide d'Enterprise Connect. La principale différence entre l'utilisation des présentations dans Content Server et dans Enterprise Connect est la manière d'accéder à la fonctionnalité. Dans Enterprise Connect, vous utilisez les menus contextuels disponibles dans le volet Liste ou les options disponibles dans les onglets Versions et Présentations du volet Aperçu pour accéder à la fonctionnalité pour les documents sélectionnés. Pour plus d'informations sur l'utilisation des présentations, consultez l'aide en ligne des présentations OpenText.
