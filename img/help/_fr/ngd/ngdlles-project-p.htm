Pour créer un projet :

  1. Avec le bouton droit de la souris, cliquez sur le dossier ou sur l'Espace de travail dans lequel vous souhaitez créer le projet, cliquez sur Ajouter un élément, puis sur Projet.

  2. Dans la zone Infos générales de la page Ajouter : Projet, procédez de la façon suivante, puis cliquez sur Suivant :

    • Saisissez un nom de projet dans la zone Nom.

    • Cliquez sur Projet vierge dans la zone Créer à partir de.

    • Indiquez les autres informations générales.

  3. Dans la zone Contenu, procédez de la façon suivante, puis cliquez sur Suivant :

    • Cliquez sur un statut pour le projet dans la liste Statut.

    • Sélectionnez un mois, un jour et une année dans les listes Date de commencement et Date prévue.

    • Indiquez les informations facultatives dans la zone Contenu.

  4. Dans la zone Intervenants, procédez de la façon suivante, puis cliquez sur Suivant :

    • Cliquez sur l'option appropriée dans la zone Intervenants.

      Vous pouvez également cliquer sur Ajouter des intervenants ou Modifier les intervenants pour ajouter des intervenants à un projet ou les modifier.

    • Cliquez sur Activé ou Désactivé dans la zone Accès public.

    • Sélectionnez l'option appropriée pour indiquer les types d'intervenants du projet qui recevront des notifications par e-mail dans la zone Notification par e-mail.

  5. Dans la zone Présentation, procédez de la façon suivante, puis cliquez sur Terminer :

    • Cliquez sur une option dans la zone N° de sous-éléments mis en valeur.

    • Indiquez les informations facultatives dans la zone Présentation.
