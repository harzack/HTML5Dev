Pour créer un raccourci par glisser-déposer :

  1. Recherchez le ou les éléments pour lesquels vous souhaitez créer un raccourci.

  2. Cliquez avec le bouton droit et faites glisser l'élément vers l'endroit où vous voulez placer le raccourci.

    Lorsque vous relâchez le bouton de la souris, un menu contextuel apparaît.

  3. Dans le menu contextuel, cliquez sur Créer un raccourci ici.

  4. Fournissez des métadonnées si nécessaire.

     

    Astuce :
    Si un raccourci du même nom existe déjà dans la destination, le nouveau raccourci est créé, avec un suffixe numérique pour le distinguer de celui déjà présent.
