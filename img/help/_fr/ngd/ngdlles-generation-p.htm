Pour créer une génération :

  1. Avec le bouton droit de la souris, cliquez sur le document ou le document composite pour lequel vous souhaitez créer une génération, puis cliquez sur Créer une génération.

  2. Sur la page Ajouter : Génération, saisissez un nom pour la génération dans la zone Nom.

    Vous pouvez aussi cliquer sur le numéro d'une version dans la liste Version pour que la génération pointe vers une version précédente.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.
