Pour ajouter des attributs de catégorie :

  1. Cliquez avec le bouton droit de la souris sur la catégorie à laquelle vous voulez ajouter un attribut, puis cliquez sur Modifier.

  2. Sur la page de la catégorie, cliquez sur Ajouter un attribut, puis sélectionnez le type d'attribut approprié.

  3. Sur la page Ajouter un attribut, saisissez le nom de l'attribut, sélectionnez les options de votre choix pour le définir, puis cliquez sur OK.

  4. Cliquez sur OK.
