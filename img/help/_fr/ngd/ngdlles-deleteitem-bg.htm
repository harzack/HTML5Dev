Vous pouvez supprimer un élément s'il n'est plus utilisé, une discussion qui n'est plus active par exemple.

Lorsque vous supprimez un élément, n'oubliez pas les points suivants :

  • Vous devez disposer des autorisations appropriées pour supprimer l'élément.

  • La suppression d'un élément de type recueil, tel qu'un dossier, un projet ou un document composite, entraîne la suppression de tous les éléments que celui-ci contient.

  • Selon votre configuration, vous pouvez ou non restaurer les éléments que vous avez supprimés. Pour plus d'informations sur la restauration des éléments supprimés, contactez votre administrateur.

 

Astuces :

  • Vous pouvez supprimer plusieurs éléments en les sélectionnant, puis en cliquant avec le bouton droit sur les éléments que vous voulez supprimer et en choisissant Supprimer. Vous recevez un message indiquant les éléments que vous ne pouvez pas supprimer, par exemple, parce que vous ne disposez pas des autorisations appropriées.

  • Si le module Corbeille de Content Server est disponible dans votre environnement, vous pouvez restaurer un élément que vous avez supprimé en accédant au volume Corbeille dans Enterprise Connect, en sélectionnant le ou les éléments que vous voulez restaurer, et en spécifiant l'endroit où les éléments restaurés doivent résider. Pour plus d'informations sur l'utilisation de la Corbeille, consultez l'aide en ligne de la Corbeille OpenText.
