Pour exporter un Schéma de Workflow :

  1. Avec le bouton droit de la souris, cliquez sur le Schéma de Workflow, puis cliquez sur Exporter.

  2. Dans la boîte de dialogue Enregistrer sous, recherchez l'emplacement où vous souhaitez enregistrer le Schéma de Workflow exporté, puis cliquez sur Enregistrer.
