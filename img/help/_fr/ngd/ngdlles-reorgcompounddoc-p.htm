Pour réorganiser le contenu d'un document composite :

  1. Avec le bouton droit de la souris, cliquez sur le document composite dont vous souhaitez réorganiser le contenu, puis cliquez sur Réorganiser.

  2. Dans la page Réorganisation du document composite, saisissez un numéro dans la zone Nouveau de chaque élément dont vous voulez modifier l'ordre.

    Vous pouvez également activer la case à cocher Document principal de l'élément que vous souhaitez traiter comme document principal.

    Vous pouvez aussi activer la case à cocher Renumérotation séquentielle pour réorganiser les autres éléments dans le document composite.

  3. Cliquez sur Mettre à jour.
