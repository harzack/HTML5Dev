Pour ajouter des groupes de tâches :

  1. Avec le bouton droit de la souris, cliquez sur la liste de tâches à laquelle vous souhaitez ajouter le groupe de tâches, cliquez sur Ajouter un élément, puis sur Groupe de tâches.

  2. Dans la page Ajouter un groupe de tâches, saisissez le nom du groupe de tâches dans la zone Nom.

    Vous pouvez également cliquer sur une étape clé dans la liste Etape clé par défaut pour les tâches pour assigner une étape clé au groupe de tâches.

  3. Cliquez sur Ajouter.

 

Note :
Il est plus facile d'ajouter de nouvelles tâches à un groupe de tâches existant que d'assigner un nouveau groupe de tâches à des tâches existantes. Si vous projetez d'utiliser un groupe de tâches pour organiser une liste de tâches, créez d'abord le groupe de tâches et ajoutez les tâches ensuite.
