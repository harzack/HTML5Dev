Pour supprimer un élément :

  1. Avec le bouton droit de la souris, cliquez sur l'élément à supprimer, puis cliquez sur Supprimer.

  2. Dans la boîte de dialogue Supprimer des éléments, cliquez sur Oui.

 

Note :
Si vous utilisez le menu contextuel pour une génération ou un raccourci afin de supprimer l'élément original, il vous faudra cliquer sur Supprimer dans la boîte de dialogue Supprimer, puis cliquer sur OK dans la boîte de dialogue Suppression de l'élément dans.

 
 

Astuce :
Dans le Volet Liste, vous pouvez sélectionner les éléments que vous souhaitez supprimer et cliquer sur Delete.
