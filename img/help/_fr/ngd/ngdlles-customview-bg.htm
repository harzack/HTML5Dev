Un affichage personnalisé est un type d'élément contenant du code HTML. Lorsque vous ajoutez un affichage personnalisé à un Espace de travail, un dossier, un projet, une liste de tâches ou un document composite, Content Server intègre ce code HTML à son interface. Cet affichage personnalisé est inséré entre les menus de navigation et les éléments du recueil ou de l'Espace de travail.

La première étape pour ajouter un affichage personnalisé consiste à créer le fichier HTML définissant cet affichage, ainsi que d'autres fichiers, tels que les fichiers image ou les feuilles de style. Par défaut, un affichage personnalisé peut contenir les éléments suivants :

  • Code HTML, JavaScript, VBScript, applets Java ou tout autre type de code pouvant être traité par un navigateur Web. Cependant, votre administrateur peut limiter votre capacité à ajouter certains types de contenu.

  • Liens vers d'autres éléments de Content Server ou d'autres pages Web.

  • Références à des fichiers audio, image ou à d'autres fichiers stockés dans Content Server ou dans un répertoire situé sur un autre serveur Web accessible.

Etant donné que le contenu d'un affichage personnalisé est inséré dans le fichier HTML de l'interface, le navigateur Web ignore les balises telles que <TITLE>, <BODY> ou <FRAMESET>. Cela signifie que les types de codes que vous pouvez inclure sont limités. Par exemple, vous ne pouvez pas créer de cadres dans un affichage personnalisé.

 

Note :
Vous pouvez ajouter un dossier de ressources au même endroit que l'affichage personnalisé afin d'y stocker les images et les autres fichiers que l'affichage personnalisé utilise. Pour éviter la suppression accidentelle de ces fichiers et vous permettre d'optimiser l'organisation de votre Espace de travail, il est recommandé de masquer l'affichage personnalisé et le dossier de ressources.

 

La création et la gestion des affichages personnalisés sont généralement réservées à certains utilisateurs ou groupes. Pour ajouter un affichage personnalisé, vous devez disposer du privilège de création d'élément pour les affichages personnalisés, ainsi que des autorisations Ajouter les éléments et Modifier sur les recueils appropriés. Toutefois, vous n'avez pas besoin d'autorisation pour modifier les sous-éléments affectés par un affichage personnalisé. Pour modifier un affichage personnalisé ajouté par un autre utilisateur, vous devez disposer de l'autorisation Modifier sur l'affichage personnalisé.

Vous déterminez les utilisateurs ou groupes auxquels un affichage personnalisé s'applique en définissant des autorisations pour cet affichage.
