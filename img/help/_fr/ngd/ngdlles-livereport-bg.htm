Les rapports LiveReport permettent d'accéder aux informations de la base de données de votre entreprise et vous permettent de rechercher, d'organiser et de présenter les informations qui y sont stockées. L'accès à la base de données à l'aide des rapports LiveReport diffère de l'utilisation de la recherche Content Server, car vous pouvez interroger non seulement la partie de la base de données relative aux éléments Content Server mais aussi le schéma de la base de données. Les rapports LiveReport obtiennent des informations en envoyant des requêtes SQL (Structured Query Language) à la base de données. Les instructions SQL pouvant modifier ou supprimer les données critiques, l'autorisation Ajouter ou Modifier pour les rapports LiveReport est réservée à l'administrateur par défaut. L'ouverture d'un Rapport LiveReport entraîne l'exécution automatique de l'instruction SQL et d'autres paramètres. Pour pouvoir ouvrir un Rapport LiveReport, vous devez disposer de l'autorisation Voir le contenu pour ce Rapport LiveReport.

Vous pouvez créer des rapports LiveReport et les placer dans n'importe quel dossier. Le volume Rapports LiveReport contient un ensemble de rapports LiveReport par défaut. Tous les rapports LiveReport par défaut que vous avez l'autorisation de voir apparaissent dans le dossier des rapports, situé dans le dossier Personnel.
