Par défaut, la page de recherche contient de nombreux attributs système. Toutefois, certains attributs système doivent être activés par un administrateur pour être affichés. Par exemple, un administrateur peut activer l'attribut OTDocumentRating de manière à permettre aux utilisateurs de rechercher les documents associés à une valeur d'évaluation donnée. L'administrateur est souvent appelé à activer de nouveaux attributs lors de l'installation d'un nouveau module ou d'une nouvelle version Content Server.

Le tableau ci-dessous contient des exemples d'attributs susceptibles d'être activés par un administrateur.

 

Note :
Le tableau affiche les noms par défaut des attributs ; votre administrateur a la possibilité de modifier ces noms.

 

Attribut Fonction
OTDocumentRating Permet de rechercher les éléments associés à une valeur d'évaluation donnée.
OTDocumentSynopsis Permet de rechercher les éléments dont la synthèse contient certaines informations.
OTDocumentUserComment Permet de rechercher les éléments dont les commentaires contiennent certaines informations.
OTDocumentUserExplanation Permet de rechercher les éléments dont les explications contiennent certaines informations.
OTProjectStatus Permet de rechercher les projets associés à un certain statut.
