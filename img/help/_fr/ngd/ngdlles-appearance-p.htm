Pour créer des apparences :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer une apparence, cliquez sur Ajouter un élément, puis cliquez sur Apparence.

  2. Sur la page Ajouter : Apparence, saisissez le nom de l'apparence dans la zone Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.

  5. Cliquez avec le bouton droit sur la nouvelle apparence, puis cliquez sur Parcourir depuis ici.

     

    Note :
    Bien que l'option de menu contextuel Parcourir depuis ici soit disponible par défaut, votre administrateur peut la supprimer si nécessaire. Si Parcourir depuis ici n'est pas disponible, contactez votre administrateur.

     

  6. Cliquez sur l'icône Modifier appropriée pour définir le contenu et les paramètres de l'apparence.
