Lorsque vous accédez pour la première fois au dossier racine Content Server de la Structure d'arborescence, Enterprise Connect vous connecte au Content Server que vous avez spécifié pendant l'installation.

Les dossiers suivants apparaissent par défaut sous le dossier racine :

  • Mon Espace de travail

  • Favoris

  • Entreprise

  • Personnel

Votre administrateur peut choisir de masquer ces dossiers ou d'en ajouter. De la même façon, il peut déterminer les éléments disponibles dans votre dossier personnel.

Vous pouvez exécuter des actions en cliquant avec le bouton droit de la souris sur les éléments de la Structure d'arborescence et du Volet Liste, puis en sélectionnant une option dans le menu qui s'affiche. Votre administrateur contrôle les commandes disponibles. Dans certains cas, une vue Web s'ouvre lorsque vous exécutez une action sur un élément. Une vue Web est une fenêtre qui affiche une partie de l'interface du Content Server accessible via un navigateur Web. La vue Web affiche uniquement les options et les commandes s'appliquant directement à la tâche que vous exécutez.

 

Note :
Si vous sélectionnez puis cliquez avec le bouton droit de la souris sur au moins deux éléments, le menu contextuel ne contient que les fonctions communes aux éléments sélectionnés.
