Pour créer un modèle de projet :

  1. Avec le bouton droit de la souris, cliquez sur le projet à partir duquel vous souhaitez créer un modèle, puis cliquez sur Créer un modèle.

  2. Dans la zone Source de la page Ajouter : Modèle de projet, cliquez sur Projet, sur Parcourir Content Server, accédez au project dont vous souhaitez créer le modèle, puis cliquez sur son lien Sélectionner.

  3. Saisissez un nom pour le modèle dans la zone Nom.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
