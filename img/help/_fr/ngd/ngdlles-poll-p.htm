Pour créer des sondages :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer le sondage, cliquez sur Ajouter un élément, puis sur Sondage.

  2. Sur la page Ajout de sondage, saisissez un nom pour le sondage dans la zone Nom.

  3. Spécifiez les autres paramètres éventuels.

  4. Cliquez sur Ajouter.
