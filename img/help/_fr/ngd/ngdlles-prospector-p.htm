Pour ajouter un prospecteur :

  1. Cliquez avec le bouton droit de la souris sur le dossier dans lequel vous souhaitez ajouter un prospecteur, cliquez sur Ajouter un élément, puis sur Prospecteur.

  2. Sur la page Ajouter : Prospecteur, saisissez un nom pour le prospecteur dans la zone Nom.

    Vous pouvez également saisir une description du prospecteur dans la zone Description.

  3. Cliquez sur une ou plusieurs vues de recherche dans la liste Dans.

  4. Saisissez une requête dans la zone Rechercher.

  5. Pour étendre ou limiter votre requête, ajoutez des critères de recherche supplémentaires dans la zone Critères facultatifs.

  6. Renseignez les champs des paramètres suivants :

    • Etat

    • Options de résultats

    • Options de notification

    • Programme de notification

  7. Cliquez sur Parcourir Content Server en regard de la zone Créer dans pour ajouter le prospecteur à un emplacement autre que le recueil actuel.

  8. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez modifier les paramètres de Prospectors existants en cliquant sur le prospecteur avec le bouton droit de la souris et en sélectionnant Modifier .
