Le menu Ajouter un élément s'affiche sur toutes les pages pour lesquelles vous avez l'autorisation d'ajouter des éléments. Les options disponibles pour l'ajout d'un élément varient selon le type d'élément. Consultez la rubrique d'aide spécifique au type d'élément pour obtenir des informations sur l'ajout d'un élément.

Lorsque vous ajoutez des éléments, souvenez-vous que les autorisations déterminent les éléments auxquels vous pouvez accéder. Par défaut, les éléments que vous ajoutez héritent des autorisations du recueil dans lequel vous le placez. Par exemple, si un groupe possède l'autorisation Voir le contenu pour un dossier, les membres de ce groupe ont l'autorisation d'afficher les documents que vous ajoutez à ce dossier. Toutefois, il est possible de modifier les autorisations d'un élément spécifique d'un dossier. Vous êtes propriétaire des éléments que vous ajoutez. Par conséquent vous possédez toutes les autorisations pour ces éléments. Pour plus d'informations sur la définition des autorisations, consultez la section Définition des autorisations.

Indication des métadonnées

En fonction de la configuration de votre environnement, il est possible que vous deviez fournir des métadonnées, comme les attributs de catégorie, sur les éléments que vous ajoutez à un dossier Content Server ou à d'autres types de recueils. Certaines métadonnées sont automatiquement extraites de l'élément et apparaissent dans les zones de la boîte de dialogue Ajouter. En fonction de votre configuration, vous pouvez avoir le droit de modifier ces zones.

Si vous ajoutez un groupe d'éléments à un recueil Content Server, la case Appliquer à tous les éléments de même type doit être visible. Lorsque vous cochez cette case, les métadonnées que vous avez spécifiées pour cet élément s'appliquent à tous les éléments suivants de même type du groupe.

Dans la boîte de dialogue Ajouter, certains paramètres sont communs à presque tous les types d'éléments que vous ajoutez. Ces paramètres, répertoriés dans le tableau suivant, ne sont pas toujours nécessaires pour ajouter avec succès un élément à un recueil Content Server.

Paramètre Description
Nom Content Server attribue un nom par défaut à tous les éléments que vous ajoutez (pour les documents par exemple, le système utilise le nom du fichier ajouté). Toutefois, il peut être utile d'attribuer un nom descriptif dans la zone Nom afin de fournir des informations sur le contenu et l'utilisation de l'élément. Cela permet également d'identifier correctement l'élément lors d'une requête de recherche ou d'un Rapport LiveReport.
Description Une description permet d'augmenter vos chances d'identifier correctement l'élément lors d'une requête de recherche ou d'un Rapport LiveReport. Content Server indexe le texte de la zone Description comme partie des métadonnées de l'élément. Plus l'élément contient de métadonnées, plus il est facile pour Content Server d'identifier son contenu et son utilisation.
Catégories La zone Catégories permet d'appliquer un ou plusieurs attributs à un élément que vous ajoutez. Pour plus d'informations sur l'utilisation des catégories, voir Utilisation des catégories.
Créer dans Par défaut, les éléments sont ajoutés au recueil actuel. Toutefois, vous pouvez ajouter l'élément à un autre recueil.

La procédure de l'exemple suivant illustre les zones que vous pouvez rencontrer lorsque vous ajoutez un élément à un recueil Content Server. Vous pouvez voir des zones supplémentaires en fonction de votre configuration, ainsi que les modules auxquels vous pouvez accéder.

  1. Sélectionnez les éléments que vous voulez ajouter, puis faites-les glisser dans le dossier Content Server approprié.

  2. Sur la page Ajouter, effectuez l'une des actions suivantes :

    • Décrivez l'élément dans la zone Description.

    • Sélectionnez le classement approprié de l'élément le cas échéant.

    • Vous pouvez également cliquer sur Modifier et entrer des informations sur la catégorie.

    • Cochez la case Appliquer à tous les éléments de même type si vous souhaitez appliquer ces valeurs à tous les éléments suivants de même type du groupe.

       

      Note :
      Si vous cliquez sur Annuler pendant l'ajout des éléments au recueil, le processus s'arrête et aucun élément n'est ajouté à l'emplacement de destination. Les fichiers déjà ajoutés ne sont pas supprimés.

       

  3. Cliquez sur OK.
