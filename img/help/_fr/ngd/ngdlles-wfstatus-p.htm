Pour créer un élément de Statut du workflow :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer un Statut du workflow, cliquez sur Ajouter un élément, puis sur Statut du workflow.

  2. Entrez un nom et une description du Statut du workflow dans les zones Nom et Description.

  3. Dans la liste Relation, cliquez sur l'une des options suivantes :

    • Tous, pour afficher uniquement les workflows dont vous êtes le responsable ou l'initiateur.

    • Lancés, pour afficher uniquement les workflows que vous avez lancés.

    • Gérés, pour afficher uniquement les workflows que vous gérez.

    • Système, qui est disponible uniquement pour l'administrateur ou les utilisateurs disposant de passe-droits.

  4. Dans les zones Workflows, cliquez sur l'une des options suivantes :

    • Non archivés, pour afficher uniquement les workflows en cours d'exécution.

    • Archivés, pour afficher uniquement les workflows archivés.

  5. Vous pouvez également exécuter les actions suivantes :

    • Cliquez sur l'icône Modifier une expression  pour modifier la requête à l'aide du générateur d'expressions.

    • Cliquez sur un type de classement dans la liste Ordre de tri pour trier la liste de workflows.

    • Cliquez sur Parcourir Content Server pour spécifier l'emplacement où vous souhaitez créer le Statut du workflow.

  6. Cliquez sur Ajouter.

 

Note :
Vous pouvez modifier un Statut du workflow en cliquant avec le bouton droit de la souris sur le Statut du workflow que vous souhaitez modifier et en sélectionnant Modifier la requête.
