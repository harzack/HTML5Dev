Pour copier un élément :

  1. Sélectionnez les éléments que vous souhaitez copier, puis cliquez avec le bouton droit de la souris et sélectionnez Copier/Déplacer vers Enterprise Connect.

  2. Dans la boîte de dialogue Enterprise Connect, accédez à l'emplacement cible, puis cliquez sur Copier.

  3. Selon la façon dont votre administrateur a configuré votre environnement, vous devrez éventuellement effectuer les opérations suivantes :

    Pour Procéder ainsi
    Définir des options de catégorie Dans la zone Catégories, effectuez l'une des opérations suivantes :
    •

    Cliquez sur Original si vous souhaitez conserver les catégories assignées à l'origine à un ou plusieurs éléments. Les catégories attribuées sont indiquées dans la zone Catégories. Cliquez sur Modifier pour modifier les catégories si nécessaire.

     

    Note :
    Si vous avez sélectionné plusieurs éléments et si vous cliquez sur Modifier dans la boîte de dialogue Copier pour modifier les catégories attribuées, la boîte de dialogue Copier de l'élément sélectionné suivant s'affiche et vous permet de modifier les catégories de cet élément. Cependant, si vous ne cliquez pas sur Modifier, les catégories spécifiées sont appliquées à tous les éléments restants.

     
    •

    Cliquez sur Destination si vous souhaitez appliquer à un ou plusieurs éléments les catégories du nouvel emplacement. Pour afficher les catégories attribuées à la destination, cliquez sur Modifier, puis sur Terminé dans la boîte de dialogue Modifier les catégories. Vous pouvez modifier les catégories si nécessaire.

    •

    Cliquez sur Fusionné si vous souhaitez appliquer les catégories des anciens et des nouveaux emplacements à l'élément ou aux éléments. Pour afficher les catégories attribuées à la destination, cliquez sur Modifier, puis sur Terminé dans la boîte de dialogue Modifier les catégories. Vous pouvez modifier les catégories si nécessaire. Si vous avez sélectionné plusieurs éléments, la boîte de dialogue Copier s'affiche pour chaque élément, vous permettant ainsi de modifier les catégories si nécessaire.

    Indiquer si l'élément hérite des classements RM (le cas échéant) Dans la zone Classements RM, effectuez l'une des opérations suivantes :
    •

    Cliquez sur Oui si vous souhaitez que l'élément hérite des classements RM assignés à la destination.

    •

    Cliquez sur Non si vous ne souhaitez pas que l'élément hérite des classements RM assignés à la destination.

  4. Si la destination comporte déjà un élément du même nom, effectuez l'opération suivante dans la boîte de dialogue Enterprise Connect :

    • Cliquez sur Oui si vous souhaitez créer une nouvelle instance de l'élément à la destination, avec un nom généré automatiquement.

    • Cliquez sur Oui pour tous si vous souhaitez résoudre les éventuels conflits d'attribution de noms avec d'autres éléments du lot que vous copiez en créant une nouvelle instance de l'élément à la destination, avec un nom généré automatiquement.

    • Cliquez sur Non si vous ne souhaitez pas copier l'élément actuel vers la destination. La copie se poursuit une fois cette option sélectionnée.

    • Cliquez sur Non pour tous si vous ne souhaitez pas copier les doublons d'éléments vers la destination. La copie se poursuit une fois cette option sélectionnée.

    • Cliquez sur Annuler si vous souhaitez arrêter la copie. Les éléments déjà copiés vers la destination y sont conservés.

     

    Note :
    si vous copiez des éléments et que vous utilisez Enterprise Connect dans Outlook, les options Non et Non pour tous ne sont pas disponibles en raison d'une restriction de fonctionnalité d'Outlook. Toutefois, si vous utilisez la boîte de dialogue Copier/Déplacer d'Enterprise Connect pour copier des éléments, ces options seront disponibles. Ces options sont également disponibles dans le volet Accès rapide d'Enterprise Connect.

     

  5. La boîte de dialogue Copie en cours indique la progression du déplacement.
