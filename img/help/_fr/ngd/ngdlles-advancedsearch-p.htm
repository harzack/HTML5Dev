Pour exécuter une recherche avancée :

  1. Dans le menu ou l'onglet Enterprise Connect, cliquez sur Rechercher, puis sur Rechercher Content Server.

     

    Astuce :
    Vous pouvez également accéder à la page de recherche avancée de Content Server en cliquant sur Recherche avancée sur la barre d'outils d'Enterprise Connect dans l'Explorateur Windows et Microsoft Outlook. En outre, vous pouvez accéder à la recherche avancée à partir de la zone de recherche dans le volet Accès rapide et dans les boîtes de dialogue Copier/Déplacer vers Enterprise Connect, Enterprise ConnectOuvrir et Enterprise ConnectEnregistrer sous. Le bouton est uniquement disponible pour les types de recueils que vous pouvez chercher et pour les recueils que vous êtes autorisé à chercher. Par exemple, Recherche avancée n'est pas disponible pour le nœud racine d'Enterprise Connect, les résultats de recherche, le dossier Favoris, le dossier Personnel et les recueils non physiques comme les Dossiers virtuels.

     

  2. Utilisez les informations du tableau suivant pour spécifier des critères de recherche.

     

    Note :
    En fonction des modules Content Server auxquels vous pouvez accéder, des options supplémentaires peuvent être disponibles. Pour plus d'informations sur la fonction de recherche dans Content Server, reportez-vous à la rubrique OpenText Content Server Aide en ligne utilisateur - Recherche Content Server (LLESWBB-H-UGD).

     

    Pour Procéder ainsi
    Indiquer l'emplacement où vous souhaitez effectuer la recherche Dans la zone Ajouter au formulaire de recherche, cliquez sur Emplacement. Ensuite, dans la zone Emplacement , cliquez sur Parcourir Content Server et sélectionnez l'emplacement où vous souhaitez exécuter la recherche.
     

    Notes :

    • Si vous ne souhaitez pas spécifier d'emplacement de recherche, Enterprise Connect effectue la recherche sur l'ensemble du référentiel Content Server.

      Les critères de recherche actuels sont effacés lorsque vous fermez la boîte de dialogue Rechercher Content Server.

    • Quand vous effectuez des recherches avancées dans Outlook, les colonnes supplémentaires que vous avez sélectionnées à l'aide du Sélecteur de champs s'affichent chaque fois que vous recherchez un ou plusieurs recueils de même type. Toutefois, si vous lancez une recherche avancée sur un ou plusieurs recueils de type différent, seules les colonnes par défaut s'affichent.

     
     

    Astuce :
    Pour affiner votre recherche, vous pouvez cliquer sur Dupliquer , sélectionner l'opérateur booléen approprié dans la liste, puis spécifier d'autres emplacements de recherche.

     
    Spécifier si la recherche doit trouver tous les mots, certains mots ou une phrase exacte Dans la zone Texte intégral, cliquez sur une option dans la liste Rechercher, puis saisissez le terme recherché.
    Exécuter une requête complexe Dans la zone Texte intégral, cliquez sur Requête complexe dans la liste Rechercher, puis saisissez le terme recherché. Pour en savoir plus, voir Recherche à l'aide de requêtes complexes.
    Spécifier un modificateur pour développer la recherche Dans la zone Texte intégral, cliquez sur une option dans la liste Modificateur, puis saisissez le terme recherché.
    Spécifier si la recherche se limite au contenu de l'élément, aux métadonnées ou aux deux Dans la zone Texte intégral, cliquez sur une option dans la liste Dans, puis saisissez le terme recherché.
    Effectuer une recherche à l'aide d'une phrase Dans la zone Ajouter au formulaire de recherche, cliquez sur Requête en langage naturel. Saisissez le terme à rechercher, puis sélectionnez l'option appropriée dans la liste Mode. Pour plus d'informations sur les requêtes en langage naturel, reportez-vous à la rubrique Recherche à l`aide de requêtes en langage naturel.
    Exécuter une recherche d'attribut système Dans la zone Ajouter au formulaire de recherche, cliquez sur Attributs système. Dans la liste Attributs supplémentaires, cliquez sur l'attribut que vous voulez inclure dans la recherche, puis spécifiez les valeurs d'attributs recherchées. Pour plus d'informations sur l'utilisation des attributs système, reportez-vous à la rubrique Utilisation des catégories et des attributs.
    Exécuter une recherche de catégorie Dans la zone Ajouter au formulaire de recherche, cliquez sur Catégories, puis sélectionnez la catégorie que vous souhaitez utiliser dans la boîte de dialogue Sélectionner une catégorie. Dans la zone Catégorie, spécifiez les valeurs des attributs de catégorie que vous souhaitez utiliser. Pour plus d'informations sur l'utilisation des catégories, reportez-vous à la rubrique Utilisation des catégories et des attributs.
    Exécuter une recherche Classifications Dans la zone Ajouter au formulaire de recherche, cliquez sur Classements. Dans la zone Classifications, cliquez sur Parcourir les Classifications pour localiser les Classifications que vous souhaitez inclure dans la recherche, ou sur Mes favoris pour sélectionner les Classifications appropriés à partir de votre dossier de favoris. Activez la case Inclure les sous-classements pour rechercher les éléments classés dans le classement sélectionné ainsi que tous ses descendants. En incluant les sous-classements, vous pouvez rechercher le Classifications sélectionné ainsi que tous ses descendants. Pour plus d'informations sur les Classifications, reportez-vous à la rubrique Utilisation de classements.
     

    Note :
    Cette option est disponible uniquement si le Classifications a été installé dans votre environnement.

     
    Rechercher un e-mail (recherche simple) Dans la zone Ajouter au formulaire de recherche, cliquez sur E-mail. Fournissez des critères de recherche, y compris OTEmailSubject, OTEmailReceivedDate et OTEmailSentDate. Vous pouvez également spécifier le nom des expéditeurs et destinataires des e-mails dans la zone Rechercher.
     

    Notes :

    • Utilisez un point-virgule pour séparer plusieurs entrées de critères de recherche dans la zone Rechercher.

    • Les recherches au moyen de caractères génériques ne sont pas prises en charge.

     
    Rechercher un e-mail (recherche avancée) Dans la zone Ajouter au formulaire de recherche, cliquez sur E-mail (recherche avancée). Cliquez dans la liste Attribut supplémentaire pour sélectionner les colonnes pour lesquelles vous souhaitez saisir des critères de recherche, puis spécifiez les valeurs recherchées.
    Rechercher des types d'élément physique Dans la zone Options avancées, cliquez sur Types d'éléments physiques. Dans la zone Types d'élément physique, cliquez sur la liste Sélectionner un type d'élément physique, puis cliquez sur Parcourir les types d'élément physique. Sur la page Parcourir les types d'élément physique, cliquez sur Sélectionner pour choisir le type d'élément approprié.
     

    Note :
    Cette option est disponible uniquement si le module Physical Objects a été installé dans votre environnement.

     
    Rechercher des attributs d'élément physique Dans la zone Ajouter au formulaire de recherche, cliquez sur Attributs d'élément physique. Dans la zone Attributs d'élément physique, spécifiez les attributs que vous souhaitez rechercher. La liste Emplacement d'origine est proposée par défaut, mais vous pouvez la supprimer si nécessaire. Cliquez sur la liste Attributs supplémentaires pour ajouter d'autres attributs à la recherche.
     

    Note :
    Cette option est disponible uniquement si le module Physical Objects a été installé dans votre environnement.

     
    Effectuer une recherche d'attributs Records Management Dans la zone Ajouter au formulaire de recherche, cliquez sur Attributs Records Management. Dans la zone Attributs Records Management, cliquez sur la liste Attributs supplémentaires, puis sélectionnez et définissez l'attribut que vous souhaitez rechercher.
     

    Note :
    Cette option est disponible uniquement si le Records Management a été installé dans votre environnement.

     
    Pour effectuer une recherche à l'aide d'un modèle Dans la liste Utiliser ce formulaire de recherche, cliquez sur le modèle que vous souhaitez utiliser. La page Rechercher s'ouvre et indique les critères de recherche spécifiés dans le modèle sélectionné. Si nécessaire, vous pouvez modifier les critères de recherche.
    Ajouter des critères Si possible, cliquez sur Dupliquer  , sélectionnez un opérateur booléen dans la liste, puis spécifiez les critères supplémentaires.
    Supprimer des critères Si possible, cliquez sur Enlever  pour les critères que vous souhaitez supprimer.
    Supprimer une option du formulaire de recherche Cliquez sur Enlever un composant  pour l'option que vous souhaitez supprimer.
    Supprimer tous les critères de recherche Cliquez sur Effacer.

  3. Cliquez sur Rechercher.

 

Notes :

  • Lorsque vous spécifiez un intervalle de dates pour les catégories ou les attributs personnalisés d'une catégorie, vous pouvez laisser vide la date dans l'une des listes Du ou Au. Si vous ne mentionnez rien dans les listes Du, Content Server recherche toutes les dates antérieures à la date qui figurent dans les listes Au. Si vous ne mentionnez rien dans les listes Au, Content Server effectue la recherche à partir de la date sélectionnée dans les listes Du jusqu'à aujourd'hui.

  • Pour certains éléments de la liste des résultats de la recherche, l'option Lancer dans le navigateur du menu contextuel peut être la seule option disponible. Cela se produit lorsque vous n'êtes pas autorisé à accéder au dossier parent de l'élément renvoyé ou si le volume ne peut pas être affiché dans Enterprise Connect. Cliquez sur Lancer dans le navigateur si vous souhaitez utiliser l'élément sélectionné dans une autre instance du navigateur. Ce comportement s'applique également aux recherches et recherches rapides Enterprise Connect.

  • Selon votre environnement, vous pouvez peut-être définir les options d'affichage pour les résultats de la recherche en cliquant sur le lien Modifier les options d'affichage. Pour plus d'informations sur les options d'affichage disponibles sur la page Options d'affichage, reportez-vous à la rubrique Définition des options d`affichage.
