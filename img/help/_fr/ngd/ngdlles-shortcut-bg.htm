Créez un raccourci pour accéder à un même élément se trouvant à différents emplacements d'Content Server. Vous pouvez par exemple créer des raccourcis dans votre Espace de travail personnel pour le dossier auquel vous accédez le plus souvent. Vous pouvez créer des raccourcis au moyen du menu contextuel et de l'option Créer un raccourci, ou en cliquant avec le bouton droit et en faisant glisser un élément vers un recueil dans Content Server, au moyen de l'option Créer un raccourci ici. L'option de glisser-déposer est uniquement disponible dans l'Explorateur Windows. Cette option est uniquement disponible pour les types de recueil pris en charge, tels que Dossiers, Projets et Tâches. Elle n'est pas disponible pour les listes de tâches ou les groupes de tâches, par exemple. Selon la façon dont votre système est configuré, vous serez peut-être invité à fournir des métadonnées pour le nouveau raccourci.

Vous pouvez copier le lien d'un élément dans le Presse-papiers Windows à l'aide de la commande Copier le lien dans le Presse-papiers. Vous pouvez ensuite coller le lien dans des e-mails et applications telles que Word ou Excel, ou des navigateurs Web. Vous pouvez choisir le type de lien à envoyer ; les options disponibles dépendent du type d'élément que vous avez sélectionné. Par exemple, vous pouvez spécifier si le lien doit ouvrir la page Vue d'ensemble ou Propriétés du document ou s'il doit l'ouvrir ou le télécharger. Pour les e-mails, les e-mails composites, les dossiers et d'autres types d'éléments tels que les discussions, les sondages ou encore les listes de tâches, vous pouvez spécifier le lien de manière à ce qu'il ouvre l'élément, ou bien sa page Propriétés.
