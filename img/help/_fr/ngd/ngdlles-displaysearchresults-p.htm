Pour définir le nombre de résultats de recherche sur chaque page :

  1. Dans l'arborescence d'Enterprise Connect, cliquez sur un recueil Content Server, Personnel, puis sur Mes paramètres.

  2. Dans la section Configuration de recherche, sélectionnez le nombre de résultats de recherche approprié à afficher sur chaque page. La valeur par défaut est 25.

  3. Cliquez sur Envoyer pour appliquer les modifications.

     

    Astuce :
    Cliquez sur Réinitialiser pour rétablir les paramètres les plus récemment enregistrés. Cliquez sur Utiliser les paramètres par défaut pour rétablir les paramètres définis par l'administrateur.
