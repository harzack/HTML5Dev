Vous pouvez utiliser les formulaires de recherche avec vue personnalisée que vous (si vous en avez l'autorisation) ou votre administrateur avez créé dans l'interface utilisateur Web de Content Server pour effectuer des recherches dans Enterprise Connect. Pour plus d'informations sur la création et l'utilisation des formulaires de recherche avec vue personnalisée, voir Utilisation des recherches enregistrées.

 

Important :
Cette fonctionnalité est disponible uniquement dans Content Server 10, mise à jour 10, ou ultérieur.

 

Lorsque vous accédez à un dossier qui contient un formulaire de recherche avec vue personnalisée, le bouton Recherche personnalisée devient disponible dans la barre d'outils Enterprise Connect. En fonction de votre version de Content Server, vous pouvez peut-être accéder à plusieurs recherches avec vue personnalisée. Cette fonctionnalité est disponible dans Outlook et l'Explorateur, ainsi que dans les boîtes de dialogue Enregistrer sous d'Enterprise Connect, Copier/Déplacer d'Enterprise Connect et Ouvrir d'Enterprise Connect. Lorsque vous exécutez une recherche avec vue personnalisée à partir des boîtes de dialogue Enregistrer sous d'Enterprise Connect et Copier/Déplacer d'Enterprise Connect, les résultats renvoient des dossiers. Lorsque vous exécutez une recherche avec vue personnalisée à partir de la boîte de dialogue Ouvrir d'Enterprise Connect, les résultats renvoient des dossiers et des éléments individuels.
