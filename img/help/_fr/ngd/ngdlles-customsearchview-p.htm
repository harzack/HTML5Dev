Pour effectuer une recherche à l'aide des formulaires de recherche avec vue personnalisée :

  1. Accédez au dossier qui contient la recherche avec vue personnalisée que vous souhaitez utiliser.

     

    Note :
    La recherche avec vue personnalisée doit déjà se trouver dans le dossier sélectionné.

     

  2. Dans la barre d'outils Enterprise Connect, cliquez sur Recherche personnalisée.

  3. Dans la boîte de dialogue Recherche avec vue personnalisée, consultez les valeurs de recherche, puis cliquez sur Rechercher.

     

    Astuce :
    Vous pouvez modifier les valeurs de recherche disponibles avant de lancer la recherche. Cependant, les modifications que vous apportez ne sont pas enregistrées.
