Si votre administrateur a activé cette fonction, vous pouvez utiliser l'option de menu contextuel Parcourir depuis ici pour lancer une nouvelle fenêtre contenant soit l'interface utilisateur Web Content Server, soit la page Vue d'ensemble de l'élément sélectionné. Cette fonctionnalité est disponible pour l'arbre Enterprise Connect et dans le Volet Liste. Pour utiliser cette fonction, vous devez disposer des autorisations appropriées pour l'élément concerné. L'option de menu contextuel Parcourir depuis ici est disponible pour les éléments suivants :

  • Documents.

  • Mon espace de travail et le Dossier Entreprise.

  • Tous les types de recueils, tels que Dossiers, Dossiers d'e-mails et Collections.

  • Les recueils résidant dans les dossiers Favoris et Personnel, ainsi que le contenu dans Dossiers virtuels.

  • Les types non-recueil et non-document, tels que Générations, Prospecteurs, Sondages, Raccourcis, URL, Schémas de workflow, Statut du workflow, Catégories, Affichages personnalisés, Rapports LiveReports et DTD XML.

Par exemple, si vous cliquez avec le bouton droit de la souris sur Mon espace de travail et que vous sélectionnez Parcourir depuis ici, une nouvelle fenêtre s'ouvre, contenant la vue de l'interface utilisateur Web de Mon espace de travail. Si vous sélectionnez Parcourir depuis ici pour un raccourci, la Vue d'ensemble de l'élément d'origine s'affiche.
