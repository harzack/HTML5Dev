Lorsque vous affichez un élément dans le Volet Lecture, une série d'onglets s'affiche en bas du volet. Ces onglets représentent les propriétés disponibles pour l'élément sélectionné. Vous pouvez également afficher une fenêtre distincte pour une propriété en cliquant sur un élément avec le bouton droit de la souris, puis en sélectionnant Propriétés et en choisissant la propriété appropriée. Si vous disposez des autorisations appropriées, vous pouvez aussi modifier certaines zones dans les onglets de propriétés. Pour plus d'informations sur la modification de ces zones, reportez-vous à la rubrique Affichage et modification des propriétés d`un élément.

Selon le type d'élément que vous sélectionnez, certains des onglets suivants apparaissent :

  • Général : affiche le nom, le type, la date de création, le créateur, l'affichage, la description, la taille, la date de modification et le propriétaire de l'élément.

     

    Notes :

    • En fonction de votre configuration, des éléments masqués (Dossiers, Documents ou Listes de tâches par exemple) peuvent être accessibles dans l'arborescence d'Enterprise Connect. Les éléments masqués sont alors visibles dans le Volet Liste d'Outlook et de l'Explorateur et ils sont également inclus dans les résultats d'une recherche.

    • Enterprise Connect ne prend pas en charge la fonction Content Server Mis en valeur.

     

  • Spécifique : affiche des informations spécifiques sur l'élément, en fonction du type de l'élément. Par exemple, la date de la version d'un document composite ou le type MIME d'un document.

  • Audit : affiche les événements, dates et noms d'utilisateur se rapportant aux actions exécutées sur un élément.

  • Catégories : affiche la catégorie de l'élément et les paramètres d'attribut. Vous pouvez modifier les informations relatives à la catégorie à partir de cette page.

  • Classements : affiche les classements des éléments sélectionnés. Vous pouvez rechercher les classements disponibles à partir de cette page.

  • Colonnes : affiche des options d'affichage des colonnes personnalisées.

     

    Note :
    Vous devez actualiser le client Enterprise Connect pour appliquer les modifications apportées.

     

  • Lien : ne s'affiche que pour les raccourcis de documents composites. L'onglet affiche le type de version (version actuelle, révision actuelle, travail en cours) du document composite auquel renvoie le raccourci.

  • Présentation : affiche les options de présentation des espaces de travail et des dossiers. A partir de cet onglet, vous pouvez afficher des images, une étiquette de nouvelles et des éléments de sous-catalogue.

  • Références : affiche les éléments Raccourci ou Génération créés à partir du document d'origine, notamment le type d'élément, le nom et la date de création, l'heure et le nom d'utilisateur.

  • Versions : affiche toutes les versions d'un document composite correspondant, y compris l'icône Version, le nom et le numéro de version, les date et heure de création et le nom de l'utilisateur qui l'a créé.

  • Statistiques : affiche tous les messages et les réponses envoyés pour un élément Discussion, y compris les noms d'utilisateur, le nombre de messages envoyés, le nombre de réponses envoyées, le nombre total de messages envoyés et la date du dernier envoi.

  • Versions : affiche les versions de l'élément, y compris le numéro, le nom et la taille de la version, les date et heure de création et le nom de l'utilisateur qui l'a créé.

  • Affichage : affiche un aperçu de l'élément sélectionné.

  • Renvoi : affiche tous les renvois vers d'autres objets Content Server.
