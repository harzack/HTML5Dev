Pour gérer les e-mails ayant le même nom

  1. Dans l'arborescence d'Enterprise Connect, cliquez sur un recueil Content Server, Personnel, puis sur Mes paramètres.

  2. Dans la section Génération de nom pour les e-mails, accédez à la zone Lorsqu'un e-mail avec le même nom est ajouté à un dossier, puis effectuez l'une des opérations suivantes :

    1. Cliquez sur Générer un nom avec un suffixe si vous souhaitez qu'Enterprise Connect génère automatiquement un nouveau nom pour le doublon d'e-mail. Une nouvelle instance de l'e-mail, avec un suffixe permettant de la différencier de l'instance déjà présente dans la destination, est ajoutée dans la destination. Dans la liste adjacente, vous devez choisir l'une des options suivantes :

      • Sélectionnez Demander pour décider si vous souhaitez qu'Enterprise Connect vous invite à spécifier si vous souhaitez ajouter l'e-mail renommé automatiquement à l'emplacement de destination. C'est le paramètre par défaut.

      • Sélectionnez Générer un nom automatiquement si vous souhaitez qu'Enterprise Connect génère automatiquement un nouveau nom pour le doublon d'e-mail sans vous demander confirmation. Vous ne serez pas invité à résoudre les conflits de dénomination.

    2. Cliquez sur Entrez un nouveau nom si vous souhaitez pouvoir saisir un nouveau nom pour le doublon d'e-mail.

  3. Cliquez sur Envoyer pour appliquer les modifications.

 

Note :
Ces paramètres s'appliquent aux e-mails, aux e-mails composites et aux remplacements d'e-mails dans l'Explorateur Windows, Microsoft Outlook, Lotus Notes et GroupWise.

 
 

Astuce :
Cliquez sur Réinitialiser pour rétablir les dernières valeurs enregistrées pour les conventions de dénomination des e-mails. Cliquez sur Utiliser les paramètres par défaut pour rétablir les paramètres définis par l'administrateur.
