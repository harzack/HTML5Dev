Pour ajouter des étapes clés :

  1. Avec le bouton droit de la souris, cliquez sur la liste de tâches à laquelle vous souhaitez ajouter la tâche, cliquez sur Ajouter un élément, puis sur Etape clé.

  2. Dans la page Ajouter une étape clé, saisissez un nom pour l'étape clé dans la zone Nom.

  3. Cliquez sur un mois, une date, une année et une heure dans les listes Date prévue.

    Vous pouvez également décrire l'étape clé et spécifier des paramètres généraux.

  4. Cliquez sur Ajouter.
