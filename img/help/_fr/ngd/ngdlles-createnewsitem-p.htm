Pour créer des nouvelles :

  1. Cliquez avec le bouton droit de la souris sur la chaîne dans laquelle vous souhaitez créer la nouvelle, puis cliquez sur Créer une nouvelle.

  2. Dans la page d'ajout d'une nouvelle, tapez le nom de la nouvelle dans la zone Nom.

  3. Cliquez sur une Date d'entrée en vigueur et une Date d'expiration dans les listes Mois, Jour, Année et Heure correspondantes.

  4. Saisissez un titre pour la nouvelle dans la zone Titre.

     

    Note :
    Si vous ne remplissez pas la zone Titre, le texte se trouvant dans la zone Nom sera utilisé comme titre.

     

  5. Saisissez le texte de votre nouvelle dans la zone Contenu.

  6. Le tableau suivant décrit comment effectuer des tâches supplémentaires pour définir la Nouvelle.

    Pour Procéder ainsi
    Ajouter une image à une nouvelle Cliquez sur Parcourir Content Server dans la zone Image, recherchez une image que vous souhaitez joindre à la Nouvelle, puis cliquez sur son lien Sélectionner.
    Ajouter une description de l'image (pour les utilisateurs ne pouvant pas la voir) Saisissez une description dans la zone Texte ALT.
    Ajouter une pièce jointe à la nouvelle Cliquez sur Parcourir Content Server dans la zone Ajouter des pièces jointes Content Server, recherchez un élément que vous souhaitez joindre à la Nouvelle, puis cliquez sur son lien Sélectionner.
    Ajouter un fichier stocké sur votre ordinateur à la nouvelle Cliquez sur Parcourir dans la zone Ajouter une pièce jointe à partir du disque local, naviguez jusqu'à l'élément, puis cliquez sur Ouvrir.

  7. Cliquez sur Ajouter.

 

Astuce :
Vous pouvez également ajouter une nouvelle en cliquant avec le bouton droit sur un élément, puis en sélectionnant Créer une nouvelle. Par défaut, l'élément est ajouté en tant que pièce jointe à la nouvelle.
