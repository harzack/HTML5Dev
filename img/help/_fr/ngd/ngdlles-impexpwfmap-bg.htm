Si vous souhaitez créer une copie de sauvegarde d'une définition de Schéma de Workflow ou la déplacer d'un serveur Content Server à un autre, vous pouvez l'exporter vers un fichier sur votre ordinateur. Une fois le Schéma de Workflow exporté, vous pouvez importer le Schéma de Workflow lorsque vous souhaitez créer un Schéma de Workflow à partir du Schéma de Workflow exporté.

Si vous souhaitez créer un Schéma de Workflow en important une définition de Schéma de Workflow, vous devez d'abord ajouter un Schéma de Workflow vide ou utiliser un Schéma de Workflow existant que vous ne souhaitez pas enregistrer. L'importation remplace les données du Schéma de workflow existant par la définition du fichier importé.

Lorsque vous exportez un Schéma de Workflow, la définition de ce Schéma de Workflow est enregistrée sous la forme d'un fichier (avec l'extension .map) sur votre ordinateur. Les pièces jointes ne sont pas exportées avec un Schéma de Workflow. Si le Schéma de Workflow comporte des pièces jointes que vous souhaitez enregistrer en même temps que celui-ci, enregistrez-les séparément.
