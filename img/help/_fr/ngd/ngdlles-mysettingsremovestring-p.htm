Pour supprimer des chaînes des noms des e-mails :

  1. Dans l'arborescence d'Enterprise Connect, cliquez sur un recueil Content Server, Personnel, puis sur Mes paramètres.

  2. Dans la zone Chaînes à supprimer de la section Génération de nom pour les e-mails, saisissez le nom de la chaîne à exclure des noms des e-mails dans la zone Nouvelle chaîne, puis cliquez sur Déplacer vers la droite pour la déplacer dans la liste Chaînes sélectionnées. Cliquez sur Supprimer pour supprimer une chaîne sélectionnée de cette liste.

    Exemple: Pour que RE: n'apparaisse pas dans les noms des e-mails, saisissez RE: dans le champ Nouvelle chaîne, puis cliquez sur Déplacer vers la droite pour ajouter RE: à la liste Chaînes sélectionnées. RE: n'apparaît pas dans les noms des e-mails. Pour autoriser l'affichage de RE: dans les noms des e-mails, sélectionnez RE: puis cliquez sur Supprimer . RE: sera supprimé de la liste Chaînes sélectionnées.

  3. Cliquez sur Envoyer pour appliquer les modifications.

     

    Astuce :
    Cliquez sur Réinitialiser pour rétablir les dernières valeurs enregistrées pour les conventions de dénomination des e-mails. Cliquez sur Utiliser les paramètres par défaut pour rétablir les paramètres définis par l'administrateur.
