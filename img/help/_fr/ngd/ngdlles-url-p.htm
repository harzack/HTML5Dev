Pour ajouter des URL :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez ajouter une URL, cliquez sur Ajouter un élément, puis sur URL.

  2. Sur la page Ajouter : URL, saisissez un nom pour l'URL dans le champ Nom.

  3. Entrez l'URL dans la zone URL.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.

 

Note :
Vous pouvez modifier une URL en accédant à ses propriétés.
