Un document texte est un type de document contenant du texte brut (un fichier créé dans le Bloc-notes Windows par exemple). La majorité des fonctions disponibles pour les autres types de documents le sont également pour les documents texte. Vous pouvez par exemple les ajouter et modifier leur contenu.

Vous pouvez utiliser un éditeur de texte intégré pour rédiger des documents texte. L'éditeur de texte permet d'ajouter des fichiers texte simples, sans avoir à les créer préalablement dans une application distincte.

Pour ajouter un document texte se trouvant sur votre ordinateur, vous devez l'ajouter en tant que document. Content Server le reconnaît en tant que fichier texte et le traite comme un document texte. Lorsque vous créez ou ajoutez un document texte, vous pouvez fournir une description de ce document, modifier ses catégories et attributs et l'ajouter à un autre recueil.
