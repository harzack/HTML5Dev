Pour effectuer une recherche à l'aide des formulaires de recherche avec vue personnalisée à partir des boîtes de dialogue d'Enterprise Connect :

  1. Procédez de l'une des façons suivantes :

    • Pour rechercher un emplacement dans lequel enregistrer un document à l'aide d'un formulaire de recherche avec vue personnalisée, dans votre application, cliquez sur Enregistrer sous dans le menu Enterprise Connect.

    • Pour rechercher un document à ouvrir à l'aide d'un formulaire de recherche avec vue personnalisée, dans votre application, cliquez sur Ouvrir dans le menu Enterprise Connect.

    • Pour rechercher un emplacement vers lequel copier ou déplacer des e-mails à l'aide d'un formulaire de recherche avec vue personnalisée, cliquez avec le bouton droit de la souris sur les e-mails à copier ou déplacer, puis sélectionnez Copier/Déplacer vers Enterprise Connect.

  2. Pour utiliser le formulaire de recherche avec vue personnalisée approprié, procédez de l'une des manières suivantes :

    • Dans la boîte de dialogue Enregistrer sous d'Enterprise Connect, accédez au dossier qui contient le formulaire de recherche personnalisée que vous souhaitez utiliser.

    • Dans la boîte de dialogue Ouvrir d'Enterprise Connect, accédez au dossier qui contient le formulaire de recherche personnalisée que vous souhaitez utiliser.

    • Dans la boîte de dialogue Copier/Déplacer d'Enterprise Connect, accédez au dossier qui contient le formulaire de recherche personnalisée que vous souhaitez utiliser.

  3. Dans la barre d'outils Enterprise Connect, cliquez sur le bouton Recherche personnalisée et, si plusieurs options sont disponibles, sélectionnez la recherche avec vue personnalisée appropriée. Le bouton est disponible uniquement si une recherche avec vue personnalisée se trouve dans le dossier sélectionné.

  4. Dans la boîte de dialogue Recherche avec vue personnalisée, consultez les valeurs de recherche, puis cliquez sur Rechercher.

     

    Astuce :
    Vous pouvez éventuellement modifier certaines des valeurs de recherche disponibles avant de lancer la recherche. Cependant, les modifications que vous apportez ne sont pas enregistrées.

     

  5. Une fois les résultats de la recherche obtenus, effectuez l'une des opérations suivantes :

    • Si vous utilisez la boîte de dialogue Enregistrer sous, cliquez sur le dossier dans lequel vous souhaitez enregistrer le document. Enregistrez le document, à l'aide des options décrites à la rubrique OpenText Enterprise Connect Plug-in for Content Server - Manuel de l'utilisateur (NGDCORE-UGD).

    • Si vous utilisez la boîte de dialogue Ouvrir, cliquez sur le document que vous souhaitez ouvrir.

    • Si vous utilisez la boîte de dialogue Copier/Déplacer vers Enterprise Connect, cliquez sur le dossier vers lequel vous souhaitez copier ou déplacer les e-mails, puis cliquez sur Copier ou Déplacer.
