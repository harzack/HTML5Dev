Pour créer des catégories :

  1. Cliquez avec le bouton droit de la souris sur l'emplacement dans lequel vous souhaitez créer la catégorie, cliquez sur Ajouter un élément, puis cliquez sur Catégorie.

  2. Sur la page Ajouter une catégorie, saisissez un nom pour la catégorie dans la zone Nom.

  3. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur Ajouter.
