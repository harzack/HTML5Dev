Pour ouvrir une discussion :

  1. Cliquez sur l'icône Fonctions de la discussion et choisissez Ouvrir.

  2. Cliquez sur une période horaire dans la liste déroulante Depuis.

  3. Cliquez sur l'une des options suivantes dans le champ Mode d'affichage :

    • Affichage imbriqué

    • Affichage du rapport

  4. Cliquez sur l'une des options suivantes dans le champ Affichage :

    • Messages seulement

    • Non-lus seulement

    • Tous
