Pour participer complètement à une discussion, vous devez posséder les autorisations adéquates. Si vous disposez de l'autorisation Lecture pour une discussion, vous pouvez ouvrir les messages et les réponses publiés par les autres utilisateurs. Pour publier des messages et des réponses dans une discussion, vous devez disposer de l'autorisation Ecriture.

Lorsque vous publiez un message ou une réponse, vous pouvez ajouter une pièce jointe, indiquée par l'icône du trombone . Les documents utilisant la création de versions avancée peuvent être mis en pièce jointe uniquement si le document est une version principale. Pour plus d'informations sur la mise en pièce jointe de documents utilisant la création de versions avancée dans un message ou une réponse, reportez-vous à la rubrique Fonctionnement de la création de versions avancée pour les fonctions de Content Server.

Après avoir lu un message ou une pièce jointe, le système marque l'élément comme lu. Vous pouvez manuellement marquer tous les messages et les réponses comme lus, même si ce n'est pas le cas. Par ailleurs, vous êtes propriétaire de chaque message ou réponse que vous avez ajouté à une discussion. Par conséquent, vous pouvez les supprimer à tout moment.
