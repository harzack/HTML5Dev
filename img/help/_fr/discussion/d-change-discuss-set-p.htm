Pour modifier les paramètres de discussion :

  1. Sélectionnez Paramètres dans le menu Outils.

  2. Sur la page Mes paramètres : Général, cliquez sur l'onglet Discussion.

  3. Sur la page Mes paramètres : Discussion, cliquez sur une option dans la liste Valeur par défaut de la zone Depuis. Le paramètre par défaut est Pas de limite de temps.

  4. Dans la liste Affichage par défaut, cliquez sur Affichage imbriqué ou sur Affichage du rapport. Le paramètre par défaut est Affichage imbriqué.

  5. Dans le champ Préférences de filtre, cliquez sur l'un des boutons radio suivants : Messages seulement, Eléments non lus seulement ou Tous. Le paramètre par défaut est Eléments non lus seulement.

  6. Dans le champ Préférences de publication, décochez la case Utiliser le résumé du contenu comme objet si vous ne voulez pas afficher un extrait du début des messages et des réponses sans titre en guise d'objet.

  7. Dans le champ Préférences de réponses, décochez la case Citer les commentaires si vous ne voulez pas afficher le contenu de l'ensemble du fil de discussion sur la page Publier une réponse.

  8. Dans le champ Afficher les options de l'élément, décochez la case Afficher les fils de discussion en contexte si vous ne voulez pas afficher la section Fils de discussions en contexte en dessous du texte de la réponse ou du message en cours.

  9. Cliquez sur Mettre à jour.
