Pour ouvrir un message ou une réponse :

  1. Ouvrez une discussion.

  2. Sur la page Discussion, cliquez sur le nom du message ou de la réponse que vous souhaitez ouvrir.

  3. Cliquez sur l'un des liens actifs dans la barre de navigation Ouvrir l'élément.

    • Message précédent, qui affiche le message publié avant l'élément actuel.

    • Elément non-lu précédent qui affiche l'élément non-lu publié avant l'élément actuel.

    • Précédent qui affiche l'élément publié avant l'élément actuel.

    • Suivant qui affiche l'élément publié après l'élément actuel.

    • Elément non-lu suivant, qui affiche l'élément non-lu publié après l'élément actuel.

    • Message suivant qui affiche le message publié après l'élément actuel.

  4. Pour ouvrir une pièce jointe, procédez de l'une des façons suivantes dans la section Pièces jointes :

    • Pour ouvrir la pièce jointe dans Content Server au format HTML, cliquez sur son nom.

    • Pour ouvrir l'élément en utilisant une application externe, cliquez sur le lien Ouvrir.

    • Pour enregistrer une copie de la pièce jointe sur votre station de travail, cliquez sur le lien Télécharger.
