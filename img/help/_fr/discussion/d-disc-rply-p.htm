Pour répondre à un message :

  1. Ouvrez une discussion.

  2. Sur la page Discussion, cliquez sur le lien du message auquel vous voulez répondre.

  3. Cliquez sur l'icône Publier une réponse .

  4. Saisissez un titre pour la réponse que vous envoyez dans le champ Objet.

  5. Saisissez le texte de votre réponse dans le champ Commentaires.

  6. Si vous voulez ajouter un élément de Content Server en pièce jointe, cliquez sur le bouton Parcourir Content Server depuis le champ Ajouter une pièce jointe Content Server. Recherchez l'élément, puis cliquez sur son lien Sélectionner.

  7. Si vous le souhaitez, vous pouvez aussi ajouter un fichier de votre ordinateur en pièce jointe. Pour ce faire, cliquez sur le bouton Parcourir depuis le champ Ajouter une pièce jointe à partir du disque local. Recherchez le fichier, puis cliquez sur le bouton Ouvrir.

  8. Cliquez sur le bouton Envoyer.
