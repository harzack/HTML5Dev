Pour ajouter une discussion :

  1. Dans le menu Ajouter un élément, cliquez sur Discussion.

  2. Sur la page Ajouter : Discussion, tapez un nom pour la discussion dans le champ Nom.

  3. Vous pouvez également spécifier les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur le bouton Ajouter.

 

Astuce :
Vous pouvez également ajouter une discussion en choisissant Discussions dans le menu Personnel, puis en cliquant sur l'icône Créer une discussion.
