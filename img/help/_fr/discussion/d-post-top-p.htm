Pour publier un message :

  1. Ouvrez la discussion dans laquelle vous souhaitez publier un message.

  2. Sur la page Discussion, cliquez sur l'icône Publier un message.

  3. Saisissez un titre pour le message dans le champ Objet.

  4. Saisissez le texte de votre message dans le champ Commentaires.

  5. Si vous voulez ajouter un élément de Content Server en pièce jointe, cliquez sur le bouton Parcourir Content Server depuis le champ Ajouter une pièce jointe Content Server. Recherchez l'élément, puis cliquez sur son lien Sélectionner.

  6. Si vous le souhaitez, vous pouvez aussi ajouter un fichier de votre ordinateur en pièce jointe. Pour ce faire, cliquez sur le bouton Parcourir depuis le champ Ajouter une pièce jointe à partir du disque local. Recherchez le fichier, puis cliquez sur le bouton Ouvrir.

  7. Cliquez sur le bouton Envoyer.
