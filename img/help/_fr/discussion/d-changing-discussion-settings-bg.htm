Content Server permet de modifier les paramètres de filtre de toutes les discussions que vous êtes autorisé à ouvrir. Les paramètres de filtre s'appliquent à toutes les discussions, mais vous pouvez modifier certains filtres pour une discussion spécifique. Pour plus d'informations sur la modification des paramètres d'une discussion spécifique, reportez-vous à la rubrique Ajout et affichage de discussions.

Pour modifier les paramètres de filtre de toutes les discussions, vous devez modifier les paramètres suivants :

  • Préférences de publication, qui utilise le début d'un message comme ligne d'objet d'une réponse.

  • Préférences de réponses, qui place le contenu des messages et des réponses entre guillemets lorsque vous rédigez une réponse.

  • Afficher les options de l'élément, qui affiche un fil complet de message en contexte lorsque que vous rédigez une réponse.
