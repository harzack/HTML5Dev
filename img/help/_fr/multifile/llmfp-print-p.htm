Pour imprimer des objets en utilisant l'imprimante par défaut :

  1. Sélectionnez la case à cocher en regard d'un ou de plusieurs noms d'objet, puis cliquez sur Imprimer.

  2. La page Eléments à imprimer affiche une synthèse des documents sélectionnés pour l'impression. Cliquez sur Impression par défaut.

  3. Dans la fenêtre de confirmation qui s'affiche, cliquez sur OK.

    Les applications associées aux objets sélectionnés pour l'impression s'ouvrent temporairement en réponse à la requête d'impression. L'écran Statut de la fonction Imprimer, qui contient une fenêtre Processus d'impression, s'affiche également lorsque les documents sont envoyés à l'imprimante par défaut.

     

    Note :
    Si certains documents sélectionnés ne peuvent pas être imprimés, un message affiche ces documents dans une fenêtre séparée. Cliquez sur OK pour fermer cette fenêtre.

     

  4. Une fois la tâche d'impression terminée, cliquez sur OK.

Pour imprimer des objets sur une imprimante précise :

  1. Sélectionnez la case à cocher en regard d'un ou de plusieurs noms d'objet, puis cliquez sur Imprimer.

  2. La page Eléments à imprimer affiche une synthèse des documents sélectionnés pour l'impression. Cliquez sur Imprimer.

  3. Dans la fenêtre de confirmation qui s'affiche, cliquez sur OK.

    Les applications associées aux objets sélectionnés pour l'impression s'ouvrent temporairement en réponse à la requête d'impression. L'écran Statut de la fonction Imprimer, qui contient une fenêtre Processus d'impression, s'affiche également lorsque les documents sont envoyés à l'imprimante par défaut.

     

    Note :
    Si certains documents sélectionnés ne peuvent pas être imprimés, un message affiche ces documents dans une fenêtre séparée. Cliquez sur OK pour fermer cette fenêtre.

     

  4. Dans la fenêtre Imprimer, sélectionnez une imprimante dans la liste Nom, puis cliquez sur OK.

  5. Une fois la tâche d'impression terminée, cliquez sur OK.

 

Note :
La première fois que vous utilisez la fonction d'impression de Multi-File Output, vous êtes invité à télécharger un fichier printdoc.cab. Cliquez sur Oui pour l'installer. Notez que certains paramétrages de sécurité peuvent empêcher l'affichage de la boîte de dialogue. Si la boîte de dialogue ne s'affiche pas, désactivez momentanément le paramétrage de sécurité afin de pouvoir télécharger cette commande.

 
 

Astuce :
Lorsque vous voulez imprimer un fichier Microsoft Power Point, une boîte de dialogue Propriétés de l'imprimante peut s'afficher derrière la page du navigateur. Dans ce cas, cliquez sur la boîte de dialogue Propriétés de l'imprimante, puis sur OK pour procéder à l'impression.
