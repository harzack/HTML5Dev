La section Imprimer de Multi-File Output permet de sélectionner un ou plusieurs objets Content Server à envoyer directement à l'imprimante. Tous les objets Content Server ne peuvent pas être imprimés. Par exemple, les workflows et les discussions ne peuvent pas être imprimés. Il est uniquement possible d'imprimer les documents que la configuration de votre poste de travail permet d'imprimer. Si votre système local ne peut pas imprimer un document, un message d'erreur est généré.

 

Note :
La Sortie multi-fichiers ne prend pas en charge les documents composites pour les objets Enterprise Library Services.
