Pour zipper et envoyer des objets par e-mail :

  1. Sélectionnez la case à cocher en regard d'un ou de plusieurs noms d'objet, puis cliquez sur Zipper et envoyer par e-mail.

  2. Dans la page des éléments à zipper et télécharger, un nom généré par Content Server remplit automatiquement le champ Nom de l'archive. Vous pouvez remplacer ce nom par un nom plus approprié.

    La section Eléments contient une synthèse des éléments de votre archive. La section Nombre d'éléments pris en charge indique le nombre d'objets contenus dans le fichier ZIP.

  3. Pour envoyer une référence URL, vous devez enregistrer votre archive sur un emplacement Content Server. Dans le champ Archiver dans, cliquez sur Parcourir Content Server pour sélectionner l'emplacement.

     

    Notes :

    • Les utilisateurs auxquels vous envoyez l'URL doivent disposer au minimum d'une autorisation Voir le contenu dans cette zone pour pouvoir accéder au lien.

    • Si vous envoyez le fichier ZIP sous forme de pièce jointe, l'option d'archivage sur Content Server n'est pas disponible. Ce champ n'apparaît pas si vous n'envoyez pas de référence URL.

     

  4. Votre adresse e-mail, telle qu'elle est enregistrée dans Content Server, s'affiche par défaut dans le champ De.

  5. Dans le champ A, saisissez les adresses e-mail des utilisateurs destinataires de l'e-mail ou cliquez sur Utilisateur pour sélectionner d'autres utilisateurs Content Server.

  6. Dans le champ CC, saisissez les adresses e-mail des utilisateurs qui seront mis en copie de cet e-mail ou cliquez sur Utilisateur pour sélectionner d'autres utilisateurs Content Server.

  7. Dans le champ Objet, saisissez l'objet de votre message.

  8. Dans le champ Message, saisissez le corps de votre message.

  9. Si votre administrateur Content Server a configuré Multi-File Output de façon que les utilisateurs soient autorisés à choisir le mode de référencement de leurs archives, les boutons radio de la section Joindre une archive s'affichent.

    • Sélectionnez En tant que fichier pour envoyer l'archive ZIP sous forme de fichier joint à un e-mail.

    • Sélectionnez En tant que référence URL pour envoyer un lien URL dans le corps de l'e-mail.

    • Si votre administrateur Content Server a configuré Sortie multi-fichiers de manière à ce qu'une seule de ces options soit disponible, seule cette option s'affichera dans la section Joindre une archive.

  10. Cliquez sur Zipper et envoyer par e-mail.

    La page Statut de la fonction Zipper et envoyer par e-mail s'affiche, indiquant le statut de toutes les actions associées à l'opération Zipper en envoyer par e-mail.

    • Si l'archive est envoyée sous forme de fichier joint, les destinataires de cet e-mail reçoivent un message contenant l'archive ZIP en tant que fichier joint.

    • Si l'archive est envoyée sous forme de référence URL, les destinataires de cet e-mail reçoivent un message contenant un lien hypertexte les envoyant vers l'archive sur Content Server.

Pour zipper et envoyer des versions secondaires par e-mail :

  1. Depuis le menu Fonctions du document à zipper et à envoyer par e-mail, sélectionnez Propriétés, puis Versions.

  2. Dans l'onglet Versions, sélectionnez le menu Fonctions de la version secondaire du document à zipper et à envoyer par e-mail.

  3. Sélectionnez Zipper et envoyer par e-mail, puis suivez les étapes décrites dans Pour zipper et envoyer des objets par e-mail.
