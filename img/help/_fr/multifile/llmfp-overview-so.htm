Multi-File Output facilite et accélère le téléchargement, l'envoi par e-mail ou l'impression d'un ou de plusieurs documents à partir de Content Server. L'installation et la configuration de la Sortie multi-fichiers ajoutent les boutons suivants à l'interface de Content Server :

  • Zipper et télécharger : compresse un ou plusieurs objets Content Server dans un fichier ZIP pour téléchargement.

  • Zipper et envoyer par e-mail : compresse, télécharge et joint un ou plusieurs objets Content Server à un nouvel e-mail ou enregistre le fichier ZIP résultant dans Content Server et envoie par e-mail une URL avec un lien hypertexte.

  • Imprimer : télécharge et imprime un ou plusieurs documents.

La Sortie multi-fichiers fonctionne avec tout type de document pris en charge pour lequel l'utilisateur dispose au moins d'autorisations Voir le contenu. Elle peut extraire le contenu de dossiers entiers, de documents composites ou d'autres recueils pris en charge dans Content Server en une seule opération. Vous pouvez également utiliser la Sortie multi-fichiers avec des collections Content Server pour rassembler des informations situées à n'importe quel emplacement dans Content Server, puis les exporter globalement (une page à la fois), en imitant le système de « panier » du commerce électronique.

Multi-File Output prend également en charge des objets OpenText Enterprise Library. Lorsque vous exécutez une tâche Multi-File Output sur des objets OpenText Enterprise Library, Content Server utilise la variante par défaut pour le contenu.

 

Notes :

  • Pour que Multi-File Output fonctionne avec les versions secondaires de document, l'utilisateur doit disposer d'autorisations Réserver.

  • Les recueils dans Content Server peuvent inclure d'autres recueils, comme des dossiers dans des dossiers. Si l'un des objets sélectionnés est un recueil incluant d'autres recueils, l'action lancée ne s'étend pas aux sous-recueils : seuls les objets situés au niveau le plus élevé sont concernés. Pour appliquer une action au contenu d'un sous-recueil, accédez au menu Fonctions de ce recueil.

  • Les types de recueil pris en charge par la Sortie multi-fichiers sont les suivants : Documents composites, Dossiers, Projets, Tâches et Workflows.

  • La Sortie multi-fichiers ne prend pas en charge les documents composites pour les objets Enterprise Library.

 

Pour effectuer une action, cochez la case en regard d'un ou de plusieurs noms d'objets et cliquez sur le bouton d'action appropriée. Par exemple, cochez la case en regard d'un document et d'une feuille de calcul, puis cliquez sur le bouton Zipper et télécharger. Si un objet n'est pas configuré pour l'action sélectionnée, un message d'erreur s'affiche.

Si un objet est configuré pour une ou plusieurs fonctions Sortie multi-fichiers, vous pouvez lancer cette fonction directement à partir de son menu Fonctions. Si le type d'objet n'est pas configuré pour les fonctions Zipper et télécharger, Zipper et envoyer par e-mail et Imprimer, ces fonctions n'apparaissent pas dans le menu Fonctions.
