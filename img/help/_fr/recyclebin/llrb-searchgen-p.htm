Note :
Seuls les administrateurs système ou les utilisateurs possédant des droits d'administrateur peuvent créer une nouvelle vue de recherche.

 

Pour créer la nouvelle vue Corbeille :

  1. Allez à la page de recherche de Content Server Outils : Recherche.

  2. Si Portée n'est pas visible, elle peut être ignorée pour l'instant.

  3. Modifiez comme suit les champs de la zone Texte intégral :

    • Définissez Recherche sur la valeur Requête complexe.

    • Définissez Modifier sur la valeur<Aucun>.

    • Définissez Dans sur la valeur Tous.

    • Définissez Champ texte sur la valeur Emplacement OT:xxxx, où xxxx est l'ID du volume de la corbeille.

  4. Cliquez sur le bouton Enregistrer votre recherche.

  5. Nommez la recherche « Corbeille » (sans les guillemets) et cliquez sur le bouton Parcourir Content Server.

  6. Sélectionnez le Dossier vue de recherche, qui se trouve dans Content Server : Système Content Server (vous pouvez sélectionner Content Server dans le menu déroulant situé en haut à gauche de la page).

  7. Cliquez sur le bouton Ajouter ou cliquez sur Continuer si l'écran Catégorie apparaît, puis cliquez à nouveau sur le bouton Ajouter.

  8. Une fois la « Vue de recherche de la corbeille » créée, il convient de la définir correctement :

    • Sélectionnez le menu Fonctions de la vue de recherche Corbeille, puis sélectionnez Propriétés : Spécifique.

    • Assurez-vous que les valeurs spécifiées à l'étape 3 sont correctes :

      • Recherche a la valeur Requête complexe.

      • Modifier a la valeur<Aucun>.

      • Dans a la valeur Tous.

      • Champ texte a la valeur EmplacementOT: xxxx, où xxxx est l'ID du volume de la corbeille.

    • Cliquez sur Portée dans le menu de gauche si cela ne s'affiche pas. Assurez-vous que Portée a la valeur Gestionnaire de recherche Entreprise.

    • Cliquez sur le bouton Mettre à jour.

Pour exclure la corbeille de la vue de recherche Entreprise existante :

 

Note :
Afin d'éviter que des éléments de la corbeille n'apparaissent dans les vues de recherche classiques, la vue de recherche Entreprise ainsi que la vue de recherche Entreprise [Toutes les versions] doivent être modifiées.

 

  1. Sélectionnez le menu Fonctions pour la vue de recherche Entreprise, puis sélectionnez Propriétés : Spécifique. Cela vous permet d'obtenir les informations relatives à la vue de recherche Entreprise.

  2. Appuyez une seule fois sur le signe « plus » situé en regard de l'option Dans tous.

  3. Modifiez comme suit les champs sans supprimer aucune option de la zone Texte intégral déjà disponible :

    • Définissez Recherche sur la valeur Requête complexe.

    • Définissez Modifier sur la valeur<Aucun>.

    • Définissez Dans sur la valeur Tous.

    • Définissez Champ texte sur la valeur not "<OTCurrentVersion>FALSE".

    • Définissez Opérateur déroulant sur la valeur Et.

    • Définissez Recherche sur la valeur Requête complexe.

    • Définissez Modifier sur la valeur<Aucun>.

    • Définissez Dans sur la valeur Tous.

    • Définissez Champ texte sur la valeur (NOT OTLocation : xxxx), où xxxx est l'ID du volume de la corbeille.

  4. Cliquez sur Portée dans le menu de gauche si cela ne s'affiche pas. Assurez-vous que Portée a la valeur Gestionnaire de recherche Entreprise.

  5. Cliquez sur le bouton Mettre à jour.

  6. Faites de même pour la vue de recherche Entreprise [Toutes les versions]. Sélectionnez Propriétés : Spécifique dans le menu Fonctions. Par défaut, vous ne devez pas appuyer sur le signe +, dans la mesure où la ligne de recherche Texte intégral doit être vide. Assurez-vous que la portée a la valeur Gestionnaire de recherche Entreprise et cliquez ensuite sur le bouton Mettre à jour.

Les étapes décrites ci-dessus permettent de s'assurer que vos performances de recherche ne sont pas affectées par la corbeille Content Server. Si tel n'est pas le cas (si par exemple votre corbeille contient un volume de données important), il est conseillé de définir un processus d'extraction individuel pour la corbeille. Contactez votre gestionnaire de compte ou votre représentant commercial OpenText pour plus de détails.

 

Note :
Lorsqu'un objet est déposé dans la corbeille et restauré et qu'une recherche est ensuite effectuée avec la portée définie comme Entreprise ou Entreprise [Toutes les versions], il se peut qu'aucun résultat ne soit renvoyé. Cela est dû au fait que le processus Admin prend un certain temps avant que l'emplacement de l'objet ne soit mis à jour. Il est recommandé, dans ce cas, d'attendre cinq (5) minutes (voire plus longtemps en fonction du nombre d'éléments à indexer), puis de répéter la recherche.
