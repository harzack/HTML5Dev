Pour modifier des attributs d'une tâche, d'une liste de tâche, d'un groupe de tâches ou d'une étape clé :

  1. Cliquez sur l'icône Fonctions d'une tâche, d'une liste de tâches, d'un groupe de tâches ou d'un point de repère et choisissez Modifier les éléments.

  2. Sur la page Modifier les éléments, sélectionnez la case à cocher de chaque attribut que vous souhaitez modifier dans la colonne Modifier.

  3. Cochez la case de chaque élément auquel vous souhaitez appliquer les modifications dans la colonne Appliquer à.

  4. Cliquez sur le bouton Mettre à jour.
