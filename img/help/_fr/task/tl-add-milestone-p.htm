Pour ajouter une étape clé à une liste de tâches :

  1. Cliquez sur le lien de la liste des tâches.

  2. Sur la page Liste de tâches, cliquez sur le bouton Ajouter un élément, puis sélectionnez Etape clé.

  3. Sur la page Ajouter : Etape clé, saisissez un nom pour l'étape clé dans le champ Nom.

  4. Le champ Date prévue indique le mois, la date, l'année et l'heure actuels remplis par défaut. Vous pouvez également cliquer sur un mois, une date, une année et une heure dans les listes Date prévue pour définir la date prévue d'achèvement de la tâche.

  5. Par défaut, le champ Créer dans contient le nom de la liste de tâches sélectionnée à la première étape. Pour changer la liste de tâches dans laquelle vous créerez cette étape clé, dans le champ Créer dans, cliquez sur Parcourir Content Server pour sélectionner une autre liste de tâches.

  6. Facultatif Ajoutez une description de la nouvelle étape clé dans le champ Description. Vous pouvez également, dans le champ Catégories, cliquez sur Modifier pour sélectionner une catégorie à appliquer à cette étape clé.

  7. Cliquez sur le bouton Ajouter.
