Vous pouvez afficher des informations sur les listes de tâches, tâches, étapes clés et utilisateurs avec une tâche assignée dans les pages d'informations suivantes :

  • Résumé de la liste de tâches, qui fournit un résumé des informations de tous les éléments contenus dans une liste de tâches.

  • Pages d'informations sur les points de repère, qui fournissent des informations sur des points de repère particuliers et les points de repère d'une liste de tâches.

    La page Tâches absentes de toute étape clé est accessible via la page d'informations sur l'étape clé.

  • Liste Détails de ressources, qui fournit un résumé de chaque utilisateur ou groupe dans une liste de tâches contenant des tâches assignées.

    La page Détails sur l'utilisateur avec une tâche assignée est accessible via la page Détailler la liste des ressources.

 

Note :
Seuls le créateur d'une tâche, le coordinateur de projet et l'utilisateur avec une tâche assignée peuvent modifier les informations d'une liste de tâches sur ses pages d'informations.

 

Pour afficher les pages d'informations

Pour afficher la... Effectuez les opérations suivantes...
Page Résumé de la liste de tâches Cliquez sur le lien de la liste des tâches. Vous accédez à la page de cette liste de tâches. Dans la barre de menu principale, située en haut de la page, cliquez sur l'élément de menu Liste de tâches. Sélectionnez Résumé.
Page Détailler la liste des étapes clés Cliquez sur le lien de la liste des tâches. Vous accédez à la page de cette liste de tâches. Dans la barre de menu principale, située en haut de la page, cliquez sur l'élément de menu Liste de tâches. Sélectionnez Détailler la liste des étapes clés.
Page Tâches absentes de toute étape clé Dans la page Détailler la liste des étapes clés, cliquez sur le lien Tâches absentes de toute étape clé.
Page Liste-Détails de ressources Cliquez sur le lien de la liste des tâches. Vous accédez à la page de cette liste de tâches. Dans la barre de menu principale, située en haut de la page, cliquez sur l'élément de menu Liste de tâches. Sélectionnez Détailler la liste des ressources.
Page Détails de l'utilisateur avec une tâche assignée Sur la page Détailler la liste des ressources, cliquez sur l'icône Vue détaillée  située dans le champ Utilisateur avec une tâche assignée.
