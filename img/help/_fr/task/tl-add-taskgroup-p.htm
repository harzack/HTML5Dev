Pour ajouter un groupe de tâches :

  1. Cliquez sur un lien de Liste de tâches ou de Groupe de tâches.

  2. Cliquez sur le bouton Ajouter un élément, puis sélectionnez Groupe de tâches.

  3. Sur la page Ajouter : Groupe de tâches, saisissez un nom pour le groupe de tâches dans le champ Nom.

  4. Facultatif Ajoutez une description du nouveau groupe de tâches dans le champ Description, cliquez sur une étape clé dans la liste Etape clé par défaut pour les tâches pour assigner une étape clé au groupe de tâches et cliquez sur Modifier dans le champ Catégories pour sélectionner une catégorie à appliquer à ce groupe de tâches.

  5. Cliquez sur le bouton Ajouter.
