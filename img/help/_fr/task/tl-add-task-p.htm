Pour ajouter une tâche :

  1. Cliquez sur un lien de Liste de tâches, de Tâche ou de Groupe de tâches.

  2. Sur la page Liste de tâches, Tâche ou Groupe de tâches, cliquez sur Tâche dans le menu Ajouter un élément.

  3. Sur la page Ajouter : Tâche, saisissez un nom pour la tâche dans le champ Nom.

  4. Dans la liste déroulante Priorité, cliquez sur Faible, Moyen ou Elevé.

  5. Dans la liste déroulante Statut, cliquez sur un statut.

  6. Facultatif Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  7. Cliquez sur le bouton Ajouter.

Pour... Effectuez les opérations suivantes...
Assigner une tâche Dans le champ Assigné à, cliquez sur l'icône Choisir un utilisateur ou un groupe . Accédez à un utilisateur ou un groupe, puis cliquez sur le lien Sélectionner >.
Spécifier les dates de commencement et d'échéance pour la tâche Dans les listes Date de début et Date d'échéance, cliquez sur un mois, une date, une année et une heure. Par défaut, la date à laquelle vous créez la tâche est insérée dans le champ Date de début.
Définir une priorité Dans le champ Priorité, cliquez sur faible, moyenne ou élevée. Par défaut, ce champ indique moyenne.
Définir un statut Dans le champ Statut, cliquez sur en attente, en cours, problème, en suspens, terminé ou annulé. Par défaut, ce champ indique en attente.
Assigner une étape clé à une tâche Si des étapes clés ont été créées, vous pouvez cliquer sur l'une d'entre elles dans la liste Etape clé.
Ajouter des instructions ou des commentaires pour la tâche Saisissez les informations dans les champs de texte Instructions et Commentaires.
Ajouter une pièce jointe Content Server Dans le champ Ajouter une pièce jointe Content Server, cliquez sur Parcourir Content Server. Cliquez sur le texte Sélectionner > en regard d'un élément que vous souhaitez joindre à votre tâche.
Ajouter une pièce jointe Desktop Dans le champ Ajouter une pièce jointe Desktop, cliquez sur Parcourir. Accédez au fichier sur votre ordinateur que vous souhaitez joindre, mettez-le en surbrillance, puis cliquez sur Ouvrir.
Assigner une catégorie Dans le champ Catégories, cliquez sur Modifier. Dans la zone Modifier la catégorie, cliquez sur Ajouter des catégories. Si votre administrateur a créé des catégories, accédez à la catégorie souhaitée, puis cliquez sur le texte Sélectionner > en regard de la catégorie que vous souhaitez assigner à cette tâche. Cliquez sur Terminé.
