Pour ajouter une liste de tâches :

  1. Dans le menu Ajouter un élément, cliquez sur Liste de tâches.

  2. Sur la page Ajouter : Liste de tâches, saisissez un nom pour la liste de tâches dans le champ Nom.

  3. Facultatif Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  4. Cliquez sur le bouton Ajouter.
