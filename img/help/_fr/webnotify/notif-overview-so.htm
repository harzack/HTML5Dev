Une notification constitue un système qui vous informe des modifications qui surviennent dans Content Server. Vous pouvez indiquer les activités ou les éléments dont vous souhaitez suivre l'évolution et la fréquence à laquelle vous souhaitez être informé par e-mail de toute modification apportée. Vous pouvez par exemple configurer des notifications pour être informé des tâches qui sont en retard tous les lundis matin à 8:00.

Les notifications sont indiquées dans le projet et dans les pages de rapports de notification. Elles peuvent également être envoyées directement à votre adresse e-mail.

 

Note :
Vous pouvez définir les notifications pour le module Pulse sur votre page Pulse en cliquant sur Paramètres utilisateur dans la barre latérale Actions. Pour en savoir plus, voir Pour modifier vos paramètres utilisateur.
