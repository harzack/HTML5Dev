Pour définir des demandes de notification spécifiques :

  1. Cliquez sur l'icône Fonctions d'un élément, puis choisissez Définir une notification.

  2. Pour chaque centre d'intérêt de la page Définir une notification, dans la liste d'éléments, cliquez sur le nom du rapport vers lequel vous souhaitez envoyer une notification.

  3. Cliquez sur Envoyer.

 

Note :
Les projets ont leurs propres rapports de notification. Pour modifier les demandes de notification de projet, procédez de la même façon que pour modifier les demandes de notification générales. Pour en savoir plus, voir Définition des demandes de notification.
