Pour définir des paramètres de notification pour un groupe :

  1. Choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, cliquez sur Nom de groupe dans la liste Rechercher, saisissez le nom d'un groupe dans la zone qui commence par, puis cliquez sur Rechercher.

  3. Cliquez sur Modifier pour le groupe dont vous souhaitez modifier les paramètres.

  4. Dans la zone Modifier la notification de la page Modifier le groupe, cliquez sur Paramètres.

  5. Dans la boîte de dialogue des paramètres de notification, modifiez les Paramètres de notification disponibles, puis cliquez sur Envoyer.

  6. Cliquez sur Terminé.
