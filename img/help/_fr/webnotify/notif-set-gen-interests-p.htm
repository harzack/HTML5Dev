Pour définir des centres d'intérêt généraux :

  1. Dans le menu Personnel, sélectionnez l'option Notification.

  2. Sur la page Mes rapports de notification, cliquez sur l'onglet du rapport à modifier.

  3. Cliquez sur le bouton Modifier les centres d'intérêt, .

  4. Pour chaque élément de centre d'intérêt de la page Définir une notification, dans la liste d'éléments, cliquez sur le rapport vers lequel vous souhaitez envoyer une notification.

  5. Facultatif Dans la zone Centres d'intérêt de l'élément spécifique, cliquez sur le lien d'un élément pour modifier ses centres d'intérêt spécifiques.

  6. Cliquez sur Envoyer.
