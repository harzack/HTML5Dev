Pour définir des demandes de notification de groupe :

  1. Dans la barre de menu générale, choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Sur la page Utilisateurs et groupes, cliquez sur Nom de groupe dans la liste Rechercher, saisissez le nom d'un groupe dans la zone qui commence par, puis cliquez sur Rechercher.

  3. Dans la zone Modifier la notification, cliquez sur Centres d'intérêt, sélectionnez le rapport dans lequel la notification doit être envoyée pour chaque élément de centre d'intérêt, puis cliquez sur Envoyer.

  4. Cliquez sur Terminé.
