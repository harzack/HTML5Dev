Pour créer un dossier d'entrepôt :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Sur la page Entrepôt de transport, cliquez sur le bouton Ajouter un élément, puis sélectionnez Dossier de l'entrepôt.

  3. Sur la page Ajouter, saisissez un nom pour le dossier de l'entrepôt dans le champ Nom, puis cliquez sur le bouton Ajouter.

 

Note :
Si vous voulez que d'autres utilisateurs accèdent aux dossiers de l'entrepôt que vous créez, vous devez leur accorder les autorisations nécessaires individuellement. Pour en savoir plus, voir Gestion des autorisations.
