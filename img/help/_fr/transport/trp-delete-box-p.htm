Pour supprimer un élément de transport :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Sur la page Entrepôt, sélectionnez la case à cocher pour chaque élément que vous souhaitez supprimer, puis cliquez sur le bouton Supprimer.

  3. Sur la page Confirmation, cliquez sur le bouton Supprimer, puis cliquez sur OK.

 

Note :
Les lots de transport ne peuvent être supprimés.
