Pour ajouter des dossiers à un atelier :

  1. Recherchez le dossier que vous souhaitez ajouter à l'atelier, cliquez sur son menu Fonctions, puis cliquez sur Ajouter à l'entrepôt.

  2. Facultatif Sur la page Ajouter, sélectionnez la case à cocher Inclure le contenu.

  3. Dans la section Ajouter à l'atelier, cliquez sur le bouton Parcourir Content Server , recherchez l'atelier dans lequel vous souhaitez ajouter le dossier, puis cliquez sur son lien Sélectionner.

  4. Dans la section Dupliquer le traitement, sélectionnez l'une des options suivantes :

    • Remplacer, qui remplace tous les éléments en double dans l'atelier par les éléments que vous ajoutez.

    • Ignorer, qui ignore les éléments en double et laisse l'élément existant dans l'atelier.

  5. Cliquez sur le bouton Ajouter à l'entrepôt.
