Pour déployer un élément :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Dans l'entrepôt de transport, recherchez l'élément que vous souhaitez déployer, cliquez sur son menu Fonctions, puis cliquez sur Vue d'ensemble.

  3. Cliquez sur le bouton Déployer.

  4. Cliquez sur OK dans la boîte de dialogue confirmation, puis cliquez à nouveau sur Déployer.
