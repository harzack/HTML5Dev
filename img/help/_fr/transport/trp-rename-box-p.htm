Pour renommer un lot de transport :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Cliquez sur le lien Lots de transport, puis cliquez sur le menu Fonctions du lot que vous devez renommer, et enfin cliquez sur Renommer.

  3. Saisissez un nouveau nom du lot dans le champ Nom, puis cliquez sur Mettre à jour.
