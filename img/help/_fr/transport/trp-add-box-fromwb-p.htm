Pour créer un lot de transport :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Sur la page Entrepôt de transport, ouvrez le recueil de lots de transport.

  3. Sur la page Lots de transport, cliquez sur Lot de transport dans le menu Ajouter un élément.

  4. Sur la page Ajouter un lot de transport, saisissez un nom pour le lot de transport dans le champ Nom.

     

    Note :
    Dans le cas où un lot de transport existe déjà sous le nom que vous avez saisi, un message d'erreur apparaît.

     

  5. Facultatif Saisissez une description du contenu du lot dans le champ Description.

  6. Facultatif Pour ajouter un lot de transport existant au dossier du lot par défaut, cliquez sur le bouton Parcourir dans la section Fichier du lot de transport, recherchez le fichier du lot que vous souhaitez charger, puis cliquez sur Ouvrir.

     

    Note :
    Le fichier doit être un fichier ZIP et ne contenir que des éléments du lot.

     

  7. Cliquez sur Ajouter.

 

Astuce :
Si vous disposez des autorisations appropriées, vous pouvez également créer un lot à partir de l'atelier. Pour ce faire, sélectionnez la case à cocher pour chaque élément à ajouter au lot, puis cliquez sur le bouton Tout ajouter au lot de transport.
