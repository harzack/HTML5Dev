Pour exporter un lot :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Dans l'entrepôt de transport, recherchez le lot de transport que vous souhaitez télécharger, cliquez sur Télécharger dans le menu Fonctions, puis enregistrez le fichier ZIP dans un emplacement externe.

 

Note :
Après le téléchargement du lot de transport, vous devez le transférer par e-mail, réseau ou clé USB vers un emplacement du système cible.
