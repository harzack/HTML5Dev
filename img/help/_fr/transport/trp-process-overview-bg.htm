Transport d'éléments

Dossiers, documents et documents composites, URL, vues personnalisées, projets, facettes et dossiers des facettes, ainsi que rapports LiveReport peuvent être transportés vers un système cible. D'autres éléments peuvent être transportés mais nécessitent une vérification supplémentaire, afin de déterminer si les dépendances pour ces éléments sont respectées. Par exemple :

  • Catégories : les catégories appliquées sur les éléments transportés seront également appliquées sur les autres éléments durant le transport.

  • Autorisations : les autorisations sur les éléments sont affichées uniquement à titre d'information.

     

    Note :
    Si le même élément avec le même GUID existe sur le système cible, les autorisations seront appliquées.
