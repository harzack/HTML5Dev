L'entrepôt de transport est une zone de Content Server destinée aux gestionnaires d'entrepôts et réservée aux fonctions exclusives de transport. L'entrepôt de transport est accessible à partir du menu Entreprise.

 

Note :
L'option Entrepôt de transport du menu Entreprise n'est visible que pour les utilisateurs dont l'accès à l'entrepôt a été autorisé par l'administrateur. Par défaut, seul l'administrateur peut accéder à l'entrepôt.

 

L'entrepôt est constitué des composants suivants :

  • Dossier de l'entrepôt, qui est un recueil utilisé pour stocker les éléments que vous préparez pour le transport. Les administrateurs d'entrepôt peuvent créer des dossiers d'entrepôt dans l'entrepôt de transport et donner des autorisations à des utilisateurs spécifiques afin qu'ils puissent accéder aux dossiers.

  • Atelier, qui est constitué des éléments Content Server. Lorsqu'un élément de Content Server est déplacé vers l'atelier, il est appelé élément de transport. Les utilisateurs disposant des autorisations appropriées peuvent accéder aux ateliers. Pour en savoir plus, voir Utilisation de l'atelier de transport.

    Les ateliers peuvent être directement ajoutés à un entrepôt ou à d'autres dossiers d'entrepôt. Content Server Les éléments doivent être ajoutés à un atelier. Ils ne peuvent pas être directement ajoutés à l'entrepôt.

  • Lot de transport, qui est le recueil dans lequel sont ajoutés les éléments provenant des ateliers lorsqu'ils sont prêts à être transportés vers le système cible.

    Lorsque vous préparez l'atelier, vous pouvez ajouter des fichiers de propriétés et des instructions en plus des autres éléments. Ces instructions peuvent être utiles au déploiement, en particulier lorsqu'un administrateur ou un administrateur d'entrepôt différent est responsable du déploiement du contenu sur le système cible.

     

    Note :
    Seuls les administrateurs d'entrepôt peuvent utiliser les lots de transport.

     

La navigation à facettes est disponible pour les éléments dans un entrepôt. Par conséquent, vous pouvez rapidement voir les éléments en fonction du type d'élément d'entrepôt (atelier, lot de transport, éléments de transport ou dossiers d'entrepôt). Vous pouvez également voir les éléments en fonction de la date de leur dernière modification ou du propriétaire de l'élément. Pour plus d'informations sur la navigation à facettes pour Content Server, voir Utilisation de la navigation à facettes.
