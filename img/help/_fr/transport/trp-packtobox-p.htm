Pour ajouter des éléments d'atelier à un lot :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Cliquez sur le lien de l'atelier qui contient l'élément que vous souhaitez ajouter à un lot de transport, puis cliquez sur le lien Ajouter au lot de transport de l'élément.

  3. Sur la page Ajouter, dans le champ Nom saisissez un nom pour le lot de transport dans lequel l'élément sera ajouté, puis cliquez sur Ajouter.

 

Note :
Vous pouvez également cliquer sur le menu Fonctions de l'élément, puis sur Ajouter au lot de transport.

Pour ajouter tous les éléments de l'atelier à un lot, cliquez sur le bouton Tout ajouter au lot de transport.
