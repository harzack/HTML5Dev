Pour déployer un atelier :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Dans l'entrepôt de transport, cliquez sur le lien de l'atelier que vous souhaitez déployer, puis cliquez sur le bouton Déployer l'atelier.

  3. Sur la page Déployer, vérifiez que les informations de l'atelier, ainsi que les éléments à déployer sont exacts, puis cliquez sur Déployer.

     

    Note :
    Pour modifier un élément avant le déploiement de l'atelier, cliquez sur le bouton Modifier de l'élément dans la colonne Actions.

     

  4. Dans la boîte de dialogue de confirmation, cliquez sur OK.

  5. Lorsque le déploiement est terminé, confirmez l'état pour chaque élément, puis cliquez sur OK.
