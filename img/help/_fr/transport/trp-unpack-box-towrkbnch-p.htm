Pour décompresser un lot vers un atelier :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Dans l'entrepôt de transport, recherchez le lot de transport que vous souhaitez ouvrir, cliquez sur son menu Fonctions, puis cliquez sur Ouvrir.

  3. Sur la page Vue d'ensemble du lot de transport, cliquez sur le bouton Décompresser vers l'atelier.

  4. Dans la section Décompresser vers de la page Décompresser, cliquez sur le bouton Parcourir Content Server, recherchez l'atelier dans lequel vous souhaitez décompresser le contenu du lot, puis cliquez sur son lien Sélectionner.

  5. Dans la section Dupliquer le traitement, cliquez sur l'un des boutons radio suivants :

    • Remplacer, qui remplace les éléments dans l'atelier par l'élément du lot de transport s'ils ont le même ID unique.

    • Ignorer, qui ignore les éléments dans l'atelier ayant le même ID unique. Ces éléments ne seront pas décompressés vers l'atelier.

  6. Cliquez sur Décompresser.

 

Astuce :
Vous pouvez également décompresser un lot vers un atelier en cliquant sur le menu Fonctions du lot de transport, puis en cliquant sur Décompresser.
