Pour ajouter un élément à un atelier :

  1. Recherchez l'élément que vous souhaitez ajouter à l'atelier, cliquez sur son menu Fonctions, puis cliquez sur Ajouter à l'entrepôt.

  2. Dans la section Ajouter à l'atelier, cliquez sur le bouton Parcourir Content Server , recherchez l'atelier dans lequel vous souhaitez ajouter l'élément, puis cliquez sur son lien Sélectionner.

  3. Dans la section Dupliquer le traitement, cliquez sur l'une des options suivantes :

    • Remplacer, qui remplace l'élément dans l'atelier par l'élément que vous ajoutez.

    • Ignorer, qui ignore l'élément et laisse l'élément existant dans l'atelier tel qu'il est.

     

    Note :
    La section Dupliquer le traitement apparaît uniquement pour les recueils.

     

  4. Cliquez sur le bouton Ajouter à l'entrepôt.

 

Note :
Vous pouvez ajouter des dossiers, y compris le contenu du dossier, à un atelier en cliquant sur le menu Fonctions du dossier, puis en cliquant sur Ajouter à l'atelier.
