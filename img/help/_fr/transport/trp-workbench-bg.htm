L'atelier est l'emplacement principal du système source dans lequel les utilisateurs stockent et travaillent sur les éléments qui vont être transportés. Les ateliers sont également utilisés sur le système cible pour vérifier et déployer les éléments qui ont été transportés.

L'atelier peut être utilisé par plusieurs utilisateurs, dès lors qu'ils disposent de l'autorisation d'accès. Les utilisateurs peuvent voir les ateliers, pour lesquels ils disposent de l'autorisation, sur la page Mes ateliers, accessible à partir du menu Personnel. Si vous disposez d'une autorisation, vous pouvez également cliquer sur le lien Entrepôt dans la colonne Emplacement de l'atelier pour accéder à l'entrepôt dans lequel se trouve l'atelier.

Lorsque vous ouvrez un atelier, la page Vue d'ensemble de l'atelier affiche la liste de tous les éléments contenus dans cet atelier. La page Vue d'ensemble de l'atelier donne les détails des éléments contenus dans l'atelier, comme la date à laquelle ils ont été ajoutés à l'atelier ou la dernière fois qu'ils ont été ajoutés à un lot de transport.

Ajout d'éléments à un atelier

Vous pouvez ajouter des dossiers, des documents et documents composites, des URL, des vues personnalisées, des projets et des rapports LiveReports à un atelier. Si vous ajoutez un dossier à un atelier, vous pouvez également inclure tout le contenu du dossier. Vous ne pouvez pas sélectionner des éléments spécifiques dans le dossier à moins de les ajouter individuellement. Si vous ajoutez un dossier et que vous sélectionnez l'option Inclure le contenu, tous les éléments du dossier seront ajoutés à l'atelier. Avant de les ajouter, la fonction Transport vérifie automatiquement si les éléments existent déjà dans l'atelier. Si un élément existe en double, vous pouvez remplacer l'élément existant ou ignorer le double et laisser l'élément existant ainsi.

 

Note :
Contrairement à la plupart des éléments dans Content Server, les éléments dans un atelier ne peuvent pas être copiés ou déplacés.

 

Ajout d'éléments vers un lot de transport

 

Note :
Les lots de transport apparaissent dans l'entrepôt et ne sont accessibles que par l'administrateur de l'entrepôt.

 

Un lot de transport est un recueil qui stocke l'ensemble des éléments prêts à être transférés vers le système cible. Le dossier Lots de transport est le recueil par défaut et apparaît automatiquement dans l'entrepôt. Les lots de transport sont créés à partir de l'entrepôt, ou à partir de l'atelier si vous disposez des autorisations appropriées, et ajoutés au recueil de lot de transport par défaut. Si un lot de transport est créé à partir de l'atelier, un message apparaît indiquant la disponibilité du lot pour le gestionnaire d'entrepôts.

Vous pouvez ajouter des lots supplémentaires au dossier Lots de transport. Des informations générales concernant les lots de transport apparaissent sur la page Vue d'ensemble du lot de transport, comme le nom, la description, la taille, la date et l'auteur de sa création, et de quelle instance de Content Server provient le lot. La page Vue d'ensemble s'affiche lorsque vous ouvrez un lot, soit en cliquant sur le lien du lot, soit en cliquant sur Ouvrir dans le menu Fonctions du lot.

Ajout d'un atelier à un lot de transport

Lorsqu'un atelier contient tous les éléments requis pour le transport et que l'atelier a été contrôlé et testé, les éléments sont ajoutés à un lot. L'ajout à un lot signifie que vous finalisez les éléments pour le transport vers le système cible. Lorsque le lot est finalisé, il est déplacé vers la zone de lots de transport de l'entrepôt dont l'accès est limité aux utilisateurs disposant des autorisations d'administration.

La fonction Ajouter au lot de transport est la dernière tâche d'atelier pour les utilisateurs non administrateurs. Cependant, l'administrateur d'entrepôt utilise toujours le lot de transport pour débuter le processus de déploiement. Pour en savoir plus, voir Déploiement d'un atelier.
