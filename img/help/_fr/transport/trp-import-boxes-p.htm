Pour importer un lot :

  1. Sur le système cible, dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Dans l'entrepôt de transport, cliquez sur le lien Lots de transport.

  3. Sur la page Lots de transport, cliquez sur le menu Ajouter un élément, puis cliquez sur Lot de transport.

  4. Sur la page Ajouter, saisissez un nom pour le lot dans le champ Nom.

  5. Facultatif Saisissez une description du lot dans le champ Description.

  6. Dans la section Fichier du lot de transport, cliquez sur le bouton Parcourir, recherchez le fichier que vous souhaitez ajouter, puis cliquez sur le bouton Ouvrir.

  7. Cliquez sur Ajouter.
