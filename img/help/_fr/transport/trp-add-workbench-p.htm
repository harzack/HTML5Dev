Pour créer un atelier :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Sur la page Entrepôt de transport, cliquez sur le bouton Ajouter un élément, puis sélectionnez Atelier.

  3. Sur la page Ajouter, saisissez un nom pour l'atelier dans le champ Nom, puis cliquez sur le bouton Ajouter.
