Pour modifier le recueil parent d'un élément :

  1. Dans le menu Entreprise, cliquez sur Entrepôt de transport.

  2. Ouvrez l'atelier contenant l'élément que vous souhaitez modifier, cliquez sur le menu Fonctions de l'élément, puis cliquez sur Ouvrir.

  3. Sur la page Vue d'ensemble, cliquez sur le bouton Modifier.

  4. Dans la section Informations sur l'objet d'origine, cliquez sur le bouton Parcourir Content Server, recherchez le répertoire parent que vous souhaitez modifier, puis cliquez sur son lien Sélectionner.

  5. Cliquez sur Enregistrer.
