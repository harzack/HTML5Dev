Pour ajouter un document texte :

  1. Dans le menu Ajouter un élément, sélectionnez Document texte.

  2. Sur la page Ajouter : Document texte, saisissez le nom du document texte dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description pour le document texte.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier la description dans les autres langues activées.

  4. Dans le champ Texte, saisissez le contenu du document texte.

  5. Facultatif Egalement dans le champ Texte, cliquez sur l'icône Vérifier l'orthographe pour rechercher des fautes d'orthographe. L'icône Vérifier l'orthographe se trouve dans le champ Texte à gauche de la zone de saisie du contenu.

     

    Note :
    La vérification de l'orthographe n'est pas disponible lorsque Java est désactivé. Pour en savoir plus, voir Utilisation des technologies d`accessibilité et du langage Java.

     

  6. Dans la liste Type MIME, sélectionnez un type MIME pour le document texte. La valeur par défaut est texte normal.

  7. Facultatif Le cas échéant, vous pouvez appliquer un classement au document texte. Depuis la liste des Classements, sélectionnez Parcourir les classements ou Mes favoris.

    1. Dans la boîte de dialogue qui apparaît, sélectionnez le classement que vous souhaitez appliquer au document.

    2. Assurez-vous que la case à cocher Action située en regard du classement est bien activée.

    3. Cliquez sur Envoyer.

  8. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce document texte.

  9. Facultatif Pour placer le document texte dans un emplacement autre que celui affiché dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server.... Naviguez jusqu'au recueil dans lequel placer le document texte, puis cliquez sur son lien Sélectionner.

  10. Cliquez sur Ajouter.
