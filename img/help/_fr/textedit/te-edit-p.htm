Pour modifier un document texte :

  1. Cliquez sur l'icône Fonctions du document texte et choisissez Modifier.

  2. Sur la page Modifier, saisissez vos modifications dans le champ texte.

  3. Facultatif Pour rechercher des fautes d'orthographe, cliquez sur le bouton Vérifier l'orthographe. Le bouton Vérifier l'orthographe se trouve sous le champ Texte, à droite du bouton Ajouter une version.

     

    Note :
    La vérification de l'orthographe n'est pas disponible lorsque Java est désactivé. Pour en savoir plus, voir Utilisation des technologies d`accessibilité et du langage Java.

     

  4. Cliquez sur Ajouter une version pour appliquer vos modifications.
