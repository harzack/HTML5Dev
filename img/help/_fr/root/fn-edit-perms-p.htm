Pour modifier les autorisations d'un élément :

  1. Cliquez sur l'icône Fonctions de l'élément et choisissez Autorisations.

  2. Sur la page Autorisations, procédez de l'une des façons suivantes :

    1. Si vous souhaitez modifier les autorisations de l'un des groupes d'Accès par défaut, dans la zone Accès par défaut sur la gauche de la page, cliquez sur le nom du groupe dont vous souhaitez modifier les autorisations. Passez maintenant à étape 3.

    2. Si vous souhaitez assigner un accès à un utilisateur ou à un groupe, dans la zone Accès assigné, cliquez sur le nom d'un utilisateur ou cliquez sur le bouton Accorder l'accès....

      1. Dans la zone Accorder l'accès sur le côté droit de la page, naviguez jusqu'à l'utilisateur ou jusqu'au groupe dont vous souhaitez modifier les autorisations.

         

        Astuce :
        Pour plus d'informations sur la recherche d'utilisateurs ou de groupes, reportez-vous à la rubrique Utilisation des utilisateurs et des groupes.

         

      2. Sélectionnez la case à cocher Accorder l'accès de cet utilisateur ou de ce groupe.

      3. Cliquez sur Envoyer. Passez maintenant à étape 3.

  3. Sur le côté droit de la page, dans la boîte de dialogue Modifier les autorisations de l'<utilisateur/du groupe>, dans la zone Accès :

    1. Sélectionnez les cases à cocher situées en regard des autorisations que vous souhaitez assigner à cet utilisateur ou à ce groupe.

    2. Cliquez sur Mettre à jour.

  4. Facultatif Si vous souhaitez modifier les autorisations d'un autre utilisateur ou groupe, passez à étape 2. Si vous avez terminé, passez à l'étape suivante.

  5. Sur le côté gauche de la page, cliquez sur Terminé.

 

Note :
Si vous ne disposez pas de l'autorisation Modifier les autorisations pour un élément donné, vous ne pouvez pas afficher la page Autorisations de cet élément.
