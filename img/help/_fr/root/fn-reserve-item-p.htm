Pour réserver un élément :

  1. Cliquez sur l'icône Fonctions d'un élément et choisissez Réserver.

  2. Sur la page Réserver, dans le champ Réservé par, cliquez sur votre nom ou sur le nom d'un groupe.

  3. Facultatif Dans le champ Option, cliquez sur le lien Télécharger pour télécharger la version la plus récente de l'élément sur votre ordinateur.

  4. Cliquez sur Envoyer.
