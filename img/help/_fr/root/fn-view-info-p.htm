Pour afficher la page Propriétés d'un élément :

  1. Cliquez sur l'icône Fonctions de l'élément, choisissez Propriétés, puis le nom de l'un des onglets. La page Propriétés de l'onglet choisi s'affiche.

  2. Facultatif A partir la page Propriétés, il est possible de consulter les informations d'un onglet en le sélectionnant tout simplement.

 

Astuce :
Lorsque vous visualisez la page Propriétés d'une version, vous pouvez vous déplacer d'une version à l'autre à l'aide de la liste située en regard de l'icône Versions, .
