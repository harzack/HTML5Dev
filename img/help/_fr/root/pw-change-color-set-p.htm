Pour modifier les paramètres de couleurs :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Paramètres.

  2. Sur la page Mes paramètres : Général, cliquez sur l'onglet Couleurs.

  3. Sur la page Mes paramètres : Couleur, sélectionnez une couleur pour chacune des sections suivantes :

    • Ligne 1

    • Ligne 2

    1. En regard de chaque ligne, cliquez sur la zone vide située à droite du champ de texte, afin d'afficher les options de couleurs pour cette ligne.

    2. Cliquez sur la couleur que vous souhaitez attribuer à cette ligne.

       

      Note :
      Le champ de texte se met à jour avec la valeur correspondant à la couleur que vous avez sélectionnée. La zone auparavant vide, située à droite du champ de texte, affiche la couleur que vous avez sélectionnée. Le tableau situé à droite affiche les résultats de votre sélection.

       

  4. Cliquez sur Mettre à jour.
