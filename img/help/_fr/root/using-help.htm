L'aide en ligne complète du système explique comment utiliser chaque caractéristique et fonction de Content Server. Elle décrit également en détail leur fonctionnement et leur utilité. L'aide répond à deux buts principaux :

  • vous aider à accomplir rapidement et efficacement une tâche particulière ;

  • vous apprendre ce que vous pouvez réaliser avec Content Server, pour ensuite appliquer ces connaissances à votre travail.

Pratiquement toutes les pages de Content Server comprennent un menu Aide dans la barre de menu générale, représenté par l'icône ?. Lorsque vous sélectionnez l'icône Aide, les options du menu Aide s'affichent. Vous pouvez choisir de faire apparaître la table des matières de l'aide ou l'aide contextuelle spécifique à la page en cours d'affichage. L'aide s'affiche dans sa propre fenêtre de navigateur, vous permettant d'obtenir des informations sans avoir à interrompre votre travail. La section de droite de la fenêtre Aide contient le texte tandis que la section de gauche contient la table des matières du système d'aide.

Il existe trois façons différentes d'utiliser l'aide en ligne :

  • Parcourir la table des matières. Pour en savoir plus, voir Table des matières.

  • Utiliser l'aide contextuelle. Pour en savoir plus, voir Aide contextuelle.

  • Si l'option a été activée par votre administrateur, vous pouvez effectuer une recherche dans l'aide. Pour en savoir plus, voir Rechercher.

Table des matières

Si vous sélectionnez Table des matières dans le menu Aide, vous accédez à la page Aide. La table des matières du système d'aide s'affiche toujours dans la section de gauche de la fenêtre d'aide. La table des matières se met constamment à jour à mesure que vous parcourez le système d'aide. Dans la table des matières, le nom de la page d'aide que vous visualisez est mis en surbrillance et les autres rubriques d'aide se trouvant au même emplacement sont indiquées.

Aide contextuelle

Pour obtenir une aide contextuelle, vous devez sélectionner Pour cette page dans le menu Aide. Une page d'aide contextuelle est une page d'aide spécifiquement conçue pour la page Content Server en cours de visualisation. Si vous ne trouvez pas les informations dont vous avez besoin sur cette page, consultez la table des matières qui s'affiche dans le volet gauche de la fenêtre d'aide.

Rechercher

Si cette fonction a été activée par votre administrateur, vous pouvez rechercher dans l'aide à partir de toute barre de menu générale de Content Server. Un champ Recherche est disponible dans la barre de menu générale de Content Server. Cliquez sur la flèche pointant vers le bas du champ de recherche. Dans le panneau de recherche qui s'affiche, dans la liste Vues de recherche, sélectionnez Aide. Notez que le champ Recherche contient désormais le texte « Aide sur la recherche ». Saisissez votre requête de recherche dans le champ de texte Recherche, puis cliquez sur le bouton Effectuer la recherche ou appuyez sur la touche Entrée.

Dans la section Table des matières, votre administrateur peut prévoir un champ d'entrée de texte à utiliser pour rechercher dans l'aide en ligne les informations particulières dont vous avez besoin. Si ce champ existe, saisissez votre requête de recherche dans le champ Poser une question au système d'aide de Content Server, situé dans le volet gauche de la fenêtre d'aide, puis cliquez sur Aller.
