Pour garantir sa conformité aux normes d'accessibilité, Content Server a été testé avec de multiples technologies d'accessibilité, telles que les logiciels de lecture d'écran pour les malvoyants. Toutefois de nombreuses technologies d'accessibilité ne prennent pas efficacement en charge les applets Java. Les applets Java sont des applications écrites dans le langage de programmation Java. Si vous utilisez les technologies d'accessibilité avec Content Server, les problèmes suivants peuvent survenir :

  • Il se peut que le lecteur d'écran ne puisse pas lire le texte depuis une applet Java.

  • Si vous ne pouvez pas utiliser la souris ou un autre dispositif de pointage et que vous devez utiliser les raccourcis clavier, il se peut que vous ne puissiez pas utiliser les applets Java de manière efficace.

Les principales fonctionnalités de Content Server qui utilisent des applets Java sont l'éditeur de texte et le vérificateur d'orthographe. Il se peut que vous ne puissiez pas utiliser les applets Java dans d'autres situations. Pour en savoir plus, contactez l'administrateur.

Utilisation du mode de navigation améliorée par le clavier

Si votre administrateur vous y autorise, vous pouvez configurer vos paramètres personnels pour utiliser le mode d'accès au clavier étendu. Ce paramètre vous permet de naviguer dans Content Server à l'aide des raccourcis clavier et désactive les fonctionnalités qui utilisent Java. Si ce paramètre n'apparaît pas sur la page Paramètres généraux, votre système est peut-être défini pour l'utilisation du mode d'accès au clavier étendu pour tous les utilisateurs. Pour en savoir plus, voir Modification de vos paramètres personnels.

La navigation dans Content Server à l'aide des raccourcis clavier implique l'utilisation des touches TAB et ENTREE, comme décrit dans le tableau ci-dessous :

Touche Action
TAB Permet de se déplacer de manière séquentielle entre les boutons, les liens et les autres éléments de l'interface utilisateur de gauche à droite et de haut en bas. Dans un menu, permet de se déplacer de haut en bas. La combinaison MAJ + TAB change la direction.
Entrée Fonctionne comme un clic de souris pour activer les boutons et les liens et ouvrir les menus. Ouvre aussi les sous-menus, tels que le sous-menu Propriétés du menu Fonctions.
