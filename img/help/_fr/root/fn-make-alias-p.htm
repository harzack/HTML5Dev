Pour créer un raccourci :

  1. Dans le menu Ajouter un élément, cliquez sur Raccourci.

  2. Sur la page Ajouter : Raccourci, saisissez un nom pour le raccourci dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description de l'élément.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Dans le champ Elément, cliquez sur le bouton Parcourir Content Server. Recherchez l'élément pour lequel vous voulez créer un raccourci, puis cliquez sur le lien Sélectionner de cet élément.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à ce raccourci.

  6. Facultatif Dans le champ Créer dans, cliquez sur Parcourir Content Server pour sélectionner un autre recueil dans lequel créer ce raccourci.

  7. Cliquez sur le bouton Ajouter.

 

Astuce :
Vous pouvez aussi créer un raccourci en cliquant sur l'icône Fonctions d'un élément et en choisissant l'option Créer un raccourci.
