Pour ajouter un élément à l'aide de la fonction Glisser-déposer :

  1. Ouvrez le recueil dans lequel vous souhaitez ajouter un élément, dans le cas où la fonction glisser-déposer est disponible pour ce recueil, l'icône d'information Glisser-déposer des fichiers apparaît.

  2. Cliquez sur un élément de votre bureau, puis déplacez-le dans la zone Parcourir du recueil.

    Attendez que le fichier soit téléchargé et que la vue Parcourir du recueil s'actualise pour afficher l'élément.

  3. Facultatif Si vous avez ajouté un élément dans un recueil avec des attributs obligatoires, cliquez sur le lien Afficher les éléments incomplets et ajoutez les attributs obligatoires ou complétez les informations obligatoires.

 

Note :
Si vous avez ajouté des éléments comportant des informations obligatoires, vous devez compléter ces informations avant que d'autres utilisateurs ne puissent les consulter.
