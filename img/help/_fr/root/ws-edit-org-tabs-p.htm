Pour déplacer un élément vers un autre onglet :

  1. Dans la barre de menu générale, effectuez l'une des opérations suivantes :

    • Dans le menu Personnel, choisissez une page de rapport.

    • Dans le menu Projet, choisissez un rapport.

  2. Sur la page <nom_du_rapport>, cliquez sur le nom de l'onglet où se trouve l'élément à déplacer, puis cliquez sur le bouton Modifier/Organiser, .

  3. En regard de l'élément que vous souhaitez déplacer, dans sa liste associée, cliquez sur le nom de l'onglet où vous souhaiter placer l'élément.

  4. Cliquez sur Envoyer.
