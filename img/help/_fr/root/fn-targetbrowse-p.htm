Pour rechercher un élément :

  1. Lorsque vous ajoutez ou enregistrez des éléments dans Content Server, vous pouvez sélectionner le recueil dans lequel stocker l'élément. Pour ce faire, cliquez sur le bouton Parcourir Content Server.

  2. Dans la boîte de dialogue qui apparaît lorsque vous cliquez sur Parcourir Content Server, vous pouvez effectuer l'une des opérations suivantes :

    • Dans la zone de texte Recherche par nom, saisissez un nom pour rechercher les recueils dans le dossier actuel ou à un niveau inférieur dans Content Server. Les termes de recherche peuvent contenir des caractères génériques, et une recherche par nom est appliquée à toutes les langues pour les objets sous l'emplacement dans lequel la recherche est effectuée.

    • Cliquez sur le nom d'un recueil pour afficher son contenu.

    • Sélectionnez un recueil dans la liste Navigation ou le bandeau horizontal pour voir le contenu de ce recueil.

  3. Facultatif Lorsque vous vous trouvez dans un recueil, vous pouvez sélectionner la case à cocher Afficher uniquement les éléments pouvant être sélectionnés pour afficher uniquement les éléments que vous pouvez sélectionner.

  4. Facultatif Cliquez sur le lien <nom_colonne> pour trier les éléments dans Vue détaillée. Par exemple, vous pouvez cliquer sur le nom de colonne Type pour trier tous les éléments par type.

  5. Cliquez sur le lien Sélectionner dans la colonne Action en regard du recueil dans lequel vous souhaitez enregistrer ou ajouter l'élément.
