Si vous avez l'habitude d'utiliser un ordinateur, la plupart des fonctions et des fonctionnalités de base de Content Server vous seront familières. Toutefois, il est nécessaire de prendre connaissance de certaines actions et certains concepts spécifiques à Content Server afin d'utiliser le système de manière efficace.

Etant donné que votre entreprise peut personnaliser Content Server et définir des autorisations afin de restreindre ce que vous pouvez afficher et effectuer, il se peut que l'apparence et le fonctionnement de votre système diffèrent des caractéristiques par défaut décrites dans l'aide. D'autres facteurs peuvent également modifier l'apparence de l'interface Content Server.
