L'option de gestion de stockage flexible vous permet de stocker des documents dans plusieurs emplacements, appelés fournisseurs de stockage. Les fournisseurs de stockage sont des répertoires externes qui vous permettent de stocker des documents en dehors de Content Server.

Chaque instance Content Server utilisant le stockage flexible dispose d'un fournisseur de stockage par défaut. Celui-ci ne peut pas être modifié ou supprimé. Les fournisseurs de stockage sont associés à des règles qui sont évaluées du haut vers le bas de la liste afin de déterminer l'emplacement de stockage des nouveaux documents. Si le document ne répond à aucune des règles définies, il est stocké dans le fournisseur de stockage par défaut, qui apparaît toujours à la fin de la liste.

Dans Content Server, chaque document affiche le nom de son fournisseur de stockage dans l'onglet Versions de la page Propriétés du document. Si vous disposez des autorisations nécessaires, vous pouvez afficher la liste de toutes les règles du fournisseur de stockage. Cette liste contient la règle du fournisseur de stockage, ainsi que le nom et le type du fournisseur de stockage auquel la règle appartient. Seuls les administrateurs peuvent modifier ou supprimer les fournisseurs de stockage et les règles qui leur sont associées.
