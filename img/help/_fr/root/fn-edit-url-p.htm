Pour modifier une URL :

  1. Cliquez sur l'icône Fonctions d'une URL, choisissez Propriétés, puis sélectionnez l'un des onglets suivants :

    • Général

    • Infos spécifiques

    • Contrôle

    • Catégories

    • Références

  2. Sur la page Propriétés de l'URL, modifiez les paramètres le cas échéant.

  3. Cliquez sur Mettre à jour.
