Content Server est une application Web utilisée pour stocker, partager et répartir les informations. Elle fournit un environnement de travail collaboratif qui aide votre entreprise à gérer les documents et les informations sur les projets, à améliorer les processus commerciaux et à partager les informations. La collaboration devient partie intégrante de chaque processus commercial, par-delà même les barrières géographiques et structurelles. Content Server rassemble les employés de votre entreprise ainsi que vos partenaires et clients impliqués dans la vie de votre entreprise.

Content Server est conçu pour vous aider à :

  • organiser vos documents et autres éléments de travail dans un emplacement central contrôlé par des autorisations ;

  • partager, rechercher et suivre des informations ;

  • réviser des documents ou autres informations dans un environnement contrôlé par version ;

  • gérer et simplifier les processus commerciaux en créant et en utilisant des workflows ;

  • communiquer des nouvelles et autres informations aux autres membres de votre entreprise ;

  • gérer le travail des équipes de projet.

Comme Content Server est une application entièrement basée sur le Web, vous n'avez besoin que d'un navigateur Web pour l'utiliser. L'ensemble puissant de fonctionnalités intégrées qui composent Content Server vous donne un accès immédiat à toutes les informations stockées dans la base de données Content Server, le référentiel central des connaissances accumulées par votre entreprise.

Mise en route

Votre entreprise vous fournira probablement des formations et des instructions pour vous guider lors de vos premières utilisations de Content Server. Pour plus d'informations sur les fonctions Content Server de base, reportez-vous aux rubriques suivantes :

  • Utilisation de l'aide en ligne de Content Server, qui fournit une introduction générale à l'aide en ligne exhaustive.

  • Exploration de Content Server, qui présente l'environnement et les éléments de travail fournis par le système.

  • Aide du Guide de prise en main, qui décrit les principales fonctions du système et explique comment les utiliser.

  • Mon espace de travail, qui permet de vous familiariser avec votre espace de travail personnel appelé Mon espace de travail.
