L'un des principaux objectifs de Content Server est de fournir un environnement qui permet aux utilisateurs de trouver des informations, de les utiliser, de les enregistrer et de les partager. Dans Content Server, le stockage des éléments dans différents emplacements, ou recueils, est identique à l'organisation des fichiers dans différents répertoires sur votre ordinateur. Toutefois, la gestion des informations dans Content Server est plus complexe. En effet, il fournit des espaces de travail et des recueils spécialisés pour faciliter la gestion des informations selon leur type et leur utilité. De plus, Content Server fournit une structure d'autorisations qui vous permet de facilement contrôler qui a accès aux informations, selon l'emplacement où elles se trouvent.

Modes d'affichage

En fonction des paramètres de navigation sélectionnés par votre administrateur, vous pouvez sélectionner différentes vues de navigation pour modifier la présentation des éléments dans les espaces de travail, les dossiers et les documents composites. Les boutons du mode navigation s'affichent à droite de l'écran, sous la barre de menu générale, juste au-dessous de la recherche :

  • Afficher la vue détaillée affiche les icônes sous forme de liste et inclut des informations complémentaires, telles que le type d'élément, son nom, sa taille et sa date de modification. Cette vue présente également les boutons de sélection multiple. Les boutons de sélection multiple sont Copier, Déplacer et Supprimer. Les boutons de sélection multiple peuvent être utilisés pour un élément ou plusieurs éléments, en sélectionnant simplement la case à cocher de sélection multiple à gauche de chaque élément de la liste.

  • Afficher sous forme de grandes icônes, , affiche de grandes icônes sur quatre colonnes pour chaque élément.

  • Afficher sous forme de petites icônes, , affiche de petites icônes sur deux colonnes pour chaque élément.

 

Note :
Les boutons du mode d'exploration ne sont disponibles que lorsque votre administrateur active ce paramètre. Dans le cas contraire, les éléments apparaissent dans la vue détaillée par défaut.

 

Espaces de travail

Les espaces de travail sont comparables à des volumes qui divisent Content Server en différentes zones opérationnelles. Dans une entreprise, certaines informations sont destinées à être partagées publiquement, d'autres à être partagées au sein d'une équipe et enfin, certaines informations restent confidentielles. Les espaces de travail sont des entités fonctionnelles de base qui stockent et organisent les informations selon le public visé.

Il existe des éléments communs à tout type d'espace de travail. Ces éléments communs sont la barre de menu générale, l'en-tête et le pied de page. Les paramètres par défaut pour ces éléments communs sont répertoriés ci-dessous. Votre administrateur peut modifier les paramètres par défaut pour chacun de ces éléments communs.

La barre de menu générale permet d'accéder aux menus Entreprise, Personnel, Outils, Aide, qui s'affiche sous la forme du caractère ?, et Recherche En fonction de vos paramètres personnalisés, d'autres options peuvent être disponibles. Lorsque vous vous déplacez dans Content Server, d'autres éléments de menu s'affichent de façon dynamique sur la barre de menu générale. Par exemple, lorsque vous êtes dans un projet, le menu Projet apparaît à la suite de Outils.

L'en-tête s'affiche au-dessus de la barre de menu générale sous la forme d'une image. A l'exception des pages d'aide administrateur et utilisateur, cette image sert de lien vers l'espace de travail Entreprise.

Vous pouvez modifier le paramètre par défaut pour l'image de l'en-tête en sélectionnant Outils, puis Paramètres à partir de la barre de menu générale. Dans l'onglet Général, sélectionnez une autre option dans la liste Page de démarrage par défaut. L'image de l'en-tête est définie comme un lien vers la page de votre choix.

Le pied de page s'affiche en tant que : <id_utilisateur> <date> OpenText Content Server <version> <copyright>.

Les trois espaces de travail principaux de Content Server sont les suivants :

  1. Espace de travail Entreprise : auquel vous accédez en cliquant sur Entreprise dans la barre de menu générale, puis en cliquant sur Espace de travail. Par défaut, vous pouvez également accéder à l'Espace de travail Entreprise en cliquant sur l'image de l'en-tête située en haut à gauche de chaque page. Seules les images d'en-têtes situées au sommet des pages d'aide utilisateur ou administrateur ne sont pas des liens vers l'espace de travail Entreprise.

  2. Mon espace de travail : auquel vous accédez en cliquant sur Personnel dans la barre de menu générale, puis en cliquant sur Mon espace de travail.

  3. Espace de travail Projet : auquel vous accédez en cliquant sur Personnel dans la barre de menu générale, puis en cliquant sur Projets. Si vous avez créé ou ajouté un projet, le nom de ce projet s'affiche sous forme de lien sur cette page. La sélection d'un nom de projet vous conduira vers l'espace de travail de cet objet.

Chaque espace de travail possède ses propres outils et caractéristiques, mais le fonctionnement, le mode de navigation et leur apparence sont toujours les mêmes.

Espace de travail Entreprise

L'Espace de travail Entreprise est le forum public de Content Server. Il permet de stocker des informations et des éléments que tous les utilisateurs de l'entreprise peuvent afficher et utiliser. Bien que tous les utilisateurs puissent généralement voir les éléments de l'espace de travail Entreprise, seul un nombre restreint a l'autorisation d'ajouter, de supprimer ou de modifier ces éléments.

 

Note :
Les autorisations contrôlent votre accès à tous les éléments de Content Server. Pour de plus amples informations sur les autorisations, reportez-vous à la section Autorisations.

 

Mon espace de travail

Chaque utilisateur dispose d'une zone de stockage privée appelée Mon espace de travail. Pour y accéder, cliquez sur Personnel dans la barre de menu générale, puis cliquez sur Mon espace de travail. Vous bénéficiez de toutes les autorisations sur votre espace de travail personnel, qui s'affiche sous la forme <votre_id_utilisateur> Accueil. Pour de plus amples informations sur votre espace de travail personnel, voir Utilisation de Ma page d`accueil.

Espace de travail Projet

Chaque projet est unique, dans la mesure où il est ajouté à Content Server en tant qu'élément, mais chacun fonctionne comme un espace de travail distinct. Chaque projet ajouté à Content Server dispose d'un espace de travail propre, accessible uniquement aux utilisateurs auxquels le projet est attribué. Pour plus d'informations sur l'espace de travail Projet, reportez-vous à la rubrique Utilisation des projets.

 

Note :
Les projets ont également la particularité de posséder leur propre modèle d'autorisations. Pour en savoir plus, voir Autorisations de projet basées sur les rôles.

 

recueils

Les recueils sont des éléments qui permettent de systématiser l'organisation des informations stockées dans Content Server. Le dossier est le type de recueil le plus répandu, car il peut stocker tout type d'élément. Les autres recueils, tels que Discussions et Documents composites, peuvent uniquement stocker certains types d'éléments. Les caractéristiques et les fonctions varient d'un recueil à l'autre. Pour plus d'informations sur les recueils et sur les autres types d'éléments, reportez-vous à la section Eléments.

 

Note :
Les espaces de travail et les recueils sont parfois réunis sous le nom d'emplacements.

 

Navigation dans les recueils

Si votre administrateur a activé la fonction de pagination, vous pouvez limiter le nombre d'éléments affichés par page lorsque vous accédez à un recueil. Les flèches en haut et au bas de chaque page vous permettent de naviguer vers les pages précédentes et suivantes à l'intérieur du recueil. Vous pouvez spécifier le nombre d'éléments à afficher sur chaque page en sélectionnant un nombre dans la liste Afficher les éléments.

 

Note :
Les outils de pagination sont également disponibles sur les pages Configurer et Navigation cible pour les recueils. Pour plus d'informations sur les outils de pagination, reportez-vous à la rubrique Interface utilisateur.
