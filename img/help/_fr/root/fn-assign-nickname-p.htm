Pour attribuer un surnom à un élément :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Général.

  2. Dans l'onglet Général de la page Propriétés, cliquez sur le bouton Modifier dans la zone Surnom.

  3. Dans la fenêtre Modifier le surnom, procédez de l'une des façons suivantes :

    • Activez le bouton radio en regard de l'un des surnoms générés automatiquement.

    • Cliquez sur le bouton radio à côté du champ de texte, puis saisissez un surnom comprenant 40 caractères alphanumériques ou moins.

       

      Note :
      Seuls les caractères alphanumériques sont autorisés dans un surnom. Les caractères spéciaux ne sont pas autorisés.

       

  4. Cliquez sur Enregistrer.

 

Astuce :
Pour restaurer le surnom dans l'ID de l'objet par défaut, cliquez sur le bouton Restaurer les valeurs par défaut.
