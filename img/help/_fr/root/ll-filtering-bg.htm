Le filtrage fournit aux utilisateurs un moyen plus rapide de rechercher des éléments dans des recueils. Lorsque les utilisateurs sont en mode de navigation dans un recueil, ils peuvent rechercher des éléments en fonction d'un type d'élément ou en tapant des mots clés qui apparaissent dans le titre de l'élément recherché. Le processus de filtrage est pratique car il permet d'afficher dans un recueil des éléments moins nombreux, mais plus pertinents.

 

Note :
Le filtrage n'est pas disponible si la barre latérale Filtre de contenu est disponible et peut être affichée.

S'il n'existe aucune correspondance aux critères de filtrage, un message d'erreur s'affiche.

Le filtrage n'est pas disponible pour les documents composites ou dans le dossier de pièces jointes de workflow.
