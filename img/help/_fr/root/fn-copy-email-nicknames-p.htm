Pour envoyer par e-mail ou copier des URL de surnom :

  1. Cliquez sur l'icône Fonctions d'un élément, choisissez Propriétés, puis Général.

  2. Dans l'onglet Général de la page Propriétés de l'élément, dans le champ Liens abrégés, cliquez sur l'un des liens suivants dans la liste :

    • Envoyer le lien abrégé 'Propriétés' par e-mail

    • Envoyer le lien abrégé 'Ouvrir' par e-mail

    • Copier le lien abrégé 'Propriétés'

    • Copier le lien abrégé 'Ouvrir'

  3. Cliquez sur OK.

  4. Si vous sélectionnez l'une des options d'e-mail ci-dessus, un e-mail contenant le nom et le Lien abrégé de l'élément est généré. Votre signature électronique n'est pas incluse dans l'e-mail. Dans la fenêtre d'e-mail, saisissez un destinataire dans le champ A. Ajoutez toute autre information dans le corps de l'e-mail avant de l'envoyer.

    Si vous sélectionnez l'une des options de copie ci-dessus, une boîte de dialogue indiquant que le lien abrégé a été copié dans le presse-papiers s'affiche. Cliquez sur OK dans cette boîte de dialogue.
