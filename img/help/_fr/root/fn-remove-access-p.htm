Pour enlever l'accès propriétaire, l'accès groupe de propriétaires ou l'accès public à un élément :

  1. Cliquez sur l'icône Fonctions d'un élément et choisissez Autorisations.

  2. Sur le côté gauche page de la page Autorisations, dans la liste Accès par défaut, cliquez sur le lien Propriétaire, Groupe de propriétaires ou Accès public.

  3. Dans la section Modifier les autorisations sur le côté droit de la page, cliquez sur Enlever.

  4. Cliquez sur OK dans la boîte de dialogue de confirmation, puis, sur le côté gauche de la page, cliquez sur Terminé.
