Pour ajouter un dossier :

  1. Dans le menu Ajouter un élément, cliquez sur Dossier.

  2. Sur la page Ajouter : Dossier, saisissez un nom pour le dossier dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, entrez une description du dossier.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Facultatif Dans le champ Classements, effectuez une sélection dans la liste Classer. Choisissez le lien Sélectionner en regard du classement que vous souhaitez appliquer à ce dossier.

  5. Facultatif Dans le champ Catégories, sélectionnez Modifier pour sélectionner ou ajouter une catégorie à appliquer à ce dossier.

  6. Facultatif Dans le champ Créer dans, sélectionnez Parcourir Content Server pour sélectionner un autre recueil dans lequel créer ce dossier.

  7. Facultatif Cliquez sur le bouton Sélectionner l'icône pour choisir une autre icône pour le dossier. Le bouton Sélectionner l'icône s'affiche uniquement si votre administrateur a activé cette fonctionnalité.

  8. Cliquez sur Ajouter pour créer le dossier.

Pour plus d'informations sur cette procédure, consultez la section Ajout et configuration de dossiers.
