L'accès à Content Server est similaire à l'accès à un site Web classique. Chaque page contient des liens hypertexte qui vous permettent de rechercher les informations dont vous avez besoin. Votre administrateur Content Server est la personne ou le groupe de personnes responsable de la gestion et du fonctionnement de Content Server dans votre entreprise. Lorsque vous utilisez le système pour la première fois, l'administrateur vous fournit les informations suivantes :

  • L'URL permettant d'accéder au système. Vous pouvez créer un lien vers l'URL dans votre navigateur Web. Si vous utilisez Microsoft Internet Explorer, vous pouvez créer un favori. Si vous utilisez Netscape ou Firefox, vous pouvez créer un marque-page. La création d'un lien vous permettra de revenir facilement à Content Server.

  • Le nom d'utilisateur et le mot de passe pour votre système.

Votre compte utilisateur Content Server permet au système de :

  • définir les privilèges et les autorisations dont vous disposez au sein du système ;

  • accéder à votre espace personnel ;

  • vous indiquer les tâches qui vous sont attribuées ;

  • suivre les informations à des fins d'audit.

Vous devez respecter la distinction majuscule/minuscule pour les mots de passe. Votre mot de passe peut par ailleurs se composer de lettres, de chiffres ou de caractères spéciaux. Les barres obliques, /, ne sont pas acceptées. Votre administrateur peut définir des restrictions supplémentaires sur votre mot de passe, par exemple exiger que tous les mots de passe comprennent au moins un chiffre. Vous pouvez modifier votre mot de passe à tout moment. Si vous oubliez votre mot de passe, contactez votre administrateur.

Vous devez vous déconnecter de Content Server lorsque vous souhaitez utiliser un compte utilisateur différent ou par mesure de sécurité lorsque vous quittez votre poste. En fermant votre navigateur Web, vous fermez automatiquement votre session sur Content Server, mais le processus complet de déconnexion peut ne pas se produire car la fonction func=ll.DoLogout peut ne pas être écrite dans le journal.
