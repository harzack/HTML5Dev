Pour rechercher des éléments à l'aide du filtrage :

  1. Accédez au recueil hébergeant les éléments que vous recherchez. Dans la liste (Tous les types d'éléments), cliquez sur le type d'élément.

     

    Note :
    Seuls les types d'éléments hébergés dans le recueil apparaissent comme options de filtrage dans la liste (Tous les types d'éléments). Si vous souhaitez rechercher tous les types d'éléments, sélectionnez l'option (Tous les types d'éléments).

     

  2. Dans le champ Filtrer par nom, saisissez le mot-clé de filtrage des éléments, puis cliquez sur le bouton Filtrer par nom.

     

    Note :
    Si l'option de filtrage n'est pas disponible, la barre latérale Filtre de contenu peut être disponible. Pour en savoir plus, voir Utilisation du bandeau du filtre de contenu.

     
 

Astuce :
Si le filtre ne correspond à aucun élément dans un recueil, un message d'erreur s'affiche. Cliquez sur le lien Afficher tous les éléments pour revenir à la page précédente.
