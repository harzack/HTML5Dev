Un surnom est un mot ou une locution à attribuer à chaque élément Content Server. Si vous n'indiquez pas de surnom pour un élément, l'ID de l'objet de cet élément est automatiquement attribué comme surnom. Lorsque vous décidez d'attribuer un surnom à un élément, vous pouvez choisir parmi les trois surnoms proposés par Content Server, ou composer votre propre mot ou locution en caractères alphanumériques.

Les surnoms peuvent être utilisés comme termes de recherche. Lorsque vous recherchez un élément par son surnom, la liste Action dans la barre de recherche contient uniquement les éléments Ouvrir et Propriétés. Pour en savoir plus, voir Utilisation de la barre de recherche Content Server.

Lorsque vous ajoutez un élément à Content Server, deux URL courtes sont générées. Ces URL courtes, appelées Liens abrégés, établissent un lien vers l'élément.

Le premier Lien abrégé référence la fonction Propriétés de l'élément. Cliquez sur Lien abrégé Propriétés pour ouvrir l'onglet Général de la page Propriétés de l'élément.

Le second Lien abrégé référence la fonction Ouvrir de l'élément. Cliquez sur Lien abrégé Ouvrir pour ouvrir l'élément.

Vous pouvez copier un Lien abrégé vers le presse-papiers de votre ordinateur ou envoyer un Lien abrégé par e-mail.

Lorsque vous envoyez un Lien abrégé par e-mail, Content Server utilise le programme de messagerie de votre système pour générer un e-mail. L'objet de cet e-mail est le nom de l'élément et le corps de l'e-mail contient le Lien abrégé.

 

Note :
Aucun en-tête ou pied de page personnalisé n'est inclus dans cet e-mail. L'e-mail contient uniquement le nom de l'élément et le Lien abrégé.
