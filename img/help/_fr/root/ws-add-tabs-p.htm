Pour ajouter un onglet à une page de rapport personnelle ou de projet :

  1. Dans la barre de menu générale, effectuez l'une des opérations suivantes :

    • Dans le menu Personnel, choisissez une page de rapport.

    • Dans le menu Projet, cliquez sur Rapports.

  2. Sur la page <nom_du_rapport>, cliquez sur le bouton Configurer les onglets, .

  3. Cliquez sur le bouton Ajouter des onglets, .

  4. Dans le champ de texte <Numéro> d'onglet, saisissez un nom pour le nouvel onglet.

  5. Activez le bouton radio Valeur par défaut située en regard de l'onglet que vous souhaitez afficher en premier sur la page de rapport.

  6. Cliquez sur Envoyer.
