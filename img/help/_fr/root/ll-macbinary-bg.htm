Important :
Les informations suivantes ne concernent pas les ordinateurs Macintosh exécutant le système d'exploitation OSX. Si vous utilisez un ordinateur Macintosh exécutant le système d'exploitation OSX, il est inutile de sélectionner la case à cocher MacBinary (définir avant de choisir un fichier).

 

Si vous utilisez un ordinateur Macintosh qui n'exécute pas le système d'exploitation OSX, certaines pages de Content Server peuvent contenir une case à cocher MacBinary (définir avant de choisir un fichier). Cette case à cocher s'affiche sur les pages qui permettent de charger les fichiers depuis votre ordinateur. Cette option permet de garantir que les fichiers que vous chargez dans Content Server sont correctement transférés. Si vous essayez de charger un fichier Macintosh dans Content Server sans avoir au préalable activé cette case à cocher, le fichier risque de ne pas être chargé dans son intégralité.

MacBinary est un format standard qui a été développé pour faciliter le transfert de documents Macintosh. Il intègre les deux composants d'un fichier Macintosh standard, fourchette de ressources et fourchette de données, sous forme de progiciel unique. Un fichier MacBinary peut être stocké sur un système Content Server fonctionnant sur un ordinateur autre que Macintosh, mais ce type de fichier téléchargé depuis Content Server peut être utilisé uniquement sur un ordinateur Macintosh.
