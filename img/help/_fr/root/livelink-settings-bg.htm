Vos paramètres personnels vous permettent de définir certaines préférences pour votre système ou votre espace de travail. Par défaut, les pages Paramètres contiennent les onglets suivants :

  • Général, qui permet de personnaliser le mode d'affichage des informations. Par exemple :

    • Dans la liste Page de démarrage par défaut, vous pouvez sélectionner la page qui s'affiche lors de votre première connexion à Content Server.

    • Si des descriptions sont fournies dans le champ Description de l'onglet Général de la page Propriétés d'un élément, vous pouvez choisir d'afficher ces descriptions en mode de vue détaillée. Pour en savoir plus, voir Interface utilisateur et Affichage et modification des propriétés d'un élément.

    • Vous pouvez remplacer la liste Navigation par un bandeau horizontal représentant le chemin hiérarchique vers l'élément actuellement affiché. Pour en savoir plus, voir Interface utilisateur.

    • Vous pouvez définir le nombre de jours d'affichage des icônes Nouveau , et Modifié, , en regard des éléments nouveaux et modifiés dans Content Server. Par défaut, l'icône Nouveau s'affiche pendant deux jours après l'ajout d'un élément à Content Server. L'icône Modifié s'affiche pendant sept jours après la modification d'un élément dans Content Server.

    • Vous pouvez utiliser le Mode d'accès au clavier étendu, qui permet de naviguer dans le système en utilisant les raccourcis clavier plutôt que la souris ou tout autre dispositif de pointage. Pour en savoir plus, voir Utilisation des technologies d'accessibilité et du langage Java.

    • Vous pouvez choisir d'afficher la barre latérale dans la vue de navigation. Si vous choisissez d'afficher la barre latérale, vous pouvez également choisir le côté de la page où elle s'affichera. Par défaut, la barre latérale apparaît sur la gauche.

    • Si votre administrateur a activé d'autres langues pour Content Server, vous pouvez modifier la Langue des métadonnées de votre système. Les noms et les descriptions qui ont été saisis dans cette langue s'afficheront.

  • Couleur, qui permet de choisir les couleurs de votre interface.

  • Discussion, qui permet d'indiquer le mode d'affichage par défaut des discussions. Pour en savoir plus, voir Modification des paramètres de discussion.

  • Enterprise Process Services, qui permet de modifier les paramètres par défaut de Enterprise Process Services tels que la durée d'affichage, l'afficheur et l'option de déléguer des processus lorsque vous êtes hors du bureau. Pour en savoir plus, voir Modification Paramètres Enterprise Process Services.

  • Workflow, qui vous permet de gérer votre participation aux processus d'exécution automatique du workflow. Pour en savoir plus, voir Spécification des paramètres du workflow et des tâches assignées.

  • Mot de passe, qui vous permet de gérer votre mot de passe.

 

Note :
Il se peut que votre administrateur ne vous autorise pas à modifier certains de ces paramètres.
