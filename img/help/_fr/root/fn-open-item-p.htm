Pour ouvrir un élément :

  • Il existe deux façons d'ouvrir un élément :

    1. Cliquez sur le lien correspondant au nom de l'élément. Le <nom_élément> d'un élément est un lien hypertexte dont l'action par défaut est d'ouvrir l'élément.

    2. Cliquez sur l'icône Fonctions d'un élément et choisissez Ouvrir.
