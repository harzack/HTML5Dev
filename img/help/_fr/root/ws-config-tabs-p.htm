Pour définir un onglet par défaut :

  1. Dans la barre de menu générale, effectuez l'une des opérations suivantes :

    • Dans le menu Personnel, choisissez une page de rapport.

    • Dans le menu Projet, choisissez un rapport.

  2. Sur la page <nom_du_rapport>, cliquez sur le bouton Configurer les onglets, .

  3. Activez le bouton radio Valeur par défaut de l'onglet que vous souhaitez afficher en premier sur la page de rapport.

  4. Cliquez sur Envoyer.
