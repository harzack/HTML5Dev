Pour modifier un raccourci :

  1. Cliquez sur l'icône Fonctions d'un raccourci, choisissez Raccourci, puis Propriétés et enfin, sélectionnez l'un des onglets suivants :

    • Général

    • Infos spécifiques

    • Contrôle

    • Catégories

  2. Selon l'onglet Propriétés que vous choisissez, modifiez les paramètres comme vous le souhaitez.

  3. Facultatif Cliquez sur un autre onglet, puis modifiez les paramètres présents dans cet onglet.

  4. Cliquez sur Mettre à jour.
