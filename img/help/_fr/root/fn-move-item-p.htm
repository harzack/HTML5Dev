Pour déplacer un ou plusieurs éléments :

  1. Depuis la vue détaillée de chaque recueil, dans la colonne Sélectionner/Désélectionner située à gauche de la colonne Type, sélectionnez la case à cocher de chaque élément à déplacer.

  2. Cliquez sur le bouton Déplacer.

  3. Facultatif Sur la page Déplacer, dans la zone Eléments :

    1. Sous le titre Nom, saisissez un nom pour les éléments.

      Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier le nom dans les autres langues activées.

    2. Sous le titre Options, sélectionnez la case à cocher Ajouter une version.

       

      Note :
      La case à cocher Ajouter une version apparaît uniquement pour les éléments multi-version. Pour en savoir plus, voir Gestion des versions et des générations.

       

  4. Dans le champ Déplacer vers, cliquez sur Parcourir Content Server.... Accédez à l'emplacement dans lequel vous voulez déplacer les éléments, et cliquez sur le lien Sélectionner de cet emplacement.

  5. Dans la zone Catégories :

    1. Cliquez sur l'un des boutons radio suivants :

      • Original

      • Destination

      • Fusionné

    2. Facultatif Cliquez sur le bouton Modifier... pour modifier les catégories de l'élément.

  6. Cliquez sur Déplacer.

  7. Cliquez sur OK.

 

Note :
Pour plus d'informations sur cette procédure, consultez la section Utilisation des éléments généraux.

Pour plus d'informations sur la vue détaillée, reportez-vous à la rubrique Espaces de travail et recueils.

 
 

Astuce :
Vous pouvez également déplacer un seul élément en cliquant sur son icône Fonctions, puis en choisissant Déplacer.
