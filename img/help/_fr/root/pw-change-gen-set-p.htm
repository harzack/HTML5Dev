Pour modifier vos paramètres généraux :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Paramètres.

  2. Sur la page Mes paramètres : Général, cliquez sur l'onglet Général.

  3. Facultatif Pour remplacer la page qui s'affiche lorsque vous vous connectez à Content Server, sélectionnez la page souhaitée dans la liste Page de démarrage par défaut.

    Le contenu initial de ce champ a été sélectionné par votre administrateur.

  4. Facultatif Si vous souhaitez afficher les descriptions des éléments, sélectionnez la case à cocher Présenter les descriptions d'élément dans l'affichage détaillé.

  5. Facultatif Pour spécifier le style de navigation, dans la liste Style de navigation, cliquez sur bandeau horizontal ou sur liste déroulante.

  6. Facultatif Pour spécifier la durée d'affichage, en jours, de l'icône Nouveau(elle) en regard des éléments, cliquez sur une valeur dans la liste Durée de l'indicateur 'Nouveau(elle)'. Sélectionnez zéro pour désactiver l'icône Nouveau(elle).

  7. Facultatif Pour spécifier la durée d'affichage, en jours, de l'icône Modifié(e) en regard des éléments, cliquez sur une valeur dans la liste Durée de l'indicateur 'Modifié(e)'. Sélectionnez zéro pour désactiver l'icône Modifié(e).

  8. Facultatif Sélectionnez la case à cocher Mode d'accès au clavier étendu pour activer la navigation à l'aide du clavier.

  9. Facultatif Pour indiquer le côté de la page où afficher la barre latérale de Content Server, dans la liste Emplacement de la barre latérale, sélectionnez Gauche ou Droite.

  10. Facultatif Sélectionnez la case à cocher Afficher la barre latérale dans la navigation pour que la barre latérale de Content Server s'affiche. Cette case est cochée par défaut.

  11. Facultatif Pour modifier la langue des métadonnées, sélectionnez une langue dans la liste Langue des métadonnées.

  12. Cliquez sur Mettre à jour.

     

    Note :
    Pour plus d'informations sur cette procédure, consultez la section Modification de vos paramètres personnels.
