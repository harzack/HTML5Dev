Pour modifier les paramètres de présentation d'un espace de travail ou d'un recueil :

  1. Cliquez sur l'icône Fonctions d'un espace de travail ou d'un recueil, choisissez Propriétés, puis Présentation.

  2. Facultatif Dans le champ N° de sous-éléments mis en valeur, cliquez sur un bouton radio pour spécifier le nombre de sous-éléments mis en valeur apparaissant sous forme de lien sous un élément mis en valeur.

  3. Facultatif Dans la zone 1ère bannière ou 2ème bannière, effectuez les étapes suivantes :

    1. Dans le champ Image, pour ajouter ou modifier une bannière d'images, cliquez sur le bouton Parcourir Content Server. Naviguez jusqu'à une image, puis cliquez sur le lien Sélectionner de cette image.

    2. Dans le champ Texte ALT, pour ajouter une description pour l'image, saisissez la description.

    3. Dans le champ URL cible, pour lier l'image à un site Web, saisissez l'URL.

  4. Cliquez sur Mettre à jour.

 

Note :
L'onglet Présentation de la page Propriétés d'un projet est unique. Pour en savoir plus, voir Personnalisation des espaces de travail Projet.
