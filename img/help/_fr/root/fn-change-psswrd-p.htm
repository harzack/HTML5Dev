Pour modifier vos paramètres de mot de passe :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Paramètres.

  2. Sur la page Mes paramètres : Général, cliquez sur l'onglet Mot de passe.

  3. Sur la page Gérer le mot de passe, saisissez votre mot de passe actuel dans le champ Mot de passe actuel.

  4. Saisissez le nouveau mot de passe dans le champ Nouveau mot de passe, puis saisissez-le une nouvelle fois dans le champ Confirmation.

  5. Cliquez sur Mettre à jour.

Pour plus d'informations sur cette procédure, consultez la section Modification de vos paramètres personnels.
