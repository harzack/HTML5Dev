Pour ajouter une URL :

  1. Dans le menu Ajouter un élément, cliquez sur URL

  2. Sur la page Ajouter : URL, saisissez un nom pour l'URL dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description de l'URL.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Dans le champ URL, saisissez l'URL.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à cette URL.

  6. Facultatif Dans le champ Créer dans, cliquez sur Parcourir Content Server pour sélectionner un autre recueil dans lequel créer cette URL.

  7. Cliquez sur Ajouter.
