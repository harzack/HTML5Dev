Pour renommer un élément :

  • Il existe deux façons de renommer un élément :

    1. Cliquez sur l'icône Fonctions d'un élément et choisissez Renommer.

      1. Sur la page Renommer de l'élément, dans le champ Nom, saisissez un nom pour l'élément.

      2. Cliquez sur Mettre à jour.

    2. Cliquez sur l'icône Fonctions de l'élément, sélectionnez Propriétés, puis Général.

      1. Dans le champ Nom, saisissez un nouveau nom.

      2. Cliquez sur Mettre à jour.
