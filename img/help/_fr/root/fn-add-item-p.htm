Pour ajouter un élément :

  1. Dans le menu Ajouter un élément, sélectionnez un type d'élément.

  2. Sur la page Ajouter d'un élément, saisissez le nom de l'élément dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Facultatif Dans le champ Description, saisissez une description de l'élément.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur l'icône Modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Facultatif Dans le champ Classements, effectuez une sélection dans la liste Classer. Choisissez le lien Sélectionner en regard du classement que vous souhaitez appliquer à cet élément.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à cet élément.

  6. Facultatif Dans le champ Créer dans, cliquez sur Parcourir Content Server pour sélectionner un autre recueil dans lequel créer cet élément.

  7. Cliquez sur Ajouter.

 

Note :
Pour plus d'informations sur l'ajout d'un document, reportez-vous à la rubrique Ajout et modification de documents.
