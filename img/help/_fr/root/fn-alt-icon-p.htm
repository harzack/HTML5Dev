Pour modifier l'icône d'un dossier :

Cette option est disponible uniquement si votre administrateur a activé cette option.

  1. Cliquez sur l'icône Fonctions du dossier, sélectionnez Propriétés, puis Général.

  2. Dans l'onglet Général de la page Propriétés du dossier, cliquez sur le bouton Sélectionner l'icône du champ Type.

  3. Sur la page Sélectionner l'icône, cliquez sur une icône.

  4. Sur la page Propriétés, cliquez sur Mettre à jour.
