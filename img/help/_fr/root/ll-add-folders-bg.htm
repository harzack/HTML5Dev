Les dossiers vous permettent d'organiser votre travail dans Content Server de la même façon que sur votre ordinateur. Le dossier est le type de recueil le plus polyvalent de Content Server, car il peut contenir la plupart des autres éléments, y compris des projets et d'autres dossiers. Un dossier situé dans un autre dossier est appelé sous-dossier.

Ajout d'un dossier

Le menu Ajouter un élément et le bouton Ajouter un dossier s'affichent dans le coin supérieur droit de chaque page d'espace de travail ou de recueil lorsque vous êtes autorisé à y ajouter un dossier ou un élément.

Lorsque vous ajoutez un dossier à Content Server, vous pouvez le décrire, modifier ses catégories et attributs et l'ajouter à un recueil différent. Pour plus d'informations sur l'ajout d'éléments, reportez-vous à la section Utilisation des éléments généraux.

Configuration d'un dossier

Si vous possédez les autorisations appropriées, vous pouvez configurer l'apparence d'un dossier. Pour en savoir plus, voir Personnalisation des espaces de travail et des recueils.

Lors de l'ajout d'un dossier, si vous y êtes autorisé par votre administrateur, une option de configuration unique vous permet de choisir un éventail de différentes icônes pour le dossier. Cela peut être utile si vous souhaitez reconnaître d'un coup d'œil le dossier ou indiquer aux utilisateurs le type d'informations contenues dans le dossier. Vous pouvez définir une icône différente lors de l'ajout d'un dossier, ou vous pouvez définir une icône différente pour un dossier existant, à condition de disposer des autorisations appropriées.

Autorisations

Par défaut, un élément que vous ajoutez à un dossier hérite des autorisations de ce dossier. Par exemple, si un groupe possède l'autorisation Voir le contenu sur un dossier, les membres de ce groupe auront l'autorisation de voir tous les éléments que vous ajoutez à ce dossier. Toutefois, il est possible de modifier les autorisations d'un élément spécifique d'un dossier. Pour en savoir plus, voir Gestion des autorisations.
