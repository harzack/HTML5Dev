Pour supprimer un onglet :

  1. Dans la barre de menu générale, effectuez l'une des opérations suivantes :

    • Dans le menu Personnel, choisissez une page de rapport.

    • Dans le menu Projet, choisissez un rapport.

  2. Sur la page <nom_du_rapport>, cliquez sur le bouton Configurer les onglets, .

  3. S'il est possible de retirer l'onglet de la page de rapport, vous pouvez sélectionner la case à cocher Enlever en regard de cet onglet.

  4. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Gestion des onglets de la page de rapports.
