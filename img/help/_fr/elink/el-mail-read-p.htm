Pour lire un e-mail :

  1. Sélectionnez Ma boîte de réception dans le menu Personnel.

  2. Sur la page Ma boîte de réception, cliquez sur le lien Nom de l'e-mail que vous souhaitez lire.

  3. Facultatif Pour afficher ou télécharger la pièce jointe d'un e-mail, cliquez sur le lien Nom de la pièce jointe, puis cliquez sur le bouton Ouvrir ou Enregistrer dans la boîte de dialogue Téléchargement de fichier.

 

Note :
Si vous utilisez également un client de bureau pour lire l'e-mail, les messages ne s'afficheront pas dans Ma boîte de réception si votre client de bureau est configuré pour supprimer les messages du serveur de messagerie après téléchargement.
