Pour... Effectuez les opérations suivantes...
Activer eLink pour un élément
1.

Cliquez sur l'icône Fonctions de l'élément, puis sélectionnez Activer l'e-mail.

2.

Sur la page Activer eLink, saisissez l'adresse e-mail de l'élément dans la zone Raccourci de l'e-mail.

3.

Le cas échéant, sélectionnez le type de stockage du contenu dans la zone Contenu stocké.

4.

Cliquez sur le bouton radio Activer.

Désactiver eLink pour un élément Cliquez sur l'icône Fonctions de l'élément, puis sélectionnez Activer l'e-mail. Sur la page Désactiver eLink, cliquez sur Désactiver.
