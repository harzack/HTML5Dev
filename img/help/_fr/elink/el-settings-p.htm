Pour spécifier les paramètres eLink :

  1. Sélectionnez eLink dans le menu Personnel.

  2. Sur votre page eLink personnelle, cliquez sur le bouton Modifier les paramètres.

    Pour... Effectuez les opérations suivantes...
    Spécifier les paramètres d'interruption Dans la section Interruption, choisissez la date (mois, jour et année) à laquelle vous souhaitez à nouveau recevoir les messages eLink.
    Spécifier le format du contenu de l'e-mail Dans la section Contenu de l'e-mail, sélectionnez Texte ou HTML dans la liste déroulante.
    Recevoir par e-mail les messages et les réponses que vous publiez Cochez la case M'envoyer par e-mail les messages / réponses que j'ai publiés.
    Recevoir une notification eLink pour les événements Dans la section Evénements globaux, cochez la case associée à chaque événement pour lequel vous souhaitez recevoir une notification eLink.

  3. Cliquez sur le bouton Envoyer.
