Pour configurer la boîte de réception eLink :

  1. Sélectionnez Ma boîte de réception dans le menu Personnel.

  2. Sur la page Ma boîte de réception, cliquez sur le bouton Configurer les paramètres.

  3. Sur la page Configurer les paramètres de messagerie, procédez comme suit :

    • Saisissez le nom DNS de votre serveur de messagerie dans le champ Serveur POP (par exemple, mail.co.entreprise.fr).

    • Saisissez le numéro de port de votre serveur de messagerie dans le champ Port POP (par exemple, 110).

    • Saisissez le nom d'utilisateur que vous utilisez pour vous connecter à votre serveur de messagerie dans le champ Nom de l'utilisateur.

    • Saisissez le mot de passe que vous utilisez pour vous connecter à votre serveur de messagerie dans le champ Mot de passe.

    • Pour recevoir automatiquement une copie des messages envoyés, cochez la case Envoyer une copie des messages envoyés vers Ma boîte de réception.

  4. Cliquez sur le bouton Envoyer.
