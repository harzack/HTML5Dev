Pour... Effectuez les opérations suivantes...
Vous abonner à une discussion activée pour eLink Sélectionnez eLink dans le menu Personnel, puis cliquez sur Ajouter un abonnement. Dans la fenêtre Ajouter un abonnement, naviguez jusqu'à la discussion activée pour eLink, puis cliquez sur le lien Sélectionner. Sur la page Abonnement eLink, cliquez sur le bouton S'abonner.
Vous vous désabonner d'une discussion activée pour eLink Sélectionnez eLink dans le menu Personnel, puis cliquez sur Modifier/Organiser. Cochez la case Se désabonner correspondant à la discussion dont vous souhaitez vous désabonner. Cliquez sur le bouton Envoyer.
 

Astuce :
Vous pouvez également vous désabonner en répondant à un message eLink et en indiquant Unsubscribe dans l'objet du message.
