Pour gérer un e-mail :

  1. Sélectionnez Ma boîte de réception dans le menu Personnel.

  2. Sur la page Ma boîte de réception, cliquez sur le lien Nom de l'e-mail que vous souhaitez gérer.

  3. Procédez de l'une des manières suivantes :

    • Cliquez sur le bouton Répondre à l'expéditeur , vérifiez que l'adresse électronique est correcte, puis cliquez sur Envoyer.

    • Cliquez sur le bouton Répondre à tous , vérifiez que les adresses e-mails sont correctes, puis cliquez sur Envoyer.

    • Cliquez sur le bouton Transférer , saisissez ou sélectionnez l'adresse de chaque utilisateur auquel vous souhaitez transférer ce message, puis cliquez sur Envoyer.

    • Cliquez sur le bouton Supprimer le message , puis sur OK dans la boîte de dialogue de confirmation.

 

Astuce :
Pour recevoir une copie d'un message envoyé, sélectionnez la case à cocher Envoyer une copie des messages envoyés vers Ma boîte de réception.
