Pour rédiger un message électronique :

  1. Sélectionnez Ma boîte de réception dans le menu Personnel.

  2. Sur la page Ma boîte de réception, cliquez sur le bouton Nouveau message .

  3. Sur la page Envoyer ce document par e-mail, saisissez l'adresse e-mail de chaque destinataire dans le champ A :, en séparant les adresses par une virgule ou un point virgule.

  4. Facultatif Dans la zone Cc :, saisissez l'adresse e-mail de chaque destinataire auquel vous souhaitez envoyer une copie de votre message. . Séparez les adresses avec une virgule ou un point virgule.

  5. Pour recevoir automatiquement une copie des messages envoyés, cochez la case Envoyer une copie des messages envoyés vers Ma boîte de réception.

  6. Dans le champ Objet, saisissez l'objet de votre e-mail.

  7. Saisissez le texte du message dans le champ Message.

  8. Cliquez sur le bouton Envoyer.

 

Astuce :
Vous pouvez également choisir des utilisateurs en cliquant sur l'icône Choisir l'utilisateur et en sélectionnant des utilisateurs sur la page Ajouter des destinataires.
