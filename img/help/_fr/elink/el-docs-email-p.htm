Pour envoyer un document par e-mail :

  1. Dans Content Server, naviguez jusqu'au document que vous souhaitez envoyer par e-mail.

  2. Cliquez sur l'icône Fonctions du document, puis sélectionnez E-mail.

  3. Sur la page Envoyer ce document par e-mail, saisissez l'adresse de chaque destinataire dans le champ A, en séparant les adresses par une virgule ou un point-virgule.

  4. Facultatif Dans la zone Cc :, saisissez l'adresse e-mail de chaque destinataire auquel vous souhaitez envoyer une copie de votre message . Séparez les adresses avec une virgule ou un point virgule.

  5. Saisissez la description du message dans le champ Objet du message.

  6. Cliquez sur le bouton radio Pièce jointe ou Lien pour définir la forme sous laquelle votre document va être envoyé.

  7. Saisissez votre message dans le champ Message.

  8. Cliquez sur le bouton Envoyer.

Astuce !Vous pouvez également choisir des utilisateurs en cliquant sur l'icône Choisir l'utilisateur et en sélectionnant des utilisateurs sur la page Ajouter des destinataires.
