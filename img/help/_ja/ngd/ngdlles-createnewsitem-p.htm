ニュース アイテムを作成するには

  1. ニュース アイテムを作成するチャンネルを右クリックして、[ニュースの作成] をクリックします。

  2. [追加: ニュース] ページの [名前] ボックスに、ニュース アイテムの名前を入力します。

  3. 該当する月、日、年、および時刻のリストから、[配信開始日時] および [有効期限日時] をクリックします。

  4. [見出し] ボックスにニュース アイテムの見出しを入力します。

     

    注意:
    [見出し] ボックスを空白のままにした場合、[名前] ボックスのテキストが見出しとして使用されます。

     

  5. [本文] ボックスにニュース アイテムの内容を入力します。

  6. 次の表に、ニュース アイテムを定義するために追加のタスクを実行する方法について説明します。

    宛先 操作
    ニュース アイテムへの画像の添付 [画像] 領域の [Content Server の参照] をクリックし、ニュース アイテムに添付する画像を指定し、対応する [選択] リンクをクリックします。
    画像の説明の追加 [ALT テキスト] ボックスに説明を入力します。
    ニュース アイテムへの添付の追加 [Content Server添付の追加] 領域で [Content Server の参照] をクリックし、ニュース アイテムに添付する画像を指定し、対応する [選択] リンクをクリックします。
    コンピューターからニュース アイテムへのファイルの添付 [デスクトップ添付の追加] 領域で [参照] をクリックし、アイテムを指定してから、[開く] をクリックします。

  7. [追加] をクリックします。

 

ヒント:
アイテムを右クリックして [ニュースの作成] を選択する方法でも、ニュースを追加できます。デフォルトでは、アイテムはニュース アイテムに添付として追加されます。
