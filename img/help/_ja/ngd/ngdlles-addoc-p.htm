ドキュメントを追加するには

  1. ドキュメントの追加先を右クリックして、[アイテムの追加] をクリックし、[ドキュメント] をクリックします。

     

    ヒント:
    [アイテムの追加] メニューから [ドキュメント] をクリックすることもできます。

     

  2. [ドキュメントの追加] ページの [ドキュメント] 領域で、[参照] をクリックして追加するファイルに移動し、[開く] をクリックします。

  3. [名前] ボックスでドキュメントの名前を入力し、必要に応じてドキュメントの説明を入力します。.

  4. [バージョン管理] 領域で、[標準 - リニア バージョン作成] または [高度 - メジャー/マイナー バージョン作成] をクリックして、バージョン作成のタイプを選択します。

    必要に応じて、クラシフィケーションとカテゴリの値を指定します。

  5. [作成場所] 領域で [Content Server の参照] をクリックし、ドキュメントを保存する場所を指定します。

  6. [追加] をクリックします。

 

注意:
目的の場所に同じ名前のドキュメントが既に存在していた場合、[Enterprise Connect] ダイアログ ボックスが開きます。以下のいずれかのオプションを選択します。

  • 既存のドキュメントのバージョンとして、新しいドキュメントを追加する場合は [はい]。

  • 重複するすべてのドキュメントを Content Server に追加する場合は [すべてはい]。

  • ドキュメントを Content Server に追加しない場合は [いいえ]。

  • 重複するドキュメントを Content Server に追加しない場合は [すべていいえ]。新しいドキュメントのみが追加されます。

  • アクションを停止するには、[キャンセル]。
