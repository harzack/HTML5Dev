アンケートを編集するには

  1. 編集するアンケートを右クリックして [編集] をクリックします。

  2. [投票コメントを参照] ページの [名前] ボックスにアンケートの名前を入力します。

  3. [配信開始日] および [有効期限日] リストで、アンケートの開始日と終了日をクリックします。

  4. [アンケート] 領域で、次の操作を実行します。

    • [質問] ボックスに質問を入力します。

    • [選択肢] ボックスに、質問に対する回答の選択肢を入力します。

    • [選択肢の形式] 領域で、各質問に対して次のいずれかのオプションを選択し、投票方法を決定します。

      • [ラジオ ボタン]

      • [チェックボックス]

  5. 次の表に、アンケートを定義するその他の方法を示します。

    目的 操作
    投票者がアンケートに入力する際の指示を追加 [指示] ボックスに指示を入力します。
    質問と回答の選択肢の追加または削除 [アンケート] 領域で、[追加] ボタンまたは [削除] ボタンをクリックします。
    ( / ).
    投票者に結果の参照とコメントの追加を許可 [オプション] 領域で、[投票する前に結果の表示を許可する] チェック ボックスまたは [匿名のコメントを許可する] チェック ボックスをオンにします。
    アンケートに画像を追加 [画像] 領域で、[Content Server の参照] をクリックして画像を指定し、対応する [選択] リンクをクリックします。
    画像に代替テキストを追加 [ALT テキスト] ボックスに代替テキストを入力します。

  6. [送信] をクリックします。
