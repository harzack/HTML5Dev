コンテンツ取得オプションを設定するには

  1. Enterprise Connect ツリーで Content Server コンテナー、[パーソナル]、[マイセッティング] の順にクリックします。

  2. [メタデータの取り込みの設定] 領域にある [コンテンツの追加時にメタデータの入力を促す] リストで次のいずれかの手順を行って、Content Server に追加するアイテムのためにメタデータを入力するように、プロンプトを表示させるタイミングを指定します。

    • 目的の Content Server フォルダーでメタデータが必須とマークされている場合に限り、メタデータの入力を指示するプロンプトを表示させる場合は、[必要時のみ] をクリックします。これがデフォルトの設定です。

    • Content Server にアイテムを追加するたびにメタデータの入力を指示するプロンプトを表示させる場合は、[常に表示] をクリックします。

  3. [送信] をクリックして変更を適用します。

     

    ヒント:
    電子メールの命名規則に従うために、最近保存した値に戻すには、[リセット] をクリックします。管理者が提供した設定に戻すには、[デフォルトを使用] をクリックします。
