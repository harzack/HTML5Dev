管理者がこの機能を有効にした場合、ユーザーは Content Server Web インターフェイスで特定のコンテンツ タイプを操作するときに Enterprise Connect にアクセスすることができます。この機能が使用可能である場合、ユーザーはサポートされているアイテムで [ファンクション] メニューにアクセスし、[Enterprise Connect ここから] を選択することで、Enterprise Connect の新しいインスタンスを開いて、選択したアイテムを表示することができます。.この機能を使用できるサブタイプは、管理者が決定します。既定では、このアクションには、[Enterprise Connect ここから] というラベルが付いていますが、管理者はこの機能に対してカスタムのラベルを作成できます。この機能の詳細については、管理者にお問い合わせください。

 

注意:
この機能は、Internet Explorer の Content Server Web インターフェイス インスタンスでのみ使用可能になります。
