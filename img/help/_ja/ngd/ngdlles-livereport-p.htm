LiveReport を作成するには

  1. LiveReport を作成する場所を右クリックして、[アイテムの追加]、[LiveReport] の順にクリックします。

  2. [追加: LiveReport] ページの [名前] ボックスにレポートの名前を入力します。

  3. [レコード件数制限] ボックスに表示可能な結果の最大数を入力します。

  4. [SQL] ボックスにレポートの SQL ステートメントを入力します。

  5. [パラメーター] リストで、SQL ステートメントに関連するパラメーターの種類をクリックします。

  6. [レポート フォーマット] リストで LiveReport のフォーマットをクリックします。

  7. 次の表に、LiveReport を定義するために追加のタスクを実行する方法について説明します。

    目的 操作
    [LiveReport の結果] ページに表示されるタイトルの追加 [タイトル] ボックスに名前を入力します。
    ユーザーがサブレポートとして LiveReport を使用する際に、新しいパラメーター値の入力を要求 [プロンプトを表示] チェック ボックスをオンにします。
    入力タイプの定義 [インプット] 領域の [タイプ] リストでインプット タイプをクリックしてから、ユーザーに入力を要求する時に表示するテキストを [プロンプト] ボックスに入力します。
    SQL ステートメントで Content Server データベースのコンテンツの変更を許可 [データベースの変更を許可] チェック ボックスをオンにします。
    この LiveReport へのサブレポートの関連付け [サブレポートの指定] 領域で [参照] をクリックし、サブレポートとして表示する LiveReport へ移動してから、[選択] リンクをクリックします。
    サブレポートへのパラメーターの割り当て それぞれの [サブレポート パラメーター #] ボックスに、既存のパラメーターの名前を入力します。
    [LiveReport の結果] ページに表示するデータベースの情報の指定 [フィールド] ボックスにデータベース カラム名を入力し、対応するカラムの表示名を [カラム タイトル] ボックスに入力します。
    選択した LiveReport フォーマットの外観の設定 LiveReport のフォーマットを選択して [設定] をクリックし、[レポート フォーマットの設定] ウィンドウで変更を行ってから、[選択] をクリックします。

  8. [追加] をクリックします。

 

ヒント:
LiveReport を右クリックして [編集] をクリックすることで、既存の LiveReport の設定を変更できます。
