閲覧ウィンドウ でアイテムを表示すると、ウィンドウの一番下に一連のタブが表示されます。これらのタブは、選択されたアイテムで使用できるプロパティに対応しています。アイテムを右クリックして [プロパティ] を選択し、適切なプロパティを選択することで、別のウインドウにプロパティを開くこともできます。権限がある場合は、[プロパティ] タブの一部のボックスを編集することもできます。これらのボックスを編集する方法の詳細については、 アイテムのプロパティの表示と変更を参照してください。

選択したアイテムのタイプに応じて、次のタブのいくつかが表示されます。

  • [一般]: [名前]、[タイプ]、[作成日時]、[作成者]、[表示]、[説明]、[サイズ]、[更新日時]、および [所有権] 情報が表示されます。

     

    注意:

    • フォルダー、ドキュメント、タスク リストなど、非表示にされたアイテムでも、設定に応じて、Enterprise Connect ツリーで完全にアクセスできることがあります。Outlook とエクスプローラーの両方の リスト ぺイン では、非表示アイテムを表示することができ、検索結果にも非表示アイテムが含まれます。

    • Enterprise Connect は Content Server の [おすすめ] 機能をサポートしません。

     

  • [特定]: アイテムのタイプに応じた特定の情報が表示されます。例えば、コンパウンド ドキュメントのリリース日や、ドキュメントの MIME タイプなどです。

  • [監査]: アイテムに実行されたアクションのイベント、日付、ユーザー名が表示されます。

  • [カテゴリ]: アイテムのカテゴリと属性の設定が表示されます。カテゴリ情報はこのページから変更できます。

  • [クラシフィケーション]: 選択したアイテムのクラシフィケーションが表示されます。このページでは、使用可能なクラシフィケーションを閲覧できます。

  • [カラム]: カスタム列の表示オプションが表示されます。

     

    注意:
    実行した変更を適用するには、Enterprise Connect クライアントを更新する必要があります。

     

  • [リンク]: コンパウンド ドキュメントのショートカットに対してのみ表示されます。タブには、ショートカットの参照先コンパウンド ドキュメントのバージョン タイプ ([現在のリリース]、[現在のリビジョン]、[作業中]) が表示されます。

  • [プレゼンテーション]: ワークスペースおよびフォルダーのプレゼンテーション オプションが表示されます。このタブから、グラフィック画像、ニュース ティッカー、サブカタログ アイテムを表示することができます。

  • [リファレンス]: オリジナル アイテムから作成されたショートカットまたはジェネレーション アイテムが表示されます。また、アイテム タイプ、名前、作成日時、ユーザー名も表示されます。

  • [リリース]: 対応するコンパウンド ドキュメントのすべてのリリースが表示されます。また、リリース アイコン、リリースの名前と番号、作成された日時、作成したユーザー名も表示されます。

  • [統計]: ディスカッション アイテムに投稿されたすべてのトピックと返信が表示されます。また、ユーザー名、投稿されたトピック数、投稿された返信数、総投稿数、最近の投稿日も表示されます。

  • [バージョン]: アイテムのバージョンが表示されます。また、バージョン番号、名前、サイズ、作成日時、作成したユーザーのユーザー名も表示されます。

  • [表示]: 選択したアイテムのプレビューが表示されます。

  • [相互参照]: 他の Content Server オブジェクトへの相互参照が表示されます。.
