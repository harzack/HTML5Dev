フォルダーまたは電子メール フォルダーを追加するには

  1. フォルダーを追加するフォルダーまたはワークスペースを右クリックし、[アイテムの追加] をクリックしてから、次のいずれかの操作を実行します。

    • [フォルダー]

    • [電子メール フォルダー]

  2. [フォルダーの追加] または [電子メール フォルダーの追加] ページにある [名前] ボックスに、フォルダーの名前を入力します。

  3. 必要に応じて、[作成場所] ボックスの横にある [Content Server の参照] をクリックし、フォルダーを作成する場所に移動して、[選択] リンクをクリックします。

  4. その他の一般的なアイテム設定を行います。

  5. [追加] をクリックします。
