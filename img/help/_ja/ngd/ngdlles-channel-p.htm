チャンネルを作成するには

  1. チャンネルを作成する場所を右クリックして、[アイテムの追加]、[チャンネル] の順にクリックします。

  2. [追加: チャンネル] ページの [名前] ボックスに、チャンネルの名前を入力します。

  3. 必要に応じて、次の手順を実行します。

    • [説明] ボックスにチャンネルの説明を入力します。

    • [画像] ボックスの横にある [Content Server の参照] をクリックしてチャンネルに添付する画像を指定し、対応する [選択] リンクをクリックします。

    • 画像を表示できないユーザーのために、[ALT テキスト] ボックスに画像の説明を入力します。

    • [作成場所] ボックスで、チャネルを作成する場所を指定します。

  4. その他の一般的なアイテム設定を行います。

  5. [追加] をクリックします。

 

ヒント:
チャンネルを右クリックしてから [開く] を選択することで、既存のチャンネルのコンテンツを表示できます。
