Enterprise Connect を使用して Content Server に電子メールを追加する場合、命名規則が電子メールに適用されます。管理者が Enterprise Connect ツリーで [マイセッティング] ノードを有効にした場合、ユーザーは電子メールの名前に関して規則を指定できます。[電子メールで使用される名前の生成] ページに表示される初期値は、管理者が設定した値です。[電子メールで使用される名前の生成] ページでは、次の操作を実行できます。

  • 電子メールの Content Server 名を生成するときに使用するメッセージ プロパティを指定します。

  • 電子メールの名前を構成する個々のメッセージ プロパティの間に使用する区切り文字を定義します。

  • 生成された電子メール名から削除する文字列を指定します。

  • 日時に使用する形式を指定します。

  • 重複する電子メールの処理方法を指定します。

 

注意:
Content Server コンテナー内で既に使用されていた電子メール名を Enterprise Connect が生成した場合、電子メールを追加する前に、名前の競合を解決するように指示されます。ファイル名の競合を解決する方法の詳細については、電子メールの操作を参照してください。
