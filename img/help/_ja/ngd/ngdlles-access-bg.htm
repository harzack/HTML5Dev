ツリー構造 で初めてルート Content Server フォルダーにアクセスすると、Enterprise Connect は、インストール時にユーザーが指定した Content Server にユーザーを接続します。

ルート フォルダーの下に、次のフォルダーが既定で表示されます。

  • 私の ワークスペース

  • [お気に入り]

  • [エンタープライズ]

  • [パーソナル]

管理者は、これらのフォルダーの非表示を選択したり、その他のフォルダーを追加したりできます。同様に、管理者は [パーソナル] フォルダーで使用できるアイテムを決定できます。

ツリー構造 や リスト ぺイン でアイテムを右クリックして、表示されたメニューからオプションを選択することで、アクションを実行することができます。管理者が、使用できるコマンドを管理します。状況により、アイテムに対するアクションの実行時に Web 表示が開きます。Web 表示は Content Server インターフェイスの一部を表示するウィンドウであり、Web ブラウザーからアクセスできます。Web 表示には、ユーザーが実行しているタスクと直接関連しているオプションとコマンドのみが表示されます。

 

注意:
複数のアイテムを選択して右クリックすると、コンテキスト メニューには、選択したアイテムに共通する機能のみが表示されます。
