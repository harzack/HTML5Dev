ドキュメント、フォルダー、ディスカッション、およびその他のアイテムを見つけるために、Content Server を検索できます。多くの検索結果を各ページに表示する方法を設定できます。十分な権限がある場合は、プロスペクターを作成して検索プロセスを自動化できます。

 

ヒント:
Enterprise Connect のクイック サーチで検索すると、Content Server の Web 検索と同じ検索結果が表示されます。Enterprise Connect での検索の詳細については、OpenText Enterprise Connect Plug-in for Content Server - ユーザ ガイド (NGDCORE-UGD)を参照してください。
