管理者がこの機能を有効にした場合、ユーザーは [ここから参照] コンテキスト メニュー項目を選択して新しいウィンドウを開き、Content Server Web UI を表示したり、ユーザーが選択したアイテムの [概要] ページを表示したりできます。この機能は、Enterprise Connect ツリーと リスト ぺイン の両方で使用できます。この機能を使用するには、アイテムに対して適切な権限が必要です。[ここから参照] コンテキスト メニュー項目は、次を含めて複数のアイテムで使用できます。

  • ドキュメント。

  • 私のワークスペースとエンタープライズ フォルダー。

  • フォルダー、電子メール フォルダー、コレクションなど、あらゆるコンテナー タイプ。

  • 仮想フォルダーにあるコンテンツなど、[お気に入り] や [パーソナル] フォルダーにあるコンテナー。

  • ジェネレーション、プロスペクター、アンケート、ショートカット、URL、ワークフロー マップ、ワークフロー ステータス、カテゴリ、カスタム ビュー、LiveReports、XML DTD など、コンテナーやドキュメントではないタイプ。

例えば、[私のワークスペース] を右クリックして [ここから参照] を選択した場合、新しいウィンドウが開き、[私のワークスペース] の Web UI 表示が表示されます。ショートカットに対して [ここから参照] を選択した場合、元のアイテムの [概要] が表示されます。
