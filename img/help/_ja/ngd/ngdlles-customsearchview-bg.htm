ユーザー (権限がある場合) または管理者が Content Server Web UI 内で作成したカスタム ビュー検索フォームを使用して、Enterprise Connect で検索を実行できます。カスタム ビュー検索フォームの作成と使用の詳細については、 保存した検索の操作を参照してください。

 

重要項目:
この機能は、Content Server 10 更新 10 以降でのみ使用できます。

 

カスタム ビュー検索フォームが入っているフォルダーにアクセスすると、[カスタム検索] ボタンが Enterprise Connect ツールバー内で使用可能になります。使用している Content Server のバージョンによっては、複数のカスタム ビュー検索にアクセスできる場合があります。この機能は、Outlook と エクスプローラーに加えて、Enterprise Connect の [名前を付けて保存]、[コピー/移動]、[開く] ダイアログ ボックスで使用できます。Enterprise Connect の [名前を付けて保存] および [Copy/Move] ダイアログ ボックスでカスタム ビュー検索を実行すると、検索ではフォルダーが返されます。Enterprise Connect の [開く] ダイアログ ボックスでカスタム ビュー検索を実行すると、検索ではフォルダーと個々のアイテムが返されます。
