Enterprise Connect では、多くの Content Server 機能を実行できます。例えば、アイテムのカテゴリを管理し、アイテムを予約したり、タスク リストを作成したりできます。

 

注意:

  • フォルダー、ドキュメント、タスク リストなど、アイテムを非表示にした場合でも、Outlook と Explorer の両方でこれらが リスト ぺイン に表示されます。また非表示のアイテムが検索結果に含まれます。

  • Enterprise Connect は Content Server の [おすすめ] 機能をサポートしません。
