このアプリケーションでは、OpenText Enterprise Connect ツリー構造 で、Content Server ワークスペースのコンテンツにアクセスできます。エクスプローラー、Microsoft Outlook、Microsoft Office などのアプリケーションを操作する際に、システムにインストールされたオプションに応じて、ユーザーはプラグインを使用することができます。

本書には、Enterprise Connect に特有の情報が記載されています。両方のアプリケーションで機能は同じであるため、本書では、メインの Content Server ヘルプ システムに記載されたトピックスを参照することがあります。Enterprise Connect のウィンドウやダイアログ ボックスでは、直接 Content Server のヘルプにアクセスできることもあります。
