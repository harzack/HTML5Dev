ユーザーが Content Server フォルダーに電子メールを追加する場合、Enterprise Connect は、管理者によって設定された命名規則を適用します。適切な権限がある場合、電子メールを Content Server に追加した後で、電子メールの名前を変更できます。また、環境の設定方法に応じて、電子メールの命名規則を設定できます。詳細については、 電子メールの命名規則の設定.

管理者が Enterprise Connect を設定した方法に応じて、電子メールを Content Server コンテナーに追加する時に、次のいずれかのオプションが検出されることがあります。

重複する電子メール名

Enterprise Connect が目的のフォルダー内で既にアイテムに割り当てられている電子メール名を生成した場合、名前の競合を解決するように指示されます。また電子メールをフォルダーに追加して、同じ名前の電子メールが目的のフォルダー内に既に存在していた場合も、同じように指示されます。同一電子メールの新しいインスタンスには同じ名前が付けられ、末尾にかっこで囲まれて番号が追加され、別のインスタンスであることが示されます。設定に応じて、電子メールの名前を変更できることがあります。詳細については、 電子メールの命名規則の設定.

電子メールのメタデータの提供

環境の設定に応じて、Enterprise Connect ツリー内のフォルダーに電子メールを追加する時に、メタデータの提供を要求されることがあります。既に一部のメタデータが提供されており、システムの設定に応じて、一部のボックスを同様に編集できる場合があります。カテゴリと属性の操作の詳細については、カテゴリの使用を参照してください。

添付

環境の設定に応じて、Enterprise Connect ツリーに電子メールを追加する時に、電子メールと添付ファイルの処理方法を指定するように要求されることがあります。次のいずれかの操作を行うように指示されることがあります。

  • 目的のフォルダーに電子メールと添付ファイルを 1 つのレコードとして保存します。

  • 目的のフォルダーに電子メールと添付ファイルを 1 つのレコードとして保存し、添付ファイルの個別のインスタンスは個別のレコードとして保存します。

OpenText Records Management モジュールの相互参照機能では、電子メールから関連する添付ファイルへリンクすることができます。

 

ヒント:
電子メールを右クリックして、[添付の一覧] をクリックすると、選択した電子メールの全添付ファイルの一覧を表示できます。[添付] ダイアログ ボックスでは、一覧表示された添付ファイルを開いたり、ダウンロードしたりできます。

 
コンパウンド電子メール

環境の設定方法に応じて、ユーザーが保存する電子メールと添付ファイルはコンパウンド電子メールとして保存される場合があります。コンパウンド電子メールとは、電子メール メッセージと添付ファイルが Enterprise Connect 内で 1 つのフォルーとして保存されるメールのことです。コンパウンド電子メールを操作すると、次のようなことがわかります。

  • アイコンはコンパウンド電子メールを示しています。

  • コンパウンド電子メールは、Enterprise Connect の リスト ぺイン 内にのみ表示され、Microsoft Outlook やエクスプローラーのツリー ペインには表示されません。

  • 環境の設定方法に応じて、コンパウンド電子メール メッセージと添付ファイルをオフラインで取得できる場合もあります。コンパウンド電子メールをオフラインで取得する場合、これらの電子メールは読み取り専用になっており、どの添付ファイルにもアクセス可能になります。管理者がオフライン機能を有効にしていない場合、コンパウンド電子メールを右クリックしても [オフライン バージョンのダウンロード] は使用可能になりません。

  • コンパウンド電子メールをデスクトップや別のプラグインにドラッグした場合、ソース サーバーの設定方法に応じて、電子メールは通常の電子メールとして処理される場合とコンパウンド電子メールとして処理される場合があります。

  • 閲覧ウィンドウでは、選択したコンパウンド電子メールの添付ファイルを表示できます。権限がある場合、開く、編集する、ダウンロードするなど、添付ファイルに対して閲覧ウィンドウからさまざまなタスクを実行できます。

     

    ヒント:
    またコンパウンド電子メールを右クリックして、[プロパティ]、[サブアイテム] の順にクリックすることで、コンパウンド電子メールの添付ファイルを表示できます。

     

  • 管理者は、ユーザーがコンパウンド電子メールをダブルクリックした場合のシステムの動作を指定できます。設定に応じて、電子メールが Outlook で開かれる場合もあれば、別の Web パネルが開かれて電子メールと添付ファイルが表示される場合もあります。Web パネルでは、権限に応じて、電子メールまたは添付ファイルを開く、ダウンロードする、編集することができます。
