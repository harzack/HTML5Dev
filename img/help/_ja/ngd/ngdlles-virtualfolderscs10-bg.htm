Enterprise Connect を使用して、仮想フォルダーにあるコンテンツにアクセスし、そのコンテンツに対して多くのタスクを実行できます。Enterprise Connect ツールバーのすぐ下にあるバナーは、仮想フォルダーで作業を行っていることを示します。仮想フォルダーの作成、設定、操作の詳細については、 仮想フォルダの使用を参照してください。

 

注意:
このリリースの Enterprise Connect では、ファセットはサポートされません。

 
 

ヒント:
管理者が移動のオプションを有効にした場合、仮想フォルダーでは、アイテムの元の場所に移動できます。アイテムを右クリックして、[元のロケーションに移動] を選択します。

 

次のセクションでは、Enterprise Connect で使用できる仮想フォルダー機能について簡単に説明します。

仮想フォルダー

Enterprise Connect で仮想フォルダーを操作する場合、次のタスクを実行できます。

  • Enterprise Connect ツリーで仮想フォルダー コンテンツを参照します。

  • 仮想フォルダーにあるドキュメントを開いて編集します。

  • 仮想フォルダーを削除して名前を変更します。

  • プレビュー ウィンドウで仮想フォルダーにあるコンテンツをプレビューします。

  • [アイテムの追加] コンテキスト メニュー アクションを使用して、仮想フォルダー内にある実際のフォルダーにコンテンツを追加します。バナーは、仮想フォルダーまたはそのコンテンツにアクセスしていることを示します。

  • 仮想フォルダーへのデスクトップ ショートカットを作成します。

     

    注意:
    コンテンツは仮想フォルダーで動的に表示されているという特性があるため、仮想フォルダーまたはコンテンツへのショートカットは、コンテンツが変更されると無効になることがあります。

     

  • 同じ仮想フォルダーに表示される同じ名前のドキュメントを開いて編集します。

     

    ヒント:
    同じ名前のドキュメントを区別するには、[Location] 列を見つけて、選択したドキュメントの親フォルダーの名前を確認します。

     

Enterprise Connect で仮想フォルダーを操作する場合、次の例外事項に注意してください。

  • 仮想フォルダーはオフラインで取得できません。

  • ドラッグでも、Enterprise Connect の [名前を付けて保存] ダイアログ ボックスでも、仮想フォルダーにコンテンツを追加することができません。[アイテムの追加] リストまたは [アイテムの追加] コンテキスト メニューは、仮想フォルダーで使用できますが、仮想フォルダーには外観、別の仮想フォルダー、またはカスタム ビューのみを追加できます。

  • 仮想フォルダーのコンテンツを検索できません。

実際のフォルダー

仮想フォルダー内にある実際のフォルダーにアクセスする場合は、次の動作に注意してください。

  • 仮想フォルダーの内部からでは、実際のフォルダーまたはそのサブフォルダーをオフラインで取得することができません。ただし、Content Server 内の実際の場所からは、これらのフォルダーをオフラインで取得できます。

  • 実際のフォルダーを右クリックすると、フル コンテキスト メニューにアクセスできます。

  • 仮想フォルダーにある実際のフォルダーには、コンテンツを追加できます。

ドキュメント

Enterprise Connect 内で仮想フォルダーにあるドキュメントを操作する場合、次の動作に注意してください。

  • [Enterprise Connect] の [開く] ダイアログ ボックスでは、読み取り専用バージョンのドキュメントを開いたり、通常のドキュメントを開いて編集したりできます。

  • [Save As New Version] 機能では、仮想フォルダーに新しいバージョンのドキュメントを保存できます。

  • リスト ウィンドウに同じファイル名のドキュメントが 2 つ表示されている場合、どちらかを開いて編集することができます。

     

    ヒント:
    同じ名前のファイルを区別するには、[Location] 列を見つけて、選択したドキュメントの親フォルダーの名前を確認します。

     

  • 仮想フォルダーにあるドキュメントを削除したり名前を変更したりできます。

  • 仮想フォルダーからデスクトップへドキュメントをドラッグできます。

  • 仮想フォルダーから実際のフォルダーへドキュメントをドラッグできます。

  • 仮想フォルダーにあるドキュメントはオフラインで取得できません。

  • 仮想フォルダーにあるドキュメントを右クリックした場合、Content Server Web UI の使用時の表示と同じように、コンテキスト メニュー アクションが表示されます。

  • Enterprise Connect ビューアーでは、仮想フォルダーにあるドキュメントを表示できます。

  • ドキュメントが保存されている親フォルダーの名前は、リスト ウィンドウで [Location] 列に表示されます。
