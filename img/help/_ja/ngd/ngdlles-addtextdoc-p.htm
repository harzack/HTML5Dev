テキスト ドキュメントを追加するには

  1. テキスト ドキュメントを追加する場所を右クリックして、[アイテムの追加]、[テキスト ドキュメント] の順にクリックします。

  2. [追加: テキスト ドキュメント] ページの [名前] ボックスに、テキスト ドキュメントの名前を入力します。

  3. [テキスト] ボックスに、テキスト ドキュメントの内容を入力します。

    オプションで [スペルチェック] アイコン をクリックしてスペル ミスをチェックすることができます。

  4. その他の一般的なアイテム設定を行います。

  5. [MIME タイプ] リストから、テキスト ドキュメントの MIME タイプを選択しますデフォルト値は text/plain です。

  6. [追加] をクリックします。

 

ヒント:
使用している Content Server のバージョンによっては、HTML MIME タイプの指定と編集ができる場合があります。
