Content Server 上の異なる場所から同じアイテムにアクセスするには、ショートカットを作成します。例えば、頻繁にアクセスするフォルダーのショートカットをユーザーのパーソナル ワークスペース に作成しておくと、簡単にアクセスできるようになります。ショートカットは、コンテキスト メニューで [ショートカットの作成] オプションを使用するか、または、アイテムを右クリックして Content Server のコンテナーにドラッグして [ここにショートカットを作成] オプションを使用することによって作成できます。ドラッグ オプションは Windows エクスプローラーでのみ利用できます。このオプションは、フォルダー、プロジェクト、およびタスクなどのサポートされているコンテナー タイプにのみ利用可能です。タスク リストやタスク グループなどには利用できません。システムの設定状況によっては、新しいショートカットのメタデータを指定するように指示される場合があります。

[リンクをクリップボードにコピー] コマンドを使用すると、Windows のクリップボードにアイテムのリンクをコピーすることができます。次にこのリンクを電子メール、Word や Excel などのデスクトップ アプリケーション、Web ブラウザーに貼り付けることができます。ユーザーは、送信するリンクのタイプを選択できますが、さらに選択したアイテム タイプに応じて使用可能なオプションを選択できます。例えば、ドキュメントの [概要] ページまたは [プロパティ] ページが開くためのリンクや、ドキュメントを開くかまたはダウンロードするためのリンクを指定することができます。電子メール、コンパウンド電子メール、フォルダー、およびその他のアイテム タイプ (ディスカッション、アンケート、タスク リストなど) の場合、リンクでそのアイテムを開くように指定したり、[プロパティ] ページが開くように指定したりできます。
