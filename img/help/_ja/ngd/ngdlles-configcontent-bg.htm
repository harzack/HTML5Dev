ユーザーは、Content Server に追加するアイテムに関連して、メタデータの入力を指示するプロンプトを表示させるタイミングを指定できます。メタデータの入力を指示するプロンプトは、属性またはその他の情報が必要な場合にのみ表示させるか、Content Server にアイテムを追加するたびにメタデータを入力するように表示させることができます。メタデータの取得に最も適したオプションを選択してください。

 

注意:
Content Server インスタンス内でコンテンツの追加、移動、コピーを行う場合は、メタデータの入力を指示するプロンプトは表示されません。
