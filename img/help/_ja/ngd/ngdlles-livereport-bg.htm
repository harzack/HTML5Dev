LiveReport では、組織のデータベース内の情報にアクセスして、保存されている情報を検索、整理および表示することができます。LiveReport を使用したデータベースへのアクセス方法は、Content Server 検索を使用したアクセスとは異なります。これは、LiveReport ではデータベース内の Content Server アイテム関連部分のみではなく、すべてのデータベース スキーマを照会できるためです。LiveReport はデータベースに SQL (Structured Query Language) ステートメントを発行することによって情報を取得します。SQL ステートメントでは重要なデータの変更や削除が可能なため、デフォルトでは LiveReport の追加権限または編集権限は、管理者に制限されています。LiveReport を開くと、自動的に SQL ステートメントおよびその他のパラメーターが実行されます。LiveReport を開くには、その LiveReport の内容参照権限が必要です。

LiveReport を作成して、任意のフォルダーに入れることができます。また、LiveReport のボリュームには、デフォルトの LiveReport セットが格納されます。参照権限が与えられているデフォルトの全 LiveReport が、[パーソナル] フォルダー内にあるレポート フォルダーに表示されます。
