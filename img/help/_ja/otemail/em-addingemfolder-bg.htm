Content Server の電子メール フォルダーは、電子メール システムにおけるフォルダーと同じように使用して作業内容をまとめることができます。

電子メール フォルダーは、標準の Content Server フォルダーに似ています。ただし、電子メール フォルダーでは、一度に多数の電子メールを 1 ページに表示できます。電子メール フォルダーに電子メールを保存する場合、次に示す情報カラムがデフォルトで追加表示されます。管理者がオプションを有効にしている場合、これらのカラムは標準の Content Server フォルダーにも表示されます。

  • 差出人

  • 宛先

  • 件名

  • 送信日

  • 受信日

フォルダーに保存するものが主に電子メール メッセージで、多数の電子メール メッセージを格納する場合は、電子メール フォルダーを使う必要があります。

電子メール フォルダーの追加

電子メール フォルダーは、アイテムを追加する権限を持っているすべてのワークスペース ページまたはコンテナ ページに追加できます。

Content Server に電子メール フォルダーを追加する場合、電子メール フォルダーに説明を加えたり、電子メール フォルダーのカテゴリと属性を変更したり、電子メール フォルダーを別のコンテナーに追加したりできます。アイテムの追加の詳細については、 一般的なアイテムの操作 を参照してください。

権限

デフォルトでは、電子メール フォルダーに追加するすべてのアイテムは追加先の電子メール フォルダーから権限を継承します。例えば、グループに電子メール フォルダーに対する内容の参照権限が割り当てられている場合、そのグループのメンバーは電子メール フォルダーに追加されるすべてのアイテムを参照する権限を持ちます。また、電子メール フォルダー内の各アイテムの権限は変更することもできます。詳細については、 権限の理解 を参照してください。
