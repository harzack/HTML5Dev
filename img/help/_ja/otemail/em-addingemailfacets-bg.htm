[コンテンツ フィルター] サイドバーにファセットとして表示されるように、次の電子メール属性を設定できます。

  • BCC

  • CC

  • 差出人

  • 添付

  • 代理人

  • 受信日

  • 送信日

  • 宛先

これらの属性をファセットとして追加するには、ファセット ボリュームへのアクセス権を持つ Content Server 管理者またはナレッジ管理者である必要があります。電子メール属性は利用可能なデータ ソースのリストに表示されます。データ ソースは、新しいファセットの追加時に割り当てることができます。ファセットの追加の詳細については、 ファセット ボリュームの操作 を参照してください。

これらの属性がファセットとして追加される場合、これらを使用して、電子メールを保存するフォルダーのビューにフィルターを適用できます。ファセットを使用してフォルダー ビューにフィルタを適用する方法の詳細については、 コンテンツ フィルター サイドバーの操作 を参照してください。

 

注意:
電子メール ファセットは電子メールのみに適用されます。電子メールの添付ファイルには適用されません。

 

Content Server にカスタム カラムとして表示されるように、次の電子メール属性を設定できます。

  • BCC

  • CC

  • 差出人

  • 添付

  • 代理人

  • 受信日

  • 送信日

  • 件名

  • 宛先

これらの属性をカスタム カラムとして追加するには、ファセット ボリュームへのアクセス権を持つ Content Server 管理者またはナレッジ管理者である必要があります。電子メール属性は利用可能なデータ ソースのリストに表示されます。データ ソースは、新しいカスタム カラムの追加時に割り当てることができます。カスタム カラムの追加の詳細については、 ファセット ボリュームの操作 を参照してください。
