You can use the Custom View Search forms that you (if you have permissions) or your administrator have created in the Content Server Web UI to perform searches in Enterprise Connect. For more information about creating and using Custom View Search forms, see Working with Saved Searches.

 

Important:
This functionality is only available in Content Server 10, update 10 or later.

 

When you access a folder that contains a Custom View Search form, the Custom Search button is available in the Enterprise Connect toolbar. Depending on which version of Content Server you are using, you may be able to access multiple Custom View Searches. This functionality is available in Outlook and Explorer, as well as the Enterprise Connect Save As, Enterprise Connect Copy/Move, and Enterprise Connect Open dialog boxes. When you run a Custom View Search from the Enterprise Connect Save As and the Enterprise Connect Copy/Move dialog boxes, the search returns folders. When you run a Custom View Search from the Enterprise Connect Open dialog box, the search returns folders and individual items.
