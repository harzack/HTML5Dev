If your administrator has enabled this feature, you can use the Browse from Here context menu item to launch a new window that contains either the Content Server Web UI or the Overview page for the item that you have selected. This feature is available in both the Enterprise Connect tree and in the List Pane. You must have the appropriate permissions for the item to be able to use this feature. The Browse from Here context menu item is available for items including the following:

  • Documents.

  • My Workspace and the Enterprise Folder.

  • All container types such as Folders, Email Folders, and Collections.

  • Containers that reside in the Favorites and Personal folders, as well as content located in Virtual Folders.

  • Non-container and non-document types such as Generations, Prospectors, Polls, Shortcuts, URLs, Workflow Maps, Workflow Status, Categories, Custom Views, LiveReports, and XML DTDs.

For example, if you right-click My Workspace and then select Browse from Here, a new window opens, containing the Web UI view of My Workspace. If you select Browse from Here for a Shortcut, you will see the Overview for the original item.
