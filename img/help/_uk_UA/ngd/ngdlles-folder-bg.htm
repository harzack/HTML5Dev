You use Folders to organize your work the same way that you work with folders on your computer. Folders can contain almost any other type of item, including Projects and other Folders. A Folder inside another Folder is sometimes called a Subfolder.

E-mail Folders are similar to regular Folders. E-mail Folders include email columns that enable you to view email metadata for any emails that reside in the Folder. When you store emails in an E-mail Folder, the following additional columns of information are provided by default.

  • From

  • To

  • Subject

  • Sent Date

  • Received Date

You should use E-mail Folders when the primary contents stored in the Folder are email messages.

You can add a Folder or E-mail Folder to every Workspace or container page where you have the permission to add items. For more information about moving and copying items, see Moving Items and Copying Items.

If you drag a Folder into the Enterprise Connect tree and a Folder already exists with the same name, you have the option of merging the new Folder with the existing one.
