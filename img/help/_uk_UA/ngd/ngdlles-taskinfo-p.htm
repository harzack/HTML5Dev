To view Task List information:

  1. Right-click the Task List for which you want to view information, and then click Task List.

  2. Choose the appropriate option for the information that you want to view.

To Do this
View a list of all Tasks and Milestones associated with the Task List Choose Workspace.
View a general summary of Task List progress, Milestones, and Resources Choose Summary.
View all Milestones for the Task List Choose Detail Milestone List.
View a summary of the assignments for a specific user or group Choose Detail Resource List.
