To add an item to a Collection:

 

Note:
Depending on how your environment is configured and on which version of Content Server you are using, you may see variations in the options available and the names of dialog boxes and lists.

 

  1. Select the item or items that you want to add to a Collection, right-click them, and then choose Collect.

  2. In the Collect dialog box, if available, select the Collect Current Version check box.

  3. In the Collection area, do one of the following to specify which Collection you want to add the selected item or items to:

    • If available, select a Collection from the list of recently used Collections.

    • If available, select a Collection from your Favorites or Personal folders.

    • Create a new Collection. In the Add Collection dialog box, specify the name, description, Classifications or Categories if required, and the location for the new Collection, and then click Add.

    • Browse Enterprise Connect to find the Collection that you want to add the items to.

  4. Click Submit and then OK.
