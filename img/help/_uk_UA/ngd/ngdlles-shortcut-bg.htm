You create a Shortcut if you want to access the same item from different locations in Content Server. For example, you may want to create Shortcuts in your personal Workspace for the folders you access most often. You can create Shortcuts using either the context menu using the Make Shortcut option, or by right-clicking and dragging an item to a container in Content Server, using the Make Shortcut Here option. The dragging option is only available in Windows Explorer. This option is only available for supported container types, such as Folders, Projects, and Tasks. It is not available for Tasklists or Taskgroups, for example. Depending on how your system is set up, you may be prompted to provide metadata for the new Shortcut.

Using the Copy Link to Clipboard command, you can copy a link for an item to the Windows Clipboard. You can then paste the link into emails, desktop applications such as Word or Excel, or web browsers. You can choose the type of link that you want to send and the options available depend on the type of item you have selected. For example, you can specify that the link opens the Overview or Properties page for Documents, or that the link opens or downloads the Document. For emails, Compound Emails, Folders and other item types such as Discussions, Polls, and Task Lists, you could specify that the link opens the item or that the Properties page opens.
