To create a Shortcut:

  1. Right-click the item for which you want to create a Shortcut, and then click Make Shortcut.

  2. On the Add Shortcut page, type a name for the Shortcut in the Name box.

  3. Next to the Create In box, click Browse Content Server, navigate to the location where you want to create the Shortcut, and then click its Select link.

  4. Specify any other general item settings.

  5. Click Add.
