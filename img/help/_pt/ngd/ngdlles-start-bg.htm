This application enables you to access the contents of Content Server workspaces in the OpenText Enterprise Connect Tree Structure. Depending on the options installed on your system, you can use the plug-in when you are working with applications such as Windows Explorer, Microsoft Outlook, or Microsoft Office.

This help system contains information that is specific to Enterprise Connect. In some cases this help system refers you to topics contained in the main Content Server help system, because the functionality is the same in both applications. In some instances you can access Content Server help directly from Enterprise Connect windows and dialog boxes.
