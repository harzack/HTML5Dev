To create a LiveReport:

  1. Right-click the location where you want to create the LiveReport, click Add Item, and then click LiveReport.

  2. On the Add LiveReport page, type a name for the report in the Name box.

  3. Type the maximum number of results allowed in the Record Limit box.

  4. Type a SQL statement for the report in the SQL box.

  5. Click parameter types that are associated with the SQL statement in the Parameter lists.

  6. Click a format for the LiveReport in the Report Format list.

  7. The following table describes how to perform additional actions to define the LiveReport.

    To Do this
    Add a title that appears on the LiveReport Results page Type a name in the Title box.
    Require users to supply new parameter values when using the LiveReport as a Subreport Select the Show Prompts check box.
    Define input types In the Inputs area, click an input type in the Type list, and then type the text that you want to display to request user input in the Prompt box.
    Allow the SQL statement to alter the contents of the Content Server database Select the Allow Database Modification check box.
    Associate a Subreport with this LiveReport In the Click-thru Sub-Report area, click Browse, navigate to the LiveReport you want to appear as a Subreport, and then click its Select link.
    Assign parameters to a Subreport Type the name of an existing parameter in each Sub- Report Param # box.
    Specify what information from the database is shown on the LiveReports Results page Type the name of the database column in the Field box, and then type a corresponding display name for the column in the Column Title box.
    Configure the appearance of the selected LiveReport format Choose a format for the LiveReport, click Configure, make changes in the Configure Report Format window, and then click Select.

  8. Click Add.

 

Tip:
You can modify the settings for an existing LiveReport by right-clicking the LiveReport, and then clicking Edit.
