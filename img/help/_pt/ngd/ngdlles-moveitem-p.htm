To move an item:

  1. Select the items that you want to move, right-click, and then select Copy/Move to Enterprise Connect.

  2. In the Enterprise Connect dialog box, navigate to the target location, and then click Move.

  3. Depending on how your administrator has configured your environment, you may be asked to do the following:

    To Do this
    Define Category options In the Categories area, do one of the following:
    •

    Click Original if you want to maintain the Categories that were originally assigned to the item or items. The assigned Categories appear in the Categories box. Click Edit to edit the Categories if required.

     

    Note:
    If you have selected multiple items, and if you click Edit in the Move dialog box to edit the assigned Categories, the Move dialog box will be displayed for the next selected item, which enables you to edit the Categories for that item. However, if you do not click Edit, the Categories that you have specified are applied to all of the remaining selected items.

     
    •

    Click Destination if you want to apply the Categories of the new location to the item or items. To view the Categories assigned to the destination, click Edit, and then Done on the Edit Categories dialog box. You can edit the Categories if required.

    •

    Click Merged if you want to apply the Categories of both the original and new locations to the item or items. To view the Categories assigned to the destination, click Edit, and then Done on the Edit Categories dialog box. You can edit the Categories if required. If you have selected multiple items, the Move dialog box will open for each item, enabling you to edit the Categories if required.

    If applicable, specify if you want the item to inherit RM Classifications In the RM Classifications area, do one of the following:
    •

    Click Yes if you want the item to inherit the RM Classifications that have been assigned to the destination.

    •

    Click No if you do not want the item to inherit the RM Classifications that have been assigned to the destination.

  4. If there is an item in the destination with the same name, do one of the following in the Enterprise Connect dialog box:

    • Click Yes if you want to create a new instance of the item in the destination, using an automatically generated name.

    • Click Yes to All if you want to resolve any naming conflicts that may arise with other items in the batch that you are moving by creating a new instance of the item in the destination, using an automatically generated name.

    • Click No if you do not want to move the current item to the destination. The move action continues after you select this option.

    • Click No to All if you do not want to move duplicate items to the destination. The move action continues after you select this option.

    • Click Cancel if you want to stop the move action. Any items that have been successfully moved will remain in the destination.

     

    Note:
    If you are moving items and are working with Enterprise Connect in Outlook, the No and No to All options will not be available due to a limitation with Outlook functionality. However, if you use the Enterprise Connect Copy/Move dialog box to move items, these options will be available. These options are also available in the Enterprise Connect Easy Access pane.

     

  5. The Moving dialog box shows the progress of the moving command.
