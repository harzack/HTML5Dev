LiveReports provide access to information in your organization's database, and enable you to find, organize, and present the information stored there. Accessing the database using LiveReports is different than using Content Server search, because you can query any part of the database schema, rather than just the part of the database related to Content Server items. LiveReports obtain information by issuing Structured Query Language (SQL) statements to the database. Since SQL statements can alter or delete critical data, the permission to add or edit LiveReports is restricted to the administrator by default. Opening a LiveReport automatically executes the SQL statement and other parameters. In order to open a LiveReport, you must have the See Contents permission for that LiveReport.

You can create LiveReports and then place them in any Folder. In addition, the LiveReports Volume contains a set of default LiveReports. All of the default LiveReports that you have permission to see appear in the Reports Folder, which is located in the Personal folder.
