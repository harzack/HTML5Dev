To create Custom Views:

  1. Right-click the location where you want to create a Custom View, click Add Item, and then click Custom View.

  2. On the Add Custom View page, click Browse, navigate to the HTML file you created that defines the Custom View, and then click Open.

  3. Specify any other general item settings.

  4. Click Add.

 

Note:
If the HTML file contains references to other files (such as images), you must add those files separately.
