To perform an advanced search:

  1. On the Enterprise Connect menu or tab, click Search, and then click Search Content Server.

     

    Tip:
    You can also access the Advanced Content Server Search page by clicking Advanced Search on the Enterprise Connect toolbar in Windows Explorer and Microsoft Outlook. In addition, you can access Advanced Search from the search box in the Easy Access pane, and in the Copy/Move to Enterprise Connect, Enterprise Connect Open, and the Enterprise Connect Save As dialog boxes. The button is only available for types of containers that you can search and for containers that you have permissions to search. For example, Advanced Search is not available for the Enterprise Connect root node, search results, the Favorites folder, the Personal folder, and non-physical containers such as Virtual Folders.

     

  2. Use the information in the following table to specify search criteria.

     

    Note:
    Additional options may be available, depending on which Content Server modules are available to you. For more information about searching in Content Server, see OpenText Content Server User Online Help - Searching Content Server (LLESWBB-H-UGD).

     

    To Do this
    Indicate where you want to search In the Add to Search Form area, click Location. Then, in the Location area, click Browse Content Server and select the location that you want to search.
     

    Notes:

    • If you do not specify where you want to search, Enterprise Connect searches the entire Content Server repository.

      The current search criteria are cleared when you close the Search Content Server dialog box.

    • When performing Advanced Searches using Outlook, additional columns that you selected using the Field Chooser will be displayed any time you search one or more containers of the same type. However, if you perform an Advanced Search on two or more containers that are different types, only the default columns will be displayed.

     
     

    Tip:
    To further define your search, you can click Duplicate , select the appropriate Boolean operator from the list, and then specify additional search locations.

     
    Specify whether the search should find all words, any words, or an exact phrase In the Full Text area, click an option in the Look For list, and then enter the search term.
    Perform a complex Query In the Full Text area, click Complex Query in the Look For list, and then enter the search term. For more information, see Searching Using Complex Queries.
    Specify a modifier to expand the search In the Full Text area, click an option in the Modifier list, and then enter the search term.
    Specify whether the search is restricted to item content, metadata, or both In the Full Text area, click an option in the Within list, and then enter the search term.
    Search using a sentence or phrase In the Add to Search Form area, click Natural Language Query. Enter the search term that you want to use, and then select the appropriate option from the Mode list. For more information about natural language queries, see Searching with Natural Language Queries.
    Perform a system attribute search In the Add to Search Form area, click System Attributes. Click the Attribute you want to include in the search in the Additional Attributes list, and then specify the Attribute values you are looking for. For more information about using System Attributes, see Working With Categories and Attributes.
    Perform a Category search In the Add to Search Form area, click Categories, and then select the Category that you want to use in the Select Category dialog box. In the Category area, specify values of the Category attributes that you want to use. For more information about using Categories, see Working With Categories and Attributes.
    Perform a Classifications search In the Add to Search Form area, click Classifications. In the Classifications area, click either Browse Classifications to locate the Classifications that you want to search, or click My Favorites, where you can select the appropriate Classifications from your Favorites folder. Select the Include Sub-Classifications check box if you want to include any sub-classifications in the search. By including Sub-classifications, you can search for the selected Classifications and all of its descendents. For more information about using Classifications, see Using Classifications.
     

    Note:
    This option is only available if Classifications has been installed in your environment.

     
    Search for email (basic search) In the Add to Search Form area, click E-mail. Provide search criteria, including OTEmailSubject, OTEmailReceivedDate, and OTEmailSentDate. You can also specify names of email senders and recipients in the Look For box.
     

    Notes:

    • Use a semicolon to separate multiple search criteria entries in the Look For box.

    • Wild card searches are not supported.

     
    Perform an advanced email search In the Add to Search Form area, click E-mail (Advanced). Click the Additional Attribute list to select the columns that you want to search criteria, and then specify the appropriate values.
    Search for Physical Item Types In the Advanced Options area, click Physical Item Types. In the Physical Item Types area, click the Select Physical Item Type list and then click Browse Physical Item Types. In the Browse Physical Item Types page, click Select to choose the appropriate item type.
     

    Note:
    This option is only available if Physical Objects has been installed in your environment.

     
    Search for Physical Item Attributes In the Add to Search Form area, click Physical Item Attributes. In the Physical Item Attributes area, specify the attributes that you want to search for. The Home Location list is provided by default, but you can remove it if required. Click the Additional Attributes list to add more attributes to the search.
     

    Note:
    This option is only available if Physical Objects has been installed in your environment.

     
    Perform a search of Records Management Attributes In the Add to Search Form area, click Records Management Attributes. In the Records Management Attributes area, click the Additional Attributes list, and select and then define the Attribute that you want to search for.
     

    Note:
    This option is only available if Records Management has been installed in your environment.

     
    To search using a Template In the Use this Search Form list, click the Template that you want to use. The Search page opens and displays the search criteria specified in the Template that you selected. If required, you can edit the search criteria.
    Add additional criteria Where available, click Duplicate , click a Boolean operator in the list, and then specify the additional criteria.
    Remove criteria Where available, click Remove for the criteria that you want to remove.
    Remove an option from the search form Click Remove Component for the option that you want to remove.
    Remove all search criteria Click Clear.

  3. Click Search.

 

Notes:

  • When you specify a date range for system attributes or custom attributes within a Category, you can leave the date in either the From or the To lists blank. If you leave the From lists blank, Content Server searches all dates prior to the date in the To lists. If you leave the To lists blank, Content Server searches from the date in the From lists to the present.

  • For some items in the search result list, the Launch in Browser option on the context menu may be the only option available. This occurs when you do not have permission to access the parent folder of the returned item or if the volume cannot be viewed using Enterprise Connect. Click Launch in Browser if you want to work with the selected item in a separate browser instance. This behavior also applies to Enterprise Connect searches and Quick Searches.

  • Depending on your environment, you may be able to set display options for your search results by clicking the Edit the Display Options link. For more information about the options available on the display options page, see Setting Display Options.
