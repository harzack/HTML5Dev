To add Documents:

  1. Right-click the location where you want to add a Document, click Add Item, and then click Document.

     

    Tip:
    You can also click Document on the Add Item menu.

     

  2. In the Document area of the Add Document page, click Browse to navigate to the file you want to add, and then click Open.

  3. Type a name for the Document in the Name box and optionally, a description of the Document.

  4. In the Version Control area, choose a type of versioning by clicking Standard - linear versioning or Advanced - major/minor versioning.

    Optionally, specify settings for Classifications and Categories.

  5. In the Create In area, click Browse Content Server to specify where you want to save the Document.

  6. Click Add.

 

Note:
If there already is a document in the destination location with the same name, the Enterprise Connect dialog box opens. Select one of the following options:

  • Yes if you want to add the new document as a version of the existing document.

  • Yes to All if you want to add all duplicate documents to Content Server.

  • No if you do not want to add the document to Content Server.

  • No to All if you do not want to add any duplicate documents to Content Server. Only new documents will be added.

  • Cancel to stop the action.
