When you first access the root Content Server Folder in the Tree Structure, Enterprise Connect connects you to the Content Server you specified during installation.

The following folders appear by default beneath the root folder:

  • My Workspace

  • Favorites

  • Enterprise

  • Personal

Your administrator can choose to hide these folders or add additional folders. As well, your administrator can determine which items are available in your Personal folder.

You can perform actions by right-clicking items in the Tree Structure and in the List Pane, and then choosing an option from the menu that appears. Your administrator controls which commands are available. In some cases, a Web view will open when you perform an action on an item. A Web view is a window that displays a portion of the Content Server interface that is accessible through a Web browser. The Web view only displays options and commands that are directly relevant to the task you are performing.

 

Note:
If you select and then right-click two or more items, the context menu will only contain functions that are common to the items that you have selected.
