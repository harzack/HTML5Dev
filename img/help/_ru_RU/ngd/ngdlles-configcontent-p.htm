To configure content capture options:

  1. In the Enterprise Connect Tree, click a Content Server container, Personal, and then My Settings.

  2. In the Metadata Capture Configuration area, do one of the following in the Prompt for metadata when content is added list to specify when you should be prompted to provide metadata for items that you add to Content Server:

    • Click Only when required if you only want to be prompted to provide metadata if the metadata is marked as required for the destination Content Server folder. This is the default setting.

    • Click Always if you want to be prompted to provide metadata for each item you add to Content Server.

  3. Click Submit to apply your changes.

     

    Tip:
    Click Reset to revert to the most recently saved values for email naming conventions. Click Use Defaults to revert to the settings provided by the administrator.
