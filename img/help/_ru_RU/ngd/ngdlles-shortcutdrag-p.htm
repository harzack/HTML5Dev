To create a Shortcut by dragging:

  1. Select the item or items for which you want to create a Shortcut.

  2. Right-click and drag the item to the location where you want the Shortcut to reside.

    When you release the mouse button, a context menu appears.

  3. On the context menu, click Create Shortcut Here.

  4. Provide metadata if required.

     

    Tip:
    If a Shortcut with the same name already exists in the destination, the new Shortcut will be created, with a numeric suffix to distinguish it from the one that already resides in the location.
