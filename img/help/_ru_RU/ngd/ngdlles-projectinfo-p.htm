To access Project information:

  1. Right-click the Project you want to work with, and then click Project.

  2. Choose the appropriate information type.

To Do this
Access a detailed Project overview, including Project Summary, Current News, and Tasks. Choose Overview.
Access the Project Workspace. Choose Workspace.
View a list of Project tasks that are assigned to you. Choose My Tasks.
Access News items related to the Project. Choose News.
Define a Project Outline. Choose Outline.
Access a list of Project participants. Choose Participants.
Access Project queries, snapshots, and reports. Choose Reports.
