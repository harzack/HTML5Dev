You can perform searches of Content Server to locate Documents, Folders, Discussions, and other items. You can configure how many search results are displayed on each page. If you have sufficient permissions, you can create Prospectors to automate the search process.

 

Tip:
If you search using Enterprise Connect quick search, you will receive the same search results as from the Content Server web search. For more information about searching in Enterprise Connect, see OpenText Enterprise Connect - User Getting Started Guide (NGDCORE-UGD).
