To add new Documents:

  1. Right-click the location where you want to add a new Document, click Add Item, and then click Document.

     

    Tip:
    You can also click Document on the Add Item menu.

     

  2. In the Document area of the Add Document page, click New, and then click the document type that you want to add in the list.

  3. Type a name for the Document in the Name box.

  4. In the Version Control area, specify a type of versioning by clicking Standard - linear versioning or Advanced - Major/Minor versioning.

  5. Specify any other general item settings.

  6. Click Add.

  7. Add content to the document that opens in its associated desktop program, and then close it.

  8. In the Finish Edit dialog box, click Save the changes to Content Server, and then click OK.
