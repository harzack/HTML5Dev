To create Channels:

  1. Right-click the location where you want to create the Channel, click Add Item, and then click Channel.

  2. On the Add Channel page, type a name for the Channel in the Name box.

  3. Optionally, do the following:

    • Type a description of the Channel in the Description box.

    • Click Browse Content Server beside the Image box, navigate to an image that you want to attach to the Channel, and then click its Select link.

    • Type a description of the image for users who are unable to see it in the ALT Text box.

    • Specify where you want to create the Channel in the Create In box.

  4. Specify any other general item settings.

  5. Click Add.

 

Tip:
You can view the contents of an existing Channel by right-clicking the Channel, and then choosing Open.
