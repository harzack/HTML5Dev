To add Milestones:

  1. Right-click the Task List to which you want to add the Task, click Add Item, and then click Milestone.

  2. On the Add Milestone page, type a name for the Milestone in the Name box.

  3. Click a month, date, year, and time in the Target Date lists to specify the Task's target completion date.

    Optionally, describe the Milestone and specify any general settings.

  4. Click Add.
