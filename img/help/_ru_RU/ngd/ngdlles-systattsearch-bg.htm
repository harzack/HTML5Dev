Many system attributes appear by default on the Search page, but some must be enabled by an administrator to appear. For example, an administrator can enable the OTDocumentRating attribute so that users can search for Documents with a particular rating. An administrator will often enable new attributes when a module or a new version of Content Server is installed.

The following table contains examples of attributes an administrator may enable.

 

Note:
The attributes are shown with their default names; however, your administrator may change these names.

 

Attribute Lets you
OTDocumentRating Search for items with a certain rating.
OTDocumentSynopsis Search for items with certain information in their synopses.
OTDocumentUserComment Search for items with certain information in their comments.
OTDocumentUserExplanation Search for items with certain information in their explanations.
OTProjectStatus Search for Projects with a certain Project Status.
