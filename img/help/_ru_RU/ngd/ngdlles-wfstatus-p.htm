To create Workflow Status items:

  1. Right-click the location where you want to create a Workflow Status, click Add Item, and then click Workflow Status.

  2. Provide a name and description of the Workflow Status in the Name and Description boxes.

  3. In the Relationship list, click one of the following:

    • All, which displays only the Workflows for which you are the manager or the initiator.

    • Initiated, which lists only the Workflows that you have initiated.

    • Managed, which lists only the Workflows that you manage.

    • System, which is available only to the administrator or users with Bypass privileges.

  4. In the Workflows areas, click one of the following:

    • Not Archived, to view only executing Workflows.

    • Archived, to view only archived Workflows.

  5. Optionally, you can do the following:

    • Click the Edit Expression icon to edit the Query using Expression Builder.

    • Click an order type in the Sort Order list to sort the list of Workflows.

    • Click Browse Content Server to specify where you want to create the Workflow Status.

  6. Click Add.

 

Note:
You can edit a Workflow Status by right-clicking the Workflow Status that you want to change and choosing Edit Query.
