To search using Custom View Search forms in Enterprise Connect dialog boxes:

  1. Do one of the following:

    • To use the Custom View Search form to find a location to save a document, in your application, click Save As in the Enterprise Connect menu.

    • To use the Custom View Search form to find a document that you want to open, in your application, click Open in the Enterprise Connect menu.

    • To use the Custom View Search form to find a location to copy or move emails to, right-click the emails that you want to relocate, and then choose Copy/Move to Enterprise Connect.

  2. To find the appropriate Custom View Search form, do one of the following:

    • In the Enterprise Connect Save As dialog box, navigate to the folder that contains the custom search form that you want to use.

    • In the Enterprise Connect Open dialog box, navigate to the folder that contains the custom search form that you want to use.

    • In the Enterprise Connect Copy/Move dialog box, navigate to the folder that contains the custom search form that you want to use.

  3. On the Enterprise Connect toolbar, click the Custom Search button and, if more than one option is available, select the appropriate Custom View Search. The button is only available if there is a Custom View Search in the selected folder.

  4. In the Custom View Search dialog box, review the search values, and then click Search.

     

    Tip:
    You may be able to edit some of the available search values before starting the search. However, the changes that you make are not saved.

     

  5. In the search result list, do one of the following:

    • If you are working with the Save As dialog box, click the folder that you want to save the document to. Save the document, using the options described in "Saving Documents" in OpenText Enterprise Connect - User Getting Started Guide (NGDCORE-UGD).

    • If you are working with the Open dialog box, click the document that you want to open.

    • If you are working with the Copy/Move to Enterprise Connect dialog box, click the folder that you want to copy or move the emails to, and then click either Copy or Move.
