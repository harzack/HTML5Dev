You set permissions to define which users or groups can see, modify attributes for, or work with an item. For example, you may want to give your assistant permissions to work with documents that you add to a folder or project.

Content Server automatically sets permissions for an item based on where the item is created, but you can modify the settings from the Permissions page. You can also assign or edit permissions to items stored in Channels, Discussions, or Task Lists by right-clicking the item and clicking Permissions.

Permissions for users are stored in an access list. Creating an item automatically adds the following default entries to the access list:

  • Owner, the user who created the item.

  • Owner Group, the item owner's work group.

  • Public Access, all users.

You cannot delete any of the default entries from the access list, but you can modify their permissions.

You must have sufficient permissions to perform this function.
