To specify the number of search results on each page:

  1. In the Enterprise Connect Tree, click a Content Server container, Personal, and then My Settings.

  2. In the Search Configuration area, select the appropriate number of search results to display on each page. The default value is 25.

  3. Click Submit to apply your changes.

     

    Tip:
    Click Reset to revert to the most recently saved setting. Click Use Defaults to revert to the settings provided by your administrator.
