Renditions is an optional Content Server module that enables you to create and store multiple, read-only versions of Documents. If your administrator has enabled this module and depending on your permissions, you can create, view, and subscribe to Renditions using Enterprise Connect. The main difference between working with Renditions in Content Server and Enterprise Connect is how you access the functionality. In Enterprise Connect, you use the context menus available in the List Pane or the options available in the Versions and Renditions tabs in the Preview Pane to access functionality for selected Documents. For more information about working with Renditions, see the online help for OpenText Renditions.
