To search using Custom View Search forms:

  1. Navigate to the folder that contains the Custom View Search that you want to use.

     

    Note:
    The Custom View Search must already reside in the selected folder.

     

  2. In the Enterprise Connect toolbar, click Custom Search.

  3. In the Custom View Search dialog box, review the search values, and then click Search.

     

    Tip:
    You can edit the available search values before starting the search. However, the changes that you make are not saved.
