A Workflow Map is a predefined implementation of a business process that usually involves multiple users and tasks. For example, a simple Workflow Map can provide a way for an employee to request and receive permission to take a vacation day and may consist of a few tasks that involve the employee, a manager, and a human resources representative. A complex Workflow Map can contain numerous steps and require the participation of dozens of users.

To create a Workflow Map, you first create a Workflow Map item, and then define the Workflow Map using Workflow Designer. For information about working with Workflow Designer, see Exploring the Workflow Map Designer. You can create a Workflow Map in any location where you have permissions to do so. Once you have added a Workflow Map, you can work with it the way you work with other items. For example, you can delete, rename, copy, or move it; make a shortcut to it; make it a News item; or add it to your Favorites page. Also, you can make a Generation of a Workflow Map.
