To edit Documents:

  1. Do one of the following to open a Document for editing:

    • Right-click the Document, and then click Edit.

       

      Note:
      To open some types of Documents for editing, you can right-click the appropriate Document, and then click Open.

       

    • Use the Enterprise Connect Open dialog box for the application.

    • Use the Open dialog box for the native application.

    • Select the appropriate Document from the Most Recently Used file list in the application.

    • Double-click the Document or a shortcut to the Document in Enterprise Connect, if your administrator has enabled this option.

  2. Save the Document using the Enterprise Connect Save As menu.
