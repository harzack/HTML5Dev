To add Tasks:

  1. Right-click the Task List to which you want to add the Task, click Add Item, and then click Task.

  2. On the Add Task page, type a name for the Task in the Name box.

  3. Click High, Medium, or Low in the Priority list.

  4. Click a status in the Status list.

  5. The following table describes how to perform additional actions to assign and define Tasks.

    To Do this
    Assign a Task Click the Select icon , navigate to a user or group, and then click its Select link.
    Specify start and due dates for the Task Click a month, date, year, and time in the Start Date and Due Date lists.
    Assign a Milestone to a Task Click a Milestone in the Milestone list. For information about creating Milestones, see To Add Milestones.
    Add instructions or comments for the Task Type information in the Instructions and Comments boxes.
    Edit a Task Right-click a Task, and then click Edit. Revise the task as required, and then click Update Task.

  6. Click Add.
