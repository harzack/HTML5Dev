A Project is a container that enables participants of a group to store information and items and communicate daily activities with other participants. Project participants can add, share, manage, and organize items within a Project, such as Tasks and Documents.

A Project Template specifies a predefined set of items based on an existing project. You can create a Project Template when you want to use the set of attributes of an existing project to create new Projects.

You can access a number of information pages from the Project folder that help you manage your Projects.
