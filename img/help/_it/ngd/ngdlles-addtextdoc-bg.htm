A Text Document is a Document whose content is written in plain text, such as a document created in Windows Notepad. You work with Text Documents using many of the same functions that you use with other Document types. For example, you can add them and edit their contents.

You can use a built-in text editor to compose Text Documents. The text editor provides a convenient way to add simple text files without having to first compose them in a separate application.

To add an existing plain-text document from your computer, you must add it as a Document. Content Server recognizes it as a plain-text file and treats it as a Text Document. When composing or adding a Text Document, you can provide a description of the Text Document, modify its Categories and Attributes, and add it to a different container.
