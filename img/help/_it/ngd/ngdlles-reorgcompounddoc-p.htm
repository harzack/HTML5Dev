To reorganize Compound Document contents:

  1. Right-click the Compound Document whose contents you want to reorganize, and then click Reorganize.

  2. On the Compound Document Reorganization page, type a number in the New box of each item whose order you want to change.

    Optionally, select the Master Document check box for the item you want to designate the master document.

    Optionally, select the Sequential Renumbering check box to resequence the other items in the Compound Document.

  3. Click Update.
