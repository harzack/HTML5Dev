To add a Folder or E-mail Folder:

  1. Right-click the Folder or Workspace where you want to add the Folder, click Add Item, and then click one of the following:

    • Folder

    • E-mail Folder

  2. On the Add Folder or Add E-mail Folder page, type a name for the Folder in the Name box.

  3. Optionally, next to the Create In box, click Browse Content Server, navigate to the location where you want to create the Folder, and then click its Select link.

  4. Specify any other general item settings.

  5. Click Add.
