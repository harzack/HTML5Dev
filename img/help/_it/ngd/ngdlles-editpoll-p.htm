To edit Polls:

  1. Right-click the Poll you want to edit, and then click Edit.

  2. On the View Poll Comments page, type a name for the Poll in the Name box.

  3. Click a start and end date for the Poll in the Effective Date and Expiration Date lists.

  4. In the Poll area, do the following:

    • Type a question in the Question box.

    • Type answer choices for the question in the Choices boxes.

    • In the Choice Selection area, select one of the following options for each question to determine the method of voting:

      • Radio Button

      • Checkbox

  5. The following table describes additional methods to define the Poll.

    To Do this
    Add instructions to help voters complete the Poll Type them in the Instructions box.
    Add or delete questions and answers In the Poll area, click the Add or Remove buttons
    ( / ).
    Allow voters to view results or add comments In the Options area, select the Allow viewing the results before voting or the Allow anonymous comments check box.
    Add an image to the Poll In the Image area, click Browse Content Server, navigate to an image, and then click its Select link.
    Add alternate text to the image Type it in the ALT Text box.

  6. Click Submit.
