To define the message properties to use in email names:

  1. In the Enterprise Connect Tree, click a Content Server container, Personal, and then My Settings.

  2. In the Name Generation for E-mails section, go to the Properties area.

  3. Click the message property that you want to use from the Available Properties list, and then click Move Right to move it to the Selected Properties list. Subject is the default value.

  4. To set the order in which the message properties appear in the email name, click a message property in the Selected Properties list, and then click Up or Down .

  5. Click Submit to apply your changes.

     

    Tip:
    Click Reset to revert to the most recently saved values for email naming conventions. Click Use Defaults to revert to the settings provided by the administrator.
