If your administrator has enabled this feature, you can access Enterprise Connect when you are working with specified content types in the Content Server web interface. When this feature is available, you can access the Function menu for a supported item, choose Enterprise Connect from Here, and a new instance of Enterprise Connect opens, displaying the item you selected. Your administrator determines which subtypes this feature is available for. By default, this action is labelled Enterprise Connect from Here, but your administrator may create a custom label for this feature. Contact your administrator for more information about this feature.

 

Note:
This feature is only available for instances of Content Server web interface in Internet Explorer.
