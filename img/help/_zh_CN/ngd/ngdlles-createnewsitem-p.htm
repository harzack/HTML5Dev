To create News items:

  1. Right-click the Channel where you want to create the News item, and then click Make News.

  2. On the Add News page, type a name for the News item in the Name box.

  3. Click an Effective Date and Expiration Date in the corresponding month, day, year, and time lists.

  4. Type a headline for the News item in the Headline box.

     

    Note:
    If you leave the Headline box blank, the text in the Name box is used as the headline.

     

  5. Type the text of your News item in the Story box.

  6. The following table describes how to perform additional tasks to define the News item.

    To Do this
    Attach an image to the News item Click Browse Content Server in the Image area, navigate to an image that you want to attach to the News item, and then click its Select link.
    Add a description of the image (for users who are unable to see it) Type a description in the ALT Text box.
    Add an attachment to the News item Click Browse Content Server in the Add Content Server Attachment area, navigate to an item that you want to attach to the News item, and then click its Select link.
    Attach a file from your computer to the News item Click Browse in the Add Desktop Attachment area, navigate to the item, and then click Open.

  7. Click Add.

 

Tip:
You can also add News by right-clicking an item, and then choosing Make News. By default, the item is added as an attachment to the News item.
