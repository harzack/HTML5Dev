To delete an item:

  1. Right-click the item you want to delete, and then click Delete.

  2. In the Delete Item dialog box, click Yes.

 

Note:
If you use the context menu for either a Generation or Shortcut to delete the original item, you will need to click Delete in the Delete dialog box, and then click OK in the Deleting Item From dialog box.

 
 

Tip:
In the List Pane, you can select the items that you want to delete and press Delete.
