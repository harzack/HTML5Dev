To create a Workflow Map:

  1. Right-click the location where you want to add a Workflow Map, click Add Item, and then click Workflow Map.

  2. On the Add Workflow Map page, type a name for the Workflow Map in the Name box.

  3. Specify any other general item settings.

  4. Click Add.

  5. Right-click the new Workflow Map and then click Edit to launch Workflow Designer, which you use to define the Workflow Map.
