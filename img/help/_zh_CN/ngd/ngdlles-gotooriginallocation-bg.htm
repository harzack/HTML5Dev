If your administrator has enabled this feature, you can use the Go to Original Location context menu action to navigate to the original location of an item that resides in a virtual container. This includes containers such as Virtual Folders, Collections, Favorites, and items in a list of search results.

This context menu option is only available if you have appropriate permissions to access the item. In addition, this option is not available if Enterprise Connect cannot navigate the path of the item.
