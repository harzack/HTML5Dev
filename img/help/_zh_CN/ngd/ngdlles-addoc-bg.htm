A Document is any electronic file that you upload from your computer to the Content Server database. Examples of Documents include spreadsheets, memos, HTML pages, images, and audio files. Using functions, you work with Documents by adding them to Content Server, editing their contents, opening them for viewing, and managing versions and generations.

 

Note:
You must have sufficient permissions for the documents you are adding to perform these functions. For more information about setting permissions, see Setting Permissions.

 

Adding Documents

You can add Documents to Content Server by uploading a document file that already exists on your computer. You can also create a new Document.

If you have the proper permissions, you can add a Document as a major or minor Version of the original. If you do not have permission to add a Document as a Version, the option to add a Document as a major or minor Version does not appear.

Duplicate Documents

If you add a document to Content Server and if a document with the same name already exists in the destination location, you can specify how the duplicate document should be handled. For example, you can add the new document as a version of the document that is already in the destination location. If you are adding multiple documents to Content Server, you can specify that all duplicates in the batch are to be added as new versions of the existing documents. You can also specify that only new documents should be added to Content Server and that any duplicate documents should not be added.

Using the Document Overview Page

The Document Overview page provides information about the Document, including a description and related information. The Document Overview page enables you to perform a number of functions on the Document, such as editing, downloading, opening, or adding a version, if you have permissions to perform these tasks.

Editing Documents

There are several ways that you can edit a Document, including using the Edit command. If you are editing a Microsoft Office or Adobe Acrobat Document, the Document opens in the appropriate application. The item is marked as reserved in Content Server until you close the application. The current version of the Document in Content Server is updated each time you save changes.

Your level of permissions for the document that you open or edit determines which versions of the document are available to you. If you have modify permissions for a document, you will be able to access the most recent version of that document. Depending on the status of the document, that could be either a major or a minor version. If you have read-only permissions for a document, you will only be able to access the most recent major version of the document.

 

Note:
You can open some Documents for editing by right-clicking the appropriate document, and then clicking Open.

 
 

Tip:
Depending on your configuration, you may be able to open a Microsoft Office document for editing by double-clicking it or a shortcut to it in Enterprise Connect.
