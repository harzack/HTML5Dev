To search for a custom set of email properties:

  1. On the Content Server Tools menu, click Search.

  2. Under Add to Search Form, click E-mail (Advanced).

  3. In the Additional Attributes list, click the property you want to search for.

  4. Enter search criteria for the property in the text box.

  5. Optional Click an option in the Modifier list, if it appears, to further define the search. For example, you can choose to look for synonyms of the search terms.

  6. Optional To add additional sets of search criteria for a single property, click the Duplicate button beside the existing criteria, and enter the additional search criteria in the text box that appears. You must select one of OR, AND, or NOT from the list to determine how the search processes the criteria for a property. For example, if you add two sets of search criteria for a property, and click AND in the list, the search only returns results that match both sets of criteria.

    To remove an additional set of search criteria for a property, click the Remove button next to the criteria.

  7. Click Search.
