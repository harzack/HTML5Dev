Un documento de texto es un documento cuyo contenido está en formato de texto simple, por ejemplo, un documento creado en el Bloc de notas de Windows. Puede trabajar con documentos de texto utilizando muchas de las mismas funciones que se utilizan con otros tipos de documentos, por ejemplo, puede agregarlos y editar su contenido.

Dispone de un editor de texto integrado que permite crear documentos de texto. El editor de texto proporciona una manera cómoda de agregar archivos de texto simple sin tener que crearlos primero en una aplicación independiente.

Para agregar un documento con formato de texto simple existente del ordenador, debe agregarlo como un documento. Content Server lo reconoce como un archivo de texto simple y lo trata como un documento de texto. Al crear o agregar un documento de texto, puede proporcionar una descripción del documento de texto, modificar sus categorías y atributos y agregarlo a un contenedor diferente.
