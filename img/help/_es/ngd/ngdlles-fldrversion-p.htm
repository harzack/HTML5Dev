Para definir opciones de establecimiento de versiones para carpetas y proyectos:

  1. Haga clic con el botón derecho del ratón en la carpeta o el proyecto cuyas opciones de establecimiento de versiones desee definir y, luego, haga clic en Control de versión.

  2. En el área Control de versión, haga clic en una de las opciones para especificar si los usuarios pueden elegir el método o deben usar el establecimiento de versión mayor/menor.

  3. En el área Aplicar a, haga clic en una de las opciones para especificar si la configuración de Control de versión se aplica sólo al elemento actual, sólo a las carpetas y los proyectos del elemento actual o a ambos.

  4. Haga clic en Enviar.
