Puede especificar el momento en que desea recibir la solicitud para introducir metadatos acerca de los elementos que agregue a Content Server. Sólo se le puede indicar que proporcione metadatos si se requiere un atributo u otra información, o para suministrar metadatos para cada elemento que agregue a Content Server. Seleccione la opción que mejor se adapte a sus prácticas recomendadas para capturar metadatos.

 

Nota:
No se le pedirá que proporcione metadatos cuando agregue, mueva o copie contenido dentro de una instancia de Content Server.
