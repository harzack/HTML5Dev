Una lista de tareas es un contenedor que ofrece una forma sencilla tanto de gestionar los datos de asignaciones de trabajo individuales y colectivas como de realizar su seguimiento. Una lista de tareas puede contener tareas, hitos y grupos de tareas, y se puede agregar en cualquier lugar. El uso de listas de tareas para gestionar las asignaciones de un proyecto es especialmente útil, ya que permite informar del estado de una tarea a los miembros de un proyecto.

Las tareas le permiten proporcionar a un usuario o grupo de usuarios una prioridad o fecha de vencimiento para una asignación así como unas instrucciones detalladas para completar una asignación. Para ello, es necesario crear una lista de tareas antes de agregar tareas o grupos de tareas. Puede agregar tantas tareas como sean necesarias a una lista de tareas, a otra tarea o a un grupo de tareas.

Un grupo de tareas se utiliza para organizar un gran número de tareas relacionadas. Puede llevar a cabo funciones en las tareas que residen en los grupos de tareas. Además, puede agregar varias tareas a un grupo de tareas y, luego, varios grupos de tareas a una lista de tareas. También puede crear grupos de tareas específicos de cada usuario y asignar a cada uno hitos independientes.

Un hito representa una fecha objetivo en el proceso de desarrollo de un proyecto como una fecha límite para terminar ciertas tareas. Los hitos ayudan a realizar un seguimiento del estado y las fechas objetivo de todas las tareas asignadas a un hito específico. La manera más eficiente de asignar un hito a una tarea o un grupo de tareas es añadiéndolo a la lista de tareas antes de agregar la tarea o el grupo de tareas. Más adelante, también puede asignar hitos o cambiar hitos previamente asignados editando el atributo del hito.

Puede tener acceso a información específica acerca de las listas de tareas consultando las páginas de información asociadas. Visualice, por ejemplo, la página Resumen de lista de tareas, que ofrece un resumen de todos los elementos incluidos en esa lista, o la página Detalle de hitos, que suministra información sobre los hitos asociados a la lista de tareas.
