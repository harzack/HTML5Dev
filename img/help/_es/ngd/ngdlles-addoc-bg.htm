Un documento es cualquier archivo electrónico que se sube desde su ordenador a la base de datos de Content Server. Entre los ejemplos de documentos se incluyen hojas de cálculo, memorandos, páginas HTML, imágenes y archivos de audio. El uso de las funciones permite trabajar con documentos; para ello, agréguelos a Content Server, edite su contenido, ábralos para su visualización y gestione versiones y generaciones.

 

Nota:
Debe tener los permisos necesarios sobre los documentos que desee agregar para ejecutar estas funciones. Para obtener más información acerca de la configuración de permisos, consulte Configuración de permisos.

 

Adición de documentos

Puede agregar documentos a Content Server mediante la subida de un archivo de documento que ya existe en el ordenador. También puede crear un documento nuevo.

Si dispone de los permisos adecuados, puede agregar un documento como versión mayor o menor que la original. Si no dispone de permiso para agregar un documento como versión, la opción para agregar un documento como versión mayor o menor no aparece.

Duplicar documentos

Si agrega un documento a Content Server, y si en la ubicación de destino ya existe un documento con el mismo nombre, puede especificar la forma de gestionar el documento duplicado. Por ejemplo, puede agregar el nuevo documento como una versión del documento que ya se encuentra en la ubicación de destino. Si desea agregar varios documentos a Content Server, puede especificar que todos los duplicados del lote se agreguen como versiones nuevas de los documentos existentes. También puede determinar que sólo se agreguen a Content Server documentos nuevos y que no se agregue ningún documento duplicado.

Uso de la página Visión general del documento

La página Visión general del documento ofrece información acerca del documento, incluida una descripción y la información relacionada. Dicha página permite realizar una serie de funciones con el documento, como editar, descargar, abrir o agregar una versión si cuenta con los permisos necesarios para realizar estas tareas.

Edición de documentos

La edición de un documento se puede realizar de varias formas, incluido el uso del comando Editar. Si va a editar un documento de Microsoft Office o Adobe Acrobat, este se abre en la aplicación correspondiente. El elemento en cuestión se marca como reservado en Content Server hasta que cierre la aplicación. La versión actual del documento de Content Server se actualiza cada vez que guarda los cambios realizados.

El nivel de permiso que tenga para el documento que abra o edite determina las versiones del documento que están disponibles para usted. Si ha modificado los permisos para un documento, podrá acceder a la versión más reciente de dicho documento. Según el estado del documento, podría ser una versión principal u otra de menor categoría. Si tiene permisos de sólo lectura para un documento, únicamente podrá acceder a la versión principal más reciente de dicho documento.

 

Nota:
Puede abrir algunos documentos para editarlos si hace clic con el botón derecho del ratón en el documento que desee y después hace clic en Abrir.

 
 

Sugerencia:
Según su configuración, puede abrir un documento de Microsoft Office para editarlo si hace doble clic en él o en un método abreviado para él en Enterprise Connect.
