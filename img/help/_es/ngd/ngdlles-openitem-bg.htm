Para algunos tipos de elementos, el uso del comando Abrir permite ver una versión de sólo lectura del elemento o el contenido de un elemento contenedor. Puede que desee abrir un elemento, por ejemplo, si desea trabajar con él de forma local. Sin embargo, para algunas aplicaciones, el uso del comando Abrir ejecuta una versión del elemento que se puede editar. Póngase en contacto con su administrador para obtener más información acerca de cómo utilizan las aplicaciones el comando Abrir en su entorno.
