La creación de una generación crea una referencia para un lanzamiento o una revisión de un documento compuesto concreto o una versión de documento y evita que otros usuarios lo modifiquen o eliminen.

Utilice generaciones para hacer que determinadas versiones de un elemento sean más visibles y accesibles. Al igual que un método abreviado, una generación apunta a una versión concreta de un elemento, generalmente a una versión anterior. Si tiene una versión antigua de un elemento que todavía se necesite, puede crear una generación de esa versión y guardarla en cualquier contenedor o Espacio de trabajo. Esto evita que la versión señalada se modifique o se elimine.

Las generaciones se tratan como elementos distintos; no se asocian al elemento (o la versión del elemento) a partir del que se crearon. Las generaciones hacen que sea más fácil encontrar y trabajar con las versiones a las que apuntan porque se pueden guardar en cualquier contenedor adecuado.

Al crear una generación, se muestra el icono Generación del elemento.

El menú Funciones de una generación se divide en dos submenús:

  • El submenú Original contiene funciones para las que tiene permiso de ejecución en el elemento original (versión a la que apunta la generación). Las funciones del submenú Original no afectan a la generación.

  • El submenú Generación contiene funciones para las que tiene permiso de ejecución en la generación. Las funciones del submenú Generación no afectan al elemento original.
