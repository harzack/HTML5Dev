Cuando se agregan mensajes de correo electrónico a Content Server con Enterprise Connect, se aplican convenciones de nomenclatura a los mensajes. Si el administrador ha habilitado el nodo Mi configuración en el árbol de Enterprise Connect, podrá especificar reglas para nombres de los mensajes de correo electrónico. El administrador define los valores iniciales de la página Generación de nombre para correos electrónicos. En dicha página, puede hacer lo siguiente:

  • Especificar las propiedades de mensajes que se usarán al generar un nombre de Content Server para un correo electrónico.

  • Definir la cadena de separador que desea usar entre las propiedades de mensajes individuales que forman el nombre del correo electrónico.

  • Especificar cualquier cadena que desee eliminar del nombre de correo electrónico generado.

  • Especificar el formato usado para fechas y horas.

  • Especificar cómo se deben gestionar los mensajes de correo electrónico duplicados.

 

Nota:
Si Enterprise Connect genera un nombre para el correo electrónico que ya se haya usado en el contenedor de Content Server, se le pedirá que solucione el conflicto de nomenclatura antes de que poder agregar el correo electrónico. Para obtener más información sobre cómo resolver conflictos de nombres de archivos, consulte Trabajo con el correo electrónico.
