Esta aplicación permite acceder al contenido de espacios de trabajo de Content Server en la Estructura de arbol de OpenText Enterprise Connect. Según las opciones instaladas en el sistema, puede usar el complemento cuando trabaje con aplicaciones como el Explorador de Windows, Microsoft Outlook o Microsoft Office.

Esta ayuda contiene información específica de Enterprise Connect. En algunos casos, cuando el funcionamiento es idéntico en ambas aplicaciones, se hace referencia a temas incluidos en la ayuda principal de Content Server. En otros casos, dispone de acceso directo a la ayuda de Content Server desde las ventanas y los cuadros de diálogo de Enterprise Connect.
