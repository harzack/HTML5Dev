Puede crear un método abreviado si desea acceder al mismo elemento desde diferentes ubicaciones de Content Server. Por ejemplo, puede que desee crear métodos abreviados en el Espacio de trabajo personal para las carpetas a las que acceda con mayor frecuencia. Para crear métodos abreviados, puede usar el menú contextual mediante la opción Crear método abreviado o hacer clic con el botón derecho del ratón y arrastrar un elemento a un contenedor en Content Server, con la opción Crear método abreviado aquí. La opción de arrastre solo está disponible en el Explorador de Windows. Esta opción solo está disponible para tipos de contenedor compatibles, como Carpetas, Proyectos y Tareas. No está disponible para Lista de tarea o Grupos de tarea, por ejemplo. Según la configuración de su sistema, puede que deba proporcionar metadatos para el nuevo método abreviado.

Con el comando Copiar vínculo en el portapapeles, puede copiar un vínculo para un elemento al portapapeles de Windows. A continuación, puede pegar el vínculo en mensajes de correo electrónico, aplicaciones de escritorio como Word o Excel, o navegadores web. Puede elegir el tipo de vínculo que desea enviar. Las opciones disponibles dependen del tipo de elemento que haya seleccionado. Por ejemplo, puede especificar que el vínculo abra la página Visión general o Propiedades para documentos, o que el vínculo abra o descargue el documento. Para correos electrónicos, correos electrónicos compuestos, carpetas y otros tipos de elemento como discusiones, encuestas y listas de tarea, puede especificar que el vínculo abra el elemento o que se abra la página Propiedades.
