Para configurar cómo Enterprise Connect gestiona archivos adjuntos de correo electrónico con el mismo nombre:

  1. En el árbol de Enterprise Connect, haga clic en un contenedor de Content Server, Personal y, a continuación, en Mi configuración.

  2. En el área Generación de nombre para correos electrónicos, en la lista Cuando se añade un correo electrónico con el mismo nombre a una carpeta, realice una de las siguientes tareas:

    • Escoja Pedir al usuario que agregue el archivo adjunto como una nueva versión de documento para permitir a los usuarios escoger si desean agregar los archivos adjuntos que tienen el mismo nombre como un documento existente en el destino como una nueva versión del documento existente. Éste es el valor predeterminado.

    • Escoja Pedir al usuario que agregue archivos adjuntos como un nuevo documento con un sufijo único para permitir a los usuarios escoger si desean agregar archivos adjuntos que tienen el mismo nombre como un documento existente en el destino como un nuevo documento con el mismo nombre y un sufijo numérico para distinguirlo del documento existente.

    • Escoja Agregar automáticamente el archivo adjunto como una nueva versión de documento para agregar automáticamente los archivos adjuntos que tienen el mismo nombre como un documento existente en el destino como una nueva versión del documento existente.

    • Escoja Agregar automáticamente archivos adjuntos como un nuevo documento con un sufijo único para agregar automáticamente archivos adjuntos que tienen el mismo nombre como un documento existente en el destino como un nuevo documento con el mismo nombre y un sufijo numérico para distinguirlo del documento existente.

  3. Haga clic en Enviar para aplicar los cambios.

     

    Sugerencia:
    Haga clic en Restablecer para revertir a los valores guardados más recientemente para gestionar los archivos adjuntos de correo electrónico con nombres duplicados. Haga clic en Utilizar los valores predeterminados para revertir a la configuración proporcionada por el administrador.
