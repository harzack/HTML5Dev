Puede crear un elemento de noticias para un canal, lo que le permite publicar dicho elemento para otros usuarios como historia de una noticia. Por ejemplo, puede que desee publicar una lista de nuevos clientes como elemento de noticias.

Puede agregar elementos de noticias a cualquier canal en la página Noticias del Espacio de trabajo personal.

Un elemento de noticias consta de un titular y una historia así como de fechas de efectividad y de caducidad para asegurarse de que las noticias son puntuales. También puede contener una imagen y archivos adjuntos. El icono Anexo  indica que un elemento de noticias incluye un archivo adjunto.

El usuario accede a los elementos de noticias y trabaja con ellos desde la página de resumen del canal en el que residen.

 

Nota:
Los elementos de noticias pueden tener varios archivos adjuntos. Puede agregar un archivo adjunto de Content Server y otro del ordenador la primera vez que cree un elemento de noticias pero puede agregar más archivos adjuntos más adelante editando el elemento de noticias.

 

Al agregar noticias a un canal, puede proporcionar una descripción del elemento de noticias, modificar sus categorías y atributos y agregar la noticia a un contenedor diferente.
