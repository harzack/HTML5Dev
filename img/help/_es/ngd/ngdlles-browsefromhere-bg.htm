Si el administrador ha habilitado esta función, podrá usar el menú contextual Examinar desde aquí para iniciar una nueva ventana que contenga la interfaz de usuario web de Content Server o la página Visión general para el artículo seleccionado. Esta función está disponible tanto en el árbol de Enterprise Connect como en el Panel de lista. Para poder usar esta función, debe disponer de los permisos adecuados para el elemento. El menú contextual Examinar desde aquí está disponible para elementos entre los que se incluyen:

  • Documentos.

  • Mi espacio de trabajo y la carpeta de empresa.

  • Todos los tipos de contenedor como carpetas, carpetas de correo electrónico y colecciones.

  • Contenedores que se encuentran en las carpetas Favoritos y Personal, así como contenido incluido en carpetas virtuales.

  • Tipos distintos a contenedores y documentos como generaciones, prospectores, sondeos, métodos abreviados, direcciones URL, esquemas de workflow, estados de workflow, categorías, vistas personalizadas, LiveReports y archivos DTD de XML.

Por ejemplo, si hace clic con el botón derecho en Mi espacio de trabajo y selecciona Examinar desde aquí, se abre una nueva ventana, que contiene la vista de la interfaz de usuario web de Mi espacio de trabajo . Si selecciona Examinar desde aquí para un método abreviado, podrá ver la Visión general del elemento original.
