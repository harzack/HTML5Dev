Las categorías agrupan atributos relacionados y definen su tipo, su orden y los valores que los usuarios proporcionarán para ellos. Al crear categorías personalizadas, puede agregar metadatos pertinentes para su organización a los elementos de la base de datos de Content Server.

Para poder crear y modificar categorías, debe tener lo siguiente:

  • Privilegio de creación de objetos para las categorías.

  • Permiso para agregar elementos al contenedor o al Espacio de trabajo donde desee almacenar la categoría.

Puede añadir una categoría a prácticamente cualquier contenedor o Espacio de trabajo donde tenga permiso para añadir elementos. Sin embargo, la mayoría de las veces las categorías se almacenan en un volumen de categorías especial, que las pone a disposición de todos los usuarios. No obstante, al agregar una categoría a otra ubicación, como una carpeta, puede ponerla a disposición de un subconjunto más pequeño de usuarios y definir permisos únicos para él.

 

Nota:
Para tener acceso al volumen de categorías, debe tener el privilegio Permisos de administración del sistema. Con este privilegio, también puede agregar carpetas de categorías al volumen para ayudar a organizar las categorías. El privilegio Permisos de administración del sistema también permite agregar una categoría a la Espacio de trabajo.

 

Incluso si agrega un único atributo personalizado, debe agregar una categoría donde guardarlo. Después de crear una categoría, puede editar sus atributos de varias formas como cambiar el atributo de bloqueado a desbloqueado, cambiar el número de filas y agregar o quitar atributos. Puede modificar los valores predeterminados, la secuencia y el esquema de atributos de una categoría.

Una vez que haya agregado una categoría, constrúyala mediante varios tipos de atributo disponibles. Defina los tipos de atributos que desea que contenga la categoría, defina sus características y diseñe su secuencia y esquema. Debe agregar al menos un atributo a cada categoría, pero puede ser tantos como necesite. Para obtener más información acerca de cómo trabajar con los atributos, consulte Trabajo con categorías.
