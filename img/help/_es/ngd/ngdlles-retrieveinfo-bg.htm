Se pueden realizar búsquedas en Content Server con objeto de localizar documentos, carpetas, discusiones y otros elementos. Puede configurar el número de resultados de búsqueda que se mostrarán por página. Si dispone de permisos suficientes, puede crear prospectores que automaticen el proceso de búsqueda.

 

Sugerencia:
Si realiza una búsqueda mediante la búsqueda rápida de Enterprise Connect, obtendrá los mismos resultados de búsqueda que con la búsqueda web de Content Server. Para obtener más información sobre la búsqueda en Enterprise Connect, consulte OpenText Enterprise Connect - User Getting Started Guide (NGDCORE-UGD).
