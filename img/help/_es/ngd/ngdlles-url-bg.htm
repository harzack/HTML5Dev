Las direcciones de Internet (URL) son parecidas a los favoritos o los marcadores de un navegador web. Al abrir una dirección de Internet (URL), se pasa esa dirección de Internet al navegador web que, entonces, muestra la página web.

Puede guardar los elementos siguientes como direcciones URL:

  • Direcciones de World Wide Web

  • Sitios FTP

  • Direcciones de correo electrónico

Aunque puede utilizar una dirección de Internet (URL) para enlazar a otro elemento de Content Server, es preferible utilizar un método abreviado.
