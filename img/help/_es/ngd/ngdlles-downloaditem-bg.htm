Según la configuración, puede descargar elementos a su ordenador para trabajar con ellos en forma local. Se recomienda reservar los documentos que descargue para garantizar que otros usuarios no puedan modificar el elemento mientras lo utiliza. Para obtener más información sobre la reserva y cancelación de la reserva de elementos, consulte Reserva y cancelación de reservas de elementos. Asimismo, puede usar el comando Enterprise Connect Descargar versión sin conexión para trabajar con elementos fuera de línea. Para obtener más información, consulte "Downloading Offline Versions " en OpenText Enterprise Connect - User Getting Started Guide (NGDCORE-UGD).
