Puede configurar permisos para definir qué usuarios o grupos pueden ver elementos, modificar sus atributos para o trabajar con ellos. Por ejemplo, puede que desee otorgar a su asistente permisos para trabajar con documentos que agregue a su carpeta o proyecto.

Content Server define automáticamente los permisos para un elemento en función de dónde se cree el elemento pero puede modificar la configuración desde la página Permisos. También puede asignar o editar permisos de elementos almacenados en canales, discusiones o listas de tareas al hacer clic con el botón derecho en el elemento apropiado y hacer clic en Permisos.

Los permisos para los usuarios se almacenan en una lista de acceso. Al crear un elemento, se agregan automáticamente las siguientes entradas de manera predeterminada a la lista de acceso:

  • Propietario: usuario que creó el elemento.

  • Grupo propietario: grupo de trabajo del propietario del elemento.

  • Acceso público: todos los usuarios.

No puede eliminar ninguna entrada predeterminada de la lista de acceso, pero puede modificar sus permisos.

Debe tener los permisos necesarios para ejecutar esta función.
