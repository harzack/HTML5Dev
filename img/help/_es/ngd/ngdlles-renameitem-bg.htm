Puede cambiar el nombre de un elemento como una carpeta o un documento a fin de describir el tipo de información que contiene. Por ejemplo, puede que desee cambiar el nombre de una carpeta si cambia el tipo de documentos que almacena.

Debe tener suficientes permisos y trabajar en línea para cambiar el nombre de los elementos.
