Un proyecto es un contenedor que permite a los participantes de un grupo almacenar información y elementos y comunicar actividades diarias a otros participantes. Los participantes del proyecto pueden agregar, compartir, gestionar y ordenar elementos dentro de un proyecto, como tareas y documentos.

Una plantilla de proyecto especifica un conjunto predefinido de elementos basado en un proyecto existente. Puede crear una plantilla de proyecto si desea utilizar el conjunto de atributos del proyecto existente para crear nuevos proyectos.

Desde la carpeta Proyecto tiene acceso a varias páginas de información que facilitan la gestión de los proyectos.
