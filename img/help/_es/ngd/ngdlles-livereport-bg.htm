LiveReports proporciona acceso a la información de la base de datos de la organización y permite buscar, organizar y presentar la información almacenada en ella. El acceso a la base de datos mediante LiveReport es distinto que mediante Content Server ya que puede consultar cualquier parte del esquema de base de datos en lugar de únicamente la parte de la base de datos relacionada con los elementos de Content Server. LiveReports obtiene la información ejecutando sentencias del lenguaje de consulta estructurado (SQL) en la base de datos. Puesto que las sentencias SQL pueden modificar o eliminar datos críticos, el permiso para agregar o editar LiveReports está restringido de manera predeterminada al administrador. Al abrir un LiveReport, se ejecuta automáticamente la sentencia SQL y otros parámetros. Para abrir un LiveReport, debe tener el permiso Ver contenido en dicho LiveReport.

Puede crear LiveReport y colocarlos en cualquier carpeta. Además, el volumen de LiveReport alberga un conjunto de LiveReport de manera predeterminada. Todos los LiveReport predeterminados que tenga permiso para ver aparecen en la carpeta Informes, ubicada en la carpeta Personal.
