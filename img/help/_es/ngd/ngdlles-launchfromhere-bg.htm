Si el administrador ha habilitado esta función, puede acceder a Enterprise Connect cuando trabaje con tipos de contenido especificados en la interfaz web de Content Server. Cuando esta función está disponible, puede acceder al menú Función para un elemento admitido, elegir Enterprise Connect desde aquí, y se abrirá una nueva instancia de Enterprise Connect en la que se mostrará el elemento seleccionado. El administrador determina para qué subtipos está disponible esta función. De forma predeterminada, esta acción se denomina Enterprise Connect desde aquí, aunque el administrador puede crear una etiqueta personalizada para esta función. Póngase en contacto con su administrador para obtener más información sobre esta función.

 

Nota:
Dicha función solo está disponible para instancias de la interfaz web de Content Server en el Explorador de Internet.
