You can select which nodes appear under the Server Node in the Enterprise Connect Tree Structure. You can also add items to the Volumes folder and the Other Items folder. In addition, you can change the order in which the items appear in the Tree Structure. A default configuration is provided, but you can modify this if required.

 

Note:
If you are using Domain Workspaces, you must move it from the Available Items list to the Server Node list to make it available to users. The Domain Workspace then becomes visible to the Enterprise Client tree and is for the Domain that the user is a member of.

 

If you want Enterprise Connect users to be able to access the Public Email Volume, their Personal Email Volume, and any delegated Email Volumes in the Enterprise Connect tree, move the Email Volume node from the Available Items list to the Server Node list. The Email Volume node only appears in the list of Available Items if you have successfully installed and configured support for Email Archiving.

 

Note:
Enterprise Connect users with administrative privileges will also be able to access the System Email Volume.

 

You can also specify which items appear under the Personal node in the Enterprise Connect Tree Structure. Some items appear by default, and you can add or remove items if required. Available items include those that appear in the Web UI Personal menu. However, the available items may vary depending on the Content Server modules installed in your environment. For some items, you may see a Web UI view.
