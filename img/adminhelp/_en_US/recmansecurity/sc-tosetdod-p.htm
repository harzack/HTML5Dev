To set DoD settings:

  1. On the Content Server Administration page, click the Configure system for Security Clearance Levels and/or Supplemental Markings link in the Security Clearance Administration section.

  2. On the Configure system for Security Clearance Levels and/or Supplemental Markings page, click one of the following radio buttons in the Configuration Settings section:

    • Supplemental Markings and Security Clearance Levels, which lets you set the Security Clearance module to have no DoD settings

    • Supplemental Markings for DoD 5015.2 Chapter 2, which lets you set the Security Clearance module to have the DoD requirements outlined in chapter 2 of the DoD 5015.2 Standards

    • Supplemental Markings and Security Clearance Level for DoD 5015.2 chapter 2 & 4, which lets you set the Security Clearance module to have the DoD requirements outlined in chapter 4 of the DoD 5015.2 Standards

    Optionally, type a value in the Declassify On Time Frame field to set the number of years past the record date that an object is declassified if the Supplemental Markings and Security Clearance Level for DoD 5015.2 chapter 2 & 4 setting is enabled.

  3. Click the Submit button.

 

Note:
If you enable the Supplemental Markings and Security Clearance Levels radio button in the Configuration Settings section, the supported objects table is pre-populated with default supported object types based on the modules that you have installed on your system. For information about supported object types, see Configuring Supported Object Types.

If you typed a new value in the Declassify On Time Frame field, you can return to the Configure System for Security Clearance Levels and/or Supplemental Markings page and click the Recalculate Declassify on Date button to update all records using the updated value.
