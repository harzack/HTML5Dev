Security Clearance lets you add and edit security clearance levels and assign users to the security clearance levels you add.

Security clearance level are hierarchical codes you can assign to objects and users. They are single attributes that are attached to each object and user, and filter which objects a user can view. A user with a high security clearance level can see an object that has a lower security clearance level, but a user with a lower security clearance level cannot see an object that has a higher security clearance level.

When editing a security clearance level, the name can be modified at any time, but the security clearance code may never be changed.

Security clearance level functionality is not available if the DoD setting is set to DoD Supplemental Markings. For more information about DoD settings, see Configuring Security Clearance Settings.
