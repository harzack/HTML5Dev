To edit a security clearance level:

  1. On the Content Server Administration page, click the Security Clearance Levels and Supplemental Markings Management link in the Security Clearance Administration section.

  2. On the Security Clearance Levels and Supplemental Markings Management page, click the Edit link for a security clearance level.

  3. On the Edit Security Clearance Levels and Supplemental Markings page, modify any of the settings.

  4. Click the Update button.

 

Note:
You can edit only the name and description of a security clearance level.
