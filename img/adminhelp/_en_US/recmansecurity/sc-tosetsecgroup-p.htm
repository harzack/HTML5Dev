To set a Security Clearance Officers group:

  1. Click the Configure system for Security Clearance Levels and/or Supplemental Markings link in the Security Clearance Administration section of the Content Server Administration page.

  2. On the Configure system for Security Clearance Levels and/or Supplemental Markings page, click the Select Group button.

  3. On the Select a User page, click Group Name in the Find drop-down list, specify any other criteria, and then click the Find button.

  4. Click the Select link for a group.

  5. Click the Submit button.
