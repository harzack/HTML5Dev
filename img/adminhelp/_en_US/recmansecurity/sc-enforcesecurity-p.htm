To enforce Security Levels and Supplemental Markings for System Administrators

  1. Click the Configure system for Security Clearance Levels and/or Supplemental Markings link in the Security Clearance Administration section of the Content Server Administration page.

  2. On the Configure system for Security Clearance Levels and/or Supplemental Markings page, click the Yes radio button in the Enforce Security Levels and Supplemental Markings for System Administrators section.

  3. Click the Submit button.
