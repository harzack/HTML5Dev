To configure multi-select buttons in Browse View:

  1. On the Content Server Administration page, click the Configure system for Security Clearance Levels and/or Supplemental Markings link in the Security Clearance Administration section.

  2. In the Security Clearance Actions in Browse View section of the Configure system for Security Clearance Levels and/or Supplemental Markings page, select the check box for each multi-select action you want users to see when they are in the Browse View.

  3. Click Submit.
