Security Clearance lets you add and edit supplemental markings with or without a DoD setting enabled. You can also add users to the supplemental markings you create. For more information about DoD settings, see Configuring Security Clearance Settings.

Supplemental markings are non-hierarchical codes that filter which objects users can see. A user or an item in Content Server can have multiple supplemental markings. Users can see objects if they have all the same supplemental markings as the object.

You can assign User Session Parameter codes to Supplemental Markings to provide additional permissions filters. Certain rules apply to what users are able to see when User Session Parameter codes are assigned to Supplemental Markings. For more information, see Configuring User Session Parameter Codes.
