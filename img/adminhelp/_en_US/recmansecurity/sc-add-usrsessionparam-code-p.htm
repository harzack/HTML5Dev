To add User Session Parameter codes:

  1. On the Content Server Administration page, click the Configure User Session Parameter Codes link in the Security Clearance Administration section.

  2. On the User Session Parameters page, click User Session Parameter in the Add Item list.

  3. On the Add User Session Parameter page, type a code for the User Session Parameter in the Code field.

  4. Optional Type a description for the code in the Description field.

  5. Click Add.
