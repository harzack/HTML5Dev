To edit a supplemental marking:

  1. On the Content Server Administration page, click the Security Clearance Levels and Supplemental Markings Management link in the Security Clearance Administration section.

  2. On the Security Clearance Levels and Supplemental Markings Management page, click the Edit link for a supplemental marking code.

  3. On the Edit Security Clearance Levels and Supplemental Markings page, modify the description, or select or clear the Restricted Data Code or Formerly Restricted Data Code check boxes.

  4. In the User Session Parameter area, use the arrows to select or remove User Session Parameters to be associated with Supplemental Marking.

     

    Note:
    You can click the View Details or Hide Details links to show or hide the User Session Parameters associated with this Supplemental Marking. If you toggle between the two options, any changes made to the metadata fields are maintained.

    If any of the values are selected, users with the Supplemental Marking must have at least one of the values passed with the user session to see the content with the Supplemental Marking.

     

  5. Click the Update button.
