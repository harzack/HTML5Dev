To add a supported object type:

  1. On the Content Server Administration page, click the Configure Supported Object Types link in the Security Clearance Administration section.

  2. On the Configure Supported Object Types page, click Object Type in the Add Item menu.

  3. On the Add Supported Object Type page, type a subtype code in the Object Type field. Optionally, select the Inherit From check box to have child objects inherit supplemental markings and security clearance levels from parent objects.

  4. Click the Submit button.

 

Note:
The Inherit From check box displays only if DoD settings are disabled. For information on DoD settings, see Configuring Security Clearance Settings.
