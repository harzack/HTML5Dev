To add users to a security clearance level:

  1. On the Content Server Administration page, click the Security Clearance Levels and Supplemental Markings Management link in the Security Clearance Administration section.

  2. On the Security Clearance Levels and Supplemental Markings Management page, click the Add/Remove Users link for a security level code.

  3. On the Set Security Clearance Level for Users page, specify any search criteria, and then click the Find button.

  4. Select the Add/Remove check box for each user you want to assign.

  5. Click the Update button.
