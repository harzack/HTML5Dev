To configure Copy and Move Inheritance Options:

  1. On the Content Server Administration page, click the Configure system for Security Clearance Levels and/or Supplemental Markings link in the Security Clearance Administration section.

  2. In the Hide the Option to Inherit Security Levels and Supplemental Markings on Copy and Move section of the Configure system for Security Clearance Levels and/or Supplemental Markings page, click the Yes radio button to allow users to choose to inherit the Security Level and Supplemental Markings on items they copy or move.

  3. In the Inherit Security Levels and Supplemental Markings on Copy and Move section, select a default option for the Supplemental Markings and Security Level parameters.

  4. Click Submit.
