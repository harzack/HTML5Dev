To send test messages:

  1. On the administration page, click the Configure System Object E-Mail Delivery link.

  2. On the Configure System Object Alert E-Mail Delivery page, do one of the following:

    • Click the Add a User icon , and then configure the recipient's email address and message type.

    • Click the Edit this user icon beside the configuration for the recipient to whom you want to send a test message.

  3. On the Configuration for the User page, click the Send Test Message button.

  4. Click the Update button.
