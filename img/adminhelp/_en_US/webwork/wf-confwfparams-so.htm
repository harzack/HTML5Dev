You can control the behavior of workflows by administering workflow features in the Workflow Administration area on the Content Server Administration page. You can administer the following workflow features:

  • Settings that control how users view workflows.

  • Access to the Workflow Agent, which can automatically processes some workflow steps, and settings that control how it works.

  • Access to the Content Server Volume where Documents and other information used in the workflow are stored.
