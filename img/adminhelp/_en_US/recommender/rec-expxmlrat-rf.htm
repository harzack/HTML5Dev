To export XML data from the Ratings tab of an item's Properties page, append &outputxml=true to the end of the URL for the page. For example, enter the following URL:

<protocol>://<host>/<URL_prefix>/.exe?func=ll&objid=<id>&objAction=Ratings&outputxml=true

where:

  • <protocol> is either http or https,

  • <host> is your fully qualified host address (for example, host.mycorp.com),

  • <URL_prefix> is the virtual directory shortcut mapped to the <Content Server_home>/cgi directory in the HTTP server, and

  • <id> is the node ID of the item.

This command outputs the Ratings tab of the item's Properties page for the user who is currently logged in to the server.

The DTD that defines this XML output is included in installations as <Content Server_home>/module/recommender_<x>_<x>_<x>/config/recommender_xml_output.dtd, where <x>_<x>_<x> is the version number of the Recommender module. For example, to view the file for the Recommender 10.5.0 module, see <Content Server_home>/module/recommender_10_5_0/config/recommender_xml_output.dtd.
