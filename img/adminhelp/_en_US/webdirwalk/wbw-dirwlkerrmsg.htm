The following table describes the error messages that may appear when constructing Directory Walker data flows.

If these errors persist, contact OpenText Customer Support for additional information.

Table 1-25: Directory Walker Error Messages

Number Message Description Remedy
1 Content Server core error An internal error occurred when Content Server attempted to create or set parameters in the configuration file for a Directory Walker process. Contact OpenText Customer Support.
2 Unable to read a configuration parameter Content Server encountered an error while reading a parameter in the configuration file for the Directory Walker process. Check the configuration file to ensure that all parameters have valid values. Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line for a Directory Walker process on the Specific tab of its Properties page.
3 Unable to set a configuration parameter Content Server encountered an error while setting a parameter in the configuration file for the Directory Walker process. Check the values that you specified when creating or updating the Directory Walker process. If you are updating a Directory Walker process, you can check the values specified on the Specific tab of its Properties page.
4 Unable to connect to the Admin server Content Server could not connect to the Admin server that manages the Directory Walker process. Check the Admin server that is associated with the Directory Walker process and ensure that it is available. If you are updating an existing Directory Walker process, you can change the Admin server by clicking a value in the Admin Server drop-down list on the Specific tab of its Properties page.
5 Unable to initialize the configuration file Content Server could not initialize the configuration file for the Directory Walker process. Check your system to ensure that adequate memory resources are available.
6 Unable to find the configuration file Content Server could not find the configuration file for the Directory Walker process. Check the configuration file's name to ensure that it matches the configuration file name that is specified in the command line for the Directory Walker process. Ensure that the configuration file is not currently in use, and check the file's permissions to ensure that the Directory Walker process can read data from it
Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line for a Directory Walker process on the Specific tab of its Properties page.
7 Unable to write data to the configuration file Content Server could not write data to the configuration file for the Directory Walker process. Check the configuration file's permissions to ensure that the Directory Walker process can write data to it. You can also check your system to ensure that adequate disk space is available and that the configuration file is not currently locked
Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line for a Directory Walker process on the Specific tab of its Properties page.
8 Unable to delete the configuration file Content Server could not delete the configuration file for the Directory Walker process. Check the configuration file name to ensure that it matches the configuration file name that is specified in the command line for the Directory Walker process. Ensure that the configuration file is not currently in use, and check the file's permissions to ensure that the Directory Walker process can write data to it.
Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line for a Directory Walker process on the Specific tab of its Properties page.
9 Unable to set the write directory Content Server could not set the write directory for the Directory Walker process. The write directory is the directory in which the data flow's read and write IPools are created. Ensure that you specified a valid directory path in the Write data to field when you created the Directory Walker process. If you are updating an existing Directory Walker process, you can change the write directory by clicking the Information button on the Specific tab of the Directory Walker Properties page, and then setting a new value in the Write field.
10 Unable to set the write area Content Server could not set the write area for the Directory Walker process. Contact OpenText Customer Support.
11 Unable to set the log file Content Server could not set the log file for the Directory Walker process. Ensure that a valid directory path and file name is specified for the log file. If you are updating an existing Directory Walker process, you can specify the absolute path and file name of the log file in the Log File field on the Specific tab of the Directory Walker Properties page.
12 Unable to set the log level Content Server could not set the log level for the Directory Walker process. Ensure that a valid log level is specified for the Directory Walker process. If you are updating an existing Directory Walker process, you can specify a log level in the Log Level drop-down list on the Specific tab of the Directory Walker Properties page.
14 Unable to set the crawl history database location Content Server could not set the crawl history database location for the Directory Walker process. The crawl history database location is the directory where you want the Directory Walker process to write its history files. Ensure that you specified the absolute path of a valid directory when you created or updated the Directory Walker process. OpenText recommends that you specify the same directory that you specify for the write directory of the Directory Walker process.
20 Unable to set the Disable parameter in the configuration file Content Server could not set the value of the Disable parameter in the configuration file for the Directory Walker process. The Disable parameter allows Content Server to temporarily remove particular directory groups from the list that the Directory Walker scans. Contact OpenText Customer Support.
21 Unable to set the Traversal parameter in the configuration file Content Server could not set the Traversal parameter in the configuration file for the Directory Walker process. The Traversal parameter works in conjunction with the value that you specify in the Depth drop-down list when creating the Directory Walker process to determine whether the Directory Walker scans directories recursively. Contact OpenText Customer Support.
22 Unable to set the directory depth Content Server could not set the depth of the directories that the Directory Walker process scans. Ensure that you specified a valid value in the Depth drop-down list when you created or updated the Directory Walker process. The following list describes the values in the Depth drop-down list:
•

Unlimited, which specifies that the Directory Walker scans all subdirectories below the top-level directories

•

Specify, which allows you to specify a non-negative number representing the exact number of subdirectory levels for the Directory Walker to scan

•

none, which specifies that the Directory Walker process scans the specified directory only

If you are updating an existing Directory Walker process, you can change the depth by clicking a value in the Depth drop-down list on the Specific tab of the Directory Walker Properties page.
23 Unable to set the minimum file size Content Server could not set the minimum required size of the files that the Directory Walker process scans. Ensure that you specified a non-negative integer value in the appropriate File Size field when you created the Directory Walker process. The minimum file size must be less than or equal to the maximum file size. If you are updating an existing Directory Walker process, you can change the file size by setting new values in the File Size fields on the Specific tab of the Directory Walker Properties page.
24 Unable to set the maximum file size Content Server could not set the maximum size of the files that the Directory Walker scans. Ensure that you specified a non-negative integer value in the appropriate File Size field when you created the Directory Walker process. This value must be greater than or equal to the minimum required size. If you are updating an existing Directory Walker process, you can change the file size by setting new values in the File Size fields on the Specific tab of the Directory Walker Properties page.
25 Unable to set the date of the oldest files that the Directory Walker process scans Content Server could not set the date of the oldest files that the Directory Walker scans. This is the first value in the date range that you specified for the directory group. Ensure that you specified a valid date in the appropriate Date Range drop-down lists when you created or updated the Directory Walker process. The date of the oldest files must be less than or equal to the date of the newest files in the range. If you are updating an existing Directory Walker process, you can change the date range by setting new values in the Date Range drop-down lists on the Specific tab of the Directory Walker Properties page.
26 Unable to set the date of the newest files that the Directory Walker process scans Content Server could not set the date of the newest files that the Directory Walker process scans. This is the second value in the date range that you specified for the directory group. Ensure that you specified a valid date in the appropriate Date Range drop-down lists when you created the Directory Walker process. The date of the newest files must be greater than or equal to the date of the oldest files in the range. If you are updating an existing Directory Walker process, you can change the date range by setting new values in the Date Range drop-down lists on the Specific tab of the Directory Walker Properties page.
28 Unable to set the top-level directories Content Server could not specify the top-level directories that you want the Directory Walker process to scan. Ensure that you specified the absolute paths of the top-level directories in the Directories field when you created the Directory Walker process, separating each directory path with a semicolon or carriage return. If you are updating an existing Directory Walker process, you can change the directory paths by setting new values in the Directories fields on the Specific tab of the Directory Walker Properties page.
29 Unable to specify the types of files that you want to include Content Server could not specify the list of file types that you want the Directory Walker process to scan when it walks the specified directories. Ensure that you specified a valid list of file name patterns that you want the Directory Walker to scan (for example, *.html; *.txt; *.doc; and so on). You specify file name patterns in the Include field when you create the Directory Walker process. If you are updating an existing Directory Walker process, you can change the file name patterns by setting new values in the Include field on the Specific tab of the Directory Walker Properties page.
30 Unable to specify the types of files that you want to exclude Content Server could not specify the list of file types that you do not want the Directory Walker process to scan when it walks the specified directories. Ensure that you specified a valid list of file name patterns that you do not want the Directory Walker to scan (for example, *.html; *.txt; *.doc; and so on). You specify these file name patterns in the Exclude field when you create the Directory Walker process. If you are updating an existing Directory Walker process, you can change the file name patterns by setting new values in the Exclude field on the Specific tab of the Directory Walker Properties page.
31 Unable to set the name of the crawl history database location Content Server could not associate a valid name with the crawl history database location that you specified. This name is the name that you specify in the Processes Prefix field when you create the Directory Walker process. Contact OpenText Customer Support.
100 Invalid command line arguments specified The Directory Walker process failed to start because some of its command line arguments are invalid. Check the command line arguments that are specified in the Command Line field on the Specific tab of the Directory Walker Properties page. If you discover errors (for example, invalid syntax), stop the process and correct them by editing the Command Template field. Then run the Directory Walker process again.
101 Unable to read the configuration file The Directory Walker process could not read the configuration file or some of its settings. Check the configuration file's permissions to ensure that the Directory Walker process can read data from it. You can also check your system to ensure that adequate disk space is available and that the configuration file is not currently locked. Finally, ensure that the configuration file is stored in the location specified in the command line.
Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process.
If you suspect that the configuration file contains invalid parameters, check the command line arguments that are specified in the Command Line field on the Specific tab of the Directory Walker Properties page. If you discover errors (for example, invalid syntax), stop the process and correct them by editing the Command Template field. Then run the Directory Walker process again.
102 Unable to open the log file The Directory Walker process could not open the log file in which it records data. Check the log file's permissions to ensure that the Directory Walker process can write data to it. You can also check your system to ensure that adequate disk space is available and that the log file is not currently locked. Finally, ensure that the log file is stored in the location specified in the Log File field on the Specific tab of the Directory Walker Properties page. By default, Directory Walker log files are stored in the Content Server_home/logs directory.
103 Unable to read the log level The Directory Walker process could not read the log level specified in the configuration file. Ensure that the log level specified on the Specific tab of the Directory Walker Properties page matches the log level specified in the configuration file. Valid values include:
•

Default, which reports errors only

•

Verbose, which reports the status of messages flowing through the processes and IPools of the data flow as well as errors

•

Debug, which reports highly detailed information about data flow status and errors

Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line arguments on the Specific tab of the Directory Walker Properties page.
104 Insufficient Directory Walker information The Directory Walker process failed to start because you have not specified a write directory or a directory to scan. Check the Specific tab of the Directory Walker Properties page to ensure that you have specified a write directory for the Directory Walker process and that you have specified the directories that you want the Directory Walker process to scan. Verify that the values specified on the Specific tab of the Directory Walker Properties page match the values specified in the configuration file.
Directory Walker configuration files are stored in the Content Server_home/config directory and have the .dw file extension. The name of a particular Directory Walker configuration file is specified in the command line for the process. You can view the command line arguments on the Specific tab of the Directory Walker Properties page.
105 Unable to set up the stop message port The Directory Walker process could not set up the stop message port. The stop message port is a port on which Content Server sends a shut down message to stop the Directory Walker process. Ensure that a valid port number is specified in the command line on the Specific tab of the Directory Walker Properties page. The port number that you specify must not be used by any other process and must be between 1025 and 65535. You can change the port number by typing a new value in the Stop Message Port field on the Specific tab of the Directory Walker Properties page.
106 Unable to read the write directory The Directory Walker process could not read the write directory in the configuration file. The write directory is the directory in which the data flow's read and write IPools are created. Ensure that a valid directory path is specified in the Write field, which you access by clicking the Information button on the Specific tab of the Directory Walker Properties page. Then ensure that the same directory path is specified in the configuration file (IPool identifier).

107 Unable to make an IPool connection The Directory Walker process could not connect to an IPool. Ensure that a valid directory path is specified in the Write field, which you access by clicking the Information button on the Specific tab of the Directory Walker Properties page. Then ensure that the same directory path is specified in the configuration file.

108 An internal IPool error has occurred The Directory Walker process encountered an error when trying to start an IPool transaction. Contact OpenText Customer Support.

116 Unable to rename the crawl history database files Content Server could not rename the crawl history database files. In normal circumstances, there are three crawl history database files, named name.new name.old, and name.junk. When a new crawl history database file is generated, the Directory Walker process renames the name.new file to name.old. At the same time, the Directory Walker process renames the name.old file to name.junk. Check the permissions of the crawl history database files to ensure that the Directory Walker process can rename them. You can also check your system to ensure that adequate disk space is available and that the crawl history database files are not currently locked. Finally, ensure that the crawl history database files are stored in the directory specified in the Crawl History Database Location field on the Specific tab of the Directory Walker Properties page. By default, crawl history database files are stored in the write directory for the process.
117 Unable to create a new crawl history database file The Directory Walker process could not create a new crawl history database file (name.new). In normal circumstances, there are three crawl history database files, named name.new name.old, and name.junk. When a new crawl history database file is generated, the Directory Walker process renames the name.new file to name.old. At the same time, the Directory Walker process renames the name.old file to name.junk. Ensure that the directory where the crawl history database files reside exists, and then check the directory's permissions to ensure that the Directory Walker process can write data to it. You can also check your system to ensure that adequate disk space is available and that the directory is not currently locked.
118 Unable to read a directory group The Directory Walker process failed to read the scanning information that is associated with a directory group. Check the values that are specified for each directory group on the Specific tab of the Directory Walker Properties page. Ensure that valid information is provided in all of the appropriate fields, and that the information matches the information in the configuration file. Then run the Directory Walker process again.

You can view the command line arguments on the Specific tab of the Directory Walker Properties page.
119 An internal IPool error has occurred The Directory Walker process encountered an internal error of an unspecified nature. Contact OpenText Customer Support.
