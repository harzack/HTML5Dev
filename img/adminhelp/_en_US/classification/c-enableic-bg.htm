The Intelligent Classification functionality allows you to create assisted and automatic Classifications that help to automate the process of classifying items. For each of these classification types, you construct a profile that defines the criteria that its items must meet. Then, for each new item that meets the criteria, Content Server either suggests the Classification to be assigned or automatically assigns the Classification to the item. For more information about assisted and automatic Classifications, see Adding Classifications.
