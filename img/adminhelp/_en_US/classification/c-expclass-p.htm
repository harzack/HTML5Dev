To export a Classification taxonomy:

  1. On the Content Server Administration page, in the Classifications Administration section, click the Taxonomy Export link.

  2. On the Taxonomy Export page, click the Browse Content Server button, and then do one of the following:

    • Click the Select link for the Classification Workspace to export all Classification Trees in the Classification Workspace.

    • Navigate to a Classification Tree or Classification, and then click the appropriate Select link.

  3. Type a name for the XML file in the Export File Name field (for example, classtree.xml).

  4. Click the Export button.

  5. On the Taxonomy Export page, click the OK button when the export process is complete.

 

Note:
If you click the OK button before the export process is complete, you will return to the Content Server Administration page and the export process continues to run in the background.
