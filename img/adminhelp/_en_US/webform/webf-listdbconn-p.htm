To modify database connection properties:

  1. On the Web Forms Connections page, click the database name link to view and modify the database connection's properties.

  2. Click Add.

 

Notes:

  • Once a database connection is created, you cannot change the RDBMS Server type.

  • The Dependent Lookups field displays regardless of the database system you select. You can view names of all current Database Lookups that require the current database connection. You cannot modify any Database Lookups on this page. If you want to modify a Database Lookup see, To Modify Web Forms Lookup Properties.

 

For information related to this procedure, see Managing Secure Database Connections.
