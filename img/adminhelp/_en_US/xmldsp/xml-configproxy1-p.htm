1. On the Content Server Administration page in the HTTP Messaging Administration section, click the Configure HTTP Proxy link.

  2. On the Specify HTTP Proxy page, in the Address box, type the IP address that the XML Workflow Interchange step will use to access the HTTP Proxy. For example, 127.0.0.1. Your system administrator can provide you with this IP address.

  3. In the Port box, type the port number of the URL.

  4. Click Save Changes.

For information related to this procedure, see Configuring HTTP Proxy.
