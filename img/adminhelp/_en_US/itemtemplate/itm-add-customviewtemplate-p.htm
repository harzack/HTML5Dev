To add a Custom View Template:

  1. Click the Open the Templates Volume link in the Item Template Administration section on the Content Server Administration page.

  2. On the Content Server Templates page, click Custom View Template on the Add Item menu.

  3. On the Add Custom View Templates page, type a name for the template in the Name field.

  4. In the Source area, do one of the following:

    • Click the Custom View radio button, click the Browse button to navigate to the customview.html file you want the template created from, and then click its Select link.

    • Click the File radio button, click the Browse button to navigate to the .xml file you want the template created from, and then click Open

  5. Specify any other general item settings.

  6. Click Add.
