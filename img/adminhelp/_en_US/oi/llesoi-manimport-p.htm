To manually import a document:

  1. On the Content Server Administration page, in the Object Importer Administration section, click the Manual Object Import link.

  2. On the Manual Object Import page, in the Log File Name field, specify a name for the log file, without an extension, that will be written to the log directory as specified in the administration pages. The name cannot contain any special characters.

  3. In the Import Control File field, click Browse... to select an import control file.

  4. Click Start Import.
