To add a table to the database:

  1. On the Content Server Administration page, in the Object Importer Administration section, click Database Administration.

  2. On the Database Administration page, in the Action column, click the missing table's Create link.

To remove a table from the database:

  1. On the Content Server Administration page, in the Object Importer Administration section, click Database Administration.

  2. On the Database Administration page, click the table's Delete link.
