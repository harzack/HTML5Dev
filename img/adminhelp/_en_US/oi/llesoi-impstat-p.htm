To view import status information:

  1. On the Content Server administration page, in the Object Importer Administration section, click Import Status.

     

    Note:
    If no import threads are currently running, clicking Import Status will display an empty status list.

     

  2. To refresh the page content, on the Import Status page, click Update Status.

  3. To view the log files, click View Log Files.

  4. To delete any rows from the oi_status table that have a Status value of “2” or “3”, click Clear Status Table.
