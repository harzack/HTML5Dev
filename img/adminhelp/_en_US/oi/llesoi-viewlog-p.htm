To view log files:

  1. On the Content Server Administration page, in the Object Importer Administration section, click View Log Files.

  2. On the View Import Log Files page, click the name of any log file you want to view.

To delete log files:

  1. On the Content Server Administration page, in the Object Importer Administration section, click View Log Files.

  2. On the View Import Log Files page, select the check box next to the log file that you want to delete.

  3. Click Delete.
