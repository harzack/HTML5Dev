The Cluster Agent port is set by the Content Server installer and appears in the org.osgi.service.http.port setting. By default, the value of this setting is 3099. You can change the TCP/IP port that the Cluster Agent uses by editing the value of this setting.

  • Syntax:

    org.osgi.service.http.port=<port>

  • Values:

    A TCP/IP port.

  • Example:

    org.osgi.service.http.port=3456
