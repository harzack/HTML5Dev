To install patches:

  1. In the Server Configuration section of the Content Server Administration page, click the Cluster Management link.

  2. On the Cluster Management page, click the Manage Master System link.

  3. Click the Available Patches tab.

  4. Click Refresh to update the list.

  5. Click Download to get the patch from the OpenText Knowledge Center.

     

    Note:
    After you download a patch, move it to the staging folder so it is available for deployment.

    Do not extract the archive (.zip or .tar.gz) in the patch staging folder that you set in To Manage Cluster Settings. Content Server extracts the contents of the patch when it deploys it.

     

  6. Click the Staged Patches tab.

  7. Select the check box for the patch that you want to install.

  8. Click Deploy.
