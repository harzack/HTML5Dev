Update Analyzer is a utility included in Content Server Cluster Management that gives you insight into the changes that a Content Server Update will make to your system. By using Update Analyzer, you can determine whether any of the existing files in your Content Server deployment need to be preserved before you apply the Update, and you can identify hotfixes that need to be removed before you apply the Content Server Update.
