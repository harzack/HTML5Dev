The Manage Cluster function analyzes and displays information about the Content Server hosts. You have to run the Manage Master System analysis before the Manage Cluster function will analyze the Content Server hosts. When the Analyze Cluster page loads, it will automatically analyze any available Content Server Hosts. It will also list information for each Host, including the following:

  • Host is the cluster being analyzed.

  • Port is the port that the Host uses.

  • Analysis Status provides the status of the analysis, indicating if it is active or complete, or if there are errors.

After the analysis is complete, information about the new patch is listed on the Review Cluster Deployment Plans page.
