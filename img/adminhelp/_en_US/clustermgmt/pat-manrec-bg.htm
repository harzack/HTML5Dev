Each time you use Cluster Management to install one or more hotfixes, Cluster Management creates a backup folder in the <Content_Server_home>/clustermanagement/recovery/ folder. The name of the backup folder is based on the date and time that you installed the hotfixes.

Example: The patchdeployment-2014-03-18-16-10-25 folder corresponds to a hotfix deployment performed on March 18, 2014 at 16:10:25.

A backup folder contains a recovery manifest and a compressed archive containing Content Server application folders and files.

The Recovery Manifest

The recovery manifest file (recovery_manifest.xml) contains information on the contents of the backup folder and on the various steps that you must take to roll back a hotfix deployment.

The information in the recovery manifest appears in three sections:

recoveryFilesArchive

This section shows the location of the compressed archive containing the Content Server application folders and files that were changed by the hotfix deployment. The folders are in the state they were in before the hotfixes were applied. Restoring these folders to the <Content_Server_home> folder reverses the effects of the hotfix deployment on the Content Server application files.

removeFiles

This section lists files that you must manually delete to complete the rollback of the hotfix deployment.

impacts

This section lists Content Server components (the Content Server database, external file store and search index) that could be modified by a hotfix. A value of true indicates that the component was changed. If Database Schema, Content in the File Store, or Search Index has a value of true, you must restore a Content Server backup set to completely roll back the hotfix deployment.

The Compressed Archive

A compressed archive file called recovery_files[.zip] contains backup copies of the Content Server files and folders that were affected by the hotfix deployment. Restoring these files and folders restores the Content Server application folder to the state it was in before the hotfix deployment.
