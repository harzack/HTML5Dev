To manage Cluster Agents:

  1. In the Server Configuration section of the Content Server Administration page, click the Cluster Management link.

  2. On the Cluster Management page, click the Manage Cluster Agents link.

  3. On the Cluster Agents page, click the Plus icon beside the desired Host.

  4. Click the online link for the desired Host.

     

    Note:
    You cannot access a Host if it is offline.
