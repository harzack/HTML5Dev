To manage Cluster Settings:

  1. Open the Manage Cluster Settings page.

    1. In the Server Configuration section of the Content Server Administration page, click Cluster Management.

    2. On the Cluster Management page, click Manage Cluster Settings.

  2. Enter the Manifest URL.

    1. Type the URL for updating the manifest in Update URL.

       

      Note:
      If you cannot access the manifest file over the Internet, see Hosting the Manifest File Locally.

       

    2. Click Check URL to validate the manifest URL.

  3. Enter the Master System Host.

    1. Click Browse to open the Select a Master System dialog box.

    2. Enable the Host that you wish to use for your Master System, and then click OK.

  4. Enter the location of the Patch Staging Folder.

     

    Tip:
    Click Reset to set the Patch Staging Folder to the default staging directory location.

     

  5. Click Save Settings.
