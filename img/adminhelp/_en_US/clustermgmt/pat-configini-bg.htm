This section describes Cluster Management configurations can only be done in the Cluster Management config.ini file. The Cluster Management config.ini file is located in the <Content_Server_home>\config\otclusteragent\ folder.

Stop the Content Server Cluster Agent service before you edit the Cluster Management config.ini file. Restart the service after you make changes to the config.ini file.
