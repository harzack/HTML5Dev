To configure community settings:

  1. In the Communities Administration area of the Content Server Administration page, click the Configure Community Settings link.

  2. On the Configure Community Settings page, do any of the following:

    1. Select or clear any of the Enable check boxes in the following areas:

      • Navigation Menu

      • Personal Frontpage

      • Indirect User Functions

      • Disable JavaScript Filtering

      • Public Member Info

      • State Overwrite

      • Calendar Appointments

      • Metrics

      • Abbreviated Links

    2. Type one of the following in the Max Personal Photo Size box:

      • A positive integer, which sets the maximum file size, in bytes, for personal photos.

      • 0, which disables personal photos.

    3. In the Calendar View area, do the following:

      • Select the Calendar View or List View radio button and the select the view style from the corresponding lists.

      • Select a start time and end time using the lists.

    4. Type the HTML code you want displayed in the footer section of emails in the HTML E-Mail Footer box.

    5. Type the text you want displayed in the footer section of emails in the Text E-Mail Footer box.

    6. Select or clear the check boxes for any Content Server item types in the Reportable Items area.

  3. Click Update.
