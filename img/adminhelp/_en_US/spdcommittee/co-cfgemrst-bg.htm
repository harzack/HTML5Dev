When you configure email restrictions, you configure the limits placed on sending email from within a community, and any warning that appears when users send email. Table 1-2 describes the email restriction settings that you can configure.

Table 1-2: Configurable Email Restriction Settings

Setting Description
External Attachments When enabled, this setting allows users to attach files uploaded from their computers to an email sent from within communities.
External E-mail Addresses Field When enabled, this setting allows users to send email from within communities to any email address.
Warning When enabled, this setting displays a warning each time a user attempts to send email from within communities.
Background Color Determines the background color for the warning displayed when users send email from within communities.
Warning Text Specifies the text of the warning displayed when users send email from within communities.
