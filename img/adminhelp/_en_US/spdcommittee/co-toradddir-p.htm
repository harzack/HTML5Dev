To add a directory:

  1. Choose Communities on the Enterprise menu.

  2. On the Community Directory page, click the <Directory_Tree_name> link of the directory tree where you want to add a directory.

  3. On the Community Directory page, click Directory on the Add Item menu.

  4. On the Add: Directory page, type a name for the directory in the Name box.

  5. Specify any other general item settings.

  6. Click Add.
