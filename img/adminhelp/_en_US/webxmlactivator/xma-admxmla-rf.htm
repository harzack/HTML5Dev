The XML Activator data flow processes allow you to connect third-party applications to data flows. This allows you to integrate Content Server and third-party applications in several ways. You can:

  • Send Content Server data flow information to a third-party application rather than indexing the information in Content Server

  • Send information from the middle of a Content Server data flow to a third-party application, as well as to the next Content Server process

  • Filter Content Server information through a third-party application before sending it to the next Content Server process

  • Index information from third-party applications in Content Server

Third-party applications do not require proprietary tools, libraries, or particular coding languages to connect to Content Server. They interface with Content Server through XML files.
