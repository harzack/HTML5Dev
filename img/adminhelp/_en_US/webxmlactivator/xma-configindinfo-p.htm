To configure index update information:

  1. In the Read Directory field, type the absolute path of the directory from which you want this process to read data.

  2. In the Index Directory field, type the absolute path of the directory in which you want the Index Engine process to create the index. You must choose an index directory on a drive that is local to the host of the Admin server that runs this process.

  3. In the Host drop-down list in the Consumer: Content Server Index section, click the shortcut of the Admin server on whose host you want the Update Distributor process to run.
