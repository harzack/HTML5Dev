Physical Objects lets you configure the settings for physical objects and specify which physical object functions are tracked. You can also maintain the location codes users create.

 

Note:
In previous versions of Physical Objects, you could add, edit, and delete physical item types in the Physical Objects Administration section of the Content Server Administration page. These features have been moved to the Physical Objects Workspace.
