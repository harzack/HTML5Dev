Note:
In previous releases, the item count for box objects was 0. Running this script will update this count on all existing boxes to reflect the objects assigned to the boxes.

 

To update box item counts:

  1. On the Content Server Administration page, click the Configure Physical Objects link in the Physical Objects Administration section.

  2. On the Configure Physical Objects page, select the Run Box Script check box.

  3. Select either the Missing value (the script is only run against boxes which have “null” as child count) or the All boxes radio button.

  4. Click the Submit button.
