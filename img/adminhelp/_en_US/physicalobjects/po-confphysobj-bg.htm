Physical Objects lets you configure physical objects. You can set the unique ID format for all physical items. You can also insert a default label type and stock definition and remove label types. When you insert a default label type, you specify a script to automate the insertion process. You can also run a script that updates the item count for boxes.

 

Note:
In previous versions of Physical Objects, you could set the default number of days to borrow a physical item and the default location to save created physical items. These features have been moved to the Physical Objects Workspace.
