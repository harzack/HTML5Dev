You can add your own icons to represent custom physical item types in Content Server. The 16 x 16 GIF images must be placed in the Content Server_home/support/physicalobjects/icons folder, and you must restart the Content Server before the new icons will become available on the Add Physical Item Type page.

 

Note:
Changing the icon for an existing physical item type will not change the icon for any physical items that were added against the physical item type before the icon was changed.
