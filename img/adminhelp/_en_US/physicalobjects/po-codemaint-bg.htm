The Administrator can batch rename location codes assigned to physical items and place these codes in the Location table.

If updating an existing code, all locations in the PhysItemData and PhysItemCO tables that match the old code are updated to the new code. Also, a new entry is inserted into the PhysObjLocation table. A blue check mark display next to all updated codes.

Free-form locations are not included in the PhysObjLocation table. The Administrator can use location code maintenance to add these locations to the table.
