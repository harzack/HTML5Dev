To set the unique ID format for all physical items:

  1. On the Content Server Administration page, click the Configure Physical Objects link in the Physical Objects Administration section.

  2. On the Configure Physical Objects page, specify the unique ID format for all physical items in the Format field, and then click the Show Next ID button.

  3. Click the Submit button.

 

Notes:

  • By default, the unique ID format is %Y0001, where %Y represents the four-digit year. Other valid formats include %m01, where %m represents the two-digit month, and %d01, where %d represents the two-digit day.

  • The format for the UniqueID here is used as a default format; you can specify different formats for each Physical Item Type if desired. You can specify different formats for each Physical Item Type from the Physical Objects Workspace, Physical Item Type Administration section.
