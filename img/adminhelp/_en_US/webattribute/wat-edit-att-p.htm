To edit an attribute:

  1. Click the Administer Additional Node Attributes link in the System Administration section on the administration page.

  2. Click the Edit link of the attribute that you want to edit.

  3. Optionally, click a new interface display type, if more than one is available, in the Type list.

  4. Optionally, type a new name in the Display Name field.

  5. For a Popup attribute type, click the Edit button, type any new values in the Valid Values field, and then click Continue Editing Attribute.

  6. Click Save Changes.
