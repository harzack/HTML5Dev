To specify attribute settings:

  1. Click the Administer Additional Node Attributes link in the System Administration section on the administration page.

  2. For each attribute, click one of the following radio buttons:

    • Exclude, which sets the attribute not to appear for Content Server items.

    • Include, which sets the attribute to appear for all Content Server items.

    • Require, which sets the attribute to appear for all Content Server items, and must be set to a valid value for each item.

  3. Click Save Changes.
