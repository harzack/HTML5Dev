Note:
You can only save a Category Folder to the Categories Volume or another Category Folder.

 

To add a Category Folder:

  1. Choose Category Folder on the Add Item menu.

  2. Type a name for the Category Folder in the Name field.

  3. Optionally, type a description of the Category Folder in the Description field.

  4. To modify the categories or attributes associated with the category, click the Edit button.

  5. To place the item in a location other than that which appears in the Create In field, click the Browse Content Server... button, navigate to the container where you want to locate the category, then click its Select link.

  6. Click the Add Item button.
