To configure mandatory search terms:

  1. On the Administration page, in the Search Administration section, click the Configure Mandatory Search Terms link.

  2. On the Configure Mandatory Search Terms page, for Apply restrictions to Users with eDiscovery rights, select the check box to enable these restrictions.

  3. For Apply restrictions to Users with System Administration rights, select the check box to enable these restrictions.

  4. In the Group text box, enter the name of the group to restrict.

  5. In the Search Term text box, enter the search term clause. A validation error is displayed if the syntax used is not correct.

  6. Click Update to use the mandatory terms you defined, or click Apply to save the settings.
