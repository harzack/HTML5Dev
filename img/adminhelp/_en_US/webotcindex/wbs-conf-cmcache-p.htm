To configure the Cache Settings search:

  1. On the Administration page, in the Search Administration section, click the Configure Search Options link.

  2. On the Configure Search Options page, in the Cache Settings section, type a value in the Cached Brokers to Keep field.

  3. Type a value in the Cached Region Sets to Keep field.

  4. Click the Update button.

 

Note:
You must click the Apply button on the Search Bar Edit page to apply your changes.

 

For more information, see Configuring the Cache Settings Search.
