The following table describes error messages for the Search Federator process.

If these errors persist, contact OpenText Customer Support for additional information.

Table 1-31: Error Messages for Search Federators

Number Message Description Remedy
132 An unknown error has occurred The Search Federator process encountered an unknown error. Contact OpenText Customer Support.
149 Invalid command line for the Search Federator There is an error in the Search Federator process' command line parameters. Ensure that the command lines in otadmin.cfg are valid. and specified in the proper format Content Server generates otadmin.cfg using its database.
150 URL is invalid The URL used to register and contact the Search Federator is invalid as specified in the search.ini file. Ensure that the search.ini parameters are valid and specified in the proper format.
153 Unable to initialize The Search Federator process failed to start because one or more parameters in the search.ini file are invalid. Ensure that the search.ini file parameters are valid and specified in the proper format.
 

Note:
The search.ini file is the configuration file for Content Server searching and indexing processes. This file contains settings that are usually set by default when indexing and searching processes and other system objects, such as data sources and partitions, are created; conversely, settings are also deleted when the processes they configure are deleted. You configure these settings when you configure searching and indexing processes and other system objects in Content Server. OpenText strongly recommends that you do not modify the settings in the search.ini file. If you do choose to modify the settings, contact OpenText Customer Support.
