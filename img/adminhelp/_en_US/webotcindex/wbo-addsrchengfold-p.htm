To add a Search Manager:

  1. On the System Administration page, click the processes_prefix Data Source Folder link of the data source folder to which you want to add a Search Manager.

  2. Click Search Manager on the Add Item menu.

  3. On the Add: Search Manager page, type a name in the Name field.

  4. To provide a description of the Search Manager on the General tab of its Properties page, type a description in the Description field.

  5. To modify the categories and attributes associated with this item, click the Edit button.

  6. Click the Add button.
