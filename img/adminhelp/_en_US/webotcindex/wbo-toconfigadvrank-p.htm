To configure advanced ranking:

  1. On the System Administration page, click the processes_prefix Data Source Folder link of the data source for which you want to configure advanced ranking.

  2. Click the Functions icon of the Search Manager, choose Properties, and then choose Ranking.

  3. On the Ranking tab of the Search Manager Properties page, configure the following parameters:

    • Query Weight

    • Field Rank is based on an item's metadata regions.

    • Date Rank

    • Type Rank

    • Object Score

    For more information about the parameters you can set, see Configuring Advanced Ranking.

  4. Click the Update button.
