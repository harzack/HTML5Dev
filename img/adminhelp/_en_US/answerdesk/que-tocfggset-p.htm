To configure general settings:

  1. In the Communities Administration area of the Content Server Administration page, click the Questions & Answers Settings link.

  2. On the Questions & Answers Settings page, do any of the following:

    • In the Default Urgency area, click one of the following radio buttons:

      • Urgent

      • Not Urgent

    • Type a positive integer or 0 in the Urgency Duration box.

    • In the Default Reply Type area, click one of the following radio buttons:

      • Answer

      • Comment

      • Neither, force the user to choose

    • Select or clear the Enable Ratings check box in the Ratings area.

    • Select or clear the Allow users to select Community Forums check box in the Target Audience area.

    • Type a positive integer in the Question List Length box.

    • Type a positive integer in the Maximum Question Search Results box.

       

      Tip:
      Leave this field blank to display all search results.

       

    • Type the terms of use in the Terms of Use box.

  3. Click Update.
