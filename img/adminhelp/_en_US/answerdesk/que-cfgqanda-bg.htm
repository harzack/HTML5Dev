When you configure questions and answers, you configure the behavior of this functionality for all users. Configuring questions and answers consists of configuring email settings and general questions and answers settings.

Configuring email settings for questions and answers includes the following tasks:

  • Specifying the suffix of the question activity report's From address.

  • Specifying whether to include a Helpful Links area in question activity reports.

  • Specifying whether to allow users to use their normal email reply function to submit a reply to a question.

  • Specifying whether to send community activity reports.

Configuring general questions and answers settings includes the following tasks:

  • Setting the default urgency.

  • Setting the urgency duration.

  • Setting the default reply type.

  • Specifying whether to allow users to rate answers and comments.

  • Specifying whether to allow users to select community forums when asking a system-wide question.

  • Setting the question list length.

  • Specifying the maximum number of questions to display on the Questions & Answers page and the My Questions page for each search.

  • Specifying the terms of use that are displayed when a user asks a question.

Specifying the From Field Suffix

When you specify the From field suffix, you specify the second part of the sender display name in question activity reports, which follows the name of the user who has authored the most relevant activity in the report. For example, if you set the suffix to Content Server Q&A, and Rita Smith answers a question, the From address in the question activity report is Rita Smith via Content Server Q&A. If Rita Smith answers a question, and Barry Ulrich comments on it, the From address is Rita Smith and others via Content Server Q&A.

Question activity reports are delivered through Content Server Notifications, which must be enabled. For more information about notifications, see Working with Notifications.

Specifying Whether to Include a Helpful Links Section

The Helpful Links section in question activity reports contains links to functions relevant to the email content, such as Unsubscribe and Help.

Allowing Email Replies to Questions

You can choose whether to allow users to reply to questions using their normal email reply function.

Specifying Whether to Send Community Activity Reports

Community activity reports are daily email reports that users receive for the communities to which they are subscribed. These reports summarize activity in the communities and directories to which users are subscribed, including recently asked questions and recently added communities. Community activity reports are delivered through Content Server's notification system, which must be enabled.

Setting the Default Urgency

Setting the default urgency for questions allows you to specify whether a question is urgent by default. If the default is not enabled, users must mark the question as urgent.

Setting the Urgency Duration

The urgency duration specifies how many days a question remains marked urgent. For example, if the urgency duration is set to 5 and a user asks a question and marks it urgent, it will no longer be marked urgent five days later. If you set the urgency duration to 0 (zero), questions will maintain their urgent status indefinitely.

Setting the Default Reply Type

When you set the default reply type for questions, you set the default value for the Type of Reply check box that appears when users reply to a question. You can set the default reply type to be an answer or comment, or you can set it to neither, which forces the user to choose a type.

Specifying Whether to Allow Users to Rate Answers and Comments

By allowing users to rate answers and comments, you provide them with a feedback mechanism to help identify the usefulness of replies to a question.

Specifying Whether to Allow Users to Select Community Forums

When you allow users to select community forums, they can target their system-wide questions to forums inside communities.

Setting the Question List Length

When you set the question list length, you specify the maximum number of questions to initially display on the Questions & Answers page and the My Questions page for each user.

Setting the Maximum Number of Question Search Results

When you set the maximum question search results, you specify the maximum number of questions to initially display on the Questions & Answers page and the My Questions page for each search. Leave this field blank to display all search results.

Specifying Terms of Use Text

You can set your organization's terms of use policy, if any, to be displayed when a user asks a question.
