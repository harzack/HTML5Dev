Description

By default, Object Exporter will traverse the entire tree from the starting location as specified in the location tag. To prevent Object Exporter from traversing the entire tree, use the <recurse>FALSE</recurse> tag.

Syntax

<recurse>FALSE</recurse>

Usage

  1. The recurse tag is optional.

  2. The only valid value for use with the recurse tag is FALSE.

Example

In this example, only the non-container objects in “Folder_001”, located in the Enterprise Workspace, will be exported to c:\docs

<export>
  <node>
    <location>Enterprise:Folder_001</location>
    <docstorage>c:\docs<docstorage>
    <recurse>FALSE</recurse>
  </node>
</export>
