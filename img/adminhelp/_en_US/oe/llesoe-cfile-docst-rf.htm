Description

 

Important:
This is a mandatory tag, regardless of object type.

 

Use the docstorage tag to specify where, in the filesystem, the documents from Content Server will be exported.

Syntax

<docstorage>c:\docs</docstorage>
<docstorage version="current">c:\docs</docstorage>

Usage

  1. This is a mandatory tag regardless of object type.

  2. The pathname specified between the <docstorage></docstorage> tags must exist on the system.

Notes

  1. By default, all versions of a document will be exported. To export only the current version, add the version=current attribute to the docstorage tag.

  2. To change the path to all documents in the generated control file, add the pathprefix attribute.

Example

In this example, the documents in “Folder_001”, located in the Enterprise Workspace, will be exported to c:\docs. However, the generated control file will show c:\temp

<export>
  <node>
    <location>Enterprise:Folder_001</location>
    <docstorage pathprefix="c:\temp">c:\docs</docstorage>
  </node>
</export>
