To Export an Object:

  1. On the Content Server administration page, in the Object Exporter Administration section, click Manual Object Export.

  2. On the Manual Object Export page, in the Log File Name field, specify a name for the log file, without an extension, that will be used for the log and output files.

     

    Note:
    A date specific value will be appended to the file name to make it unique.

     

  3. In the Export Control File field, click Browse... to select an export control file from the file system.

  4. Click Start Export to start the export process. When the export is finished, the administrator will be notified via email.
