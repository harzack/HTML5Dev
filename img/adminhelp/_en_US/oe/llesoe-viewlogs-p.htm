To View a Log File:

  1. On the Content Server administration page, in the Object Exporter Administration section, click View Log Files.

  2. On the View Export Log Files page, click the name of any log file you want to view.

To Delete a Log File:

  1. On the Content Server administration page, in the Object Exporter Administration section, click View Log Files.

  2. On the View Export Log Files page, select the check box next to the log file that you want to delete.

  3. Click Delete, then click OK to confirm that you want to delete the selected log files.
