To Configure LiveReports Security

  1. On the Content Server Administration page, in the LiveReports Administration section, click the Configure LiveReports Security link.

  2. On the Configure LiveReports Security page, enable Prevent LiveReports from modifying the database. Once enabled it cannot be reversed.

  3. The Disable Database Modification dialog box appears. Enable I understand the risk involved by enabling this setting and want to continue, and then click OK.

  4. On the Configure LiveReports Security page, click Save Changes.
