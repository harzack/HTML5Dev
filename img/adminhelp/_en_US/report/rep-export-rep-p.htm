To export a LiveReport:

  1. Click the Export LiveReports link in the LiveReports Administration section on the Content Server Administration page.

  2. If prompted, type the password for the Admin user in the Administrator Password box, and then click Log-in.

  3. On the Export LiveReport page, select the check boxes of the LiveReports that you want to export to a single text file.

  4. Do one of the following:

    • If you want to export the selected LiveReports, click the Export button.

    • If you want to export and delete the selected LiveReports, click the Export and Delete button.

  5. Save the LiveReport file to a location of your choice.
