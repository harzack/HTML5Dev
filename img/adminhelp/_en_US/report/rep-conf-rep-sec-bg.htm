To protect the security of your Content Server data, you may wish to make it impossible for LiveReports to make changes to the Content Server database. You can permanently prevent all LiveReports from modifying the database by using the Configure LiveReports link in the LiveReports Administration section of the Content Server Administration page.

 

Important:
This action is not reversible.Â Once you have enabled LiveReports security, you cannot restore the ability of LiveReports to modify the database.
