To change or move the external document storage directory:

  1. Record the directory path currently displayed in the External Directory field.

  2. Type the path of the new storage directory as it is mapped/mounted on the primary Content Server host in the External Directory field.

  3. Click the Set Directory button. Content Server prompts you to Restart the Server and the Web application server.

  4. On the Restart Server page, stop the Server.

  5. Move the external database files from the old directory to the new directory that you set in step 2.

  6. Restart the Server and the Web application server.

  7. On the Restart Server page, click the Continue link.
