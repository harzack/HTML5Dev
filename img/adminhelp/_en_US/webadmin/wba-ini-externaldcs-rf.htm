• Description:

    Content Serverâ€™s Document Conversion Service (DCS) converts items from their native file formats (for example, Microsoft Word or Excel, or Adobe PDF) to a simple text format (for example, HTML or text) for viewing or indexing in Content Server.

    This parameter is used to configure the SEA Servlet (LLServlet) to allow View as Web Page requests from external users.

  • Syntax

    [ExternalDCS]
    host=<IP Address of the internal Content Server>
    port=<DCS_Port_Number>
    viewcache=<SEASERVLET_HOME>/viewcache
    cachesize=<10>

     

    Note:
    OpenText recommends that you share the viewcache subfolder, with full share permissions for your users.

     
     

    Tip:
    You can obtain the port number by accessing the Document Conversion Service tab on the Properties page for the default System Object Volume.

     

  • Values:

    The values in the [ExternalDCS] section of the opentext.ini file are for the Admin server, and can be used for the primary host or a different computer.
