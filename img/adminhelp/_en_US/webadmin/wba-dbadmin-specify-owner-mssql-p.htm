To specify the owner of a Microsoft SQL Server database:

  1. In the SQL Server Name field, type the Microsoft SQL Server alias.

  2. In the MSSql User Name field, type the name of the Microsoft SQL Server user that can log in to the Microsoft SQL Server database that you want to select.

  3. In the Password field, type the password of the Microsoft SQL Server user.

  4. In the SQL Database Name field, type the name of the Microsoft SQL Server database that you want to select.

  5. Click the Continue button. Depending on whether this is a new or existing Content Server installation, whether the database was originally created on the current host, and whether the database needs upgrading, one of the following pages appears:

    • Content Server Database Upgrade Confirmation

    • Install Modules

    • Administering Admin Servers
