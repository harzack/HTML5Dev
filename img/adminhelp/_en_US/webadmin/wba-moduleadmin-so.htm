Content Server components are organized into software modules. There are two types of modules: core modules and optional modules.

Core modules, such as Content Server Workspaces, Content Server Document Management, and Content Server Projects, are automatically installed when Content Server is installed. Optional modules, such as Content Server Forms, must be installed by the Content Server Administrator either during or after the initial installation of Content Server. Optional modules enable you to extend the functionality of Content Server to meet your organization's specific needs. These may be new modules that you have purchased or modules that were available at the time of installation, but you chose not to install. For information on installing optional modules during a Content Server installation, see the Content Server First-Time Installation Guide.

Optional modules are sometimes distributed on a CD-ROM, but most often are available for download from the Open Text Knowledge Center. In either case, documentation for the specific module and installation instructions will accompany the module.
