In the [llserver] section, you can change the number of threads that the Server uses in its operation.

id

  • Description:

    This is Open Text proprietary information.

max

  • Description:

    The maximum number of threads that are used by the server. For more information, see Configuring Performance Settings.

  • Syntax:

    max=32

  • Values:

    An integer greater than or equal to 1. The default value is dependent on the type of hardware being used.

    The more threads you allow, the more system resources the server consumes.

min

  • Description:

    The minimum number of threads that are used by the server. For more information, see Configuring Performance Settings.

  • Syntax:

    min=8

  • Values:

    An integer greater than, or equal to, 1. The default value is dependent on the type of hardware being used.

    The more threads you allow, the more system resources the server consumes.
