When a user deletes a document, Content Server places that document in the Deleted Objects Volume instead of removing it from the system. The document undelete function allows any user with system administration rights to restore deleted documents from the Deleted Objects Volume to the Content Server workspaces.

The document undelete function only applies to Content Server documents, and no other item type, including compound documents.

When users delete renditions or versions of documents, the deleted items are not added to the Deleted Objects Volume. The document undelete function is available only when a user deletes the document node.

As a user with system administration rights, you can view any aliases to the deleted document until the document is purged.
