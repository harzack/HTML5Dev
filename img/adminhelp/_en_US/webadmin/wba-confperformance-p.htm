To configure performance settings:

  1. Click the Configure Performance Settings link on the Administration page.

  2. On the Configure Performance Settings page, do any of the following:

    • Type the maximum number of user log-in sessions the server will cache in the Maximum Number of Sessions field.

    • Type the number of threads the server will use in the Number of Threads field.

    • Select or clear the Receive before Send check box.

    • Select or clear the Enable Web Caching check box.

    • Type the size (in KB) used for file copy operations in the File Copy Buffer Size field.

  3. Click the Save Changes button.
