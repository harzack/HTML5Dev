To install a module on Windows:

  1. Download and run the module's executable file.

  2. In the module's Welcome window, click the Next button.

  3. In the Select Content Server Services window, select the check box beside the Content Server instance on which you want the module installed, and then click the Next button.

  4. In the Setup Complete window, click the Finish button.

  5. In the Module Administration section of the Administration page, click the Install Modules link.

  6. In the Installable Modules section, select the check box beside the module you want installed, and then click the Install button.

  7. Restart the server and Web Application server, and then click the Continue link on the Restart Content Server page.
