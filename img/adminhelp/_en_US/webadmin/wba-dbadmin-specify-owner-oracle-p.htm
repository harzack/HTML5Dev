To specify the owner of an Oracle database:

  1. In the Oracle User Name field, type the name of the Oracle user who owns tables of the Content Server database that you want to select.

  2. In the Password field, type the password of the Oracle user.

  3. In the Service Name field, type the service name (database alias) of the Oracle server.

  4. Click the Continue button. Depending on whether this is a new or existing Content Server installation, whether the database was originally created on the current host, and whether the database needs upgrading, one of the following pages appears:

    • Content Server Database Upgrade Confirmation

    • Install Modules

    • Default Admin Server Confirmation

    • Data Source Resynchronization Confirmation
