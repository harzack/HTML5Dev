Stopping the Servers under UNIX

Stopping or starting theContent Server process under UNIX also shuts down or starts the Admin server process.

To stop the servers under UNIX:

  1. Log on to the UNIX host with the user name that Content Server uses.

  2. At the command prompt, change to the directory where Content Server is installed.

  3. Type the following command, and then press ENTER:

    ./stop_llserver

Starting the Servers under UNIX

To start the servers under UNIX:

  1. Log on to the UNIX host with the user name that Content Server uses.

  2. At the command prompt, change to the directory where Content Server is installed, and then do one of the following:

    • To start both of the servers, type the following command, and then press ENTER:

      ./start_llserver

    • To start the Admin server only, for example, in the case of a secondary Content Server host, type the following command, and then press ENTER:

      ./start_lladmin
