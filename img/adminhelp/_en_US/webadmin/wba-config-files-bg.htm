Some system configuration settings cannot be made from the Administration pages. In these cases, you must edit a system-configuration file directly to implement a change. The opentext.ini file used most often for manual configurations. The default location of the opentext.ini file is default_installation_directory/config/. The mime.types and mime.gif files, both located in the config directory, are also files that are modified directly.

When you edit the opentext.ini file directly in a text editor, you must stop and restart the Content Server services and the Web application server for the changes to take effect.
