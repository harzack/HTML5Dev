To log in to the Microsoft SQL Server:

  1. Type the Microsoft SQL Server alias in the SQL Server Name field.

  2. Type the Microsoft SQL Server system administrator user name (the default is sa) in the System User field.

  3. Type the password for the system user in the System Password field.

  4. Type the name of the master database (the default is master) in the Master Database Name field.

  5. Click the Log-in button. One of the following pages appears:

    • Create Content Server Tables

    • Microsoft SQL Server Maintenance
