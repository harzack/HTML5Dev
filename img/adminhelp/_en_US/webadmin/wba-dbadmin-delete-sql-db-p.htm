To delete a SQL Server database:

  1. On the Microsoft SQL Server Maintenance page, click the Delete a Microsoft SQL Server Database link.

  2. In the Database Name drop-down list, click the name of the SQL Server database that you want to delete.

  3. Click the Delete Database button.

  4. Click the OK button in the dialog box that opens.

  5. If you have completed all desired tasks on the Microsoft SQL Server Maintenance page, click the Return to previous page link.
