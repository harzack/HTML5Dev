To view the distributed agent workers:

  1. On the Content Server administration page, under the System Administration heading, click Distributed Agent Status.

  2. On the Distributed Agent Status page you will see:

    • Worker ID: lists the IDs of the distributed agent workers.

    • Last Update: displays the date and time that the worker ID status was last updated.

    • Status: displays the status for each worker process. Possible values include Idle, Waiting and Running.

    • Status Description: describes the status of the worker ID.

    • Actions: displays the actions that you can apply to each worker. Possible values include Pause and Resume.
