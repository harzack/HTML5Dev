To install a module on UNIX:

  1. Copy the <module_name>.tar file to the <Content Server_home> directory of the Content Server instance on which you want the module installed.

  2. At the shell prompt, type the following command, tar -xvf <module_name>.tar, and then press ENTER.

  3. In the Module Administration section of the Administration page, click the Install Modules link.

  4. In the Installable Modules section, select the check box beside the module you want installed, and then click the Install button.

  5. Restart the server and Web Application server, and then click the Continue link on the Restart Content Server page.
