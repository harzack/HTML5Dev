To change the password used to connect to the database:

  1. In the Database Administration section of the Content Server Administration page, click the Change Current Database link.

  2. On the Deselect Content Server Database page, clear the check box for each Data Source, and then click Continue.

     

    Note:
    Do not close the browser window.

     

  3. Change the usersâ€™ database password in the database you are using.

  4. Return to the Content Server window, and click the Select Existing Database link.

  5. On the Select RDBMS Type page, click the radio button for your database type, and then click Continue.

  6. On the Specify Content Server Database Owner page, enter the required information, and then click Continue.

  7. On the Content Server Administrator User Log-in page, enter your log in credentials, and then click Log-in.

  8. On the Install Module page, click Continue.

  9. On the Content Server Admin Server Configuration Confirmation page, click Continue.
