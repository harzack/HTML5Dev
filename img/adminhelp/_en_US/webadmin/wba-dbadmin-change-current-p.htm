To connect the current Content Server instance to a database:

 

Note:
Before changing the database to which Content Server is connected, your current database must be deselected. See To Deselect the Current Content Server Database for more information.

 

  1. Do one of the following:

    • Click the Create New Database link to create a new database.

    • Click the Select Existing Database link to select an existing database.

    The Congratulations page appears after you connect the current Content Server installation to a new or existing Content Server database. It confirms that the connection to the database was completed successfully.

  2. On the Congratulations page, you have the following options:

    • Click the Go to the Content Server home page link to go to the default Content Server start page.

    • Click the Admin Home link to go to the Content Server administration page.

 

Note:
If you connect this Content Server installation to an existing Content Server database and click Continue on the Data Source Resynchronization Confirmation page, you must reconfigure the data source objects, then individually resynchronize all registered Admin servers.
