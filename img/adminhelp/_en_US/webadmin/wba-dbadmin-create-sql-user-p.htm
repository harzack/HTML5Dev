To create a SQL Server user:

  1. On the Microsoft SQL Server Maintenance page, click the Create A New User link.

  2. Type the name that you want to give the new SQL Server user in the User Name field.

  3. Type a password for the new SQL Server user in the Password field.

  4. Type the password again in the Verify Password field.

  5. Click the name of the SQL Server database in which you want this user to store its tables in the Database Name drop-down list.

  6. Click the Create User button.

  7. Do one of the following:

    • If you are creating a new Content Server database, click the Return to previous page link to return to the Create Content Server Tables page.

    • Otherwise, if you have completed all tasks on the Microsoft SQL Server Maintenance page, click the Return to previous page link to return to the Maintain Current Database page.
