Content Server allows you to store Documents in multiple locations outside Content Server. By default, Content Server Documents are stored in an external storage directory that is created at the time Content Server is installed. Storage Providers and Storage Provider Rules can be added, deleted, or modified by the Content Server administrator.

 

Note:
While some storage providers are delivered with Content Server, other storage providers require that you install additional software. For example, the Archive Storage Provider requires that you install OpenText Archiving for Content Server.
