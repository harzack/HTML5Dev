The Specify Content Server Database Owner page appears when you are connecting the current Content Server installation to an existing database, after you click the Continue button on the Select RDBMS Type page.

The purpose of the page is to prompt you to specify the user name and password of the RDBMS account that owns the Content Server database tables.

You may need to obtain this information from the system administrator who is responsible for the database.

Depending on your RDBMS type, proceed to one of the following sections:

  • Specifying the Owner of an Oracle Database

  • Specifying the Owner of a Microsoft SQL Server Database
