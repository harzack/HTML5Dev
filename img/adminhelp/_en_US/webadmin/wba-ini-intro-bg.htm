The default installation path for the opentext.ini file is <Content Server_home>/config/opentext.ini. The opentext.ini file is the main Content Server configuration file for both primary and secondary Content Server installations.

The opentext.ini file contains settings such as database connection options, paths to files, date formats, debugging options, and logging options. The file is created automatically during the Content Server installation process. At that time, the default options are set and many settings are configured dynamically. Some settings, such as the encoded Administrator's password and Notification settings, are changed by Content Server as necessary.

Most of the settings that appear in the opentext.ini file can be changed on the Administration pages, but some settings must be changed by manually editing the opentext.ini file in a text editor. For more information, see Modifying the opentext.ini file.
