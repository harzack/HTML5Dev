To update a language pack for Content Server:

  1. Download the updated language pack that you want to install, and unzip it in the root Content Server installation directory.

     

    Note:
    Do not change the default name of the updated language pack.

     

  2. On the Content Server Administration page, in the Languages section, click the Install Language Packs link.

  3. On the Install Language Packs page, in the Installable Content Server Language Packs section, select the Install check box for each language pack update that you want to install, and then click the Install button.

  4. If prompted to do so, restart your servers.

  5. Click Continue.
