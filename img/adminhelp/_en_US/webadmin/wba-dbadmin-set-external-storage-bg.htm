Content Server databases can store documents either internally or externally on the file system. Whether a given database uses internal or external document storage is determined at the time that the Content Server database tables are created.

You use the External Document Directory page to set or change the path to the external file system location in which an external-storage database stores its data. The External Document Storage page appears under the following two circumstances:

  • You are creating a new Content Server database and have selected the External Documents check box on the Create Content Server Tables page.

  • You are administering an existing external-storage database and have accessed this page via the External Storage Settings link on the Maintain Current Database page. (This link does not appear on the Maintain Current Database page of existing internal-storage databases.)
