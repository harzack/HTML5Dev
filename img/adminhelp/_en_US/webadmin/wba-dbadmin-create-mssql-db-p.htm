Note:
You must first log in to the Microsoft SQL Server as a user with administrator privileges.

 

To create a Microsoft SQL Server database:

  1. In the Database Administration section of the Administration page, click the Maintain current Database link.

  2. On the Maintain Current Database page, click the Microsoft SQL Server Maintenance Tasks link.

  3. On the Microsoft SQL Server Administrator Log-in, type your SQL user name into the appropriate fields, and then click the Log-in button.

  4. On the Microsoft SQL Server Maintenance page, click Create a New Microsoft SQL Server Database.

  5. In the Database Name field, type the name that you want to assign to the database, for example, Content Server.

  6. In the Data File Specification field, type a path and file name, for example, C:\Store\<file_name>.mdf.

  7. In the Data File Size field, type a size in MB for the Data File (minimum is 5 MB).

     

    Tip:
    You can estimate an appropriate data file size using the following formula:

    # documents X # versions X # KB avg. doc. size = tablespace size

     

  8. In the Log File Specification field, type a path and file name, for example, C:\Store\<file_name>.ldf.

  9. In the Log File Size field, type a size in MB for the file (minimum is 5MB). Use the Data File Size example as a guide.

  10. Click Create Database.

  11. If you have completed all desired tasks on the Microsoft SQL Server Maintenance page, click the Return to previous page link.
