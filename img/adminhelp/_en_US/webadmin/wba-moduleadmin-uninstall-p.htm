To uninstall a module:

  1. In the Module Administration section of the Administration page, click the Uninstall Modules link.

  2. On the Uninstall Modules page, click the Uninstall button beside the module you want to remove.

  3. In the confirmation dialog box, click the OK button.

  4. Restart the server and Web application server, and then click the Continue link on the Restart Content Server page.
