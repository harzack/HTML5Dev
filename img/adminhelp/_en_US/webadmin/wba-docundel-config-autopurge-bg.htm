From the administration pages, you can automatically purge multiple documents from Content Server according to the deletion date of the documents.

Every day, Content Server removes all documents from the Deleted Objects Volume that have remained there for longer than the number of days specified.

 

Note:
The Consumer Five Minute Agent must be enabled to use the AutoPurge feature. For more information, see Configuring Scheduled Activities.
