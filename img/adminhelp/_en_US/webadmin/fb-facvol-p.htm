To view the facet volume:

  1. If you are an administrator or a Knowledge Manager, from the Global Menu Bar, select Tools then Facets Volume.

     

    Note:
    If you are an administrator, you can also access the Facets Volume from the administration home page. Select Server Configuration then Configure Facets. Next select View Facet Volume.

     

  2. From the Facet Volume you can also access the Facet Volume Control Panel by selecting the Facet Volume's functions button then selecting Control Panel.
