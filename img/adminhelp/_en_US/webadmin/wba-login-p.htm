To log in to the Content Server Administration page:

  1. Open the Content Server system-administration URL in a Web browser.

  2. On the Administrator Log-in page, type the administration password in the Password field.

  3. Click the Log-in button.

 

Tip:
If you installed Content Server on Windows, you can also access the Administration page by clicking the Windows Start menu, clicking Programs, opening <Content Server_folder_name>, and then clicking Content Server Administration.
