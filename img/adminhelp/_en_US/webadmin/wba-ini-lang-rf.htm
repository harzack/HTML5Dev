ShortTimeFormat

  • Description:

    Specifies whether Content Server displays times in 12-hour format, for example 02:41 PM, or 24-hour format, for example 14:41.

  • Syntax:

    ShortTimeFormat=%I:%M %p

  • Values:

    The default setting is a 12-hour format: %I:%M %p.

ShortDateFormat

  • Description:

    Specifies how Content Server displays the day, month, and year.

  • Syntax:

    ShortDateFormat=%m/%d/%Y

  • Values:

    The default setting is a two-digit month, the day, and the year: %m/%d/%Y.

    Other characteristics are governed by the ShortDateSeqFormat parameter.

ShortDateSeqFormat

  • Description:

    Specifies the order in which Content Server displays the day, month, and year, and what characters separate the elements of the date.

  • Syntax:

    ShortDateSeqFormat={1,1,2,1,'/','/'}

  • Values:

    The default setting is month, day, and year, all separated by a slash: {1,1,2,1,'/','/'}.

    Other characteristics are governed by the TwoDigitYears parameter.

UserNameDisplayFormat

  • Description:

    A format in which a Content Server user's name displays throughout Content Server, in fields such as Created by, Owned by, and User. Change user name formats only on the Configure User Name Display page of the Content Server administration interface.

  • Syntax:

    UserNameDisplayFormat=1

  • Values:

    Valid values are:

    • 1, which displays the Log-in ID. This is the default value.

    • 2, which displays FirstName LastName

    • 3, which displays FirstName MiddleInitial LastName

    • 4, which displays LastName, FirstName

    • 5, which displays LastName, FirstName MiddleInitial

    • 6, which displays LastName FirstName
