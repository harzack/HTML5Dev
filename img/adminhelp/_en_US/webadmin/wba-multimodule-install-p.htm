To install a language pack for a Content Server module:

  1. Download the language pack that you want to install, then move it to the C:\<Content Server_home>\langpkgstaging\module location, where C:\<Content Server_home> is the location of your Content Server installation.

     

    Note:
    Do not change the default name of the language pack.

     

  2. On the Content Server Administration page, in the Languages section, click the Install Language Packs link.

  3. On the Install Language Packs page, in the Installable Module Language Packs section, select the Install check box for each language that you want to install, and then click the Install button.

  4. If prompted to do so, restart your servers.

  5. Click Continue.

 

Note:
Once you install a language pack, you must enable it in the Configure Languages section in order to apply the new language to the entire system. For information about enabling language packs, see Configuring Languages.
