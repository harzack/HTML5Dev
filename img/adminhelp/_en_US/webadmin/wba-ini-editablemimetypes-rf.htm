• Description:

    The [EditableMimeTypes] section is a list of MIME types that can be edited in Content Server using the Text Edit feature. For more information about the Text Edit feature, see Adding and Editing Documents.

  • Values:

    The following is an example of the [EditableMimeTypes] section in the opentext.ini file:

    [EditableMimeTypes]
    text/html=TRUE
    text/plain=TRUE
    text/tab-separated-values=TRUE
    text/xml=TRUE
    text/xsl=TRUE
    text/x-setext=TRUE
    text/x-sgml=TRUE
