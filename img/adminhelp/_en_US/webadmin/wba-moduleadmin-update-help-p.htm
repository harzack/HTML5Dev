To update the Admin and User Online Help:

  1. In the Search Administration section of the Administration page, click the Open the System Object Volume link.

  2. On the Content Server System page, click the Help Data Source Folder link or the Admin Help Data Source Folder link.

  3. Click the Help Data Flow Manager or Admin Help Data Flow Manager link.

  4. In the Process section on the Help Data Flow Manager page, click the Functions icon of the Directory Walker process, and then choose Start.

 

Note:
OpenText strongly recommends you delete and recreate the Help Data Source Folder and the Admin Help Data Source Folder whenever you add or remove language packs, or when the system default locale is modified.
