Stopping the servers under Windows

To stop the servers under Windows:

  1. Right-click the My Computer icon on the Desktop, and then choose Manage.

  2. Double-click the Services and Applications icon.

  3. Double-click the Services icon.

  4. To stop the Content Server (OTCS) service, right-click its name, and then choose Stop.

  5. To stop the Content Server Admin (OTCS) service, right-click its name, and then choose Stop.

Starting the servers under Windows

To start the servers under Windows:

  1. Right-click the My Computer icon on the Desktop, and then choose Manage.

  2. Double-click the Services and Applications icon.

  3. Double-click the Services icon.

  4. To start the Content Server (OTCS) service, right-click its name, and then choose Start.

  5. To start the Content Server Admin (OTCS) service, right-click the name of the Admin server, and then choose Start.
