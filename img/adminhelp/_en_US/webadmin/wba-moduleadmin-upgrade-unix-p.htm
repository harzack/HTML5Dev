To upgrade a Content Server module on UNIX:

  1. Copy the <module_name>.tar file to the <_home> directory of the Content Server instance on which the current module is installed.

  2. At the system prompt, type the following command, and then press ENTER:tar -xvf <module_name>.tar

  3. In the Module Administration section of the Administration page, click the Upgrade Modules link.

  4. On the Upgrade Modules page, select the check box beside the module you want upgraded, and then click the Upgrade button.

  5. Restart the server and Web application server, and then click the Continue link on the Restart Content Server page.
