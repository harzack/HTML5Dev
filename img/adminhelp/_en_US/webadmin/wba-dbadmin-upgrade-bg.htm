The Upgrade this Database link opens the Content Server Database Upgrade Confirmation page. The only circumstance under which you need to upgrade a database is when you connect the current Content Server installation to a database created with an earlier version of Content Server. In this case, the Content Server Database Upgrade Confirmation page appears automatically as part of the database selection procedure.

For a database that is up to date, the Content Server Database Upgrade Confirmation page displays the current schema version of the overall Content Server database and of certain administration modules that use tables in the Content Server database.

You can view the current schema version of the Content Server database and certain administration modules by clicking the Verify this Database link on the Maintain Current Database page.

The Content Server Database Upgrade Confirmation page appears when the following two conditions are met:

  • You are connecting the current Content Server installation to a database.

  • The database to which you are connecting needs to be upgraded in order to work properly with the current version of Content Server.

If these conditions are met, the Content Server Database Confirmation page appears after you have specified the database owner on the Specify Content Server Database Owner page and have logged in to the database on the Content Server Administrator User Log-in page.

The Database Upgrade Status page appears when you connect the current Content Server installation to a Content Server database that needs to be upgraded in order to work properly with the Content Server installation.

When the Database Upgrade Status page first appears, it displays the progress of the upgrade and refreshes its display every few seconds. When the upgrade is complete, the page displays the Continuelink.

Depending on whether this is a new or existing Content Server installation and whether the database was originally created on the current host, one of the following pages appears:

  • Install Modules

  • Default Admin Server Confirmation

  • Data Source Resynchronization Confirmation
