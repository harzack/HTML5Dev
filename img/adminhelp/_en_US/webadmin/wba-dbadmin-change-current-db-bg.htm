For a variety of reasons, you may want to change the Content Server database to which the current Content Server installation is connected. You have the option of connecting Content Server to a new or existing database. The first step in changing the database is to deselect the current database. If you want to save the indices of the current database, back up the contents of the directories shown.

Changing the Password that Content Server Uses to Connect to the Database

You can change the password used by Content Server to connect to the database on the Content Server Administration pages.
