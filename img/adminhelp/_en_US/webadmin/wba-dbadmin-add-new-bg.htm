The Select RDBMS Type page appears after you click the Create New Database link on the Database Administration page. You can specify either the Oracle Server or the Microsoft SQL Server (Windows only) as your RDBMS type.

The Select RDBMS Type page appears after you click the Select Existing Database link on the Database Administration page. You can select the database to connect to a new or existing Content Server installation.

Creating a Microsoft SQL Server Database

When you create a new Content Server database to connect to a new or existing Content Server installation, you access the Microsoft SQL Maintenance page from the Create Content Server Tables page. For more information about creating Content Server tables see, Creating Content Server Tables.

When you create a new database, you must perform the following tasks in the order shown, prior to returning to the previous page:

  • Create a SQL Server database, which will contain the Content Server database tables

  • Create a Microsoft SQL Server user, who will be able to log in to the SQL Server database
