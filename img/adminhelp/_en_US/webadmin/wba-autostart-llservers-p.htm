To modify the startup settings of a Server under Windows:

  1. Right-click the My Computer icon on the Desktop, and then choose Manage.

  2. Double-click the Services icon.

  3. Double-click the Services and Applications icon.

  4. In the Computer Management window, right-click the name of the server whose startup setting you want to modify and choose Properties.

  5. Click the Startup type drop-down list and do one of the following:

    • To set the server to manual startup, choose Manual, and then click OK.

    • To set the server to automatic startup, choose Automatic, and then click OK.

To modify the startup settings of a Server under UNIX:

  1. Log in to UNIX as root.

  2. Add the full path of start_llserver, which is in the ContentServer_home directory, to the boot script of the UNIX computer running the servers.

 

Note:
If you do not have root privileges or are unsure about modifying the boot script, consult your UNIX system administrator.

There are several other ways to set up the servers to start automatically on UNIX operating systems.
