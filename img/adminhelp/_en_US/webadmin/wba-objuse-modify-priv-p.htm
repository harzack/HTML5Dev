To modify object or usage privileges:

  1. Click the Administer Object and Usage Privileges in the System Administration section of the Content Server Administration page.

  2. On the Administer Object and Usage Privileges page, click Edit Restrictions beside the object or usage privilege that you wish to modify.

  3. To grant the privilege to new users or groups, search for the users or groups to whom you want to grant the object or usage privilege, enable Add to group beside the group or user name, and then click Submit.

    To remove the privilege from existing users or groups, click the name of the user or group on the left, and then click Remove From Group on the right.

  4. When you have finished making modifications, click Done.
