The [UndeletePurge] section controls the automatic purging of documents from the Deleted Documents Volume.

To modify this parameter, OpenText recommends that you use the Configure Autopurge page, rather than edit the opentext.ini file directly. For more information, see Configuring AutoPurge.

OlderThan

  • Description:

    Specifies the time, in days, after which a deleted document is purged from Content Server.

  • Syntax:

    OlderThan=7

  • Values:

    An integer. The default value is OlderThan=7.
