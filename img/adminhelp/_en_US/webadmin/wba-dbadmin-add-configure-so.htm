The Database Administration page appears when you:

  • Install a new Content Server instance after you click the Save Changes button on the Configure Server Parameters page.

  • Change the database to which an existing Content Server instance is connected after you click the Continue button on the DeselectContent Server Database page.

In either of these cases, you have the option of selecting an existing database or creating a new one.

A SQL Server database is created as part of the Content Server installation, so it is normally not necessary to create additional databases. However, you can create additional databases if necessary.
