To restore documents from the Deleted Objects Volume:

  1. Navigate to the deleted document using a Report from LiveReports by selecting Reports from the Personal global menu.

  2. On the My Reports page, click LiveReports. Next, select one of:

    • A custom report you created to access documents in the Deleted Objects Volume

    • Deleted Documents

    • Deleted Documents by [User]. If you select this option, you will be prompted to enter the userid of the user.

  3. From the list of documents in the LiveReport:Deleted Documents page, click the Functions icon of the deleted document that you want to restore, then click Restore.

  4. If the original path of the deleted document still exists, the Move to field on the Restore page displays that path by default. To change that path, click the Browse Content Server button.

    In the Move dialog box, navigate to the location where you want to restore the document, then click its Select link.

  5. Click the Restore button.

  6. To modify permissions, click the restored document's Functions icon, and then choose Permissions.
