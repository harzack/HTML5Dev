To limit the Admin account log-in by IP address:

  1. In the Server Configuration section of the Administration page, click the Limit the Admin Account Log-in link.

  2. On the Limit the Admin Account Log-in page, do one of the following:

    • To add an IP address, type it in the provided field, and then click the Add IP Address button.

    • To delete an IP address, click the address in the associated list, and then click the Delete button.

     

    Note:
    To allow Admin to log on locally on the Content Server computer, you must permit logon from the IP address 127.0.0.1.

     

  3. Click the OK button.

  4. Restart the Server and, if applicable, the web application server
