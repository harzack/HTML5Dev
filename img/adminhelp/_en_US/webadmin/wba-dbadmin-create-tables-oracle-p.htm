To create Content Server tables under an Oracle Server RDBMS:

  1. On the Create Content Server Tables page, click the radio button that corresponds with your RDBMS type.

  2. In the Create Content Server Tables page, click the name of the Oracle user that you want to own the tables in the User Name list.

  3. Type the password of the selected Oracle user in the Password field.

  4. If you want Content Server to store documents and other items outside the database, select the External Document Storage check box and then type the absolute path of the directory where you want the Content Server database to store items in the adjacent field.

  5. Click the Create Tables button.

    Depending on whether you selected internal or external storage and whether this is a new or existing Content Server installation, one of the following pages appears:

    • External Document Directory

    • Install Modules

    • Create New Update Enterprise Index
