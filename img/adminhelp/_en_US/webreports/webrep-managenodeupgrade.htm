The WebReports Node Administration page identifies existing WebReports using syntax which is out-of-date for the current release. WebReports that continue to use out-of-date syntax might not work as they did in previous releases. Updating the syntax using this utility will result in the selected WebReports being updated with the current syntax in order to maintain the same functionality.

Current reasons for upgrade are limited to:

  • New variable syntax usage

  • Syntax around LLURL:FORMEDIT needs updating for editable parameter

  • Syntax for Sort tag has changed
