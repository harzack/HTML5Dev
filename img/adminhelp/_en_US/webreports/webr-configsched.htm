It is not necessary for Content Server notifications to be enabled for WebReports scheduling to function. However, Content Server Change Agents must be enabled on your Content Server instance. The [options] section of the opentext.ini file for at least one instance of Content Server will contain EnableAgents=TRUE if Change Agents are enabled. Normally, Change Agents are enabled, but if you are unsure about how to configure Change Agents for your particular Content Server instance, please consult OpenText support.
