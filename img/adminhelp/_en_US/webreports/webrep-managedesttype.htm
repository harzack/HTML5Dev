To manage destination media types:

  1. On the Content Server administration page, under WebReports Administration, select Manage Destination Media Types.

  2. Optional If you want to remove a media type to which a WebReport can be output, click the “-” button next to the media type you want removed.

  3. Optional If you want to add a media type to which a WebReport can be output, click the “+” button at the bottom of the page. In the text field, type the new media type.

  4. Click Apply.
