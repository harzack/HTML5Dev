Use this page to manage which directories are used by Content Server Applications Management.

Use the Content Server Applications Directory field to set which directory on the server is used to store folders for installed applications.
Use the Content Server Applications Staging Directory field to set which directory on the server is used to store folders for applications to be installed or upgraded, or that have been uninstalled.

The directories should already exist on the server.

Click the Submit button to submit the form. Validation will be performed on the fields to ensure that the values entered are legitimate directories.
Click the Reset button to clear any changes made since the page was last refreshed.
