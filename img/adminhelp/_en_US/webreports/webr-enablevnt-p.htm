To enable events for WebReports:

  1. From the Content Server administration page, under System Administration section, click Administer Event Auditing.

  2. Click Set Auditing Interests and then select WebReports from the list.

  3. Check the interests as desired and then click Set Interests.

  4. RUN and EXPORT audit events are not enabled for WebReports by default. For each WebReport for which you want these auditing events to run:

    1. Select Properties from that WebReport's Functions menu.

    2. On the Properties page, select the Specific tab.

    3. Select Run/Export Audits Enabled.

    4. Click Update.
