By default, the WebReports conversion agent is configured to run every 5 minutes, or 300 seconds. In some instances, it might be desirable to have the agent to run in more frequent intervals. In this case, the agent can be set to run in intervals as short as 60 seconds. This can be set in the Manage WebReports Conversion page. The top section of this page contains a form that changes the sleep interval setting for the WebReports conversion agent.
