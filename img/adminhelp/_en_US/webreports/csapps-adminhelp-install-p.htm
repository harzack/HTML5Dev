To install an application:

  1. Copy the application folder to the <Content Server_home>\csapplicationsstaging directory on the server.

  2. On the Content Server administration page, under Content Server Applications Administration, click Applications Management.

  3. On the Applications Management page, select the Install box beside the name of the application you want to install.

  4. On the Install Application page:

    1. Optional If the application contains Content Server nodes then the form contains the Target Container field. Use the Browse For Container button to browse to a Content Server source object where the application's Content Server nodes will be installed.

    2. Optional Click Preview to see details of actions that will be performed on the target system by the installation process.

    3. Click Submit.
