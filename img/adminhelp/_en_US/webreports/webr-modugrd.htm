To upgrade the module from an earlier version:

  1. Ensure that you have first run the WebReports installer to place the modules into the <Content Server_home>/staging directory.

  2. Open the Content Server Administration page in your Web browser and login as the Content Server administrator user. Do not attempt to install this, or any other module containing database schema, unless logged in as the Content Server administrator.

  3. In the Module Administration section, click Upgrade Modules.

  4. Select the box beside Content Server WebReports Standard and click Upgrade.

  5. Restart the Content Server service when prompted and click Continue.
