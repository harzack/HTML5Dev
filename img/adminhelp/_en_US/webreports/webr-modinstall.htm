To install the module in Content Server:

  1. Ensure that you have first run the WebReports installer to place the modules into the <Content Server_home>/staging directory. You will now need to install the module in the Content Server instance. These next steps will guide you through the installation to Content Server.

  2. Open the Content Server Administration page in your Web browser and login as the Content Server administrator user. Do not attempt to install this, or any other module containing database schema, unless logged in as the Content Server administrator.

  3. Under the Module Administration section, click Install Modules.

  4. Check the box beside Content Server WebReports Standard and click Install.

  5. Restart the Content Server service when prompted and then click Continue.
