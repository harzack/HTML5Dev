To enable search integration:

  1. On the Content Server administration page, under WebReports Administration, select Manage Search Query Integration.

  2. On the Manage Search Query Integration page, select the Search integration is disabled... box.

  3. Click OK to confirm that you will be restarting the Content Server services. This feature is not activated until Content Server is restarted.

  4. In the Search Button Title field, provide a name for the button.

  5. Click Apply.

  6. Restart the Content Server services.
