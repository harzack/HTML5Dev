The Manage Category Data Source Configuration page allows you to define data source settings for the Category Data Source.

Category Data Source options include:

  • Maximum Number of Categories - a limitation on the number of categories that may be included in a query. Please note that exceeding a moderate number of categories for a query may result in query performance degradation.
  • Number of Attributes to Display Across - the number of attributes that should be displayed across each line. Keep this number low to ensure visibility of all attribute names without having to scroll across the page.
  • Multi-Valued Attribute Delimiter in Report Display Strings - the character(s) that will be used to separate multi-valued attributes on a report display.
  • Maximum Number of Results - The Maximum Number of Results that may be set by WebReport Designer for Category Data Source.
