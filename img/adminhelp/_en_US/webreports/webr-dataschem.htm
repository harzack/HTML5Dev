A Content Server database table called WEBREPORTS is created when WebReports is installed. The purpose of this table is to store WebReports schedule information. This table has a two column key: USERID and NODEID.NODEID is the nodeid of the webreport.

An additional table called WEBREPORTSTATS is created when the module is installed or upgraded.

 

Note:
Uninstalling the WebReports module does not remove these tables. In order to uninstall these tables, you need to access the WebReports Database Administration option under the Content Server administration pages prior to uninstalling the module.
