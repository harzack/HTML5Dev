The Manage Destination Media Types page can be used to add, edit and delete the destination media types that a WebReport can be output to.

Any existing destination media types will be displayed when the page is loaded. You can change the name of a media type by amending the data in the Media Type field for any media type.

Use the Add Row button () to add a new media type.

Use the Delete Row button () to delete an existing media type.

Note that although the content of the page will change when the Add or Delete Row buttons are clicked, your changes will not be saved until either the Update or Apply buttons is clicked.

Click on the Update button to save your changes and return to the WebReports Administration section of the admin index. Click on the Apply button to save your changes and stay on the current page. Click on the Cancel button to return to the WebReports Administration section of the admin index without saving your changes.
