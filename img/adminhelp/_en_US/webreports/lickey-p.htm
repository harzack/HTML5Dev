To enter the license key:

  1. On the Content Server Administration page, under WebReports Administration, select WebReports Licensing.

  2. Optional To initiate a 30 day trial, click the START TRIAL link.

  3. Enter the license key details exactly as provided, including:

    • Company Name

    • Expiration Date. This is only applicable with evaluation licenses.

    • Number of WebReport users. Note that this may not always be the same as the number of general Content Server users.

    • License Key

  4. Click Save Changes and Content Server will respond, confirming that a valid key has been provided.
