Each time a WebReport is run, a cached file is created for the current version of the reportview file. This functionality is similar to that used for WebForms. These cached views can be found at: <Content Server_hom>/support/webreports/views

Because it is sometimes desirable for users to execute multiple versions of the same WebReport, all versions of cached files are maintained in this folder. The administrator can delete the contents of this folder at any time. These files are created as soon as any user runs a WebReport for the first time.
