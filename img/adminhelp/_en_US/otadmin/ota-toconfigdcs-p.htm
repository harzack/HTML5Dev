To configure the Document Conversion Service (DCS):

  1. On the System Volume Object page, click an Admin server's Functions icon, choose Properties, and then choose Document Conversion Service.

  2. Do any of the following:

    • Click a value in the Log Level drop-down list to change the default log level (which is 1)

    • Type a location for the Temporary File in the Temporary File Directory field to specify the Temporary File directory

    • Click a value in the Conversion Timeout drop-down list to change the default conversion timeout setting (which is 30 seconds)

    • Click a value in the Number of Threads drop-down list to change the default number of document conversion threads (which is 2)

    • Click a value in the Queue Size drop-down list to change the default queue size (which is 50)

  3. Click the Update button.

 

Note:
For more information about configuring DCS settings, see Setting Up the Document Conversion Service.
