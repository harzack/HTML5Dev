To configure search index verification:

  1. On the Administration page, in the Search Administration section, click Open the System Object Volume.

  2. On the System Volume Object page, click the Enterprise Data Source Folder's Functions icon, and then choose Maintenance.

  3. On the Data Source Maintenance page, click the Configure search index verification radio button.

  4. Click the OK button.

  5. On the Configure Index Verification page, do one of the following, based on the status of your system:

    • Click the Run Verify Again button to perform the index verification again.

    • Click the Re-extract Missing Objects button to re-export objects that are missing from the index.

    • Click the OK button to return to the System Volume Object page.

 

Note:
The status of the index verification is displayed on the Search Index Verify Log page for one week after the verification completes.
