To edit a Backup Manager:

  1. On the System Volume Object page, click theprocesses_prefix Data Source Folder link of the data source folder whose Backup Manager you want to change.

  2. Click the Backup Manager's Functions icon, choose Properties, and then choose Specific.

  3. On the Specific tab of the Backup Manager's Properties page, edit the parameters of the Backup Manager.

  4. Click the Update button.
