Modifying and Deleting Prospectors Importer Processes

You can modify a Prospectors' importer process to enable or disable system management, change the Admin server on whose host the process runs, or change when the Prospectors importer process runs. For more information about editing importer processes, see To Configure an Importer Process.

Regenerating Prospectors Query Files

You must regenerate the Prospector Query files if the Query files are deleted or become corrupt.
