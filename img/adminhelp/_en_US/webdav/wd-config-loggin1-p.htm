To specify the logging level:

  1. On the Content Server Administration page, click Specify Logging Level in the Weblink Coserver Configuration area.

  2. On the Specify Weblink Coserver Logging Level page, click one of the following logging options in the Logging Level list:

    • Fatal errors only

    • Errors only

    • Errors and warnings

    • Full debugging information

    • Complete connection dump

  3. Click Save Changes.

 

Note:
You must restart Content Server for these changes to take effect.
