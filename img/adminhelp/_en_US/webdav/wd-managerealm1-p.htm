To allow all authenticated users:

  1. On the Content Server Administration page, click Specify Weblink Authentication in the Weblink Coserver area.

  2. Select Allow all authenticated users.

  3. Click Save Changes, and then restart the Content Server server.

To accept users from specified realms:

  1. On the Content Server Administration page, click Specify Weblink Authentication in the Weblink Coserver area.

  2. Select Accept users from the following realms.

  3. To add a realm, type the name of the realm in the Add Realm field and then click Add Realm.

  4. To delete a realm, select a realm from the Delete field and then click Delete.

  5. Click Save Changes, and then restart the Content Server server.
