When Custom authentication is specified, WebDAV authenticates using an Adminstrator specified Java plug-in class, implemented in compliance with the WebDAV Authentication Plug-in API.

To configure this authentication method, the class name and .jar file for the plug-in must be provided.

 

Note:
For detailed information on Custom authentication, see "Configuring Custom Authentication" in the OpenText WebDAV Installation and Administration Guide.
