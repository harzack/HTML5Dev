To specify a static coserver port:

  1. On the Content Server Administration page, click Specify Coserver Port in the Weblink Coserver Configuration area.

  2. On the Specify Weblink Coserver Port page, click Static in the Port Allocation list, and then type the port number you want to use in the Port Number field that opens.

  3. Click Save Changes.

 

Note:
You must restart Content Server for these changes to take effect.
