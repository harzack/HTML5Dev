To configure IIS as a local Web server during an install or upgrade:

  1. On the Configure Local Web Server page, click Microsoft Internet Information Server (IIS) in the Web Server drop-down list.

  2. To configure the IIS Web server, click Run the Installer.

  3. During the installation, confirm the Web server, the Content Server service name, and the virtual directory alias, and then complete the Web server configuration.

  4. On the Configure Local Web Server page, click Continue.

  5. Complete the module installation.

     

    Notes:

    • You must run the installer on the same machine that hosts the IIS Web server.

    • If you save the installer to disk to run at a later time, do not rename the file because the name contains the parameters necessary for the installer to configure the Web server.
