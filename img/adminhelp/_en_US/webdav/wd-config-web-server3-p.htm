To configure a different remote Web server for Windows:

  1. On the Content Server Administration page, click Configure Web Server.

  2. On the Configure Web Server page, click Configure Remote Web Server.

  3. On the Configure Remote Web Server page, click a Web server in the Web Server list.

  4. Type the Content Server IP address in the IP Address of this Content Server field.

  5. Click Windows.

  6. Click Run the Installer to configure the Web server or save the installer to disk.

     

    Notes:

    • You must run the installer on the same machine that hosts the Web server.

    • If you save the installer to disk to run at a later time, do not rename the file, because the name contains the parameters necessary for the installer to configure the Web server.
