A data flow is a set of linked processes that process data from a single source in Content Server. There are seven types of data flow processes you can configure: the Enterprise Extractor Process, the Directory Walker Process, the XML Activator Process, the Document Conversion Process, the Importer Process, the Merge Process, and Proxies. Each data flow has a Data Flow Manager, which manages all the processes that make up the data flow. A Data Flow Manager is also a Folder that contains all the processes in a data flow.

After you add the processes that make up a data flow, you can configure them individually.
