The file dcsviewrules.txt contains the rules for viewing items, finding similar items, hit highlighting search results, and synopsis and profile generation. Each rules file also determines the way the DCS behaves if errors occur during text extraction or conversion. The OpenText Document Filters installation program does not overwrite the dcsviewrules.txt file, but sample files with suggested configurations are provided for you as examples in the Content Server_home\config\config-reference folder.

Most converted documents are displayed in View as Web Page, as raster images using the try DCSIMImageView parameter. Word, Excel, or RTF documents are displayed as text which can be searched and indexed using the try DCSIMTextView parameter.
