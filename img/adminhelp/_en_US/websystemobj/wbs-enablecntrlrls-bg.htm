Enabling a control rule activates the rule in the system. This means that Content Server begins monitoring the site for the conditions that you have defined in the rule and takes the appropriate actions when those conditions are met. When you add a control rule for the first time, it is disabled by default. When you are ready to activate the rule, you enable it.

You disable a control rule when you want to temporarily deactivate the rule in the system. For example, you disable a control rule before editing it. For information about editing a control rule, see Administering Control Rules.
