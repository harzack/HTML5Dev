OpenText Document Filters is a core module of Content Server 10.5, and is installed automatically on 64-bit Linux environments.

The following libraries are required, so you must verify the following libraries are installed:

  • Red Hat Enterprise Linux (RHEL 6) (libXfont)

  • xorg-x11-server-Xvfb-1.1.1-48.76.el5

  • Xrender (libXrender-0.9.1-3.1)
