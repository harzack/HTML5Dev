When Content Server users perform searches, Content Server temporarily caches the search results in the Content Server database. Content Server uses the cached search results to allow users to switch between show summary and hide summary views and to save their results without having to rerun the search.

You can set the amount of time that Content Server caches search results in the Content Server database. The default setting is 30 minutes. After that time expires, Content Server clears the search results from the database cache.
