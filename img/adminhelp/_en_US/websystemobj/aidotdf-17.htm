The language support for OpenText Document Filters (OTDF) is based on your Content Server installation, and provides text extraction for the formats specified in the latest Content Server Release Notes. Text Extraction is supported for all languages with Unicode-based formats such as Microsoft Office formats.

The supported language groups are format specific for non-Unicode-based formats:

  • Western

  • Eastern European

  • Far Eastern

  • Arabic

  • Hebrew (for Word 2007 and 2010 documents)

  • Thai (for Excel and PowerPoint 2007 to 2013 documents) for Windows only

  • Other groups

 

Note:
On UNIX operating systems, only Western languages are supported for View as Web Page - Text and for View as Web Page - Raster.
