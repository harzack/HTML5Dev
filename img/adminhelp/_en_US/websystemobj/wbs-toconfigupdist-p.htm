To configure an Update Distributor process:

  1. On the System Object Volume page, click the processes_prefix Data Source Folder link of a data source that contains the Update Distributor process that you want to configure.

  2. Click the processes_prefix Data Flow Manager link.

  3. On the Data Flow Manager page, click the Functions icon of the Update Distributor process , and then choose Stop Indexing Processes.

  4. Click the Functions icon of the Update Distributor process, choose Properties, and then choose Specific.

  5. On the Specific tab of the Update Distributor Properties page, edit the parameters of the Update Distributor process.

  6. Click the Update button.

  7. On the Data Flow Manager page, click the Functions icon of the Update Distributor process that you configured, and then choose Start Indexing Processes.

 

Tip:
You can also configure the Update Distributor process when you view the partition map to which the process belongs.
