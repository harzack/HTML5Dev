To start or stop a partition map:

  1. On the Administration page, under Search Administration, click Open the System Object Volume.

  2. On the System Object Volume page, click the processes_prefix Data Source Folder link of the data source folder that contains the partition map that you want to stop.

  3. On the Data Source Folder page, click a partition map's Functions icon, and then choose one of the following:

    • Start, which starts all indexing and searching processes

    • Stop, which stops all indexing and searching processes

    • Restart, which stops and then starts all indexing and searching processes
