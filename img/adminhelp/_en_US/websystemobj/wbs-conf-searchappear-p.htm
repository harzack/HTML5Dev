To create and manage search bar layouts:

  1. On the Administration page, in the Appearances Administration section, click the Search Bar Administration link.

  2. To configure default search bar modes, click the Edit button to open the Search Bar Edit page.

  3. On the Search Bar Edit page, select options to enable, disable, and change settings of the behavior of Content Server Search for users.

For more information about the Search Bar Edit page, see Configuring the General Settings.
