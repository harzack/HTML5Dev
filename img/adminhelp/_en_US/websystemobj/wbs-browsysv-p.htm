To access the System Administration page in the System Object Volume:

  1. On the administration page, click the Open the System Object Volume link.

  2. If prompted, type the user name of a Content Server user that has system administration rights in the Username field, type the corresponding password in the Password field, and then click the Log-in button. If you have already logged in as a user with system administration rights during the current Web browser session, Content Server does not prompt you to log in. For more information about using the System Object Volume, see Using the System Object Volume.
