To add a data source folder:

  1. On the System Object Volume page, click Data Source Folder on the Add Item menu.

  2. On the Add: Data Source Folder page, type a name for the data source folder in the Name field.

  3. To provide a description of the data source folder on the General tab of its Properties page, type descriptive text in the Description field.

  4. To modify the categories and attributes associated with this item, click the Edit button beside the Categories field.

  5. Click the Add button.
