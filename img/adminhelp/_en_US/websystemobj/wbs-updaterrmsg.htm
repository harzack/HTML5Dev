The following table describes error messages for the Update Distributor process.

If these errors persist, contact OpenText Customer Support for additional information.

Table 1-29: Error Messages for Update Distributors

Number Message Description Remedy
129 Unable to load JNI library The Update Distributor process cannot load the dynamically linked file jniipool.dll (on Windows) or libjniipool.so (on Solaris). Ensure that the jniipool.dll or libjniipool.so file exists in the bin directory and is readable and executable. The file should be part of your Content Server distribution.
If the file does not exist or is empty, contact OpenText Customer Support.
131 Insufficient memory space available The Update Distributor process ran out of memory. Increase the amount of memory allocated to the Update Distributor process.
132 An unknown error has occurred The Update Distributor process encountered an unknown error. Contact OpenText Customer Support.
149 Invalid command line for the Update Distributor There is an error in the Update Distributor process' command line parameters. Ensure that the command lines in otadmin.cfg are valid. and specified in the proper format Content Server generates otadmin.cfg using its database.
150 URL is invalid The URL used to register and contact the Update Distributor process is invalid as specified in the search.ini file. Ensure that the search.ini file parameters are valid and specified in the proper format.
152 Invalid partition specification specified The Update Distributor process received an invalid partition specification. Ensure that the partition specifications in the search.ini file are valid and specified in the proper format.
153 Unable to initialize The Update Distributor process failed to start because one or more of the parameters in the search.ini file are invalid. Ensure that the search.ini file parameters are valid and specified in the proper format.
170 Insufficient memory space available The Index Engine process ran out of memory. Increase the amount of memory allocated to the Index Engine process.
171 Unable to contact Index Engine The Update Distributor process is unable to contact one or more Index Engines. Restart the Update Distributor and Index Engines.
172 Insufficient disk space for the Index Engine The Index Engine process ran out of disk space. Increase the amount of disk space available to the Index Engine and Update Distributor.
173 All partitions full All specified partitions are full. The Update Distributor process is unable to process update or addition requests. Add a new partition to the grid. For more information about adding partitions, see To Add a Partition.
174 An IPool error has occurred The Update Distributor process encountered an IPool error. Contact OpenText Customer Support.
 

Note:
The search.ini file is the configuration file for Content Server searching and indexing processes. This file contains settings that are usually set by default when indexing and searching processes and other system objects, such as data sources and partitions, are created; conversely, settings are also deleted when the processes they configure are deleted. You configure these settings when you configure searching and indexing processes and other system objects in Content Server. OpenText strongly recommends that you do not modify the settings in the search.ini file. If you do choose to modify the settings, contact OpenText Customer Support.
