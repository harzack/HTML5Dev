To configure a partition map:

  1. On the Administration page, under Search Administration, click Open the System Object Volume.

  2. On the System Object Volume page, click the processes_prefix Data Source Folder link of the data source folder that contains the partition map you want to configure.

  3. Click the partition map's Functions icon, choose Properties, and then choose Specific.

  4. On Specific tab of the Partition Map Properties page, edit the parameters of the partition map.

  5. Click the Update button.
