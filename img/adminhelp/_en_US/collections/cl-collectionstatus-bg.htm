The Collection Status section of the Collections General Settings page identifies Collection operations active in the background. Shutting down Content Server will terminate the operations displayed here. Click the Send E-mail Notice button to send an email with a warning to notify the initiator of each Collection operation that their process will be interrupted.

Collection Status
Active
Displays the following information about each Collection process currently running:
•

Name – name of the Collection

•

Location – Content Server location where the documents or items are being collected

•

Size – number of items in the Collection

•

Task – type of Collection option or action that is running

•

Start Date – date and time the Collection process started

•

Initiating User – name of the user creating the Collection

 

Note:
If Content Server is shut down, these Collection processes will terminate.

 
Click the Send E-mail Notice button to send an email with a warning to notify the initiator of each Collection operation about a potential interruption.
 

Note:
These initiating Users must have their E-Mail or Alternative E-Mail fields filled out on their My Profile page. For details, see Working with Users and Groups and Configuring Mail Settings.
