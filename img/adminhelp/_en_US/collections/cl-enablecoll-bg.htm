By default, Collections is enabled, and all Users have the ability to create Collections by selecting Collection from the Add Item menu. However, you can restrict which Users and Groups can access the Collection menu option from the Administer Object and Usage Privileges page in the administration pages. For example, if there is a specific Group that is responsible for your Content Server architecture, you can restrict the right to create Collections to this Group only. For more information on adding items to a Collection, see Collecting Items.
