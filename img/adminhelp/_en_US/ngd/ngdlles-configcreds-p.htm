To configure how Enterprise Connect credentials are stored:

  1. On the Content Server Administration page, click Enterprise Connect Administration, and then click the Configure Enterprise Connect System Settings link.

  2. In the Credentials Management Configuration area, do one of the following:

    • Select the Allow Enterprise Connect client to store user credentials on disk check box if you want to store user credentials locally. By default, the check box is selected.

    • Clear the Allow Enterprise Connect client to store user credentials on disk check box if you do not want to store user credentials locally.

  3. When you have made your selection, do one of the following:

    • Click Submit to apply the changes.

    • Click Reset to return to the last saved configuration.

    • Click Set Default to return to the default configuration, and then click OK to apply the default settings.
