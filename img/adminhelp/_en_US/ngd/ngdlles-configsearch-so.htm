You can perform the following tasks on the Configure Enterprise Connect Search Settings page:

  • Configure search profiles.

  • Specify the number of returned items displayed on each page when users perform a search.

  • Include Shortcuts and Cross-References in search results.

 

Important:
Users must initiate a new Enterprise Connect session in order to access any administrative changes that you make.
