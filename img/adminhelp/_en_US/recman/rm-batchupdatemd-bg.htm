Metadata is the term used to describe a characteristic of a managed object. For example, Status, Status Date, Storage Medium, and RSI are pieces of information assigned to a managed item. The metadata that appears for RM classified objects is configured on the Records Management Administration pages.

You can update the Status, Essential, Storage Medium, and RSI metadata values for multiple managed objects that have a specific RM Classification assigned. When you perform a batch update on RM Classification metadata, alll items with that RM Classification assigned, including items classified with the RM Classification’s sub-items, are updated with the new value that you specify. The number of classified items and sub-items is displayed once you perform the update.

If the Status parameter is updated using the batch update process, the item’s ACL permissions are also updated if the Status code is configured to update ACL permissions. For more information about ACL permissions, see Understanding Permissions.
