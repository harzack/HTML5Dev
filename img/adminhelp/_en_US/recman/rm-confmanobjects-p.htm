To configure managed objects:

 

Note:
You can configure the Managed Object options at any time, but the options are not enabled until the Managed check box is selected for that object type.

 

To... On the Content Server Administration page, click the Managed Objects link in the Records Management Administration section, and then do the following...
Enable object management Select the Managed check box beside any of the object types, and then click the Update button.
Enable child objects to inherit parent object RM Classifications Select the Inherit from check box beside any of the container object types, and then click the Update button.
Ensure the downgrade of permissions if object is marked as Official Select the Downgrade Permissions check box beside any of the object types, and then click the Update button.
Enable the Titling feature Select the Use Titling check box beside any object type, and then click the Update button.
Select which item types can be added to user based Holds Select the User Hold check box beside any object type, and then click the Update button.
Set the default RM Classification for a specific object type Click the Select button, browse to a classification, click the Select link for a Classification, and then click the Update button.

 

Note:
If you modify any Managed check boxes, you must restart the Content Server services for the changes to take effect.
