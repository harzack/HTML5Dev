1. In the Records Management Administration section of the Content Server Administration page, click the Managed Objects link.

  2. On the Managed Objects page, click the Add Item menu , and then select the object type you want to add.

 

Important:
When you add a managed object type to the Managed Objects page, it is not automatically managed. You must select the Managed check box for each object type to enable the selected Managed Object options.

 
 

Note:
Once you add an additional managed object to the Managed Objects page, you cannot delete it. If you no longer need to manage that object type, clear the Managed check box and restart the Content Server services.
