To change deleted user ownership:

  1. On the Content Server Administration page, click the Change Ownership link in the Records Management Administration section.

  2. Click a user in the Deleted User drop-down list.

  3. Search for a new owner by clicking the Find button beside the New Owner field, and then performing a search on the Select a User page.

    Optionally, select the Classification Only check box to have the change in ownership apply to only the classification.

  4. Click the Submit button.

 

Note:
Deleted users who have not created objects are not returned in the Deleted User drop-down list.
