To batch update RM Classification metadata:

  1. On the Content Server Administration page, click the Batch Update RM Classification Metadata link in the Records Management Administration section.

  2. In the RM Classification list, select the RM Classification that is assigned to the items for which you want to update the metadata.

    The File Number and Name fields are automatically populated when you select an RM Classification.

  3. In the Metadata Field area, select a new value for each existing metadata field you want to change, select the corresponding Update check box, and then click Update.

     

    Note:
    When you click the Update button, a dialog window displays asking if you want to continue with the update. When you click Yes, the dialog window closes and a progress indicator appears showing the progress of the update. You can navigate away from the page while the update is in progress.
