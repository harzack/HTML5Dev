To configure Auto Purge:

  1. Click the Configure Auto Purge link in the Recycle Bin Administration section on the Content Server Administration page.

  2. On the Configure Auto Purge page, modify any of the Auto Purge settings as described in the following table.

    To... Do the following...
    Enable the Scheduler Click Enable beside Status in the Auto Purge section.
    Schedule Auto Purge Specify the days and times you want to run Auto Purge in the Schedule section.
    Specify how many days you want Recycle Bin to retain Content Server items Enter a number in the Days Old box.
    Enable a Delete Limit Click Enable beside Status in the Delete Limit section.
    Delete Limit Threshold Specify the maximum number of root items you want Recycle Bin to delete in the Number of Root Items box.
     

    Tip:
    A root item is an object that lives directly under the day-level folders.

     

     

    Note:
    If Auto Purge is not performing in accordance with its configured settings, disable the Auto Purge Scheduler and then restart the Content Server service. This prevents any further Auto Purge actions.
