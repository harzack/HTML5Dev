You can modify Auto Purge configuration settings for the Content Server Recycle Bin module from the Configure Auto Purge page.

  • Auto Purge allows you to enable, disable, and schedule Auto Purge activities for your Content Server object types. For example, you can set Auto Purge to run during off hours or to go on every five minutes.

  • Delete Limit allows you to specify how many items you want Recycle Bin to delete at one time. The default setting is 1000 items.
