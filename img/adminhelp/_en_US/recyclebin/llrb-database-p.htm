The information in the Status and Action columns indicates the actions that are necessary to ensure that the Recycle Bin schema objects are in the correct state:

  • If the Status is missing, a create link appears beside the Table Name. Click create to create the schema object.

  • If the status is schema change, an update link appears beside the Table Name. Click update to update the schema object.

     

    Tip:
    After you click update, the status of the schema object becomes present.

     

  • If the status is present, a delete link appears beside the Table Name. No action is necessary. If you want to delete the schema object, click delete.

     

    Tip:
    To delete all of the Recycle Bin schema objects, the best practice is to start at the bottom of the page and work up.
