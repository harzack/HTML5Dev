The Recycle Bin schema objects are essential to the functioning of Recycle Bin:

  • If you have installed Recycle Bin for the first time, the schema objects must exist before deleted Content Server items can be stored in the Recycle Bin.

  • If you have upgraded Recycle Bin, you may need to upgrade the Recycle Bin schema objects to ensure that deleted Content Server items can be stored in the Recycle Bin

The Recycle Bin Database Administration page is where you create, update and delete Recycle Bin schema objects.
