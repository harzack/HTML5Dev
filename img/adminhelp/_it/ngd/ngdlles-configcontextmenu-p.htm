To configure the context menu:

  1. On the Content Server Administration page, click Enterprise Connect Administration, and then click the Configure Enterprise Connect Advanced Settings link.

  2. In the Action Configuration area, in the Exclude these actions from the context menu box, enter the command ID of any additional context menu items that you do not want Enterprise Connect users to access.

     

    Notes:

    • Use a comma to separate the command IDs.

    • Do not use spaces between the commas and the command IDs.

    • Command ID values are not case sensitive.

     

  3. When you have specified the appropriate options, do one of the following:

    • Click Submit to apply the changes.

    • Click Reset to return to the last saved configuration.

    • Click Set Default to return to the default configuration, and then click OK to apply the default settings.
