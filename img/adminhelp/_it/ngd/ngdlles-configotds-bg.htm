Note:
This procedure only applies if you are using Enterprise Connect Client version 10.2.3 or earlier. This setting is not required for Enterprise Connect Client version 10.3 or higher.

 

If you are using Enterprise Connect Client version 10.2.3 or earlier, and if you are using CA SiteMinder or RSA Access Manager with Content Server and OpenText Directory Services (OTDS), you must select the Allow Content Server to retrieve OTDS ticket from OTDS server (Only for Enterprise Connect Client 10.2.3 or earlier) check box. By default, this option is not selected.
