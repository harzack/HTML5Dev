To specify which subtypes can be downloaded to the desktop:

  1. On the Content Server Administration page, click Enterprise Connect Administration, and then click the Configure Enterprise Connect Advanced Settings link.

  2. In the Download to Desktop Configuration area, in the Include Subtypes box, enter the values of any additional subtypes that you want users to be able to download to their desktop. Use a comma to separate the values. The following values are provided by default, and you can remove them as required:

    • 0 for Folders.

    • 136 for Compound Documents.

    • 144 for Documents.

    • 202 for Projects.

    • 298 for Collections.

    • 557 for Compound Emails.

    • 628 for Favorites.

    • 749 for Email.

    • 751 for Email Folders.

  3. When you have made your selection, do one of the following:

    • Click Submit to apply the changes.

    • Click Reset to return to the last saved configuration.

    • Click Set Default to return to the default configuration, and then click OK to apply the default settings.
