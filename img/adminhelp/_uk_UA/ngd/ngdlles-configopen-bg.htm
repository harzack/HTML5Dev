You can configure how Microsoft Office documents open when users double-click them in Enterprise Connect, in either Windows Explorer or Microsoft Outlook. This setting also applies to Shortcuts that point to Office documents. The Always open document as read-only option is the default setting, and you can use this option if you want users to open a read-only version when they double-click a Word, Excel, PowerPoint, Project, or Visio document. Use the Open document for editing if permissions allow setting if you want users to be able to immediately edit Office documents that they open by double-clicking.

 

Note:
Users can only edit documents for which they have Modify and Reserve permissions.
