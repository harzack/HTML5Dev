重要項目:
ショートカットを検索結果に含めるには、Content Server 10 アップデート 7 以降を使用する必要があります。相互参照を検索結果に含めるには、OpenText Records Management (RM) 10.2.0 パッチ 1 以降を使用する必要があります。使用している Content Server と RM のバージョンがそれよりも古い場合、このチェック ボックスは表示されません。

 

Enterprise Connect では、ショートカットと相互参照の検索がサポートされています。この機能を使用可能にするには、必要なバージョンの Enterprise Connect、Content Server、および RM がシステムにインストールされていなければなりません。また、Content Server は、この機能が有効となるように設定されていなければなりません。RM の操作の詳細については、Open Text Records Management - インストールおよび管理ガイド (LLESRCM-IGD)を参照してください。Content Server の設定の詳細については、Open Text Administration Help - Content Server (LLESCOR-H-AGM)を参照してください。

この機能を有効にした場合、ユーザーは、ショートカットと相互参照を自分の Enterprise Connect のクイック検索と詳細検索のどちらにも含めることができます。例えば、検索に含まれるフォルダーに他のフォルダーへのショートカットまたは相互参照が含まれる場合、参照先のそれらのフォルダーが検索結果に含められます。
