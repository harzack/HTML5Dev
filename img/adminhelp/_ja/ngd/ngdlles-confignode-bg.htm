Enterprise Connect ツリー構造 の [サーバー] ノードに表示されるノードを選択できます。また [ボリューム] フォルダーと [その他のアイテム] フォルダーにアイテムを追加することもできます。さらに、アイテムが ツリー構造 に表示される順序を変更できます。デフォルト設定が指定されますが、必要に応じて変更できます。

 

注意:
ドメイン ワークスペースを使用している場合は、ユーザーが利用できるようにするため、それを [利用可能なアイテム] リストから [サーバー ノード] リストに移動する必要があります。そのようにすることにより、ドメイン ワークスペースが Enterprise Client ツリーから見える状態になります。それは、ユーザーがメンバとして属するドメインに対するものです。

 

Enterprise Connect ユーザーが、Enterprise Connect ツリーのパブリック電子メール ボリューム、パーソナル電子メール ボリューム、および委任された電子メール ボリュームにアクセスできるようにするには、[利用可能なアイテム] リストの [電子メール ボリューム] ノードを [サーバー ノード] リストに移動します。Email Archiving のサポートのインストールおよび設定が正常になされている場合、[電子メール ボリューム] ノードは [使用可能なアイテム] のリストにのみ表示されます。

 

注意:
管理権限のある Enterprise Connect ユーザーは、システム電子メール ボリュームにもアクセスできます。

 

Enterprise Connect ツリー構造 の [パーソナル] ノードに表示されるアイテムを指定することもできます。デフォルトでいくつかのアイテムが表示されますが、必要に応じてアイテムを追加または削除できます。利用可能なアイテムには、Web UI の [パーソナル] メニューに表示されるアイテムも含まれます。ただし、ご使用の環境にインストールされている Content Server モジュールに応じて、利用可能なアイテムが変わる場合があります。いくつかのアイテムについては、Web UI 表示が確認できます。
