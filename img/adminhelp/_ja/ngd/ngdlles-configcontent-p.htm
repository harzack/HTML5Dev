コンテンツの取り込みの設定を行うには

  1. [Content Server 管理] ページで [Enterprise Connect の管理] をクリックし、次に [Enterprise Connect 詳細設定] リンクをクリックします。

  2. [メタデータの取り込みの設定] 領域の [コンテンツの追加時にメタデータの入力を促す] リストで次のいずれかの操作を実行し、Content Server に追加したアイテムのメタデータを入力するようにユーザーに要求するタイミングを指定します。

    • 宛先の Content Server フォルダーでメタデータが必須として指定されている場合に限り、[必要時のみ] をクリックして、ユーザーにメタデータの入力を要求します。これはデフォルトの設定です。

    • ユーザーが Content Server に追加したアイテムごとにメタデータの入力を求める場合は、[常に表示] をクリックします。

  3. [メタデータの取り込みの設定] 領域の [Content Server 内でのコンテンツの移動/コピー時に追加モジュールからのメタデータの入力を促す] リストで次のいずれかの操作を実行することによって、他の Content Server モジュールがインストールされている環境において、メタデータを入力するようにユーザーに要求するタイミングを指定します。

    • Content Server 内でアイテムをコピーまたは移動するたびに、毎回、メタデータの入力をユーザーに求める場合は、[常に表示] をクリックします。これはデフォルトの設定です。

    • ユーザーが Content Server 内でアイテムをコピーまたは移動する際に、ソースと宛先のカテゴリ要件が異なる場合にのみ、メタデータの入力を求めるには、[表示しない] をクリックします。

  4. 選択が完了した後、次のいずれかの操作を実行します。

    • [送信] をクリックして変更を適用します。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • デフォルト設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックしてデフォルト設定を適用します。
