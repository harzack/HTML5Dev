コンテキスト メニューを設定するには

  1. [Content Server 管理] ページで [Enterprise Connect の管理] をクリックし、次に [Enterprise Connect 詳細設定] リンクをクリックします。

  2. [アクションの設定] 領域の [コンテキストメニューからこれらのアクションを除外する] ボックスに、Enterprise Connect ユーザーがアクセスしないようにする追加のコンテキスト メニュー アイテムのコマンド ID を入力します。

     

    注意:

    • コマンド ID を区切るには、コンマを使用します。

    • コンマとコマンド ID の間にスペースを入力しないでください。

    • コマンド ID の値は大文字と小文字を区別しません。

     

  3. 適切なオプションを指定した後、次のいずれかの操作を実行します。

    • 変更を適用するには、[送信] をクリックします。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • 既定の設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックして既定の設定を適用します。
