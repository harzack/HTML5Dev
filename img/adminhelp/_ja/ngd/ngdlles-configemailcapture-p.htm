電子メールの添付ファイルを取得するための設定を行うには

  1. [Content Server 管理] ページで [Enterprise Connect の管理] をクリックし、次に [Enterprise Connect の電子メール設定] リンクをクリックします。

  2. [電子メールの取り込みの設定] セクションに移動します。

  3. 電子メールのサブタイプを操作する場合は、[電子メールを、電子メールのサブタイプとして保存] をクリックして電子メールのサブタイプ (749) を電子メールに適用します。これがデフォルトの設定です。[添付付きの E メールを追加する場合] リストから次のいずれかのオプションを選択して、添付ファイルを保存する方法を指定します。

    • 電子メールと添付ファイルを 1 つのアイテムとして保存する場合は、[添付の個別コピーを追加しない] をクリックします。これがデフォルトの設定です。

    • 電子メールと添付ファイルを単一のアイテムとして保存し、さらにすべての添付ファイルを別個のアイテムとして保存する場合は、[添付の個別コピーを追加する] をクリックします。

    • 添付ファイルの保存方法の決定をユーザーに許可する場合は、[ユーザーの決定を確認する] をクリックします。ユーザーは電子メールと添付ファイルを単一のレコードとして保存するか、または電子メールと添付ファイルを一緒に保存してから、添付ファイルの追加のコピーを個別に保存することができます。

  4. 重複電子メールの特定方法を指定するには、次の操作を実行します。

    • Enterprise Connect でメッセージ ID の比較を使用して重複電子メールを特定するには、[重複する電子メールの確認には、メッセージ ID を使用します。] チェック ボックスをオンにします。既定ではこのチェック ボックスはオフになっているため、名前の比較のみが使用されます。

    • Enterprise Connect で外部 ID の比較を使用して重複電子メールを特定するには、[重複する電子メールの確認には、外部メッセージ ID を使用します。] チェック ボックスをオンにします。既定では、このチェック ボックスはオフになっています。

       

      注意:
      [重複する電子メールの確認には、メッセージ ID を使用します。] チェック ボックスがオンになっている場合に限り、このオプションを選択できます。

       

  5. コンパウンド電子メールのサブタイプを操作する場合は、[電子メールを添付の有無にかかわらずコンパウンド電子メールとして保存] をクリックしてコンパウンド電子メールのサブタイプ (557) を適用します。

  6. 選択が完了した後、次のいずれかの操作を実行します。

    • [送信] をクリックして変更を適用します。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • デフォルト設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックしてデフォルト設定を適用します。
