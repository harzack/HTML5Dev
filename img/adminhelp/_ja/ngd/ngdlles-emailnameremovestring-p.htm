電子メール名から文字列を削除するには

  1. [Content Server 管理] ページで [Enterprise Connect 管理] をクリックし、次に [Enterprise Connect電子メールの設定] リンクをクリックします。

  2. [電子メールで使用される名前の生成] 領域に移動します。

  3. [削除する文字列] 領域の [新規文字列] ボックスに、電子メール名から除外する文字列の名前を入力し、[右へ移動] をクリックして文字列を [選択された文字列] リストに移動します。選択された文字列をこのリストから削除するには、[削除] をクリックします。

    例: 電子メール名に RE: が表示されないように指定するには、[新規文字列] フィールドに RE: を入力し、次に [右へ移動] をクリックして RE: を [選択された文字列] リストに追加します。 RE: は、電子メール名に表示されなくなります。RE: が電子メール名に表示されるようにするには、RE: を選択して [削除] をクリックします。 RE: は [選択された文字列] リストから削除されます。

  4. 選択が完了した後、次のいずれかの操作を実行します。

    • 変更を適用するには、[送信] をクリックします。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • 既定の設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックして既定の設定を適用します。
