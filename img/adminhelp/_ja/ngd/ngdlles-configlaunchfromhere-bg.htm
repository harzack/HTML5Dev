ユーザーが Content Server Web インタフェースで特定のコンテンツ タイプを操作している時に、ユーザーが Enterprise Connect にアクセスできるようにすることができます。この機能が利用可能な場合は、ユーザーはサポートされているアイテムの [ファンクション] メニューにアクセスし、[Enterprise Connect ここから] を選択することができ、選択されたアイテムは新しい Windows Explorer インスタンスに表示されます。管理者の場合は、この機能を利用できるサブタイプを設定します。既定では、このアクションには [Enterprise Connect ここから] というラベルが付いています。必要に応じて、このラベルの名前を変更できます。

 

注意:
この機能は、Internet Explorer でのみ利用できます。さらに、指定したアイテムの Content Server Web インタフェースでのみ利用できます。
