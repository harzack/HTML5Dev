カラム表示を設定するには

  1. [Content Server 管理] ページで [ Enterprise Connect 管理] をクリックし、次に [構成 Enterprise Connect ビュー列設定の参照] リンクをクリックします。

  2. [参照ビュー列設定の構成] ページで、設定するコンテナーの [編集の設定] をクリックします。

     

    注意:
    Enterprise Connect クライアントでは、ユーザーは、設定に応じてコンテナーのカラムを追加、削除、または再整理することができます。カラムの操作の詳細については、「OpenText Enterprise Connect Plug-in for Content Server - ユーザ ガイド (NGDCORE-UGD)」を参照してください。

     

  3. 選択したコンテナーの設定ページでは、表示されているカラムごとに次の操作を実行します。

    1. [利用可能] カラムで、ユーザーに表示しないカラムのチェック ボックスをオフにします。デフォルトで一部のカラムがオンになっている場合があります。その場合は、ユーザーに表示しないすべてのカラムのチェック ボックスをオフにする必要があります。

       

      ヒント:
      すべてのカラムをオフまたはオンにするには、[利用可能] カラムの一番上にあるチェック ボックスをクリックします。

       

    2. [デフォルトで表示] カラムでは、ユーザーに既定で表示するカラムのチェック ボックスをオンにします。

       

      ヒント:
      すべてのカラムをオフまたはオンにするには、[デフォルトで表示] カラムの一番上にあるチェック ボックスをクリックします。

       

    3. [列幅] カラムで、特定のカラムに適用する幅をピクセルで指定します。

       

      注意:
      値を適用しない場合は、Enterprise Connect によってカラムのサイズが自動的に設定されます。

       

    4. [表示順] カラムで、[上] および [下] を使用して、選択したカラムを適切な位置に配置します。

  4. 設定が完了した後、次のいずれかの操作を実行します。

    • [送信] をクリックして変更を適用します。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • デフォルト設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックしてデフォルト設定を適用します。
