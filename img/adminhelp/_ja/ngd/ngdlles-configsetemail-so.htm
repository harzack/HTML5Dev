[電子メールの設定] ページで、次のタスクを実行できます。

  • 電子メール アーカイブを設定します。

  • ユーザーが Content Server に追加した電子メールおよび添付ファイルがどのように処理されるかを指定します。

  • Enterprise Connect を使用して Content Server に追加された電子メールで使用する命名規則と規則を指定します。

  • 重複する電子メール添付ファイル名の処理方法を指定します。

 

重要項目:
ユーザーは、管理上の変更にアクセスするために、新しい Enterprise Connect セッションを開始する必要があります。

  • 電子メールの添付ファイル、重複電子メール、および電子メールのサブタイプを取得するための設定

  • 電子メールの命名規則の定義

  • 同じ名前の電子メール添付ファイルの処理
