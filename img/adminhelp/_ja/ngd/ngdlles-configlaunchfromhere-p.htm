Content Server Web インタフェースから Enterprise Connect にアクセスできるようにするには

  1. [Content Server 管理] ページで [Enterprise Connect 管理] をクリックし、次に [Enterprise Connect の詳細設定] リンクをクリックします。

  2. [アクションの設定] 領域で、次の操作を実行します。

    • [[ここから Enterprise ConnectEnterprise Connect] のラベル] ボックスに、コンテキスト メニューに表示するラベルを入力します。既定では、[Enterprise Connect ここから] が使用されます。

    • [ここから Enterprise Connect Enterprise Connect] のサブタイプ] ボックスで、この機能を利用できるようにするサブタイプを追加または削除します。既定では、この機能は次のサブタイプでサポートされています。

      • フォルダー (0)

      • エンタプライズ ワークスペース (141)

      • パーソナル ワークスペース (142)

      • プロジェクト (202)

      • 電子メール フォルダー (751)

      • 仮想フォルダー (899)

  3. 選択が完了した後、次のいずれかの操作を実行します。

    • 変更を適用するには、[送信] をクリックします。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • 既定の設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックして既定の設定を適用します。
