OTDS Web アクセス管理を有効にするには

  1. [Content Server 管理] ページで [Enterprise Connect 管理] をクリックし、次に [Enterprise Connect システムの設定 ] リンクをクリックします。

  2. [OTDS Webアクセス管理の設定] 領域で、[Content Server コンテンツ サーバーに OTDS サーバーからの OTDS チケットの取得を許可します (OTDS Enterprise Connect Client 10.2.3 以前のみ) ] チェック ボックスをオンにします。

  3. 選択が完了した後、次のいずれかの操作を実行します。

    • 変更を適用するには、[送信] をクリックします。

    • 最後に保存した設定に戻るには、[リセット] をクリックします。

    • 既定の設定に戻すには [デフォルトの設定] をクリックし、[OK] をクリックして既定の設定を適用します。
