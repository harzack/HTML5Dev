ユーザーが Enterprise Connect を参照する際のカラムの動作を指定できます。フォルダー、コンパウンド フォルダー、マイルストーンなどの Content Server コンテナーの場合は、既定でユーザーに表示されるカラムと、Outlook および Windows Explorer で作業中にユーザーが追加または削除できるカラムを指定できます。選択したカラムが表示される順序や、カラムのカスタムの幅を指定することもできます。ワークフロー、プロジェクトなどのコンテナー リストに含まれていないコンテナー タイプでこれらの設定を指定する場合は、[その他] オプションを使用できます。

利用できるカラムはコンテナーのタイプに応じて異なりますが、Enterprise Connect の既定のカラム、Records Management などの Content Server モジュール固有のカラム、Content Server の動的なグローバル カラムを含めることができます。Content Server の動的なローカル カラムはフォルダー固有であるため、このページには表示されません。ただし、ユーザーはこのローカル カラムを Explorer および Outlook のカラム選択で使用することができます。

ユーザーは、Enterprise Connect クライアント内の設定にあわせて、カラムの設定を追加、削除、または再整理することができます。ユーザーがクライアントの既定のカラム表示をカスタマイズした場合、このページで行った変更が適用されるのは、ユーザーがカラム表示を既定の設定にリセットしたか、Explorer または Outlook のカラム選択を使用して個々のカラムを追加した場合に限られます。
