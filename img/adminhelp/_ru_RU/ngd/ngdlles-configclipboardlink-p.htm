To configure link types:

  1. On the Content Server Administration page, click Enterprise Connect Administration, and then click the Configure Enterprise Connect Advanced Settings link.

  2. In the Send Document as Link Configuration area, in the Choose the type of link to send list, do one of the following:

    • Click Overview to specify that links direct users to the Overview page for the document. Emails and Compound Emails links direct users to the Properties page, and Enterprise Connect Containers links direct users to the specified container. This is the default setting.

    • Click Download to specify that links enable users to download the document. For all other types of items, the link directs users to the Properties page of the item.

    • Click Properties to specify that links for documents, emails, and Compound Emails direct users to the Properties page. Links to containers direct users to the specified container.

  3. When you have made your selection, do one of the following:

    • Click Submit to apply the changes.

    • Click Reset to return to the last saved configuration.

    • Click Set Default to return to the default configuration, and then click OK to apply the default settings.
