You can specify what type of link is used when users add a link to a document to an email when working in Outlook. This same setting also applies when users are working in Microsoft Explorer and use the Send To command to add a link to a document to an email.

The following options are available:

  • Overview: For documents, the link directs users to the Overview page for the document. Links for emails and Compound Emails direct users to the Properties page of the email. Links for containers enable users to browse the specified container. This is the default setting.

  • Download: The link enables users to download the selected document. For all other types of items, the link directs users to the Properties page of the item.

  • Properties: For documents, the link directs users to the Properties page for the document. Links for emails and Compound Emails direct users to the Properties page of the email. Links for containers enable users to browse the specified container.
