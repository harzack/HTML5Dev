So aktivieren Sie OTDS Web Access Management:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung und anschließend auf die Verknüpfung Systemeinstellungen für Enterprise Connect konfigurieren.

  2. Aktivieren Sie im Bereich OTDS Web Access Management-Konfiguration das Kontrollkästchen Abrufen des OTDS-Tickets von OTDS-Server durch Content Server zulassen (nur Enterprise Connect Client 10.2.3 oder früher).

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
