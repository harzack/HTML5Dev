So konfigurieren Sie unterstützte Versionen des Enterprise Connect Frameworks:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung und anschließend auf die Verknüpfung Systemeinstellungen für Enterprise Connect konfigurieren.

  2. Führen Sie im Bereich Support-Konfiguration der Rückwärtskompatibilität einen der folgenden Schritte aus:

    • Wählen Sie Rückwärtskompatibilität mit älteren Versionen des Enterprise Connect-Clients vermeiden, wenn Sie erzwingen möchten, dass Benutzer umgehend ein Upgrade ihrer Instanz von Enterprise Connect Framework durchführen. Sie können die Nachricht bei Bedarf aktualisieren.

    • Deaktivieren Sie das Kontrollkästchen Rückwärtskompatibilität mit älteren Versionen des Enterprise Connect-Clients vermeiden, wenn Sie zulassen möchten, dass Benutzer ihre aktuelle Version von Enterprise Connect Framework verwenden.

    • Passen Sie die Upgrade-Nachricht bei Bedarf an.

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
