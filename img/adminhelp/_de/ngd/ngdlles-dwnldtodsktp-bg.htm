Die Option Auf den Desktop herunterladen ist nur für die Untertypen verfügbar, die Sie den Benutzern zur Verfügung stellen. Wenn Benutzer mehrere Objekte auswählen und die Funktion Auf den Desktop herunterladen bei einem dieser Objekte nicht unterstützt wird, ist die Option Auf den Desktop herunterladen nicht im Kontextmenü verfügbar.

Standardmäßig können die Untertypen Ordner, Zusammengesetzte Dokumente, Dokumente, Projekte, Sammlungen, Favoriten, E-Mails, und E-Mail-Ordner auf den Desktop heruntergeladen werden. Die IDs für diese Untertypen werden automatisch im Feld Untertypen einschließen angezeigt. Sie können Untertypen nach Bedarf hinzufügen oder entfernen. Weitere Informationen zu anderen Untertypwerten finden Sie unter Objekttypnummer und zugeordnete Namen. Weitere Informationen zur Verwendung der Funktion Auf den Desktop herunterladen finden Sie unter OpenText Enterprise Connect Plug-in for Content Server - Benutzerhandbuch (NGDCORE-UGD).

 

Hinweis:
Wenn Benutzer zusammengesetzte E-Mails auf ihre Desktops herunterladen, werden diese als einzelne E-Mail-Binärdatei angezeigt.
