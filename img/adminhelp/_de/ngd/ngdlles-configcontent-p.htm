So konfigurieren Sie die Einstellungen für die Inhaltsspeicherung:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung, und klicken Sie dann auf die Verknüpfung Erweiterte Einstellungen für Enterprise Connect konfigurieren.

  2. Führen Sie im Bereich Konfiguration der Metadatenspeicherung eine der folgenden Aktionen in der Liste Beim Hinzufügen von Inhalt zur Eingabe von Metadaten auffordern aus, um festzulegen, wann Benutzer aufgefordert werden sollen, Metadaten für Objekte anzugeben, die sie zu Content Server hinzufügen:

    • Klicken Sie auf Nur wenn erforderlich, um Benutzer nur dann aufzufordern, Metadaten einzugeben, wenn die Metadaten als für den Content Server-Zielordner erforderlich gekennzeichnet sind. Dies ist die Standardeinstellung.

    • Klicken Sie auf Immer, um Benutzer bei jedem Objekt, das sie zu Content Server hinzufügen, zur Eingabe von Metadaten aufzufordern.

  3. Führen Sie im Bereich Konfiguration der Metadatenspeicherung eine der folgenden Aktionen in der Liste Beim Verschieben/Kopieren von Inhalt in Content Server zur Eingabe von Metadaten aus weiteren Modulen auffordern aus, um festzulegen, wann Benutzer aufgefordert werden sollen, Metadaten anzugeben, wenn andere Content Server-Module in der Umgebung installiert sind:

    • Klicken Sie auf Immer, wenn Benutzer immer aufgefordert werden sollen, Metadaten anzugeben, wenn Sie Elemente innerhalb des Content Servers kopieren oder verschieben. Dies ist die Standardeinstellung.

    • Klicken Sie auf Nie, wenn Benutzer nur dann beim Kopieren oder Verschieben von Elementen innerhalb des Content Servers aufgefordert werden sollen, Metadaten anzugeben, wenn unterschiedliche Kategorieanforderungen für die Quelle und das Ziel vorliegen.

  4. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
