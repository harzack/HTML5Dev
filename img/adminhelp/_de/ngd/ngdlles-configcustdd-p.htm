So konfigurieren Sie Objekttypen beim Ziehen von Ordnern:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung, und klicken Sie dann auf die Verknüpfung Erweiterte Einstellungen für Enterprise Connect konfigurieren.

  2. Führen Sie im Bereich Benutzerdefinierte Drag-und-Drop-Konfiguration einen der folgenden Schritte aus:

    • Geben Sie im Feld Zusammengesetztes Dokument die Erweiterungen an, die zur Identifizierung von Ordnern verwendet werden sollen, die als zusammengesetzte Dokumente behandelt werden sollen, z. B. CD. Das Feld enthält standardmäßig keine Einträge.

    • Geben Sie im Feld E-Mail-Ordner die Erweiterungen an, die zur Identifizierung von Ordnern verwendet werden sollen, die als E-Mail-Ordner behandelt werden sollen, z. B. EM. Das Feld enthält standardmäßig keine Einträge.

     

    Hinweis:
    Wenn Sie mehrere Erweiterungen angeben, trennen Sie die Namen mit Kommas. Verwenden Sie kein Leerzeichen zwischen den Namen.

     

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
