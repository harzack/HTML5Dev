Auf der Seite E-Mail-Einstellungen konfigurieren können Sie Folgendes durchführen:

  • Einstellungen für die E-Mail-Archivierung konfigurieren.

  • Festlegen, wie E-Mails und Anlagen behandelt werden sollen, wenn Benutzer diese zu Content Server hinzufügen.

  • Namenskonventionen und Regeln für E-Mails angeben, die von Enterprise Connect zu Content Server hinzugefügt wurden.

  • Festlegen, wie doppelte E-Mail-Anlagennamen behandelt werden.

 

Wichtig:
Die Benutzer müssen eine neue Enterprise Connect-Sitzung initiieren, um auf die von Ihnen vorgenommenen administrativen Änderungen zugreifen zu können.

 

Inhaltsverzeichnis  

  • E-Mail-Archivierung konfigurieren

  • Einstellungen für das Erfassen von E-Mail-Anlagen, doppelten E-Mails und E-Mail-Untertypen konfigurieren

  • Namenskonventionen für E-Mails definieren

  • E-Mail-Anlagen mit demselben Namen behandeln
