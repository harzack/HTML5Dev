So konfigurieren Sie Einstellungen für das Erfassen von E-Mail-Anlagen:

  1. Klicken Sie auf der Seite Content ServerVerwaltung auf Enterprise Connect-Verwaltung und anschließend auf die Verknüpfung E-Mail-Einstellungen für Enterprise Connect konfigurieren.

  2. Navigieren Sie zum Bereich E-Mail-Konfiguration.

  3. Wenn Sie mit E-Mail-Untertypen arbeiten möchten, klicken Sie auf E-Mails als Untertyp speichern, um den E-Mail-Untertyp (749) auf E-Mails anzuwenden. Dies ist die Standardeinstellung. Wählen Sie eine der folgenden Optionen aus der Liste Beim Hinzufügen von E-Mails mit Anlagen, um festzulegen, wie Anlagen gespeichert werden sollen:

    • Klicken Sie auf Keine separaten Kopien der Anlagen hinzufügen, wenn die E-Mails und die Anlagen als ein einzelnes Objekt gespeichert werden sollen. Dies ist die Standardeinstellung.

    • Klicken Sie auf Separate Kopien der Anlagen hinzufügen, wenn die E-Mail und die Anlagen als ein einzelnes Objekt und alle Anlagen zusätzlich als separate Objekte gespeichert werden sollen.

    • Klicken Sie auf Benutzer nach Entscheidung fragen, wenn Sie dem Benutzer die Entscheidung über das Speichern von Anlagen überlassen möchten. Sie können die E-Mail und die Anlage als einen einzelnen Datensatz speichern oder die E-Mail und die Anlage zusammen speichern und dann eine zusätzliche Kopie der Anlage separat speichern.

  4. Wenn Sie festlegen möchten, wie doppelte E-Mails herausgestellt werden sollen, gehen Sie folgendermaßen vor:

    • Aktivieren Sie das Kontrollkästchen Nachrichten-ID für die Duplikaterkennung verwenden, wenn Enterprise Connect doppelte E-Mails anhand des Vergleichs von Nachrichten-IDs herausstellen soll. Standardmäßig ist dieses Kontrollkästchen deaktiviert, und es wird der Namensvergleich verwendet.

    • Aktivieren Sie optional das Kontrollkästchen Externe Nachrichten-ID für die Duplikaterkennung verwenden, wenn Enterprise Connect doppelte E-Mails anhand des Vergleichs von externen Nachrichten-IDs herausstellen soll. Standardmäßig ist dieses Kontrollkästchen deaktiviert.

       

      Hinweis:
      Sie können diese Option nur wählen, wenn das Kontrollkästchen Nachrichten-ID für die Duplikaterkennung verwenden aktiviert ist.

       

  5. Wenn Sie mit Untertypen für zusammengesetzte E-Mails arbeiten möchten, klicken Sie auf E-Mails mit oder ohne Anlagen als zusammengesetzte E-Mails speichern, um den Untertyp (557) für zusammengesetzte E-Mails anzuwenden.

  6. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
