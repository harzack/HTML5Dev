Standardmäßig sind Benutzer berechtigt, vorherige Versionen des Enterprise Connect Frameworks für die Verbindung zu Content Server zu verwenden. Sie können jedoch erzwingen, dass Benutzer ein Upgrade auf die aktuelle Version des Enterprise Connect Frameworks durchführen. Wenn Sie diese Funktion aktivieren, erhalten Benutzer bei Initiierung einer neuen Sitzung eine Nachricht mit der Aufforderung, ein Upgrade auf die aktuelle Version des Enterprise Connect Frameworks durchzuführen. Benutzer können sich dann erst anmelden, wenn das Upgrade erfolgreich abgeschlossen wurde. Für die Nachricht wird ein Standardtext bereitgestellt. Sie können bei Bedarf jedoch auch eine benutzerdefinierte Nachricht erstellen. So können Sie beispielsweise Informationen zur Kontaktperson im Zusammenhang mit dem Upgrade angeben oder eine Verknüpfung zu einem Upgrade-Prozess bereitstellen.

Weitere Informationen zum Upgrade des Enterprise Connect Frameworks und -Moduls finden Sie unter „Upgrading the Framework“ in OpenText Enterprise Connect - Installation Guide (NGDCORE-IGD).
