So konfigurieren Sie Einstellungen für die E-Mail-Archivierung:

  1. Klicken Sie auf der Seite Content ServerVerwaltung auf Enterprise Connect-Verwaltung und anschließend auf die Verknüpfung E-Mail-Einstellungen für Enterprise Connect konfigurieren.

  2. Führen Sie im Bereich E-Mail-Archivierungskonfiguration einen der folgenden Schritte aus:

    • Aktivieren Sie das Kontrollkästchen E-Mail-Archivierung für Enterprise Connect archivieren. Dies betrifft E-Mails, die über den Posteingang zu Content Server hinzugefügt wurden., wenn Sie die E-Mail-Archivierung mithilfe von Enterprise Connect aktivieren möchten.

    • Deaktivieren Sie das Kontrollkästchen E-Mail-Archivierung für Enterprise Connect archivieren. Dies betrifft E-Mails, die über den Posteingang zu Content Server hinzugefügt wurden., wenn Sie die E-Mail-Archivierung mithilfe von Enterprise Connect nicht aktivieren möchten. E-Mails werden stattdessen im ausgewählten Content Server-Container gespeichert und nicht archiviert. Das Kontrollkästchen ist standardmäßig deaktiviert.

  3. Führen Sie im Bereich Auflösung des Auftragsposteingangs einen der folgenden Schritte aus, um anzugeben, wie der Auftragsposteingang bestimmt wird:

    • Klicken Sie auf Basierend auf Server oder Administratorgruppe, um den Auftragsposteingang unter Verwendung von Einstellungen für den Server oder die Administratorgruppe auszuwählen. Dies ist die Standardeinstellung.

    • Klicken Sie auf Basierend auf Eigenschaft mit Hexadezimalwert, und geben Sie dann den entsprechenden Eigenschaftenwert in hexadezimaler Form in das Feld hex ein, um den Auftragsposteingang mithilfe einer Eigenschaft auszuwählen.

     

    Achtung
    In der Enterprise Connect-Konfiguration müssen Sie dieselben Werte verwenden wie bei der Konfiguration des Moduls Email Management Services.

     

  4. Führen Sie einen der folgenden Schritte aus:

    • Aktivieren Sie das Kontrollkästchen Aufträge an Auftragsposteingang senden, um Archivierungsaufträge direkt an den Auftragsposteingang zu senden. Dies ist die Standardeinstellung.

    • Deaktivieren Sie das Kontrollkästchen Aufträge an Auftragsposteingang senden, wenn Sie Archivierungsaufträge per E-Mail an den Posteingang senden möchten.

     

    Achtung
    In der Enterprise Connect-Konfiguration müssen Sie dieselben Werte verwenden wie bei der Konfiguration des Moduls Email Management Services.

     

  5. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
