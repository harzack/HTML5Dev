Auf der Seite Durchsuchenansichtsspalten-Einstellungen konfigurieren können Sie für jeden Containertyp Folgendes durchführen:

  • Angeben, welche Spalten Enterprise Connect-Benutzern zur Verfügung stehen.

  • Angeben, welche Spalten Benutzern standardmäßig angezeigt werden.

  • Die Reihenfolge festlegen, in der die Spalten angezeigt werden.

  • Eine benutzerdefinierte Breite für jede Spalte anzeigen.

 

Wichtig:
Die Benutzer müssen eine neue Enterprise Connect-Sitzung initiieren, um auf die von Ihnen vorgenommenen administrativen Änderungen zugreifen zu können.
