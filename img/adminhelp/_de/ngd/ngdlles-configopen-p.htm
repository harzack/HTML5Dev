So konfigurieren Sie Aktionen für Doppelklicken in Microsoft Office-Dokumenten:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung, und klicken Sie dann auf die Verknüpfung Erweiterte Einstellungen für Enterprise Connect konfigurieren.

  2. Klicken Sie im Bereich Aktionskonfiguration in der Liste Beim Doppelklicken eines Microsoft Office-Dokuments auf eine der folgenden Optionen:

    • Klicken Sie auf Dokument immer schreibgeschützt öffnen, wenn eine schreibgeschützte Version des Dokuments geöffnet werden soll.

    • Klicken Sie auf Dokument zur Bearbeitung öffnen, sofern Berechtigung vorhanden, wenn die Benutzer Dokument bearbeiten können sollen.

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
