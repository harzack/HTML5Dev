Sie können Benutzern, die in der Content Server-Web-Benutzeroberfläche an angegebenen Inhaltstypen arbeiten, den Zugriff auf Enterprise Connect ermöglichen. Wenn diese Funktion aktiviert ist, können Benutzer auf das Menü Funktion für ein unterstütztes Element zugreifen, die Option Enterprise Connect hier öffnen wählen, und das ausgewählte Element wird in einer neuen Windows Explorer-Instanz angezeigt. Als Administrator bestimmen Sie, für welche Untertypen diese Funktion verfügbar ist. Standardmäßig trägt diese Aktion die Bezeichnung Enterprise Connect hier öffnen. Sie können diese Bezeichnung bei Bedarf umbenennen.

 

Hinweis:
Diese Funktion ist nur in Internet Explorer verfügbar. Außerdem ist sie nur in der Content Server-Web-Benutzeroberfläche für von Ihnen angegebene Elemente verfügbar.
