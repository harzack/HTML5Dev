Mit dieser Einstellung können Sie festlegen, ob E-Mails und deren Anhänge archiviert werden sollen, wenn Benutzer sie über ihr Postfach zu Content Server hinzufügen. Standardmäßig ist die E-Mail-Archivierung nicht aktiviert. Weitere Informationen zum Aktivieren der E-Mail-Archivierung und zum Archivieren von E-Mails mit Enterprise Connect finden Sie unter OpenText Enterprise Connect - Installation Guide (NGDCORE-IGD) .

 

Hinweis:
Wenn Sie die E-Mail-Archivierung aktivieren, werden alle übrigen E-Mail-Einstellungen auf der Seite E-Mail-Einstellungen für Enterprise Connect konfigurieren nicht auf die archivierten E-Mails angewendet. Beim Archivieren von E-Mails und deren Anhängen werden stattdessen die Einstellungen angewendet, die Sie bei der Konfiguration des Moduls Email Management Services festgelegt haben.

 

Um die E-Mail-Archivierung zu aktivieren, müssen Sie zuerst angeben, wie der entsprechende Auftragsposteingang für die einzelnen Benutzer festgelegt wird. Standardmäßig richtet sich der Auftragsposteingang nach dem Server oder der Administratorgruppe, der der Benutzer angehört. Sie können auch eine Active Directory-Benutzereigenschaft verwenden, die als Hexadezimalwert definiert ist. Weitere Informationen finden Sie unter OpenText Email Archiving for Microsoft Exchange - Administration Guide (EA-AGD).

 

Hinweis:
Dieser Wert muss dem Wert entsprechen, der bei der Konfiguration des Moduls Email Management Services festgelegt wurde. Microsoft Exchange 2013 erfordert eine auf Eigenschaften beruhende Auflösung des Auftragsposteingangs.

 

Sie müssen außerdem angeben, ob Archivierungsaufträge direkt in den Auftragsposteingang gesendet werden oder ob sie per E-Mail gesendet werden. Standardmäßig ist das Kontrollkästchen Aufträge an Auftragsposteingang senden aktiviert. Dies ist im Allgemeinen die effizienteste Methode zum Senden von Archivierungsaufträgen. Damit dies möglich ist, müssen Sie allerdings die Berechtigungen für den Auftragsposteingang ändern. Weitere Informationen finden Sie unter OpenText Email Archiving for Microsoft Exchange - Administration Guide (EA-AGD).

 

Hinweis:
Damit die E-Mail-Archivierungsfunktion verwendet werden kann, muss die entsprechende Version des Moduls Email Management installiert sein. Eine Fehlermeldung im Abschnitt E-Mail-Archivierungskonfiguration weist darauf hin, ob ein Upgrade ausgeführt werden muss.
