Auf der Seite Erweiterte Einstellungen konfigurieren können Sie Folgendes durchführen:

  • Angeben, wie Microsoft Office-Dokumente geöffnet werden sollen, wenn Benutzer darauf in Enterprise Connect doppelklicken.

  • Angeben, wie zusammengesetzte E-Mails geöffnet werden sollen, wenn Benutzer darauf in Enterprise Connect doppelklicken.

  • Angeben, welche Optionen im Kontextmenü angezeigt werden.

  • Einstellungen für die Funktion Enterprise Connect hier öffnen angeben.

  • Angeben, welche Benutzer aufgefordert werden, Metadaten einzugeben, wenn Sie Objekte zu Content Server hinzufügen.

  • Angeben, welchen Verknüpfungstyp Benutzer in die Zwischenablage kopieren können.

  • Angeben, welche Ordnertypen Benutzer offline verwenden können.

  • Angeben, welche Objekttypen Benutzer auf ihren Desktop herunterladen können.

  • Angeben, wie Benutzer auf den Befehl "Kopieren/Verschieben nach Enterprise Connect" zugreifen können.

  • Den Subtyp anpassen, der beim Ziehen bestimmter Ordnertypen angewendet wird.

 

Wichtig:
Die Benutzer müssen eine neue Enterprise Connect-Sitzung initiieren, um auf die von Ihnen vorgenommenen administrativen Änderungen zugreifen zu können.

 

Inhaltsverzeichnis  

  • Aktionen für Doppelklicken in Microsoft Office-Dokumenten konfigurieren

  • Doppelklickaktionen für zusammengesetzte E-Mails konfigurieren

  • Kontextmenüaktionen konfigurieren

  • Zugriff auf Enterprise Connect von der Content Server-Web-Benutzeroberfläche aus ermöglichen

  • Optionen für Inhaltsspeicherung konfigurieren

  • Verknüpfungstypen konfigurieren

  • Angabe von Link-Pfaden

  • Untertypen angeben, die offline verwendet werden können

  • Festlegen, welche Untertypen auf den Desktop heruntergeladen werden können

  • Den Befehl Kopieren/Verschieben nach Enterprise Connect konfigurieren

  • Objekttypen beim Ziehen von Ordnern konfigurieren
