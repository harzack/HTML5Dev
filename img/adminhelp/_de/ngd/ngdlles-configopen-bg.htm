Sie können konfigurieren, wie Microsoft Office-Dokumente in Windows Explorer oder Microsoft Outlook geöffnet werden sollen, wenn Benutzer darauf in Enterprise Connect doppelklicken. Diese Einstellung gilt auch für Verknüpfungen, die auf Office-Dokumente verweisen. Die Option Dokument immer schreibgeschützt öffnen ist die Standardeinstellung. Mit dieser Option wird für die Benutzer eine schreibgeschützte Version geöffnet, wenn sie auf Word-, Excel-, PowerPoint-, Project- oder Visio-Dokumente doppelklicken. Mit der Option Dokument zur Bearbeitung öffnen, sofern Berechtigung vorhanden können Sie festlegen, dass Benutzer Office-Dokumente, die sie durch Doppelklicken geöffnet haben, direkt bearbeiten können.

 

Hinweis:
Benutzer können nur solche Dokumente bearbeiten, für die sie die Berechtigung zum Ändern und Reservieren besitzen.
