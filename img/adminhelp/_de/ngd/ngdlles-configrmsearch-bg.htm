Wichtig:
Um Verknüpfungen in die Suchergebnisse einzuschließen, müssen Sie Content Server 10 mit Update 7 oder höher verwenden. Um Querverweise in die Suchergebnisse einzuschließen, müssen Sie OpenText Records Management (RM) 10.2.0 Patch 1 oder höher verwenden. Wenn Sie ältere Versionen von Content Server und RM verwenden, wird dieses Kontrollkästchen nicht angezeigt.

 

Enterprise Connect unterstützt die Suche nach Verknüpfungen und Querverweisen. Um diese Funktion verfügbar zu machen, müssen die erforderlichen Versionen von Enterprise Connect, Content Server und RM auf Ihrem System installiert sein. Außerdem müssen Sie Content Server konfigurieren, um diese Funktion verwenden zu können. Weitere Informationen zum Arbeiten mit RM, finden Sie unter OpenText Records Management - Installation and Administration Guide (LLESRCM-IGD). Weitere Informationen zum Konfigurieren von Content Server finden Sie unter Open Text Administration Help - Content Server (LLESCOR-H-AGM).

Wenn Sie diese Funktion aktivieren, können Benutzer Verknüpfungen und Querverweise bei Schnellsuchen und erweiterten Suchen in Enterprise Connect mit einschließen. Wenn z. B. ein Ordner bei einer Suche Verknüpfungen oder Querverweise zu anderen Ordnern enthält, werden diese referenzierten Ordner in die Suchergebnisse mit eingeschlossen.
