So ermöglichen Sie den Zugriff auf Enterprise Connect von der Content Server-Web-Benutzeroberfläche aus:

  1. Klicken Sie auf der Seite "Content Server-Verwaltung" auf Enterprise Connect-Verwaltung und anschließend auf die Verknüpfung Erweiterte Einstellungen für Enterprise Connect konfigurieren.

  2. Gehen Sie im Bereich Aktionskonfiguration wie folgt vor:

    • Geben Sie im Feld Bezeichnung für "Enterprise Connect hier öffnen" die Bezeichnung ein, die im Kontextmenü angezeigt werden soll. Standardmäßig wird Enterprise Connect hier öffnen verwendet.

    • Fügen Sie im Feld Untertypen für "Enterprise Connect hier öffnen" die Untertypen hinzu, für die diese Funktion verfügbar sein soll, oder entfernen Sie sie. Standardmäßig wird diese Funktion von den folgenden Untertypen unterstützt:

      • Ordner (0)

      • Unternehmensarbeitsbereiche (141)

      • Persönliche Arbeitsbereiche (142)

      • Projekte (202)

      • E-Mail-Ordner (751)

      • Virtuelle Ordner (899)

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
