Sie können festlegen, welcher Verknüpfungstyp verwendet wird, wenn Benutzer eine Verknüpfung zu einem Dokument bei der Arbeit in Outlook zu einer E-Mail hinzufügen. Dieselbe Einstellung gilt auch, wenn Benutzer in Microsoft Explorer arbeiten und einer E-Mail über den Befehl Senden an eine Verknüpfung zu einem Dokument hinzufügen.

Die folgenden Optionen sind verfügbar:

  • Überblick: Bei Dokumenten leitet die Verknüpfung Benutzer zur Seite Überblick für das Dokument um. Verknüpfungen für E-Mails und zusammengesetzte E-Mails leiten Benutzer zur Seite Eigenschaften der E-Mail um. Mit Verknüpfungen für Container können Benutzer den angegebenen Container durchsuchen. Dies ist die Standardeinstellung.

  • Herunterladen: Mit dieser Verknüpfung können Benutzer das gewählte Dokument herunterladen. Bei allen anderen Objekttypen leitet die Verknüpfung die Benutzer zur Seite Eigenschaften des Objekts um.

  • Eigenschaften: Bei Dokumenten leitet die Verknüpfung Benutzer zur Seite Eigenschaften für das Dokument um. Verknüpfungen für E-Mails und zusammengesetzte E-Mails leiten Benutzer zur Seite Eigenschaften der E-Mail um. Mit Verknüpfungen für Container können Benutzer den angegebenen Container durchsuchen.
