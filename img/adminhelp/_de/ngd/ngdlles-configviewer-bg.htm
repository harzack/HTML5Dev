Der Enterprise Connect Viewer ermöglicht es Benutzern, Objekte in unterstützten Formaten mithilfe der Registerkarte Anzeigen im Lesefenster selbst dann anzuzeigen, wenn die ursprüngliche Anwendung nicht installiert ist. So kann ein Benutzer z. B. ein Objekt, das in Microsoft Project erstellt wurde, selbst dann anzeigen, wenn Project nicht installiert ist. Der Enterprise Connect Viewer ermöglicht es Benutzern, das ausgewählte Objekt zu drucken, zu durchsuchen, zu kopieren und seine Ansicht anzupassen. Der Enterprise Connect Viewer wird mit Enterprise Connect automatisch installiert. Eine zusätzliche Konfiguration ist nicht erforderlich.

Als Alternative zum Enterprise Connect Viewer können Sie OpenText Document Filters nutzen, um Benutzern das Anzeigen von Objekten in unterstützten Formaten zu ermöglichen. Im Bereich Viewer-Technologie auf der Seite Anzeigeeinstellungen konfigurieren können Sie angeben, welchen Viewer Sie verwenden möchten. Enterprise Connect Viewer wird standardmäßig verwendet. Weitere Informationen zu Document Filters finden Sie unter OpenText Document Filters - Administrator's Guide (LLESCF-AGD).
