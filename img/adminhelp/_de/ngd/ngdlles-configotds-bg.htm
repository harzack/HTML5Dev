Hinweis:
Diese Vorgehensweise gilt nur, wenn Sie Enterprise Connect Client Version 10.2.3 oder früher verwenden. Diese Einstellung ist für Enterprise Connect Client Version 10.3 oder höher nicht erforderlich.

 

Wenn Sie Enterprise Connect Client Version 10.2.3 oder früher und CA SiteMinder oder RSA Access Manager zusammen mit Content Server und OpenText Directory Services (OTDS) verwenden, müssen Sie das Kontrollkästchen Abrufen des OTDS-Tickets von OTDS-Server durch Content Server zulassen (nur Enterprise Connect Client 10.2.3 oder früher) aktivieren. Standardmäßig ist diese Option nicht aktiviert.
