Auf der Seite Systemeinstellungen konfigurieren können Sie Folgendes durchführen:

  • Erzwingen, dass Benutzer ein Upgrade auf die aktuelle Version des Enterprise Connect-Clients durchführen.

  • Angeben, wie Anmeldeinformationen gespeichert werden.

  • Auf Verwaltungsseiten für Module zugreifen, die Sie zur Unterstützung von Enterprise Connect konfigurieren müssen.

 

Wichtig:
Die Benutzer müssen eine neue Enterprise Connect-Sitzung initiieren, um auf die von Ihnen vorgenommenen administrativen Änderungen zugreifen zu können.

 

Inhaltsverzeichnis  

  • Vorherige Versionen von Enterprise Connect Framework unterstützen

  • Speicherung von Enterprise Connect-Anmeldeinformationen konfigurieren

  • Auf Verwaltungsseiten für externe Module zugreifen
