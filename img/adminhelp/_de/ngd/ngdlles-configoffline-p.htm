So geben Sie Untertypen an, die offline verwendet werden können:

  1. Klicken Sie auf der Seite Content Server-Verwaltung auf Enterprise Connect-Verwaltung, und klicken Sie dann auf die Verknüpfung Erweiterte Einstellungen für Enterprise Connect konfigurieren.

  2. Geben Sie im Bereich Download Offline-Konfiguration im Feld Untertypen einschließen die Werte aller weiteren Untertypen ein, die Benutzer offline verwenden können. Trennen Sie die Werte mit einem Komma. Folgende Werte werden standardmäßig angegeben und können ggf. entfernt werden:

    • 0 für Ordner.

    • 136 für zusammengesetzte Dokumente.

    • 144 für Dokumente.

    • 202 für Projekte.

    • 298 für Sammlungen.

    • 628 für Favoriten.

    • 749 für E-Mails.

    • 751 für E-Mail-Ordner.

     

    Hinweise:

    • Sie können keine Kanäle für die Offline-Verwendung herunterladen.

    • Damit Benutzer zusammengesetzte E-Mails offline verwenden können, fügen Sie den Untertyp 557 hinzu.

     

  3. Wenn Sie Ihre Wahl getroffen haben, führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Senden, um die Änderungen anzuwenden.

    • Klicken Sie auf Zurücksetzen, um zur letzten gespeicherten Konfiguration zurückzukehren.

    • Klicken Sie auf Standard festlegen, um zur Standardkonfiguration zurückzukehren, und anschließend auf OK, um die Standardeinstellungen anzuwenden.
