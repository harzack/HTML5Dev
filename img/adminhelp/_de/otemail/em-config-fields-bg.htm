Sie können die Optionen der Benutzeroberfläche so konfigurieren, dass bestimmte E-Mail-Felder angezeigt werden, wenn Benutzer eine E-Mail zu Content Server hinzufügen. Die Felder E-Mail-Betreff, Von, An, CC, Sendedatum und Empfangsdatum sind Standardfelder, und werden im Dialogfeld E-Mail hinzufügen in Outlook angezeigt. Die Felder Im Namen von und BCC sind keine Standardfelder und müssen konfiguriert werden, wenn Sie im Dialogfeld E-Mail hinzufügen in Outlook angezeigt werden sollen.

Sie können konfigurieren, ob E-Mail-Felder ein- oder ausgeblendet werden sollen, und die E-Mail-Felder im Dialogfeld E-Mail hinzufügen in Outlook als bearbeitbar festlegen:

  • Anzeigen zeigt das entsprechende E-Mail-Feld als schreibgeschütztes Feld an.

  • Änderbar macht das E-Mail-Feld bearbeitbar. Das E-Mail-Feld kann nur als Änderbar markiert werden, wenn es als Anzeigen markiert ist.
