Exemple 6-13 : Pour mettre à jour la description d'un document en anglais et en français. Ce code peut fonctionner ou non car le type d'opération concerné est sync. Si le document existe, la description sera mise à jour. Dans le cas contraire, une erreur sera générée car la balise <file> n'a pas été spécifiée. Dans cet exemple, la case « Ajouter un titre à l'emplacement » est cochée : 

<import>
  <node type="document" action="sync">
    <location>Enterprise:Projects:Documents</location>
    <title language="en">logo.jpg</title>
    <description language="en">Final draft</description>
    <description language="fr">Version Finale</description>
  </node>
</import>

Exemple 6-14 : Pour mettre à jour la description d'un document. Ce code peut fonctionner ou non car le type d'opération concerné est sync. Si le document existe, la description sera mise à jour. Dans le cas contraire, une erreur sera générée car la balise <file> n'a pas été spécifiée. Dans cet exemple, la case « Ajouter un titre à l'emplacement » est décochée :

<import>

    <location>Enterprise:Projects:Documents:logo.jpg</location>
    <description language="en">Final drafts</description>
    <description language="fr">Version Finale</description>

Exemple 6-15 : Pour créer un dossier « Documents » dans le dossier « Projets » de l'espace de travail Entreprise, en octroyant les autorisations « Client » de groupe Voir et Voir le contenu et en retirant des autorisations du groupe public. Les descriptions sont créées en anglais et en français. Afin de pouvoir appliquer les descriptions, ces deux langues doivent être installées et activées dans le système :

<import>
  <node type="folder" action="create">
    <location>Enterprise:Projects</location>
    <title language="en">Documents</title>
    <description language="en">This folder stores documents</description>
    <description language="fr">Ce dossier stocke des documents</description>
    <acl group="Client" permissions="100000000"></acl>
    <acl standard="world" permissions="000000000"></acl>
  </node>
</import>

Exemple 6-16 : Pour mettre à jour un dossier existant nommé « Documents » situé dans l'espace de travail Entreprise, en ajoutant un surnom. Dans cet exemple, la case « Ajouter un titre à l'emplacement » est activée. La description en anglais pour le dossier est mise à jour, tandis que celle en français est supprimée :

<import>
  <node type="folder" action="update">
    <location>Enterprise</location>
    <title language="en">Documents</title>
    <description language="en">This folder stores obsolete documents</description>
    <description clear="true" language="fr"></description>
    <nickname>docs</nickname>
  </node>
</import>

Exemple 6-17 : Pour renommer un dossier existant de « Documents » en « Mes documents » lorsque la case « Ajouter un titre à l'emplacement » est décochée :

<import>
  <node type="folder" action="update">
    <location>Enterprise:Projects:Documents</location>
    <title language="en">My Documents</title>
    <title language="en">Mes Documents</title>
  </node>
</import>
