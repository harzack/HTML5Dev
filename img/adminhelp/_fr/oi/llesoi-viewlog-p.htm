Pour afficher des fichiers journaux :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur Afficher les fichiers journaux.

  2. Sur la page Afficher les fichiers journaux d'importation, cliquez sur le nom du fichier journal que vous souhaitez afficher.

Pour supprimer des fichiers journaux :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur Afficher les fichiers journaux.

  2. Sur la page Afficher les fichiers journaux d'importation, activez la case à cocher en regard du fichier journal à supprimer.

  3. Cliquez sur Supprimer.
