Description

Utilisez type="tasklist" sur la balise <node> pour créer, mettre à jour ou supprimer une liste de tâches.

Syntaxe

<node type="tasklist" action="create"></node>

Utilisation

La création, mise à jour ou suppression d'une liste de tâches peut nécessiter un certain nombre de balises subordonnées, telles que la balise location . Ces balises subordonnées sont détaillées dans Descriptions et syntaxe des balises Importateur d'objet.

Exemple

Pour créer une liste de tâches nommée « New Task List » dans l'espace de travail Enterprise :

<import>
  <node type="tasklist" action="create">
    <location>Enterprise</location>
    <title language="en">New Task List</title>
  </node>
</import>

Pour consulter d'autres exemples de balises tasklist , consultez la section Exemples de fichier de contrôle d'importation de tâches.
