Exemple 6-18 : Pour créer un document nommé « My Guidelines » dans le dossier « Documents » avec le type mime défini et une catégorie appliquée à tous les sous-éléments : 

<import>
  <node type="document" action="create">
    <location>Enterprise:Projects:Documents</location>
    <title language="en">My Guidelines</title>
    <file>c:\temp\guidelines.doc<\file>
    <mime>application/msword</mime>
    <category name="Content Server Categories:Test">
      <attribute name="date">20060214</attribute>
      <attribute name="internal">TRUE</attribute>
      <attribute name="user">Admin</attribute>
      <subitems>reapply</subitems>
    </category>
  </node>
</import>

Exemple 6-25 : Pour mettre à jour la description d'un document. Ce code peut fonctionner ou non car le type d'opération concerné est sync. Si le document existe, la description sera mise à jour. Dans le cas contraire, une erreur sera générée car la balise <file> n'a pas été spécifiée. Dans cet exemple, la case « Ajouter un titre à l'emplacement » est cochée :

<import>
  <node type="document" action="sync">
    <location>Enterprise:Projects:Documents</location>
    <title language="en">logo.jpg</title>
    <description language="en">Final drafts</description>
  </node>
</import>
