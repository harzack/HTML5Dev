Pour ajouter une table à la base de données :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur Administration de la base de données.

  2. Sur la page Administration de la base de données, cliquez sur le lien Créer de la table manquante dans la colonne Action.

Pour supprimer une table de la base de données :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur Administration de la base de données.

  2. Sur la page Administration de la base de données, cliquez sur le lien Supprimer de la table concernée.
