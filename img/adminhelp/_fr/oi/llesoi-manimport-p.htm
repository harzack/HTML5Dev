Pour importer manuellement un document :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur le lien Importation manuelle d'objet.

  2. Sur la page Importation manuelle d'objet, dans le champ Nom du fichier journal, entrez un nom pour le fichier journal, sans extension ; ce nom sera enregistré dans le répertoire des fichiers journaux tel qu'indiqué dans les pages d'administration. Le nom ne doit pas inclure de caractères spéciaux.

  3. Dans le champ Importer le fichier de contrôle, cliquez sur Parcourir... pour sélectionner un fichier de contrôle de l'importation.

  4. Cliquez sur Démarrer l'importation.
