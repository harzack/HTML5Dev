Pour afficher des informations relatives à l'état de l'importation :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur Etat de l'importation.

     

    Note :
    Si aucun thread d'importation n'est en cours d'exécution, le fait de cliquer sur Etat de l'importation aura pour effet d'afficher une liste d'état vide.

     

  2. Pour actualiser le contenu de la page, cliquez sur Statut de l'importation sur la page Mettre à jour le statut.

  3. Pour afficher les fichiers journaux, cliquez sur Afficher les fichiers journaux.

  4. Pour supprimer les lignes de la table oi_status ayant une valeur Etat égale à « 2 » ou « 3 », cliquez sur Effacer la table d'état.
