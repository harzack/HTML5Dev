url

Description

Utilisez type="url" sur la balise <node> pour créer, mettre à jour ou supprimer une URL.

Syntaxe

<node type="url" action="create"></node>

Utilisation

  1. La création ou la mise à jour d'une URL peut inclure une balise subordonnée, la Balise url, décrite ci-dessous.

  2. La création, mise à jour ou suppression d'une URL peut également nécessiter un certain nombre de balises subordonnées, telles que la balise location . Ces autres balises subordonnées sont détaillées dans Descriptions et syntaxe des balises Importateur d'objet.

Exemple

Pour créer une URL nommée « OpenText » dans le dossier « Links » de l'espace de travail Entreprise :

<import>
  <node type="url" action="create">
    <location>Enterprise:Links</location>
    <title language="en">OpenText</title>
    <url>http://www.opentext.com</url>
  </node>
</import>

Pour consulter d'autres exemples de balises url , consultez la section Exemples de fichier de contrôle d'importation d'URL.

Balise url

La balise url permet de spécifier une URL. La syntaxe de cette balise est la suivante : 

<url>http://www.opentext.com</url>

Utilisation Notes
1.

La balise url ne peut exister seule et doit être contenue dans une balise <node type="url" .

2.

La balise url est obligatoire dans le cas suivant :

  • <node type="url" action="create">

3.

La balise url est facultative dans le cas suivant :

  • <node type="url" action="update">
Veillez à ce que l'URL soit précédée d'un protocole admis. Il peut s'agir par exemple des protocoles http et https.
