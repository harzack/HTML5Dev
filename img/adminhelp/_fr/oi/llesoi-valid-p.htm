Pour valider un fichier de contrôle d'importation :

  1. Dans la section Administration de l'importateur d'objet de la page d'administration de Content Server, cliquez sur le lien Validation.

  2. Sur la page Validation, cliquez sur le bouton à droite du champ Importer le fichier de contrôle pour sélectionner un fichier de contrôle d'importation.

  3. Dans le champ Action, cliquez sur Démarrer la validation.

  4. Une fois la validation terminée, la page des résultats de la validation s'affiche. En cas d'erreurs, ces dernières apparaissent sur la page des résultats. Si aucune erreur n'a été détectée, la page des résultats de la validation contient un lien vers la page Importation manuelle d'objet, ce qui vous permet d'exécuter immédiatement votre fichier de contrôle d'importation.
