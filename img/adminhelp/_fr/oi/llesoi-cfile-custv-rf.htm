Description

Utilisez type="customview" sur la balise <node> pour créer, mettre à jour ou supprimer une vue personnalisée.

Syntaxe

<node type="customview" action="create"></node>

Utilisation

La création, mise à jour ou suppression d'une vue personnalisée peut nécessiter des balises subordonnées, telles que la balise location . Ces balises subordonnées sont détaillées dans Descriptions et syntaxe des balises Importateur d'objet.

Exemple

Pour importer une vue personnalisée nommée « Customview 001 » dans l'espace de travail Entreprise :

<import>
  <node type="customview" action="create">
    <location>Enterprise</location>
    <title language="en">Customview 001</title>
    <file>C:\OI\Files\My Custom View.html</file>
  </node>
</import>
