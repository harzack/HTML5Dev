Pour supprimer un formulaire de recherche :

  1. Sur la page d'administration, cliquez sur le lien Ouvrir le volume des objets système.

  2. Cliquez sur le lien Modèles de recherche personnels.

  3. Procédez de l'une des manières suivantes :

    • Pour supprimer un formulaire de recherche de système, cliquez sur le lien Admin, cliquez ensuite sur l'icône Fonctions du formulaire de recherche, puis choisissez Supprimer.

    • Pour supprimer le formulaire de recherche personnel d'un utilisateur, cliquez sur le nom de connexion de l'utilisateur approprié, cliquez ensuite sur l'icône Fonctions du formulaire de recherche, puis choisissez Supprimer.

  4. Dans la boîte de dialogue qui vous invite à confirmer la suppression, cliquez sur le bouton OK.

 

Astuces :

  • Vous pouvez supprimer tous les modèles de recherche, les modèles de recherche système uniquement, ou les modèles de recherche personnels uniquement en cliquant sur le lien Réinitialiser le modèle de recherche système par défaut de la page Réinitialiser les modèles de recherche. Pour en savoir plus, consultez le document Pour réinitialiser les modèles de recherche.

  • Si vous avez activé les domaines et que vous supprimez tous les modèles de recherche et réinitialisez le modèle système par défaut, vous devez réappliquer les autorisations Ajouter un élément et Supprimer pour le dossier Modèles de recherche personnels.
