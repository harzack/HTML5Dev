Pour supprimer une valeur de meilleurs résultats :

  1. Sur la page Administration, dans la section Administration des meilleurs résultats, cliquez sur le lien Gérer les éléments de meilleurs résultats.

  2. Sur la page Gérer les éléments de meilleurs résultats, saisissez une valeur de meilleurs résultats dans le champ Rechercher vos Eléments de vos meilleurs résultats, puis cliquez sur le bouton Aller.

  3. Sélectionnez la case à cocher de chaque élément pour lequel vous souhaitez enlever la valeur de meilleurs résultats.

  4. Cliquez sur le bouton Enlever les entrées des meilleurs résultats.

 

Astuce :
Vous pouvez activer la case à cocher en haut de la liste pour enlever toutes les valeurs de meilleurs résultats.
