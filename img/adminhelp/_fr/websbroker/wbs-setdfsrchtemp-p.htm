Pour gérer le nombre de formulaires de recherche et de modèles de recherche :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur les liens Gérer les modèles de recherche , puis Réinitialiser les modèles de recherche.

  2. Sur la page Réinitialiser les modèles de recherche, dans la section Formulaires de recherche, saisissez une valeur entre 0 et 100 dans la zone de texte Maximum par utilisateur pour définir le nombre de formulaires de recherche personnels que chaque utilisateur Content Server est autorisé à créer. La valeur par défaut est 20.

  3. Dans la section Modèles de résultats de la recherche, saisissez une valeur entre 0 et 100 dans la zone de texte Maximum par utilisateur pour définir le nombre de Modèles de résultats de la recherche que chaque utilisateur Content Server est autorisé à enregistrer. La valeur par défaut est 20.

  4. Cliquez sur Mettre à jour.
