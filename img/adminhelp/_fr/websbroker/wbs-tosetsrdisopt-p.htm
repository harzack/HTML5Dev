Pour définir les options d'affichage des résultats de recherche pour les Modèles de recherche depuis la page d'administration

  1. Sur la page d'administration, cliquez sur le lien Ouvrir le volume des objets système.

  2. Cliquez sur le lien Modèles de recherche personnels.

  3. Cliquez sur le lien Admin.

  4. Cliquez sur l'icône Fonctions du modèle de résultats de recherche, puis cliquez sur le lien Modifier les options d'affichage.

  5. Sur la page Options d'affichage, spécifiez les options d'affichage souhaitées.

  6. Cliquez sur le bouton Mettre à jour.
