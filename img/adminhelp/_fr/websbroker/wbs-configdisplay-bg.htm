Lorsqu'un utilisateur effectue une recherche avancée ou une recherche à partir de la barre de recherche d'Enterprise Server, l'apparence de la page Résultats de recherche est déterminée par le modèle de résultats de recherche par défaut de l'utilisateur. Si aucun modèle de résultats de recherche par défaut n'a été défini pour l'utilisateur, l'apparence de la page Résultats de recherche est déterminée par les modèles de recherche système par défaut. Pour plus d'informations sur les formulaires de recherche, voir Gestion des modèles de recherche.

Vous pouvez définir la façon dont les résultats de recherche sont affichés à l'aide du modèle de recherche système par défaut ou de tout autre modèle de recherche système sur la page Options d'affichage. Vous pouvez définir les mêmes options d'affichage pour chaque formulaire de recherche, utiliser la configuration des options d'affichage par défaut, ou personnaliser les options d'affichage pour chaque formulaire de recherche.

Par exemple, vous pouvez définir deux modèles de recherche système : un pour le service juridique et un autre pour la comptabilité, chacun avec une page Résultats de recherche différente. Dans le Modèle de recherche Service juridique, vous pouvez ajouter le champ « Numéro de dossier » en tant que Champs de résultat affiché. Ce champ sera ensuite affiché sur la page Résultats de recherche lors de toute recherche utilisant le modèle de recherche Service juridique. Vous pouvez ensuite ajouter le champ de métadonnées « Nom du client » en tant que Champs de résultat affiché sur le Modèle de recherche Comptabilité. Ce champ sera ensuite affiché sur la page Résultats de recherche lors de toute recherche utilisant le modèle de recherche Comptabilité.

Les options suivantes peuvent être configurées sur la page Options d'affichage :

  • Disposition de la page de résultats

  • Nombre de résultats

  • Afficher les résumés/descriptions

  • Afficher les locutions clés

  • Afficher le chemin complet de l'emplacement

  • Champs de résultat

  • Trier par

Pour des informations plus détaillées sur ces paramètres, reportez-vous à la rubrique Définition des options d`affichage.

 

Notes :

  • Il n'est pas possible de supprimer les champs de résultat obligatoires. Ces champs, le cas échéant, s'affichent dans la section Obligatoire de la page Options d'affichage. Pour plus d'informations sur les champs de recherche obligatoires, reportez-vous à la rubrique Configuration des champs de résultats de recherche requis.

  • Les options disponibles dans le champ Trier par dépendent des régions d'index qui ont été configurées comme pouvant être triées. Pour plus d'informations, reportez-vous à la rubrique Configuration des régions d'index.

 

Les options d'affichage des Modèles de recherche peuvent être définies depuis les pages d'administration. Lorsque vous vous connectez au serveur Enterprise en tant qu'utilisateur administrateur, vous pouvez également modifier les modèles de recherche en sélectionnant Modèles de recherche dans le menu déroulant Personnel.
