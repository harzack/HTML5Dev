Pour définir les options d'affichage des résultats de recherche pour un Modèle de recherche système depuis le menu Personnel :

  1. Connectez-vous à Enterprise Server en tant qu'utilisateur Admin.

  2. Cliquez sur le menu Personnel, puis sur Modèles de recherche.

  3. Cliquez sur l'icône Fonctions du modèle de résultats de recherche, puis cliquez sur le lien Modifier les options d'affichage.

  4. Sur la page Options d'affichage, spécifiez les options d'affichage souhaitées.

  5. Cliquez sur le bouton Mettre à jour.
