Pour modifier un formulaire de recherche :

  1. Sur la page d'administration, cliquez sur le lien Ouvrir le volume des objets système.

  2. Cliquez sur le lien Modèles de recherche personnels.

  3. Procédez de l'une des manières suivantes :

    • Pour localiser un formulaire de recherche système, cliquez sur le lien Admin.

    • Pour localiser le formulaire de recherche personnel d'un utilisateur Content Server, cliquez sur le nom de connexion de l'utilisateur approprié.

  4. Cliquez sur l'icône Fonctions du formulaire de recherche, puis choisissez Modifier ou Propriétés, puis Spécifique.

  5. Modifiez les critères de recherche sur la page Recherche.

  6. Cliquez sur le bouton Mettre à jour.

 

Notes :

  • Vous pouvez également modifier le modèle de recherche système par défaut en cliquant sur le lien Modifier le modèle de système par défaut de la page Administration, puis en recréant le modèle.

  • Vous pouvez également modifier les formulaires de recherche créés en les recréant sur la page Recherche. Pour en savoir plus, voir To Reset the Default Search Template.
