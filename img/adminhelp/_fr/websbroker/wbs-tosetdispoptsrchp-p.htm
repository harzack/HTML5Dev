Pour définir les options d'affichage des résultats de recherche pour les Formulaires de recherche depuis la page Recherche :

  1. Connectez-vous à Enterprise Server en tant qu'utilisateur Admin.

  2. Cliquez sur le lien Recherche avancée ; dans la liste déroulante Utiliser ce formulaire de recherche, sélectionnez le Formulaire de recherche que vous souhaitez modifier.

  3. Dans la liste déroulante Style d'affichage des résultats, cliquez sur le lien Modifier les options d'affichage.

  4. Sur la page Options d'affichage, spécifiez les options d'affichage souhaitées.

  5. Cliquez sur le bouton Mettre à jour.
