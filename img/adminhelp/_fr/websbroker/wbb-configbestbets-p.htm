Pour configurer les paramètres de meilleurs résultats :

  1. Sur la page Administration, dans la section Administration des meilleurs résultats, cliquez sur le lien Configurer les paramètres de meilleurs résultats.

  2. Sur la page Configurer les paramètres de meilleurs résultats, dans la section Options des meilleurs résultats, sélectionnez la case à cocher Activer pour permettre à Content Server de surveiller le statut de ce processus.

  3. Saisissez un nombre de résultats de recherche dans le champ Nombre à afficher. La valeur par défaut est 5.

  4. Saisissez le libellé de la valeur de meilleurs résultats dans le champ Libellé de la page des propriétés.

  5. Saisissez le libellé de la date d'expiration dans le champ Libellé de la date d'expiration.

  6. Dans la section Correspondance des modèles de meilleurs résultats, dans le champ Nombre maximum de mots clés à vérifier, saisissez le nombre de termes de recherche à utiliser ou sélectionnez la case à cocher Aucune limite. La valeur par défaut est 4.

  7. Dans le champ Longueur minimum des mots clés, saisissez le nombre de caractères qu'un modificateur de racine de terme de recherche doit comporter avant d'être traduit en requête SQL. La valeur par défaut est 4.

  8. Dans la section Correspondance des modèles de surnom, sélectionnez la case à cocher Inclure les surnoms pour activer la recherche de surnoms.

  9. Dans le champ Nombre maximum de mots clés à vérifier, saisissez le nombre de termes de recherche à utiliser ou sélectionnez la case à cocher Aucune limite. La valeur par défaut est 3.

  10. Dans le champ Longueur minimum des mots clés, saisissez le nombre de caractères qu'un modificateur de racine de terme de recherche doit comporter avant d'être traduit en requête SQL. La valeur par défaut est 4.

  11. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Vous pouvez modifier la langue utilisée dans les champs Libellé de la page des propriétés et Libellé de la date d'expiration en saisissant une valeur Xlate au lieu de texte brut.
