Pour réinitialiser le modèle de recherche système par défaut :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur les liens Gérer les modèles de recherche, puis Réinitialiser les modèles de recherche.

  2. Sur la page Réinitialiser les modèles de recherche, cliquez sur le bouton radio Réinitialiser le modèle de recherche système par défaut.

  3. Cliquez sur Envoyer.

  4. Cliquez sur OK dans la boîte de dialogue qui vous invite à confirmer la suppression.
