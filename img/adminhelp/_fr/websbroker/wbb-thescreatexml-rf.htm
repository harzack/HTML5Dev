Exemple de fichier de modèle XML

Le code suivant est un exemple de modèle XML fourni par OpenText (inclus dans les installations Content Server sous la forme Content Server_accueil /config/sample_thesaurus.xml). Notez la première entrée <Headword>. Le mot « déjeuner » est le mot tête ou l'entrée principale, pour la définition du thésaurus. Le terme déjeuner pouvant être à la fois un nom ou un verbe, l'entrée du thésaurus pour déjeuner contient deux définitions distinctes, l'une pour le nom et l'autre pour le verbe.
