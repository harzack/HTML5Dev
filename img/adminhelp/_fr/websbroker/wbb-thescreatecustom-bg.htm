Vous pouvez créer un thésaurus personnalisé manuellement dans un éditeur de texte ou vous pouvez écrire un script traduisant vos données dans la syntaxe requise par l'utilitaire Personnalisation du thésaurus. L'utilitaire Personnalisation du thésaurus est un fichier exécutable intitulé buildths.bat (sous Microsoft Windows) ou buildths.sh (sous UNIX).

Avant de créer un thésaurus personnalisé, vous devez créer un fichier de modèle XML contenant vos propres mots synonymes spécialisés. Avec l'utilitaire Personnalisation du thésaurus, OpenText fournit un exemple de fichier de modèle XML que vous pouvez modifier. Par exemple, vous pouvez y ajouter des synonymes, les traductions de mots existants ou de nouveaux mots. Dans une entrée de thésaurus, le mot principal est appelé un mot tête.

Nom de la balise Contenu de la balise Emplacement de la balise
<OTThesaurus> Toutes les balises du modèle de thésaurus Au début du fichier de modèle
<Headword> Les informations relatives à un mot tête ou à un groupe de synonymes Dans la balise <OTThesaurus>
<Headword_Text> Le mot tête Dans la balise <Headword>
<Meaning> Les informations relatives à un sens particulier du mot tête. Un mot tête peut avoir plusieurs sens. Dans la balise <Headword>
<Meaning_Text> Le texte distinguant un sens particulier Dans la balise <Meaning>
<PartofSpeech> La partie porteuse de sens d'un mot tête (comme un nom, un verbe, un adverbe, un adjectif, etc.) Dans la balise <Meaning>
<Synonym> Un synonyme du mot tête Dans la balise <Meaning>

Après avoir créé un fichier de modèle XML, transférez le fichier du thésaurus en tant que paramètre dans l'utilitaire Personnalisation du thésaurus afin de créer le fichier du thésaurus. Les moteurs de recherche de Content Server utilisent ce fichier pour fournir des synonymes dans les résultats de la recherche. Le fichier du thésaurus généré avec cet utilitaire n'est pas mis à jour de manière dynamique. Si vous voulez ajouter des entrées à votre thésaurus, vous devez générer manuellement un nouveau fichier de thésaurus.
