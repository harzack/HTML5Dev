Une fois une collection créée et les éléments collectés, vous pouvez créer une image de disque de la collection. Une image de disque permet de copier tous les éléments d'une collection ou seulement certains d'entre eux vers le disque. Cette action peut être utilisée pour partager le contenu d'une collection avec des personnes ou des groupes extérieurs à l'environnement Content Server.

Lorsque vous configurez les paramètres de création de l'image de disque, vous pouvez spécifier des informations sur les collections et objets collectés pouvant être capturés, les tailles de disque à choisir et le moment d'envoi de la notification par e-mail. Vous pouvez également restreindre l'option Créer une image de disque à des utilisateurs et groupes Content Server spécifiques.

Spécification des paramètres de création de l'image de disque

Pour configurer l'option Créer une image de disque, vous devez d'abord configurer les paramètres utilisés lorsque vous écrivez des collections sur un disque dans la zone Paramètres de création d'image de disque des pages Administration.

 

Important :
Pour que le processus de création de l'image de disque fonctionne, vous devez avoir configuré une mémoire cache de fichier Content Server, également appelée cache Admin distant (RAC).

 

Sur la page Paramètres de création d'image de disque, vous pouvez configurer les paramètres suivants :

Paramétrage Description
Tailles de disque disponibles Définit les tailles de disque disponibles aux utilisateurs pendant le processus de création de l'image de disque et spécifie la valeur par défaut qui devrait apparaître. La taille du disque correspond à la taille prévue de la sortie du disque. Si la taille de la collection dépasse la taille spécifiée, une image de disque supplémentaire est créée.
Collection Définit les informations Content Server sur la Collection qui peut être incluse dans l'image de disque et indique si les utilisateurs sont en mesure de modifier ces paramètres. Par exemple, si vous voulez toujours que la description de la collection soit incluse dans les images de disque et que vous ne souhaitez pas que les utilisateurs puissent l'exclure, sélectionnez la case à cocher Actif, mais pas Configuration utilisateur.
Contenu Définit les informations Content Server sur les éléments de la collection qui peut être incluse dans l'image de disque et indique si les utilisateurs sont en mesure de modifier ces paramètres. Par exemple, si vous voulez toujours que les descriptions des éléments de la collection soient incluses dans les images de disque et que vous ne souhaitez pas que les utilisateurs puissent les exclure, sélectionnez la case à cocher Actif, mais pas Configuration utilisateur.
Options de notification Définit le point dans le processus de création de l'image de disque où la notification par e-mail doit être envoyée. Pour plus d'informations sur la notification par e-mail et le contenu des messages électroniques, consultez Description de la notification de création d`image de disque.
Nombre maximal d'objets par image Permet aux utilisateurs de spécifier le nombre d'objets distincts à inclure dans chaque image. Cette valeur, ainsi que les limitations de format du système de fichiers de disque, entraînent la génération de nombreux répertoires de sortie lorsque le volume de sortie dépasse la valeur et les limitations de format. Vous pouvez désactiver la case si vous ne voulez pas que les utilisateurs spécifient un paramètre Nombre maximal d'objets par image.
 

Note :
Pour plus d'informations sur les formats du système de fichiers de disque, consultez Création et utilisation des images de disque.

 
Administration des notifications Définit le point dans le processus de création de l'image de disque où la notification par e-mail doit être envoyée à l'administrateur Content Server. Vous pouvez désactiver ou activer cette option, préciser le nombre d'éléments dans un processus de création d'image de disque qui génère une notification à l'administrateur, et spécifier l'adresse de courrier électronique à laquelle cette notification doit être envoyée.
 

Note :
Cette option n'est pas configurable par les utilisateurs pendant le processus de création Créer une image de disque.

 
Zone Mappage de répertoire :  
Serveur d'administration Définit l'emplacement de l'antémémoire du fichier Content Server où sont conservées les images de disque. Vous pouvez cliquer sur le lien Paramètres de l'antémémoire de fichier pour sélectionner une antémémoire de fichier spécifique et modifier des valeurs supplémentaires. Pour plus d'informations sur les paramètres de l'antémémoire de fichier, reportez-vous à la rubrique Configuration de la mémoire cache de fichier du serveur d`administration.
Valeur du schéma Définit l'URL permettant d'accéder à l'image de disque. Cette valeur apparaît dans le message de confirmation et le courrier électronique de notification lors de la création d'une image de disque. Lorsque vous spécifiez cette valeur, vous pouvez utiliser un serveur FTP, ou fournir un emplacement de partage réseau UNC ou NFS.

 

Note :
Par défaut, lorsqu'une image de disque est créée, elle est supprimée de l'antémémoire de fichier le lendemain de la création. Vous pouvez modifier cette valeur en changeant le paramètre Heure d'expiration dans l'onglet Antémémoire de fichier de la page Propriétés du serveur d'administration. Vous pouvez accéder rapidement à cette page en cliquant sur le lien Paramètres de la mémoire cache de fichier dans la zone Serveur d'administration de la page Paramètres de création d'image de disque.

 

Pour plus d'informations sur la création d'images de disque, reportez-vous à la section Création et utilisation des images de disque

Restriction de l'option Créer une image de disque

L'option Créer une image de disque s'affiche par défaut dans la liste Autres actions lorsque vous affichez une collection. Vous pouvez restreindre cette option à des utilisateurs ou groupes spécifiques en mettant à jour le privilège d'utilisation Créer une image de disque. Par exemple, si vous avez un groupe qui est responsable de diffuser des informations à des sources tierces, et que vous voulez seulement que ce groupe puisse copier les informations vers une image de disque, vous pouvez limiter cette option à ce Groupe exclusivement. Pour plus d'informations sur la restriction des privilèges d'utilisation relatifs aux objets, reportez-vous à la rubrique Gestion des privilèges relatifs aux objets et à l`utilisation.
