Pour configurer des paramètres d'image de disque :

  1. Sur la page Administration, dans la section Administration des collections, cliquez sur le lien Paramètres de création d'image de disque.

  2. Sur la page Paramètres de création d'image de disque, dans la zone Tailles de disque disponibles :

    1. Sélectionnez la case à cocher Disponible en regard de chaque type Libellé que vous souhaitez afficher en tant qu'option sur la page Créer une image de disque.

    2. Sélectionnez le bouton radio Par défaut en regard du type Libellé que vous voulez désigner par défaut sur la page Créer une image de disque.

  3. Facultatif Dans la zone Collections :

    1. Sélectionnez la case à cocher Actif en regard de chaque type Purger aussi que vous souhaitez inclure lors de la création d'une image de disque. Ces types représentent différentes informations de collection à inclure lors de la création d'une image de disque.

    2. Sélectionnez la case à cocher Configuration utilisateur en regard de chaque type Purger aussi dont vous souhaitez autoriser la configuration par les utilisateurs.

  4. Facultatif Dans la zone Contenu :

    1. Sélectionnez la case à cocher Actif en regard de chaque type Purger aussi pour spécifier si les informations de contenu sur les éléments collectés doivent être incluses lors de la création d'une image de disque.

    2. Sélectionnez la case à cocher Configuration utilisateur en regard de chaque type Purger aussi dont vous souhaitez autoriser la configuration des informations de contenu par les utilisateurs.

  5. Dans la zone Options de notification :

    1. Sélectionnez la case à cocher Disponible en regard de chaque type de notification que vous souhaitez afficher comme option sur la page Créer une image de disque.

    2. Cliquez sur le bouton Par défaut pour spécifier le type de notification que vous voulez voir apparaître par défaut sur la page Créer une image de disque.

  6. Cochez la case Nombre maximum d'éléments par image pour autoriser un utilisateur à spécifier le nombre maximum d'éléments par image de disque.

  7. Selon sa configuration, vous pouvez voir Options de langue des métadonnées. Dans la zone Options de langue des métadonnées :

    1. Dans le champ Langue par défaut, effectuez une sélection dans la liste.

    2. Sélectionnez la case à cocher Configuration utilisateur pour autoriser l'utilisateur à configurer la langue des métadonnées pour l'image de disque.

  8. Dans la zone Notification de l'administrateur, pour configurer les e-mails de notification pour la création d'une image de disque, procédez comme suit :

    1. Cochez la case Activé.

    2. Dans le champ Niveau de notification, entrez un nombre positif. Lorsque ce nombre est atteint, une notification est envoyée à l'adresse e-mail indiquée dans le champ Adresse e-mail.

    3. Dans le champ Adresse e-mail, entrez l'adresse e-mail de l'utilisateur ou de l'administrateur qui devrait recevoir la notification de création de l'image de disque.

  9. Dans la zone Mappage de répertoire :

    1. Facultatif Pour modifier les paramètres de mémoire cache de fichier, cliquez sur le lien Paramètres de la mémoire cache de fichiers.

      Dans l'onglet Mémoire cache de fichiers, modifiez les informations selon les besoins, puis cliquez sur Mettre à jour.

    2. Dans le champ Valeur du schéma, entrez l'URL permettant d'accéder aux images de disque après la création.

  10. Cliquez sur Mettre à jour.
