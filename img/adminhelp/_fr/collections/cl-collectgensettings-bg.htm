Configurez certains paramètres des collections sur la page Paramètres généraux des collections, qui contrôle les paramètres Télécharger comme feuille de calcul.

Vous pouvez configurer les paramètres suivants :

Paramétrage Description
Paramètres Télécharger comme feuille de calcul
Nombre maximal d'éléments
Définit le nombre maximal d'Objets pouvant être exportés à l'aide de Télécharger comme feuille de calcul. Si un plus grand nombre d'objets est sélectionné, l'utilisateur reçoit un message d'erreur. La valeur par défaut est 250000.
Sélectionnez la case à cocher Aucune limite pour ne pas restreindre le nombre d'objets pouvant être exportés.
Seuil de compression
Définit la valeur maximale de la taille des feuilles de calcul téléchargées, en kilo-octets (Ko). Les feuilles de calcul plus importantes sont compressées et converties en fichier ZIP pour téléchargement. La valeur par défaut est 5120.
Sélectionnez la case à cocher Aucune limite pour ne pas convertir les feuilles de calcul téléchargées en fichiers ZIP.
Nombre d'éléments par feuille de calcul
Définit le nombre maximal d'Objets pouvant être placés dans une seule feuille de calcul. Si le nombre d'éléments est plus important, alors plusieurs feuilles de calcul sont créées puis compressées dans un seul fichier ZIP. La valeur par défaut est 30000.
Sélectionnez la case à cocher Aucune limite afin de placer tous les éléments dans une seule feuille de calcul.
