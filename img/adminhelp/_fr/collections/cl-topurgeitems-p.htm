Pour éliminer des enregistrements d'audit des éléments collectés :

  1. Sur la page Administration, dans la section Administration des collections, cliquez sur le lien Eliminer les enregistrements d'audit des éléments de collections.

  2. Sur la page Eliminer les enregistrements d'audit des éléments de collections, saisissez le nombre de jours de stockage des informations sur les éléments collectés dans le champ Eliminer les données antérieures à.

  3. Cliquez sur Eliminer.
