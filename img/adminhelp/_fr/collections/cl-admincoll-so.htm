Les collections vous permettent de stocker des pointeurs vers les éléments OpenText Content Server. Grâce à ces pointeurs, vous pouvez rapidement et facilement accéder aux éléments Content Server originaux et organiser les informations à partir de plusieurs sites dans Content Server dans un emplacement unique.

Par exemple, vous pouvez regrouper des informations liées à un projet d'équipe à partir de plusieurs emplacements dans Content Server et les collecter de sorte que toutes les informations se trouvent dans un emplacement facile d'accès et d'entretien. La collection peut ensuite être utilisée comme emplacement central pour tous les documents de ce projet et par tous les membres du projet.

Lorsque vous administrez des collections, vous pouvez activer des options de collections spécifiques pour les rendre disponibles aux utilisateurs Content Server, configurer la capacité à créer une image de disque des éléments collectés et maintenir les enregistrements d'audit pour les éléments collectés.
