Pour configurer les paramètres généraux des collections

  1. Sur la page Administration, dans la section Administration des collections, cliquez sur le lien Paramètres généraux des collections.

  2. Sur la page Paramètres généraux des collections, dans la section Télécharger comme feuille de calcul, pour Nombre maximal d'éléments, saisissez le nombre maximal d'Objets pouvant être exportés à l'aide de Télécharger comme feuille de calcul. La valeur par défaut est 250000. Vous pouvez également sélectionner la case à cocher Aucune limite pour que le nombre d'objets exportés ne soit pas restreint.

  3. Pour Seuil de compression, saisissez la valeur maximale de la taille des feuilles de calcul téléchargées, en kilo-octets (Ko). La valeur par défaut est 5120. Vous pouvez également sélectionner la case à cocher Aucune limite pour que les feuilles de calcul téléchargées converties ne soient pas converties en fichiers ZIP.

  4. Pour Nombre d'éléments par feuille de calcul, saisissez le nombre maximal d'Objets pouvant être placés dans une seule feuille de calcul. La valeur par défaut est 30000. Vous pouvez également sélectionner la case à cocher Aucune limite afin de placer tous les éléments dans une seule feuille de calcul.

  5. Cliquez sur Enregistrer.
