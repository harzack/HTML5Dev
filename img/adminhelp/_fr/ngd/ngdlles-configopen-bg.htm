A partir de l'Explorateur Windows ou de Microsoft Outlook, vous pouvez configurer la façon dont s'ouvrent les documents Microsoft Office lorsque l'utilisateur double-clique dessus dans Enterprise Connect. Ce paramètre s'applique également aux raccourcis pointant vers des documents Office. L'option Toujours ouvrir un document en lecture seule est la valeur par défaut. Vous pouvez utiliser cette option si vous souhaitez que l'utilisateur ouvre une version en lecture seule lorsqu'il double-clique sur un document Word, Excel, PowerPoint, Project ou Visio. Utilisez l'option Ouvrir le document pour modification si les autorisations le permettent si vous souhaitez que l'utilisateur puisse modifier immédiatement un document Office qu'il ouvre en double-cliquant dessus.

 

Note :
L'utilisateur peut uniquement modifier un document pour lequel il dispose des autorisations de modification et de réservation.
