Pour gérer les e-mails portant le même nom :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres de messagerie d'Enterprise Connect.

  2. Accédez à la zone Génération de nom pour les e-mails.

  3. Dans la liste Lorsqu'un e-mail avec le même nom est ajouté à un dossier, effectuez l'une des opérations suivantes :

    • Cliquez sur Générer un nom avec un suffixe si vous souhaitez qu'Enterprise Connect génère automatiquement un nouveau nom pour le doublon d'e-mail. Une nouvelle instance de l'e-mail, avec un suffixe permettant de la différencier de l'instance déjà présente dans la destination, est ajoutée dans la destination. Dans la liste adjacente, vous devez choisir l'une des options suivantes :

      • Sélectionnez Demander à l'utilisateur de décider si vous souhaitez qu'Enterprise Connect invite les utilisateurs à spécifier s'ils souhaitent ajouter l'e-mail renommé automatiquement à l'emplacement de destination. C'est le paramètre par défaut.

      • Sélectionnez Générer un nom automatiquement si vous souhaitez qu'Enterprise Connect génère automatiquement un nouveau nom pour le doublon d'e-mail sans demander de confirmation préalable à l'utilisateur. L'utilisateur ne sera pas invité à résoudre les conflits de dénomination.

    • Cliquez sur Autoriser l'utilisateur à saisir un nouveau nom si vous souhaitez que les utilisateurs puissent saisir un nouveau nom pour le doublon d'e-mail.

  4. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.

 

Note :
Ces paramètres s'appliquent aux e-mails, aux e-mails composites et aux remplacements d'e-mails dans l'Explorateur Windows, Microsoft Outlook, Lotus Notes et GroupWise.
