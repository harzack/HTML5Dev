A partir de l'Explorateur Windows ou de Microsoft Outlook, vous pouvez configurer la façon dont s'ouvrent les e-mails composites lorsque l'utilisateur double-clique dessus dans Enterprise Connect. Ouvrir le message E-mail composé est le paramètre par défaut. Vous pouvez utiliser cette option si vous souhaitez que les e-mails composites s'ouvrent dans Outlook lorsque les utilisateurs cliquent dessus. Utilisez le paramètre Ouvrir les sous-éléments E-mail composé si vous souhaitez que l'e-mail et les pièces jointes à ouvrir s'affichent dans un panneau Web distinct lorsque les utilisateurs double-cliquent sur un e-mail composite. A partir de ce panneau Web, les utilisateurs peuvent modifier, ouvrir ou télécharger l'e-mail ou les pièces jointes, en fonction des autorisations dont ils disposent. Ce panneau offre les mêmes fonctionnalités que l'onglet Sous-éléments du volet Aperçu. Ces paramètres s'appliquent aux e-mails composites auxquels peuvent accéder les utilisateurs et aux e-mails composites renvoyés dans les résultats de recherche.
