Le Enterprise Connect Viewer permet aux utilisateurs d'afficher les éléments dans les formats pris en charge via l'onglet Afficher du Volet Lecture, même si l'application native n'est pas installée. Par exemple, un utilisateur peut afficher un élément créé dans Microsoft Projet même si Project n'est pas installé sur l'ordinateur. La Enterprise Connect Viewer permet aux utilisateurs d'imprimer, de rechercher, de copier et de régler l'affichage de l'élément sélectionné. La Enterprise Connect Viewer est installée automatiquement avec Enterprise Connect et ne nécessite aucune configuration supplémentaire.

A la place de la Enterprise Connect Viewer, vous pouvez utiliser OpenText Document Filters pour permettre aux utilisateurs d'afficher les éléments dans les formats pris en charge. Vous pouvez spécifier le visualiseur à utiliser dans la zone Technologie d'affichage de la page Configurer les paramètres d'affichage. Pour plus d'informations sur les Document Filters, reportez-vous à la rubrique OpenText Document Filters - Administrator's Guide (LLESCF-AGD).
