Pour afficher des éléments masqués dans Enterprise Connect :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres d'affichage d'Enterprise Connect.

  2. Dans la section Configuration des éléments masqués, effectuez l'une des opérations suivantes :

    • Activez la case Afficher les éléments masqués pour afficher tous les éléments masqués dans Enterprise Connect.

    • Désactivez la case Afficher les éléments masqués si vous ne souhaitez pas afficher les éléments masqués dans Enterprise Connect. Il s'agit de la valeur par défaut.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
