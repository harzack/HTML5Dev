Pour chaque type de conteneur, vous pouvez effectuer les tâches suivantes à partir de la page Configurer les paramètres d'affichage des colonnes :

  • Spécifier les colonnes disponibles aux utilisateurs d'Enterprise Connect.

  • Spécifier les colonnes accessibles par défaut aux utilisateurs.

  • Définir l'ordre dans lequel apparaissent les colonnes.

  • Personnaliser la largeur de chaque colonne.

 

Important :
Les utilisateurs doivent ouvrir une nouvelle session Enterprise Connect pour accéder aux modifications administratives que vous apportez.
