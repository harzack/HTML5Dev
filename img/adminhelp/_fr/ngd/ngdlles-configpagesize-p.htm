Pour définir le nombre de résultats de recherche sur chaque page :

  1. Sur la page Administration de Content Server, cliquez sur Administration d'Enterprise Connect, puis cliquez sur le lien Configurer les paramètres de recherche d'Enterprise Connect.

  2. Dans la section Configuration des résultats de recherche, sélectionnez le nombre de résultats de recherche approprié à afficher sur chaque page dans la liste Taille de la page de recherche. La valeur par défaut est 25.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
