Pour configurer des versions prises en charge de la structure Enterprise Connect :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres système de Enterprise Connect.

  2. Dans la zone Configuration de la prise en charge de la compatibilité ascendante, effectuez l'une des opérations suivantes :

    • Sélectionnez Empêcher la compatibilité ascendante avec les versions antérieures du client Enterprise Connect si vous souhaitez forcer les utilisateurs à mettre à niveau immédiatement leur instance de la structure Enterprise Connect. Vous pouvez mettre à jour le message si nécessaire.

    • Désactivez Empêcher la compatibilité ascendante avec les versions antérieures du client Enterprise Connect si vous souhaitez autoriser les clients à utiliser leur version actuelle de la structure Enterprise Connect.

    • Personnalisez le message de mise à niveau, si nécessaire.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
