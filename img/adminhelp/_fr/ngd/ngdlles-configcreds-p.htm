Pour configurer la façon dont les informations d'identification d'Enterprise Connect sont stockées :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres système d'Enterprise Connect.

  2. Dans la zone Configuration de la gestion des informations d'identification, effectuez l'une des opérations suivantes :

    • Cochez la case Autoriser le client Enterprise Connect à stocker les informations d'identification de l'utilisateur sur le disque si vous voulez stocker les informations d'identification d'utilisateur en local. Par défaut, cette case est cochée.

    • Désactivez la case à cocher Autoriser le client Enterprise Connect à stocker les informations d'identification de l'utilisateur sur le disque si vous ne voulez pas stocker les informations d'identification d'utilisateur en local.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
