Pour configurer l'affichage des colonnes :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres d'affichage des colonnes d'Enterprise Connect.

  2. Sur la page Configurer les paramètres d'affichage des colonnes, cliquez sur Modifier la configuration pour le conteneur que vous souhaitez configurer.

     

    Note :
    Dans le client Enterprise Connect, les utilisateurs peuvent ajouter, supprimer et réorganiser les colonnes des conteneurs selon leurs préférences. Pour plus d'informations sur l'utilisation des colonnes, reportez-vous à la rubrique OpenText Enterprise Connect Plug-in for Content Server - Manuel de l'utilisateur (NGDCORE-UGD).

     

  3. Sur la page de configuration du conteneur sélectionné, procédez comme suit pour chacune des colonnes :

    1. Dans la colonne Disponible, décochez la case de toutes les colonnes que vous souhaitez rendre inaccessibles aux utilisateurs. Par défaut, certaines colonnes peuvent être sélectionnées. Vous devez donc décocher manuellement toutes les colonnes à masquer.

       

      Astuce :
      Cochez ou décochez la case en haut de la colonne Disponible pour sélectionner ou désélectionner toutes les colonnes.

       

    2. Dans la colonne Afficher par défaut, cochez la case des colonnes que vous souhaitez rendre accessibles par défaut aux utilisateurs.

       

      Astuce :
      Cochez ou décochez la case en haut de la colonne Afficher par défaut pour sélectionner ou désélectionner toutes les colonnes.

       

    3. Dans la colonne Largeur des colonnes, spécifiez la largeur, en pixels, à appliquer aux colonnes spécifiques.

       

      Note :
      Si vous n'appliquez aucune valeur, Enterprise Connect dimensionne automatiquement les colonnes.

       

    4. Dans la colonne Ordre de l'affichage, utilisez les boutons Haut et Bas pour placer la colonne sélectionnée à l'endroit voulu.

  4. Une fois la configuration effectuée, procédez de l'une des manières suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
