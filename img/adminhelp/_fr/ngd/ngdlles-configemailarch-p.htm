Pour configurer les paramètres d'archivage des e-mails :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres d'email d'Enterprise Connect.

  2. Dans la zone Configuration de l'archivage des e-mails, effectuez l'une des opérations suivantes :

    • Sélectionnez l'option Activer l'archivage des e-mails pour Enterprise Connect. Cette option s'applique aux e-mails ajoutés à Content Server depuis la boîte de réception si vous voulez activer l'archivage des e-mails au moyen d'Enterprise Connect.

    • Désactivez l'option Activer l'archivage des e-mails pour Enterprise Connect. Cette option s'applique aux e-mails ajoutés à Content Server depuis la boîte de réception si vous ne voulez pas activer l'archivage des e-mails au moyen d'Enterprise Connect. Au lieu de cela, les e-mails sont enregistrés dans le recueil Content Server sélectionné et ne sont pas archivés. Par défaut, la case est désactivée.

  3. Dans la zone Résolution de la messagerie de commandes, effectuez l'une des actions suivantes pour spécifier la façon dont la messagerie de commandes est déterminée :

    • Cliquez sur A partir d'un serveur ou d'un groupe d'administration pour sélectionner la messagerie de commandes au moyen de paramètres d'un groupe d'administration ou d'un serveur. C'est le paramètre par défaut.

    • Cliquez sur A partir d'une propriété présentant la valeur hex, puis tapez la valeur de propriété appropriée sous forme hexadécimale dans la zone hex pour sélectionner la messagerie de commandes au moyen d'une propriété.

     

    Attention
    Vous devez utiliser les mêmes valeurs dans la configuration Enterprise Connect que celles que vous avez utilisées lors de la configuration du module Email Management Services.

     

  4. Procédez de l'une des façons suivantes :

    • Cochez la case Publier des commandes dans la messagerie de commandes pour publier les commandes d'archivage directement dans la messagerie de commandes. C'est le paramètre par défaut.

    • Décochez la case Publier des commandes dans la messagerie de commandes si vous voulez envoyer les commandes d'archivage dans la messagerie par e-mail.

     

    Attention
    Vous devez utiliser les mêmes valeurs dans la configuration Enterprise Connect que celles que vous avez utilisées lors de la configuration du module Email Management Services.

     

  5. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
