Vous pouvez exécuter les tâches suivantes sur la page Configurer les paramètres avancés :

  • Spécifier la façon dont s'ouvrent les documents Microsoft Office lorsque l'utilisateur double-clique dessus dans Enterprise Connect.

  • Spécifier la façon dont s'ouvrent les e-mails composites lorsque l'utilisateur double-clique dessus dans Enterprise Connect.

  • Spécifier les options qui s'affichent dans le menu contextuel.

  • Spécifier les paramètres de la fonction Enterprise Connect depuis ici.

  • Spécifier quand les utilisateurs doivent entrer les métadonnées dans le cadre de l'ajout d'éléments à Content Server.

  • Spécifier le type de lien que les utilisateurs peuvent copier dans le Presse-papiers.

  • Spécifier les types de dossiers que les utilisateurs peuvent récupérer hors ligne.

  • Spécifier les types d'éléments que les utilisateurs peuvent télécharger sur leur bureau.

  • Spécifier les emplacements où les utilisateurs peuvent accéder à la commande Copier/Déplacer vers Enterprise Connect.

  • Personnaliser le sous-type appliqué lors du glisser-déposer de certains types de fichiers.

 

Important :
Les utilisateurs doivent ouvrir une nouvelle session Enterprise Connect pour accéder aux modifications administratives que vous apportez.
