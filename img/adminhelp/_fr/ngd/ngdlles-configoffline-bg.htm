Vous devez spécifier les sous-types que les utilisateurs d'Enterprise Connect peuvent récupérer hors ligne. Si les utilisateurs cliquent avec le bouton droit de la souris sur un élément d'un sous-type ne pouvant pas être utilisé hors ligne, l'option Télécharger la version hors ligne n'est pas disponible. Si les utilisateurs sélectionnent plusieurs éléments et que l'un d'entre eux est exclu de l'utilisation hors ligne, l'option de téléchargement pour utilisateur hors ligne n'est pas disponible dans le menu contextuel.

Par défaut, les sous-types Dossiers, Documents composites, Documents, Projets, Collections, Favoris, E-mails et Dossiers d'e-mails peuvent être téléchargés pour une utilisation hors ligne. Les ID de ces sous-types s'affichent automatiquement dans la zone Inclure les sous-types. Vous pouvez ajouter ou supprimer des sous-types si nécessaire. Pour plus d'informations sur les autres valeurs de sous-type, consultez la section Correspondance entre le nom et le numéro des types de nœud.
