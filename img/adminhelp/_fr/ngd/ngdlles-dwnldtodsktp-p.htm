Pour spécifier les sous-types qui peuvent être téléchargés sur le bureau :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la section Configuration de la fonction Télécharger sur le bureau, dans la zone Inclure les sous-types, saisissez les valeurs des sous-types supplémentaires que les utilisateurs peuvent télécharger vers leur bureau. Utilisez une virgule pour séparer les valeurs. Les valeurs suivantes sont indiquées par défaut (si nécessaire, vous pouvez les supprimer) :

    • 0 pour les dossiers.

    • 136 pour les documents composites.

    • 144 pour les documents.

    • 202 pour les projets.

    • 298 pour les collections.

    • 557 pour les e-mails composites.

    • 628 pour les favoris.

    • 749 pour les e-mails.

    • 751 pour les dossiers d'e-mails.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
