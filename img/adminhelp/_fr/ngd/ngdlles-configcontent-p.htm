Pour configurer les paramètres de capture de contenu :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la zone Configuration de la capture des métadonnées, effectuez l'une des opérations suivantes dans la liste Demander les métadonnées lors de l'ajout de contenu pour spécifier le moment auquel les utilisateurs doivent être invités à fournir les métadonnées des éléments qu'ils ajoutent à Content Server :

    • Cliquez sur Uniquement si nécessaire pour inviter les utilisateurs à fournir les métadonnées uniquement si celles-ci sont obligatoires pour le dossier Content Server de destination. C'est le paramètre par défaut.

    • Cliquez sur Toujours pour inviter les utilisateurs à fournir les métadonnées de chaque élément qu'ils ajoutent à Content Server.

  3. Dans la zone Configuration de la capture des métadonnées, effectuez l'une des opérations suivantes dans la liste Demander les métadonnées de modules complémentaires lors du déplacement/de la copie au sein de Content Server pour spécifier le moment auquel les utilisateurs doivent être invités à fournir les métadonnées quand de nouveaux modules Content Server sont installés dans votre environnement :

    • Cliquez sur Toujours si vous souhaitez que les utilisateurs soient toujours invités à fournir des métadonnées dès qu'ils copient ou qu'ils déplacent des éléments au sein de Content Server. C'est le paramètre par défaut.

    • Cliquez sur Jamais si vous souhaitez que les utilisateurs soient invités à fournir des métadonnées uniquement quand ils copient ou déplacent des éléments au sein de Content Server et lorsque les conditions de catégories sont différentes pour la source et pour la destination.

  4. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
