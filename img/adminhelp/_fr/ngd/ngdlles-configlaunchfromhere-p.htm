Pour rendre Enterprise Connect accessible à partir de l'interface Web de Content Server :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la zone Configuration des actions, procédez de la façon suivante :

    • Dans la zone Etiquettes pour « Enterprise Connect depuis ici », saisissez le libellé que vous souhaitez afficher dans le menu contextuel. Le libellé Enterprise Connect depuis ici est utilisé par défaut.

    • Dans la zone Sous-types pour « Enterprise Connect depuis ici », ajoutez ou supprimez les sous-types pour lesquels vous souhaitez que cette fonctionnalité soit disponible. Par défaut, les sous-types suivants prennent en charge cette fonctionnalité :

      • Dossiers (0)

      • Espaces de travail Entreprise (141)

      • Espaces de travail Personnel (142)

      • Projets (202)

      • Dossiers d'e-mails (751)

      • Dossiers virtuels (899)

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
