Vous pouvez autoriser les utilisateurs à accéder à Enterprise Connect lorsqu'ils utilisent des types de contenu spécifiques dans l'interface Web de Content Server. Lorsque cette fonction est disponible, les utilisateurs peuvent accéder au menu Fonction d'un élément pris en charge et choisir Enterprise Connect depuis ici ; l'élément sélectionné apparaît alors dans une nouvelle instance de l'Explorateur Windows. En tant qu'administrateur, déterminez les sous-types pour lesquels cette fonction sera disponible. Par défaut, cette action est intitulée Enterprise Connect depuis ici. Si nécessaire, vous pouvez renommer cet intitulé.

 

Note :
cette fonction est uniquement disponible dans Internet Explorer. En outre, elle sera disponible uniquement dans l'interface Web de Content Server pour les éléments que vous spécifiez.
