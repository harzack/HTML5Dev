Pour configurer les paramètres de capture des pièces jointes d'e-mails :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres d'email d'Enterprise Connect.

  2. Accédez à la section Configuration de la capture des e-mails.

  3. Si vous souhaitez utiliser les sous-types d'e-mails, cliquez sur Enregistrer des e-mails comme sous-type d'e-mails pour appliquer le sous-type d'e-mail (749) aux e-mails. C'est le paramètre par défaut. Dans la liste Lors de l'ajout d'e-mails contenant des pièces jointes, choisissez l'une des options suivantes pour spécifier la méthode d'enregistrement des pièces jointes :

    • Cliquez sur Ne pas ajouter de copies séparées des pièces jointes si vous souhaitez enregistrer l'e-mail et les pièces jointes en tant qu'élément unique. C'est le paramètre par défaut.

    • Cliquez sur Ajouter des copies séparées des pièces jointes si vous souhaitez enregistrer l'e-mail et les pièces jointes en tant qu'élément unique et enregistrer toutes les pièces jointes en tant qu'éléments séparés.

    • Cliquez sur Demander à l'utilisateur de décider si vous souhaitez autoriser les utilisateurs à déterminer comment enregistrer les pièces jointes. Ils peuvent soit enregistrer l'e-mail et les pièces jointes en tant qu'enregistrement unique, soit enregistrer l'e-mail et les pièces jointes ensemble, puis enregistrer séparément une copie supplémentaire des pièces jointes.

  4. Si vous souhaitez définir une méthode d'identification des doublons d'e-mails, procédez comme suit :

    • Activez la case Utiliser l'ID de message pour rechercher les e-mails en double si vous souhaitez qu'Enterprise Connect identifie les doublons d'e-mails en comparant l'ID des messages. Par défaut, cette case à cocher est désactivée et seule la comparaison basée sur le nom est utilisée.

    • Vous pouvez également activer la case Utiliser l’ID des messages externes pour rechercher les e-mails en double si vous souhaitez qu'Enterprise Connect identifie les doublons d'e-mails en comparant les ID externes. Par défaut, cette case est désactivée.

       

      Note :
      Vous pouvez activer cette option uniquement si vous avez activé la case Utiliser l'ID de message pour rechercher les e-mails en double.

       

  5. Si vous souhaitez utiliser les sous-types d'e-mails composites, cliquez sur Enregistrer vos e-mails (avec ou sans pièce jointe) sous forme d'e-mails composés pour appliquer le sous-type d'e-mails composites (557).

  6. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
