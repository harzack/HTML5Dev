Pour configurer la structure d'arborescence et le volet Liste :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres d'affichage d'Enterprise Connect.

  2. Dans la zone Serveur de la section Configuration de l'arbre, de la liste et du volet de lecture, procédez comme suit pour spécifier les éléments qui apparaissent sous le nœud Serveur :

    • Dans la liste Eléments disponibles, sélectionnez l'élément que vous voulez afficher sous le nœud du serveur, cliquez sur Déplacer vers la gauche pour déplacer l'élément dans la liste Nœud du serveur. Incluez le Nœud Volumes et le nœud Autres éléments dans cette liste si vous voulez exposer ces éléments et leur contenu sous le nœud du serveur. Les éléments restants de la liste Eléments disponibles ne sont pas affichés.

    • Sélectionnez l'élément que vous voulez faire apparaître sous le Nœud Volumes ou le nœud Autres éléments, puis cliquez sur Déplacer vers la gauche ou Déplacer vers la droite pour déplacer l'élément dans la liste appropriée. Incluez le Nœud Volumes et le nœud Autres éléments dans la liste Nœud Serveur si vous voulez exposer ces éléments et leur contenu sous le nœud du serveur.

    • Pour modifier l'ordre d'une liste, sélectionnez l'élément que vous voulez déplacer, puis cliquez sur Haut ou Bas .

  3. Pour sélectionner les éléments qui apparaissent sous le nœud Eléments personnels, accédez à la section Nœud Personnel et procédez comme suit :

    • Sélectionnez l'élément que vous voulez faire apparaître sous le nœud Eléments personnels, puis cliquez sur Déplacer vers la droite pour déplacer l'élément dans la liste Eléments à utiliser. Les éléments restants de la liste Eléments disponibles ne sont pas affichés.

    • Pour modifier l'ordre d'une liste, sélectionnez l'élément que vous voulez déplacer, puis cliquez sur Haut ou Bas .

     

    Note :
    L'élément Mes paramètres permet aux utilisateurs d'Enterprise Connect d'ouvrir la page Mes paramètres Enterprise Connect, depuis laquelle ils peuvent contrôler les paramètres de génération de noms d'e-mails. Cette page inclut tous les paramètres disponibles dans la zone Génération de nom pour les e-mails, dans la page Configurer les paramètres de messagerie d'Enterprise Connect. Les utilisateurs pourront remplacer des paramètres que vous avez appliqués. Par défaut, cet élément se trouve dans la liste Eléments à utiliser. Si vous ne souhaitez pas que les utilisateurs puissent accéder à cette page, déplacez-la dans la liste Eléments disponibles. Pour plus d'informations sur les règles de dénomination des e-mails, consultez Définition des conventions de dénomination des e-mails.

     

  4. Une fois vos sélections faites, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
