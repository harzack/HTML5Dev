Ce paramètre vous permet de spécifier si les e-mails et leurs pièces jointes doivent être archivés lorsque les utilisateurs les ajoutent à Content Server à partir de leur boîte de réception. Par défaut, l'archivage des e-mails n'est pas activé. Pour plus d'informations sur la façon d'activer l'archivage des e-mails et la façon dont les utilisateurs peuvent archiver les e-mails au moyen d'Enterprise Connect, voir OpenText Enterprise Connect - Installation Guide (NGDCORE-IGD) .

 

Note :
Si vous activez l'archivage des e-mails, tous les autres paramètres d'e-mails dans la page Configurer les paramètres de messagerie d'Enterprise Connect ne s'appliquent pas aux e-mails qui sont archivés. Au lieu de cela, lorsque les e-mails et leurs pièces jointes sont archivés, les paramètres que vous avez spécifiés dans le cadre de la configuration du module Email Management Services sont appliqués.

 

Pour activer l'archivage des e-mails, vous devez d'abord spécifier la façon dont la messagerie de commandes appropriée est déterminée pour chaque utilisateur. Par défaut, la messagerie de commandes est déterminée en fonction du serveur ou du groupe d'administration auquel l'utilisateur appartient. Vous pouvez également utiliser une propriété d'utilisateur Active Directory, qui est définie en tant que valeur hexadécimale. Pour plus d'informations, consultez la section OpenText Email Archiving for Microsoft Exchange - Administration Guide (EA-AGD).

 

Note :
Cette valeur doit être identique à la valeur spécifiée dans la configuration du module Email Management Services. Microsoft Exchange 2013 requiert une résolution de la messagerie de commandes basée sur les propriétés.

 

Vous devez également spécifier si les commandes d'archivage sont publiées directement dans la messagerie de commandes, ou si elles sont envoyées par e-mail. Par défaut, la case Publier des commandes dans la messagerie de commandes est cochée. Il s'agit généralement de la méthode la plus efficace pour publier des commandes d'archivage. Toutefois, pour activer cette fonctionnalité, vous aurez besoin de modifier les autorisations définies sur la messagerie de commandes. Pour plus d'informations, consultez la section OpenText Email Archiving for Microsoft Exchange - Administration Guide (EA-AGD).

 

Note :
Vous devez avoir installé la version appropriée du module Email Management pour pouvoir utiliser la fonctionnalité d'archivage des e-mails. Un message d'erreur dans la section Configuration de l'archivage des e-mails vous indique si une mise à niveau est nécessaire.
