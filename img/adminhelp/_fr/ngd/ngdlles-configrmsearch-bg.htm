Important :
Pour que les raccourcis soient inclus dans les résultats de recherche, vous devez utiliser Content Server 10, avec mise à jour 7, ou ultérieur. Pour que les renvois soient inclus dans les résultats de recherche, vous devez utiliser OpenText Records Management (RM) 10.2.0, correctif 1, ou ultérieur. Si vous utilisez des versions antérieures de Content Server et RM, cette case à cocher n'apparaît pas.

 

Enterprise Connect prend en charge la recherche de raccourcis et de renvois. Pour que cette fonctionnalité soit disponible, les versions requises d'Enterprise Connect, de Content Server et de RM doivent être installées sur votre système. Content Server doit également être configuré. Pour plus d'informations sur l'utilisation de RM, reportez-vous à la rubrique OpenText Records Management - Installation and Administration Guide (LLESRCM-IGD). Pour plus d'informations sur la configuration de Content Server, reportez-vous à la rubrique Open Text Administration Help - Content Server (LLESCOR-H-AGM).

Une fois cette fonctionnalité activée, les raccourcis et les renvois pourront être inclus dans les résultats de recherche rapide et de recherche avancée Enterprise Connect. Par exemple, si un dossier inclu dans une recherche contient des raccourcis ou des renvois vers d'autres dossiers, ces dossiers référencés seront inclus dans les résultats de recherche.
