Pour configurer les actions par double-clic pour les e-mails composites :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la zone Configuration de l'action, cliquez sur l'une des options suivantes dans la liste Lorsque vous double-cliquez sur un E-mail composé :

    • Ouvrir le message E-mail composé pour ouvrir l'e-mail sélectionné dans Outlook. Il s'agit de la valeur par défaut.

    • Ouvrir les sous-éléments E-mail composé pour lancer un panneau Web distinct.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
