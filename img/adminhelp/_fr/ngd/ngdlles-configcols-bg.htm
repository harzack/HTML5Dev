Vous pouvez spécifier le comportement des colonnes lorsque les utilisateurs naviguent dans Enterprise Connect. Pour les conteneurs Content Server, tels que les dossiers, les dossiers composites et les étapes clés, vous pouvez définir les colonnes par défaut auxquelles les utilisateurs peuvent accéder ainsi que celles qu'ils sont autorisés à ajouter ou supprimer lorsqu'ils utilisent Outlook et l'Explorateur Windows. Vous pouvez également indiquer l'ordre dans lequel les colonnes sélectionnées apparaissent et personnaliser leur largeur. L'option Autre vous permet de définir ces paramètres pour les types de conteneur qui ne figurent pas dans la liste, comme les workflows ou les projets.

Les colonnes disponibles varient en fonction des types de conteneur, mais peuvent être les colonnes par défaut d'Enterprise Connect, les colonnes spécifiques aux modules Content Server (comme Records Management) et les colonnes globales dynamiques de Content Server. Les colonnes globales dynamiques de Content Server étant uniques à chaque dossier, elles n'apparaissent pas sur cette page. Elles srestent cependant accessibles par les utilisateurs dans les sélecteurs de colonnes d'Outlook et de l'Explorateur.

Les utilisateurs peuvent ajouter, supprimer ou réorganiser les paramètres de colonnes selon leurs préférences à partir du client Enterprise Connect. Si un utilisateur a personnalisé l'affichage par défaut des colonnes sur son client, toutes les modifications que vous apportez sur cette page s'appliquent uniquement si l'utilisateur rétablit les paramètres d'affichage des colonnes par défaut ou s'il ajoute chaque colonne à l'aide des sélecteurs de colonnes d'Outlook ou de l'Explorateur. Si vous spécifiez qu'une colonne n'est pas disponible, elle sera automatiquement supprimée de l'interface utilisateur client.
