L'option Télécharger sur le bureau ne peut être utilisée que pour les sous-types que vous rendez disponibles pour les utilisateurs. Si les utilisateurs sélectionnent plusieurs éléments et que l'un de ces éléments ne prend pas en charge la fonction Télécharger sur le bureau, l'option Télécharger sur le bureau ne sera pas disponible via le menu contextuel.

Par défaut, les sous-types Dossiers, Documents composites, Documents, Projets, Collections, Favoris, E-mails et Dossiers d'e-mails peuvent être téléchargés sur le bureau. Les ID de ces sous-types s'affichent automatiquement dans la zone Inclure les sous-types. Vous pouvez ajouter ou supprimer des sous-types si nécessaire. Pour plus d'informations sur les autres valeurs de sous-type, consultez la section Correspondance entre le nom et le numéro des types de nœud. Pour plus d'informations sur l'utilisation de la fonction Télécharger sur le bureau, reportez-vous à la rubrique OpenText Enterprise Connect Plug-in for Content Server - Manuel de l'utilisateur (NGDCORE-UGD).

 

Note :
Si les utilisateurs téléchargent des e-mails composites sur leur bureau, ils apparaîtront sous la forme d'un e-mail unique binaire.
