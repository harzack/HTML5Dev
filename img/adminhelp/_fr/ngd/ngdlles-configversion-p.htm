Pour configurer des paramètres de gestion des versions :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres système de Enterprise Connect.

  2. Dans la zone Configuration de la gestion des versions, cliquez sur l'une des options suivantes dans la liste Lorsque des documents sont enregistrés pendant une session de modification :

    • Conserver uniquement la dernière version si vous souhaitez ne conserver que la version finale créée par les utilisateurs pendant une session de modification.

    • Conserver toutes les versions si vous souhaitez conserver chaque version créée par les utilisateurs pendant une session de modification.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
