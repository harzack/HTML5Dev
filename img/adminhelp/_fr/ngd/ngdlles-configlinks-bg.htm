Vous devez configurer les paramètres d'administration de certains modules Content Server, tels que Email Services, pour pouvoir utiliser Enterprise Connect. Cliquez sur les liens dans la zone Paramètres de module externe pour accéder directement à la page administrative appropriée. Un message indique si vous devez installer un module requis. Pour plus d'informations sur l'installation et la configuration des modules requis, consultez la section « System Requirements » dans OpenText Enterprise Connect - Installation Guide (NGDCORE-IGD).
