Vous pouvez exécuter les tâches suivantes sur la page Configurer les paramètres d'affichage :

  • Configurer l'arborescence et le volet Liste d'Enterprise Connect.

  • Sélectionner le visualiseur à utiliser dans le Volet Lecture du client Enterprise Connect.

  • Spécifier si des éléments masqués sont affichés dans Enterprise Connect.

 

Important :
Les utilisateurs doivent ouvrir une nouvelle session Enterprise Connect pour accéder aux modifications administratives que vous apportez.
