Pour configurer des types d'objets lors du glisser-déposer de dossiers :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la zone Configuration Glisser-déposer personnalisée, effectuez l'une des opérations suivantes :

    • Dans la zone Document composite, spécifiez les extensions que vous voulez utiliser pour identifier les dossiers devant être traités comme documents composites, par exemple, CD. Par défaut, cette zone ne comporte aucune valeur.

    • Dans la zone Dossier d'e-mails, spécifiez les extensions que vous voulez utiliser pour identifier les dossiers devant être traités comme dossiers d'e-mails, par exemple, EM. Par défaut, cette zone ne comporte aucune valeur.

     

    Note :
    Si vous spécifiez plusieurs extensions, utilisez des virgules pour séparer les noms. Ne laissez pas d'espaces entre les noms.

     

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
