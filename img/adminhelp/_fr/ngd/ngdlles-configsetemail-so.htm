Vous pouvez exécuter les tâches suivantes sur la page Configurer les paramètres de messagerie :

  • Configurer les paramètres d'archivage des e-mails.

  • Spécifier comment les e-mails et les pièces jointes sont traités lorsque les utilisateurs les ajoutent à Content Server.

  • Spécifier les conventions et règles de dénomination utilisées pour les e-mails ajoutés à Content Server à l'aide d'Enterprise Connect.

  • Spécifier la façon dont les noms des pièces jointes d'e-mail en double sont gérés.

 

Important :
Les utilisateurs doivent ouvrir une nouvelle session Enterprise Connect pour accéder aux modifications administratives que vous apportez.
