Pour configurer les types de lien :

  1. Sur la page Administration de Content Server, cliquez sur Administration Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés d'Enterprise Connect.

  2. Dans la liste Envoyer le document sous forme de configuration de lien de la zone Choisir le type de lien à envoyer, effectuez l'une des opérations suivantes :

    • Cliquez sur Vue d'ensemble pour spécifier que les liens redirigent les utilisateurs vers la page Vue d'ensemble du document. Les liens des e-mails et des e-mails composites redirigent les utilisateurs vers la page Propriétés et les liens des recueils Enterprise Connect redirigent les utilisateurs vers le recueil spécifié. C'est le paramètre par défaut.

    • Cliquez sur Télécharger pour spécifier que les liens permettent aux utilisateurs de télécharger le document. Pour tous les autres types d'élément, le lien dirige les utilisateurs vers la page Propriétés de l'élément.

    • Cliquez sur Propriétés pour spécifier que les liens des documents, des e-mails et des e-mails composites redirigent les utilisateurs vers la page Propriétés. Les liens vers les recueils redirigent les utilisateurs vers le recueil spécifié.

  3. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
