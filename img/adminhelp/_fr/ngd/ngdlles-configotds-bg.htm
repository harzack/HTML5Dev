Note :
Cette procédure s'applique uniquement si vous utilisez la version 10.2.3 ou antérieure du client Enterprise Connect. Ce paramètre n'est pas nécessaire pour la version 10.3 (ou supérieure) du client Enterprise Connect.

 

Si vous utilisez la version 10.2.3 ou antérieure du client Enterprise Connect et si vous utilisez CA SiteMinder ou RSA Access Manager avec Content Server et OpenText Directory Services (OTDS), vous devez cocher la case Autoriser Content Server à récupérer le ticket OTDS à partir du serveur OTDS (uniquement pour le client Enterprise Connect 10.2.3 ou une version antérieure). Par défaut, cette option n'est pas sélectionnée.
