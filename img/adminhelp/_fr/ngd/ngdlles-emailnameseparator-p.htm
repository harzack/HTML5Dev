Pour définir le séparateur entre les propriétés de message :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres de messagerie de Enterprise Connect.

  2. Accédez à la zone Génération de nom pour les e-mails.

  3. Dans la zone Caractère de séparation, saisissez une valeur appropriée, par exemple & ou -. Si vous ne spécifiez aucune valeur, un espace est utilisé pour séparer les propriétés de message dans le nom de l'e-mail.

     

    Important :
    ?, *, /, \, <, >, |, ., : et " ne peuvent pas être utilisés comme séparateur.

     

  4. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
