Pour définir les propriétés de message à utiliser dans les noms des e-mails :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres de messagerie de Enterprise Connect.

  2. Accédez à la zone Génération de nom pour les e-mails, puis à la zone Propriétés.

  3. Cliquez sur la propriété de message que vous souhaitez utiliser dans la liste Propriétés disponibles, puis cliquez sur Déplacer vers la droite pour la déplacer dans la liste Propriétés sélectionnées. Objet est la valeur par défaut.

  4. Pour définir l'ordre dans lequel les propriétés de message apparaissent dans le nom de l'e-mail, cliquez sur une propriété de message dans la liste Propriétés sélectionnées, puis cliquez sur Haut ou Bas .

  5. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
