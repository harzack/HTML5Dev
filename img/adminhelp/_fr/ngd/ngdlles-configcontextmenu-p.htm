Pour configurer le menu contextuel :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres avancés de Enterprise Connect.

  2. Dans la zone Exclure ces actions du menu contextuel de la section Configuration des actions, saisissez l'ID de commande des autres éléments de menu contextuel auxquels vous ne souhaitez pas que les utilisateurs d'Enterprise Connect puissent accéder.

     

    Notes :

    • Séparez les ID de commande par une virgule.

    • N'utilisez pas d'espaces entre les virgules et les ID de commande.

    • Les valeurs d'ID de commande ne sont pas sensibles à la casse.

     

  3. Une fois les options appropriées spécifiées, exécutez l'une des opérations suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
