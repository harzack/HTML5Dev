Vous pouvez spécifier le type des liens vers un document que les utilisateurs peuvent ajouter à un e-mail dans Outlook. Ce paramètre s'applique également dans l'Explorateur Microsoft avec la commande Envoyer à pour ajouter un lien vers un document dans un e-mail.

Les options disponibles sont les suivantes :

  • Vue d'ensemble : Dans le cas d'un document, le lien dirige les utilisateurs vers la page Aperçu du document. Les liens des e-mails et des e-mails composites redirigent les utilisateurs vers la page Propriétés de l'e-mail. Les liens des recueils permettent aux utilisateurs de parcourir le recueil spécifié. C'est le paramètre par défaut.

  • Télécharger : Le lien permet aux utilisateurs de télécharger le document sélectionné. Pour tous les autres types d'élément, le lien dirige les utilisateurs vers la page Propriétés de l'élément.

  • Propriétés : Dans le cas d'un document, le lien dirige les utilisateurs vers la page Propriétés du document. Les liens des e-mails et des e-mails composites redirigent les utilisateurs vers la page Propriétés de l'e-mail. Les liens des recueils permettent aux utilisateurs de parcourir le recueil spécifié.
