Pour supprimer des chaînes des noms des e-mails :

  1. Sur la page Administration de Content Server, cliquez sur Administration de Enterprise Connect, puis cliquez sur le lien Configurer les paramètres de messagerie de Enterprise Connect.

  2. Accédez à la zone Génération de nom pour les e-mails.

  3. Dans la section Chaînes à supprimer, saisissez le nom de la chaîne à exclure des noms des e-mails dans la zone Nouvelle chaîne, puis cliquez sur Déplacer vers la droite pour déplacer la chaîne dans la liste Chaînes sélectionnées. Cliquez sur Supprimer pour supprimer une chaîne sélectionnée de cette liste.

    Exemple: Pour que RE: n'apparaisse pas dans les noms des e-mails, saisissez RE: dans le champ Nouvelle chaîne, puis cliquez sur Déplacer vers la droite pour ajouter RE: à la liste Chaînes sélectionnées. RE: n'apparaît pas dans les noms des e-mails. Pour que RE: apparaisse dans les noms des e-mails, sélectionnez RE: puis cliquez sur Supprimer . RE: sera supprimé de la liste Chaînes sélectionnées.

  4. Une fois votre sélection faite, exécutez l'une des actions suivantes :

    • Cliquez sur Envoyer pour appliquer les modifications.

    • Cliquez sur Réinitialiser pour revenir à la dernière configuration enregistrée.

    • Cliquez sur Définir la valeur par défaut pour revenir à la configuration par défaut, puis sur OK pour appliquer les paramètres par défaut.
