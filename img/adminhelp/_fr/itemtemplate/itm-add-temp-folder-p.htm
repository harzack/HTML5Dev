Pour créer un dossier Modèle de projet :

  1. Cliquez sur le lien Ouvrir le volume Modèles dans la section Administration des modèles d'élément de la page Administration Content Server.

  2. Cliquez sur Dossier Modèle dans le menu Ajouter un élément.

  3. Pour utiliser un nom différent du nom par défaut, saisissez-le dans le champ Nom.

  4. Pour ajouter une description du contenu du dossier, saisissez-la dans le champ Description.

  5. Pour modifier les catégories ou les attributs associés à ce dossier, cliquez sur le bouton Modifier.

  6. Pour ajouter le dossier à un recueil différent, cliquez sur le bouton Parcourir Content Server, naviguez jusqu'au recueil, puis cliquez sur le lien Sélectionner.

  7. Cliquez sur Ajouter un élément.
