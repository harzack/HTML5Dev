Pour ajouter un modèle de vue personnalisée :

  1. Cliquez sur le lien Ouvrir le volume Modèles dans la section Administration des modèles d'élément de la page Administration Content Server.

  2. Sur la page Modèles Content Server, cliquez sur Modèle de vue personnalisée du menu Ajouter un élément.

  3. Sur la page Ajouter des modèles de vue personnalisée, saisissez un nom de modèle dans le champ Nom.

  4. Dans le champ Source, effectuez l'une des opérations suivantes :

    • Cliquez sur le bouton radio Affichage personnalisé, cliquez sur le bouton Parcourir et recherchez le fichier affichagepersonnalisé.html à partir duquel vous voulez créer le modèle, puis cliquez sur son lien Sélectionner.

    • Cliquez sur le bouton radio Fichier, cliquez sur le bouton Parcourir et recherchez le fichier .xml à partir duquel vous voulez créer le modèle, puis cliquez sur Ouvrir

  5. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  6. Cliquez sur Ajouter.
