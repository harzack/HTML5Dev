Vous pouvez exporter des modèles de projet et des modèles de vue personnalisée. Une copie du modèle est alors enregistrée dans un fichier XML, qui peut être importé dans Content Server ou exporté vers un autre périphérique de stockage.

Lorsque vous exportez un modèle de vue personnalisée ou de projet, toutes les données stockées dans LLNode sont exportées.
