Pour ajouter un modèle de projet :

  1. Procédez de l'une des manières suivantes :

    • Cliquez sur le lien Ouvrir le volume Modèles, dans la section Administration des modèles d'élément de la page Administration Content Server, puis cliquez sur Modèle de projet dans le menu Ajouter un élément.

    • Cliquez sur l'icône Fonctions du projet et choisissez Créer un modèle.

  2. Pour utiliser un nom différent du nom par défaut, saisissez-le dans le champ Nom.

  3. Pour ajouter une description du modèle de projet, saisissez-la dans le champ Description.

  4. Pour créer un modèle à partir d'un projet existant, procédez comme suit :

    • Cliquez sur le bouton radio Projet.

    • Cliquez sur le bouton Parcourir Content Server, naviguez jusqu'au projet, puis cliquez sur son lien Sélectionner.

  5. Pour créer un modèle à partir d'un fichier XML, procédez comme suit :

    • Cliquez sur le bouton radio Fichier (*.xml).

    • Cliquez sur le bouton Parcourir, naviguez jusqu'au fichier XML, puis cliquez sur le bouton Ouvrir.

  6. Pour modifier les catégories ou les attributs associés à l'élément, cliquez sur le bouton Modifier.

  7. Pour ajouter le modèle à un recueil différent, cliquez sur le bouton Parcourir Content Server, naviguez jusqu'au recueil, puis cliquez sur le lien Sélectionner.

  8. Cliquez sur Ajouter un élément.
