Le privilège permettant de créer des modèles de formulaire implique la possibilité de créer des tables dans la base de données Content Server lors du chargement des modèles. Une table de base de données pour un modèle de formulaire particulier peut stocker les données saisies dans les formulaires basés sur ce modèle. C'est pourquoi il peut être souhaitable de limiter le privilège de création de modèles de formulaire à un petit nombre d'utilisateurs connaissant bien le schéma Content Server. Cela vous permet de maintenir l'intégrité de votre base de données Content Server.

Par défaut, le privilège de création de modèles de formulaire n'est pas limité. Ainsi, tous les utilisateurs ont par défaut la possibilité d'ajouter des modèles à Content Server. La gestion du droit de création de formulaires est semblable à celle des privilèges de création des autres objets. Pour plus d'informations sur la gestion des privilèges relatifs aux objets et à l'utilisation, voir Gestion des privilèges relatifs aux objets et à l`utilisation.
