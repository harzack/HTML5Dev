Si vous disposez des autorisations appropriées, vous pouvez configurer les espaces de travail et les recueils dans Content Server afin de les rendre plus utilisables, instructifs et reconnaissables pour les autres utilisateurs. Vous pouvez configurer des paramètres d'un espace de travail ou d'un recueil sur la page Configurer ou dans l'onglet Présentation de la page Propriétés.

Sur la page Configurer, vous pouvez :

  • Indiquer un mode d'exploration par défaut lorsque le recueil est ouvert.

     

    Note :
    L'administrateur peut désactiver la possibilité pour les utilisateurs de choisir leurs propres vues dans les recueils. Si l'administrateur a désactivé cette option, les éléments ne s'affichent que dans la vue détaillée.

     

  • désigner certains éléments comme Mis en valeur, afin qu'ils apparaissent dans une section Eléments mis en valeur distincte, au-dessus des autres éléments d'une page ; Les éléments qui apparaissent dans la section Eléments mis en valeur sont toujours répertoriés dans l'autre vue détaillée avec les autres éléments non mis en valeur dans le recueil.

  • Désigner certains éléments comme Masqué, afin de masquer les éléments auxquels vous souhaitez que les utilisateurs n'aient pas accès ou dont vous n'avez pas besoin, mais que vous ne souhaitez pas supprimer.

  • Supprimer les éléments dont vous n'avez plus besoin.

Dans l'onglet Présentation de la page Propriétés, vous pouvez :

  • définir le nombre de liens à afficher avec un élément mis en valeur, tel qu'un espace de travail ou un recueil ;

  • ajouter des bannières d'images ;

  • activer un bandeau de nouvelles. Pour en savoir plus, voir Utilisation des chaînes.

 

Note :
L'onglet Présentation de la page Propriétés d'un projet est unique. Pour en savoir plus, voir Personnalisation des espaces de travail Projet.

 

Vous pouvez également établir des personnalisations avancées en créant une apparence ou un affichage personnalisé. Pour plus d'informations sur les apparences, reportez-vous à la rubrique Utilisation des apparences. Pour plus d'informations sur les affichages personnalisés, reportez-vous à Utilisation des affichages personnalisés.
