Pour configurer un espace de travail ou un recueil :

  1. Cliquez sur l'icône Fonctions de l'espace de travail ou du recueil, puis choisissez Configurer.

  2. Tous éléments de l'espace de travail ou du recueil apparaissent dans une liste. Pour chacun de ces éléments, vous pouvez :

    1. Facultatif Dans le champ Nom, saisir un nom pour l'élément.

    2. Sélectionner l'une des options de visualisation suivantes pour déterminer le type d'affichage de l'élément dans l'espace de travail ou le recueil :

      • Liste

      • Mis en valeur

      • Masqué

      • Supprimer

      Pour plus d'informations sur les options de visualisation, voir Personnalisation des espaces de travail et des recueils.

  3. Cliquez sur Mettre à jour.

 

Astuce :
Vous pouvez également configurer chaque élément à partir de la page Propriétés de l'élément en sélectionnant l'onglet Général. Vous pouvez changer le nom dans le champ Nom, et changer les options d'affichage dans le champ Affichage.
