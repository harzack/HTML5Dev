Si la fonction de stockage flexible est activée, vous pouvez sélectionner un autre emplacement de stockage pour les éléments classés spécifiés en configurant les règles du fournisseur de stockage.

Lorsque des objets gérés sont ajoutés à Content Server, ils sont évalués sur la base des règles définies. Vous pouvez créer une expression complexe pour des classements multiples reliés avec les modificateurs Et, Sauf ou Ou. Lors de l'utilisation des règles du fournisseur de stockage, Content Server évalue les éléments classés dans l'ordre croissant, puis stocke les éléments selon les critères que vous avez spécifiés. Pour plus d'informations sur le stockage flexible, reportez-vous à la rubrique Gestion de stockage flexible.
