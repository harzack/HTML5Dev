Pour activer des types d'objets gérés :

  1. Dans un éditeur de texte, ouvrez le fichier <content_server_home>/config/opentext.ini.

  2. Ajoutez la ligne suivante à la section [Classification] :

    SubTypes={136,144,145,0,<n>}, où <n> représente la valeur du type d'objet que vous souhaitez activer (par exemple, 140 correspond à la valeur d'une URL).

  3. Vous pouvez également ajouter les valeurs suivantes au paramètre SubTypes afin d'activer les types d'objet OpenText Enterprise Library dossier, élément et variante :

    823, 824, 825

  4. Enregistrez et fermez le fichier opentext.ini.

  5. Redémarrez le serveur de contenu et le serveur d'application Web.

 

Note :
Les valeurs 136, 144, 145 et 0 représentent les types d'objet géré par défaut (à savoir : les documents composites, les documents, les documents texte et les dossiers).

Les valeurs 823, 824 et 825 représentent les types d'objet OpenText Enterprise Library (à savoir : dossier, élément et variante).
