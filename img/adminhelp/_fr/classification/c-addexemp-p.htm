Pour ajouter un document exemple :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions d'un classement ou d'un Arbre de classement, choisissez Propriétés, puis Profil.

  2. Cliquez sur le bouton Ajouter un document exemple.

  3. Sur la page Sélectionner un document exemple, recherchez le document que vous souhaitez ajouter, puis cliquez sur le lien Sélectionner correspondant.

  4. Répétez les étapes 2 et 3 pour chaque document exemple que vous voulez ajouter.

  5. Cliquez sur le bouton Définir une requête de profil. Pour plus d'instructions sur la définition d'une Requête de profil, reportez-vous à Pour définir la Requête de profil.

 

Note :
La prise en charge de documents exemple de la bibliothèque Enterprise est limitée à la variante par défaut d'éléments.

 
 

Astuce :
Vous pouvez également sélectionner le menu Fonctions d'un document ou d'un élément de bibliothèque Enterprise et choisir Définir comme exemple.
