Pour exporter une taxonomie de classement :

  1. Sur la page Administration Content Server, cliquez sur le lien Exporter la taxonomie dans la section Administration des classements.

  2. Sur la page Exporter la taxonomie, cliquez sur le bouton Parcourir Content Server, puis effectuez l'une des opérations suivantes :

    • Cliquez sur le lien Sélectionner pour que l'espace de travail Classement exporte tous les Arbre de classement dans l'espace de travail Classement.

    • Recherchez un Arbre de classement ou un classement, puis cliquez sur le lien Sélectionner approprié.

  3. Dans le champ Nom du fichier d'exportation, saisissez le nom du fichier XML exporté (arborclass.xml, par exemple).

  4. Cliquez sur le bouton Exporter.

  5. Sur la page Exporter les taxonomies, cliquez sur le bouton OK une fois le processus d'exportation terminé.

 

Note :
Si vous cliquez sur le bouton OK avant que le processus d'exportation ne soit terminé, vous retournez à la page Administration Content Server et le processus d'exportation continue de s'exécuter à l'arrière-plan.
