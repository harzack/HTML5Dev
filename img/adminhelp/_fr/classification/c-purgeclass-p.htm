Pour éliminer de la base de données certains classements rejetés :

  1. Sur la page Administration Content Server, cliquez sur le lien Eliminer les classements rejetés pour les objets dans la section Administration des classements.

  2. Sur la page Eliminer les classements rejetés pour les objets, indiquez dans le champ Nb de jours (depuis la date de création) l'ancienneté des entrées que vous voulez éliminer (l'élimination est effectuée en partant de la date la plus récente).

  3. Cliquez sur le bouton Eliminer.
