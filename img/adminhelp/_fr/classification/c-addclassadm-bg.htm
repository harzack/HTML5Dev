Pour ajouter des éléments à l'espace de travail Classement, vous devez disposer du privilège Création d'objets et des autorisations Voir et Modifier. Pour de plus amples informations sur les autorisations, reportez-vous à la section Autorisations.

Les trois types d'élément suivants peuvent être ajoutés à l'espace de travail Classement :

  • les Arbre de classement, stockés au premier niveau de l'espace de travail Classement et contenant des hiérarchies de classements ;

  • les classements, stockés à l'intérieur d'Arbre de classement ou d'autres classements ;

  • les raccourcis de classement, stockés à l'intérieur d'Arbre de classement et de classements. Il s'agit de raccourcis vers des Arbre de classement ou vers des classements.

Lorsque vous ajoutez un classement, vous devez également définir le type de gestion à attribuer au classement pour les objets gérés, tels que les documents, documents composites, documents texte et dossiers. Il existe trois types de gestion :

  • Manuel : le système n'assigne pas le classement à un élément automatiquement. Seuls les administrateurs et les utilisateurs disposant des autorisations appropriées auront la possibilité d'assigner ce classement.

  • Assisté : le système propose le classement pour les éléments répondant aux critères définis dans le profil du classement et les place sur la page Gérer les objets en attente (Outils > Administration Content Server > Administration des classements). Sur cette page, ils peuvent être acceptés ou rejetés par les administrateurs et utilisateurs Content Server disposant des autorisations appropriées.

  • Automatique : le système assigne automatiquement le classement aux éléments répondant aux critères définis dans le profil du classement.

     

    Note :
    Si vous utilisez OpenText Records Management, le classement RM ne prend pas en charge le type de gestion Assisté dans l'onglet Extras des classements RM. Pour plus d'informations sur la configuration de classements avec Records Management, reportez-vous à OpenText Records Management - Installation and Administration Guide (LLESRCM-IGD).

     

Vous pouvez également définir si vous souhaitez que d'autres utilisateurs puissent sélectionner le classement.

Pour obtenir plus d'informations sur la création manuelle de classements dans Content Server, voir Ajout de classements.
