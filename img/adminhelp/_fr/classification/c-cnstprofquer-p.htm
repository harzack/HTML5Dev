Pour définir la Requête de profil :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions d'un classement ou d'un Arbre de classement, choisissez Propriétés, puis Profil.

  2. Cliquez sur le bouton Définir une requête de profil.

  3. Sur la page Requête de profil de classement, procédez de l'une des façons suivantes :

    • Dans la première zone de texte, saisissez des phrases définies par l'utilisateur que vous voulez utiliser pour définir le classement.

    • Dans la seconde zone de texte, modifiez les phrases clé du document exemple (si vous avez utilisé un ou plusieurs documents exemple et que vous voulez modifier les phrases clé extraites automatiquement).

    • Dans la troisième zone de texte, modifiez les phrases clé de la requête complexe du classement importé (si vous avez importé une requête complexe de classement et que vous voulez modifier les phrases clé extraites automatiquement).

  4. Dans la section Attributs système, spécifiez vos critères de recherche.

  5. Cliquez sur le bouton Enregistrer en tant que profil.

 

Note :
Vous ne pouvez pas ajouter des options avancées à la page Requête de profil de classement (par exemple, Requête en langage naturel ou Portée). Content Server ignore ces critères. Vous pouvez toutefois affiner la Requête de profil en utilisant les liens Catégories, Parcourir Content Server, Types Enterprise Library, Types de propriétés Enterprise Library et Types XML.
