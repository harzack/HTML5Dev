Pour tester une Requête de profil sur des données Enterprise et Enterprise Library :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions d'un classement ou d'un Arbre de classement, choisissez Propriétés, puis Profil.

     

    Note :
    Si la commande Profil n'est pas présente, les types d'objet OpenText Enterprise Library ne sont pas classables. Vous devez les rendre classables. Pour plus d'informations, reportez-vous à la rubrique Objets Enterprise Library Services au sein d'un classement.

     

  2. Cliquez sur le bouton Définir une requête de profil.

    Dans la section Texte intégral de la page Requête de profil de classement, saisissez une phrase pour définir le classement.

  3. Dans la section Texte intégral, modifiez la phrase clé du document exemple.

  4. Dans la section Texte intégral, modifiez la phrase clé de la requête complexe du classement importé.

  5. Dans la section Attributs système, spécifiez, éventuellement, d'autres critères de recherche.

  6. Cliquez sur le bouton Rechercher.
