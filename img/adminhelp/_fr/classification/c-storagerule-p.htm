Pour configurer des règles de stockage :

  1. Sur la page d'administration de Content Server, dans la section Paramètres du fournisseur de stockage, cliquez sur le lien Configurer les règles de stockage.

  2. Sur la page Configurer les règles de stockage, cliquez sur l'icône Modifier.

  3. Sur la page Ajouter une nouvelle règle, cliquez sur Classements '?' assignés à un nœud dans la liste déroulante Règle.

  4. Dans la liste déroulante Classer, cliquez sur l'une des options suivantes :

    • Parcourir les classements

    • Mes favoris

  5. Dans la fenêtre Parcourir les classements ou Mes classements favoris, cliquez sur le lien Sélectionner pour le classement que vous souhaitez ajouter à la règle de stockage. Vous pouvez également cliquer sur le lien Modifier pour ajouter un autre classement en utilisant un des modificateurs.

  6. Dans la liste déroulante Fournisseur de stockage logique, cliquez sur le fournisseur de stockage pour la règle.

  7. Cliquez sur le bouton Envoyer.
