Pour définir un taux de pertinence pour un profil de classement :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions d'un classement ou d'un Arbre de classement, choisissez Propriétés, puis Profil.

  2. Procédez de l'une des manières suivantes :

    • Cliquez sur le bouton radio Utiliser par défaut pour utiliser la valeur par défaut du taux de pertinence des scores.

    • Cliquez sur le bouton radio Utiliser spécifique pour saisir une valeur personnalisée, puis saisissez une valeur entre 1 et 100 dans le champ de saisie.
