Pour gérer les objets en attente :

  1. Sur la page Administration Content Server, cliquez sur le lien Gérer les objets en attente dans la section Administration des classements.

  2. Sur la page Gérer les objets en attente, effectuez l'une des opérations suivantes :

    • Pour accepter ou rejeter le classement proposé pour un objet, cliquez sur Accepter ou sur Rejeter dans les listes déroulantes Statut correspondantes, puis cliquez sur le bouton Envoyer les changements de statut.

    • Pour accepter les classements proposés pour tous les objets, cliquez sur le bouton Définir tous les statuts sur Accepter, puis cliquez sur le bouton Envoyer les changements de statut.

 

Astuce :
Si vous ne voulez ni accepter, ni rejeter le classement proposé pour un objet, vous pouvez laisser l'élément en attente. Lorsque vous cliquez sur le bouton Envoyer les changements de statut, le système affiche la liste des objets en attente qui n'ont pas encore été affichés ou, s'il n'en reste aucun, une liste vide. Pour afficher les objets en attente qui ont déjà été affichés, retournez à la page Administration Content Server et cliquez sur le lien Gérer les objets en attente.
