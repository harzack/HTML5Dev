Pour importer une taxonomie de classement :

  1. Sur la page Administration Content Server, cliquez sur le lien Importer la taxonomie dans la section Administration des classements.

  2. Sur la page Importer la taxonomie, effectuez l'une des opérations suivantes :

    • Pour sélectionner un fichier XML stocké dans le système de fichiers de votre ordinateur, cliquez sur le bouton Parcourir, recherchez le fichier XML contenant l'Arbre de classement à importer et cliquez deux fois sur son nom.

    • Pour sélectionner un fichier XML stocké dans Content Server, cliquez sur le bouton Parcourir Content Server, recherchez le fichier XML contenant l'Arbre de classement à importer et cliquez sur son lien Sélectionner.

  3. Cliquez sur le niveau de messages d'erreur que vous souhaitez afficher dans la liste déroulante Niveau d'alerte :

    • Erreur grave permet de notifier au destinataire les erreurs système critiques.

    • Erreur permet de notifier au destinataire les erreurs système générales.

    • Avertissement permet de notifier au destinataire les erreurs non critiques dans le système.

    • Informations permet de notifier au destinataire le statut général et d'autres informations système.

    • Aucun est l'option par défaut.

    Pour plus d'informations sur la transmission des messages d'erreur, reportez-vous à la rubrique Activation de la recherche d`erreurs et des e-mails d`alerte.

  4. Cliquez sur le bouton Modifier dans le champ Catégories si vous souhaitez définir ou modifier les catégories à appliquer à l'Arbre de classement et à ses classements.

  5. Cliquez sur le bouton Importer.

  6. Sur la page Importer les taxonomies, cliquez sur le bouton OK une fois le processus d'importation est terminé.

 

Note :
Si vous avez appliqué des catégories au volume de classements, ces catégories apparaissent par défaut dans le champ Catégories.
