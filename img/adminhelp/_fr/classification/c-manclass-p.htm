Pour gérer un classement :

  1. Dans l'espace de travail Classement, cliquez sur l'icône Fonctions de l'Arbre de classement ou du classement que vous souhaitez gérer et sélectionnez Gérer le classement.

  2. Sur la page Gérer l'Arbre de classement ou Gérer le classement, procédez comme suit :

    • Pour supprimer des éléments du classement, cliquez sur Supprimer dans les listes déroulantes Action correspondantes. Ensuite, cliquez sur le bouton Envoyer. Vous pouvez également cliquer sur le bouton Définir toutes les actions sur Dissocier. Ensuite, cliquez sur le bouton Envoyer.

    • Pour reclasser des éléments dans ce classement, cliquez sur Reclasser dans les listes déroulantes Action correspondantes, puis cliquez sur le bouton Envoyer. Dans la fenêtre Parcourir les classements, recherchez ensuite le classement que vous souhaitez assigner à ces éléments. puis cliquez sur le lien Sélectionner. Vous pouvez également cliquer sur le bouton Définir toutes les actions sur Reclasser. Ensuite, cliquez sur le bouton Envoyer.
