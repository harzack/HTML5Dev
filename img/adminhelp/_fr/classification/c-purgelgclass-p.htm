Pour éliminer des classements :

  1. Sur la page Administration Content Server, cliquez sur le lien Eliminer les classements dans la section Administration des classements.

  2. Sur la page Eliminer les classements, cliquez sur le bouton Parcourir Content Server, recherchez la hiérarchie de classements que vous souhaitez éliminer, puis cliquez sur le lien Sélectionner approprié.

  3. Cliquez sur le bouton Eliminer.

  4. Sur la page Eliminer les classements : Réussite, cliquez sur le lien Continuer.

 

Note :
Content Server ne peut pas éliminer les hiérarchies de classement qui contiennent certains types d'objets (par exemple, des objets de type Records Management). Vous devez supprimer ces types d'objet manuellement avant de procéder à l'élimination.
