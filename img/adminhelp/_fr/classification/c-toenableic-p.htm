Pour activer la fonctionnalité de Classement intelligent :

  1. Sur la page Administration Content Server, cliquez sur le lien Activer la fonctionnalité de classement intelligent.

  2. Sur la page Activer la fonctionnalité de classement intelligent, cochez la case Activer le classement intelligent.

  3. Cliquez sur le bouton Enregistrer les modifications.
