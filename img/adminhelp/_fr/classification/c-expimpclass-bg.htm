Vous exportez et importez des taxonomies de classement lorsque vous souhaitez copier une hiérarchie de classements (arbres de classement ou classements et tous les classements qu'ils contiennent) d'une installation Content Server vers une autre, ou au sein d'une même installation Content Server. Supposons, par exemple, que vous ayez défini un Arbre de classement nommé Documents de marketing, contenant de nombreux classements, et que vous souhaitiez créer une hiérarchie de classements identique sur une autre installation Content Server ; il vous suffit d'exporter et d'importer l'Arbre de classement.

Vous pouvez procéder de la sorte pour un espace de travail Classement dans sa totalité, un Arbre de classement ou un classement. Lorsque vous exportez un espace de travail Classement, tous les Arbre de classement et classements contenus dans cet espace de travail sont exportés. Lorsque vous exportez un classement, celui-ci devient un Arbre de classement. Supposons, par exemple, que vous ayez un Arbre de classement nommé Développement produits, dans lequel figure un classement nommé Plannings de projet. Vous pouvez exporter le classement Plannings de projet et l'importer dans une autre installation Content Server. Dans ce cas, le classement Plannings de projet est importé sous forme d'Arbre de classement.

Si vous souhaitez élaborer une hiérarchie de classements, sans avoir à ajouter chaque classement à l'espace de travail Classement individuellement, vous pouvez créer un fichier XML définissant une hiérarchie de classements (une taxonomie de classement XML), puis l'importer. Pour obtenir plus d'informations sur la création d'une taxonomie XML, reportez-vous à la rubrique Construction et importation d'une taxonomie XML.

 

Note :
Si vous n'avez pas activé la fonctionnalité Classement intelligent, lorsque vous importez des arbres de classement, le type de gestion est défini sur « manuel » et tous les profils associés sont désactivés. Pour plus d'informations sur l'activation de Classement intelligent, reportez-vous à la rubrique Activation de la fonctionnalité de Classement intelligent.

 

Exportation de taxonomies de classement

Lorsque vous exportez un espace de travail Classement ou une partie de son contenu, Content Server écrit les informations dans un fichier XML. Le fichier XML contient toutes les informations définissant les Arbre de classement et les classements, ainsi que la hiérarchie de classements qu'ils peuvent contenir (par exemple, les noms et les types de gestion de chaque classement). Le système stocke ensuite ce fichier XML dans le répertoire temp du répertoire où est installée l'application Content Server (par exemple, <content_server_home>/temp), puis il le transfère vers l'espace de travail de l'utilisateur qui effectue l'exportation.

 

Note :
Selon les activités en cours sur le système, il est possible que le fichier XML contenant les Arbre de classement et les classements exportés n'apparaisse pas immédiatement dans Mon espace de travail.

 

Importation de taxonomies de classement

Lorsque vous importez une taxonomie de classement dans l'espace de travail Classement, Content Server lit dans le fichier XML correspondant les informations qui définissent le ou les Arbre de classement et la hiérarchie de classements qu'ils contiennent. Lorsque vous lancez le processus d'importation, le système construit cette hiérarchie de classements dans l'espace de travail Classement.

Lorsque vous importez une taxonomie de classement, vous pouvez également appliquer des catégories. Les catégories s'appliquent à tous les nouveaux Arbre de classement et aux classements créés lors du processus d'importation. Par défaut, lorsque vous spécifiez les paramètres d'importation, toutes les catégories appliquées au volume de classements apparaissent dans le champ Catégories.

Vous pouvez supprimer ces catégories, en ajouter de nouvelles ou modifier les catégories à appliquer aux classements importés. Pour plus d'informations sur les catégories, reportez-vous à la rubrique Utilisation des catégories et des attributs.

Une fois l'Arbre de classement importé, activez ses classements assistés et automatiques en régénérant le fichier de requête de classement. Pour en savoir plus, voir Maintenance des classements.

 

Note :
Selon les activités en cours sur le système, il est possible que l'Arbre de classement et les classements importés n'apparaissent pas immédiatement dans l'espace de travail Classement. Lorsque vous importez un Arbre de classement comportant un grand nombre de classements (nombre supérieur à 1 000) et que l'importation échoue, vous pouvez éliminer l'Arbre de classement. Ce processus d'élimination, tout comme la dissociation, permet de supprimer tous les classements qui ont été importés, mais est plus rapide. Pour en savoir plus, voir Pour éliminer des classements.
