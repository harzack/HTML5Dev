Pour configurer des objets gérés :

  1. Sur la page Administration Content Server, cliquez sur le lien Objets gérés dans la section Administration des classements.

  2. Effacez ou sélectionnez la case à cocher Gérés en regard de chacun des types d'objet suivants :

    • Document composite

    • Document

    • Dossier Enterprise Library

    • Elément Enterprise Library

    • Définition de type d'objet Enterprise Library

    • Variante Enterprise Library

    • Dossier

    • Document de texte

  3. Cliquez sur le bouton Mettre à jour.
