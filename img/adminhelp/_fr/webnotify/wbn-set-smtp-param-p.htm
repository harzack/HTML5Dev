Pour définir les paramètres SMTP :

  1. Sur la page Administration Content Server, dans la section Administration des notifications, cliquez sur Configurer la notification.

  2. Dans le champ ID du serveur SMTP de la zone Paramètres SMTP, saisissez le nom du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est mail.

  3. Dans le champ Port SMTP, saisissez le port de réception du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est 25.

  4. Dans le champ Nom de l'hôte Content Server, saisissez le nom DNS complet de l'hôte principal Content Server. Par exemple, contentserverhost.mycorp.com.

  5. Cliquez sur Envoyer.

     

    Note :
    Si vous configurez pour la première fois les notifications, vous devez également activer les notifications et configurer les paramètres d'e-mail et les paramètres de rapport. Pour plus d'informations, voir les rubriques Pour activer des notifications, Pour définir les paramètres d'e-mail et Pour configurer les paramètres de rapport.
