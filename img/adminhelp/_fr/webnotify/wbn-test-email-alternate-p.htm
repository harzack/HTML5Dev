Pour tester l'envoi d'e-mails aux autres adresses e-mail :

  1. Sur la page Administration Content Server, dans la section Administration des notifications, cliquez sur Tester l'envoi par e-mail.

  2. Dans le champ Serveur SMTP, saisissez le nom du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est mail.

  3. Dans le champ Port SMTP, saisissez le port de réception du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est 25.

  4. Dans le champ De, saisissez l'adresse e-mail de l'expéditeur qui apparaîtra dans les e-mails de test.

  5. Dans le champ Réponse, saisissez une adresse e-mail.

  6. Dans le champ Destinataire, saisissez l'adresse e-mail à laquelle vous souhaitez envoyer le message de test.

  7. Vous pouvez également saisir un objet dans le champ Objet.

  8. Vous pouvez également saisir le texte du message dans le champ Corps du texte

  9. Cliquez sur Envoyer un message de test. La colonne Statut indique si le message a abouti ou non et la colonne Journal consigne les étapes impliquées dans chaque test de message.
