Pour configurer les paramètres de rapport de notification :

  1. Sur la page Administration Content Server, dans la section Administration des notifications, cliquez sur Configurer la notification.

  2. Dans la section Paramètres du rapport de la page Configurer la notification, saisissez un entier dans le champ Nombre maximal d'événements par transmission. Cet entier représente le nombre maximal d'événements couvert à chaque exécution de la notification.

  3. Cliquez sur le type de rapport par défaut souhaité dans la liste Type de rapport par défaut. Le seul type de rapport disponible actuellement est Rapport de notifications par défaut.

  4. Cliquez sur l'une des options d'affichage suivantes dans la liste Options de rapport par défaut :

    • Autre fenêtre pour afficher la case à cocher Ouvrir une fenêtre de consultation HTML lors de la sélection de liens sur la page Paramètres de rapport de notification.

    • <Aucun> pour interdire l'option Autre fenêtre aux utilisateurs.

  5. Cliquez sur la période au terme de laquelle Content Server doit effacer les messages de notification dans la liste Vider les messages antérieurs.

  6. Cliquez sur la période au terme de laquelle Content Server doit effacer les activités programmées dans la liste Vider les événements en suspens.

  7. Dans chacune des trois zones Planification des notifications par défaut, procédez comme suit :

    1. Si vous le souhaitez, définissez un nouveau nom pour l'étape dans le champ Nom.

    2. Activez les cases à cocher correspondant aux jours d'envoi des messages de notification dans la section Les jours suivants.

    3. Activez les cases à cocher correspondant aux heures d'envoi des messages de notification pour les jours sélectionnés dans la section Aux heures suivantes.

    4. Activez les cases à cocher correspondant aux minutes d'envoi des messages de notification pour chaque heure sélectionnée dans la section Aux minutes suivantes.

  8. Cliquez sur Envoyer.

     

    Note :
    Si vous configurez pour la première fois la notification, vous devez également activer la notification et configurer SMTP et les paramètres de l'e-mail. Pour plus d'informations, voir les rubriques Pour activer des notifications, Pour définir les paramètres SMTP : et Pour définir les paramètres d'e-mail :.
