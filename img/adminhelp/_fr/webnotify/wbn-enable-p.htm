Pour activer des notifications :

  1. Dans la section Administration des notifications de la page Administration Content Server, cliquez sur Configurer la notification.

  2. Dans la section Activer les notifications de la page Configurer la notification, cliquez sur le bouton radio Activer.

  3. Cliquez sur Envoyer.

  4. Redémarrez le serveur et le serveur d'application Web. Pour plus d'informations sur le redémarrage des serveurs, reportez-vous à la rubrique Arrêt et démarrage des serveurs.

     

    Note :
    Si vous configurez pour la première fois les notifications, vous devez également définir les paramètres SMTP, de rapport et d'e-mail. Pour plus d'informations, voir les rubriques Pour définir les paramètres SMTP :, Pour configurer les paramètres de rapport et Pour définir les paramètres d'e-mail :.
