Vous pouvez effacer les événements de notification programmés sur la page Vider les événements en suspens. Vous pouvez activer ou désactiver des événements sur la page Configuration des activités programmées. Vous pouvez également définir la fréquence à laquelle Content Server efface ces événements dans la zone Paramètres du rapport de la page Configurer la notification.
