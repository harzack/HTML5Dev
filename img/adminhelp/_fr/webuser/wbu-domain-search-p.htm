Pour configurer la recherche de domaines Content Server :

  1. S'il n'existe pas, créez le groupe interdomaine Tous les domaines.

  2. Cliquez sur le lien Ouvrir le volume des objets système de la section Administration de recherche sur la page Administration.

  3. Cliquez sur l'icône Fonctions du dossier Système Content Server, puis sélectionnez Autorisations.

  4. Dans la section Accès assigné, cliquez sur l'icône Accorder l'accès.

  5. Localisez le groupe interdomaine, cochez la case Accorder l'accès, puis cliquez sur le bouton Envoyer.

  6. Cliquez sur le nom du groupe interdomaine, accordez à ce groupe les autorisations Voir et Voir le contenu. Appliquez ces autorisations à l'élément actuel et à tous ses sous-éléments, puis cliquez sur le bouton Mettre à jour.

  7. Pour limiter les autorisations relatives aux vues de recherche, allez dans le dossier Vues de recherche et ajoutez le domaine approprié à l'Accès assigné pour une vue particulière, puis définissez les autorisations pour ce domaine.

  8. Pour limiter les autorisations relatives aux modèles de recherche, allez dans le dossier Modèles de recherche personnels et accordez au groupe interdomaine l'autorisation d'ajouter des éléments.
