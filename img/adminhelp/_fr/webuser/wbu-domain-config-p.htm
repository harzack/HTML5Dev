Pour configurer des domaines :

  1. Cliquez sur le lien Configurer le domaine de la section Administration d'utilisateurs et de groupes sur la page Administration.

  2. Effectuez l'une des actions suivantes :

    • Pour activer les domaines, activez la case à cocher Activer la prise en charge de domaines.

    • Pour créer le groupe interdomaine Tous les domaines, sélectionnez la case à cocher Créer, puis saisissez un nom pour le groupe Tous les domaines dans le champ Nom de groupe.

    • Pour spécifier le nom d'affichage du domaine, entrez un nom dans le champ Nom d'affichage du domaine.

    • Pour spécifier le nom du domaine système, entrez un nom dans le champ Nom de domaine du système.

  3. Cliquez sur le bouton Envoyer.
