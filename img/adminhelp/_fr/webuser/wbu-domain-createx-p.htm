Pour créer un groupe interdomaine

  1. Choisissez Utilisateurs et groupes dans le menu Entreprise.

  2. Cliquez sur Groupe interdomaine dans le menu Ajouter un élément.

  3. Saisissez un nom dans le champ Nom de groupe interdomaine.

  4. Cliquez sur le bouton Envoyer.

    La page Modifier le groupe s'affiche, vous permettant d'ajouter des membres au groupe, de le renommer ou de le supprimer.
