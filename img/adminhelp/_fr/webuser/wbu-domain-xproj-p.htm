Pour créer un projet interdomaine :

  1. Créez un projet ou ouvrez un projet existant.

  2. Choisissez Intervenants dans le menu Projet.

  3. Cliquez sur le bouton Ajouter des intervenants.

  4. Recherchez les intervenants que vous voulez ajouter.

  5. Pour chaque intervenant, cliquez sur Coordinateur, Membre ou Invité dans la liste déroulante Rôle.

  6. Cliquez sur le bouton Envoyer.

  7. Cliquez sur le bouton Terminé.
