Lorsqu'une étape de XML Workflow Interchange envoie un message à un système externe, il est possible d'accéder à celui-ci directement via Internet ou par le biais d'un proxy HTTP. En configurant un proxy HTTP externe, vous pouvez offrir un niveau de sécurité supplémentaire entre votre ordinateur et Internet si le proxy HTTP se trouve derrière un pare-feu.
