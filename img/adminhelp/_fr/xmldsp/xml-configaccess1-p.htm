1. Sur la page Administration Content Server de la section Administration de messagerie HTTP, cliquez sur le lien Configurer l'accès HTTP externe.

  2. Sur la page Configurer l'accès HTTP externe, saisissez l'URL complète que les systèmes externes peuvent utiliser pour accéder à l'étape de XML Workflow Interchange dans la zone URL XMLDSP.

  3. Cliquez sur Enregistrer les modifications.

     

    Astuce :
    si vous avez configuré votre serveur HTTP de façon à autoriser l'accès HTTP à l'aide d'un numéro de port non standard, vous pouvez ajouter le numéro de port après le nom de l'hôte. Séparez le nom de l'hôte et le numéro de port par un point-virgule.

     

Pour plus d'informations sur cette procédure, consultez la section Configuration de l'accès HTTP externe.
