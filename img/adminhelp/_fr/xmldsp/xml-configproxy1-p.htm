1. Sur la page Administration Content Server de la section Administration de messagerie HTTP, cliquez sur le lien Configurer le proxy HTTP.

  2. Sur la page Spécifier le proxy HTTP, dans la zone Adresse, saisissez l'adresse IP que l'étape de XML Workflow Interchange va utiliser pour accéder au proxy HTTP. Exemple : 127.0.0.1. Votre administrateur système vous fournira cette adresse IP.

  3. Dans la zone Port, tapez le numéro de port de l'URL.

  4. Cliquez sur Enregistrer les modifications.

Pour plus d'informations sur cette procédure, consultez la section Configuration du proxy HTTP.
