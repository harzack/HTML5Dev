La page Configurer les emplacements exclus vous permet de spécifier les dossiers ou éléments Content Server que vous souhaitez exclure de l'extraction. Si l'élément Content Server est un type de recueil, comme un dossier, le contenu du recueil est également exclu de l'extraction. Si vous excluez un emplacement après indexation des objets, ceux-ci resteront dans l'index de recherche jusqu'à ce que vous procédiez à une élimination et à une réindexation de la base de données.

Si l'élément Content Server est un recueil, comme un dossier, le contenu du recueil est également exclu de l'extraction. Cependant, si vous excluez un emplacement déjà indexé, l'élément reste dans l'index jusqu'à ce que vous procédiez à une élimination et à une réindexation.

Pour configurer des emplacements exclus :

  1. Sur Pour définir les paramètres généraux de l'Extracteur, dans le champ Emplacements exclus de la section Paramètres d'exclusion, cliquez sur le lien liste de modification/révision.

  2. Sur la page Configurer les emplacements exclus, cliquez sur le bouton Parcourir Content Server pour spécifier les emplacements que l'Extracteur doit exclure des opérations d'indexation de recherche.

  3. Si nécessaire, cliquez sur le bouton Ajouter un emplacement , ou le lien, pour ajouter d'autres emplacements. Cliquez sur le bouton Supprimer un emplacement pour supprimer les emplacements inutiles.

  4. Cliquez sur Mettre à jour pour utiliser les emplacements exclus que vous avez définis, ou sur Annuler.
