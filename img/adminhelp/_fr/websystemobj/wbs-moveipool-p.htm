Pour déplacer un IPool :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données qui contient l'IPool à déplacer.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur l'icône Fonctions du processus producteur du flux de données, puis sélectionnez Arrêter. Les processus Extracteur Entreprise, Directory Walker et Producteur XML Activator sont tous des processus producteurs.

  4. Cliquez sur un intervalle d'actualisation dans la liste déroulante Actualiser. Une fois les données entièrement traitées dans le flux de données, cliquez sur Ne pas actualiser dans la liste déroulante Actualiser.

  5. Pour chaque processus de flux de données en cours d'exécution ou programmé pour être exécuté, cliquez sur l'icône Fonctions, puis sélectionnez Arrêter.

  6. Cliquez sur le lien du nom du processus chargé d'écrire les données dans l'IPool à déplacer.

  7. Dans l'onglet Infos spécifiques de la page Propriétés du processus, cliquez sur le bouton Informations.

  8. Sur la page Informations iPool, saisissez le chemin du nouveau répertoire d'écriture iPool tel qu'il est mappé/monté sur l'hôte du serveur d'administration sur lequel ce processus est exécuté dans le champ Ecrire.

  9. Cliquez sur le bouton Mettre à jour.

  10. Cliquez sur le nom du processus qui lit les données à partir du répertoire IPool que vous avez déplacé.

  11. Dans l'onglet Infos spécifiques de la page Propriétés du processus, cliquez sur le bouton Informations.

  12. Sur la page Informations iPool, saisissez le chemin du nouveau répertoire de lecture iPool tel qu'il est mappé/monté sur l'hôte du serveur d'administration sur lequel ce processus est exécuté dans le champ Lire.

  13. Cliquez sur le bouton Mettre à jour.

  14. Cliquez sur l'icône Fonctions du flux de données, puis sélectionnez Reprendre.
