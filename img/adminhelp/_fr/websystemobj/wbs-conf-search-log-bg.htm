Si vous voulez enregistrer l'activité de recherche dans votre système Content Server, vous pouvez définir un journal de recherche. Les enregistrements du journal de recherche résultent des recherches dans les requêtes et les résultats. Vous pouvez configurer Content Server pour enregistrer les journaux de recherche dans l'un des fichiers suivants ou dans les deux :

  • Content Server_home/logs/threadn.out, où n représente un numéro de thread. Pour enregistrer les journaux de recherche dans les fichiers threadn.out, vous devez également configurer la journalisation du serveur. Pour en savoir plus, voir Configuration du journal du serveur.

  • Content Server_home/logs/search.log
