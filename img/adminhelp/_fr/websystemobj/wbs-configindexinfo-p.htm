Pour configurer les informations du moteur d'indexation :

  1. Dans le champ Répertoire de lecture, saisissez le chemin absolu du répertoire à partir duquel vous voulez que le processus du moteur d'indexation lise les données.

  2. Dans le champ Répertoire d'indexation, saisissez le chemin absolu du répertoire dans lequel vous voulez que le processus du moteur d'indexation crée l'index. Vous devez choisir un répertoire d'indexation sur un lecteur local de l'hôte du serveur d'administration qui exécute ce processus.

  3. Dans la liste déroulante Hôte de la section Client : Index Content Server, cliquez sur le raccourci du serveur d'administration sur l'hôte à partir duquel le processus du moteur d'indexation doit s'exécuter.
