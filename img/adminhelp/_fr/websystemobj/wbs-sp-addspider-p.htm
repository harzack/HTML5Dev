Vous devez créer un flux de données Spider, décrit dans Pour ajouter un nouveau flux de données Spider, avant d'ajouter un processus Spider.

Pour ajouter un Spider :

  1. Sur la page Volume des objets système, cliquez sur le lien Dossier de la source de données Spider.

  2. Sur la page Dossier de la source de données Spider, cliquez sur le lien Gestionnaire de flux de données Spider.

  3. Sur la page Gestionnaire de flux de données Spider, dans le menu Ajouter un élément, cliquez sur Spider.

  4. Sur la page Ajouter : Spider, dans le champ Nom, saisissez la chaîne de caractères que vous voulez utiliser comme identificateur pour le Spider.

  5. Dans le champ Description, saisissez le texte donnant une description du Spider sur l'onglet Général de la page Propriétés.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte à partir duquel le Spider doit s'exécuter.

  7. Dans la zone Serveurs Spider, saisissez :

    • Le nom de l'Hôte de chaque module de serveur Spider installé sur votre système Content Server. Si nécessaire, sous Action, cliquez sur bouton Dupliquer pour ajouter d'autres serveurs Spider.

    • Le numéro de Port du serveur Spider.

    • Le Mot de passe pour accéder au serveur Spider, si défini.

    • Les Caractéristiques du projet pour cette source de données à indexer. Cliquez sur le lien Création de la liste des projets pour afficher une liste des projets du serveur Spider, ainsi que leurs numéros de port, définis sur l'Hôte, puis cliquez sur les projets à inclure. Vous pouvez également saisir les caractéristiques directement, à l'aide du format nom du projet, nom du port, séparé par ; (point-virgule). Si nécessaire, vous pouvez également mettre les projets en surbrillance puis les supprimer.

    • Sous Action, cliquez sur le bouton Gérer le serveur Spider pour modifier les paramètres du serveur Spider, y compris les URL de site Web et les caractéristiques de navigation.

    • Si nécessaire, cliquez sur le bouton Supprimer pour supprimer les définitions du serveur Spider inutiles.

  8. Dans la section Infos sur Interchange Pool, dans le champ Ecrire les données dans, sous Zone principale IPool, saisissez le chemin absolu du répertoire des données Interchange Pool (IPool) dans lequel le Spider devra écrire les données de l'index.

  9. Si nécessaire, dans la liste déroulante Processus, sélectionnez le processus qui doit suivre le Spider dans le flux de données.

  10. Dans la section Options de lancement, à partir de la liste déroulante Programme, cliquez sur l'une des options suivantes :

    • Manuel, option par défaut permettant de lancer manuellement le processus Spider à chaque fois que vous souhaitez parcourir à nouveau les répertoires spécifiés.

    • Programmé, option permettant de programmer le jour et l'heure d'exécution du processus Spider. Si les répertoires parcourus par les projets du serveur Spider changent fréquemment, vous pouvez programmer le jour et l'heure ou l'intervalle auxquels le processus Spider devra extraire régulièrement les données du serveur Spider.

  11. Cliquez sur le bouton radio qui représente le moment auquel vous souhaitez exécuter le processus Spider :

    • A cette heure : cette option permet de programmer l'exécution du processus Spider à une heure spécifique certains jours de la semaine. Cliquez sur les valeurs dans les listes déroulantes Heure, puis sélectionnez les cases à cocher appropriées pour indiquer les jours de la semaine d'exécution du processus Spider.

    • Chaque : cette option permet de programmer l'exécution du processus Spider à un intervalle régulier. Cliquez sur les valeurs des listes déroulantes pour indiquer les horaires et la durée de l'intervalle d'exécution du processus Spider.

  12. Dans la section Options d'arrêt, saisissez une Valeur maximale du code de sortie correcte pour spécifier le numéro de code d'erreur maximum pour lequel le serveur d'administration redémarrera automatiquement le Spider en cas d'échec. Ne modifiez pas la valeur maximale du code de sortie correcte, sauf si le service d'assistance clientèle d'OpenText vous le demande.

  13. Cliquez sur le bouton Ajouter.
