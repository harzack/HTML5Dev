Pour resynchroniser un processus de flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le processus à resynchroniser.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur le nom du processus à resynchroniser.

  4. Dans l'onglet Infos spécifiques de la page Propriétés du processus, cliquez sur le bouton Resynchroniser.
