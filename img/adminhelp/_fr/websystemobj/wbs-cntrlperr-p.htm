Pour ajouter une règle de contrôle basée sur des erreurs de processus :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur Ouvrir le volume des objets système.

  2. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du préfixe_processus Dossier de la source de données. Choisissez Propriétés, puis Règles de contrôle.

  3. Sur la page Règles de contrôle, cliquez sur le bouton Ajouter une nouvelle règle de contrôle .

  4. Sur la page Ajouter une nouvelle règle de contrôle, saisissez un nom pour la règle dans le champ Nom.

  5. Dans la section Modèles, dans la liste déroulante, cliquez sur Erreur du processus dans le flux de données. Vous pouvez modifier la description de la règle dans le champ Description.

  6. Dans la section Conditions, cliquez sur Erreur du processus de flux de données dans la liste déroulante Type de condition. Dans la section Rapports, vous pouvez entrer un message à envoyer avec le rapport dans le champ Message.

  7. Cliquez sur le bouton Mettre à jour.
