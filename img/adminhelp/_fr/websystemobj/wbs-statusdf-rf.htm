La page Statut s'affiche une fois un flux de données d'indexation créé. Cette page fournit la liste des objets qui ont été créés et affiche les numéros de port associés aux processus de flux de données. Vous pouvez consulter ces informations sur la page Statut afin de vous assurer que tous vos processus de flux de données ont été créés avec succès.

Pour revenir à la page Volume des objets système, cliquez sur le bouton Continuer.

Si vous créez l'index Entreprise dans le cadre de la création d'une nouvelle base de données Content Server, la page Félicitations s'affiche.
