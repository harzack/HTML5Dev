Pour contrôler l'accès IP aux composants de recherche

  1. Sur la page d'administration, cliquez sur le lien Contrôler l'accès IP pour la recherche de composants.

  2. Sur la page Contrôler l'accès IP aux composants de recherche, saisissez une adresse IP dans le champ situé en regard du bouton Ajouter une adresse IP, puis cliquez sur le bouton Ajouter une adresse IP.

    Répétez l'étape 2 pour chaque adresse IP que vous souhaitez ajouter.

  3. Cliquez sur le bouton OK.

  4. Redémarrez le serveur d'administration.

 

Avertissement
Si vous n'ajoutez pas d'adresse IP pour tous les ordinateurs qui hébergent les serveurs et les serveurs d'administration, le système de recherche risque de ne pas répondre.

 
 

Astuce :
Pour interdire l'accès au réseau d'un ordinateur, cliquez sur son adresse IP dans la liste située en regard du bouton Supprimer, cliquez sur le bouton Supprimer, puis sur OK et redémarrez le serveur d'administration.
