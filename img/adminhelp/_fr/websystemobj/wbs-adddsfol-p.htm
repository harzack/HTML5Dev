Pour ajouter un dossier source de données :

  1. Sur la page Volume des objets système, cliquez sur Dossier de la source de données dans le menu Ajouter un élément.

  2. Sur la page Ajouter Dossier sources de données, saisissez un nom pour le dossier source de données dans le champ Nom.

  3. Pour fournir une description du dossier de la source de données dans l'onglet Général de sa page Propriétés, saisissez un texte descriptif dans le champ Description.

  4. Pour modifier les catégories et attributs associés à cet élément, cliquez sur le bouton Modifier en regard du champ Catégories.

  5. Cliquez sur le bouton Ajouter.
