Pour configurer une source de données Directory Walker :

  1. Sur la page Volume des objets système, cliquez sur Source de données Directory Walker dans le menu Ajouter un élément.

  2. Si vous avez supprimé une source de données Content Server et que vous souhaitez associer ses requêtes et ses formulaires de recherche enregistrés à cette source de données Directory Walker, choisissez le nom de la vue de recherche de la source de données supprimée dans la liste déroulante Remplacement de vue de recherche.

  3. Dans le champ Préfixe du processus, saisissez un identificateur unique pour tous les objets associés à ce flux de données d'indexation. Cet identificateur correspond au nom d'affichage des objets associés à cet index sur la page Administration système et au nom d'affichage de la vue de recherche de l'index dans la liste Vues de recherche sur la page Recherche.

  4. Dans le champ Partitions, saisissez le nombre de partitions à créer pour la source de données Directory Walker.

  5. Dans le champ Port, saisissez une valeur représentant la série des numéros des ports sur lesquels vous souhaitez que les processus associés à cette source de données soient en réception. Le numéro de port que vous spécifiez et au moins les onze numéros de port suivants ne doivent pas être utilisés par une autre source de données de votre système. Le nombre de numéros de port consécutifs utilisés dépend du nombre de partitions que vous indiquez dans le champ Partitions. La configuration d'une source de données Directory Walker nécessite huit numéros de port et, pour chaque partition, quatre numéros de port supplémentaires. Les valeurs valables vont de 1025 à 65535.
