1. Sur la page Volume des objets système, cliquez sur le lien <préfixe> Dossier de la source de données correspondant au dossier de la source de données qui contient le gestionnaire de flux de données dans lequel vous souhaitez ajouter le processus T.

  2. Cliquez sur le lien <préfixe> Gestionnaire de flux de données.

  3. Cliquez sur le lien Processus T pour le processus que vous souhaitez modifier.

  4. Sur la page Propriétés de T, modifiez les paramètres du processus T afin de refléter toute modification que vous voulez faire.

  5. Dans la section iPools, cliquez sur le lien Plus de détails pour modifier les processus iPool utilisés dans le processus T.

  6. Vous pouvez également cliquer sur Démarrer pour démarrer le processus.

  7. Vous pouvez également cliquer sur Resynchroniser pour réinitialiser la valeur du champ Modèle de commande.

  8. Cliquez sur Mettre à jour.

 

Astuce :
Pour plus d'informations sur les champs de la page Propriétés T, voir Configuration d'un processus T.
