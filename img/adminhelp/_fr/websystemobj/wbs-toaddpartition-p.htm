Pour ajouter une partition :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données à laquelle vous souhaitez ajouter une partition.

  2. Cliquez sur le lien préfixe_processus Schéma de partition.

  3. Cliquez sur Partition dans le menu Ajouter un élément.

  4. A l'étape de la page Planification de l'assistant d'ajout de partition, consultez les informations affichées, puis cliquez sur le bouton Suivant(e).

     

    Note :
    Vous devez disposer de toutes les informations requises telles que les valeurs et la taille de l'index, le nom de la partition, l'emplacement de l'index et les quatre ports inutilisés avant de lancer l'assistant. Commencez par établir votre planification pour déterminer une configuration appropriée.

    Votre serveur nécessite une quantité de mémoire vive supplémentaire pour prendre en charge la nouvelle partition de l'index. Pour une partition standard avec une taille maximale de contenu de disque de 50 000 Mo et une taille maximale de mémoire des métadonnées de 1 000 Mo, vous devez fournir une quantité de mémoire vive supplémentaire de 2,4 Go avant de créer la nouvelle partition.

    Tout port spécifié doit être libre et inutilisé. Vous pouvez utiliser les liens Vérifier le port dans l'assistant pour vous en assurer. Consultez votre administrateur réseau si vous n'êtes pas certain des ports à utiliser. Il n'est pas nécessaire de spécifier les ports pour vos serveurs d'administration ou vos processus de serveur existants. Spécifiez uniquement les ports requis pour la nouvelle partition.

    Vous devez spécifier le chemin d'accès exact de votre nouvel index en tant qu'emplacement de l'index, par exemple C:\opentext\index\enterprise\index2. Le dernier sous-dossier (index2 dans l'exemple) ne doit pas encore exister. Il sera créé pour vous. N'utilisez pas le dossier à la racine de l'index (généralement C:\opentext\index ou C:\opentext\index\enterprise). Il est important de spécifier le dossier avec attention. L'assistant ne suggère pas de dossier approprié et ne vous empêche pas d'en utiliser un inapproprié.

     

  5. A l'étape Partition, procédez de la façon suivante :

    • Saisissez un nom pour la partition dans le champ Nom.

    • Pour personnaliser la quantité d'espace disque à allouer au contenu indexé dans l'index de la partition, saisissez une valeur dans le champ Taille maximale du contenu du disque.

    • Pour personnaliser la quantité d'espace mémoire à allouer aux métadonnées indexées dans l'index de la partition, saisissez une valeur dans le champ Taille maximale de la mémoire des métadonnées.

  6. Cliquez sur le bouton Suivant.

  7. A l'étape Moteur d'indexation, procédez de la façon suivante :

    • Dans le champ Nom, saisissez un nom pour le moteur d'indexation.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le moteur d'indexation.

    • Saisissez un numéro de port inutilisé dans le champ Port Admin pour spécifier le port sur lequel le moteur d'indexation sera exécuté sur un serveur d'administration. Cliquez sur le lien Vérifier le port pour vérifier si un numéro de port est disponible.

    • Saisissez un numéro de port inutilisé dans le champ Port du serveur pour spécifier le port que les processus d'échange utiliseront pour communiquer avec le moteur d'indexation.

    • Saisissez le chemin absolu pour un nouvel emplacement de l'index dans le champ Répertoire d'index de l'index de la partition.

  8. Cliquez sur le bouton Suivant.

  9. A l'étape Moteurs de recherche, fournissez les informations permettant d'ajouter un moteur de recherche à chaque fédérateur de recherche de votre système d'indexation et de recherche. Saisissez les informations suivantes dans chaque section de la page :

    • Dans le champ Nom, saisissez un nom pour le moteur de recherche.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le moteur de recherche.

    • Saisissez un numéro de port inutilisé dans le champ Port Admin pour spécifier le port sur lequel le moteur d'indexation sera exécuté sur un serveur d'administration.

    • Saisissez un numéro de port inutilisé dans le champ Port du serveur pour spécifier le port que les processus d'échange utiliseront pour communiquer avec le moteur d'indexation.

    • Saisissez le chemin absolu pour un nouvel emplacement de l'index dans le champ Répertoire d'index de l'index de la partition. Le moteur d'indexation que vous ajoutez effectuera les recherches dans cet index.

  10. A l'étape Résumé, consultez les informations que vous avez fournies pour la nouvelle partition, le moteur d'indexation et chaque moteur d'indexation, puis cliquez sur le bouton Terminer pour ajouter les processus.

 

Note :
Si vous rencontrez un problème et quittez l'assistant, il peut être nécessaire de supprimer le schéma de partition incomplet manuellement.

Après avoir quitté l'assistant et créé la nouvelle partition, redémarrez le schéma de partition. Si vous le souhaitez, vous pouvez faire basculer les anciennes partitions en mode Lecture seule ou Mettre à jour uniquement. (Que vous le fassiez ou non dépend de votre configuration et de vos besoins.)

Pour en savoir plus, voir Gestion des partitions.
