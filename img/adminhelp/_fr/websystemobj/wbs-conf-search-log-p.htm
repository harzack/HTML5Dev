Pour activer le journal de recherche :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de débogage.

  2. Sélectionnez la case à cocher Consigner les requêtes et les résultats de recherche pour enregistrer les requêtes et les résultats de recherche dans le fichier Content Server_home/logs/search.log.

  3. Pour ajouter des journaux de recherche aux fichiers Content Server_home/logs/threadn.out, cochez la case Ajouter la recherche et l'enregistrement de l'objet système aux journaux de thread.

  4. Cliquez sur le bouton Mettre à jour.

  5. Relancez le serveur sur l'hôte Content Server principal.
