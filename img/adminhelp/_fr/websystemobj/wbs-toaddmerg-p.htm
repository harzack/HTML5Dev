Pour ajouter un processus de fusion :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le gestionnaire de flux de données dans lequel vous voulez ajouter le processus de fusion.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Fusion dans le menu Ajouter un élément.

  4. Sur la page Ajouter Fusion, saisissez un nom pour le processus Fusion dans le champ Nom.

  5. Pour fournir une description du processus de fusion dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Fusion doit s'exécuter.

  7. Pour permettre à Content Server de surveiller le statut de ce processus, cochez la case Activer la gestion du système.

  8. Pour spécifier le répertoire d'exécution du processus de fusion, saisissez le chemin d'accès absolu du répertoire dans le champ Répertoire de lancement.

  9. Dans le champ Lire les données à partir de, saisissez le chemin de répertoire absolu de l'IPool à partir duquel le processus Fusion devra lire les données.

  10. Si le flux de données contient déjà des processus producteurs ou intermédiaires, leurs noms s'affichent dans la liste déroulante Processus. Dans cette liste, cliquez sur le processus auquel le processus de fusion doit succéder dans le flux de données.

  11. Dans le champ Ecrire les données dans, saisissez le chemin de répertoire absolu de l'IPool dans lequel vous voulez que le processus Fusion écrive les données fusionnées.

  12. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, les noms des processus apparaissent dans la liste déroulante Processus. Dans cette liste, cliquez sur le processus que le processus de fusion doit précéder dans le flux de données.

  13. Dans la section Options de lancement, modifiez les paramètres du processus de fusion.

  14. Dans la section Options d'arrêt, modifiez les paramètres du processus de fusion.

  15. Cliquez sur le bouton Ajouter.
