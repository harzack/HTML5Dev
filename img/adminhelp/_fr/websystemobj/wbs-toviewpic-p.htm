Pour afficher une image du flux de données :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données d'un dossier source de données.

  2. Cliquez sur l'icône Fonctions du gestionnaire de flux de données, puis sélectionnez Afficher l'image du flux de données.

 

Astuce :
Pour afficher une image du flux de données lorsque vous parcourez le contenu du gestionnaire de flux de données, cliquez sur l'icône Fonctions du gestionnaire de flux de données, puis sélectionnez Afficher l'image du flux de données.
