Pour configurer le mode par défaut :

  1. Dans la section Administration des apparences de la page d'administration, cliquez sur le lien Administration de la barre de recherche.

  2. Sur la page Administration de la barre de recherche, cliquez sur le bouton Modifier pour une configuration de barre de recherche.

  3. Sur la page Modification de la barre de recherche, cliquez sur une option dans la liste déroulante Mode par défaut.

  4. Dans la section Bouton de recherche, cliquez sur le bouton radio Icône ou Texte pour sélectionner l'apparence par défaut de la barre de recherche et le libellé.

  5. Dans le champ Texte du bouton, saisissez un libellé. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

  6. Dans le champ Largeur de la barre de recherche, saisissez une valeur comprise entre 100 et 400 pour déterminer, en pixels, la taille d'affichage de la barre de recherche.

  7. Dans la section Lien Aide, sélectionnez la case à cocher Afficher.

  8. Cliquez sur le bouton OK.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.

     

Pour plus d'informations sur les options de mode par défaut de la barre de recherche, reportez-vous à la rubrique Configuration des paramètres généraux.
