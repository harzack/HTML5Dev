Pour ajouter une règle de contrôle basée sur des messages IPool :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du lien préfixe_processus Dossier de la source de données, choisissez Propriétés, puis Règles de contrôle.

  2. Sur la page Règles de contrôle, cliquez sur le bouton Ajouter une nouvelle règle de contrôle .

  3. Sur la page Ajouter une nouvelle règle de contrôle, saisissez un nom pour la règle dans le champ Nom.

  4. Dans la liste déroulante Modèles, cliquez sur Nombre de messages dans le flux de données. Vous pouvez modifier la description de la règle dans le champ Description.

  5. Dans la section Conditions, cliquez sur l'un des éléments suivants de la liste déroulante Si le nombre de messages iPool est :

    • < (inférieur à)

    • > = (supérieur ou égal à)

    • = (égal à)

  6. Saisissez un nombre dans le champ des messages iPool.

  7. Dans la liste déroulante élément, cliquez sur la description de l'IPool qui sera surveillé par cette règle. Dans la section Rapports, vous pouvez entrer un message à envoyer avec le rapport dans le champ Message.

  8. Cliquez sur le bouton Mettre à jour.
