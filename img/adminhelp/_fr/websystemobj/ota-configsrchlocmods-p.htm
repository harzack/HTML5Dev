Pour configurer des modificateurs d'emplacement de recherche :

  1. Dans la section Administration de recherche de la page Administration, cliquez sur le lien Configurer les modificateurs d'emplacement de recherche.

  2. Cliquez sur le lien Suivre raccourcis.

    Le lien Suivre raccourcis est fourni par défaut, car les modificateurs d'emplacement de recherche sont modulaires. Les modules Content Server peuvent fournir des modificateurs d'emplacement supplémentaires. Les modules avec une interface Administration sont répertoriés sur la page Configurer des modificateurs d'emplacement de recherche.

  3. Sur la page Configurer des modificateurs d'emplacement de recherche, dans la section Statut, sélectionnez le niveau d'extension d'un emplacement de recherche :

    • Désactivé

    • Modificateur disponible en option

    • Appliquer toujours le modificateur

  4. Dans la section Type d'objet, les types d'objets sélectionnés actuellement sont affichés. Cliquez sur le lien liste de modification/révision pour ouvrir la page Configurer les types d'objets.

  5. Utilisez la page Configurer les types d'objets pour sélectionner les recueils Content Server où votre modificateur d'emplacement s'applique.

  6. Cliquez sur Mettre à jour.

  7. Sur la page Configurer des modificateurs d'emplacement de recherche, dans la section Restrictions, saisissez une valeur numérique dans la zone de texte pour spécifier le nombre maximal de dossiers liés.

  8. Sélectionnez le mode d'application du nombre maximal :

    • Tronquer au maximum : limite la recherche au nombre de fichiers spécifié.

    • Annuler si le maximum est dépassé : arrête la recherche avec un message d'erreur.

    • Ne pas appliquer de modificateur si le maximum est dépassé : permet la poursuite de la recherche sans aucun modificateur.

  9. Cliquez sur Mettre à jour pour utiliser les paramètres Modificateurs d'emplacement de recherche définis, ou sur Appliquer pour enregistrer les paramètres.
