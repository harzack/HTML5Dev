La page Configurer les types d'élément exclus vous permet de spécifier les types de nœud ou d'élément Content Server que vous souhaitez exclure de l'extraction. Le fait d'exclure un type d'élément de l'extraction implique que ni le contenu, ni les métadonnées associées au type d'élément défini ne sont indexés par le processus Extracteur Entreprise. Si le type d'élément Content Server correspondant au type de nœud défini est un élément de type recueil (un dossier, par exemple), seules les métadonnées associées à l'élément de type recueil sont exclues de l'extraction. En d'autres termes, le contenu et les métadonnées associées à ses contenus sont extraits, sauf si leurs types de nœud ont également été exclus.

Pour configurer des types d'élément exclus :

  1. Sur Pour définir les paramètres généraux de l'Extracteur, dans le champ Types d'élément de la section Paramètres d'exclusion, cliquez sur le lien liste de modification/révision.

  2. Sur la page Configurer les types d'élément exclus, sélectionnez la case à cocher Tout sélectionner, ou sélectionnez des types d'élément individuels pour spécifier ceux que l'Extracteur doit exclure des opérations d'indexation de recherche.

    Vous pouvez « passer la souris » sur des types d'élément individuels pour afficher leurs numéros d'ID dans le texte de pointage. C'est utile lorsque deux nœuds portent le même nom, ce qui est très rare mais possible.

  3. Si vous ne trouvez pas de type d'élément Content Server, consultez la section Types d'élément ne pouvant pas être recherchés pour voir s'il figure sur la liste.

  4. Cliquez sur Mettre à jour pour utiliser les paramètres de types d'élément exclus que vous avez définis, ou cliquez sur Réinitialiser les valeurs par défaut pour rétablir les paramètres système par défaut, comme pour une nouvelle installation Content Server, ou cliquez sur Annuler.
