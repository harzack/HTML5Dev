1. Dans la section Administration de recherche de la page Administration, cliquez sur le lien Configurer les paramètres de l'Extracteur Entreprise.

  2. Sur la page Configurer les paramètres de l'Extracteur Entreprise, cliquez sur le lien Définir les paramètres généraux de l'Extracteur.

  3. Dans la section Paramètres généraux de la page Définir les paramètres généraux de l'Extracteur, choisissez l'un des niveaux de consignation suivants dans la liste déroulante Niveau de journal :

    • Normale : désigne le niveau de consignation par défaut. Les problèmes graves, ainsi que les activités de démarrage et d'arrêt du système sont consignés. L'avancement minimum de l'Extracteur est consigné.

    • Information : spécifie le niveau du journal détaillé.

    • Détaillé : spécifie le niveau du journal de débogage, ce qui génère un journal détaillé pour permettre le débogage.

       

      Note :
      Content Server Le niveau de débogage doit être à 2 ou supérieur à cette valeur pour que la consignation de l'Extracteur fonctionne.

       

  4. Dans le champ Nombre maximum d'éléments par extraction, sous Nombre d'éléments, pour :

    • Extraire à nouveau

    • Mises à jour de pertinence

    • Ajouter/Modifier

    • Workflows lancés

    • Tous les autres

    Saisissez une valeur pour le nombre maximum de requêtes, nouvelles ou mises à jour, de mise à jour d'index à extraire de la base de données Content Server à chacune de ses exécutions. La valeur par défaut pour chaque type d'itérateur est 5000.

    Sous Options, utilisez la liste déroulante pour sélectionner :

    • Extraction normale

    • Suspendre l'extraction

    • Rejeter les demandes d'indexation

    Dans Nombre d'éléments, pour Tous les autres, saisissez une valeur pour le nombre maximum de tous les autres éléments nouveaux ou mis à jour. La valeur par défaut est 5000.

  5. Dans le champ Nombre maximum de versions à extraire, saisissez une valeur pour le nombre maximum de versions par document extraites dans la vue de recherche Entreprise [Toutes les versions].

    Sélectionnez la case à cocher Toutes les versions pour permettre la mise à jour de toutes les versions des documents lorsque des modifications sont apportées.

     

    Note :
    Pour que l'Extracteur ne ralentisse pas le système, la valeur Nombre maximum de versions à extraire est limitée à la valeur Nombre d'éléments, même si elle est définie sur illimitée.

     

  6. Dans le champ Inclure les listes de contrôle d'accès, sélectionnez la case à cocher pour placer les informations d'autorisation ACL (liste de contrôle d'accès) dans les IPools.

  7. Dans le champ Contenu d'objet EFS, sélectionnez un bouton radio pour indiquer comment le contenu d'objet EFS (magasin de fichiers externe) est traité :

    • Incorporer une copie du contenu d'objet dans les messages IPool : incorporée dans les messages IPool pour augmenter les performances d'indexation.

    • Autoriser le DCS à lire le contenu d'objet depuis EFS : transmis via certaines étapes de flux de données en tant que fichiers séparés.

       

      Note :
      Les fichiers de données sont référencés directement dans EFS, plutôt que d'être extraits à nouveau vers le disque. Les fichiers ne sont pas supprimés.

      OpenText Si ce paramètre est utilisé, le service d'assistance clientèle vous recommande de définir votre serveur d'administration sur le mode En lecture seule pour EFS afin de prévenir les pertes de données.

       

  8. Dans le champ Contenu d'objet non EFS, sélectionnez un bouton radio pour indiquer comment le contenu d'objet non EFS (magasin de fichiers non externe) est traité :

    • Incorporer une copie du contenu d'objet dans les messages IPool : incorporée dans les messages IPool pour augmenter les performances d'indexation.

    • Créer une copie temporaire du contenu d'objet qui sera supprimée par le DCS après traitement : le contenu d'un objet est transmis via certaines étapes de flux de données en tant que fichiers séparés.

      Dans le champ Chemin d'accès au fichier, saisissez un chemin d'accès au répertoire Content Server valide.

       

      Note :
      Le chemin d'accès au fichier doit être défini pour ce paramètre pour fonctionner. Les fichiers sont extraits du disque à cet emplacement, puis supprimés une fois traités ultérieurement dans le flux de données.

       

  9. Dans le champ Taille totale de la section Limites IPool, spécifiez une limite, en Mo, de taille maximale pour le contenu d'un document qui figurera dans les IPools, pour tous les objets.

  10. Dans le champ Taille du contenu d'élément unique, spécifiez une limite, en Mo, de taille maximale pour le contenu d'un document qui figurera dans les IPools, pour tous les objets.

  11. Dans le champ Nombre d'éléments, définissez le nombre maximum d'éléments extraits et regroupés dans un seul message IPool par le processus Extracteur Entreprise.

  12. Dans le champ Types d'élément de la section Paramètres d'exclusion, cliquez sur le lien liste de modification/révision pour ouvrir Configuration des types d'élément exclus afin de spécifier les types de nœud ou d'élément Content Server que vous souhaitez exclure de l'extraction.

    Le nombre de types d'éléments exclus est également affiché.

  13. Dans le champ Types de volume, cliquez sur le lien liste de modification/révision pour ouvrir Configuration des types de volume exclus afin de spécifier les types de volume Content Server que vous souhaitez exclure de l'extraction.

    Le nombre de types de volumes exclus est également affiché.

  14. Dans le champ Emplacements, cliquez sur le lien liste de modification/révision pour ouvrir Configuration des emplacements exclus afin de spécifier les emplacements Content Server que vous souhaitez exclure de l'extraction.

    Le nombre d'emplacements exclus est également affiché.

  15. Dans le champ Types de présentation, cliquez sur le lien liste de modification/révision pour ouvrir Configuration des types de présentation afin de spécifier les présentationsContent Server que vous souhaitez exclure de l'extraction.

    Le nombre de types de présentations exclus est également affiché.

  16. Cliquez sur Mettre à jour pour utiliser les paramètres que vous avez définis, ou cliquez sur Réinitialiser les valeurs par défaut pour restaurer les paramètres existants lorsque vous avez ouvert cette page, ou cliquez sur Annuler.
