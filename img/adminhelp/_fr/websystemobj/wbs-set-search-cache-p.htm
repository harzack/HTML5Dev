Pour modifier le délai de stockage des résultats de la recherche dans la mémoire cache :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de débogage.

  2. Dans le champ Expiration du délai de stockage des résultats de la recherche dans la mémoire cache, saisissez le nombre de minutes pendant lesquelles vous souhaitez que Content Server stocke les résultats de recherche dans la mémoire cache de sa base de données.

  3. Cliquez sur le bouton Mettre à jour.
