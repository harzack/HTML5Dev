Lorsque les utilisateurs Content Server effectuent une recherche, Content Server place temporairement les résultats de la recherche dans la mémoire cache de la base de données Content Server. Content Server utilise les résultats de recherche mis en mémoire cache pour permettre aux utilisateurs d'afficher et de masquer le résumé et d'enregistrer leurs résultats sans avoir à exécuter la recherche à nouveau.

Vous pouvez définir la durée pendant laquelle Content Server conserve les résultats de recherche dans la mémoire cache de la base de données Content Server. Le paramètre par défaut est de 30 minutes. A la fin de cette période, Content Server efface les résultats de recherche de la mémoire cache de la base de données.
