Pour assurer la maintenance d'un flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données qui contient le flux de données à configurer.

  2. Cliquez sur l'icône Fonctions du gestionnaire du flux de données, sélectionnez Propriétés, puis Infos spécifiques.

  3. Dans l'onglet Infos spécifiques de la page Propriétés du gestionnaire de flux de données, cliquez sur l'un des boutons suivants :

    • Reprendre pour lancer les processus d'un flux de données interrompu.

    • Interrompre pour arrêter les processus d'un flux de données.

    • Vider pour arrêter les processus de flux de données et supprimer tous les messages IPool en attente dans le flux de données.

    • Tester pour envoyer un message au flux de données afin de déterminer si le message passe bien d'un bout à l'autre du flux de données. Lorsque vous testez un flux de données, Content Server arrête le flux de données, élimine tous les messages existants, puis envoie le message test.

    • Resynchroniser pour recréer les répertoires iPool conformément aux informations relatives au flux de données stocké dans la base de données Content Server.
