Onglet Infos spécifiques de la page Propriétés Distributeur de mise à jour

Elément Description
Etat Indique si le processus Distributeur de mise à jour est en cours d'exécution. Vous ne pouvez pas modifier ce champ.
Hôte Indique le serveur d'administration sur lequel s'exécute le processus.
Activer la gestion du système Détermine si Content Server contrôle ce processus pour détecter le retour de messages d'erreur. S'il retourne un message d'erreur, Content Server l'enregistre dans la base de données. La gestion du système est activée par défaut. Pour plus d'informations sur la configuration de Content Server en vue de l'envoi d'e-mails d'alerte lorsqu'un processus de flux de données rencontre des erreurs, reportez-vous à la rubrique Pour activer la recherche d'erreurs et l'envoi d'e-mails.
Valeur maximale du code de sortie correcte Lorsqu'un processus rencontre une erreur, il renvoie un code d'erreur. Si le code d'erreur est inférieur au nombre spécifié dans le champ Valeur maximale du code de sortie correcte, le processus essaye de redémarrer. Si le numéro du code d'erreur est supérieur au numéro inscrit dans le champ Valeur maximale du code de sortie correcte, le processus ne tente pas de redémarrer et devra être redémarré manuellement. OpenText recommande de ne pas modifier la Valeur maximale du code de sortie correcte sauf si le service d'assistance clientèle d'OpenText vous le demande.
Port Admin Indique le port sur lequel s'exécute le processus Distributeur de mise à jour. Le serveur d'administration du processus utilise ce port pour le démarrer et l'arrêter. Vous pouvez vérifier la disponibilité du numéro de port spécifié en cliquant sur le lien Vérifier le port.
Utilisation max. de la mémoire par le processus Indique la quantité de mémoire allouée au processus Distributeur de mise à jour.
IPools Cliquez sur le bouton Informations pour afficher les informations sur les IPools à partir desquels le processus Distributeur de mise à jour lit les données ou dans lesquels il écrit.
Démarrer/Arrêter Permet de démarrer ou d'arrêter le processus du moteur d'indexation ou tous les processus d'indexation (le processus Distributeur de mise à jour et les processus du moteur d'indexation appartenant à un système d'indexation et de recherche). Que vous arrêtiez tous les processus d'indexation ou uniquement le processus Distributeur de mise à jour, l'indexation s'arrête. Si les processus d'indexation ne sont pas en cours d'exécution, les boutons Démarrer s'affichent ; si les processus d'indexation sont en cours d'exécution ou s'ils sont programmés, les boutons Arrêter s'affichent.
Redémarrer Permet d'arrêter puis de démarrer tous les processus d'indexation appartenant à un système d'indexation et de recherche.
Resynchroniser Permet de resynchroniser les informations sur le processus Distributeur de mise à jour avec les informations de la base de données Content Server.
