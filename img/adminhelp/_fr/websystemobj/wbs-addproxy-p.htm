Pour ajouter un proxy :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le flux de données auquel vous voulez ajouter le proxy.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Proxy dans le menu Ajouter un élément.

  4. Sur la page Ajouter Proxy, saisissez un nom pour le proxy dans le champ Nom.

  5. Pour fournir une description du proxy dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le proxy doit s'exécuter.

  7. Dans le champ Lire les données à partir de, saisissez le chemin absolu de l'IPool à partir duquel le proxy devra lire les données.

  8. Si le flux de données contient déjà au moins un processus producteur ou intermédiaire, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que devra suivre le proxy dans le flux de données.

  9. Dans le champ Ecrire les données dans, saisissez le chemin absolu de l'IPool dans lequel le proxy devra écrire les données.

  10. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que précèdera le proxy dans le flux de données.

  11. Cliquez sur le bouton Ajouter.
