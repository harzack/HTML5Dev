Pour configurer l'affichage des résultats de recherche :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Configurer les options de recherche.

  2. Sur la page Configurer les options de recherche, dans la section Affichage des résultats de recherche, saisissez une valeur dans le champ Agents mis en mémoire cache à conserver.

  3. Saisissez une valeur dans le champ Longueur du nom pour les colonnes Nom et Emplacement de la page Résultats de recherche.

  4. Saisissez une valeur dans le champ Longueur de chemin pour la longueur du chemin Fil d'Ariane.

  5. Cliquez sur le bouton radio Vrai dans le champ Fil d'Ariane pour afficher le chemin des emplacements de recueil.

  6. Cliquez sur le bouton Mettre à jour.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.
