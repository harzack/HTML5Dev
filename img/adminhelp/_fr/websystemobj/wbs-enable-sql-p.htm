Pour activer les journaux de connexion à la base de données :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de débogage.

  2. Sur la page Configurer les paramètres du débogage, activez la case à cocher Enregistrer les connexions. Cela modifie le même paramètre que la case à cocher Journal SQL de la page Configurer les paramètres du serveur.

  3. Cliquez sur le bouton Mettre à jour.

  4. Relancez le serveur Content Server sur l'hôte Content Server principal.
