Pour ajouter un processus Importateur :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le flux de données auquel vous voulez ajouter un processus Importateur.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Importateur dans le menu Ajouter un élément.

  4. Sur la page Ajouter Importateur, saisissez un nom pour le processus Importateur dans le champ Nom.

  5. Pour fournir une description du processus Importateur dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Importateur doit s'exécuter.

  7. Pour permettre à Content Server de surveiller le statut de ce processus, cochez la case Activer la gestion du système.

  8. Pour spécifier le répertoire d'exécution du processus Importateur, saisissez-en le chemin absolu dans le champ Répertoire de lancement. Si vous ne spécifiez pas de répertoire, le processus Importateur est exécuté dans le répertoire Content Server_home/bin.

  9. Cliquez sur la tâche qui sera effectuée par le processus Importateur dans la liste déroulante Importer la définition de tâche.

  10. Pour configurer la tâche exécutée par le processus Importateur, cliquez sur le bouton Configurer.

  11. Dans le champ Lire les données à partir de, saisissez le chemin absolu de l'IPool à partir duquel le processus Importateur devra lire les données. Si le flux de données contient déjà au moins un processus, la liste déroulante connecté(e) à apparaît.

  12. Dans la liste déroulante connecté à, cliquez sur le processus qui précèdera le processus Importateur dans le flux de données.

     

    Astuce :
    Dans la plupart des flux de données, le processus Importateur suit le processus Distributeur de mise à jour.

     

  13. Dans la section Options de lancement, définissez les paramètres de lancement du processus Importateur.

  14. Cliquez sur le bouton Ajouter.
