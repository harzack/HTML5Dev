Pour définir des options sur les pages de configuration de texte intégral :

  1. Dans la section Administration des apparences de la page d'administration, cliquez sur le lien Administration de la barre de recherche.

  2. Sur la page Administration de la barre de recherche, cliquez sur le bouton Modifier pour une configuration de barre de recherche.

  3. Sur la page Configurer les options de recherche, dans la section Texte intégral, cliquez sur le lien Configurer associé à l'un des éléments suivants :

    • Types pouvant faire l'objet de recherches

    • Créé par

    • Une fois les modifications apportées

    • Sélection de vue de recherche

  4. Dans la section Eléments de menu de la page Configurer, effectuez l'une des opérations suivantes :

    • Dans la liste Disponible(s), cliquez sur les éléments que vous souhaitez voir apparaître dans la liste déroulante de la barre de recherche, puis sur le bouton Afficher pour les déplacer vers la liste Affiché(s).

    • Dans la liste Affiché(s), cliquez sur les éléments que vous ne souhaitez pas voir apparaître dans la liste déroulante de la barre de recherche, puis sur le bouton Enlever pour les déplacer vers la liste Disponible(s).

    • Cliquez sur les boutons Déplacer vers le haut et Déplacer vers le bas pour modifier l'ordre des éléments dans la liste Affiché(s).

     

    Astuce :
    Pour retirer un élément de menu de la liste Disponible(s), cliquez sur le bouton Enlever l'élément de menu sélectionné .

     

  5. Cochez la case Afficher le texte dans le menu.

  6. Dans la section Modifier, procédez de la façon suivante :

    • Saisissez le nom de l'élément de menu dans le champ Libellé. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

    • Saisissez le chemin relatif de l'image dans le champ Image.

    • Cliquez sur le niveau de retrait dans la liste déroulante Retrait.

    • Saisissez la requête LQL dans le champ Requête.

       

      Notes :

      • Vous devez sélectionner un élément de menu dans la liste Disponible(s) ou Affiché(s) pour effectuer toute modification.

      • Pour ajouter un élément de menu dans la liste Disponible(s), définissez les paramètres dans la section Modifier, puis cliquez sur le bouton Ajouter un nouvel élément de menu .

       

  7. Vérifiez les paramètres de l'élément de menu dans la section Aperçu.

  8. Cliquez sur le bouton Envoyer.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer ces paramètres.

     

Pour plus d'informations sur les options, reportez-vous à la rubrique Configuration de la recherche Texte intégral.
