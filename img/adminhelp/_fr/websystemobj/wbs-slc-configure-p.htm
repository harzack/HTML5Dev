Pour configurer les vues de recherche dans un ordre personnalisé :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur le lien Ouvrir le volume des objets système.

  2. Cliquez sur l'icône Fonctions du dossier Vues de recherche et choisissez Configurer les vues de recherche.

  3. Sur la page Dossier Vues de recherche, pour Ordre des vues de recherche dans la barre de recherche et dans le composant Vue de recherche, sélectionnez l'ordre alphabétique dans lequel les vues de recherche sont affichées. Vous pouvez définir une configuration personnalisée en sélectionnant le bouton radio Personnaliser. Les vues de recherche Disponible et Affiché de votre système Content Server sont affichées de sorte que vous puissiez les disposer dans l'ordre de votre choix.

  4. Dans Ajouter l'option 'A partir d'ici' à la barre de recherche, sélectionnez la case à cocher pour afficher l'option A partir d'ici dans la liste Vues de recherche de la barre de recherche dans la recherche avancée.

  5. Dans Taille du menu du composant Vue de recherche pour la recherche avancée, saisissez la hauteur, en lignes, du menu du composant Vue de recherche pour la recherche avancée.

  6. Cliquez sur Mettre à jour.
