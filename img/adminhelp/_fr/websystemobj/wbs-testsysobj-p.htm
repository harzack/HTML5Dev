Pour tester les objets système pour diagnostiquer des erreurs :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Afficher le statut de l'objet système.

  2. Si vous avez spécifié des destinataires d'e-mails, cochez la case Rapport d'e-mail, sur la page Afficher le statut de l'objet système, afin d'envoyer des e-mails d'alerte relatifs aux objets système aux destinataires que vous spécifiez sur la page Configurer l'envoi des e-mails d'alerte de l'objet système par messagerie.

  3. Cliquez sur le bouton Test.

  4. Cliquez sur le bouton Continuer pour revenir à la page d'administration.
