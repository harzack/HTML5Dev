Pour déplacer un processus de flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données qui contient le processus de flux de données à déplacer.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur l'icône Fonctions du processus à déplacer, puis sélectionnez Arrêter.

  4. Cliquez sur le lien du nom du processus à déplacer.

  5. Dans la liste déroulante Serveur d'administration de l'onglet Infos spécifiques sur la page Propriétés du processus, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus doit s'exécuter.

  6. Cliquez sur le bouton Informations.

  7. Sur la page Informations iPool, saisissez le chemin du répertoire d'écriture tel qu'il est mappé/monté sur l'hôte du nouveau serveur d'administration dans le champ Ecrire. Le champ Ecrire n'est pas disponible si le processus de flux de données à déplacer n'écrit pas les données dans un répertoire.

  8. Saisissez le chemin du répertoire de lecture tel qu'il est mappé/monté sur l'hôte du nouveau serveur d'administration dans le champ Lire. Le champ Lire n'est pas disponible si le processus de flux de données à déplacer ne lit pas les données à partir d'un répertoire (par exemple, un processus Directory Walker).

  9. Cliquez sur le bouton Mettre à jour.

  10. Cliquez sur l'icône Fonctions du processus déplacé, puis sélectionnez Démarrer.
