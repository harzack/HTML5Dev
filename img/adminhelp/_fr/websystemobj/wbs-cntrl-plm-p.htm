Pour configurer l'emplacement des partitions :

  1. Cliquez sur l'icône Fonctions du schéma de partition Entreprise, choisissez Propriétés, puis Gestionnaire d'emplacement des partitions.

  2. Dans la section Méthode de la page Schéma de la partition, cliquez sur l'un des boutons suivants :

    • Répartir les partitions sur les emplacements actifs, l'une après l'autre

    • Utiliser les emplacements actifs séquentiellement, jusqu'à ce qu'ils soient pleins

  3. Dans la section Emplacements des partitions, cliquez sur un serveur dans la liste déroulante Serveur d'administration.

  4. Dans le champ Chemin d'accès de l'emplacement, saisissez le chemin d'accès d'un répertoire d'indexation et d'un répertoire de sauvegarde. Vous pouvez également cliquer sur le bouton Parcourir pour naviguer dans le système de fichiers du serveur d'administration.

  5. Dans la liste déroulante Disponible, cliquez sur le nombre de partitions.

  6. Cochez la case Actif pour chaque emplacement dans lequel vous souhaitez créer des partitions.

  7. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Vous pouvez également spécifier un chemin d'accès au répertoire dans le champ Chemin d'accès de l'emplacement en cliquant sur le bouton Parcourir, puis en naviguant dans le système de fichiers du serveur d'administration pour le sélectionner. Vous pouvez ajouter ou supprimer des emplacements du serveur d'administration en cliquant sur le bouton Ajouter un nouvel emplacement ou , respectivement.
