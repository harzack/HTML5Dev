Vous configurez les paramètres d'un processus Importateur pour gérer la ligne de commande, les informations IPool et les options de lancement et d'arrêt, ou pour resynchroniser les données associées au processus. Configurez les processus Importateur sur leurs pages Propriétés.

Onglet Infos spécifiques de la page Propriétés du processus Importateur

Elément Description
Etat Cet élément spécifie si le processus Importateur est ou non en cours d'exécution. Vous ne pouvez pas modifier ce champ.
Activer la gestion du système Détermine si Content Server contrôle ce processus pour détecter le retour de messages d'erreur. S'il retourne un message d'erreur, Content Server l'enregistre dans la base de données. La gestion du système est activée par défaut. Pour plus d'informations sur la configuration de Content Server en vue de l'envoi d'e-mails d'alerte lorsqu'un processus de flux de données rencontre des erreurs, reportez-vous à la rubrique Pour activer la recherche d'erreurs et l'envoi d'e-mails.
Serveur d'administration Cet élément spécifie le raccourci du serveur d'administration sur l'hôte duquel le processus Importateur s'exécute. Ne modifiez pas le raccourci du serveur d'administration destiné au processus Importateur sauf en cas de nécessité (par exemple, si l'hôte du serveur d'administration n'est plus disponible).
Modèle de commande Cet élément spécifie le modèle utilisé pour générer la ligne de commande du processus Importateur. Ne modifiez pas cette valeur, sauf si le service d'assistance clientèle d'OpenText vous le demande.
Ligne de commande Cet élément spécifie la ligne de commande utilisée par Content Server pour exécuter le processus Importateur. Vous ne pouvez pas modifier la valeur de ce champ.
Répertoire de démarrage Spécifie le répertoire dans lequel le processus Importateur s'exécute. Ne modifiez pas cette valeur, sauf si le service d'assistance clientèle d'OpenText vous le demande.
Bouton Informations Cet élément permet de charger la page Gestion de lien des données de flux, qui spécifie le répertoire à partir duquel le processus Importateur lit les données. Ne modifiez pas ces valeurs, sauf si vous déplacez un IPool ou si le service d'assistance clientèle d'OpenText vous le demande.
Importer la définition de tâche Tâche effectuée par l'importateur à l'aide des données récupérées par le processus Distributeur de mise à jour (par exemple, Classification).
Options de démarrage Cet élément permet de spécifier les paramètres de démarrage du processus Importateur.
•

Manuel vous permet de démarrer le processus Importateur manuellement dès lors que vous souhaitez l'exécuter.

•

Programmé - Permet de programmer le jour et l'heure d'exécution du processus Importateur.

•

Continuel - Entraîne l'exécution en continu du processus Importateur

Bouton Démarrer/Arrêter Cet élément permet de démarrer ou d'arrêter le processus Importateur selon l'état actuel de ce dernier. Si le processus n'est pas en cours d'exécution, le bouton Démarrer s'affiche. Si le processus est en cours d'exécution ou programmé pour être exécuté, le bouton Arrêter s'affiche.
Bouton Resynchroniser Cet élément resynchronise les informations relatives à ce processus dans le fichier otadmin.cfg du serveur d'administration du processus avec les informations de la base de données Content Server.
Bouton Mettre à jour Permet de soumettre au serveur Content Server les modifications que vous avez apportées à cette page.
Bouton Réinitialiser Rétablit les informations de cette page à l'état qui était le leur à l'ouverture de la page.
