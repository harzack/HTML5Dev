Pour régler le stockage de la mémoire :

  1. Sur la page Administration, sous Administration de recherche, cliquez sur Ouvrir le volume des objets système.

  2. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier de la source de données contenant le schéma de partition pour lequel vous souhaitez régler les paramètres de mémoire des métadonnées.

  3. Cliquez sur l'icône Fonctions d'un schéma de partition, choisissez Propriétés, puis Paramètres de mémoire des métadonnées.

  4. Sur l'onglet Paramètres de mémoire des métadonnées du schéma de partition, cliquez sur l'icône Modifier de l'un des modes de partition pour régler les paramètres de stockage de la mémoire :

    • Lecture/Ecriture

    • Mettre à jour uniquement

    • Lecture seule

    • Hors service

       

      Note :
      En mode Hors service, un index accepte les mises à jour et les suppressions, mais pas les nouveaux objets. De plus, un remplacement complet d'un objet en mode Hors service déplace l'objet vers une autre partition et le supprime de la partition d'origine à des fins d'équilibrage.

       
     

    Note :
    Open Text recommande généralement de ne pas utiliser les modes En lecture seule et Mettre à jour uniquement.

     

  5. Sur la page Modifier les paramètres de mémoire des métadonnées, dans la section Affectations de régions, effectuez l'une des opérations suivantes :

    • Cliquez sur une région de texte pour la sélectionner ou maintenez la touche CTRL pour en sélectionner plusieurs.

    • Utilisez la zone de texte de sélection de mémoire pour sélectionner les régions qui enregistrent le nombre de mégaoctets que vous indiquez.

    La modification de la quantité de mémoire vive requise et la modification de la quantité de mémoire utilisée sont affichées lorsque vous sélectionnez les régions.

  6. Cliquez sur les icônes Flèche gauche Flèche droite pour déplacer les régions afin de les stocker dans la Mémoire, sur le Disque, ou en mode Récupération du disque.

  7. Dans la section Affectations de groupe, cliquez sur le bouton Mémoire ou Disque pour affecter toutes les régions d'un type donné à la mémoire ou au disque.

     

    Note :
    Le Mode Mémoire faible dans le moteur de recherche permet d'indexer davantage d'objets dans une partition utilisant le mode disque pour stocker des métadonnées de texte, et présente une légère baisse des performances de recherche. La modification de ce paramètre entraîne la reconstruction des index pour les régions de métadonnées utilisant le stockage Disque.

    Par défaut, ce mode est activé. Vous pouvez l'activer à l'aide du paramètre du bouton radio Mode Mémoire faible dans la section Paramètres

     

  8. Dans la section Emplacement par défaut, cliquez sur le bouton radio Mémoire ou Disque pour définir l'emplacement par défaut des nouvelles régions.

     

    Avertissement
    Lorsque vous modifiez une partition sur En lecture seule, le moteur d'indexation pour cette partition s'arrête et cesse d'indexer tout élément mis à jour ou supprimé.

     

  9. Cliquez sur le bouton Appliquer.

  10. Revenez sur l'onglet Paramètres de mémoire des métadonnées décrit dans Pour régler les paramètres de mémoire des métadonnées, et cliquez sur le bouton Redémarrer les processus d'indexation.
