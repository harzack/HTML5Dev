Le tableau suivant décrit les messages d'erreur relatifs au processus du moteur d'indexation.

Si ces erreurs persistent, contactez le service d'assistance clientèle d'OpenText pour obtenir plus d'informations.

Tableau 1-29 : Messages d'erreur relatifs aux moteurs d'indexation

Numéro Message Description Solution
132 Une erreur inconnue s'est produite Le processus du moteur d'indexation a rencontré une erreur inconnue. Contactez l'assistance clientèle d'OpenText.
149 Ligne de commande non valable pour le moteur d'indexation. Les paramètres de ligne de commande du processus du moteur d'indexation comportent une erreur. Vérifiez que les lignes de commande dans le fichier otadmin.cfg sont valables et spécifiées au format correct auquel Content Server génère le fichier otadmin.cfg à l'aide de sa base de données.
150 L'URL n'est pas valable L'URL utilisée pour enregistrer et contacter le moteur d'indexation n'est pas valable telle qu'elle est spécifiée dans le fichier search.ini. Assurez-vous que les paramètres du fichier search.ini sont valables et que leur format est correct.
153 Impossible d'initialiser Le processus du moteur d'indexation n'a pas pu démarrer car un ou plusieurs paramètres du fichier search.ini ne sont pas valables. Assurez-vous que les paramètres du fichier search.ini sont valables et que leur format est correct.
171 Echec de l'ajout d'une partition Le processus de moteur d'indexation n'a pas pu ajouter une nouvelle partition. Contactez l'assistance clientèle d'OpenText.
174 Impossible de créer un index vide Le processus de moteur d'indexation n'a pas pu créer un nouvel index vide. Contactez l'assistance clientèle d'OpenText.
175 Impossible de lire l'index existant Le processus de moteur d'indexation n'a pas pu lire un index existant. Contactez l'assistance clientèle d'OpenText.
176 Impossible de restaurer l'index Le processus de moteur d'indexation n'a pas pu restaurer un index. Contactez l'assistance clientèle d'OpenText.
180 Impossible de démarrer le moteur d'indexation Un processus du moteur d'indexation est peut-être déjà en cours d'exécution. Si aucun moteur d'indexation n'est en cours d'exécution, contactez le service d'assistance clientèle d'OpenText.
181 Impossible de démarrer le moteur d'indexation Vous ne pouvez pas démarrer un moteur d'indexation alors qu'un index est en cours de restauration. Attendez la fin complète de la restauration de l'index.
 

Note :
Le fichier search.ini est le fichier de configuration pour les processus d'indexation et de recherche de Content Server. Ce fichier contient des paramètres qui sont généralement définis par défaut lors de la création des processus d'indexation et de recherche et d'autres objets système, tels que les sources de données et les partitions ; ces paramètres sont supprimés lorsque les processus qu'ils configurent sont supprimés. La configuration de ces paramètres a lieu lors de la configuration des processus d'indexation et de recherche et des autres objets système dans Content Server. OpenText vous recommande vivement de ne pas modifier les paramètres du fichier search.ini. Si vous décidez de modifier ces paramètres, contactez le service d'assistance clientèle d'OpenText.
