Pour configurer la recherche de similarités :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Configurer les options de recherche.

  2. Sur la page Configurer les options de recherche, dans la section Rechercher similarités, cliquez sur le bouton radio Activer pour ajouter la commande Rechercher similarités au menu Fonctions des documents.

  3. Cliquez sur le bouton Mettre à jour.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.
