Dans Content Server 10, vous pouvez lier une configuration de barre de recherche à une apparence, puis appliquer cette apparence à un dossier. Pour plus d'informations sur les apparences, reportez-vous à la rubrique Utilisation des apparences.

Pour afficher une barre de recherche personnalisée dans un dossier spécifique :

  1. Recherchez le dossier dans Content Server dans lequel vous souhaitez ajouter une apparence. Pour en savoir plus, voir Utilisation des apparences.

  2. Ouvrez l'apparence.

  3. Cliquez sur le lien de la section Composants Content Server dans le coin inférieur droit de la page.

  4. Dans le menu de la barre de recherche, sélectionnez celui que vous souhaitez utiliser.

  5. Cliquez sur Envoyer.

  6. De retour sur la page Modifier pour l'apparence, cliquez sur le lien Paramètres de la partie supérieure.

  7. Sur la page Paramètres, choisissez comment appliquer l'apparence puis appliquez-la.

Comme les apparences sont affichées en fonction des autorisations de l'utilisateur, il est possible qu'une barre de recherche différente s'affiche, selon l'apparence que l'utilisateur consulte dans ce dossier. Pour définir les apparences générales, sur la page Administration, dans la section Administration des apparences, cliquez sur le lien Ouvrir le volume Apparences.

Une autre manière d'obtenir une interface de recherche personnalisée dans un dossier est d'utiliser une recherche avec vue personnalisée. Pour en savoir plus, voir Pour créer une recherche de vue personnalisée.
