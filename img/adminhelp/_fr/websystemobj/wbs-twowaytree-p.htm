Pour ajouter un T :

  1. Sur la page Volume des objets système, cliquez sur le lien <préfixe> Dossier de la source de données correspondant au dossier de la source de données qui contient le gestionnaire de flux de données dans lequel vous souhaitez ajouter le processus T.

  2. Cliquez sur le lien <préfixe> Gestionnaire de flux de données.

  3. Dans le menu Ajouter un élément, cliquez sur

    T.

  4. Sur la page Ajouter : T, saisissez un nom pour le processus T dans le champ Nom.

  5. Dans le champ Description, saisissez une description du processus T.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte à partir duquel le processus T doit s'exécuter.

  7. Vous pouvez cocher la case Activer la gestion du système.

  8. Dans le champ Mode de sortie, sélectionnez le bouton radio Copier les messages IPool dans toutes les sorties, pour envoyer tous les messages IPool à plusieurs processus de sortie IPool, ou sur Distribuer les objets IPool (contenu de message) parmi toutes les sorties, pour répartir tous les messages IPool parmi les processus de sortie.

     

    Note :
    Lorsque vous sélectionnez le mode de sortie de distribution, les champs Région de filtre et Valeur de filtre sont désactivés dans la section Ecrire les données dans.

     

  9. Dans la zone Lire les données à partir de, dans le champ Zone principale IPool, saisissez le chemin absolu du répertoire de l'IPool à partir duquel vous souhaitez que le processus T lise les données.

    Dans la liste déroulante Processus, sélectionnez le processus qui doit précéder le processus T dans le flux de données.

  10. Si vous le souhaitez, dans la zone Ecrire les données dans, cliquez sur le bouton Ajouter un lien d'écriture IPool afin de configurer jusqu'à cinq processus qui suivent le processus T.

    Dans le champ Zone principale IPool, saisissez le chemin absolu du répertoire de l'IPool dans lequel vous souhaitez que le processus T écrive les données.

    Dans la liste déroulante Processus, sélectionnez le processus qui doit suivre le processus T dans le flux de données.

    Dans le champ Région de filtre, saisissez le nom de la région à appliquer en tant que filtre.

    Dans le champ Valeur de filtre, indiquez une valeur autorisée pour la région spécifiée dans le paramètre Région de filtre.

  11. Dans la section Options de démarrage, cliquez sur Manuel ou sur Programmé, puis procédez de l'une des façons suivantes :

    • Dans les listes déroulantes A cette heure, définissez les jours de la semaine et l'heure d'exécution du processus T.

    • Cliquez sur le bouton radio Intervalle, puis cliquez sur des valeurs des listes déroulantes pour sélectionner l'intervalle entre deux exécutions du processus T.

      Cliquez sur le bouton radio Continuel pour que le processus T s'exécute continuellement.

  12. Dans la section Options d'arrêt, saisissez une valeur dans le champ Valeur maximale du code de sortie correcte; afin de spécifier le numéro de code d'erreur maximum pour lequel le serveur d'administration redémarre automatiquement le processus T en cas d'échec.

  13. Cliquez sur le bouton Ajouter.

 

Astuce :
Pour plus d'informations sur les paramètres disponibles durant l'ajout d'un processus T, voir Ajout d'un processus T.
