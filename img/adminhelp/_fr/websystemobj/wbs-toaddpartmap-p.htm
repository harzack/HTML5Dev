Pour ajouter un schéma de partition :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données à laquelle vous souhaitez ajouter un schéma de partition.

  2. Cliquez sur le lien préfixe_processus Schéma de partition.

  3. Saisissez un nom pour le schéma de partition dans le champ Nom.

  4. Pour afficher une description du schéma de partition dans l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  5. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  6. Cliquez sur le bouton Ajouter.
