Pour créer une vue de recherche pour un projet Content Server et son contenu :

  1. Allez à la page Espace de travail Projet du projet pour lequel vous voulez créer une vue de recherche.

  2. Saisissez le texte suivant dans le champ Rechercher qui se trouve dans la barre de recherche dans la partie supérieure de l'espace de travail Projet : * (asterisk)

  3. Cliquez sur A partir d'ici dans la liste déroulante, puis cliquez sur le bouton Aller.

  4. Assurez-vous que la barre de recherche affiche Rechercher dans Content Server.

  5. Sur la page Résultats de recherche, cliquez sur le bouton Enregistrer la requête.

  6. Sur la page Ajouter Requête, saisissez un nom unique pour la vue de recherche dans le champ Nom.

  7. Pour afficher une description de la vue de recherche dans l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  8. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  9. Cliquez sur le bouton Parcourir Content Server, sélectionnez le dossier système et cliquez sur le lien Sélectionner dans la colonne Actions du Dossier Vues de recherche.

  10. Cliquez sur le bouton Ajouter.

 

Note :
Vous créez une vue de recherche à partir d'un projet Content Server lorsque vous voulez autoriser les utilisateurs Content Server à formuler des requêtes dans le contenu du projet.
