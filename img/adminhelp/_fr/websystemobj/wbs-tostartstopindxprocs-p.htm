Pour lancer ou arrêter les processus d'indexation :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier de la source de données auquel sont associés les processus d'indexation que vous voulez démarrer ou arrêter.

  2. Cliquez sur le lien préfixe_processus Schéma de partition.

  3. Sur la page Schéma de partition, procédez de la façon suivante :

    • Cliquez sur l'icône Fonctions du processus Distributeur de mise à jour, puis choisissez Lancer les processus d'indexation pour démarrer tous les processus.

    • Cliquez sur l'icône Fonctions du processus Distributeur de mise à jour, puis choisissez Arrêter les processus d'indexation pour arrêter tous les processus.

    • Cliquez sur l'icône Fonctions du processus Distributeur de mise à jour, puis choisissez Redémarrer les processus d'indexation pour redémarrer tous les processus.

    • Cliquez sur le nom d'un processus, puis sur le bouton Démarrer du processus sur la page Propriétés pour démarrer un seul processus.

    • Cliquez sur le nom d'un processus, puis sur le bouton Arrêter du processus sur la page Propriétés pour arrêter un seul processus.

 

Astuce :
Vous pouvez également démarrer, arrêter ou redémarrer tous les processus d'indexation en cliquant sur le nom d'un processus, puis en cliquant sur le bouton Démarrer, Arrêter ou Redémarrer de la page Propriétés.
