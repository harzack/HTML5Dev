Pour ajouter un processus Extracteur Entreprise :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le gestionnaire de flux de données auquel vous voulez ajouter le processus Extracteur Entreprise.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Extracteur Entreprise dans le menu Ajouter un élément.

  4. Sur la page Ajouter Extracteur Entreprise, saisissez un nom pour le processus Extracteur Entreprise dans le champ Nom.

  5. Pour fournir une description du processus Extracteur Entreprise dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci d'un serveur d'administration qui exécute un hôte Content Server principal. Le processus Extracteur Entreprise doit être exécuté sur un hôte Content Server principal, c'est-à-dire un hôte qui exécute un serveur. Pour plus d'informations sur les installations Content Server principales, reportez-vous au Guide d'installation Content Server.

  7. Pour permettre à Content Server de surveiller le statut de ce processus, cochez la case Activer la gestion du système.

  8. Pour spécifier le répertoire d'exécution du processus Extracteur Entreprise, saisissez le chemin absolu du répertoire dans le champ Répertoire de lancement. Ce répertoire doit être le répertoire Content Server_accueil/bin du serveur d'administration qui exécute le processus Extracteur Entreprise.

  9. Dans le champ Ecrire les données dans, saisissez le chemin absolu de l'IPool (relatif au serveur d'administration sur lequel le processus Extracteur s'exécute) dans lequel vous souhaitez que le processus Extracteur Entreprise écrive les données qu'il extrait de la base de données Content Server.

  10. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, les noms des processus apparaissent dans la liste déroulante Processus. Dans cette liste, cliquez sur le processus que devra précéder le processus Extracteur Entreprise dans le flux de données.

     

    Astuce :
    Dans la plupart des flux de données Extracteur Entreprise, le processus Extracteur Entreprise précède un processus Conversion de documents. Dans un flux de données d'indexation de gros volumes, le processus Extracteur Entreprise peut précéder un processus de conversion de documents ou un processus de fusion.

     

  11. Dans la section Options de lancement, cliquez sur Programmé dans la liste déroulante, sur le bouton radio Chaque, puis sur des valeurs des listes déroulantes pour sélectionner l'intervalle d'exécution du processus Extracteur Entreprise.

  12. Dans la section Options d'arrêt, cliquez sur le bouton radio Terminer, puis saisissez -1 dans le champ Valeur maximale du code de sortie correcte. Le processus Extracteur Entreprise se termine alors dès qu'il s'arrête. Le champ Valeur maximale du code de sortie correcte spécifie le numéro de code d'erreur le plus élevé pour lequel le serveur d'administration redémarre automatiquement le processus Extracteur Entreprise en cas d'échec. La valeur du champ Valeur maximale du code de sortie correcte est fixée par défaut à -1, ce qui signifie que le serveur d'administration ne redémarrera pas ce processus en cas d'échec.

  13. Cliquez sur le bouton Ajouter.
