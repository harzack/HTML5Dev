Lorsque vous ajoutez des processus Extracteur Entreprise ou Conversion de documents multiples à un flux de données, vous devez également ajouter des processus de fusion. Ces processus regroupent les messages IPool de plusieurs processus et les inscrivent dans un seul IPool pour le processus de flux de données suivant.

Les flux de données pour l'indexation de gros volumes peuvent être configurés de différentes façons, selon les besoins particuliers de chaque site. Le schéma suivant illustre un flux de données avec deux processus Extracteur Entreprise.
