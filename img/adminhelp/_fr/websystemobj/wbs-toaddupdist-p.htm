Pour ajouter un processus Distributeur de mise à jour :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier de la source de données à laquelle vous souhaitez ajouter un processus Distributeur de mise à jour.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Sur la page Gestionnaire de flux de données, cliquez sur Distributeur de mise à jour dans le menu Ajouter un élément.

  4. A l'étape de la page Planification dans l'assistant d'ajout de distributeur de mise à jour, consultez les informations affichées, puis cliquez sur le bouton Suivant(e).

  5. A l'étape Distributeur de mise à jour, procédez de la façon suivante :

    • Dans la section Distributeur de mise à jour, saisissez un nom pour le distributeur de mise à jour dans le champ Nom. Pour démarrer le distributeur de mise à jour et chaque moteur d'indexation une fois qu'ils sont créés, sélectionnez la case à cocher Démarrer automatiquement.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le Distributeur de mise à jour.

    • Saisissez un numéro de port dans le champ Port Admin pour spécifier le port sur lequel le Distributeur de mise à jour sera exécuté sur un serveur d'administration.

    • Dans la section Infos sur Interchange Pool, cliquez dans la liste déroulante Lire les données à partir de sur le répertoire IPool à partir duquel le processus Distributeur de mise à jour lira les données.

    • Dans l'une des listes déroulantes Ecrire les données dans, cliquez sur les répertoires IPool dans lesquels le processus Distributeur de mise à jour écrira les données.

  6. Cliquez sur le bouton Suivant.

  7. A l'étape Moteurs d'indexation, effectuez les opérations suivantes pour chaque moteur d'indexation :

    • Dans le champ Nom, saisissez un nom pour le moteur d'indexation.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le moteur d'indexation.

    • Saisissez un numéro de port dans le champ Port Admin pour spécifier le port sur lequel le moteur d'indexation sera exécuté sur un serveur d'administration.

    • Dans le champ Port du serveur, saisissez le numéro du port que les processus d'échange utiliseront pour communiquer avec le moteur d'indexation.

    • Dans le champ Répertoire d'indexation, saisissez le chemin absolu de l'index de la partition.

  8. Cliquez sur le bouton Suivant.

  9. A l'étape Résumé, consultez les informations que vous avez fournies pour le nouveau processus Distributeur de mise à jour et chaque moteur d'indexation, puis cliquez sur le bouton Terminer pour ajouter les processus.

 

Note :
Vous pouvez vérifier la disponibilité d'un numéro de port en cliquant sur le lien Vérifier le port. Vous pouvez également ajouter un processus de distributeur de mise à jour à une source de données spécifique lorsque vous affichez le schéma de partition de la source de données.
