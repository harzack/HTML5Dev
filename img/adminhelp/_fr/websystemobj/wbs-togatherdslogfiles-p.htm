Pour rassembler des fichiers journaux de recherche de la source de données :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du dossier source de données Entreprise, puis sélectionnez Maintenance.

  2. Sur la page Maintenance des sources de données, cliquez sur le bouton radio Rassembler les fichiers journaux pour cette source de données.

  3. Cliquez sur le bouton OK.

  4. Sur la page Rassembler les fichiers journaux pour cette source de données, cliquez sur une valeur dans la liste déroulante Nombre de lignes à inclure à partir de la fin de chaque fichier journal.

  5. Sélectionnez la case à cocher Inclure les fichiers INI de recherche, les fichiers journaux DCS et les fichiers de configuration du serveur d'administration pour rassembler ces fichiers.

  6. Cliquez sur le bouton OK.

  7. Sur la page Rassembler les fichiers journaux pour cette source de données, cliquez sur le bouton Télécharger, puis indiquez l'emplacement de téléchargement des fichiers archivés.

  8. Cliquez sur le bouton OK.
