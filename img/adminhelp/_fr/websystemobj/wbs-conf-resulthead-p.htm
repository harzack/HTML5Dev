Pour configurer la recherche En-tête des résultats de recherche :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Configurer les options de recherche.

  2. Sur la page Configurer les options de recherche, dans la section En-tête des résultats de recherche, consultez les informations dans le champ Légende.

  3. Dans le champ Modèle de compte des résultats, saisissez les chaînes à afficher pour différentes classes d'utilisateurs :

    • Estimation : le modèle pour générer la chaîne d'en-tête pour les utilisateurs réguliers ayant consulté des comptes de résultats de recherches estimés par défaut.

    • Exact : le modèle pour les utilisateurs dotés d'autorisations Administration système, ainsi que ceux inclus dans le groupe de privilèges « Afficher les Comptes de résultats non autorisés ». Ces utilisateurs peuvent voir le compte de résultats de recherche réel du moteur de recherche.

    Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut. Pour plus d'informations, voir « Définition des comptes de résultats de recherche » dans Gestion des processus de recherche.

  4. Cliquez sur le lien Modifier restrictions Afficher les Comptes de résultats non autorisés pour modifier les utilisateurs ou les groupes que vous souhaitez ajouter ou modifier.

  5. Cliquez sur le bouton Mettre à jour.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.
