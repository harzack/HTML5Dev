Content Server propose des systèmes complets d'aide en ligne utilisateur et administrateur qui permettent aux utilisateurs et aux administrateurs Content Server d'accéder aux informations sur les tâches qu'ils réalisent. Vous pouvez générer un index des fichiers d'aide en ligne utilisateur et administrateur si vous souhaitez que les utilisateurs Content Server puissent rechercher le texte de l'aide depuis la page Recherche ou directement depuis le système d'aide en ligne pour utilisateur ou administrateur.

 

Note :
Seuls l'utilisateur Admin et les autres utilisateurs Content Server disposant des autorisations adaptées peuvent rechercher ou accéder au système d'aide en ligne administrateur.

 

Content Server crée l'index de l'aide en ligne utilisateur et administrateur sur l'hôte Content Server principal, représenté par le raccourci (généralement Default) de son serveur d'administration.

 

Astuce :
Si vous ajoutez, désinstallez ou mettez à niveau un module après avoir créé l'index de l'aide en ligne utilisateur, l'index de l'aide en ligne doit être actualisé. Pour en savoir plus, voir Mise à jour de l`aide en ligne utilisateur et administrateur.
