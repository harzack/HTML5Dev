Pour configurer une partition :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier source de données contenant la partition à configurer.

  2. Cliquez sur le lien préfixe_processus Schéma de partition.

  3. Sur la page Schéma de partition, cliquez sur le nom d'une partition.

  4. Dans l'onglet Infos spécifiques de la page Propriétés de la partition, dans la section Paramètres, modifiez les paramètres suivants :

    • Mode : indique le mode de la partition.

    • Taille maximale du contenu du disque : indique la quantité maximale d'espace disque que le contenu des documents peut occuper dans l'index de la partition.

    • Taille maximale de la mémoire des métadonnées : indique la quantité maximale d'espace mémoire que les métadonnées des documents peuvent occuper dans l'espace mémoire total alloué à l'index d'une partition.

    • Taille de la mémoire de l'accumulateur : indique la quantité d'espace mémoire alloué à l'accumulateur de contenu.

  5. Dans la section Paramètres de rééquilibrage en mode Lecture/Ecriture, modifiez les paramètres suivants :

    • Démarrer le rééquilibrage à partir de, qui permet de définir le pourcentage de mémoire utilisée auquel aucune mise à jour n'est acceptée et tous les nouveaux objets sont envoyés sur d'autres partitions, quand une partition est en mode Lecture / Ecriture.

    • Arrêter le rééquilibrage au-dessous de, qui permet de définir le pourcentage de mémoire utilisée auquel les mises à jour sont acceptées et les nouveaux objets sont toujours envoyés sur d'autres partitions.

    • Mises à jour uniquement au-dessus de, qui permet de définir le pourcentage de mémoire utilisée auquel les nouveaux objets sont envoyés sur d'autres partitions, quand une partition est en mode Lecture / Ecriture.

  6. Cliquez sur le bouton Mettre à jour.

 

Note :
Vous devez redémarrer les processus d'indexation et de recherche (le processus Distributeur de mise à jour, les moteurs d'indexation, les fédérateurs de recherche et les moteurs de recherche) avant que les paramètres que vous configurez pour une partition prennent effet.
