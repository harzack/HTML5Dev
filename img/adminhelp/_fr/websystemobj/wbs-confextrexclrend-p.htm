La page Configurer les types de présentation vous permet de spécifier les types de présentation que vous souhaitez extraire et indexer. Une présentation est une version d'un document dans un format différent. Par exemple, une version PDF d'un document Microsoft Word est une présentation du document Microsoft Word d'origine. Un type de présentation est une chaîne de caractères.

Etant donné que les documents source des présentations sont indexés, l'indexation des présentations est souvent redondante. Si vous autorisez l'extraction et l'indexation de certains types de présentation, l'utilisateur qui recherche un document peut accéder à la source et à la présentation correspondante dans les résultats de recherche. Ceci permet aux utilisateurs de choisir le format d'affichage du document.

 

Note :
La page Configurer les types de présentation n'est disponible que si le module Renditions est installé. Pour en savoir plus, voir « Content Server Services and Modules » dans OpenText Content Server - Installation Guide (LLESCOR-IGD).

Indexer une présentation peut s'avérer très utile si l'objet principal est une image. Par exemple, lorsqu'une version a été convertie en texte par un processus Reconnaissance optique de caractères (OCR), la présentation est plus efficace pour envoyer les données vers l'index.

 

Pour configurer des types de présentation exclus :

  1. Sur Pour définir les paramètres généraux de l'Extracteur, dans le champ Types de présentation de la section Paramètres d'exclusion, cliquez sur le lien liste de modification/révision.

  2. Sur la page Configurer les types de présentation exclus, sélectionnez la case à cocher Tout sélectionner, ou sélectionnez des types de présentation individuels pour spécifier ceux que l'Extracteur doit exclure des opérations d'indexation de recherche.

  3. Cliquez sur Mettre à jour pour utiliser les types de présentation exclus que vous avez définis, ou sur Annuler.
