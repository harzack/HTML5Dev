Après avoir ajouté un processus T, vous pouvez afficher et modifier la configuration depuis la page Propriétés T du processus T. Outre les paramètres spécifiés durant la création du processus, vous pouvez afficher et modifier les éléments suivants :

Paramétrage Action
Etat Informations sur le statut actuel du processus T.
Serveur d'administration Le nom du serveur d'administration sur l'hôte à partir duquel le processus T doit s'exécuter.
Activer la gestion du système Sélectionnez cette option pour permettre à Content Server de surveiller le statut du processus T. Lorsque cette option est activée, si le processus retourne des messages d'erreur, Content Server les enregistre dans la base de données.
 

Astuce :
Pour plus d'informations sur la configuration de Content Server en vue de l'envoi d'e-mails d'alerte lorsqu'un processus de flux de données rencontre des erreurs, reportez-vous à la rubrique Pour activer la recherche d'erreurs et l'envoi d'e-mails.

 
Mode de sortie
Sélectionnez :
•

Copier les messages IPool dans toutes les sorties, pour envoyer tous les messages IPool à cinq processus de sortie IPool maximum

•

Distribuer les objets IPool (contenu de message) parmi toutes les sorties, pour distribuer tous les messages IPool parmi cinq processus de sortie IPool maximum

 

Note :
Lorsque vous sélectionnez le mode de sortie de distribution, les champs Région de filtre et Valeur de filtre sont désactivés dans la zone Ecrire les données dans.

 
Ligne de commande
Modèle de commande
Consultez et modifiez le modèle de commande. Il s'agit d'une commande générée par le système qui indique au serveur d'administration comment créer la ligne de commande du processus T. Elle est enregistrée dans le fichier otadmin.cfg du serveur d'administration sur l'hôte.
 

Important :
Ne modifiez pas cette valeur, sauf si le service d'assistance clientèle d'OpenText vous le demande.

 
Ligne de commande La ligne de commande utilisée par le serveur Enterprise pour exécuter le processus T.
iPools
Cliquez sur le lien Plus de détails pour afficher les processus Lire les données à partir de et Ecrire les données dans utilisés dans le processus T.
Vous pouvez également supprimer le processus lié en cliquant sur le bouton Enlever ce lien , ou ajouter un processus lié en cliquant sur le bouton Ajouter un lien . Vous pouvez afficher davantage d'informations sur chaque processus en cliquant sur le bouton Infos sur le lien .
Options de démarrage
Programme
Sélectionnez :
•

Manuel, si vous souhaitez que le processus T traite les données lorsque vous le lancez vous-même

•

Programmé, si vous souhaitez que le processus T s'exécute automatiquement

Pour A cette heure, définissez l'heure dans les listes déroulantes, puis cliquez sur les jours de la semaine auxquels le processus T doit s'exécuter.
Pour Intervalle, sélectionnez l'intervalle selon lequel le processus T doit s'exécuter.
Pour Attente, définissez les valeurs des listes déroulantes afin de sélectionner l'intervalle entre deux exécutions du processus T.
Sélectionnez Continuel pour que le processus T s'exécute continuellement. En cas d'arrêt inopiné du processus, son serveur d'administration le redémarre automatiquement.
Options de démarrage
Valeur maximale du code de sortie correcte Indiquez le numéro de code d'erreur le plus élevé pour lequel le serveur d'administration redémarre automatiquement le processus T en cas d'échec.
 

Astuce :
Si vous spécifiez la valeur -1, le processus T se termine immédiatement lorsqu'il s'arrête et le service d'administration ne le redémarre pas en cas d'échec.

 
Ne modifiez pas ce paramètre, sauf si le service d'assistance clientèle d'OpenText vous le demande.
Actions
Processus Cliquez sur Démarrer ou Arrêter, pour contrôler le processus T.
Option Cliquez sur Resynchroniser pour réécrire l'entrée du Modèle de commande dans le fichier otadmin.cfg du serveur d'administration hôte.

 

Note :
Seules les informations ajoutées à Content Server après modification des paramètres du processus T sont envoyées au processus ajouté au flux de données. Pour envoyer les données d'index actuelles au processus, vous devez effectuer une réindexation. Pour plus d'informations sur la réindexation, voir Vérification du contenu et réindexation.

 

Pour plus d'informations sur les paramètres configurés durant la création d'un processus T, voir Ajout d'un processus T. Pour obtenir des instructions sur la configuration d'un processus T, voir Pour configurer le processus T.
