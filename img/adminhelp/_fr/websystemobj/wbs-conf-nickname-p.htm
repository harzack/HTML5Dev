Pour configurer la recherche de surnoms :

  1. Dans la section Administration des apparences de la page d'administration, cliquez sur le lien Administration de la barre de recherche.

  2. Sur la page Administration de la barre de recherche, cliquez sur le bouton Modifier pour une configuration de barre de recherche.

  3. Sur la page Modification de la barre de recherche, dans la section Surnom, sélectionnez la case à cocher Inclure.

  4. Dans le champ Texte de l'invite, saisissez le texte que vous souhaitez voir apparaître dans la liste déroulante Sélectionner le type de recherche au dessus de la barre de recherche. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

  5. Dans la section Composant, cochez la case Afficher, puis sélectionnez l'action dans la liste déroulante.

  6. Cliquez sur le bouton OK.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.

     

Pour plus d'informations sur les options de surnom, reportez-vous à la rubrique Surnom.
