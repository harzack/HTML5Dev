Pour configurer la recherche de texte intégral :

  1. Dans la section Administration des apparences de la page d'administration, cliquez sur le lien Administration de la barre de recherche.

  2. Sur la page Administration de la barre de recherche, cliquez sur le bouton Modifier pour le Texte intégral.

  3. Sur la page Modification de la barre de recherche, cochez la case Inclure.

  4. Dans le champ Texte de l'invite, saisissez le texte que vous souhaitez voir apparaître dans la barre de recherche. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

  5. Dans la section Composant, cochez la case Afficher située en regard de chacun des composants de recherche à afficher par défaut, puis cliquez sur le paramètre du composant par défaut. Pour plus d'informations sur les liens Configurer, reportez-vous à la rubrique Configuration de la liste déroulante Types pouvant faire l'objet de recherches, Configuration de la liste déroulante Créé par, Configuration de la liste déroulante Une fois les modifications apportées ou Configuration de la liste déroulante Sélection de vue de recherche.

 

Note :
Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.

 

Pour plus d'informations sur les options de texte intégral, reportez-vous à la rubrique Configuration de la recherche Texte intégral.
