Pour définir les niveaux de consignation de cette source de données :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du dossier source de données Entreprise, puis sélectionnez Maintenance.

  2. Sur la page Maintenance des sources de données Content Server, cliquez sur le bouton radio Définir les niveaux de consignation pour cette source de données.

  3. Cliquez sur le bouton OK.

  4. Sur la page Définir les niveaux de consignation pour cette source de données, cliquez sur une valeur dans la liste déroulante Niveau de consignation :

    • Niveau d'informations : tous les types de messages.

    • Niveau de statut : les messages de statut périodiques, les messages d'avertissement et tous les messages d'erreur.

    • Niveau d'avertissement : les messages d'avertissement et tous les messages d'erreur.

    • Niveau d'erreur : les messages d'erreur typiques et graves.

    • Niveau d'erreur grave : les messages d'erreur graves.

  5. Sélectionnez les types de processus dont vous souhaitez définir les niveaux de consignation dans la liste déroulante :

    • Tous : les sources de données de recherche et d'indexation.

    • En cours de recherche : uniquement les sources de données en cours de recherche.

    • Indexation : uniquement les sources de données en cours d'indexation.

  6. Sélectionnez la case à cocher Inclure le serveur DCS pour définir également le niveau de journal pour le service de conversion de documents.

  7. Cliquez sur le bouton OK.

La page est actualisée de manière à afficher un rapport pour chaque Objet système concerné, le niveau de consignation sélectionné et le nom du nœud.
