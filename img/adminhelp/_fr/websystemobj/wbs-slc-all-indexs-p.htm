Pour créer une vue de recherche à partir de tous les index :

  1. Dans le menu Outils, choisissez Rechercher.

  2. Sur la page Recherche, cliquez sur Tous les mots dans la liste déroulante Rechercher.

  3. Saisissez le texte suivant (y compris les guillemets) dans le champ de la section Texte intégral : "<otdoc>"

  4. Cliquez sur le lien Vues de recherche dans la section Options avancées pour afficher les vues de recherche disponibles sur votre site Content Server.

  5. Maintenez la touche CTRL enfoncée tout en cliquant sur le nom de chaque vue de recherche dans la liste Vues de recherche.

  6. Cliquez sur le bouton Enregistrer la requête.

  7. Sur la page Ajouter Requête, saisissez un nom unique pour la vue de recherche dans le champ Nom.

  8. Pour afficher une description de la vue de recherche dans l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  9. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  10. Cliquez sur le bouton Parcourir Content Server, sélectionnez le dossier système et cliquez sur le lien Sélectionner dans la colonne Actions du Dossier Vues de recherche.

  11. Cliquez sur le bouton Ajouter.

 

Note :
Vous créez une vue de recherche qui comprend tous les index existants sur un site Content Server lorsque vous voulez appliquer des requêtes à tous les index disponibles en même temps.
