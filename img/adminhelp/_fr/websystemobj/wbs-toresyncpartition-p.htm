Pour resynchroniser une partition :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier source de données contenant la partition à resynchroniser.

  2. Cliquez sur le lien préfixe_processus Schéma de partition.

  3. Sur la page Schéma de partition, cliquez sur l'icône Fonctions d'une partition, choisissez Propriétés, puis Infos spécifiques.

  4. Dans l'onglet Infos spécifiques de la page Propriétés Partition, cliquez sur le bouton Resynchroniser.
