Pour créer une vue de recherche pour un dossier Content Server et son contenu :

  1. Allez à la racine du dossier Content Server pour lequel vous voulez créer une vue de recherche.

  2. Dans le champ Recherche, en haut de la page, entrez un astérisque (*).

  3. Cliquez sur A partir d'ici dans la liste déroulante, puis cliquez sur le bouton Aller.

  4. Sur la page Résultats de recherche, cliquez sur le bouton Enregistrez votre recherche.

  5. Sur la page Ajouter Requête, saisissez un nom unique pour la vue de recherche dans le champ Nom.

  6. Pour afficher une description de la vue de recherche dans l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  7. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  8. Cliquez sur le bouton Parcourir Content Server, sélectionnez le dossier système et cliquez sur le lien Sélectionner dans la colonne Actions du Dossier Vues de recherche.

  9. Cliquez sur le bouton Ajouter.

 

Note :
Vous créez une vue de recherche à partir d'un dossier Content Server lorsque vous voulez autoriser les utilisateurs Content Server à formuler des requêtes dans le dossier et son contenu.
