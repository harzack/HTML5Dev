Pour ajouter un processus Conversion de documents :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le flux de données auquel vous voulez ajouter un processus Conversion de documents.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Conversion de documents dans le menu Ajouter un élément.

  4. Sur la page Ajouter Conversion de documents, saisissez un nom pour le processus Conversion de documents dans le champ Nom.

  5. Pour fournir une description du processus Conversion de documents dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Conversion de documents doit s'exécuter.

  7. Dans le champ Lire les données à partir de, saisissez le chemin absolu de l'IPool à partir duquel le processus Conversion de documents devra lire les données.

  8. Si le flux de données contient déjà au moins un processus producteur ou intermédiaire, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que suivra le processus Conversion de documents dans le flux de données.

     

    Astuce :
    Dans la plupart des flux de données, le processus Conversion de documents suit un processus producteur, tel qu'un processus Extracteur Entreprise ou un processus Directory Walker.

     

  9. Dans le champ Ecrire les données dans, saisissez le chemin absolu de l'IPool dans lequel vous voulez que le processus Conversion de documents enregistre les données converties.

  10. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que précèdera le processus Conversion de documents dans le flux de données.

     

    Astuce :
    Dans la plupart des flux de données, le processus Conversion de documents précède le processus Distributeur de mise à jour.

     

  11. Dans la section Options de lancement, définissez les paramètres de lancement du processus Conversion de documents.

  12. Dans la section Options d'arrêt, définissez les paramètres d'arrêt du processus Conversion de documents.

  13. Cliquez sur le bouton Ajouter.
