La page Administration de la barre de recherche vous permet de personnaliser l'apparence de la barre de recherche Content Server, disponible sur la plupart des pages Content Server. Vous pouvez visualiser les configurations de barre de recherche disponibles, en ajouter une nouvelle en cliquant sur le bouton Ajouter une nouvelle configuration de barre de recherche ou en modifier les configurations existantes. Vous pouvez également supprimer n'importe quelle configuration de barre de recherche existante, à l'exception de Barre de recherche standard, en cliquant sur le bouton Enlever .

Cliquez sur le bouton Modifier d'une configuration de barre de recherche pour ouvrir la page Modification de la barre de recherche avec les paramètres pour ce mode de barre de recherche. Depuis cette page, vous pouvez définir l'apparence par défaut et les composants disponibles sur la barre de recherche Content Server.

 

Astuce :
Lorsque vous créez une configuration de la barre de recherche personnalisée, vous pouvez l'utiliser en créant tout d'abord une apparence dans un dossier, puis dans la section Composants Content Server en sélectionnant la barre de recherche personnalisée. Pour plus de détails, voir Pour afficher une barre de recherche personnalisée dans un dossier spécifique et Utilisation des apparences.

 

Vous pouvez modifier les configurations standard de recherche système, de même que celles que vous avez créées, comme décrit dans le tableau suivant.

Nom Description
Activer le lien Derniers résultats Sélectionnez la case à cocher Activer le lien Derniers résultats pour afficher un lien pour permettre à vos utilisateurs de revenir à leur dernière page Résultats de recherche.
Barre de recherche standard La configuration par défaut de la barre de recherche qui fournit la recherche en Texte intégral pour élaborer les Requêtes contenant des mots clés ou des Requêtes complexes constituées à l'aide du langage LQL (Live Query Language).
Simple (exemple)
Une configuration simplifiée basée sur une version du formulaire de recherche enregistré.
Sélection de vue de recherche (exemple) Une configuration qui permet de masquer la liste déroulante Vue de recherche sur la barre de recherche et de déterminer l'affichage et l'ordre des vues de recherche.
Langage naturel (exemple) Une configuration qui permet aux utilisateurs de saisir des requêtes sous forme de question ou sur une ou plusieurs lignes de texte. A partir de ces critères, Content Server détermine les critères et les mots clés de la recherche, puis lance la recherche.
