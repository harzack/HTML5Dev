Pour ajouter un gestionnaire de flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien Dossier de la source de données correspondant au dossier source de données dans lequel vous souhaitez créer un gestionnaire de flux de données.

  2. Cliquez sur Gestionnaire de flux de données dans le menu Ajouter un élément.

  3. Sur la page Ajouter Gestionnaire de flux de données, saisissez un nom pour le gestionnaire de flux de données dans le champ Nom.

  4. Pour fournir une description du gestionnaire de flux de données dans l'onglet Général de la page Propriétés, saisissez un texte descriptif dans le champ Description.

  5. Pour modifier les catégories associées au gestionnaire de flux de données, cliquez sur le bouton Modifier en regard du champ Catégories.

  6. Cliquez sur le bouton Ajouter.
