Un flux de données est un ensemble de processus liés qui traitent les données à partir d'une seule source dans Content Server. Vous pouvez configurer les sept types de processus de flux de données suivants : le processus Extracteur Entreprise, le processus Directory Walker, le processus XML Activator, le processus Conversion de documents, le processus Importateur, le processus de fusion et les proxies. Chaque flux de données dispose d'un gestionnaire de flux de données, qui gère tous les processus constituant le flux de données. Un gestionnaire de flux de données est aussi un dossier qui contient tous les processus d'un flux de données.

Lorsque vous avez ajouté les processus constituant le flux de données, vous pouvez configurer ses processus.
