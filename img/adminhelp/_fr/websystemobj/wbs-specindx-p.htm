Pour configurer un moteur d'indexation :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à une source de données contenant le moteur d'indexation à configurer.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Sur la page Gestionnaire de flux de données, cliquez sur l'icône Fonctions du processus Distributeur de mise à jour et sélectionnez Arrêter les processus d'indexation.

  4. Cliquez sur l'icône Fonctions du processus Distributeur de mise à jour, sélectionnez Propriétés, puis Moteurs d'indexation.

  5. Dans l'onglet Moteurs d'indexation de la page Moteurs d'indexation, cliquez sur le lien du nom du moteur d'indexation à configurer.

  6. Dans l'onglet Spécifique de la page Propriétés du moteur d'indexation, modifiez les paramètres du moteur d'indexation.

  7. Cliquez sur le bouton Mettre à jour.

  8. Sur la page Gestionnaire de flux de données, cliquez sur l'icône Fonctions du moteur d'indexation configuré, puis choisissez Lancer les processus d'indexation.

 

Astuce :
Vous pouvez également configurer les moteurs d'indexation lorsque vous affichez le schéma de partition auquel ils appartiennent.
