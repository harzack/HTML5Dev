Pour ajouter une règle de contrôle basée sur des flux de données bloqués :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du lien préfixe_processus Dossier de la source de données, choisissez Propriétés, puis Règles de contrôle.

  2. Sur la page Règles de contrôle, cliquez sur le bouton Ajouter une nouvelle règle de contrôle.

  3. Sur la page Ajouter une nouvelle règle de contrôle, saisissez un nom pour la règle dans le champ Nom.

  4. Cliquez sur Flux de données bloqué dans la liste déroulante Modèles. Vous pouvez modifier la description de la règle dans le champ Description.

  5. Dans la section Conditions, saisissez un nombre dans le champ Si bloqué pendant plus de, puis cliquez sur les minutes, heures ou jours dans la liste déroulante.

  6. Dans la liste déroulante élément, cliquez sur la totalité du flux de données. Dans la section Rapports, vous pouvez entrer un message à envoyer avec le rapport dans le champ Message.

  7. Cliquez sur le bouton Mettre à jour.
