Pour activer ou désactiver le journal du serveur et définir les options d'enregistrement de threads :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de débogage.

  2. Sur la page Configurer les paramètres du débogage, cliquez sur l'une des valeurs suivantes dans la liste déroulante Niveau de débogage Content Server :

    • 1, qui permet la création de fichiers journaux de niveau 1 (threadn.out, sockserv1.out et llserver.out) ;

    • 2, qui permet la création de fichiers journaux de niveau 2 (niveau 1 + receivern.out) ;

    • 11, qui permet la création de fichiers journaux de niveau 11 (niveau 2 + llclientnnn.out, servletclientnnn.out, llindexupdatennn.out et indexupdateOutnnn.out) ;

    • 0, qui désactive la journalisation du débogage.

  3. Pour ajouter des informations sur les séquences (temps nécessaire à la journalisation) dans les enregistrements de thread, cochez la case Consigner les séquences Content Server.

  4. Pour ajouter des informations sur les variables d'environnement aux journaux de thread, cochez la case Journal détaillé.

  5. Pour ajouter des informations sur les actions des objets système, y compris sur les actions liées aux requêtes de recherche et aux résultats, cochez la case Ajouter la recherche et l'enregistrement de l'objet système aux journaux de thread.

  6. Cliquez sur le bouton Mettre à jour.

  7. Relancez le serveur Content Server sur l'hôte Content Server principal.
