Pour configurer les options d'arrêt d'un processus de flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier source de données qui contient le processus dont vous souhaitez définir les options d'arrêt.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur l'icône Fonctions du processus dont vous voulez définir les options d'arrêt, puis sélectionnez Arrêter.

  4. Cliquez sur le nom du processus dont vous voulez définir les options d'arrêt.

  5. Pour interrompre le processus aussitôt qu'il s'arrête (automatiquement ou manuellement), cliquez sur le bouton radio Terminer.

  6. Pour envoyer un message d'arrêt au processus lorsqu'il s'arrête (automatiquement ou manuellement), cliquez sur le bouton radio Port des messages d’arrêt, puis saisissez un numéro de port disponible (entre 1025 et 65535) dans le champ correspondant.

  7. Pour exécuter un fichier exécutable lorsque le processus s'arrête (automatiquement ou manuellement), cliquez sur le bouton radio Exécutable d'arrêt, puis saisissez le chemin absolu du fichier exécutable (tel qu'il est mappé/monté sur l'hôte du serveur d'administration du processus) dans le champ correspondant.

  8. Pour changer la valeur maximale du code de sortie pour laquelle le serveur d'administration redémarrera automatiquement ce processus en cas d'échec, saisissez un numéro de code d'erreur dans le champ Valeur maximale du code de sortie correcte. Ne modifiez pas cette option, sauf si le service d'assistance clientèle d'OpenText vous le demande.

  9. Cliquez sur le bouton Mettre à jour.

  10. Cliquez sur l'icône Fonctions du processus, puis sélectionnez Démarrer.
