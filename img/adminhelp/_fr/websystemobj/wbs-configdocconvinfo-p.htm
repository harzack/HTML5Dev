Pour configurer les informations de conversion de documents :

  1. Dans le champ Répertoire de lecture, saisissez le chemin absolu du répertoire à partir duquel vous voulez que le processus Conversion de documents lise les données.

  2. Dans le champ Répertoire d'écriture, saisissez le chemin absolu du répertoire dans lequel vous voulez que le processus Conversion de documents enregistre ses données. OpenText vous recommande de choisir un répertoire d'écriture sur un lecteur local de l'hôte du serveur d'administration.

  3. Dans la liste déroulante Hôte de la section Intermédiaire : Conversion de documents, cliquez sur le raccourci du serveur d'administration dont l'hôte sera utilisé pour exécuter le processus Conversion de documents.
