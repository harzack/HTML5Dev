Pour ajouter une règle de contrôle pour la création de partitions selon la date :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du lien Entreprise Dossier de la source de données, choisissez Propriétés, puis Règles de contrôle.

  2. Sur la page Règles de contrôle, cliquez sur le bouton Ajouter une nouvelle règle de contrôle .

  3. Sur la page Ajouter une nouvelle règle de contrôle, saisissez un nom pour la règle dans le champ Nom.

  4. Cliquez sur Création de partitions selon la date dans la liste déroulante Modèles. Vous pouvez modifier la description de la règle dans le champ Description.

  5. Dans la section Conditions, cliquez sur la date et l'heure de déclenchement dans les listes déroulantes Modifier la date de déclenchement au. Dans la section Actions, vous pouvez activer l'une des cases à cocher suivantes :

    • Descendre les partitions « Lecture/Ecriture » sur « Mettre à jour uniquement »

    • Descendre les partitions « Mettre à jour uniquement » sur « Lecture seule »

     

    Avertissement
    Lorsque vous modifiez une partition sur En lecture seule, le moteur d'indexation pour cette partition s'arrête et cesse d'indexer tout élément mis à jour ou supprimé.

     

  6. Cliquez sur le nombre de partitions souhaité dans la liste déroulante Créer.

  7. Saisissez le modèle de nom de la partition dans le champ Modèle de nom de la partition.

  8. Saisissez le modèle de répertoire de la partition dans le champ Modèle de répertoire de la partition.

  9. Saisissez le nom du processus de sauvegarde pour la partition dans le champ Modèle de nom du processus de sauvegarde.

  10. Saisissez le nom du répertoire de sauvegarde dans le champ Modèle de nom du répertoire de sauvegarde.

  11. Cliquez sur la durée devant s'écouler avant la date du prochain déclenchement dans la liste déroulante Intervalle. Dans la section Rapports, vous pouvez entrer un message à envoyer avec le rapport dans le champ Message.

  12. Cliquez sur le bouton Mettre à jour.
