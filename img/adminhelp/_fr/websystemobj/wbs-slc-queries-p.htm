Pour créer une vue de recherche à partir d'une requête :

  1. Dans le menu Outils, cliquez sur Rechercher.

  2. Sur la page Recherche, spécifiez les critères de recherche.

  3. Cliquez sur le bouton Enregistrer la requête.

  4. Sur la page Ajouter Requête, saisissez un nom unique pour la vue de recherche dans le champ Nom.

  5. Pour afficher une description de la vue de recherche dans l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  6. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  7. Cliquez sur le bouton Parcourir Content Server, sélectionnez le dossier système et cliquez sur le lien Sélectionner dans la colonne Actions du Dossier Vues de recherche.

  8. Cliquez sur le bouton Ajouter.

 

Note :
Vous créez une vue de recherche à partir d'une requête lorsque vous voulez appliquer des requêtes à l'ensemble de données défini par les critères de recherche dans la vue de recherche.
