Pour créer un index des rubriques de l'aide en ligne administrateur :

  1. Sur la page Volume des objets système, cliquez sur Sources de données de l'aide administrateur dans le menu Ajouter un élément. Si l'index de l'aide en ligne administrateur existe déjà, Sources de données de l'aide administrateur n'apparaît pas dans le menu Ajouter un élément. Si vous avez supprimé une source de données Content Server et que vous voulez associer ses requêtes et ses formulaires de recherche enregistrés à cet index d'aide en ligne administrateur, cliquez sur le nom de la vue de recherche de la source de données supprimée dans la liste déroulante Remplacement de vue de recherche.

  2. Dans le champ Répertoire principal, saisissez le chemin absolu du répertoire dans lequel vous voulez créer l'index de l'aide en ligne administrateur. Le répertoire principal par défaut est Content Server_home/index/adminhelp. OpenText vous recommande de choisir un répertoire sur un lecteur de l'hôte Content Server par défaut.

  3. Saisissez un identificateur unique pour tous les objets système associés à cette source de données dans le champ Préfixe du processus. Si vous souhaitez définir le nombre de partitions qui doivent composer cet index, saisissez un nombre dans le champ Partitions.

  4. Cliquez sur le bouton Créer des processus.

  5. Sur la page Statut de la création d'un flux de données, cliquez sur le bouton Continuer.

 

Note :
OpenText vous recommande vivement de supprimer et de recréer le Dossier de la source de données : Aide administrateur à chaque fois que vous ajoutez ou que vous supprimez des lots de langues pour vous assurer que les résultats de recherche de l'aide sont disponibles pour tous les lots de langues installés.
