Pour configurer la recherche Mise en évidence des résultats du résumé :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Configurer les options de recherche.

  2. Sur la page Configurer les options de recherche, dans la section Mise en évidence des résultats du résumé, cliquez sur le bouton radio Activer pour activer la mise en évidence des expressions de requête dans le résumé de documents.

  3. Cliquez sur le bouton Mettre à jour.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.
