La page Modification de la barre de recherche contrôle les paramètres de la barre de recherche Content Server, accessible sur la plupart des pages Content Server.

Toute modification éventuelle de la barre de recherche sera visible par tous vos utilisateurs lorsque vous cliquez sur Appliquer car il s'agit de la configuration par défaut de la barre de recherche.

 

Astuce :
Vous pouvez également enregistrer vos paramètres d'options d'affichage à utiliser en tant que résultats de recherche en cliquant sur le bouton Enregistrer en tant que formulaire.

 

Vous pouvez activer, désactiver et modifier les paramètres par défaut des composants suivants :

  • Paramètres généraux

  • Texte intégral

  • Surnom

  • Requête en langage naturel

Configuration des paramètres généraux

La section Paramètres généraux permet de configurer le mode de recherche, l'apparence du bouton de recherche et son libellé. Elle permet également d'activer un lien Aide sur la barre de recherche.

Le paramètre Mode par défaut permet de déterminer le mode de recherche affiché dans la barre de recherche par défaut, tel que décrit dans le tableau ci-dessous.

Option Description
Texte intégral Permet aux utilisateurs de créer des requêtes qui contiennent des mots clés ou des requêtes complexes LQL (Live Query Language).
Surnom Permet aux utilisateurs de saisir des surnoms comme termes de recherche. Les utilisateurs peuvent attribuer des surnoms à des éléments.
Requête en langage naturel Permet aux utilisateurs de saisir des requêtes sous forme de question ou sur une ou plusieurs lignes de texte. A partir de ces critères, Content Server détermine les critères et les mots clés de la recherche, puis lance la recherche.

Les boutons radio Bouton de recherche permettent de déterminer l'apparence par défaut de la barre de recherche et, le cas échéant, de modifier son libellé.

 

Astuce :
Vous pouvez modifier la langue utilisée pour le Bouton de recherche ainsi que d'autres champs de texte, en saisissant une valeur Xlate au lieu de texte brut. Si vous laissez le texte Recherche par défaut, celui-ci s'affiche dans la langue des utilisateurs. Si vous modifiez le texte, tous les utilisateurs, quelle que soit leur langue, verront le texte tel que vous l'avez saisi.

 

La zone de texte Largeur de la barre de recherche définit la taille d'affichage de la barre de recherche, comprise entre 100 et 400 pixels.

En sélectionnant la case à cocher Lien Aide, un lien Aide est affiché sur la barre de recherche du mode de recherche sélectionné.

Les options et paramètres de la barre de recherche changent selon le mode de recherche que les utilisateurs choisissent. Vous pouvez définir les paramètres par défaut des recherches de texte intégral, de surnom et en langage naturel dans les sections suivantes de la page Modification de la barre de recherche.

Configuration de la recherche Texte intégral

La section Texte intégral détermine l'apparence du texte intégral par défaut dans la barre de recherche.

La case à cocher Inclure permet de déterminer si la recherche en Texte intégral est disponible dans les Options avancées.

Le champ Texte de l'invite permet de contrôler le texte qui s'affiche pour les recherches de type Texte intégral. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

Dans la zone Composant, vous pouvez spécifier les composants de la recherche Texte intégral à afficher par défaut et configurer leur apparence et leur comportement. La zone Composant comprend les paramètres suivants :

  • Mode de recherche

  • Opérateur

  • Types pouvant faire l'objet de recherches (Configurer)

  • Créé par (Configurer)

  • Une fois les modifications apportées (Configurer)

  • Sélection de vue de recherche (Configuration de vue de recherche)

  • Recherche avancée

  • Recherche avancée

Lorsque vous cochez la case Afficher associée à un composant, celui-ci s'affiche par défaut dans la barre de recherche.

Configuration du mode de recherche

Le composant Mode de recherche vous permet de définir le mode de recherche par défaut de la barre de recherche lorsque vous effectuez une recherche de type Texte intégral ou Requête en langage naturel. Content Server renvoie une liste d'éléments correspondant au mode de recherche sélectionné, tel que décrit dans le tableau ci-dessous.

Option Description
Tous les mots Renvoie une liste d'éléments contenant tous les mots de la requête.
N'importe quel mot Renvoie une liste d'éléments contenant n'importe quel mot de la requête.
Phrase exacte Renvoie une liste d'éléments contenant la phrase exacte de la requête si deux mots clés ou plus sont encadrés par des guillemets.
Requête complexe Renvoie les données cibles à récupérer. Les requêtes complexes sont des instructions uniques, formulées en langage LQL, qui vous permettent de cibler avec précision les données que vous souhaitez récupérer. Par exemple, pour rechercher des éléments Content Server contenant le mot rapport mais pas le mot frais, saisissez la requête complexe rapport AND-NOT frais.

Configuration de la liste déroulante Modificateur

Le composant Modificateur vous permet de définir le modificateur par défaut de la barre de recherche lorsque vous effectuez une recherche de type Texte intégral. Les opérateurs permettent aux utilisateurs Content Server d'indiquer la relation qu'ils souhaitent inclure entre le terme saisi et la recherche.

Option Description
Aucun Aucun modificateur n'est indiqué.
Synonymes de Mots tirés de l'entrée du thésaurus pour le terme indiqué.
Relatif à Mots dérivés de la partie principale du terme indiqué.
Recherche phonétique Mots dont la prononciation est similaire à celle du terme indiqué, ce qui peut s'avérer utile lorsque l'utilisateur n'est pas sûr de l'orthographe du terme.
 

Note :
Le modificateur de recherche Recherche phonétique est actuellement pris en charge uniquement en anglais.

 
Mot commençant par Mots commençant par le terme indiqué.
Mot se terminant par Mots se terminant par le terme indiqué.

Configuration de la liste déroulante Types pouvant faire l'objet de recherches

Le composant Types pouvant faire l'objet de recherches permet de définir l'apparence des éléments de menu affichés dans la liste Type d'objet durant une recherche de type Texte intégral. La liste déroulante Type d'objet permet aux utilisateurs de définir les types d'éléments Content Server à rechercher.

Cliquez sur le lien Configurer correspondant au composant Types pouvant faire l'objet de recherches pour ouvrir la page Configurer. La section Eléments de menu vous permet de définir l'apparence de la liste déroulante Type d'objet, tel que décrit dans le tableau ci-dessous.

Option Description
Disponible Répertorie les éléments de menu Type d'objet que vous pouvez déplacer vers la liste Affiché. Les éléments affichés sont inclus dans la liste déroulante Type d'objet sur la barre de recherche.
Vous pouvez également ajouter de nouveaux éléments à la liste Disponible en cliquant sur le bouton Ajouter un nouvel élément de menu , ou supprimer des éléments en les sélectionnant puis en cliquant sur le bouton Enlever l'élément de menu sélectionné .
Affiché(s) Répertorie les éléments Type d'objet qui s'affichent sur la barre de recherche. Vous pouvez retirer des éléments de la liste Affiché ou en ajouter. Vous pouvez également modifier l'ordre des éléments de la liste.
Afficher le texte dans le menu En sélectionnant la case à cocher, le texte est affiché dans la sélection de menu quand un utilisateur Content Server clique sur la liste.

Le cadre Aperçu vous permet de visualiser l'apparence de la liste Affiché avant de soumettre vos modifications.

La section Modifier vous permet de configurer le libellé par défaut, une image, une valeur de retrait et la chaîne de requête LQL de la liste déroulante Type d'objet, tel que décrit dans le tableau ci-dessous.

Option Description
Libellé Permet de déterminer le libellé de la liste déroulante Type d'objet. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.
Image Indique l'emplacement de l'image utilisée. Utilisez un chemin relatif basé dans le répertoire de prise en charge Content Server.
Retrait Permet de déterminer le niveau de retrait des éléments de menu dans la liste déroulante Type d'objet : Aucun, 1, ou 2.
Requête Indique la requête LQL associée à la liste déroulante Type d'objet.

Configuration de la liste déroulante Sélection de vue de recherche

Le composant Sélection de vue de recherche affiche le lien Vues dans la barre de recherche quand des utilisateurs Content Server effectuent une recherche Texte intégral ou en langage naturel. Il vous permet également de définir l'ordre dans lequel les vues de recherche sont affichées.

Lorsque la case à cocher Afficher n'est pas sélectionnée pour le composant Sélection de vue de recherche, la liste déroulante Vues de recherche n'est pas affichée dans la barre de recherche. Toutefois, vous pouvez toujours sélectionner la vue de recherche dans laquelle effectuer une recherche par défaut, tel que décrit dans le tableau ci-dessous.

Option Description
A partir d'ici Permet aux utilisateurs d'effectuer une recherche dans le Dossier ou l'Espace de travail dans lequel ils se trouvent.
Entreprise Permet aux utilisateurs d'effectuer une recherche des versions actuelles des documents.
Entreprise [Toutes les versions] Permet aux utilisateurs d'effectuer une recherche de toutes les versions des documents.

Cliquez sur le lien Configuration de vue de recherche pour ouvrir la page Dossier Vues de recherche pour définir l'apparence du composant Vues de recherche dans la Recherche avancée.

La page Dossier Vues de recherche vous permet de sélectionner l'ordre alphabétique dans lequel les vues de recherche sont affichées, ou de définir une configuration personnalisée.

La section Ordre des vues de recherche dans la barre de recherche et dans le composant Vue de recherche vous permet de sélectionner l'ordre alphabétique dans lequel les vues de recherche sont affichées. Vous pouvez également définir une configuration personnalisée en sélectionnant le bouton radio Personnaliser. Les vues de recherche Disponible et Affiché de votre système Content Server sont affichées de sorte que vous puissiez les disposer dans l'ordre de votre choix.

La case à cocher Ajouter l'option 'A partir d'ici' à la barre de recherche affichera l'option A partir d'ici dans la liste Vues de recherche.

Le champ Taille du menu du composant Vue de recherche pour la recherche avancée permet de déterminer le nombre maximal de vues de recherche à afficher dans la liste Vues de recherche.

Recherche avancée

Le composant Recherche avancée de la page Modification de la barre de recherche affiche le lien Recherche avancée sur le volet de recherche quand les utilisateurs Content Server indiquent une recherche de type Texte intégral ou en langage naturel.

Modificateur d'emplacement

La case à cocher Modificateur d'emplacement affiche le composant Emplacement dans la liste Ajouter au formulaire de recherche.

La liste déroulante vous permet de spécifier le modificateur d'emplacement par défaut Content Server à utiliser, en fonction des modules Content Server installés sur votre système. Pour en savoir plus, consultez le document Configuration des modificateurs d'emplacement de recherche.

Surnom

Le composant Surnom détermine les paramètres de surnom par défaut disponibles dans la barre de recherche. Un surnom est un mot ou une phrase associée à un élément Content Server et comportant une URL courte générée pour les fonctions Propriétés et Ouvrir.

La case à cocher Inclure détermine si la recherche Surnom est disponible dans la barre de recherche.

Le champ Texte de l'invite contrôle le texte qui s'affiche pour le Surnom dans la liste déroulante Sélectionner le type de recherche au-dessus de la barre de recherche. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

Dans la section Composant, vous pouvez spécifier les composants de la recherche Surnom à afficher par défaut et configurer leur apparence et leur comportement.

La case à cocher Afficher détermine si le composant Action est affiché dans la barre de recherche. Deux actions sont proposées dans la liste déroulante, Ouvrir et Propriétés. Ces actions permettent aux utilisateurs Content Server d'accéder à la commande Ouvrir ou à la page Propriétés de l'élément lorsque la recherche est fructueuse.

Configuration de la recherche Requête en langage naturel

Le composant Requête en langage naturel détermine les paramètres par défaut de la recherche en langage naturel disponibles dans la barre de recherche.

La case à cocher Inclure permet de déterminer si la recherche en langage naturel est disponible dans les Options avancées.

Le champ Texte de l'invite contrôle le texte qui s'affiche dans la barre de recherche pour les recherches en langage naturel. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

Vous pouvez spécifier les composants de la recherche en langage naturel à afficher par défaut et configurer leur apparence et leur comportement.

Option Description
Mode RLN La recherche en langage naturel propose deux modes :
•

Content Server Aware est le type de requête utilisé pour les requêtes portant sur des personnes, des dates et des types d'éléments Content Server (par exemple, des documents et des dossiers). Les requêtes Content Server Aware utilisent un ensemble de règles permettant de déterminer le contexte en récupérant des informations sur les types de fichiers et les attributs à partir du texte saisi.

•

Extraction de mots-clés est utilisé lorsque vous ne disposez d'aucune information précise concernant les personnes, les dates ou les éléments Content Server que vous recherchez. L'extraction de mots clés permet d'identifier les cinq mots ou locutions de la requête qui ne sont pas des mots vides (mots qui n'ajoutent aucune valeur sémantique à une phrase, par exemple : un, et, et le) et recherche tous les éléments contenant certains de ces mots ou tous ces mots.

Vous pouvez choisir l'un de ces modes comme mode par défaut dans la barre de recherche.
Sélection de vue de recherche Le composant Sélection de vue de recherche affiche le lien Vues dans la barre de recherche quand des utilisateurs Content Server effectuent une recherche Texte intégral ou en langage naturel. Il vous permet également de définir l'ordre dans lequel les vues de recherche sont affichées.
Recherche avancée Affiche Recherche avancée par défaut dans la barre de recherche.
Modificateur d'emplacement
Affiche le composant Emplacement dans la liste Ajouter au formulaire de recherche. La liste déroulante vous permet de spécifier le modificateur d'emplacement par défaut Content Server à utiliser, en fonction des modules Content Server installés sur votre système. Pour en savoir plus, consultez le document Configuration des modificateurs d'emplacement de recherche.
