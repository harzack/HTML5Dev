La page Configurer les types de volume exclus vous permet de spécifier les types de volume que vous souhaitez exclure de l'extraction. Le fait d'exclure un type de volume de l'extraction implique que ni le contenu, ni les métadonnées associées au type de volume défini, ni les éléments qu'il contient ne sont indexés par le processus Extracteur Entreprise.

 

Note :
Exclure un type de volume exclut tous les éléments stockés dans le volume, même si ces nœuds sont normalement extraits. Par exemple, les documents sont normalement extraits, mais les documents stockés dans les volumes exclus ne seront pas extraits.

Si vous excluez un type de volume déjà indexé, les objets restent dans l'index. Pour les supprimer, éliminez et effectuez une réindexation, ou exécutez une vérification de l'index de recherche. Pour plus de détails, voir Vérification du contenu et réindexation et Configuration de la vérification de l'index.

 

Pour configurer des types de volume exclus :

  1. Sur Pour définir les paramètres généraux de l'Extracteur, dans le champ Types de volume de la section Paramètres d'exclusion, cliquez sur le lien liste de modification/révision.

  2. Sur la page Configurer les types de volume exclus, sélectionnez la case à cocher Tout sélectionner, ou sélectionnez des types de volume individuels pour spécifier ceux que l'Extracteur doit exclure des opérations d'indexation de recherche.

  3. Cliquez sur Mettre à jour pour utiliser les paramètres des types de volume exclus que vous avez définis, ou cliquez sur Réinitialiser les valeurs par défaut pour rétablir les paramètres système par défaut, comme pour une nouvelle installation Content Server, ou cliquez sur Annuler.
