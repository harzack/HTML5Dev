Lors de l'ajout d'un gestionnaire de flux de données, Content Server active la gestion du système par défaut. Cela signifie que Content Server surveille automatiquement le statut des IPools pour le flux de données. Vous pouvez désactiver la gestion du système une fois que le gestionnaire du flux de données est créé en désactivant la case à cocher Activer la gestion du système de la page Règles de contrôle.

Si vous créez les composants d'index de votre source de données individuellement, vous pouvez ajouter un gestionnaire de recherche à un dossier source de données, mais, avant de le faire, vous devez ajouter un gestionnaire de flux de données au dossier source de données. Pour plus d'informations sur l'ajout de gestionnaires de flux de données, reportez-vous à la rubrique Ajout de composants d'index. Pour plus d'informations sur les gestionnaires de recherche, reportez-vous à la rubrique Gestion des processus de recherche.
