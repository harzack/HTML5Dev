Pour utiliser les données stockées dans Content Server, vous devez pouvoir les retrouver rapidement et facilement. Pour cette raison, la création des index et la maintenance de leur intégrité sont deux tâches essentielles que l'administrateur Content Server doit réaliser.

Les administrateurs Content Server créent les index en concevant des flux de données qui extraient et traitent les données qu'ils souhaitent indexer. A mesure que la taille du référentiel Content Server augmente, les administrateurs doivent être capables d'optimiser la fonctionnalité de recherche et d'indexation de Content Server en fonction de la demande croissante sur le système. Cette étape exige une parfaite compréhension de l'architecture des systèmes de recherche et d'indexation de Content Server.

Lorsque vous êtes connecté en tant qu'administrateur de Content Server, la barre de menu générale affiche le menu Admin sur presque toutes les pages par défaut. Depuis le menu Admin, vous pouvez sélectionner Administration Content Server pour naviguer vers la page Administration ou sélectionner Navigateur d'administration de recherche pour accéder au navigateur d'administration de recherche.

Navigateur d'administration de recherche

Le navigateur d'administration de recherche vous permet d'explorer les fonctions de Content Server pour la configuration et la gestion des fonctionnalités, des processus et des composants de recherche. Les liens dans le navigateur ne déclenchent pas d'actions ou de modifications de configuration. Les liens vous guident vers la page de Content Server qui contient la fonction.

Le panneau des tâches, s'il est présent, contient des liens vers les pages Administration de Content Server qui contiennent les commandes des activités décrites.
