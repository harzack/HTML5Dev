Pour configurer la consignation dans un journal du serveur d'administration s'exécutant sur l'hôte Content Server principal :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de débogage.

  2. Pour spécifier le nom et/ou l'emplacement du fichier journal du serveur d'administration, cochez la case Fichier journal dans la section OTAdmin (Serveur Admin), puis saisissez le chemin absolu du fichier dans le champ correspondant.

  3. Pour spécifier le niveau de consignation du fichier journal du serveur d'administration, cliquez sur l'une des valeurs suivantes dans la liste déroulante Niveau de consignation :

    • 1, pour le journal par défaut ;

    • 2, pour le journal détaillé ;

    • 3, pour le journal de débogage.

  4. Pour consigner le flux de données entre le serveur et le serveur d'administration, cochez la case Fichier journal d'entrée, puis saisissez le chemin absolu du fichier journal dans le champ correspondant.

  5. Pour consigner le flux de données entre le serveur d'administration et le serveur, cochez la case Fichier journal de sortie, puis saisissez le chemin absolu du fichier journal dans le champ correspondant.

  6. Cliquez sur le bouton Mettre à jour.

  7. Relancez le serveur d'administration sur l'hôte Content Server principal.
