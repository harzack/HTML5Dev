Pour configurer les informations Directory Walker :

  1. Dans le champ Répertoire d'écriture de la section Informations Directory Walker, saisissez le chemin absolu du répertoire où vous voulez que le processus Directory Walker enregistre les données. OpenText vous recommande de choisir un répertoire d'écriture sur un lecteur local de l'hôte du serveur d'administration.

  2. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Directory Walker doit s'exécuter.

  3. Dans le champ Répertoires, saisissez les chemins d'accès absolus des répertoires de niveau supérieur dans lesquels le processus Directory Walker devra chercher des documents, en séparant les chemins par des points-virgules (;) ou en saisissant chaque chemin sur une ligne individuelle.

  4. Dans le champ Inclure, saisissez la liste des modèles de noms de fichier que le processus Directory Walker devra recueillir, en les séparant par des points-virgules (;) (par exemple, *.html; *.txt; *.doc; etc.).

  5. Dans le champ Exclure, saisissez la liste des modèles de noms de fichier que le processus Directory Walker devra ignorer, en les séparant par des points-virgules (;) (par exemple, *.jpg; *.exe; *.doc; etc.). Si vous ne spécifiez pas de valeurs dans les champs Inclure et Exclure, le processus Directory Walker recueille tous les fichiers dans les répertoires spécifiés. Si vous définissez un type de fichiers à inclure, le processus Directory Walker exclut automatiquement tous les autres types de fichiers. De la même façon, si vous définissez un type de fichiers à exclure, le processus inclut automatiquement tous les autres types de fichiers. Si vous parcourez un répertoire d'un système d'exploitation dans lequel les noms de fichier sont sensibles à la casse, vous pouvez saisir les modèles de noms de fichier couvrant toutes les combinaisons de casse possibles. Par exemple, *.[hH] [tT] [mM] [lL] comprend *.HTML, *.hTml, etc.

  6. Pour définir le nombre de niveaux de sous-répertoires sous les répertoires de niveau supérieur que le processus Directory Walker doit parcourir, cliquez sur l'une des options suivantes de la liste déroulante Profondeur :

    • Illimité, qui permet au processus Directory Walker d'analyser tous les niveaux de sous-répertoires sous les répertoires de niveau supérieur ;

    • Spécifier, qui vous permet de définir le nombre de niveaux de sous-répertoires analysés par le processus Directory Walker ;

    • aucun(e), qui permet au processus Directory Walker d'analyser uniquement les répertoires de niveau supérieur.

  7. Pour limiter les fichiers recueillis par le processus Directory Walker à ceux compris entre certaines dates, saisissez l'intervalle de dates au format aaaa/mm/jj dans les champs Intervalle de dates.

  8. Pour limiter les fichiers recueillis par le processus Directory Walker à ceux compris entre deux tailles de fichier spécifiques, saisissez l'intervalle de taille des fichiers (en octets) dans les champs Taille du fichier.
