Pour afficher la page de statut de l'objet système :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Afficher le statut de l'objet système.

  2. Sur la page Afficher le statut de l'objet système, si des erreurs sont présentes, vous pouvez cliquer sur le lien vers le recueil comportant une erreur.

  3. Cliquez sur le bouton Continuer pour revenir à la page d'administration.
