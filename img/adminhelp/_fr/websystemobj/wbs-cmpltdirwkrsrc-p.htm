Pour finir de configurer une source de données Directory Walker :

  1. Pour lancer les processus de flux de données dès leur création, vérifiez que la case Lancer les processus dans le flux de données est activée.

  2. Cliquez sur le bouton Créer des processus.

  3. Sur la page Statut de la création d'un flux de données, cliquez sur le bouton Continuer.
