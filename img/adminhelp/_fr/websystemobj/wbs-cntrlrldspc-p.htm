Pour ajouter une règle de contrôle basée sur l'espace disque :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur Ouvrir le volume des objets système.

  2. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du préfixe_processus Dossier de la source de données. Choisissez Propriétés, puis Règles de contrôle.

  3. Sur la page Règles de contrôle, cliquez sur le bouton Ajouter une nouvelle règle de contrôle .

  4. Sur la page Ajouter une nouvelle règle de contrôle, saisissez un nom pour la règle dans le champ Nom.

  5. Dans la section Modèles, cliquez sur Espace disque dans la liste déroulante. Vous pouvez modifier la description de la règle dans le champ Description.

  6. Sélectionnez l'une des options suivantes dans la liste déroulante Si :

    • <

    • > =

    • =

  7. Saisissez un nombre dans le champ Mo d'espace disque libre.

  8. Dans la liste déroulante élément, cliquez sur la description de l'IPool qui sera surveillé par cette règle. Dans la section Rapports, vous pouvez entrer un message à envoyer avec le rapport dans le champ Message.

  9. Cliquez sur le bouton Mettre à jour.
