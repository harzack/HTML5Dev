Pour arrêter tous les processus dans un flux de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données de la source de données contenant le flux de données dont vous souhaitez arrêter les processus.

  2. Cliquez sur l'icône Fonctions du gestionnaire de flux de données, puis sélectionnez Interrompre.
