Pour masquer des éléments de l'affichage WebDAV :

  1. Sur la page Administration de Content Server, cliquez sur Configurer les paramètres d'accès WebDAV de la zone Administration WebDAV.

  2. Sur la page Configurer l'accès à WebDAV, cochez l'une des cases suivantes :

    • Masquer les éléments désignés comme masqués, pour masquer les éléments désignés comme masqués pour tous les utilisateurs à l'exception des administrateurs système.

    • Masquer les catégories et les volumes des rapports, pour masquer les catégories et les volumes de rapports pour tous les utilisateurs à l'exception des administrateurs système.

  3. Cochez la case de chaque élément Content Server à masquer dans le dossier WebDAV.

  4. Cliquez sur Enregistrer les modifications.

 

Note :
Vous devez redémarrer le serveur Content Server pour que ces modifications prennent effet.
