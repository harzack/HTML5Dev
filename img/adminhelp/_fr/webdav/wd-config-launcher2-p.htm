Pour activer l'affichage Dossier :

  1. Sur la page Administration de Content Server, cliquez sur Configurer les lanceurs client WebDAV dans la zone Administration WebDAV.

  2. Sur la page Configuration du lanceur client WebDAV, cochez la case Activer l'action « Affichage dossier WebDAV ».

  3. Cliquez sur Enregistrer les modifications.
