Le serveur coopérant Weblink dépend d'un module Weblink pour votre serveur Web. Le module Weblink communique avec son serveur Web au moyen d'un protocole TCP/IP sur un port spécifique. Par défaut, ce port est choisi dynamiquement à chaque démarrage du serveur coopérant Weblink. Pour exécuter le serveur Web et Content Server sur des hôtes différents, vous devez choisir un numéro de port fixe pour le module à utiliser.

Pour vous connecter à une instance WebDAV sur un serveur distant, ce dernier doit être configuré de sorte à être en écoute sur un port statique.
