Pour activer l'authentification d'en-tête personnalisé :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans Serveur coopérant Weblink.

  2. Sur la page Spécifier l'authentification Weblink, sélectionnez En-tête personnalisé. Une fois cette option sélectionnée, un champ Nom d'en tête s'affiche.

  3. Saisissez le nom de l'en-tête dans le champ Nom d'en tête.

  4. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
