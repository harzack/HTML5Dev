L'authentification Basique/Content Server est la méthode d'authentification utilisée par défaut par WebDAV.

Avec cette méthode, à moins que l'authentification des services d'annuaire ne soit configurée pour Content Server, les utilisateurs sont authentifiés par leurs nom d'utilisateur et mot de passe Content Server à l'aide de l'authentification HTTP Basic.

Lorsque Content Server utilise l'authentification des services d'annuaire, le serveur Web doit être configuré pour authentifier les utilisateurs.

 

Note :
Pour plus d'informations sur l'authentification Basique/Content Server, consultez la norme Internet RFC2617.
