Les lanceurs client WebDAV permettent de se connecter rapidement à Content Server au moyen des clients WebDAV les plus courants susceptibles d'être à la disposition des utilisateurs de votre organisation. Vous pouvez choisir de lancer les applications client appropriées lorsque les utilisateurs veulent modifier des documents créés dans des applications Microsoft Office. Cette fonction est disponible pour les clients équipés de l'API client Microsoft Sharepoint, un composant facultatif d'Office 2003/2007/2010/2013. Vous pouvez également configurer les dossiers Web Microsoft de sorte à afficher le contenu de l'élément incorporé dans l'interface utilisateur de Content Server. Pour que cet affichage fonctionne correctement, les utilisateurs doivent utiliser Microsoft Internet Explorer version 5.5 ou ultérieure et disposer de Microsoft WebFolders.
