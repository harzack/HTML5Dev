Pour activer l'authentification personnalisée :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sur la page Spécifier l'authentification Weblink, sélectionnez Personnalisé. Une fois cette option sélectionnée, les champs Nom de classe et Fichier Jar s'affichent.

  3. Saisissez le nom de la classe dans le champ Nom de classe.

  4. Saisissez le nom du fichier Jar dans le champ Fichier Jar.

  5. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
