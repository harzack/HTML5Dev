Pour configurer un serveur Web distant différent pour UNIX/Linux :

  1. Sur la page Administration Content Server, cliquez sur Configurer le serveur Web.

  2. Sur la page Configurer le serveur Web, cliquez sur Configurer le serveur Web distant.

  3. Sur la page Configurer le serveur Web distant, cliquez sur un serveur Web dans la liste Serveur Web.

  4. Saisissez l'adresse IP de Content Server dans le champ Adresse IP de ce Content Server.

  5. Cliquez sur UNIX/Linux.

  6. Entrez le nom d'un répertoire d'installation dans le champ Répertoire d'installation.

  7. Cliquez sur Obtenir les plug-ins.

  8. Enregistrez le fichier .tar dans le répertoire d'installation spécifié.

  9. À l'invite de commandes, tapez la commande suivante puis appuyez sur la touche ENTRÉE :

    tar -xvf weblink351.tar

  10. Cliquez sur Afficher les modifications afin d'enregistrer un fichier comportant les modifications à apporter au serveur Web. Pour plus d'informations sur la procédure à suivre pour configurer votre serveur Web en fonction de ce fichier, voir la section « Configuring the Web Servers and Verifying Livelink ECM - WebDAV (Configuration des serveurs Web et vérification de Livelink ECM - WebDAV) » dans OpenText WebDAV Installation and Administration Guide (Manuel d'installation et d'administration de OpenText WebDAV).
