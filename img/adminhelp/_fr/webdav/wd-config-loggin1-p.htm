Pour spécifier un niveau de consignation :

  1. Sur la page Administration Content Server, cliquez sur Spécifier le niveau de consignation dans la zone Configuration du serveur coopérant Weblink.

  2. Sur la page Spécifier le niveau de consignation du serveur coopérant Weblink, cliquez sur l'une des options de consignation suivantes dans la liste Niveau de consignation :

    • Erreurs fatales uniquement

    • Erreurs uniquement

    • Erreurs et avertissements

    • Informations complètes de débogage

    • Terminer l'archivage de connexion

  3. Cliquez sur Enregistrer les modifications.

 

Note :
Vous devez redémarrer Content Server pour que ces modifications prennent effet.
