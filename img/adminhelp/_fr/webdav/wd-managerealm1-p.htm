Pour autoriser tous les utilisateurs authentifiés :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sélectionnez Autoriser tous les utilisateurs authentifiés.

  3. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.

Pour accepter les utilisateurs de domaines spécifiés :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sélectionnez Accepter les utilisateurs des domaines suivants.

  3. Pour ajouter un domaine, saisissez son nom dans le champ Ajouter domaine, puis cliquez sur Ajouter un domaine.

  4. Pour supprimer un domaine, sélectionnez-le dans le champ Supprimer, puis cliquez sur Supprimer.

  5. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
