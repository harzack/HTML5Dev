Pour activer l'authentification Kerberos :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sur la page Spécifier l'authentification Weblink, sélectionnez Kerberos/NEGOTIATE. Une fois cette option sélectionnée, les champs d'édition Nom principal du service, Domaine Kerberos, Serveur KDC et Fichier de table de clés s'affichent.

  3. Entrez le nom principal du service dans le champ Nom principal du service.

  4. Entrez le nom du domaine Kerberos dans le champ Domaine Kerberos.

  5. Entrez le nom du serveur KDC dans le champ Serveur KDC.

  6. Entrez le nom du fichier de table de clés dans le champ Fichier de table de clés.

  7. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
