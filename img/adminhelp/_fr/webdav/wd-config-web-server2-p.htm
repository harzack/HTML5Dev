Pour configurer Microsoft Internet Information Server (IIS) en tant que serveur Web distant :

  1. Sur la page Administration Content Server, cliquez sur Configurer le serveur Web.

  2. Sur la page Configurer le serveur Web, cliquez sur Configurer le serveur Web distant.

  3. Sur la page Configurer le serveur Web distant, cliquez sur Microsoft Internet Information Server (IIS) dans la liste déroulante Serveur Web.

  4. Saisissez l'adresse IP de Content Server dans le champ Adresse IP de ce Content Server.

  5. Cliquez sur Exécuter le programme d'installation afin de configurer le serveur Web ou d'enregistrer le programme d'installation sur le disque.

     

    Notes :

    • Vous devez exécuter le programme d'installation sur l'ordinateur hébergeant le serveur Web IIS.

    • Si vous enregistrez le programme d'installation sur le disque pour l'exécuter ultérieurement, ne renommez pas le fichier car son nom contient les paramètres nécessaires à la configuration du serveur Web.
