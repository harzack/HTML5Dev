Pour activer l'authentification Basique/Content Server :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sur la page Spécifier l'authentification Weblink, sélectionnez Basique/Content Server.

  3. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
