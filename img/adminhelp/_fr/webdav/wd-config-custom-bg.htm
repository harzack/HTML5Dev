Lorsque l'authentification personnalisée est spécifiée, WebDAV réalise l'authentification en utilisant une classe de module d'extension Java spécifiée par l'administrateur, implémentée conformément à l'API du module d'extension d'authentification WebDAV.

Pour configurer cette méthode d'authentification, le nom de la classe et le fichier .jar du module d'extension doivent être fournis.

 

Note :
Pour des informations détaillées sur l'authentification personnalisée, voir « Configuring Custom Authentication (Configuration de l'authentification personnalisée) » dans OpenText WebDAV Installation and Administration Guide (Manuel d'installation et d'administration de OpenText WebDAV).
