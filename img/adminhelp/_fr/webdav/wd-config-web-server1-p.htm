Pour configurer un serveur Web local :

  1. Sur la page Administration Content Server, cliquez sur Configurer le serveur Web.

  2. Sur la page Configurer le serveur Web, cliquez sur Configurer le serveur Web local.

  3. Sur la page Configurer le serveur Web local, cliquez sur un serveur Web dans la liste Serveur Web.

  4. Procédez de l'une des manières suivantes :

    • Si vous avez spécifié Microsoft Internet Information Server (IIS) à l'étape 3, cliquez sur Exécuter le programme d'installation afin de configurer le serveur Web ou d'enregistrer le programme d'installation sur le disque.

       

      Note :
      Si vous enregistrez le programme d'installation sur le disque pour l'exécuter ultérieurement, ne renommez pas le fichier car son nom contient les paramètres nécessaires à la configuration du serveur Web.

       

    • Si vous avez spécifié un serveur Web autre qu'IIS à l'étape 3, cliquez sur Afficher les modifications afin d'enregistrer un fichier comportant les modifications à apporter au serveur Web. Pour plus d'informations sur la procédure à suivre pour configurer votre serveur Web en fonction de ce fichier, voir « Configuring the Web Servers and Verifying WebDAV (Configuration des serveurs Web et vérification de WebDAV) » dans OpenText WebDAV Installation and Administration Guide (Manuel d'installation et d'administration de OpenText WebDAV).
