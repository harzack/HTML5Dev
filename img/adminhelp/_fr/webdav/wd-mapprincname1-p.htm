Pour configurer la méthode de mappage :

  1. Sur la page Administration Content Server, cliquez sur Spécifier l'authentification Weblink dans la section Serveur coopérant Weblink.

  2. Sur la page Spécifier l'authentification Weblink, cliquez sur l'une des options suivantes dans la liste Méthode de mappage des noms principaux pour les utilisateurs Content Server :

    • Nom identique

    • Domaine et nom

    • Géré par l'utilisateur

  3. Quand le module Content Server Directory Services est installé et configuré correctement, vous pouvez choisir d'utiliser le mappage du module des services d'annuaire pour mapper un nom principal sur un utilisateur Content Server, ou l'une des méthodes de mappage identifiées ci-dessus. Pour utiliser l'une des méthodes de mappage identifiées ci-dessus, sélectionnez Annuler le mappage des services d'annuaire.

  4. Cliquez sur Enregistrer les modifications, puis redémarrez le serveur Content Server.
