Pour lancer une application client afin de modifier un document :

  1. Sur la page Administration de Content Server, cliquez sur Configurer les lanceurs client WebDAV dans la zone Administration WebDAV.

  2. Sur la page Configurer les lanceurs client WebDAV, cochez la case Activer l'action Modification WebDAV pour les documents Microsoft Office.

  3. Cliquez sur Enregistrer les modifications.
