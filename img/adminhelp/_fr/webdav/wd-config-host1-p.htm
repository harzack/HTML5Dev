Pour configurer les traductions de l'hôte :

  1. Sur la page Administration Content Server, cliquez sur Configurer les traductions de l'hôte dans la section Configuration du serveur coopérant Weblink.

  2. Sur la page Configurer les traductions de l'hôte du serveur coopérant Weblink, cliquez sur Ajouter une traduction.

  3. Saisissez le nom d'un serveur intermédiaire dans le champ Nom d'hôte du proxy.

  4. Saisissez le nom d'un serveur client dans le champ Définir l'hôte.

    Le cas échéant, entrez le protocole dans le champ Définir le schéma pour spécifier le protocole utilisé par le serveur client.

  5. Cliquez sur Enregistrer les modifications.

 

Notes :

  • Si le serveur intermédiaire se connecte via un port non-standard, vous pouvez ajouter ce port dans le champ Nom d'hôte du proxy. Par exemple, hostname:443.

  • Vous devez redémarrer Content Server pour que ces modifications prennent effet.
