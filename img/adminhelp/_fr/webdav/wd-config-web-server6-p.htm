Configurer IIS en tant que serveur Web local lors d'une installation ou d'une mise à niveau :

  1. Sur la page Configurer le serveur Web local, cliquez sur Microsoft Internet Information Server (IIS) dans la liste déroulante Serveur Web.

  2. Pour configurer le serveur Web IIS, cliquez sur Exécuter le programme d'installation.

  3. Pendant l'installation, vérifiez le serveur Web, le nom du service Content Server et l'alias du répertoire virtuel, puis terminez la configuration du serveur Web.

  4. Sur la page Configurer le serveur Web local, cliquez sur Continuer.

  5. Terminez l'installation du module.

     

    Notes :

    • Vous devez exécuter le programme d'installation sur l'ordinateur hébergeant le serveur Web IIS.

    • Si vous enregistrez le programme d'installation sur le disque pour l'exécuter ultérieurement, ne renommez pas le fichier car son nom contient les paramètres nécessaires à la configuration du serveur Web.
