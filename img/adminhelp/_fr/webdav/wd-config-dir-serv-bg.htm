Lorsque le module des services d'annuaire Content Server est installé et correctement configuré, Content Server se base sur le serveur Web pour authentifier les utilisateurs.

Pour configurer WebDAV pour l'authentification à l'aide du module Content Server Directory Services, vous devez configurer manuellement les règles de contrôle d'accès dans le répertoire virtuel WebDAV de votre serveur Web pour qu'elles correspondent à celles du répertoire virtuel de Content Server. Par exemple, si vous accédez à Content Server à l'aide d'un répertoire virtuel appelé livelink et à WebDAV à l'aide d'un répertoire virtuel appelé livelinkdav, vous devez configurer les règles de contrôle d'accès du répertoire virtuel livelinkdav pour qu'elles correspondent à celles du répertoire virtuel livelink.

 

Note :
Pour des informations détaillées sur la configuration de l'authentification à l'aide du module des services d'annuaire Content Server, voir « Configuring Authentication using the Content Server Directory Services Module (Configuration de l'authentification à l'aide du module des services d'annuaire Content Server) » dans OpenText WebDAV Installation and Administration Guide (Manuel d'installation et d'administration de OpenText WebDAV).
