Pour configurer un autre serveur Web local lors d'une installation ou d'une mise à niveau :

  1. Sur la page Configurer le serveur Web local, cliquez sur un serveur Web dans la liste Serveur Web.

  2. Cliquez sur Afficher les modifications.

  3. Enregistrez le fichier contenant la description des modifications à apporter au serveur Web spécifié.

  4. Sur la page Configurer le serveur Web local, cliquez sur Continuer.

  5. Terminez l'installation du module.
