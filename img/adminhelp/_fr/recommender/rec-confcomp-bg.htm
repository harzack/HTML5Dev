Les recommandations et évaluations sont présentées aux utilisateurs dans les composants de l'Assistant de recommandation. Il s'agit de sections d'éléments connexes de l'interface utilisateur graphique figurant sur la page Recommandations de chaque utilisateur et dans l'onglet Evaluations de la page Propriétés de chaque document. Par exemple, le composant Nouveautés, qui apparaît sur la page Recommandations, répertorie la liste des éléments les plus récents ajoutés à Content Server.

 

Note :
Le composant Nouveautés utilise le paramètre de serveur Indicateur « Nouveau » pour déterminer les éléments nouveaux. Pour plus d'informations sur la configuration de la durée de cet indicateur, reportez-vous à la rubrique Configuration des paramètres de base du serveur.

Permet de spécifier la taille de l'échantillon pour le composant. La taille de l'échantillon détermine le nombre d'éléments que le système utilise pour effectuer les recommandations. Synthèse automatique
•

Synthèse
Lorsqu'un élément est ajouté à Content Server ou modifié de quelque manière que ce soit, le système l'extrait. Ce processus attribue des métadonnées à l'élément et l'indexe de sorte qu'il puisse faire l'objet d'une recherche.
Ce paramètre permet de spécifier si vous souhaitez extraire la synthèse d'un élément lors de la première consultation de l'onglet Evaluations de la page Propriétés de cet élément. Les synthèses ne peuvent faire l'objet d'une recherche qu'une fois extraites. Si cette case est désactivée, les synthèses sont extraites uniquement lorsqu'elles sont modifiées, celles qui ne sont pas modifiées ne peuvent donc pas faire l'objet d'une recherche.
 

Note :
L'extraction est une opération coûteuse lorsqu'elle est effectuée sur plusieurs éléments dans un laps de temps restreint. Le paramétrage de l'Assistant de recommandation pour une extraction automatique des synthèses peut ralentir le fonctionnement des installations Content Server les plus importantes.
