Pour configurer le paramétrage système de l'Assistant de recommandation :

  1. Sur la page Administration, cliquez sur le lien Configurer les paramètres système de l'Assistant de recommandation dans la section Administration de l'Assistant de recommandation.

  2. Sur la page Configurer les paramètres système de l'Assistant de recommandation, dans la section Agent de l'Assistant de recommandation :

    1. Pour activer l'agent de l'Assistant de recommandation, cochez la case Activer.

    2. Pour que l'agent de l'Assistant de recommandation assure le suivi de l'utilisateur Admin, cochez la case Suivi de l'utilisateur Admin.

  3. Dans la section Tableau des évaluations :

    1. Pour configurer les types d'élément pouvant être évalués, cliquez sur l'icône Types d'élément.

    2. Sur la page Configurer : Types d'éléments, sélectionnez les types d'éléments que vous souhaitez évaluer.

    3. Cliquez sur Mettre à jour.

  4. Dans la section Tableau de résumé :

    1. Pour afficher le tableau de synthèse, effectuez l'une des opérations suivantes :

      • Pour afficher les entrées du tableau de résumé dont le nombre de références est supérieur à un nombre spécifique, cliquez sur un nombre de la liste Nombre de références supérieur à, puis cliquez sur Afficher.

      • Pour afficher tout le contenu du tableau de résumé, ne faites pas de sélection dans la liste Nombre de références supérieur à, puis cliquez sur Afficher.

    2. Pour spécifier le nombre de jours pendant lesquels vous souhaitez que les données soient conservées dans le tableau de résumé, sélectionnez un nombre dans la liste Automatique.

    3. Pour éliminer toutes les données du tableau de résumé, cliquez sur Eliminer maintenant.

  5. Cliquez sur Mettre à jour.
