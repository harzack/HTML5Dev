Pour exporter des données au format XML à partir de l'onglet Evaluations de la page Propriétés d'un élément, ajoutez &outputxml=true à la fin de l'URL de la page. Saisissez par exemple l'URL suivante :

<protocole>://<hôte>/<préfixe_URL> /.exe?func=ll&objid=<ID>&objAction=Ratings&outputxml=true

où :

  • <protocole> est soit http, soit https ;

  • <hôte> est l'adresse complète de l'hôte (par exemple, host.mycorp.com)

  • <préfixe_URL> est le raccourci du répertoire virtuel mappé au répertoire <Content Server_accueil>/cgi dans le serveur HTTP et

  • <id> est l'ID de nœud de l'élément.

Cette commande génère la sortie de l'onglet Evaluations de la page Propriétés de l'élément pour l'utilisateur actuellement connecté au serveur.

Le DTD qui définit cette sortie XML est inclus dans les installations telles que <Content Server_home>/module/recommender_<x> _<x>_<x>/config/recommender_xml_output.dtd, où <x>_<x>_<x> est le numéro de version du module Recommender. Par exemple, pour afficher le fichier pour le module Recommender 10.5.0, voir <Content Server_home> /module/recommender_10_5_0/config/recommender_xml_output.dtd.
