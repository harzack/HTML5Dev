Pour configurer un composant de l'Assistant de recommandation :

  1. Sur la page Administration, cliquez sur le lien Configurer les composants de l'Assistant de recommandation dans la section Administration de l'Assistant de recommandation.

  2. Sur la page Configurer les composants de l'Assistant de recommandation, sélectionnez la case à cocher Activer associée à un composant pour l'afficher sur la page Recommandations et sur la page Evaluations par défaut.

  3. Cliquez sur le bouton Action pour ouvrir la page Configurer avec les paramètres utilisateur pour ce composant.

  4. Sur la page Configurer : <composant> :

    1. Spécifiez les paramètres que vous voulez rendre visibles aux utilisateurs.

    2. Cliquez sur Mettre à jour.

  5. Sur la page Configurer les composants de l'Assistant de recommandation, cliquez sur Mettre à jour.
