Vous ajoutez une icône d'évaluation lorsque vous voulez représenter les évaluations assignées aux éléments de Content Server par des icônes personnalisées. Pour plus d'informations sur l'ajout d'icônes d'évaluation, reportez-vous à la rubrique Personnalisation de l'assistant de recommandation.

Utilisation des icônes d'évaluation

Les notes d'évaluation allant de 0 à 5, avec la possibilité de demi-points (par exemple, 3,5), vous devez insérer 11 images au total, soit une icône par évaluation potentielle. Les noms de fichier pour chaque icône doivent suivre une convention spécifique.

Tableau 1-2 : Noms de fichier des icônes d'évaluation

Nom d'icône Note
<nom>0.gif 0
<nom>05.gif 0.5
<nom>1.gif 1
<nom>15.gif 1.5
<nom>2.gif 2
<nom>25.gif 2.5
<nom>3.gif 3
<nom>35.gif 3.5
<nom>4.gif 4
<nom>45.gif 4.5
<nom>5.gif 5

où <nom> est le nom que vous souhaitez attribuer à votre icône d'évaluation dans l'interface utilisateur graphique.

Les fichiers des icônes d'évaluation doivent être stockés dans les deux répertoires suivants :

  • <Content Server_accueil> /support/recommender/ratinggifs/<nom> et

  • <Content Server_accueil>/module/recommender_< version>/support/ratinggifs/<nom>

où :

  • <Content Server_accueil> est l'emplacement de votre installation Content Server.

  • <nom> est le nom du fichier de l'image d'évaluation.

  • <version> est le numéro de version du module Recommender. Par exemple, 10_5_0 pour la version 10.5.0.

La taille recommandée pour chaque fichier d'icône d'évaluation est de 48 pixels en largeur pour 10 pixels en hauteur ; l'arrière-plan des icônes d'évaluation doit être transparent.
