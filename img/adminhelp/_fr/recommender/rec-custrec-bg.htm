La personnalisation du fonctionnement du module de l'Assistant de recommandation inclut la sélection d'une icône d'évaluation, le changement des libellés d'évaluation et l'exportation de données de recommandation et d'évaluation au format XML pour une utilisation dans vos applications personnelles. Vous pouvez également ajouter des icônes d'évaluation personnelles. Pour en savoir plus, voir Ajout d'icônes d'évaluation.

Sélection d'une icône d'évaluation

Lorsque vous sélectionnez une icône d'évaluation, vous sélectionnez l'icône représentant les évaluations assignées aux éléments dans Content Server. Par exemple, si vous choisissez l'image d'une étoile , l'image représente une évaluation de 3. Plusieurs icônes sont disponibles par défaut.

Modification des libellés des évaluations

Lorsque vous modifiez un libellé d'évaluation, vous modifiez le texte qui figure dans la liste déroulante Evaluation de l'onglet Evaluations de la page Propriétés d'un élément. Par exemple, pour une évaluation de 4, vous pouvez changer le libellé de Très intéressant (par défaut) à Très bon.

Exportation de données au format XML

L'exportation de données de recommandation ou d'évaluation au format XML vous permet d'utiliser ces données dans vos applications personnelles. Vous pouvez, par exemple, mettre les données à la disposition des utilisateurs à l'aide d'une interface de portail personnalisée. Pour en savoir plus, voir Exportation de données de recommandation au format XML et Exportation de données d'évaluation au format XML.
