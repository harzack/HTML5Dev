Pour modifier les libellés d'évaluation :

  1. Sur la page Administration, cliquez sur le lien Configurer les composants de l'Assistant de recommandation dans la section Administration de l'Assistant de recommandation.

  2. Cliquez sur l'icône Configurer du composant Ma révision.

  3. Sur la page Configurer : Ma révision, dans chaque champ Libellé, saisissez le texte que vous voulez faire apparaître pour les libellés d'évaluation dans la liste déroulante Ma révision de la page Evaluations. Vous pouvez modifier la langue utilisée en saisissant une valeur Xlate au lieu de texte brut.

  4. Cliquez sur Mettre à jour.

  5. Pour modifier le texte d'un libellé d'évaluation, saisissez le texte dans le champ Libellé correspondant.
