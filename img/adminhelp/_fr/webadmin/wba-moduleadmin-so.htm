Les composants Content Server sont organisés en modules logiciels. Il existe deux types de modules : les modules principaux et les modules facultatifs.

Les modules principaux, comme les Espaces de travail Content Server, Content Server Document Management et Projets Content Server, sont installés automatiquement avec Content Server. Les modules facultatifs, comme Content Server Forms, doivent être installés par l'administrateur Content Server pendant ou après l'installation initiale de Content Server. Les modules facultatifs complètent les fonctionnalités de Content Server en fonction des besoins de votre entreprise. Il peut s'agir de modules nouvellement achetés ou de modules disponibles à l'installation, que vous avez choisi de ne pas installer. Pour plus d'informations sur l'installation de modules facultatifs en même temps que Content Server, reportez-vous au manuel Content Server First-Time Installation Guide.

Les modules facultatifs sont parfois distribués sur CD-ROM, mais sont le plus souvent disponibles en téléchargement depuis la Base de connaissances Open Text. Dans les deux cas, le module est accompagné d'une documentation spécifique et d'instructions d'installation.
