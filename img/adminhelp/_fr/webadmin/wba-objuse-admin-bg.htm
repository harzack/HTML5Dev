Un administrateur Content Server gère les privilèges relatifs aux objets et à l'utilisation pour protéger les données Content Server et l'intégrité du système :

  • Les privilèges de l'objet permettent aux utilisateurs et aux groupes de créer certains types d'élément Content Server

  • Les privilèges d'utilisation permettent aux utilisateurs et aux groupes d'effectuer certains types de tâche Content Server
