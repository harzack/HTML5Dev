Note :
Vous devez tout d'abord vous connecter au serveur Microsoft SQL Server en tant qu'utilisateur doté de privilèges d'administrateur.

 

Pour créer une base de données Microsoft SQL Server :

  1. Dans la section Administration de la base de données de la page Administration, cliquez sur le lien Maintenance de la base de données active.

  2. Sur la page Maintenance de la base de données active, cliquez sur le lien Tâches de maintenance Microsoft SQL Server.

  3. Sur la page Connexion de l'administrateur Microsoft SQL Server, saisissez votre nom d'utilisateur SQL dans les champs appropriés, puis cliquez sur le bouton Connexion.

  4. Sur la page Maintenance Microsoft SQL Server, cliquez sur Créer une nouvelle base de données Microsoft SQL Server.

  5. Dans le champ Nom de la base de données, saisissez le nom à assigner à la base de données, par exemple, Content Server.

  6. Dans le champ Chemin d'accès au fichier de données, saisissez un chemin et un nom de fichier, par exemple, C:\Store\<nom_fichier>.mdf.

  7. Entrez la taille en Mo du fichier de données (minimum 5 Mo) dans le champ Taille du fichier de données.

     

    Astuce :
    Vous pouvez estimer une taille de fichier de données en vous servant de la formule suivante :

    # documents X # versions X # KB avg. doc. size = tablespace size

     

  8. Dans le champ Chemin d'accès au fichier journal, saisissez un chemin et un nom de fichier, par exemple, C:\Store\<nom_fichier>.ldf.

  9. Entrez la taille en Mo du fichier (minimum 5 Mo) dans le champ Taille du fichier journal. Utilisez l'exemple Taille du fichier de données comme guide.

  10. Cliquez sur Créer une base de données.

  11. Si vous avez terminé les tâches qui vous intéressent sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Revenir à la page précédente.
