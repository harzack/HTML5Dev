Pour démarrer ou arrêter provisoirement un traitement d'agent distribué :

  1. Sur la page Administration Content Server, sous le titre Administration système, cliquez sur Statut de l'agent distribué.

  2. Sur la page Statut de l'agent distribué, dans la colonne Actions, si le traitement a été arrêté, cliquez sur le lien Reprendre. Si le traitement est en cours d'exécution, cliquez sur le lien Pause.

     

    Note :
    Le système peut reprendre un traitement arrêté, ou suspendre un traitement repris si nécessaire.
