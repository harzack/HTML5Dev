• Description :

    La section [FetchMimeTypes] contient les types MIME de fichiers que Content Server doit toujours télécharger au lieu de tenter de les ouvrir ou de les extraire.

     

    Important :
    OpenText vous recommande de ne modifier aucune des options de cette section.

     

  • Valeurs :

    Voici un exemple de la section [FetchMimeTypes] du fichier opentext.ini :

    [FetchMimeTypes]
    type1=application/x-msdownload
    type2=application/octet-stream
