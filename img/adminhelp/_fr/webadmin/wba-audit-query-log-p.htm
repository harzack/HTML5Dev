Pour consulter le journal de l'audit :

  1. Cliquez sur Gérer l'audit des événements de la section Administration système sur la page Administration.

  2. Sur la page Gérer l'audit des événements, cliquez sur Interroger le journal d'audit.

  3. Dans la section Eléments cible, procédez de l'une des façons suivantes :

    • Pour afficher le journal de l'audit de tous les éléments, cliquez sur le bouton radio Tous.

    • Pour afficher le journal d'audit d'un type d'élément particulier, cliquez sur le bouton radio Par type, puis sélectionnez un type d'élément dans la liste.

       

      Note :
      Si vous sélectionnez Utilisateur ou groupe, les résultats retournés correspondent aux types d'éléments suivants : Utilisateur, Groupe, Groupe interdomaine et Générateur.

       

    • Pour afficher le journal d'audit d'un seul élément, cliquez sur Elément unique. Cliquez sur le bouton Parcourir Content Server, naviguez jusqu'à l'élément, puis cliquez sur son lien Sélectionner.

    • Pour afficher le journal d'audit d'événements exécutés sur un compte utilisateur ou un groupe, cliquez sur Utilisateur/Groupe unique, puis recherchez un utilisateur ou un groupe.

  4. Cliquez sur le type d'événement pour lequel vous voulez afficher les résultats de l'audit dans la liste Type d'événement.

     

    Astuce :
    Cliquez sur <N'importe lequel> pour afficher les résultats de tous les types d'événements.

     

  5. Dans la section Exécutants, procédez de l'une des façons suivantes :

    • Pour afficher le journal d'audit d'événements exécutés par tout utilisateur ou groupe, cliquez sur Tous.

    • Pour afficher le journal d'audit d'événements exécutés par un utilisateur spécifique, cliquez sur Utilisateur spécifique, puis sélectionnez un utilisateur ou un groupe.

  6. Cliquez sur un mois, un jour et une année dans les listes A partir du et Jusqu'au pour définir la plage temporelle de la recherche.

  7. Cliquez sur une valeur de la liste Lignes par page pour définir le nombre d'éléments apparaissant sur une page.

  8. Cliquez sur Envoyer la requête.

 

Note :
Une requête trop vague peut potentiellement renvoyer un grand nombre de résultats, selon la taille de la base de données. OpenText vous recommande donc de limiter les résultats en précisant un type d'élément et d'événement, un utilisateur et/ou un intervalle de dates. De plus, les combinaisons impossibles ne renvoient aucun résultat. Par exemple, si vous sélectionnez Dossier comme type d'élément et Ajouter une version comme type d'événement, la requête ne va renvoyer aucun résultat.
