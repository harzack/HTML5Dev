Dans la section [llview], vous pouvez corriger les problèmes de consommation des ressources excessive par les processus llview. Ces processus sont créés à chaque fois qu'un utilisateur ouvre un document dans Content Server.

CancelTimeOut

  • Description :

    Envoie à Content Server une instruction de terminer tout processus llview dépassant le nombre indiqué de secondes.

  • Syntaxe :

    CancelTimeOut=10

  • Valeurs :

    Un entier supérieur ou égal à 0.
