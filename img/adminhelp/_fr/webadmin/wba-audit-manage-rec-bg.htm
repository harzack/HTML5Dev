La page Résultats de l'interrogation du journal d'audit répertorie tous les événements et éléments inclus dans l'audit et correspondant aux critères spécifiés sur la page Interroger le journal d'audit. Vous pouvez modifier les critères en cliquant sur le bouton Retour à la page Requête et en modifiant les informations.

 

Astuce :
Pour afficher d'autres pages de résultats, cliquez sur le bouton Précédent ou Suite, en bas de la liste de résultats. Pour revenir à la page Administration, cliquez sur le lien Accueil Admin.

Les informations du journal d'audit sont stockées dans des tables de la base de données Content Server. Vous pouvez augmenter la capacité de stockage de votre base de données en supprimant régulièrement les données inutiles. Vous pouvez vider le journal de l'audit en fonction de la date, de l'utilisateur et du type d'événement. Lorsque vous éliminez les données du journal d'audit, Content Server supprime des lignes de la table DAuditNew de la base de données Content Server. Si des lignes correspondantes existent dans la table DAuditMore, elles sont également supprimées.

 

Avertissement
L'élimination est irréversible. Pour être sûr d'éliminer les bons éléments ou événements, demandez au journal de l'audit d'afficher les éléments et événements à éliminer avant de procéder à toute élimination. Pour en savoir plus, voir Gestion des enregistrements d'audit.

 

Après avoir défini les événements à inclure dans l'audit, vous pouvez consulter les journaux d'audit pour vérifier l'utilisation de la base de données et diagnostiquer les problèmes. Les utilisateurs de Content Server peuvent afficher le journal d'audit des éléments uniques en cliquant sur l'icône Fonctions , puis sur Infos et Audit.
