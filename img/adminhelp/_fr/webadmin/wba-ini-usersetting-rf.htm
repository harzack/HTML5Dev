La section [UserSetting] définit les paramètres de couleur de l'interface utilisateur Content Server et l'affichage du nom d'utilisateur Content Server dans l'interface.

Les paramètres de couleur se rapportent au jeu de couleurs visible sur la plupart des pages Content Server. En général, la plupart des pages présentent des données sous forme de lignes et de colonnes. Les utilisateurs Content Server modifient les couleurs par défaut des en-têtes de colonne et des lignes en accédant au menu Outils.

L'affichage du nom de l'utilisateur est un format permettant d'identifier la façon dont apparaissent les noms dans Content Server. L'administrateur change le format des noms en effectuant des modifications sur la page Configurer l'affichage du nom d'utilisateur de la page Administration.

RowColorOptions

  • Description :

    Fournit les six options de valeur de couleur hexadécimale que les utilisateurs peuvent sélectionner pour les lignes 1 et 2. Les utilisateurs peuvent modifier la couleur par défaut de la ligne via Paramètres dans le menu Outils. L'administrateur peut, s'il le souhaite, ajouter un nombre illimité d'options de couleur en spécifiant toute valeur de couleur hexadécimale valide dans le fichier opentext.ini.

  • Syntaxe :

    RowColorOptions=#FFFFFF,#EEEEEE,#FFFFCC,#CCFFFF,#CCFFCC,#DDDDFF

  • Valeurs :

    Liste de valeurs de couleurs hexadécimales valides, séparées par des virgules.

ColumnHeaderColorOptions

  • Description :

    Fournit les quatre options de valeur de couleur hexadécimale que les utilisateurs peuvent sélectionner pour les en-têtes de colonnes qui apparaissent sur la plupart des pages Content Server. Les utilisateurs peuvent modifier la couleur par défaut de la colonne d'en-tête via Paramètres dans le menu Outils. L'administrateur peut, s'il le souhaite, ajouter un nombre illimité d'options de couleur en spécifiant toute valeur de couleur hexadécimale valide dans le fichier opentext.ini.

  • Syntaxe :

    ColumnHeaderColorOptions=#CCCCCC,#A0B8C8,#83D8A4,#CCCC99

  • Valeurs :

    Liste de valeurs de couleurs hexadécimales valides, séparées par des virgules.

Row1Color

  • Description :

    Identifie une valeur de couleur hexadécimale pour Ligne 1. Toute modification apportée à ce champ influence uniquement les paramètres INI de l'ordinateur local et non les autres utilisateurs du système Content Server.

  • Syntaxe :

    Row1Color=#EEEEEE

  • Valeurs :

    Toute valeur de couleur hexadécimale valide. La valeur par défaut de la couleur est #EEEEEE.

Row2Color

  • Description :

    Identifie une valeur de couleur hexadécimale pour Ligne 2. Toute modification apportée à ce champ influence uniquement les paramètres INI de l'ordinateur local et non les autres utilisateurs du système Content Server.

  • Syntaxe :

    Row2Color=#FFFFFF

  • Valeurs :

    Toute valeur de couleur hexadécimale valide. La valeur par défaut de la couleur est #FFFFFF.

ColumnHeaderColor

  • Description :

    Identifie une valeur de couleur hexadécimale pour les en-têtes de colonne. Toute modification apportée à ce champ influence uniquement les paramètres INI de l'ordinateur local et non les autres utilisateurs du système Content Server.

  • Syntaxe :

    ColumnHeaderColor=#CCCCCC

  • Valeurs :

    Toute valeur de couleur hexadécimale valide. La valeur par défaut de la couleur est #CCCCCC.

UserNameAppendID

  • Description :

    Format permettant d'ajouter l'ID de connexion Content Server à l'affichage du nom dans Content Server. Seul l'administrateur peut modifier le paramètre UserNameAppendID sur la page Configurer l'affichage du nom d'utilisateur de l'interface d'administration.

  • Syntaxe :

    UserNameAppendID=0

  • Valeurs :

    0 ou 1. La valeur par défaut est 0, ou false, ce qui signifie que l'ID de connexion Content Server n'est pas ajouté à l'affichage du nom. L'autre valeur valide est 1, ou true, ce qui signifie que l'ID de connexion Content Server est ajouté à l'affichage du nom.

    Exemples :

    • Si UserNameAppendID=0, le format d'affichage du nom est : John Q Public.

    • Si UserNameAppendID=1, le format d'affichage du nom est : John Q Public (JQPublic).
