Dans l'éventualité (peu probable) où une modification de votre base de données invaliderait votre empreinte système, le statut de votre licence apparaîtrait comme Invalid Fingerprint sur la Présentation de licence Content Server.

Une empreinte système non valide n'a pas d'incidence sur votre déploiement. Vos utilisateurs peuvent continuer d'accéder normalement à Content Server. Le système ne passe pas en mode d'administration. Cependant, si le statut de votre licence est Invalid Fingerprint, OpenText vous recommande de contacter votre service d'assistance clientèle pour obtenir de l'aide.
