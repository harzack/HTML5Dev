• Description :

    Les entrées de la section [HHIncludedMimeTypes] déterminent les types d'éléments dont les résultats peuvent être mis en évidence dans Content Server. Seuls les éléments qui correspondent à des types MIME compris dans cette section ont la commande Mise en évidence des résultats disponible dans le menu Fonctions de Content Server. La section [HHIncludedMimeTypes] est prioritaire par rapport à la section [HHExcludedMimeTypes], qui détermine les types d'éléments pouvant être exclus de la mise en évidence dans Content Server. Cependant, si la section [HHIncludedMimeTypes] ne figure pas dans le fichier opentext.ini ou si elle est vide, Content Server utilise la section [HHExcludedMimeTypes] pour déterminer les éléments dont les résultats peuvent être mis en évidence. Pour plus d'informations sur la section [HHExcludedMimeTypes], voir [HHExcludedMimeTypes].

  • Syntaxe :

    mime1=image/jpeg

  • Valeurs :

    Chaque entrée de la section [HHIncludedMimeTypes] suit le format suivant : format : mime<n>=<type_mime>, où <n> est un nombre unique.

    Le format des types MIME que vous souhaitez définir doit être valable. Pour afficher des types MIME courants, voir le tableau situé dans [HHExcludedMimeTypes].
