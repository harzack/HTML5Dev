La section [chicklet] contient le fichier image qui apparaît dans l'encadré du coin supérieur droit. Vous pouvez remplacer l'image par défaut par le fichier image de votre entreprise.

Pour une visualisation optimale, votre fichier image doit présenter les propriétés suivantes :

Propriété Valeur
Taille décodée en octets 7064
Dimensions de l'image en pixels 83 x 30
Couleur True Color RVB 24 bits
Palette graphique Aucune
Transparence Non

Le fichier image se trouve dans le répertoire <Content Server_home>/support, où <Content Server_accueil> est la racine de votre installation Content Server. Si vous modifiez le fichier image, vous devez redémarrer le serveur et le serveur d'application Web.
