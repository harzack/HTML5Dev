Pour configurer des messages système :

  1. Cliquez sur le lien Configurer les messages système de la section Configuration du serveur de la page Administration.

  2. Pour ajouter un message système, procédez comme suit :

    • Cliquez sur le bouton Ajouter un message.

    • Saisissez le nom du message dans le champ Nom.

    • Saisissez un titre descriptif au message dans le champ Titre. Ce texte s'affichera dans le bandeau de nouvelles.

    • Vous pouvez entrer une URL dans le champ URL. En spécifiant une URL, vous convertissez le titre en lien pouvant être sélectionné pour accéder à une autre page Web. Il peut s'agir de toute page Web accessible à Content Server sur Internet, sur l'intranet de votre entreprise ou dans Content Server proprement dit.

    • Cliquez sur la date à laquelle le message doit commencer à être diffusé aux utilisateurs dans les listes déroulantes Date d'entrée en vigueur.

    • Cliquez sur les boutons et pour déplacer l'élément vers le haut ou le bas dans le menu.

    • Cliquez sur Envoyer pour enregistrer les changements et revenir à la page d'administration. Cliquez sur le bouton Annuler pour annuler vos modifications et renvoyer la page à ses valeurs enregistrées précédentes de façon à pouvoir recommencer.

  3. Pour modifier un message système, procédez comme suit :

    • Cliquez pour mettre en surbrillance le nom du message que vous voulez éditer dans la boîte de dialogue à gauche de la page Configurer les messages système.

    • Vous pouvez remarquer que les champs à droite de la page sont à présent remplis à l'aide des informations liées au nom du message que vous avez mis en surbrillance. Modifiez les informations dans les champs appropriés.

    • Cliquez sur les boutons et pour déplacer l'élément sélectionné vers le haut ou le bas dans l'ordre d'affichage.

     

    Astuce :
    Pour supprimer un message, cliquez pour mettre en surbrillance le nom du message que vous voulez supprimer de la boîte de dialogue à gauche de la page Configurer les messages système. Cliquez sur le bouton Supprimer .
