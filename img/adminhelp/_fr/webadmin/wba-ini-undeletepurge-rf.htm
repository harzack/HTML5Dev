La section [UndeletePurge] contrôle l'élimination automatique des documents du Volume Documents supprimés.

Pour modifier ce paramètre, OpenText vous recommande d'utiliser la page Configurer l'élimination automatique, plutôt que de modifier directement le fichier opentext.ini. Pour en savoir plus, voir Configuration de l'élimination automatique.

OlderThan

  • Description :

    Spécifie le nombre de jours après lequel un document supprimé est éliminé de Content Server.

  • Syntaxe :

    OlderThan=7

  • Valeurs :

    Un entier. La valeur par défaut est OlderThan=7.
