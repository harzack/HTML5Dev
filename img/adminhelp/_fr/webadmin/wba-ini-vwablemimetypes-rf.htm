• Description :

    La section [ViewableMimeTypes] répertorie les types MIME de documents pour lesquels l'ouverture doit être traitée comme une extraction. Content Server transmet directement le document au navigateur Web sans tenter de le convertir au préalable au format HTML pour l'affichage.

  • Valeurs :

    Voici un exemple de la section [ViewableMimeTypes] du fichier opentext.ini :

    MimeType_1=image/gif
    MimeType_2=image/jpeg
    MimeType_3=text/html
    MimeType_4=text/plain
    MimeType_5=application/pdf
    MimeType_6=application/x-zip-compressed
    MimeType_7=text/xml
    MimeType_8=message/rfc822
    MimeType_9=application/x-macbinary
    MimeType_10=image/pjpeg
