Par défaut, la section [Home] ne figure pas dans le fichier opentext.ini. Vous n'aurez besoin de l'ajouter que pour modifier la valeur par défaut du paramètre EditOrganizeMaxItems, décrit ci-dessous.

EditOrganizeMaxItems

  • Description :

    Définit le nombre maximum d'éléments affichés par page par Content Server lorsqu'un utilisateur modifie/organise des éléments, par exemple des favoris ou des projets dans des onglets.

    Par exemple, si un utilisateur, membre de 150 projets, clique sur le lien Modifier/Organiser tandis que l'onglet Tous des projets est ouvert dans son espace de travail Personnel, Content Server affiche uniquement le nom des 100 premiers projets sur la première page. Pour visualiser les 50 autres projets, l'utilisateur doit cliquer sur 2 de 2 dans la liste Page située en bas de la page.

  • Syntaxe :

    EditOrganizeMaxItems=100

  • Valeurs :

    Un entier supérieur ou égal à 0. La valeur par défaut est 100.
