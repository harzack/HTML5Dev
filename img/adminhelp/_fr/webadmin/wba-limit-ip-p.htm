Pour limiter la connexion du compte Admin au moyen d'une adresse IP :

  1. Dans la section Configuration du serveur de la page Administration, cliquez sur le lien Limiter la connexion du compte Admin.

  2. Sur la page Limiter la connexion du compte Admin, procédez de l'une des façons suivantes :

    • Pour ajouter une adresse IP, saisissez-la dans le champ prévu à cet effet, puis cliquez sur le bouton Ajouter une adresse IP.

    • Pour supprimer une adresse IP, cliquez sur l'adresse à supprimer dans la liste associée, puis cliquez sur le bouton Supprimer.

     

    Note :
    Pour autoriser l'utilisateur administrateur à se connecter localement à l'ordinateur Content Server, vous devez autoriser la connexion à partir de l'adresse IP 127.0.0.1.

     

  3. Cliquez sur le bouton OK.

  4. Redémarrez le serveur et, le cas échéant, le serveur d'application Web.
