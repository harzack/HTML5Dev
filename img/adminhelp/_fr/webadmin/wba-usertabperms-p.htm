Pour définir les autorisations à partir de l'onglet Utilisateur :

  1. Sur la page Administration, sélectionnez Administration d'utilisateurs et de groupes.

  2. Sélectionnez Autorisations à partir de l'onglet de l'utilisateur.

  3. Pour modifier les privilèges pour tous les utilisateurs, procédez comme suit :

    • Cliquez sur le lien <Accès public>.

    • Activez ou désactivez la case à cocher Autoriser la modification personnelle pour chaque onglet.

    • Cliquez sur le bouton Mettre à jour.

  4. Pour permettre à un nouvel utilisateur ou groupe de bénéficier des privilèges Autorisation de modifier n'importe qui, procédez comme suit :

    • Cliquez sur l'icône Accorder l'accès .

    • Recherchez l'utilisateur ou le groupe auquel vous souhaitez accorder l'accès.

    • Activez la case à cocher Accorder accès.

    • Cliquez sur le bouton Envoyer.

    • Activez la case à cocher Autoriser la modification de tous pour chaque onglet qui devra être disponible.

    • Cliquez sur le bouton Mettre à jour.

  5. Cliquez sur le bouton Terminé.
