Pour ajouter des catégories de documents :

  1. Accédez à la page d'administration, sous Configuration du serveur et sélectionnez Configurer les facettes. Sélectionnez Configurer des catégories de documents.

     

    Note :
    Vous pouvez aussi accéder aux Définitions de catégories de documents à partir de n'importe quelle page de l'espace de travail en sélectionnant Outils puis Volume à facettes dans la barre de menus globale. Sélectionnez le bouton des fonctions des Facettes, puis Panneau de configuration. Sélectionnez Configurer des catégories de documents.

     

  2. Pour ajouter une nouvelle catégorie de documents, dans la zone de saisie Catégories de documents, tapez le nom de la nouvelle catégorie de documents. Cliquez sur le bouton Ajouter une nouvelle catégorie de documents à droite de la zone de saisie Catégories de documents.

  3. Si vous préférez configurer la nouvelle catégorie de documents ultérieurement, cliquez sur le bouton OK pour l'enregistrer et revenir à la page Configurer les facettes.

     

    Note :
    Vous pouvez supprimer n'importe quelle catégorie de documents en cliquant sur le bouton de suppression, , situé dans le champ Catégories de documents sous la colonne Actions. Vous serez invité à confirmer si vous souhaitez supprimer la catégorie de documents. Cliquez sur OK pour supprimer la catégorie de documents. Une fois que vous avez cliqué sur OK dans la boîte de dialogue, la catégorie de documents est supprimée.

     

  4. Si vous souhaitez poursuivre et configurer votre nouvelle catégorie de documents immédiatement, cliquez sur le nom de la nouvelle catégorie de documents dans la section Nom de la catégorie. La page Définitions de raccourcis de type MIME s'affichera.

     

    Note :
    Vous pouvez choisir d'ajouter un nouveau raccourci de type MIME dans une catégorie de documents existante. Si vous voulez ajouter un nouveau raccourci de type MIME pour une catégorie de documents existante, cliquez sur le nom de cette catégorie de documents dans la section Nom de la catégorie. La page Définitions de raccourcis de type MIME s'affichera.

     

  5. Dans la page Définitions de raccourcis de type MIME, dans la zone de saisie Raccourcis de type MIME, tapez le nom de raccourci de votre nouvelle catégorie de documents. Cliquez sur le bouton Ajouter un nouveau raccourci, à droite de la zone de saisie Raccourcis de type MIME afin d'ajouter le nouveau raccourci à la liste Nom du raccourci.

    Si vous préférez configurer votre nouveau raccourci ultérieurement, cliquez sur le bouton OK pour le sauvegarder et retourner à la page Définitions de catégories de documents.

    Vous pouvez supprimer n'importe quel raccourci de type MIME en cliquant sur le bouton de suppression, , situé dans le champ Raccourcis de type MIME sous la colonne Actions. Vous serez invité à confirmer si vous souhaitez supprimer la catégorie de documents. Cliquez sur OK pour supprimer le raccourci de type MIME.

    Pour continuer et configurer vos nouveaux raccourcis, cliquez sur le nom du nouveau raccourci dans la section Nom du raccourci. La page Configurer le raccourci de type MIME s'affichera.

  6. Dans la page Configurer le raccourci de type MIME, sélectionnez les cases en regard de tous les types MIME que vous voulez ajouter à votre nouvelle catégorie de documents. Une fois que vous avez terminé, cliquez sur le bouton Mettre à jour de l'écran.
