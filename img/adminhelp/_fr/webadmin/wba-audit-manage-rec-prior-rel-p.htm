Note :
Vous pouvez gérer les enregistrements d'audit créés dans les versions précédentes après avoir mis à niveau une base de données à partir d'une version précédente de Content Server.

 

Pour gérer les enregistrements d'audit créés dans les versions précédentes :

  1. Sur la page Administration Content Server, dans la section Administration système, cliquez sur Gérer l'audit des événements.

  2. Sur la page Gérer l'audit des événements, cliquez sur Gérer les enregistrements d'audit créés dans les versions précédentes.

     

    Note :
    Cliquer sur l'une des options suivantes peut demander beaucoup de temps. Une fois votre sélection faite, une page de progression s'affiche. Patientez jusqu'à ce que la page de progression se mette à jour, puis cliquez sur Accueil Admin.

     

  3. Sur la page Gérer les enregistrements d'audit créés dans les versions précédentes, procédez de l'une des façons suivantes :

    • Pour convertir des enregistrements d'audit à partir de versions précédentes de Content Server, cliquez sur Convertir les enregistrements d'audit créés dans les versions précédentes.

    • Pour supprimer des enregistrements d'audit créés dans des versions précédentes de Content Server, cliquez sur Eliminer les enregistrements d'audit créés dans les versions précédentes.
