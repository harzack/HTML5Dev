Pour définir les événements à inclure dans l'audit :

  1. Cliquez sur le lien Gérer l'audit des événements de la section Administration système sur la page Administration.

  2. Sur la page Gérer l'audit des événements, cliquez sur Définir les centres d'intérêt à inclure dans l'audit.

  3. Sur la page Définir les centres d'intérêt à inclure dans l'audit, dans la section Evénements, pour ajouter un type d'événement que vous souhaitez auditer, sélectionnez la case à cocher du type d'événement.

    Pour arrêter l'audit d'un type d'événement, désactivez la case à cocher de ce type d'événement.

  4. Facultatif Dans la section Options, si vous souhaitez activer l'audit de toute action pouvant faire l'objet d'un audit réalisée par les utilisateurs bénéficiant du privilège Administration système, sélectionnez Forcer l'audit de tous les événements réalisés par des administrateurs système.

  5. Cliquez sur Définir les centres d'intérêt.

  6. Cliquez sur le lien Accueil Admin pour revenir à la page Administration Livelink.
