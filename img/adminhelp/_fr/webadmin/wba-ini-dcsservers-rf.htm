La section [DCSservers] compile les serveurs du service de conversion de documents (DCS) en cours d'exécution dans Content Server. Le DCS convertit des documents au format HTML ou texte brut pour permettre leur affichage et indexation. Les DCS sont gérés par les serveurs d'administration. Les clients de DCS, tels que LLView et la mise en évidence des résultats, utilisent cette section pour connaître les serveurs DCS disponibles. Cette liste est actualisée à chaque fois que Content Server doit charger un serveur d'administration.

Server1

  • Description :

    Fournit une liste de serveurs DCS disponibles.

    La valeur de ce paramètre est définie automatiquement et mise à jour régulièrement par le serveur d'administration.

     

    Important :
    Ne modifiez pas la valeur de ce paramètre, sauf si le service d'assistance clientèle d'OpenText vous le demande.

     

  • Valeurs :

    Un entier. La valeur de ce paramètre est générée automatiquement.

SpawnExe

  • Description :

    Indique le nom du binaire DCS utilisé par Content Server pour effectuer des opérations de conversion de documents. La valeur de ce paramètre est définie automatiquement et mise à jour régulièrement par le serveur d'administration.

     

    Important :
    Ne modifiez pas la valeur de ce paramètre, sauf si le service d'assistance clientèle d'OpenText vous le demande.

     

  • Valeurs :

    Sous Windows, la valeur par défaut est dcs.exe. Pour les autres plates-formes, la valeur par défaut est dcs.

SpawnIni

  • Description :

    Indique le fichier ini à utiliser lorsque le DCS n'est pas continu. La valeur de ce paramètre est définie automatiquement et mise à jour régulièrement par le serveur d'administration.

     

    Important :
    Ne modifiez pas la valeur de ce paramètre, sauf si le service d'assistance clientèle d'OpenText vous le demande.

     

  • Valeurs :

    Un chemin absolu. La valeur par défaut est <Content Server_home> /config/opentext.ini, où <Content Server_accueil> est la racine de votre installation Content Server.
