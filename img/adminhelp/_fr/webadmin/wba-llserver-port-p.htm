Pour spécifier le numéro de port du serveur :

  1. Cliquez sur le lien Spécifier le port du serveur de la section Configuration du serveur sur la page Administration.

  2. Entrez un numéro de port inutilisé et compris entre 1025 et 65535, qui sera le port récepteur du serveur, dans le champ Numéro de port.

    Sous UNIX, seul l'utilisateur racine dispose des privilèges nécessaires pour exécuter des processus sur les ports 1 à 1024. Pour cette raison, évitez d'utiliser cette série de numéros pour Content Server.

  3. Dans le champ Astuce concernant la famille, sélectionnez la famille d'adresses sur lesquelles Content Server doit recevoir des requêtes.

  4. Cliquez sur le bouton Enregistrer les modifications. Un message d'erreur s'affiche, indiquant que le serveur ne répond pas.

  5. Redémarrez le serveur et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer le serveur Content Server.
