Pour supprimer des tables Content Server :

  1. Sur la page Administration, sélectionnez Administration de la base de données.

  2. Sur la page Administration de la base de données, sélectionnez Maintenance de la base de données active.

  3. Sur la page Maintenance de la base de données active, sélectionnez Tâches de maintenance Microsoft SQL Server. Vous pouvez devoir saisir votre mot de passe administrateur MS SQL.

  4. Sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Supprimer les tables Content Server Microsoft SQL Server.

  5. Dans la liste Nom de la base de données, cliquez sur le nom de la base du serveur SQL qui contient les tables Content Server à supprimer.

  6. Dans la liste déroulante Nom de l'utilisateur, cliquez sur le nom d'utilisateur du serveur SQL utilisé par le serveur pour se connecter à cette base de données du serveur SQL.

  7. Dans le champ Mot de passe, saisissez le mot de passe de l'utilisateur SQL Server.

  8. Cliquez sur le bouton Supprimer les tables.

  9. Cliquez sur le bouton OK de la boîte de dialogue qui s'affiche.

  10. Si vous avez terminé les tâches qui vous intéressent sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Revenir à la page précédente.
