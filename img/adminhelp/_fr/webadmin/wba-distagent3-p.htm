Pour actualiser un traitement d'agent distribué :

  1. Sur la page Administration Content Server, sous le titre Administration système, cliquez sur Statut de l'agent distribué.

  2. Sur la page Statut de l'agent distribué, dans la liste déroulante située à droite de la page, sélectionnez l'une des options de l'heure pour actualiser.

  3. Cliquez sur Appliquer.

     

    Note :
    La sélection que vous avez faite dans la liste déroulante n'est pas enregistrée. Lorsque vous quittez la page Statut de l'agent distribué, la liste déroulante Actualiser rétablit le statut Ne pas actualiser par défaut.
