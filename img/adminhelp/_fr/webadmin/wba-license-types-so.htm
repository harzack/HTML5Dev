Vous pouvez afficher le type de licence dont vous disposez dans le champ Type de licence de la page Administration Gérer les licences. Trois types de licence sont disponibles :

Licence de production

Une licence de production Content Server offre des fonctionnalités complètes pour un nombre spécifié d'utilisateurs avec licence. Une licence de production est associée à une version spécifique de Content Server.

Licence temporaire

Une licence Content Server temporaire offre les mêmes fonctionnalités qu'une licence de production, mais comporte une date d'expiration. La date d'expiration est toujours une date spécifique, elle ne varie pas en fonction de la date à laquelle vous appliquez la licence temporaire.

Licence de non-production

Une licence de non-production offre les mêmes fonctionnalités qu'une licence de production, mais elle est délivrée pour une utilisation dans tous les environnements Content Server utilisés pour prendre en charge un environnement de production. Par exemple, vous pouvez appliquer une licence de non-production à un environnement de développement ou à un environnement de test d'acceptation utilisateur.
