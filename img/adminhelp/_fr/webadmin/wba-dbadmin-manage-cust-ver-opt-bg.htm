Exécution de jeux de tests de diagnostic prédéfinis

La page Vérifier la base de données Content Server comprend cinq boutons radio (de Diagnostic niveau 1 à Diagnostic niveau 5) qui permettent de sélectionner un ensemble de votre choix. Les tests exécutés à chaque niveau sont énumérés sous chaque bouton radio.

Exécution d'un jeu de tests de diagnostic personnalisé

Si aucun des ensembles de tests prédéfinis ne correspond à vos besoins, sélectionnez un ensemble de tests personnalisé sur la page Options de vérification personnalisées.

Pour accéder à la page Options de vérification personnalisées, cliquez sur le lien Options de vérification personnalisées de la page Vérifier la base de données Content Server. Suivant que votre base de données utilise un stockage des documents interne ou externe, Content Server affiche un ensemble d'options de test différent.

La page Rapport de vérification de la base de données Content Server initiale apparaît lorsque vous cliquez sur le bouton Etablir diagnostic de la page Vérifier base de données Content Server ou Options de vérification personnalisées (version de stockage interne ou externe).

La page Rapport de vérification de la base de données Content Server initiale vous avertit que la vérification est en cours et affiche un lien permettant de visualiser le statut ou les résultats de la vérification.

La deuxième page Rapport de vérification de la base de données Content Server apparaît lorsque vous cliquez sur le lien cliquez ici de la première page Rapport de vérification de la base de données Content Server.

Si la vérification est en cours, la deuxième page Rapport de vérification de la base de données Content Server affiche le statut du processus. Lorsque celui-ci est terminé, la page affiche les résultats. Si aucune erreur n'est détectée, elle affiche le message « Bravo ! Aucune erreur n'a été trouvée dans cette base de données. »

Options de vérification personnalisées

Pour accéder à la page Options de vérification personnalisées, cliquez sur le lien Options de vérification personnalisées de la page Vérifier la base de données Content Server.

L'ensemble de tests qui s'affiche varie selon que votre base de données Content Server est définie pour stocker les documents en interne ou en externe. Cette rubrique d'aide répertorie les tests disponibles pour une base de données de stockage interne. Les tests disponibles pour une base de données de stockage externe sont également répertoriés.

Si aucun des ensembles de tests prédéfinis de la page Vérifier la base de données Content Server ne correspond à vos besoins, sélectionnez un ensemble de tests personnalisé sur la page Options de vérification personnalisées.
