Pour désinstaller un module :

  1. Dans la section Administration du module de la page Administration, cliquez sur le lien Désinstaller des modules.

  2. Sur la page Désinstaller des modules, cliquez sur le bouton Désinstaller en regard du module que vous souhaitez supprimer.

  3. Dans la boîte de dialogue de confirmation, cliquez sur OK.

  4. Redémarrez le serveur et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer Content Server.
