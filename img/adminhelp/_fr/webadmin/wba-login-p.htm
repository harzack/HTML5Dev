Pour vous connecter à la page Administration Content Server :

  1. Ouvrez l'URL d'administration système Content Server dans un navigateur Web.

  2. Sur la page Connexion de l'administrateur, saisissez le mot de passe d'administration dans le champ Mot de passe.

  3. Cliquez sur le bouton Connexion.

 

Astuce :
Si vous avez installé Content Server sur Windows, vous pouvez également accéder à la page Administration en cliquant sur le menu Démarrer de Windows, sur Programmes, en ouvrant <nom_du_dossier_Content Server>, puis en cliquant sur Administration Content Server.
