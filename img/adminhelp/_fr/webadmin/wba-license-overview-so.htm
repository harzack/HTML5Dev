La Présentation de la licence Content Server résume les fonctionnalités de votre licence Content Server :

Etat

Pour plus d'informations sur les statuts des licences, voir Statuts des licences.

Nom du produit

Le produit disposant de cette licence OpenText : OpenText Content Server.

Version sous licence

La version du produit disposant de cette licence OpenText.

Type de licence

Pour plus d'informations sur les types de licence, voir Types de licence.

Nom de l'entreprise

Le nom de l'entreprise à laquelle la licence est attribuée.

Date d'expiration :

Si le type de votre licence est Temporary License, une date d'expiration apparaît.

Utilisateurs avec licence

Nombre maximum d'utilisateurs pris en charge par cette licence OpenText.

Utilisateurs actifs

Nombre d'utilisateurs existant actuellement dans Content Server.
