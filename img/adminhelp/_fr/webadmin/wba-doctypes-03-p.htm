Pour ajouter un nouveau fichier d'icônes :

  1. Créez un fichier d'icônes dans le format d'image General Internet Format, avec l'extension de fichier .gif.

  2. Créez quatre copies du nouveau fichier d'icônes comme suit :

    • Créez une icône de 16 × 16 pixels dans le répertoire <Content Server_home>/support/webdoc.

    • Créez une icône de 32 × 32 pixels dans le répertoire <Content Server_home>/support/webdoc.

    • Créez une icône de 32 × 32 pixels dans le répertoire <Content Server_home>/support/mimetypeimages.

    • Créez une icône de 145 × 145 pixels dans le répertoire <Content Server_home>/support/mimetypeimages.

  3. Associez le nouveau fichier d'icônes au(x) type(s) MIME souhaité(s). Pour en savoir plus, voir Pour modifier des correspondances icône-type MIME.
