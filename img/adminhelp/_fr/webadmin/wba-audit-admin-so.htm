Content Server permet de suivre les événements qui se produisent dans la base de données Content Server. Un événement est une action que les utilisateurs accomplissent sur un élément Content Server, tel qu'un document ou un dossier. L'audit des événements permet de contrôler le mode d'utilisation du système par les usagers de Content Server et de déterminer qui est à l'origine des opérations. Par exemple, il est possible d'identifier les utilisateurs qui ont supprimé ou déplacé des éléments.

A partir de la page Gérer l'audit des événements, vous pouvez exécuter les tâches suivantes :

  • Pour visualiser tous les centres d'intérêt actuellement inclus dans l'audit

  • Définition des centres d'intérêt à inclure dans l'audit

  • Gestion des enregistrements d'audit créés dans les versions précédentes

  • Consultation du journal de l'audit

  • Elimination des données du journal d'audit

  • Pour configurer les paramètres de sécurité de l'audit
