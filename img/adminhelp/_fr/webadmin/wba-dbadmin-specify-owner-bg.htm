La page Spécifier le propriétaire de la base de données Content Server apparaît lorsque vous connectez l'installation Content Server active à une base de données existante, après avoir cliqué sur le bouton Continuer de la page Sélectionner le type de SGBDR. Elle a pour but de vous demander le nom d'utilisateur et le mot de passe du compte de SGBDR auquel appartiennent les tables de la base de données Content Server.

Vous pouvez obtenir ces informations auprès de l'administrateur système responsable de la base de données.

Selon le type de votre SGBDR, passez à l'une des sections suivantes :

  • Spécification du propriétaire d'une base de données Oracle

  • Spécification du propriétaire d'une base de données Microsoft SQL Server
