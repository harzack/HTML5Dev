Pour ajouter un élément au menu Entreprise :

  1. Cliquez sur le lien Eléments de menu Entreprise supplémentaires, dans la section Configuration du serveur de la page Administration.

  2. Dans le champ Nom, saisissez le nom de l'élément. Il s'agit du texte que vous souhaitez afficher dans le menu Entreprise.

  3. Entrez une URL dans le champ URL.

  4. Cliquez sur Ajouter un élément de menu.

  5. Cliquez sur les boutons et pour déplacer l'élément vers le haut ou le bas dans le menu.

  6. Cliquez sur Envoyer.

  7. Redémarrez les serveurs.

 

Note :
Cliquez sur le bouton pour supprimer l'élément sélectionné du menu Entreprise.
