Pour installer un lot de langue pour un module Content Server :

  1. Téléchargez le lot de langue que vous souhaitez installer, puis déplacez-le vers l'emplacement C:\<Content Server_home>\langpkgstaging\module, où C:\<Content Server_home> correspond à l'emplacement de votre installation Content Server.

     

    Note :
    Ne changez pas le nom par défaut d'un lot de langue.

     

  2. Dans la section Langues de la page Administration Content Server, cliquez sur le lien Installer les lots de langues.

  3. Dans la section Lots de langues de module pouvant être installés de la page Installer les lots de langues, sélectionnez la case à cocher Installer pour chaque langue que vous souhaitez installer, puis cliquez sur le bouton Installer.

  4. Redémarrez vos serveurs lorsque vous y êtes invité.

  5. Cliquez sur Continuer.

 

Note :
Une fois que vous avez installé un lot de langue, vous devez l'activer dans la section Configurer les langues afin d'appliquer la nouvelle langue à l'ensemble du système. Pour plus d'informations sur l'activation des lots de langues, voir Configuration des langues.
