La section [Workflow] comprend des paramètres qui définissent le comportement du workflow.

ExcludedNodeSubTypes

  • Description :

    Détermine les éléments exclus de l'attribut de référence d'élément.

  • Syntaxe :

    ExcludedNodeSubTypes={731,732,900,901,902,903,904,905,906,919,920}

  • Valeurs :

    ID de type de nœud des éléments à exclure de l'attribut de référence d'élément. Les ID de types de nœuds que vous spécifiez pour ce paramètre doivent être compris entre des accolades et chaque ID de type de nœud doit être séparé du précédent par une virgule.

    Par exemple, le paramètre ExcludedNodeSubTypes={141,142} exclut l'espace de travail Entreprise, qui est l'ID de type de nœud 141, et l'espace de travail Personnel, qui est l'ID de type de nœud 142, de la liste d'éléments que vous pouvez conserver dans un attribut de référence d'élément.

IHLogging

  • Description :

    Contrôle la consignation de l'étape du gestionnaire d'élément.

  • Syntaxe :

    IHLogging={1}

  • Valeurs :

    • 0

      Désactive la consignation du gestionnaire d'élément avec horodatage.

    • 1

      Active la consignation du gestionnaire d'élément.

    • 11

      Active la consignation du gestionnaire d'élément avec horodatage.
