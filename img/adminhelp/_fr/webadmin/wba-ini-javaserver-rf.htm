La section [javaserver] vous permet de modifier les paramètres de la machine virtuelle s'exécutant dans Content Server. Vous pouvez optimiser les performances du serveur d'application Java Web en entrant de nouveaux arguments de machine virtuelle. Par exemple :
JavaVMOption_4=-Xms256M
JavaVMOption_5=-Xmx256M

 

Note :
Sur un serveur multiprocesseur de 64 bits, le processus Java utilise davantage de mémoire pour chaque unité centrale supplémentaire en cours d'utilisation. Pour limiter la quantité de mémoire utilisée par le javaserver lors d'un démarrage de Content Server, ajoutez l'argument suivant à la section [javaserver] du fichier opentext.ini : JavaVMOption_5=Xmx256M.

 

Pour plus d'informations sur les arguments que vous pouvez utiliser, identifiez votre installation Java SDK, puis accédez à <Java_SDK_installation_path> \jre\bin\client\Xusage.txt. Le fichier texte répertorie et présente tous les arguments.
