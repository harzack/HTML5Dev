Depuis la page Configurer les paramètres d'affichage, vous pouvez définir la valeur HREF DE BASE dans une page HTML incluant une vue personnalisée. Si aucune valeur n'est présente sur cette page, les valeurs par défaut dans la requête HTTP sont utilisées. Les paramètres que vous pouvez mettre à jour sont :

  • Si vous souhaitez ajouter des commentaires à chaque page HTML générée dans Content Server pour décrire quel fichier de modèle WebLingo a généré cette page, vous pouvez sélectionner la case à cocher Générer des commentaires WebLingo Filename. Principalement à des fins de débogage.

  • Protocole : une liste à partir de laquelle vous pouvez sélectionner http ou https.

  • Hôte : un champ de texte vous permettant de saisir le nom d'hôte du serveur à inclure dans la BASE HREF URL.

  • Port : un champ de texte vous permettant de saisir le port du serveur Web utilisé dans la BASE HREF URL.

Pour plus d'informations sur l'affichage de ces paramètres dans le fichier opentext.ini, voir [BaseHref].
