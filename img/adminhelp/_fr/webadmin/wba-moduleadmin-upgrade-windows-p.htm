Pour mettre à niveau un module sous Windows :

  1. Téléchargez et exécutez le fichier exécutable du module.

  2. Dans la fenêtre de bienvenue, cliquez sur le bouton Suivant.

  3. Dans la fenêtre de sélection de services Content Server, cochez la case en regard de l'instance de Content Server sur laquelle est installé le module actuel, puis cliquez sur le bouton Suivant.

  4. Cliquez sur le bouton Terminer.

  5. Dans la section Administration du module de la page Administration, cliquez sur le lien Mettre des modules à niveau.

  6. Sur la page Mettre des modules à niveau, sélectionnez la case à cocher située en regard de chaque module à mettre à niveau, puis cliquez sur le bouton Mettre à niveau.

  7. Redémarrez le serveur Livelink et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer le serveur Content Server.
