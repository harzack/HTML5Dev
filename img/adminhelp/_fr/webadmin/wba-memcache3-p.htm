Pour lancer ou arrêter un processus memcached :

  1. Sur la page Administration, dans la section Administration système cliquez sur Statistiques Memcached.

  2. Sur la page Statistiques Memcached, sous la colonne Nom, sélectionnez le nom du processus memcached que vous voulez lancer ou arrêter.

  3. Sur la page Propriétés <nom_processus_memcache>, cliquez sur l'onglet Spécifique.

  4. Facultatif Sous la section Actions, si le processus est arrêté, cliquez sur le bouton Démarrer. Si le processus est en cours d'exécution, cliquez sur le bouton Arrêter.

  5. Cliquez sur Mettre à jour.
