Le chemin d'accès par défaut de l'installation pour le fichier opentext.ini est <Content Server_home>/config/opentext.ini. Le fichier opentext.ini constitue le principal fichier de configuration Content Server pour les installations principales et secondaires de Content Server.

Le fichier opentext.ini contient des paramètres tels que les options de connexion aux bases de données, les chemins d'accès aux fichiers, les formats de date, les options de débogage et les options relatives aux journaux. Le fichier est créé automatiquement au cours de la procédure d'installation de Content Server. Les options par défaut sont alors définies et de nombreux paramètres sont configurés de manière dynamique. Certains paramètres, tels que le mot de passe codé de l'administrateur et les paramètres de notification, sont modifiés par Content Server si nécessaire.

La plupart des paramètres qui apparaissent dans le fichier opentext.ini peuvent être modifiés dans les pages Administration, mais certains paramètres doivent être modifiés manuellement en modifiant le fichier opentext.ini dans un éditeur de texte. Pour en savoir plus, voir Modification du fichier opentext.ini.
