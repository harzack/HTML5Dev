Pour configurer les paramètres d'affichage

  1. Cliquez sur le lien Configurer les paramètres d'affichage de la section Administration système sur la page Administration Content Server.

  2. Facultatif Pour ajouter des commentaires pour décrire quel fichier de modèle WebLingo a généré une page HTML Content Server particulière, sélectionnez la case à cocher Générer des commentaires WebLingo Filename.

  3. Facultatif Dans la section Paramètres Href de base :

    1. Dans le champ Protocole, sélectionnez http ou https dans la liste.

    2. Dans le champ Hôte, saisissez le nom de l'hôte.

    3. Dans le champ Port, saisissez le numéro de port.
