Pour associer un type MIME à une catégorie :

  1. Dans la section Administration système de la page Administration Content Server, cliquez sur le lien Gérer les types MIME et les catégories.

  2. Sur la page Gérer les types MIME et les catégories, cliquez sur l'icône Ajouter une catégorie, , pour ajouter des catégories existantes dans Content Server à la liste Catégories.

  3. Dans la liste Types MIME, mettez chaque type MIME que vous voulez associer en surbrillance. Mettez chaque catégorie à laquelle vous voulez associer le type MIME en surbrillance, puis cliquez sur l'icône Associer la catégorie.

  4. Cliquez sur Envoyer.
