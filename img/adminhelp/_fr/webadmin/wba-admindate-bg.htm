Vous pouvez configurer les formats utilisés pour la date et l'heure afin de les aligner sur ceux de votre entreprise. Si plusieurs langues sont activées, les formats de date et d'heure peuvent être configurés individuellement pour chaque langue. Vous pouvez spécifier le format des paramètres de date suivants :

  • Fuseau horaire, qui vous permet de configurer Content Server pour calculer et afficher automatiquement l'heure correcte dans le fuseau horaire local de chaque utilisateur. Par défaut, Content Server affiche l'heure locale à l'emplacement du serveur.

  • Entrée, qui spécifie la façon dont la date et l'heure doivent être entrées lorsque ces informations sont demandées par Content Server.

  • Format court, qui détermine comment Content Server affiche les dates au format court. Ce format comprend le jour de la semaine, qui précède l'affichage de la date.

  • Format long, qui détermine comment Content Server affiche les dates au format long.

Par défaut, les trois paramètres de date sont identiques : mois/jour/année, avec un mois à deux chiffres, une année à 4 chiffres, une horloge de 12 heures et la barre oblique (/) comme séparateur.
