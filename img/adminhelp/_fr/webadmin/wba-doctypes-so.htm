Par défaut, Content Server reconnaît une centaine de types MIME. MIME, pour Multipurpose Internet Mail Extensions, est une norme utilisée pour identifier les formats de fichiers. Vous pouvez modifier la liste des types MIME que Content Server reconnaît en modifiant le fichier Content Server_home/config/mime.types. Les types MIME de ce fichier sont affichés dans la liste Type MIME de l'onglet Spécifique de la page Propriétés d'un document.

De même, vous pouvez modifier la liste des correspondances icône-type MIME en modifiant le fichier Content Server_home/config/mime.gif.

Pour plus d'informations sur la modification d'autres fichiers de configuration système, reportez-vous à la rubrique Modification des fichiers de configuration système.
