Pour installer un module sous Windows :

  1. Téléchargez et exécutez le fichier exécutable du module.

  2. Dans la fenêtre de bienvenue du module, cliquez sur le bouton Suivant.

  3. Dans la fenêtre de sélection de services Content Server, activez la case à cocher en regard de l'instance de Content Server sur laquelle vous souhaitez installer le module, puis cliquez sur le bouton Suivant.

  4. Dans la fenêtre Installation terminée, cliquez sur le bouton Terminer.

  5. Dans la section Administration du module de la page Administration, cliquez sur le lien Installer des modules.

  6. Dans la section Modules que vous pouvez installer, cochez la case en regard du module à installer, puis cliquez sur le bouton Installer.

  7. Redémarrez le serveur et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer Content Server.
