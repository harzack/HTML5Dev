La page Connexion de l'administrateur de Microsoft SQL Server s'affiche dans les cas suivants :

  • Lorsque vous créez une base de données Content Server. La page apparaît une fois que vous cliquez sur le bouton Continuer de la page Sélectionner le type de SGBDR.

  • Lorsque vous avez cliqué sur le lien Tâches de maintenance Microsoft SQL Server sur la page Maintenance de la base de données active.

Sur cette page, vous devez indiquer le nom d'utilisateur et le mot de passe d'un compte SGBDR détenant les privilèges Administrateur.
