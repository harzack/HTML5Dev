Pour mettre à jour un lot de langue pour Content Server :

  1. Téléchargez le lot de langue mis à jour que vous souhaitez installer et décompressez-le dans le répertoire d'installation racine de Content Server.

     

    Note :
    Ne changez pas le nom par défaut du lot de langue mis à jour.

     

  2. Dans la section Langues de la page Administration Content Server, cliquez sur le lien Installer les lots de langues.

  3. Dans la section Lots de langues Content Server pouvant être installés de la page Installer les lots de langues, sélectionnez la case à cocher Installer pour chaque mise à jour de lot de langue que vous souhaitez installer, puis cliquez sur le bouton Installer.

  4. Redémarrez vos serveurs lorsque vous y êtes invité.

  5. Cliquez sur Continuer.
