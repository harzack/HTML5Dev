Pour vous connecter au serveur Microsoft SQL Server :

  1. Dans le champ Nom du serveur SQL, saisissez le raccourci de Microsoft SQL Server.

  2. Dans le champ Utilisateur système, saisissez le nom d'utilisateur de l'administrateur système de Microsoft SQL Server (sa par défaut).

  3. Dans le champ Mot de passe système, saisissez le mot de passe de l'utilisateur système.

  4. Dans le champ Nom de la base de données principale, saisissez le nom de la base principale (master par défaut).

  5. Cliquez sur le bouton Connexion. Une des pages suivantes apparaît :

    • Créer des tables Content Server

    • Maintenance Microsoft SQL Server
