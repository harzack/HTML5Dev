Content Server vous permet de stocker des documents dans plusieurs emplacements en dehors de Content Server. Par défaut, les documents Content Server sont stockés dans un répertoire de stockage externe créé au moment de l'installation de Content Server. Les fournisseurs de stockage et leurs règles peuvent être ajoutés, supprimés ou modifiés par l'administrateur Content Server.

 

Note :
Certains fournisseurs de stockage sont proposés avec Content Server, mais d'autres fournisseurs de stockage requièrent l'installation d'un logiciel supplémentaire. Par exemple, le fournisseur de stockage d'archives requiert l'installation de l'archivage OpenText pour Content Server.
