Pour créer des tables Content Server sous un SGBDR Microsoft SQL Server :

  1. Sur la page Créer des tables Content Server, cliquez sur le bouton radio correspondant à votre type de SGBDR.

  2. Sur la page Créer des tables Content Server, cliquez sur le nom de la base de données Microsoft SQL Server dans laquelle vous souhaitez créer les tables.

  3. Sur la page Créer les tables Content Server, cliquez sur le nom de l'utilisateur de Microsoft SQL Server qui sera propriétaire des tables dans la liste Nom de l'utilisateur Microsoft SQL. (L'utilisateur sélectionné doit avoir des privilèges de connexion sur la base SQL serveur choisie.)

  4. Dans le champ Mot de passe, saisissez le mot de passe de l'utilisateur Microsoft SQL Server sélectionné.

  5. Si vous voulez que Content Server enregistre des documents ou d'autres éléments en dehors de la base de données, désactivez la case à cocher Documents externes puis entrez le chemin d'accès absolu du répertoire où vous voulez que la base de données Content Server enregistre les éléments.

  6. Cliquez sur le bouton Créer les tables.

    Selon que vous avez choisi un stockage interne ou externe et qu'il s'agit d'une installation Content Server nouvelle ou existante, une des pages suivantes apparaît :

    • Répertoire de documents externe

    • Installation de modules

    • Index Entreprise - Créer une nouvelle mise à jour
