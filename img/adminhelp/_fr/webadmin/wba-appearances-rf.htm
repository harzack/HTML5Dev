Une apparence est un recueil qui stocke les documents et le code HTML. Elle permet de personnaliser certains emplacements, un peu comme vous pouvez concevoir une page Web. Pour plus d'informations sur les apparences, reportez-vous à la rubrique Utilisation des apparences.

Les apparences générales, destinées à être appliquées à plusieurs emplacements distincts, doivent être ajoutées au volume Apparences. Pour ajouter des apparences générales, vous devez disposer de l'autorisation Ajouter les éléments sur le volume des apparences.

Le volume Apparences est accessible à partir de la page Administration.

  1. Sélectionnez Administration des apparences.

  2. Ensuite, sélectionnez Ouvrir le volume Apparences.

  3. Pour créer une apparence générale, cliquez sur le bouton Ajouter un élément et sélectionnez Apparence générale.

  4. Entrez un nom pour la nouvelle apparence générale dans le champ Nom. Vous pouvez également saisir ou sélectionner des éléments dans les champs Description, Catégories et Créer dans.

  5. Cliquez sur le bouton Ajouter.
