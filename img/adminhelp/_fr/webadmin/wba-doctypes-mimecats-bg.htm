Vous pouvez associer n'importe quel type MIME disponible à une catégorie existante. Lorsqu'un type MIME est associé à plus d'une catégorie et que les utilisateurs ajoutent des objets de ce type MIME, ils sont redirigés vers une deuxième page pendant le processus Ajouter un élément qui leur permet de choisir la catégorie applicable à l'élément.

Pour autoriser les utilisateurs à choisir une catégorie lorsqu'ils ajoutent un élément, si l'élément qu'ils ajoutent présente un type MIME avec plus d'une catégorie lui étant associée, vous devez sélectionner la case à cocher Afficher les catégories sur une deuxième page Ajouter un élément sur la page Gérer le contrôle d'élément. Pour plus d'informations sur les paramètres de contrôle d'élément, voir Gestion du contrôle d'élément.
