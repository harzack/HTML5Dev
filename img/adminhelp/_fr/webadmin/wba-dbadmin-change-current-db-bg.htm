Pour de multiples raisons, vous pouvez avoir besoin de vous connecter à une base de données Content Server autre que celle à laquelle l'installation Content Server actuelle est connectée. Vous avez alors la possibilité de vous connecter à une base nouvelle ou à une base existante. La première opération pour procéder au changement consiste à désélectionner la base de données courante. Pour enregistrer les index de la base de données courante, créez une copie de sauvegarde des répertoires indiqués.

Modification du mot de passe utilisé par Content Server pour se connecter à la base de données

Vous pouvez modifier le mot de passe utilisé par Content Server pour se connecter à la base de données sur les pages Administration Content Server.
