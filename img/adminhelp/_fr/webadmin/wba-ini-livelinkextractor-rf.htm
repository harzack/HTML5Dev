Ce paramètre contrôle l'ordre dans lequel l'Extracteur traite les informations dans la table DTreeNotify. Un processus Extracteur Entreprise est un processus de flux de données qui extrait des données de la base de données Content Server et les écrit dans un flux de données, où il est indexé. La plupart des paramètres de l'Extracteur sont accessibles dans Content Server sur la page Administration des paramètres généraux de l'Extracteur. Pour en savoir plus, consultez le document Définition des paramètres généraux de l`Extracteur.

 

Important :
Un seul processus étant requis pour extraire les données de la base de données Content Server, OpenText vous recommande vivement de ne prévoir qu'un processus Extracteur Entreprise pour chaque système Content Server.

 

wantDescendingExtractor

  • Description :

    Définit l'ordre dans lequel l'Extracteur traite les informations dans la table DTreeNotify. Vous pouvez traiter les informations des mises à jour les plus récentes aux plus anciennes ou des plus anciennes aux plus récentes.

     

    Note :
    Pour définir une règle de contrôle de création de partitions selon la date, l'ordre d'extraction décroissant doit être défini sur false.

     

  • Syntaxe :

    wantDescendingExtractor=TRUE

  • Valeurs :

    TRUE ou FALSE. Par défaut, ce paramètre n'apparaît pas dans le fichier opentext.ini, ce qui équivaut au paramètre wantDescendingExtractor=TRUE.

    Pour traiter les informations des plus anciennes aux plus récentes, vous devez définir le paramètre sur wantDescendingExtractor=FALSE.
