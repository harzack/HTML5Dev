Vous pouvez définir des règles binaires individuelles et créer une association ordonnée entre les règles et les fournisseurs de stockage afin de déterminer l'emplacement de stockage des documents. Pour modifier les fournisseurs de stockage, les règles et leurs associations, il n'est pas nécessaire de redémarrer les serveurs ou le serveur d'application Web.

Vous pouvez configurer autant de règles de stockage que vous le souhaitez. Cependant, OpenText vous recommande de maintenir le nombre de règles de stockage en dessous de 12. Les règles de stockage sont évaluées dès qu'un document est ajouté à Content Server ou dès le démarrage d'une tâche de déplacement de contenu. Lorsque vous ajoutez un document, les règles sont évaluées dans l'ordre à partir du haut de la liste. Si le document ne répond à aucune des règles définies, il est stocké dans le fournisseur de stockage par défaut, qui apparaît toujours à la fin de la liste et que vous ne pouvez pas configurer ou supprimer.

Une règle de stockage est constituée d'un nom de règle, d'une ou de plusieurs conditions et d'une cible de stockage sélectionnée si toutes les conditions sont remplies. Après la création d'une règle, vous pouvez la modifier, la supprimer et modifier son ordre dans la liste. Lorsque vous ajoutez une nouvelle règle, celle-ci apparaît au-dessus de la règle actuelle par défaut.

Pour chaque type de règle que vous ajoutez, vous spécifiez la valeur de la règle, représentée ci-dessous par le signe ? dans le champ Valeur.

Les règles suivantes sont disponibles par défaut lorsque Content Server est installé pour la première fois :

  • La taille du fichier en octets est supérieure à '?'

  • Le nom de la catégorie est '?'

  • Le type Mime est '?'

  • La taille du fichier en octets est inférieure à '?'

  • Le nom du noeud contient '?'

  • Toujours (valeur ignorée)

  • Toute autre valeur d'attribut du noeud est '?'

  • La valeur de l'attribut ? non spécifique est ?

  • La valeur de l'attribut ? est ?

  • Toutes les miniatures

  • Espace disponible du volume (Mo)

 

Note :
Si aucune règle n'a été définie, seule l'icône Ajouter apparaît dans la colonne Modifier de la page Configurer les règles de stockage.
