Si vous rencontrez des difficultés au niveau de votre base de données Content Server, utilisez les tests de diagnostic proposés sur la page Vérifier la base de données Content Server pour contrôler divers aspects de la base. Pour accéder à cette page, cliquez sur le lien Vérifier cette base de données sur la page Maintenance de la base de données active.

Vous avez la possibilité d'exécuter des tests de diagnostic prédéfinis ou personnalisés.
