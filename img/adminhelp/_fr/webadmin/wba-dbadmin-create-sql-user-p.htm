Pour créer un utilisateur SQL Server :

  1. Sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Créer un nouvel utilisateur.

  2. Entrez le nom à attribuer au nouvel utilisateur SQL Server dans le champ Nom de l'utilisateur.

  3. Entrez le mot de passe du nouvel utilisateur du serveur SQL dans le champ Mot de passe.

  4. Saisissez une nouvelle fois le mot de passe dans le champ Confirmation.

  5. Cliquez sur le nom de la base de données du serveur SQL qui va contenir les tables de cet utilisateur dans la liste déroulante Nom de la base de données.

  6. Cliquez sur le bouton Créer un utilisateur.

  7. Procédez de l'une des manières suivantes :

    • Si vous créez une nouvelle base Content Server, cliquez sur le lien Revenir à la page précédente pour revenir à la page Créer des tables Content Server.

    • Sinon, si vous avez terminé les tâches que vous vouliez accomplir sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Revenir à la page précédente pour revenir à la page.
