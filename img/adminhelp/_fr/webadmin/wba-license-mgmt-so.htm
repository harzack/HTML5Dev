Sur la page Gestion des licences, vous pouvez acquérir une licence ou appliquer un fichier de licence.

Pour obtenir un fichier de licence :

  1. Sur la page Configuration de la licence, une empreinte système identifiant votre installation Content Server de manière unique apparaît. Copiez l'empreinte système afin de pouvoir l'utiliser pour générer un fichier de licence pour Content Server ou pour un module Content Server facultatif.

  2. Connectez-vous au site http://productactivation.opentext.com/ContentServer à l'aide de l'ouverture de session et du mot de passe qui vous ont été fournis lorsque vous avez acheté Content Server.

  3. Si vous disposez de plusieurs environnements Content Server, sélectionnez l'environnement auquel s'applique la licence.

     

    Astuce :
    Une seule licence est requise pour plusieurs instances de Content Server qui se connectent à la même base de données.

     

  4. Sélectionnez le produit et le type de fichier de licence appropriés et utilisez votre empreinte système pour générer un fichier de licence.

Pour appliquer un fichier de licence :

  1. En regard d'Emplacement de la licence, cliquez sur Choix du fichier.

  2. Recherchez l'emplacement de votre fichier de licence et sélectionnez-le.

    La page s'actualise et une section Détails du nouveau fichier de licence apparaît. Cette section contient des informations sur le fichier de licence que vous allez appliquer.

  3. Pour appliquer cette licence, cliquez sur Appliquer le fichier de licence.

    Lorsque le fichier de licence est appliqué avec succès, la page Gérer les licences apparaît, et Valid apparaît dans le champ Statut de la Présentation de la licence Content Server.

  4. Si vous avez appliqué une licence de module, redémarrez Content Server. (Si vous avez appliqué une licence Content Server globale, il n'est pas nécessaire de redémarrer Content Server.)
