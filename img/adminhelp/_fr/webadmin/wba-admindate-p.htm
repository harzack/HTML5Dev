Pour définir des formats de date et d'heure :

  1. Dans la page Administration, sous Configuration du serveur ou Langues, cliquez sur le lien Gérer la date/l'heure.

  2. Dans la section Configuration du fuseau horaire, pour remplacer l'heure du serveur, cochez la case Activer le décalage horaire.

  3. Dans la section Format de date Entrée, effectuez les opérations suivantes :

    • Dans la liste déroulante Ordre des dates, choisissez l'ordre dans lequel les champs d'entrée du jour, du mois et de l'année apparaîtront. Le paramètre par défaut lors de l'installation est MM/JJ/AAAA. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

    • Cochez la case Année en 2 parties pour obtenir deux listes déroulantes pour les saisies d'année. Une liste se référera au siècle, 19 et 20. L'autre à la décennie, 00 à 99. Par défaut, pour ce paramètre, la case n'est pas cochée.

    • Cochez la case Horloge à cycle de 24 heures pour afficher les heures au format 24 heures, par exemple, 14:41. Le paramètre par défaut affiche l'heure au format 12 heures, par exemple, 02:41. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

  4. Dans la section Format de date Format court, effectuez les opérations suivantes :

    • Dans la liste déroulante Ordre des dates, choisissez l'ordre dans lequel les champs d'affichage du jour, du mois et de l'année apparaîtront. Le paramètre par défaut lors de l'installation est MM/JJ/AAAA. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

    • Sélectionnez la case à cocher Année à 2 chiffres pour afficher les années sous forme de nombre à deux chiffres (par exemple, 09 pour 2009). Le paramètre par défaut à l'installation affiche l'année sous forme de nombre à quatre chiffres (par exemple, 2009 pour 2009). Toute modification que vous apportez est affichée dans le champ Exemple ci-dessous.

    • Entrez les caractères à utiliser pour séparer les éléments de la date dans les champs Séparateurs de date. Dans le premier champ, entrez le caractère à utiliser entre le premier et le deuxième élément. Dans le second, entrez le caractère à utiliser entre le deuxième et le troisième élément. Le paramètre par défaut à l'installation utilisera le caractère slash, /, comme séparateur. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

  5. Dans la section Format de date Format long, effectuez les opérations suivantes :

    • Dans la liste déroulante Ordre des dates, choisissez l'ordre dans lequel les champs d'affichage du jour, du mois et de l'année apparaîtront. Le paramètre par défaut lors de l'installation est MM/JJ/AAAA. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

    • Sélectionnez la case à cocher Année à 2 chiffres pour afficher les années sous forme de nombre à deux chiffres (par exemple, 09 pour 2009). Le paramètre par défaut à l'installation affiche l'année sous forme de nombre à quatre chiffres (par exemple, 2009 pour 2009). Toute modification que vous apportez est affichée dans le champ Exemple ci-dessous.

    • Entrez les caractères à utiliser pour séparer les éléments de la date dans les champs Séparateurs de date. Dans le premier champ, entrez le caractère à utiliser entre le premier et le deuxième élément. Dans le second, entrez le caractère à utiliser entre le deuxième et le troisième élément. Le paramètre par défaut à l'installation utilisera le caractère slash, /, comme séparateur. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

    • Cochez la case Horloge à cycle de 24 heures pour afficher les heures au format 24 heures, par exemple, 14:41. Le paramètre par défaut affiche l'heure au format 12 heures, par exemple, 02:41. Tout changement que vous pouvez apporter s'affichera dans le champ Exemple ci-dessous.

  6. Cliquez sur le bouton OK pour enregistrer les modifications. Vous devrez redémarrer votre serveur d'administration et le serveur d'application Web une fois que vous aurez modifié les formats de date et d'heure. Une fois les serveurs redémarrés, cliquez sur le lien Continuer de la page Redémarrer Content Server.
