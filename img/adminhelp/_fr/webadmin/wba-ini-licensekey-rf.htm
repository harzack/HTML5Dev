The parameters in the [LicenseKeyInfo] section are set at installation time.

To modify these parameters, OpenText recommends that you use the Configure Server Parameters page, rather than edit the <Content Server_home>/config/opentext.ini file directly. For further information, see Configuring Basic Server Parameters.

CompanyName

  • Description:

    The name of your organization.

  • Syntax:

    CompanyName=ABC Company

  • Values:

    Characters and integers.

ExpireDate

  • Description:

    The number of days after which the license expires.

  • Syntax:

    ExpireDate=1000

  • Values:

    Integers greater than, or equal to, 1.

NumUsers

  • Description:

    The number of users permitted by the license.

  • Syntax:

    NumUsers=1000

  • Values:

    Integers greater than, or equal to, 1.
