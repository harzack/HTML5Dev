Pour configurer le menu Fonctions :

  1. Dans la section Configuration du serveur sur la page Administration, cliquez sur le lien Configurer le menu Fonctions.

  2. Sur la page Configurer le menu Fonctions, cliquez sur l'un des boutons radio suivants pour chacune des fonctions :

    • Principal, qui affiche la fonction dans la section principale du menu Fonctions.

    • Plus, qui affiche la fonction dans la section masquée du menu Fonctions.

  3. Cliquez sur le bouton Enregistrer les modifications.

 

Note :
Lorsque Content Server est mis à niveau vers la version 10.0.0, l'utilisateur administrateur doit aller sur la page Configurer le menu Fonctions et cliquer sur le bouton Enregistrer les modifications pour que le menu Plus apparaisse dans Content Server.
