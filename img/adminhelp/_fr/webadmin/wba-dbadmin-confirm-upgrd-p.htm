Pour confirmer la mise à niveau :

  1. Sur la page Confirmation de la mise à niveau de la base de données Content Server, cliquez sur le bouton radio Mettre la base de données Livelink <nom_base_de_données> Content Server à niveau.

  2. Cliquez sur le bouton Continuer.

  3. Redémarrez le serveur et le serveur d'application Web.

  4. Cliquez sur le lien Continuer de la page Redémarrer le serveur. La page Statut de la mise à niveau de la base de données s'affiche.
