Lorsqu'un utilisateur supprime un document, Content Server le place dans le volume des objets supprimés au lieu de le supprimer du système. La fonction Récupération d'un document permet à tout utilisateur bénéficiant des droits d'administration système de restaurer les documents supprimés du volume des objets supprimés vers les espaces de travail de Content Server.

La fonction Récupération d'un document ne s'applique qu'aux documents Content Server et à aucun autre type d'élément, notamment les documents composites.

Lorsque les utilisateurs suppriment des présentations ou des versions de documents, les éléments supprimés ne sont pas ajoutés au volume des objets supprimés. La fonction de récupération d'un document est disponible uniquement lorsqu'un utilisateur supprime le nœud du document.

En tant qu'utilisateur bénéficiant de droits d'administration système, vous pouvez visualiser tous les raccourcis des documents supprimés jusqu'à ce que le document soit éliminé définitivement.
