Pour spécifier le propriétaire d'une base de données Microsoft SQL Server :

  1. Dans le champ Nom du serveur SQL, saisissez le raccourci de Microsoft SQL Server.

  2. Dans le champ Nom de l'utilisateur MSSql, saisissez le nom de l'utilisateur Microsoft SQL Server qui peut se connecter à la base de données Microsoft SQL Server que vous sélectionnez.

  3. Dans le champ Mot de passe, saisissez le mot de passe de l'utilisateur Microsoft SQL Server.

  4. Dans le champ Nom de la base de données SQL, saisissez le nom de la base de données Microsoft SQL Server à sélectionner.

  5. Cliquez sur le bouton Continuer. Selon qu'il s'agit d'une installation Content Server nouvelle ou existante, que la base a été créée initialement sur l'hôte courant ou non et qu'elle a ou non besoin d'être mise à niveau, une des pages suivantes s'affiche :

    • Confirmation de la mise à niveau de la base de données Content Server

    • Installation de modules

    • Gestion des serveurs d'administration
