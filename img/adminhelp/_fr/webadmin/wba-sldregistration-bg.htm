System Landscape Directory (SLD) est un référentiel central des informations relatives à tous les produits ou modules installés sur Netweaver. Vous pouvez utiliser la page Enregistrement SLD dans Content Server pour générer un fichier SLD avec les informations de Content Server. Lorsque le fichier SLD est généré, Netweaver utilise les informations pour ajouter Content Server à son SLD.
