Pour ajouter une facette :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Volume Facettes.

  2. Accédez au dossier de facette personnalisé dans lequel vous souhaitez créer une nouvelle facette ou créez un dossier de facette dans le volume à facettes, puis accédez à ce nouveau dossier.

  3. Dans le menu Ajouter un élément, sélectionnez Facette.

  4. Dans le champ Nom, saisissez un nom unique pour la nouvelle facette.

  5. Facultatif Dans le champ Description, saisissez la description de la nouvelle facette.

  6. Dans la liste Source de données, choisissez une source de données pour la nouvelle facette.

  7. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à cette facette.

  8. Facultatif Pour placer la facette dans un emplacement autre que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server..., naviguez jusqu'au recueil dans lequel vous voulez placer la facette, puis cliquez sur le lien Sélectionner. Seules les zones de Content Server permettant d'enregistrer une facette peuvent être sélectionnées.

  9. Cliquez sur Ajouter pour enregistrer votre nouvelle facette et revenir au volume Facettes.
