Dans les pages Administration, vous pouvez éliminer automatiquement plusieurs documents de Content Server en fonction de la date de suppression des documents.

Tous les jours, Content Server supprime tous les documents du volume des objets supprimés qui sont restés là plus longtemps que le nombre de jours spécifié.

 

Note :
L'Agent client 5 minutes doit être activé pour utiliser la fonction Elimination automatique. Pour en savoir plus, voir Configuration des activités programmées.
