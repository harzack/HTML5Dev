Pour modifier le format de date pour les facettes :

  1. Ouvrez le fichier opentext.ini à des fins de modification et placez le paramètre suivant sous [Lang_en_US] : MonthYearFormat=%B %Y. L'exemple donné ici devrait permettre d'afficher le nom complet du mois suivi de l'année en quatre chiffres.

  2. Après avoir modifié le fichier opentext.ini, vous devez redémarrer le serveur et le serveur d'application Web pour appliquer les modifications.
