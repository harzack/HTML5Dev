Pour afficher ou modifier les propriétés d'un processus memcached :

  1. Sur la page Administration, dans la section Administration système cliquez sur Statistiques Memcached.

  2. Sur la page Statistiques Memcached, sous la colonne Nom, cliquez sur le nom du processus memcached que vous voulez afficher.

  3. Sur la page Propriétés <nom_processus_memcache>, cliquez sur l'onglet Général. Effectuez ensuite l'une des actions suivantes :

    1. Facultatif Si vous voulez changer le nom du processus, dans le champ Nom, saisissez le nouveau nom du processus.

    2. Facultatif Dans le champ Description, saisissez une description du processus.

  4. Cliquez sur Mettre à jour.
