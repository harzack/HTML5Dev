Pour agrandir une base de données SQL Server :

  1. Sur la page Administration, sélectionnez Administration de la base de données.

  2. Sur la page Administration de la base de données, sélectionnez Maintenance de la base de données active.

  3. Sur la page Maintenance de la base de données active, sélectionnez Tâches de maintenance Microsoft SQL Server. Vous pouvez devoir saisir votre mot de passe administrateur MS SQL.

  4. Sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Agrandir une base de données Microsoft SQL Server.

  5. Dans la liste Base de données à agrandir, cliquez sur le nom de la base de données du serveur SQL concernée.

  6. Dans la liste Fichier de données, cliquez sur le nom du fichier de données dans lequel un espace supplémentaire doit être attribué à cette base de données de serveur SQL.

  7. Dans le champ Attribution du fichier de données, saisissez la quantité d'espace supplémentaire (en Mo) à attribuer à cette base de données de serveur SQL dans le fichier de données.

  8. Cliquez sur le bouton Agrandir base de données.

  9. Si vous avez terminé les tâches qui vous intéressent sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Revenir à la page précédente.
