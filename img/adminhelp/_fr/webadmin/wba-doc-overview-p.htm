Pour activer ou désactiver la page de vue d'ensemble d'un document :

  1. Dans la section Configuration du serveur de la page Administration, cliquez sur le lien Configurer la présentation.

  2. Sur la page Configurer la présentation, cliquez sur le lien Configurer la fonction de vue d'ensemble d'un document.

  3. Sur la page Configurer la fonction de vue d'ensemble d'un document, cliquez sur l'un des boutons radio suivants :

    • Activer, pour activer les pages de vue d'ensemble d'un document et d'une version en tant que cibles par défaut.

    • Désactiver, pour désactiver les pages de vue d'ensemble d'un document et d'une version en tant que cibles par défaut.

  4. Cliquez sur le bouton Enregistrer les modifications pour enregistrer vos changements et revenir à la page Administration, ou cliquez sur le bouton Annuler pour réinitialiser la page sur ses valeurs précédentes.
