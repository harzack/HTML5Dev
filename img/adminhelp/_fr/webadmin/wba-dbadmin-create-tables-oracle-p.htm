Pour créer des tables Content Server sur un SGBDR serveur Oracle :

  1. Sur la page Créer des tables Content Server, cliquez sur le bouton radio correspondant à votre type de SGBDR.

  2. Sur la page Créer les tables Content Server, cliquez sur le nom de l'utilisateur Oracle qui sera propriétaire des tables dans la liste Nom de l'utilisateur.

  3. Dans le champ Mot de passe, saisissez le mot de passe de l'utilisateur Oracle sélectionné.

  4. Si vous voulez que Content Server enregistre des documents ou d'autres éléments en dehors de la base de données, désactivez la case à cocher Stockage de documents externe puis entrez le chemin d'accès absolu du répertoire où vous voulez que la base de données Content Server enregistre les éléments.

  5. Cliquez sur le bouton Créer les tables.

    Selon que vous avez choisi un stockage interne ou externe et qu'il s'agit d'une installation Content Server nouvelle ou existante, une des pages suivantes apparaît :

    • Répertoire de documents externe

    • Installation de modules

    • Index Entreprise - Créer une nouvelle mise à jour
