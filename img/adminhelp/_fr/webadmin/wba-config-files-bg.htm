Certains paramètres de configuration système ne peuvent pas être définis à partir des pages Administration. Dans ce cas, vous devez modifier un fichier de configuration système directement pour implémenter une modification. Le fichier opentext.ini le plus souvent utilisé pour les configurations manuelles. L'emplacement par défaut du fichier opentext.ini est installation_défaut_répertoire/config/. Les fichiers mime.types et mime.gif, situés dans le répertoire config, sont également modifiés directement.

Lorsque vous modifiez directement le fichier opentext.ini dans un éditeur de texte, vous devez arrêter, puis redémarrer les services Content Server et le serveur d'application Web pour appliquer les modifications.
