Pour purger les documents du volume des objets supprimés :

 

Attention
Une fois que vous avez éliminé les documents supprimés, ils sont définitivement supprimés de Content Server.

 

  1. Cliquez sur le lien Eliminer les documents supprimés, dans la section Administration de la récupération de documents de la page Administration.

  2. Sur la page Eliminer les documents supprimés, saisissez un entier dans le champ Jours d'existence. La valeur par défaut est 7.

  3. Cliquez sur le bouton Eliminer.

    Content Server supprime tous les documents du volume des objets supprimés qui sont restés là plus longtemps que le nombre de jours spécifié.
