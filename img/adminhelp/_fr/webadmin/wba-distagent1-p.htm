Pour afficher les traitements d'agent distribués :

  1. Sur la page Administration Content Server, sous le titre Administration système, cliquez sur Statut de l'agent distribué.

  2. Sur la page Statut de l'agent distribué vous verrez s'afficher :

    • ID de traitement : répertorie les ID des traitements d'agent distribués.

    • Dernière mise à jour : affiche la date et l'heure auxquelles le statut de l'ID de traitement a été mis à jour pour la dernière fois.

    • Statut : affiche le statut pour chaque processus de traitement. Parmi les valeurs possibles figurent Inactif, Attente et En cours d'exécution.

    • Description du statut : décrit le statut de l'ID de traitement.

    • Actions : affiche les actions que vous pouvez appliquer à chaque traitement. Parmi les valeurs possibles figurent Pause et Reprendre.
