Pour spécifier le propriétaire d'une base de données Oracle :

  1. Dans le champ Nom de l'utilisateur Oracle, saisissez le nom de l'utilisateur Oracle qui est le propriétaire des tables de la base Content Server à sélectionner.

  2. Dans le champ Mot de passe, saisissez le mot de passe de l'utilisateur Oracle.

  3. Dans le champ Nom de service, saisissez le nom de service (raccourci de base de données) du serveur Oracle.

  4. Cliquez sur le bouton Continuer. Selon qu'il s'agit d'une installation Content Server nouvelle ou existante, que la base a été créée initialement sur l'hôte courant ou non et qu'elle a ou non besoin d'être mise à niveau, une des pages suivantes s'affiche :

    • Confirmation de la mise à niveau de la base de données Content Server

    • Installation de modules

    • Confirmation du serveur d'administration par défaut

    • Confirmation de la resynchronisation de la source de données
