La page de sélection du service vous permet de configurer la manière dont les groupes sont assignés aux utilisateurs dans Content Server. Vous pouvez spécifier l'une des options suivantes :

  • Liste déroulante, sélectionnée par défaut. Lorsque cette option est sélectionnée, les pages Créer un utilisateur et Modifier l'utilisateur utilisent une liste déroulante pour regrouper tous les groupes publics extraits du système. Lorsqu'un utilisateur est créé, tous les groupes publics sont répertoriés.

  • Boîte de dialogue Service, utilisée pour rechercher et sélectionner un service auquel un utilisateur peut être assigné. Lorsque vous sélectionnez cette option, les pages Créer un utilisateur et Modifier l'utilisateur contiennent le champ supplémentaire Service, qui permet de rechercher un groupe particulier et de le sélectionner comme service de l'utilisateur. Cette option permet au système d'extraire les utilisateurs d'un service dans un ensemble restreint de groupes publics, plutôt que d'extraire tous les groupes publics du système Content Server. Cette option est adaptée aux systèmes comportant un grand nombre de groupes.

 

Note :
Lorsque la sélection du service est modifiée, l'événement d'audit Configuration modifiée est enregistré. Pour en savoir plus, voir Gestion des centres d'intérêt de l'audit.
