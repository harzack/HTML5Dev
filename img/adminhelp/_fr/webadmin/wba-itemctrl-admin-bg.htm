La page Gérer le contrôle d'élément vous permet d'effectuer les actions suivantes :

  1. personnaliser les fonctions Réserver et Annuler la réservation de Content Server

  2. définir un nombre maximal par défaut de versions pour les éléments Content Server

  3. spécifier des paramètres pour la création de versions avancées

  4. activer le processus Ajout avancé des éléments. Cela permet aux utilisateurs de choisir une catégorie lorsqu'ils ajoutent un élément, si l'élément en question est d'un type MIME associé à plus d'une catégorie. Pour que cette activation prenne effet, vous devez sélectionner la case à cocher Afficher les catégories sur une deuxième page Ajouter un élément. Pour en savoir plus, voir Association de types MIME et de catégories. Pour obtenir des informations générales sur les paramètres des types MIME, voir Gestion des icônes et des types MIME.

Les paramètres que vous spécifiez sur la page Gérer le contrôle d'élément deviennent les valeurs par défaut pour les éléments Content Server multiversions ou pouvant être réservés.
