Pour supprimer un fournisseur de stockage de document externe :

  1. Sur la page Administration Content Server, dans la zone Paramètres du fournisseur de stockage, cliquez sur le lien Configurer les fournisseurs de stockage.

  2. Sur la page Configurer les fournisseurs de stockage, dans la colonne Actions du fournisseur de stockage que vous souhaitez supprimer, cliquez sur le lien Supprimer.

  3. Cliquez sur OK dans la boîte de dialogue de confirmation.

  4. Redémarrez les serveurs.
