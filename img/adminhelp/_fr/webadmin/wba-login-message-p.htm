Pour configurer le message de l'écran de connexion :

  1. Sur la page Administration Content Server, sous Configuration du serveur, cliquez sur Configurer la présentation.

  2. Sur la page Configurer la présentation, cliquez sur Configurer le message de l'écran de connexion.

  3. Sur la page Configurer le message de connexion, sous la colonne Langue, si vous avez installé un seul lot de langues, cette langue s'affiche dans cette colonne et vous ne pouvez pas modifier ce champ.

    Si vous avez installé plusieurs lots de langues, une liste s'affiche. Dans cette liste, sélectionnez le lot de langues à utiliser pour afficher votre message de connexion.

  4. La colonne Code de langue ne peut pas être modifiée. Si un seul lot de langues est installé, le code de cette langue s'affiche dans cette colonne.

    Si plusieurs lots de langues sont installés, le code de la langue sélectionnée dans la liste de la colonne Langue s'affiche dans cette colonne.

  5. Dans la colonne Message de l'écran de connexion HTML, saisissez dans la zone de texte le message que vous souhaitez afficher sur l'écran de connexion à tous les utilisateurs se connectant à votre système.

  6. Cliquez sur Enregistrer les modifications.
