La section [AFORM] vous permet d'ajouter un paramètre pour résoudre les problèmes de formulaires PDF dans Content Server. Ajoutez cette section, avec le paramètre ci-dessous, pour que Content Server créé des fichiers de débogage de formulaires.

wantDebug

  • Description :

    Envoie à Content Server une instruction de création de fichiers de débogage de formulaires décrivant la récupération de données à partir des formulaires PDF.

  • Syntaxe :

    wantDebug=MORE

  • Valeurs :

    MORE
