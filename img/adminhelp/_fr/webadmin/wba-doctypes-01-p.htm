Pour modifier la liste des types MIME :

  1. Ouvrez le fichier Content Server_home/config/mime.types dans un éditeur de texte.

  2. Pour ajouter un nouveau type MIME au fichier mime.types, ajoutez le type MIME sur sa propre ligne au format :

    <type_MIME> <extension_fichier1> <extension_fichier2> ...

     

    Note :
    Il n'est pas nécessaire de mapper une extension de fichier au type MIME.

     

  3. Pour associer une nouvelle extension de fichier à un type MIME existant, ajoutez l'extension sur la ligne du type MIME, en la séparant des autres extensions de fichier par un espace.

  4. Pour supprimer une entrée du type MIME, supprimez la ligne du type MIME du fichier mime.types.

     

    Astuce :
    Si vous n'êtes pas sûr de vouloir supprimer le type MIME de manière permanente, ajoutez un commentaire à la ligne du type MIME en insérant un caractère # au début de la ligne au lieu de le supprimer. Ainsi, lorsque vous aurez besoin d'utiliser le type MIME ultérieurement, vous pourrez supprimer le caractère #.

     

  5. Enregistrez et fermez le fichier mime.types.

  6. Redémarrez les serveurs d'administration et le serveur d'application Web.

    Si la liste mise à jour des types MIME ne s'affiche pas immédiatement dans la liste Type MIME de la page Infos spécifiques d'un document, vous devrez peut-être recharger votre navigateur Web.
