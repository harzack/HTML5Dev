Arrêt des serveurs sous UNIX

L'arrêt ou le démarrage du processus Content Server sous UNIX arrête ou démarre également le processus du serveur d'administration.

Pour arrêter les serveurs sous UNIX :

  1. Connectez-vous à l'hôte UNIX avec le nom d'utilisateur utilisé par Content Server.

  2. Lorsque vous y êtes invité, allez dans le répertoire d'installation de Content Server.

  3. Saisissez la commande suivante et appuyez sur ENTREE :

    ./stop_llserver

Arrêt des serveurs Livelink sous UNIX

Pour arrêter les serveurs Livelink sous UNIX :

  1. Connectez-vous à l'hôte UNIX avec le nom d'utilisateur utilisé par Content Server.

  2. Lorsque vous y êtes invité, allez dans le répertoire d'installation de Content Server, puis effectuez l'une des opérations suivantes :

    • Pour démarrer les deux serveurs, saisissez la commande suivante, puis appuyez sur la touche ENTREE :

      ./start_llserver

    • Pour démarrer uniquement le serveur d'administration (par exemple, dans le cas d'un hôte Content Server secondaire), saisissez la commande suivante, puis appuyez sur ENTREE :

      ./start_lladmin
