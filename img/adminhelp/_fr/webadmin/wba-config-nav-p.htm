Pour configurer des options de recueil :

  1. Cliquez sur le lien Configurer la présentation de la section Configuration du serveur de la page Administration.

  2. Cliquez sur le lien Configurer des options de recueil sur la page Configurer la présentation.

  3. Dans la section Navigation, cliquez sur le bouton radio Bandeau horizontal pour sélectionner l'affichage par lien hypertexte ou sur le bouton radio Liste déroulante pour sélectionner l'affichage par liste déroulante.

  4. Pour empêcher les utilisateurs de modifier leur style de navigation, désactivez la case à cocher Autoriser les remplacements utilisateur.

  5. Pour inclure des apparences dans la boîte de dialogue Navigation cible, sélectionnez la case à cocher Parcourir les apparences. Cette case est désactivée par défaut.

  6. Dans la section Pagination, cochez la case Activer la pagination pour activer la pagination. La pagination est activée par défaut.

  7. Dans la liste déroulante Nombre d'éléments affichés par page, sélectionnez le nombre d'éléments qui apparaîtront sur une page.

  8. Dans le champ Personnalisation des colonnes, activez la case Autoriser la personnalisation des colonnes de la liste de navigation pour permettre aux utilisateurs de personnaliser l'affichage des colonnes dans la liste de navigation. Par défaut, la personnalisation des colonnes est désactivée.

  9. Dans le champ Eléments mis en valeur, sélectionnez Supprimer les Eléments mis en valeur de la liste de navigation si, une fois qu'un élément a été sélectionné comme élément mis en valeur, vous voulez le supprimer de l'affichage de la liste. Les éléments mis en valeur apparaissent dans la liste de navigation par défaut.

  10. Dans la section Filtrage, cochez la case Activer le filtrage pour activer le filtrage. Le filtrage est activé par défaut.

  11. Cliquez sur le bouton Enregistrer les modifications pour enregistrer vos changements et revenir à l'écran précédent, ou cliquez sur le bouton Réinitialiser pour réinitialiser la page sur ses valeurs précédentes.
