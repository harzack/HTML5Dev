Pour éliminer les données du journal d'audit :

  1. Dans la section Administration système de la page Administration, cliquez sur le lien Gérer l'audit des événements.

  2. Sur la page Gérer l'audit des événements, cliquez sur le lien Eliminer des événements du journal de l'audit.

  3. Dans la section Eléments cible, procédez de l'une des façons suivantes :

    • Pour éliminer tous les éléments, cliquez sur le bouton radio Tous.

    • Pour éliminer certains éléments, cliquez sur le bouton radio Par type, puis sélectionnez un élément dans la liste déroulante.

       

      Note :
      Si vous sélectionnez Utilisateur ou Groupe, vous éliminez automatiquement les types d'éléments suivants : Utilisateur, Groupe, Groupe interdomaine et Générateur.

       

    • Pour éliminer un élément unique, cliquez sur le bouton radio Elément unique, puis cliquez sur le bouton Parcourir Content Server, naviguez jusqu'à l'élément et cliquez sur le lien Sélectionner.

    • Pour éliminer les éléments appartenant à un utilisateur ou à un groupe spécifique, cliquez sur le bouton radio Utilisateur/Groupe unique, puis recherchez un utilisateur ou un groupe.

  4. Pour éliminer le journal de l'audit d'un type d'événement spécifique, cliquez sur l'événement dans la liste déroulante Type d'événement.

     

    Astuce :
    Pour éliminer le journal de l'audit de tous les types d'événements, cliquez sur <N'importe lequel>.

     

  5. Dans la section Exécutant, effectuez l'une des opérations suivantes :

    • Pour éliminer le journal de l'audit de tous les utilisateurs et groupes, cliquez sur le bouton radio Tous.

    • Pour éliminer le journal de l'audit d'un utilisateur spécifique, cliquez sur le bouton radio Utilisateur spécifique, puis recherchez un utilisateur ou un groupe.

  6. Pour éliminer des éléments ou événements du journal de l'audit couvrant une période précise, sélectionnez un mois, un jour et une année dans les listes déroulantes A partir du et Jusqu'au.

  7. Cliquez sur le bouton Eliminer.

  8. Cliquez sur le lien Accueil Admin pour revenir à la page Administration Livelink.
