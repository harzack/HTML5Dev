Pour créer un utilisateur Oracle :

  1. Sur la page Maintenance du serveur Oracle, cliquez sur le lien Créer un nouvel utilisateur.

  2. Dans la section Créer un nouvel utilisateur, saisissez un nom unique dans le champ Nom de l'utilisateur.

  3. Dans le champ Mot de passe, saisissez un mot de passe pour le nouvel utilisateur, puis saisissez-le une seconde fois dans le champ Confirmation.

  4. Dans la liste Espace de stockage par défaut, cliquez sur le nom de l'espace dans lequel vous voulez créer les tables de la nouvelle base Content Server.

  5. Cliquez sur le bouton Créer un utilisateur.

  6. Procédez de l'une des manières suivantes :

    • Si vous créez une nouvelle base Content Server, cliquez sur le lien Revenir à la page précédente pour revenir à la page Créer des tables Content Server.

    • Sinon, si vous avez terminé les tâches que vous vouliez accomplir sur la page Maintenance du serveur Oracle, cliquez sur le lien Revenir à la page précédente pour revenir à la page Maintenance de la base de données active.
