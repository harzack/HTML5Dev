Vous pouvez afficher le statut de votre licence Content Server sur la page Administration Gérer les licences. Le statut d'une licence Content Server peut être le suivant :

Valide

Un statut de licence valide indique que Content Server dispose d'une licence pour une utilisation par un nombre spécifié d'utilisateurs.

Non valide

Un statut de licence non valide révèle une anomalie dans votre licence Content Server :

Version non valide

Votre licence s'applique à une version de Content Server différente de celle que vous exécutez.

Empreinte non valide

Des modifications apportées à votre environnement Content Server ont entraîné la modification de l'empreinte de votre système, de telle sorte qu'elle ne correspond pas à l'empreinte dans votre fichier de licence Content Server.

Expirée

La date d'expiration spécifiée dans votre licence Content Server temporaire a été dépassée. Lorsque sa licence temporaire expire, Content Server fonctionne en Mode d'administration.

Nombre d'utilisateurs en trop

Le nombre d'utilisateurs dans votre environnement Content Server est supérieur à la valeur Utilisateurs avec licence dans votre licence Content Server.

 

Astuce :
Si le statut de votre licence est Nombre d'utilisateurs en trop, les fonctionnalités de Content Server ne sont pas affectées et aucun utilisateur n'est bloqué.

 
Non valide

Votre licence Content Server est non valide pour un motif autre que ceux énoncés ci-dessus.

Pas de licence

Vous n'avez pas appliqué de licence Content Server à votre installation Content Server. Lorsque Content Server ne dispose pas d'une licence, il fonctionne en Mode d'administration.
