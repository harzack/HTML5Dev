La section [unix_lang], qui s'applique à Unix uniquement, contient des informations système spécifiques sur la commande locale de l'hôte Content Server. Pour plus d'informations sur les fonctions et les attributs de la commande locale, adressez-vous à votre administrateur système.

LL_LC_ALL

  • Description :

    Langage par défaut utilisé pour les entrées et les sorties du système. Ce paramètre correspond à l'argument LANG de la commande locale. Ne modifiez pas la valeur par défaut.

  • Syntaxe :

    LL_LC_ALL=C

  • Valeurs :

    Les valeurs varient en fonction de la plate-forme. La valeur par défaut, C, convient à toutes les plates-formes. Elle correspond au standard ANSI C 7 bits.

NLS_LANG

  • Description :

    Spécifie le paramètre NLS_LANG interne Oracle.

  • Valeurs :

    Les valeurs varient en fonction du système d'exploitation. Par exemple, un ordinateur Solaris anglais (U.S.) utilise par défaut le paramètre american_america.us7ascii. Reportez-vous à votre documentation Oracle pour déterminer la valeur appropriée à votre installation.

NLS_SORT

  • Description :

    Variable d'environnement client Oracle qui détermine la méthode de tri des résultats de recherche renvoyés par la base de données.

  • Syntaxe :

    NLS_SORT=binary

  • Valeurs :

    Toute méthode de tri Oracle valide. La valeur par défaut est binary.
