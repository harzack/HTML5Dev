• Description :

    ’Le service de conversion de documents (DCS) de Content Server convertit les éléments à partir de leur format de fichier natif (par exemple, Microsoft Word ou Excel, ou Adobe PDF) en format texte simple (par exemple, HTML ou texte) pour les afficher ou les indexer dans Content Server.

    Ce paramètre est utilisé pour configurer le SEA Servlet (LLServlet) afin d'autoriser les demandes d'affichage en tant que page Web des utilisateurs externes.

  • Syntaxe

    [ExternalDCS]
    hôte=<adresse IP du Content Server interne>
    port=<Numéro_port_DCS>
    afficher_cache=<SEASERVLET_ACCUEIL>/afficher_cache
    taille_cache=<10>

     

    Note :
    OpenText vous recommande de partager le sous-dossier viewcache, avec des autorisations de partages totales pour vos utilisateurs.

     
     

    Astuce :
    Vous pouvez obtenir le numéro de port en accédant à l'onglet Service de conversion de documents sur la page Propriétés pour le volume des objets système par défaut.

     

  • Valeurs :

    Les valeurs dans la section [ExternalDCS] du fichier opentext.ini sont destinées au serveur d'administration, et peuvent être utilisées pour l'hôte principal ou un autre ordinateur.
