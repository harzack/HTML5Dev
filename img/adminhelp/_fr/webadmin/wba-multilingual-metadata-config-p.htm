Pour configurer les paramètres de métadonnées multilingues :

  1. Dans la section Metadonnées de la page Administration Content Server, cliquez sur le lien Configurer les métadonnées multilingues.

  2. Effectuez l'une des actions suivantes :

    • Pour activer une langue, cochez la case Activé de la langue souhaitée.

    • Pour spécifier la langue comme étant la valeur par défaut pour le serveur Content Server, cliquez sur le bouton Valeur système par défaut correspondant à cette langue.

    • Pour modifier le nom de la langue, cliquez sur l'icône Modifier de la langue, entrez une nouvelle langue dans les champs Langue ou Langue (locale), puis cliquez sur Enregistrer.

    • Pour supprimer une langue qui a été ajoutée, cliquez sur l'icône Enlever correspondant à la langue, puis sur Oui dans la fenêtre de confirmation.

  3. Cliquez sur le bouton OK.
