Pour configurer les règles de détection de type MIME :

  1. Dans la section Administration système de la page Administration Content Server, cliquez sur le lien Configurer la détection du type MIME.

  2. Sur la page Configurer la détection du type MIME, mettez en surbrillance toute méthode de détection de type MIME et utilisez la flèche Droite pour déplacer les types MIME vers le champ Méthodes sélectionnées.

  3. Lorsque les méthodes que vous souhaitez utiliser apparaissent dans le champ Méthodes sélectionnées, utilisez les flèches Haut et Bas pour les placer dans l'ordre où elles doivent être utilisées.

  4. Cliquez sur Enregistrer les modifications.
