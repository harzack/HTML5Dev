La page Statut de l'agent distribué affiche les numéros d'identification de traitement des traitements du cluster, ainsi que leur statut sur Content Server. De plus, les informations suivantes sont affichées pour chaque ID de traitement :

  • La date et l'heure auxquelles le traitement a été mis à jour pour la dernière fois.

  • Le statut du traitement. Parmi les valeurs possibles figurent Inactif, Attente ou En cours d'exécution.

  • La description du statut du traitement.

  • Les actions que vous pouvez réaliser sur ce traitement. Les actions peuvent être Pause ou Reprendre.

Un administrateur réalisera peu d'actions sur la page Statut de l'agent distribué. Cette page est principalement utilisée à des fins informatives. Cependant, des processus peuvent être en cours d'exécution sur votre matériel et conduire à saturation de l'ordinateur. Dans ce cas, vous pouvez actualiser le statut des traitements sur la page Statut de l'agent distribué, et si nécessaire, suspendre un traitement sur un ordinateur saturé.
