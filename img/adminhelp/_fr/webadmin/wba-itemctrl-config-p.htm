Pour configurer des paramètres de contrôle d'élément :

  1. Dans la section Administration système de la page Administration Content Server, cliquez sur le lien Gérer le contrôle d'élément.

  2. Dans la zone Limite de la version de la page Gérer le contrôle d'élément, saisissez une valeur numérique dans le champ Définir la valeur par défaut de la limite de version, afin de définir un nombre maximal de versions par défaut pour les éléments Content Server.

     

    Note :
    La valeur par défaut de ce champ est illimitée. Pour rétablir la limite de version sur une valeur illimitée, saisissez Illimité ou -1 dans le champ Définir la valeur par défaut de la limite de version.

     

  3. Dans la zone Réserver :

    1. Dans le champ Fonction de réserve du groupe, sélectionnez la case à cocher Afficher la liste de groupes pour permettre aux groupes de réserver ou d'annuler la réservation des éléments Content Server.

    2. Dans le champ Annuler la réservation du document, sélectionnez la case à cocher Activer 'Ajouter une nouvelle version' par défaut pour permettre automatiquement aux utilisateurs d'ajouter une nouvelle version à un élément lorsqu'ils annulent la réservation.

  4. Dans la zone Contrôle de la version avancée des documents :

    1. Dans le champ Contrôle de la version avancée, sélectionnez la case à cocher Activer la création avancée de versions principales/secondaires pour des documents du système, afin de permettre aux utilisateurs d'ajouter des versions à un document en tant que versions principales ou secondaires de l'original.

    2. Dans le champ Accès à la version principale uniquement, sélectionnez la case à cocher Pour les utilisateurs qui bénéficient uniquement d'un accès à la version principale, masquer les documents qui ne possèdent que des versions secondaires, afin de masquer les documents comportant uniquement des versions secondaires aux utilisateurs non autorisés à les voir.

  5. Dans la zone Ajout d'élément avancé, dans le champ Attributs obligatoires, sélectionnez la case à cocher Afficher les catégories sur une deuxième page Ajouter un élément pour activer le processus d'ajout d'élément avancé en deux étapes.

  6. Cliquez sur Enregistrer les modifications.
