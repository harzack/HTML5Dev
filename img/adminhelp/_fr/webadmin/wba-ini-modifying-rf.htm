La plupart des paramètres de configuration apparaissant dans les pages Administration, il n'est généralement pas nécessaire de modifier manuellement le fichier opentext.ini. Ne modifiez pas manuellement le fichier opentext.ini, sauf si vous y êtes explicitement invité par :

  • une instruction dans un document Content Server, tel que le présent document en ligne ou le manuel Content Server Installation Guide (Manuel d'installation Content Server)

  • OpenText le service d'assistance clientèle d'Open Text.

Si vous modifiez directement le fichier opentext.ini dans un éditeur de texte, vous devez arrêter, puis redémarrer le serveur et le serveur d'application Web pour appliquer les modifications.

Le fichier opentext.ini se trouve dans le répertoire <Content Server_home>/config, où <Content Server_home> est la racine de votre installation Content Server.

Le fichier opentext.ini est organisé conformément à la structure standard de fichier Windows INI. Il se compose de plusieurs sections, chacune portant un libellé entre crochets droits. Chaque section contient aucun ou plusieurs paramètres, sous la forme <nom>=<valeur> . L'ordre de ces sections est indifférent. Les titres de sections et les chaînes de noms ne font pas l'objet d'une distinction minuscules/majuscules, contrairement aux chaînes de valeurs qui en font toujours l'objet.

 

Note :
Les sections décrites concernent les paramètres de Content Server. Des modules facultatifs peuvent ajouter d'autres sections au fichier opentext.ini qui ne sont pas décrites ici.

 

Pour plus d'informations sur la modification d'autres fichiers de configuration système, reportez-vous à la rubrique Modification des fichiers de configuration système.
