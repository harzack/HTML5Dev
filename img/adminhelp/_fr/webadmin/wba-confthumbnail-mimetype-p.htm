Pour configurer les types MIME afin de générer des miniatures :

  1. Sur la page Administration Content Server, sous Configuration du serveur, cliquez sur le lien Configurer les miniatures.

  2. Sur la page Configurer les options de miniature, pour Types MIME, cliquez sur le lien liste de modification/révision afin d'ouvrir la page Configurer les types MIME de miniature.

  3. Sur la page Configurer les types MIME de miniature, cliquez sur les cases à cocher pour les types MIME à extraire vers l'index de recherche.

  4. Cliquez sur le bouton Mettre à jour.

Types MIME pris en charge

Vous pouvez générer des miniatures lors de la conversion de documents pour certains types de documents, en fonction de l'analyse du document. La page Configurer les options de miniature définit l'Extracteur pour lequel les types MIME Content Server sont susceptibles d'être concernés par la génération de miniature. Définir les miniatures sur cette page produit uniquement des miniatures si le format de fichier est pris en charge et si le type MIME est défini dans la section [Thumbnail] du fichier dcsrules.txt.
