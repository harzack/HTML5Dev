Pour configurer les paramètres de base du serveur :

  1. Dans la section Configuration du serveur de la page Administration Content Server, cliquez sur le lien Configurer les paramètres du serveur.

  2. Sur la page Configurer les paramètres du serveur, modifiez l'un des paramètres du serveur puis cliquez sur le bouton Enregistrer les modifications.

 

Note :
Pour plus d'informations sur les paramètres de cette page, voir Configuration des paramètres de base du serveur.
