Pour afficher la liste de la totalité ou d'un sous-ensemble de groupes :

  1. Connectez-vous à Content Server en tant qu'Admin ou en tant qu'utilisateur disposant du privilège de création/modification de groupes ou des droits d'administration utilisateur.

  2. Choisissez Utilisateurs et groupes dans le menu Entreprise.

  3. Cliquez sur Nom du groupe dans la liste déroulante Recherche.

  4. Dans le champ qui commence par, saisissez votre terme de recherche. Content Server effectue une recherche d'après le critère qui commence par, sans distinction entre majuscules et minuscules.

    Par exemple, la saisie des caractères dur affiche les groupes dont le nom commence par les caractères dur ou Dur.

  5. Cliquez sur le bouton Rechercher.

 

Astuce :
Cliquez sur le bouton Rechercher sans saisir de terme de recherche pour afficher toutes les entrées du champ spécifié. Notez toutefois qu'une telle recherche peut prendre beaucoup de temps et solliciter fortement les ressources système.
