La section [Lang_<xx_XX>, où <xx_XX> indique les paramètres régionaux de votre version Content Server, contrôle la manière dont Content Server traite les dates, les heures et les noms d'utilisateur qui dépendent des paramètres régionaux.

ShortTimeFormat

  • Description :

    Définit l'affichage de Content Server au format 12 heures, par exemple, 02:41, ou au format 24 heures, par exemple, 14:41.

  • Syntaxe :

    ShortTimeFormat=%I:%M %p

  • Valeurs :

    Le paramètre par défaut est le format 12 heures : %I:%M %p.

ShortDateFormat

  • Description :

    Définit le format d'affichage de Content Server pour le jour, le mois et l'année.

  • Syntaxe :

    ShortDateFormat=%m/%d/%Y

  • Valeurs :

    Le paramètre par défaut est le format mois, jour et année à deux chiffres : %m/%d/%Y.

    D'autres caractéristiques sont contrôlées par le paramètre ShortDateSeqFormat.

ShortDateSeqFormat

  • Description :

    Définit l'ordre du format d'affichage de Content Server pour le jour, le mois et l'année, ainsi que les caractères séparant les éléments de la date.

  • Syntaxe :

    ShortDateSeqFormat={1,1,2,1,'/','/'}

  • Valeurs :

    Le paramètre par défaut et le format mois, jour et année, séparé par une barre oblique : {1,1,2,1,'/','/'}.

    D'autres caractéristiques sont contrôlées par le paramètre TwoDigitYears.

InputTimeFormat

  • Description :

    Définit si Content Server accepte les entrées d'heure au format 12 heures, par exemple, 02:41, ou au format 24 heures, par exemple, 14:41.

  • Syntaxe :

    InputTimeFormat=%I:%M %p

  • Valeurs :

    Le paramètre par défaut est le format 12 heures : %I:%M %p.

InputDateFormat

  • Valeurs :

    D'autres caractéristiques sont contrôlées par le paramètre InputDateSeqFormat.

UserNameDisplayFormat

  • Description :

    Format d'affichage des noms d'utilisateur Content Server dans des champs tels que Créé par, Propriétaire et Utilisateur. Modifiez les formats des noms d'utilisateur uniquement sur la page Configurer l'affichage du nom d'utilisateur des pages de l'interface d'administration Content Server.

  • Syntaxe :

    UserNameDisplayFormat=1

  • Valeurs :

    Les valeurs possibles sont :

    • 1, qui affiche l'ID de connexion. Il s'agit de la valeur par défaut.

    • 2, qui affiche Prénom Nom

    • 3, qui affiche Prénom Initiale du second prénom Nom

    • 4, qui affiche Nom, Prénom

    • 5, qui affiche Nom, Prénom Initiale du second prénom

    • 6, qui affiche Nom Prénom
