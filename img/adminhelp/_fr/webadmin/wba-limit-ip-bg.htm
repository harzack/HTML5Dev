Par défaut, vous pouvez vous connecter à Content Server en tant qu'utilisateur administrateur à partir de toute adresse IP ayant accès au réseau informatique de votre organisation. Cependant, pour des raisons de sécurité, vous pouvez restreindre les possibilités de connexion à Content Server en tant qu'utilisateur administrateur. Vous pouvez configurer Content Server pour autoriser l'utilisateur administrateur à se connecter uniquement à partir d'une adresse IP approuvée.

Les valeurs que vous ajoutez au champ Liste d'adresses IP autorisées peuvent comporter une adresse IP explicite ou une adresse IP contenant un astérisque (*), qui fonctionne comme un caractère générique pour remplacer des portions de l'adresse. L'utilisation d'astérisques vous permet d'inclure un groupe d'ordinateurs ayant en commun des parties de leur adresse IP.

 

Important :
Si vous avez installé OpenText Directory Services, les paramètres que vous définissez sur cette page empêchent également l'utilisateur otadmin@otds.admin de se connecter à partir d'une adresse IP non autorisée.
