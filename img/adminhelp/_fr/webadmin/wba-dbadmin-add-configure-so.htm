La page Administration de la base de données s'affiche lorsque vous :

  • installez une nouvelle instance de Content Server (après avoir cliqué sur le bouton Enregistrer les modifications de la page Configurer les paramètres du serveur) ;

  • modifiez la base de données à laquelle est connectée une instance de Content Server (après avoir cliqué sur le bouton Continuer de la page Désélectionner base de données Content Server).

Dans les deux cas, vous pouvez sélectionner une base existante ou en créer une.

Une base de données SQL Server étant créée dans le cadre de l'installation Content Server, il n'est en principe pas nécessaire de créer des bases de données supplémentaires. Cependant, vous pouvez en créer au besoin.
