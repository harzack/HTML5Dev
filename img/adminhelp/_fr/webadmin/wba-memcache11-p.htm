Pour afficher les statistiques memcached :

  1. Sur la page Administration, dans la section Administration système cliquez sur Statistiques Memcached.

  2. Sur la page Statistiques Memcached vous verrez s'afficher :

    • Statut : affiche le statut pour chaque processus mis en mémoire cache. Parmi les valeurs possibles figurent Inactif et En cours d'exécution.

    • Nom : affiche le nom du processus mis en mémoire cache sous forme de lien. Cliquez sur ce lien pour afficher la page de détails pour ce processus.
