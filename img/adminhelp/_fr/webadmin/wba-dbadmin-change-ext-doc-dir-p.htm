Pour changer ou déplacer le répertoire de stockage de documents externe :

  1. Notez le chemin de répertoire affiché dans le champ Répertoire externe.

  2. Dans le champ Répertoire externe, saisissez le chemin d'accès au nouveau répertoire de stockage tel qu'il est mappé ou monté sur l'hôte principal Content Server.

  3. Cliquez sur le bouton Définir répertoire. Vous êtes invité à redémarrer le serveur Content Server et le serveur d'application Web.

  4. Sur la page Redémarrer le serveur, arrêtez le serveur.

  5. Déplacez les fichiers externes de la base de données de l'ancien répertoire vers le nouveau (défini à l'étape 2).

  6. Redémarrez le serveur et le serveur d'application Web.

  7. Sur la page Redémarrer le serveur, cliquez sur le lien Continuer.
