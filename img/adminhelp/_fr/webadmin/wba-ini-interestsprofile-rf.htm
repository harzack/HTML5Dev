La section [InterestsProfile] définit le paramètre de configuration pour Paramètres du rapport dans Administration des notifications. Pour définir ce paramètre, OpenText vous recommande d'utiliser la page Configurer la notification, plutôt que de modifier directement le fichier opentext.ini.

EventNo

  • Description :

    Cette valeur contraint Content Server à traiter uniquement ce nombre d'événements par cycle.

  • Syntaxe :

    EventNo=1000

  • Valeurs :

    Un entier supérieur ou égal à zéro.

    La valeur par défaut est EventNo=1000.
