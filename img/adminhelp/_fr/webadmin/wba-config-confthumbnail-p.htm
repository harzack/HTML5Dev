Pour configurer des options de miniature :

  1. Sur la page Administration Content Server, sous Configuration du serveur, cliquez sur le lien Configurer les miniatures.

  2. Sur la page Configurer les options de miniature, pour Générer les miniatures, cliquez sur le bouton radio Activer, si besoin. Ce dernier est activé par défaut.

  3. Pour Versions de miniature, cliquez sur le bouton radio Version actuelle uniquement, activé par défaut, pour générer les miniatures uniquement pour la version actuelle de l'objet, ou sur le bouton radio Toutes les versions.

  4. Pour Types MIME, cliquez sur le lien liste de modification/révision pour ouvrir la page Configurer les types MIME de miniature. Pour en savoir plus, consultez le document Pour configurer les types MIME afin de générer des miniatures.

     

    Note :
    Les miniatures sont générées lors de l'indexation. Les types MIME de miniature fonctionnent comme une liste blanche d'indexation et sont prioritaires par rapport à la liste d'exclusions de type MIME de l'Extracteur dans la section [ExcludedMimeTypes] du fichier opentext.ini.

    Pour que la génération de miniature fonctionne, le type MIME doit être sélectionné sur la page Configurer les types MIME de miniature et le type MIME doit être défini dans la section [Thumbnail] du fichier dcsrules.txt.

     

  5. Pour Afficher les miniatures, cliquez sur le bouton radio Activer pour afficher les miniatures à la place des grandes icônes. Ce dernier est activé par défaut.

  6. Cliquez sur le bouton Enregistrer les modifications.
