Pour modifier des correspondances icône-type MIME :

  1. Ouvrez le fichier Content Server_home/config/mime.gif dans un éditeur de texte.

  2. Pour associer un nouveau type MIME à un fichier d'icônes existant, saisissez le nouveau type MIME après les types MIME existants, en les séparant par des espaces.

  3. Pour associer un nouveau fichier d'icônes avec des types MIME nouveaux ou existants, ajoutez une entrée dans le fichier mime.gif au format suivant :

    <nom_gif> <nom_fichier> <type_MIME1> <type_MIME2> ...

  4. Vous pouvez utiliser le caractère # pour ajouter un commentaire à une entrée d'icône. Tout ce qui suit le caractère # est considéré comme commentaire et ignoré par Content Server.

  5. Assurez-vous que les types MIME que vous référencez dans le fichier mime.gif sont également répertoriés dans le fichier Content Server_home/config/mime.types. Pour en savoir plus, voir Modification de la liste des types MIME.

  6. N'associez pas un type MIME à plusieurs fichiers d'icônes.

  7. Enregistrez et fermez le fichier mime.gif.

  8. Redémarrez les serveurs d'administration et le serveur d'application Web.

 

Astuce :
Si une nouvelle icône n'apparaît pas immédiatement dans Content Server, vous devrez peut-être recharger votre navigateur Web.
