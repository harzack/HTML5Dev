Pour modifier le mot de passe utilisé pour se connecter à la base de données :

  1. Dans la section Administration de la base de données de la page Administration Content Server, cliquez sur le lien Modifier la base de données active.

  2. Sur la page Désélectionner la base de données Content Server, désactivez la case à cocher pour chaque source de données, puis cliquez sur Continuer .

     

    Note :
    Ne fermez pas la fenêtre de navigation.

     

  3. Modifiez le mot de passe de la base de données de l'utilisateur dans la base de données que vous utilisez.

  4. Retournez à la fenêtre Content Server, puis cliquez sur le lien Sélectionner la base de données existante.

  5. Sur la page Sélectionner le type de SGBDR, cliquez sur le bouton radio correspondant à votre type de base de données, puis cliquez sur Continuer.

  6. Sur la page Spécifier le propriétaire de la base de données Content Server, saisissez les informations requises, puis cliquez sur Continuer.

  7. Sur la page Connexion de l'administrateur Content Server, saisissez vos informations d'authentification, puis cliquez sur Connexion.

  8. Sur la page Installer le module, cliquez sur Continuer.

  9. Sur la page Confirmation de la configuration du serveur d'administration Content Server, cliquez sur Continuer.
