Sur la page Définir les centres d'intérêt à inclure dans l'audit, vous pouvez spécifier les événements audités. Tableau 21-1 répertorie les événements que vous pouvez auditer. Le tableau ci-dessous répertorie les événements pouvant faire l'objet d'un audit disponibles lorsque Content Server est installé. Si des modules facultatifs sont installés, des centres d'intérêt à inclure dans l'audit supplémentaires peuvent être disponibles.

 

Notes :

  • Lorsqu'un événement d'audit n'est pas enregistré dans un sous-système de registre, il est journalisé comme Obsolète. Un événement d'audit obsolète a un ID égal à zéro (0).

  • L'option Forcer l'audit de tous les événements réalisés par des administrateurs système permet d'activer l'audit de toute action pouvant faire l'objet d'un audit réalisée par des utilisateurs bénéficiant du privilège Administration système, quels que soient les autres paramètres des centres d'intérêt de l'audit.
