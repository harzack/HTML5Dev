Dans le cadre de l'administration de Content Server, deux paramètres relatifs aux utilisateurs sont les paramètres de mot de passe des comptes utilisateur, ainsi que l'affichage du nom d'utilisateur. Vous pouvez définir des restrictions sur les mots de passe, telles que la longueur minimale du mot de passe, un intervalle d'expiration et si les mots de passe doivent contenir ou non des chiffres. Vous pouvez également définir la façon dont Content Server affiche les noms d'utilisateur sur votre système. Les noms d'affichage peuvent être configurés individuellement pour toutes les langues activées sur le système.

Pour plus d'informations sur les paramètres par défaut du mot de passe, voir [Passwords].

Vous pouvez prévisualiser chacune des options dans le champ Exemple avant de prendre une décision finale.
