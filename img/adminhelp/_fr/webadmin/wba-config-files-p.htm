Pour modifier un fichier de configuration système :

  1. Connectez-vous à l'hôte Content Server principal en tant qu'utilisateur système défini pour l'exécution des serveurs.

  2. Ouvrez le fichier approprié dans un éditeur de texte.

  3. Modifiez le fichier selon les besoins pour implémenter le changement.

  4. Enregistrez et fermez le fichier.

  5. Redémarrez les services Content Server et le serveur d'application Web.
