Vous pouvez définir les autorisations pour que les utilisateurs puissent modifier les onglets des informations relatives aux utilisateurs. Les utilisateurs peuvent bénéficier des privilèges Modifier les autorisations personnelles ou Autorisation de modifier n'importe qui dans l'onglet Général ou dans l'onglet Personnel. Par défaut, les utilisateurs disposant du privilège Droits d'administration système bénéficient aussi des privilèges Autorisation de modifier n'importe qui. Vous pouvez attribuer aux autres utilisateurs le privilège Autorisation de modifier n'importe qui pour l'onglet Personnel, mais pas pour l'onglet Général. Vous pouvez faire bénéficier à tous les utilisateurs du privilège Modification des autorisations personnelles pour les deux onglets ou pour l'un ou l'autre.
