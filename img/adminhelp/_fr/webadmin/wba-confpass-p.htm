Pour configurer les paramètres du mot de passe :

  1. Cliquez sur le lien Configurer les paramètres du mot de passe de la section Administration d'utilisateurs et de groupes sur la page Administration.

  2. Dans le champ Nombre minimum de caractères, entrez le nombre minimum de caractères requis pour les mots de passe.

  3. Cochez la case Activer pour l'une des options suivantes :

    • Les mots de passe doivent contenir un chiffre

    • Le mot de passe ne peut pas commencer par un chiffre

    • Le mot de passe ne peut pas se terminer par un chiffre

    • Le mot de passe modifié doit être différent

    • Modifier le mot de passe à la première connexion

  4. Pour définir tous les mots de passe afin qu'ils expirent au bout d'une période donnée, cliquez sur le bouton radio Connecté dans la section Expiration du mot de passe, puis saisissez un entier dans le champ Jours avant expiration.

    Si vous le souhaitez, activez Les utilisateurs disposant du droit 'création/modification d'utilisateurs' peuvent définir l'expiration du mot de passe pour chaque compte utilisateur pour :

    • autoriser les utilisateurs bénéficiant du privilège Peut créer/modifier des groupes à définir l'expiration d'un mot de passe sur des comptes utilisateur qu'ils contrôlent, et

    • autoriser les utilisateurs bénéficiant du privilège Droits d'administration utilisateur à définir Mot de passe n'expire jamais sur des comptes utilisateur.

     

    Astuce :
    Les privilèges Peut créer/modifier des groupes et Droits d'administration utilisateur peuvent être définis sur la page Ajouter un nouvel utilisateur. Pour en savoir plus, voir Utilisation des utilisateurs.

     

  5. Pour empêcher les utilisateurs de réutiliser leur mot de passe pendant un certain nombre de jours, cliquez sur une valeur de la liste déroulante Nombre de jours pour empêcher la réutilisation de mot de passe.

  6. Pour empêcher les utilisateurs de choisir un nouveau mot de passe avant un certain nombre de jours, cliquez sur une valeur de la liste déroulante Nombre de jours requis entre les modifications de mot de passe.

  7. Cliquez sur le bouton Enregistrer les modifications.

 

Notes :

  • La modification des paramètres d'expiration du mot de passe requiert la mise à jour par Content Server de la base de données utilisateur. Si vous effectuez cette modification, une boîte de dialogue s'ouvre lorsque vous tentez d'enregistrer vos modifications. Vous pouvez ainsi voir la progression de la mise à jour.

  • Le nombre maximum de caractères qu'un mot de passe peut contenir est défini sur 16 par défaut. Pour modifier ce paramètre, vous devez modifier le fichier opentext.ini. Pour en savoir plus, voir [Passwords].
