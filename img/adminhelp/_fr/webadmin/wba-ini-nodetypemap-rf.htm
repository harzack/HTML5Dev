Chaque type de nœud défini dans Content Server est identifié par son sous-type. Un sous-type est un nombre entier unique d'identification, enregistré dans les nœuds LLNode et WebNode du type correspondant.

Pour voir le nombre total de types de nœuds afin de nommer les mappages, exécutez un rapport système et affichez la section « Types de noeuds ».

Le tableau suivant montre quelques-uns des types de nœuds courants pour une installation standard de Content Server. Si vous installez des modules facultatifs, il est possible que votre système comporte des types de nœud ne figurant pas dans cette liste. De même, les personnalisations apportées à Content Server peuvent créer des types de nœuds personnalisés ne figurant pas dans cette liste.
