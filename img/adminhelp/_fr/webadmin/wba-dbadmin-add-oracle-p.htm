Pour ajouter un serveur Oracle au SGBDR :

  1. Sur la page Administration de la base de données, cliquez sur le lien Créer une nouvelle base de données.

  2. Sur la page Sélectionner le type de SGBDR, cliquez sur le bouton radio Serveur Oracle, puis sur le bouton Continuer.

  3. Sur la page Connexion de l'administrateur du serveur Oracle, saisissez vos informations de connexion dans les champs appropriés, puis cliquez sur le bouton Connexion.
