Pour créer un nouvel espace de stockage :

  1. Sur la page Maintenance du serveur Oracle, cliquez sur le lien Créer un nouvel espace de stockage.

  2. Attribuez un nom unique à l'espace de stockage (par exemple _TS) dans le champ Nom de l'espace de stockage.

    Vous pouvez voir les noms déjà utilisés dans la liste déroulante Espace de stockage par défaut, dans la section Créer un utilisateur de la page Maintenance du serveur Oracle.

  3. Dans le champ Chemin d'accès, saisissez le chemin d'accès absolu au fichier de données de l'espace de stockage à créer (par exemple, C:\orant\database\filename.ora ou /usr/oracle/database/filename.dbf). Le répertoire spécifié doit déjà exister et l'utilisateur Windows ou UNIX créé pour le serveur Oracle doit posséder une autorisation en écriture sur celui-ci.

  4. Dans le champ Taille, indiquez, en Mo, la taille du fichier de données de l'espace de stockage (5 Mo minimum), en suivant les instructions de la section Créer un nouvel espace de stockage de la page Maintenance du serveur Oracle.

  5. Cliquez sur le bouton Créer un espace de stockage.

  6. Procédez de l'une des manières suivantes :

    • Si vous créez une base de données Content Server et que vous n'avez pas encore créé l'utilisateur Oracle, reportez-vous à la rubrique Création d'un utilisateur Oracle.

    • Si vous avez terminé les tâches que vous vouliez accomplir sur la page Maintenance du serveur Oracle, cliquez sur le lien Revenir à la page précédente.
