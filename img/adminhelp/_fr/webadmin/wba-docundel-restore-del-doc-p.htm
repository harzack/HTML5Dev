Pour restaurer les documents du volume des objets supprimés :

  1. Recherchez le document supprimé à l'aide d'un rapport de LiveReports en sélectionnant Rapports dans le menu général Personnel.

  2. Sur la page Mes rapports, cliquez sur Rapports LiveReport. Sélectionnez ensuite l'un des éléments suivants :

    • Un rapport personnalisé que vous avez créé pour accéder aux documents du volume des objets supprimés

    • Documents supprimés

    • Documents supprimés par [Utilisateur]. Si vous sélectionnez cette option, vous êtes invité à saisir l'ID utilisateur de l'utilisateur.

  3. Dans la liste de documents de la page LiveReport : Documents supprimés, cliquez sur l'icône Fonctions du document supprimé que vous souhaitez restaurer, puis cliquez sur Restaurer.

  4. Si le chemin d'accès d'origine du document supprimé existe toujours, le champ Déplacer vers de la page Restaurer affiche ce chemin par défaut. Pour modifier ce chemin d'accès, cliquez sur le bouton Parcourir Content Server.

    Dans la boîte de dialogue Déplacer, recherchez l'emplacement où vous souhaitez restaurer le document, puis cliquez sur son lien Sélectionner.

  5. Cliquez sur le bouton Restaurer.

  6. Pour modifier les autorisations, cliquez sur l'icône Fonctions du document restauré, puis sélectionnez Autorisations.
