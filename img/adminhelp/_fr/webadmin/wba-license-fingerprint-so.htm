Une empreinte système identifie votre déploiement Content Server de manière unique, à l'aide des éléments d'informations provenant de votre base de données Content Server. (Les informations sont codées et hachées afin qu'elles ne soient pas lisibles, même par OpenText.) L'empreinte système est identique pour toutes les instances de Content Server dans votre déploiement, à condition que chaque instance se connecte à la même base de données Content Server.
