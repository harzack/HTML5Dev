Content Server fonctionne en mode d'administration quand :

  • Sa licence temporaire a expiré.

  • Il ne dispose pas d'une licence.

Lorsque Content Server est en mode d'administration, seuls les utilisateurs avec le privilège System administration rights peuvent se connecter. Les utilisateurs ordinaires ne peuvent pas se connecter à Content Server. Si un utilisateur ne bénéficiant pas des droits d'administration système essaye de se connecter à Content Server lorsqu'il est en mode d'administration, le message suivant apparaît :

Pour que Content Server quitte le mode d'administration, appliquez une licence valide.
