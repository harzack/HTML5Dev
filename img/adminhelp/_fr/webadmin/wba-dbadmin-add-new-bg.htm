La page Sélectionner le type de SGBDR s'affiche lorsque vous cliquez sur le lien Créer une nouvelle base de données de la page Administration de la base de données. Vous pouvez spécifier le type de SGBDR Serveur Oracle ou Microsoft SQL Server (Windows uniquement).

La page Sélectionner le type de SGBDR apparaît lorsque vous cliquez sur le lien Sélectionner la base de données existante sur la page Administration de la base de données. Vous pouvez sélectionner la base de données pour vous connecter à une nouvelle installation Content Server ou à une installation existante.

Création d'une base de données Microsoft SQL Server

Lorsque vous créez une base de données Content Server pour vous connecter à une installation Content Server nouvelle ou existante, vous accédez à la page Maintenance Microsoft SQL à partir de la page Créer des tables Content Server. Pour plus d'informations sur la création de tables Content Server, reportez-vous à la rubrique Création des tables Content Server.

Lorsque vous créez une nouvelle base de données, vous devez effectuer les tâches suivantes dans l'ordre indiqué, avant de revenir à la page précédente :

  • Créez une base de données SQL Server, qui contiendra les tables de la base Content Server ;

  • Créez un utilisateur Microsoft SQL Server, qui sera en mesure de se connecter à la base de données SQL Server.

Les tables d'une base de données Content Server résident dans une base de données du serveur SQL. Content Server enregistre ses données dans un fichier de données et son journal dans un fichier journal.
