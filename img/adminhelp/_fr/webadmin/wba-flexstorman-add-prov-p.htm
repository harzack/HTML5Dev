Pour ajouter un fournisseur de stockage de documents externe :

  1. Sur la page Administration Content Server, dans la zone Paramètres du fournisseur de stockage, cliquez sur le lien Configurer les fournisseurs de stockage.

  2. Sur la page Configurer les fournisseurs de stockage, cliquez sur le type de fournisseur de stockage que vous souhaitez ajouter dans la liste Ajouter un élément.

     

    Note :
    Si aucun module facultatif n'est installé, le seul fournisseur de stockage disponible est Stockage de documents externe.

     

  3. Sur la page Ajouter : Nouveau fournisseur de stockage logique, saisissez un nom pour le fournisseur de stockage dans le champ Nom.

  4. Dans le champ Configuration, saisissez le chemin absolu vers le fournisseur de stockage.

     

    Note :
    Le chemin absolu s'applique aux répertoires de stockage externe. Si le fournisseur de stockage n'est pas un répertoire externe, vous devrez peut-être spécifier un chemin de configuration différent dans le champ Configuration. Après avoir ajouté un fournisseur de stockage, il est nécessaire de lui configurer des règles.

     

  5. Cliquez sur Envoyer.
