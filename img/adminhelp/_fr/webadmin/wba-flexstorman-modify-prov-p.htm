Pour modifier un fournisseur de stockage de document externe :

  1. Sur la page Administration Content Server, dans la zone Paramètres du fournisseur de stockage, cliquez sur le lien Configurer les fournisseurs de stockage.

  2. Sur la page Configurer les fournisseurs de stockage, cliquez sur l'icône Modifier, en regard du fournisseur de stockage que vous souhaitez modifier.

  3. Dans le champ Configuration, saisissez un chemin absolu vers le nouveau fournisseur de stockage.

     

    Note :
    Le chemin absolu s'applique aux répertoires de stockage externe. Si le fournisseur de stockage n'est pas un répertoire externe, vous devrez peut-être spécifier un chemin de configuration différent dans le champ Configuration.

    Si vous modifiez le chemin de configuration du fournisseur de stockage, vous devez déplacer manuellement le contenu existant de l'ancien emplacement vers le nouveau.

     

  4. Cliquez sur Envoyer.
