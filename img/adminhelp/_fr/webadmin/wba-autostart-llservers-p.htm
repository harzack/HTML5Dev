Pour modifier les paramètres de démarrage d'un serveur sous Windows :

  1. Cliquez avec le bouton droit de la souris sur l'icône Poste de travail du bureau, puis sélectionnez Gérer.

  2. Double-cliquez sur l'icône Services.

  3. Double-cliquez sur l'icône Services et applications.

  4. Dans la fenêtre Gestion de l'ordinateur, cliquez sur le nom du serveur dont vous voulez modifier le paramètre de démarrage, puis sélectionnez Propriétés.

  5. Cliquez sur la liste déroulante Type de démarrage et effectuez l'une des opérations suivantes :

    • Pour configurer le serveur afin qu'il démarre manuellement, sélectionnez Manuel, puis cliquez sur OK.

    • Pour configurer le serveur afin qu'il démarre automatiquement, sélectionnez Automatique, puis cliquez sur OK.

Pour modifier les paramètres de démarrage d'un serveur sous UNIX :

  1. Connectez-vous à UNIX en tant que root.

  2. Ajoutez le chemin complet du répertoire start_llserver (qui est le répertoire ContentServer_home) au script de démarrage de l'ordinateur UNIX exécutant les serveurs.

 

Note :
Si vous ne disposez pas des privilèges racine ou si vous ne savez pas exactement comment modifier le script d'initialisation, consultez votre administrateur système UNIX.

Sur les systèmes d'exploitation UNIX, vous disposez de plusieurs autres possibilités pour configurer le démarrage automatique des serveurs.
