Pour mettre à niveau un module Content Server sous UNIX :

  1. Copiez le fichier <nom_module>.tar dans le répertoire <_home> de l'instance Content Server sur laquelle le module actuel est installé.

  2. A l'invite du système d'exploitation, tapez la commande suivante puis appuyez sur la touche ENTRÉE :tar -xvf <nom_module>.tar.

  3. Dans la section Administration du module de la page Administration, cliquez sur le lien Mettre des modules à niveau.

  4. Sur la page Mettre des modules à niveau, sélectionnez la case à cocher située en regard du module à mettre à niveau, puis cliquez sur le bouton Mettre à niveau.

  5. Redémarrez le serveur Livelink et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer le serveur Content Server.
