Le lien Mettre cette base de données à niveau conduit à la page Confirmation de la mise à niveau de la base de données Content Server. La mise à niveau d'une base n'est obligatoire que dans le cas où vous connectez l'installation Content Server courante à une base créée à l'aide d'une version antérieure de Content Server. Dans ce cas, la page Confirmation de la mise à niveau de la base de données Content Server s'affiche automatiquement dans le cadre de la procédure de sélection de la base.

Pour une base qui est à jour, la page Confirmation de la mise à niveau de la base de données Content Server affiche la version du schéma actuel de la base Content Server globale et de certains modules d'administration qui utilisent des tables de la base Content Server.

Vous pouvez visualiser la version du schéma actuel de la base Content Server et de certains modules d'administration en cliquant sur le lien Vérifier cette base de données de la page Maintenance de la base de données active.

La page Confirmation de la mise à niveau de la base de données Content Server apparaît lorsque les deux conditions suivantes sont réunies :

  • Vous connectez l'installation Content Server à une base de données.

  • La base à laquelle vous vous connectez doit être mise à niveau pour fonctionner correctement avec la version actuelle de Content Server.

Si ces conditions sont réunies, la page Confirmation de la mise à niveau de la base de données Content Server apparaît après que vous avez spécifié le propriétaire de la base de données sur la page Spécifier le propriétaire de la base de données Content Server et que vous vous êtes connecté à la base de données sur la page Connexion de l'administrateur Content Server.

La page Statut de la mise à niveau de la base de données apparaît lorsque vous connectez l'installation Content Server courante à une base de données Content Server qui doit être mise à jour pour fonctionner correctement avec l'installation Content Server.

Lorsque la page Statut de la mise à niveau de la base de données apparaît pour la première fois, elle affiche la progression de la mise à jour puis elle s'actualise automatiquement à un intervalle de quelques secondes. Une fois la mise à jour terminée, la page affiche le lien Continuer.

Selon que vous utilisez une installation Content Server nouvelle ou existante et que la base de données a été créée initialement sur l'hôte courant, une des pages suivantes apparaît :

  • Installation de modules

  • Confirmation du serveur d'administration par défaut

  • Confirmation de la resynchronisation de la source de données
