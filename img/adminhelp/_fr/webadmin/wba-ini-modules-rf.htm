• Description :

    La section [Modules] comporte des informations relatives aux modules. Les valeurs de cette section sont définies par Content Server et par les modules eux-mêmes.

    Pour plus d'informations sur les fichiers de configuration des modules et certains paramètres de la section [Modules], voir Création d'un module Content Server dans le manuel Content Server Module Developer's Guide (Manuel du développeur du module Content Server).

     

    Important :
    OpenText vous recommande de ne modifier aucun des paramètres de cette section.

     

  • Valeurs :

    Il existe trois lignes pour chaque module. Voici par exemple la définition du module Discussions :

    • ospaces_discussion={'discussion'}. . .

    • module_11=discussion. . .

    • discussion=_3_0_0

    La première ligne comporte la liste des OSpaces composant le module. Il s'agit d'une liste d'un ou plusieurs noms d'OSpace séparés par des virgules. Chaque nom doit être délimité par des apostrophes et la liste complète doit être comprise entre accolades : {'<valeur>','<valeur>'}.

    La deuxième ligne est utilisée pour permettre l'itération parmi la liste des modules installés.

    La troisième ligne définit le numéro de version du module.
