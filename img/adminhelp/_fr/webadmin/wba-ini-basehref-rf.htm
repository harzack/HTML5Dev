La section [BaseHref] définit les options pour la définition de la valeur HREF DE BASE dans une page HTML qui contient une vue personnalisée.

Protocole

  • Description :

    Soit http, soit https.

  • Syntaxe :

    Protocol=http

Hôte

  • Description :

    Le nom d'hôte du serveur à inclure dans l'URL HREF DE BASE.

  • Syntaxe :

    Host=myhostname

Port

  • Description :

    Le port du serveur Web utilisé dans l'URL HREF DE BASE.

  • Syntaxe :

    Port=3000

  • Valeurs :

    Un entier positif.
