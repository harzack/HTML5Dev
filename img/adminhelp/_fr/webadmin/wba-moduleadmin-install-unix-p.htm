Pour installer un module sous UNIX :

  1. Copiez le fichier <nom_module>.tar dans le répertoire <Content Server_home> de l'instance Content Server sur laquelle vous voulez installer le module.

  2. A l'invite de commandes, saisissez la commande suivante : tar -xvf <nom_module>.tar, puis appuyez sur la touche ENTREE.

  3. Dans la section Administration du module de la page Administration, cliquez sur le lien Installer des modules.

  4. Dans la section Modules que vous pouvez installer, cochez la case en regard du module à installer, puis cliquez sur le bouton Installer.

  5. Redémarrez le serveur et le serveur d'application Web, puis cliquez sur le lien Continuer de la page Redémarrer Content Server.
