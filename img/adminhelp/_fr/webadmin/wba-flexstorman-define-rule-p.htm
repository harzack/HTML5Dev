Pour définir une règle de fournisseur de stockage :

  1. Dans la zone Paramètres du fournisseur de stockage de la page Administration Content Server, cliquez sur le lien Configurer les règles de stockage.

  2. Dans la colonne Modifier de la zone Règles de stockage, sur la page Configurer les règles de stockage, cliquez sur l'icône Ajouter une nouvelle règle avant celle-ci, .

    La nouvelle règle apparaît au-dessus de la règle active.

  3. Dans la liste Règle de la page Ajouter une nouvelle règle, sélectionnez une règle.

  4. Dans le champ Valeur, saisissez une valeur pour la règle.

  5. Facultatif Dans le champ Description, saisissez une description de votre nouvelle règle.

     

    Note :
    Bien que le champ Description soit facultatif, OpenText vous recommande de toujours saisir une description lors de la création d'une nouvelle règle de stockage.

     

  6. Les prochains champs auxquels vous pouvez accéder sont déterminés par la règle que vous sélectionnez dans étape 3. Si votre nouvelle règle contenait une variable, indiquée par ?, vous devez saisir une valeur pour cette variable. Ces champs sont obligatoires.

    1. Dans le champ Valeur, saisissez la valeur souhaitée pour définir cette règle.

    2. Dans le champ Définition de l'attribut, saisissez une valeur pour la définition de l'attribut que vous souhaitez définir pour cette règle.

    3. Dans le champ Valeur d'attribut, saisissez une valeur pour l'attribut que vous souhaitez définir pour cette règle.

  7. Dans la liste Fournisseur de stockage logique, cliquez sur le fournisseur de stockage que vous souhaitez associer à la règle.

  8. Cliquez sur Envoyer.
