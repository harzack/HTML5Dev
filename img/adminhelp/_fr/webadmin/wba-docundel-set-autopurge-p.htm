Pour définir l'élimination automatique des documents dans le volume des objets supprimés :

 

Attention
Une fois que vous avez éliminé les documents supprimés, ils sont définitivement supprimés de Content Server.

 

  1. Cliquez sur le lien Configurer l'élimination automatique de la section Administration de la récupération de documents sur la page Administration.

  2. Sur la page Configurer l'élimination automatique, saisissez un entier dans le champ Jours d'existence. La valeur par défaut est 7.

  3. Cliquez sur le bouton Envoyer.
