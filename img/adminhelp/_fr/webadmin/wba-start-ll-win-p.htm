Arrêt des serveurs sous Windows

Pour arrêter les serveurs sous Windows :

  1. Cliquez avec le bouton droit de la souris sur l'icône Poste de travail du bureau, puis sélectionnez Gérer.

  2. Double-cliquez sur l'icône Services et applications.

  3. Double-cliquez sur l'icône Services.

  4. Pour arrêter le service Content Server (OTCS), cliquez avec le bouton droit de la souris sur son nom, puis cliquez sur Arrêter.

  5. Pour arrêter le service d'administration Content Server (OTCS), cliquez avec le bouton droit de la souris sur son nom, puis cliquez sur Arrêter.

Démarrage des serveurs sous Windows

Pour démarrer les serveurs sous Windows :

  1. Cliquez avec le bouton droit de la souris sur l'icône Poste de travail du bureau, puis sélectionnez Gérer.

  2. Double-cliquez sur l'icône Services et applications.

  3. Double-cliquez sur l'icône Services.

  4. Pour démarrer le service Content Server (OTCS), cliquez avec le bouton droit de la souris sur son nom, puis cliquez sur Démarrer.

  5. Pour démarrer le service d'administration Content Server (OTCS), cliquez avec le bouton droit de la souris sur son nom, puis cliquez sur Démarrer.
