La page Administration vous permet d'ajouter rapidement et sans difficulté des URL au menu Entreprise. Vous pouvez ajouter l'URL de toute page Web à laquelle Content Server est en mesure d'accéder à l'aide du protocole HTTP sur Internet, sur l'intranet de votre organisation ou dans Content Server proprement dit. Les éléments ajoutés s'affichent au-dessous du séparateur qui suit les éléments par défaut du menu Entreprise.
