Pour définir le DOCTYPE déclaré pour les pages HTML :

  1. Sur la page Administration Content Server, sous Configuration du serveur, cliquez sur le lien Configurer la présentation.

  2. Sur la page Configurer la présentation, cliquez sur le lien DOCTYPE par défaut pour les pages HTML.

  3. Sur la page DOCTYPE déclaré pour les pages HTML, dans la zone de texte DOCTYPE par défaut pour les pages HTML, saisissez votre nouvelle déclaration du type de document.

  4. Cliquez sur Enregistrer les modifications.
