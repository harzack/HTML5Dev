Pour afficher les modules pouvant être désinstallés :

  1. Dans la section Administration du module de la page Administration, cliquez sur le lien Afficher les modules pouvant être désinstallés.

  2. Sur la page Afficher les modules pouvant être désinstallés, dans la colonne Nom, prenez connaissance de la liste de modules Content Server pouvant être désinstallés de votre système.

    Si la colonne Dépendants d'un module contient une entrée, le module dépendant doit être préalablement désinstallé.

  3. Lorsque vous avez identifié les modules à désinstaller, voir Pour désinstaller un module :.
