Une fois que vous avez installé un lot de langue, vous pouvez utiliser la page Administration pour activer, désactiver et modifier la langue par défaut du système. Les utilisateurs peuvent voir toutes les langues qui sont activées et sélectionner celle qu'ils veulent utiliser avec Content Server lorsqu'ils se connectent. Les utilisateurs ne peuvent pas voir ou sélectionner des langues installées mais non activées.

 

Note :
La langue par défaut actuelle du système ne peut pas être désactivée ou supprimée. Si vous voulez désactiver une langue, vous devez d'abord activer et spécifier une autre langue comme valeur par défaut.

 

Chaque langue a un nom par défaut, un nom de langue local et un code de langue associé. Par exemple, la langue anglaise est répertoriée en tant que « Anglais (Etats-Unis) » aussi bien pour la langue que pour la langue locale, mais chaque nom peut être modifié si besoin. Le code de langue est une valeur par défaut. Elle ne peut pas être modifiée.
