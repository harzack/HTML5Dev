Pour configurer les paramètres de performances :

  1. Cliquez sur le lien Configurer les paramètres de performances de la page Administration.

  2. Sur la page Configurer les paramètres de performances, procédez de l'une des façons suivantes :

    • Saisissez un nombre entier représentant le seuil de droits d'utilisateur dans le champ Nombre de droits.

    • Saisissez le nombre maximal de sessions de connexion utilisateur que le serveur met en mémoire cache dans le champ Nombre maximal de sessions.

    • Saisissez le nombre de threads que le serveur utilise dans le champ Nombre de threads.

    • Activez ou désactivez la case à cocher Recevoir avant d'envoyer.

    • Activez ou désactivez la case à cocher Activer la mise en mémoire cache des éléments Web.

    • Saisissez la taille (en Ko) utilisée pour les opérations de copie de fichier dans le champ Taille du tampon pour la copie de fichiers.

  3. Cliquez sur le bouton Enregistrer les modifications.
