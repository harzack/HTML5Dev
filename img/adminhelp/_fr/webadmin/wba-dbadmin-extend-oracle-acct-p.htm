Pour agrandir un espace de stockage :

  1. Sur la page Maintenance du serveur Oracle, cliquez sur le lien Agrandir l'espace de stockage.

  2. Dans la liste Espace de stockage à agrandir, cliquez sur le nom de l'espace concerné.

  3. Dans le champ Chemin d'accès, saisissez le chemin absolu d'accès au fichier destiné à agrandir l'espace de stockage (par exemple, C:\orant\database\filename.ora ou /usr/oracle/database/filename.dbf). Le répertoire spécifié doit déjà exister et l'utilisateur Windows ou UNIX créé pour le serveur Oracle doit posséder une autorisation en écriture sur celui-ci.

  4. Dans le champ Taille, indiquez une taille (en Mo) pour le nouveau fichier. En vous appuyant sur la taille des précédents fichiers de données créés pour cet espace et sur le temps qu'il a fallu pour les remplir, vous pouvez déterminer approximativement la taille à attribuer au niveau fichier.

  5. Cliquez sur le bouton Agrandir espace de stockage.

  6. Si vous avez terminé les tâches que vous vouliez accomplir sur la page Maintenance du serveur Oracle, cliquez sur le lien Revenir à la page précédente.
