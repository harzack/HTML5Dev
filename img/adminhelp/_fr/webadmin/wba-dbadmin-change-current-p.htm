Pour connecter l'instance courante de Content Server à une base de données :

 

Note :
Avant de modifier la base de données à laquelle Content Server est connecté, vous devez désélectionner votre base de données active. Pour plus d'informations, consultez Pour désélectionner la base de données Content Server active.

 

  1. Procédez de l'une des manières suivantes :

    • Cliquez sur le lien Créer une nouvelle base de données.

    • Cliquez sur le lien Sélectionner base de données existante.

    La page Félicitations s'affiche lorsque vous connectez l'installation Content Server actuelle à une nouvelle base de données ou à une base de données existante Content Server. Elle confirme que la connexion à la base de données s'est correctement déroulée.

  2. Sur la page Félicitations, vous disposez des options suivantes :

    • Cliquez sur le lien Allez à la page d'accueil de Content Server pour accéder à la page de démarrage par défaut de Content Server.

    • Cliquez sur le lien Accueil Admin pour accéder à la page Administration de Content Server.

 

Note :
Si vous connectez cette installation Content Server à une base de données Content Server existante et cliquez sur Continuer de la page Confirmation de la resynchronisation de la source de données, vous devez reconfigurer les objets source de données, puis resynchroniser un par un tous les serveurs d'administration enregistrés.
