Pour ajouter un dossier Facette :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Volume Facettes.

  2. Dans le menu Ajouter un élément, sélectionnez Dossier des facettes.

     

    Astuce :
    Votre administrateur a peut-être activé un bouton Ajouter un dossier Facette à gauche du menu Ajouter un élément.

     

  3. Dans le champ Nom, saisissez un nom unique pour le nouveau dossier des facettes.

  4. Facultatif Dans le champ Description, saisissez la description du dossier des facettes.

  5. Facultatif Dans le champ Catégories, cliquez sur Modifier... pour sélectionner ou ajouter une catégorie à appliquer à ce dossier des facettes.

  6. Facultatif Pour placer le dossier des facettes dans un emplacement autre que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server..., naviguez jusqu'au recueil dans lequel vous voulez placer le dossier des facettes, puis cliquez sur le lien Sélectionner. Seules les zones de Content Server permettant d'enregistrer un dossier des facettes peuvent être sélectionnées.

  7. Cliquez sur Ajouter pour enregistrer votre nouveau dossier des facettes et revenir au volume Facettes.
