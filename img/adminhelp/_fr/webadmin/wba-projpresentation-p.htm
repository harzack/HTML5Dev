Pour configurer les paramètres de projet :

  1. Cliquez sur le lien Configurer la présentation de la section Configuration du serveur de la page Administration.

  2. Cliquez sur le lien Paramètres de projet de la page Configurer la présentation.

  3. Dans la section Page de démarrage, cliquez au choix sur le bouton radio Vue d'ensemble ou Espace de travail.

    Cette opération vous permet de définir la première page apparaissant à l'écran lorsqu'un utilisateur ouvre un projet.

  4. Dans la section Overview.html, cliquez sur le bouton radio Activer ou Désactiver.

  5. Dans la section Navigation, activez la case à cocher Menu Projet pour afficher le menu Projet dans tous les espaces de travail Projet.

  6. Procédez de l'une des manières suivantes :

    • Activez la case à cocher Bandeau d'icônes de projet, puis les cases à cocher des éléments que vous souhaitez afficher dans le bandeau d'icônes.

    • Désactivez la case à cocher Bandeau d'icônes de projet si vous ne souhaitez voir apparaître le bandeau d'icônes.

  7. Dans la section Vue d'ensemble Paramètres, activez les cases à cocher des sections que vous souhaitez rendre disponibles pour les coordinateurs de projet lorsqu'ils choisissent les sections à afficher sur la page Vue d'ensemble du projet.

  8. Dans la liste déroulante N° d'éléments, cliquez sur le nombre maximum d'éléments autorisé pour chaque section sur la page Vue d'ensemble du projet.

  9. Dans la section Indicateur de statut, saisissez une valeur pour chaque indicateur d'état dans les champs correspondants.

     

    Note :
    Si vous modifiez les valeurs par défaut de l'indicateur de statut et qu'il existe déjà des projets, les nouvelles valeurs s'appliquent immédiatement à ces projets. Avant d'effectuer une modification, prévenez les coordinateurs de projet.

     

  10. Cliquez sur le bouton Envoyer.
