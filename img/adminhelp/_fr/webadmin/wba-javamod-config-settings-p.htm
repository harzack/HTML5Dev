To administer Java module settings:

  1. In the Java Module Administration section of the Administration page, click the Administer Java Module link.

  2. Optional On the Administer Java Module page, in the JVM Settings area, in the Maximum Heap Size text field, type a positive integer. In the associated dropdown list, select either MB or GB.

  3. Optional In the RMI Port # field, type a new port number.

  4. Optional In the Application Server field, select the Configured check box to enable Java-based features in Content Server.

  5. Optional Select the Secure Attribute For Cookies check box to only allow cookies to be transmitted over a secure connection.

  6. Optional In the Allowed Client IPs area, type a client IP address in the Client IP text field. Next, click Add IP.

  7. Click Submit.

  8. Restart the server and the Web application server.
