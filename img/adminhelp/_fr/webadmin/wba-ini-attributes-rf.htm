La section [Attributes] définit les options relatives à la mise en application d'attributs complexes.

AttributeMaxRows

  • Description :

    Définit le nombre maximal de valeurs récurrentes autorisé dans l'interface utilisateur.

  • Syntaxe :

    AttributeMaxRows=50

  • Valeurs :

    Un entier compris entre 1 et 100. La valeur par défaut est 50.
