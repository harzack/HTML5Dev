Pour ajouter un processus memcached :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur le lien Ouvrir le volume des objets système.

  2. Sur la page Ouvrir le volume des objets système, cliquez sur Processus Memcached.

  3. Dans le menu Ajouter un élément, sélectionnez Processus Memcached.

  4. Dans le champ Nom, entrez un nom pour votre nouveau processus.

  5. Dans le champ Numéro de port, entrez un numéro de port disponible pour votre nouveau processus. Une fois que vous avez saisi le numéro de port, cliquez sur le lien Vérifier le port pour vous assurer qu'il est disponible.

  6. Facultatif Dans le champ Description, saisissez une description du nouveau processus.

  7. Facultatif Dans le champ Hôte, modifiez l'hôte par défaut en vous servant de la liste associée.

  8. Facultatif Dans le champ Utilisation de la mémoire, modifiez le paramètre d'utilisation de la mémoire par défaut en vous servant de la liste associée.

  9. Facultatif Dans le champ Ligne de commande supplémentaire, entrez des options de ligne de commande supplémentaires à exécuter avec votre nouveau processus.
