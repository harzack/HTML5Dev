Pour supprimer un utilisateur ou des tables Oracle :

  1. Sur la page Maintenance du serveur Oracle, cliquez sur le lien Supprimer des utilisateurs et des tables.

  2. Dans la liste Nom de l'utilisateur, cliquez sur le nom de l'utilisateur Oracle à supprimer ou dont vous voulez supprimer les tables.

  3. Procédez de l'une des manières suivantes :

    • Pour supprimer l'utilisateur Oracle sélectionné et toutes les tables dont il est propriétaire, cliquez sur le bouton radio Supprimer des utilisateurs et des tables.

    • Si vous supprimez uniquement les tables Content Server que possède l'utilisateur Oracle sélectionné, cliquez sur le bouton radio Supprimer les tables.

  4. Cliquez sur le bouton Supprimer.

  5. Cliquez sur le bouton OK de la boîte de dialogue qui s'affiche.

  6. Si vous avez terminé les tâches que vous vouliez accomplir sur la page Maintenance du serveur Oracle, cliquez sur le lien Revenir à la page précédente.
