Pour configurer le bandeau :

  1. Sur la page Administration, sélectionnez Configuration du serveur.

  2. Sélectionnez Configurer la présentation, puis Configurer le bandeau.

  3. Pour activer le bandeau pour tous les utilisateurs, vérifiez que la case Activer le bandeau est activée. Par défaut, le bandeau est activé pour tous les utilisateurs.

  4. Pour vous assurer que le bandeau est activé pour tous les utilisateurs, vous devez également sélectionner au moins un volet du bandeau. Par défaut, la case du volet du bandeau Filtre de contenu est activée.

    Sélectionnez la case à cocher Pulse à partir d'ici pour afficher le flux d'activités des mises à jour Pulse.

  5. Cliquez sur le bouton Enregistrer les modifications pour enregistrer vos changements et revenir à la page précédente, ou cliquez sur le bouton Réinitialiser pour annuler vos modifications.

     

    Notes :

    • S'il n'y a qu'un seul volet de bandeau à activer, le fait de le désactiver entraîne également la désactivation du bandeau. Pour vous assurer que le bandeau est activé pour les utilisateurs, vous devez sélectionner la case à cocher Activer la barre latérale et un volet du bandeau.

    • Vous pouvez également configurer le bandeau à partir de la page Panneau de commande du volume à facettes. Sélectionnez Outils puis Volume à facettes dans la barre de menu globale. A partir du menu de fonctions Facettes, sélectionnez Panneau de configuration. Sélectionnez Configurer le bandeau.
