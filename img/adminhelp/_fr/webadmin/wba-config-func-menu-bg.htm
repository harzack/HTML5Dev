Sur la page Configurer le menu Fonctions, vous pouvez déterminer pour chaque élément l'endroit où les fonctions vont s'afficher dans le menu Fonctions. Vous pouvez décider d'afficher la fonction d'un élément dans la partie principale du menu Fonctions ou de la masquer dans le sous-menu Plus. Les fonctions des éléments sont classées selon leur similitude, les fonctions les plus courantes s'affichant en haut du menu.

Par exemple, l'ouverture, le téléchargement ou l'affichage d'un document en tant que page Web sont des concepts similaires, ils sont donc regroupés.

La liste suivante décrit les différents types de fonctions :

  • Groupe 1 OBTENIR, qui permet à l'utilisateur de récupérer l'élément ou ses données.

  • Groupe 2 AFFECTER LE CONTENU, qui permet à l'utilisateur de modifier un aspect de l'élément.

  • Groupe 3 AFFECTER L'EMPLACEMENT, qui permet à l'utilisateur de copier, déplacer ou créer un élément à partir d'un autre élément.

  • Groupe 4 SUIVRE, qui informe l'utilisateur lorsqu'un événement se produit sur l'élément.

  • Groupe 5 COMMUNIQUER, qui permet à l'utilisateur de donner des informations sur l'élément à d'autres utilisateurs.

  • Groupe 6 AFFECTER L'ACCES, qui permet à l'utilisateur de contrôler ou autoriser l'accès aux éléments.

  • Groupe 7 SOUS-ZONES, qui permet à l'utilisateur d'accéder aux fonctions du module.

  • Groupe 8 RECHERCHE EN COURS, qui permet à l'utilisateur de rechercher des éléments similaires.

  • Groupe 9 SUPPRIMER, qui permet à l'utilisateur de supprimer des éléments.

  • Groupe 10 DETAILS, qui permet à l'utilisateur d'afficher, de supprimer ou de modifier les informations relatives à l'élément.

 

Note :
Que les fonctions de l'élément apparaissent dans le menu principal ou dans le sous-menu Plus, les autorisations des éléments standard permettent uniquement aux utilisateurs de voir les fonctions pour lesquelles ils bénéficient d'une autorisation.

Lorsque Content Server est mis à niveau vers la version 10.0.0, l'utilisateur administrateur doit aller sur la page Configurer le menu Fonctions et cliquer sur le bouton Enregistrer les modifications pour que le menu Plus apparaisse dans Content Server.
