• Description :

    La section [EditableMimeTypes] est une liste de types MIME qui peuvent être édités dans Content Server en utilisant la fonction Modification de texte. Pour plus d'informations sur la fonction Modification de texte, reportez-vous à la rubrique Ajout et modification de documents.

  • Valeurs :

    Voici un exemple de la section [EditableMimeTypes] du fichier opentext.ini :

    [EditableMimeTypes]
    text/html=TRUE
    text/plain=TRUE
    text/tab-separated-values=TRUE
    text/xml=TRUE
    text/xsl=TRUE
    text/x-setext=TRUE
    text/x-sgml=TRUE
