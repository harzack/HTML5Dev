Pour éliminer la table des données BLOB de votre base de données :

  1. Cliquez sur le bouton Eliminer de la page Eliminer les données supprimées.

    La boîte de dialogue Eliminer les données supprimées qui s'affiche indique le nombre d'enregistrements supprimés qui vont être éliminés. Une fois l'élimination terminée, la ligne Statut contient la mention Terminé.

  2. Cliquez sur le bouton OK pour fermer la boîte de dialogue.
