Exemple 4-1 : Pour que l'attribut système Date de création soit disponible comme filtre de contenu dans Content Server :

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Volume Facettes.

  2. Ouvrez le dossier Facettes système par défaut.

  3. Dans le menu Ajouter un élément, sélectionnez Facette.

  4. Dans le champ Nom, saisissez « Date de création ».

  5. Dans le champ Description, saisissez « Cette facette va filtrer le contenu par date de création. »

  6. Dans la liste Source de données, choisissez « Date de création ».

  7. Cliquez sur Ajouter.

  8. Dans le dossier Facettes système par défaut, sélectionnez l'arborescence de facettes Facettes système par défaut.

  9. Sur la page Facettes système par défaut, cliquez sur le signe plus à côté de Facettes système par défaut.

  10. Dans la zone de liste qui s'affiche, sélectionnez Date de création.

  11. Cliquez sur Mettre à jour.

  12. Naviguez jusqu'à la page Facettes système par défaut, ouvrez la facette Date de création.

  13. Sur la page de propriétés Date de création, sélectionnez l'onglet Spécifique.

  14. Dans l'onglet Spécifique, dans le champ Valeurs uniques minimum, augmentez la valeur à « 4 ».

  15. Cliquez sur Mettre à jour.

Exemple 4-2 : Pour créer une facette à partir d'un attribut de catégorie et rendre cette facette disponible sous la forme d'un filtre de contenu vers un dossier uniquement :

 

Note :
Cet exemple suppose que vous avez préalablement créé une catégorie appelée « Emplacements de bureau », avec un attribut de catégorie Texte : Menu contextuel appelé « Emplacement de bureau ». L'attribut de catégorie « Emplacement de bureau » contient une liste de noms de villes. Par exemple : Chicago, Londres, New York, Paris, Seattle, Toronto.

Cet exemple suppose également que vous avez préalablement créé un dossier dans l'espace de travail Entreprise appelé « Bureaux ».

 

  1. Dans le menu Outils de la barre de menu générale, sélectionnez Volume Facettes.

  2. Ouvrez le dossier Facettes système par défaut.

  3. Dans le menu Ajouter un élément, sélectionnez Dossier des facettes.

  4. Dans le champ Nom, saisissez « Emplacements de bureau ».

  5. Dans le champ Description, saisissez « Contient les facettes et l'arborescence de facettes du filtre de contenu Emplacements de bureau, affiché sur la page Entreprise :Bureaux. ».

  6. Cliquez sur Ajouter.

  7. Ouvrez le dossier des facettes Emplacements de bureau.

  8. Dans le menu Ajouter un élément, sélectionnez Facette.

  9. Dans le champ Nom, saisissez « Emplacement de bureau ».

  10. Dans le champ Description, saisissez « Cette facette va filtrer le contenu en fonction de l'attribut de catégorie Emplacement de bureau. »

  11. Dans la liste Source de données, choisissez « Catégorie :Emplacements de bureau :Emplacement de bureau ».

  12. Cliquez sur Ajouter.

  13. Naviguez jusqu'au dossier Facettes système par défaut. Dans le menu Ajouter un élément, sélectionnez Arborescence de facettes.

  14. Dans le champ Nom, saisissez « Emplacements de bureau ».

  15. Cliquez sur Ajouter.

  16. Ouvrez l'arborescence de facettes Emplacements de bureau.

  17. Cliquez sur le signe plus en regard d'Emplacements de bureau.

  18. Dans la zone de liste qui s'affiche, sélectionnez Emplacement de bureau.

  19. Cliquez sur Mettre à jour.

  20. Ouvrez l'arborescence de facettes Emplacements de bureau.

  21. Sélectionnez l'onglet Disponibilité.

  22. Sélectionnez le bouton Afficher uniquement à des emplacements spécifiques.

  23. Cliquez sur Parcourir Content Server.... Naviguez jusqu'au dossier Bureaux précédemment créé dans l'espace de travail Entreprise.

  24. Cliquez sur Mettre à jour.
