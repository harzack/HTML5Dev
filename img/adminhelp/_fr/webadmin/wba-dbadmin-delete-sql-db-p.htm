Pour supprimer une base de données du serveur SQL :

  1. Sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Supprimer une base de données Microsoft SQL Server.

  2. Dans la zone de liste déroulante Nom de la base de données, cliquez sur le nom de la base de données du serveur SQL à supprimer.

  3. Cliquez sur le bouton Supprimer la base de données.

  4. Cliquez sur le bouton OK de la boîte de dialogue qui s'affiche.

  5. Si vous avez terminé les tâches qui vous intéressent sur la page Maintenance Microsoft SQL Server, cliquez sur le lien Revenir à la page précédente.
