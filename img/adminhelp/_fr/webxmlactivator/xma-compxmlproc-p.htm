Pour terminer l'installation d'un processus XML Activator

  1. Dans le champ Balise de contenu, saisissez le nom de la balise XML qui identifie le contenu.

  2. Dans le champ Attribut de balise, saisissez le nom de l'attribut qui représente l'attribut de codage.

  3. Dans le champ Valeur d'attribut de balise, saisissez la valeur qui représente le type de codage des fichiers XML.

  4. Dans la liste déroulante Codage, choisissez l'un des types de codage suivants pour le contenu de vos fichiers XML :

    • Base64 définit le type de codage des fichiers XML sur Base64.

    • Aucun ne définit aucun type de codage pour les fichiers XML.

  5. Dans le champ Ecrire les données dans, saisissez le chemin absolu de l'IPool dans lequel vous voulez que le processus producteur XML Activator écrive les données.

  6. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que le processus producteur XML Activator doit précéder dans le flux de données.

  7. Spécifiez les options de lancement d'un processus de flux de données du processus de producteur XML Activator dans la section Options de lancement.

  8. Spécifiez les options d'arrêt d'un processus de flux de données du processus de producteur XML Activator dans la section Options d'arrêt.

  9. Cliquez sur le bouton Ajouter.
