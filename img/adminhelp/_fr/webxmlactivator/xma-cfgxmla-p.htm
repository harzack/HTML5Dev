Pour configurer un processus XML Activator :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données de l'index pour lequel vous voulez configurer un processus XML Activator.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur l'icône Fonctions du processus XML Activator, puis sélectionnez Arrêter.

  4. Cliquez sur le nom du processus XML Activator.

  5. Dans l'onglet Infos spécifiques de la page Propriétés XML Activator, Configuration d'un processus XML Activator du processus.

  6. Cliquez sur le bouton Mettre à jour.

  7. Cliquez sur l'icône Fonctions du processus XML Activator, puis sélectionnez Démarrer.
