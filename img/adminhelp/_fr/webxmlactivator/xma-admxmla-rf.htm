Les processus de flux de données XML Activator vous permettent de connecter des applications tierces à des flux de données. Cela vous permet d'intégrer Content Server et des applications tierces de plusieurs façons. Vous pouvez :

  • envoyer des informations de flux de données Content Server à une application tierce plutôt que d'indexer les informations dans Content Server ;

  • envoyer des informations depuis le milieu d'un flux de données Content Server à une application tierce, ainsi qu'au processus Content Server suivant ;

  • filtrer des informations Content Server via une application tierce avant de les envoyer au processus Content Server suivant ;

  • indexer des informations d'applications tierces dans Content Server.

Les applications tierces ne nécessitent pas de bibliothèques, d'outils exclusifs ou de langages de code particuliers pour se connecter à Content Server. Elles se connectent à Content Server via des fichiers XML.
