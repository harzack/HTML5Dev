Pour configurer les informations de mise à jour de l'index :

  1. Dans le champ Répertoire de lecture, saisissez le chemin absolu du répertoire dans lequel vous voulez que ce processus lise les données.

  2. Dans le champ Répertoire d'indexation, saisissez le chemin absolu du répertoire dans lequel vous voulez que le processus du moteur d'indexation crée l'index. Vous devez choisir un répertoire d'indexation sur un lecteur local de l'hôte du serveur d'administration qui exécute ce processus.

  3. Dans la liste déroulante Hôte de la section Client : Index Content Server, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Distributeur de mise à jour doit s'exécuter.
