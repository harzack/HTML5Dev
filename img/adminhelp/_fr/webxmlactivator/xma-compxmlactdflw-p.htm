Pour terminer un flux de données producteur XML Activator :

  1. Pour lancer les processus de flux de données dès qu'ils sont créés, cochez la case Lancer les processus dans le flux de données.

  2. Cliquez sur le bouton Créer des processus.

  3. Sur la page Statut de la création d'un flux de données, cliquez sur le bouton Continuer.
