Sortie multi-fichiers facilite et accélère les opérations suivantes :

  • Télécharger : compresser un ou plusieurs objets OpenText Content Server dans un fichier zip en vue de les télécharger.

  • E-mail : compresser, télécharger et lier un ou plusieurs objets à un nouveau message électronique ou enregistrer le fichier zip qui en résulte à nouveau dans Content Server et envoyer par e-mail une URL avec lien hypertexte.

  • Imprimer : télécharger et imprimer un ou plusieurs documents, qu'ils soient simples ou multiples, à partir de Content Server.

La Sortie multi-fichiers fonctionne avec tout type de document pris en charge pour lequel l'utilisateur dispose au moins d'autorisations Voir le contenu. Elle peut exporter le contenu de dossiers entiers, de documents composites ou d'autres recueils pris en charge en une seule opération. Vous pouvez également utiliser la Sortie multi-fichiers avec des Collections pour rassembler des informations de n'importe où dans Content Server, puis les exporter globalement (une page à la fois), en imitant le système de "panier" du commerce électronique.

Multi-File Output prend également en charge des objets OpenText Enterprise Library. Lorsque vous exécutez une tâche Sortie multi-fichiers sur des objets OpenText Enterprise Library, Content Server utilise la variante par défaut pour le contenu.

 

Notes :

  • Content Server permet de disposer de versions majeures et mineures des documents. Pour que la Sortie multi-fichiers fonctionne avec les versions secondaires de document, l'utilisateur doit disposer d'autorisations Réserver.

  • Les recueils dans Content Server peuvent inclure d'autres recueils (par exemple, dossiers dans des dossiers). Si un des objets sélectionnés est un recueil incluant d'autres recueils, l'action lancée ne s'étend pas aux sous-recueils ; par exemple, seuls les objets situés au niveau supérieur sont concernés. Pour appliquer une action au contenu d'un sous-recueil, accédez au menu Fonctions de ce recueil.

  • La Sortie multi-fichiers prend en charge les types de recueil suivants : Documents composites, Dossiers, Projets, Tâches et Workflows.

  • Multi-File Output ne prend pas en charge les documents composites pour les objets OpenText Enterprise Library.
