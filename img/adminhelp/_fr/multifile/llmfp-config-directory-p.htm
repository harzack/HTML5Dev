Pour configurer les paramètres généraux :

  1. Dans Administration des sorties multi-fichiers sur la page Administration Content Server, cliquez sur le lien Paramètres généraux.

  2. Sur la page Paramètres généraux, entrez le chemin de l'emplacement dans lequel vous voulez ajouter les fichiers compressés. Par défaut, le chemin est le suivant : OPENTEXT\temp\multifile.

  3. Dans le champ Durée de vie de la mémoire cache SMTP, entrez le nombre de secondes pendant lequel vous voulez que le cache soit valide avant que Content Server ne tente d'obtenir de nouveaux paramètres. Par défaut, ce champ est défini sur 300.

  4. Cochez la case Activer l'accélérateur SSL du répartiteur de charge pour accélérer l'envoi/la réception de données pour les applications Web. Par défaut, cette case est désactivée.

  5. Dans la zone Nombre maximal d'adresses, entrez le nombre maximal d'adresses e-mail autorisées lors de l'envoi d'e-mails dans Nombre maximal d'adresses e-mail. Par défaut, ce champ est défini sur 10.

  6. Dans la section Options des pièces jointes, cliquez sur l'un des boutons radio suivants :

    • Permettre à l'utilisateur de choisir, qui permet à l'utilisateur de choisir si les pièces jointes doivent être ajoutées en tant que fichiers ou URL. Par défaut, cette option est cochée.

    • Utiliser uniquement des fichiers, qui permet d'inclure les pièces jointes en tant que fichier uniquement.

    • Utiliser uniquement des URL, qui permet d'inclure des pièces jointes sous formes de références à des URL vers l'emplacement du fichier dans Content Server.

  7. Entrez le nombre de Ko que les utilisateurs sont autorisés à télécharger, envoyer par e-mail ou imprimer (une fois que le fichier a été compressé) dans les champs Limiteur de bande passante. Par défaut, tous les champs sont définis sur 5120 Ko.

  8. Sélectionnez les objets qui auront des fonctionnalités de téléchargement, envoi par e-mail et impression appliquées dans le champ Configuration d'objets multi-fichiers. Par défaut, les fonctionnalités de téléchargement, envoi par e-mail et impression ne sont appliquées qu'aux Documents et Dossiers.

  9. Cliquez sur Envoyer pour enregistrer votre paramètre.
