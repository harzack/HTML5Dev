Pour configurer les rapports sur les éléments évalués :

  1. Dans la zone Gestion des communautés sur la page Gestion de Content Server, cliquez sur le lien Configurer les rapports des éléments évalués.

  2. Sur la page Configurer les rapports des éléments évalués, effectuez l'une des actions suivantes :

    • Dans la zone Envoyer des rapports, activez la case à cocher Activer pour pouvoir envoyer des rapports sur les éléments évalués par e-mail.

    • Dans la zone Types d'élément, activez les cases à cocher correspondant aux types d'élément à inclure dans les rapports sur les éléments évalués.

  3. Cliquez sur Mettre à jour.
