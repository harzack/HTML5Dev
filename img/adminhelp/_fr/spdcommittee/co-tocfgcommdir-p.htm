Pour configurer l'annuaire des communautés :

  1. Cliquez sur Annuaire des communautés dans le bandeau Console de navigation, puis sur Apparence.

  2. Sur la page Affichage de la page d'accueil, cliquez sur le nom d'un module dans la liste Modules de contenu inactifs, puis sur le bouton Activer pour activer un module de contenu inactif.

  3. Pour désactiver un module de contenu actif, cliquez sur un nom de module dans la liste Colonne de gauche ou Colonne de droite, puis sur le bouton Supprimer .

  4. Pour déplacer les modules de contenu actifs entre Colonne de gauche ou Colonne de droite, cliquez sur le nom du module, puis cliquez sur le bouton ou .

  5. Pour modifier la largeur des colonnes de gauche et de droite, cliquez sur l'un des boutons radio suivants dans la section Mise en page :

    • Colonne de gauche large

    • Colonnes égales

    • Colonne de droite large

  6. Pour ajouter un bandeau sous la Console de navigation des annuaires de communauté, saisissez un titre pour le bandeau dans le champ Titre, puis saisissez le texte du bandeau dans le champ Corps.

     

    Astuce :
    Pour ajouter un autre bandeau, cliquez sur le bouton Ajouter un bandeau , puis répétez cette étape.

     

  7. Cliquez sur Mettre à jour.
