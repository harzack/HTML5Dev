Pour configurer les restrictions d'e-mail :

  1. Dans la zone Gestion des communautés de la page Gestion de Content Server, cliquez sur le lien Configurer les restrictions d'e-mail.

  2. Sur la page Configurer les restrictions d'e-mail, activez ou désactivez les cases à cocher Activer suivantes :

    • Pièces jointes externes

    • Champ Adresses e-mail externes

    • Avertissement

      1. Cliquez sur le bouton radio Couleur de fond correspondant à la couleur de fond souhaitée pour l'avertissement.

      2. Saisissez le texte d'avertissement dans le champ Texte d'avertissement.

  3. Cliquez sur Soumettre.
