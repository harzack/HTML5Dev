Pour configurer le paramétrage de la communauté :

  1. Dans la zone Gestion des communautés sur la page Gestion de Content Server, cliquez sur le lien Configurer le paramétrage de la communauté.

  2. Sur la page Configurer le paramétrage de la communauté, effectuez l'une des actions suivantes :

    1. Activez ou désactivez les cases à cocher Activer dans les sections suivantes :

      • ‎Console de navigation

      • Page d'accueil personnelle

      • Fonctions utilisateurs indirects

      • Informations publiques sur les membres

      • Remplacement du statut

      • Rendez-vous de calendrier

      • Mesures

      • Liens abrégés

    2. Saisissez l'un des éléments suivants dans le champ Taille max. de la photo personnelle :

      • Un entier positif, qui définira la taille maximale des fichiers, en octets, pour les photos personnelles ;

      • 0, pour désactiver les photos personnelles.

    3. Dans la zone Affichage du calendrier, procédez comme suit :

      • Activez le bouton radio Affichage du calendrier ou Affichage liste, puis sélectionnez le style d'affichage pour les listes correspondantes.

      • A l'aide des listes proposées, sélectionnez une heure de début et une heure de fin.

    4. Saisissez le code HTML que vous souhaitez afficher dans la section du pied-de-page des e-mails dans le champ Pied de page d'e-mail HTML.

    5. Saisissez le texte que vous souhaitez afficher dans la section du pied-de-page des e-mails dans le champ Pied de page d'e-mail Texte.

    6. Activez ou désactivez les cases à cocher pour les types d'élément Content Server dans la zone Eléments auditables.

  3. Cliquez sur Mettre à jour.
