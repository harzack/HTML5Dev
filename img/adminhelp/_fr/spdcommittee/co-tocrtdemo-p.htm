Pour créer un modèle de démonstration :

  1. Dans la zone Gestion des communautés de la page Gestion de Content Server, cliquez sur le lien Ouvrir le volume des modèles de communauté.

  2. Sur la page Volume des modèles de communauté, cliquez sur le lien Créer un modèle de démonstration.

  3. Sur la page Nouveau modèle de démonstration de communauté, cliquez sur Contrôles du modérateur dans le bandeau Console de navigation, puis sur Configuration.

  4. Sur la page Configurer le paramétrage de la communauté, configurez le modèle comme vous le souhaitez à l'aide des onglets suivants :

    • Page d'accueil

    • Bannière de la page d'accueil

    • ‎Console de navigation

    • Configuration

    • Résumé

     

    Astuce :
    Vous pouvez ajouter des rôles supplémentaires en répétant ces procédures à partir de l'étape étape 6 de la procédure Pour ajouter un modèle de communauté.

     

  5. Cliquez sur Terminé, puis sur le lien Lorsque vous avez fini d'apporter des modifications, cliquez ici.
