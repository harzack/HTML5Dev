Pour ajouter un dossier de modèles de communauté :

  1. Dans la zone Gestion des communautés de la page Gestion de Content Server, cliquez sur le lien Ouvrir le volume des modèles de communauté.

  2. Sur la page Volume des modèles de communauté, cliquez sur Dossier de modèles de communauté dans le menu Ajouter un élément.

  3. Sur la page Ajouter un dossier de modèles de communauté, saisissez le nom du dossier dans le champ Nom.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
