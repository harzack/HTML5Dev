Pour ajouter une structure d'annuaires :

  1. Cliquez sur Communautés dans le menu Entreprise.

  2. Sur la page Annuaire des communautés, cliquez sur Structure d'annuaires dans le menu Ajouter un élément.

  3. Sur la page Ajouter une structure d'annuaires, saisissez le nom de la structure d'annuaires dans le champ Nom.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.
