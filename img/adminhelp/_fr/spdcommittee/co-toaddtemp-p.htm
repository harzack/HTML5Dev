Pour ajouter un modèle de communauté :

  1. Dans la zone Gestion des communautés de la page Gestion de Content Server, cliquez sur le lien Ouvrir le volume des modèles de communauté.

  2. Sur la page Volume des modèles de communauté, cliquez sur Modèle de communauté dans le menu Ajouter un élément.

  3. Sur la page Créer un modèle de communauté, saisissez le nom du modèle dans le champ Nom.

  4. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  5. Cliquez sur Ajouter.

  6. Sur la page Autorisations, cliquez sur le bouton Ajouter un rôle .

  7. Saisissez un nom de rôle dans le champ Nom du rôle, par exemple Responsable ou Modérateur.

  8. Facultatif Vous pouvez ajouter des informations supplémentaires dans les champs Description du rôle et Remarques.

  9. Cliquez sur Oui dans la liste Modérateur pour assigner le rôle de modérateur de communauté.

  10. Dans la zone Autorisations, accordez les autorisations pour ce rôle en sélectionnant les cases à cocher correspondantes. Par exemple, si ce rôle est pour un Modérateur de communauté, activez la case à cocher Modifier les autorisations pour accorder les autorisations maximales du rôle.

  11. Cliquez sur Ajouter, puis sur OK.

     

    Astuce :
    Vous pouvez ajouter d'autres rôles en répétant cette procédure à partir de l'étape étape 6.

     

  12. Cliquez sur Terminé.

  13. Cliquez sur le lien <Nouveau modèle de communauté>.

  14. Sur la page Nouveau modèle de communauté, cliquez sur Contrôles du modérateur dans le bandeau Console de navigation, puis choisissez Configuration.

  15. Sur la page Configurer le paramétrage de la communauté, configurez le modèle comme vous le souhaitez à l'aide des onglets suivants :

    • Page d'accueil

    • Bannière de la page d'accueil

    • ‎Console de navigation

    • Configuration

    • Résumé

  16. Cliquez sur Terminé.

  17. Sur la page Configurer le paramétrage de la communauté, cliquez sur le lien Cliquez ici une fois les modifications terminées.
