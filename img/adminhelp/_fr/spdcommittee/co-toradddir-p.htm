Pour ajouter un annuaire :

  1. Choisissez Communautés dans le menu Entreprise.

  2. Sur la page Annuaire des communautés, cliquez sur le lien <Nom de structure d'annuaires> de la structure d'annuaires où vous souhaitez ajouter un annuaire.

  3. Sur la page Annuaire des communautés, cliquez sur Annuaire dans le menu Ajouter un élément.

  4. Sur la page Ajouter un annuaire, saisissez un nom pour l'annuaire dans le champ Nom.

  5. Spécifiez les autres paramètres généraux nécessaires pour l'élément.

  6. Cliquez sur Ajouter.
