Pour configurer des paramètres eLink avancés :

  1. Dans la section Administration eLink de la page Administration, cliquez sur le lien Paramètres eLink avancés.

  2. Sur la page Paramètres eLink avancés, procédez comme suit dans la section Documents par e-mail :

    • Sélectionnez la case à cocher Activer l'envoi de documents par e-mail via le menu Fonctions pour permettre aux utilisateurs d'envoyer des documents par e-mail à partir du menu Fonctions.

    • Sélectionnez la case à cocher Accuser réception du contenu de l'e-mail entrant pour envoyer un e-mail à la réception de contenu d'un dossier ou espace de travail activé pour l'envoi par e-mail.

    • Cliquez sur une option dans la liste Option de lien de document e-mail pour choisir le type de lien inclus dans les messages :

      • Ouvrir, pour ouvrir le document.

      • Propriétés, pour ouvrir l'onglet Général du document.

      • Afficher, pour ouvrir l'onglet Spécifiques du document.

  3. Procédez comme suit dans la section Discussions par e-mail :

    • Sélectionnez la case à cocher Autoriser les utilisateurs ne se servant pas de Content Server à publier des messages dans des discussions pour autoriser les utilisateurs n'ayant pas accès à Content Server à participer aux discussions activées par e-mail.

    • Sélectionnez la case à cocher Autoriser les utilisateurs ne bénéficiant pas d'autorisation de modification à publier des discussions pour autoriser les utilisateurs sans droits de publication dans une discussion à publier des sujets.

    • Sélectionnez la case à cocher Autoriser les utilisateurs à résilier leur abonnement automatisé à un groupe pour autoriser les utilisateurs d'un groupe à se désabonner d'une discussion.

    • Sélectionnez la case à cocher Envoyer sujet/réponse uniquement aux membres du groupe si vous voulez envoyer des sujets ou des réponses uniquement aux utilisateurs directement abonnés à la discussion (mais non aux utilisateurs abonnés via un groupe).

  4. Procédez comme suit dans la zone Ma boîte de réception :

    • Sélectionnez la case à cocher Mon client de messagerie pour autoriser les utilisateurs à accéder au client de messagerie léger Content Server via l'option Ma boîte de réception du menu Personnel.

       

      Note :
      Pour changer la valeur de la case Mon client de messagerie, vous devez redémarrer le serveur. Pour plus d'informations sur le redémarrage du serveur, voir Arrêt et démarrage des serveurs.

       

    • Spécifiez les paramètres de connexion pour utiliser un serveur e-mail POP3 au lieu des paramètres configurés par les utilisateurs.

    • Sélectionnez l'option Utiliser le nom d'utilisateur Content Server comme nom d'utilisateur POP3 pour demander aux utilisateurs de se connecter au serveur POP3 à l'aide de leurs informations d'authentification Content Server.

  5. Dans la section Signature professionnelle, sélectionnez l'option Inclure la signature professionnelle dans les e-mails sortants pour ajouter une signature aux messages eLink, puis saisissez le texte de la signature dans le champ Signature professionnelle.

  6. Cliquez sur le bouton Envoyer.

  7. Si vous avez sélectionné l'option Mon client de messagerie, redémarrez le système, puis cliquez sur le lien Continuer.
