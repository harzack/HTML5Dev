Pour configurer des paramètres généraux :

  1. Dans la section Administration eLink sur la page Administration, cliquez sur le lien Configurer eLink.

  2. Sur la page de configuration d'eLink, effectuez les opérations suivantes dans la section Général :

    • Cliquez sur l'une des options suivantes dans la liste Contenu de l'e-mail :

      • Texte, pour qu'eLink envoie par défaut des e-mails au format texte seulement. Tous les clients de messagerie sont en mesure de gérer ce type de message.

      • HTML, pour qu'eLink envoie par défaut des e-mails au format HTML.

    • Sélectionnez la case à cocher M'envoyer par e-mail les messages/réponses que j'ai publiés pour qu'eLink envoie un e-mail aux utilisateurs à chaque fois qu'ils publient un message de discussion ou une réponse.

    • Cliquez sur l'une des options suivantes dans la liste Activer le style de remplacement pour les e-mails des discussions :

      • Désactivé, pour que eLink affiche les e-mails des discussions au format standard.

      • Activé, pour que les e-mails des discussions d'eLink intègrent, dans le champ A partir de, le nom de l'utilisateur qui a publié le message ou la réponse et référencent le nom de la discussion entre parenthèses dans le champ Objet, avant l'objet lui-même.

    • Dans le champ Messages non transmis, saisissez un nombre entier pour indiquer le nombre de jours pendant lesquels eLink doit tenter de renvoyer les messages non transmis avant de les rejeter.

    • Dans le champ Nom d'hôte virtuel, saisissez le nom de domaine que votre serveur de messagerie héberge virtuellement pour eLink.

    • Sélectionnez la case à cocher Journal détaillé pour activer le journal détaillé des transactions eLink dans Content Server.

  3. Dans le champ Taille maximale des messages sortants de la section Pièces jointes, saisissez la taille maximale autorisée pour la pièce jointe.

  4. Dans la section SMTP, procédez comme suit :

    • Dans le champ Serveur, saisissez le nom du serveur de messagerie par l'intermédiaire duquel eLink envoie les e-mails sortants.

    • Dans le champ Port, saisissez le numéro du port de réception du serveur SMTP. Le port par défaut est 25.

    • Cliquez sur le bouton Tester la connexion pour tester les paramètres de connexion SMTP des e-mails sortants.

  5. Dans la section POP3, procédez comme suit :

    • Dans le champ Serveur, saisissez le nom du serveur de messagerie sur lequel sont stockés les e-mails entrants d'eLink.

    • Dans le champ Port, saisissez le numéro du port de réception du serveur POP3. Le port par défaut est 110.

    • Dans le champ Nom d'utilisateur, saisissez le nom de la boîte de réception du serveur POP3 qui reçoit les e-mails d'eLink.

    • Dans le champ Mot de passe, saisissez le mot de passe correspondant au nom d'utilisateur POP3 d'eLink.

    • Cliquez sur le bouton Tester la connexion pour tester les paramètres de connexion POP3 des e-mails entrants.

  6. Dans la section Filtres des messages entrants, effectuez la procédure suivante :

    • Dans la liste Rejeter les messages contenant ces en-têtes X, saisissez les en-têtes des messages entrants que eLink doit rejeter.

    • Dans la zone de liste Rejeter les messages contenant cet objet, saisissez l'objet des e-mails entrants qu'eLink doit rejeter.

    • Dans la zone de liste Rejeter les messages en provenance de ces adresses, saisissez l'intégralité ou une partie des adresses e-mails des messages entrants qu'eLink doit rejeter.

    • Dans la liste Accepter les messages pour les domaines de la liste uniquement, sinon laisser vide sans spécifier de restrictions, saisissez les domaines des messages que eLink doit accepter. Les domaines non répertoriés seront rejetés.

    • Dans la liste Rejeter les pièces jointes de ces types MIME ou ayant ces extensions de fichier, saisissez les types MIME ou les extensions de fichier que eLink doit rejeter.

  7. Cliquez sur le bouton Envoyer.

  8. Redémarrez le système, puis cliquez sur le lien Continuer.
