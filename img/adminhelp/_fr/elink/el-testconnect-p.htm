Pour tester les connexions eLink :

  1. Dans la section Administration eLink sur la page Administration, cliquez sur le lien Configurer eLink.

  2. Dans la section SMTP, cliquez sur le bouton Tester la connexion pour tester les paramètres de connexion SMTP des e-mails sortants.

  3. Dans la section POP3, cliquez sur le bouton Tester la connexion pour tester les paramètres de connexion POP3 des e-mails entrants.
