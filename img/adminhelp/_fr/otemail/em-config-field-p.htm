Pour configurer les champs d'e-mail :

  1. Sur la page Administration Content Server Content Server, cliquez sur le lien Configurer les services d'e-mail de la section Administration Email Services.

  2. Sur la page Configurer les champs d'e-mail, sélectionnez les options Afficher et Modifiable pour les champs d'e-mail.

     

    Note :
    L'option Afficher active la case Modifiable du champ d'e-mail. Lorsque l'option Afficher n'est pas cochée, le champ n'apparaît pas dans la boîte de dialogue Ajouter un e-mail, mais les propriétés du champ d'e-mail seront tout de même enregistrées.

     

  3. Cliquez sur Envoyer.
