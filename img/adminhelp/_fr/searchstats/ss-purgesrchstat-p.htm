Pour éliminer les statistiques de recherche :

  1. Sur la page Gérer les statistiques de recherche Content Server, cliquez sur le lien Eliminer les statistiques de recherche.

  2. Sur la page Eliminer les statistiques de recherche, spécifiez vos critères d'élimination.

  3. Cliquez sur le bouton OK ou Eliminer toutes les statistiques.
