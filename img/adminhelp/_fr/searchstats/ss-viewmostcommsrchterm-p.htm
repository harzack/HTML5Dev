Pour afficher les termes les plus souvent recherchés :

  1. Sur la page Afficher les termes recherchés Content Server, cliquez sur le lien Afficher les termes les plus souvent recherchés.

  2. Sur la page Afficher les termes les plus souvent recherchés, effectuez une sélection dans la liste déroulante Nombre à afficher (la valeur par défaut est 10).

  3. Saisissez le nombre de jours requis dans le champ Au cours des derniers/dernières (laisser ce champ vide affiche tous les jours).

  4. Cliquez sur le bouton Afficher les résultats.
