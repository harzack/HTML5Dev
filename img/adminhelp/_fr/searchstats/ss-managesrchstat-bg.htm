Lorsque vous vous connectez à Content Server en tant qu'administrateur, vous pouvez gérer les statistiques de recherche et identifier la manière dont les utilisateurs exploitent les recherches.

Cette fonction permet d'effectuer les opérations suivantes :

  • Afficher le récapitulatif des statistiques

  • Afficher les termes recherchés

  • Afficher l'utilisation de vue de recherche

  • Afficher l'utilisation du modèle

  • Eliminer les statistiques de recherche

Afficher le récapitulatif des statistiques

La page Afficher le récapitulatif des statistiques vous permet d'afficher des statistiques de recherche générales pour les utilisateurs de Livelink. Ces statistiques comprennent : le total des recherches, le nombre de formulaires et de vues de recherche utilisés, les sources des requêtes de recherche et les types de recherche.

Affichage des termes recherchés

L'affichage des termes recherchés vous permet de consulter les termes les plus fréquemment recherchés, ou de rechercher un terme particulier. La sélection de ce lien vous permet de choisir une des options suivantes :

  • Afficher les termes les plus souvent recherchés. Cette page fournit une liste des termes les plus souvent recherchés y compris leur fréquence de recherche ainsi que la date et l'heure de leur dernière recherche.

  • Afficher les recherches non productives. Cette page fournit une liste des termes les plus souvent recherchés par les utilisateurs de Content Server mais qui n'ont engendré aucun résultat. Une recherche incluant plusieurs vues de recherche ou emplacements indique Multiple dans la colonne de vue de recherche.

  • Afficher les statistiques relatives à un terme spécifique. Cette page fournit des données sur un terme spécifique, y compris les termes recherchés dans la même requête, ainsi que le nombre de résultats renvoyés. Lorsqu'une chaîne de texte figure entre guillemets (p.ex., « connexion »), la recherche génère des résultats similaires aux recherches de terme unique.

Affichage de l'utilisation de vue de recherche

L'utilisation de vue de recherche permet d'afficher le nombre de vues de recherche, la vue de recherche la plus utilisée et les utilisations spécifiques de vue de recherche.

Affichage de l'utilisation du formulaire

L'utilisation du formulaire vous permet d'afficher le nombre de formulaires, les formulaires les plus utilisés et les utilisations spécifiques du formulaire.

Elimination des statistiques de recherche

L'option Eliminer les statistiques de recherche vous permet d'éliminer manuellement ou automatiquement les données. L'élimination manuelle comme automatique consigne les données dans le journal d'audit Content Server standard à des fins de suivi.
