Pour afficher les statistiques relatives à un terme spécifique :

  1. Sur la page Afficher les termes recherchés Content Server, cliquez sur le lien Afficher les statistiques relatives à un terme spécifique.

  2. Sur la page Afficher les statistiques relatives à un terme spécifique, saisissez un terme spécifique dans le champ Terme.

  3. Effectuez une sélection dans la liste déroulante Nombre de requêtes (la valeur par défaut est 10).

  4. Cliquez sur le bouton Afficher les résultats.
