Pour ajouter un processus de sauvegarde :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier source de données dans lequel vous souhaitez ajouter un processus de sauvegarde.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de sauvegarde. Si ce lien n'existe pas, vous devez ajouter un gestionnaire de sauvegarde. Pour en savoir plus, voir Pour ajouter un gestionnaire de sauvegarde.

  3. Sur la page préfixe_processus Gestionnaire de sauvegarde, cliquez sur Processus de sauvegarde dans le menu Ajouter un élément.

  4. Sur la page Ajouter Processus de sauvegarde, saisissez un nom dans le champ Nom.

  5. Dans l'onglet Général de la page Propriétés, vous pouvez ajouter une description du processus de sauvegarde dans le champ Description.

  6. Pour que Content Server contrôle ce processus et détecte les messages d'erreur retournés, cochez la case Activer la gestion du système.

  7. Cliquez sur le nom de la partition à sauvegarder dans la liste déroulante Partition.

  8. Pour modifier le format de la chaîne de libellé associée à la sauvegarde, modifiez le texte dans le champ Libellé de sauvegarde.

     

    Note :
    Le libellé permet d'identifier la sauvegarde. La valeur par défaut est Source_Données %m%d%Y_%T%r, où Source_Données représente une chaîne identifiant la source de données, %m le mois, %d le jour, %Y l'année, %T le type de sauvegarde effectué et %r l'ID de partition. Pour obtenir la liste complète des variables permettant la mise en forme d'un modèle de libellé, reportez-vous à la rubrique Variables Date et Heure.

     

  9. Cliquez sur l'un des types de sauvegarde suivants dans la liste déroulante Type de sauvegarde :

  10. Cliquez sur le niveau de transmission des messages d'erreur de votre choix dans la liste déroulante Après message d'erreur :

    • Aucun, qui est l'option par défaut ;

    • Informations, qui permet de notifier au destinataire le statut général et d'autres informations système ;

    • Avertissement, qui permet de notifier au destinataire les erreurs non critiques dans le système ;

    • Erreur, qui permet de notifier au destinataire les erreurs système générales ;

    • Erreur grave, qui permet de notifier au destinataire les erreurs système critiques.

     

    Note :
    Pour plus d'informations sur la transmission des messages d'erreur, reportez-vous à la rubrique Activation de la recherche d'erreurs et des e-mails d'alerte.

     

  11. Dans la section Options de lancement, activez le bouton radio correspondant au moment où vous voulez que le processus de sauvegarde ait lieu :

    • A cette heure : cette option permet de programmer un processus de sauvegarde pour qu'il s'exécute à une heure précise certains jours de la semaine. Cliquez sur les valeurs dans les listes déroulantes, puis activez les cases à cocher pour indiquer l'heure et les jours de la semaine où vous voulez que le processus de sauvegarde ait lieu.

    • Chaque : cette option permet de programmer un processus de sauvegarde pour qu'il s'exécute à un intervalle régulier. Cliquez sur les valeurs dans les listes déroulantes pour spécifier les unités de temps et la durée de l'intervalle auquel vous voulez que le processus de sauvegarde s'exécute.

  12. Cliquez sur le bouton Ajouter.

  13. Sur la page préfixe_processus Gestionnaire de sauvegarde, cliquez sur l'icône Fonctions du processus de sauvegarde, puis sélectionnez Démarrer.
