Pour configurer le service de conversion de documents (DCS) :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions d'un serveur d'administration, choisissez Propriétés, puis Service de conversion de documents.

  2. Effectuez l'une des actions suivantes :

    • Cliquez sur une valeur dans la liste déroulante Niveau de consignation pour changer le niveau de consignation par défaut (1).

    • Saisissez l'emplacement du répertoire du fichier temporaire dans le champ Répertoire du fichier temporaire.

    • Cliquez sur une valeur dans la liste déroulante Délai d'attente de la conversion pour changer le délai d'attente par défaut de la conversion (30 secondes).

    • Cliquez sur une valeur dans la liste déroulante Nombre de threads pour changer le nombre de threads de conversion de documents par défaut (2).

    • Cliquez sur une valeur dans la liste déroulante Taille de la file d'attente pour changer la taille de la file d'attente par défaut (50).

  3. Cliquez sur le bouton Mettre à jour.

 

Note :
Pour plus d'informations sur la configuration des paramètres DCS, reportez-vous à la rubrique Configuration du service de conversion de documents.
