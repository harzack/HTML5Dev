Pour resynchroniser un serveur d'administration :

  1. Dans l'onglet Infos spécifiques de la page Propriétés de chaque objet source de la base de données Content Server, vérifiez que les chemins d'accès des répertoires et les numéros de port sont corrects pour chaque objet.

  2. Sur la page Pour accéder au volume des objets système, cliquez sur l'icône Fonctions d'un serveur d'administration et choisissez Resynchroniser.

  3. Sur la page Resynchroniser, cliquez sur le bouton OK.

  4. Sur la page Resynchroniser le serveur d'administration, cliquez sur le lien Continuer.
