L'onglet Infos spécifiques de la page Propriétés du gestionnaire de sauvegarde affiche les informations suivantes :

Elément Description
Modèle de libellé Modèle utilisé pour associer un libellé à une sauvegarde. Ce libellé permet d'identifier la sauvegarde. La valeur par défaut est Source_Données %m%d%Y_%T%r, où Source_Données représente une chaîne identifiant la source de données, %m le mois, %d le jour, %Y l'année, %T le type de sauvegarde effectué et %r l'ID de partition. Pour obtenir la liste complète des variables permettant la mise en forme d'un modèle de libellé, reportez-vous à la rubrique Variables Date et Heure.
Opération de sauvegarde (UNIX uniquement) Spécifie si les processus de sauvegarde sauvegardent les index en copiant les fichiers vers un répertoire de destination ou en exécutant un script spécifié. L'option de script est disponible uniquement sur les systèmes d'exploitation UNIX. Pour plus d'informations sur l'écriture de ce script, contactez le service d'assistance clientèle d'OpenText.
Répertoire de destination Chemin absolu du répertoire dans lequel les processus de sauvegarde stockent les sauvegardes pour la source de données. Chaque partition a son propre répertoire de destination, indiquant l'emplacement de stockage de la sauvegarde de cette partition.
