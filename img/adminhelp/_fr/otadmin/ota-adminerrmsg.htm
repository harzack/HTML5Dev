Le tableau ci-dessous décrit les erreurs relatives au processus système du serveur d'administration qui peuvent apparaître lorsque ce dernier ne peut pas invoquer de processus de flux de données.

Si ces erreurs persistent, contactez le service d'assistance clientèle d'OpenText pour obtenir plus d'informations.

Tableau 1-23 : Messages d'erreur du processus système du serveur d'administration

Numéro Message Description Solution
1000 Les paramètres système spécifiés ne sont pas valables Le serveur d'administration n'a pas pu démarrer le processus car certains de ses paramètres système sont incorrects. Examinez le contenu du fichier admserv.log pour obtenir des informations détaillées sur l'opération effectuée par le serveur d'administration. Le fichier admserv.log est stocké dans le répertoire des journaux de votre installation Content Server.
1001 Impossible d'allouer la mémoire Le serveur d'administration n'a pas pu allouer suffisamment de mémoire pour démarrer le processus. Examinez le contenu du fichier admserv.log pour obtenir des informations détaillées sur l'opération effectuée par le serveur d'administration. Le fichier admserv.log est stocké dans le répertoire des journaux de votre installation Content Server.
