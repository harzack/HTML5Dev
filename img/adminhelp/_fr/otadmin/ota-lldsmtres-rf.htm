Content Server affiche la page Résultats de maintenance de sources de données lorsque vous effectuez certaines tâches de maintenance sur un dossier source de données (par exemple l'élimination et la réextraction de données de la source de données). La page Résultats de maintenance de sources de données enregistre le statut de ces opérations de maintenance et identifie les actions que Content Server a effectuées pour accomplir ces opérations.

Les liens figurant sur cette page renvoient à l'aide disponible pour les tâches que vous pouvez effectuer dans Content Server à ce stade. Cliquez sur l'un des liens suivants pour continuer :

  • Pour réindexer le contenu Entreprise

  • Pour réindexer le contenu des index non-Entreprise

  • Pour vérifier le contenu de l'index Entreprise

  • Sauvegarde et restauration des index

 

Note :
Les résultats du processus de vérification d'index sont affichés sur la page Pour vérifier le contenu de l'index Entreprise.

 

Cliquez sur le bouton Continuer pour revenir à la page Administration système.
