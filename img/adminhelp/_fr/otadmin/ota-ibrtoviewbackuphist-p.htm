Pour afficher l'historique des sauvegardes d'une source de données :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier source de données dont vous souhaitez afficher l'historique.

  2. Cliquez sur l'icône Fonctions du gestionnaire de sauvegarde, choisissez Propriétés, puis Historique.
