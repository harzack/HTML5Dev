Pour modifier un gestionnaire de sauvegarde :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données du dossier source de données dont vous souhaitez modifier le gestionnaire de sauvegarde.

  2. Cliquez sur l'icône Fonctions du gestionnaire de sauvegarde, choisissez Propriétés, puis Infos spécifiques.

  3. Dans l'onglet Infos spécifiques de la page Propriétés du gestionnaire de sauvegarde, modifiez les paramètres du gestionnaire de sauvegarde.

  4. Cliquez sur le bouton Mettre à jour.
