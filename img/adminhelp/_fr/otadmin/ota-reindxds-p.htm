Pour réindexer le contenu des index non Entreprise :

  1. Vous devez vous assurer que tous les processus du flux de données sont programmés ou en cours d'exécution.

  2. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du dossier source de données contenant les données à réindexer, puis sélectionnez Maintenance.

  3. Sur la page Maintenance de sources de données, activez l'une des cases d'option suivantes :

    • Extraire à nouveau les données à partir de la source : cette option permet d'extraire à nouveau les données à partir de la source sans éliminer l'index existant.

    • Eliminer le flux de données, reconstruire l'index, puis extraire les données de la source : cette option permet de supprimer le contenu de l'index avant d'extraire à nouveau les données à partir de la source.

  4. Cliquez sur le bouton OK.

  5. Sur la page Résultats de maintenance de sources de données, cliquez sur le bouton Continuer.
