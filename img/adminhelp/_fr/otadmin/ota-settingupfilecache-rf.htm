La mémoire cache de fichier du serveur d'administration est un dossier de stockage géré par un serveur d'administration. La mémoire cache de fichier s'appelle également cache admin distant (RAC) ; avec la mémoire cache locale du fichier du serveur, elle forme un système de mémoire cache de fichier qui permet d'améliorer l'efficacité de certaines opérations Content Server, telles que la mise en évidence des résultats. La mise en évidence des résultats est la fonction de Content Server permettant de mettre en évidence les termes de requête dans le contenu des résultats de la recherche.

La configuration d'une mémoire cache de fichier du serveur d'administration passe par l'activation, l'élimination ou l'affichage des informations la concernant (par exemple, le nombre d'éléments contenus dans la mémoire cache de fichier). Lorsque vous installez ou mettez à niveau Content Server, une mémoire cache de fichier du serveur d'administration est créée et activée sur l'ordinateur sur lequel se trouve le serveur d'administration par défaut.

Afin de configurer une mémoire cache de fichier du serveur d'administration pour un autre serveur d'administration de votre installation Content Server, vous devez créer le répertoire de la mémoire cache de fichier. Vous pouvez ensuite activer la mémoire cache de fichier en spécifiant son répertoire. Si vous activez plusieurs mémoires cache de fichier du serveur d'administration dans votre installation Content Server, la mise en mémoire cache de fichier peut continuer si l'un des serveurs d'administration devient indisponible. Vous pouvez à tout moment désactiver une mémoire cache de fichier du serveur d'administration dans votre installation Content Server.

 

Note :
Si vous choisissez de désactiver la mémoire cache de fichier du serveur d'administration, la commande Rechercher similarités et la mise en évidence des résultats ne fonctionnent plus sur les listes de documents.

 

Lorsque vous redémarrez un serveur d'administration qui gère une mémoire cache de fichier du serveur d'administration, le contenu de la mémoire cache de fichier est conservé. Vous pouvez supprimer le contenu de la mémoire cache de fichier du serveur d'administration en l'éliminant. Pour éliminer une mémoire cache de fichier du serveur d'administration, vous devez définir l'un des paramètres de traitement suivants :

  • Eliminer : le contenu d'une mémoire cache de fichier du serveur d'administration est automatiquement supprimé, en fonction des critères que vous avez définis (par exemple, tous les documents sont supprimés lorsqu'ils occupent plus d'espace de mémoire cache de fichier qu'une quantité définie).

  • Notifier : vous recevez un e-mail lorsque les critères que vous avez définis pour l'élimination sont satisfaits.

  • Eliminer et notifier : le contenu de la mémoire cache de fichier du serveur d'administration est automatiquement supprimé, en fonction des critères que vous avez définis. Vous recevez un e-mail vous indiquant que le contenu de la mémoire cache de fichier a été supprimé.
