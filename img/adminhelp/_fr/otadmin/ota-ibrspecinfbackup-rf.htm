Onglet Infos spécifiques de la page Propriétés d'un processus de sauvegarde

Elément Description
Etat Indique si le processus de sauvegarde est en cours d'exécution. Si vous voulez modifier les paramètres de la sauvegarde alors que le processus est en cours, cliquez sur le bouton Arrêter.
Activer la gestion du système Détermine si Content Server contrôle ce processus pour détecter le retour de messages d'erreur. S'il retourne un message d'erreur, Content Server l'enregistre dans la base de données. La gestion du système est activée par défaut. Pour plus d'informations sur la configuration de Content Server en vue de l'envoi d'e-mails d'alerte lorsqu'un processus de flux de données rencontre des erreurs, reportez-vous à la rubrique Pour activer la recherche d'erreurs et l'envoi d'e-mails.
Partition Vous permet d'indiquer la partition que vous voulez sauvegarder au cours du processus de sauvegarde. Si vous n'avez qu'une seule partition, celle-ci apparaît par défaut.
Libellé de sauvegarde Modèle utilisé pour associer un libellé à une sauvegarde. Ce libellé permet d'identifier la sauvegarde. La valeur par défaut est Source_Données %m%d%Y_%T%r, où Source_Données représente une chaîne identifiant la source de données, %m le mois, %d le jour, %Y l'année, %T le type de sauvegarde effectué et %r l'ID de partition. Pour obtenir la liste complète des variables permettant la mise en forme d'un modèle de libellé, reportez-vous à la rubrique Variables Date et Heure.
Type de sauvegarde Spécifie si vous voulez une sauvegarde Complète (nouvelle sauvegarde de l'intégralité de l'index) ou une sauvegarde Différentielle (sauvegarde des modifications apportées à l'index depuis la dernière sauvegarde uniquement).
Après message d'erreur Spécifie le niveau de messages d'erreurs désiré.
Options de démarrage Permet de spécifier les options de démarrage du processus de sauvegarde :
•

A cette heure : cette option permet de programmer un processus de sauvegarde pour qu'il s'exécute à une heure précise certains jours de la semaine.

•

Chaque : cette option permet de programmer un processus de sauvegarde pour qu'il s'exécute à un intervalle régulier.

Pour plus d'informations sur les processus de flux de données, reportez-vous à la rubrique Configuration des options de lancement d'un processus de flux de données.
Démarrer/Arrêter Permet de démarrer ou d'arrêter le processus de sauvegarde, selon son statut actuel. Si le processus n'est pas en cours d'exécution, le bouton Démarrer s'affiche. Si le processus est en cours d'exécution, le bouton Arrêter s'affiche. Pour plus d'informations sur le démarrage et l'arrêt des processus de flux de données, reportez-vous à la rubrique Maintenance des flux de données.
Resynchroniser Permet de resynchroniser les informations relatives au processus de sauvegarde (dans le fichier otadmin.cfg du serveur d'administration du processus) avec les informations présentes dans la base de données Content Server. Pour plus d'informations sur la resynchronisation des processus de flux de données, reportez-vous à la rubrique Pour resynchroniser les processus de flux de données.
Mettre à jour Permet de soumettre au serveur les modifications que vous avez apportées à cette page.
Réinitialiser Rétablit les informations de cette page à l'état qui était le leur à l'ouverture de la page.
