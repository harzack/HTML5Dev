Content Server affiche la page Resynchroniser le serveur d'administration lorsque vous confirmez que vous souhaitez resynchroniser un serveur d'administration spécifique. Pour plus d'informations sur les resynchronisations, reportez-vous à la rubrique Resynchronisation des serveurs d'administration.

La page Resynchroniser le serveur d'administration commence par afficher un message indiquant qu'une resynchronisation est en cours. La page est actualisée très fréquemment, indiquant les objets système qui ont été synchronisés par Content Server jusque-là. Pour chaque objet système, la page affiche l'action éventuelle effectuée par Content Server pour resynchroniser l'objet. Une fois la resynchronisation terminée, la page Resynchroniser le serveur d'administration affiche un message de confirmation et un lien Continuer.

Si la resynchronisation ne peut pas s'effectuer correctement, vous pouvez consulter le fichier journal affiché par Content Server sur la page Resynchroniser le serveur d'administration, puis corriger les erreurs avant de resynchroniser à nouveau le serveur d'administration.

Pour revenir à l'onglet Général de la page Propriétés du serveur d'administration une fois la resynchronisation terminée, cliquez sur le lien Continuer.
