Pour vérifier le contenu de l'index Entreprise :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur Ouvrir le volume des objets système.

  2. Sur la page Volume des objets système, cliquez sur l'icône Fonctions du dossier source de données Entreprise, puis sélectionnez Maintenance.

  3. Sur la page Maintenance des sources de données, cliquez sur le bouton radio Configurer la vérification de l'index de recherche.

  4. Cliquez sur le bouton OK.

  5. Sur la page Rechercher le fichier journal de vérification d'index, procédez de l'une des façons suivantes, en fonction du statut de votre système :

    • Cliquez sur le bouton Réexécuter la vérification pour réeffectuer la vérification d'index.

    • Si vous cliquez sur le bouton Réextraire les objets manquants, les objets manquants dans l'index sont exportés de nouveau.

    • Cliquez sur le bouton OK pour revenir à la page Volume des objets système.

 

Note :
Le statut de la vérification d'index apparaît sur la page Rechercher le fichier journal de vérification d'index une fois la vérification terminée.
