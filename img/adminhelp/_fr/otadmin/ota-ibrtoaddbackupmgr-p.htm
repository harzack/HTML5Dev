Pour ajouter un gestionnaire de sauvegarde :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données d'un dossier source de données.

  2. Dans le menu Ajouter un élément, cliquez sur Gestionnaire de sauvegarde.

  3. Sur la page Ajouter Gestionnaire de sauvegarde, saisissez un nom dans le champ Nom.

  4. Dans l'onglet Général de la page Propriétés, vous pouvez ajouter une description du gestionnaire de sauvegarde dans le champ Description.

  5. Si la liste déroulante Opération de sauvegarde est disponible (systèmes UNIX uniquement), cliquez sur l'une des opérations de sauvegarde suivantes :

    • Copier les fichiers d'index vers leur destination : cette option permet de copier les fichiers d'index dans le répertoire spécifié dans le champ Répertoire de destination ;

    • Exécuter le script de sauvegarde spécifié : cette option permet d'exécuter un script de sauvegarde (systèmes UNIX uniquement).

  6. Pour changer le modèle de libellé par défaut, modifiez le texte dans le champ Modèle de libellé.

     

    Note :
    La valeur par défaut est Source_Données %m%d%Y_%T%r, où Source_Données représente une chaîne identifiant la source de données, %m le mois, %d le jour, %Y l'année, %T le type de sauvegarde effectué et %r l'ID de partition. Pour obtenir la liste complète des variables permettant la mise en forme d'un modèle de libellé, reportez-vous à la rubrique Variables Date et Heure.

     

  7. Dans le champ Répertoire de destination de chaque partition que vous avez créée, saisissez le chemin absolu du répertoire dans lequel vous voulez que Content Server sauvegarde l'index.

  8. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  9. Cliquez sur le bouton Ajouter.
