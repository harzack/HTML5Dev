Pour sauvegarder un index à l'aide de l'Assistant de sauvegarde :

  1. Sur la page Volume des objets système, cliquez sur l'icône Fonctions de la source de données dont vous voulez sauvegarder l'index, puis sélectionnez Maintenance.

  2. Sur la page Maintenance de sources de données, cliquez sur le bouton radio Sauvegarder l'index, puis cliquez sur le bouton OK. Si vous n'avez pas créé de gestionnaire de sauvegarde auparavant, vous êtes invité à en ajouter un maintenant. Sur la page Confirmation de la création d'un gestionnaire de sauvegarde, cliquez sur le bouton OK. Sur la page Ajouter Gestionnaire de sauvegarde, saisissez les informations requises, puis cliquez sur le bouton Ajouter.

  3. Sur la page Sauvegarde de sources de données, cliquez sur le bouton radio Démarrer l'Assistant de sauvegarde dans la section Options, puis cliquez sur le bouton OK.

  4. Sur la page Assistant de sauvegarde de sources de données, à l'étape Partitions, activez les cases à cocher correspondant aux partitions à sauvegarder, puis cliquez sur le bouton Suivant.

  5. A l'étape Niveau d'alerte, cliquez sur l'un des niveaux de message d'erreur suivants dans la liste déroulante Niveau d'alerte :

    • Aucun, qui est l'option par défaut ;

    • Informations, qui permet de notifier au destinataire le statut général et d'autres informations système ;

    • Avertissement, qui permet de notifier au destinataire les erreurs non critiques dans le système ;

    • Erreur, qui permet de notifier au destinataire les erreurs système générales ;

    • Erreur grave, qui permet de notifier au destinataire les erreurs système critiques.

     

    Note :
    Pour plus d'informations sur la transmission des messages d'erreur, reportez-vous à la rubrique Pour activer la recherche d'erreurs et l'envoi d'e-mails.

     

  6. Cliquez sur le bouton Suivant.

  7. A l'étape Type de sauvegarde, activez l'une des cases d'option suivantes :

    • Complet

    • Différentiel

     

    Note :
    Si la source de données ne dispose pas d'un gestionnaire de sauvegarde, vous pouvez uniquement effectuer une sauvegarde complète. Lorsque le bouton radio Différentiel n'est pas disponible, cela signifie que vous n'avez pas encore créé de sauvegarde complète.

     

  8. Cliquez sur le bouton Suivant.

  9. A l'étape Destination, modifiez le chemin absolu du répertoire pour la sauvegarde dans le champ Destination ou acceptez la valeur par défaut, puis cliquez sur le bouton Suivant.

  10. A l'étape Libellé, saisissez un identificateur pour la sauvegarde dans le champ Libellé ou acceptez la valeur par défaut, puis cliquez sur le bouton Suivant.

  11. A l'étape Résumé, passez en revue les options de l'assistant. Procédez de l'une des manières suivantes :

    • Pour continuer avec ces options, cliquez sur le bouton Sauvegarde.

    • Pour changer l'une de ces options, cliquez sur le bouton Retour.

    • Pour quitter l'assistant sans sauvegarder l'index, cliquez sur le bouton Quitter l'assistant.

  12. Cliquez sur le bouton Vérifier le statut pour actualiser la progression du processus. Si vous avez activé la transmission de messages d'erreur pour l'Assistant de sauvegarde, Content Server envoie un message d'alerte une fois la sauvegarde terminée.

     

    Note :
    Si vous avez sélectionné plusieurs partitions à sauvegarder, vous serez invité à revenir à l'étape 7 afin de terminer la sauvegarde pour les autres partitions spécifiées.

     

  13. Cliquez sur le bouton OK pour quitter l'Assistant de sauvegarde.
