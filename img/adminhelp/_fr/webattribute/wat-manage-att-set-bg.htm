Les attributs de nœud supplémentaires s'appliquent à tous les éléments de Content Server. Sur la page Gérer les attributs de nœud supplémentaires, vous pouvez ajouter, afficher, modifier ou supprimer un attribut de nœud supplémentaire.

Une fois que des attributs de nœud supplémentaires ont été créés, vous devez définir leur utilisation dans Content Server.

Après avoir ajouté, modifié ou supprimé des attributs système sur cette page, vous devez redémarrer le serveur et le serveur d'application Web afin que les ajouts ou les modifications soient pris en compte. Pour plus d'informations sur le redémarrage des serveurs, reportez-vous à la rubrique Arrêt et démarrage des serveurs.

La page Configurer les spécifications de valeur d'attribut vous permet de spécifier que, pour certains types d'objets, les utilisateurs n'ont pas à définir de valeurs, et ce même pour des attributs requis définis sur la page Gérer les attributs de nœud supplémentaires.

Cette fonction permet de réutiliser des attributs tout en définissant des comportements différents pour chaque type d'élément.
