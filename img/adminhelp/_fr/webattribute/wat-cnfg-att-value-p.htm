Pour configurer les spécifications de valeur d'attribut :

  1. Cliquez sur le lien Configurer les spécifications de valeur d'attribut de la section Administration système sur la page Administration.

  2. Pour chaque type d'élément, cochez l'option Obligatoire pour que les utilisateurs puissent définir une valeur pour les attributs obligatoires.

  3. Cliquez sur Envoyer.
