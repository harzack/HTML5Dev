Pour définir des paramètres d'attribut :

  1. Cliquez sur le lien Gérer les attributs d'objet supplémentaires de la section Administration système sur la page Administration.

  2. Pour chaque attribut, cliquez sur l'un des boutons radio suivants :

    • Exclure, auquel cas l'attribut n'apparaît pas pour les éléments Content Server

    • Inclure, auquel cas l'attribut apparaît pour tous les éléments Content Server

    • Demander, auquel cas l'attribut apparaît pour tous les éléments Content Server et doit avoir une valeur valide pour chacun d'entre eux

  3. Cliquez sur Enregistrer les modifications.
