Un attribut est un élément d'information au sujet d'un élément. Par défaut, les attributs définis sur la page Gérer les attributs de nœud supplémentaires s'appliquent à tous les éléments de Content Server.

En affectant des attributs de nœud supplémentaires aux éléments, vous aidez les utilisateurs à les rechercher dans la base de données Content Server au moyen d'un ou plusieurs attributs spécifiques et de la ou des valeurs qui leur ont été affectées.

Lorsque vous définissez un attribut de type Inclure ou Obligatoire, les éléments ajoutés à Content Server par les utilisateurs adoptent cet attribut. Les utilisateurs peuvent définir les valeurs d'attribut de tout élément pour lequel ils possèdent une autorisation Modifier les attributs.

OpenText vous recommande de créer, supprimer ou modifier des attributs de nœud supplémentaires pendant que les éventuels utilisateurs de la base de données Content Server concernée sont peu nombreux. Après avoir modifié les attributs d'objet supplémentaires, vous devez redémarrer les serveurs et le serveur d'application Web. Pour plus d'informations sur le redémarrage des serveurs, reportez-vous à la rubrique Arrêt et démarrage des serveurs.

Attributs de nœud standard

Chaque type d'élément Content Server possède un ensemble par défaut d'attributs qui n'est pas nécessairement identique pour chacun d'entre eux. Les attributs affichés pour un élément sur la page Propriétés correspondent à ses attributs par défaut. Par exemple, le type d'élément Document possède l'ensemble par défaut d'attributs suivant :

  • Nom

  • Description

  • Date de dernière modification

  • Propriétaire

  • Date de création

  • Réservé

  • Réservé(e) par

  • Réservé(e) le

  • Numéro de la version

  • Version

  • Propriétaire de la version

  • Version modifiée

  • Nom du fichier

  • Taille du fichier

  • Type MIME

  • Type de fichier
