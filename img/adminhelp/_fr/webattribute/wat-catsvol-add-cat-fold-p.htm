Note :
Vous pouvez enregistrer un dossier de catégorie uniquement dans le volume des catégories ou dans un autre dossier de catégorie.

 

Pour ajouter un dossier de catégorie :

  1. Choisissez Dossier de catégorie dans le menu Ajouter un nouvel élément.

  2. Saisissez un nom pour le dossier de catégorie dans le champ Nom.

  3. Vous pouvez également saisir une description du dossier Catégorie dans le champ Description.

  4. Pour modifier les catégories ou attributs associés à la catégorie, cliquez sur le bouton Modifier.

  5. Pour placer l'élément dans un emplacement autre que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server..., naviguez jusqu'au recueil dans lequel vous voulez rechercher la catégorie, puis cliquez sur le lien Sélectionner.

  6. Cliquez sur le bouton Ajouter un élément.
