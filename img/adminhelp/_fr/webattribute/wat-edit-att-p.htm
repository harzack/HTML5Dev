Pour modifier un attribut :

  1. Cliquez sur le lien Gérer les attributs d'objet supplémentaires de la section Administration système sur la page Administration.

  2. Cliquez sur le lien Modifier de l'attribut à modifier.

  3. Vous pouvez également cliquer sur un nouveau type d'affichage d'interface (s'il en existe plusieurs) dans la liste Type.

  4. Vous pouvez également saisir un nouveau nom dans le champ Libellé d'affichage.

  5. Pour un type d'attribut Menu contextuel, cliquez sur le bouton Modifier, saisissez les nouvelles valeurs dans le champ Valeurs valables et cliquez sur Continuer la modification de l'attribut.

  6. Cliquez sur Enregistrer les modifications.
