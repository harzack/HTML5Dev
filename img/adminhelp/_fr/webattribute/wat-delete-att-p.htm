Pour supprimer un attribut de nœud :

  1. Cliquez sur le lien Gérer les attributs d'objet supplémentaires de la section Administration système sur la page Administration.

  2. Cliquez sur le lien Modifier de l'attribut à supprimer.

  3. Cliquez sur le bouton Supprimer l'attribut, puis sur OK dans la boîte de dialogue de confirmation.
