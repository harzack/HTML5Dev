Pour créer un attribut de nœud supplémentaire :

  1. Cliquez sur le lien Gérer les attributs d'objet supplémentaires de la section Administration système sur la page Administration.

  2. Sur la page Gérer les attributs de nœud supplémentaires, cliquez sur le lien Ajouter un nouvel attribut.

  3. Sur la page Ajouter un nouvel attribut, saisissez un nom unique pour l'attribut dans le champ Nom.

  4. Cliquez sur l'un des types d'attributs dans la liste Type.

  5. Pour Texte : Champ, saisissez le nombre maximum de caractères (jusqu'à 254) que les utilisateurs pourront saisir pour l'attribut dans le champ Longueur du texte.

  6. Dans le champ Libellé d'affichage, saisissez le nom de l'attribut tel qu'il apparaîtra dans l'interface utilisateur.

  7. Pour un type d'attribut Menu contextuel, cliquez sur le bouton Modifier, saisissez chaque valeur du menu contextuel sur une ligne séparée et cliquez sur Continuer la modification de l'attribut.

  8. Pour un attribut de type Zone de liste déroulante, cliquez sur la valeur par défaut requise dans la liste déroulante.

  9. Cliquez sur le bouton Créer un attribut. Content Server vérifie que la valeur par défaut spécifiée est correcte. Si vous recevez un message d'erreur, cliquez sur le bouton Précédent de votre navigateur Web pour retourner à la page Ajouter un nouvel attribut et spécifiez une valeur correcte.
