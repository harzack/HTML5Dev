Pour ajouter une catégorie :

  1. Sélectionnez Catégorie dans le menu Ajouter un élément.

  2. Sur la page Ajouter : Catégorie, saisissez un nom pour la catégorie dans le champ Nom.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les noms dans les autres langues activées.

  3. Dans le champ Description, vous pouvez éventuellement saisir une description du rapport LiveReport.

    Si plusieurs langues sont installées et activées sur votre système, cliquez sur le bouton Cliquez pour modifier les valeurs multilingues pour modifier les descriptions dans les autres langues activées.

  4. Dans le champ Catégories, vous pouvez éventuellement cliquer sur Modifier pour sélectionner ou ajouter une catégorie à appliquer à cette catégorie.

  5. Pour placer l'élément dans un emplacement autre que celui qui apparaît dans le champ Créer dans, cliquez sur le bouton Parcourir Content Server..., naviguez jusqu'au recueil dans lequel vous voulez rechercher la catégorie, puis cliquez sur le lien Sélectionner.

  6. Cliquez sur Ajouter.

 

Note :
Après avoir créé une catégorie, vous devez lui ajouter des attributs. Pour en savoir plus, voir Gestion des attributs de nœud.
