Content Server permet de définir de solides catégories ayant, chacune, un certain nombre d'attributs à une ou plusieurs valeurs. Une catégorie est un jeu d'attributs permettant de classer, grouper ou rechercher les éléments qu'il définit. Vous pouvez configurer une catégorie en choisissant les types d'attributs qu'elle doit contenir, leur ordre et leur disposition, ainsi que les valeurs par défaut et autorisées de chaque attribut.

En attribuant des catégories aux éléments de Content Server, vous aidez les utilisateurs à rechercher des éléments dans la base de données en précisant les éléments d'une catégorie particulière ou ceux dont les attributs ont une valeur spécifique. Les utilisateurs peuvent associer des catégories aux éléments qu'ils ajoutent et définir leurs attributs s'ils disposent des autorisations Voir et Voir le contenu pour la catégorie concernée. Par défaut, toutes les catégories que vous gérez sont accessibles à tous les utilisateurs.

Un élément Content Server associé à une catégorie adopte les attributs de la catégorie en question, en plus du jeu d'attributs par défaut associé à l'élément. Lorsque vous créez une catégorie, vous devez lui attribuer au moins un attribut.

 

Note :
Lorsque vous effectuez une mise à niveau de Content Server, les catégories et attributs de la version antérieure ne deviennent fonctionnels que lorsque vous réexportez les données Entreprise. Cette fonctionnalité extrait une nouvelle fois les données de la base de données Content Server sans supprimer l'index Entreprise.
