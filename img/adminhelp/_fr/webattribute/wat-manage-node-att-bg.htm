Une fois les attributs de cette page ajoutés, modifiés ou supprimés, vous devez redémarrer le serveur et le serveur d'application Web afin que les modifications soient prises en compte. Pour plus d'informations sur le redémarrage des serveurs, reportez-vous à la rubrique Arrêt et démarrage des serveurs.

Le nom d'attribut du champ Nom est utilisé comme nom de colonne dans la table de la base de données Content Server DTree. Pour être valide :

  • Un nom d'attribut doit contenir uniquement des caractères alphabétiques et des traits de soulignement.

  • Si le type de données d'attribut sélectionné est Date, respectez le format d'entrée des dates de Content Server en utilisant quatre chiffres pour l'année.

  • Les noms d'attributs ne respectent pas la casse. Par exemple, le nom test1 équivaut à Test1.

  • Les noms d'attributs ne doivent pas contenir de mots-clés (comme Longueur, Taille, Table ou Date), réservés à l'usage de votre SGBDR.

Pour obtenir la liste complète des mots clés réservés, reportez-vous à la documentation de votre logiciel SGBDR. Le nom d'un attribut supprimé ne peut pas être réutilisé.

Une fois l'attribut créé, vous ne pouvez pas modifier la longueur du texte ou le type de données. En revanche, vous pouvez modifier le type d'affichage de l'interface. S'il s'avère nécessaire de modifier un de ces paramètres, vous devez supprimer l'attribut puis le recréer sous un autre nom.
