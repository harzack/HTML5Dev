Il est inutile d'activer les notifications Content Server pour que la planification des WebReports fonctionne. Cependant, vous devez activer l'option Content Server Agents de modification sur votre instance Content Server. La section [options] du fichier opentext.ini d'au moins une instance de Content Server contient EnableAgents=TRUE si les Agents de modification sont activés. Normalement, les Agents de modification sont activés, mais si vous ne savez pas comment configurer les Agents de modification de votre propre instance Content Server, consultez l'assistance d'OpenText.
