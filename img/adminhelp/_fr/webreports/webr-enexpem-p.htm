Pour activer l'exportation vers un e-mail :

  1. Sur la page Administration Content Server, dans la section Administration des notifications, cliquez sur Configurer la notification.

  2. Sur la page Configurer la notification, dans la zone Paramètres SMTP :

    1. Dans le champ ID du serveur SMTP, saisissez le nom du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est mail.

    2. Dans le champ Port SMTP, saisissez le port de réception du serveur SMTP. Le paramètre par défaut de la plupart des serveurs SMTP est 25.

    3. Dans le champ Content Server Nom de l'hôte, saisissez le nom DNS complet de l'hôte principal Content Server. Par exemple, « contentserverhost.mycorp.com ».

     

    Note :
    Bien que ces détails doivent être renseignés, il est inutile d'activer les notifications Content Server.

     

  3. Cliquez sur Envoyer.
