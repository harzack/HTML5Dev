Pour gérer les programmes :

  1. A partir de la page Administration Content Server, sous Administration WebReports, cliquez sur Gérer les programmes WebReports.

  2. Chaque ligne du rapport représente un programme défini par un utilisateur pour un WebReport. Pour chaque programme, vous pouvez :

    1. Facultatif Désactiver le programme en désélectionnant sa case à cocher Activé. Les informations de planification sont conservées, mais le rapport n'est plus exécuté jusqu'à ce que l'utilisateur répertorié ou l'administrateur le réactive.

    2. Facultatif Supprimer le programme en sélectionnant sa case à cocher Supprimer le programme.

       

      Note :
      Content Server supprime automatiquement les programmes si l'utilisateur qui les a définis est supprimé ou si le WebReport lui-même est supprimé.

       

  3. Cliquez sur Appliquer.
