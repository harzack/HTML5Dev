Pour stocker des WebReports dans le volume Rapports :

  1. Lorsque vous enregistrez votre WebReport, au cours d'une opération d'ajout, de déplacement ou de copie : dans le champ Créer dans, cliquez sur le lien sélectionné en regard de Content Server.

  2. Cliquez ensuite sur le lien sélectionné en regard de Rapports Content Server.

     

    Astuce :
    Pour voir les WebReports que vous avez enregistrés dans le volume Rapports, sélectionnez dans la barre d'outils Content Server le menu Personnel ou Projet. Ensuite, sélectionnez Rapports. Sur la page Mes rapports, sélectionnez l'onglet Rapports LiveReport.
