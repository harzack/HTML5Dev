La fonction Catégorie comme source de données permet la configuration, à l'aide de la souris, des rapports sur les catégories et attributs Content Server. Les paramètres d'administration de cette fonction permettent un contrôle de base du format de la sortie. En outre, certains paramètres facilitent la définition de limites pour le coût d'exécution des rapports, en termes de performances.

  • Nombre maximal de catégories détermine le nombre de catégories qu'un utilisateur peut sélectionner et sur lesquelles exécuter un rapport. Plus ce nombre est élevé, plus la requête envoyée à la base de données est complexe, lorsqu'un utilisateur sélectionne ce nombre maximal de catégories.

  • Nombre d'attributs à afficher sur la ligne détermine le nombre d'attributs à sélectionner présentés par ligne sur la page de configuration de la source de données des WebReports (pas dans le rapport proprement dit).

  • Délimiteur d'attribut à plusieurs valeurs spécifie le caractère utilisé pour séparer les valeurs multiples renvoyées par un rapport.

  • Nombre maximal de résultats définit une limite pour le nombre de résultats renvoyés par tous les rapports utilisant la fonction Catégorie comme source de données. Les résultats renvoyés au-delà de ce nombre sont ignorés.
