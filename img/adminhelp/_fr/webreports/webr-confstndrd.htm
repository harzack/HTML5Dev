Pour configurer une conversion :

  1. Configurez un moteur de conversion externe, par exemple Adlib eXpress, afin qu'il surveille un dossier d'entrée pour détecter tous les nouveaux fichiers nécessitant une conversion.

  2. Configurez le même moteur de conversion afin qu'il stocke les documents convertis dans un dossier de sortie spécifique.

  3. Configurez le moteur de conversion afin qu'il supprime le fichier source du dossier d'entrée.

  4. Configurez le moteur de conversion afin que l'extension du fichier de destination, par exemple .pdf, remplace celle du type source, par exemple .html, .doc ou .xls, au lieu d'ajouter cette nouvelle extension à l'ancienne.

  5. Sur la page Administration Content Server, sélectionnez Gérer la conversion WebReports.

  6. Définissez un intervalle de conversion dans le champ Intervalle de veille de l'agent de conversion. Ce paramètre déterminera en partie le temps nécessaire au fichier converti pour réapparaître dans Content Server. Le moteur de conversion doit vérifier le contenu du dossier d'entrée. La fréquence de vérification par défaut est de « 300 » secondes, soit 5 minutes.

  7. Dans le champ Répertoire d'entrée, spécifiez le chemin d'accès au dossier d'entrée.

  8. Dans le champ Répertoire de sortie, spécifiez le chemin d'accès au dossier de sortie.

     

    Important :
    Il est important que le Content Server dispose des autorisations nécessaires pour écrire un fichier dans le dossier d'entrée et supprimer un fichier dans le dossier de sortie.

     

  9. Facultatif Si vous souhaitez configurer la conversion par bon de travail Adlib eXpress, sélectionnez Cochez cette case.

    1. Dans le champ Répertoire d'entrée (lors de l'utilisation de bons de travail XML), saisissez le répertoire d'entrée à utiliser lorsque des fichiers sont convertis avec des bons de travail XML.

    2. Dans le champ Répertoire de sortie (lors de l'utilisation de bons de travail XML), saisissez le répertoire de sortie à utiliser lorsque des fichiers sont convertis avec des bons de travail XML.

    3. Facultatif Si vous souhaitez activer l'interface pour les bons de travail XML dans l'onglet Destination d'un WebReport, sélectionnez l'option Utiliser un moteur de conversion externe.

  10. Cliquez sur Appliquer.
