Pour configurer l'agent de planification :

  1. A partir des pages Administration Content Server, sous Administration WebReports, cliquez sur Gérer les programmes WebReports.

  2. Dans le champ Intervalle de veille de l'agent de planification, saisissez la valeur de l'intervalle de veille pour l'agent de planification des WebReports. Il s'agit de la durée d'attente de l'agent entre les exécutions.

  3. Cliquez sur Enregistrer les modifications.

  4. Redémarrez Content Server.

  5. Facultatif Si vous avez défini l'Intervalle de veille sur une valeur inférieure à 300, vous devez accéder à la page Propriétés du WebReport :

    1. Sélectionnez l'onglet Destination.

    2. Cliquez sur le bouton Saisissez les minutes (0-59).

    3. Dans le champ Minutes, saisissez le nombre de minutes. Par exemple, si vous avez saisi la valeur « 120 » dans le champ Intervalle de veille de l'agent de planification, saisissez « 2 » dans le champ Minutes pour que le WebReport s'exécute toutes les deux minutes.
