La fonction Déclencheur WR permet aux événements survenant dans Content Server de déclencher un WebReport. Les différents sous-types Content Server peuvent être activés ou désactivés au moyen des paramètres d'administration. Si vous activez la fonction pour un sous-type, l'option Déclencheur WR s'affiche sur les menus Fonctions de ce nœud. Si cette fonction est désactivée pour un sous-type particulier (non sélectionnée), aucun code associé à WebReports ne s'exécute pour les nœuds de ce sous-type et l'option Déclencheur WR ne s'affiche pas dans le menu Fonctions de ces nœuds. Si tous les sous-types sont désactivés, la fonction Déclencheur WR n'a aucun impact sur les performances. Lorsque WebReports est installé ou mis à niveau vers une nouvelle version, avec cette fonction, tous les sous-types sont désactivés par défaut.
