Le module WebReports est fourni avec différents exemples d'affichages de rapports. Une fois le module installé, ces affichages de rapports sont présents dans le dossier suivant de Content Server : <Content Server_home>\module\webreports_x_x_x\examplereportviews

Comme ces affichages de rapports peuvent être utiles pour tous les développeurs de WebReports, il convient souvent de les placer dans un emplacement standard, accessible aux communautés d'utilisateurs appropriées.

Par exemple, vous pouvez créer une zone de Content Server appelée « Développement de WebReports  » et contenant des exemples de WebReports. Vous pouvez générer ces WebReports à partir des affichages de rapports prédéfinis du module. Vous pouvez également télécharger les fichiers d'affichages de rapports dans Content Server, où les développeurs peuvent les récupérer.
