Pour mettre à niveau un module de version antérieure :

  1. Vérifiez tout d'abord que vous avez exécuté l'installateur de WebReports pour placer les modules dans le répertoire <Content Server_home>/staging.

  2. Ouvrez la page Administration Content Server dans votre navigateur Web et connectez-vous en tant qu'utilisateur administrateur Content Server. Ne tentez pas d'installer ce module, ou tout autre module contenant un schéma de base de données, si vous n'êtes pas connecté en tant qu'administrateur Content Server.

  3. Dans la section Administration du module, cliquez sur Mettre des modules à niveau.

  4. Sélectionnez la case à cocher en regard de Content Server WebReports Standard, puis cliquez sur Mettre à niveau.

  5. Lorsque vous y êtes invité, redémarrez le service Content Server, puis cliquez sur Continuer.
