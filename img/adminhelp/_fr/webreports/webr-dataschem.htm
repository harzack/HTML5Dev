Une table de la base de données Content Server appelée WEBREPORTS est créée lorsque WebReports est installé. Cette table a pour objet de stocker les informations de planification des WebReports. Elle comporte deux clés de colonnes : USERID et NODEID. NODEID est le nodeid du WebReport.

Une table supplémentaire, appelée WEBREPORTSTATS, est créée lorsque le module est installé ou mis à niveau.

 

Note :
Ces tables ne sont pas supprimées lorsque le module WebReports est désinstallé. Pour désinstaller ces tables, vous devez accéder à l'option Administration de la base de données WebReports dans les pages Administration Content Server avant de désinstaller le module. Pour plus d'informations sur l'Administration de la base de données WebReports, voir Administration d'une base de données WebReports et bridage vers des versions antérieures.
