Pour vérifier les mises à jour :

  1. Créez un nouveau WebReport à partir du menu Ajouter un élément de Content Server.

  2. Cliquez sur le menu Fonctions de WebReports et sélectionnez Modifier l'affichage du rapport.

  3. Cliquez sur le bouton Vérifier les mises à jour dans l'angle supérieur droit de la page.

  4. Vérifiez les informations affichées dans la fenêtre contextuelle, puis cliquez sur OK pour envoyer ces informations.

  5. Une nouvelle fenêtre de navigateur s'ouvre et affiche l'état de votre installation. Les résultats indiquent si une version plus récente est disponible ou si un correctif est présent pour votre système. Suivez toutes les indications affichées sur la page de résultats pour que votre installation WebReports reste à jour.
