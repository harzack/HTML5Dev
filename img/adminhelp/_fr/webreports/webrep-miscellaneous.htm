Cette page permet de définir les valeurs de différentes clés dans la section WebReports du fichier opentext.ini.

Actuellement, la seule clé pouvant être définie est EmailAddressMaxCharacters. Cette clé limite le nombre de caractères pouvant être utilisés pour la chaîne d'adresse e-mail d'un WebReport dont la destination est un e-mail.

Si cette valeur n'est pas définie sur un entier, la valeur de 10 000 caractères est utilisée par défaut.
