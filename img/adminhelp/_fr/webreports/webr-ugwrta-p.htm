Pour activer le déclenchement WR utilisateur/de groupe :

  1. Sur la page Administration Content Server, sous Administration WebReports, cliquez sur Déclenchement WR utilisateur/de groupe.

  2. Sur la page Déclenchement WR utilisateur/de groupe, sélectionnez la case à cocher Activer les déclencheurs WR de l'utilisateur/du groupe.

  3. Chaque ligne du tableau vous permet de définir un événement WR utilisateur/de groupe différent pour déclencher un WebReport :

    1. Sélectionnez l'utilisateur et/ou le groupe auxquels cet événement doit être appliqué. Procédez de l'une des manières suivantes :

      • Dans la colonne Utilisateur/Groupe, si vous souhaitez appliquer ce déclenchement à des utilisateurs ou groupes particuliers, utilisez le bouton associé pour sélectionner chaque utilisateur et/ou groupe.

      • Dans la colonne Global, si vous souhaitez appliquer ce déclenchement à tous les utilisateurs et groupes, sélectionnez la case à cocher.

    2. A partir de la colonne Evénement de déclenchement, sélectionnez l'événement qui déclenchera le WebReport. Si vous le souhaitez, vous pouvez sélectionner tous les événements de déclenchement disponibles.

    3. Dans la colonne Héritage, sélectionnez Membres directs ou Membres directs et indirects.

    4. Dans la colonne Action WebReport, utilisez le bouton Parcourir Content Server pour sélectionner le dossier dans lequel ce WebReport sera généré et le type d'action que WebReports effectuera.

    5. Facultatif La dernière colonne vous permet de supprimer la ligne que vous venez d'ajouter ou d'ajouter une nouvelle ligne au tableau afin de saisir un autre événement de déclenchement.
