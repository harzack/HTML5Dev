To configure Adlib eXpress job ticket conversion:

  1. Below the section to specify Input and Output directories, there is a checkbox to enable the use of XML Job Tickets with conversion.

  2. When you select this box and enable XML Job Tickets, the following will occur:

    • A second set of directories will appear to allow the user to specify an additional set of directories for when files are converted using XML Job Tickets.

    • The interface for XML Job Tickets will be enabled in the destination tab for a WebReports, provided the Use External Conversion Engine box is checked.
