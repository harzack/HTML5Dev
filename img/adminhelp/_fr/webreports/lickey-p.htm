Pour entrer la clé de licence :

  1. Sur la page Administration Content Server, sélectionnez Gérer la clé de licence WebReports.

  2. Facultatif Pour lancer un essai de 30 jours, cliquez sur le lien LANCER L'ESSAI.

  3. Entrez les détails de la clé de licence, exactement comme ils sont fournis, notamment les éléments suivants :

    • Nom de l'entreprise.

    • Date d'expiration. S'applique aux licences d'évaluation uniquement.

    • Nombre d'utilisateurs de WebReports. Notez que ce nombre peut être parfois différent du nombre d'utilisateurs généraux de Content Server.

    • Clé de licence

  4. Cliquez sur Enregistrer les modifications. Content Server répond afin de confirmer qu'une clé valide a été fournie.
