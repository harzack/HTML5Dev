Par défaut, l'agent de conversion WebReports est configuré pour une exécution toutes les 5 minutes, ou 300 secondes. Dans certaines situations, il peut être souhaitable que l'agent soit exécuté plus fréquemment. Dans ce cas, l'agent doit être configuré pour une exécution au maximum toutes les 60 secondes. Ce paramètre est configurable sur la page Gérer la conversion WebReports. La section supérieure de cette page contient un formulaire permettant de modifier l'intervalle de veille pour l'agent de conversion WebReports.
