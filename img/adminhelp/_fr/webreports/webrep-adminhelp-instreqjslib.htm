Utilisez la page Installer la bibliothèque Requests.js pour installer la bibliothèque requests.js, ainsi que les fichiers associés, à l'emplacement de la bibliothèque JavaScript : /img/webreports/library/

Lorsque la page est chargée, elle est prérenseignée avec des valeurs par défaut.

Si les champs Paramètre de préfixe d'URL et Chemin d'accès au répertoire de support (Image) sont laissés vides, l'installateur inclut du code JavaScript pour obtenir ces valeurs en utilisant les informations disponibles dans l'URL lors du chargement du fichier requests.js sur le client.

Si vous spécifiez le paramètre de préfixe d'URL, veillez à respecter la casse normalement saisie par les utilisateurs. Par exemple, si vous indiquez /CS/cs.exe, un utilisateur connecté avec /cs/cs.exe devra probablement être connecté à nouveau, ce qui cause des problèmes pour toute requête nécessitant une réponse. En cas de doute, laissez ce champ vide.

Cliquez sur Installer pour envoyer le formulaire et installer la bibliothèque requests.js. Cliquez sur Réinitialiser pour effacer toutes les modifications et restaurer les valeurs par défaut. Cliquez sur Annuler pour annuler et revenir à la page Administration principale.
