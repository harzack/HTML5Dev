La section Administration WebReports comporte les paramètres et options suivants :

  • Installer la bibliothèque Requests.js : fournit une interface permettant d'installer la bibliothèque de requêtes, et les fichiers associés, sur l'emplacement de la bibliothèque JavaScript. Pour en savoir plus, voir Installer la bibliothèque Requests.js.

  • Gérer la configuration de la source de données de la catégorie : mettez à jour le nombre maximal de catégories et de paramètres d'affichage d'attributs. Pour en savoir plus, voir Catégorie comme source de données.

  • Gérer l'intégration de la requête de recherche : gère la possibilité d'appeler WebReports directement à partir de l'écran de recherche. Pour en savoir plus, voir Gérer l'intégration de la requête de recherche.

  • Gérer les déclenchements WR utilisateur/de groupe : détermine l'utilisateur et/ou le groupe pouvant déclencher WebReports. Pour en savoir plus, voir Gestion du déclenchement WR utilisateur/de groupe.

  • Gérer la conversion WebReports : définissez les répertoires d'entrée et de sortie pour gérer la conversion PDF. Pour en savoir plus, voir Gérer la conversion WebReports.

  • Gérer les programmes WebReports : activez, désactivez ou supprimez définitivement des programmes distincts. Pour en savoir plus, voir WebReports Programmation.

  • Gérer l'écriture de scripts WebReports : activez ou désactivez l'écriture de scripts de WebReports distincts. Pour en savoir plus, voir Gérer l'écriture de scripts WebReports.

  • Gérer les services WebReports : activez, désactivez ou restreignez la fonctionnalité des services WebReports. Pour en savoir plus, voir Restriction des services WebReports.

  • Gérer les déclencheurs WR : déterminez les types de nœuds pouvant déclencher WebReports. Pour en savoir plus, voir Gestion du déclencheur WR.

  • Paramètres WebReports divers : définissez les valeurs de différentes clés dans la section WebReports du fichier opentext.ini. Pour en savoir plus, voir Paramètres WebReports divers.

  • Gestion des licences de WebReports : définissez ou modifiez la clé de licence WebReports et affichez l'état des licences. Pour en savoir plus, voir Fonctions de clé de licence.

  • Administration des nœuds WebReports : identifiez et mettez à jour les affichages de rapport utilisant une syntaxe obsolète. Pour en savoir plus, voir Gestion des nœuds.

  • Générateur de sous-balises WebReports : utilisé pour reconstituer toutes les sous-balises, y compris celles créées dans le dossier des sous-balises. Pour en savoir plus, voir Générateur de sous-balises WebReports.

  • Administration de la base de données WebReports : utilisé pour gérer toute action prise en charge sur la base de données WebReports. Pour en savoir plus, voir Administration d'une base de données WebReports et bridage vers des versions antérieures.
