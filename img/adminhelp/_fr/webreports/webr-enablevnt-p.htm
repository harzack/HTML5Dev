Pour activer les événements WebReports :

  1. A partir de la page Administration Content Server, dans la section Administration système, cliquez sur le lien Gérer l'audit des événements.

  2. Cliquez sur Définir les centres d'intérêt à inclure dans l'audit, puis sélectionnez WebReports dans la liste.

  3. Cochez les centres d'intérêt souhaités, puis cliquez sur Définir les centres d'intérêts.

  4. Par défaut, Les événements d'audit RUN et EXPORT ne sont pas activés pour les WebReports. Si vous souhaitez que ces événements d'audit se déclenchent pour un WebReport, procédez comme suit :

    1. Dans le menu Fonctions du WebReport, sélectionnez Propriétés.

    2. Sur la page Propriétés, sélectionnez l'onglet Spécifique.

    3. Sélectionnez Exécution/Exportation des audits activée.

    4. Cliquez sur Mettre à jour.
