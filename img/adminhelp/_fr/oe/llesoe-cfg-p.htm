Pour configurer l'exportateur d'objet :

  1. Créez un répertoire de travail, visible pour Content Server, où le ou les fichiers de sortie sont créés et enregistrés temporairement avant d'être déplacés une fois le processus terminé.

  2. Créez un répertoire de journaux où l'exportateur d'objet copiera les fichiers journaux. Assurez-vous que ce répertoire correspond au chemin d'accès <Content Server_home>\object_exporter .

  3. Créez un répertoire de destination, visible pour Content Server, vers lequel l'exportateur d'objet déplacera le ou les fichiers de sortie une fois le processus terminé.

  4. Dans la section Administration de l'exportateur d'objet de la page d'administration de Content Server, cliquez sur Configurer.

  5. Sur la page Configurer l'exportateur d'objet, dans la section Répertoire :

    1. Dans le champ Travail, indiquez le chemin complet du répertoire que vous avez créé à l'étape 1. C'est le répertoire où l'exportateur d'objet traitera l'exportation.

    2. Dans le champ Journal, indiquez le chemin complet du répertoire que vous avez créé à l'étape 2. C'est le répertoire dans lequel l'exportateur d'objet enregistrera les fichiers journaux et qui doit toujours se trouver dans le chemin d'accès <Content Server_home>\object_exporter .

    3. Dans le champ Destination, indiquez le chemin complet du répertoire que vous avez créé à l'étape 3. C'est le répertoire vers lequel l'exportateur d'objet déplacera le ou les fichiers de sortie une fois le processus terminé.

  6. Dans le champ Objets par lot de la section Avancé, indiquez le nombre d'objets à inclure dans chaque fichier de sortie.

  7. Cliquez sur Mettre à jour.

  8. Redémarrez les serveurs lorsque vous y serez invité.
