Pour exporter un objet :

  1. Dans la section Administration de l'exportateur d'objet de la page d'administration de Content Server, cliquez sur Exportation manuelle d'objet.

  2. Dans le champ Nom du fichier journal de la page Exportation manuelle d'objet, indiquez le nom du fichier journal, sans l'extension, qui sera utilisé pour les fichiers journaux et de sortie.

     

    Note :
    Une valeur spécifique à la date sera ajoutée au nom du fichier afin de le rendre unique.

     

  3. Dans le champ Exporter le fichier de contrôle, cliquez sur Parcourir... pour sélectionner un fichier de contrôle de l'exportation dans le système de fichiers.

  4. Cliquez sur Démarrer l'exportation pour lancer le processus d'exportation. Une fois l'exportation terminée, l'administrateur recevra une notification par e-mail.
