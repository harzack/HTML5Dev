Description

Par défaut, l'exportateur d'objet parcourt toute l'arborescence depuis l'emplacement de démarrage tel que défini dans la balise location . Pour éviter que l'exportateur d'objet parcoure toute l'arborescence, utilisez la balise <recurse>FALSE</recurse>.

Syntaxe

<recurse>FALSE</recurse>

Utilisation

  1. La balise recurse est facultative.

  2. La seule valeur valide pouvant être utilisée avec la balise recurse est FALSE.

Exemple

Dans cet exemple, seuls les objets non-recueil du dossier « Folder_001 », situé dans l'espace de travail Entreprise, seront exportés vers c:\docs

<export>
  <node>
    <location>Enterprise:Folder_001</location>
    <docstorage>c:\docs<docstorage>
    <recurse>FALSE</recurse>
  </node>
</export>
