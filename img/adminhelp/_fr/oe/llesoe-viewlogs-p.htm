Pour afficher un fichier journal :

  1. Dans la section Administration de l'exportateur d'objet de la page d'administration de Content Server, cliquez sur le lien Afficher les fichiers journaux.

  2. Sur la page Afficher les fichiers journaux d'exportation, cliquez sur le nom du fichier journal que vous voulez consulter.

Pour supprimer un fichier journal :

  1. Dans la section Administration de l'exportateur d'objet de la page d'administration de Content Server, cliquez sur le lien Afficher les fichiers journaux.

  2. Sur la page Afficher les fichiers journaux d'exportation, activez la case à cocher en regard du fichier journal à supprimer.

  3. Cliquez sur Supprimer, puis sur OK pour confirmer que vous souhaitez bien supprimer les fichiers journaux sélectionnés.
