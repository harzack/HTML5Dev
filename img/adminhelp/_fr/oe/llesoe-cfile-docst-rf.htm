Description

 

Important :
Cette balise est obligatoire, quel que soit le type d'objet.

 

Utilisez la balise docstorage pour spécifier vers quel emplacement du système de fichiers seront exportés les documents de Content Server.

Syntaxe

<docstorage>c:\docs</docstorage>
<docstorage version="current">c:\docs</docstorage>

Utilisation

  1. Cette balise est obligatoire, quel que soit le type d'objet.

  2. Le nom de chemin spécifié entre les balises <docstorage></docstorage> doit exister dans le système.

Notes

  1. Par défaut, toutes les versions d'un document seront exportées. Pour exporter uniquement la version en cours, ajoutez l'attribut version=current à la balise docstorage .

  2. Pour modifier le chemin d'accès à tous les documents dans le fichier de contrôle généré, ajoutez l'attribut pathprefix .

Exemple

Dans cet exemple, les documents du dossier « Folder_001 », situé dans l'espace de travail Entreprise, seront exportés vers c:\docs. Toutefois, le fichier de contrôle généré affichera c:\temp

<export>
  <node>
    <location>Enterprise:Folder_001</location>
    <docstorage pathprefix="c:\temp">c:\docs</docstorage>
  </node>
</export>
