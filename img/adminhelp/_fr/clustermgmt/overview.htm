OpenText Content Server Cluster Management déploie et gère des correctifs sur l'instance principale d'un cluster Content Server. Gestion des clusters fait partie des modules Content Server principaux, et vous administrez Gestion des clusters depuis les pages Administration de Content Server. Gestion des clusters permet d'effectuer les fonctions suivantes :

  • Détecter les dépendances avec d'autres correctifs et modules.

  • Enlever les correctifs obsolètes.

  • Vérifier les correctifs installés avant de les déployer sur tout le cluster.

Gestion des clusters dispose des composants suivants :

  • Historique d'audit du cluster

  • Gérer les agents de cluster

  • Gérer le cluster

  • Gérer les paramètres du cluster

  • Gérer le système maître
