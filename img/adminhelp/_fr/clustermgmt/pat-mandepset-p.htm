Pour gérer les paramètres du cluster :

  1. Dans la section Configuration du serveur de la page Administration Content Server, cliquez sur le lien Gestion des clusters.

  2. Sur la page Gestion des clusters, cliquez sur le lien Gérer les paramètres du cluster.

  3. Sur la page Gérer les paramètres du cluster, saisissez l'URL pour mettre à jour le manifeste dans Mettre l'URL à jour

  4. Cliquez sur Vérifier l'URL pour valider l'URL du manifeste.

  5. Saisissez l'emplacement de l'URL ou cliquez sur Parcourir pour configurer l'Hôte.

  6. Saisissez l'emplacement du répertoire pour le Dossier de mise en lots des correctifs.

     

    Astuce :
    Cliquez sur Réinitialiser pour configurer le Dossier de mise en lots des correctifs de l'emplacement du répertoire de mise en lots par défaut.

     

  7. Cliquez sur Enregistrer les paramètres.
