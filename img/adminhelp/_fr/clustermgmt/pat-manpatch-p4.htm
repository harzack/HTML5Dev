Pour installer les correctifs :

  1. Dans la section Configuration du serveur de la page Administration Content Server, cliquez sur le lien Gestion des clusters.

  2. Sur la page Gestion des clusters, cliquez sur le lien Gérer le système maître.

  3. Cliquez sur l'onglet Correctifs disponibles.

  4. Cliquez sur Actualiser pour mettre la liste à jour.

  5. Cliquez sur Télécharger pour obtenir le correctif depuis le OpenText Knowledge Center.

     

    Note :
    Après avoir téléchargé un correctif, il doit être placé dans le répertoire de mise en lots pour pouvoir être téléchargé vers la base de données.

     

  6. Cliquez sur l'onglet Correctifs mis en lots.

  7. Sélectionnez la case à cocher pour le correctif que vous souhaitez installer.

  8. Cliquez sur Déployer.
