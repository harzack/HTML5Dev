Pour afficher l'historique d'audit du cluster :

  1. Dans la section Configuration du serveur de la page Administration Content Server, cliquez sur le lien Gestion des clusters.

  2. Sur la page Gestion des clusters, cliquez sur le lien Historique d'audit du cluster.

  3. Sur la page Historique d'audit, sélectionnez une instance Content Server dans la liste Hôtes.

  4. Cliquez sur le lien Détails pour afficher les détails de l'événement.
