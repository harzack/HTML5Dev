Les paramètres du cluster configurent les informations sur les correctifs pour la gestion des clusters. Sur la page Configurer les paramètres de déploiement, vous fournissez les informations suivantes :

  • Mettre l'URL à jour définit l'URL qui est utilisée pour mettre à jour le manifeste du système. Les répertoires par défaut pour Mettre l'URL à jour sont les suivants :

    • Windows, http://mimage.opentext.com/support/ecm/contentserver/10.5/manifest_win.xml

    • Solaris 10, http://mimage.opentext.com/support/ecm/contentserver/10.5/manifest_sol10.xml

    • Solaris 11, http://mimage.opentext.com/support/ecm/contentserver/10.5/manifest_sol11.xml

    • Linux 5, http://mimage.opentext.com/support/ecm/contentserver/10.5/manifest_lnx5.xml

    • Linux 6, http://mimage.opentext.com/support/ecm/contentserver/10.5/manifest_lnx6.xml

  • Hôte est le modèle système utilisé pour les autres systèmes dans le cluster.

  • Dossier de mise en lots des correctifs est l'emplacement sur l'instance Content Server où les nouveaux correctifs sont mis en lots pour la base de données. Le répertoire de mise en lots par défaut se trouve dans <Content Server_accueil>/clustermanagement/staging
