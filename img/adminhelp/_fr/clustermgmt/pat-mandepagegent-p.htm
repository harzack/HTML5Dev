Pour gérer les agents de cluster :

  1. Dans la section Configuration du serveur de la page Administration Content Server, cliquez sur le lien Gestion des clusters.

  2. Sur la page Gestion des clusters, cliquez sur le lien Gérer les agents de cluster.

  3. Sur la page Agents cluster, cliquez sur l'icône Plus en regard de l'hôte souhaité.

  4. Cliquez sur le lien en ligne de l'hôte souhaité.

     

    Note :
    Vous ne pouvez pas accéder à un hôte s'il est hors ligne.
