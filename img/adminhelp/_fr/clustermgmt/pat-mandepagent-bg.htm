La fonction Agents de cluster affiche les hôtes pour Content Server. La page Agents de cluster affiche toutes les informations suivantes sur les hôtes et les instances disponibles :

  • Hôte est le nom de l'instance de Content Server.

  • Port est le numéro du port que l'instance de Content Server utilise.

  • Etat de l'agent indique si l'hôte est actif, inactif, ou hors ligne.

  • Dernière mise à jour indique quand a eu lieu la dernière modification d'une instance de Content Server.

  • Statut indique si l'hôte est en ligne ou hors ligne.Si le statut est en ligne, des informations détaillées sur l'hôte sont disponibles.
