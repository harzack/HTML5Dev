La fonction Historique d'audit du cluster affiche les événements qui se produisent sur un cluster avec un déploiement et une analyse des correctifs. La page Historique d'audit affiche tous les événements sur les hôtes disponibles, y compris les informations suivantes :

  • Type d'événement est le type d'événement qui s'est produit.

  • Date est la date et l'heure de l'événement.

  • Résultat est le résultat de l'événement. Le résultat est soit OK soit Erreur.

  • Source est l'adresse réseau du cluster.

  • Détails répertorie les informations concernant l'événement.
