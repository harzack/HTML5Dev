La fonction Gérer le cluster analyse et affiche les informations sur les hôtes de Content Server. Vous devez exécuter l'analyse de Gérer le système maître avant que la fonction Gérer le cluster n'analyse les hôtes de Content Server. Lorsque la page Analyser le cluster charge, elle analyse automatiquement tout hôte de Content Server disponible. Elle répertorie également les informations pour chaque hôte, y compris ce qui suit :

  • Hôte est le cluster en cours d'analyse.

  • Port est le port utilisé par l'hôte.

  • Etat de l'analyse fournit l'état de l'analyse, en indiquant si elle est active ou terminée, ou s'il y a des erreurs.

Une fois l'analyse terminée, les informations sur le nouveau correctif sont répertoriées sur la page Vérifier le plan de déploiement du cluster.
