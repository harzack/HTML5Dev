Note :
Dans les versions précédentes, le nombre d'éléments pour les objets de boîte était égal à 0. Exécutez ce script pour mettre à jour ce nombre sur toutes les boîtes existantes afin de refléter les objets assignés aux boîtes.

 

Pour mettre à jour le nombre d'éléments de boîte :

  1. Sur la page Administration Content Server, cliquez sur le lien Configurer les objets physiques de la section Administration d'objets physiques.

  2. Sur la page Configurer les objets physiques, activez la case à cocher Exécuter le script des boîtes.

  3. Sélectionnez le bouton radio Valeur manquante (exécute le script uniquement sur les boîtes dont le nombre d'enfants est « null ») ou Toutes les boîtes.

  4. Cliquez sur le bouton Envoyer.
