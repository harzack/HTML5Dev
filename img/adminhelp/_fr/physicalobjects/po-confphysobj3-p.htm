Pour supprimer un type de libellé :

  1. Sur la page Administration Content Server, cliquez sur le lien Configurer les objets physiques de la section Administration d'objets physiques.

  2. Sur la page Configurer les objets physiques, activez la case à cocher en regard d'un type de libellé, dans la section Types de libellé.

  3. Cliquez sur le bouton Envoyer.

 

Note :
Lorsque vous supprimez un type de libellé, la définition du libellé et tous les libellés créés sous ce type sont supprimés.
