Pour définir le format d'ID unique pour tous les éléments physiques :

  1. Sur la page Administration Content Server, cliquez sur le lien Configurer les objets physiques de la section Administration d'objets physiques.

  2. Sur la page Configurer les objets physiques, dans le champ Format, indiquez le format d'ID unique pour tous les éléments physiques, puis cliquez sur le bouton Afficher l'ID suivant.

  3. Cliquez sur le bouton Envoyer.

 

Notes :

  • Par défaut, le format d'ID unique est %Y0001, %Y représentant les quatre chiffres de l'année. Les autres formats possibles sont %m01, %m représentant les deux chiffres du mois et %d01, %d représentant les deux chiffres du jour.

  • Le format utilisé ici pour l'ID unique est un format par défaut ; vous pouvez spécifier différents formats pour chaque type d'élément physique, si besoin est. Vous pouvez spécifier différents formats pour chaque type d'élément physique à partir de l'espace de travail Objets physiques dans la section Administration des types d'éléments physiques.
