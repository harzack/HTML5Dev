L'administrateur peut renommer les codes d'emplacement assignés aux éléments physiques par lots et mettre ces codes dans la table d'emplacements.

Lors de la mise à jour d'un code existant, tous les emplacements des tables PhysItemData et PhysItemCO correspondant à l'ancien code sont actualisés avec le nouveau code. De même, les nouvelles entrées sont ajoutées à la table PhysObjLocation. Une coche bleue signale un code mis à jour.

Les emplacements de formulaire libre ne figurent pas dans la table PhysObjLocation. L'administrateur peut utiliser la fonction de gestion des codes d'emplacement pour ajouter ces emplacements à la table.
