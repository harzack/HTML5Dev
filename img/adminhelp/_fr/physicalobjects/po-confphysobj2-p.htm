Pour insérer une définition de papier et de type de libellé par défaut :

  1. Sur la page Administration Content Server, cliquez sur le lien Configurer les objets physiques de la section Administration d'objets physiques.

  2. Sur la page Configurer les objets physiques, spécifiez un fichier de script dans le champ Exécuter un script.

  3. Cliquez sur le bouton Envoyer.
