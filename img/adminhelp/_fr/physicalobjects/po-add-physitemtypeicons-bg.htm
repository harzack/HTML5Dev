Vous avez la possibilité d'ajouter vos propres icônes pour représenter les types d'éléments physiques personnalisés dans Content Server. Placez les images GIF 16 x 16 dans le dossier Content Server_home/support/physicalobjects/icons. Vous devez ensuite redémarrer Content Server pour que les nouvelles icônes apparaissent sur la page Ajouter un nouveau type d'élément physique.

 

Note :
Lorsque vous modifiez l'icône d'un type d'élément physique existant, la modification ne s'applique pas à l'icône des éléments physiques ajoutés à ce type avant la modification.
