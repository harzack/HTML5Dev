Le module Physical Objects permet de configurer les paramètres des objets physiques et de spécifier les fonctions des objets physiques dont le suivi est effectué. Vous pouvez également conserver les codes d'emplacement créés par les utilisateurs.

 

Note :
Les versions précédentes du module Physical Objects permettaient d'ajouter, de modifier et de supprimer des types d'éléments physiques via la section Administration d'objets physiques de la page Administration Content Server. Ces fonctions font désormais partie de l'espace de travail Objets physiques.
