Le module Physical Objects permet de configurer des objets physiques. Vous pouvez définir le format d'ID unique pour tous les éléments physiques. Vous pouvez également insérer une définition de papier et de type de libellé par défaut, et supprimer des types de libellé. Lorsque vous insérez un type de libellé par défaut, vous devez spécifier un script qui automatise le processus d'insertion. Vous pouvez également exécuter un script pour mettre à jour le nombre d'éléments pour les boîtes.

 

Note :
Les versions précédentes du module Physical Objects permettaient de définir le nombre de jours par défaut pour prêter un élément physique ainsi que l'emplacement d'enregistrement par défaut des éléments physiques créés. Ces fonctions font désormais partie de l'espace de travail Objets physiques.
