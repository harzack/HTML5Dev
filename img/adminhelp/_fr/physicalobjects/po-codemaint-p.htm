Pour renommer les codes d'emplacement par lots :

  1. Sur la page Administration Content Server, cliquez sur le lien Conserver les codes d'emplacement de la section Administration d'objets physiques.

  2. Sur la page Conserver les codes d'emplacement, activez la case à cocher en regard du code.

  3. Saisissez un nouveau code dans le champ Nouveau code.

  4. Cliquez sur le bouton Envoyer.
