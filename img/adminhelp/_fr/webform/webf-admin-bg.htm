Les consultations de base de données Web Forms sont exécutées à l'aide de JavaScript dans la vue du modèle HTML et consistent en une connexion et une consultation de base de données sécurisées. Avant de créer une consultation de base de données sécurisée, vous devez créer une connexion de base de données sécurisée qui fournisse une connexion à la base de données contenant les informations à consulter.

Gérer des connexions sécurisées à la base de données

Le lien Gérer des connexions sécurisées à la base de données de la section Administration des consultations de bases de données des formulaires Web vous permet d'afficher, de modifier et de créer des connexions de base de données. Une connexion de base de données Web Forms vous permet de vous connecter à une base de données Content Server ou SQL externe afin de les utiliser lors d'une consultation de base de données sécurisée dans une vue HTML d'un modèle de formulaire. Vous ne pouvez pas supprimer une connexion de base de données si des consultations de base de données sécurisées utilisent cette connexion.

Par défaut, seul l'administrateur peut créer des connexions et des consultations de base de données sécurisées. Tous les utilisateurs peuvent effectuer des consultations. Les utilisateurs de Web Forms qui ne disposent pas des autorisations nécessaires pour utiliser la fonction de consultation de base de données obtiendront une erreur.

 

Note :
Si vous souhaitez accorder des autorisations aux utilisateurs qui veulent avoir la possibilité de créer des connexions et des consultations de base de données, ou si vous souhaitez limiter les utilisateurs qui effectuent des consultations de base de données Web Forms, vous devez modifier les limitations d'accès à ces fonctions sur la page Gérer les privilèges relatifs aux objets et à l'utilisation dans la section Administration système. Pour plus d'informations sur la configuration des limitations concernant la fonction de consultation de base de données Web Forms, la création de connexions ou de consultations, reportez-vous à Gestion des privilèges relatifs aux objets et à l`utilisation.
