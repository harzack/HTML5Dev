Pour créer une connexion sécurisée à une base de données :

  1. Dans la section Administration des consultations de bases de données des formulaires Web, cliquez sur le lien Gérer des connexions sécurisées à la base de données.

  2. Sur la page Connexions à des formulaires Web, cliquez sur Connexion à des formulaires Web de la base de données dans le menu Ajouter un élément.

  3. Sur la page Ajouter Connexion à des formulaires Web de la base de données, cliquez sur le type de base de données relationnelle auquel vous souhaitez vous connecter dans la liste Type de serveur SGBDR.

     

    Note :
    Le serveur Microsoft SQL n'est disponible dans le menu Type de serveur SGBDR que si le serveur Content Server est exécuté sur un serveur Microsoft Windows.

    Selon que vous avez sélectionné le système de base de données Oracle ou Microsoft SQL dans la liste Type de serveur SGBDR, des champs différents s'affichent dans la page Ajouter : Connexion à des formulaires Web de la base de données.

     

    • Pour le serveur Microsoft SQL, spécifiez les informations suivantes :

      Champ Contenu
      Nom de l'utilisateur Le nom du compte utilisateur du serveur Microsoft SQL ayant les autorisations d'accès à la base de données.
      Mot de passe Le mot de passe du compte utilisateur du serveur Microsoft SQL ayant les autorisations d'accès à la base de données.
      Le nom de la base de données du serveur SQL Le nom de la base de données du serveur Microsoft SQL à laquelle vous souhaitez vous connecter.
      Nom du serveur SQL Le nom du serveur Microsoft SQL auquel vous souhaitez vous connecter.

    • Pour le serveur Oracle, vous devez spécifier le nom d'utilisateur et le mot de passe utilisés pour la connexion au serveur Oracle, ainsi que le Nom de service, qui est la chaîne de connexion pour le service de la base de données à laquelle vous souhaitez vous connecter.

  4. Le champ Recherches dépendantes s'affiche quel que soit le système de base de données sélectionné. Vous ne pouvez pas modifier ce champ lors de la création d'une connexion à la base de données.

  5. Renseignez les paramètres Nom, Description, Catégories et Créer dans de la même manière que lors de l'ajout d'élément à Content Server.

     

    Note :
    Le nom que vous spécifiez pour la connexion à la base de données doit être unique parmi toutes les connexions de base de données.

     

  6. Cliquez sur Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Gestion de connexions de base de données sécurisées.
