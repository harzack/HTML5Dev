Vous pouvez utiliser les consultations de base de données sécurisées pour créer une instruction SQL pouvant être exécutée en réponse à l'action d'un utilisateur sur une vue HTML Web Form.

Une consultation nécessite une connexion de base de données Web Forms sécurisée. Cette connexion définit la base de données sur laquelle l'instruction SQL est exécutée. La connexion de base de données sécurisée doit d'abord être créée.

L'instruction SQL à exécuter définit l'instruction SQL qui sera exécutée dans la connexion de base de données spécifiée et peut éventuellement accepter des paramètres issus de la vue HTML Web Form. Pour définir les valeurs récupérées du formulaire de l'instruction SQL, utilisez la syntaxe de définition SQL standard en ajoutant :A<n> à l'instruction SQL pour représenter le paramètre. Par exemple, :A1 sera remplacé par la valeur du premier paramètre et :A-2 par la valeur du deuxième paramètre.

Les consultations de base de données sécurisées peuvent être configurées dans la page Gestion des consultations de base de données sécurisées de la section Administration des consultations de bases de données des formulaires Web en ajoutant un élément de consultation de base de données Web Forms. Si vous cochez la case Résultat du filtre basé sur les autorisations, l'instruction SQL à exécuter doit inclure les colonnes DataID, OwnerID et PermID du tableau DTree dans une base de données Content Server. Avec ces trois colonnes, la recherche s'effectue et seules les lignes pour lesquelles l'utilisateur actuel bénéficie d'une Visualisation de l'accès sont renvoyées dans les données de sortie. Les lignes pour lesquelles l'utilisateur actuel ne bénéficie pas d'une Visualisation de l'accès sont supprimées des données de sortie.

Le Nom doit être unique et la valeur du Nom doit être spécifiée dans le JavaScript de la vue HTML Web Form afin de préciser la recherche à exécuter. Nom, Description, Catégories et Créer dans sont des paramètres standard qui peuvent être utilisés lors de l'ajout de tout élément à Content Server.
