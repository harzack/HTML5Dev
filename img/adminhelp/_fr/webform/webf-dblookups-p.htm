Pour créer une consultation de base de données :

  1. Dans la section Administration des consultations de bases de données Web Forms, cliquez sur le lien Gérer des recherches sécurisées dans la base de données.

  2. Sur la page Recherches formulaires Web, cliquez sur Consultation de bases de données Web Forms dans le menu Ajouter un élément.

  3. Sur la page Ajouter page de consultation de base de données formulaires Web, indiquez les informations suivantes :

    • Connexion de base de données Web Form, qui est le nom de l'objet de connexion représentant le système de base de données auquel vous souhaitez vous connecter.

    • Instruction SQL à exécuter, qui est l'instruction SQL exécutée dans la connexion de base de données spécifiée.

    • Résultat du filtre basé sur les autorisations, qui indique si les résultats de la consultation doivent être limités par les autorisations de l'utilisateur. Si cette option est sélectionnée, vous devez spécifier les colonnes DataID, OwnerID et PermID du tableau DTree dans le champ Instruction SQL à exécuter.

  4. Indiquez les paramètres Nom, Description, Catégories et Créer dans de la même manière que lors de l'ajout d'élément à Content Server.

  5. Cliquez sur Ajouter.

Pour plus d'informations sur cette procédure, consultez la section Pour créer une consultation de base de données.
