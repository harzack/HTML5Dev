Pour ajouter le fichier JavaScript inclus :

  1. Dans la section Administration des consultations de bases de données Web Forms, cliquez sur le lien Ajouter un fichier JavaScript personnalisé à tous les modèles exportés au format HTML.

  2. Dans le champ Chemin et nom du fichier inclus, ajoutez le fichier JavaScript inclus personnalisé.

     

    Note :
    Le chemin doit être relatif au préfixe d'assistance du serveur Content Server. Le préfixe d'assistance figure à la page Fichier JavaScript inclus personnalisé.

     

  3. Cliquez sur Envoyer.

Pour plus d'informations sur cette procédure, consultez la section Ajout d'un fichier JavaScript à tous les modèles exportés au format HTML.
