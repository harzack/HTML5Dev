Vous pouvez spécifier un fichier JavaScript inclus personnalisé pour chaque modèle de formulaire que vous souhaitez exporter au format HTML. Votre serveur Web doit disposer des permissions d'écriture pour le fichier JavaScript inclus personnalisé, sinon les vues des modèles ne pourront pas inclure le fichier JavaScript.
