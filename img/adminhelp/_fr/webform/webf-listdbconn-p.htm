Pour modifier les propriétés d'une connexion de base de données :

  1. Sur la page Connexions à des formulaires Web, cliquez sur le lien du nom d'une base de données pour afficher et modifier les propriétés de la connexion de base de données.

  2. Cliquez sur Ajouter.

 

Notes :

  • Une fois qu'une connexion de base de données est créée, vous ne pouvez pas modifier le type de serveur SGBDR.

  • Le champ Consultations connexes s'affiche quel que soit le système de base de données sélectionné. Vous pouvez voir les noms de toutes les consultations de base de données en cours qui nécessitent une connexion de base de données active. Vous ne pouvez pas modifier les consultations de base de données sur cette page. Si vous souhaitez modifier une consultation de base de données, consultez Pour modifier les propriétés de consultation de Web Forms.

 

Pour plus d'informations sur cette procédure, consultez la section Gestion de connexions de base de données sécurisées.
