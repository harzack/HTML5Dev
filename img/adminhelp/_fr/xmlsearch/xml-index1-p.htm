Pour ajouter un fichier DTD XML :

  1. Sur la page d'administration, cliquez sur le lien Ouvrir le volume DTD XML.

  2. Cliquez sur DTD XML dans le menu Ajouter un élément.

  3. Sur la page Ajouter DTD XML, cliquez sur le bouton Parcourir, recherchez le fichier DTD XML, sélectionnez son nom, puis cliquez sur le bouton Ouvrir.

  4. Saisissez un nom pour le fichier DTD XML ou l'exemple de document XML dans le champ Nom.

  5. Pour afficher une description de la DTD XML ou de l'exemple de document XML dans l'onglet Général de sa page Propriétés, saisissez-la dans le champ Description.

  6. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  7. Pour spécifier un recueil pour la DTD XML que vous ajoutez, cliquez sur le bouton Parcourir Content Server pour rechercher le recueil de votre choix.

  8. Cliquez sur le bouton Ajouter.

  9. Pour rendre disponible à la recherche les régions du fichier DTD XML, cochez la case Pour recherche sur la page Régions pour recherche.

  10. Cliquez sur le bouton Mettre à jour.

 

Astuce :
Pour modifier les régions d'un fichier DTD XML dans le volume de DTD XML, cliquez sur l'icône Fonctions du fichier, puis sélectionnez Définir les régions. Pour sélectionner ou désélectionner toutes les régions XML, activez ou désactivez la case à cocher située à côté du titre de colonne Pour recherche dans la partie supérieure du tableau Régions.
