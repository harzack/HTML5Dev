Les codes de paramètre de session utilisateur sont ajoutés et gérés via les pages d'administration de Security Clearance. Les codes de paramètre de session utilisateur sont utilisés avec les marquages supplémentaires pour fournir des filtres d'autorisations supplémentaires.

Lorsque des paramètres de session utilisateur sont associés à un marquage supplémentaire, les règles suivantes s'appliquent :

  • Si un code de paramètre de session utilisateur est transmis à l'ouverture de session, un marquage supplémentaire avec des codes de paramètre de session utilisateur ne peut être évalué que par les utilisateurs disposant des marquages supplémentaires et codes de paramètre de session utilisateur appropriés.

  • Si aucun code de paramètre de session utilisateur n'est assigné à un marquage supplémentaire, les utilisateurs peuvent afficher le contenu avec le marquage à tout emplacement s'ils disposent du marquage.

  • Si aucun code n'est transmis à l'ouverture de session, les critères de marquages supplémentaires avec des codes de paramètres de session utilisateur ne sont pas satisfaits et l'utilisateur ne peut pas les consulter. Toutefois, l'utilisateur peut toujours consulter les éléments avec un marquage supplémentaire sans code de paramètre de session utilisateur.
