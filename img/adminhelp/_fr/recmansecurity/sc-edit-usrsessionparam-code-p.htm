Pour modifier des codes de paramètre de session utilisateur :

  1. A la section Administration de Security Clearance de la page Administration Content Server, cliquez sur le lien Configurer les codes de paramètre de session utilisateur.

  2. A la page Paramètres de session utilisateur, cliquez sur le lien Modifier du code souhaité dans la colonne Actions.

  3. A la page Modifier le paramètre de session utilisateur, modifiez la description, puis cliquez sur Mettre à jour.
