Pour définir un groupe de responsables d'habilitation de sécurité :

  1. Cliquez sur le lien Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires de la section Administration de Security Clearance dans la page Administration Content Server.

  2. Dans la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires, cliquez sur le bouton Sélectionner un groupe.

  3. Sur la page Sélectionner un utilisateur, cliquez sur Nom de groupe dans la liste déroulante Rechercher, spécifiez les critères de votre choix, puis cliquez sur le bouton Rechercher.

  4. Cliquez sur le lien Sélectionner d'un groupe.

  5. Cliquez sur le bouton Envoyer.
