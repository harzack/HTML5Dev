Le module Security Clearance permet d'ajouter et de modifier des marquages supplémentaires avec ou sans paramètre DoD activé. Vous pouvez également ajouter des utilisateurs aux marquages supplémentaires créés. Pour plus d'informations sur les paramètres DoD, reportez-vous à la section Configuration des paramètres d'habilitation de sécurité.

Les marquages supplémentaires sont des codes non hiérarchiques permettant de filtrer les objets pouvant être visualisés par les utilisateurs. Un utilisateur ou un élément de Content Server peut posséder plusieurs marquages supplémentaires. Les utilisateurs peuvent voir des objets s'ils possèdent tous les mêmes marquages supplémentaires que l'objet.

Vous pouvez assigner des codes de paramètres de session utilisateur à des marquages supplémentaires pour fournir des filtres d'autorisations supplémentaires. Certaines règles s'appliquent à ce que les utilisateurs peuvent voir lorsque des codes de paramètres de session utilisateur sont assignés aux marquages supplémentaires. Pour en savoir plus, voir Configuration des codes de paramètre de session utilisateur.
