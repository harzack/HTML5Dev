Pour configurer les boutons de sélection multiple en mode navigation :

  1. Dans la page Administration Content Server, cliquez sur le lien Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires de la section Administration de Security Clearance.

  2. A la section Actions d'habilitation de sécurité en mode navigation de la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires, cochez la case de chaque action de sélection multiple à activer pour les utilisateurs lorsqu'ils se trouvent en mode d'exploration.

  3. Cliquez sur Envoyer.
