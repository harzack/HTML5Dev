Pour ajouter un marquage supplémentaire :

  1. Dans la page Administration Content Server, cliquez sur le lien Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires de la section Administration de Security Clearance.

  2. Dans la page Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires, cliquez sur Marquage supplémentaire dans le menu Ajouter : Elément.

  3. Dans la page d'ajout de niveaux d'habilitation de sécurité et marquages supplémentaires, saisissez un code de marquage supplémentaire dans le champ Code.

  4. Saisissez une description du marquage supplémentaire dans le champ Description.

  5. Facultatif Cliquez sur le bouton radio Code de données restreintes afin d'empêcher les utilisateurs de saisir des informations dans les sections Déclasser le et Bridé le de la page Définir l'habilitation de sécurité.

  6. Facultatif Cliquez sur le bouton radio Code de données précédemment restreintes afin d'empêcher les utilisateurs de saisir des informations dans les sections Déclasser le et Bridé le de la page Définir l'habilitation de sécurité.

  7. Dans la zone Paramètre de session utilisateur, utilisez les flèches pour sélectionner les paramètres de session utilisateur à associer au marquage supplémentaire.

     

    Note :
    Vous pouvez cliquez sur le lien Afficher les détails ou Masquer les détails pour afficher ou masquer les paramètres de session utilisateur associés au marquage supplémentaire. Si vous basculez d'une option à l'autre, les modifications apportées aux champs de métadonnées sont conservées.

    Si des valeurs sont sélectionnées, les utilisateurs dotés du marquage supplémentaire doivent avoir au moins une des valeurs transmise à leur session pour afficher le contenu doté du marquage supplémentaire.

     

  8. Cliquez sur le bouton Envoyer.

 

Note :
Si un paramètre DoD est activé et si vous cliquez sur le bouton radio Code de données restreintes ou Code de données précédemment restreintes, toutes les modifications des champs Déclasser le et Bridé le de la page Définir l'habilitation de sécurité sont ignorées.
