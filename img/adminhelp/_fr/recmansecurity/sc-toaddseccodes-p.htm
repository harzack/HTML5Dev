Pour ajouter un niveau d'habilitation de sécurité :

  1. Dans la page Administration Content Server, cliquez sur le lien Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires de la section Administration de l'habilitation de sécurité.

  2. Dans la page Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires, cliquez sur Niveau d'habilitation de sécurité dans le menu Ajouter : Elément.

  3. Dans la page d'ajout de niveaux d'habilitation de sécurité et marquages supplémentaires, saisissez une valeur dans le champ Niveau.

  4. Saisissez un nom pour le niveau d'habilitation de sécurité dans le champ Nom.

    Vous pouvez également saisir une description du niveau dans le champ Description.

  5. Cliquez sur le bouton Envoyer.
