Pour activer les mises à jour de contenu de boîte :

  1. Cliquez sur le lien Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires de la section Administration de Security Clearance dans la page Administration Content Server.

  2. Dans la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires, cliquez sur le bouton radio Oui de la section Mettre à jour le contenu de la boîte.

  3. Cliquez sur le bouton Envoyer.
