Pour ajouter un type d'objet pris en charge :

  1. Dans la section Administration de l'habilitation de sécurité de la page Administration Content Server, cliquez sur le lien Configurer les types d'objets pris en charge.

  2. Dans la page Configurer les types d'objets pris en charge, cliquez sur Type d'objet dans le menu Ajouter un élément.

  3. Dans la page Ajouter un type d'objet pris en charge, tapez un code de sous-type dans le champ Type d'objet. Facultatif : activez la case à cocher Hériter à partir de afin que les objets enfants héritent des marquages supplémentaires et niveaux d'habilitation de sécurité des objets parents.

  4. Cliquez sur le bouton Envoyer.

 

Note :
La case à cocher Hériter à partir de apparaît uniquement si les paramètres DoD sont désactivés. Pour plus d'informations sur les paramètres DoD, reportez-vous à la section Configuration des paramètres d'habilitation de sécurité.
