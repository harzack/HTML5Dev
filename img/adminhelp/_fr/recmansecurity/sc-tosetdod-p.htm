Pour définir des paramètres DoD :

  1. Dans la page Administration Content Server, cliquez sur le lien Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires de la section Administration de Security Clearance.

  2. Dans la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires, cliquez sur un des boutons radio suivants de la section Paramètres de configuration :

    • Marquages supplémentaires et niveaux d'habilitation de sécurité permet de définir le module Security Clearance sans paramètre DoD.

    • Marquages supplémentaires pour compatibilité DoD 5015.2 Chapitre 2 permet de définir le module Security Clearance avec les exigences DoD spécifiées dans le chapitre 2 du document DoD 5015.2 Standards.

    • Marquages supplémentaires et niveaux d'habilitation de sécurité pour la compatibilité DoD 5015.2 Chapitres 2 et 4 permet de définir le module Security Clearance avec les exigences DoD spécifiées dans le chapitre 4 du document DoD 5015.2 Standards.

    Facultatif : saisissez une valeur dans le champ Plage temporelle de déclassement pour définir le nombre d'années pour le déclassement d'un objet après la date d'enregistrement si le paramètre Marquages supplémentaires et niveaux d'habilitation de sécurité pour la compatibilité DoD 5015.2 Chapitres 2 et 4 est activé.

  3. Cliquez sur le bouton Envoyer.

 

Note :
Si vous activez le bouton radio Marquages supplémentaires et niveaux d'habilitation de sécurité dans la section Paramètres de configuration, la table des objets pris en charge est pré-remplie avec des types d'objets pris en charge par défaut en fonction des modules que vous avez installés sur votre système. Pour plus d'informations sur les types d'objets pris en charge, reportez-vous à la rubrique Configuration des types d'objets pris en charge.

Si vous avez saisi une nouvelle valeur dans le champ Plage temporelle de déclassement, vous pouvez revenir à la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires et cliquer sur le bouton Recalculer la date de déclassement pour mettre à jour tous les documents selon la nouvelle valeur.
