Pour modifier un type d'objet pris en charge :

  1. Dans la section Administration de l'habilitation de sécurité de la page Administration Content Server, cliquez sur le lien Configurer les types d'objets pris en charge.

  2. Dans la page Configurer les types d'objets pris en charge, cliquez sur le lien Modifier d'un type d'objet.

  3. Modifiez les paramètres souhaités.

  4. Cliquez sur le bouton Envoyer.
