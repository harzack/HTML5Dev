Pour ajouter des utilisateurs à un marquage supplémentaire :

  1. Dans la page Administration Content Server, cliquez sur le lien Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires de la section Administration de Security Clearance.

  2. Dans la page Gestion des niveaux d'habilitation de sécurité et marquages supplémentaires, cliquez sur le lien Ajouter/Enlever des utilisateurs d'un code de marquage supplémentaire.

  3. Dans la page de définition de marquages supplémentaires pour les utilisateurs, spécifiez des critères de recherche, puis cliquez sur le bouton Rechercher.

  4. Activez la case à cocher Ajouter/Enlever pour chaque utilisateur à assigner.

  5. Cliquez sur le bouton Mettre à jour.
