Pour appliquer les niveaux de sécurité et les marquages supplémentaires pour les administrateurs système

  1. Cliquez sur le lien Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires de la section Administration de l'habilitation de sécurité dans la page Administration Content Server.

  2. Dans la page Configurer le système pour les niveaux d'habilitation de sécurité et/ou les marquages supplémentaires, cliquez sur le bouton radio Oui de la section Appliquer les niveaux de sécurité et les marquages supplémentaires pour les administrateurs système.

  3. Cliquez sur le bouton Envoyer.
