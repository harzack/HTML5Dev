Pour ajouter des codes de paramètre de session utilisateur :

  1. A la section Administration de Security Clearance de la page Administration Content Server, cliquez sur le lien Configurer les codes de paramètre de session utilisateur.

  2. A la page Paramètres de session utilisateur, cliquez sur Paramètre de session utilisateur dans la liste Ajouter un élément.

  3. A la page Ajouter un paramètre de session utilisateur, saisissez le code d'un paramètre de session utilisateur dans le champ Code.

  4. Facultatif Saisissez une description du code dans le champ Description.

  5. Cliquez sur Ajouter.
