1. Dans la section Administration de Records management de la page Administration Content Server, cliquez sur le lien Objets gérés.

  2. Dans la page Objets gérés, cliquez sur le menu Ajouter un élément , puis sélectionnez le type d'objet à ajouter.

 

Important :
Lorsque vous ajoutez un type d'objet géré à la page Objets gérés, celui-ci n'est pas géré automatiquement. Vous devez cocher la case Géré correspondant à chaque type d'objet pour activer les options d'objet géré sélectionnées.

 
 

Note :
Lorsque vous avez ajouté un objet géré supplémentaire à la page Objets gérés, vous ne pouvez plus le supprimer. Si vous ne souhaitez plus gérer ce type d'objet, décochez la case Géré et relancez les services Content Server.
