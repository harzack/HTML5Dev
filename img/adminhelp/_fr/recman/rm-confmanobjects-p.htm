Pour configurer des objets gérés :

 

Note :
Vous pouvez configurer les options Objet géré à tout moment, mais ces options ne seront activées que lorsque vous aurez coché la case Géré pour le type d'objet concerné.

 

Pour... Dans la section Administration de Records management de la page Administration Content Server, cliquez sur le lien Objets gérés, puis procédez comme suit...
Activer la gestion d'objets Activez la case à cocher Géré située en regard des types d'objets concernés, puis cliquez sur le bouton Mettre à jour.
Permettre aux objets enfants d'hériter des classements RM de leur élément parent Activez la case à cocher Hériter à partir de située en regard des types d'objets de recueil concernés, puis cliquez sur le bouton Mettre à jour.
Assurer que les autorisations sont bridées lorsqu'un objet est officialisé Activez la case à cocher Brider les autorisations située en regard des types d'objets concernés, puis cliquez sur le bouton Mettre à jour.
Activer la fonction de titrage Activez la case à cocher Utiliser le titrage située en regard des types d'objets concernés, puis cliquez sur le bouton Mettre à jour.
Sélectionnez les types d'éléments pouvant se voir appliquer des attentes basées sur utilisateur Activez la case à cocher Attente utilisateur située en regard des types d'objets concernés, puis cliquez sur le bouton Mettre à jour.
Définir le classement RM par défaut d'un type d'objet particulier Cliquez sur le bouton Sélectionner, accédez au classement de votre choix, cliquez sur le lien Sélectionner correspondant, puis sur le bouton Mettre à jour.

 

Note :
Si vous modifiez une ou plusieurs des cases à cocher Géré, vous devez redémarrer les services Content Server pour que les changements prennent effet.
