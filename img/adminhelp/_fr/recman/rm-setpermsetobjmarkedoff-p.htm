To set permission settings for objects marked as official:

  1. On the Livelink Administration page, click the Set Classify and Mark Official link in the Records Management Administration section.

  2. On the Configure Access on Classify and Mark Official page, click one of the following radio buttons in the Mark Official section:

    • Use existing access restrictions, which downgrades an object's existing Access List permissions to See, See Contents when a user marks the object as official

    • Add to access list, which adds the permissions from the Primary RM classification Access List to an object's existing Access List permissions, and then downgrades them to See, See Contents when a user marks the object as official. The Records Managers group is also added with full rights.

    • Replace access list, which removes an object's existing Access List permissions, replaces them with the Primary RM classification Access List permissions, and then downgrades them to See, See Contents when a user marks the object as official

  3. Click the Update button.

 

Note:
Downgrades are applied only to subtypes that have the Downgrade Permissions setting enabled.
