Le tableau suivant décrit les messages d'erreur relatifs au processus du moteur de recherche.

Si ces erreurs persistent, contactez le service d'assistance clientèle d'OpenText pour obtenir plus d'informations.

Tableau 1-31 : Messages d'erreur des moteurs de recherche

Numéro Message Description Solution
132 Une erreur inconnue s'est produite Le processus du moteur de recherche a rencontré une erreur inconnue. Contactez l'assistance clientèle d'OpenText.
149 Ligne de commande non valable pour les moteurs de recherche Les paramètres de ligne de commande du moteur de recherche comportent une erreur. Vérifiez que les lignes de commande dans le fichier otadmin.cfg sont valables et spécifiées au format correct auquel Content Server génère le fichier otadmin.cfg à l'aide de sa base de données.
150 L'URL n'est pas valable L'URL utilisée pour enregistrer et contacter le moteur de recherche n'est pas valable comme indiqué dans le fichier search.ini. Assurez-vous que les paramètres du fichier search.ini sont valables et que leur format est correct.
153 Impossible d'initialiser Le processus du moteur de recherche n'a pas pu démarrer car un ou plusieurs paramètres du fichier search.ini ne sont pas valables. Assurez-vous que les paramètres du fichier search.ini sont valables et que leur format est correct.
 

Note :
Le fichier search.ini est le fichier de configuration pour les processus d'indexation et de recherche de Content Server. Ce fichier contient des paramètres qui sont généralement définis par défaut lors de la création des processus d'indexation et de recherche et d'autres objets système, tels que les sources de données et les partitions ; ces paramètres sont supprimés lorsque les processus qu'ils configurent sont supprimés. La configuration de ces paramètres a lieu lors de la configuration des processus d'indexation et de recherche et des autres objets système dans Content Server. OpenText vous recommande vivement de ne pas modifier les paramètres du fichier search.ini. Si vous décidez de modifier ces paramètres, contactez le service d'assistance clientèle d'OpenText.
