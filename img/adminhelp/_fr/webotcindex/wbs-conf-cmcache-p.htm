Pour configurer la recherche de paramètres de mémoire cache :

  1. Dans la section Administration de recherche de la page d'administration, cliquez sur le lien Configurer les options de recherche.

  2. Sur la page Configurer les options de recherche, dans la section Paramètres de mémoire cache, saisissez une valeur dans le champ Agents mis en mémoire cache à conserver.

  3. Saisissez une valeur dans le champ Ensembles de régions de mémoire cache à conserver.

  4. Cliquez sur le bouton Mettre à jour.

     

    Note :
    Vous devez cliquer sur le bouton Appliquer sur la page Modification de la barre de recherche pour appliquer les modifications.

     

Pour en savoir plus, voir Configuration de la recherche Paramètres de mémoire cache.
