Le moteur de recherche prend désormais en charge un nouveau modificateur de région pour les requêtes de recherche, l'opérateur « like ». Lorsque cet opérateur est appliqué à des régions de type PartNum, il effectue une création de jetons intense pour tenter de trouver des séquences souvent présentes dans des numéros de parties et des noms de fichiers.

Sur les nouvelles installations et les mises à niveau de Content Server 10.5, ou versions ultérieures, le langage LQL (Live Query Language) et le langage OTSQL (Open Text Structured Query Language) prennent en charge le mot-clé « like » dans la recherche Texte intégral de la recherche avancée. Ces langages reconnaissent le nouveau mot-clé QLLIKE et activent le mode Requête complexe.

Les régions régulières de type « Text » doivent être configurées explicitement pour la prise en charge du mot-clé « like » sur la page Modificateurs de région. Sur les nouvelles installations, la région OTName est configurée pour la prise en charge de « like » au moyen d'entrées par défaut sur la page Modificateurs de région de chaque gestionnaire de recherche.

Toute région déclarée de type PartNum dans le fichier LLFieldDefinitions.txt bénéficie de la prise en charge de « like », par définition. Dans Content Server 10.5, il n'existe pas de régions de type PartNum par défaut.

Le mot-clé « like » fonctionne alors comme un non-opérateur et effectue une requête normale, non modifiée, de la région. Si une région n'est pas configurée pour « like », ce mot-clé fonctionne comme un AND des termes dans l'expression.

Si la région avec des propriétés de numéro de partie est incluse dans la liste des régions de recherche par défaut, l'opération QLLIKE est utilisée automatiquement pour cette région.

Pour configurer des modificateurs de région pour le mot-clé like :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données contenant les régions d'index auxquelles vous souhaitez appliquer des modificateurs de région pour le mot-clé « like ».

  2. Cliquez sur l'icône Fonctions du Gestionnaire de recherche, puis sur Propriétés et sélectionnez Modificateurs de région.

  3. Dans la zone Modificateur like, dans la première zone de texte, saisissez la liste des régions de recherche configurées pour utiliser le modificateur « like ».

  4. Dans la seconde zone de texte, saisissez la liste des régions de recherche configurées pour l'application par défaut du modificateur « like ».

  5. Redémarrez les moteurs d'indexation pour tous les serveurs d'administration dans le cluster Content Server afin que les modifications prennent effet.
