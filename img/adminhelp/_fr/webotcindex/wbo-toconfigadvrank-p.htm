Pour configurer un classement avancé :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant à la source de données pour laquelle vous voulez configurer un classement avancé.

  2. Cliquez sur l'icône Fonctions du gestionnaire de recherche, sélectionnez Propriétés, puis Classement.

  3. Dans l'onglet Classement de la page Propriétés du gestionnaire de recherche, configurez les paramètres suivants :

    • Pondération de la requête

    • Classement du champ, basé sur les régions de métadonnées d'un élément.

    • Classement par date

    • Classement par type

    • Score de l'objet

    Pour plus d'informations sur les paramètres pouvant être définis, reportez-vous à la rubrique Configuration du classement avancé.

  4. Cliquez sur le bouton Mettre à jour.
