Pour ajouter un fédérateur de recherche :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier de la source de données à laquelle vous souhaitez ajouter un fédérateur de recherche.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de recherche.

  3. Cliquez sur Fédérateur de recherche dans le menu Ajouter un élément.

  4. A l'étape de la page Planification de l'assistant d'ajout du fédérateur de recherche, lisez les informations affichées, puis cliquez sur le bouton Suivant.

  5. A l'étape Fédérateur de recherche, procédez comme suit :

    • Dans le champ Nom, saisissez un nom pour le fédérateur de recherche. Si vous souhaitez lancer le fédérateur de recherche et chaque moteur d'indexation après les avoir ajoutés, sélectionnez la case à cocher Démarrer automatiquement.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le fédérateur de recherche.

    • Saisissez un numéro de port dans le champ Port Admin afin de spécifier le port utilisé par le fédérateur de recherche pour s'exécuter sur un serveur d'administration.

    • Saisissez un numéro de port dans le champ Port de recherche pour spécifier le port sur lequel le fédérateur de recherche recevra les requêtes du gestionnaire de recherche.

  6. Cliquez sur le bouton Suivant.

  7. A l'étape Moteurs de recherche, fournissez les informations requises pour ajouter un moteur de recherche à chacune des partitions de votre système d'indexation et de recherche. Tous les moteurs de recherche ajoutés seront gérés par le fédérateur de recherche ajouté. Saisissez les informations suivantes dans chaque section de la page :

    • Dans le champ Nom, saisissez un nom pour le moteur de recherche.

    • Cliquez sur un serveur d'administration dans la liste déroulante Hôte pour spécifier le serveur sur lequel sera exécuté le moteur de recherche.

    • Saisissez un numéro de port dans le champ Port Admin pour spécifier le port utilisé par le moteur de recherche pour s'exécuter sur un serveur d'administration.

    • Dans le champ Port du serveur, saisissez le numéro du port utilisé par les processus d'échange pour communiquer avec le moteur de recherche.

    • Dans le champ Répertoire d'index, saisissez le chemin absolu de l'index de partition dans lequel le moteur de recherche effectuera les recherches.

  8. Cliquez sur le bouton Suivant.

  9. A l'étape Résumé, vérifiez les informations que vous avez fournies pour le nouveau fédérateur de recherche et chaque moteur de recherche, puis cliquez sur le bouton Terminer pour ajouter les processus.

 

Note :
Vous pouvez vérifier la disponibilité d'un numéro de port en cliquant sur le lien Vérifier le port. Vous pouvez également ajouter des fédérateurs de recherche et des moteurs de recherche à une source de données particulière lorsque vous affichez le schéma de partition de la source de données. Pour plus d'informations sur l'affichage des schémas de partition, reportez-vous à la rubrique Utilisation des schémas de partition.
