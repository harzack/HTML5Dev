Pour gérer les objets qui ont échoué :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de l'Extracteur Entreprise.

  2. Sur la page Configurer les paramètres de l'Extracteur Entreprise, cliquez sur le lien Gérer les objets dont aucun contenu n'a été extrait.

  3. Sur la page Gérer les objets dont aucun contenu n'a été extrait, cochez les cases associées aux éléments desquels vous souhaitez que l'Extracteur Entreprise tente à nouveau d'extraire le contenu, puis cliquez sur le bouton Réessayer.

  4. Cochez les cases associées aux éléments à enlever et cliquez sur le bouton Enlever.

  5. Sur la page Gérer les objets dont aucun contenu n'a été extrait, cliquez sur le bouton Terminé.
