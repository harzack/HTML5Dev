Pour configurer les filtres de recherche

  1. Dans la section Administration de recherche de la page Administration, cliquez sur le lien Configurer les filtres de recherche.

  2. Sur la page Configurer les filtres de recherche, dans la section Paramètres généraux, pour Statut , cochez la case Activer pour afficher les filtres de recherche dans un panneau à côté des résultats de recherche.

  3. Dans Filtre par région, sélectionnez les régions à afficher en cliquant sur les boutons fléchés gauche et droit pour les faire basculer vers les colonnes Disponible et Actif, puis cliquez sur les boutons fléchés haut ou bas pour définir l'ordre des régions.

     

    Note :
    Vous pouvez sélectionner les options de recherche des régions d'index sur la page Administration du gestionnaire de recherche. Pour plus de détails, consultez Configuration des régions d'index

    La région de recherche d'auteurs est mal configurée. Vous devez la supprimer de la colonne Actif ou sélectionner la case à cocher Pour recherche correspondante sur la page Administration du gestionnaire de recherche.

     

  4. Dans le champ Nombre de filtres, cliquez sur une valeur dans la liste déroulante pour indiquer le nombre maximum de régions à afficher à l'aide des filtres de recherche. Vous pouvez augmenter le nombre maximum de valeurs de région de filtre de recherche prises en compte par le moteur de recherche. Pour en savoir plus, consultez le document Ensembles de facettes incomplets.

  5. Dans le champ Nombre à prérécupérer, cliquez sur une valeur dans la liste déroulante pour indiquer le nombre de régions que Content Server doit stocker dans sa mémoire cache pour afficher plus rapidement les résultats de recherche.

  6. Dans la section Filtres de texte, dans Moins, cliquez sur une valeur dans la liste déroulante pour indiquer le nombre maximum de valeurs à afficher lorsque Moins... est sélectionné. .

  7. Dans Plus, cliquez sur une valeur dans la liste déroulante pour indiquer le nombre maximum de valeurs à afficher lorsque Plus... est sélectionné.

  8. Dans la section Filtres par date, cliquez sur une valeur dans la liste déroulante pour indiquer le nombre maximum de valeurs à afficher pour une période donnée.

  9. Dans la section Afficher les nombres, cliquez sur une valeur dans la liste déroulante pour indiquer le seuil d'affichage des comptes de résultats :

    • Toujours : toujours afficher les comptes de résultats.

    • Jamais : ne jamais afficher les comptes de résultats.

    • Une valeur comprise entre > 1 et > 100 : afficher les comptes de résultats lorsque plusieurs valeurs sont spécifiées.

    Vous pouvez utiliser ce paramètre pour indiquer que, pour les petites valeurs de compte, si l'utilisateur ne se trouve pas dans le groupe privilégié « Afficher les Comptes de résultats non autorisés », le compte n'est pas affiché. OpenText recommande l'utilisation de ce paramètre pour les installations de Content Server dans lesquelles la sécurité est un point crucial ou dans lesquelles l'administrateur système ne souhaite afficher aucun compte pour les filtres de recherche.

    Les utilisateurs dotés du privilège « Afficher les Comptes de résultats non autorisés » peuvent voir le vrai compte de valeurs des filtres de recherche, plutôt qu'une estimation. Les comptes des filtres de recherche sont toujours affichés pour les utilisateurs du groupe « Afficher les Comptes de résultats non autorisés », même lorsque l'option Jamais est sélectionnée. La section Afficher les nombres s'applique uniquement aux utilisateurs qui ne se trouvent pas dans ce groupe.

  10. Cliquez sur Mettre à jour pour utiliser les paramètres de filtres de recherche définis ou sur Appliquer pour enregistrer les paramètres.
