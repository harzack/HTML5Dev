A l'aide des réglages proposés par la Configuration des paramètres de l'Extracteur Entreprise, vous pouvez :

  • Modifier les paramètres qui contrôlent les processus de l'Extracteur Entreprise lorsqu'il ne reçoit pas de contenu.

  • Afficher les éléments dont le contenu a été reçu.

Quand vous modifiez les paramètres qui contrôlent le processus Extracteur Entreprise lorsque les objets ne parviennent pas à extraire du contenu, vous pouvez configurer le nombre de tentatives d'extraction d'objet du Content Server et l'intervalle entre chaque tentative. Passé le nombre de tentatives, Content Server ne tentera plus de récupérer le contenu de cet objet. Il enverra une notification indiquant que le seuil a été dépassé et vous pouvez demander à ce que tous les Extracteurs Entreprise soient arrêtés.

Depuis la page Gérer les objets dont aucun contenu n'a été extrait, vous pouvez également gérer les éléments dont le contenu n'a pas pu être extrait. Vous pouvez : Afficher la liste des éléments et leur emplacement, voir les éléments pour lesquels la limite a été atteinte, tenter à nouveau le processus d'extraction et enlever des éléments.
