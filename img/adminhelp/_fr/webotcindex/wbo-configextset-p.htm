Pour définir les opérations à suivre en cas de non-obtention de contenu :

  1. Sur la page d'administration, cliquez sur le lien Configurer les paramètres de l'Extracteur Entreprise.

  2. Sur la page Configurer les paramètres de l'Extracteur Entreprise, cliquez sur le lien Définir les opérations à entreprendre en cas de non-obtention de contenu.

  3. Sur la page Définir les opérations à entreprendre en cas de non-obtention de contenu, activez la case Activé pour permettre aux opérations de s'effectuer lorsque le processus d'extraction ne parvient pas à obtenir de contenu.

  4. Dans la section Arrêt en cas d'échec, cliquez sur le bouton radio Oui pour interrompre le processus de l'Extracteur.

  5. Dans la section Nombre de tentatives, cliquez sur une valeur dans la liste déroulante pour définir le nombre de tentatives de récupération du contenu de l'objet par le processus de l'Extracteur.

  6. Dans la section Durée entre les tentatives, cliquez sur les valeurs dans la liste déroulante pour définir l'intervalle entre le Nombre de tentatives.

  7. Dans la section Nombre maximum d'objets pour le nouvel essai, cliquez sur une valeur dans la liste déroulante pour restreindre le nombre de tentatives de récupération de contenu par le processus d'extraction.

  8. Dans la section Niveau de notification en cas d'échec, cliquez sur un niveau d'avertissement dans la liste déroulante pour définir le niveau de notification de l'échec.

    Content Server envoie une notification et interrompt tous les processus de l'Extracteur Entreprise.

  9. Cliquez sur le bouton Mettre à jour.
