Pour ajouter un gestionnaire de recherche :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier de la source de données auquel vous voulez ajouter un gestionnaire de recherche.

  2. Cliquez sur Gestionnaire de recherche dans le menu Ajouter un élément.

  3. Sur la page Ajouter Responsable de recherche, saisissez un nom dans le champ Nom.

  4. Pour fournir une description du gestionnaire de recherche sous l'onglet Général de sa page Propriétés, saisissez une description dans le champ Description.

  5. Pour modifier les catégories et les attributs associés à cet élément, cliquez sur le bouton Modifier.

  6. Cliquez sur le bouton Ajouter.
