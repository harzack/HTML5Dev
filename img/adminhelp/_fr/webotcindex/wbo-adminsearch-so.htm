Configuration de la JVM de recherche

Si le processus de recherche n'est pas optimal, vous pouvez spécifier une JVM personnalisée fournie par le service d'assistance clientèle d'OpenText. Pour en savoir plus, voir Configuration de la JVM de recherche.

Configuration des options de recherche

Lors de la configuration des options de recherche Content Server, vous activez, désactivez et modifiez les paramètres qui altèrent le fonctionnement de la recherche Content Server pour les utilisateurs. Pour en savoir plus, voir Configuration des options de recherche.

Filtres de recherche

Les filtres de recherche sont affichés dans un panneau situé en regard des résultats de recherche, comportant une barre de titre et une liste de valeurs, indiquant une valeur pour les métadonnées et un chiffre indiquant la fréquence à laquelle cette valeur apparaît dans les résultats. Pour en savoir plus, voir Filtres de recherche.

Gestion des processus de recherche

Les fédérateurs de recherche et les moteurs de recherche sont les processus de recherche des systèmes d'indexation et de recherche. Un fédérateur de recherche est un processus qui gère un ou plusieurs moteurs de recherche et qui leur distribue les recherches. Un moteur de recherche est un processus qui effectue la recherche dans l'index d'une partition. Les responsables de recherche servent de récipients aux processus de recherche et jouent un rôle dans l'exécution des recherches. Pour en savoir plus, voir Gestion des processus de recherche.

Création des vues de recherche

Une vue de recherche est un domaine de recherche défini par un ensemble de critères de recherche (requête) appliqués à un ou plusieurs index. Lorsque vous créez un flux de données d'indexation, Content Server crée automatiquement une vue de recherche qui représente l'ensemble de toutes les données de l'index obtenu. En tant qu'administrateur, vous pouvez également créer des vues de recherche personnalisées en enregistrant les requêtes dans Dossier - Vues de recherche. Les vues de recherche personnalisées permettent aux utilisateurs Content Server de formuler des requêtes usuelles sans avoir à saisir à nouveau des critères de recherche complexes. Pour en savoir plus, voir Création des vues de recherche.

Configuration des régions d'index

L'index d'une source de données est constitué de régions, chacune d'elles stockant un type d'information différent. Vous pouvez configurer les options de recherche pour les régions afin de personnaliser le mode de recherche utilisé par les utilisateurs. Pour en savoir plus, voir Configuration des régions d'index.

Elimination des régions d'index

Vous pouvez synchroniser la liste des régions des bases de données Content Server avec celles qui se trouvent dans l'index de recherche en éliminant les régions qui n'existent pas dans l'index de recherche. Pour en savoir plus, voir Pour éliminer les régions d'index.

Configuration du classement avancé

Vous configurez le classement avancé lorsque vous administrez le gestionnaire de recherche pour l'index d'une source de données. Les responsables de recherche jouent un rôle dans l'exécution des recherches dans les systèmes d'indexation et de recherche. La configuration du classement avancé vous permet de déterminer les éléments les plus appropriés pour votre site. Elle peut aider les utilisateurs Content Server à localiser les informations qui leur sont les plus utiles. Lorsqu'un utilisateur envoie une recherche Content Server, Content Server assigne un score aux éléments Content Server qui correspondent aux critères de recherche spécifiés. Les éléments sont répertoriés dans les résultats de la recherche dans l'ordre de leurs scores (du plus élevé au plus faible). Pour en savoir plus, voir Configuration du classement avancé.

Configuration des mappages de liens hypertextes

Les mappages de liens hypertextes permettent aux utilisateurs d'accéder à des informations dans une source de données Directory Walker lors de l'affichage des résultats de recherche. Les documents contenus dans les sources de données Directory Walker sont identifiés par leur nom de chemin complet. Lorsque ces documents sont renvoyés dans une recherche, ils apparaissent sur la page Résultats de recherche sous leur nom de chemin. Pour en savoir plus, voir Configuration des mappages de liens hypertextes.

Gestion des formulaires de recherche

Les formulaires de recherche sont des configurations de la page Recherche, que vous créez et enregistrez. Ils vous permettent de réutiliser une configuration particulière de la page de recherche sans avoir à la recréer à chaque fois. Pour plus d'informations, reportez-vous à la rubrique Gestion des modèles de recherche.

Gestion du téléchargement comme feuille de calcul

Les utilisateurs peuvent créer une collection de leurs résultats de recherche, puis télécharger la collection comme feuille de calcul. En tant qu'administrateur, vous pouvez restreindre l'utilisation de cette fonction à certains utilisateurs ou groupes sur la page Gérer les privilèges relatifs aux objets et à l'utilisation. Pour en savoir plus, voir Gestion du téléchargement comme feuille de calcul.

Administration des meilleurs résultats

Les Meilleurs résultats vous permettent de définir une valeur pouvant être recherchée pour un élément Content Server. Pour en savoir plus, voir Administration des meilleurs résultats.

Recherche des éléments supprimés

Lorsque les utilisateurs suppriment des éléments dans Content Server, ceux-ci sont placés dans le volume des documents supprimés. Grâce à la fonctionnalité Récupérer, les administrateurs Content Server peuvent restaurer les documents supprimés sur un espace de travail Content Server. Pour plus d'informations, reportez-vous à la rubrique Gestion de la récupération des documents.

Gestion des statistiques de recherche

Lorsque vous vous connectez à Content Server en tant qu'administrateur, vous pouvez utiliser la fonction Gérer les statistiques de recherche. Celle-ci permet d'afficher le récapitulatif des statistiques, les termes recherchés, l'utilisation de vue de recherche et l'utilisation du formulaire, ainsi que d'éliminer les statistiques de recherche. Pour en savoir plus, voir Gestion des statistiques de recherche.
