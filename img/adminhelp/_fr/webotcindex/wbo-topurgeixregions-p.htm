Vous devrez peut-être supprimer les régions non utilisées de la représentation Content Server des régions de source de données. Seules les régions présentes dans le modèle Content Server, mais pas dans l'index de recherche peuvent être éliminées.

Pour éliminer les régions d'index :

  1. Sur la page Administration, dans la section Administration de recherche, cliquez sur le lien Ouvrir le volume des objets système.

  2. Cliquez sur le lien dossier de la source de données de l'entreprise.

  3. Sur la page Dossier de la source de données de l'entreprise, cliquez sur l'icône Fonctions du gestionnaire de recherche Entreprise, choisissez Propriétés, puis Purger les régions.

  4. Sous Purger, cliquez sur la case à cocher supérieure pour sélectionner ou déselectionner toutes les cases à cocher.

  5. Ou, sélectionnez la case à cocher pour chaque région à supprimer des régions du gestionnaire de recherche.

  6. Cliquez sur Mettre à jour ou Appliquer pour supprimer les régions que vous avez sélectionnées.

 

Note :
Si aucun nom de région n'est affiché sur cette page, c'est que toutes les régions dans Content Server existent également dans l'index de recherche.
