Pour configurer un fédérateur de recherche :

  1. Sur la page Administration système, cliquez sur le lien < préfixe_processus> Dossier de la source de données correspondant à une source de données contenant le fédérateur de recherche à configurer.

  2. Cliquez sur le lien <préfixe_processus> Gestionnaire de recherche.

  3. Sur la page Gestionnaire de recherche, cliquez sur l'icône Fonctions du fédérateur de recherche, puis choisissez Arrêter les processus de recherche.

  4. Cliquez sur l'icône Fonctions du fédérateur de recherche, choisissez Propriétés, puis Spécifique.

  5. Dans l'onglet Spécifique de la page Propriétés du fédérateur de recherche, modifiez les paramètres du fédérateur de recherche.

  6. Cliquez sur le bouton Mettre à jour.

  7. Cliquez sur l'icône Fonctions du fédérateur de recherche configuré, puis sélectionnez Lancer les processus de recherche.

 

Astuce :
Vous pouvez également configurer les fédérateurs de recherche lorsque vous affichez le schéma de partition auquel ils appartiennent.
