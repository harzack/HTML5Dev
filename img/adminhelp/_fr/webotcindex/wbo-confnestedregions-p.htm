To configure a nested index region:

  1. Open the Content Server_home/config/LLFieldDefinitions.txt file in a text editor.

  2. In the nested section of the LLFieldDefinitions.txt file, type NESTED and the name of the outer region you want to index.

     

    Notes:

    • You must use a separate line for each outer region in the LLFieldDefinitions.txt file.

    • The space between NESTED and the name of the outer region is a tab space.

     

  3. Save and close the LLFieldDefinitions.txt file.

  4. Reconstruct the index. For information about reconstructing the index, see Data Source Maintenance Results.
