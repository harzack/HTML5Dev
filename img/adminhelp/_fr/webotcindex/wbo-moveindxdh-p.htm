Pour déplacer un index vers le lecteur local d'un autre hôte Content Server :

  1. Sur la page Administration système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier de la source de données du répertoire d'indexation à déplacer.

  2. Cliquez sur l'icône Fonctions du lien préfixe_processus Gestionnaire de flux de données, puis sélectionnez Interrompre.

  3. Cliquez sur l'icône Fonctions du lien préfixe_processus Gestionnaire de recherche, puis sélectionnez Arrêter.

  4. Déplacez le répertoire d'indexation vers son nouvel emplacement.

  5. Cliquez sur le lien préfixe_processus Schéma de partition.

  6. Sur la page Schéma de partition, cliquez sur le lien correspondant au nom d'un moteur d'indexation.

  7. Dans l'onglet Infos spécifiques de la page Propriétés du moteur d'indexation, cliquez sur le raccourci correspondant au serveur d'administration vers l'hôte duquel vous avez déplacé le répertoire d'indexation dans la liste déroulante Hôte.

  8. Saisissez le nouveau chemin d'accès au répertoire d'indexation dans le champ Répertoire d'indexation tel qu'il est mappé sur le serveur d'administration sélectionné dans la liste déroulante Hôte (par exemple, nouveau_chemin/index).

  9. Cliquez sur le bouton Mettre à jour.

  10. Cliquez sur le lien correspondant au nom d'un moteur de recherche chargé d'effectuer des recherches dans le répertoire d'indexation à déplacer.

  11. Dans l'onglet Infos spécifiques de la page Propriétés du moteur de recherche, saisissez le nouveau chemin d'accès au répertoire d'indexation dans le champ Répertoire d'indexation tel qu'il est mappé sur le serveur d'administration sélectionné dans la liste déroulante Hôte (par exemple, nouveau_chemin/index).

  12. Dans la liste déroulante Hôte, cliquez sur le raccourci correspondant au serveur d'administration vers l'hôte duquel vous avez déplacé le répertoire d'indexation.

  13. Cliquez sur le bouton Mettre à jour. Si vous voulez autoriser d'autres moteurs de recherche à effectuer des recherches dans l'index de partition à déplacer, modifiez les répertoires d'indexation pour d'autres moteurs de recherche.

  14. Retournez à la page préfixe_processus Dossier de la source de données.

  15. Cliquez sur l'icône Fonctions du lien préfixe_processus Gestionnaire de recherche, puis sélectionnez Démarrer.

  16. Cliquez sur l'icône Fonctions du lien préfixe_processus Gestionnaire de flux de données, puis sélectionnez Reprendre.
