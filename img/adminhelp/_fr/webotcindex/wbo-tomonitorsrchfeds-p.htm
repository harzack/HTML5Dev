Pour afficher le statut d'un fédérateur de recherche :

  1. Sur la page Administration système, cliquez sur le lien < préfixe_processus> Dossier de la source de données correspondant à une source de données contenant le fédérateur de recherche dont vous souhaitez afficher le statut.

  2. Affichez la colonne Statut du gestionnaire de recherche.
