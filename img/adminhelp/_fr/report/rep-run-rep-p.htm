Pour exécuter un rapport LiveReport :

  1. Dans le menu général, sélectionnez Personnel. Ensuite, sélectionnez Rapports.

  2. Sur la page Gérer mes requêtes, sélectionnez l'onglet LiveReports.

  3. Localisez le rapport LiveReport que vous voulez exécuter à partir de la liste et cliquez sur le nom de ce rapport LiveReport ou sur son icône.
