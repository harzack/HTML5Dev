Pour exporter un rapport LiveReport :

  1. Cliquez sur le lien Exporter les rapports LiveReport de la section Administration des rapports LiveReport sur la page Administration Content Server.

  2. Si vous y êtes invité, saisissez le mot de passe d'utilisateur administrateur dans le champ Mot de passe de l'administrateur, puis cliquez sur Connexion.

  3. Sur la page Exporter le rapport LiveReport, activez les cases à cocher des rapports LiveReport que vous souhaitez exporter vers un seul fichier texte.

  4. Procédez de l'une des manières suivantes :

    • Pour exporter les rapports LiveReport sélectionnés, cliquez sur le bouton Exporter.

    • Pour exporter et supprimer les rapports LiveReport sélectionnés, cliquez sur le bouton Exporter et supprimer.

  5. Enregistrez le fichier de rapport LiveReport à l'emplacement de votre choix.
