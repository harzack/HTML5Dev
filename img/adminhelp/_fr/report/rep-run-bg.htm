Lorsque vous exécutez un rapport LiveReport interactif, une nouvelle page apparaît, vous demandant de spécifier certains paramètres d'entrée. Si un bouton Parcourir apparaît, cliquez sur ce dernier pour sélectionner l'un des choix disponibles. Si aucun bouton Parcourir n'apparaît, vous devez taper directement le texte dans le champ de paramètre d'entrée fourni. Dans tous les cas, cliquez sur le bouton Mettre à jour pour appliquer la ou les valeurs au rapport LiveReport.

Après avoir exécuté un rapport LiveReport, Content Server affiche les résultats du rapport. La bannière de la page contient le titre du rapport LiveReport. La présentation des résultats dans le corps de la page dépend du rapport LiveReport. Les résultats apparaissent sous la forme d'un tableau, d'un graphique circulaire, d'un graphique à barres ou d'un graphique en courbes.

Si le rapport LiveReport que vous exécutez est lié à un sous-rapport, vous pouvez exécuter celui-ci en cliquant sur les résultats ou sur le lien Détails d'un résultat. Par exemple, un rapport LiveReport peut présenter les éléments Content Server sous la forme d'un graphique circulaire, chaque secteur représentant le nombre d'éléments appartenant à un utilisateur. Ce rapport peut être accompagné d'un sous-rapport signalant tous les éléments détenus par un utilisateur donné. En cliquant sur l'un des secteurs, vous pouvez exécuter le sous-rapport.
