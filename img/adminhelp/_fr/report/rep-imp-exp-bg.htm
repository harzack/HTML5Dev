A partir de Content Server, vous pouvez exporter des rapports LiveReport vers un fichier texte. Les fichiers texte exportés peuvent ensuite être importés dans l'installation Content Server de votre choix.

Au lieu de créer un rapport LiveReport de toute pièce, vous pouvez importer un rapport LiveReport exporté depuis Content Server vers un fichier texte. Ce fichier texte peut contenir un ou plusieurs rapports LiveReport.
