Pour configurer la sécurité des rapports LiveReports

  1. Sur la page Administration Content Server, dans la section Administration des rapports LiveReports, cliquez sur le lien Configurer la sécurité LiveReports.

  2. Sur la page Configurer la sécurité LiveReports, activez Empêcher LiveReports de modifier la base de données. L'activation de ce paramètre est définitive.

  3. La boîte de dialogue Désactiver la modification de la base de données apparaît. Activez Je comprends le risque lié à l'activation de ce paramètre et souhaite poursuivre, puis cliquez sur OK.

  4. Sur la page Configurer la sécurité LiveReports, cliquez sur Enregistrer les modifications.
