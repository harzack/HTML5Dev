Pour importer un rapport LiveReport à partir d'un fichier texte :

  1. Cliquez sur le lien Importer les rapports LiveReport de la section Administration des rapports LiveReport sur la page Administration Content Server.

  2. Si vous y êtes invité, saisissez le mot de passe d'utilisateur Admin dans le champ Mot de passe d'utilisateur Admin, puis cliquez sur Connexion.

  3. Facultatif Spécifiez un emplacement où stocker le rapport LiveReport.

    1. Sur la page Importer le rapport LiveReport, cliquez sur Parcourir Content Server si vous voulez importer le rapport LiveReport vers un emplacement Content Server autre que celui qui est affiché dans le champ Créer dans.

       

      Note :
      L'emplacement par défaut dans le champ Créer dans est le volume Rapports LiveReport Content Server. Si vous souhaitez afficher le rapport LiveReport dans les onglets Statut des espaces de travail Personnel des utilisateurs, vous devez les stocker dans le volume Rapports LiveReport Content Server.

       

    2. Parcourez le recueil Content Server vers lequel vous souhaitez importer le rapport LiveReport, puis cliquez sur le lien Sélectionner du recueil. La boîte de dialogue Sélectionner le recueil de création se ferme et le chemin de l'emplacement sélectionné s'affiche dans le champ Créer dans.

  4. Dans la section Fichier, cliquez sur Choix du fichier. Sélectionnez le fichier texte d'un rapport LiveReport à télécharger, puis cliquez sur Ouvrir. La boîte de dialogue se ferme et le chemin du fichier sélectionné s'affiche dans le champ Fichier.

  5. Pour que le rapport LiveReport importé remplace un rapport LiveReport existant du même nom, activez Ecraser les rapports LiveReport existants. Si vous n'activez pas cette case, Content Server affiche un message d'erreur lorsque vous tentez d'importer un rapport LiveReport portant le même nom qu'un rapport LiveReport existant.

  6. Dans la section Action, cliquez sur Ajouter.
