Pour fournir un accès pour les utilisateurs à l'entrepôt de transport :

  1. Cliquez sur le lien Gérer les privilèges relatifs aux objets et à l'utilisation dans la section Administration système de la page Administration Content Server.

  2. Dans la section Privilèges d'utilisation de la page Gérer les privilèges relatifs aux objets et à l'utilisation, cliquez sur le lien Modifier les restrictions pour le type d'utilisation Administration de l'entrepôt.

  3. Sur la page Modifier le groupe, recherchez les utilisateurs ou les groupes que vous souhaitez ajouter au groupe Gestionnaire d'entrepôt, sélectionnez la case à cocher Ajouter au groupe, puis cliquez sur Envoyer.

  4. Dans la zone Modifier le groupe, cliquez sur Terminé.

 

Astuce :
Pour enlever l'accès à des utilisateurs ou groupes spécifiques, cliquez sur le lien de l'utilisateur ou du groupe de la page Modifier le groupe, puis cliquez sur Enlever du groupe.
