Pour configurer la JVM de recherche :

  1. Sur la page d'administration, cliquez sur le lien Configurer la JVM de recherche.

  2. Dans le champ Chemin relatif vers JRE :, saisissez le chemin relatif vers l'environnement Java Runtime Environment fourni par le service d'assistance clientèle d'OpenText pour être utilisé dans Content Server.

  3. Dans le champ Arguments supplémentaires, saisissez les arguments JVM à ajouter.

  4. Cliquez sur le bouton Mettre à jour.

  5. Arrêtez et démarrez tous les processus de recherche, puis resynchronisez les serveurs d'administration.

 

Note :
Certaines versions de JRE, y compris JRE 6, Mise à jour 20, qui est fourni avec Content Server, allouent initialement une importante quantité de mémoire. Pour les partitions de recherche relativement peu remplies, on compte jusqu'à 200 Mo d'allocation de mémoire. Cette mémoire est utilisée à mesure que les partitions sont remplies et ne semble pas affecter les partitions plus importantes.
