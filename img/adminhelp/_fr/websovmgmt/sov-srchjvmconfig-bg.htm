Vous pouvez spécifier une JVM (machine virtuelle Java) personnalisée si les performances des processus de recherche ne sont pas optimales. Vous devez utiliser la JVM fournie par le service d'assistance clientèle d'OpenText.

Pour appliquer la JVM personnalisée, vous devez spécifier le chemin Java Runtime Environment (JRE) relatif et les arguments JVM sur la page Configuration de la JVM de recherche. En mettant à jour ces paramètres, vous vous assurez que le chemin JRE et les valeurs des arguments JVM sont définis ou ajoutés à la table LLSystemData, selon que les valeurs existent ou non.

Après la modification des paramètres sur cette page, vous devez :

  • interrompre tout processus de recherche

  • redémarrer les serveurs d'administration

  • resynchroniser les serveurs d'administration

  • lancer les processus de recherche
