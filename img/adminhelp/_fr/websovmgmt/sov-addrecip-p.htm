Pour ajouter des destinataires d'e-mails :

  1. Sur la page Configurer l'envoi des e-mails d'alerte de l'objet système par messagerie, cliquez sur l'icône Ajouter un utilisateur .

  2. Saisissez un nom pour cette configuration de messagerie dans le champ Nom de la configuration.

  3. Dans la section Envoyer à, effectuez l'une des opérations suivantes :

    • Cliquez sur le bouton radio à côté de l'icône Rechercher un utilisateur. Cliquez ensuite sur l'icône Rechercher un utilisateur, recherchez un utilisateur Content Server, puis cliquez sur le lien Sélectionner dans la colonne Actions de l'utilisateur Content Server qui doit recevoir les e-mails d'alerte relatifs aux objets système.

    • Cliquez sur le bouton radio Autre, puis saisissez une adresse e-mail valable identifiant le compte de messagerie auquel Content Server envoie des e-mails d'alerte relatifs aux objets système.

    • Le cas échéant, cliquez sur le bouton radio Adresse e-mail de l'administrateur Content Server pour envoyer des e-mails d'alerte relatifs aux objets système au compte de messagerie attribué actuellement à l'administrateur Content Server sur le site Content Server.

  4. Dans la section Options de rapport, activez les cases à cocher suivantes de votre choix :

    • Erreurs graves, qui informe le destinataire des erreurs système critiques, comme l'arrêt d'un moteur de recherche ;

    • Erreurs, qui informe le destinataire des erreurs système générales, comme l'arrêt d'un processus de flux de données ;

    • Avertissements, qui notifie le destinataire des erreurs non critiques dans le système, comme un flux de données ne répondant plus aux messages Ipool ;

    • Informations, qui informe le destinataire du statut général et d'autres informations système ;

    • Corrections, qui notifie le destinataire qu'une erreur précédemment notifiée a été rectifiée.

  5. Dans la section Niveau d'informations, cliquez sur l'un des boutons radio suivants :

    • Message de texte détaillé fournit aux destinataires une explication détaillée de l'alerte d'objet système au format texte brut ;

    • Message HTML détaillé fournit aux destinataires une explication détaillée de l'alerte d'objet système au format HTML ;

    • Message de texte court fournit aux destinataires le nombre d'erreurs rapportées dans le système Content Server

       

      Note :
      Créez des messages courts pour les destinataires qui accèdent à leurs e-mails depuis des radiomessageurs, téléphones cellulaires et autres appareils de poche.

       

  6. Cochez la case correspondant aux jours où Content Server doit envoyer des e-mails d'alerte relatifs aux objets système à ce destinataire.

  7. Cochez la case correspondant aux heures où Content Server doit envoyer des e-mails d'alerte relatifs aux objets système à ce destinataire.

  8. Cliquez sur le bouton Mettre à jour.
