Pour configurer les champs de résultats de recherche requis :

  1. Sur la page Administration, dans la section Administration des meilleurs résultats, cliquez sur le lien Configurer les champs de résultats de recherche requis.

    Sur la page Administration, dans la section Administration des meilleurs résultats, cliquez sur le lien Gérer les éléments de meilleurs résultats.

  2. Dans la colonne Disponible, choisissez les champs que vous souhaitez rendre obligatoires, puis cliquez sur la flèche vers la droite pour les ajouter à la colonne Obligatoire. Vous pouvez également cliquer sur un champ de la colonne Obligatoire puis cliquer sur la flèche vers le haut ou le bas pour réorganiser les champs.

  3. Cliquez sur Appliquer.

 

Astuce :
Lorsque vous êtes connecté en tant qu'utilisateur doté de droits d'administration système, vous pouvez également modifier les champs de résultat de recherche obligatoires sur la page Options d'affichage, dans la section Obligatoire, en cliquant sur le lien Configurer les champs de résultats de recherche requis. Pour plus d'informations sur la page Options d'affichage, reportez-vous à Configuration des options d'affichage des résultats de recherche.
