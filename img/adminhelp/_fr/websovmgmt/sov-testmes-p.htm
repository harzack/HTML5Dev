Pour envoyer des messages de test :

  1. Sur la page d'administration, cliquez sur le lien Configurer l'envoi d'e-mails relatifs aux objets système.

  2. Sur la page Configurer l'envoi des e-mails d'alerte de l'objet système par messagerie, procédez de l'une des façons suivantes :

    • Cliquez sur l'icône Ajouter un utilisateur  , puis configurez l'adresse e-mail et le type de message du destinataire.

    • Cliquez sur l'icône Modifier cet utilisateur  à côté de la configuration du destinataire à qui vous souhaitez envoyer un message de test.

  3. Sur la page Configuration de l'utilisateur, cliquez sur le bouton Envoyer un message de test.

  4. Cliquez sur le bouton Mettre à jour.
