Une fois que vous avez configuré l'envoi d'e-mails, configurez Content Server pour l'envoi d'e-mails d'alerte relatifs aux objets système. Content Server envoie ces e-mails aux utilisateurs Content Server, à l'administrateur Content Server ou aux autres comptes de messagerie configurés.

Avant que les destinataires puissent recevoir des e-mails d'alerte relatifs aux objets système, vous devez configurer les paramètres d'un serveur de messagerie. Fournissez le nom et le numéro de port du serveur SMTP qu'utilisera Content Server. Un indicateur de statut de la connexion SMTP indique si la connexion SMTP actuelle est valide. Fournissez aussi les informations d'envoi et de réception associées aux e-mails d'alerte relatifs aux objets système envoyés par Content Server, notamment :

  • L'adresse e-mail à laquelle les destinataires des e-mails d'alerte relatifs aux objets système peuvent répondre.

  • Le texte qui apparaît sur la ligne A partir de des e-mails d'alerte relatifs aux objets système. Par défaut, ce texte est Messagerie automatisée Content Server. Si vous avez plusieurs installations Content Server principales, saisissez un texte qui permettra de distinguer l'origine des e-mails d'alerte relatifs aux objets système.

  • Un nom représentant l'expéditeur des e-mails d'alerte relatifs aux objets système. L'expéditeur par défaut est Content Server. Certains serveurs de messagerie nécessitent l'inscription du nom d'un expéditeur dans les paramètres de configuration par défaut.
