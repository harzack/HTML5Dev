Pour configurer les paramètres du courrier :

  1. Sur la page Configurer l'envoi des e-mails d'alerte de l'objet système par messagerie, saisissez le nom du serveur SMTP dans le champ Serveur SMTP.

  2. Saisissez le numéro de port récepteur du serveur SMTP dans le champ Port SMTP.

  3. Saisissez une adresse e-mail de réponse à laquelle les destinataires peuvent répondre dans le champ Réponse.

  4. Saisissez le texte à afficher sur la ligne De de tous les e-mails d'alerte relatifs aux objets système dans le champ De.

  5. Saisissez un nom correspondant à l'expéditeur de l'e-mail d'alerte relatif aux objets système dans le champ Expéditeur.
