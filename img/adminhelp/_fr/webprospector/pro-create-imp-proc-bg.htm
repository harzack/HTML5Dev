Les prospecteurs analysent et récupèrent des données en communiquant avec un processus d'importation de prospecteurs. Un processus d'importation de prospecteurs est un processus de flux de données qui analyse les nouvelles données une fois qu'elles ont été indexées et qui identifie les éléments correspondant aux critères de recherche d'un prospecteur. Dans les flux de données habituels, le processus d'importation de prospecteurs est effectué après le processus Distributeur de mise à jour. Les processus d'importation de prospecteurs sont automatiquement ajoutés à chaque flux de données créé à l'aide des modèles d'index Content Server. Si vous voulez ajouter des processus d'importation de prospecteurs à des flux de données existants, vous devez le faire manuellement. Vous pouvez, par exemple, créer un processus d'importation de prospecteurs dans le flux de données Entreprise pour permettre aux utilisateurs d'effectuer une recherche dans les vues de recherche Entreprise et Entreprise [Toutes les versions].

 

Notes :

  • Vous ne pouvez ajouter qu'un processus d'importation de prospecteurs à chaque flux de données.

  • Vous ne pouvez pas créer de processus d'importation de prospecteurs dans la source de données de l'aide utilisateur ou de l'aide administrateur.

  • Vous pouvez ajouter des prospecteurs à un flux de données OpenText Enterprise Library. Les utilisateurs peuvent effectuer des recherches dans la vue de recherche Enterprise Library [Toutes les variantes]. Un prospecteur ignore les vues de recherche Enterprise Library [Toutes les versions] et effectue une recherche dans les vues de recherche Entreprise Library [Toutes les variantes].

  • Les recherches dans les autres vues de recherche Enterprise Library ne donnent pas les résultats attendus.

  • Les prospecteurs ne respectent pas les définitions des vues de recherche. Si vous spécifiez le nom d'une vue de recherche à surveiller dans le champ Dans d'un prospecteur, la vue de recherche est ignorée et la source de données entière de la vue de recherche est alors surveillée.
