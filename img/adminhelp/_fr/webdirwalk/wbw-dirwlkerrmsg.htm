Le tableau suivant décrit les messages d'erreur susceptibles de s'afficher en créant des flux de données Directory Walker.

Si ces erreurs persistent, contactez le service d'assistance clientèle d'OpenText pour obtenir plus d'informations.

Tableau 1-24 : Messages d'erreur Directory Walker

Numéro Message Description Solution
1 Erreur Content Server principale Une erreur interne s'est produite lors de la tentative de création ou de définition de paramètres par Content Server dans le fichier de configuration d'un processus Directory Walker. Contactez l'assistance clientèle d'OpenText.
2 Impossible de lire un paramètre de configuration Content Server a rencontré une erreur lors de la lecture d'un paramètre dans le fichier de configuration du processus Directory Walker. Vérifiez le fichier de configuration pour vous assurer que tous les paramètres présentent des valeurs valables. Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher la ligne de commande d'un processus Directory Walker dans l'onglet Spécifique de sa page Propriétés.
3 Impossible de définir un paramètre de configuration Content Server a rencontré une erreur lors de la configuration d'un paramètre dans le fichier de configuration du processus Directory Walker. Vérifiez les valeurs spécifiées lors de la création ou de la mise à jour du processus Directory Walker. Si vous mettez à jour un processus Directory Walker, vous pouvez vérifier les valeurs spécifiées dans l'onglet Spécifique de sa page Propriétés.
4 Impossible de se connecter au serveur d'administration La connexion de Content Server au serveur d'administration qui gère le processus Directory Walker a échoué. Vérifiez le serveur d'administration associé au processus Directory Walker pour vous assurer qu'il est disponible. Si vous mettez à jour un processus Directory Walker existant, vous pouvez changer de serveur d'administration en cliquant sur une valeur dans la liste déroulante Serveur d'administration, située dans l'onglet Spécifique de sa page Propriétés.
5 Impossible d'initialiser le fichier de configuration Content Server n'a pas pu initialiser le fichier de configuration du processus Directory Walker. Vérifiez que les ressources mémoires disponibles de votre système sont adéquates.
6 Fichier de configuration introuvable Content Server n'a pas pu trouver le fichier de configuration du processus Directory Walker. Vérifiez le nom du fichier de configuration pour vous assurer qu'il correspond au nom spécifié sur la ligne de commande du processus Directory Walker. Vérifiez que le fichier de configuration n'est pas en cours d'utilisation et que ses autorisations n'empêchent pas le processus Directory Walker de lire les données qu'il contient.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher la ligne de commande d'un processus Directory Walker dans l'onglet Spécifique de sa page Propriétés.
7 Impossible d'écrire des données dans le fichier de configuration Content Server n'a pas pu écrire de données dans le fichier de configuration du processus Directory Walker. Vérifiez que les autorisations du fichier de configuration n'empêchent pas le processus Directory Walker d'écrire des données dans celui-ci. Vous pouvez également contrôler votre système pour vous assurer qu'il dispose de suffisamment d'espace disque disponible et que le fichier de configuration n'est pas verrouillé.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire de configuration Content Server_home/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher la ligne de commande d'un processus Directory Walker dans l'onglet Spécifique de sa page Propriétés.
8 Impossible de supprimer le fichier de configuration Content Server n'a pas pu supprimer le fichier de configuration du processus Directory Walker. Vérifiez le nom du fichier de configuration pour vous assurer qu'il correspond au nom spécifié sur la ligne de commande du processus Directory Walker. Vérifiez que le fichier de configuration n'est pas en cours d'utilisation et que ses autorisations n'empêchent pas le processus Directory Walker d'écrire des données dans celui-ci.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher la ligne de commande d'un processus Directory Walker dans l'onglet Spécifique de sa page Propriétés.
9 Impossible de définir le répertoire d'écriture Content Server n'a pas pu définir le répertoire d'écriture du processus Directory Walker. Le répertoire d'écriture est celui dans lequel sont créés les IPools de lecture et d'écriture du flux de données. Assurez-vous d'avoir spécifié un chemin de répertoire valable dans le champ Ecrire les données dans lors de la création du processus Directory Walker. Si vous mettez à jour un processus Directory Walker existant, vous pouvez changer le répertoire d'écriture en cliquant sur le bouton Informations situé dans l'onglet Spécifique de la page Propriétés du processus Directory Walker, puis en définissant une nouvelle valeur dans le champ Ecriture.
10 Impossible de définir la zone d'écriture Content Server n'a pas pu définir la zone d'écriture du processus Directory Walker. Contactez l'assistance clientèle d'OpenText.
11 Impossible de définir le fichier journal Content Server n'a pas pu définir le fichier journal du processus Directory Walker. Assurez-vous qu'un chemin d'accès de répertoire et un nom de fichier valables ont été spécifiés pour le fichier journal. Si vous mettez à jour un processus Directory Walker existant, vous pouvez spécifier le chemin absolu et le nom du fichier journal dans le champ Fichier journal situé dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
12 Impossible de définir le niveau de consignation Content Server n'a pas pu définir le niveau de journal du processus Directory Walker. Vérifiez que le niveau de consignation spécifié pour le processus Directory Walker est correct. Si vous mettez à jour un processus Directory Walker existant, vous pouvez spécifier un niveau de journal dans la liste déroulante Niveau de journal située dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
14 Impossible de définir l'emplacement de la base de données Historique des fichiers parcourus Content Server n'a pas pu définir l'emplacement de la base de données Historique des fichiers parcourus du processus Directory Walker. L'emplacement de la base de données Historique des fichiers parcourus est le répertoire dans lequel vous souhaitez que le processus Directory Walker écrive ses fichiers d'historique. Vérifiez que vous avez spécifié le chemin absolu d'un répertoire valable lors de la création ou de la mise à jour du processus Directory Walker. OpenText recommande de choisir le même répertoire que celui spécifié pour le répertoire d'écriture du processus Directory Walker.
20 Impossible de définir le paramètre Désactiver du fichier de configuration Content Server n'a pas pu définir la valeur du paramètre Disable dans le fichier de configuration du processus Directory Walker. Le paramètre Disable permet à Content Server de supprimer temporairement des groupes de répertoires particuliers dans la liste que doit parcourir Directory Walker. Contactez l'assistance clientèle d'OpenText.
21 Impossible de définir le paramètre Traversal du fichier de configuration Content Server n'a pas pu définir le paramètre Traversal du fichier de configuration du processus Directory Walker. Le paramètre Traversal est utilisé avec la valeur sélectionnée dans la liste déroulante Profondeur lors de la création du processus Directory Walker pour déterminer si Directory Walker doit parcourir les répertoires de manière récurrente. Contactez l'assistance clientèle d'OpenText.
22 Impossible de définir la profondeur du répertoire Content Server n'a pas pu définir la profondeur des répertoires que le processus Directory Walker doit parcourir. Vérifiez que vous avez spécifié une valeur valable dans la liste déroulante Profondeur lors de la création ou de la mise à jour du processus Directory Walker. La liste suivante décrit les différentes valeurs disponibles dans la liste déroulante Profondeur :
•

Illimité pour spécifier que le processus Directory Walker doit parcourir tous les sous-répertoires au-dessous des répertoires de niveau supérieur

•

Spécifier qui permet d'entrer une valeur positive indiquant le nombre exact de niveaux de sous-répertoires que le processus Directory Walker doit parcourir

•

Aucun pour spécifier que le processus Directory Walker doit uniquement parcourir le répertoire spécifié

Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier la profondeur en cliquant sur une valeur dans la liste déroulante Profondeur située dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
23 Impossible de définir la taille de fichier minimum Content Server n'a pas pu définir la taille minimum requise pour les fichiers que le processus Directory Walker doit parcourir. Vérifiez que vous avez spécifié une valeur entière positive dans le champ Taille du fichier approprié à la création du processus Directory Walker. La taille de fichier minimum doit être inférieure ou égale à la taille maximum. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier la taille de fichier en définissant de nouvelles valeurs dans les champs Taille du fichier de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
24 Impossible de définir la taille de fichier maximum Content Server n'a pas pu définir la taille maximum requise pour les fichiers que le processus Directory Walker doit parcourir. Vérifiez que vous avez spécifié une valeur entière positive dans le champ Taille du fichier approprié à la création du processus Directory Walker. La taille de fichier maximum doit être supérieure ou égale à la taille minimum. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier la taille de fichier en définissant de nouvelles valeurs dans les champs Taille du fichier de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
25 Impossible de définir la date des fichiers les plus anciens parcourus par le processus Directory Walker Content Server n'a pas pu définir la date des fichiers les plus anciens que le processus Directory Walker doit parcourir. Il s'agit de la première valeur dans l'intervalle de dates spécifié pour le groupe de répertoires. Vérifiez que vous avez spécifié une date valable dans les listes déroulantes Intervalle de dates appropriées lors de la création ou de la mise à jour du processus Directory Walker. La date des fichiers les plus anciens doit être inférieure ou égale à la date des fichiers les plus récents dans l'intervalle. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier l'intervalle de dates en définissant de nouvelles valeurs dans les listes déroulantes Intervalle de dates de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
26 Impossible de définir la date des fichiers les plus récents parcourus par le processus Directory Walker Content Server n'a pas pu définir la date des fichiers les plus récents que le processus Directory Walker doit parcourir. Il s'agit de la deuxième valeur dans l'intervalle de dates spécifié pour le groupe de répertoires. Vérifiez que vous avez spécifié une date valable dans les listes déroulantes Intervalle de dates appropriées lors de la création du processus Directory Walker. La date des fichiers les plus récents doit être supérieure ou égale à la date des fichiers les plus anciens dans l'intervalle. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier l'intervalle de dates en définissant de nouvelles valeurs dans les listes déroulantes Intervalle de dates de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
28 Impossible de définir les répertoires de niveau supérieur Content Server n'a pas pu spécifier les répertoires de niveau supérieur que le processus Directory Walker doit parcourir. Vérifiez que vous avez spécifié le chemin d'accès absolu aux répertoires de niveau supérieur dans le champ Répertoires à la création du processus Directory Walker, en les séparant les uns des autres par un point-virgule ou un retour-chariot. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier le chemin d'accès aux répertoires en définissant de nouvelles valeurs dans les champs Répertoires, dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
29 Impossible de spécifier les types de fichiers que vous souhaitez inclure Content Server ne peut pas spécifier la liste des types de fichiers que le processus Directory Walker doit analyser lorsqu'il parcourt les répertoires spécifiés. Assurez-vous d'avoir spécifié des modèles de noms de fichier valables pour permettre au processus Directory Walker de les parcourir (par exemple, *.html; *.txt; *.doc, etc.). Vous indiquez des modèles de noms de fichier dans le champ Inclure lors de la création du processus Directory Walker. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier les modèles de noms de fichier en définissant de nouvelles valeurs dans le champ Inclure de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
30 Impossible de spécifier les types de fichiers que vous souhaitez exclure Content Server ne peut pas spécifier la liste des types de fichiers que le processus Directory Walker ne doit pas analyser lorsqu'il parcourt les répertoires spécifiés. Assurez-vous d'avoir spécifié des modèles de noms de fichier valables pour permettre au processus Directory Walker de les parcourir (par exemple, *.html; *.txt; *.doc, etc.). Vous indiquez ces modèles de noms de fichier dans le champ Exclure lors de la création du processus Directory Walker. Si vous mettez à jour un processus Directory Walker existant, vous pouvez modifier les modèles de noms de fichier en définissant de nouvelles valeurs dans le champ Exclure de l'onglet Spécifique de la page Propriétés du processus Directory Walker.
31 Impossible de définir le nom de l'emplacement de la base de données Historique des fichiers parcourus Content Server n'a pas pu associer un nom valable à l'emplacement de la base de données Historique des fichiers parcourus que vous avez spécifié. Il s'agit du nom que vous avez spécifié dans le champ Préfixe du processus à la création du processus Directory Walker. Contactez l'assistance clientèle d'OpenText.
100 Arguments de ligne de commande spécifiés non valables Le processus Directory Walker n'a pas pu démarrer parce que certains de ses arguments de ligne de commande sont incorrects. Vérifiez les arguments de la ligne de commande spécifiés dans le champ Ligne de commande de l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker. En cas d'erreurs (syntaxe incorrecte par exemple), arrêtez le processus et corrigez-les en modifiant le champ Modèle de commande. Exécutez à nouveau le processus Directory Walker.
101 Impossible de lire le fichier de configuration Le processus Directory Walker n'a pas pu lire le fichier de configuration ou certains de ses paramètres. Vérifiez que les autorisations du fichier de configuration n'empêchent pas le processus Directory Walker de lire des données de celui-ci. Vous pouvez également contrôler votre système pour vous assurer qu'il dispose de suffisamment d'espace disque disponible et que le fichier de configuration n'est pas verrouillé. Pour terminer, assurez-vous que le fichier de configuration est bien stocké dans l'emplacement spécifié sur la ligne de commande.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire de configuration Content Server_home/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus.
Si vous pensez que le fichier de configuration contient des paramètres incorrects, vérifiez les arguments de ligne de commande spécifiés dans le champ Ligne de commande, dans l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker. En cas d'erreurs (syntaxe incorrecte par exemple), arrêtez le processus et corrigez-les en modifiant le champ Modèle de commande. Exécutez à nouveau le processus Directory Walker.
102 Impossible d'ouvrir le fichier journal Le processus Directory Walker n'a pas pu ouvrir le fichier journal dans lequel il enregistre des données. Vérifiez que les autorisations du fichier journal n'empêchent pas le processus Directory Walker d'écrire des données dans celui-ci. Vous pouvez également contrôler votre système pour vous assurer qu'il dispose de suffisamment d'espace disque disponible et que le fichier journal n'est pas verrouillé. Pour terminer, vérifiez que le fichier journal est bien stocké dans l'emplacement spécifié dans le champ Nom du journal de l'onglet Spécifique de la page Propriétés du processus Directory Walker. Par défaut, les fichiers journaux Directory Walker sont stockés dans le répertoire Content Server_accueil/logs.
103 Impossible de lire le niveau de consignation Le processus Directory Walker n'a pas pu lire le niveau de journal spécifié dans le fichier de configuration. Assurez-vous que le niveau de consignation spécifié dans l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker correspond au niveau de consignation spécifié dans le fichier de configuration. Les valeurs possibles sont :
•

Par défaut pour consigner les erreurs uniquement

•

Détaillé pour consigner le statut des messages acheminés à travers les processus et les IPools du flux de données, ainsi que les erreurs

•

Déboguer pour consigner des informations extrêmement détaillées au sujet du statut des flux de données et des erreurs

Les fichiers de configuration Directory Walker sont stockés dans le répertoire de configuration Content Server_home/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher les arguments de la ligne de commande dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
104 Informations sur Directory Walker insuffisantes Le processus Directory Walker n'a pas pu démarrer parce que vous n'avez pas spécifié de répertoire d'écriture ou de répertoire à parcourir. Dans l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker, vérifiez que vous avez spécifié le répertoire d'écriture ainsi que les répertoires que le processus Directory Walker doit parcourir. Assurez-vous que les valeurs définies dans l'onglet Spécifique de la page Propriétés du processus Directory Walker correspondent aux valeurs définies dans le fichier de configuration.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire de configuration Content Server_home/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher les arguments de la ligne de commande dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
105 Impossible de configurer le port des messages d’arrêt Le processus Directory Walker n'a pas pu configurer le port des messages d'arrêt. Le port des messages d'arrêt est un port sur lequel Content Server envoie un message d'arrêt pour interrompre le processus Directory Walker. Assurez-vous qu'un numéro de port valable est spécifié sur la ligne de commande de l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker. Le numéro de port que vous spécifiez ne doit pas être utilisé par un autre processus et doit être compris entre 1 025 et 65 535. Pour modifier le numéro de port, saisissez une nouvelle valeur dans le champ Port de message d'arrêt de l'onglet Spécifique sur la page Propriétés de Directory Walker.
106 Impossible de lire de répertoire d'écriture Le processus Directory Walker n'a pas pu lire le répertoire d'écriture dans le fichier de configuration. Le répertoire d'écriture est celui dans lequel sont créés les IPools de lecture et d'écriture du flux de données. Assurez-vous que le chemin de répertoire spécifié dans le champ Ecriture (auquel vous pouvez accéder en cliquant sur le bouton Informations de l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker) est valable et que le même chemin de répertoire est spécifié dans le fichier de configuration (identificateur IPool).
Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher les arguments de la ligne de commande dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
107 Impossible d'établir une connexion IPool. Le processus Directory Walker n'a pas pu se connecter à un IPool. Assurez-vous que le chemin de répertoire spécifié dans le champ Ecriture (auquel vous pouvez accéder en cliquant sur le bouton Informations de l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker) est valable et que le même chemin de répertoire est spécifié dans le fichier de configuration.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher les arguments de la ligne de commande dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
108 Une erreur interne IPool s'est produite. Le processus Directory Walker a rencontré une erreur lors de la tentative de lancement d'une transaction IPool. Contactez l'assistance clientèle d'OpenText.
109 Impossible de lire la zone d'écriture Le processus Directory Walker n'a pas pu lire la zone d'écriture du fichier de configuration. Contactez l'assistance clientèle d'OpenText.
110 Impossible de lire les informations de la base de données Historique des fichiers parcourus Le processus Directory Walker n'a pas pu lire l'emplacement de la base de données Historique des fichiers parcourus dans le fichier de configuration. L'emplacement de la base de données Historique des fichiers parcourus est le répertoire dans lequel le processus Directory Walker écrit ses fichiers d'historique. Contactez l'assistance clientèle d'OpenText.
116 Impossible de renommer les fichiers de la base de données Historique des fichiers parcourus Content Server n'a pas pu renommer les fichiers de la base de données Historique des fichiers parcourus. Il existe normalement trois fichiers de la base de données Historique des fichiers parcourus : nom.new nom.old et nom.junk. Lorsqu'un nouveau fichier de la base de données Historique des fichiers parcourus est créé, le processus Directory Walker renomme le fichier nom.new en nom.old. Il renomme aussi le fichier nom.old en nom.junk. Vérifiez les autorisations des fichiers de la base de données Historique des fichiers parcourus pour vous assurer que le processus Directory Walker est en mesure de les renommer. Vous pouvez également vérifier votre système pour vous assurer que l'espace disponible est suffisant et que les fichiers de la base de données Historique des fichiers parcourus ne sont pas verrouillés. Pour terminer, assurez-vous que ces fichiers sont stockés dans le répertoire spécifié dans le champ Emplacement de la base de données Historique des fichiers parcourus de l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker. Par défaut, ils sont stockés dans le répertoire d'écriture du processus.
117 Impossible de créer un nouveau fichier de la base de données Historique des fichiers parcourus Le processus Directory Walker n'a pas pu créer un nouveau fichier de la base de données Historique des fichiers parcourus (nom.new). Il existe normalement trois fichiers de la base de données Historique des fichiers parcourus : nom.new nom.old et nom.junk. Lorsqu'un nouveau fichier de la base de données Historique des fichiers parcourus est créé, le processus Directory Walker renomme le fichier nom.new en nom.old. Il renomme aussi le fichier nom.old en nom.junk. Vérifiez que le répertoire des fichiers de la base de données Historique des fichiers parcourus existe et que ses autorisations n'empêchent pas le processus Directory Walker d'écrire des données dans celui-ci. Contrôlez également votre système pour vous assurer que l'espace disque approprié est disponible et que le répertoire n'est pas verrouillé.
118 Impossible de lire un groupe de répertoires Le processus Directory Walker n'a pas pu lire les informations d'analyse associées à un groupe de répertoires. Vérifiez les valeurs spécifiées pour chaque groupe de répertoires dans l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker. Assurez-vous que tous les champs appropriés contiennent des valeurs correctes et qu'elles correspondent à celles du fichier de configuration. Exécutez à nouveau le processus Directory Walker.
Les fichiers de configuration Directory Walker sont stockés dans le répertoire Content Server_accueil/config et portent l'extension .dw. Le nom d'un fichier de configuration Directory Walker particulier est spécifié sur la ligne de commande du processus. Vous pouvez afficher les arguments de la ligne de commande dans l'onglet Spécifique de la page Propriétés du processus Directory Walker.
119 Une erreur interne IPool s'est produite. Le processus Directory Walker a rencontré une erreur interne de type non spécifié. Contactez l'assistance clientèle d'OpenText.
120 Impossible de lire le nom de l'emplacement de la base de données Historique des fichiers parcourus Le processus Directory Walker n'a pas pu lire le nom associé à l'emplacement de la base de données Historique des fichiers parcourus. Il s'agit du nom que vous avez spécifié dans le champ Préfixe du processus à la création du processus Directory Walker. Contactez l'assistance clientèle d'OpenText.
