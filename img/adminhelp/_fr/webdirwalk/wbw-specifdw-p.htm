Pour configurer les paramètres d'un processus Directory Walker :

  1. Sur la page Volume des objets système, cliquez sur le lien préfixe_processus Dossier de la source de données correspondant au dossier de la source de données qui contient le processus Directory Walker à configurer.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur l'icône Fonctions du processus préfixe_processus Directory Walker, puis sélectionnez Arrêter.

  4. Cliquez sur le lien préfixe_processus Directory Walker.

  5. Dans l'onglet Infos spécifiques de la page Propriétés du processus Directory Walker, modifiez les paramètres du processus Directory Walker.

  6. Cliquez sur le bouton Mettre à jour.

  7. Cliquez sur l'icône Fonctions du processus préfixe_processus Directory Walker, puis sélectionnez Démarrer.
