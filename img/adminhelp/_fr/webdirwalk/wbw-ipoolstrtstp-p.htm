Pour définir iPool ainsi que les options d'arrêt et de lancement :

  1. Dans le champ Ecrire les données dans, saisissez le chemin absolu du répertoire dans lequel vous souhaitez que le processus Directory Walker enregistre les données extraites.

  2. Si le flux de données contient déjà au moins un processus intermédiaire ou consommateur, la liste déroulante connecté(e) à s'affiche. Dans cette liste, cliquez sur le processus que vous souhaitez placer après le processus Directory Walker dans le flux de données.

     

    Astuce :
    Dans la plupart des flux de données Directory Walker, le processus Directory Walker précède un processus Conversion de documents.

     

  3. Dans la section Options de lancement, cliquez sur Manuel dans la première liste déroulante. Ceci vous permet de lancer manuellement le processus Directory Walker lorsque vous souhaitez parcourir à nouveau les répertoires spécifiés.

  4. Dans la section Options d'arrêt, cliquez sur le bouton radio Terminer, puis saisissez 99 dans le champ Valeur maximale du code de sortie correcte. Ce champ spécifie la Valeur maximale du code de sortie correcte pour laquelle le serveur d'administration redémarrera automatiquement le processus Directory Walker en présence d'une erreur. Ne modifiez pas la Valeur maximale du code de sortie correcte, sauf si le service d'assistance clientèle d'OpenText vous le demande.

  5. Cliquez sur le bouton Ajouter.
