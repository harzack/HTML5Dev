Pour configurer un processus Directory Walker :

  1. Sur la page Ouvrir le volume des objets système, cliquez sur le lien préfixe_processusDossier de la source de données correspondant au dossier de la source de données qui contient le flux de données auquel vous voulez ajouter un processus Directory Walker.

  2. Cliquez sur le lien préfixe_processus Gestionnaire de flux de données.

  3. Cliquez sur Directory Walker dans le menu Ajouter un élément.

  4. Sur la page Ajouter Directory Walker, saisissez le nom du processus Directory Walker dans le champ Nom.

  5. Vous pouvez décrire le processus Directory Walker dans l'onglet Général de la page Propriétés. Pour cela, saisissez la description dans le champ Description.

  6. Dans la liste déroulante Hôte, cliquez sur le raccourci du serveur d'administration sur l'hôte duquel le processus Directory Walker doit s'exécuter.

     

    Note :
    Vous devez attendre jusqu'à dix minutes après la création d'une nouvelle source de données Directory Walker avant que la nouvelle source de données ne soit répertoriée dans la liste Vues de recherche sur la page Recherche Content Server.
