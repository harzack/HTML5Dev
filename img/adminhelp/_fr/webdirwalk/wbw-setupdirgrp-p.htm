Pour définir les groupes de répertoires :

  1. Dans le champ Répertoires du premier groupe de répertoires, saisissez les chemins absolus des répertoires de niveau supérieur que doit analyser le processus Directory Walker. Séparez les différents chemins par des points-virgules (;) ou faites une nouvelle entrée pour chaque chemin.

  2. Dans le champ Inclure, saisissez la liste des modèles de noms de fichier que doit recueillir le processus Directory Walker. Séparez chaque entrée par un point-virgule (;) (par exemple, *.html; *.txt; *.doc; etc.).

  3. Dans le champ Exclure, saisissez la liste des modèles de noms de fichier que doit ignorer le processus Directory Walker. Séparez chaque entrée par un point-virgule (;) (par exemple, *.jpg; *.exe; *.dll; etc.).

    Si vous ne renseignez pas les champs Inclure et Exclure, le processus Directory Walker collecte tous les fichiers contenus dans les répertoires spécifiés. Si vous définissez un type de fichiers à inclure, le processus Directory Walker exclut automatiquement tous les autres types de fichiers. De la même façon, si vous définissez un type de fichiers à exclure, le processus inclut automatiquement tous les autres types de fichiers. Si vous parcourez un répertoire d'un système d'exploitation dans lequel les noms de fichier sont sensibles à la casse, vous pouvez saisir les modèles de noms de fichier couvrant toutes les combinaisons de casse possibles. Par exemple, *.[hH] [tT] [mM] [lL] comprend *.HTML, *.hTml, etc.

  4. Pour définir le nombre de niveaux de sous-répertoires sous les répertoires de niveau supérieur que le processus Directory Walker doit parcourir, cliquez sur l'une des options suivantes de la liste déroulante Profondeur :

    • Illimité, qui permet au processus Directory Walker d'analyser tous les niveaux de sous-répertoires sous les répertoires de niveau supérieur ;

    • Spécifier, qui vous permet de définir le nombre de niveaux de sous-répertoires analysés par le processus Directory Walker ;

    • aucun(e), qui permet au processus Directory Walker d'analyser uniquement les répertoires de niveau supérieur.

  5. Pour limiter les fichiers parcourus par le processus Directory Walker à ceux compris entre certaines dates, sélectionnez un intervalle de dates dans les listes déroulantes Intervalle de dates.

  6. Pour limiter les fichiers parcourus par le processus Directory Walker à ceux compris entre deux tailles de fichier spécifiques, saisissez l'intervalle de taille des fichiers (en octets) dans les champs Taille du fichier.

  7. Si vous souhaitez définir de nouveaux groupes de répertoires dans la section Groupes de répertoires, répétez les étapes 1 à 6.
