Pour configurer la suppression automatique :

  1. Cliquez sur le lien Configurer la suppression automatique dans la section Administration de la corbeille de la page Administration Content Server.

  2. Sur la page Configurer la suppression automatique, modifiez les paramètres de votre choix, comme décrit dans le tableau suivant.

    Pour... Effectuez les opérations suivantes...
    Activer la planification Cliquez sur le bouton radio Activer en regard de Statut dans la section Suppression automatique.
    Planifier la suppression automatique Spécifiez les jours et heures d'exécution de la suppression automatique dans la section Planification.
    Définissez le nombre de jours pendant lesquels vous voulez conserver les éléments Content Server dans le module Corbeille. Saisissez un chiffre dans le champ Jours d'existence.
    Activer une limite de suppression Cliquez sur le bouton radio Activer en regard de Statut dans la section Limite de suppression.
    Supprimer la limite de suppression Spécifiez le nombre maximum d'éléments racine que vous souhaitez supprimer du module Corbeille dans le champ Nombre d'éléments racine.
     

    Astuce :
    Un élément racine est un objet qui se trouve immédiatement sous les dossiers au niveau des jours.

     

     

    Note :
    Si la suppression automatique ne fonctionne pas conformément au paramétrage, désactivez la Planification de suppression automatique puis redémarrez le service Content Server. Cela empêche toute action ultérieure de suppression automatique.
