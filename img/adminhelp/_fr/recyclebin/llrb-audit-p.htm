Pour configurer l'audit :

  1. Ouvrez la page Administration de Content Server.

  2. Dans la section Administration système, cliquez sur Gérer l'audit des événements.

  3. Cliquez sur Définir les centres d'intérêt à inclure dans l'audit.

  4. Activez les cases Mettre à la corbeille et Restaurer pour générer les informations d'audit relatives à ces types d'événement.

  5. Cliquez sur Définir les centres d'intérêt.
