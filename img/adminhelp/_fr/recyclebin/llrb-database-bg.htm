Les objets de schéma Corbeille sont essentiels au bon fonctionnement du module Corbeille :

  • Si vous installez Corbeille pour la première fois, les objets de schéma doivent exister avant que les éléments Content Server supprimés puissent être stockés dans Corbeille.

  • Si vous mettez à niveau la Corbeille, une mise à niveau des objets de schéma Corbeille peut être nécessaire pour pouvoir stocker les éléments Content Server supprimés dans la Corbeille.

La page Administration de la base de données de la Corbeille vous permet de créer, mettre à jour et supprimer des objets de schéma Corbeille.
