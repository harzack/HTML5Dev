Vous pouvez modifier les paramètres de configuration de la suppression automatique du module Corbeille de Content Server à partir de l'écran de configuration de la suppression automatique.

  • La suppression automatique permet d'activer, de désactiver et de programmer les activités de suppression automatique pour vos types d'objet Content Server. Vous pouvez, par exemple, définir l'exécution de la suppression automatique en dehors des heures de bureau ou toutes les cinq minutes.

  • La limite de suppression permet de spécifier le nombre d'éléments contenus dans le module Corbeille que vous souhaitez supprimer en une seule fois. La valeur par défaut est 1000.
