La page Volume Workflow vous permet d'afficher le contenu du volume du workflow. Vous pouvez également définir des autorisations sur les workflows et les pièces jointes copiées dans le volume du workflow. Lorsqu'un élément est ajouté en tant que pièce jointe de workflow, il hérite des autorisations du dossier Pièces jointes du workflow. Par défaut, il s'agit de toutes les autorisations. Tous les utilisateurs peuvent effectuer une recherche dans les pièces jointes de workflow, même s'ils n'ont pas d'autorisation sur le workflow.

Si l'accès public n'est pas configuré pour un workflow, les utilisateurs ne peuvent pas le lancer car Content Server stocke les pièces jointes ajoutées au dossier Pièces jointes du workflow dans le volume du workflow. La page Volume Workflow vous permet d'assigner des privilèges utilisateur permettant d'afficher et d'ajouter des pièces jointes. De plus, si votre site Content Server prend en charge les domaines Content Server, cette page vous permet de contrôler les domaines qui peuvent lancer le workflow.

Blocage des recherches dans les pièces jointes de workflow par les utilisateurs

Pour empêcher les utilisateurs d'effectuer des recherches dans les pièces jointes de workflow sans changer les autorisations dans le workflow ou les pièces jointes, vous pouvez exclure manuellement le volume du workflow de l'index de recherche Content Server, puis recréer l'index de recherche Content Server.
