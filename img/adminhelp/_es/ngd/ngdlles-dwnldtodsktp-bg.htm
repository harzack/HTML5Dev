The Download to Desktop option is only available for those subtypes that you make available to users. If users select multiple items and one of those items does not support the Download to Desktop feature, the Download to Desktop option will not be available in the context menu.

By default Folders, Compound Documents, Documents, Projects, Collections, Favorites, Email, and Email Folders subtypes can be downloaded to the desktop. The IDs for these subtypes automatically appear in the Include Subtypes box. You can add or remove subtypes as required. For more information about other subtype values, see Node Type Number to Name Mappings. For more information about using the Download to Desktop feature, see OpenText Enterprise Connect - User Getting Started Guide (NGDCORE-UGD).

 

Note:
If users download Compound Emails to their desktops, they will appear as a single email binary.
